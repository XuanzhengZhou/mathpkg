A class of $L$-structures $K$ equipped with a notion of strong substructure $\preceq$ is an \text{abstract elementary class} (AEC) if:
\begin{itemize}
\item $K$ and $\preceq$ are closed under isomorphism.
\item If $A \preceq B$, then $A \subseteq B$.
\item $\preceq$ is a partial order.
\item (Coherence) If $A_0 \subseteq A_1 \subseteq A_2$, $A_0 \preceq A_1$, $A_1 \preceq A_2$, then $A_0 \preceq A_2$ iff $A_0 \preceq A_1$.
\item (Tarski-Vaught chains) The union of a $\preceq$-increasing chain is a strong extension of each member.
\item (L\"owenheim-Skolem) There exists a cardinal $\operatorname{LS}(K)$ such that every structure in $K$ has a strong substructure of cardinality $\leq \operatorname{LS}(K)$.
\end{itemize}
