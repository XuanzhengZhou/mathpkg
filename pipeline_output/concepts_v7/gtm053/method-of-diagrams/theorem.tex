For an $L$-structure $A$, let $L_A = L \cup \{c_a : a \in A\}$ be the language expansion and $A_A$ the natural expansion. Define:
\begin{itemize}
\item $\operatorname{Diag}(A) = \{S : \text{atomic or negated atomic } L_A\text{-sentence such that } A_A \vDash S\}$ (the \text{diagram}).
\item $\operatorname{CDiag}(A) = \{S : L_A\text{-sentence such that } A_A \vDash S\}$ (the \text{complete diagram}).
\end{itemize}
For an $L$-structure $B$:
\begin{enumerate}
\item There is an expansion $B_A$ to $L_A$ with $B_A \vDash \operatorname{Diag}(A)$ iff $A \subseteq B$.
\item There is an expansion $B_A$ to $L_A$ with $B_A \vDash \operatorname{CDiag}(A)$ iff $A \preceq B$.
\end{enumerate}
