Given a saturated model $\mathbf{A}$ of a complete theory $T$, the following are equivalent:
\begin{enumerate}
\item For any two $n$-tuples $\bar{a}, \bar{b}$ in $\mathbf{A}$,
$$\operatorname{qftp}(\bar{a}) = \operatorname{qftp}(\bar{b}) \text{ iff there exists } \pi \in \operatorname{Aut}(\mathbf{A}) \text{ such that } \pi : \bar{a} \mapsto \bar{b}.$$
\item Every $L$-formula with $n$ free variables is equivalent (modulo $T$) to a quantifier-free $L$-formula.
\end{enumerate}
Here $\operatorname{qftp}_A(\bar{a}) := \{Q(\bar{x}) : \text{quantifier-free}, A \vDash Q(\bar{a})\}$ denotes the quantifier-free type.
