The \text{constructible universe} $L$ is the class of sets obtained by transfinite iteration of basic Godel operations, starting from $\varnothing$. Formally, $L = \bigcup_{\alpha \in \mathrm{Ord}} L_\alpha$, where:
\begin{itemize}
\item $L_0 = \varnothing$,
\item $L_{\alpha+1} = \operatorname{Def}(L_\alpha)$ (sets definable with parameters in $L_\alpha$),
\item $L_\lambda = \bigcup_{\alpha < \lambda} L_\alpha$ for limit $\lambda$.
\end{itemize}
All ZFC axioms hold in $L$, and CH (indeed GCH) holds in $L$. The axiom "$V = L$" (every set is constructible) is independent of ZFC.
