For an $L$-structure $A$ and an $L$-formula $P(x_1, \ldots, x_n)$, the \text{definable set} determined by $P$ is
$$P(A) = \{ \bar{a} \in A^n : A \vDash P(\bar{a}) \}.$$
Sets of this form are called definable. Any subset of $A^n$ can be viewed as an $n$-ary relation; $P(\bar{v})$ then determines an $L$-definable relation. If $P(A)$ coincides with the graph of an operation $f : A^{n-1} \rightarrow A$, then $f$ is called an $L$-definable operation.
