Cohen's \text{forcing} method constructs a \text{generic extension} $V[G]$ of the universe $V$ (or a countable transitive model $M$) by adding a \text{generic filter} $G$ over a \text{forcing poset} $\mathbb{P}$. Key elements:
\begin{itemize}
\item A \text{forcing notion} $\mathbb{P} = (P, \leq)$ is a partially ordered set with a maximum element $1$.
\item A filter $G \subseteq \mathbb{P}$ is \text{generic} over $M$ if $G$ intersects every dense subset of $\mathbb{P}$ that lies in $M$.
\item The \text{forcing relation} $p \Vdash \phi$ ("$p$ forces $\phi$") is defined in $M$ and determines truth in $M[G]$.
\end{itemize}
To make CH false, one adds $\aleph_2$ Cohen reals to a model of GCH, resulting in $2^{\aleph_0} \geq \aleph_2$.
