The theory $\mathrm{RCF}$ of real closed fields is the theory of the field of real numbers in the language $L_1\mathrm{Ar}$ with an ordering $<$ (definable as $\exists z (z \neq 0 \land x + z^2 = y)$). It includes:
\begin{enumerate}
\item Axioms of ordered fields.
\item The intermediate value property for polynomials: for every polynomial $f(x)$ over the field, if $f(a) \cdot f(b) < 0$ for some $a < b$, then there exists $c$ with $a < c < b$ and $f(c) = 0$.
\item Every positive element has a square root.
\item Every polynomial of odd degree has a root.
\end{enumerate}
