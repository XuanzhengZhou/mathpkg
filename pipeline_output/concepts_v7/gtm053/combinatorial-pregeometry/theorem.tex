A \text{combinatorial pregeometry} $(M, \operatorname{cl})$ consists of a set $M$ and a finitary closure operator $\operatorname{cl} : \mathcal{P}(M) \to \mathcal{P}(M)$ satisfying:
\begin{enumerate}
\item[(i)] $U \subseteq \operatorname{cl}(U)$ and $\operatorname{cl}(\operatorname{cl}(U)) = \operatorname{cl}(U)$.
\item[(ii)] (Exchange property) If $v \in \operatorname{cl}(U \cup \{w\}) \setminus \operatorname{cl}(U)$, then $w \in \operatorname{cl}(U \cup \{v\})$.
\item[(iii)] (Finite character) $\operatorname{cl}(U) = \bigcup \{\operatorname{cl}(U') : U' \subseteq U \text{ finite}\}$.
\end{enumerate}
A \text{combinatorial geometry} is a pregeometry where $\operatorname{cl}(\{u\}) = \{u\}$ for all $u \in M$. Given a pregeometry, the associated geometry is $M \setminus \operatorname{cl}(\varnothing)$ factored by the equivalence $u \sim v \Leftrightarrow \operatorname{cl}(u) = \operatorname{cl}(v)$.
