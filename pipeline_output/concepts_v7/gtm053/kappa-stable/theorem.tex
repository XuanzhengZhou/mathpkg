A complete theory $T$ is \text{$\kappa$-stable} if for every set $C$ of cardinality $\kappa$, the expanded theory $T_C$ has at most $\kappa$ complete $1$-types. $T$ is \text{stable} if it is $\kappa$-stable for some infinite $\kappa$. $T$ is \text{$\aleph_0$-stable} (or $\omega$-stable) if it is $\aleph_0$-stable; this is the strongest stability condition and implies $\kappa$-stability for all $\kappa$. $T$ is \text{superstable} if it is $\kappa$-stable for all $\kappa \geq 2^{\operatorname{card} T}$.
