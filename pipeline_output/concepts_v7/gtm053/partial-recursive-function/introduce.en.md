Partial recursive functions are the standard formalization of computable functions. The $\mu$-operator (unbounded search) distinguishes recursive functions from primitive recursive ones. Kleene's normal form theorem shows every partial recursive function can be expressed using a single application of the $\mu$-operator to a primitive recursive predicate. The class of recursive sets (those with recursive characteristic functions) equals the class of algorithmically decidable sets.
