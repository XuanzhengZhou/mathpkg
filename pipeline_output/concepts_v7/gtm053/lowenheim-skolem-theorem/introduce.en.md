The Löwenheim-Skolem theorem reveals a fundamental limitation of first-order logic: it cannot control cardinality. Any theory with an infinite model has models of every infinite cardinality (upward via compactness, downward via Skolem functions). This "Skolem paradox" shows that first-order set theory, if consistent, has a countable model despite proving the existence of uncountable sets.
