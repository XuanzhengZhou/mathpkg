A Godelian set formalizes the set of all true sentences under a given interpretation. Any interpretation of a first-order language determines a Godelian set, and conversely, Godel's completeness theorem states that every Godelian set arises as the set of true formulas under some interpretation. The completeness theorem thus establishes the equivalence of semantic truth and syntactic consistency.
