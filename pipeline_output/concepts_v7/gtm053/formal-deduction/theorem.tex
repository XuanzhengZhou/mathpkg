A \text{deduction} of a formula $P$ from a set of formulas $\mathcal{E}$ in a language $L$ of class $\mathfrak{L}_1$ is a finite sequence of formulas $P_1, \ldots, P_n = P$ such that for each $i$, one of the following holds:
\begin{itemize}
\item $P_i$ is an axiom of $L$ (logical axiom or one of the special axioms),
\item $P_i \in \mathcal{E}$,
\item $P_i$ is obtained from earlier formulas in the sequence by a rule of deduction (Modus Ponens or Generalization).
\end{itemize}
We write $\mathcal{E} \vdash P$ if such a deduction exists.
