A structure $M$ (or a definable set in an ambient structure) is called \text{minimal} if $\operatorname{rk} M = 1$ and for every partition $M = S_1 \dot{\cup} S_2$ into definable subsets, $\operatorname{rk} S_1 = 0$ or $\operatorname{rk} S_2 = 0$. $M$ is \text{strongly minimal} if every $M' \equiv M$ is minimal. The theory of a strongly minimal structure is uncountably categorical.
