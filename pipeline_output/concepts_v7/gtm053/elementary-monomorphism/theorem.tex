A partial map $h: A \rightarrow B$ with $\operatorname{dom} h = X \subset A$ and $\operatorname{range} h = Y \subset B$ is called an \text{elementary monomorphism} if for any $a_1, \ldots, a_n \in X$ and any $L$-formula $P$,
$$A \vDash P(a_1, \ldots, a_n) \text{ iff } B \vDash P(h(a_1), \ldots, h(a_n)).$$
