The Peano axioms formalize arithmetic in first-order logic. The induction schema (an infinite set of axioms, one per formula) replaces the second-order induction principle. First-order Peano arithmetic is incomplete (Godel's first incompleteness theorem), in contrast to the second-order version, which is categorical but has no complete deductive system.
