A predimension is a counting function used in Hrushovski's method for constructing new stable structures via amalgamation. The intuition is that each new function or relation should impose a predictable number of algebraic dependencies. The Hrushovski inequality $\delta(X) \geq 0$ for all finite $X$ defines a class of structures, from which an existentially closed (algebraically closed) model is taken.
