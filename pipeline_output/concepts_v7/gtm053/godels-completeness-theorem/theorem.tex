Let $L$ be a first-order language.
\begin{enumerate}
\item[(a)] Any Gödelian set $T$ of $L$-formulas is the set of $\phi$-true formulas $T_\phi L$ for some interpretation $\phi$ of $L$ in a set $M$ with $\operatorname{card} M \leq \operatorname{card}(\text{alphabet of }L) + \aleph_0$.
\item[(b)] Any set of formulas $\mathcal{E}$ containing $\mathrm{Ax}\,L$ that is consistent can be embedded in a Gödelian set.
\end{enumerate}
