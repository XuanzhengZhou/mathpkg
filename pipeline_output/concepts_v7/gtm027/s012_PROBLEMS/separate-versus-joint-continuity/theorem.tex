A function $f(x, y)$ on a product space $X \times Y$ is said to be \emph{continuous in $x$} if for each fixed $y \in Y$, the function $x \mapsto f(x, y)$ is continuous. Similarly, $f(x, y)$ is \emph{continuous in $y$} if for each fixed $x \in X$, the function $y \mapsto f(x, y)$ is continuous.

If $f$ is continuous on the product space $X \times Y$ (jointly continuous), then $f(x, y)$ is continuous in $x$ and continuous in $y$ separately. However, the converse is false. A classical counterexample is the real-valued function on $\mathbb{R}^2$ defined by
\[
f(x, y) = \begin{cases}
\dfrac{xy}{x^2 + y^2}, & (x, y) \neq (0, 0), \\[6pt]
0, & (x, y) = (0, 0).
\end{cases}
\]
This function is continuous in each variable separately but is not jointly continuous at the origin.
