Let $f$ and $g$ be real-valued continuous functions on a topological space $X$ (with the usual topology on $\mathbb{R}$), and let $a \in \mathbb{R}$ be a fixed real number. Then:

\begin{itemize}
\item[(a)] The function $a f$, defined by $(a f)(x) = a f(x)$, is continuous.
\item[(b)] The function $|f|$, defined by $|f|(x) = |f(x)|$, is continuous.
\item[(c)] The map $F(x) = (f(x), g(x))$ from $X$ to $\mathbb{R}^2$ is continuous with respect to the usual topology on the Euclidean plane.
\item[(d)] The functions $f + g$, $f - g$, and $f \cdot g$ are continuous. If $g(x) \neq 0$ for all $x \in X$, then $f / g$ is also continuous.
\item[(e)] The functions $\max[f, g] = \frac{(f+g) + |f-g|}{2}$ and $\min[f, g] = \frac{(f+g) - |f-g|}{2}$ are continuous.
\end{itemize}
