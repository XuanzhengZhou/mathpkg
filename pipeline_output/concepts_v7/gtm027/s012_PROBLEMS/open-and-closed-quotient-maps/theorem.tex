A continuous surjection $f: X \to Y$ is a \textbf{quotient map} if and only if $f^{-1}[U]$ is open in $X$ implies $U$ is open in $Y$. The map $f$ is an \textbf{open} (resp. \textbf{closed}) quotient map if it maps open (resp. closed) sets to open (resp. closed) sets. Every open or closed continuous surjection is a quotient map, but the converse fails.
