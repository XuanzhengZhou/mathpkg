This proposition classifies the homeomorphism types of intervals in the real line with the usual topology. Open, closed, and half-open intervals each form a single homeomorphism class, but no two of these three classes are homeomorphic to each other. This is a standard exercise in elementary topology that illustrates how topological properties like compactness distinguish different interval types.
