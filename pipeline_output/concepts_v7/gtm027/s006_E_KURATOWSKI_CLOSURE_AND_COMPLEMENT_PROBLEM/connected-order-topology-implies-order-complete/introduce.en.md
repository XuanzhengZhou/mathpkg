In a linearly ordered set endowed with the order topology, connectedness forces order-completeness. If the space is connected in the topological sense, then every nonempty subset that is bounded above must have a supremum. This is part of a characterization: for order topologies, connectedness is equivalent to the combination of order-completeness and the absence of gaps.
