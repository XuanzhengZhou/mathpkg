A topological space $X$ is a \textbf{$T_0$-space} (or Kolmogorov space) if and only if for each pair $x$ and $y$ of distinct points, there is a neighborhood of one point to which the other does not belong. Equivalently, $X$ is a $T_0$-space if and only if for distinct points $x$ and $y$, either $x \notin \{y\}^-$ or $y \notin \{x\}^-$.
