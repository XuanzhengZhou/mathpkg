This theorem shows that adjoining all accumulation points to a set always yields a closed set. It is the key lemma needed to prove that the closure of a set equals the union of the set with its accumulation points. The proof uses the fact that any point outside this union has a neighborhood disjoint from $A$, establishing openness of the complement.
