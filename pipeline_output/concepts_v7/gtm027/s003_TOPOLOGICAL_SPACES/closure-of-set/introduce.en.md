The closure of a set is the intersection of all closed supersets—equivalently, the smallest closed set containing it. It is a fundamental operation in topology: a set is closed precisely when it equals its own closure. The closure can also be characterized as the union of the set with its accumulation points, linking the algebraic closure axioms with the geometric notion of limit points.
