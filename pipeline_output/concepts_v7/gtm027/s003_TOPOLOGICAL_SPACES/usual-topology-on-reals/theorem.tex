The \textbf{usual topology} for the set of real numbers $\mathbb{R}$ is the family of all subsets $A \subset \mathbb{R}$ such that for each $x \in A$ there exist numbers $a, b$ with $a < x < b$ and the open interval $\{y : a < y < b\} \subset A$. That is, a set is open iff it contains an open interval about each of its points.
