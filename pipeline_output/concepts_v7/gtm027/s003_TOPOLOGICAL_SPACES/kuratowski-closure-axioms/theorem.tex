A \textbf{closure operator} on a set $X$ is an operator assigning to each subset $A \subset X$ a subset $A^{c} \subset X$ satisfying the following four \textbf{Kuratowski closure axioms}:

\begin{enumerate}
\item[(a)] If $\emptyset$ is the void set, $\emptyset^{c} = \emptyset$.
\item[(b)] For each $A$, $A \subset A^{c}$.
\item[(c)] For each $A$, $A^{cc} = A^{c}$ (idempotence).
\item[(d)] For each $A$ and $B$, $(A \cup B)^{c} = A^{c} \cup B^{c}$ (preserves finite unions).
\end{enumerate}
