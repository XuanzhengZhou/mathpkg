Continuous convergence is a mode of convergence for nets of continuous functions that is stronger than pointwise convergence. A net of functions converges continuously to a limit function if the evaluation map, considered jointly in the function index and the point argument, respects convergence: whenever points converge to $x$, the corresponding function values converge to the limit function's value at $x$, uniformly in the function index.
