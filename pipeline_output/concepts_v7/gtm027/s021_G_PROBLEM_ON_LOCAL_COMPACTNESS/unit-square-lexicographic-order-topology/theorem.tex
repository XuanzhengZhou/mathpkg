Let $X = [0,1] \times [0,1]$ be the unit square with the dictionary (lexicographic) order: $(a,b) < (c,d)$ if and only if $a < c$ or $a = c$ and $b < d$. With the order topology induced by this ordering, $X$ is compact, connected, and Hausdorff. It satisfies the first axiom of countability but is not separable, and hence is not metrizable.
