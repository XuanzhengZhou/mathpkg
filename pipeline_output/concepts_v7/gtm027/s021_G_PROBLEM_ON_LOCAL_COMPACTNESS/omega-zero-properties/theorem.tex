Let $\Omega$ be the first uncountable ordinal and let $\Omega_0$ be the set of all ordinal numbers strictly less than $\Omega$, equipped with the order topology. Then $\Omega_0$ is:

\begin{enumerate}
\item locally compact,
\item Hausdorff,
\item satisfies the first axiom of countability,
\item sequentially compact,
\end{enumerate}

but $\Omega_0$ is not compact.
