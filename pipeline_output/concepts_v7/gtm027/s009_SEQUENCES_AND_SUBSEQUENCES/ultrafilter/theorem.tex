A filter $\mathcal{F}$ in a set $X$ is an ultrafilter if and only if it is properly contained in no filter in $X$. If $\mathcal{F}$ is an ultrafilter in $X$ and $A \cup B \in \mathcal{F}$, then either $A \in \mathcal{F}$ or $B \in \mathcal{F}$. In particular, for every subset $A$ of $X$, either $A \in \mathcal{F}$ or $X \setminus A \in \mathcal{F}$.
