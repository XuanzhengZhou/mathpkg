A convergence class is an axiomatic specification of which nets converge to which points in a set $X$, characterized by four conditions: constant nets converge, subnets preserve convergence, non-convergence is witnessed by a subnet with no convergent subnet, and iterated limits compose properly. Kelley showed that convergence classes are in one-to-one correspondence with topologies on $X$, providing a purely net-theoretic foundation for topological convergence.
