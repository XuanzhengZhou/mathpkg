Let $e$ be a convergence class for a set $X$. For each subset $A$ of $X$, let $A^c$ be the set of all points $s$ such that, for some net in $A$, the net converges $(e)$ to $s$. Then the operator $c$ is a Kuratowski closure operator on $X$, and $A^{cc} = A^c$ for all $A \subset X$. Consequently, the topology $\mathfrak{J}$ associated with the operator $c$ has the property that a net converges relative to $\mathfrak{J}$ if and only if it converges $(e)$. This establishes a one-to-one correspondence between topologies for $X$ and convergence classes on $X$.
