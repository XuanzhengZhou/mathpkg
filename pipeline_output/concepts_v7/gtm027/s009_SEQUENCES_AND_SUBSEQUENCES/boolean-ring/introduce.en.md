A Boolean ring is a ring with unit in which every element is idempotent ($r^2 = r$). Boolean rings are algebraically equivalent to Boolean algebras: the natural partial order $r \geq s \iff rs = s$ yields join $r \vee s = r + s + rs$ and meet $r \wedge s = rs$. The prototypical example is the power set of any set under symmetric difference and intersection, establishing the connection between algebraic and set-theoretic Boolean structures.
