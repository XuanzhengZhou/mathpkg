Let $R$ be an equivalence relation on a set $X$. For a subset $A \subseteq X$, the \text{image of $A$ under $R$} is
$$R[A] = \{\, y \in X : (x,y) \in R \text{ for some } x \in A \,\} = \bigcup \{\, D \in X/R : D \cap A \neq \varnothing \,\}.$$
For a point $x \in X$, we write $R[x] = R[\{x\}]$, which is the equivalence class containing $x$.
