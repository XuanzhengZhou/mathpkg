A real number $x$ is a \textbf{non-negative integer} iff $x$ belongs to every inductive set. The set $\omega$ of non-negative integers is defined to be the intersection of the members of the family of all inductive sets. Each member of $\omega$ is actually non-negative because the set of all non-negative numbers is inductive. The set $\omega$ is itself an inductive set and is a subset of every other inductive set.
