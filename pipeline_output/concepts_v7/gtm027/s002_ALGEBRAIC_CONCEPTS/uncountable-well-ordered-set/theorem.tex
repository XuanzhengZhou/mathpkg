\textbf{Summary 22.} There is an uncountable set $\Omega'$, which is linearly ordered by a relation $<$ in such a way that:
\begin{enumerate}
\item[(a)] Every non-void subset of $\Omega'$ (relative to the order $<$) has a first element (i.e., $\Omega'$ is well-ordered);
\item[(b)] $\Omega'$ has a greatest element, denoted $\Omega$;
\item[(c)] For each $x \in \Omega'$ with $x \neq \Omega$, the set $\{y : y \in \Omega' \text{ and } y < x\}$ is countable;
\item[(d)] $\Omega$ has uncountably many predecessors; that is, $\{y : y \in \Omega' \text{ and } y < \Omega\}$ is uncountable.
\end{enumerate}
