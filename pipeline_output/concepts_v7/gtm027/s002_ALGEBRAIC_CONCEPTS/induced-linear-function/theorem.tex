Suppose $f$ is a linear function on $X$ to $Y$ and $g$ is a linear map of $X$ onto $Z$ such that the null space of $f$ contains the null space of $g$. Then there is a unique linear function $h$ on $Z$ to $Y$ such that $f = h \circ g$ (explicitly, $h(z)$ is the unique member of $f \circ g^{-1}[z]$). The function $h$ is said to be \textbf{induced} by $f$ and $g$.

A particular consequence is that each linear function may be written as a projection into a quotient space followed by a one-to-one linear function.
