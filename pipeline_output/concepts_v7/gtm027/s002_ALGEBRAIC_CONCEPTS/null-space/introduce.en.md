The null space (or kernel) of a linear function is the set of all vectors mapped to zero. It is always a subspace of the domain. The null space measures injectivity: a linear function is one-to-one exactly when its null space is trivial. This is the linear-algebraic analogue of the kernel of a group homomorphism.
