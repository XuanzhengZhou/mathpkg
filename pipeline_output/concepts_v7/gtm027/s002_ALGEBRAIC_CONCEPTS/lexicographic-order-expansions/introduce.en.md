The set $E$ of all $b$-adic expansions is linearly ordered by the lexicographic (dictionary) order: compare digits left-to-right and the first differing digit determines the order. This order is compatible with the usual order on real numbers under the correspondence $r$, which is order-preserving. Kelley uses this to prove Theorem 14.
