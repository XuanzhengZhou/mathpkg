Suppose that for each member $a$ of an index set $A$ there is given a set $X_a$. The \textbf{Cartesian product} of the family $\{X_a : a \in A\}$ is the set of all functions $x$ on $A$ such that $x_a \in X_a$ for each $a \in A$. The value $x_a$ is called the \textbf{$a$-th coordinate} of the point $x$ of the product.
