For each non-negative integer $p$ let $\omega_p = \{q : q \in \omega \text{ and } q \leq p\}$.

\textbf{Theorem 13.} Suppose $a$ is given and $F(g)$ is given whenever $g$ is a function whose domain is of the form $\omega_p$ for some $p \in \omega$. Then there is a unique function $f$ on $\omega$ such that
$$f(0) = a \quad \text{and} \quad f(p+1) = F(f \mid \omega_p)$$
for each $p \in \omega$. (Here $f \mid \omega_p$ is the function $f$ restricted to the set $\omega_p$.)
