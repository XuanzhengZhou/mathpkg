Let $\{U_n\}_{n \in \omega}$ be a sequence of subsets of $X \times X$ such that $U_0 = X \times X$, each $U_n$ contains the diagonal $\Delta$, and $U_{n+1} \circ U_{n+1} \circ U_{n+1} \subset U_n$ for each $n$. Then there exists a non-negative real-valued function $d$ on $X \times X$ such that:
\begin{enumerate}
\item[(a)] $d(x,y) + d(y,z) \geq d(x,z)$ for all $x, y, z \in X$;
\item[(b)] $U_n \subset \{(x,y) : d(x,y) < 2^{-n}\} \subset U_{n-1}$ for each positive integer $n$.
\end{enumerate}
If each $U_n$ is symmetric, then $d$ can be chosen to be a pseudo-metric satisfying condition (b).
