When B is a base (or subbase) for a uniformity on X, the sections U[x] for U in B form a base (or subbase) for the neighborhood system of each point x in the uniform topology. This directly generalizes the metric space fact that open balls about a point form a neighborhood base, since the sets V_{d,r}[x] are precisely the open r-balls centered at x.
