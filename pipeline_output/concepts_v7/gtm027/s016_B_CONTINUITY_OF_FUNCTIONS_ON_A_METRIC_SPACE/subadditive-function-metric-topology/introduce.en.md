This theorem provides a method for constructing new metrics from existing ones using subadditive functions. Given a suitable subadditive function $f$, the composition $e = f \circ d$ yields a new metric on the same set with the identical metric topology. The standard example $f(x) = x/(1+x)$ produces a bounded metric from any given metric, a technique frequently used to show that every metric space is topologically equivalent to a metric space of finite diameter.
