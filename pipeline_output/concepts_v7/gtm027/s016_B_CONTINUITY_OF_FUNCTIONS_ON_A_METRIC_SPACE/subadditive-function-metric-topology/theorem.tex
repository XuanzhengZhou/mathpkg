Let $f$ be a continuous real-valued function defined on the set of all non-negative real numbers, such that:
\begin{enumerate}
\item $f(x) = 0$ if and only if $x = 0$,
\item $f$ is non-decreasing,
\item $f(x + y) \leq f(x) + f(y)$ for all non-negative numbers $x$ and $y$ (i.e., $f$ is subadditive).
\end{enumerate}
If $(X, d)$ is a metric space and $e(x, y) = f(d(x, y))$, then $(X, e)$ is a metric space, and the metric topology of $(X, e)$ is identical with that of $(X, d)$.

A particular case which occurs frequently in the literature: $f(x) = x / (1 + x)$.
