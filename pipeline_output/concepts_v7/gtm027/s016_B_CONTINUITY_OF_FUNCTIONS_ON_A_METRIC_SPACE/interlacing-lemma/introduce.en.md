The interlacing lemma concerns two sequences in the ordinal space $\Omega_0$ (the set of countable ordinals with the order topology) satisfying the interlacing condition $x_n \leq y_n \leq x_{n+1}$. Under this condition, both sequences converge to the same limit. This lemma is a key technical tool for analyzing the topological properties of ordinal spaces, particularly for establishing normality and studying accumulation points.
