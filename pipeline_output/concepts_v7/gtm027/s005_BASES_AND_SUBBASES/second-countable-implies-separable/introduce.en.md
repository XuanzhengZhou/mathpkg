Theorem 14 establishes that every second countable space is separable. The proof is constructive: choose one point from each member of a countable base to obtain a countable set $A$. The complement of the closure of $A$ is an open set containing no non-void base member, hence must be empty, so $A$ is dense. The converse is false in general, as uncountable discrete spaces are separable (via themselves) but not second countable.
