Theorem 22 collects the fundamental properties of connected components. Every connected set is contained in a unique component. Each component is always closed (because its closure remains connected). Distinct components are necessarily separated. This provides a canonical decomposition of any topological space into maximal connected, pairwise separated, closed subsets. The converse -- that points always lying in the same half of any separation must belong to the same component -- is false in general.
