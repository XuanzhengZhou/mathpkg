\text{Let } \mathfrak{u} \text{ be a uniformity for } X \text{, and for each member } U \text{ of } \mathfrak{u} \text{ and each } x \in X \text{ define } V_U(x) = U[x] \text{. Then } \mathfrak{u} \text{ is directed by } \subset \text{ and } (V, \subset) \text{ is a uniform neighborhood system for } X \text{ whose associated uniformity coincides with } \mathfrak{u}.
