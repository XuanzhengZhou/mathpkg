\text{Let } (X, \mathfrak{u}) \text{ be a uniform space such that for each member } U \text{ of some subbase for } \mathfrak{u} \text{ there is a finite cover } A_1, \cdots, A_n \text{ of } X \text{ such that } A_i \times A_i \subset U \text{ for each } i. \text{ Then the space } (X, \mathfrak{u}) \text{ is totally bounded.}
