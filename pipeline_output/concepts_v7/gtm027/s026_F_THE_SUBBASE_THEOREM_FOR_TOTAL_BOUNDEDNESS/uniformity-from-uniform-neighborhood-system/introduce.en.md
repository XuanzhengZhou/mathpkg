# Uniformity from a Uniform Neighborhood System

This proposition shows how to construct a uniformity from a uniform neighborhood system. Given a uniform neighborhood system $(V, \geq)$ on a set $X$, the sets of the form $\{(x,y) : y \in V_a(x)\}$ for indices $a$ in the directed set form a base for a uniformity on $X$. The resulting uniformity is tightly coupled with the original neighborhood system: the entourage neighborhoods $U[x]$ and the system neighborhoods $V_a(x)$ mutually approximate each other.
