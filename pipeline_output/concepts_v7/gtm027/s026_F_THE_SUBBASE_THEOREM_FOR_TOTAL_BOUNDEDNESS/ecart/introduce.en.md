# Écart

An écart (French for "gap" or "deviation") is Kelley's term for what is now commonly called a pseudometric: a distance function on a set satisfying all the axioms of a metric except the separation axiom — distinct points may be at distance zero. Every écart defines a uniformity on the set via the entourages $\{(x,y) : e(x,y) < \varepsilon\}$. The gage of a uniformity is precisely the family of all écarts that are uniformly continuous on the product, providing a metric characterization of uniform spaces.
