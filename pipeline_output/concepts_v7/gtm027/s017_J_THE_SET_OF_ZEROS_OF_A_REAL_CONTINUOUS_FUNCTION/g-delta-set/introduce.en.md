A $G_\delta$ set is a countable intersection of open sets in a topological space. This concept is fundamental in descriptive set theory and general topology, providing a measure of complexity for subsets beyond the open/closed dichotomy. In metric spaces, every closed set is a $G_\delta$, but in general topological spaces this property characterizes a special class of normal spaces. The notation $G_\delta$ originates from the German "Gebiet" (region) for open sets and "Durchschnitt" (intersection), with $\delta$ denoting countable intersection.
