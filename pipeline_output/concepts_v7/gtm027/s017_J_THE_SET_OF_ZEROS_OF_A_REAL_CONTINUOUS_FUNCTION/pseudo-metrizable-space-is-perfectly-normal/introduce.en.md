Every pseudo-metrizable space is both normal and has the property that every closed set is a $G_\delta$, thus is perfectly normal. In a pseudo-metric space, a closed set $C$ can be written as the countable intersection $\bigcap_{n} \{x : d(x, C) < 1/n\}$ of open neighborhoods, making it a $G_\delta$. Normality of pseudo-metric spaces follows from the standard Urysohn function $f(x) = d(x, A) / (d(x, A) + d(x, B))$. This theorem places perfectly normal spaces as a generalization of metrizable spaces.
