In a normal topological space, every closed $G_\delta$ set can be realized as the zero set of some continuous real-valued function. Combined with the forward direction, this establishes that the zero sets of continuous real-valued functions on a normal space are precisely the closed $G_\delta$ sets. The proof relies on Urysohn's lemma applied countably many times: writing $A = \bigcap_{n} U_n$ with $U_n$ open, one constructs functions $f_n$ vanishing on $A$ and equaling $1$ outside $U_n$, then sums them into a single function $f = \sum 2^{-n} f_n$ whose zero set is exactly $A$.
