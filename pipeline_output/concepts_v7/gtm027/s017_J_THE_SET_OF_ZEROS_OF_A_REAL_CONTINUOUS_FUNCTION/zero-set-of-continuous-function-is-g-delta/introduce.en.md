Every continuous real-valued function on a topological space has a zero set that is a $G_\delta$. This follows from the fact that $\{0\}$ is a $G_\delta$ in the real line and the preimage of a $G_\delta$ under a continuous function is again a $G_\delta$. This result is the forward direction of the correspondence between $G_\delta$ sets and zero sets of continuous functions, which is fully realized in normal spaces.
