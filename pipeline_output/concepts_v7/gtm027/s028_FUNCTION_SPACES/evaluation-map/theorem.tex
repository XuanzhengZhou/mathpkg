Let $F$ be a family of functions, each on a set $X$ to a topological space $Y$. For each point $x \in X$, the \textbf{evaluation at $x$} (or the \textbf{projection into the $x$-th coordinate space}) is the function
$$e_x : F \to Y$$
defined by
$$e_x(f) = f(x)$$
for all $f \in F$.

Relative to the topology $\wp$ of pointwise convergence on $F$, the evaluation map $e_x$ is both continuous and open. Moreover, $\wp$ is the smallest (coarsest) topology on $F$ such that each evaluation $e_x$ is continuous. A function $g$ from a topological space $Z$ to $F$ is continuous relative to $\wp$ if and only if the composition $e_x \circ g$ is continuous for each $x \in X$.
