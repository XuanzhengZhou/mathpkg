Let $f$ be a function. The \textbf{domain} of $f$ is the set of all first coordinates of members of $f$. The \textbf{range} (or set of values) of $f$ is the set of all second coordinates. A function $f$ is \textbf{on} $X$ if and only if $X$ is its domain. It is \textbf{onto} $Y$ if and only if $Y$ is its range. If the range of $f$ is a subset of $Y$, then $f$ is \textbf{to} $Y$ (or \textbf{into} $Y$).
