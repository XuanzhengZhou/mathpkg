The domain and range are fundamental attributes of a function. A function is said to be "on" a set if that set is its domain, "onto" a set if that set equals its range, and "into" (or "to") a set if its range is a subset of that set. This terminology is standard throughout Kelley's text and distinguishes the set-theoretic function from its target specification. A function can be onto at most one set (its unique range) but into many different supersets.
