The notation $R[A]$ generalizes the function image $f(A)$ to arbitrary relations. It collects all elements that are $R$-related to some element of $A$. Kelley uses square brackets to distinguish this from function application. This notation is essential for expressing continuity (inverse images of open sets) and for working with net convergence.
