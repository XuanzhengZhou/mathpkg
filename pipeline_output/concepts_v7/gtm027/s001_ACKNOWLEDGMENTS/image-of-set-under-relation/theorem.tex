If $R$ is a relation and $A$ is a set, then $R[A]$ is the set of all $R$-relatives of points of $A$:
$$
R[A] = \{y: xRy \text{ for some } x \in A\}.
$$
If $A$ is the domain of $R$, then $R[A]$ is the range of $R$. For arbitrary $A$, $R[A]$ is contained in the range of $R$. If $R \subset S$, then $R[A] \subset S[A]$ for every $A$.
