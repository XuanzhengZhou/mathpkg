Kelley defines an ordering with striking simplicity: it is merely a transitive relation. This minimalist definition encompasses quasi-orderings (preorders) and partial orders alike. A relation orders a set if it is transitive on that set. While this definition is very general -- even $X \times X$ qualifies as an ordering -- more interesting orderings satisfy additional conditions like antisymmetry. This broad definition allows unified treatment of various order-theoretic concepts throughout the text.
