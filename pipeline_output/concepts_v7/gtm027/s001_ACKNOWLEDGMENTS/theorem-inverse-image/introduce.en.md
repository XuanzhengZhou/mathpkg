Theorem 7 establishes that the inverse image operation $f^{-1}[\cdot]$ preserves all basic set-theoretic operations: relative complement, union, and intersection, for both binary and arbitrary families. This is a crucial property used throughout topology -- for example, the inverse image of an open set under a continuous function is open. Unlike the forward image, which only preserves unions, the inverse image behaves perfectly with respect to all Boolean operations, making it particularly well-behaved.
