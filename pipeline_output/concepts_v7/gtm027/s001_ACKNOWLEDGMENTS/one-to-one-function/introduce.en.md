A one-to-one function (injection) maps distinct elements of the domain to distinct elements of the range. Equivalently, the inverse relation is itself a function. One-to-one functions are essential for comparing sizes of sets (cardinality) via equipollence and for constructing inverse functions. In Kelley's set-theoretic framework, this property is expressed concisely: $f^{-1}$ is a function exactly when $f$ is injective.
