For each $c$ in $A$, the projection $P_c$ of the product space $\bigwedge \{X_a : a \in A\}$ into the $c$-th coordinate space $X_c$ is an open mapping. That is, if $V$ is an open subset of the product space, then $P_c[V]$ is an open subset of $X_c$.

In particular, if $V = \{y : y_a \in U_a \text{ for } a \text{ in } F\}$ is a basic open set where $F$ is a finite subset of $A$ and $U_a$ is open in $X_a$, then
$$P_c[V] = \begin{cases}
U_c & \text{if } c \in F,\\
X_c & \text{if } c \notin F.
\end{cases}$$
