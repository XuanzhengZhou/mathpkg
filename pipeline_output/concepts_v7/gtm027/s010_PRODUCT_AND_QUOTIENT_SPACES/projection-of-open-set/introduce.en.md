This theorem establishes that every projection map $P_c$ from a product space to a coordinate space is an open mapping — the image of any open set under the projection is again open. The proof is straightforward for basic open sets: a basic open set restricts only finitely many coordinates, and projecting onto a coordinate either yields the corresponding open set (if that coordinate was restricted) or the entire coordinate space (if it was not). This should be contrasted with the fact that projections are generally \emph{not} closed mappings; the set $\{(x,y) : xy = 1\}$ in the Euclidean plane is closed but its projections onto both axes are not.
