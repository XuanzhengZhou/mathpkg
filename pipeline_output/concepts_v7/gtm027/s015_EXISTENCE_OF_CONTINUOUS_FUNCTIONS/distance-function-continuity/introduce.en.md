In a pseudo-metric space, the distance from a variable point to a fixed set $A$ is a continuous function. The proof follows from the inequality $|D(A,x) - D(A,y)| \leq d(x,y)$, showing the distance function is actually Lipschitz continuous with constant 1. This theorem is used to prove that every pseudo-metric space is normal: the distance functions to disjoint closed sets yield separating open sets.
