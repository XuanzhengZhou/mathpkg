A \textbf{pseudo-metric} for a set $X$ is a function $d : X \times X \to \mathbb{R}$ such that:
\begin{enumerate}
\item[(a)] $d(x,x) = 0$ for all $x \in X$;
\item[(b)] $d(x,y) = d(y,x)$ for all $x, y \in X$ (symmetry);
\item[(c)] $d(x,z) \leq d(x,y) + d(y,z)$ for all $x, y, z \in X$ (triangle inequality).
\end{enumerate}
A \textbf{pseudo-metric space} is a pair $(X,d)$ where $d$ is a pseudo-metric for $X$.
