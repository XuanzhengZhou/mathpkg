A topological space is \textbf{regular} iff for each point $x$ and each neighborhood $U$ of $x$ there is a closed neighborhood $V$ of $x$ such that $V \subset U$; equivalently, the family of closed neighborhoods of each point is a base for the neighborhood system of the point.

An equivalent formulation: for each point $x$ and each closed set $A$, if $x \notin A$, then there are disjoint open sets $U$ and $V$ such that $x \in U$ and $A \subset V$.
