Let $F$ be a family of continuous functions, each member $f$ being on a topological space $X$ to a topological space $Y_f$. Let the evaluation map $e : X \to \prod \{Y_f : f \in F\}$ be defined by $e(x)_f = f(x)$. Then:
\begin{enumerate}
\item[(a)] $e$ is a continuous function on $X$ to the product space.
\item[(b)] $e$ is an open map of $X$ onto $e[X]$ if $F$ distinguishes points and closed sets.
\item[(c)] $e$ is one-to-one if and only if $F$ distinguishes points.
\end{enumerate}
