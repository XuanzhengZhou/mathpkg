In a pseudo-metric space, the distance from a variable point to a fixed set $A$ is a continuous function of the point. The proof follows from the triangle inequality, which yields $|D(A,x) - D(A,y)| \leq d(x,y)$, showing that the distance function is in fact Lipschitz with constant 1. This result is fundamental for constructing separating functions in normal spaces and for proving that pseudo-metric spaces are normal.
