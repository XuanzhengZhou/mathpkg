\begin{definition}[Discrete Non-Metrizable Uniform Space]
Let $\Omega_0$ be the set of all ordinals which are less than the first uncountable ordinal $\Omega$. For each $a \in \Omega_0$, define
\[
U_a = \{(x, y) \in \Omega_0 \times \Omega_0 : x = y \text{ or } (x \geq a \text{ and } y \geq a)\}.
\]
Then the family $\{U_a : a \in \Omega_0\}$ is a base for a uniformity $\mathfrak{u}$ on $\Omega_0$. The uniform space $(\Omega_0, \mathfrak{u})$ has a discrete (hence metrizable) topology, but the uniformity $\mathfrak{u}$ itself is not metrizable.
\end{definition}
