The \textbf{Cantor discontinuum} (middle third set) is the set of all members of the closed unit interval $[0,1]$ which have a triadic expansion in which the digit one does not occur, using only irrational triadic expansions (i.e., expansions which are not identically zero from some point on). Equivalently, it is the set obtained from $[0,1]$ by successive deletion of open middle thirds.
