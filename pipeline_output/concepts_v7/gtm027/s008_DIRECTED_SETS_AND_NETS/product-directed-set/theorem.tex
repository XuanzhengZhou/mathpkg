If $(D, \geq)$ and $(E, >)$ are directed sets, then the Cartesian product $D \times E$ is directed by $\gg$, where $(d, e) \gg (f, g)$ if and only if $d \geq f$ and $e > g$. The directed set $(D \times E, \gg)$ is the product directed set.

More generally, if for each $a$ in a set $A$ we are given a directed set $(D_a, >_a)$, the Cartesian product $\bigwedge \{D_a : a \in A\}$ is the set of all functions $d$ on $A$ such that $d_a = d(a)$ is a member of $D_a$ for each $a$ in $A$. The product directed set is $(\bigwedge \{D_a : a \in A\}, \geq)$ where, if $d$ and $e$ are members of the product, $d \geq e$ if and only if $d_a >_a e_a$ for each $a$ in $A$.
