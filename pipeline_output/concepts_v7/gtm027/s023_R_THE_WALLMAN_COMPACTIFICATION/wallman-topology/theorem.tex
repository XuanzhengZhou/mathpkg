For each open set $U$ of $X$, let $U^* = \{\alpha \in w(X) : \text{there exists } A \in \alpha \text{ with } A \subset U\}$. The family $\{U^* : U \text{ is open in } X\}$ forms a base for a compact topology on $w(X)$, called the \textbf{Wallman topology}. The correspondence $U \mapsto U^*$ preserves finite intersections and unions.
