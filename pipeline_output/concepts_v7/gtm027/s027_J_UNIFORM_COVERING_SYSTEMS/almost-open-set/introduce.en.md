A set is almost open if it differs from an open set by a meager set—specifically, it can be made open by removing a meager set and then adding another meager set. This is a Baire category refinement of the notion of openness. Almost open sets form a $\sigmaehBalgebra containing all Borel sets, making them a crucial bridge between topology and measure-theoretic (category) considerations in topological groups.
