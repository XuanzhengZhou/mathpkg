This fundamental theorem bridges the global notion of an open set with the local notion of neighborhoods. If a set contains a neighborhood of each of its points, then each point belongs to some open subset of the set, and the union of all such open subsets recovers the set, proving it is open. Conversely, an open set itself serves as a neighborhood for each of its points.
