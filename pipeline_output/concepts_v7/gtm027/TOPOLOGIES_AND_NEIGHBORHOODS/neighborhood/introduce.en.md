A neighborhood of a point is any set that contains an open set containing the point. Neighborhoods need not themselves be open, but every open set containing $x$ is a neighborhood of $x$. This concept localizes the notion of "nearness" to individual points and provides an alternative (and equivalent) foundation for topology via neighborhood systems, as developed by Hausdorff.
