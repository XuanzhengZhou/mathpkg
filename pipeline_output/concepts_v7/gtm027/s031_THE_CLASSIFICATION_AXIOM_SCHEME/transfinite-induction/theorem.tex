\textbf{(Transfinite Induction/Recursion)} Given a function $g$, there is a unique function $f$ whose domain is an ordinal (possibly the class $R$ of all ordinals) such that for each ordinal $x$ in the domain of $f$, $f(x) = g(f|_{x})$, where $f|_{x}$ is the restriction of $f$ to the ordinal $x$. If the domain of $f$ is not all of $R$, then $f$ is a set.
