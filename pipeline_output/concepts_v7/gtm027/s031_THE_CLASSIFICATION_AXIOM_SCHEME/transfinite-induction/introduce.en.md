Transfinite induction (or definition by transfinite recursion) is the fundamental tool for constructing functions along ordinals. Given a rule $g$ that determines the value at an ordinal from all previous values, there exists a unique function $f$ satisfying $f(x) = g(f|_x)$ for all ordinals $x$ in its domain. The domain is either all ordinals or some initial segment. This theorem is essential for the proofs of the Well-Ordering Theorem, the Hausdorff Maximal Principle, and many constructions in set theory and topology.
