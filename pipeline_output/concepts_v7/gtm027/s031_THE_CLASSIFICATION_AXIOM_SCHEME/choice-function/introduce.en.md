A choice function is a simultaneous selection of an element from each set in its domain. The Axiom of Choice asserts the existence of a global choice function defined on all non-empty sets. Choice functions are used throughout mathematics to select representatives: a basis from a vector space, a point from each equivalence class, or a maximal element in inductive proofs.
