The singleton $\{x\}$ is defined using a technical device: if $x$ is a set, $\{x\}$ has $x$ as its sole member; if $x$ is not a set (proper class), then $\{x\} = \mathcal{U}$. This convention simplifies computations by returning $\mathcal{U}$ when an operation is applied outside its natural domain.
