In Kelley's system, a cardinal number is an ordinal number not equipotent to any smaller ordinal (the "initial ordinal" definition). The power $P(x)$ of a set is the unique cardinal number equipotent to it, whose existence follows from the Well-Ordering Theorem. This definition works only with the Axiom of Choice. Without choice, the more general definition using Scott's trick or the axiom of regularity would be needed.
