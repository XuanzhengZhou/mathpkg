In Kelley's set theory, a set is defined as a class that can be a member of another class. This elegant definition distinguishes sets from proper classes: proper classes are "too large" to be members of other classes. This dichotomy prevents Russell's paradox by restricting class membership to sets. The definition is compatible with the Classification Axiom Scheme, which also requires the "$u$ is a set" condition.
