The Axiom of Choice in Kelley's system is a strong global form: there exists a single choice function that selects an element from every non-empty set simultaneously. This is equivalent to the usual formulations. From it, the Well-Ordering Theorem (every set can be well-ordered) and the Hausdorff Maximal Principle are derived via transfinite induction. The axiom is used throughout topology, notably in Tychonoff's product theorem and in the construction of compactifications.
