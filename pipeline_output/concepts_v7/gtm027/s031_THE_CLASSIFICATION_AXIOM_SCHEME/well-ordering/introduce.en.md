A well-ordering is a linear order with the property that every non-empty subset has a least element. This simple definition has profound consequences: a well-ordered class admits transfinite induction, is order-isomorphic to a unique ordinal, and every element (except the last) has an immediate successor. Well-orderings are central to set theory via the Well-Ordering Theorem and the theory of ordinals and cardinals.
