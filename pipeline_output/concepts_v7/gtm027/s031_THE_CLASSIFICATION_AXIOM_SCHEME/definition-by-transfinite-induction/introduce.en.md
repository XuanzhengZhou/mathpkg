It is possible to define a function on the entire class $R$ of ordinals by specifying its value at each ordinal in terms of its restriction to smaller ordinals. If $g(0) = \mathcal{U}$, then $f = 0$. This is the fundamental theorem justifying definitions by transfinite induction, used throughout set theory.
