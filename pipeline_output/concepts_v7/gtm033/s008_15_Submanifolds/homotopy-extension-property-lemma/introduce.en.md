This is a $C^r$ version of the homotopy extension property, providing a key technical tool for constructing homotopies in the smooth category. The proof uses a smooth bump function $\lambda$ to interpolate between $f$ on the base and $g$ on a neighborhood of the closed set $Z$. This lemma forms the foundation for the homotopy extension theorem (Theorem 1.4) and the covering homotopy theorem for vector bundles.
