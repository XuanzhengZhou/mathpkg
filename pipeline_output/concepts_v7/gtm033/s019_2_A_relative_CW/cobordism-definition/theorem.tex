\text{Two compact manifolds } M_0, M_1 \text{ without boundary are called } \textit{cobordant}, \text{ denoted } M_0 \sim M_1, \text{ if there exists a compact manifold } W \text{ such that } \partial W \approx M_0 \times 0 \cup M_1 \times 1.
\]

\text{The set of cobordism classes is denoted by } \mathfrak{N}^n; \text{ the cobordism class of } M \text{ is } [M].
\]

\text{For oriented cobordism: } (M_0, \omega_0) \text{ and } (M_1, \omega_1) \text{ are cobordant if there is a compact oriented } (W, \theta) \text{ and an orientation-preserving diffeomorphism } (\partial W, \partial\theta) \approx (M_0 \times 0, -\omega_0) \cup (M_1 \times 1, \omega_1). \text{ The set of these classes is } \Omega^n.
