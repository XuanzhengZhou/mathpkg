\text{Let } E_{s,k} \to G_{s,k} \text{ be the Grassmann bundle, } E^*_{s,k} \text{ its one-point compactification (Thom space). The Thom homomorphism}
\[
\tau: \pi_{n+k}(E^*_{s,k}) \to \mathfrak{N}^n
\]
\text{is defined by } \tau([f]) = [f^{-1}(G_{s,k})] \text{ for } f \text{ transverse to } G_{s,k}.
\]

\text{For } s \geqslant k + n + 1 \text{ and } k > n+1: \tau \text{ is a bijection.}
\]

\text{For oriented cobordism there is a similar isomorphism } \tilde{\tau}: \pi_{n+k}(\tilde{E}^*_{s,k}) \to \Omega^n \text{ using the oriented Grassmann bundle } \tilde{\gamma}_{s,k}.
