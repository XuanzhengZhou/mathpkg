A point $x \in M$ is critical for a $C^1$ map $f: M \to N$ if the linear map $T_x f: M_x \to N_{f(x)}$ is not surjective. The set of critical points of $f$ is denoted $\sum_f$.

A point $y \in N$ is a regular value of $f$ if $y \in N - f(\sum_f)$, i.e., every $x \in f^{-1}(y)$ is not a critical point. (In particular, any $y \notin f(M)$ is vacuously a regular value.)
