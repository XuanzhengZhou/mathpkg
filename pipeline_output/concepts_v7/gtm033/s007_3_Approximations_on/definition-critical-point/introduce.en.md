A critical point of a $C^1$ map is a point where the derivative (tangent map) fails to be surjective. The set of critical points is denoted $\sum_f$. A regular value is any point in the target that is not the image of a critical point---equivalently, every preimage of a regular value is a regular point where the derivative is surjective. The Morse-Sard theorem states that for sufficiently smooth maps, the set of critical values has measure zero, so regular values are abundant.
