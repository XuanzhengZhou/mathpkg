Let $X$ and $Y$ be arbitrary spaces. The space $C_S(X, Y)$ is the set $C(X, Y)$ with the strong topology, defined as follows. Let $\Gamma_f \subset X \times Y$ denote the graph of $f$. If $W \subset X \times Y$ is an open set containing $\Gamma_f$, let
$$\mathcal{N}(f, W) = \{g \in C(X, Y): \Gamma_g \subset W\}.$$
These sets, for all $f$ and $W$, form a base for the strong topology.

When $X$ is paracompact and $Y$ is metric, $C_S(X, Y)$ has an equivalent base comprising all sets of the form
$$\mathcal{N}(f, \varepsilon) = \{g: d(g(x), f(x)) < \varepsilon(x), \text{ for all } x \in X\}$$
where $f \in C(X, Y)$ and $\varepsilon \in C(X, \mathbb{R}_+)$ are arbitrary.
