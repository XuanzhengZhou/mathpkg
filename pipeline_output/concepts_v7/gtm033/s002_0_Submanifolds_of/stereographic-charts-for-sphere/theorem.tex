For $i = 1, \ldots, 2n+2$ define maps

$$\varphi_i : U_i \to \mathbb{R}^n,$$
$$\varphi_i(x) = (x_1, \ldots, \hat{x}_j, \ldots, x_{n+1}) \quad \text{if } i = 2j-1 \text{ or } 2j,$$

where $\hat{x}_j$ denotes omission of the $j$-th coordinate. Each $\varphi_i$ maps $U_i$ homeomorphically onto the open $n$-disk

$$B = \{y \in \mathbb{R}^n : |y| < 1\},$$

and $\varphi_i^{-1}: B \to \mathbb{R}^{n+1}$ is analytic.
