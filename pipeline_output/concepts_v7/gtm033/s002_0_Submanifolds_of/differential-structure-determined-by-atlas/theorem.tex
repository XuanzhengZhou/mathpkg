To determine a $C^r$ differential structure on a topological space $M$, it suffices to give a single $C^r$ atlas contained in it. Thus $\mathbb{R}^n$ has a unique $C^r$ differential structure containing the identity map of $\mathbb{R}^n$. More generally, every open set $U \subset \mathbb{R}^n$ has a unique $C^r$ differential structure containing the inclusion map $U \hookrightarrow \mathbb{R}^n$.
