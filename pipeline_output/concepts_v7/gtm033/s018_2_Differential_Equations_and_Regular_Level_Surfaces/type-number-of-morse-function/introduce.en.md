The $k$'th type number $v_k(f)$ of a Morse function $f$ counts the number of critical points of index $k$. The tuple $(v_0, \ldots, v_n)$ is called the type of $f$. Type numbers are central to the Morse inequalities, which relate the topology of the manifold (via Betti numbers) to the critical point data of any Morse function on it.
