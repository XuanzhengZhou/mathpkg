Proposition 1.0 is the easy Whitney embedding theorem restated in the language of function spaces: for a compact manifold, embeddings into sufficiently high-dimensional Euclidean space are dense in the weak $C^r$ topology. This is an approximation result showing that any $C^r$ map can be perturbed slightly to become an embedding (or immersion), provided the codimension is large enough.
