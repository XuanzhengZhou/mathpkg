A $C^r$ partition of unity is a fundamental technical tool on manifolds. It is a family of nonnegative smooth (or $C^r$) functions, each supported inside a given open set of a cover, whose supports are locally finite (each point has a neighborhood where only finitely many functions are nonzero), and which sum to 1 everywhere. Partitions of unity enable the gluing of locally defined objects — vector fields, Riemannian metrics, differential forms, and maps — into globally defined ones. The existence of $C^r$ partitions of unity for any open cover is a defining property of paracompact manifolds.
