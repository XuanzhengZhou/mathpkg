The boundary of a $C^r$ manifold $(M, \Phi)$ is defined to be the set of points $x \in M$ which are boundary points for some (hence any) chart. A point $x \in M$ is a boundary point for a chart $(\varphi, U) \in \Phi$ if $\varphi(x)$ lies in the boundary $\partial H$ of a halfspace $H \subset \mathbb{R}^n$ containing $\varphi(U)$. The boundary is denoted by $\partial M$.
