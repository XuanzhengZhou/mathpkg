Theorem 1.4 provides a more flexible isotopy extension: given an isotopy defined only on an open set $U$, one can extend it to a global compactly supported diffeotopy that agrees with the original isotopy near a compact subset $A \subset U$. The openness of the isotopy track $\hat{F}(U \times I)$ is crucial for the partition-of-unity construction of the extending vector field.
