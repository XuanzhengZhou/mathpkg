Let $\xi = (p, E, B)$ be a vector bundle. An \text{orientation} for $\xi$ is a family $\omega = \{\omega_x\}_{x \in B}$ where $\omega_x$ is an orientation of the fibre $E_x$, such that $\xi$ has an atlas $\Phi$ with the following property: if $\varphi: \xi|_U \to \mathbb{R}^n$ is in $\Phi$, then the fibre map $\varphi_x: (E_x, \omega_x) \to (\mathbb{R}^n, \omega_n)$ is orientation preserving, where $\omega_n$ is the standard orientation of $\mathbb{R}^n$. The family $\omega$ is called a \text{coherent family of orientations} of the fibres, and $\Phi$ is an \text{oriented atlas} belonging to $\xi$.

If $\xi$ has an orientation $\omega$, then $\xi$ is called \text{orientable} and the pair $(\xi, \omega)$ is an \text{oriented vector bundle}. If $\xi$ is a $C^r$ vector bundle for $r \geqslant 1$ and $\omega$ is an orientation, then $\xi$ has a $C^r$ atlas belonging to $\omega$.

Let $F: \eta \to \xi$ be a bimorphism. If $\xi$ has an orientation $\omega$, there is a unique orientation $\theta$ of $\eta$ such that $F$ maps fibres to fibres preserving orientation. Consequently, the pullback $f^*\xi$ of an oriented bundle $(\xi, \omega)$ carries a natural induced orientation $f^*\omega$.
