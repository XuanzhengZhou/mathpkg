A tangent vector to a $C^{r+1}$ manifold $M$ (with atlas $\Phi = \{(\varphi_i, U_i)\}$ indexed by $A$) is an equivalence class $[x,i,a]$ of triples
$$(x,i,a) \in M \times A \times \mathbb{R}^n$$
under the equivalence relation:
$$[x,i,a] = [y,j,b]$$
if and only if $x = y$ and
$$D(\varphi_j \varphi_i^{-1})(\varphi_i(x))a = b.$$
The set of all tangent vectors is denoted $TM$, the tangent bundle of $M$.
