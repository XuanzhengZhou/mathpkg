An alternative and geometrically intuitive definition of a tangent vector identifies it as an equivalence class of $C^1$ curves emanating from a point, where two curves are equivalent if they have the same velocity in some (hence any) coordinate chart. This definition is equivalent to the chart-based triple definition, with the curve $t \mapsto \varphi_i^{-1}(\varphi_i(x) + ta)$ providing the correspondence. This curve-based perspective connects the abstract definition to the physical notion of a tangent vector as a velocity.
