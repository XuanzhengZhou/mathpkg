\text{Let } f: M \to [a,b] \text{ be a } C^{r+1} \text{ map on a compact } \partial\text{-manifold, } 1 \leqslant r \leqslant \omega. \text{ Suppose } f \text{ has no critical points and } f(\partial M) = \{a,b\}. \text{ Then there is a } C^r \text{ diffeomorphism } F: f^{-1}(a) \times [a,b] \to M \text{ such that the diagram }

\[
\begin{CD}
f^{-1}(a) \times [a,b] @>{F}>{\approx}> M \\
@V{\pi}VV @VV{f}V \\
[a,b] @= [a,b]
\end{CD}
\]

\text{commutes, where } \pi \text{ is projection. In particular all level surfaces of } f \text{ are diffeomorphic.}
