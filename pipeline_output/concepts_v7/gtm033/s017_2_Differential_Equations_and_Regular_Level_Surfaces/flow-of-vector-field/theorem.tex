\text{Let } X \text{ be a } C^r \text{ vector field on a manifold } M, 1 \leqslant r \leqslant \omega. \text{ For each } x \in M \text{ there is a maximal open interval } J(x) \subset \mathbb{R} \text{ about } 0 \text{ and a } C^{r+1} \text{ integral curve } \eta^x: (J(x),0) \to (M,x) \text{ such that } \frac{d}{dt}\eta^x(t) = X(\eta^x(t)).
\]

\text{The set } \Omega = \{(t,x) \in \mathbb{R} \times M : t \in J(x)\} \text{ is open, and the flow } \eta: \Omega \to M, (t,x) \mapsto \eta_t(x) \text{ is } C^r. \text{ When } M \text{ is compact and } \partial M = \varnothing, \Omega = \mathbb{R} \times M \text{ and each } \eta_t \text{ is a } C^r \text{ diffeomorphism, so } t \mapsto \eta_t \text{ is a one-parameter group of diffeomorphisms.}
