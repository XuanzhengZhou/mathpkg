Let $M$ be a $C^{r+1}$ manifold with a Riemannian metric $\langle\cdot,\cdot\rangle$. For every linear map $\lambda: M_x \to \mathbb{R}$ there exists a unique tangent vector $X_\lambda \in M_x$ such that

$$\lambda(Y) = \langle X_\lambda, Y \rangle \quad \text{for all } Y \in M_x.$$

If $f: M \to \mathbb{R}$ is a $C^{r+1}$ map, the gradient vector field $\text{grad } f$ is the $C^r$ vector field on $M$ defined by

$$\text{grad } f(x) = X_{Df_x}.$$

Thus $\langle \text{grad } f(x), Y \rangle = Df_x(Y)$ for all $Y \in M_x$.
