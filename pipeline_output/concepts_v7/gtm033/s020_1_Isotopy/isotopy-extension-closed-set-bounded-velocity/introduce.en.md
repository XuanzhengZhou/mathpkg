Theorem 1.7 generalizes the isotopy extension to closed subsets with bounded velocity. It is the analog of Theorem 1.4 for the bounded velocity setting, replacing the compact set $A$ with a closed set and using the bounded velocity condition instead of compact support for the resulting diffeotopy.

The additional conclusion that $\operatorname{Supp} G \subset F(U \times I)$ provides control over where the extending diffeotopy is active — it is supported only within the image of the original isotopy. This localization property is crucial for applications where one needs to extend isotopies without disturbing the manifold outside a specified region.
