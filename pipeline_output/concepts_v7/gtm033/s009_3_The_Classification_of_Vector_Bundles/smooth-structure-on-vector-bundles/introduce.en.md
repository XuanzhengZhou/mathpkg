This theorem states that the notion of a $C^r$ vector bundle over a smooth manifold is essentially independent of the differentiability class $r$: every $C^r$ bundle admits a compatible $C^\infty$ structure, and this structure is unique up to $C^\infty$ isomorphism. The proof uses the classification theorem: take a $C^r$ classifying map into a Grassmannian, approximate it by a $C^\infty$ map (which is homotopic to the original), and pull back the universal bundle. The same result holds with $C^\infty$ replaced by $C^\omega$ (real analytic), using the analytic approximation theorem.
