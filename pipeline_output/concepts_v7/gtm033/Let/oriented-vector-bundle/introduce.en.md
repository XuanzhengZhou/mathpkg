An oriented vector bundle is a vector bundle equipped with a coherent choice of orientation on each fibre. Coherence means there exists a vector bundle atlas whose local trivializations are orientation-preserving at each fibre. Equivalently, an oriented $C^r$ vector bundle is a $C^r$ vector bundle with a maximal $C^r$ oriented atlas. Pullbacks and bimorphisms preserve orientations naturally.
