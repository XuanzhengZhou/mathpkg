Let $\xi = (p, E, B)$ be a vector bundle. An \text{orientation} for $\xi$ is a family $\omega = \{\omega_x\}_{x \in B}$ where $\omega_x$ is an orientation of the fibre $E_x$ such that $\xi$ has an atlas $\Phi$ with the following property: if $\varphi: \xi|U \to \mathbb{R}^n$ is in $\Phi$ then

$$\varphi_x: (E_x, \omega_x) \to (\mathbb{R}^n, \omega_n)$$

is orientation preserving. We call $\omega$ a \text{coherent family of orientations} of the fibres. The atlas $\Phi$ is an \text{oriented atlas} belonging to $\xi$.

If $\xi$ has an orientation $\omega$, then $\xi$ is called \text{orientable} and the pair $(\xi, \omega)$ is an \text{oriented vector bundle}. If $\xi$ is a $C^r$ vector bundle for any $r \geqslant 1$, and $\omega$ is an orientation, then $\xi$ has a $C^r$ atlas which belongs to $\omega$. An oriented $C^r$ vector bundle can be defined as a $C^r$ vector bundle together with a maximal $C^r$ oriented atlas.
