A CW-complex is a space $X$ which can be expressed as $X = \bigcup_{n=0}^{\infty} X_n$ where $X_0 \subset X_1 \subset \cdots$, $X_0$ is a discrete subset and $X_{n+1}$ is obtained from $X_n$ by attaching $n$-cells via continuous maps of their boundaries into $X_{n-1}$. It is required that a subset of $X$ is closed provided its intersection with each closed cell is closed. If $X = X_n$, $X$ is an $n$-dimensional CW-complex. If the number of cells is finite, $X$ is a finite CW-complex.
