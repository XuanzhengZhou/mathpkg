A topological space $X$ is called:
\begin{itemize}
\item \text{Hausdorff} if every pair of distinct points have disjoint neighborhoods;
\item \text{normal} if for every pair of disjoint closed sets $A, B$ there is a continuous map $f: X \to [0,1]$ with $f(A) = 0$ and $f(B) = 1$;
\item \text{paracompact} if every open cover $\mathcal{U} = \{U_{\lambda}\}_{\lambda \in \Lambda}$ has a locally finite open refinement $\mathcal{V} = \{V_{\gamma}\}_{\gamma \in \Gamma}$.
\end{itemize}
