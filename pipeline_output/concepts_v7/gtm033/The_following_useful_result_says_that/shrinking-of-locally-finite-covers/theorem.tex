If $X$ is normal and $\mathcal{U} = \{U_{\lambda}\}_{\lambda \in \Lambda}$ is a locally finite open cover, then $\mathcal{U}$ has a \text{shrinking}, that is, an open cover $\mathcal{V} = \{V_{\lambda}\}_{\lambda \in \Lambda}$ such that $\overline{V}_{\lambda} \subset U_{\lambda}$ for each $\lambda$.
