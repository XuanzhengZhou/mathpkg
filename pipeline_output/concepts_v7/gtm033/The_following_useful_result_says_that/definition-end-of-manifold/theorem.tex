Let $M$ be a connected noncompact surface. An \text{end} of $M$ is an equivalence class $[K, C]$ of pairs $(K, C)$ where $K \subset M$ is compact, $C$ is a component of $M - K$ whose closure is not compact, and $[K_1, C_1] = [K_2, C_2]$ if there exists $(K, C)$ with $K_1 \cup K_2 \subset K$ and $C_1 \cap C_2 \supset C$.
