A map $f: U \to \mathbb{R}^n$ is called $C^1$ if $Df_x$ exists for every $x \in U$ and the map $Df: U \to L(\mathbb{R}^m, \mathbb{R}^n)$, $x \mapsto Df_x$, is continuous. By recursion, $f$ is $C^r$ ($2 \leqslant r < \infty$) if $Df$ is $C^{r-1}$. If $f$ is $C^r$ for all $r$, it is called $C^{\infty}$ or \text{smooth}. We say $\infty \leqslant \omega$.
