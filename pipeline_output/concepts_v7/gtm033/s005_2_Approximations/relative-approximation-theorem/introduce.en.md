Theorem 2.5 is the relative approximation theorem, which refines Theorem 2.4. It allows one to improve the differentiability of a $C^r$ map to $C^s$ ($s > r$) on a neighborhood of a closed set $K$, while keeping the map unchanged outside an open set $W$ and staying arbitrarily close to the original map in the strong $C^r$ topology. This is a crucial technical tool for globalization arguments on manifolds.
