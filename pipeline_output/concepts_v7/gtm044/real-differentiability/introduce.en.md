A function $f: U \subset \mathbb{R}^n \to \mathbb{R}$ is differentiable at a point if it can be approximated to first order by an affine linear function (a tangent $n$-plane). This is the standard notion of total differentiability in multivariable real analysis, expressed via the existence of the gradient $\nabla f(a) = (c_1, \ldots, c_n)$. A vector-valued map is differentiable if each component function is.
