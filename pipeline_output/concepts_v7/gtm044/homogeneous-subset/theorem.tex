A homogeneous subset of $k^n$ is any subset $S$ of $k^n$ satisfying

$$x \in S \text{ implies } cx \in S \quad \text{for all } c \in k.$$

Hence a set is homogeneous iff it is $\varnothing$, $\{0\}$, or consists of a nonempty union of $1$-subspaces of $k^n$. In view of this definition, a homogeneous set in $k^n$ can be regarded as a set of points in $\mathbb{P}^{n-1}(k)$. (One regards $\{0\} \subset k^n$ as $\varnothing \subset \mathbb{P}^{n-1}(k)$.)
