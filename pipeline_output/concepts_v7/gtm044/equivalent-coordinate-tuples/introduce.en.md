Two nonzero $(n+1)$-tuples in $k^{n+1}$ are equivalent if one is a nonzero scalar multiple of the other. This equivalence relation captures the fact that all coordinate sets of a given point in $\mathbb{P}^n(k)$ represent the same $1$-subspace. The projective space can be viewed as the set of equivalence classes of $k^{n+1} \setminus \{0\}$ under this relation.
