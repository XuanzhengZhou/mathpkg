A \textbf{basis} (or \textbf{base}) for an ideal $\mathfrak{a}$ in a ring $R$ is any collection $\{a_\gamma\}$ of elements $a_\gamma \in \mathfrak{a}$ (with $\gamma$ in some indexing set $\Gamma$) such that
$$\mathfrak{a} = \{ r_{\gamma_1} a_{\gamma_1} + \cdots + r_{\gamma_k} a_{\gamma_k} \mid r_{\gamma_i} \in R \text{ and } \gamma_i \in \Gamma \}.$$
We write $\mathfrak{a} = \{a_\gamma\}$, or $\mathfrak{a} = (a_1, a_2, \ldots)$ if $\Gamma$ is countable, and $\mathfrak{a} = (a_1, \ldots, a_n)$ if $\Gamma$ is finite. If $\Gamma$ is finite, we say $\mathfrak{a}$ has a \textbf{finite basis}.
