Let $D$ be any unique factorization domain of characteristic zero. Then $f \in D[X]$ has a repeated (nonconstant) factor if and only if $f$ and its formal derivative $f'$ have a common factor. Thus
$$f \text{ has a repeated factor iff } \mathcal{D}(f) = 0.$$

In particular:
If $D = \mathbb{C}[X_1, \ldots, X_{i-1}, X_{i+1}, \ldots, X_t]$, then $f(X_i) \in D[X_i]$ has a repeated factor (involving $X_i$) if and only if $\mathcal{D}_{X_i}(f) = 0$.
