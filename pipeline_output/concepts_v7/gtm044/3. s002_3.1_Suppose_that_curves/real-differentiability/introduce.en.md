Real differentiability of a function $f: \mathbb{R}^n \to \mathbb{R}$ at a point $a$ means that $f$ can be approximated to first order by an affine linear function near $a$, with the error going to zero faster than the distance to $a$. A vector-valued map is differentiable when each component is. When all partial derivatives exist and are continuous, the function is $C^n$, which is a stronger and often more convenient condition. This definition is the foundation for the real implicit function theorem and serves as a template for the complex-analytic analogue used throughout algebraic geometry.
