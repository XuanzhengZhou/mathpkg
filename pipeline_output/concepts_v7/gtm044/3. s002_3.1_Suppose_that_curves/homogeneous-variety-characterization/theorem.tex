Let $V = \mathbb{V}(q_1, \ldots, q_r) \subset k^n$ be an affine variety. Then $V$ is a homogeneous set if and only if $V$ can be defined by a set of homogeneous polynomials. Specifically, if $V = \mathbb{V}(q_1, \ldots, q_r)$ is homogeneous, then $V$ is also the zero set of the set of all homogeneous components of all the $q_i$.
