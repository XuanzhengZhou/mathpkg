A projective set is any subset of a projective space $\mathbb{P}^{n-1}(k)$. It is represented by a homogeneous set in $k^n$, namely the union of all $1$-subspaces corresponding to points in the projective set. This correspondence allows one to transfer constructions and properties between the affine cone (a homogeneous subset of $k^n$) and the projective set it represents, which is a fundamental technique in algebraic geometry.
