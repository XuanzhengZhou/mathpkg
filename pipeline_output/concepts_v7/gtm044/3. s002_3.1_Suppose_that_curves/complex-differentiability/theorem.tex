A function $f(X): \mathbb{C}_X \to \mathbb{C}_Y$ is complex-differentiable at $a = a_1 + i a_2 \in \mathbb{C}_X$ if there exists $c \in \mathbb{C}$ such that

$$\lim_{x \to a} \frac{f(x) - [f(a) + c(x - a)]}{|x - a|} = 0.$$

If $f$ is complex-differentiable at all points in a neighborhood of $(a) \in \mathbb{C}_{X_1, \ldots, X_n}$, then in particular at $(a)$ all partial derivatives $\partial^n f / (\partial X_i)^n$ exist, which means $f$ is analytic in each variable separately. This in turn implies $f$ itself is analytic (Hartogs' theorem).
