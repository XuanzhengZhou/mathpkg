The Monodromy Theorem is a fundamental result in complex analysis. It states that if an analytic function element (a convergent power series on some open disk $\mathcal{O}$) can be analytically continued along every polygonal path within a simply connected domain $\Omega$, then all these continuations are consistent: the function element extends uniquely to a single-valued analytic function on the entire domain $\Omega$. In other words, on simply connected domains, the monodromy obstruction to global analytic continuation vanishes. This theorem is the key ingredient in proving Lemma 8.13 and, consequently, the connectedness of complex algebraic curves.
