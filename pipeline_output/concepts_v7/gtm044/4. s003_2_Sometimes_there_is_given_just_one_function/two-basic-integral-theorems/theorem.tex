Let $f(X)$ be a function analytic at each point of an open set containing a closed disk $\bar{\Delta}$ in $\mathbb{C}$ with boundary $\partial \Delta$, and suppose that within $\Delta = \bar{\Delta} \setminus \partial \Delta$ there are exactly $N$ zeros of $f(X)$, counted with multiplicity. Then:

\textbf{(3.8.1) Cauchy integral formula.}
For any point $b \in \Delta$,
$$\frac{1}{2\pi i} \int_{\partial \Delta} \frac{f(X)}{X - b} \, dX = f(b).$$

\textbf{(3.8.2) Argument principle.}
If $f(X) \neq 0$ on $\partial \Delta$, then
$$\frac{1}{2\pi i} \int_{\partial \Delta} \frac{f'(X)}{f(X)} \, dX = N.$$
