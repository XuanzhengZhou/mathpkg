Let $f$ and $g$ be any two functions defined on a neighborhood of $P \in \mathcal{T}$ with values in a field. Define
$$f_P^\sim + g_P^\sim = (f + g)_P^\sim,$$
$$f_P^\sim \cdot g_P^\sim = (f \cdot g)_P^\sim,$$
$$-(f_P^\sim) = (-f)_P^\sim.$$
If there is a neighborhood of $P$ on which $f$ is never zero, then define
$$\frac{1}{f_P^\sim} = \left(\frac{1}{f}\right)_P^\sim.$$
