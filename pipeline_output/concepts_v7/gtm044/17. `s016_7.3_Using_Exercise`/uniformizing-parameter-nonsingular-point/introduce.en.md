Theorem 2.31 provides a simple geometric method to obtain a uniformizing parameter at any nonsingular point of a plane curve: any coordinate axis not tangent to the curve at the point serves as a uniformizing parameter. This follows from the implicit function theorem (Theorem 3.6 of Chapter II), which guarantees that near a nonsingular point the curve can be expressed as a graph of an analytic function, making one coordinate a power series in the other. This result makes uniformizing parameters computationally accessible.
