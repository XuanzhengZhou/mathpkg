Let $K$ and $k$ be fields. If each element in $K$ is assigned a value in $k \cup \{\infty\}$, and if this assignment satisfies properties (2.5.1) and (2.5.2), then the set of elements assigned finite values forms a subring $R$ of $K$, and for each $a \in K$, $a \notin R$ implies $1/a \in R$.

Conversely, let $K$ be a field. If $R$ is any subring of $K$ such that for each $a \in K$, $a \notin R$ implies $1/a \in R$, then there is a field $k$ such that each element of $K$ is assigned a value in $k \cup \{\infty\}$, this assignment satisfying (2.5.1) and (2.5.2).
