Let $C \subset \mathbb{C}_{X_1, \ldots, X_n}$ be an irreducible curve. An element $U$ in the function field of $C$ is called a uniformizing parameter (or uniformizing variable) at $P \in C$ if the part of $C$ about $P$ can be represented by power series in $U$:
$$\left( \sum_j c_{1j} U^j, \ldots, \sum_j c_{nj} U^j \right),$$
where $P = (c_{10}, \ldots, c_{n0})$.
