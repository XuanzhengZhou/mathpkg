In a locally convex real TVS, every nonempty closed convex set is the intersection of all closed half-spaces that contain it. Given any point outside the set, applying strict separation to the closed convex set and the singleton yields a closed half-space containing the set but not the point. This result is a geometric version of the Hahn-Banach theorem and is fundamental in convex analysis.
