A subset is contained in a nonempty closed convex set if and only if its image under every continuous linear form is contained in the image of the convex set under the same form. The necessity is trivial, and the sufficiency follows from the fact that a closed convex set is the intersection of closed half-spaces; if a point were outside the set, some continuous linear form would witness this.
