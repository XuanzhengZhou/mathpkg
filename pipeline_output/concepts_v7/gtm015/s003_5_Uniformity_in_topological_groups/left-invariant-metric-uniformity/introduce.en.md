For a metrizable topological group, any left-invariant compatible metric $d$ generates exactly the left uniform structure. This follows from the identity $d(x, y) = d(e, x^{-1}y)$, which shows that the metric entourages $\{(x, y) : d(x, y) < \varepsilon\}$ coincide with the left-uniformity entourages $\{(x, y) : x^{-1}y \in V\}$ where $V = \{x : d(e, x) < \varepsilon\}$. This result justifies using left-invariant metrics to study completeness in topological groups.
