A valued group is a pair $$(G, |\cdot|)$$, where $$G$$ is a group and $$x \mapsto |x|$$ is a value on $$G$$, i.e., a function $$|\cdot|: G \to \mathbb{R}$$ satisfying:
\begin{enumerate}
\item $$|x| \geq 0$$ for all $$x \in G$$;
\item $$|x| = 0$$ if and only if $$x = e$$;
\item $$|x^{-1}| = |x|$$ for all $$x \in G$$;
\item $$|xy| \leq |x| + |y|$$ for all $$x, y \in G$$.
\end{enumerate}
When $$G$$ is written additively, the axioms take the form: (i') $$|0| = 0$$, (ii') $$|x| > 0$$ when $$x \neq 0$$, (iii') $$|-x| = |x|$$ for all $$x$$, (iv') $$|x + y| \leq |x| + |y|$$ for all $$x, y$$.
