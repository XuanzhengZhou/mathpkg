This elementary inequality states that the function $t \mapsto t/(1+t)$ is subadditive on $[0, \infty)$. It is a key technical tool for constructing bounded metrics from unbounded ones: given any metric $d$, the formula $D(x,y) = d(x,y)/(1+d(x,y))$ defines an equivalent metric bounded by $1$, and this lemma ensures the triangle inequality for $D$. This construction is used repeatedly in the theory of metrizable topological vector spaces.
