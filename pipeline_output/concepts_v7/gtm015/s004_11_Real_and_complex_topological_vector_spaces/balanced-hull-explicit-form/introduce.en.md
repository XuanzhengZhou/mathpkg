The balanced hull of a subset $S$ of a vector space is the smallest balanced set containing $S$. A set is balanced if $\lambda A \subset A$ for all $|\lambda| \leq 1$. This theorem gives the explicit construction: the balanced hull of $S$ is $DS = \{\lambda x : x \in S, |\lambda| \leq 1\}$, where $D$ is the closed unit disc (or interval) in $\mathbb{K}$. Balanced sets are fundamental in TVS theory because every neighborhood of zero can be replaced by a balanced one, forming a convenient neighborhood base.
