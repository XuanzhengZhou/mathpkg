We denote by $\mathcal{M}$ the set of all functions $f: X \to \mathbb{R}$ that are measurable with respect to a $\sigma$-ring $\mathcal{S}$. For $f \in \mathcal{M}$ we write
$$N(f) = \{x \in X : f(x) \neq 0\}.$$
We denote by $\mathcal{N}$ the set of all $f \in \mathcal{M}$ that vanish a.e. (with respect to $\mu$), that is, $\mu(N(f)) = 0$. With respect to the pointwise operations, $\mathcal{M}$ is a real algebra and $\mathcal{N}$ is an algebra ideal of $\mathcal{M}$. If $f \in \mathcal{M}$ we write
$$[f] = f + \mathcal{N}$$
for the canonical image of $f$ in the quotient algebra $\mathcal{M}/\mathcal{N}$; thus $[f] = [g]$ if and only if $f = g$ a.e. We write $[\mathcal{M}] = \mathcal{M}/\mathcal{N}$.
