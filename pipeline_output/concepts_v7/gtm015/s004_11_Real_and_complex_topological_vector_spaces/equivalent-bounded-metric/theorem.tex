If $(X, d)$ is a metric space, the formula
$$D(x, y) = \frac{d(x, y)}{1 + d(x, y)}$$
defines an equivalent metric $D$ on $X$. In particular, a sequence in $X$ is convergent for $d$ if and only if it is convergent for $D$. Moreover, a sequence in $X$ is Cauchy for $d$ if and only if it is Cauchy for $D$; thus $X$ is complete for $d$ if and only if it is complete for $D$.
