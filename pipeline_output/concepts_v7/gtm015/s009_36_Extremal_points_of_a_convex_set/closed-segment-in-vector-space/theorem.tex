Let $E$ be a vector space over $\mathbb{K}$ and $u, v \in E$. The **closed segment** $[u, v]$ is
$$[u, v] = \{\alpha u + (1 - \alpha)v : 0 \leq \alpha \leq 1\}.$$
Then $[u, v]$ is convex and $[u, v] = \operatorname{conv}\{u, v\}$. If $E$ is a separated topological vector space, then $[u, v]$ is compact and closed in $E$.
