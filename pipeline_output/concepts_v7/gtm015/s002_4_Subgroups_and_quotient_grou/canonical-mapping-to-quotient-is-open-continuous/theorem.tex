\text{Let } H \text{ be a subgroup of a topological group } G, \text{ let } G/H \text{ be the space of left cosets equipped with the quotient topology, and let } \pi: G \to G/H \text{ be the canonical mapping. Then:}
\begin{enumerate}
\item[(i)] \pi \text{ is an open, continuous mapping.}
\item[(ii)] \text{For each } a \in G, \text{ the neighborhoods of } \pi(a) \text{ are precisely the sets } \pi(aV), \text{ where } V \text{ is a neighborhood of the neutral element } e \text{ of } G.
\end{enumerate}
