\text{Let } E \text{ and } F \text{ be normed spaces over } \mathbb{K} \text{ and let } T: E \to F \text{ be a continuous linear mapping. The norm of } T, \text{ denoted } \|T\|, \text{ is defined by}

$$\|T\| = \sup \{ \|Tx\| : \|x\| \leq 1 \}.$$

\text{One has } 0 \leq \|T\| < \infty \text{ and } \|Tx\| \leq \|T\| \|x\| \text{ for all } x \in E. \text{ Moreover, } \|T\| \text{ is the minimal constant satisfying this inequality.}
