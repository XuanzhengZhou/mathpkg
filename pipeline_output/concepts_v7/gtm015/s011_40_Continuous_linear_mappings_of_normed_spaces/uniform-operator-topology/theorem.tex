\text{Let } E \text{ and } F \text{ be normed spaces over } \mathbb{K} \text{ and let } \mathcal{L}(E, F) \text{ be the space of continuous linear mappings. If } T_n, T \in \mathcal{L}(E, F), \text{ then}

$$\|T_n - T\| \to 0$$

\text{if and only if } \|T_n x - Tx\| \to 0 \text{ uniformly in } \|x\| \leq 1, \text{ i.e., } T_n \to T \text{ uniformly on the closed unit ball of } E.

\text{The topology on } \mathcal{L}(E, F) \text{ derived from the operator norm is called the uniform operator topology (or norm topology), denoted } \tau_n.
