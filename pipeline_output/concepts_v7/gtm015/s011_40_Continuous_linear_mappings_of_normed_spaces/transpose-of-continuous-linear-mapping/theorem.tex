\text{Let } E \text{ and } F \text{ be normed spaces over } \mathbb{K} \text{ and let } T \in \mathcal{L}(E, F). \text{ The transpose (or adjoint) of } T \text{ is the mapping } T' \in \mathcal{L}(F', E') \text{ defined by}

$$T'(g) = g \circ T \quad (g \in F').$$

\text{Equivalently, } (T'g)(x) = g(Tx) \text{ for all } x \in E \text{ and } g \in F'.

\text{Since } T' \in \mathcal{L}(F', E'), \text{ one may form } (T')' \in \mathcal{L}(E'', F''), \text{ denoted } T''.
