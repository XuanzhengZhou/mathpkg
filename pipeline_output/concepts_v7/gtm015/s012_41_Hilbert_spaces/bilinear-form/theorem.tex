If $E$ is a real vector space, a \textbf{bilinear form} on $E$ is a function $\varphi \colon E \times E \to \mathbb{R}$ that is linear in each of its variables, i.e., for all $x, y, z \in E$ and all real scalars $\alpha$:

\begin{enumerate}
\item[(i)] $\varphi(x + y, z) = \varphi(x, z) + \varphi(y, z)$,
\item[(ii)] $\varphi(\alpha x, y) = \alpha\,\varphi(x, y)$,
\item[(iii)] $\varphi(x, y + z) = \varphi(x, y) + \varphi(x, z)$,
\item[(iv)] $\varphi(x, \alpha y) = \alpha\,\varphi(x, y)$.
\end{enumerate}

The form is called \textbf{symmetric} if $\varphi(y, x) = \varphi(x, y)$ for all $x, y$; \textbf{positive} if $\varphi(x, x) \geq 0$ for all $x$; and \textbf{strictly positive} (or \textbf{positive definite}) if $\varphi(x, x) > 0$ for all $x \neq \theta$.

A \textbf{real inner product} on $E$ is a symmetric, strictly positive, bilinear form on $E$.
