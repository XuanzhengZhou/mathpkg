A linear variety in a vector space $E$ over $\mathbb{K}$ is a set $V$ of the form
$$V = x + N = \{x + y : y \in N\},$$
where $x \in E$ and $N$ is a linear subspace of $E$. Thus, a linear variety is a translate of a linear subspace, i.e., a coset of $E$ relative to a linear subspace. The subspace $N$ is uniquely determined by $V$ via
$$N = \{y - z : y, z \in V\}.$$
One has $0 \in V$ if and only if $V$ is a linear subspace; a linear subspace is called a \textbf{homogeneous variety}. Equivalently, $V$ is homogeneous if and only if $V + V \subset V$.
