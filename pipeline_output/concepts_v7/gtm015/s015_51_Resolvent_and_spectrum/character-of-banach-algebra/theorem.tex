A character of $A$ is an algebra epimorphism $f: A \to \mathbb{C}$; we write $\mathcal{X}$ for the set of all characters of $A$. Every character $f$ is a continuous linear form with $\|f\| = 1$, and its kernel is a maximal ideal. The correspondence $M \mapsto f_M$ is a bijection from the set of maximal ideals onto $\mathcal{X}$.
