The spectrum of a bounded operator on a normed space decomposes into four disjoint parts: point spectrum (eigenvalues), residual spectrum (injective but non-dense range), continuous spectrum (injective with dense range but not surjective), and compression spectrum (non-injective). This classification provides a fine-grained analysis of the ways an operator can fail to be invertible.
