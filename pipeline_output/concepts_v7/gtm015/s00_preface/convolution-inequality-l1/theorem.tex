If $x, y \in \ell^1(\mathbb{Z})$, then for each $m \in \mathbb{Z}$,
$$\sum_{n=-\infty}^{\infty} |x(m-n) y(n)| < \infty,$$
so that the series
$$\sum_{n=-\infty}^{\infty} x(m-n) y(n)$$
is absolutely convergent. Denoting its sum by $z(m)$, we have
$$|z(m)| \leq \sum_{n=-\infty}^{\infty} |x(m-n) y(n)|.$$
Summing over $m$,
$$\sum_{m=-\infty}^{\infty} |z(m)| \leq \sum_{m=-\infty}^{\infty} \sum_{n=-\infty}^{\infty} |x(m-n) y(n)| = \|x\|_1 \|y\|_1,$$
so that $z \in \ell^1(\mathbb{Z})$ and $\|z\|_1 \leq \|x\|_1 \|y\|_1$.
