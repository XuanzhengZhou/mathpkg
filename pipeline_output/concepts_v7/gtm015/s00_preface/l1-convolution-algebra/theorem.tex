With convolution as product, $\ell^1(\mathbb{Z})$ is a commutative, associative algebra with unity element $e_0$. Specifically, for all $x, y, z \in \ell^1(\mathbb{Z})$:
\begin{itemize}
\item \textbf{Commutativity:} $xy = yx$.
\item \textbf{Associativity:} $(xy)z = x(yz)$.
\item \textbf{Bilinearity:} $(x + y)z = xz + yz$ and $x(y + z) = xy + xz$, and $(cx)y = c(xy) = x(cy)$ for $c \in \mathbb{C}$.
\item \textbf{Unity:} $e_0 x = x e_0 = x$, where $e_0(m) = \delta_{0m}$.
\item \textbf{Norm inequality:} $\|xy\|_1 \leq \|x\|_1 \|y\|_1$.
\end{itemize}
Moreover, $\ell^1(\mathbb{Z})$ is complete with respect to the metric $d(x, y) = \|x - y\|_1$, hence it is a commutative Banach algebra.
