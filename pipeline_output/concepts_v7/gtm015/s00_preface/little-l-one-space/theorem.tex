We write $\ell^1(\mathbb{Z})$ for the set of all sequences $(a_n)_{n \in \mathbb{Z}}$ with $a_n \in \mathbb{C}$ and
$$\sum_{n=-\infty}^{\infty} |a_n| < \infty.$$
Such a sequence may be regarded as a function $x: \mathbb{Z} \to \mathbb{C}$ with $x(n) = a_n$ for all $n \in \mathbb{Z}$. Thus $\ell^1(\mathbb{Z})$ is the set of all functions $x: \mathbb{Z} \to \mathbb{C}$ such that
$$\sum_{n=-\infty}^{\infty} |x(n)| < \infty.$$
Two elements $x, y \in \ell^1(\mathbb{Z})$ are equal if and only if $x(n) = y(n)$ for all $n \in \mathbb{Z}$. For $x \in \ell^1(\mathbb{Z})$, the $\ell^1$-norm is defined by
$$\|x\|_1 = \sum_{n=-\infty}^{\infty} |x(n)|.$$
