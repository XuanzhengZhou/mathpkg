Define pointwise (term-by-term) operations on $\ell^1(\mathbb{Z})$ by
$$(x + y)(n) = x(n) + y(n), \quad (cx)(n) = c \cdot x(n) \quad (c \in \mathbb{C}).$$
Then $\ell^1(\mathbb{Z})$ is a vector space over $\mathbb{C}$. Moreover, for all $x, y \in \ell^1(\mathbb{Z})$ and $c \in \mathbb{C}$:
$$\|x + y\|_1 \leq \|x\|_1 + \|y\|_1,$$
$$\|cx\|_1 = |c| \cdot \|x\|_1.$$
Consequently, $\ell^1(\mathbb{Z})$ is a normed vector space with respect to the $\ell^1$-norm.
