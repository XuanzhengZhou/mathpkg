For $x \in \ell^1(\mathbb{Z})$, the $\ell^1$-norm is defined by
$$\|x\|_1 = \sum_{n=-\infty}^{\infty} |x(n)|.$$
The formula
$$d(x, y) = \|x - y\|_1$$
defines a metric $d$ on $\ell^1(\mathbb{Z})$ satisfying:
$$d(x, z) \leq d(x, y) + d(y, z), \quad d(x, y) = d(y, x), \quad d(x, y) \geq 0,$$
and $d(x, y) = 0$ if and only if $x = y$.
