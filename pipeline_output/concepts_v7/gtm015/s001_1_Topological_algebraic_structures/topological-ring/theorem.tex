Let $A$ be a ring (with associative multiplication). A topology on $A$ is said to be compatible with the ring structure if the mappings $(x, y) \mapsto x + y$, $x \mapsto -x$, and $(x, y) \mapsto xy$ are continuous. A topological ring is a pair $(A, \tau)$, where $A$ is a ring and $\tau$ is a topology on $A$ compatible with the ring structure. Such a topology is, in particular, compatible with the additive group structure, thus the additive group of a topological ring is an abelian topological group.
