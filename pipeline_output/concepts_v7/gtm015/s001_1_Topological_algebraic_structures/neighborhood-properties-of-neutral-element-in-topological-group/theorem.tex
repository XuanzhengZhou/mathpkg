If $G$ is a topological group, then the class $\mathcal{V}$ of all neighborhoods of the neutral element $e$ has the following properties:
\begin{enumerate}
\item[(1)] $e \in V$ for all $V \in \mathcal{V}$;
\item[(2)] if $V, W \in \mathcal{V}$ then $V \cap W \in \mathcal{V}$;
\item[(3)] if $V \in \mathcal{V}$ then there exists $W \in \mathcal{V}$ such that $WW \subset V$;
\item[(4)] if $V \in \mathcal{V}$ then $V^{-1} \in \mathcal{V}$;
\item[(5)] if $V \in \mathcal{V}$ and $a \in G$ then $aVa^{-1} \in \mathcal{V}$;
\item[(6)] if $V \in \mathcal{V}$ and $W \supset V$ then $W \in \mathcal{V}$.
\end{enumerate}
