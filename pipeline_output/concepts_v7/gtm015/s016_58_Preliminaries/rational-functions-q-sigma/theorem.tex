If $\sigma$ is a nonempty subset of $\mathfrak{S}$ (the Riemann sphere), write $Q(\sigma)$ for the set of all restrictions $f|\sigma$ where $f$ is a rational function whose poles lie outside $\sigma$. Equivalently, $f \in Q(\sigma)$ if $f: \sigma \to \mathbb{C}$ is finite-valued and extends to a rational function on $\mathfrak{S}$ with poles in $\mathfrak{S} \setminus \sigma$.
