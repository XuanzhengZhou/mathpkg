For a subset $\sigma$ of the Riemann sphere, the set $Q(\sigma)$ of restrictions of rational functions with poles off $\sigma$ forms a function algebra. This algebra serves as the starting point for the $\sigma$-analytic functional calculus, where rational functions are extended to operator arguments via the Riesz-Dunford functional calculus.
