If $\sigma$ is a spectral set for $T \in \mathcal{L}(E)$, then the rational functional calculus $\Phi: Q(\sigma) \to \mathcal{L}(E)$ extends uniquely to a continuous unital algebra homomorphism $\Phi: \mathcal{A}(\sigma) \to \mathcal{L}(E)$. In particular, $f(T)$ is defined for all $\sigma$-analytic functions $f$.
