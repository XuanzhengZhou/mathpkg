When $\sigma$ is a spectral set for an operator $T$, the rational functional calculus extends continuously to all $\sigma$-analytic functions. This provides a powerful calculus that applies to functions uniformly approximable by rational functions with poles off $\sigma$, generalizing the Riesz-Dunford functional calculus.
