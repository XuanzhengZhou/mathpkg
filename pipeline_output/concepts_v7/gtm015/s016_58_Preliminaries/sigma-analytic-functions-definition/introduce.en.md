The algebra $\mathcal{A}(\sigma)$ of $\sigma$-analytic functions is the uniform closure of rational functions with poles off $\sigma$. This is the maximal class of functions to which the $\sigma$-analytic functional calculus for operators can be extended, generalizing the rational functional calculus via uniform approximation.
