\begin{proposition}
A linear form $f$ on $\mathcal{L}(H)$ is ultraweakly continuous if and only if there exists a pair of sequences $(x_i)$, $(y_i)$ of vectors in $H$, with $\sum_{i=1}^{\infty} \|x_i\|^2 < \infty$ and $\sum_{i=1}^{\infty} \|y_i\|^2 < \infty$, such that
$$f(T) = \sum_{i=1}^{\infty} (Tx_i \mid y_i)$$
for all $T \in \mathcal{L}(H)$.
\end{proposition}
