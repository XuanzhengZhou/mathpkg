The set of all neighborhoods of the identity, directed by reverse inclusion, provides an index set for constructing an approximate identity in $L^1(G)$. For each neighborhood $V$, one chooses a nonnegative continuous function supported in $V$ with integral $1$, yielding a net that converges to an approximate identity.
