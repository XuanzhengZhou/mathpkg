\begin{definition}
For each $x \in H$, the function
$$p_x(T) = \|Tx\| \quad (T \in \mathcal{L}(H))$$
is a seminorm on $\mathcal{L}(H)$. The topology on $\mathcal{L}(H)$ generated by the set of all such seminorms $p_x$ $(x \in H)$ is called the \text{strong operator topology} (or ``strong topology''), denoted $\tau_s$. This is the topology of pointwise convergence on $H$.
\end{definition}
