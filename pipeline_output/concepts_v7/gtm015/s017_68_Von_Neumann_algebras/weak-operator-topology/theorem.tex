\begin{definition}
For each pair of vectors $x, y \in H$, the function
$$\omega_{x,y}(T) = (Tx \mid y) \quad (T \in \mathcal{L}(H))$$
is a linear form on $\mathcal{L}(H)$. The initial topology on $\mathcal{L}(H)$ for the set of all such linear forms $\omega_{x,y}$ $(x, y \in H)$ is called the \text{weak operator topology} (or ``weak topology''), denoted $\tau_w$.
\end{definition}
