Let $E$ be a vector space over $\mathbb{K}$ and let $p, q$ be seminorms on $E$. Define:
\begin{itemize}
\item $p \prec q$ if and only if $p \leq M q$ for some constant $M > 0$;
\item $p \sim q$ if and only if $p \prec q$ and $q \prec p$.
\end{itemize}
Then $\sim$ is an equivalence relation on the set of all seminorms on $E$, and $p \sim q$ if and only if $\tau_p = \tau_q$.
