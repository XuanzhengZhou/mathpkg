Let $(X, \mathcal{S}, \mu)$ be a measure space and let $p \geq 1$ be a fixed real number. Define $\mathcal{L}^p = \mathcal{L}^p(X, \mathcal{S}, \mu)$ as the set of all measurable functions $f: X \to \mathbb{C}$ such that $|f|^p$ is integrable with respect to $\mu$ (i.e., $|f|^p \in \mathcal{L}^1$). For $f \in \mathcal{L}^p$, define
$$\|f\|_p = \left( \int |f|^p \, d\mu \right)^{1/p}.$$
If $p > 1$, the conjugate exponent $q$ is defined by $q = p/(p-1)$, so that $q > 1$ and $p^{-1} + q^{-1} = 1$ (equivalently, $p + q = pq$).
