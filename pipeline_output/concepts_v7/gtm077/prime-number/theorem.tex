An integer $a$ (different from $\pm 1$) is called a prime number (or prime) if its only integral factorizations are the trivial ones, i.e., if $a = bc$ with integers $b, c$ implies $b = \pm 1$ or $c = \pm 1$. The units $\pm 1$ are not counted as primes.

Equivalently, for a prime $p$: if $p \mid ab$, then $p \mid a$ or $p \mid b$.
