If $ab = ac$, then if $a \neq 0$, $b = c$.

To see this we determine an ideal $m$ such that $am = (\delta)$ is a principal ideal. Then

$$amb = amc, \quad (\alpha)b = (\alpha)c.$$

The latter equation asserts that $\alpha$ times every number from $b$ is of the form $\alpha$ times a number from $c$, that is, every number of $b$ belongs to $c$, and likewise the converse is true; thus $b = c$.

And now we obtain a new definition of divis

Conversely if every number in $a$ is also a number in $c$, then for all integers $\lambda_{ik}$ there exist integers $\mu_{pk}$ for which

$$\sum_i \lambda_{ik} \alpha_i = \sum_p \mu_{pk} \gamma_p;$$

then for each $d = (\delta_1, \ldots, \delta_s)$

$$\sum_k \sum_i \lambda_{ik} \alpha_i \delta_k = \sum_k \sum_p \mu_{pk} \gamma_p \delta_k,$$

that is, each number in $a d$ belongs to $c d$. Now let us choose $d$ so that $c d = (\delta)$ is a principal ideal $(\delta \neq 0)$. If $a d = (\rho_1, \rho_2, \ldots)$, then each $\rho_i$ is a number from $(\delta)$; thus it is of the form $\lambda_i \delta$ with integral $\lambda_i$ and hence

$$(\rho_1, \rho_2, \ldots) = (\delta)(\lambda_1, \lambda_2, \ldots),$$

$$a d = c d \cdot (\lambda_1, \lambda_2, \ldots),$$

$$a = c \cdot (\lambda_1, \lambda_2, \ldots),$$

i.e., $c | a$.

As an immediate consequence of this theorem we emphasize:

Let $a$ be an ideal which is not $= (0)$.

The integer $\alpha$ occurs in $a$ if and only if $a | (\alpha)$. If $a | (\alpha)$ and $a | (\beta)$, then also $a | (\lambda \alpha + \mu \beta)$ for all integers $\lambda, \mu$.

It follows from $a b = (1)$ that $a = (1)$ and $b = (1)$.

If each of two ideals is a divisor of the other, then they are equal.
