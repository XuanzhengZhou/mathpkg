Two ideals $\mathfrak{a}$ and $\mathfrak{b}$ are called equivalent ($\mathfrak{a} \sim \mathfrak{b}$) if there exist nonzero integers $\alpha, \beta$ in $K$ such that $\alpha\mathfrak{a} = \beta\mathfrak{b}$. The equivalence classes form a finite Abelian group, the ideal class group of $K$. Its order $h$ is the class number of $K$. The class group is trivial ($h=1$) iff every ideal is principal, i.e., the ring of integers is a UFD.
