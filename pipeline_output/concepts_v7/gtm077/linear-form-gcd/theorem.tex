The range of values of an arbitrary linear form in $n$ variables with integral coefficients, not all vanishing,
$$a_1x_1 + a_2x_2 + \cdots + a_nx_n$$
where $x_i$ run through all integers, is identical with the range of values of a single form $d \cdot x$, where $d$ is the GCD of the coefficients $a_1,\ldots,a_n$.
