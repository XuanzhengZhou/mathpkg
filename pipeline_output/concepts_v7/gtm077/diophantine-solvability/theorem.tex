The range of values of an arbitrary linear form $a_1 x_1 + a_2 x_2 + \cdots + a_n x_n$ (with fixed integral coefficients $a_i$, not all zero) as $x_1, \ldots, x_n$ run through all integers is identical with the set of multiples of $d$, where $d = (a_1, a_2, \ldots, a_n)$.

Consequently, the Diophantine equation
$$k = a_1 x_1 + a_2 x_2 + \cdots + a_n x_n$$
is solvable in integers $x_1, \ldots, x_n$ if and only if $d \mid k$.
