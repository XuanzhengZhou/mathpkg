For a rational integer $n$, the Gauss sum is defined as
$$G(n, d) = \sum_{x \bmod |n|} \left(\frac{x}{n}\right) e^{2\pi i x d / n},$$
where $\left(\frac{x}{n}\right)$ is the Jacobi symbol. The fundamental property is
$$G^2(1, d) = d.$$
More generally, $G(m, d) = \left(\frac{m}{|d|}\right) G(1, d)$ for $m$ coprime to $d$.
