For a prime ideal $\mathfrak{p}$ not dividing $2$ in an arbitrary algebraic number field $k$, and for an integer $\alpha$ of $k$ not divisible by $\mathfrak{p}$, the quadratic residue symbol $\left(\frac{\alpha}{\mathfrak{p}}\right)$ is defined as the unique value $\pm 1$ such that
$$\left(\frac{\alpha}{\mathfrak{p}}\right) \equiv \alpha^{(N(\mathfrak{p})-1)/2} \pmod{\mathfrak{p}}.$$
This extends multiplicatively to all odd ideals $\mathfrak{a}$: $\left(\frac{\alpha}{\mathfrak{a}}\right) = \prod \left(\frac{\alpha}{\mathfrak{p}_i}\right)^{a_i}$ for $\mathfrak{a} = \prod \mathfrak{p}_i^{a_i}$.
