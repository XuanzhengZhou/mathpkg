No element among the elements $B_1, \ldots, B_r$ can be a $p$th power of an element of $\mathfrak{G}$.

If we had $B_m = C^p$ then the system obtained from the $B_1, \ldots, B_r$ by replacing $B_m$ with $C$ and possibly changing the numbering would also be independent, but obviously of higher rank than the maximal system $B_1, \ldots, B_r$, which is impossible.
