If all elements of a group are powers of a single element $A$, the group is called cyclic, and $A$ is called a generator. The elements of the group are
$$E, A, A^2, \ldots, A^{h-1}, \quad A^h = E.$$

**Theorem 28.** A finite Abelian group $\mathfrak{G}$ of order $h$ is cyclic if and only if for each prime $p \mid h$, the number of elements $A$ with $A^p = E$ is exactly $p$ (i.e., the basis number $e(p) = 1$ for all $p \mid h$).
