A complete residue system modulo $n$ is a full set of representatives, one from each of the $|n|$ residue classes. The simplest example is $\{0, 1, \ldots, n-1\}$, but any $|n|$ pairwise incongruent integers suffice. These systems are essential for constructing residue class groups and for proving counting theorems in modular arithmetic.
