A system of complex numbers is called a number field (or, more briefly, a field) if it contains more than one number and if along with the numbers $\alpha$ and $\beta$ it always contains $\alpha + \beta$, $\alpha - \beta$, $\alpha\beta$, and, if $\beta \neq 0$, $\alpha/\beta$.
