If $q$ is a prime factor of $p-1$, $p$ an odd prime, and $(a, p) = 1$, then the congruence $x^q \equiv a \pmod{p^\alpha}$ is solvable if and only if it is solvable modulo $p$, which is equivalent to
$$a^{(p-1)/q} \equiv 1 \pmod{p}.$$
When solvable, there are $q$ solutions incongruent modulo $p^\alpha$.
