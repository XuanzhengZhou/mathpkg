Let $p$ be an odd prime ideal in $k$, $\alpha$ an arbitrary integer in $k$ which is not divisible by $p$. We call $\alpha$ a quadratic residue mod $p$ and set

$$\left( \frac{\alpha}{p} \right) = +1,$$

if there is an integer $\xi$ in $k$ such that $\alpha \equiv \xi^2 \pmod{p}$. In the other case we call $\alpha$ a quadratic nonresidue mod $p$ and we set

$$\left( \frac{\alpha}{p} \right) = -1.$$

Finally we set

$$\left( \frac{\alpha}{p} \right) = 0 \quad \text{if } \alpha \equiv 0 \pmod{p}.$$

By Theorem 84 we see, as in §16, that for each integer $\alpha$ the symbol $\left( \frac{\alpha}{p} \right)$ denotes that one of the three numbers 0, 1, $-1$ for which

$$\alpha^{(N(p)-1)/2} \equiv \left( \frac{\alpha}{p} \right) \pmod{p}. \tag{164}$$

If we have to deal with residue symbols in different number fields, we will distinguish them by attaching an index.

For integers $\alpha, \beta$ we have, once again,

$$\left( \frac{\alpha}{p} \right) = \left( \frac{\beta}{p} \right), \quad \text{if } \alpha \equiv \beta \pmod{p}$$

$$\left( \frac{\alpha \cdot \beta}{p} \right) = \left( \frac{\alpha}{p} \right) \left( \frac{\beta}{p} \right).$$

We now assign a sum to each nonzero integer or fraction $\omega$ of $k$ in the following manner:

Let $d$ denote the different of $k$ and let $d\omega$ be represented as the quotient of relatively prime integral ideals $a$ and $b$:

$$\omega = \frac{b}{a\delta}; \quad (a, b) = 1.$$

By Theorem 101 the trace $S(v\omega)$ is a rational integer for each integer $v$ which is divisible by $a$. Consequently for integral $v$ the number

$$e^{2\pi i S(v\omega)}$$

depends only on the residue class to which $v$ belongs mod $a$. If we now form the sum

$$C(\omega) = \sum_{\mu \bmod a} e^{2\pi i S(\mu^2\omega)},$$

(166)

where $\mu$ runs through some complete system of residues mod $a$, then we obtain a number which depends only on $\omega$, and which is independent of the special choice of the system of residues. We call such a sum a Gauss sum in $k$ which belongs to the denominator $a$. We agree here that an addition to the $\sum$ sign, like “$\mu \bmod a$” is to mean that the summation letter $\mu$ is to run through a complete system of residues mod $a$ possibly with further side conditions which will be stated.

In the rational number field these $C(\omega)$ are formally different from the Gauss sums defined in §52; however, as we will see at once, the latter can be reduced to the $C(\omega)$. If the denominator $a$ is $= 1$, then obviously $C(\omega) = 1$.
