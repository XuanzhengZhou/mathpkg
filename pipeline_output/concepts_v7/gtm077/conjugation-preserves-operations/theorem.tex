For numbers $\alpha, \beta$ in $K(\theta)$,
$$\alpha_i \pm \beta_i = (\alpha \pm \beta)_i, \quad
\alpha_i \beta_i = (\alpha\beta)_i, \quad
\frac{\alpha_i}{\beta_i} = \left(\frac{\alpha}{\beta}\right)_i$$
for $i = 1, 2, \ldots, n$. That is, the map $\alpha \mapsto \alpha_i$ is a field isomorphism of $K(\theta)$ onto $K(\theta_i)$. Moreover, if a polynomial relation with coefficients in $k$ holds among several numbers of $K(\theta)$, the same relation holds for each set of corresponding conjugates.
