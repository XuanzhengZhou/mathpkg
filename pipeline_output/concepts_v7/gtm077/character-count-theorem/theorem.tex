There are exactly $h$ distinct functions $\chi(A)$ which have the property: $\chi(AB) = \chi(A) \cdot \chi(B)$ and $\chi(A)$ is not $= 0$ for all elements $A$ of $\mathfrak{G}$. Each $\chi$ is an $h$th-root of unity.

Each such function $\chi(A)$ is called a group character or character of $\mathfrak{G}$.

Among the characters $\chi(A)$ there is one which is $= 1$ for all $A$; it is called the principal character. Conversely, there exists exactly one element, namely $E$, such that $\chi(E) = 1$ for every character.

The element $A$ is determined uniquely by the $h$ numbers $\chi_n(A)$, where $\chi_n$ are the $h$ characters for $n = 1, 2, \ldots, h$. For if a second element $B$ had the same values $\chi_n(B)$, then we would have $\chi_n(AB^{-1}) = 1$ for all $n$, and $AB^{-1}$ would be the unit element, thus $A = B$.
