In this way the class number is connected with the decomposition laws, and thereby in certain cases a remarkably simple expression for the class number can be derived, of which no other way has led until now.

If $F(m)$ denotes the number of integral ideals of the field whose norm is equal to the positive number $m$, then obviously

$$Z(t) = \sum_{m=1}^{t} F(m).$$

Here $\sum_{m=1}^{t}$ means that the summation index $m$ runs through all rational integers for which $1 \leq m \leq t$. Now, moreover, for two rational integers $a, b$

$$F(ab) = F(a) \cdot F(b) \quad \text{if } (a, b) = 1. \tag{89}$$

For, from two integral ideals $a$ and $b$ with $N(a) = a, N(b) = b$, an ideal $c = ab$ arises with $N(c) = ab$. And conversely if $c$ is an integral ideal with norm $ab$, let us set

$$(c, a) = a_1, \quad (c, b) = b_1; \tag{90}$$

from this it follows by multiplication that

$$a_1 b_1 = (c^2, c a, c b, ab) = c \left( c, a, b, \frac{ab}{c} \right) = c.$$

By passage to the conjugate, we obtain from (90) that $N(a_1)$, as a divisor of $a^n$, is thus coprime to $b$ and $N(b_1)$ is coprime to $a$, while the product $N(a_1) \cdot N(b_1) = ab$. Consequently, $N(a_1) = a, N(b_1) = b$, and $c$ is thus decomposed into two factors whose norms are $a$ and $b$ respectively. The assertion (89) follows from this.
