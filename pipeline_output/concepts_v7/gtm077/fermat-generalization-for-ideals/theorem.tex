For each integer $\alpha$ in $K$ and each prime ideal $\mathfrak{p}$,
$$\alpha^{N(\mathfrak{p})} \equiv \alpha \pmod{\mathfrak{p}}.$$
If $\alpha$ is not divisible by $\mathfrak{p}$, then
$$\alpha^{N(\mathfrak{p})-1} \equiv 1 \pmod{\mathfrak{p}}.$$
More generally, for any ideal $\mathfrak{a}$, if $(\alpha, \mathfrak{a}) = 1$ then $\alpha^{\varphi(\mathfrak{a})} \equiv 1 \pmod{\mathfrak{a}}$.
