For an odd prime $p$ and integer $a$, the Legendre symbol $\left(\frac{a}{p}\right)$ is defined as:
$$\left(\frac{a}{p}\right) = \begin{cases}
+1 & \text{if $a$ is a quadratic residue mod $p$ and } p \nmid a,\\
-1 & \text{if $a$ is a quadratic nonresidue mod $p$},\\
0 & \text{if } p \mid a.
\end{cases}$$
Equivalently,
$$\left(\frac{a}{p}\right) \equiv a^{(p-1)/2} \pmod{p}.$$
