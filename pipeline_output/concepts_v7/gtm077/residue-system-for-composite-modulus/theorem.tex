If $a_1, a_2, \ldots, a_n$ are pairwise relatively prime integers, then a complete residue system modulo $A = a_1 a_2 \cdots a_n$ is obtained in the form
$$L(x_1, \ldots, x_n) = \frac{A}{a_1} c_1 x_1 + \frac{A}{a_2} c_2 x_2 + \cdots + \frac{A}{a_n} c_n x_n$$
if the $x_i$ independently run through a complete residue system modulo $a_i$, where the $c_i$ are arbitrary integers relatively prime to $a_i$.
