If $\alpha$ and $\beta$ are ideals distinct from $(0)$, then there is always a number $\omega$ for which

$$(\omega, \alpha b) = \alpha.$$

This $\omega$ then obviously has a decomposition $\omega = \alpha c$ where $(c, b) = 1$. Thus the theorem asserts that each $\alpha$ can be made into a principal ideal by multiplication with such a $c$ which is relatively prime to the given $b$.
