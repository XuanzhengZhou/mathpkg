For a quadratic field $k = \mathbb{Q}(\sqrt{d})$ with discriminant $d$, the class number $h$ is given by:
\begin{itemize}
\item If $d < 0$: $\displaystyle h = \frac{w}{2|d|}\left|\sum_{n=1}^{|d|-1} n\left(\frac{n}{d}\right)\right|$,
\item If $d > 0$: $\displaystyle h = \frac{1}{\log\varepsilon_0}\left|\sum_{n=1}^{d-1} \left(\frac{n}{d}\right)\log\sin\frac{n\pi}{d}\right|$,
\end{itemize}
where $w$ is the number of roots of unity ($w=2$ for $d<-4$, $w=4$ for $d=-4$, $w=6$ for $d=-3$) and $\varepsilon_0 > 1$ is the fundamental unit for real quadratic fields.
