In a finite group, the powers $A, A^2, A^3, \ldots$ of an element $A$ cannot all be distinct. The set of exponents $q$ for which $A^q = E$ forms a module of integers, hence consists of all multiples of a unique positive integer $a$, called the order of $A$. Thus:
$$A^r = E \iff a \mid r.$$
The order $a$ is also the number of distinct powers of $A$, i.e., the size of the cyclic subgroup generated by $A$.
