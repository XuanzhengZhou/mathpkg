A system of complex numbers is called a number field (or simply a field) if it contains more than one number and if along with $\alpha$ and $\beta$ it always contains $\alpha + \beta$, $\alpha - \beta$, $\alpha\beta$, and, if $\beta \neq 0$, $\alpha/\beta$. The term domain of rationality (Kronecker) is also used. Every number field contains the field of rational numbers $k(1)$ as a subfield.
