An integral polynomial $f(x)$ of degree $k$ has no more than $k$ incongruent roots modulo a prime $p$, unless $f(x) \equiv 0 \pmod{p}$, in which case all coefficients are divisible by $p$.

The theorem is true for the polynomials of degree 0, the constants. For if $f(x) = c_0$ is independent of $x$, then $f(x) \equiv 0 \pmod{p}$ has either 0 solutions—when $p$ does not divide $c_0$—or it has more than 0 solutions—namely every integer if $c_0$ is divisible by $p$, that is, the polynomial $f(x) \equiv 0 \pmod{p}$.
