For two integral polynomials $f(x)$ and $g(x)$, $f(x)$ is said to be divisible by $g(x)$ mod $n$ if there is an integral polynomial $g_1(x)$ such that

$$f(x) \equiv g(x)g_1(x) \pmod{n}.$$

If moreover $a$ is an integer such that

$$f(a) \equiv 0 \pmod{n},$$

then $a$ is called a root of $f(x)$ mod $n$.

If $a$ is a root of $f(x)$ mod $n$ and $a \equiv b \pmod{n}$, then obviously $b$ is also a root of $f(x)$ mod $n$.
