The $n$ numbers $\omega^{(1)}, \ldots, \omega^{(n)}$ in $K(\theta)$ form a fundamental system if and only if there is no linear relation

$$\sum_{i=1}^{n} u_i \omega^{(i)} = 0$$

with coefficients in $k$ except if all $u_i = 0$.

The $n$ numbers $\omega^{(i)}$ of this type are said to be linearly independent. For, in the notation used above it would follow from (37) that

$$0 = \sum_{i=1}^{n} u_i \sum_{k=1}^{n} c_{ik} \theta^{k-1}$$

and as before, if the $u_i$ belong to $k$ and are not all $= 0$,

$$\sum_{i=1}^{n} u_i c_{ik} = 0, \quad (k=1, \ldots, n);$$

thus

$$\|\copyright_{ik}\| = 0.$$

$$\sum_{i=1}^{n} u_i c_{ik} = 0 \quad (k=1, \ldots, n)$$
