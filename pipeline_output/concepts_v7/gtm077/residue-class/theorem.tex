The relation $a \equiv b \pmod{n}$ partitions the integers into $|n|$ equivalence classes, called residue classes modulo $n$. Two integers belong to the same residue class if and only if they are congruent modulo $n$. Each residue class consists of all integers of the form $a + kn$ for a fixed representative $a$ and $k \in \mathbb{Z}$.

A complete system of residues modulo $n$ is a set of $|n|$ integers, one from each residue class.
