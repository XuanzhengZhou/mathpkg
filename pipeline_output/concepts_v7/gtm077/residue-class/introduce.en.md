A residue class modulo $n$ is an equivalence class of the congruence relation. The integers are partitioned into exactly $|n|$ such classes, which form the ring $\mathbb{Z}/n\mathbb{Z}$ under addition and multiplication of representatives. The concept of a complete residue system — one representative from each class — is fundamental for counting arguments in modular arithmetic.
