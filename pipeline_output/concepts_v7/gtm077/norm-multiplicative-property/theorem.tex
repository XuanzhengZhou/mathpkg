For two ideals $a$ and $b$, we always have

$$N(a b) = N(a) \cdot N(b).$$

Let $\alpha$ be a number divisible by $a$ such that $(\alpha, a b) = a$. If we let $\xi_i$ ($i = 1, 2, \ldots, N(b)$) run through a complete system of residues mod $b$ and let $\eta_k$ ($k = 1, 2, \ldots, N(a)$) run through a complete system of residues mod $a$, then no two of the numbers $\alpha \xi_i + \eta_k$ are congruent mod $a b$. On the other hand, each integer $\rho$ is congruent mod $a b$ to one of these numbers $\alpha \xi_i + \eta_k$. For let $\eta_k$ be determined so that

$$\eta_k \equiv \rho \pmod{a}$$

and then let $\xi$ be determined so that

$$\alpha \xi \equiv \rho - \eta_k \pmod{a b}.$$

Since $(\alpha, a b) = a$ and $a | \rho - \eta_k$ this congruence can be solved by Theorem 78 and $\xi$ can be determined mod $b$ so that $\xi$ can be chosen equal to $\xi_i$. Consequently the $N(a) \cdot N(b)$ numbers $\alpha \xi_i + \eta_k$ form a complete system of residues mod $a b$ and thus there must also be $N(a b)$ of them.
