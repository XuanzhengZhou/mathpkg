A system $S$ of elements $A, B, C, \ldots$ is called a group if the following conditions hold:

\begin{enumerate}
\item[(i)] There is a rule of composition (binary operation) such that from elements $A$ and $B$ a unique element $C \in S$ is obtained, written $C = AB$.

\item[(ii)] Associative law: For all $A, B, C \in S$,
$$A(BC) = (AB)C.$$

\item[(iii)] Cancellation laws: For all $A, A', B \in S$,
$$AB = A'B \implies A = A',$$
$$BA = BA' \implies A = A'.$$

\item[(iv)] Solvability of equations: For all $A, B \in S$, there exist $X, Y \in S$ such that
$$AX = B, \quad YA = B.$$
\end{enumerate}

If $S$ contains only finitely many elements, condition (iv) follows from (i) and (iii).
