In an algebraic number field $K(\theta)$ of degree $n$ (with $\theta$ an integer), the set of all integers of the field forms a module of rank $n$. There exist $n$ integers $\omega_1, \ldots, \omega_n$ such that every integer $\alpha$ in $K$ can be represented uniquely as
$$\alpha = x_1 \omega_1 + x_2 \omega_2 + \cdots + x_n \omega_n$$
with rational integers $x_i$. Such a set $\{\omega_i\}$ is called an integral basis (or minimal basis) of the field.
