Since each class complex consists of $h_0/2^{e_0}$ classes in the strict sense, the odd ideals $r$ with $Q(\omega, r) = +1$ are identical with the odd ideals which lie in the $h_0/2$ strict classes of this group $\mathfrak{G}(\omega)$.
