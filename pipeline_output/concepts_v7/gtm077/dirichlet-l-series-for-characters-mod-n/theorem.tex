For each residue character $\chi$ modulo $n$, the Dirichlet $L$-series
$$L(s, \chi) = \sum_{m=1}^{\infty} \frac{\chi(m)}{m^s}$$
converges for $\Re(s) > 0$ when $\chi$ is not the principal character. If $\chi$ is not principal, $L(1, \chi) \neq 0$. For the principal character $\chi_0$,
$$L(s, \chi_0) = \zeta(s) \prod_{p \mid n} \left(1 - \frac{1}{p^s}\right),$$
which has a simple pole at $s=1$.
