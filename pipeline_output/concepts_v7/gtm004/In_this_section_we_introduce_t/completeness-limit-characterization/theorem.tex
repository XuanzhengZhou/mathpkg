Let $\mathfrak{A}$ be an abelian category and let
$$\cdots \subseteq X^{p-1} \subseteq X^p \subseteq \cdots \subseteq X$$
be a filtered object. Define $X_q^p = X^p / X^q$ for $q < p$, with the natural maps $\sigma_q^p : X_{q-1}^p \to X_q^p$ and $\varrho_q^p : X_q^p \to X_q^{p+1}$.

\begin{enumerate}
\item If the filtration is right complete, then for each $p$,
$$X^p = \varprojlim_q (X_q^p, \sigma_q^p).$$

\item If the filtration is left complete, then for each $q$,
$$X_q = \varinjlim_p (X_q^p, \varrho_q^p).$$
\end{enumerate}
