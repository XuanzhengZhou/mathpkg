The homomorphic relation $\Theta$ from $E_\infty$ to $\Gamma^+$ is an isomorphism if and only if

$$\operatorname{coker} \bar{\alpha}' = 0, \quad \ker \alpha'' = 0.$$

For a filtered (graded) differential group $C$, these conditions are equivalent to:

$$I_p \cap \alpha^{-1}(0) = 0, \quad \bar{\alpha}' \bar{U}_p = \bar{U}_{p+1}, \quad \text{for all } p,$$

where $\bar{\alpha}'$ is induced by $\bar{\alpha}$.
