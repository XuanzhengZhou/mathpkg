Let $X$ be a filtered object in an abelian category $\mathfrak{A}$ with filtration
$$\cdots \subseteq X^{p-1} \subseteq X^p \subseteq \cdots \subseteq X.$$
Suppose that for all $p$, the inclusion $\nu^p : X^p \to X$ factorizes as
$$X^p \xrightarrow{\nu^p} Y \xrightarrow{\mu} X,$$
where $\mu$ is independent of $p$. Set $Y_p = Y/X^p$ and let $\eta'_p : Y \to Y_p$ be the projection. Then $\xi_p : X_p \to X_{p+1}$ induces $\xi'_p : Y_p \to Y_{p+1}$, and the resulting diagram is bicartesian.

If $(X; \eta_p) = \varprojlim (X_p, \xi_p)$, then $(Y; \eta'_p) = \varprojlim (Y_p, \xi'_p)$.
