Let $\varphi : A \to B$ and $\psi : B \to C$ be homomorphisms of $\Lambda$-modules. The sequence
$$A \xrightarrow{\varphi} B \xrightarrow{\psi} C$$
is called \textbf{exact} (at $B$) if
$$\ker \psi = \operatorname{im} \varphi.$$

A sequence
$$A_0 \to A_1 \to \cdots \to A_n \to A_{n+1}$$
is called \textbf{exact} if it is exact at $A_1, \ldots, A_n$.

As special cases:
\begin{itemize}
  \item[(a)] $0 \to A \xrightarrow{\varphi} B$ is exact (at $A$) if and only if $\varphi$ is injective.
  \item[(b)] $A \xrightarrow{\varphi} B \to 0$ is exact (at $B$) if and only if $\varphi$ is surjective.
\end{itemize}
