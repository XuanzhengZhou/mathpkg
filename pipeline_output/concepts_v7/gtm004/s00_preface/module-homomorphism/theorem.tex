Let $A, B$ be $\Lambda$-modules. A \textbf{homomorphism} (or \textbf{map}) of $\Lambda$-modules is a homomorphism of abelian groups $\varphi : A \to B$ such that
$$\varphi(\lambda a) = \lambda(\varphi a)$$
for all $a \in A$, $\lambda \in \Lambda$.

The identity map $1_A : A \to A$ is a homomorphism of $\Lambda$-modules.

If $\varphi$ is surjective, we write $\varphi : A \twoheadrightarrow B$. If $\varphi$ is injective, we write $\varphi : A \rightarrowtail B$. We call $\varphi$ an \textbf{isomorphism} and write $\varphi : A \xrightarrow{\cong} B$ if there exists a homomorphism $\psi : B \to A$ such that $\psi \varphi = 1_A$ and $\varphi \psi = 1_B$.
