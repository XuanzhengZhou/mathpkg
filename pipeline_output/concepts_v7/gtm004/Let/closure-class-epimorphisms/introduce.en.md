The closure operation on classes of epimorphisms adds all epimorphisms against which $\mathcal{E}$-projective objects automatically lift. A closed class satisfies $\mathcal{E} = C(\mathcal{E})$ and is the natural setting for relative homological algebra. Closed projective classes provide the right framework for defining derived functors and satellites in the relative context.
