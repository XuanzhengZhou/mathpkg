A \textit{homotopy} $\Sigma : \varphi \to \psi$ between two chain maps $\varphi, \psi : C \to D$ is a morphism of degree $+1$ of graded modules $\Sigma : C \to D$ such that
$$\psi - \varphi = \partial \Sigma + \Sigma \partial,$$
i.e., for all $n \in \mathbb{Z}$,
$$\psi_n - \varphi_n = \partial_{n+1} \Sigma_n + \Sigma_{n-1} \partial_n.$$
We say $\varphi$ and $\psi$ are \textit{homotopic}, written $\varphi \simeq \psi$, if such a $\Sigma$ exists.
