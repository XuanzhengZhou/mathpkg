A $\Lambda$-module is \textit{cofree} if it is the direct product of modules $\Lambda^* = \operatorname{Hom}_{\mathbb{Z}}(\Lambda, \mathbb{Q}/\mathbb{Z})$. Note that this is consistent with the description of $\mathbb{Q}/\mathbb{Z}$ as a cofree abelian group, since $\operatorname{Hom}_{\mathbb{Z}}(\mathbb{Z}, \mathbb{Q}/\mathbb{Z}) \cong \mathbb{Q}/\mathbb{Z}$.
