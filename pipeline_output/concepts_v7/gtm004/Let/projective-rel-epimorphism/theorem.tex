Let $\mathfrak{U}$ be an abelian category and let $\mathcal{E}$ be a class of epimorphisms in $\mathfrak{U}$. An object $P$ in $\mathfrak{U}$ is called \emph{projective rel $\varepsilon$}, where $\varepsilon : B \to C$ is in $\mathcal{E}$, if
$$\varepsilon_* : \mathfrak{U}(P, B) \longrightarrow \mathfrak{U}(P, C)$$
is surjective. $P$ is called \emph{$\mathcal{E}$-projective} if it is projective rel $\varepsilon$ for every $\varepsilon \in \mathcal{E}$.
