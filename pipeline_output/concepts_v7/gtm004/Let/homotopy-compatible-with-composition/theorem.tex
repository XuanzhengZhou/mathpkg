Let $\varphi \simeq \psi : C \to D$ and $\varphi' \simeq \psi' : D \to E$ be chain maps. Then
$$\varphi' \varphi \simeq \psi' \psi : C \to E.$$
In particular, if $\varphi \simeq \psi$, then for any chain map $\alpha$, $\alpha \varphi \simeq \alpha \psi$ and $\varphi \alpha \simeq \psi \alpha$ whenever the compositions are defined.
