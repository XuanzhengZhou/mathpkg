Theorem 9.2 proves the existence of a maximal essential extension of a module $A$ inside an injective module containing it. The proof uses Zorn's Lemma on the inductive set of essential extensions of $A$ contained in the given injective module. The maximal element is shown to be injective by a splitting argument, establishing the link between essential extensions and injective hulls.
