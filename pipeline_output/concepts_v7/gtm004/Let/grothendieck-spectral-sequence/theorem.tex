Let $\mathfrak{A} \xrightarrow{F} \mathfrak{B} \xrightarrow{G} \mathfrak{C}$ be additive functors between abelian categories with enough injectives. Assume that if $I$ is an injective object of $\mathfrak{A}$, then $F(I)$ is $G$-acyclic (i.e., $R^q G(F(I)) = 0$ for $q \geq 1$). Then for each object $A$ of $\mathfrak{A}$ there is a spectral sequence $\{E_n(A)\}$ with
$$E_1^{p,q} = (R^p G)(R^{q-p} F)(A) \Rightarrow R^q(GF)(A),$$
which converges finitely to the graded object associated with $\{R^q(GF)(A)\}$, suitably filtered.
