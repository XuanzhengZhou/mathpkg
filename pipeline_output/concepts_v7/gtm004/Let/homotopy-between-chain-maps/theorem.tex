Let $\varphi, \psi : C \to D$ be two chain maps between chain complexes $C$ and $D$. A \emph{homotopy} $\Sigma : \varphi \to \psi$ is a morphism of graded modules of degree $+1$, $\Sigma : C \to D$, such that
$$\psi - \varphi = \partial_D \Sigma + \Sigma \partial_C,$$
i.e., for each $n \in \mathbb{Z}$,
$$\psi_n - \varphi_n = \partial_{n+1} \Sigma_n + \Sigma_{n-1} \partial_n.$$
If such a $\Sigma$ exists, we say $\varphi$ and $\psi$ are \emph{homotopic}, written $\varphi \simeq \psi$. A \emph{contracting homotopy} is a homotopy $\Sigma : 0 \to 1_C$, satisfying $\partial \Sigma + \Sigma \partial = 1$.
