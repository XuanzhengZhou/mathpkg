Any two projective resolutions $C$ and $D$ of a module $A$ are canonically of the same homotopy type. That is, there exist chain maps $\varphi : C \to D$ and $\psi : D \to C$, both inducing the identity on $H_0 = A$, such that $\psi \varphi \simeq 1_C$ and $\varphi \psi \simeq 1_D$. Moreover, the homotopy class of the homotopy equivalence $\varphi$ is uniquely determined.
