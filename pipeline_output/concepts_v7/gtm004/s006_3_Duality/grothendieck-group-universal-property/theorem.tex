Let $S$ be an abelian semigroup. Define an equivalence relation on $S \times S$ by $(a,b) \sim (c,d)$ if and only if there exists $u \in S$ such that $a + d + u = b + c + u$. The \textit{Grothendieck group} $\operatorname{Gr}(S) = (S \times S)/{\sim}$ is an abelian group with $[a,b] + [c,d] = [a+c, b+d]$. The map $\iota : S \to \operatorname{Gr}(S)$ given by $\iota(a) = [a,0]$ is a homomorphism, and $\iota$ is injective if and only if $S$ is a cancellation semigroup.

Moreover, $\iota$ has the universal property: for any abelian group $A$ and homomorphism $\sigma : S \to A$, there exists a unique group homomorphism $\tau : \operatorname{Gr}(S) \to A$ such that $\tau \iota = \sigma$. Thus $\operatorname{Gr}$ is left adjoint to the forgetful functor from abelian groups to abelian semigroups.
