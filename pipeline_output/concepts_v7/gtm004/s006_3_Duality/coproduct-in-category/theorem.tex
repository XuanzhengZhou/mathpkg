Let $\{X_i\}_{i \in I}$ be a family of objects in $\mathfrak{C}$ indexed by a set $I$. A \textit{coproduct} of the $X_i$ is an object $X$ together with morphisms $q_i : X_i \to X$ (called \textit{injections}) such that for any object $Y$ and morphisms $f_i : X_i \to Y$, there exists a unique morphism $f : X \to Y$ satisfying $f q_i = f_i$ for all $i \in I$. Equivalently, $(X; q_i)$ is a coproduct in $\mathfrak{C}$ if and only if it is a product in $\mathfrak{C}^{\text{opp}}$. We write $X = \coprod_i X_i$ and $f = \langle f_i \rangle$.
