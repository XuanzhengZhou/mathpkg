For a small category $\mathfrak{C}$, the assignment $A \mapsto \mathfrak{C}(A, -)$ is a full embedding
$$
\mathfrak{C}^{\text{opp}} \hookrightarrow \mathfrak{S}^{\mathfrak{C}},
$$
called the \textit{Yoneda embedding}. The set of morphisms $\mathfrak{C}(A', A)$ is in natural bijection with the set of natural transformations from $\mathfrak{C}(A, -)$ to $\mathfrak{C}(A', -)$.
