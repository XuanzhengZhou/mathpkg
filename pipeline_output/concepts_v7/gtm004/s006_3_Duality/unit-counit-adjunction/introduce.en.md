An adjunction can be equivalently described by two natural transformations: the unit $\varepsilon : 1_{\mathfrak{C}} \to GF$ and the counit $\delta : FG \to 1_{\mathfrak{D}}$, satisfying triangular identities. This formulation is often more convenient for computations. The unit embeds each object into its "free" envelope, while the counit provides the "evaluation" map from the free object back.
