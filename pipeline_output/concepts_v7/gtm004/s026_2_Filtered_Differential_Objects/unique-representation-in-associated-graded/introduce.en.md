When a filtered object $M$ satisfies both the separatedness condition (intersection of all filtration levels is zero) and the exhaustiveness condition (union of all filtration levels equals $M$), every non-zero element acquires a unique filtration degree. This degree is the first $p$ for which the element belongs to $M^{(p)}$. Under these conditions, there is a one-to-one correspondence between non-zero elements of $M$ and homogeneous non-zero elements of $Gr(M)$, so that the associated graded faithfully represents the elements of $M$ — only the extension data between successive filtration levels is lost.
