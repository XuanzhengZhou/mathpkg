The associated graded object functor $Gr$ maps a filtered object to the graded object formed by its successive subquotients $B^{(p)} / B^{(p-1)}$. This functor forgets the extension data that relates successive filtration steps, retaining only the homogeneous components. In the context of spectral sequences, the $E$-term is precisely $H \circ Gr$ applied to a filtered differential object.
