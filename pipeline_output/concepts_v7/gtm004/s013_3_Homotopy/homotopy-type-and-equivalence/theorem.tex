\text{Two chain complexes}~C, D~\text{are of the same \text{homotopy type} (or homotopic) if they are isomorphic in the homotopy category, that is, if there exist chain maps}~\varphi : C \rightarrow D~\text{and}~\psi : D \rightarrow C~\text{such that}

$$\psi \varphi \simeq 1_C \quad \text{and} \quad \varphi \psi \simeq 1_D.$$

\text{The chain map}~\varphi~(\text{or}~\psi)~\text{is then called a \text{homotopy equivalence}.}
