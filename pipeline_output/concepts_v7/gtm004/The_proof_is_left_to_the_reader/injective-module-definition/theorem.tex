A $\Lambda$-module $I$ is called \textbf{injective} if for every homomorphism $\alpha : A \rightarrow I$ and every monomorphism $\mu : A \rightarrow B$, there exists a homomorphism $\beta : B \rightarrow I$ such that $\beta \circ \mu = \alpha$. In other words, the following diagram can be completed:

\[
\begin{CD}
0 @>>> A @>\mu>> B\\
@. @V\alpha VV @.\\
@. I @.
\end{CD}
\]

Equivalently, every homomorphism into $I$ can be extended from a given domain $A$ to any larger domain $B$ containing $A$ as a submodule (via the monomorphism $\mu$).
