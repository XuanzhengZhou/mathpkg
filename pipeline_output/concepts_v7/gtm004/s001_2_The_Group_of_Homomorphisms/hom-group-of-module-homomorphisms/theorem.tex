Let $\Lambda$ be a ring and let $A, B$ be $\Lambda$-modules. Let $\operatorname{Hom}_{\Lambda}(A, B)$ denote the set of all $\Lambda$-module homomorphisms from $A$ to $B$. This set has the structure of an abelian group: if $\varphi \colon A \to B$ and $\psi \colon A \to B$ are $\Lambda$-module homomorphisms, then $\varphi + \psi \colon A \to B$ is defined by
$$
(\varphi + \psi)(a) = \varphi(a) + \psi(a) \qquad \text{for all } a \in A.
$$
Note that $\operatorname{Hom}_{\Lambda}(A, B)$ is not, in general, a $\Lambda$-module in any obvious way.
