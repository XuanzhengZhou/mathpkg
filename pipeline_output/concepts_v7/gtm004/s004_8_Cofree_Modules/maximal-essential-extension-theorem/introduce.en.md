Theorem 9.2 establishes the existence of maximal essential extensions. Given a module $A$ embedded in an ambient module $M$, the set of essential extensions of $A$ inside $M$ is inductive under inclusion (using Proposition 9.1 to verify that unions of chains remain essential extensions), so Zorn's Lemma provides a maximal element $E$. When the ambient $M$ is an injective module $I$, this maximal essential extension $E$ has the striking property that it admits no proper essential monomorphism and is a direct summand of $I$, hence is itself injective. This result is the crucial step in constructing the injective envelope.
