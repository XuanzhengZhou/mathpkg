For a $\Lambda$-module $I$ the following statements are equivalent:
\begin{enumerate}
\item[(1)] $I$ is injective.
\item[(2)] For every exact sequence $A \xrightarrow{\mu} B \xrightarrow{\varepsilon} C$ of $\Lambda$-modules, the induced sequence
$$0 \to \operatorname{Hom}_{\Lambda}(C, I) \xrightarrow{\varepsilon^{*}} \operatorname{Hom}_{\Lambda}(B, I) \xrightarrow{\mu^{*}} \operatorname{Hom}_{\Lambda}(A, I) \to 0$$
is exact.
\item[(3)] If $\mu : I \to B$ is a monomorphism, then there exists $\beta : B \to I$ such that $\beta \mu = 1_I$.
\item[(4)] $I$ is a direct summand in every module which contains $I$ as a submodule.
\item[(5)] $I$ is a direct summand in a cofree module.
\end{enumerate}
