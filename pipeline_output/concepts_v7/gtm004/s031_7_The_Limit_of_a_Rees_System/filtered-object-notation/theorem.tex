\textbf{Notation (8.4).} For a filtered object
$$
\cdots \subseteq X^{p-1} \subseteq X^p \subseteq \cdots \subseteq X, \quad -\infty < p < \infty
$$
in an abelian category $\mathfrak{A}$, we write
$$
X^{p-1} \xrightarrow{\xi_p} X^p \xrightarrow{\nu_p} X \xrightarrow{\eta_p} X_p \xrightarrow{\xi_p} X_{p+1},
$$
where $\xi_p, \nu_p$ are the inclusions, $\eta_p$ is the cokernel of $\nu_p$, so that $X_p = X / X^p$.
