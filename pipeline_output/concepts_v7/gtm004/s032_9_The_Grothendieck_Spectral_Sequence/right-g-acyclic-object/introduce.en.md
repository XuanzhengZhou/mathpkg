A right $G$-acyclic object is one on which all higher right derived functors $R^q G$ vanish for $q \geq 1$. This notion generalizes that of an injective object: every injective is $G$-acyclic for any left exact $G$, but $G$-acyclicity is a strictly weaker condition. It provides the essential hypothesis for the Grothendieck spectral sequence, where one requires that $F$ sends injectives to $G$-acyclic objects rather than to injectives. In practice, one often verifies $G$-acyclicity by showing that certain resolutions can be used to compute $R^q G$ directly.
