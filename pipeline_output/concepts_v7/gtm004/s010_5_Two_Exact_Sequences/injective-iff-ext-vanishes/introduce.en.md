Corollary 5.6 is the dual of Corollary 5.5: a module $B$ is injective if and only if $\operatorname{Ext}_\Lambda(A, B) = 0$ for all $A$. The proof is obtained by dualizing the argument for projectivity, using the Hom-Ext sequence in the first variable. This establishes that Ext provides a unified homological test for both projective and injective modules, reflecting the symmetry between the two notions.
