Let the following commutative diagram of $\Lambda$-modules have exact rows.
$$
\begin{CD}
A @>{\mu}>> B @>{\varepsilon}>> C @>>> 0 \\
@V{\alpha}VV @V{\beta}VV @V{\gamma}VV \\
0 @>>> A' @>{\mu'}>> B' @>{\varepsilon'}>> C'
\end{CD}
$$
Then there exists a connecting homomorphism $\omega : \ker \gamma \to \operatorname{coker} \alpha$ such that the following sequence is exact:
$$
\ker \alpha \xrightarrow{\mu_*} \ker \beta \xrightarrow{\varepsilon_*} \ker \gamma \xrightarrow{\omega} \operatorname{coker} \alpha \xrightarrow{\mu'_*} \operatorname{coker} \beta \xrightarrow{\varepsilon'_*} \operatorname{coker} \gamma.
$$
If $\mu$ is monomorphic, so is $\mu_*$; if $\varepsilon'$ is epimorphic, so is $\varepsilon'_*$.
