A divisible module over an integral domain is one in which every element can be divided by any non-zero scalar — for each element $d$ and non-zero scalar $\lambda$, there exists some $c$ satisfying $\lambda c = d$. Uniqueness of $c$ is not required. Over a principal ideal domain, divisibility exactly characterizes injective modules (Theorem 7.1). Classic examples include the additive group of rationals $\mathbb{Q}$ and the quotient $\mathbb{Q}/\mathbb{Z}$ as $\mathbb{Z}$-modules.
