A zero object in a category is an object that is both initial and terminal: there is exactly one morphism from any object to it and exactly one morphism from it to any object. Zero objects exist in many algebraic categories such as groups, abelian groups, and modules (where the trivial group or zero module serves as a zero object). Any two zero objects in a category are isomorphic.
