Let $U^\gamma : \mathfrak{M}_{\Lambda'} \rightarrow \mathfrak{M}_\Lambda$ be the change-of-rings functor induced by the ring homomorphism $\gamma : \Lambda \rightarrow \Lambda'$.

(i) If $\Lambda'$ is projective as a left $\Lambda$-module via $\gamma$, then $U^\gamma$ sends projective $\Lambda'$-modules to projective $\Lambda$-modules.

(ii) If $\Lambda'$ is flat as a right $\Lambda$-module via $\gamma$, then $U^\gamma$ sends injective $\Lambda'$-modules to injective $\Lambda$-modules.
