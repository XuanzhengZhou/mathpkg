Let $U^\gamma : \mathfrak{M}_{\Lambda'} \rightarrow \mathfrak{M}_\Lambda$ be the change-of-rings functor induced by $\gamma : \Lambda \rightarrow \Lambda'$. Then the natural transformation

$$\theta : \operatorname{Ext}_{\Lambda'}^n(A', B') \rightarrow \operatorname{Ext}_\Lambda^n(U^\gamma A', U^\gamma B')$$

constructed via the first adjoint method coincides with the natural transformation

$$\bar{\theta} : \operatorname{Ext}_{\Lambda'}^n(A', B') \rightarrow \operatorname{Ext}_\Lambda^n(U^\gamma A', U^\gamma B')$$

constructed via the second adjoint method.
