Let
\[
D \xrightarrow{\alpha} D \xrightarrow{\beta} E \xrightarrow{\gamma} D
\]
be an exact couple of bigraded objects with $\deg \alpha = (1,0)$, $\deg \beta = (0,0)$, $\deg \gamma = (-1,-1)$.

The map $\alpha : D \to D$ is \text{positively stationary} if, given $q$, there exists $p_0$ such that $\alpha : D^{p,q} \to D^{p+1,q}$ is an isomorphism for $p \geq p_0$.

The map $\alpha$ is \text{negatively stationary} if, given $q$, there exists $p_0$ such that $\alpha : D^{p,q} \to D^{p+1,q}$ is an isomorphism for $p \leq p_0$.

If $\alpha$ is both positively and negatively stationary, it is called \text{stationary}.
