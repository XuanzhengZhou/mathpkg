$E(A, B)$ is the set of all equivalence classes of extensions of the module $A$ by the module $B$. It always contains at least the class of the split extension $A \oplus B$. This set can be endowed with an abelian group structure via the Baer sum, and it is naturally isomorphic to $\operatorname{Ext}^1_A(A, B)$, the first right derived functor of $\operatorname{Hom}_A(-, B)$, thus linking the concrete extension problem to the machinery of homological algebra.
