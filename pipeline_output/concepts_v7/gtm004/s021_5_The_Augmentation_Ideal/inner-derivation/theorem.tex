A derivation $d : G \to A$ is called an \textbf{inner derivation} (or \textbf{principal crossed homomorphism}) if there exists an element $a \in A$ such that
$$d(x) = (x-1)a \quad \text{for all } x \in G.$$
The set of all inner derivations forms a subgroup of $\operatorname{Der}(G, A)$, denoted by $\operatorname{Ider}(G, A)$.
