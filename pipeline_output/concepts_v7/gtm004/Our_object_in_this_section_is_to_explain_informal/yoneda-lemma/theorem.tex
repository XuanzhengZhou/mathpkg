Let $\mathfrak{C}$ be a category, let $A$ be an object of $\mathfrak{C}$, and let $F: \mathfrak{C} \to \mathfrak{S}$ be a functor to the category of sets. For any natural transformation $\tau: \mathfrak{C}(A, -) \to F$, the element $\tau_A(1_A) \in F(A)$ determines $\tau$ entirely. Specifically, for any $\varphi: A \to B$,
$$\tau_B(\varphi) = (F\varphi)(\tau_A(1_A)).$$
Conversely, for any $\kappa \in F(A)$, the rule
$$\tau_B(\varphi) = (F\varphi)(\kappa), \quad \varphi \in \mathfrak{C}(A, B),$$
defines a natural transformation $\tau: \mathfrak{C}(A, -) \to F$. Thus there is a bijection between $F(A)$ and the set of natural transformations $\mathfrak{C}(A, -) \to F$.
