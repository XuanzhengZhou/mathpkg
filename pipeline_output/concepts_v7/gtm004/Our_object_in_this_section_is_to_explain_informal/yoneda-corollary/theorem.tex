By taking $F = \mathfrak{C}(A', -)$ in the Yoneda Lemma (Proposition 4.1), there is a bijection between the set of morphisms $\mathfrak{C}(A', A)$ and the set of natural transformations $\mathfrak{C}(A, -) \to \mathfrak{C}(A', -)$. Thus the assignment $A \mapsto \mathfrak{C}(A, -)$ is an embedding (called the \text{Yoneda embedding}) of $\mathfrak{C}^{\text{opp}}$ into the functor category $\mathfrak{S}^{\mathfrak{C}}$, and this embedding is full.
