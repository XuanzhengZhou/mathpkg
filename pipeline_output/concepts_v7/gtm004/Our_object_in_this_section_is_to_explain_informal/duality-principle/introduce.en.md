The duality principle is a powerful meta-theorem in category theory: every categorical statement has a dual obtained by reversing all morphisms and interchanging "mono" and "epi." If the original statement is provable, so is its dual — one need only apply the proof to the opposite category. This principle can be refined: if a theorem holds for all categories satisfying some self-dual conditions $Q$, then both the theorem and its dual apply to any particular category satisfying $Q$. The principle is most effective for abelian categories, which are self-dual.
