A morphism $\varphi$ in $\mathfrak{C}$ is a monomorphism in $\mathfrak{C}$ if and only if it is an epimorphism as a morphism of $\mathfrak{C}^{\text{opp}}$. Consequently, a statement about epimorphisms and monomorphisms which is true in any category must remain true if the prefixes ``epi-'' and ``mono-'' are interchanged and all arrows are reversed. More generally, if a theorem $\mathcal{T}$ holds in any category $\mathfrak{C}$ satisfying axioms $A, B, \ldots$, then the dual theorem $\mathcal{T}^{\text{opp}}$ holds in any category $\mathfrak{C}$ satisfying the dual axioms $A^{\text{opp}}, B^{\text{opp}}, \ldots$.
