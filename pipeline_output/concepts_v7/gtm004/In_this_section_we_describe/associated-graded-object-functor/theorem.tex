Let $\mathfrak{B}$ be an abelian category and let $(\mathfrak{B}, f)$ be the category of filtered objects in $\mathfrak{B}$. The \textbf{associated graded object functor}
$$
Gr : (\mathfrak{B}, f) \to \mathfrak{B}^{\mathbb{Z}}
$$
assigns to a filtered object $B$ with filtration $\cdots \subseteq B^{(p-1)} \subseteq B^{(p)} \subseteq \cdots \subseteq B$ the graded object whose $p^{\text{th}}$ component is
$$
Gr(B)^p = B^{(p)} / B^{(p-1)}.
$$
This is a functor from the category of filtered objects to the category of $\mathbb{Z}$-graded objects in $\mathfrak{B}$.
