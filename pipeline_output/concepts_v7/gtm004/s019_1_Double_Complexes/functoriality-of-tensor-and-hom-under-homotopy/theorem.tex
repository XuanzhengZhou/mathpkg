Chain maps $\varphi : C \rightarrow C'$, $\psi : D \rightarrow D'$ induce chain maps
$$\varphi \otimes \psi : C \otimes_{\Lambda} D \rightarrow C' \otimes_{\Lambda} D',$$
$$\operatorname{Hom}(\varphi, \psi) : \operatorname{Hom}_{\Lambda}(C', D) \rightarrow \operatorname{Hom}_{\Lambda}(C, D').$$

Moreover, if $\varphi \simeq \varphi'$ and $\psi \simeq \psi'$ (chain homotopic), then
$$\varphi \otimes \psi \simeq \varphi' \otimes \psi',$$
$$\operatorname{Hom}(\varphi, \psi) \simeq \operatorname{Hom}(\varphi', \psi').$$
