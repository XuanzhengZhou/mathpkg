If $\mathfrak{A}$ has enough projectives then the $J$-homology
$$H_*(J, -) : [\mathfrak{U}, \mathfrak{A}] \rightarrow [\mathfrak{B}, \mathfrak{A}]$$
exists. It may be computed as the left $\mathcal{E}_1'$-derived functor of the Kan extension,
$$H_n(J, T) = (L_n^{\mathcal{E}_1'} \tilde{J}) T, \quad n = 0, 1, \dots,$$
where $\mathcal{E}_1'$ is the projective class in $[\mathfrak{U}, \mathfrak{A}]$ obtained by applying Theorem 4.1 to the projective class $\mathcal{E}_1$ of all epimorphisms in $[\mathfrak{U}_d, \mathfrak{A}]$.
