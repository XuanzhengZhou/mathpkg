Let $\mathfrak{A}$ have enough projectives and exact coproducts. If $R \in [\mathfrak{U}, \mathfrak{A}]$ is an $\mathcal{E}_0$-projective functor, then
$$(L_n^{\mathcal{E}_1} \tilde{J}) R = 0 \quad \text{for } n \geq 1,$$
where $\mathcal{E}_0$ is the class of all split epimorphisms in $[\mathfrak{U}_d, \mathfrak{A}]$ and $\mathcal{E}_1$ is the class of all epimorphisms.
