A functor $F: \mathfrak{C} \to \mathfrak{D}$ is called:
\begin{itemize}
\item \text{full} if $F$ maps $\mathfrak{C}(A, B)$ \text{onto} $\mathfrak{D}(FA, FB)$ for all objects $A, B$ in $\mathfrak{C}$.
\item \text{faithful} if $F$ maps $\mathfrak{C}(A, B)$ \text{injectively} to $\mathfrak{D}(FA, FB)$ for all objects $A, B$ in $\mathfrak{C}$.
\item a \text{full embedding} if $F$ is full, faithful, and one-to-one on objects.
\end{itemize}
If $F$ is a full embedding, then $F(\mathfrak{C})$ is a full subcategory of $\mathfrak{D}$.
