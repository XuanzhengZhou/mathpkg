A morphism $f: A \to B$ in a category $\mathfrak{C}$ is called an \text{isomorphism} (or \text{invertible}) if there exists a morphism $g: B \to A$ in $\mathfrak{C}$ such that
$$gf = 1_A, \quad fg = 1_B.$$
The morphism $g$ is itself invertible and is uniquely determined by $f$; we write $g = f^{-1}$, so that
$$(f^{-1})^{-1} = f.$$
The composite of two invertible morphisms is again invertible. Two objects $A$ and $B$ are \text{isomorphic}, written $A \cong B$, if there exists an invertible morphism $f: A \to B$.
