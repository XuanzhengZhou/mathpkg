A free $\Lambda$-module $P$ on a set $S$ satisfies the following universal property: any set-theoretic function $f: S \to M$ from the basis $S$ into the underlying set of any $\Lambda$-module $M$ extends uniquely to a $\Lambda$-module homomorphism $\varphi: P \to M$. Concretely, if $f(s) = m_s$, then the unique extension sends $\sum \lambda_s s$ to $\sum \lambda_s m_s$. This property characterizes free modules up to isomorphism.
