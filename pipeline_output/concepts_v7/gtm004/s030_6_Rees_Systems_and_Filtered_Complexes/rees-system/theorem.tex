A \textbf{Rees system} in an abelian category $\mathfrak{A}$ is a diagram (6.9) consisting of:

\begin{itemize}
\item Two exact couples sharing the same $E$-term:
  $$\textcircled{1} = (D, E; \alpha, \beta, \gamma), \quad \textcircled{2} = (\bar{D}, E; \bar{\alpha}, \bar{\beta}, \bar{\gamma})$$
\item A morphism $\xi : \bar{D} \to D$
\item A differential object $\Gamma$ with morphisms $\varphi : D \to \Gamma$ and $\bar{\varphi} : \Gamma \to \bar{D}$
\end{itemize}

such that the following relations (6.10) hold:

$$\bar{\varphi} \circ \varphi = \xi, \quad \xi \bar{\alpha} = \alpha \xi, \quad \beta = \bar{\beta} \xi, \quad \gamma = \xi \bar{\gamma}$$

and the triangles formed by $(D, \Gamma, \bar{D}, \varphi, \bar{\varphi}, \xi)$ are compatible with the exact couple structures.

Every exact couple $EC$ may be regarded as a Rees system by setting $\textcircled{1} = EC$, $\textcircled{2} = EC$, $\Gamma = 0$, $\xi = 1$.
