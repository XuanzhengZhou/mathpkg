Let $\mathfrak{U}$ be an abelian category and let $\mathcal{E}$ be a class of epimorphisms in $\mathfrak{U}$. The object $P$ in $\mathfrak{U}$ is called \textbf{projective rel $\mathcal{E}$}, where $\mathcal{E} : B \rightarrow C$ is an epimorphism in $\mathfrak{U}$, if
$$\mathcal{E}_* : \mathfrak{U}(P, B) \rightarrow \mathfrak{U}(P, C)$$
is surjective.
