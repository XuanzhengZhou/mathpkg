The closure $C(\mathcal{E})$ of a class of epimorphisms collects all epimorphisms
against which every $\mathcal{E}$-projective object already has the lifting property.
It is a genuine closure operator: it is extensive ($\mathcal{E} \subseteq C(\mathcal{E})$)
and idempotent ($C(C(\mathcal{E})) = C(\mathcal{E})$). The closure operation is
fundamental for setting up a well-behaved relative homological algebra, as it
ensures that the class of allowable epimorphisms is saturated with respect to
the projectives it defines.
