Given an exact couple $\mathit{EC} = \{D, E, \alpha, \beta, \gamma\}$, set $d_0 = \beta\gamma : E \to E$. Then $d_0^2 = 0$. Define the derived couple $\mathit{EC}_1 = \{D_1, E_1, \alpha_1, \beta_1, \gamma_1\}$ by
$$D_1 = \alpha D, \quad E_1 = H(E, d_0),$$
and, for $x \in D$ and $z \in \ker d_0$,
$$\alpha_1(\alpha x) = \alpha(\alpha x), \quad \beta_1(\alpha x) = [\beta x], \quad \gamma_1[z] = \gamma(z),$$
where $[z]$ denotes the homology class of the cycle $z$.
