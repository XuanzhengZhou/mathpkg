\text{Using the projective presentation}

$$\mathbb{Z} \xrightarrow{\mu} \mathbb{Z} \xrightarrow{\varepsilon} \mathbb{Z}_r$$

\text{where } \mu \text{ is multiplication by } r \text{, one obtains the exact sequence}

$$\text{Hom}(\mathbb{Z}_r, \mathbb{Z}) \to \text{Hom}(\mathbb{Z}, \mathbb{Z}) \xrightarrow{\mu^*} \text{Hom}(\mathbb{Z}, \mathbb{Z}) \to \text{Ext}(\mathbb{Z}_r, \mathbb{Z})$$

\text{Since } \mu^* \text{ is again multiplication by } r \text{, we obtain}

$$\text{Ext}(\mathbb{Z}_r, \mathbb{Z}) \cong \mathbb{Z}_r.$$
