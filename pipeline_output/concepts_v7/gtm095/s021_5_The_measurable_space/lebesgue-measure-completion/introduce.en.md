Lebesgue measure on the real line is the measure corresponding to the generalized distribution function $G(x) = x$, which assigns to each interval $(a, b]$ its length $b - a$. The measure is first defined on the Borel $\sigma$-algebra via Caratheodory's extension theorem and then completed to the system of Lebesgue-measurable sets by including all subsets of Borel null sets. The completion process ensures that every subset of a set of measure zero is itself measurable (with measure zero), which is essential for the development of integration theory.
