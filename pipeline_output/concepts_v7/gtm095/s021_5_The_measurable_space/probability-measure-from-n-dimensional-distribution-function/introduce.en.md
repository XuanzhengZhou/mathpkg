Theorem 2 generalizes the one-dimensional correspondence (Theorem 1) to $n$ dimensions. Given an $n$-dimensional distribution function $F_n$, there exists a unique probability measure $P$ on $(R^n, \mathcal{B}(R^n))$ whose value on any rectangle $(a, b]$ is given by the iterated difference $\Delta_{a_1 b_1} \cdots \Delta_{a_n b_n} F_n$. The proof follows the same strategy as Theorem 1, using Caratheodory's extension theorem on the algebra generated by $n$-dimensional half-open rectangles. This result is fundamental for the study of random vectors and multivariate probability distributions.
