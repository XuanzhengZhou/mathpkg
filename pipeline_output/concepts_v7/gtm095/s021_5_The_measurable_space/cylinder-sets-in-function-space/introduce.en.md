A cylinder set in the function space $R^T$ is the collection of all real-valued functions on $T$ whose values at finitely many prescribed points $t_1, \ldots, t_n$ fall into specified sets (intervals, Borel sets, or a Borel set in $R^n$). These cylinder sets form the building blocks for defining $\sigma$-algebras on function spaces. Three variants are distinguished by the type of constraining set: intervals of the form $(a_k, b_k]$, arbitrary Borel sets on the line, or a single Borel set in the product space $R^n$.
