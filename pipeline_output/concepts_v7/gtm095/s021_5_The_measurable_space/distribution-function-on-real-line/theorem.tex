\textbf{Definition 1.} A function $F = F(x)$, $x \in R$, is called a \textit{distribution function} (on the real line $R$) if it satisfies:

\begin{enumerate}
\item[(i)] $F(x)$ is nondecreasing;
\item[(ii)] $F(-\infty) = 0$, $F(+\infty) = 1$, where
$$F(-\infty) = \lim_{x \downarrow -\infty} F(x), \quad F(+\infty) = \lim_{x \uparrow \infty} F(x);$$
\item[(iii)] $F(x)$ is continuous on the right and has a limit on the left at each $x \in R$.
\end{enumerate}

Given a probability measure $P$ on $(R, \mathcal{B}(R))$, the function $F(x) = P(-\infty, x]$ is a distribution function.
