Let $\mathcal{B}(R^T)$, $\mathcal{B}_1(R^T)$, and $\mathcal{B}_2(R^T)$ be the smallest $\sigma$-algebras corresponding respectively to the cylinder sets

$$\mathcal{I}_{t_1, \ldots, t_n}(I_1 \times \cdots \times I_n),$$

$$\mathcal{I}_{t_1, \ldots, t_n}(B_1 \times \cdots \times B_n),$$

$$\mathcal{I}_{t_1, \ldots, t_n}(B^n),$$

where $I_k$ is a set of the form $(a_k, b_k]$, $B_k$ is a Borel set on the line, and $B^n$ is a Borel set in $R^n$. It is clear that

$$\mathcal{B}(R^T) \subseteq \mathcal{B}_1(R^T) \subseteq \mathcal{B}_2(R^T).$$
