\textit{Absolutely continuous measures} are those for which the corresponding distribution functions satisfy

$$F(x) = \int_{-\infty}^{x} f(t)\,dt,$$

where $f = f(x)$, $x \in R$, is a nonnegative Riemann-integrable (or Lebesgue-integrable) function with

$$\int_{-\infty}^{\infty} f(x)\,dx = 1.$$

The function $f = f(x)$ is called the \textit{density} of the distribution function $F = F(x)$ (or the \textit{density of the probability distribution}, or simply the \textit{density}), and $F = F(x)$ is called \textit{absolutely continuous}.
