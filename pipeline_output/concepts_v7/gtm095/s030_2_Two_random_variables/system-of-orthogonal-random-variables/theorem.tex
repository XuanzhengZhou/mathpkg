A set $M \subseteq L^2$ is called a system of orthogonal random variables if $\xi \perp \eta$ for every pair $\xi, \eta \in M$ with $\xi \neq \eta$, i.e.,

$$\mathsf{E}[\xi \eta] = 0 \quad \text{for all } \xi, \eta \in M,\ \xi \neq \eta.$$

If in addition $\|\xi\| = \sqrt{\mathsf{E}[\xi^2]} = 1$ for every $\xi \in M$, then $M$ is called an orthonormal system of random variables.
