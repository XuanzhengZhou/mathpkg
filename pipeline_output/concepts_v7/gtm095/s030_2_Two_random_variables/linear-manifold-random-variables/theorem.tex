Let $\eta_1, \eta_2, \ldots$ be a sequence of random variables in $L^2$.

The linear manifold spanned by $\eta_1, \eta_2, \ldots$, denoted

$$\mathcal{L} = \mathcal{L}\{\eta_1, \eta_2, \ldots\},$$

is the set of all finite linear combinations:

$$\mathcal{L}\{\eta_1, \eta_2, \ldots\} = \left\{ \sum_{i=1}^{n} a_i \eta_i : n \geq 1,\ a_i \in \mathbb{R} \right\}.$$

The closed linear manifold spanned by $\eta_1, \eta_2, \ldots$, denoted

$$\overline{\mathcal{L}} = \overline{\mathcal{L}}\{\eta_1, \eta_2, \ldots\},$$

is the closure of $\mathcal{L}$ in the $L^2$ norm, i.e., the set of all random variables in $\mathcal{L}$ together with their mean-square limits.
