A probability distribution is symmetric about the origin precisely when its characteristic function is purely real. This provides a simple and practically useful criterion: checking whether a characteristic function takes only real values is equivalent to verifying that the underlying distribution is symmetric. The result follows because the sine function is odd and the cosine function is even: the imaginary part vanishes by symmetry, while the real part persists.
