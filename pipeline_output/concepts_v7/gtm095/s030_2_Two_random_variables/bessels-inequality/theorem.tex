Let $M = \{\eta_1, \eta_2, \ldots\}$ be an orthonormal system of random variables in $L^2$ and let $\xi \in L^2$. Then Bessel's inequality holds:

$$\sum_{i=1}^{\infty} |(\xi, \eta_i)|^2 \leq \|\xi\|^2.$$

Equality is attained if and only if

$$\xi = \lim_{n \to \infty} \sum_{i=1}^{n} (\xi, \eta_i) \eta_i,$$

where the limit is taken in the mean-square sense (i.e., in the $L^2$ norm).
