Two random variables $\xi$ and $\eta$ in $L^2$ are said to be orthogonal (denoted $\xi \perp \eta$) if

$$(\xi, \eta) \equiv \mathsf{E}[\xi \eta] = 0.$$

For random variables with zero mean, orthogonality coincides with uncorrelatedness, since $\operatorname{Cov}(\xi, \eta) = \mathsf{E}[\xi \eta] - \mathsf{E}[\xi]\mathsf{E}[\eta]$.
