Let $\eta_1, \eta_2, \ldots$ be a sequence of linearly independent random variables in $L^2$. An orthonormal sequence $\varepsilon_1, \varepsilon_2, \ldots$ is constructed recursively as follows:

Set $\varepsilon_1 = \eta_1 / \|\eta_1\|$.

If $\varepsilon_1, \ldots, \varepsilon_{n-1}$ have been selected and are orthonormal, define

$$\varepsilon_n = \frac{\eta_n - \hat{\eta}_n}{\|\eta_n - \hat{\eta}_n\|},$$

where $\hat{\eta}_n$ is the projection of $\eta_n$ onto the linear manifold $\mathcal{L}(\varepsilon_1, \ldots, \varepsilon_{n-1})$ generated by $\eta_1, \ldots, \eta_{n-1}$:

$$\hat{\eta}_n = \sum_{k=1}^{n-1} (\eta_n, \varepsilon_k) \varepsilon_k.$$

Since $\eta_1, \ldots, \eta_n$ are linearly independent and $\mathcal{L}\{\eta_1, \ldots, \eta_{n-1}\} = \mathcal{L}\{\varepsilon_1, \ldots, \varepsilon_{n-1}\}$, we have $\|\eta_n - \hat{\eta}_n\| > 0$, so the construction is well-defined.
