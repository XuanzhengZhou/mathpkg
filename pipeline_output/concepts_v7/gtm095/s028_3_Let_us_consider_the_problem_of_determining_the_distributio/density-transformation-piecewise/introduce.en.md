When the transformation $\varphi$ is not globally monotone, the density of $\eta = \varphi(\xi)$ is obtained by summing contributions from each monotone branch of $\varphi$. For each branch where $\varphi$ is strictly monotone, the single-branch formula applies using the corresponding local inverse $h_k$. The indicator functions ensure that each branch contributes only over its domain. An important example is $\eta = \xi^2$, where two branches (positive and negative square roots) contribute.
