Let $X$ and $\tilde{X}$ be random elements defined on a probability space $(\Omega, \mathcal{F}, \mathbb{P})$ with values in a separable metric space $E$. Let $P_X$ and $P_{\tilde{X}}$ be their probability distributions. Then

$$L(P_X, P_{\tilde{X}}) \leq d_{\mathbb{P}}(X, \tilde{X}),$$

where $L(\cdot, \cdot)$ is the Levy-Prokhorov metric and

$$d_{\mathbb{P}}(X, \tilde{X}) = \inf\{\varepsilon > 0 : \mathbb{P}\{\rho(X, \tilde{X}) \geq \varepsilon\} \leq \varepsilon\}.$$
