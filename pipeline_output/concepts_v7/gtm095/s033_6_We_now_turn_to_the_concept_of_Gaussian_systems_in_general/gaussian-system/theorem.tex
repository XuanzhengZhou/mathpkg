A collection of random variables $\xi = (\xi_\alpha)$, where $\alpha$ belongs to some index set $\mathfrak{A}$, is a \textit{Gaussian system} if the random vector $(\xi_{\alpha_1}, \ldots, \xi_{\alpha_n})^*$ is Gaussian for every $n \geq 1$ and all indices $\alpha_1, \ldots, \alpha_n$ chosen from $\mathfrak{A}$.
