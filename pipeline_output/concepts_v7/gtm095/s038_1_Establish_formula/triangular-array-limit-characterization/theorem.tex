Let $\xi_{n,1}, \ldots, \xi_{n,n}$ be, for each $n \geq 1$, independent identically distributed random variables, and set
$$T_n = \sum_{k=1}^{n} \xi_{n,k}, \quad n \geq 1.$$
A random variable $T$ can be a limit in distribution of the sums $T_n$ (i.e., $T_n \xrightarrow{d} T$ as $n \to \infty$) if and only if $T$ is infinitely divisible.
