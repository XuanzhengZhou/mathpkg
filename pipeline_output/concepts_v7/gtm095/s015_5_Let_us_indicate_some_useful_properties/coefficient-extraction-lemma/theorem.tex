The number $N(r; n)$ of solutions $(X_1, \ldots, X_n)$ of the equation $X_1 + \cdots + X_n = r$ with nonnegative integer-valued $X_j$ subject to constraints $X_j \in \{k_j^{(1)}, k_j^{(2)}, \ldots\}$ equals the coefficient of $x^r$ in the expansion of the product

$$A(x) = A_1(x) \cdots A_n(x),$$

where $A_j(x) = x^{k_j^{(1)}} + x^{k_j^{(2)}} + \cdots$ is the generating function encoding the allowed values for $X_j$.

In particular, when $X_j \in \{0, 1, 2, \ldots\}$ with no restrictions, $A(x) = (1 + x + x^2 + \cdots)^n = (1-x)^{-n}$ and

$$N(r; n) = C_{n+r-1}^r.$$
