Let $N(r; n)$ be the number of nonnegative integer-valued solutions $(X_1, \ldots, X_n)$ of the equation
$$X_1 + \cdots + X_n = r$$
subject to constraints $X_j \in \{k_j^{(1)}, k_j^{(2)}, \ldots\}$, $j = 1, \ldots, n$. Define generating functions
$$A_j(x) = x^{k_j^{(1)}} + x^{k_j^{(2)}} + \cdots, \quad j = 1, \ldots, n.$$
Then $N(r; n)$ equals the coefficient of $x^r$ in the expansion of $A(x) = A_1(x) \cdots A_n(x)$.
