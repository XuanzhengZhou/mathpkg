If $(a_n)_{n \geq 0} \leftrightarrow A(x)$ and $(b_n)_{n \geq 0} \leftrightarrow B(x)$, then for any constants $c, d$,
$$(c a_n + d b_n)_{n \geq 0} \leftrightarrow cA(x) + dB(x),$$
and the \text{convolution property} holds:
$$\left( \sum_{i=0}^{n} a_i b_{n-i} \right)_{n \geq 0} \leftrightarrow A(x) B(x).$$
