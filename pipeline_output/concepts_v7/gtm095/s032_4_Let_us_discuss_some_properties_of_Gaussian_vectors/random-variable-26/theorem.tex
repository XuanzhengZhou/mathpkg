(a) The components of a Gaussian vector are uncorrelated if and only if they are independent.

(b) A vector $\xi = (\xi_1, \ldots, \xi_n)^*$ is Gaussian if and only if, for every vector $\lambda = (\lambda_1, \ldots, \lambda_n)^* \in R^n$ the random variable $(\xi, \lambda) = \lambda_1 \xi_1 + \cdots + \lambda_n \xi_n$ has a Gaussian distribution.
