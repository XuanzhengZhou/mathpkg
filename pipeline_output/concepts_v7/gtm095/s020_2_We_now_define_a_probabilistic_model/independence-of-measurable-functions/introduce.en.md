Lemma 4. Let $\mathcal{A}_1, \mathcal{A}_2, \ldots, \mathcal{A}_n$ be algebras of events independent with respect to the measure $P$. Then the $\sigma$-algebras $\sigma(\mathcal{A}_1), \sigma(\mathcal{A}_2), \ldots, \sigma(\mathcal{A}_n)$ are also independent with respect to this measure. This lemma provides a key technical result used in proving theorems.
