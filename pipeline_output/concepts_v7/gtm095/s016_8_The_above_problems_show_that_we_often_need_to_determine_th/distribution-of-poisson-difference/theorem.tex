Let $\xi_1 \sim \text{Poisson}(\lambda_1)$ and $\xi_2 \sim \text{Poisson}(\lambda_2)$ be independent. Then for $k = 0, \pm 1, \pm 2, \ldots$,
$$P(\xi_1 - \xi_2 = k) = e^{-(\lambda_1 + \lambda_2)} \left(\frac{\lambda_1}{\lambda_2}\right)^{k/2} I_k\!\left(2\sqrt{\lambda_1 \lambda_2}\right),$$
where $I_k(2\lambda)$ is the modified Bessel function of the first kind of order $k$:
$$I_k(2\lambda) = \lambda^k \sum_{r=0}^{\infty} \frac{\lambda^{2r}}{r! \, \Gamma(k+r+1)}.$$
