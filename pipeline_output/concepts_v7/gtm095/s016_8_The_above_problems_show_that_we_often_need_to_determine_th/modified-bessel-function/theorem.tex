The modified Bessel function of the first kind of order $k$ ($k = 0, \pm 1, \pm 2, \ldots$) is defined by

$$I_k(2\lambda) = \lambda^k \sum_{r=0}^{\infty} \frac{\lambda^{2r}}{r! \, \Gamma(k + r + 1)}, \quad \lambda \in \mathbb{R}.$$

It satisfies the generating function identity

$$e^{\lambda(t + 1/t)} = \sum_{k=-\infty}^{\infty} t^k I_k(2\lambda).$$
