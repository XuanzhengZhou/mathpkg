Let $\xi_1$ and $\xi_2$ be independent random variables with Poisson distributions with parameters $\lambda_1 > 0$ and $\lambda_2 > 0$ respectively:

$$P(\xi_i = k) = \frac{\lambda_i^k e^{-\lambda_i}}{k!}, \quad k = 0, 1, 2, \ldots, \quad i = 1, 2.$$

Then the sum $\xi_1 + \xi_2$ has a Poisson distribution with parameter $\lambda_1 + \lambda_2$.
