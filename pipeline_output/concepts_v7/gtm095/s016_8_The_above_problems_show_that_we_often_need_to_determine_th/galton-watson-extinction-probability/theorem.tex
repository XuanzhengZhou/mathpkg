Let $\xi_k$ be a Galton-Watson branching process with offspring generating function $G(x) = \sum_{k=0}^{\infty} p_k x^k$, where $p_k = P(\eta = k)$ is the offspring distribution. Let $F_k(x) = E x^{\xi_k}$ be the generating function of the $k$-th generation size. Then
$$F_{k+1}(x) = F_k(G(x)) = G(F_k(x)).$$
The extinction probability $q = \lim_{N \to \infty} P(\xi_N = 0)$ satisfies the fixed-point equation
$$q = G(q), \quad 0 \leq q \leq 1.$$
Moreover: (a) if $E\eta > 1$, then $0 < q < 1$; (b) if $E\eta \leq 1$, then $q = 1$.
