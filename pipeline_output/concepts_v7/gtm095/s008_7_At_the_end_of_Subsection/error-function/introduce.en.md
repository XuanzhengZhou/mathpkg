The error function $\operatorname{erf}(x)$ is a special function commonly used in probability theory and statistics in connection with the normal distribution. It is defined as the integral of $e^{-t^2}$ from $0$ to $x$, normalized so that $\operatorname{erf}(x) \to 1$ as $x \to \infty$. The error function is closely related to the standard normal distribution function $\Phi(x)$ through simple algebraic relations.
