Let $S_n = \xi_1 + \cdots + \xi_n$ where $\xi_i$ are independent Bernoulli random variables with $P(\xi_i = 1) = p$, $P(\xi_i = 0) = q$. Then for any $\varepsilon > 0$ and all $0 \leq p \leq 1$,

$$P\left\{ \frac{S_n}{n} - p \leq -\varepsilon \right\} \leq e^{-2n\varepsilon^2},$$

$$P\left\{ \left| \frac{S_n}{n} - p \right| \geq \varepsilon \right\} \leq 2e^{-2n\varepsilon^2}.$$
