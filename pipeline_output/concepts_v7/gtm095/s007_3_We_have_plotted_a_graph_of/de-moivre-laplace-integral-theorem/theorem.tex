Let $S_n = \xi_1 + \cdots + \xi_n$ where $\xi_i$ are independent Bernoulli random variables with $P(\xi_i = 1) = p$, $P(\xi_i = 0) = q$, $p + q = 1$. Then for any $a < b$,

$$P_n(a, b] = P\left\{a\sqrt{npq} < S_n - np \leq b\sqrt{npq}\right\}$$

is closely approximated by the integral

$$\frac{1}{\sqrt{2\pi}} \int_a^b e^{-x^2/2} \, dx$$

as $n \to \infty$.
