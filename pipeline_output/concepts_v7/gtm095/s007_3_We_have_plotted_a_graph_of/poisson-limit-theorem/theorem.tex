Let

$$P_n(k) = \begin{cases} C_n^k p^k q^{n-k}, & k = 0, 1, \ldots, n, \\ 0, & k = n + 1, n + 2, \ldots, \end{cases}$$

and suppose that $p = p(n)$ is a function of $n$ such that $p(n) \to 0$ as $n \to \infty$ and $np(n) \to \lambda$ where $\lambda > 0$. Then for each fixed $k = 0, 1, 2, \ldots$,

$$P_n(k) \to \pi_k, \quad n \to \infty,$$

where

$$\pi_k = \frac{\lambda^k e^{-\lambda}}{k!}.$$

The set $\{\pi_k, k = 0, 1, 2, \ldots\}$ defines the Poisson probability distribution ($\pi_k \geq 0$, $\sum_{k=0}^{\infty} \pi_k = 1$).
