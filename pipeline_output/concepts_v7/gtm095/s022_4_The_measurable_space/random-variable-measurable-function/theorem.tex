Let $(\Omega, \mathcal{F})$ be a measurable space and let $(R, \mathcal{B}(R))$ be the real line with the system $\mathcal{B}(R)$ of Borel sets.

A real function $\xi = \xi(\omega)$ defined on $(\Omega, \mathcal{F})$ is an $\mathcal{F}$-measurable function, or a random variable, if

$$\{\omega : \xi(\omega) \in B\} \in \mathcal{F}$$

for every $B \in \mathcal{B}(R)$; equivalently, if the inverse image $\xi^{-1}(B) \equiv \{\omega : \xi(\omega) \in B\}$ is a measurable set in $\Omega$.

When $(\Omega, \mathcal{F}) = (R^n, \mathcal{B}(R^n))$, the $\mathcal{B}(R^n)$-measurable functions are called Borel functions.

The simplest example of a random variable is the indicator $I_A(\omega)$ of an arbitrary measurable set $A \in \mathcal{F}$.

A random variable $\xi$ having a representation $\xi(\omega) = \sum_{i=1}^{\infty} x_i I_{A_i}(\omega)$ with $\sum A_i = \Omega$, $A_i \in \mathcal{F}$, is called discrete. If the sum is finite, it is called simple.
