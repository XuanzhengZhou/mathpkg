Let $\eta = \eta(\omega)$ be an $\mathcal{F}_\xi$-measurable random variable, where $\mathcal{F}_\xi = \sigma(\xi)$ is the $\sigma$-algebra generated by $\xi$. Then there is a Borel function $\varphi$ such that $\eta = \varphi \circ \xi$, i.e., $\eta(\omega) = \varphi(\xi(\omega))$.

Conversely, if $\varphi$ is a Borel function, the function $\eta = \varphi \circ \xi$ is $\mathcal{F}_\xi$-measurable.
