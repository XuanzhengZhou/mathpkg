This theorem generalizes Kolmogorov's extension theorem from the countable product space $R^\infty$ to an arbitrary product space $R^T$ for any index set $T$. The key structural insight is that every set in $\mathcal{B}(R^T)$ is determined by countably many coordinates, so the consistency of finite-dimensional distributions ensures a unique extension. The theorem provides the measure-theoretic foundation for defining random processes with arbitrary index sets, including the construction of Wiener measure on $R^{[0,\infty)}$ as a fundamental example.
