Let $\varphi = \varphi(x)$ be a Borel function and $\xi = \xi(\omega)$ a random variable. Then the composition $\eta = \varphi \circ \xi$, i.e., the function $\eta(\omega) = \varphi(\xi(\omega))$, is also a random variable.

This follows from

$$\{\omega : \eta(\omega) \in B\} = \{\omega : \varphi(\xi(\omega)) \in B\} = \{\omega : \xi(\omega) \in \varphi^{-1}(B)\} \in \mathcal{F}$$

for $B \in \mathcal{B}(R)$, since $\varphi^{-1}(B) \in \mathcal{B}(R)$.

Therefore if $\xi$ is a random variable, so are $\xi^n$, $\xi^+ = \max(\xi, 0)$, $\xi^- = -\min(\xi, 0)$, $|\xi| = \xi^+ + \xi^-$, and so on.
