This lemma describes the structure of measurable functions when the underlying $\sigma$-algebra is atomic (generated by a countable partition). In this case, measurability forces the function to be constant on each atom $D_k$ of the partition, since any measurable set is a union of atoms. This result is fundamental for understanding conditional expectations with respect to discrete $\sigma$-algebras and for constructing simple random variables.
