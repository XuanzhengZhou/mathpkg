Let $\mathcal{E}$ be a system of sets such that $\sigma(\mathcal{E}) = \mathcal{B}(R)$. A necessary and sufficient condition that a function $\xi = \xi(\omega)$ is $\mathcal{F}$-measurable is that

$$\{\omega : \xi(\omega) \in E\} \in \mathcal{F}$$

for all $E \in \mathcal{E}$.

**Corollary.** A necessary and sufficient condition for $\xi = \xi(\omega)$ to be a random variable is that $\{\omega : \xi(\omega) < x\} \in \mathcal{F}$ for every $x \in R$, or that $\{\omega : \xi(\omega) \leq x\} \in \mathcal{F}$ for every $x \in R$.

The proof is immediate, since each of the systems $\mathcal{E}_1 = \{x : x < c, c \in R\}$ and $\mathcal{E}_2 = \{x : x \leq c, c \in R\}$ generates the $\sigma$-algebra $\mathcal{B}(R)$.
