An ordered set $(\eta_1(\omega), \ldots, \eta_n(\omega))$ of random variables is called an $n$-dimensional random vector.

Equivalently, every random element $X(\omega)$ with values in $(R^n, \mathcal{B}(R^n))$ is an $n$-dimensional random vector, representable as

$$X(\omega) = (\xi_1(\omega), \ldots, \xi_n(\omega)),$$

where $\xi_k = \pi_k \circ X$ and $\pi_k$ is the projection onto the $k$th coordinate. Each $\xi_k$ is an ordinary random variable, since for $B \in \mathcal{B}(R)$,

$$\{\omega : \xi_k(\omega) \in B\} = \{\omega : X(\omega) \in (R \times \cdots \times B \times \cdots \times R)\} \in \mathcal{F}.$$
