For random elements $X$ and $Y$ defined on the same probability space $(\Omega, \mathcal{F}, \mathbb{P})$ and taking values in a metric space $(E, \mathcal{E}, \rho)$, the \text{Ky Fan metric} $d_{\mathbb{P}}(X, Y)$ is defined by

$$d_{\mathbb{P}}(X, Y) = \inf \left\{ \varepsilon > 0 : \mathbb{P}\left\{\omega : \rho(X(\omega), Y(\omega)) \geq \varepsilon\right\} \leq \varepsilon \right\}.$$
