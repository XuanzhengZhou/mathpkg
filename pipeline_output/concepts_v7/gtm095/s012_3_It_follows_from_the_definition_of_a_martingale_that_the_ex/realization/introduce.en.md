Definition 2. A random variable $\tau = \tau(\omega)$ that takes the values $1, 2, \ldots, n$ is called a stopping time (with respect to decompositions $(\mathcal{D}_k)_{1 \leq k \leq n}, \mathcal{D}_1 \preceq \mathcal{D}_2 \preceq \cdots \preceq \mathcal{D}_n)$ if, for any $k = 1, \ldots, n$, the random variable $I_{\{\tau=k\}}(\omega)$ is $\mathcal{D}_k$-measurable. This definition introduces a fundamental concept in probability theory.
