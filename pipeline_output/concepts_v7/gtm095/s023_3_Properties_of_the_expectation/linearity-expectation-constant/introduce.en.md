The expectation operator is linear with respect to multiplication by a constant. If the expectation of a random variable exists, then scaling the random variable by a constant simply scales its expectation by the same constant. This property holds for both positive and negative constants $c$, using the decomposition $\xi = \xi^+ - \xi^-$ to handle the sign.
