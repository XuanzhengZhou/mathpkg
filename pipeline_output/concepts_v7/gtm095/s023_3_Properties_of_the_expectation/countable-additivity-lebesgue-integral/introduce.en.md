The indefinite Lebesgue integral $Q(A) = \int_A \xi \, dP$, viewed as a set function on the $\sigma$-algebra $\mathcal{F}$, is a signed measure that is absolutely continuous with respect to $P$. The proof for nonnegative $\xi$ follows from the corollary to the monotone convergence theorem; the general case uses the decomposition $\xi = \xi^+ - \xi^-$. This result sets the stage for the Radon--Nikodým theorem.
