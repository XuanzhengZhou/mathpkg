An integrable random variable (one with finite absolute expectation) is finite almost surely. If there were a set of positive probability on which $|\xi| = \infty$, the expectation $E|\xi|$ would be at least $\infty \cdot P(A) = \infty$, contradicting the integrability hypothesis.
