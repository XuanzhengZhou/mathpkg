Let $\xi$ and $\eta$ be random variables with a two-dimensional density $f_{\xi,\eta}(x,y)$. Then $\xi$ and $\eta$ are independent if and only if
$$f_{\xi,\eta}(x,y) = f_{\xi}(x) f_{\eta}(y)$$
(where the equality is understood to hold almost surely with respect to two-dimensional Lebesgue measure).
