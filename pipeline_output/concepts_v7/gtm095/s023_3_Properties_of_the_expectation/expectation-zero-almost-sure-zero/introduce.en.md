If a nonnegative random variable has zero expectation, it must be zero almost surely. The proof uses the sets $A_n = \{\omega : \xi(\omega) \geq 1/n\}$ and the fact that $0 \leq E(\xi I_{A_n}) \leq E\xi = 0$, which forces $P(A_n) = 0$ for all $n$, and hence $P(\xi > 0) = 0$.
