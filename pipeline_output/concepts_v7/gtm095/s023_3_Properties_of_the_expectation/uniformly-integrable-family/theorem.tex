A family $\{\xi_n\}_{n \geq 1}$ of random variables is said to be uniformly integrable (with respect to the measure $P$) if
$$\sup_n \int_{\{|\xi_n| > c\}} |\xi_n| \, P(d\omega) \to 0, \quad c \to \infty,$$
or, equivalently,
$$\sup_n E\bigl[|\xi_n| I_{\{|\xi_n| > c\}}\bigr] \to 0, \quad c \to \infty.$$
