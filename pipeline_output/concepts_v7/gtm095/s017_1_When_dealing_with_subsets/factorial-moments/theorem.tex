Let $\xi$ be an integer-valued random variable taking values $0, 1, 2, \ldots$. The factorial moment of order $r$ is defined by

$$m_{(r)} = E[\xi(\xi - 1) \cdots (\xi - r + 1)], \quad r = 1, 2, \ldots.$$

The factorial moments can be obtained from the generating function $G_\xi(x) = E x^\xi$ via differentiation:

$$m_{(r)} = G_\xi^{(r)}(1),$$

where $G_\xi^{(r)}$ denotes the $r$th derivative of $G_\xi$.
