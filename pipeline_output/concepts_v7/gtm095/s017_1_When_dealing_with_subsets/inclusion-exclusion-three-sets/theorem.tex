Let $A, B, C$ be subsets of a finite set $\Omega$. Then

$$N(A \cup B \cup C) = [N(A) + N(B) + N(C)] - [N(AB) + N(AC) + N(BC)] + N(ABC),$$

where $AB = A \cap B$, $AC = A \cap C$, $BC = B \cap C$, $ABC = A \cap B \cap C$.

Similarly, for the complement of the union:

$$N(\overline{A} \cap \overline{B} \cap \overline{C}) = N(\Omega) - [N(A) + N(B) + N(C)] + [N(AB) + N(AC) + N(BC)] - N(ABC).$$
