Let $I \subseteq I_\xi$. A decomposition of $I$ is an unordered collection of disjoint nonempty sets $I_p$ such that $\sum_p I_p = I$. Then

$$m_\xi(I) = \sum_{\sum_{p=1}^q I_p = I} \prod_{p=1}^q s_\xi(I_p),$$

$$s_\xi(I) = \sum_{\sum_{p=1}^q I_p = I} (-1)^{q-1}(q-1)! \prod_{p=1}^q m_\xi(I_p),$$

where $\sum_{\sum_{p=1}^q I_p = I}$ denotes summation over all decompositions of $I$, with $1 \leq q \leq N(I)$, where $N(I)$ is the number of elements of $I$.
