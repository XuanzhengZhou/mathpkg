If $\nu = (\nu_1, \ldots, \nu_k)$ is a vector whose components are nonnegative integers, we put

$$\nu! = \nu_1! \cdots \nu_k!, \quad |\nu| = \nu_1 + \cdots + \nu_k, \quad t^{\nu} = t_1^{\nu_1} \cdots t_k^{\nu_k}.$$

We also write $s_{\xi}^{(\nu)} = s_{\xi}^{(\nu_1, \ldots, \nu_k)}$ and $m_{\xi}^{(\nu)} = m_{\xi}^{(\nu_1, \ldots, \nu_k)}$.
