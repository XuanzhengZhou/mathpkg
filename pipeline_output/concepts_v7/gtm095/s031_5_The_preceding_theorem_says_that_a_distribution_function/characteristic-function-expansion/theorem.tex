Let $\xi = (\xi_1, \ldots, \xi_k)^*$ be a random vector with $E|\xi_i|^n < \infty$ for some $n \geq 1$, $i = 1, \ldots, k$. Then the mixed moments $E(\xi_1^{\nu_1} \cdots \xi_k^{\nu_k})$ exist for all nonnegative $\nu_1, \ldots, \nu_k$ with $\nu_1 + \cdots + \nu_k \leq n$, and the partial derivatives

$$\frac{\partial^{\nu_1 + \cdots + \nu_k}}{\partial t_1^{\nu_1} \cdots \partial t_k^{\nu_k}} \varphi_\xi(t_1, \ldots, t_k)$$

exist and are continuous for $\nu_1 + \cdots + \nu_k \leq n$. The Taylor expansion is

$$\varphi_\xi(t_1, \ldots, t_k) = \sum_{\nu_1 + \cdots + \nu_k \leq n} \frac{i^{\nu_1 + \cdots + \nu_k}}{\nu_1! \cdots \nu_k!} m_\xi^{(\nu_1, \ldots, \nu_k)} t_1^{\nu_1} \cdots t_k^{\nu_k} + o(|t|^n),$$

where $|t| = |t_1| + \cdots + |t_k|$ and $m_\xi^{(\nu_1, \ldots, \nu_k)} = E \xi_1^{\nu_1} \cdots \xi_k^{\nu_k}$ are the mixed moments.
