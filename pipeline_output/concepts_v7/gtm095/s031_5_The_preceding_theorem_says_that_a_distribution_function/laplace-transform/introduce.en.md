The Laplace transform of a nonnegative random variable is defined as $\hat{F}(\lambda) = E e^{-\lambda X}$ for $\lambda \geq 0$. It is analogous to the characteristic function but uses the real variable $\lambda$ instead of $it$, making it particularly useful for distributions supported on $[0,\infty)$. Bernstein's criterion completely characterizes which functions arise as Laplace transforms: they are exactly the completely monotone functions with $f(0) = 1$.
