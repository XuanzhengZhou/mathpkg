Let $(\Omega, \mathcal{F}, P)$ be a probability space. For $p \geq 1$, the space $L^p = L^p(\Omega, \mathcal{F}, P)$ is the collection of random variables $\xi = \xi(\omega)$ such that

$$E|\xi|^p \equiv \int_{\Omega} |\xi|^p \, dP < \infty.$$

Define $\|\xi\|_p = (E|\xi|^p)^{1/p}$. Then:

$$\|\xi\|_p \geq 0, \quad \|c\xi\|_p = |c| \|\xi\|_p \text{ for constant } c,$$

and by Minkowski's inequality,

$$\|\xi + \eta\|_p \leq \|\xi\|_p + \|\eta\|_p.$$

Identifying random variables that coincide almost surely, the collection of equivalence classes $[L^p]$ with operations $[\xi] + [\eta] = [\xi + \eta]$, $a[\xi] = [a\xi]$, and norm $\|[\xi]\|_p = \|\xi\|_p$ forms a normed linear space.
