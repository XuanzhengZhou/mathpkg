Let $L^\infty = L^\infty(\Omega, \mathcal{F}, P)$ be the space (of equivalence classes of) random variables $\xi = \xi(\omega)$ for which $\|\xi\|_\infty < \infty$, where $\|\xi\|_\infty$, the \textbf{essential supremum} of $\xi$, is defined by

$$\|\xi\|_\infty := \operatorname{ess\,sup} |\xi| := \inf\{0 \leq c \leq \infty : P(|\xi| > c) = 0\}.$$

The function $\|\cdot\|_\infty$ is a norm, and $L^\infty$ is complete in this norm.
