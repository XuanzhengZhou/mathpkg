A sequence $\xi_1, \xi_2, \ldots$ of random variables \textbf{converges in distribution} to a random variable $\xi$ (notation: $\xi_n \xrightarrow{d} \xi$ or $\xi_n \xrightarrow{\text{law}} \xi$) if

$$E f(\xi_n) \to E f(\xi), \quad n \to \infty,$$

for every bounded continuous function $f = f(x)$.
