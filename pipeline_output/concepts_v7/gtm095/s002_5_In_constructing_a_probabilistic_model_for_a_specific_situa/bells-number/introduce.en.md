Bell numbers count the number of ways to partition a set of $N$ labeled elements into nonempty disjoint subsets — the total number of possible decompositions of a sample space of size $N$. They satisfy a simple recurrence and grow super-exponentially. In probability, Bell numbers enumerate all possible algebras of events on a finite sample space, since each decomposition generates a unique algebra.
