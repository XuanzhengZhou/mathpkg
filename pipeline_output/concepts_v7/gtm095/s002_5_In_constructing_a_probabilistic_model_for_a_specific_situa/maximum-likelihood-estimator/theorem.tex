Given a parametric family of probability distributions $\{\text{P}_\theta : \theta \in \Theta\}$ and an observed outcome $x$, a \text{maximum likelihood estimator} $\hat{\theta}$ of the parameter $\theta$ is a value that maximizes the likelihood function $L(\theta) = \text{P}_\theta(\{x\})$ (or, in the continuous case, the density $f_\theta(x)$):
$$\hat{\theta} = \arg\max_{\theta \in \Theta} L(\theta).$$
