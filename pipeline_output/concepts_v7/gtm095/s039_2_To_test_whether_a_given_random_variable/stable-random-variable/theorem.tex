A random variable $T$, its distribution function $F(x)$, and its characteristic function $\varphi(t)$ are called \textbf{stable} if, for every $n \geq 1$, there exist constants $a_n > 0$, $b_n$, and independent random variables $\xi_1, \ldots, \xi_n$, distributed like $T$, such that

$$a_n T + b_n \stackrel{d}{=} \xi_1 + \cdots + \xi_n$$

or, equivalently, $F((x - b_n)/a_n) = F^{*n}(x)$, or

$$[\varphi(t)]^n = [\varphi(a_n t)] \, e^{i b_n t}.$$
