A random variable $T$ is stable if and only if its characteristic function $\varphi(t)$ has the form $\varphi(t) = \exp \psi(t)$, where

$$\psi(t) = it\beta - d|t|^\alpha \left(1 + i\theta \frac{t}{|t|} \, G(t, \alpha)\right),$$

with $0 < \alpha < 2$, $\beta \in \mathbb{R}$, $d \geq 0$, $|\theta| \leq 1$, $t/|t| = 0$ for $t = 0$, and

$$G(t, \alpha) = \begin{cases}
\tan \frac{1}{2}\pi\alpha, & \text{if } \alpha \neq 1,\\[4pt]
\dfrac{2}{\pi} \log |t|, & \text{if } \alpha = 1.
\end{cases}$$
