\textbf{Theorem 3.7.} \text{(Inverse Function Theorem).} Suppose $f: [a, b] \to \mathbb{R}$ is continuous on $[a, b]$ and differentiable on $(a, b)$ with $f'(x) > 0$ for all $x \in (a, b)$. Then $f$ is strictly increasing. For each $y \in [f(a), f(b)]$ there is a unique point $x = g(y) \in [a, b]$ such that $f(x) = y$. The function $g = f^{-1}$ is continuous on $[f(a), f(b)]$ and differentiable at each point of $(f(a), f(b))$ with
$$g'(y) = \bigl[f'(g(y))\bigr]^{-1}.$$
