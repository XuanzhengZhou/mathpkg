Let $\mathbb{C}^0$ be the product set $\mathbb{R}^2 = \mathbb{R} \times \mathbb{R}$, whose elements are ordered pairs $(x, y)$ of real numbers. Define addition and multiplication by
$$(x, y) + (x', y') = (x + x', y + y'),$$
$$(x, y)(x', y') = (xx' - yy', xy' + x'y).$$

With $i = (0, 1)$, every element can be written as $z = x + iy$, $x, y \in \mathbb{R}$. The real part and imaginary part are
$$\operatorname{Re}(z) = x, \quad \operatorname{Im}(z) = y.$$

The \textbf{complex conjugate} is $z^* = x - iy$. The \textbf{modulus} is
$$|z| = (z^*z)^{1/2} = (x^2 + y^2)^{1/2}.$$
