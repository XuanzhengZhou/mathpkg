Higher derivatives are defined inductively:
$$f^{(0)}(x) = f(x), \quad f^{(1)}(x) = f'(x), \quad f^{(k+1)}(x) = (f^{(k)})'(x), \quad k = 0, 1, 2, \ldots$$

The function $f: (a, b) \to \mathbb{C}$ is of \textbf{class $C^k$} ($k$-times continuously differentiable) if each of $f, f', \ldots, f^{(k)}$ is continuous on $(a, b)$. It is of \textbf{class $C^\infty$} (infinitely differentiable or smooth) if $f^{(k)}$ is continuous for every integer $k \geq 0$.
