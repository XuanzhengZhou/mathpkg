Set membership is the fundamental binary relation between an element and a set. The notation $x \in A$ (read "x is in A" or "x belongs to A") asserts that $x$ is an element of the set $A$. Its negation $x \notin A$ asserts that $x$ is not an element of $A$. A set is completely determined by specifying which objects are members of it.
