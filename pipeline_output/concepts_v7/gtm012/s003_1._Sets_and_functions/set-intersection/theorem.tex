The intersection of two sets $A$ and $B$ is defined as
$$A \cap B = \{x \mid x \in A \text{ and } x \in B\}.$$

More generally, if $J$ is an index set and $\{A_j\}_{j \in J}$ is a family of sets, their intersection is
$$\bigcap_{j \in J} A_j = \{x \mid x \in A_j \text{ for all } j \in J\}.$$

For a finite family $A_1, A_2, \ldots, A_m$, the intersection is denoted
$$A_1 \cap A_2 \cap \cdots \cap A_m \quad \text{or} \quad \bigcap_{j=1}^{m} A_j.$$

For a countably infinite family $A_1, A_2, \ldots$, the intersection is denoted
$$\bigcap_{j=1}^{\infty} A_j.$$
