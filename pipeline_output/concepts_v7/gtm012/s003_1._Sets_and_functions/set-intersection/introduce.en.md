The intersection of two sets $A$ and $B$ is the set consisting of all elements that belong to both $A$ and $B$. This binary operation generalizes to arbitrary families of sets via an index set $J$: the intersection $\bigcap_{j \in J} A_j$ collects all elements that belong to every set $A_j$ simultaneously. Intersection is commutative, associative, and idempotent, and it distributes over union.
