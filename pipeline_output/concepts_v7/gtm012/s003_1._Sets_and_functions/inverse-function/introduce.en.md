The inverse of a bijective function $f: A \to B$ is the unique function $f^{-1}: B \to A$ that "undoes" $f$, satisfying $f^{-1} \circ f = \mathrm{id}_A$ and $f \circ f^{-1} = \mathrm{id}_B$. The existence of an inverse characterizes bijections: a function is bijective if and only if it has a two-sided inverse. The inverse function reverses the direction of the original mapping.
