A function $f$ from a set $A$ to a set $B$, written
$$f: A \to B,$$
assigns to each element $x \in A$ a uniquely determined element $f(x) \in B$. The set $A$ is called the domain of $f$, and $B$ is called the codomain. The elements $f(x)$ assigned by $f$ are called the values of $f$.

A real-valued function on $A$ is a function $f: A \to \mathbb{R}$, where $\mathbb{R}$ is the set of real numbers. A complex-valued function on $A$ is a function $f: A \to \mathbb{C}$, where $\mathbb{C}$ is the set of complex numbers.
