A function $f: A \to B$ is a rule that assigns to each element $x$ of the domain $A$ a unique element $f(x)$ of the codomain $B$. Functions are fundamental objects in mathematics, providing a formal language for mappings, transformations, and correspondences between sets. Special cases include real-valued functions (codomain $\mathbb{R}$) and complex-valued functions (codomain $\mathbb{C}$).
