An injective function (also called one-to-one) maps distinct elements of the domain to distinct elements of the codomain. The contrapositive formulation is equally useful: if $f(x) = f(y)$ then $x = y$. Injectivity ensures that no two different inputs produce the same output, making the function a reversible mapping from its domain onto its image.
