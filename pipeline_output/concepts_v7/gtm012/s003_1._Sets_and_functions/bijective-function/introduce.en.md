A bijective function (or bijection) is a function that is simultaneously injective and surjective. It establishes a perfect one-to-one correspondence between the domain and codomain: every element of the codomain is the image of exactly one element of the domain. Bijections are precisely the invertible functions, making them central to the study of set cardinality and isomorphism.
