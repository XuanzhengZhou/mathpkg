A surjective function (also called onto) is one whose image equals its entire codomain: every element of the codomain is the image of at least one element of the domain. Surjectivity guarantees that the function "covers" the target set completely, with no element of the codomain left unmatched. Together with injectivity, it forms one half of the definition of a bijection.
