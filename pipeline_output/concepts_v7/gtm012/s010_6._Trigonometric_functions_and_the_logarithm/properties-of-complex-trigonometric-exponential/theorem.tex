The sine, cosine, and exponential functions have the following properties:

\begin{enumerate}
\item[(a)] $\exp(iz) = \cos z + i \sin z$ for all $z \in \mathbb{C}$.
\item[(b)] $\sin(z + 2\pi) = \sin z$, $\cos(z + 2\pi) = \cos z$, and $\exp(z + 2\pi i) = \exp z$ for all $z \in \mathbb{C}$.
\item[(c)] If $w \in \mathbb{C}$ and $w \neq 0$, there exists $z \in \mathbb{C}$ such that $w = \exp(z)$. If also $w = \exp(z')$, then there is an integer $n$ such that $z' = z + 2n\pi i$.
\end{enumerate}
