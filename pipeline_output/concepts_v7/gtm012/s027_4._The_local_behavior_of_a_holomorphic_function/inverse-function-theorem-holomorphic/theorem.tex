Suppose $f$ is holomorphic in an open set $\Omega$, and suppose $z_0 \in \Omega$ with $f'(z_0) \neq 0$. Let $w_0 = f(z_0)$. Then there exist $\delta > 0$ and $\varepsilon > 0$ such that for each $w$ with $|w - w_0| < \varepsilon$, there is a unique $z$ with $|z - z_0| < \delta$ such that $f(z) = w$. The inverse function $g(w) = z$ defined by $g(f(z)) = z$ is holomorphic on the disc $|w - w_0| < \varepsilon$, with derivative

$$g'(w) = \frac{1}{f'(g(w))}.$$
