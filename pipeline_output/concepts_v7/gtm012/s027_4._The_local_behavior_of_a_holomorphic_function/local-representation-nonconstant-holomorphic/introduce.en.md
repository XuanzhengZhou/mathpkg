Lemma 4.1 provides a canonical local representation for any nonconstant holomorphic function near a point. If $f$ is holomorphic and not constant near $z_0$, it can be written as $f(z) = a_0 + a_m(z - z_0)^m h(z)$ where $h$ is holomorphic and $h(z_0) = 1$. The integer $m \geq 1$ is the order of the first non-vanishing derivative at $z_0$, so near $z_0$ the function behaves like a shifted $m$-th power. This lemma is the key structural result underlying the Inverse Function Theorem and the description of local behavior when the derivative vanishes.
