Let $\mathcal{L}$ be the set of all smooth functions $u: \mathbb{R} \to \mathbb{C}$ such that for every integer $k \geq 0$ and every $a \in \mathbb{R}$,

$$\lim_{t \to +\infty} e^{at} D^{k} u(t) = 0.$$

Equivalently, for each integer $k \geq 0$, each $a \in \mathbb{R}$, and each $M \in \mathbb{R}$, the function

$$e^{at} D^{k} u(t)$$

is bounded on the interval $[M, \infty)$.

The set $\mathcal{L}$ is a vector space: if $u, v \in \mathcal{L}$ and $b \in \mathbb{C}$, then $bu$ and $u + v$ are in $\mathcal{L}$.
