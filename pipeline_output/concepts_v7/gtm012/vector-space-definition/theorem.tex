A \textbf{real vector space} is a set $X$ with an operation of addition (satisfying associativity, commutativity, existence of zero, existence of negatives) and scalar multiplication $\mathbb{R} \times X \to X$ satisfying for all $a, b \in \mathbb{R}$ and $x, y \in X$:
\begin{itemize}
\item[V5.] $a(bx) = (ab)x$,
\item[V6.] $(a + b)x = ax + bx$,
\item[V7.] $a(x + y) = ax + ay$,
\item[V8.] $1x = x$.
\end{itemize}
A \textbf{complex vector space} is defined analogously with $\mathbb{C}$ as the scalar field.
