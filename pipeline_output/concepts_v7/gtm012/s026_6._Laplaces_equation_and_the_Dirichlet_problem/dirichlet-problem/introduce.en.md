The Dirichlet problem is the prototypical boundary value problem for Laplace's equation. Given a domain and prescribed values on its boundary, one seeks a harmonic function in the interior that attains those boundary values continuously. For the unit disc, the problem asks: find a function harmonic inside the disc and continuous up to the boundary, matching a given continuous function $g$ on the unit circle. Physically, this models the steady-state temperature distribution on a disc whose edge is held at a fixed (but possibly varying) temperature. The solution is given by convolution of the boundary data with the Poisson kernel.
