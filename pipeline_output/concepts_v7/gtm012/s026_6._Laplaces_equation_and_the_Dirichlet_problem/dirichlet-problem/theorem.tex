The \text{Dirichlet problem for the unit disc} is the boundary value problem: find a function $u$ continuous on the closed unit disc

$$\{(x, y) \mid x^2 + y^2 \leq 1\},$$

harmonic in the open disc $x^2 + y^2 < 1$, and equal on the boundary to a given continuous function $g$:

$$u(x, y) = g(x, y), \quad x^2 + y^2 = 1.$$

In polar coordinates $(r, \theta)$, this becomes: find $u = u(r, \theta)$ satisfying

$$r^2 \frac{\partial^2 u}{\partial r^2} + r \frac{\partial u}{\partial r} + \frac{\partial^2 u}{\partial \theta^2} = 0, \quad 0 \leq r < 1,$$

with boundary condition

$$u(1, \theta) = g(\theta), \quad \theta \in \mathbb{R},$$

where $u$ and $g$ are periodic in $\theta$.
