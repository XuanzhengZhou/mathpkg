Suppose $f$ is holomorphic on $0 < |z - z_0| < R$ and has Laurent expansion $f(z) = \sum_{n=-\infty}^{\infty} a_n (z-z_0)^n$. The isolated singularity at $z_0$ is:
\begin{itemize}
\item \textbf{Removable} if $a_n = 0$ for all $n < 0$. Then $f$ extends holomorphically to $z_0$.
\item \textbf{Pole of order $m$} if $a_{-m} \neq 0$ and $a_n = 0$ for $n < -m$. Then $\lim_{z \to z_0} |f(z)| = \infty$.
\item \textbf{Essential} if $a_n \neq 0$ for infinitely many negative $n$. By Casorati-Weierstrass, $f$ maps any punctured neighborhood of $z_0$ densely in $\mathbb{C}$.
\end{itemize}
