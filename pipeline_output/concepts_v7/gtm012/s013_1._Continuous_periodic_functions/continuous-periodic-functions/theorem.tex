Let $\mathbb{C}$ denote the set of all continuous functions $u: \mathbb{R} \to \mathbb{C}$ that are periodic with period $2\pi$. The set $\mathbb{C}$ is a vector space with the operations of pointwise addition and scalar multiplication:
\begin{align*}
(u + v)(x) &= u(x) + v(x), \quad u, v \in \mathbb{C}, \; x \in \mathbb{R}; \\
(au)(x) &= a u(x), \quad u \in \mathbb{C}, \; a \in \mathbb{C}, \; x \in \mathbb{R}.
\end{align*}
This set includes the trigonometric functions $\sin nx$, $\cos nx$, and $\exp(inx) = \cos nx + i \sin nx$ for each integer $n$.
