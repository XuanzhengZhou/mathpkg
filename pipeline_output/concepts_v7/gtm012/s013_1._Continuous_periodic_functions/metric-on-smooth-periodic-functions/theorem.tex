On $\mathcal{P}$, define the metric
$$d'(u, v) = \sum_{k=0}^{\infty} 2^{-k-1} \, |D^k u - D^k v| \, \bigl[1 + |D^k u - D^k v|\bigr]^{-1}.$$
Then $d'(u, v) \geq 0$, with $d'(u, v) = 0$ if and only if $u = v$, $d'(u, v) = d'(v, u)$, and $d'$ satisfies the triangle inequality. Moreover, $d'(u, v) < 1$ for all $u, v \in \mathcal{P}$.
