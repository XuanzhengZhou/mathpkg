A periodic function is a function that repeats its values at regular intervals, called periods. If a function has period $a$, it also has period $na$ for any integer $n$, so the set of periods forms a subgroup of $\mathbb{R}$. For the study of Fourier series, it is standard to normalize all periods to $2\pi$ by a linear change of variable, since $\sin(nx)$ and $\cos(nx)$ are $2\pi$-periodic and serve as the fundamental building blocks.
