A complex-valued function $u$ defined on the real line $\mathbb{R}$ is said to be \emph{periodic} with period $a \neq 0$ if
$$u(x + a) = u(x)$$
for each $x \in \mathbb{R}$. If $u$ is periodic with period $a \neq 0$, then $u$ is also periodic with period $na$ for each integer $n$. A function $u$ is called \emph{periodic} (without qualification) if it is periodic with period $2\pi$.
