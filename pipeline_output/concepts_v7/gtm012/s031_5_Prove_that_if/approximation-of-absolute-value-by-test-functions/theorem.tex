Let $h: \mathbb{R} \to \mathbb{R}$ be defined by
$$h(t) = |t| \quad \text{for } t \leq 0,$$
$$h(t) = 0 \quad \text{for } t > 0.$$
There exists a sequence $(h_n)_{n=1}^{\infty} \subset \mathcal{L}$ such that
$$|h_n - h|_{0,a,M} \to 0 \quad \text{as } n \to \infty$$
for each $a \in \mathbb{R}$ and $M \in \mathbb{R}$.
