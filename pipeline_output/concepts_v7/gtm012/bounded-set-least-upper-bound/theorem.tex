A nonempty subset $A \subset \mathbb{R}$ is said to be \textbf{bounded above} if there is an $x \in \mathbb{R}$ such that every $y \in A$ satisfies $y \leq x$. Such a number $x$ is called an \textbf{upper bound} for $A$.

A number $x \in \mathbb{R}$ is said to be a \textbf{least upper bound} for a nonempty set $A \subset \mathbb{R}$ if $x$ is an upper bound, and if every other upper bound $x'$ satisfies $x' \geq x$. If such $x$ exists it is unique, denoted
$$x = \operatorname{lub} A.$$

A set is \textbf{bounded below} and \textbf{greatest lower bound} are defined analogously, with $\operatorname{glb} A$ denoting the greatest lower bound.
