Euler's number $e$ is defined as the sum of the reciprocal factorial series $e = \sum_{n=0}^{\infty} 1/n!$, or equivalently as the value $E(1)$ of the natural exponential function at $x = 1$. This series definition is particularly convenient in analysis because it places $e$ directly in the context of power series and differential equations, avoiding the more geometric definition as a limit of $(1 + 1/n)^n$ and instead making the connection to the exponential function $E$ immediate.
