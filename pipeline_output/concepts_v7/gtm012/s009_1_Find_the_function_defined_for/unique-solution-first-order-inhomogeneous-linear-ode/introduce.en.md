Adding a forcing term $h(x)$ to the homogeneous equation $f' = af$ yields the inhomogeneous equation $f' = af + h$. The solution formula $f(x) = cE(ax) + E(ax)\int_0^x E(-at)h(t)dt$ decomposes into the homogeneous solution plus a particular solution obtained via the integrating factor method. The key insight in the proof is to write $f = f_1 f_0$ with $f_0(x) = E(ax)$ and reduce to a simple integration problem for $f_1$, a technique that generalizes to higher-order equations.
