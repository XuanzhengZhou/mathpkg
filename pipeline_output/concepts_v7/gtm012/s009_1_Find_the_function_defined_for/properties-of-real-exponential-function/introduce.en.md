The real exponential function $E(x) = \sum_{n=0}^{\infty} x^n/n!$ is positive everywhere, strictly increasing (since $E' = E > 0$), and provides a bijection from $\mathbb{R}$ onto $(0, \infty)$. Its defining algebraic property is the functional equation $E(x+y) = E(x)E(y)$, which together with $E(1) = e$ implies $E(m/n) = e^{m/n}$ for all rational exponents. These properties justify identifying $E(x)$ with the familiar notation $e^x$ and make $E$ the natural bridge between power series and the exponential function of elementary calculus.
