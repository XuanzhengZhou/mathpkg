The exponential function extends naturally from real to complex arguments via the same power series $E(z) = \sum_{n=0}^{\infty} z^n/n!$, which converges absolutely for all $z \in \mathbb{C}$. This definition establishes the notation $e^z$ and $\exp z$ for the complex exponential. Because the series has infinite radius of convergence (by the ratio test), $E$ is an entire function — analytic on the whole complex plane — and provides the foundation for defining complex powers, logarithms, and trigonometric functions via Euler's formula.
