The exponential function arises naturally as the unique solution to $f' = f$ with $f(0) = 1$. Its power series converges everywhere. Key properties include the functional equation $e^{x+y} = e^x e^y$, strict monotonicity on $\mathbb{R}$, and surjectivity onto $(0, \infty)$. These properties extend partially to $\mathbb{C}$, where $e^{z+w} = e^z e^w$ but the function is not injective due to $2\pi i$-periodicity.
