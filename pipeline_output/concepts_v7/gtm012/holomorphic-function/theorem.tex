A function $f$ defined on an open set $\Omega \subset \mathbb{C}$ is \textbf{complex differentiable} at $z_0 \in \Omega$ if the limit
$$f'(z_0) = \lim_{z \to z_0} \frac{f(z) - f(z_0)}{z - z_0}$$
exists. If $f$ is complex differentiable at every point of $\Omega$, it is said to be \textbf{holomorphic} on $\Omega$.
