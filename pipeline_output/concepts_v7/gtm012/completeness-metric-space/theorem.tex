A sequence $(x_n)_{n=1}^{\infty}$ in a metric space $(S, d)$ \textbf{converges} to $x \in S$ if $d(x_n, x) \to 0$. It is a \textbf{Cauchy sequence} if for each $\varepsilon > 0$ there is $N$ such that $d(x_n, x_m) < \varepsilon$ whenever $n, m \geq N$.

The metric space $(S, d)$ is \textbf{complete} if every Cauchy sequence converges to a point of $S$.
