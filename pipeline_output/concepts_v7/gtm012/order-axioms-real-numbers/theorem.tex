There exists a subset $P \subset \mathbb{R}$, the set of \textbf{positive} elements, satisfying:

\begin{itemize}
\item[O1.] If $x \in \mathbb{R}$, then exactly one of the following holds: $x \in P$, $x = 0$, or $-x \in P$.
\item[O2.] If $x, y \in P$, then $x + y \in P$.
\item[O3.] If $x, y \in P$, then $xy \in P$.
\end{itemize}
