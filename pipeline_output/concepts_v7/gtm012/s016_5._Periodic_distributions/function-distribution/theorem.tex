If $v$ is a continuous periodic function defined on $[0, 2\pi]$, then we define a linear functional $F = F_v$ by

$$F_v(u) = \frac{1}{2\pi} \int_0^{2\pi} v(x)u(x) \, dx, \quad u \in \mathcal{C}.$$

Then $F_v: \mathcal{C} \to \mathbb{C}$ is linear, and

$$|F_v(u_n) - F_v(u)| \leq |v|\,|u_n - u|.$$

Therefore $F_v$ is continuous on $\mathcal{C}$. Its restriction to the subspace $\mathcal{P}$ is a periodic distribution. We say that a periodic distribution $F$ is a \textit{function} if there is a $v \in \mathcal{C}$ such that $F = F_v$ on $\mathcal{P}$.
