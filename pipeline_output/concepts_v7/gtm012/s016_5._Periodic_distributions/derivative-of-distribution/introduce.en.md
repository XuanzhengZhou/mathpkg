The derivative of a periodic distribution $F$ is defined by transferring the derivative to the test function with a minus sign: $(DF)(u) = -F(Du)$. This definition is motivated by integration by parts for smooth functions and ensures that every periodic distribution is infinitely differentiable. For a function distribution $F_v$ with $v \in \mathcal{P}$, we have $D(F_v) = F_{Dv}$, so the distributional derivative extends the classical derivative.
