The set of all periodic distributions is denoted by $\mathcal{P}'$. It is a vector space with operations defined pointwise: if $F, G \in \mathcal{P}'$, $u \in \mathcal{P}$, $a \in \mathbb{C}$, then

$$(F + G)(u) = F(u) + G(u),$$

$$(aF)(u) = aF(u).$$

If $v, w$ are continuous periodic functions, then

$$F_v + F_w = F_{v+w}, \quad F_{av} = aF_v.$$
