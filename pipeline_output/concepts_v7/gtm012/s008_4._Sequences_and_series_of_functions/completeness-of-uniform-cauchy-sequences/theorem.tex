Suppose $(f_n)_{n=1}^{\infty}$ is a sequence of bounded functions from a set $S$ to $\mathbb{C}$ which is a uniform Cauchy sequence. Then there is a unique bounded function $f: S \to \mathbb{C}$ such that $(f_n)_{n=1}^{\infty}$ converges uniformly to $f$. If $S$ is a metric space and each $f_n$ is continuous, then $f$ is continuous.
