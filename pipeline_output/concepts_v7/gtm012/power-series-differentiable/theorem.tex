The function $f$ defined by the power series
$$f(x) = \sum_{n=0}^{\infty} a_n (x - x_0)^n, \quad |x - x_0| < R,$$
is differentiable on $(x_0 - R, x_0 + R)$ and
$$f'(x) = \sum_{n=1}^{\infty} n a_n (x - x_0)^{n-1}, \quad |x - x_0| < R.$$

\textbf{Corollary.} $f$ is infinitely differentiable and
$$a_k = (k!)^{-1} f^{(k)}(x_0).$$
