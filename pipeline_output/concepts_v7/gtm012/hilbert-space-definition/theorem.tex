Suppose $H$ is a vector space over $\mathbb{R}$ or $\mathbb{C}$. An \textbf{inner product} on $H$ is a function $( \cdot, \cdot ): H \times H \to \mathbb{C}$ (or $\mathbb{R}$) such that:
\begin{itemize}
\item $(au, v) = a(u,v)$,
\item $(u_1+u_2, v) = (u_1,v) + (u_2,v)$,
\item $(v, u) = (u,v)^*$,
\item $(u,u) > 0$ if $u \neq 0$.
\end{itemize}
Then $\|u\| = (u,u)^{1/2}$ is a norm. If $H$ is complete with respect to this norm, $H$ is a \textbf{Hilbert space}.
