The classes $C^n$ and $C^\infty$ classify functions according to how many continuous partial derivatives they possess. A function is $C^n$ if all partial derivatives up to order $n$ exist and are continuous. The class $C^\infty$ (smooth functions) requires this for every $n$, which is a strictly stronger condition than being $C^n$ for any finite $n$.
