The Poisson kernel provides the solution to the Dirichlet problem on the unit disc. For each $r < 1$, $P_r$ is a smooth positive function whose Fourier coefficients are $r^{|n|}$. As $r \to 1^-$, the family $\{P_r\}$ forms an approximate identity, so convolutions $g * P_r$ converge to $g$.
