A set $A$ is \textbf{finite} if either $A = \varnothing$ or there is a bijection from $A$ to the set $\{1, 2, \ldots, n\}$ for some positive integer $n$.

A set $A$ is said to be \textbf{countable} if there is a surjective function $f: \mathbb{Z}_+ \to A$.

A set is \textbf{countable or finite} if there is a surjective function $f: \mathbb{Z}_+ \to A$.
