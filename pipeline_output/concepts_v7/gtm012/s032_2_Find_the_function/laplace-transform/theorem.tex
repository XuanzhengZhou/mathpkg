Let $f \colon \mathbb{R} \to \mathbb{C}$ be a continuous function satisfying
\begin{enumerate}
\item $\operatorname{supp}(f) \subset [M, \infty)$ for some $M \in \mathbb{R}$,
\item $|f(t)| \leq K e^{at}$ for all $t \in \mathbb{R}$, for some constants $K, a \in \mathbb{R}$.
\end{enumerate}
For $\operatorname{Re} z > a$, the Laplace transform $Lf$ of $f$ is defined by
$$Lf(z) = \int_{-\infty}^{\infty} e^{-zt} f(t) \, dt = \int_{M}^{\infty} e^{-zt} f(t) \, dt.$$
