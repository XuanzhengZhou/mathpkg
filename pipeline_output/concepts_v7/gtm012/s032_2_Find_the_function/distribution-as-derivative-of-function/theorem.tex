Let $F$ be a distribution of order $k-2$, where $k \geq 2$ is an integer. Then there exists a continuous function $f \colon \mathbb{R} \to \mathbb{C}$ such that
$$F = D^{k} F_{f},$$
where $F_{f}$ denotes the regular distribution determined by $f$, and $D^{k}$ is the $k$-th distributional derivative. Moreover, this representation is unique: if $D^{k}F_{f} = D^{k}F_{g}$ for two continuous functions $f, g$, then $F_{f} = F_{g}$.
