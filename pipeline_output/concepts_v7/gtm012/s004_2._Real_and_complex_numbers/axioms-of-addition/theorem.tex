For any $x, y, z \in \mathbb{R}$, the following axioms hold:
\begin{itemize}
\item[A1.] $(x + y) + z = x + (y + z)$ \hfill (associativity of addition)
\item[A2.] $x + y = y + x$ \hfill (commutativity of addition)
\item[A3.] There exists an element $0 \in \mathbb{R}$ such that $x + 0 = x$ for every $x \in \mathbb{R}$ \hfill (additive identity)
\item[A4.] For each $x \in \mathbb{R}$ there exists an element $-x \in \mathbb{R}$ such that $x + (-x) = 0$ \hfill (additive inverse)
\end{itemize}
