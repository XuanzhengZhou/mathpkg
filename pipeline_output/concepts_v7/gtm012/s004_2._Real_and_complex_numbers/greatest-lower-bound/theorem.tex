A number $x \in \mathbb{R}$ is said to be a \textit{greatest lower bound} for a nonempty set $A \subset \mathbb{R}$ if $x$ is a lower bound for $A$, and every other lower bound $x'$ for $A$ satisfies $x' \leq x$. If such an $x$ exists, it is unique and we write
\begin{equation*}
x = \operatorname{glb} A.
\end{equation*}
