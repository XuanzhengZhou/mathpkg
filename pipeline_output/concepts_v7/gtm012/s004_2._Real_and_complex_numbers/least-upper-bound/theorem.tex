A number $x \in \mathbb{R}$ is said to be a \textit{least upper bound} for a nonempty set $A \subset \mathbb{R}$ if $x$ is an upper bound for $A$, and every other upper bound $x'$ for $A$ satisfies $x' \geq x$. If such an $x$ exists, it is unique and we write
\begin{equation*}
x = \operatorname{lub} A.
\end{equation*}
