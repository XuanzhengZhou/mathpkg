A function is bounded if its image is contained within some ball of finite radius in the codomain. For complex-valued functions, this means there is a uniform bound $M$ on the absolute value of the function over the entire domain. Boundedness is an important property that continuous functions on compact sets automatically enjoy (Theorem 1.4), though continuous functions on non-compact domains may be unbounded.
