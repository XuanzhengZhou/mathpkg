Theorem 1.4 is a fundamental result in analysis: a continuous real- or complex-valued function on a compact metric space is necessarily bounded and attains its extreme values. This is one of the main reasons compactness is so useful -- it guarantees that continuous functions behave well. In particular, for a continuous function on a closed bounded interval $[a, b] \subset \mathbb{R}$, the function achieves a maximum and minimum on that interval.
