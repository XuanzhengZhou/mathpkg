A metric space $(S, d)$ is said to be complete if every Cauchy sequence in $S$ converges to a point of $S$.

For example, $\mathbb{R}^n$ is complete with respect to the usual metric. A sequence in $\mathbb{R}^n$ converges if and only if each of the $n$ corresponding coordinate sequences converges in $\mathbb{R}$, and is Cauchy if and only if each coordinate sequence is Cauchy in $\mathbb{R}$.
