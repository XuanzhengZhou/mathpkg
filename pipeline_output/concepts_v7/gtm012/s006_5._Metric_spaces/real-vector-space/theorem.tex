A vector space over $\mathbb{R}$, or a real vector space, is a set $X$ with two operations:

\begin{itemize}
\item \textbf{Addition:} $+: X \times X \to X$ satisfying
  \begin{itemize}
  \item[V1.] $x + (y + z) = (x + y) + z$ for all $x, y, z \in X$.
  \item[V2.] $x + y = y + x$ for all $x, y \in X$.
  \item[V3.] There exists $0 \in X$ such that $x + 0 = x$ for all $x \in X$.
  \item[V4.] For each $x \in X$, there exists $-x \in X$ such that $x + (-x) = 0$.
  \end{itemize}
\item \textbf{Scalar multiplication:} $\mathbb{R} \times X \to X$ satisfying
  \begin{itemize}
  \item[V5.] $(a + b)x = ax + bx$ for all $a, b \in \mathbb{R}, x \in X$.
  \item[V6.] $a(x + y) = ax + ay$ for all $a \in \mathbb{R}, x, y \in X$.
  \item[V7.] $(ab)x = a(bx)$ for all $a, b \in \mathbb{R}, x \in X$.
  \item[V8.] $1x = x$ for all $x \in X$.
  \end{itemize}
\end{itemize}

The elements of $X$ are called vectors and the elements of $\mathbb{R}$ are called scalars.
