Suppose $(S, d)$ and $(S', d')$ are metric spaces. A function $f: S \to S'$ is said to be uniformly continuous if for each $\varepsilon > 0$ there is a $\delta > 0$ such that

$$d'(f(x), f(y)) < \varepsilon \quad \text{whenever } d(x, y) < \delta.$$

The crucial difference from pointwise continuity is that $\delta$ depends only on $\varepsilon$, not on the point $x \in S$.
