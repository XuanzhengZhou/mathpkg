A metric on a set $S$ is a function $d: S \times S \to \mathbb{R}$ with the following properties:

\begin{itemize}
\item[D1.] $d(x, x) = 0$, and $d(x, y) > 0$ if $x, y \in S$, $x \neq y$.
\item[D2.] $d(x, y) = d(y, x)$ for all $x, y \in S$.
\item[D3.] $d(x, z) \leq d(x, y) + d(y, z)$ for all $x, y, z \in S$.
\end{itemize}

The inequality D3 is called the triangle inequality. The value $d(x, y)$ is referred to as the distance from $x$ to $y$.
