A linear transformation (or linear map) is a function between vector spaces that preserves the vector space operations of addition and scalar multiplication. Linear transformations are the structure-preserving maps in the category of vector spaces. Key associated objects include the kernel (null space) $N(T) = \{x \mid T(x) = 0\}$ and the range (image) $R(T) = \{T(x) \mid x \in X\}$, both of which are subspaces.
