Let $F$ be a periodic distribution and $u$ be a smooth periodic function. Then $F * u$ is a smooth periodic function and satisfies:

\textit{Bilinearity:}
$$(aF) * u = a(F * u) = F * (au), \quad a \in \mathbb{C},$$
$$(F + G) * u = F * u + G * u,$$
$$F * (u + v) = F * u + F * v.$$

\textit{Translation:}
$$T_t(F * u) = (T_t F) * u.$$

\textit{Differentiation:}
$$D^k(F * u) = (D^k F) * u = F * (D^k u), \quad \text{for all } k \geq 0.$$
