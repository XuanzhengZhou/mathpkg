% P Polyhedron Definition
% From GTM035, Section s008
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

A subset $\Pi$ of $\mathbb{C}^n$ is a $p$-polyhedron if there exist polynomials $P_1, \ldots, P_s$ such that
$$\Pi = \{z \in \mathbb{C}^n \mid |z_j| \leq 1, \text{ all } j, \text{ and } |P_k(z)| \leq 1, k = 1, 2, \ldots, s\}.$$
