% Boundary Removal Lemma
% From GTM035, Section s009
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Fix $x \in X \setminus S$. There exists a neighborhood $U$ of $x$ with the following property: If $\beta$ is a boundary, then $\beta \setminus U$ is also a boundary.
