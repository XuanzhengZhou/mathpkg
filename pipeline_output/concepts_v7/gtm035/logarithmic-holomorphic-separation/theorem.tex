% Logarithmic Holomorphic Separation
% From GTM035, Section s010
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $K$ be a compact set in $\mathbb{C}^N$ and $U_1, U_2$ open sets covering $K$ with $U_1 \cap U_2 \subset \{\text{Re } z_1 < 0\}$. Then the function $(\log z_1)/z_1$, holomorphic on $U_1 \cap U_2$, can be written as $h_1 - h_2$ with $h_j \in H(U_j)$ on a suitable neighborhood of $K$.
