% Riesz Banach Approximation Theorem
% From GTM035, Section s001
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $\mathcal{L}$ be a linear subspace of $C(X)$ and fix $g$ in $C(X)$. If for every measure $\mu$ on $X$
$$\mu \perp \mathcal{L} \text{ implies } \mu \perp g,$$
then $g$ lies in the closure of $\mathcal{L}$. In particular, if
$$\mu \perp \mathcal{L} \text{ implies } \mu = 0,$$
then $\mathcal{L}$ is dense in $C(X)$.
