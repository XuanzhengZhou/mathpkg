% Fiber Cardinality Bound Theorem
% From GTM035, Section s012
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $(A, X, \mathcal{M})$ be a uniform algebra with $X$ the maximal ideal space. Fix $f \in A$. If there exists a set of positive measure on which $f^{-1}(\lambda)$ has cardinality $\leq n$, then this bound extends to a neighborhood.
