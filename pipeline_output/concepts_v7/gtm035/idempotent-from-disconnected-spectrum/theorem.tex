% Idempotent From Disconnected Spectrum
% From GTM035, Section s005
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Assume there is an element $x$ in $\mathfrak{A}$ such that $\sigma(x)$ is disconnected. Then $\mathfrak{A}$ contains a nontrivial idempotent.
