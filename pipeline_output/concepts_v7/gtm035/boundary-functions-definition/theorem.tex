% Boundary Functions Definition
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $w_2, \ldots, w_n$ be smooth boundary functions on $\Gamma$. Then $w = (w_2, \ldots, w_n)$ is a map of $\Gamma$ into $\mathbb{C}^{n-1}$. For $x \in H_1$, these provide the boundary data for the Bishop disk construction.
