% Isoperimetric Inequality Via Rational Approximation
% From GTM035, Section s022
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $\Omega$ be a closed Jordan domain in the plane with smooth boundary. Let $A = \text{area}(\Omega)$ and $L = \text{length}(\partial\Omega)$. Then $2A/L \leq \text{dist}(\bar{z}, R(\Omega))$.
