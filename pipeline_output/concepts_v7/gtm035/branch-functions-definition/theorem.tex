% Branch Functions Definition
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

$\mathcal{K}$ is the collection of all $2^n$ functions of the form $\sum_{j=1}^{n} c_j \rho_j \beta_j$ on $\gamma_1$, where each $\rho_j$ is a constant $= 1$ or $= -1$. These are the algebraic branch functions of the constructed example.
