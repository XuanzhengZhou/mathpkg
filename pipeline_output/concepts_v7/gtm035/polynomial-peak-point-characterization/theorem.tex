% Polynomial Peak Point Characterization
% From GTM035, Section s007
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $K$ be a compact set in $\mathbb{C}$. $\mathbb{C} \setminus K$ is connected if and only if for each $x^0 \in \mathbb{C} \setminus K$ we can find a polynomial $P$ such that
$$|P(x^0)| > \max_K |P|.$$
