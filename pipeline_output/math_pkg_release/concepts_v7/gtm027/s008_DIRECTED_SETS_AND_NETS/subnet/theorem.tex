A net $\{T_m, m \in E\}$ is a subnet of a net $\{S_n, n \in D\}$ if and only if there is a function $N$ on $E$ with values in $D$ such that

\begin{enumerate}
\item[(a)] $T = S \circ N$, or equivalently, $T_i = S_{N_i}$ for each $i$ in $E$; and
\item[(b)] for each $m$ in $D$ there is $n$ in $E$ with the property that, if $p \geq n$, then $N_p \geq m$.
\end{enumerate}

If $S$ is eventually in a set $A$, then every subnet $S \circ N$ of $S$ is also eventually in $A$.
