This proposition shows that in a first countable space, one can always replace a given countable local base with a nested (decreasing) one. Given a countable local base $\{U_n\}$, defining $V_n = \bigcap_{k \leq n} U_k$ yields a new local base with $V_1 \supset V_2 \supset \cdots$. This nesting property is technically convenient for sequential arguments and limit constructions.
