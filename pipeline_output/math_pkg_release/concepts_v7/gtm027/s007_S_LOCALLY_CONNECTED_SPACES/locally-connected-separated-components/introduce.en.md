In a locally connected space, distinct connected components can be separated by a partition of the whole space into two separated subsets, each containing one of the components. This strengthens the fact that components are maximal connected subsets: in a locally connected space, different components are not merely disjoint but are separated in the sense that neither intersects the closure of the other. The result follows from property (a), since each component of an open set is open, allowing a clean separation by taking unions of components.
