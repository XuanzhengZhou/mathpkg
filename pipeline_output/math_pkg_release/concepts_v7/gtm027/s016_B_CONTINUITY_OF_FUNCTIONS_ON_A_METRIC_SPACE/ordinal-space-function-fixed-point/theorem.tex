If $f$ is a function on $\Omega_0$ to $\Omega_0$ such that $f(x) \geq x$ for each $x$, then for some $x$ in $\Omega_0$, the point $(x, x)$ is an accumulation point of the graph of $f$.

(Define a sequence inductively by $x_{n+1} = f(x_n)$, observe that $x_n \leq f(x_n) \leq x_{n+1}$, and use the interlacing lemma.)
