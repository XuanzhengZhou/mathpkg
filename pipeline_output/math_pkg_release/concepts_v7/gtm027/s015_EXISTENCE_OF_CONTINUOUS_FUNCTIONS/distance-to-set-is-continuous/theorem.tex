Let $(X,d)$ be a pseudo-metric space and let $A \subset X$ be a fixed subset. Define the distance from a point $x$ to $A$ by $D(A,x) = \inf \{d(x,z) : z \in A\}$. Then the function $x \mapsto D(A,x)$ is continuous on $X$ with respect to the pseudo-metric topology.

Moreover, $|D(A,x) - D(A,y)| \leq d(x,y)$ for all $x,y \in X$.
