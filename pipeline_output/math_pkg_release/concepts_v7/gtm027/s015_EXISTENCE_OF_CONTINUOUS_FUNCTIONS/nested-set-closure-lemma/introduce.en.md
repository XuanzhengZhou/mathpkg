This lemma strengthens the nested set function lemma by imposing closure-nested conditions on the family $F_t$. When each $F_t$ is open and the closure of each earlier-indexed set is contained in the later one, the resulting function $f(x) = \inf\{t : x \in F_t\}$ is continuous, with open strict sub-level sets and closed non-strict super-level sets. This is the key construction in Urysohn's lemma.
