In a first countable space, accumulation points of a set can be detected by convergent sequences lying in the set minus the point. This is part (a) of Theorem 8, which establishes that sequential convergence is adequate for describing the topology in first countable spaces. The proof constructs a nested countable neighborhood base and selects a point from each neighborhood intersected with $A \setminus \{s\}$.
