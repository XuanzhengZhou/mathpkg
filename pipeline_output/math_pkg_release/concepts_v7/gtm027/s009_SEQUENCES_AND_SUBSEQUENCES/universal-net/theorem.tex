A net in a set $X$ is universal if and only if for every subset $A$ of $X$, the net is eventually in $A$ or eventually in $X \setminus A$.

If a universal net is frequently in a set, then it is eventually in that set. Consequently, a universal net in a topological space converges to each of its cluster points.

If a net is universal, then every subnet is also universal. If $S$ is a universal net in $X$ and $f : X \to Y$, then $f \circ S$ is a universal net in $Y$.
