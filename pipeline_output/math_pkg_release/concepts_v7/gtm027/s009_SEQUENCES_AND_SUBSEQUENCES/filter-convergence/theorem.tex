A filter $\mathcal{F}$ converges to a point $x$ in a topological space $X$ if and only if each neighborhood of $x$ belongs to $\mathcal{F}$ (that is, the neighborhood system of $x$ is a subfamily of $\mathcal{F}$).

A subset $U$ of $X$ is open if and only if $U$ belongs to every filter which converges to a point of $U$.

A point $x$ is an accumulation point of a set $A$ if and only if $A \setminus \{x\}$ belongs to some filter which converges to $x$.

If $\phi_x$ is the collection of all filters which converge to a point $x$, then $\bigcap\{\mathcal{F} : \mathcal{F} \in \phi_x\}$ is the neighborhood system of $x$.
