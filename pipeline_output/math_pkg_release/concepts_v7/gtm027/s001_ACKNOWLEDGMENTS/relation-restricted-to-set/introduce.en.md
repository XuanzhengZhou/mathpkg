Restricting a relation to a subset allows localized reasoning: a relation might be transitive on a subset even if it is not transitive globally. Kelley formalizes this with the intersection $R \cap (X \times X)$. This concept is used when discussing orders on subsets (subspaces of ordered spaces) and when a convergence structure is only defined for certain nets.
