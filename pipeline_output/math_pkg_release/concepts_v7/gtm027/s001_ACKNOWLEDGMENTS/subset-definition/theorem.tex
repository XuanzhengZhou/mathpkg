A set $A$ is a \text{subset} of a set $B$, written $A \subset B$, iff every member of $A$ is a member of $B$:
$$
A \subset B \iff (\forall x)(x \in A \implies x \in B).
$$
The void set $0$ is a subset of every set. The inclusions
$$
0 \subset A \cap B \subset A \subset A \cup B
$$
are valid for every pair of sets $A$ and $B$.
