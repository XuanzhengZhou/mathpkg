An equivalence class is the set of all elements equivalent to a given element. Kelley proves the fundamental property: equivalence classes partition the domain — they are either identical or disjoint. This partition property is what makes quotient constructions possible: the set of equivalence classes forms the quotient set, which inherits a topology via the quotient map in topological contexts.
