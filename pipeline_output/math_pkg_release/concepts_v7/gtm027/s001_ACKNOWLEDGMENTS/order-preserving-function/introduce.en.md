An order-preserving (isotone, monotone) function between ordered sets maps comparable elements to comparable elements while respecting the order: if $u \leq v$ then $f(u) \leq f(v)$. When both domain and codomain are chains and the function is one-to-one, the inverse is automatically isotone. Order-preserving functions are central to order theory and appear throughout topology in the study of continuous functions on ordered spaces and in the construction of the Stone-Cech compactification.
