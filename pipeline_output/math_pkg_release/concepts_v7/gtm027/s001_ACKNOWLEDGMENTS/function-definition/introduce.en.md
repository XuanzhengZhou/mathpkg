A function is defined set-theoretically as a set of ordered pairs with the property that no two distinct pairs share the same first coordinate. The domain is the set of all first coordinates, and the value $f(x)$ is the unique second coordinate paired with $x$. This definition reduces the intuitive notion of a function to pure set theory: the graph of a function contains all information about it. Two functions are identical exactly when they have the same domain and agree at every point.
