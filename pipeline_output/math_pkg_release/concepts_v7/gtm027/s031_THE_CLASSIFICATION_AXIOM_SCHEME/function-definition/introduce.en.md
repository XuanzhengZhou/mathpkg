In Kelley's system, a function is a single-valued relation: a class of ordered pairs with the property that no two distinct pairs share the same first coordinate. The value $f(x)$ is defined as $\bigcap \{y : (x,y) \in f\}$, the intersection of all second coordinates corresponding to $x$. This definition handles both the case where $x$ is in the domain (unique value) and where it is not (value is the universe $\mathcal{U}$).
