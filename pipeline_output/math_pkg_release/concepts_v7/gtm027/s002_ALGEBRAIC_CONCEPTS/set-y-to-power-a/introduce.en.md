When all factors are the same set $Y$, the product reduces to $Y^A$, the set of all functions from $A$ to $Y$. This unifies two fundamental constructions: $\mathbb{R}^n$ is $\mathbb{R}^{\{1,\dots,n\}}$, and the set of all sequences in $Y$ is $Y^{\omega}$. Kelley uses this perspective for function spaces in topology.
