An inductive set is a set of real numbers containing $0$ and closed under the successor operation $x \mapsto x+1$. Kelley uses this concept to define the set $\omega$ of non-negative integers as the intersection of all inductive sets. This is a standard set-theoretic construction of the natural numbers within the real numbers.
