Theorem 7.4 establishes the basic comparison and separation properties of the compact-open topology $\mathcal{C}$. Since singletons are compact, each subbasic set of the pointwise topology $\wp$ belongs to $\mathcal{C}$, so $\wp \subset \mathcal{C}$. Moreover, $\mathcal{C}$ inherits the Hausdorff property from the range space $Y$, and if $Y$ is regular and all functions in $F$ are continuous, then $(F,\mathcal{C})$ is regular. The regularity proof uses the fact that a compact image $f[K]$ can be separated from a closed set by a closed neighborhood.
