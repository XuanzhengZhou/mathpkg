A perfectly normal space is a normal space in which every closed set is a $G_\delta$. This property implies that every closed set is the zero set of some continuous real-valued function, by the theorem of Section J. Perfect normality is a separation property strictly between normality and metrizability: every metric space is perfectly normal, but perfectly normal spaces need not be metrizable. Notable examples include all metric spaces and all linearly ordered topological spaces.
