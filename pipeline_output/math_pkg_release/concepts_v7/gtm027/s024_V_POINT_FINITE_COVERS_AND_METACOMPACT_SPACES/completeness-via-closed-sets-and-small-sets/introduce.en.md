A uniform space is complete exactly when every family of closed sets with the finite intersection property that contains arbitrarily small sets has nonempty intersection. This characterization is the uniform analog of the compactness criterion via closed sets with the finite intersection property, with the additional requirement that the family contain small sets. It provides a useful tool for proving completeness without explicit reference to nets.
