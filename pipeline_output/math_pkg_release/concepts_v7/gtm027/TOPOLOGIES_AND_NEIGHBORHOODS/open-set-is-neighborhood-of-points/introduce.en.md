This corollary restates the characterization of open sets more succinctly: a set is open precisely when it belongs to the neighborhood system of every point it contains. It follows directly from the theorem that a set is open iff it contains a neighborhood of each of its points, since an open set is itself an open neighborhood of each of its members.
