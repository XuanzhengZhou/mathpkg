For $u \in \mathbb{C}$, the \emph{supremum norm} (or uniform norm) is defined by
$$|u| = \sup_{x \in \mathbb{R}} |u(x)| = \sup_{x \in [0, 2\pi]} |u(x)|.$$
This norm satisfies:
\begin{enumerate}
\item $|u| \geq 0$, and $|u| = 0$ if and only if $u(x) = 0$ for all $x$;
\item $|au| = |a| \, |u|$ for all $a \in \mathbb{C}$, $u \in \mathbb{C}$;
\item $|u + v| \leq |u| + |v|$ for all $u, v \in \mathbb{C}$.
\end{enumerate}
The metric on $\mathbb{C}$ induced by this norm is $d(u, v) = |u - v|$.
