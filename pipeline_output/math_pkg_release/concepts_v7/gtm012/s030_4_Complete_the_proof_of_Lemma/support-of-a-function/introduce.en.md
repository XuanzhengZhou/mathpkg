The support of a function is the closure of the set where the function is nonzero. It is a fundamental concept in analysis for describing where a function is "active." The condition $f(t) = 0$ for $t \leq M$ can be expressed concisely as $\operatorname{supp}(f) \subset [M, \infty)$. The support generalizes naturally to distributions.
