A \textbf{linear combination} of vectors $x_1, \ldots, x_n$ is any vector $x = \sum a_j x_j$ where $a_j$ are scalars. The vectors are \textbf{linearly independent} if no nontrivial linear combination equals zero. A set $S \subset X$ \textbf{spans} $X$ if every vector is a linear combination of elements of $S$.

A \textbf{basis} is a linearly independent set that spans $X$. If $X$ has a finite basis, it is \textbf{finite-dimensional} and the number of elements in any basis is the same, called the \textbf{dimension} of $X$.
