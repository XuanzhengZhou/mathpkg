A function $f: (a, b) \to \mathbb{C}$ is \textbf{differentiable} at $x \in (a, b)$ if the limit
$$f'(x) = \lim_{y \to x} \frac{f(y) - f(x)}{y - x}$$
exists. If $f$ is differentiable at each point of $(a, b)$, it is said to be \textbf{differentiable}. If $f'$ is itself differentiable, the second derivative is $f'' = (f')'$. Higher derivatives are defined inductively:
$$f^{(0)} = f, \quad f^{(k+1)} = (f^{(k)})'.$$
$f$ is of \textbf{class $C^k$} if $f, f', \ldots, f^{(k)}$ are continuous; $f$ is \textbf{$C^\infty$} if all derivatives exist and are continuous.
