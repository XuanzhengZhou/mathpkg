Every continuous $2\pi$-periodic function $v$ induces a periodic distribution $F_v$ defined by integration against $v$. This embedding of continuous functions into the space of distributions is injective: distinct continuous functions yield distinct distributions. The definition uses the normalized integral $\frac{1}{2\pi}\int_0^{2\pi}$ to simplify Fourier series formulas. Not every periodic distribution is a function distribution — the Dirac $\delta$-distribution is the canonical counterexample.
