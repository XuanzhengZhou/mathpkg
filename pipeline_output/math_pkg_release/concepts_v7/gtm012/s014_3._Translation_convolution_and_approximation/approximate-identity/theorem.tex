A sequence $(\varphi_n)_{n=1}^{\infty} \subset \mathcal{C}$ is called an \textbf{approximate identity} if it satisfies:
\begin{enumerate}
\item[(i)] $\varphi_n(x) \geq 0$ for all $n$ and all $x$;
\item[(ii)] $\displaystyle \frac{1}{2\pi} \int_0^{2\pi} \varphi_n(x) \, dx = 1$ for all $n$;
\item[(iii)] for each $0 < \delta < \pi$,
$$\int_{\delta}^{2\pi - \delta} \varphi_n(x) \, dx \to 0 \quad \text{as} \quad n \to \infty.$$
\end{enumerate}
