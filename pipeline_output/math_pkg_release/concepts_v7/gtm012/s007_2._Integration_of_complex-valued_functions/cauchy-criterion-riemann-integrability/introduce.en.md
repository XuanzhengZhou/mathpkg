The Cauchy criterion for Riemann integrability states that a bounded function is integrable if and only if its Riemann sums for different partitions can be made arbitrarily close to each other by taking the mesh sufficiently small. This is an intrinsic characterization that does not require knowing the value of the integral in advance. The proof uses the completeness of the complex numbers.
