A subset of a metric space is bounded if it can be contained entirely within some open ball of finite radius. This generalizes the notion of boundedness from the real line: a set $A \subset \mathbb{R}$ is bounded if it is contained in some interval $(-M, M)$. As shown by Proposition 6.1, compactness implies boundedness, but the converse is false in general.
