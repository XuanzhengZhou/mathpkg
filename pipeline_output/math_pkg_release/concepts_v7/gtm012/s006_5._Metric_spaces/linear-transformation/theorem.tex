Suppose $X$ and $Y$ are vector spaces, either both real or both complex. A function $T: X \to Y$ is said to be linear if for all vectors $\mathbf{x}, \mathbf{x}' \in X$ and all scalars $a$,

\begin{align*}
T(\mathbf{x} + \mathbf{x}') &= T(\mathbf{x}) + T(\mathbf{x}'),\\
T(a\mathbf{x}) &= a T(\mathbf{x}).
\end{align*}
