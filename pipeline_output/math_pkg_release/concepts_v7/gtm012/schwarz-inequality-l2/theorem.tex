For $u, v \in \mathcal{C}$:
\[
|(u, v)| \leq \|u\| \, \|v\|,
\]
where $\|u\| = (u, u)^{1/2} = \left(\frac{1}{2\pi} \int_0^{2\pi} |u(x)|^2 \, dx\right)^{1/2}$.

\textbf{Corollary.} The function $u \mapsto \|u\|$ is a norm on $\mathcal{C}$, satisfying the triangle inequality $\|u + v\| \leq \|u\| + \|v\|$.
