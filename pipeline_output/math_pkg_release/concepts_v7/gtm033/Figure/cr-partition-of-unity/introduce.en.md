A $C^r$ partition of unity is a collection of smooth nonnegative functions on a manifold, each supported in a given open set of a cover, such that they sum to 1 everywhere. The local finiteness condition guarantees the sum is well-defined. Partitions of unity are the fundamental tool for gluing locally defined maps into global ones on manifolds.
