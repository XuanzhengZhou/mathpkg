This theorem classifies all compact connected orientable surfaces that admit a Morse function with exactly one saddle. Removing disks around minima and maxima reduces to the case of a surface with only a saddle (Theorem 3.3, yielding $H_2$), and then capping boundary circles with disks produces the four possible topological types.
