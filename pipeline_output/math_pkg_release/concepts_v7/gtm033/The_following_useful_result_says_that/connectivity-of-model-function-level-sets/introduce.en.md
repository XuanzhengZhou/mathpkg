This lemma establishes key properties of an explicit model Morse function $g$ on $\mathbb{R}^2$ with critical points at $(0,0)$ (saddle), $(-1,0)$ (minimum), and $(2,0)$ (minimum). The level sets above the saddle value are connected, and sublevel sets for sufficiently large values are disks. This model function serves as the reference for the disk characterization theorem.
