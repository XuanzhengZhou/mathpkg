Let $M$ be a manifold of dimension $n$. The cotangent bundle $T^*M$ is defined like the tangent bundle $TM$ using the dual vector space $(\mathbb{R}^n)^* = L(\mathbb{R}^n, \mathbb{R})$ instead of $\mathbb{R}^n$. As a set, $T^*M = \bigcup_{x \in M} M_x^*$ where $M_x^* = L(M_x, \mathbb{R})$. If $(\varphi, U)$ is a chart on $M$, a natural chart on $T^*M$ is the map
$$T^*U \to \varphi(U) \times (\mathbb{R}^n)^*$$
which sends $\lambda \in M_x^*$ to $(\varphi(x), \lambda \circ \varphi_x^{-1})$. The projection map $p: T^*M \to M$ sends $M_x^*$ to $x$.
