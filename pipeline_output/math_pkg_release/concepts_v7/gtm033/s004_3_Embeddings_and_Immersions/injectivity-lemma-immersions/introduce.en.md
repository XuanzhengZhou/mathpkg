The Local Injectivity Lemma states that if an immersion $f$ is injective on the closure of an open set $W$, then any map $g$ sufficiently $C^1$-close to $f$ on $W$ is also injective on $W$. The proof uses a compactness argument: if the lemma fails, one can construct sequences $a_n, b_n$ converging to the same point, leading to a contradiction with $f$ being an immersion. This lemma is essential for proving that embeddings form an open set in the strong topology.
