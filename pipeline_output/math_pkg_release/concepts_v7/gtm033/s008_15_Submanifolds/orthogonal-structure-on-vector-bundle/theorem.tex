An \textbf{inner product} or \textbf{orthogonal structure} (of class $C^r$) on a vector bundle $\xi = (p, E, B)$ is a family $\alpha = \{\alpha_x\}_{x \in B}$ where each $\alpha_x$ is an inner product (symmetric, bilinear, positive definite $2$-form) on the vector space $E_x$, such that the map
$$(x, y, z) \mapsto \alpha_x(y, z),$$
defined on $\{(x, y, z) \in B \times E \times E : x = p(y) = p(z)\}$, is $C^r$.

The pair $(\xi, \alpha)$ is called an \textbf{orthogonal vector bundle}. Using Gram-Schmidt orthogonalization, an orthogonal bundle admits an \textbf{orthogonal atlas} $\Phi = \{\varphi_i, U_i\}$ where each $\varphi_{ix}: \xi_x \to \mathbb{R}^n$ is an isometry, and transition functions $g_{ij}: U_i \cap U_j \to GL(n)$ take values in $O(n)$.
