Let $P_1, P_2, \ldots$ be probability measures on $(R, \mathcal{B}(R)), (R^2, \mathcal{B}(R^2)), \ldots$ respectively, possessing the consistency property

$$P_{n+1}(B \times R) = P_n(B), \quad B \in \mathcal{B}(R^n), \quad n = 1, 2, \ldots$$

Then there is a unique probability measure $P$ on $(R^\infty, \mathcal{B}(R^\infty))$ such that

$$P(\mathcal{I}_n(B)) = P_n(B), \quad B \in \mathcal{B}(R^n),$$

for $n = 1, 2, \ldots$, where $\mathcal{I}_n(B) = \{x \in R^\infty : (x_1, \ldots, x_n) \in B\}$ is the cylinder set in $R^\infty$ with base $B \in \mathcal{B}(R^n)$.
