Let $L_\theta(\omega) = \log p_\theta(\omega)$ be the log-likelihood function for the Bernoulli scheme with parameter $\theta$. The Fisher information $I_n(\theta)$ is defined by

$$I_n(\theta) = E_\theta \left[ \frac{\partial L_\theta(\omega)}{\partial \theta} \right]^2.$$

For the Bernoulli scheme, $\frac{\partial L_\theta(\omega)}{\partial \theta} = \frac{\sum_{i=1}^n (x_i - \theta)}{\theta(1 - \theta)}$, and

$$I_n(\theta) = \frac{n}{\theta(1 - \theta)}.$$
