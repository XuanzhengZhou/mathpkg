Here are some examples of martingales.

Example 1. Let $\eta_1, \ldots, \eta_n$ be independent Bernoulli random variables with

$$P(\eta_k = 1) = P(\eta_k = -1) = \frac{1}{2},$$

$$S_k = \eta_1 + \cdots + \eta_k \quad \text{and} \quad \mathcal{D}_k = \mathcal{D}_{\eta_1, \ldots, \eta_k}.$$

We observe that the decompositions $\mathcal{D}_k$ have a simple structure:

$$\mathcal{D}_1 = \{D^+, D^-\},$$

where $D^+ = \{\omega : \eta_1 = +1\}, D^- = \{\omega : \eta_1 = -1\};$$

$$\mathcal{D}_2 = \{D^{++}, D^{+-}, D^{-+}, D^{--}\},$$

where $D^{++} = \{\omega : \eta_1 = +1, \eta_2 = +1\}, \ldots, D^{--} = \{\omega : \eta_1 = -1, \eta_2 = -1\}$, etc.

It is also easy to see that $\mathcal{D}_{\eta_1, \ldots, \eta_k} = \mathcal{D}_{S_1, \ldots, S_k}$.

Let us show that $(S_k, \mathcal{D}_k)_{1 \leq k \leq n}$ form a martingale. In fact, $S_k$ is $\mathcal{D}_k$-measurable, and by (12) and (18) of Sect. 8

$$E(S_{k+1} \mid \mathcal{D}_k) = E(S_k + \eta_{k+1} \mid \mathcal{D}_k)$$

$$= E(S_k \mid \mathcal{D}_k) + E(\eta_{k+1} \mid \mathcal{D}_k) = S_k + E\eta_{k+1} = S_k.$$

If we put $S_0 = 0$ and take $D_0 = \{\Omega\}$, the trivial

$$\xi_k = \mathsf{E}(\xi_{k+1} \mid \mathcal{D}_k) = \mathsf{E}[\mathsf{E}(\xi_{k+2} \mid \mathcal{D}_{k+1}) \mid \mathcal{D}_k]$$
$$= \mathsf{E}(\xi_{k+2} \mid \mathcal{D}_k) = \cdots = \mathsf{E}(\xi_n \mid \mathcal{D}_k).$$
