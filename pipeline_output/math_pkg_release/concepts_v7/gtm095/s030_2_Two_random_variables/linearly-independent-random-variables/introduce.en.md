Linear independence for random variables is defined almost surely: a nontrivial linear combination cannot vanish with probability one unless all coefficients are zero. This notion is essential for constructing orthonormal systems via Gram-Schmidt orthogonalization. Linear independence is equivalent to the covariance matrix being strictly positive definite, which implies its Gram determinant and all eigenvalues are positive.
