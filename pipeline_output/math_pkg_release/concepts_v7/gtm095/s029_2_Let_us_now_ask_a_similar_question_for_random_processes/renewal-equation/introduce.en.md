The renewal equation governs the renewal function $m(t) = E N_t$, the expected number of renewals by time $t$. The equation $m(t) = F(t) + \int_0^t m(t-x) dF(x)$ decomposes the expected count into the probability the first renewal occurs by $t$, plus expected subsequent renewals integrated over possible first arrival times. This Wiener-Hopf type equation has a unique solution within the class of functions bounded on finite intervals.
