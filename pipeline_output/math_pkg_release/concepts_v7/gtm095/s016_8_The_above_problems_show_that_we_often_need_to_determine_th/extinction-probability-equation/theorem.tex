Let $\xi_n$ be a Galton–Watson branching process with offspring generating function $G(x) = \sum_{k=0}^{\infty} p_k x^k$. Let $q = \lim_{N \to \infty} P(\xi_N = 0)$ be the extinction probability. Then $q$ satisfies the fixed-point equation

$$q = G(q), \quad 0 \leq q \leq 1.$$

Moreover:
(a) if $E\eta > 1$ (supercritical), then $0 < q < 1$;
(b) if $E\eta \leq 1$ (critical or subcritical), then $q = 1$.
