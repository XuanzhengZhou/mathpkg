Comparing terms in $t^{\lambda}$ on the right-hand sides of (38) and (42), and using $|\lambda^{(1)}| + \cdots + |\lambda^{(q)}| = |\lambda^{(1)} + \cdots + \lambda^{(q)}|$, we obtain (40).

Moreover,

$$\log \varphi_{\xi}(t) = \log \left[ 1 + \sum_{1 \leq |\lambda| \leq n} \frac{i|\lambda|}{\lambda!} m_{\xi}^{(\lambda)} t^{\lambda} + o(|t|^n) \right].$$

For small $z$ we have the expansion

$$\log(1 + z) = \sum_{q=1}^{n} \frac{(-1)^{q-1}}{q} z^q + o(z^q).$$

Using this in (43) and then comparing the coefficients of $t^{\lambda}$ with the corresponding coefficients on the right-hand side of (38), we obtain (41).
