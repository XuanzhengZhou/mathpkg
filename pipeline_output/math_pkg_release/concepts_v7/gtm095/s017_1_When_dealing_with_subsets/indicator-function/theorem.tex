The \text{indicator function} $I_A(\omega)$ of a set $A \subseteq \Omega$ is defined by
$$I_A(\omega) = \begin{cases} 1, & \omega \in A, \\ 0, & \omega \notin A. \end{cases}$$
For the complement $\overline{A} = \Omega \setminus A$, we have $I_{\overline{A}} = 1 - I_A$. For intersections, $I_{A \cap B} = I_A \cdot I_B$. The number of elements is $N(A) = \sum_{\omega \in \Omega} I_A(\omega)$.
