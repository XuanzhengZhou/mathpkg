The measure $\lambda$ corresponding to the generalized distribution function $G(x) = x$ is called \textit{Lebesgue measure} on $(R, \mathcal{B}(R))$.

The measure $\lambda$, initially defined on $\mathcal{B}([0,1])$, extends naturally to the system $\overline{\mathcal{B}}([0,1])$ of Lebesgue-measurable sets: if $\Lambda \in \overline{\mathcal{B}}([0,1])$ and there exist $A, B \in \mathcal{B}([0,1])$ with $A \subseteq \Lambda \subseteq B$ and $\lambda(B \setminus A) = 0$, then define $\overline{\lambda}(\Lambda) = \lambda(A)$. The set function $\overline{\lambda}$ is a probability measure on $([0,1], \overline{\mathcal{B}}([0,1]))$, called \textit{Lebesgue measure} on the system of Lebesgue-measurable sets.
