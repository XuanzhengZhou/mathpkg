Let $T$ be an arbitrary set and let $R^T$ be the collection of real functions $x = (x_t)$ defined for $t \in T$. For $t_1, \ldots, t_n \in T$, cylinder sets are defined in three forms:

$$\mathcal{I}_{t_1, \ldots, t_n}(I_1 \times \cdots \times I_n) = \{x: x_{t_1} \in I_1, \ldots, x_{t_n} \in I_n\},$$

$$\mathcal{I}_{t_1, \ldots, t_n}(B_1 \times \cdots \times B_n) = \{x: x_{t_1} \in B_1, \ldots, x_{t_n} \in B_n\},$$

$$\mathcal{I}_{t_1, \ldots, t_n}(B^n) = \{x: (x_{t_1}, \ldots, x_{t_n}) \in B^n\},$$

where $I_k$ is a set of the form $(a_k, b_k]$, $B_k$ is a Borel set on the line, and $B^n$ is a Borel set in $R^n$.
