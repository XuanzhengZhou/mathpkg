\textbf{Theorem 2.} Let $F_n = F_n(x_1, \ldots, x_n)$ be a distribution function on $R^n$. Then there exists a unique probability measure $P$ on $(R^n, \mathcal{B}(R^n))$ such that

$$P(a, b] = \Delta_{a_1 b_1} \cdots \Delta_{a_n b_n} F_n(x_1, \ldots, x_n)$$

for arbitrary $a = (a_1, \ldots, a_n)$, $b = (b_1, \ldots, b_n)$ with $a_i \leq b_i$.
