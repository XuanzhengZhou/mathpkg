A triangular array of random variables $\xi_{n1}, \ldots, \xi_{nn}$ (independent within each row) satisfies the \textbf{condition of asymptotic negligibility} if for every $\varepsilon > 0$,

$$\max_{1 \leq k \leq n} \mathbb{P}\bigl\{\,|\xi_{nk}| \geq \varepsilon\,\bigr\} \rightarrow 0, \quad n \rightarrow \infty.$$
