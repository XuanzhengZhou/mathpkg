(a) for each $\omega \in \Omega$ the function $Q(\omega; B)$ is a probability measure on $(E, \mathcal{E})$;

(b) for each $B \in \mathcal{E}$ the function $Q(\omega; B)$, as a function of $\omega$, is a version of the conditional probability $P(X \in B | G)(\omega)$, i.e.

$$Q(\omega; B) = P(X \in B | G)(\omega) \quad \text{(a. s.)}.$$
