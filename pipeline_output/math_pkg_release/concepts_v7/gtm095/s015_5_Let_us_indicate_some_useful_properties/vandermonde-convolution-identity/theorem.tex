Vandermonde's convolution identity states that for nonnegative integers $n$, $m$, and $k$,

$$\sum_{j=0}^{k} C_n^j \, C_m^{k-j} = C_{n+m}^k.$$

Equivalently, with $C_n^{n-k} = C_n^k$, this can be written as

$$\sum_{j=1}^{n} C_n^j \, C_{m}^{k-j} = C_{n+m}^k.$$

The proof follows from the equality $(1+x)^n (1+x)^m = (1+x)^{n+m}$ and the convolution property of generating functions.
