Let $\xi \sim \mathcal{N}(m, \sigma^2)$ be a normal random variable and $\eta = e^\xi$. Then $\eta$ has the lognormal distribution with density

$$f_\eta(y) = \begin{cases} \dfrac{1}{\sqrt{2\pi}\,\sigma y} \exp\!\left[-\dfrac{(\log(y/M))^2}{2\sigma^2}\right], & y > 0, \\[10pt] 0, & y \leq 0, \end{cases}$$

where $M = e^m$.
