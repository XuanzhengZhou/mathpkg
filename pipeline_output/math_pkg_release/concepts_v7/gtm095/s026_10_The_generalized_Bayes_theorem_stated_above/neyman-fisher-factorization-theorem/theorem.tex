Let $\mathcal{P} = \{P_{\theta}, \theta \in \Theta\}$ be a dominated family of probability measures on $(\Omega, \mathcal{F})$, i.e., $P_{\theta} \ll \lambda$ for all $\theta \in \Theta$ for some $\sigma$-finite measure $\lambda$. A $\sigma$-subalgebra $\mathcal{G} \subseteq \mathcal{F}$ is sufficient for $\mathcal{P}$ if and only if for each $\theta \in \Theta$ there exists a $\mathcal{G}$-measurable function $\hat{g}_{\theta}^{(\lambda)} = \hat{g}_{\theta}^{(\lambda)}(\omega)$ such that the Radon-Nikodym derivative admits the factorization
$$\frac{dP_{\theta}}{d\lambda}(\omega) = \hat{g}_{\theta}^{(\lambda)}(\omega)\, h(\omega)$$
where $h = h(\omega)$ is an $\mathcal{F}$-measurable function independent of $\theta$.
