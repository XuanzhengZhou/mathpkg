A morphism $f: A \to B$ in a category $\mathfrak{C}$ is called an \textbf{isomorphism} (or \textbf{invertible}) if there exists a morphism $g: B \to A$ in $\mathfrak{C}$ such that
$$gf = 1_A, \quad fg = 1_B.$$

The morphism $g$ is uniquely determined by $f$; we write $g = f^{-1}$, so that $(f^{-1})^{-1} = f$.
