Let $\beta : B_1 \to B_2$ be a homomorphism of $\Lambda$-modules. The induced map
$$\beta_* = \operatorname{Hom}_\Lambda(A, \beta) : \operatorname{Hom}_\Lambda(A, B_1) \to \operatorname{Hom}_\Lambda(A, B_2)$$
is defined by $\beta_*(\varphi) = \beta \circ \varphi$ for $\varphi \in \operatorname{Hom}_\Lambda(A, B_1)$. This is a homomorphism of abelian groups and satisfies:
\begin{itemize}
\item $(\beta' \beta)_* = \beta'_* \beta_*$
\item $(1_B)_* = 1_{\operatorname{Hom}_\Lambda(A, B)}$
\end{itemize}
