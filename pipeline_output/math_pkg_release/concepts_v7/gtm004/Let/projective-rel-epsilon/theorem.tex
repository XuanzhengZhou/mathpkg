Let $\mathfrak{U}$ be an abelian category and let $\mathcal{E}$ be a class of epimorphisms in $\mathfrak{U}$. An object $P$ in $\mathfrak{U}$ is called \textit{projective rel} $\varepsilon$, where $\varepsilon : B \to C$ is an epimorphism in $\mathfrak{U}$, if $\varepsilon_* : \mathfrak{U}(P, B) \to \mathfrak{U}(P, C)$ is surjective.
