The homotopy relation $\simeq$ on the set of chain maps from $C$ to $D$ is an equivalence relation:
\begin{itemize}
\item Reflexivity: $\varphi \simeq \varphi$ (take $\Sigma = 0$).
\item Symmetry: If $\psi - \varphi = \partial \Sigma + \Sigma \partial$, then $\varphi - \psi = \partial(-\Sigma) + (-\Sigma)\partial$, so $\varphi \simeq \psi$ implies $\psi \simeq \varphi$.
\item Transitivity: If $\psi - \varphi = \partial \Sigma + \Sigma \partial$ and $\chi - \psi = \partial T + T \partial$, then $\chi - \varphi = \partial(\Sigma + T) + (\Sigma + T)\partial$.
\end{itemize}
