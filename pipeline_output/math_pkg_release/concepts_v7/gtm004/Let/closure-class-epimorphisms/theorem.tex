The \textit{closure} $C(\mathcal{E})$ of a class $\mathcal{E}$ of epimorphisms consists of those epimorphisms $\varepsilon$ in $\mathfrak{U}$ such that every $\mathcal{E}$-projective object of $\mathfrak{U}$ is also projective rel $\varepsilon$. The class $\mathcal{E}$ is called \textit{closed} if $\mathcal{E} = C(\mathcal{E})$.
