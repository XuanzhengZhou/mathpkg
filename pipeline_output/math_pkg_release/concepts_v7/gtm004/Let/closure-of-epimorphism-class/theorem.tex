Let $\mathcal{E}$ be a class of epimorphisms in an abelian category $\mathfrak{U}$. The \emph{closure} $C(\mathcal{E})$ of $\mathcal{E}$ is the class of all epimorphisms $\varepsilon$ in $\mathfrak{U}$ such that every $\mathcal{E}$-projective object of $\mathfrak{U}$ is also projective rel $\varepsilon$. The class $\mathcal{E}$ is called \emph{closed} if $\mathcal{E} = C(\mathcal{E})$. One always has $\mathcal{E} \subseteq C(\mathcal{E})$ and $C(C(\mathcal{E})) = C(\mathcal{E})$.
