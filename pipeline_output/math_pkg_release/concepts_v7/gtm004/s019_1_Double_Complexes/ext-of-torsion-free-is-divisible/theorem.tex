If $A$ is a torsion-free abelian group, then $\operatorname{Ext}_{\mathbb{Z}}(A, B)$ is a divisible abelian group for every abelian group $B$.

Equivalently, for torsion-free $A$, the functor $\operatorname{Ext}_{\mathbb{Z}}(A, -)$ takes values in divisible groups, i.e., $\operatorname{Ext}_{\mathbb{Z}}(A, B)$ is an injective $\mathbb{Z}$-module for all $B$.
