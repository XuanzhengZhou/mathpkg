This theorem proves that the $J$-homology of a functor $T: \mathfrak{U} \rightarrow \mathfrak{A}$ along $J: \mathfrak{U} \rightarrow \mathfrak{B}$ exists whenever $\mathfrak{A}$ has enough projectives. The $J$-homology is defined as the left derived functor of the Kan extension $\tilde{J}$ with respect to the projective class $\mathcal{E}_1'$ induced from the class of all epimorphisms. This construction provides a homology theory for functors between small categories with values in an abelian category with enough projectives.
