\text{Using the projective presentation}

$$\mathbb{Z} \xrightarrow{\mu} \mathbb{Z} \xrightarrow{\varepsilon} \mathbb{Z}_r$$

\text{where } \mu \text{ is multiplication by } r \text{, one obtains the exact sequence}

$$\text{Hom}(\mathbb{Z}_r, \mathbb{Z}_q) \to \text{Hom}(\mathbb{Z}, \mathbb{Z}_q) \xrightarrow{\mu^*} \text{Hom}(\mathbb{Z}, \mathbb{Z}_q) \to \text{Ext}(\mathbb{Z}_r, \mathbb{Z}_q)$$

\text{Since } \mu^* \text{ is multiplication by } r \text{ on } \text{Hom}(\mathbb{Z}, \mathbb{Z}_q) \cong \mathbb{Z}_q \text{, we obtain}

$$\text{Ext}(\mathbb{Z}_r, \mathbb{Z}_q) \cong \mathbb{Z}_{(r,q)}$$

\text{where } (r,q) \text{ denotes the greatest common divisor of } r \text{ and } q.
