Let $M$ be a filtered object satisfying the adequate filtration conditions

$$\bigcap_p M^{(p)} = 0, \qquad \bigcup_p M^{(p)} = M.$$

Then for every $x \in M$, $x \neq 0$, there exists a unique integer $p$ such that

$$x \notin M^{(r)} \text{ for all } r < p, \qquad x \in M^{(r)} \text{ for all } r \geq p.$$

Consequently, each non-zero element of $M$ is represented by a unique homogeneous non-zero element in $Gr(M)$, and conversely each homogeneous non-zero element of $Gr(M)$ represents a non-zero element of $M$. Thus the only information lost in passing from $M$ to $Gr(M)$ is the data of the module extensions $M^{(p-1)} \hookrightarrow M^{(p)}$.
