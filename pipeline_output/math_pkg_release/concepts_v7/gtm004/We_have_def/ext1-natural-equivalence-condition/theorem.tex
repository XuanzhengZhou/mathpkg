For a homomorphism $\alpha : A' \rightarrow A$, the induced natural transformation

$$\alpha^* : \operatorname{Ext}_\Lambda^1(A, -) \rightarrow \operatorname{Ext}_\Lambda^1(A', -)$$

is a natural equivalence if and only if $\alpha$ factors as

$$A' \xrightarrow{\sigma} A \oplus P \xrightarrow{\pi_A} A$$

where $P$ is a projective $\Lambda$-module, $\pi_A$ is the projection, and $\sigma$ is an isomorphism.
