This theorem describes how the change-of-rings functor $U^\gamma$ interacts with projective and injective modules. If $\Lambda'$ is projective as a $\Lambda$-module, then $U^\gamma$ preserves projectivity — this follows because the left adjoint $\bar{F}$ preserves epimorphisms. Dually, if $\Lambda'$ is flat as a right $\Lambda$-module, then $U^\gamma$ preserves injectivity because the left adjoint $F$ preserves monomorphisms. These results are essential for comparing homological algebra over different base rings.
