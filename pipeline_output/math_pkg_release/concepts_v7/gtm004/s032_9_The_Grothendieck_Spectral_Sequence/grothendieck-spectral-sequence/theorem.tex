Let $F: \mathfrak{A} \to \mathfrak{B}$ and $G: \mathfrak{B} \to \mathfrak{C}$ be left exact additive functors between abelian categories with enough injectives. Assume that if $I$ is an injective object of $\mathfrak{A}$, then $F(I)$ is $G$-acyclic. Then for each object $A$ of $\mathfrak{A}$ there exists a spectral sequence $\{E_n(A)\}$ such that

$$
E_1^{p,q} = (R^p G)(R^{q-p} F)(A) \Rightarrow R^q(GF)(A),
$$

which converges finitely to the graded object associated with $\{R^q(GF)(A)\}$, suitably filtered.
