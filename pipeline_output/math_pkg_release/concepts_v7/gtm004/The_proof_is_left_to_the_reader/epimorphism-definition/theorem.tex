A $\Lambda$-module homomorphism $\varepsilon : B \rightarrow C$ is called \textbf{epimorphic} (or an \textbf{epimorphism}) if for any two homomorphisms $\alpha_1, \alpha_2 : C \rightarrow M$,

$$\alpha_1 \circ \varepsilon = \alpha_2 \circ \varepsilon \quad \implies \quad \alpha_1 = \alpha_2.$$

In other words, $\varepsilon$ is right-cancellative with respect to composition of module homomorphisms.
