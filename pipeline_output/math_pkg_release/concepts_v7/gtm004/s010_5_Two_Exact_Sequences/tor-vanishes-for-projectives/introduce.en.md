Proposition 8.1 establishes that the Tor functor vanishes when either argument is projective. This parallels the vanishing of Ext for projective (or injective) modules. The proof uses the fact that a projective presentation of a projective module splits, so the kernel in the definition of Tor is trivial. This property is essential for computing Tor groups via projective resolutions and is analogous to the acyclicity of projectives with respect to any left derived functor.
