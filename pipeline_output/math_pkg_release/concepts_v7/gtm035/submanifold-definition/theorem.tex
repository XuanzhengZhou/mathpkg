% Submanifold Definition
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $\Omega$ be an open set in $\mathbb{C}^n$. A closed subset $\Sigma \subset \Omega$ is a $k$-dimensional submanifold of $\Omega$ of class $C^e$ if for each $x_0 \in \Sigma$ there exist a neighborhood $U$ of $x_0$ and real-valued functions $\rho_1, \ldots, \rho_{2n-k}$ in $C^e(U)$ with linearly independent gradients such that $\Sigma \cap U = \{x \in U : \rho_1(x) = \cdots = \rho_{2n-k}(x) = 0\}$.
