% Cech Cohomology Definition
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

For a compact space $X$ with open cover $\mathcal{U} = \{U_\alpha\}$, the Čech cohomology group $H^p(\mathcal{U}, \mathbb{Z})$ is defined via alternating cochains $c^p(U_{\alpha_0}, \ldots, U_{\alpha_p})$ with integer values, with coboundary operator $\delta$.
