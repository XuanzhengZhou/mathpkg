% Boundary For Function Algebra
% From GTM035, Section s009
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

A boundary for $\mathcal{F}$ is a closed subset $E$ of $X$ such that
$$|f(x)| \leq \max_E |f|, \quad \text{all } f \in \mathcal{F}, x \in X.$$
