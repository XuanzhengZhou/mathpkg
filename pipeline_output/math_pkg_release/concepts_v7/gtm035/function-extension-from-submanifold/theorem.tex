% Function Extension From Submanifold
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Fix a compact set $K$ on $\Sigma$. Let $u$ be a function of class $C^e$ defined on $\Sigma$. Then there exists a function $U$ of class $C^1$ in $\mathbb{C}^n$ extending $u$ with controlled derivatives.
