% Dense Subalgebra From Perturbation
% From GTM035, Section s014
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Assume that there is a constant $k < 1$ such that $|R(z) - R(z')| \leq k|z - z'|$ for $z, z' \in D$. Then $[z, \bar{z} + R(z)]$ is dense in $C(D)$.
