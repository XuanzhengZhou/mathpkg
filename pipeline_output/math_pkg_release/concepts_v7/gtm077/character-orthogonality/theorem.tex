For the $h$ characters $\chi$ of a finite Abelian group $\mathfrak{G}$:
$$\sum_{A \in \mathfrak{G}} \chi(A) = \begin{cases} h & \text{if } \chi \text{ is the principal character}, \\ 0 & \text{otherwise}. \end{cases}$$

Dually, for a fixed element $A \in \mathfrak{G}$:
$$\sum_{\chi} \chi(A) = \begin{cases} h & \text{if } A = E, \\ 0 & \text{otherwise}. \end{cases}$$

Consequently, an element $A$ is uniquely determined by its $h$ character values $\chi_n(A)$. If $A$ has order $f$, then among the $h$ values $\chi_n(A)$, each $f$th root of unity occurs exactly $h/f$ times.
