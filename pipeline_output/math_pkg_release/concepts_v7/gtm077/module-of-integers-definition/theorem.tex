A system $M$ of integers in a number field $K$ is called a module if it is closed under addition and subtraction and contains a nonzero element. The elements of $M$ form a torsion-free Abelian group under addition. If $M$ contains $k$ independent elements, it is a $k$-rank module. An $n$-rank module is characterized by containing $n$ linearly independent numbers over $\mathbb{Q}$.
