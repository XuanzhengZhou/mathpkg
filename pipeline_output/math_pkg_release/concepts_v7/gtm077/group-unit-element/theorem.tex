In every group there exists a uniquely determined element $E$ (the unit element) such that $AE = A$ for every $A$. Moreover, for each $A$ there exists a uniquely determined inverse element $A^{-1}$ with $A A^{-1} = A^{-1} A = E$. Powers $A^m$ are defined for all integers $m$, satisfying $A^{m+n} = A^m A^n$.
