Let $a_1 > 0$, $a_2 > 0$. Define the sequence $(a_i)$ by repeated division with remainder:
$$a_i = q_i a_{i+1} + a_{i+2}, \quad 0 \leq a_{i+2} < a_{i+1}, \quad i = 1, 2, \ldots$$
The process terminates when $a_{k+2} = 0$ for some $k$. Then
$$(a_1, a_2) = (a_2, a_3) = \cdots = (a_k, a_{k+1}) = (a_{k+1}, 0) = a_{k+1},$$
i.e., the last non-vanishing remainder $a_{k+1}$ is the GCD of $a_1$ and $a_2$.
