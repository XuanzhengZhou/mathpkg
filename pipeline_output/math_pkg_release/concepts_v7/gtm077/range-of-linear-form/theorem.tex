The range of values of an arbitrary linear form
$$a_1 x_1 + a_2 x_2 + \cdots + a_n x_n$$
in $n$ variables with integral coefficients, not all vanishing, as $x_1,\ldots,x_n$ run through all integers, is identical with the range of values of a certain form of one variable $d \cdot x$, where $d$ is the GCD of the coefficients $a_1,\ldots,a_n$.
