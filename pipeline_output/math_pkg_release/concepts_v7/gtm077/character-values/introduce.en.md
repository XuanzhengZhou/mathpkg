Theorem 32 describes how character values are distributed: if an element $A$ has order $f$, then all characters take values that are $f$th roots of unity, and each such root occurs equally often across the $h$ characters. This equidistribution property follows from the orthogonality relations and is essential for the analytic applications of characters in L-series.
