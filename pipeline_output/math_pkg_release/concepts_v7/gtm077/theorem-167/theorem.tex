If $l$ is an integral ideal without odd prime factors and $\lambda$ is an integer with the decomposition $\lambda = \text{lr}$, where $r$ is an odd integral ideal, then

$$\left( \frac{\lambda}{\alpha} \right) \left( \frac{\alpha}{r} \right) = (-1)^{\sum_{p=1}^{n_1} ((\text{sgn} \alpha^{(p)} - 1)/2)((\text{sgn} \lambda^{(p)} - 1)/2)}$$

if the odd number $\alpha$ is a quadratic residue mod 4l and relatively prime to $\lambda$.
