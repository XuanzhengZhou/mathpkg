A nonempty set $S$ of integers is called a module (or $\mathbb{Z}$-module) if it is closed under addition and subtraction, i.e., for all $m, n \in S$:
$$m + n \in S, \quad m - n \in S.$$
Equivalently, $S$ is closed under taking integer linear combinations of its elements. If a module contains a single nonzero number, it contains infinitely many.
