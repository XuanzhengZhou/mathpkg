If $E$ is a vector space over $\mathbb{K}$ and $p$ is a seminorm on $E$, then:
\begin{enumerate}
\item[(i)] The sets
$$B = \{x : p(x) \leq 1\}, \quad U = \{x : p(x) < 1\}$$
are balanced and convex;
\item[(ii)] For all $x, y \in E$,
$$|p(x) - p(y)| \leq p(x - y).$$
\end{enumerate}
If, moreover, $E$ is a TVS and $p$ is continuous, then $B$ is a balanced convex body (i.e., a balanced, closed, convex set with nonempty interior).
