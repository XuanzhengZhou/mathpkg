When a family of linear maps $f_\iota: E \to G$ all share the same target TVS $G$, the fundamental system of neighborhoods at the origin for the initial topology simplifies significantly. Instead of choosing a different neighborhood $V_\iota$ in $G$ for each $\iota$ in the finite set $J$, one can use a single neighborhood $V$ common to all maps. This is because finite intersections of neighborhoods are again neighborhoods. This simplification is particularly important for weak topologies, where $G = \mathbb{K}$ and each $f_\iota$ is a linear functional.
