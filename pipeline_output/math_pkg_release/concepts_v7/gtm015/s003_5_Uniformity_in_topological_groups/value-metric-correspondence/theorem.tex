If $$(G, |\cdot|)$$ is a valued group, then the formula $$d(x, y) = |x^{-1}y|$$ defines a left-invariant metric $$d$$ on $$G$$. Conversely, the value is expressible in terms of the metric by $$|x| = d(e, x)$$. This establishes a one-to-one correspondence between values on a group and left-invariant metrics on the group.
