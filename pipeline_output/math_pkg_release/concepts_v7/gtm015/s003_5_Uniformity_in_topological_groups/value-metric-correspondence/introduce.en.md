There is a natural bijection between values on a group $G$ and left-invariant metrics on $G$. Given a value $|\cdot|$, the metric $d(x, y) = |x^{-1}y|$ is left-invariant; conversely, given a left-invariant metric $d$, the function $|x| = d(e, x)$ is a value. This correspondence allows one to freely translate between the algebraic language of values and the geometric language of metrics in the study of topological groups.
