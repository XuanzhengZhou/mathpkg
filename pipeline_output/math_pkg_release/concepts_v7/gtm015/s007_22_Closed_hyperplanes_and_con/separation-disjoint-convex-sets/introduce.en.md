The fundamental separation theorem: two disjoint nonempty convex sets in a real TVS, one of which is open, can be separated by a closed hyperplane. The open set lies strictly to one side. If both are open, the separation is strict. The proof uses the set of differences, which forms an open convex set not containing the origin, and applies the geometric Hahn-Banach theorem.
