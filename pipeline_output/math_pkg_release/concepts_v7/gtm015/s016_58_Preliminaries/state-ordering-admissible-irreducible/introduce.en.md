The convex set of states on a $C^*$-algebra carries a natural partial order: $g \leq f$ means $f - g$ is a state (up to scaling). A state is pure (extremal) if and only if the only states it dominates are its scalar multiples. This order-theoretic purity is equivalent to the irreducibility of the associated GNS representation.
