Let $\varphi$ and $\psi$ be Hermitian sesquilinear forms on a complex vector space $A$. We write $\psi \leq \varphi$ in case $\psi(x, x) \leq \varphi(x, x)$ for all $x \in A$. This is a partial ordering, with $\psi \leq \varphi$ and $\varphi \leq \psi$ implying $\psi = \varphi$ by the polarization identity.
