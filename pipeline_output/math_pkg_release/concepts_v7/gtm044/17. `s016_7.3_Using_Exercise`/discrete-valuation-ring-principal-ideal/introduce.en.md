Corollary 2.21 states that every discrete rank one valuation ring is a principal ideal ring: every ideal is generated by a single element, namely any element of minimal order in the ideal. This follows directly from Lemma 2.20, since the ideal consists of all elements with order at least $\mathrm{ord}(\mathfrak{a})$, which is precisely the principal ideal generated by an element of order $\mathrm{ord}(\mathfrak{a})$. This property is fundamental for the algebraic study of nonsingular points on curves.
