Let $\mathfrak{q}$ be an ideal of a ring and let $\mathfrak{p}_1, \ldots, \mathfrak{p}_t$ be prime ideals. If $\mathfrak{q}$ is not contained in any $\mathfrak{p}_i$, then there exists an element $x \in \mathfrak{q}$ such that $x \notin \mathfrak{p}_i$ for all $i = 1, \ldots, t$. More precisely, there exist elements $p_i^*$ ($i = 1, \ldots, t$) such that $p_i^* \notin \mathfrak{p}_i$ but $p_i^* \in \mathfrak{p}_j$ for all $j \neq i$, and then $\sum_{i=1}^t p_i^* \in \mathfrak{q}$ but lies in none of $\mathfrak{p}_1, \ldots, \mathfrak{p}_t$.
