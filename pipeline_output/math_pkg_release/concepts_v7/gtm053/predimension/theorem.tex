In a fusion of two structures $(M, L_1)$ and $(M, L_2)$ each carrying a combinatorial pregeometry with dimension functions $d_1$ and $d_2$, the \text{predimension} $\delta(X)$ for finite $X \subseteq M$ is a linear combination
$$\delta(X) = d_1(X) + \alpha \cdot d_2(X)$$
uniquely determined by the \text{free fusion principle}: the new relations from $L_2$ should impose exactly the expected number of independent constraints. For a unary function $f$ on a field, the predimension is $\delta(X) = \operatorname{trd}(X \cup f(X)) - |X|$.
