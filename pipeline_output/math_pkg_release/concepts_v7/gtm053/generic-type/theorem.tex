For a stable theory $T$, let $A \preceq B$ be a small elementary submodel and $I \subseteq B$ an $A$-indiscernible set. The \text{average type} of $I$ over $C$ (with $A \subseteq C \subseteq B$) is
$$\operatorname{Av}_C(I) = \{Q(x) : L_C\text{-formula such that } B \vDash Q(i) \text{ for all but finitely many } i \in I\}.$$
When $I$ is a maximal $A$-indiscernible set whose average type equals a given type $p$, the cardinal $|I|$ is an important invariant. For algebraically closed fields, the average type of an algebraically independent set is the \text{generic type} of the field.
