An $L$-sentence $S$ is a \text{logical consequence} of a set of $L$-sentences $E$, written $E \vDash S$, if:
\begin{itemize}
\item For finite $E$: $A \vDash E$ implies $A \vDash S$ for every $L$-structure $A$.
\item For infinite $E$: there exists a finite $E^0 \subset E$ such that $E^0 \vDash S$.
\end{itemize}
$S$ is called \text{logically valid}, written $\vDash S$, if $A \vDash S$ for every $L$-structure $A$.
