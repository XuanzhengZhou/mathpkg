Given a predimension $\delta$ on a class of structures $\mathcal{H}$, the \text{Hrushovski inequality} is
$$\delta(X) \geq 0 \quad \text{for all finite } X \subseteq M.$$
The class $\mathcal{H}_\delta$ consists of all structures $(M, L_1 \cup L_2)$ satisfying this inequality. A structure in $\mathcal{H}_\delta$ is \text{algebraically closed} (existentially closed) if no proper extension in $\mathcal{H}_\delta$ adds new realizations of quantifier-free types.
