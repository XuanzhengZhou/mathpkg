An embedding $h: A \rightarrow B$ is called an \text{elementary embedding} if for any $L$-formula $P(x_1, \ldots, x_n)$ and any $a_1, \ldots, a_n \in A$,
$$A \vDash P(a_1, \ldots, a_n) \text{ iff } B \vDash P(h(a_1), \ldots, h(a_n)).$$
In this situation $A$ is called an \text{elementary substructure} of $B$ and $B$ an \text{elementary extension} of $A$, written $A \preceq B$ or $A \preceq_h B$.
