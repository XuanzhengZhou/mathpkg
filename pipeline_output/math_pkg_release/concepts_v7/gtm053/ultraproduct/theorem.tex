Let $\mathbf{A}_i = (A_i, L)$, $i \in I$, be a family of $L$-structures and $U$ an ultrafilter on $I$. Define the equivalence $\varphi \approx_U \psi$ if $\{i : \varphi(i) = \psi(i)\} \in U$. The quotient $\prod_{i \in I} A_i / \approx_U$ forms the domain of the \text{filtered product} $\prod_{i \in I} \mathbf{A}_i / U$.
\begin{itemize}
\item For an $n$-ary relation symbol $p$: $p(\tilde{\varphi}_1, \ldots, \tilde{\varphi}_n)$ holds iff $\{i : \mathbf{A}_i \vDash p(\varphi_1(i), \ldots, \varphi_n(i))\} \in U$.
\item For an $n$-ary operation symbol $f$: $f(\tilde{\varphi}_1, \ldots, \tilde{\varphi}_n) = \tilde{\varphi}_{n+1}$ is defined analogously.
\item For a constant $c$: $c = \tilde{\varphi}$ where $\varphi(i) = c^{\mathbf{A}_i}$.
\end{itemize}
When $U$ is an ultrafilter, this is called an \text{ultraproduct}. When all $\mathbf{A}_i = \mathbf{A}$, it is called an \text{ultrapower}, written $\mathbf{A}^I / U$.
