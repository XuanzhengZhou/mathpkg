A set of formulas $T$ in a first-order language $L$ is called \text{Gödelian} if it is:
\begin{itemize}
\item Complete: for every closed formula $P$, either $P \in T$ or $\neg P \in T$,
\item Does not contain a contradiction: there is no $P$ such that both $P \in T$ and $\neg P \in T$,
\item Closed under deduction: if $T \vdash P$ then $P \in T$,
\item Contains all logical axioms $\mathrm{Ax}\,L$.
\end{itemize}
