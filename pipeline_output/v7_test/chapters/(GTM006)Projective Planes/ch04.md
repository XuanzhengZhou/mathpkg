3. Planar Ternary Rings

If $\mathcal{S}$ is any set then a ternary operation $T$ on $\mathcal{S}$ is a rule which assigns to any three ordered elements $a, b, c$ of $\mathcal{S}$ a unique element $T(a, b, c)$ of $\mathcal{S}$. A non empty set $G$ with a binary operation $\cdot$ is called a loop if:

(i) $a \cdot x=b$ has a unique solution in $x$ for any $a, b \in G$.

(ii) $y \cdot a=b$ has a unique solution in $y$ for any $a, b \in G$.

(iii) $G$ has an element $e$ such that $e \cdot x=x \cdot e=x$ for all $x \in G$ (the element $e$ is called an identity of $G$).
