Let \(Q = \bigcap_{i=1}^{n} Q_i\). By Theorem 2.7(ii), \(\operatorname{Rad} Q = \bigcap_{i=1}^{n} \operatorname{Rad} Q_i = \bigcap_{i=1}^{n} P = P\); in particular, \(Q \subset P \subset \operatorname{Rad} Q\). One then verifies that if \(ab \in Q\) and \(a \notin Q\), then \(b \in P\) using the fact that each \(Q_i\) is \(P\)-primary.
