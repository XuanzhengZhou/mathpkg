Given a multiplicative set \(S\) in a ring \(R\) and an ideal \(I\) disjoint from \(S\), Zorn's Lemma guarantees the existence of an ideal maximal among those containing \(I\) and disjoint from \(S\). Any such maximal ideal is necessarily prime. This result is fundamental for constructing prime ideals with specified avoidance properties and is frequently used in the case \(I = 0\).
