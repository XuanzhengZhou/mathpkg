Any two reduced primary decompositions of a given submodule have the same number of components and the same set of associated prime ideals. In particular, the set of isolated primes is uniquely determined by the submodule. However, the primary components themselves are not necessarily unique — only the components corresponding to isolated primes are uniquely determined. Embedded primes and their components may vary; Exercise 18 in the text demonstrates three distinct reduced primary decompositions of the same ideal in \(F[x,y]\).
