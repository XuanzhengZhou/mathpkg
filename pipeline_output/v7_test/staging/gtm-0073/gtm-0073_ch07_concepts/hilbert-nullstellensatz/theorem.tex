(Hilbert Nullstellensatz) Let $F$ be an algebraically closed extension field of a field $K$ and $I$ a proper ideal of $K[x_1, \ldots, x_n]$. Define
\[
V(I) = \{(a_1, \ldots, a_n) \in F^n \mid g(a_1, \ldots, a_n) = 0 \text{ for all } g \in I\},
\]
\[
J(V(I)) = \{f \in K[x_1, \ldots, x_n] \mid f(a_1, \ldots, a_n) = 0 \text{ for all } (a_1, \ldots, a_n) \in V(I)\}.
\]
Then
\[
\operatorname{Rad} I = J(V(I)).
\]

In words: $f$ vanishes on every common zero of $I$ in $F^n$ if and only if some power of $f$ belongs to $I$.
