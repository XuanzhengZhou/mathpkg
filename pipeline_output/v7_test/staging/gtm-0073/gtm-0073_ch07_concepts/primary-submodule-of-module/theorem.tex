Let $B$ be a module over a commutative ring $R$ with identity. A proper submodule $A$ of $B$ is \textbf{primary} if for any $r \in R$ and $b \in B \setminus A$,
\[
rb \in A \implies r^n B \subset A \text{ for some } n > 0.
\]

If $A$ is a primary submodule of $B$, then $Q_A = \{ r \in R \mid r^n B \subset A \text{ for some } n > 0 \}$ is a primary ideal of $R$, and $\operatorname{Rad} Q_A$ is a prime ideal $P$. The submodule $A$ is said to \textbf{belong to} $P$ (or is $P$-\textbf{primary} in $B$) if $P = \operatorname{Rad} Q_A = \{ r \in R \mid r^n B \subset A \text{ for some } n > 0 \}$.
