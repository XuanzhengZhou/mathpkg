Let $R$ be a commutative ring with identity and $B$ an $R$-module. A submodule $C$ of $B$ has a \textbf{primary decomposition} if
\[
C = A_1 \cap A_2 \cap \cdots \cap A_n,
\]
with each $A_i$ a $P_i$-primary submodule of $B$ for some prime ideal $P_i$ of $R$.

The decomposition is \textbf{reduced} (or irredundant) if no $A_i$ contains $\bigcap_{j \neq i} A_j$ and the prime ideals $P_1, \ldots, P_n$ are all distinct.

If $P_j \not\subset P_i$ for all $j \neq i$, then $P_i$ is called an \textbf{isolated} prime ideal of $C$. Otherwise $P_i$ is called an \textbf{embedded} prime ideal.
