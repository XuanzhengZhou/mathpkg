Let $R$ be a commutative ring with identity, $J$ an ideal contained in every maximal ideal of $R$, and $A$ a finitely generated $R$-module. Then the following are equivalent:
\begin{enumerate}
  \item every element of $1_R + J = \{1_R + j \mid j \in J\}$ is a unit in $R$;
  \item if $JA = A$, then $A = 0$;
  \item if $B$ is a submodule of $A$ such that $A = B + JA$, then $A = B$;
  \item $A/JA$ is a vector space over $R/J$ and any basis of this vector space can be lifted to a minimal generating set of $A$ as an $R$-module.
\end{enumerate}
