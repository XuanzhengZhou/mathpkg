Nakayama's Lemma is one of the most important tools in commutative algebra. Its most frequently used form states that if \(J\) is contained in every maximal ideal (e.g., the Jacobson radical) and \(A\) is a finitely generated module with \(JA = A\), then \(A = 0\). Equivalently, if \(B\) is a submodule of a finitely generated module \(A\) and \(A = B + JA\), then \(A = B\). The key step in the proof: choosing a minimal generating set and using the unit property of elements in \(1_R + J\) to eliminate generators.
