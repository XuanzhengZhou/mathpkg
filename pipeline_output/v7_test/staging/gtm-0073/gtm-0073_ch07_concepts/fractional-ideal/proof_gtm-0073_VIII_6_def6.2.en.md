This is a definition with immediate structural observations. The inverse of an invertible fractional ideal is unique because if \(IJ = JI = R\) then \(J \subset I^{-1}\); conversely \(I^{-1} = RI^{-1} = (JI)I^{-1} = J(II^{-1}) \subset JR = RJ \subset J\), so \(J = I^{-1}\).
