An element \(s\) in a ring extension is integral over the base ring \(R\) precisely when the subring \(R[s]\) is finitely generated as an \(R\)-module. The most subtle direction (iii) \(\Rightarrow\) (i) uses the determinant trick: express the action of \(s\) on a generating set of a faithful module \(T\) as a matrix, then the characteristic polynomial \(\det(M - sI_n)\) annihilates \(T\); faithfulness forces \(s\) to satisfy the monic polynomial \(\det(M - xI_n)\). This result is fundamental for the theory of integral extensions.
