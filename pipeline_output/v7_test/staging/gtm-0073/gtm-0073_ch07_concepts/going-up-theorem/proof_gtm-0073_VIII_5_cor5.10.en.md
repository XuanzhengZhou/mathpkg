As in the proof of Theorem 5.9, \(R \setminus P\) is a multiplicative set in \(S\). Since \(Q_1 \cap R = P_1 \subset P\), we have \(Q_1 \cap (R \setminus P) = \emptyset\). By Theorem 2.2 there is a prime ideal \(Q\) of \(S\) that contains \(Q_1\) and is maximal in the set of all ideals containing \(Q_1\) and disjoint from \(R \setminus P\). The same argument as in Theorem 5.9 shows that \(Q \cap R = P\).
