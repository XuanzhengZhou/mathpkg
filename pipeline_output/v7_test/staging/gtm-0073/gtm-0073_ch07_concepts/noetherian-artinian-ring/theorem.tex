A ring \(R\) is \textbf{left} [resp. \textbf{right}] \textbf{Noetherian} if \(R\) satisfies the ascending chain condition on left [resp. right] ideals. \(R\) is said to be \textbf{Noetherian} if \(R\) is both left and right Noetherian.

A ring \(R\) is \textbf{left} [resp. \textbf{right}] \textbf{Artinian} if \(R\) satisfies the descending chain condition on left [resp. right] ideals. \(R\) is said to be \textbf{Artinian} if \(R\) is both left and right Artinian.
