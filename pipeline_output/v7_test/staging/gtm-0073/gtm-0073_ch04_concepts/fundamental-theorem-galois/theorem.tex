If $F$ is a finite dimensional Galois extension of $K$, then there is a one-to-one correspondence between the set of all intermediate fields of the extension and the set of all subgroups of the Galois group $\mathrm{Aut}_K F$ (given by $E \mapsto E' = \mathrm{Aut}_E F$) such that:

(i) the relative dimension of two intermediate fields is equal to the relative index of the corresponding subgroups; in particular, $\mathrm{Aut}_K F$ has order $[F : K]$;

(ii) $F$ is Galois over every intermediate field $E$, but $E$ is Galois over $K$ if and only if the corresponding subgroup $E' = \mathrm{Aut}_E F$ is normal in $G = \mathrm{Aut}_K F$; in this case $G/E'$ is (isomorphic to) the Galois group $\mathrm{Aut}_K E$ of $E$ over $K$.
