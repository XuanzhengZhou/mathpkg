Let $F$ be an algebraic extension field of $K$, with $\mathrm{char}\,K = p \neq 0$. If $F$ is separable over $K$, then $F = KF^{p^n}$ for each $n \geq 1$. If $[F : K]$ is finite and $F = KF^p$, then $F$ is separable over $K$. In particular, $u \in F$ is separable over $K$ if and only if $K(u^p) = K(u)$.
