Let $F$ be an extension field of $K$. An element $u$ of $F$ is said to be algebraic over $K$ provided that $u$ is a root of some nonzero polynomial $f \in K[x]$. If $u$ is not a root of any nonzero $f \in K[x]$, $u$ is said to be transcendental over $K$. $F$ is called an algebraic extension of $K$ if every element of $F$ is algebraic over $K$; otherwise $F$ is a transcendental extension of $K$.
