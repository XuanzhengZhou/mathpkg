Let $F$ be a finite dimensional extension of $K$. Then for all $u,v \in F$ and $a \in K$:

(i) $N_K^F(u)N_K^F(v) = N_K^F(uv)$ and $N_K^F(a) = a^{[F:K]}$;

(ii) $T_K^F(u) + T_K^F(v) = T_K^F(u+v)$ and $T_K^F(a) = [F:K]a$;

(iii) if $E = K(u)$, then $N_K^F(u) = (N_K^E(u))^{[F:E]}$ and $T_K^F(u) = [F:E]T_K^E(u)$.
