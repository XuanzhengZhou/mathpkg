If $F$ is an algebraic Galois extension of $K$ (possibly infinite dimensional), then the conclusions of the Fundamental Theorem 2.5 remain valid with the following modifications: all intermediate fields are closed, and the statements about dimension and index are omitted (since they may be infinite). Furthermore:

(i) $F$ is Galois over every intermediate field $E$;

(ii') an intermediate field $E$ is Galois over $K$ if and only if $E'$ is normal in $\mathrm{Aut}_K F$, in which case $\mathrm{Aut}_K F / E' \cong \mathrm{Aut}_K E$.
