If $F$ is an extension field of $K$, then there is a one-to-one correspondence between the closed intermediate fields of the extension and the closed subgroups of the Galois group, given by $E \mapsto E' = \mathrm{Aut}_E F$. The inverse is $H \mapsto H'$ (the fixed field of $H$). An object $X$ is closed if $X = X''$.
