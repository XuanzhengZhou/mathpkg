If $K$ is a field and $f \in K[x]$ is an irreducible polynomial, then there exists a simple extension field $F = K(u)$ of $K$ such that:

(i) $u$ is a root of $f$;

(ii) $K(u)$ is unique up to $K$-isomorphism;

(iii) if $E$ is any extension of $K$ containing a root $v$ of $f$, then there is a $K$-monomorphism $K(u) \to E$ mapping $u$ to $v$.
