Let $G$ be a nonempty set with a binary operation $G \times G \to G$.
\begin{enumerate}
\item A \textbf{semigroup} is a nonempty set $G$ together with an \textit{associative} binary operation: $a(bc) = (ab)c$ for all $a,b,c \in G$.
\item A \textbf{monoid} is a semigroup $G$ that contains a \textit{(two-sided) identity element} $e \in G$ such that $ae = ea = a$ for all $a \in G$.
\item A \textbf{group} is a monoid $G$ such that for every $a \in G$ there exists a \textit{(two-sided) inverse element} $a^{-1} \in G$ with $a^{-1}a = aa^{-1} = e$.
\item A semigroup $G$ is \textbf{abelian} (or \textbf{commutative}) if its binary operation is commutative: $ab = ba$ for all $a,b \in G$.
\end{enumerate}
The \textbf{order} of a group $G$ is the cardinal number $|G|$. $G$ is \textit{finite} [resp.\ \textit{infinite}] if $|G|$ is finite [resp.\ infinite].
