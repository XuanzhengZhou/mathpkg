Two finitely generated abelian groups are isomorphic if and only if they have the same rank and the same invariant factors (up to order). Equivalently, two finitely generated abelian groups are isomorphic if and only if they have the same rank and the same elementary divisors (up to order). The invariant factors $m_1, \ldots, m_t$ and the elementary divisors are uniquely determined by the group.
