($\Rightarrow$) If $A$ is invertible with inverse $A^{-1}$, define $f: E \to F$ by its matrix $A$ relative to $U,V$ (Theorem 1.2) and $g: F \to E$ by its matrix $A^{-1}$ relative to $V,U$. Then $gf = 1_E$ and $fg = 1_F$ by Theorem 1.3, so $f$ is an isomorphism.
($\Leftarrow$) If $f$ is an isomorphism with matrix $A$ and $f^{-1}$ has matrix $B$, then $AB = I_n = BA$ by Theorem 1.3, so $A$ is invertible.
