There are three types of \textbf{elementary row operations} [resp. \textbf{elementary column operations}] on a matrix over a ring $R$ with identity:
\begin{enumerate}
\item Interchange two rows [resp. columns];
\item Multiply a row [resp. column] by a unit of $R$;
\item Add a multiple of one row [resp. column] to another row [resp. column].
\end{enumerate}
An $n \times n$ \textbf{elementary matrix} is a matrix obtained by performing a single elementary row [or column] operation on the identity matrix $I_n$.
