For any $n \times n$ matrix $A = (a_{ij})$ over a commutative ring $R$ with identity,
\[
|A| = \sum_{j=1}^{n} (-1)^{i+j} a_{ij} |A_{ij}|
\]
for each $i = 1, 2, \ldots, n$, and
\[
|A| = \sum_{i=1}^{n} (-1)^{i+j} a_{ij} |A_{ij}|
\]
for each $j = 1, 2, \ldots, n$, where $A_{ij}$ is the $(n-1) \times (n-1)$ submatrix obtained by deleting row $i$ and column $j$ of $A$.
The first formula is the \textbf{expansion along row $i$}; the second is the \textbf{expansion along column $j$}.
