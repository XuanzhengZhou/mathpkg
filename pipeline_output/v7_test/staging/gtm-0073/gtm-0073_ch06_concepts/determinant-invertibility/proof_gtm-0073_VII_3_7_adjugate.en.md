If $A$ is invertible with inverse $A^{-1}$, then $|A||A^{-1}| = |AA^{-1}| = |I_n| = 1_R$, so $|A|$ is a unit. Conversely, if $|A|$ is a unit, define $A^0$ by $(A^0)_{ij} = (-1)^{i+j}|A_{ji}|$. Using the Laplace expansion (Theorem 3.6), verify that $A A^0 = |A| I_n = A^0 A$. Then $A^{-1} = |A|^{-1} A^0$ is the inverse of $A$.
