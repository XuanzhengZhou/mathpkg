Let $M$ be the set of all $n \times m$ matrices over a division ring $D$ and let $A, B \in M$.
\begin{enumerate}
\item $A$ is equivalent to $E_r^{n,m}$ if and only if $\operatorname{rank} A = r$.
\item $A$ is equivalent to $B$ if and only if $\operatorname{rank} A = \operatorname{rank} B$.
\item The matrices $E_r^{n,m}$ ($r = 1, 2, \ldots, \min(n,m)$) constitute a set of canonical forms for the relation of equivalence on $M$.
\end{enumerate}
where $E_r^{n,m}$ is the $n \times m$ matrix with $I_r$ in the upper left corner and zeros elsewhere.
