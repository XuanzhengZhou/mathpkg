Let $\phi : E \to E$ be a linear transformation of a vector space $E$ over a field $K$.
\begin{itemize}
\item A nonzero vector $u \in E$ is an \textbf{eigenvector} of $\phi$ if there exists $k \in K$ such that $\phi(u) = ku$.
\item An element $k \in K$ is an \textbf{eigenvalue} of $\phi$ if there exists a nonzero $u \in E$ with $\phi(u) = ku$.
\item The set of all eigenvalues of $\phi$ is called the \textbf{spectrum} of $\phi$.
\end{itemize}
$k \in K$ is an eigenvalue of $\phi$ if and only if $p_\phi(k) = 0$.
