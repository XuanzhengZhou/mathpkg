(Cramer's Rule) Let $A = (a_{ij})$ be the matrix of coefficients of the system of $n$ linear equations in $n$ unknowns
\[
\sum_{j=1}^{n} a_{ij} x_j = b_i \quad (i = 1, \ldots, n)
\]
over a field $K$. If $|A| \neq 0$, then the system has a unique solution given by:
\[
x_j = |A|^{-1} \left( \sum_{i=1}^{n} (-1)^{i+j} b_i |A_{ij}| \right) \quad (j = 1, 2, \ldots, n).
\]
