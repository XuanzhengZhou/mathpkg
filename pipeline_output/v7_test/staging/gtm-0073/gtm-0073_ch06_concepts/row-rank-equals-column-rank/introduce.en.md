For any matrix over a division ring, the row rank and column rank are always equal. This fundamental result is proved via duality: the rank of a linear transformation equals its row rank (Theorem 2.3) and the rank of the dual equals the original rank (Proposition 2.4), while the dual map's matrix is the same matrix (Proposition 1.10), linking row rank to column rank.
