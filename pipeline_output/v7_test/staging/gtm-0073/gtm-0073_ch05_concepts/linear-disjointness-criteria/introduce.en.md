This lemma provides practical criteria for verifying linear disjointness: it suffices to check linear independence on a subring that generates the field (such as a polynomial ring), and even a single $K$-basis of that ring suffices. This is particularly useful when $E = K(S)$ is a purely transcendental extension and $R = K[S]$ is the polynomial ring.
