A separating transcendence base is a transcendence base $S$ such that the algebraic part $F/K(S)$ is separable (not just algebraic). A field extension is separably generated if it admits such a base. Every separable algebraic extension has the empty set as a separating transcendence base. Every purely transcendental extension is trivially separably generated. Not every transcendence base of a separably generated extension is separating.
