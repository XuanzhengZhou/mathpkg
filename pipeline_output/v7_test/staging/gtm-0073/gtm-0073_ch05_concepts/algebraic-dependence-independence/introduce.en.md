Algebraic dependence generalizes the concept of linear dependence: a set $S$ is algebraically dependent over $K$ if some nonzero polynomial with coefficients in $K$ vanishes on finitely many elements of $S$. Every subset of an algebraically independent set is algebraically independent. In particular, the empty set is algebraically independent. A singleton $\{u\}$ is algebraically dependent if and only if $u$ is algebraic over $K$. Every element of an algebraically independent set is necessarily transcendental over $K$.
