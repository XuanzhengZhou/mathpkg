Let $\sigma \colon K_1 \xrightarrow{\cong} K_2$ be an isomorphism of fields.
Let $S_1$ [resp.\ $S_2$] be an algebraically independent subset of some extension field of $K_1$ [resp.\ $K_2$] with $|S_1| = |S_2|$.
If $\varphi \colon S_1 \to S_2$ is a bijection, then $\sigma$ extends uniquely to an isomorphism
\[
\bar{\sigma} \colon K_1(S_1) \xrightarrow{\cong} K_2(S_2)
\]
such that $\bar{\sigma}(s) = \varphi(s)$ for all $s \in S_1$.
In particular, if $K_1 = K_2 = K$ and $\sigma$ is the identity, then any bijection between algebraically independent sets of equal cardinality extends to a $K$-isomorphism of the fields they generate.
