Lemma 3.11 constructs an important injective: if $J$ is a divisible group, $\operatorname{Hom}_{\mathbb{Z}}(R,J)$ with the $R$-module structure $(rf)(x) = f(xr)$ is injective. This is the coinduction functor preserving injectivity.
