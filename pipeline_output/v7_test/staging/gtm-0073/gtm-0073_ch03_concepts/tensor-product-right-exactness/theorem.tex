If $A \xrightarrow{f} B \xrightarrow{g} C \to 0$ is an exact sequence of left $R$-modules, then for every right $R$-module $D$,
\[
D \otimes_R A \xrightarrow{1_D \otimes f} D \otimes_R B \xrightarrow{1_D \otimes g} D \otimes_R C \to 0
\]
is an exact sequence of abelian groups. An analogous statement holds for an exact sequence in the first variable.
