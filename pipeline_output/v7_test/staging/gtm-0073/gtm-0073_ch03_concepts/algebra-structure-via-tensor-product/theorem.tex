Let $K$ be a commutative ring with identity and $A$ a unitary left $K$-module. Then $A$ is a $K$-algebra if and only if there exists a $K$-module homomorphism $\pi : A \otimes_K A \to A$ (the \emph{product map}) such that the associativity diagram commutes. $A$ has an identity if and only if there is a $K$-module homomorphism $I : K \to A$ (the \emph{unit map}) such that the unit diagram commutes.
