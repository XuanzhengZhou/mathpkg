Let $B$ and $C$ be submodules of a module $A$ over a ring $R$.
\begin{enumerate}
\item[(i)] There is an $R$-module isomorphism $B/(B \cap C) \cong (B + C)/C$;
\item[(ii)] if $C \subset B$, then $B/C$ is a submodule of $A/C$, and there is an $R$-module isomorphism $(A/C)/(B/C) \cong A/B$.
\end{enumerate}
