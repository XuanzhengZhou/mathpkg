If $R$ is a ring and $B$ is a submodule of an $R$-module $A$, then there is a one-to-one correspondence between the set of all submodules of $A$ containing $B$ and the set of all submodules of $A/B$, given by $C \mapsto C/B$. Hence every submodule of $A/B$ is of the form $C/B$, where $C$ is a submodule of $A$ which contains $B$.
