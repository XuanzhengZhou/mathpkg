Let $R$ be a ring. $0 \to A \xrightarrow{\varphi} B \xrightarrow{\psi} C$ is an exact sequence of $R$-modules if and only if for every $R$-module $D$,
\[
0 \to \operatorname{Hom}_R(D, A) \xrightarrow{\bar{\varphi}} \operatorname{Hom}_R(D, B) \xrightarrow{\bar{\psi}} \operatorname{Hom}_R(D, C)
\]
is an exact sequence of abelian groups (where $\bar{\varphi}(f) = \varphi f$ and $\bar{\psi}(g) = \psi g$).
