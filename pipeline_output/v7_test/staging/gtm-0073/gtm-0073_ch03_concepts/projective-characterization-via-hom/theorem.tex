The following conditions on a module $P$ over a ring $R$ are equivalent:
\begin{enumerate}
\item[(i)] $P$ is projective;
\item[(ii)] if $\psi : B \to C$ is any $R$-module epimorphism, then $\bar{\psi} : \operatorname{Hom}_R(P, B) \to \operatorname{Hom}_R(P, C)$ is an epimorphism;
\item[(iii)] if $0 \to A \to B \to C \to 0$ is any short exact sequence, then $0 \to \operatorname{Hom}_R(P, A) \to \operatorname{Hom}_R(P, B) \to \operatorname{Hom}_R(P, C) \to 0$ is exact.
\end{enumerate}
