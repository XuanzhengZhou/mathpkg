The order ideal $\mathcal{O}_a = \{r \mid ra = 0\}$ generalizes the order of an element in a group. The torsion submodule $A_t$ consists of elements with nonzero order ideals. Over a PID, cyclic modules are isomorphic to $R/\mathcal{O}_a$.
