An $R$-module $A$ is \emph{cyclic} if it is generated by a single element, that is,
\[
A = Ra = \{ra \mid r \in R\}
\]
for some $a \in A$. The element $a$ is called a \emph{generator} of $A$.

If $R$ has an identity, then every unitary cyclic $R$-module is isomorphic to $R/J$ for some left ideal $J$ of $R$ (specifically, $J = \mathcal{O}_a = \{r \in R \mid ra = 0\}$). Over a principal ideal domain $R$, every cyclic module has a unique order $r \in R$ (up to associates) such that $A \cong R/(r)$.
