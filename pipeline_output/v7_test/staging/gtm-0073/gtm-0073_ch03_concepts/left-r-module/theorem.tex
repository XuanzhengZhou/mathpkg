Let $R$ be a ring. A (left) $R$-module is an additive abelian group $A$ together with a function $R \times A \to A$ (the image of $(r,a)$ being denoted by $ra$) such that for all $r,s \in R$ and $a,b \in A$:
\begin{enumerate}
\item[(i)] $r(a + b) = ra + rb$.
\item[(ii)] $(r + s)a = ra + sa$.
\item[(iii)] $r(sa) = (rs)a$.
\end{enumerate}
If $R$ has an identity element $1_R$ and
\begin{enumerate}
\item[(iv)] $1_R a = a$ for all $a \in A$,
\end{enumerate}
then $A$ is said to be a \emph{unitary} $R$-module. If $R$ is a division ring, then a unitary $R$-module is called a (left) \emph{vector space}.
