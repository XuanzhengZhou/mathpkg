The following conditions on a module $J$ over a ring $R$ are equivalent:
\begin{enumerate}
\item[(i)] $J$ is injective;
\item[(ii)] if $\theta : A \to B$ is any $R$-module monomorphism, then $\bar{\theta} : \operatorname{Hom}_R(B, J) \to \operatorname{Hom}_R(A, J)$ is an epimorphism;
\item[(iii)] if $0 \to A \to B \to C \to 0$ is any short exact sequence, then $0 \to \operatorname{Hom}_R(C, J) \to \operatorname{Hom}_R(B, J) \to \operatorname{Hom}_R(A, J) \to 0$ is exact.
\end{enumerate}
