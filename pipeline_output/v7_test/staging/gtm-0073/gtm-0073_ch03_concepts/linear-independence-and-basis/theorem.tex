A subset $X$ of an $R$-module $A$ is said to be \emph{linearly independent} provided that for distinct $x_1, \ldots, x_n \in X$ and $r_i \in R$,
\[
r_1x_1 + r_2x_2 + \cdots + r_nx_n = 0 \Rightarrow r_i = 0 \text{ for every } i.
\]
A set that is not linearly independent is \emph{linearly dependent}. If $A$ is generated by a set $Y$, we say $Y$ \emph{spans} $A$. A linearly independent subset of $A$ that spans $A$ is called a \emph{basis}. The empty set is a basis of the zero module.
