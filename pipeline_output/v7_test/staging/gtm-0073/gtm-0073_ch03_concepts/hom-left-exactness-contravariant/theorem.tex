Let $R$ be a ring. $A \xrightarrow{\theta} B \xrightarrow{\zeta} C \to 0$ is an exact sequence of $R$-modules if and only if for every $R$-module $D$,
\[
0 \to \operatorname{Hom}_R(C, D) \xrightarrow{\bar{\zeta}} \operatorname{Hom}_R(B, D) \xrightarrow{\bar{\theta}} \operatorname{Hom}_R(A, D)
\]
is an exact sequence of abelian groups (where $\bar{\zeta}(f) = f\zeta$ and $\bar{\theta}(g) = g\theta$).
