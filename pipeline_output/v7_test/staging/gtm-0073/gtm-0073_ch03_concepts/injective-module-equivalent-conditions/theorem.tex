The following conditions on a unitary module $J$ over a ring $R$ with identity are equivalent:
\begin{enumerate}
\item[(i)] $J$ is injective;
\item[(ii)] every short exact sequence $0 \to J \to B \to C \to 0$ is split exact (hence $B \cong J \oplus C$);
\item[(iii)] $J$ is a direct summand of any module $B$ containing $J$ as a submodule.
\end{enumerate}
