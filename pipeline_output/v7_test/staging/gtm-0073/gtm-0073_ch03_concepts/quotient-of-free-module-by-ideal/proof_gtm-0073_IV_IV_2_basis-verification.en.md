If $u + IF \in F/IF$, write $u = \sum r_j x_j$. Then $u + IF = \sum (r_j + I)(x_j + IF)$, so $\pi(X)$ generates $F/IF$. For linear independence: if $\sum (r_k + I)\pi(x_k) = 0$, then $\sum r_k x_k \in IF$, so $\sum r_k x_k = \sum s_j u_j$ with $s_j \in I$. Expanding gives coefficients in $I$, so by linear independence over $R$, each $r_k \in I$. Hence $r_k + I = 0$ in $R/I$, and $\pi(X)$ is linearly independent.
