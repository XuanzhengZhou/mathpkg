Let $\mathcal{C}$ be a category with a zero object $0$ (and hence zero morphisms). A kernel of a morphism $f : C \to D$ is a difference kernel of the pair $(f, 0_{C,D})$, where $0_{C,D} : C \to D$ is the zero morphism. Equivalently, a morphism $k : K \to C$ is a kernel of $f$ if and only if:
\begin{enumerate}
\item[(i)] $k$ is a monomorphism with $f k = 0_{K,D}$;
\item[(ii)] if $h : B \to C$ is a morphism with $f h = 0_{B,D}$, then there exists a unique morphism $\bar{h} : B \to K$ such that $k \bar{h} = h$.
\end{enumerate}

Dually, a cokernel $t : D \to E$ of $f : C \to D$ is a difference cokernel of the pair $(f, 0_{C,D})$, characterized by:
\begin{enumerate}
\item[(iii)] $t$ is an epimorphism with $t f = 0_{C,E}$;
\item[(iv)] if $g : D \to F$ is a morphism with $g f = 0_{C,F}$, then there exists a unique morphism $\bar{g} : E \to F$ such that $\bar{g} t = g$.
\end{enumerate}
The kernel is unique up to equivalence and is sometimes denoted $\operatorname{Ker} f$; the cokernel is denoted $\operatorname{Coker} f$.
