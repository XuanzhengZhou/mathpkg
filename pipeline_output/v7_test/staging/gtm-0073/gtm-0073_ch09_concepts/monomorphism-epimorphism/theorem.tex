A morphism $f : C \to D$ of a category $\mathcal{C}$ is monic (or a monomorphism) if for all objects $B$ and morphisms $g, h \in \hom(B, C)$,
\[
fh = fg \implies h = g.
\]
The morphism $f$ is epic (or an epimorphism) if for all objects $E$ and morphisms $k, t \in \hom(D, E)$,
\[
kf = tf \implies k = t.
\]
