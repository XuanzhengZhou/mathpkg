A covariant functor $T : \mathcal{D} \to \mathcal{C}$ has a left adjoint if and only if there exists for each object $C$ of $\mathcal{C}$ an object $S(C)$ of $\mathcal{D}$ and a morphism $u_C : C \to T(S(C))$ such that $(S(C), u_C)$ is a universal element of the functor $\hom_{\mathcal{C}}(C, T(-)) : \mathcal{D} \to \mathcal{S}$.
