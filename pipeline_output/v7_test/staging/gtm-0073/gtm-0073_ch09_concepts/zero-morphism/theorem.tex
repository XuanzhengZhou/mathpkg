Let $\mathcal{C}$ be a category with a zero object $0$. For each ordered pair of objects $C, D$ of $\mathcal{C}$, there is a unique morphism $0_{C,D} : C \to D$, called the zero morphism, such that:
\begin{enumerate}
\item[(i)] $f \circ 0_{C,D} = 0_{C,E}$ for any morphism $f : D \to E$;
\item[(ii)] $0_{C,D} \circ g = 0_{B,D}$ for any morphism $g : B \to C$.
\end{enumerate}
Explicitly, $0_{C,D}$ is the composition $C \to 0 \to D$ of the unique morphisms to and from the zero object.
