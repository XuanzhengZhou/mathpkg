Let $\mathcal{C}$ and $\mathcal{D}$ be categories and $T : \mathcal{C} \times \mathcal{D} \to \mathcal{S}$ a functor contravariant in the first variable and covariant in the second. Suppose that for each object $C$ of $\mathcal{C}$, the functor $T(C, -) : \mathcal{D} \to \mathcal{S}$ is representable. Then there exists a covariant functor $S : \mathcal{C} \to \mathcal{D}$ such that $S(C)$ represents $T(C, -)$ for each $C$, and there is a natural isomorphism of bifunctors
\[
\hom_{\mathcal{D}}(S(-), -) \cong T(-, -).
\]
