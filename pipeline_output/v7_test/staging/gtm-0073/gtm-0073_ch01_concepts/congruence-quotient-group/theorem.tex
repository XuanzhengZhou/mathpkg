\begin{theorem}
Let $G$ be a monoid [resp. group] and let $R$ be an equivalence relation on $G$ such that $a_1 \sim a_2$ and $b_1 \sim b_2$ imply $a_1 b_1 \sim a_2 b_2$. Then the set $G/R$ of all equivalence classes is a monoid [resp. group] under the binary operation $\bar{a}\bar{b} = \overline{ab}$. If $G$ is abelian, then $G/R$ is abelian.
\end{theorem}
