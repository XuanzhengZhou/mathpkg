If an equivalence relation on a monoid or group is compatible with the binary operation (a congruence relation), then the set of equivalence classes inherits the algebraic structure. This theorem provides the foundation for constructing quotient groups and underlies the definition of $\mathbb{Z}_m$ as $\mathbb{Z}$ modulo congruence modulo $m$.
