\begin{theorem}
If $f: G \rightarrow H$ is a homomorphism of groups and $N$ is a normal subgroup of $G$ contained in the kernel of $f$, then there is a unique homomorphism $\bar{f}: G/N \rightarrow H$ such that $\bar{f}(aN) = f(a)$ for all $a \in G$. Moreover, $\operatorname{Im} \bar{f} = \operatorname{Im} f$ and $\operatorname{Ker} \bar{f} = (\operatorname{Ker} f)/N$. $\bar{f}$ is an isomorphism if and only if $f$ is an epimorphism and $N = \operatorname{Ker} f$.
\end{theorem}
