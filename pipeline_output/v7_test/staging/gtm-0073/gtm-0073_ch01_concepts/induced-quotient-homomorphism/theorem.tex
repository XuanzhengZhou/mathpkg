\begin{theorem}
If $f : G \rightarrow H$ is a homomorphism of groups, $N \triangleleft G$, $M \triangleleft H$, and $f(N) < M$, then $f$ induces a homomorphism $\bar{f} : G/N \rightarrow H/M$, given by $aN \mapsto f(a)M$. $\bar{f}$ is an isomorphism if and only if $\operatorname{Im} f \vee M = H$ and $f^{-1}(M) \subset N$. In particular if $f$ is an epimorphism such that $f(N) = M$ and $\operatorname{Ker} f \subset N$, then $\bar{f}$ is an isomorphism.
\end{theorem}
