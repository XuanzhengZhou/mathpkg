\begin{definition}
An object $I$ in a category $\mathcal{C}$ is said to be \textbf{universal} (or \textbf{initial}) if for each object $C$ of $\mathcal{C}$ there exists one and only one morphism $I \rightarrow C$. An object $T$ of $\mathcal{C}$ is said to be \textbf{couniversal} (or \textbf{terminal}) if for each object $C$ of $\mathcal{C}$ there exists one and only one morphism $C \rightarrow T$.
\end{definition}
