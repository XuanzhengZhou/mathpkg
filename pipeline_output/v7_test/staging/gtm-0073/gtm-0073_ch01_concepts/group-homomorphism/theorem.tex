\begin{definition}
Let $G$ and $H$ be semigroups. A function $f: G \rightarrow H$ is a \textbf{homomorphism} provided $f(ab) = f(a)f(b)$ for all $a,b \in G$. If $f$ is injective as a map of sets, $f$ is a \textbf{monomorphism}. If $f$ is surjective, $f$ is an \textbf{epimorphism}. $f$ is an \textbf{isomorphism} if it is both injective and surjective. A homomorphism $G \rightarrow G$ is an \textbf{endomorphism} of $G$, and an isomorphism $G \rightarrow G$ is an \textbf{automorphism} of $G$.
\end{definition}
