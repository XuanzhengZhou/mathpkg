\begin{theorem}
If $f : G \rightarrow H$ is an epimorphism of groups, then the assignment $K \mapsto f(K)$ defines a one-to-one correspondence between the set $S_f(G)$ of all subgroups $K$ of $G$ which contain $\operatorname{Ker} f$ and the set $S(H)$ of all subgroups of $H$. Under this correspondence normal subgroups correspond to normal subgroups.
\end{theorem}
