Let $G$ be a nonempty set equipped with a binary operation $G \times G \to G$, written multiplicatively.

\begin{itemize}
\item $G$ is a \textbf{semigroup} if the operation is associative:
  $a(bc) = (ab)c$ for all $a,b,c \in G$.
\item $G$ is a \textbf{monoid} if it is a semigroup and there exists a (two-sided)
  \textbf{identity element} $e \in G$ such that $ae = ea = a$ for all $a \in G$.
\item $G$ is a \textbf{group} if it is a monoid and for every $a \in G$ there exists a
  (two-sided) \textbf{inverse element} $a^{-1} \in G$ such that
  $a^{-1}a = aa^{-1} = e$.
\item $G$ is \textbf{abelian} (commutative) if $ab = ba$ for all $a,b \in G$.
\end{itemize}

The \textbf{order} of $G$ is the cardinal number $|G|$. $G$ is \textbf{finite} if $|G|$ is
finite; otherwise $G$ is \textbf{infinite}.
