\begin{definition}
A \textbf{concrete category} is a category $\mathcal{C}$ together with a function $\sigma$ that assigns to each object $A$ of $\mathcal{C}$ a set $\sigma(A)$ (called the \textbf{underlying set} of $A$) in such a way that:
\begin{enumerate}
    \item[(i)] every morphism $A \rightarrow B$ of $\mathcal{C}$ is a function on the underlying sets $\sigma(A) \rightarrow \sigma(B)$;
    \item[(ii)] the identity morphism of each object $A$ of $\mathcal{C}$ is the identity function on the underlying set $\sigma(A)$;
    \item[(iii)] composition of morphisms in $\mathcal{C}$ agrees with composition of functions on the underlying sets.
\end{enumerate}
\end{definition}
