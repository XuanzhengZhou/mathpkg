\begin{definition}
The permutations $\sigma_1,\sigma_2,\ldots,\sigma_r$ of $S_n$ are said to be \textbf{disjoint} provided that for each $1 \leq i \leq r$, and every $k \in I_n$, $\sigma_i(k) \neq k$ implies $\sigma_j(k) = k$ for all $j \neq i$. In other words, $\sigma_1,\sigma_2,\ldots,\sigma_r$ are disjoint if and only if no element of $I_n$ is moved by more than one of $\sigma_1,\ldots,\sigma_r$.
\end{definition}
