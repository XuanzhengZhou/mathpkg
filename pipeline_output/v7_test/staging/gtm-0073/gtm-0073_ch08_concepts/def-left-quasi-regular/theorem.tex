An element $a$ in a ring $R$ is said to be \textbf{left quasi-regular} if there exists $r \in R$ such that $r + a + ra = 0$. The element $r$ is called a \textit{left quasi-inverse} of $a$. A (right, left or two-sided) ideal $I$ of $R$ is said to be \textbf{left quasi-regular} if every element of $I$ is left quasi-regular. Similarly, $a \in R$ is \textbf{right quasi-regular} if there exists $r \in R$ such that $a + r + ar = 0$.
