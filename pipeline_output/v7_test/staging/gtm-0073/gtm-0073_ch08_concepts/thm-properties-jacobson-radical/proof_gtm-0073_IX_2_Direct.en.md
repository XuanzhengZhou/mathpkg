(i) If R is primitive, it has a faithful simple module A. The annihilator of A is 0. Since J(R) is contained in this annihilator (Theorem 2.3(i)), J(R) = 0. (ii) If R is simple, J(R) is either 0 or R. If J(R) = 0, then R is semisimple and must be primitive (since J(R) being the intersection of left primitive ideals being 0 forces the existence of left primitive ideals). (iii) Follows from simplicity: the ideal J(R) is either 0 or R.
