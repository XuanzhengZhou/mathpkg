A ring $Q(R)$ is a \textbf{left quotient ring} of $R$ if $R$ is a subring of $Q(R)$, every regular element of $R$ is a unit in $Q(R)$, and every element of $Q(R)$ can be written as $a^{-1}b$ where $a,b \in R$ and $a$ is regular in $R$. Similarly, $Q(R)$ is a \textbf{right quotient ring} if every element can be written as $ba^{-1}$ with $a$ regular.
