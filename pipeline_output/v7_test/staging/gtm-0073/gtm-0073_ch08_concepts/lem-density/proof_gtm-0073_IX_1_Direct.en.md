Proof by induction on n = dim_D V. If n = 0, V = 0 and a != 0. Since A is simple, Ra = A, so there exists r in R with ra = a != 0 and rV = 0. Suppose n > 0 and true for smaller dimensions. Let {u_1, ..., u_{n-1}, u} be a D-basis of V. Let W be spanned by the first n-1 vectors and I = alpha(W) the left annihilator. Then Iu is an R-submodule of A. Since u not in W, by induction there exists r with ru != 0 and rW = 0. Thus Iu != 0, so Iu = A by simplicity. Hence there exists s in I with su = a. But sW = 0 and sa = a != 0, giving the needed element.
