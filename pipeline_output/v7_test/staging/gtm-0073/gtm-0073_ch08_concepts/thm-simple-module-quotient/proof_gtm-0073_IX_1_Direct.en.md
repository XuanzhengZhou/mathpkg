Every simple R-module is cyclic: if a != 0 in A, then Ra = A since Ra is a nonzero submodule. The map theta: R -> A given by r -> ra is an R-module epimorphism with kernel I, a left ideal. By the First Isomorphism Theorem, R/I is isomorphic to A. The submodules of R/I correspond to left ideals J with I subset J subset R, so R/I (and hence A) has no proper submodules iff I is maximal. The condition RA != 0 (i.e., R(R/I) != 0) is equivalent to I being regular. Conversely, if I is a regular maximal left ideal, then R/I is a simple R-module.
