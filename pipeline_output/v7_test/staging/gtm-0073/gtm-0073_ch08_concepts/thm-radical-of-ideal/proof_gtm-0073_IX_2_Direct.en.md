(i) I cap J(R) is an ideal of I. If a in I cap J(R), then a is left quasi-regular in R, and the quasi-inverse r = -a - ra lies in I. Thus I cap J(R) subset J(I). Conversely, if a in J(I), then for any r in R, (ra)^2 in IJ(I) subset J(I), so ra is left quasi-regular in R. Hence Ra is a left quasi-regular left ideal of R, so a in J(R). Thus J(I) subset I cap J(R). (ii) If J(R) = 0, then J(I) = I cap 0 = 0. (iii) Apply (i) with I = J(R): J(J(R)) = J(R) cap J(R) = J(R).
