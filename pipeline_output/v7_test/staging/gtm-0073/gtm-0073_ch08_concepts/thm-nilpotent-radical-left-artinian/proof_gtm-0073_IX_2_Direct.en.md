Consider the descending chain J(R) superset J(R)^2 superset J(R)^3 superset ... Since R is left Artinian, this stabilizes: J(R)^k = J(R)^{k+1} = ... for some k. If J(R)^k != 0, pick a nonzero element a in J(R)^k minimal such that the left ideal J(R)^k a is nonzero. Then derive a contradiction using quasi-regularity of elements in J(R). Thus J(R)^k = 0. The last statement follows from Theorem 2.12.
