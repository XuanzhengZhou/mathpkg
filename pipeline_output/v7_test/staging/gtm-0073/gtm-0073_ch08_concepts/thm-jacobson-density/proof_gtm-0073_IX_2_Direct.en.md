For each r in R, define alpha_r: A -> A by alpha_r(a) = ra. Since the R-action commutes with D = Hom_R(A,A), each alpha_r is a D-linear transformation. The map alpha: R -> Hom_D(A,A) is a ring homomorphism. Since A is faithful (alpha(A) = 0), alpha is injective, so R is isomorphic to Im(alpha). To show Im(alpha) is dense: given D-linearly independent {u_1,...,u_n} and arbitrary {v_1,...,v_n} in A, for each i let V_i be spanned by all u_j except u_i. By Lemma 1.11, there exists r_i in R with r_i u_i != 0 and r_i V_i = 0. Then by simplicity, R(r_i u_i) = A, so there exists s_i with s_i r_i u_i = v_i. Also there exists t_i with t_i r_i u_i = u_i (by another application of Lemma 1.11). Then s_i r_i t_i r_i u_i = s_i r_i u_i = v_i. Define r = sum s_i r_i. Then for each i, r u_i = v_i because all cross-terms vanish (r_j u_i = 0 for j != i since u_i in V_j).
