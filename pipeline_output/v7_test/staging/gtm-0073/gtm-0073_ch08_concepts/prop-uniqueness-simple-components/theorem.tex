Let $R$ be a semisimple left Artinian ring.
\begin{itemize}
\item[(i)] There exist ideals $I_1, \ldots, I_n$ of $R$, each a simple ring, such that $R = I_1 \times I_2 \times \cdots \times I_n$ (internal direct product).
\item[(ii)] If $J$ is any simple ideal of $R$, then $J = I_k$ for some $k$.
\item[(iii)] If $R = J_1 \times \cdots \times J_m$ with each $J_k$ a simple ideal, then $n = m$ and (after reindexing) $I_k = J_k$ for $k = 1, \ldots, n$.
\end{itemize}
