For $i = 1,2$ let $V_i$ be a vector space of finite dimension $n_i$ over the division ring $D_i$.
\begin{itemize}
\item[(i)] If there is an isomorphism of rings $\operatorname{Hom}_{D_1}(V_1,V_1) \cong \operatorname{Hom}_{D_2}(V_2,V_2)$, then $\dim_{D_1}V_1 = \dim_{D_2}V_2$ and $D_1$ is isomorphic to $D_2$.
\item[(ii)] If there is an isomorphism of rings $\operatorname{Mat}_{n_1}D_1 \cong \operatorname{Mat}_{n_2}D_2$, then $n_1 = n_2$ and $D_1$ is isomorphic to $D_2$.
\end{itemize}
