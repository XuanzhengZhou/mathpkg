(=>) If I is a nilpotent ideal and K any prime ideal, then I^n = 0 in K for some n, so I subset K. Thus I subset P(R). If R is semiprime, P(R) = 0, so the only nilpotent ideal is 0. (<=) Suppose R has no nonzero nilpotent ideals. Show every nonzero a has a prime ideal not containing it. If a != 0, then RaR != 0 (otherwise a in A_r(R) which is nilpotent). Construct an m-system avoiding 0 and containing a; by Zorn, get a prime ideal not containing a.
