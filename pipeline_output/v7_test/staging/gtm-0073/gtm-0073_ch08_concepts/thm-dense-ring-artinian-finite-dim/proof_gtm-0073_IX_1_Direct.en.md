If R is left Artinian and dim_D V is infinite, pick an infinite linearly independent set {u_1, u_2, ...}. Let I_n be the left annihilator in R of {u_1, ..., u_n}. By density, each I_n properly contains I_{n+1}, giving an infinite strictly descending chain, contradicting left Artinian. So dim_D V must be finite. If dim_D V = n is finite, then for any theta in Hom_D(V,V), the density condition for a basis {u_1,...,u_n} with targets theta(u_i) gives an element of R that agrees with theta on the basis, hence everywhere. Thus R = Hom_D(V,V). Conversely, if dim_D V is finite, Hom_D(V,V) is isomorphic to Mat_n(D), which is Artinian.
