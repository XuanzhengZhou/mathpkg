Let $W$ be a finite-dimensional vector space, $M$ a $d$-dimensional subspace, $L = \wedge^d M \subset V = \wedge^d W$. For $x \in \operatorname{GL}(W)$ and $\mathbf{x} \in \mathfrak{gl}(W)$:

(a) $(\wedge^d x)L = L$ if and only if $x(M) = M$.
(b) $(d\wedge^d)(\mathbf{x})L \subset L$ if and only if $\mathbf{x}(M) \subset M$.
