Since indices for type B2 are $\langle \alpha, \beta \rangle = -1$, $\langle \beta, \alpha \rangle = -2$, we have $\alpha(t_{\beta}) = -1$, $\beta(t_{\alpha}) = 1$. From Lemma 32.5, there exist scalars $a, b, c, d \in K$ (independent of $x, y$) governing the commutation formulas. Using the action of Int $n_{\alpha}$ and Int $n_{\beta}$ and the fact that $\alpha + 2\beta \notin \Phi$, one determines these scalars: $a = b = 1, c = 2, d = -1$. The identity $(n_{\alpha}n_{\beta})^{4} = t_{\alpha}$ follows from the commutation relations and the definition of $t_{\alpha}$.
