Let $\Phi_{\alpha\beta}$ be of type $B_2$, with positive roots $\alpha, \beta, \alpha+\beta, 2\alpha+\beta$. Set $x_{\alpha+\beta} = \operatorname{Ad} n_{\beta}(x_{\alpha})$, $x_{2\alpha+\beta} = \operatorname{Ad} n_{\alpha}(x_{\alpha+\beta})$. Then for all $x, y \in \mathbf{G}_a$:
(a) $\varepsilon_{\beta}(y)\varepsilon_{\alpha}(x) = \varepsilon_{\alpha}(x)\varepsilon_{\beta}(y)\varepsilon_{\alpha+\beta}(xy)\varepsilon_{2\alpha+\beta}(x^2 y)$,
  $\varepsilon_{\alpha+\beta}(y)\varepsilon_{\alpha}(x) = \varepsilon_{\alpha}(x)\varepsilon_{\alpha+\beta}(y)\varepsilon_{2\alpha+\beta}(2xy)$.
(b) $\operatorname{Ad} n_{\alpha}(x_{\alpha+\beta}) = -x_{\alpha+\beta}$, $\operatorname{Ad} n_{\alpha}(x_{2\alpha+\beta}) = x_{\beta}$,
  $\operatorname{Ad} n_{\beta}(x_{\alpha+\beta}) = -x_{\alpha}$, $\operatorname{Ad} n_{\beta}(x_{2\alpha+\beta}) = x_{2\alpha+\beta}$.
(c) $(n_{\alpha}n_{\beta})^{4} = (n_{\beta}n_{\alpha})^{4} = t_{\alpha}$.
