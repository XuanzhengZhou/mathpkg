For a root system $\Phi_{\alpha\beta}$ of type $G_2$, let the positive roots be $\alpha, \beta, \alpha+\beta, 2\alpha+\beta, 3\alpha+\beta, 3\alpha+2\beta$. Setting
$$x_{\alpha+\beta} = \operatorname{Ad} n_\beta(x_\alpha), \quad x_{2\alpha+\beta} = \operatorname{Ad} n_\alpha(x_{\alpha+\beta}),$$
the commutation relations between root groups are determined by the Chevalley commutator formula.
