If G is solvable, this is Proposition 19.4. In general, for x in C_G(S), show x lies in a Borel subgroup containing S. x lies in some Borel B (22.2). S acts on G/B with fixed point set X, which is closed hence complete, and nonempty (contains class of B). Both S and x stabilize X. By Borel Fixed Point Theorem, there exists a fixed point whose stabilizer is a Borel subgroup containing both S and x. Since C_G(S) in this Borel subgroup is connected (solvable case), x in C_G(S)^circ.
