We first introduce some notations and prove an auxiliary lemma. If $v$ is a valuation of a field $K$ we shall denote by $L(v)$ the set of all valuations $v'$ of $K$ such that $R_v < R_{v'} < K$. In other words, $L(v)$ is the set of all non-trivial valuations $v'$ such that $v$ is composite with and is non-equivalent to $v'$. We denote by $E(v)$ the set of distinct (i.e., non-equivalent) extensions of $v$ to $K^\star$.
