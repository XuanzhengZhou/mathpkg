Let $A$ be a local ring. The following properties are equivalent:

(a) $A$ is a Macaulay ring;
(b) There exists a system of parameters of $A$ which is a prime sequence;
(c) Every system of parameters of $A$ is a prime sequence.

The equivalence of (a) and (b) has already been established. It is clear that (c) implies (b) since a local ring admits at least one system of parameters. Now, if $A$ is a Macaulay ring and if $\{a_1, \cdots, a_d\}$ is a system of parameters of $A$, we have $\dim (A / (Aa_1 + \cdots + Aa_d)) = 0 = d - d$, whence Theorem 2 shows that $\{a_1, \cdots, a_d\}$ is a prime sequence; thus (a) implies (c). Q.E.D.
