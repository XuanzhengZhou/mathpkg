Let $f(x) \in \Phi[x]$ have positive degree. Then $f(x)$ has a multiple root in its splitting field if and only if $(f, f') \neq 1$, where $f'$ is the formal derivative of $f$. In particular, if $f$ is irreducible, then $f$ has a multiple root if and only if $f' = 0$. In characteristic $0$, an irreducible polynomial always has distinct roots. In characteristic $p \neq 0$, an irreducible polynomial $f$ has multiple roots if and only if $f(x) = g(x^p)$ for some $g(x) \in \Phi[x]$.
