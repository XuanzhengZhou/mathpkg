Theorem 14: For an infinite base field, any extension generated by a separable algebraic element and an arbitrary algebraic element is simple. The proof constructs a primitive element of the form $\xi + \gamma\eta$ for a suitable $\gamma \in \Phi$, using the fact that an infinite field avoids the finite set of values that would make two conjugates coincide.
