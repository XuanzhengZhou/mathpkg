Let $P$ be an extension field of $\Phi$ and let $M$ be a vector space over $\Phi$. Then $P \otimes_\Phi M$ can be considered as a vector space over $P$ via
$$\rho\left(\sum \rho_i \otimes x_i\right) = \sum \rho\rho_i \otimes x_i, \quad \rho, \rho_i \in P, \; x_i \in M.$$
Denote this space by $M_P$. Then $\dim_P M_P = \dim_\Phi M$. If $A$ is a linear transformation in $M$ over $\Phi$, then $1 \otimes A$ is a linear transformation in $M_P$ over $P$ with the same matrix relative to the extended basis $\{1 \otimes e_i\}$.
