A polynomial $f(x)$ over a field is called separable if it has no multiple roots in its splitting field. The formal derivative provides an algebraic criterion for detecting multiple roots: $f$ has a multiple root if and only if $\gcd(f, f') \neq 1$. For irreducible polynomials, this criterion shows that in characteristic 0 all irreducible polynomials are separable, while in characteristic $p > 0$, an irreducible $f$ is inseparable if and only if $f(x) = g(x^p)$ for some polynomial $g$. This distinction between characteristic 0 and $p$ is fundamental in field theory.
