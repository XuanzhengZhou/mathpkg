An algebraic extension $P/\Phi$ is called separable if every element $\rho \in P$ is separable over $\Phi$ (i.e., its minimum polynomial has distinct roots). Let $A/\Phi$ be the algebraic closure of $\Phi$ in some extension. Then the subset $\Sigma \subseteq A$ of elements separable over $\Phi$ is a subfield containing $\Phi$, and every element of $A$ separable over $\Sigma$ is already in $\Sigma$.

An algebraic extension $A/\Phi$ is called purely inseparable if the only separable elements over $\Phi$ are those in $\Phi$. An algebraic element $\rho$ is purely inseparable over $\Phi$ if its minimum polynomial has only one distinct root.
