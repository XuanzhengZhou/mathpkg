Let $P$ be a field and $\mathfrak{A}$ a set of endomorphisms of $(P, +)$ such that:
\begin{enumerate}
\item[(i)] $\mathfrak{A}$ is a subring of $\mathfrak{L}(P, P)$ (containing the identity).
\item[(ii)] $\mathfrak{A}$ is a subspace of $\mathfrak{L}(P, P)$ as right vector space over $P$.
\item[(iii)] $[\mathfrak{A}:P]_R = n < \infty$.
\end{enumerate}
Let $\Phi = \{\alpha \in P \mid \alpha_R A = A \alpha_R \text{ for all } A \in \mathfrak{A}\}$. Then $\Phi$ is a subfield of $P$, $[P:\Phi] = n$, and $\mathfrak{A} = \mathfrak{L}_\Phi(P, P)$.

This establishes a bijection between:
\begin{itemize}
\item $\mathcal{R}$ = the set of subrings $\mathfrak{A}$ of $\mathfrak{L}(P,P)$ which are also finite-dimensional right $P$-subspaces,
\item $\mathcal{F}$ = the set of subfields $\Phi$ of $P$ of finite co-dimension,
\end{itemize}
via the maps $R(\Phi) = \mathfrak{L}_\Phi(P, P)$ and $F(\mathfrak{A}) = \{\alpha \in P \mid \alpha_R A = A \alpha_R, \forall A \in \mathfrak{A}\}$.
