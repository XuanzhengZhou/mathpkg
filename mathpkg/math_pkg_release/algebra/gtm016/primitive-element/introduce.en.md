An element $x \in H$ is primitive if $\varepsilon_H(x) = 0$ and $\Delta_H x = x \otimes I + I \otimes x$. The set of primitive elements is denoted $D(H)$. An element $x$ is primitive modulo $S$ if $\varepsilon_H(x) = 0$ and $\Delta_H x = x \otimes I + I \otimes x + \sum_{i \geq 3} {}_i x \otimes x_i$ with ${}_i x, x_i \in S^0$.
