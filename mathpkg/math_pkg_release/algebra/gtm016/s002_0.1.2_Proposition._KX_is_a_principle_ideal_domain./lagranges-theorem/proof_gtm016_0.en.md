The left cosets of $H$ partition $G$. Each coset $xH$ has cardinality $|H|$ (the map $h \mapsto xh$ is a bijection). There are $(G:H)$ distinct cosets. Therefore $|G| = (G:H)|H|$. The element order statement follows by applying the theorem to the cyclic subgroup generated by the element.
