By induction on $G:1$. Use the class equation $G:1 = C(G):1 + \sum G:G_{x_i}$. If any proper centralizer $G_{x_i}$ has order divisible by $p$, induction applies. Otherwise $p$ divides each $G:G_{x_i}$, so $p$ divides $C(G):1$. Since $C(G)$ is Abelian, it has an element of order $p$.
