Write $G = \prod G_{p_i}$. Each $G_{p_i}$ has an element $g_i$ whose order is the exponent of $G_{p_i}$. Then $g = \prod g_i$ has order $\prod p_i^{e_i}$ which equals $\operatorname{Exp} G$.
