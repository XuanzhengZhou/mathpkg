A \emph{representation} of a $C^*$-algebra $A$ is a $\ast$-homomorphism $\pi: A \to \mathcal{L}(\mathcal{H})$. $\pi$ is \emph{nondegenerate} if $[\pi(A)\mathcal{H}] = \mathcal{H}$. $\pi$ is \emph{cyclic} if $\mathcal{H} = [\pi(A)\xi]$ for some $\xi$. $\pi$ and $\sigma$ are \emph{equivalent} if $\sigma(x) = U\pi(x)U^*$ for a unitary $U$. $\pi$ is \emph{irreducible} if $\pi(A)$ has no nontrivial closed invariant subspaces.
