Let $\mathcal{H}$ be a separable Hilbert space and $\pi$ a representation of a separable abelian $C^*$-algebra on $\mathcal{H}$. Then $\pi$ is multiplicity-free if and only if $\pi$ has a cyclic vector. An abelian von Neumann algebra $\mathcal{R}$ on $\mathcal{H}$ is maximal abelian if and only if $\mathcal{R}$ has a cyclic vector.
