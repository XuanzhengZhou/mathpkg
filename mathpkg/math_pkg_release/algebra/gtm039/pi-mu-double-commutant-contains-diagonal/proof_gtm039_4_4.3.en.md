Let $\mathcal{B}_0$ be the class of Borel sets whose characteristic functions give diagonal operators in $\pi_\mu(A)''$. Using an approximate unit $\{e_j\}$ for a CCR ideal, show $L_{E_j} \to L_f$ strongly where $f = \chi_{\hat{A}_n \setminus q(S_{k\ell})}$, so these sets belong to $\mathcal{B}_0$. The $S_{k\ell}$ separate points (by the functional calculus), so $\mathcal{B}_0$ is a countably generated separating $\sigma$-field. By 3.3.5, $\mathcal{B}_0$ is the full Borel $\sigma$-field. Thus all diagonal operators are in $\pi_\mu(A)''$. $\square$
