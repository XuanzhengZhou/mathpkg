Let $A = f(N^\infty)$, $B = g(N^\infty)$. Using the sieve basis, suppose no Borel set separates. Construct inductively sequences $m_1,m_2,\ldots$ and $n_1,n_2,\ldots$ such that $f(U_{m_1\ldots m_k})$ and $g(U_{n_1\ldots n_k})$ cannot be separated for any $k$. Let $\mu = (m_1,m_2,\ldots)$, $\nu = (n_1,n_2,\ldots)$. Then $f(\mu) \neq g(\nu)$, so there exist disjoint open neighborhoods. For large $k$, their preimages are contained in these neighborhoods, contradiction. $\square$
