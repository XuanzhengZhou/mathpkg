If $f(x)g(x) = 1$, then taking degrees gives $\deg f(x) + \deg g(x) = \deg 1 = 0$. Since degrees are non-negative integers (or $-\infty$), the only way this sum can be $0$ is if $\deg f(x) = 0$ and $\deg g(x) = 0$. Hence $f(x) = a \in \mathfrak{A}$ and $g(x) = b \in \mathfrak{A}$ with $ab = 1$, so $a$ and $b$ are units in $\mathfrak{A}$. For the ring of integers $I$, the only units are $\pm 1$. If $\mathfrak{J}$ is a field, every non-zero element is a unit, so the units of $\mathfrak{J}[x]$ are exactly the non-zero constant polynomials.
