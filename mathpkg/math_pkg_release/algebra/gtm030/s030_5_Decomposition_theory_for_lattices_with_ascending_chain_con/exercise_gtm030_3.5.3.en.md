Prove that, if the elements $a_1, a_2, \dots, a_n$ are independent and $(a_1 \cup \dots \cup a_n) \cap a_{n+1} = 0$, then the elements $a_1, a_2, \dots, a_{n+1}$ are independent. Prove that the set $a_1, a_2, \dots, a_n$ is independent if and only if $(a_1 \cup \dots \cup a_i) \cap a_{i+1} = 0$, for $i = 1, 2, \dots, n-1$.
