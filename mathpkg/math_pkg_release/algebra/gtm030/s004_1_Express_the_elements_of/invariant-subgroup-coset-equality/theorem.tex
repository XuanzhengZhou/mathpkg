If $\mathfrak{H}$ is an invariant subgroup of $\mathfrak{G}$, then for all $y \in \mathfrak{G}$,
\[
\mathfrak{H}y = y\mathfrak{H}.
\]
Hence, for an invariant subgroup, the right coset determined by any element coincides with the left coset determined by that element, and there is only one coset decomposition.
