For any element $xh \in x\mathfrak{H}$, its inverse is $(xh)^{-1} = h^{-1}x^{-1}$. Since $\mathfrak{H}$ is a subgroup, $h^{-1} \in \mathfrak{H}$, so $h^{-1}x^{-1} \in \mathfrak{H}x^{-1}$. Thus the set of inverses of elements of $x\mathfrak{H}$ is contained in the left coset $\mathfrak{H}x^{-1}$. Conversely, any element of $\mathfrak{H}x^{-1}$ has the form $hx^{-1}$ for some $h \in \mathfrak{H}$, and $(hx^{-1})^{-1} = xh^{-1} \in x\mathfrak{H}$, so $hx^{-1}$ is the inverse of an element of $x\mathfrak{H}$. Therefore the set of inverses of $x\mathfrak{H}$ is exactly $\mathfrak{H}x^{-1}$. If $x\mathfrak{H} = y\mathfrak{H}$, then taking inverses gives $\mathfrak{H}x^{-1} = \mathfrak{H}y^{-1}$, so the correspondence is well-defined. It is clearly bijective, with inverse map $\mathfrak{H}x \mapsto x^{-1}\mathfrak{H}$.
