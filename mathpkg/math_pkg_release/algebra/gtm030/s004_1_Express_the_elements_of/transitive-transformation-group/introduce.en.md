A transformation group $\mathfrak{G}$ is transitive on $S$ if it can map any element of $S$ to any other element. Equivalently, the congruence relation modulo $\mathfrak{G}$ has exactly one equivalence class — the whole set $S$. Transitivity is a fundamental property studied in the theory of group actions and permutation groups.
