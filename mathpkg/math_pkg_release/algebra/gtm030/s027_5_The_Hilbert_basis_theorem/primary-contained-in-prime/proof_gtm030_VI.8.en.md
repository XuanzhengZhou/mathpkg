If $z \equiv 0 \pmod{\mathfrak{P}} = \mathfrak{R}(\mathfrak{Q})$, then $z^r \equiv 0 \pmod{\mathfrak{Q}}$ for some integer $r$. Since $\mathfrak{Q} \subseteq \mathfrak{P}'$, we have $z^r \equiv 0 \pmod{\mathfrak{P}'}$. Since $\mathfrak{P}'$ is prime, $z \equiv 0 \pmod{\mathfrak{P}'}$. Hence $\mathfrak{P} \subseteq \mathfrak{P}'$.
