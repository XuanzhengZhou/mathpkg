An ideal $\mathfrak{P}$ of a commutative ring $\mathfrak{A}$ is **prime** if for any $a, b \in \mathfrak{A}$,
$$ab \equiv 0 \pmod{\mathfrak{P}}$$
implies that either $a \equiv 0 \pmod{\mathfrak{P}}$ or $b \equiv 0 \pmod{\mathfrak{P}}$.

Equivalently, $\mathfrak{A}/\mathfrak{P}$ is an integral domain. The zero ideal $0$ is prime if and only if $\mathfrak{A}$ is an integral domain.
