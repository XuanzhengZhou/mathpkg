An element $u \in \mathfrak{Q} : \mathfrak{C}$ satisfies $uc \equiv 0 \pmod{\mathfrak{Q}}$ for all $c \in \mathfrak{C}$. If $u \notin \mathfrak{P}$, then since $\mathfrak{Q}$ is primary, $c \equiv 0 \pmod{\mathfrak{Q}}$ for all $c \in \mathfrak{C}$, which would mean $\mathfrak{C} \subseteq \mathfrak{Q} \subseteq \mathfrak{P}$, contradicting the hypothesis that $\mathfrak{C} \not\subseteq \mathfrak{P}$. Hence $u \in \mathfrak{P}$. But then $u^r \equiv 0 \pmod{\mathfrak{Q}}$ for some $r$. Since $u \in \mathfrak{Q} : \mathfrak{C}$, we have $u \cdot c \equiv 0 \pmod{\mathfrak{Q}}$ for some $c \in \mathfrak{C}$ not in $\mathfrak{P}$. Then $uc \equiv 0 \pmod{\mathfrak{Q}}$, so choosing $c \notin \mathfrak{P}$ and using the primariness of $\mathfrak{Q}$, we must actually have $u \in \mathfrak{Q}$. Thus $\mathfrak{Q} : \mathfrak{C} = \mathfrak{Q}$.
