Let $\mathfrak{g}$ be a Noetherian subring of a ring $\mathfrak{A}$. An element $a \in \mathfrak{A}$ is integral over $\mathfrak{g}$ (a $\mathfrak{g}$-integer) if and only if there exists a finitely generated $\mathfrak{g}$-submodule of $\mathfrak{A}$ that contains all the powers $1, a, a^2, \dots$ of $a$.
