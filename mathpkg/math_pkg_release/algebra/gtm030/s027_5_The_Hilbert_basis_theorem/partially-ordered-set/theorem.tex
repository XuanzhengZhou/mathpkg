A **partially ordered set** is a system consisting of a set $S$ and a relation $\geq$ ("greater than or equals" or "contains") satisfying the following postulates:

\begin{itemize}
\item[P1.] $a \geq b$ and $b \geq a$ hold if and only if $a = b$.
\item[P2.] If $a \geq b$ and $b \geq c$, then $a \geq c$.
\end{itemize}

If $a$ and $b$ are any elements of $S$, we may have $a \geq b$ or not; in the latter case we write $a \ngeq b$. If $a \geq b$ and $a \neq b$, we write $a > b$. We use $b \leq a$ and $b < a$ as alternatives for $a \geq b$ and $a > b$.

If every pair of elements is comparable ($a \geq b$ or $b \geq a$), then $S$ is said to be **linearly ordered** or is a **chain**.
