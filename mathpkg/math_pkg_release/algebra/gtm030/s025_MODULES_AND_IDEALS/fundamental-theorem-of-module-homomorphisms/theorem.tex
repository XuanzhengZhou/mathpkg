$$
\text{Let } \eta: \mathfrak{M} \to \mathfrak{M}' \text{ be an } \mathfrak{A}\text{-module homomorphism.}
\text{ Then the image } \mathfrak{M}\eta \text{ is a submodule of } \mathfrak{M}',
\text{ the kernel } \mathfrak{R} = \ker \eta \text{ is a submodule of } \mathfrak{M},
\text{ and there is a canonical isomorphism:}
$$
$$
\mathfrak{M}\eta \cong \mathfrak{M} / \mathfrak{R}.
$$
