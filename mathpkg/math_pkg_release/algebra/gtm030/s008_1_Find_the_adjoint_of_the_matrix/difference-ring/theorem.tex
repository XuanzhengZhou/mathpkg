Let $\mathfrak{B}$ be an ideal of the ring $\mathfrak{A}$. The set $\bar{\mathfrak{A}} \equiv \mathfrak{A}/\mathfrak{B}$ of cosets $a + \mathfrak{B}$ forms a commutative group under addition:
$$(a + \mathfrak{B}) + (c + \mathfrak{B}) = (a + c) + \mathfrak{B}.$$
Multiplication of cosets is defined by
$$(a + \mathfrak{B})(c + \mathfrak{B}) = ac + \mathfrak{B},$$
which is well-defined because $\mathfrak{B}$ is an ideal. The system $\mathfrak{A}/\mathfrak{B}, +, \cdot$ is called the difference ring (quotient ring, residue class ring) of $\mathfrak{A}$ relative to the ideal $\mathfrak{B}$.
