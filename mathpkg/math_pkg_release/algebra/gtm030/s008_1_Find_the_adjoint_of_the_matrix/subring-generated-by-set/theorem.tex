The intersection of any collection of subrings of a ring is a subring. If $S$ is any subset of the ring $\mathfrak{A}$, the intersection of the subrings containing $S$ is called the subring generated by $S$, denoted by $[[S]]$.

$[[S]]$ is characterized by:
(1) $[[S]]$ is a subring;
(2) $[[S]] \supseteq S$;
(3) if $\mathfrak{B}$ is any subring containing $S$, then $\mathfrak{B} \supseteq [[S]]$.
