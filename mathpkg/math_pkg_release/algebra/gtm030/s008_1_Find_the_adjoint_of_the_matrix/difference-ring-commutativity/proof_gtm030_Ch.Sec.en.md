If $\mathfrak{A}$ is commutative, then for any cosets:
$$(a + \mathfrak{B})(c + \mathfrak{B}) = ac + \mathfrak{B} = ca + \mathfrak{B} = (c + \mathfrak{B})(a + \mathfrak{B}).$$
Thus $\mathfrak{A}/\mathfrak{B}$ is commutative.
