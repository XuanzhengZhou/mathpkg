For any element $a$ in the ring, we have:
$$a0 = a(0 + 0) = a0 + a0.$$

Now add $-a0$ to both sides of this equation:
$$a0 + (-a0) = (a0 + a0) + (-a0).$$

The left side is $0$ by definition of the additive inverse. The right side simplifies using associativity:
$$0 = a0 + (a0 + (-a0)) = a0 + 0 = a0.$$

Thus $a0 = 0$. The proof that $0a = 0$ follows by an entirely symmetric argument, using the right distributive law:
$$0a = (0 + 0)a = 0a + 0a,$$
and adding $-0a$ to both sides yields $0a = 0$.
