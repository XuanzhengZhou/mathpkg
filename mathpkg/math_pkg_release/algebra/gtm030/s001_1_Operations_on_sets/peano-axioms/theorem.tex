The set $P$ of natural numbers satisfies the following axioms:
\begin{enumerate}
\item $P$ is not vacuous.
\item There exists a 1-1 mapping $a \mapsto a^+$ of $P$ into itself. ($a^+$ is the immediate successor of $a$.)
\item The set of images under the successor mapping is a proper subset of $P$.
\item Any subset of $P$ that contains an element that is not a successor and that contains the successor of every element in the set coincides with $P$. This is called the axiom of induction.
\end{enumerate}
All the properties of $P$ are consequences of these axioms. The unique non-successor is denoted $1$. Also $1^+ = 2$, $2^+ = 3$, etc.
