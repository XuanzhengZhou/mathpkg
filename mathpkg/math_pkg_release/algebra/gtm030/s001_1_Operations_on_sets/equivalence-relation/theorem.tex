A relation $\sim$ on a set $S$ is called an equivalence relation if it satisfies the following conditions:
\begin{enumerate}
\item $a \sim a$ (reflexive property).
\item $a \sim b$ implies $b \sim a$ (symmetric property).
\item $a \sim b$ and $b \sim c$ imply that $a \sim c$ (transitive property).
\end{enumerate}
