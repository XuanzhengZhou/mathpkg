Let $\sim$ be an equivalence relation on $S$ with quotient set $\bar{S}$. Let $\alpha : S \to T$ be a mapping with the property that if $a \sim b$, then $a\alpha = b\alpha$ (i.e., $\alpha$ is constant on each equivalence class). Then the rule $\bar{a} \mapsto a\alpha$ defines a mapping $\bar{\alpha} : \bar{S} \to T$, called the induced mapping, satisfying $\bar{a}\bar{\alpha} = a\alpha$, so that $\alpha = \nu \bar{\alpha}$ where $\nu$ is the natural mapping.
