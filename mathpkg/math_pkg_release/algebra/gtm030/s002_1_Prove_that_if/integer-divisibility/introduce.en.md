Divisibility is a fundamental relation on the set of integers. An integer $b$ divides $a$, written $b \mid a$, if there exists an integer $c$ such that $a = bc$. This relation is transitive: if $b \mid a$ and $a \mid c$, then $b \mid c$. When two integers divide each other and are non-zero, they must differ only by a sign: $a = \pm b$. The divisibility relation provides the framework for defining prime numbers, greatest common divisors, and the fundamental theorem of arithmetic.
