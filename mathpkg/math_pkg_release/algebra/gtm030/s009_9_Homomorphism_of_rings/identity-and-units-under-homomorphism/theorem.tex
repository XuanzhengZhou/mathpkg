Let $$\eta: \mathfrak{A} \to \mathfrak{A}'$$ be a ring homomorphism.

\begin{enumerate}
\item If $$\mathfrak{A}$$ has an identity $$1$$, then $$1' = 1\eta$$ is an identity for $$\mathfrak{A}\eta$$.
\item If $$u \in \mathfrak{A}$$ is a unit with inverse $$v$$, then $$u' = u\eta$$ is a unit in $$\mathfrak{A}\eta$$ with inverse $$v' = v\eta$$.
\item If $$1\eta = 0$$, then $$\mathfrak{A}\eta = 0$$.
\item If $$\mathfrak{A}$$ is a division ring, then either $$\mathfrak{A}\eta = 0$$ or $$\mathfrak{A}\eta$$ is also a division ring.
\end{enumerate}
