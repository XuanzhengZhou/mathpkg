The product set generalizes the Cartesian product to an arbitrary index set $J$. Each element is a function $\alpha \mapsto g_{\alpha}$ from the index set to the disjoint union of the groups, with the constraint that the value at index $\alpha$ lies in $\mathfrak{G}_{\alpha}$. When $J = \{1, 2, \dots\}$ is the set of positive integers, the product set consists of all sequences $(g_1, g_2, \dots)$ with $g_i \in \mathfrak{G}_i$. The product set serves as the underlying set for both the complete direct product and the restricted direct product.
