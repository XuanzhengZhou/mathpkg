This proposition states that adjoining several commuting elements to a ring can be performed step by step: the subring generated by $A$ and $u_1, \ldots, u_r$ is the same as first adjoining $u_1$ to $A$, then adjoining $u_2$ to the result, and so on. The proof uses induction on the number of elements and the basic property that $A[u_1, \ldots, u_s]$ contains $A[u_1, \ldots, u_{s-1}]$ together with $u_s$, and conversely $A[u_1, \ldots, u_{s-1}][u_s]$ contains all the $u_i$ and $A$. This reduction is essential for extending results about polynomial rings in one variable to the several-variable case.
