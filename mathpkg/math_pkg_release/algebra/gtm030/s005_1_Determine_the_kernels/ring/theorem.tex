A \text{ring} is a system consisting of a set $\mathfrak{A}$ and two binary compositions in $\mathfrak{A}$ called addition and multiplication such that

\begin{enumerate}
\item $\mathfrak{A}$ together with addition $(+)$ is a commutative group.
\item $\mathfrak{A}$ together with multiplication $(\cdot)$ is a semi-group.
\item The distributive laws hold:
  $$a(b + c) = ab + ac$$
  $$(b + c)a = ba + ca$$
\end{enumerate}

Thus the assumptions are that $a + b$ and $ab \in \mathfrak{A}$ and satisfy:

\begin{itemize}
\item[A1] $(a + b) + c = a + (b + c)$.
\item[A2] $a + b = b + a$.
\item[A3] There is an element $0$ such that $a + 0 = a = 0 + a$.
\item[A4] For each $a$ there is a negative $-a$ such that $a + (-a) = 0 = -a + a$.
\item[M] $(ab)c = a(bc)$.
\end{itemize}

The system $\mathfrak{A}, +$ is called the \text{additive group} and the system $\mathfrak{A}, \cdot$ is called the \text{multiplicative semi-group} of the ring.
