Let $\mathfrak{S}$ be an arbitrary commutative semi-group that has an identity $1$ and that satisfies the cancellation law. If $a, b \in \mathfrak{S}$, we say that $b$ is a **factor** or **divisor** of $a$ if there exists an element $c \in \mathfrak{S}$ such that $a = bc$. We write $b \mid a$. This relation is transitive and reflexive. An element $u$ is a unit if and only if $u \mid 1$. The units are the trivial factors since they are factors of every element of $\mathfrak{S}$.
