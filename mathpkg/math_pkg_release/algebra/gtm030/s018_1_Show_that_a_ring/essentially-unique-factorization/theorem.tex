A factorization $a = p_1 p_2 \cdots p_s$ of $a$ into irreducible elements is **essentially unique** if for any other factorization $a = p_1' p_2' \cdots p_t'$ with $p_i'$ irreducible, we have $t = s$ and $p_i' \sim p_i$ for a suitable arrangement of the factors.

If $\mathfrak{S}$ has units $\neq 1$ and $s > 1$, then given a factorization $a = p_1 p_2 \cdots p_s$, one obtains other factorizations $a = p_1' p_2' \cdots p_s'$ by setting $p_i' = u_i p_i$ where the $u_i$ are units such that $u_1 u_2 \cdots u_s = 1$. These factorizations are regarded as essentially the same as the original.
