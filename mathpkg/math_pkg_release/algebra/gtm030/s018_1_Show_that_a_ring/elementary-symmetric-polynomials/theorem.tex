Let $x_1, x_2, \cdots, x_r$ be algebraically independent over a commutative ring $\mathfrak{A}$ with identity. The \text{elementary symmetric polynomials} $p_1, p_2, \cdots, p_r$ are defined by

$$F(x) = \prod_{i=1}^r (x - x_i) = x^r - p_1 x^{r-1} + p_2 x^{r-2} - \cdots + (-1)^r p_r,$$

where

$$p_1 = \sum_i x_i, \quad p_2 = \sum_{i < j} x_i x_j, \quad p_3 = \sum_{i < j < k} x_i x_j x_k, \quad \cdots, \quad p_r = x_1 x_2 \cdots x_r.$$
