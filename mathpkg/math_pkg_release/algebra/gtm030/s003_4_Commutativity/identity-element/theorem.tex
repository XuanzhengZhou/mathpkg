An element $e$ of a semigroup $\mathfrak{S}$ is called a \text{left identity} (unit, unity) if $ea = a$ for every $a$ in $\mathfrak{S}$. Similarly, $f$ is a \text{right identity} if $af = a$ for every $a$ in $\mathfrak{S}$.

If an element is both a left identity and a right identity, it is called a \text{two-sided identity} (or simply an identity).
