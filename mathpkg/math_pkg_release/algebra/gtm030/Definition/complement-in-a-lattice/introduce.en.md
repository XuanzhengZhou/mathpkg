In a bounded lattice, a complement of an element $a$ is an element $a'$ whose join with $a$ is the top element $1$ and whose meet with $a$ is the bottom element $0$. A relative complement of $b$ with respect to $a$ (where $b \leq a$) is an element $b_1 \leq a$ such that $b \cup b_1 = a$ and $b \cap b_1 = 0$. In a complemented modular lattice, relative complements always exist.
