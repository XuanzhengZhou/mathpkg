Show that any Boolean algebra defines a ring relative to the two compositions
$$a \oplus b = (a \cup b') \cap (a' \cup b), \quad a \odot b = a \cup b.$$
Show that $a \oplus b = 1 + a + b$ and $a \odot b = a + b + ab$ where $+$ and $\cdot$ are as defined in the text.
