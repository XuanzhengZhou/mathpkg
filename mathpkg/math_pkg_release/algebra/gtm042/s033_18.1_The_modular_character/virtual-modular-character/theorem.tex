Property (iii) (additivity on exact sequences) allows one to extend the definition of the modular character to the Grothendieck group $R_k(G)$. For any element $x \in R_k(G)$, one defines its \textbf{virtual modular character} $\phi_x: G_{\text{reg}} \to A$ by linear extension.

If $x = d(y)$ with $y \in R_K(G)$ (where $d: R_K(G) \to R_k(G)$ is the decomposition map), then $\phi_x$ is the restriction to $G_{\text{reg}}$ of the virtual ordinary character $\chi_y$ of $y$.
