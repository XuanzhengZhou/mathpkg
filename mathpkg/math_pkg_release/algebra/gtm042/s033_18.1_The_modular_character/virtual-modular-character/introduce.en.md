The additivity property of Brauer characters on exact sequences allows extending the modular character map from $k[G]$-modules to the Grothendieck group $R_k(G)$, yielding virtual modular characters. For an element $x = d(y)$ coming from $R_K(G)$ via the decomposition map, the virtual modular character $\phi_x$ is simply the restriction to $G_{\text{reg}}$ of the ordinary virtual character $\chi_y$. This establishes the fundamental relationship between ordinary and modular character rings.
