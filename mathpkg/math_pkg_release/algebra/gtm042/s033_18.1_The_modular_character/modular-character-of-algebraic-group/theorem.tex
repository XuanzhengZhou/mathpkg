Let $G$ be a linear algebraic group over an algebraically closed field $k$ of characteristic $p$. Define $G_{\text{reg}}$ as the set of semisimple elements of $G$.

If $E$ is a linear representation of $G$ and $s \in G_{\text{reg}}$, then $\phi_E(s)$ is defined as the sum of the multiplicative representatives of the eigenvalues of $s_E$. The \textbf{modular character} $\phi_E$ thus defined is a class function on $G_{\text{reg}}$ with values in $A$.
