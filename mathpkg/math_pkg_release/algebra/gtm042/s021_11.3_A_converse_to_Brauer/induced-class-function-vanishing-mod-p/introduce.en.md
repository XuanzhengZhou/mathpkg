Lemma 11 is a crucial technical result in the converse direction of Brauer's theorem on induced characters. It states that if a subgroup $H$ of $G$ contains no conjugate of the product $C \times P$ — where $C$ is generated by a $p'$-element $x$ and $P$ is a Sylow $p$-subgroup of the centralizer $Z(x)$ — then any class function $\psi$ on $H$, when induced to $G$, yields a class function $\psi'$ whose value at $x$ is divisible by $p$. This lemma provides the key vanishing condition needed to establish that the elementary subgroups form the smallest family for which Brauer's induction theorem holds.
