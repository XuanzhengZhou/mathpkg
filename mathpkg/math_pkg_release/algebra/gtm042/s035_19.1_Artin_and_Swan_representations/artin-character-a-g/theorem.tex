Let $E$ be a field complete with respect to a discrete valuation, let $F/E$ be a finite Galois extension with Galois group $G$, and assume $E$ and $F$ have the same residue field. Let $i_G$ be the ramification function on $G$. Define the Artin character $a_G$ by

$$a_G(s) = -i_G(s) \quad \text{if } s \neq 1,$$

$$a_G(1) = \sum_{s \neq 1} i_G(s).$$

The function $a_G$ is a class function on $G$ with integer values.
