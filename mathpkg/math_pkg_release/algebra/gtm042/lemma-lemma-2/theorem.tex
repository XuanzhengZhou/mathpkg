Remarks

(1) Theorem 13 expresses the fact that the maps Res and Ind are adjoints of each other.

(2) Instead of the bilinear form $\langle \alpha, \beta \rangle$, we can use the scalar product $(\alpha | \beta)$ defined in 2.3. We have the same formula:

$$(\psi | \text{Res } \varphi)_{\text{H}} = (\text{Ind } \psi | \varphi)_{\text{G}}.$$

(3) We mention also the following useful formula

$$\text{Ind}(\psi \cdot \text{Res } \varphi) = (\text{Ind } \psi) \cdot \varphi.$$

It can be checked by a simple calculation, or deduced from the formula $\text{Ind}(\text{W}) \otimes \text{E} \cong \text{Ind}(\text{W} \otimes \text{Res E})$, cf. 3.3, example 5.
