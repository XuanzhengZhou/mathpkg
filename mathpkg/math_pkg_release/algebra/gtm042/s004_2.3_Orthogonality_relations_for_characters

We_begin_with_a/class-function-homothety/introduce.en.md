Proposition 6 is a technical lemma that bridges class functions and irreducible representations. It shows that the operator obtained by averaging a class function over the group action on an irreducible representation is always a scalar multiple of the identity. This is a refinement of Schur's lemma: the intertwining condition is replaced by the class function property. The result is essential for proving that the irreducible characters span the space of class functions.
