If $\mathfrak{a} = Ae$ with $e^2 = e$, then $A = Ae \oplus A(1-e)$, so $\mathfrak{a}$ is a direct factor of a free module and thus projective. Conversely, if $\mathfrak{a}$ is a direct summand of $_A A$, then there exists a projection $\pi: A \to \mathfrak{a}$ which is a left $A$-module homomorphism. Let $e = \pi(1)$. Then for any $a \in A$, $\pi(a) = a\pi(1) = ae$, so $\mathfrak{a} = Ae$. Moreover, $e = \pi(1) = \pi(e) = e\pi(1) = e^2$, so $e$ is idempotent.
