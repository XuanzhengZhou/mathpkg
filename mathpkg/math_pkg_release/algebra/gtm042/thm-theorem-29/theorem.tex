Let $f$ be a class function on $G$ with values in $Q(m)$.

(a) In order that $f$ belong to $Q \otimes R(G)$, it is necessary and sufficient that $\sigma_t(f) = \Psi^t(f)$ for each $t$ prime to $m$.

(b) In order that $f$ belong to $Q \otimes R_Q(G)$, it is necessary and sufficient that $f$ have values in $Q$, and that $\Psi^t(f) = f$ for each $t$ prime to $m$

(i.e., we must have $f(x) = f(y)$ if $x$ and $y$ generate the same subgroup of $G$).

(Recall that $\sigma_t$ is the automorphism of $Q(m)$ which takes an $m$th root of unity to its $t$th power, and that $\Psi^t(f)$ is the function $x \mapsto f(x^t)$.)
