If $C$ is a category with small hom-sets, the hom-functor defines a bifunctor
$$\hom_C: C^{\mathrm{op}} \times C \to \mathbf{Set},$$
contravariant in the first argument and covariant in the second. The functors
$$\hom_C(-, c): C^{\mathrm{op}} \to \mathbf{Set}, \qquad \hom_C(b, -): C \to \mathbf{Set}$$
satisfy the interchange condition necessary to make $\hom_C$ a bifunctor.
