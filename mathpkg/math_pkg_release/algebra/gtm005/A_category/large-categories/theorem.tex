To construct large categories, assume there is a sufficiently large set $U$, called a \emph{universe}. A set $x$ is called \emph{small} if it is a member of the universe $U$. Then:
\begin{itemize}
\item $\mathbf{Set}$: the category whose objects are all small sets $U$, with arrows all functions from one small set to another.
\item $\mathbf{Set}_*$: pointed sets; objects are small sets each with a selected base point, arrows are base-point-preserving functions.
\item $\mathbf{Ens}$: category of all sets and functions within a (variable) set $V$.
\item $\mathbf{Cat}$: the category of all small categories, with arrows all functors between them.
\end{itemize}
