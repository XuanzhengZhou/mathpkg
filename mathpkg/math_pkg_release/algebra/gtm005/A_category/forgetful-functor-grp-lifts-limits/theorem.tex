Let $U : \mathbf{Grp} \rightarrow \mathbf{Set}$ be the forgetful functor. If $H : J \rightarrow \mathbf{Grp}$ is such that the composite $UH$ has a limit $L$ and a limiting cone $v : L \rightarrow UH$ in $\mathbf{Set}$, then there is exactly one group structure on the set $L$ for which each arrow $v_j : L \rightarrow UH_j$ of the cone $v$ is a morphism of groups; moreover, this group $L$ is a limit of $H$ with $v$ as limiting cone.
