In $\mathbf{Top}$, take each object $F_n$ to be a circle $S^1$, and each arrow $f_n : S^1 \rightarrow S^1$ to be the continuous map wrapping the domain circle $S^1$ uniformly $p$ times around the codomain circle. The inverse limit set $L$ then becomes a topological space when we introduce just those open sets in $L$ necessary to make all the functions $\mu_n : L \rightarrow S^1$ continuous. This $L$ is the limit space in $\mathbf{Top}$; it is known as the $p$-adic solenoid.
