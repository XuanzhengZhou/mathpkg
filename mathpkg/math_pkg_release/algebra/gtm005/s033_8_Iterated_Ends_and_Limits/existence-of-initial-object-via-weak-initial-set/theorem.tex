Let $D$ be a small-complete category, and let $S$ be a small full subcategory of $D$ which is weakly initial in $D$ (i.e., for every object $d \in D$, there exists an object $s \in S$ and an arrow $s \to d$). Then the limit of the inclusion functor $F: S \hookrightarrow D$ exists and is an initial object of $D$.
