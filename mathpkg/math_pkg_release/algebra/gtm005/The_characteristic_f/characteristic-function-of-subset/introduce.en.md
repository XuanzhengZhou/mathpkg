The characteristic function of a subset $S \subset X$ assigns 0 to elements in $S$ and 1 to elements outside $S$, sending each element of $X$ to the two-element set $\{0,1\}$. This construction encodes arbitrary subsets into the simplest non-trivial subset $\{0\} \subset \{0,1\}$ via a pullback square. Characteristic functions are widely used in probability theory as indicator functions and in logic where $\{0,1\}$ is interpreted as the set of truth values with 0 representing truth.
