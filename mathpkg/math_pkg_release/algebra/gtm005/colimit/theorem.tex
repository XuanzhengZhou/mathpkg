Let $C$ and $J$ be categories and $\Delta: C \to C^J$ the \textbf{diagonal functor} sending each object $c$ to the constant functor $\Delta c$ (value $c$ on all objects of $J$, $1_c$ on all arrows). For a functor $F: J \to C$, a \textbf{colimit} of $F$ is a universal arrow $\langle r, u \rangle$ from $F$ to $\Delta$. It consists of an object $r = \operatorname{Colim} F$ (or $\varinjlim F$) and a natural transformation $u: F \to \Delta r$ (a \textbf{cocone}) which is universal among natural transformations $F \to \Delta c$. There is a natural bijection
$$C(\operatorname{Colim} F, c) \cong \operatorname{Nat}(F, \Delta c) = \operatorname{Cone}(F, c).$$
