A \emph{subcategory} $S$ of a category $C$ is a collection of some of the objects and some of the arrows of $C$, which includes
\begin{itemize}
\item with each arrow $f$, both the object $\dom f$ and the object $\cod f$;
\item with each object $s$, its identity arrow $1_s$;
\item with each pair of composable arrows $s \rightarrow s' \rightarrow s''$, their composite,
\end{itemize}
such that these collections of objects and arrows themselves constitute a category $S$.
