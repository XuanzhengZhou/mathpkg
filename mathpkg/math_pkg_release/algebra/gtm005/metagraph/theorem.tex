A \emph{metagraph} consists of objects $a, b, c, \ldots$, arrows $f, g, h, \ldots$, and two operations:

\begin{enumerate}
\item \textbf{Domain}, which assigns to each arrow $f$ an object $a = \text{dom} f$;
\item \textbf{Codomain}, which assigns to each arrow $f$ an object $b = \text{cod} f$.
\end{enumerate}

These operations on $f$ are indicated by displaying $f$ as an actual arrow starting at its domain (or ``source'') and ending at its codomain (or ``target''):
$$f: a \rightarrow b \quad \text{or} \quad a \xrightarrow{f} b.$$
