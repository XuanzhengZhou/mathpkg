Let $O$ be a set of objects. An $O$-graph $A$ is a set $A$ (of arrows) equipped with two functions
$$\partial_0, \partial_1 : A \to O,$$
where $\partial_0 f = \text{domain } f$ and $\partial_1 f = \text{codomain } f$.

A morphism $D : A \to B$ of $O$-graphs is a function $D : A \to B$ such that $\partial_0 \circ D = \partial_0$ and $\partial_1 \circ D = \partial_1$ (i.e., $D$ preserves domains and codomains).
