A functor $K: A \to C$ is dense if and only if the nerve-like functor $C \to [A^{\mathrm{op}}, \mathbf{Set}]$ sending $c$ to $\hom_C(K-, c)$ is fully faithful. Dually, $K$ is codense if and only if the functor $C^{\mathrm{op}} \to [A, \mathbf{Set}]$ sending $c$ to $\hom_C(c, K-)$ is fully faithful.
