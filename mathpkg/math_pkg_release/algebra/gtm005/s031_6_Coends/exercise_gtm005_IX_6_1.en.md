For $S : C^{\text{op}} \times C \rightarrow \text{Set}$, prove that the set $\text{Wedge}(*, S)$ of all wedges $\omega : * \rightarrow S$ from the one-point set $*$ to $S$ is an end of $S$, with ending wedge given by $\omega \mapsto \omega_c(*) \in S(c, c)$. Compare with the explicit description of a limit in $\text{Set}$ as a set of cones.
