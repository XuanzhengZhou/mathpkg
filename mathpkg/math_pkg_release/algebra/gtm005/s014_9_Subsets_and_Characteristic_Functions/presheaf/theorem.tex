A \emph{presheaf} on a small category $C$ is a contravariant functor $F \colon C^{\mathrm{op}} \to \mathbf{Sets}$. In other words, $F$ assigns to each object $x$ of $C$ a set $F(x)$ and to each morphism $f \colon x \to y$ in $C$ a function $F(f) \colon F(y) \to F(x)$, satisfying $F(\mathrm{id}_x) = \mathrm{id}_{F(x)}$ and $F(g \circ f) = F(f) \circ F(g)$. The category $\mathbf{Sets}^{C^{\mathrm{op}}}$ of all presheaves on $C$ (with natural transformations as morphisms) is an elementary topos.

In topological contexts, $C$ is typically the partially ordered set of open sets of a topological space $X$; a presheaf $F$ then assigns to each open set $U$ a set $F(U)$. For example, $F(U)$ may be the set of all continuous real-valued functions on $U$. When $F$ satisfies additional gluing conditions it is called a \emph{sheaf}.
