A natural numbers object (NNO) in a category is an object $N$ equipped with a zero element and a successor operation, satisfying the universal property of primitive recursion. This categorical formulation captures the essence of the natural numbers without relying on set-theoretic encodings: any function on $N$ can be uniquely defined by specifying its value at zero and how it commutes with the successor. In a topos used as a foundation for mathematics, the NNO provides the arithmetic backbone needed to develop number theory and analysis.
