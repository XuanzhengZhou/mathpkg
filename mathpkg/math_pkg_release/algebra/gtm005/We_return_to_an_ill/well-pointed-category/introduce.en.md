A well-pointed category is one in which morphisms are determined by their action on global elements (morphisms from the terminal object $1$). This property captures the set-theoretic intuition that two functions are equal if and only if they agree on every element of the domain. In the context of topos theory, well-pointedness is a desirable property when using a topos as a foundation for mathematics, as it ensures the category behaves like the category of sets where equality of maps can be checked pointwise.
