For a category $C$ with small hom-sets and each object $a \in C$, the \textbf{covariant hom-functor}

$$
C(a, -) = \hom(a, -): C \to \mathbf{Set}
$$

has object function sending each object $b$ to the set $\hom(a, b)$, and arrow function sending each arrow $k: b \to b'$ to the function

$$
\hom(a, k): \hom(a, b) \to \hom(a, b')
$$

defined by the assignment $f \mapsto k \circ f$ for each $f: a \to b$. This function is sometimes written $k_*$ and called ``composition with $k$ on the left.''
