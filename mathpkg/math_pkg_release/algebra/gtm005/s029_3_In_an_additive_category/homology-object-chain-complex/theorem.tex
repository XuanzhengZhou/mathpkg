For a chain complex $C = \{c_n, \partial_n\}$ in an abelian category $A$, with $\partial_n \partial_{n+1} = 0$ for all $n$, the $n$-th homology object is defined as the quotient

$$H_n C = \operatorname{Ker} \left( \partial_n : c_n \rightarrow c_{n-1} \right) / \operatorname{Im} \left( \partial_{n+1} : c_{n+1} \rightarrow c_n \right).$$

This quotient is taken in the sense of the quotient subobject (cf. Exercise 3.6), i.e., $H_n C = \operatorname{coker}(\operatorname{Im} \partial_{n+1} \hookrightarrow \operatorname{Ker} \partial_n)$.
