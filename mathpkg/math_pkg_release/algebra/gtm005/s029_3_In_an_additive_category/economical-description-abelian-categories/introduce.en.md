Abelian categories admit an economical axiomatic description that avoids prescribing an abelian group structure on each hom-set in advance. Instead, the axioms only require a null object, binary products and coproducts, kernels and cokernels, and the condition that every monic is a kernel and every epi is a cokernel. From these purely categorical conditions, the addition of arrows can be defined via the formula derived from the additive category construction, endowing each hom-set with an abelian group structure and yielding a full abelian category.
