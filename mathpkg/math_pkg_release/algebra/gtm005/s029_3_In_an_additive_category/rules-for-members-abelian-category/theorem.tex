In any abelian category $A$, the following rules hold for members:

\begin{enumerate}
\item[(i)] $f : a \rightarrow b$ is monic if and only if, for all $x \in_m a$, $f x \equiv 0$ implies $x \equiv 0$.
\item[(ii)] $f : a \rightarrow b$ is monic if and only if, for all $x, y \in_m a$, $f x \equiv f y$ implies $x \equiv y$.
\item[(iii)] $g : b \rightarrow c$ is epi if and only if, for all $z \in_m c$, there exists $y \in_m b$ with $g y \equiv z$.
\item[(iv)] $h : r \rightarrow s$ is the zero arrow if and only if, for all $x \in_m r$, $h x \equiv 0$.
\item[(v)] A sequence $a \xrightarrow{f} b \xrightarrow{g} c$ is exact at $b$ if and only if $g f = 0$ and to every $y \in_m b$ with $g y \equiv 0$ there exists $x \in_m a$ with $f x \equiv y$.
\item[(vi)] (Subtraction) Given $g : b \rightarrow c$ and $x, y \in_m b$ with $g x \equiv g y$, there exists a member $z \in_m b$ with $g z \equiv 0$; moreover, any $f : b \rightarrow d$ with $f x \equiv 0$ has $f y \equiv f z$ and any $h : b \rightarrow a$ with $h y \equiv 0$ has $h x \equiv -h z$.
\end{enumerate}
