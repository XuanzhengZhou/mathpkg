Given a monad $\langle T, \eta, \mu \rangle$ in a category $X$, consider to each object $x \in X$ a new object $x_T$ and to each arrow $f : x \to Ty$ in $X$ a new arrow $f^\flat : x_T \to y_T$. These new objects and arrows constitute a category $X_T$ when the composite of $f^\flat : x_T \to y_T$ with $g^\flat : y_T \to z_T$ is defined by

$$g^\flat \circ f^\flat = (\mu_z \circ Tg \circ f)^\flat : x_T \to z_T.$$

The identity arrow $1_{x_T}$ is $(\eta_x)^\flat : x_T \to x_T$.

Define functors $F_T : X \to X_T$ and $G_T : X_T \to X$ by

$$F_T x = x_T, \qquad F_T(f : x \to y) = (\eta_y \circ f)^\flat,$$
$$G_T(x_T) = Tx, \qquad G_T(f^\flat : x_T \to y_T) = \mu_y \circ Tf.$$

Then $\langle F_T, G_T \rangle$ is an adjunction with $F_T \dashv G_T$, whose unit is $\eta$, counit is $\varepsilon_T$ given by $(\varepsilon_T)_{y_T} = (1_{Ty})^\flat : (Ty)_T \to y_T$, and the monad defined by this adjunction in $X$ is exactly the original monad $\langle T, \eta, \mu \rangle$.
