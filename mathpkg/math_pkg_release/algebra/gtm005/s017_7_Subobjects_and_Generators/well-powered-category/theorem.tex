\text{A category is called \textbf{well-powered} when each of its objects has only a set of subobjects (i.e., the collection of equivalence classes of monics with codomain $a$ forms a set, not a proper class, for each object $a$).}

\text{Equivalently, for every object $a$, there is only a set of isomorphism classes of monics with codomain $a$.}
