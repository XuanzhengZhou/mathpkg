A cogenerating set (or cogenerator) is the dual notion to a generating set. Instead of distinguishing arrows via morphisms into the tested object, a cogenerating set distinguishes arrows via morphisms out of the tested object. Given two distinct parallel arrows $h, h': a \to b$, there must be an object $q$ in the cogenerating set and a morphism $g: b \to q$ such that $gh \neq gh'$. The existence of a small cogenerating set is a crucial hypothesis of the Special Adjoint Functor Theorem. Classic examples include the two-point set in Set and the unit interval in compact Hausdorff spaces.
