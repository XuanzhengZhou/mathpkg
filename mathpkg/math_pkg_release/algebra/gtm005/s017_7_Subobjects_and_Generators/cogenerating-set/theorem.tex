\text{Dually, a set $Q$ of objects is a \textbf{cogenerating set} for the category $C$ when to every parallel pair $h \neq h': a \to b$ of arrows of $C$ there is an object $q \in Q$ and an arrow $g: b \to q$ with $gh \neq gh'$.}

\text{A single object $q$ is a \textbf{cogenerator} when $\{q\}$ is a cogenerating set.}

\text{Example: any two-point set is a cogenerator in $\mathbf{Set}$. The unit interval $I = [0,1]$ is a cogenerator for $\mathbf{CompHaus}$ (by Urysohn's Lemma).}
