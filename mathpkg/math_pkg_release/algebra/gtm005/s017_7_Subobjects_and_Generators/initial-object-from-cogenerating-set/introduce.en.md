This theorem provides sufficient conditions for a complete category to possess an initial object. Given a small-complete category $D$ with small hom-sets, a small cogenerating set $Q$, and the property that every set of subobjects has an intersection (a pullback), one can construct an initial object as the intersection of all subobjects of the product of all objects in $Q$. The cogenerating condition ensures there is at most one arrow from the constructed object to any object, while the intersection property yields the existence of the required arrow. This result is a critical lemma in proving the Special Adjoint Functor Theorem.
