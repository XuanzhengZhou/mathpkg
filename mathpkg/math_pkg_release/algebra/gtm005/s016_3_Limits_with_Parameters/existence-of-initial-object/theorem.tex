Let $D$ be a small-complete category with small hom-sets. Then $D$ has an initial object if and only if it satisfies the following

\textbf{Solution Set Condition.} There exists a small set $I$ and an $I$-indexed family $k_i$ of objects of $D$ such that for every object $d \in D$ there exists an index $i \in I$ and an arrow $k_i \rightarrow d$ in $D$.
