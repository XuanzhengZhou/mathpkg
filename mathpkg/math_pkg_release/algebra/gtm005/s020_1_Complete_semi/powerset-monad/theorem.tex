Let $\mathcal{P}$ be the covariant power set functor on $\mathbf{Set}$. For each set $X$, let $\eta_X : X \to \mathcal{P}X$ send each $x \in X$ to the one-point set $\{x\}$, while $\mu_X : \mathcal{P}\mathcal{P}X \to \mathcal{P}X$ sends each set of sets into its union. Then $\langle \mathcal{P}, \eta, \mu \rangle$ is a monad $\mathcal{P}$ on $\mathbf{Set}$ (the power set monad, also called the Manes monad).
