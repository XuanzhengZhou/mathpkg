Every $\mathcal{P}$-algebra $(X, h)$ for the power set monad carries a natural complete semi-lattice structure. The structure map $h : \mathcal{P}X \to X$ is used to define a partial order: $x \leq y$ if and only if $h\{x, y\} = y$. For any subset $S \subset X$, the supremum is given by $\sup S = hS$. The algebra axioms for $h$ ensure that this indeed defines a complete semi-lattice. This is one half of the equivalence between $\mathcal{P}$-algebras and complete semi-lattices.
