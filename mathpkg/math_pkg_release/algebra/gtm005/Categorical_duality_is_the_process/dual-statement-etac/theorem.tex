For a statement $\Sigma$ of the elementary theory of an abstract category, the dual statement $\Sigma^{*}$ is obtained by simultaneously replacing each atomic constituent of $\Sigma$ as follows:

\begin{itemize}
\item ``$a = \operatorname{dom} f$'' is replaced by ``$a = \operatorname{cod} f$''
\item ``$b = \operatorname{cod} f$'' is replaced by ``$b = \operatorname{dom} f$''
\item ``$i = 1_a$'' remains ``$i = 1_a$'' (identity arrows are self-dual)
\item ``$h = g \circ f$'' is replaced by ``$h = f \circ g$'' (composition order reversed)
\end{itemize}

If $\Sigma$ involves a diagram, the dual statement $\Sigma^{*}$ involves that diagram with all arrows reversed. Note that the dual of the dual is the original statement: $\Sigma^{**} = \Sigma$.
