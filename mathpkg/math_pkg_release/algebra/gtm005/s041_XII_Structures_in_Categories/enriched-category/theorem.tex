Given a closed monoidal category $(V, \otimes, I, [-, -])$, a \textbf{$V$-category} (or \textbf{enriched category}) $C$ consists of:
\begin{itemize}
\item A set of objects $\operatorname{Ob}C$;
\item For each $a, b \in C$, a hom-object $C(a, b) \in V$;
\item Composition morphisms $C(b, c) \otimes C(a, b) \rightarrow C(a, c)$ in $V$;
\item Identity morphisms $I \rightarrow C(a, a)$ in $V$;
\end{itemize}
satisfying associativity and unit axioms expressed as commuting diagrams in $V$.
