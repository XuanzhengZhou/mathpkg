A functor $T: M \to A$ has a colimit if and only if it has a left Kan extension along the (unique) functor $K_1: M \to \mathbf{1}$, and then $\operatorname{Colim} T$ is the value of $\operatorname{Lan}_{K_1} T$ on the unique object of $\mathbf{1}$.

Dually, a functor $T: M \to A$ has a limit if and only if it has a right Kan extension along $K_1: M \to \mathbf{1}$, and then $\operatorname{Lim} T$ is the value of $\operatorname{Ran}_{K_1} T$ on the unique object of $\mathbf{1}$.
