A \textbf{metagraph} consists of objects $a, b, c, \ldots$, arrows $f, g, h, \ldots$, and two operations:
\begin{itemize}
\item \textbf{Domain}, which assigns to each arrow $f$ an object $a = \operatorname{dom} f$;
\item \textbf{Codomain}, which assigns to each arrow $f$ an object $b = \operatorname{cod} f$.
\end{itemize}
These operations on $f$ are indicated by displaying $f$ as an actual arrow starting at its domain and ending at its codomain:
$$f: a \rightarrow b \quad \text{or} \quad a \xrightarrow{f} b.$$
