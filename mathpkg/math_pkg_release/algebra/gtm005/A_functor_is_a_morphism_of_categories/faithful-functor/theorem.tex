A functor $T: C \rightarrow B$ is \textbf{faithful} (or an \textbf{embedding}) when to every pair $c, c'$ of objects of $C$ and to every pair $f_1, f_2: c \rightarrow c'$ of parallel arrows of $C$, the equality $Tf_1 = Tf_2: Tc \rightarrow Tc'$ implies $f_1 = f_2$.

Equivalently, via the function on hom-sets
$$T_{c, c'}: \hom(c, c') \rightarrow \hom(Tc, Tc'), \quad f \mapsto Tf,$$
$T$ is faithful when every such function is injective.
