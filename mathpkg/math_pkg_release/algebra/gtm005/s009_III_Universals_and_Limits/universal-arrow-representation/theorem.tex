\textbf{Proposition 1.} For a functor $S: D \to C$, a pair $\langle r, u: c \to Sr \rangle$ is universal from $c$ to $S$ if and only if the function sending each $f': r \to d$ into $Sf' \circ u: c \to Sd$ is a bijection of hom-sets

\[
D(r, d) \cong C(c, Sd).
\]

This bijection is natural in $d$. Conversely, each natural isomorphism $D(r, -) \cong C(c, S -)$ is determined in this way by a unique arrow $u: c \to Sr$.
