A generating set (or generator) in a category $\mathcal{C}$ is a set of objects that collectively "separates" parallel morphisms: two distinct arrows $h, h' : c \to d$ can be distinguished by precomposition with some arrow emanating from an object in the generating set. A single generating object suffices in many categories: $\mathbb{Z}$ generates both $\mathbf{Ab}$ and $\mathbf{Grp}$, while the one-point set generates $\mathbf{Set}$. The concept is essential for the Adjoint Functor Theorem, where solution sets are constructed using generating objects, and is dual to the notion of a cogenerating set.
