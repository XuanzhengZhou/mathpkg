A cogenerating set (or cogenerator) is the dual concept to a generating set: rather than separating arrows from above via precomposition, a cogenerating set separates them from below via postcomposition. A single cogenerator object $q$ satisfies that for any parallel pair $h \neq h' : a \to b$, there exists an arrow $g : b \to q$ distinguishing them. Cogenerators are a crucial hypothesis of the Special Adjoint Functor Theorem; for example, the unit interval $I$ cogenerates the category of compact Hausdorff spaces (by Urysohn's Lemma), yielding the Stone-Čech compactification as a left adjoint.
