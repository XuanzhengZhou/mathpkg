This theorem provides sufficient conditions for the existence of an initial object purely in terms of subobjects and cogenerators. Given a small cogenerating set $Q$, one forms the product of all objects in $Q$ and then takes the intersection $r$ of all subobjects of this product. The cogenerating property ensures that for any object $d$, there is at most one arrow $r \to d$ (otherwise their equalizer would be a proper subobject smaller than the intersection). Existence of an arrow $r \to d$ is established via a pullback construction using the set of all arrows from $d$ to objects in $Q$. This result is a key lemma in the proof of the Special Adjoint Functor Theorem.
