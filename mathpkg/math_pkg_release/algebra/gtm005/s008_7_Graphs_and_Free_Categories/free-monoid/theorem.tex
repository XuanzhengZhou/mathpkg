The \textbf{free monoid} $FX$ generated by a set $X$ consists of all finite strings $x_1 x_2 \cdots x_n$ of elements $x_i \in X$. Multiplication is juxtaposition, and the empty string is the unit. For any monoid $M$, any function $f: X \rightarrow UM$ (where $UM$ is the underlying set) extends uniquely to a monoid morphism $g: FX \rightarrow M$.
