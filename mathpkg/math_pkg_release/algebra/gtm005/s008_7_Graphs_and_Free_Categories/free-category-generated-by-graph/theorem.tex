Let $G$ be a directed graph with object set $O$ and arrow set $A$. The \textbf{free category} $C = C_G$ generated by $G$ is:
\begin{itemize}
\item Objects: the vertices of $G$.
\item Arrows: finite \textbf{paths} (strings) of composable arrows in $G$,
$$\langle a_1, f_1, a_2, \ldots, f_{n-1}, a_n \rangle: a_1 \to a_n,$$
where each $f_i: a_i \to a_{i+1}$ is an arrow of $G$.
\item Composition: concatenation (juxtaposition) of paths.
\item Identities: paths of length 1, $\langle a \rangle$.
\end{itemize}
Every path decomposes uniquely into length-2 paths. There is a canonical morphism of graphs $P: G \to UC$ sending each arrow $f: a \to b$ to the path $\langle a, f, b \rangle$.
