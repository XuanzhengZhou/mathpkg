For a category $C$, let $R$ be a function assigning to each pair of objects $a, b$ a binary relation $R_{a,b}$ on $\hom_C(a,b)$. There exists a \textbf{quotient category} $C/R$ and a functor $Q: C \rightarrow C/R$, bijective on objects, such that:
\begin{enumerate}
\item If $f R f'$ then $Qf = Qf'$;
\item If $H: C \rightarrow D$ is any functor with $Hf = Hf'$ whenever $f R f'$, then $H = H' \circ Q$ for a unique functor $H': C/R \rightarrow D$.
\end{enumerate}
