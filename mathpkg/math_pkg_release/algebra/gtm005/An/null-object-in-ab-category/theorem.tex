The following properties of an object $z$ in an $\text{Ab}$-category $A$ are equivalent:

\begin{enumerate}
\item[(i)] $z$ is initial;
\item[(ii)] $z$ is terminal;
\item[(iii)] $1_z = 0 : z \rightarrow z$;
\item[(iv)] the abelian group $A(z,z)$ is the zero group.
\end{enumerate}

In particular, any initial (or any terminal) object in $A$ is a null object.
