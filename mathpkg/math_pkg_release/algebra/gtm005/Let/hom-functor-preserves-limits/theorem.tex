For any category $C$ with small hom-sets, each hom-functor $C(c, -): C \rightarrow \mathbf{Set}$ preserves all limits; in particular, all small limits.

More generally, if $C$ has hom-sets in a category $\mathbf{Ens}$ of sets, then each hom-functor $C(c, -): C \rightarrow \mathbf{Ens}$ preserves all limits which exist in $C$.

Equivalently, the contravariant hom-functor $C(-, c) = C^{\text{op}}(c, -): C^{\text{op}} \rightarrow \mathbf{Set}$ carries small colimits in $C$ to limits in $\mathbf{Set}$.
