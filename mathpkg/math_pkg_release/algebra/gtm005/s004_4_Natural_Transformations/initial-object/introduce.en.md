An initial object is the categorical dual of a terminal object. It is an object from which every object receives a unique arrow. In Set, the empty set is initial; in Grp, the trivial group is initial; in Rng, $\mathbb{Z}$ is initial. Initial objects are fundamental to the definition of universal arrows and adjunctions.
