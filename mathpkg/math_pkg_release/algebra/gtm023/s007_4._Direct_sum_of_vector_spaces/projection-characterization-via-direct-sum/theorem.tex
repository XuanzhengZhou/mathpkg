Let $\varphi: E \to E$ be a projection operator on a vector space $E$ (i.e., $\varphi^2 = \varphi$). Then
$$E = \ker \varphi \oplus \operatorname{Im} \varphi$$
and
$$\varphi = \iota_{\operatorname{Im} \varphi} \oplus 0_{\ker \varphi}.$$

Conversely, if $E = E_1 \oplus E_2$ is a direct decomposition of $E$, then the linear mapping
$$\varphi = \iota_{E_1} \oplus 0_{E_2}$$
is a projection operator on $E$.
