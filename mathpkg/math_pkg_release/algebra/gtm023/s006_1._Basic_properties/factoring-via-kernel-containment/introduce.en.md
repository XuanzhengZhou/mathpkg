This proposition provides a necessary and sufficient algebraic condition for one linear map to factor through another. If the kernel of $\varphi$ is contained in the kernel of $\psi$, then $\psi$ descends to a map on the image of $\varphi$, which can then be extended to the full codomain $F$. The proof uses the first isomorphism theorem and the fact that any linear map defined on a subspace can be extended to the whole space.
