The free vector space over a set $X$ is a vector space having $X$ as a basis. It consists of all finitely supported $\Gamma$-valued functions on $X$. The free vector space satisfies a universal property: any set map from $X$ to a vector space extends uniquely to a linear map from $C(X)$. This construction shows that sets and vector spaces are intimately connected.
