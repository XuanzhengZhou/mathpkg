Proposition IV establishes that any finitely generated nonzero vector space admits a finite basis, and more generally, any linearly independent subset of a finite generating set can be extended to a basis using additional vectors from that generating set. The proof proceeds by selecting a minimal generating subsystem (for part i) and by induction on the size of the generating set (for part ii). This result ensures that finite-dimensional spaces have well-defined bases whose size — the dimension — is uniquely determined.
