Let $E$ and $F$ be vector spaces over $\Gamma$. A set mapping $\varphi: E \to F$ is called a \textbf{linear mapping} (or linear transformation) if

$$\varphi(x + y) = \varphi x + \varphi y \quad (x, y \in E)$$

and

$$\varphi(\lambda x) = \lambda \varphi x \quad (\lambda \in \Gamma, \; x \in E).$$

Equivalently, $\varphi$ preserves linear combinations:
$$\varphi\left(\sum_i \lambda^i x_i\right) = \sum_i \lambda^i \varphi x_i.$$

If $F = \Gamma$, then $\varphi$ is called a \textbf{linear function} (or linear form) on $E$. From the additivity condition it follows that $\varphi(0) = 0$.
