The free vector space over a set $X$ is the vector space of all finitely supported functions from $X$ to the field $\Gamma$, with pointwise operations. The elements of $X$ can be identified with the characteristic functions $f_a$, forming a distinguished basis. This construction gives a canonical way to turn any set into a vector space where the original set serves as a basis — every vector is a unique finite formal linear combination of elements of $X$. It is the "most general" vector space generated by $X$, satisfying a universal property.
