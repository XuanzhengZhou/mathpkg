Let $A$ be a set and assume that for some pairs $X, Y$ ($X \in A, Y \in A$) a relation, denoted by $X \leq Y$, is defined which satisfies the following conditions:

\begin{itemize}
\item[(i)] $X \leq X$ for every $X \in A$ (Reflexivity)
\item[(ii)] if $X \leq Y$ and $Y \leq X$ then $X = Y$ (Antisymmetry)
\item[(iii)] If $X \leq Y$ and $Y \leq Z$, then $X \leq Z$ (Transitivity).
\end{itemize}

Then $\leq$ is called a partial order in $A$.

A partially ordered set is called linearly ordered or a chain if for every pair $X, Y$ either $X \leq Y$ or $Y \leq X$.

Let $A_1$ be a subset of the partially ordered set $A$. Then an element $A \in A$ is called an upper bound for $A_1$ if $X \leq A$ for every $X \in A_1$.

A maximal element of $A$ is an element $A \in A$ such that the relation $A \leq X$ implies $A = X$.
