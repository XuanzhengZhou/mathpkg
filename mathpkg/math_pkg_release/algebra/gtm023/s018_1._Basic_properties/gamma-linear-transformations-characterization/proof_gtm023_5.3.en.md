If $\varphi \in A_{\Gamma}(E; E)$, then $\varphi(\alpha x) = \alpha \varphi x$ for all $x \in E$, $\alpha \in \Gamma$, which means $\varphi \circ \varepsilon_{\alpha} = \varepsilon_{\alpha} \circ \varphi$. Conversely, if this commutation holds, then for any $x \in E$, $\varphi(\alpha x) = \varphi \varepsilon_{\alpha} x = \varepsilon_{\alpha} \varphi x = \alpha \varphi x$, so $\varphi \in A_{\Gamma}(E; E)$.
