Assume $\mu = f^k$ with $\deg f = p$ and $f$ irreducible. Let $N_\kappa$ be the number of irreducible subspaces of dimension $p\kappa$ ($1 \leq \kappa \leq k$) in a decomposition of $E$. Set $\psi = f(\varphi)$. Then
$$N_j = \frac{1}{p} \left[ r(\psi^{j+1}) + r(\psi^{j-1}) - 2r(\psi^j) \right] \quad j = 1, \ldots, k,$$
where $r(\psi^j) = \operatorname{rank}(\psi^j)$. In particular,
$$N_k = \frac{1}{p} r(\psi^{k-1}) \geq 1$$
and $N_j = 0$ for $j > k$.
