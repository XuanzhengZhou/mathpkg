From the theorem we have
$$\dim F = \deg \nu$$
while according to sec. 13.2 $\nu$ divides $\mu$.

Hence $\nu = \mu$ if and only if $\deg \nu = \deg \mu$; i.e., if and only if
$$\dim F = \deg \mu.$$
