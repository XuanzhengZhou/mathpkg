The Jordan canonical matrix provides a standard matrix representation for a linear transformation. For an irreducible transformation with $\mu = f^k$ and $\deg f = p$, a basis is constructed from a generator $e$ via $a_{ij} = f(\varphi)^{i-1} \varphi^{j-1} e$, yielding $k$ Jordan blocks of size $p$. The companion matrix of $f$ appears as each block. For general $\varphi$, the matrix is block diagonal with such blocks.
