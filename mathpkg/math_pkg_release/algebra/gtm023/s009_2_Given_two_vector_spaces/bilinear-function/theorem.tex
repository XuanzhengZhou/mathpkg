A bilinear function in $E \times F$ is a mapping $\Phi: E \times F \rightarrow \Gamma$ satisfying
$$\Phi(\lambda_1 x_1 + \lambda_2 x_2, y) = \lambda_1 \Phi(x_1, y) + \lambda_2 \Phi(x_2, y)$$
and
$$\Phi(x, \mu_1 y_1 + \mu_2 y_2) = \mu_1 \Phi(x, y_1) + \mu_2 \Phi(x, y_2).$$

If $\Phi$ is a bilinear function in $E \times F$ and $E_1 \subset E$, $F_1 \subset F$ are subspaces, then $\Phi$ induces a bilinear function $\Phi_1$ in $E_1 \times F_1$ called the restriction.

Every bilinear function $\Phi_1$ in $E_1 \times F_1$ may be extended to a bilinear function in $E \times F$.
