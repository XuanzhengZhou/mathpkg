A group $G$ is called commutative or abelian if for each $x, y \in G$
$$xy = yx.$$

In an abelian group one often writes $x + y$ instead of $xy$ and calls $x + y$ the sum of $x$ and $y$. Then the unit element is denoted by $0$. As an example, the set $\mathbb{Z}$ of integers with usual addition is an abelian group.
