The set $S_n$ of all permutations of the set $\{1, \ldots, n\}$, together with the composition defined by
$$(\sigma \tau)i = \sigma(\tau i), \quad i = 1, \ldots, n,$$
forms a group, called the \textbf{symmetric group} of $n$ objects. The identity element is the identity permutation. The order of $S_n$ is $n!$.
