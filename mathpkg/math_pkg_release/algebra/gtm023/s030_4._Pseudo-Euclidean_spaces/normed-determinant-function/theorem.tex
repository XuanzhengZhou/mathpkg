A determinant function $\Delta$ in a pseudo-Euclidean space $E$ of dimension $n$ and index $s$ is called a \text{normed determinant function} if it satisfies the relation
$$\Delta(x_1 \ldots x_n) \Delta(y_1 \ldots y_n) = (-1)^{n-s} \det(x_i, y_j).$$

A normed determinant function can be defined by
$$\Delta = \pm \frac{\Delta_0}{\sqrt{(-1)^{n-s}} \alpha},$$
where $\Delta_0$ is an arbitrary determinant function. Consequently, there exist exactly two normed determinant functions $\Delta$ and $-\Delta$ in $E$.
