Injective linear maps preserve linear independence: if a set $S$ of vectors is linearly independent in the domain, then its image $\varphi(S)$ remains linearly independent in the codomain. This fails for non-injective maps (e.g., the zero map collapses everything to $\{0\}$, which is linearly dependent). The proof uses the definition of linear independence and the injectivity of $\varphi$ to pull a linear relation on $\varphi(S)$ back to one on $S$.
