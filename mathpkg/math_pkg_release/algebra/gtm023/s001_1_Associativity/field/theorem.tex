A \textbf{field} is a set $\Gamma$ on which two binary laws of composition, called respectively addition and multiplication, are defined such that:

\begin{enumerate}
  \item $\Gamma$ is a commutative group with respect to the addition.
  \item The set $\Gamma \setminus \{0\}$ is a commutative group with respect to the multiplication.
  \item Addition and multiplication are connected by the distributive law,
  $$(\alpha + \beta)\gamma = \alpha \gamma + \beta \gamma, \quad \alpha, \beta, \gamma \in \Gamma.$$
\end{enumerate}

The rational numbers $\mathbb{Q}$, the real numbers $\mathbb{R}$, and the complex numbers $\mathbb{C}$ are fields with respect to the usual operations.
