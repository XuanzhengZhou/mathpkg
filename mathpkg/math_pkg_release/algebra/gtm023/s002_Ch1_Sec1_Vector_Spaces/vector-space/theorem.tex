A **vector (linear) space** $E$ over a field $\Gamma$ is a set of elements (vectors) with two operations:

**I. Addition:** $E$ is an additive group under $+: E \times E \to E$ satisfying:
- I.1 $(x+y)+z = x+(y+z)$ (associative)
- I.2 $x+y = y+x$ (commutative)
- I.3 There exists $0 \in E$ with $x+0 = 0+x = x$
- I.4 For each $x$, there exists $-x$ with $x+(-x)=0$

**II. Scalar multiplication:** $\Gamma \times E \to E$, $(\lambda, x) \mapsto \lambda x$, satisfying:
- II.1 $(\lambda\mu)x = \lambda(\mu x)$
- II.2 $(\lambda+\mu)x = \lambda x + \mu x$ and $\lambda(x+y) = \lambda x + \lambda y$
- II.3 $1 \cdot x = x$

If $\Gamma = \mathbb{R}$ (resp. $\mathbb{C}$), $E$ is a real (resp. complex) vector space.
