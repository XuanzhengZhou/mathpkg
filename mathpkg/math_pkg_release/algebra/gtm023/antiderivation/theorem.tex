Let $\omega$ be a fixed involution in $A$ (an endomorphism with $\omega^2 = \iota$). A linear transformation $\Omega: A \to A$ is called an antiderivation with respect to $\omega$ if
$$\Omega(xy) = \Omega x \cdot y + \omega x \cdot \Omega y.$$

A derivation is an antiderivation with respect to $\iota$. The kernel of an antiderivation is a subalgebra. If $A$ has a unit element $e$, then $\Omega e = 0$.
