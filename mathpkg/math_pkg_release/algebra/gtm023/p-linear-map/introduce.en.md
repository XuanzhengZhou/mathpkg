A $p$-linear map is a function of $p$ vector arguments that is linear in each argument separately. When $p=2$ these are bilinear maps, when $p=1$ these are ordinary linear maps. $p$-linear maps form the foundation of multilinear algebra. Examples include the tensor product and determinant functions.
