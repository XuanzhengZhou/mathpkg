Let $A$ be an $n \times n$-matrix with row vectors $a_1, \ldots, a_n$. Then

$$\det A = \Delta(a_1, \ldots, a_n),$$

where $\Delta$ is the determinant function on $\Gamma^n$ that assumes the value $1$ at the standard basis $e_1, \ldots, e_n$.

Consequently, the determinant as a function of the row vectors has the following properties:
\begin{enumerate}
  \item The determinant is linear with respect to every row-vector.
  \item If two row-vectors are interchanged, the determinant changes sign.
  \item The determinant does not change if a multiple of another row-vector is added to a row-vector.
  \item The determinant is different from zero if and only if the row-vectors are linearly independent.
\end{enumerate}

The same properties hold when the determinant is considered as a function of the column-vectors.
