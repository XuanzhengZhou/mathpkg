Apply the Gauss elimination to the following system:

$$\eta^1 + 2\eta^2 + 3\eta^3 + 4\eta^4 = 5,$$
$$2\eta^1 + \eta^2 + 4\eta^3 + \eta^4 = 2,$$
$$3\eta^1 + 4\eta^2 + \eta^3 + 5\eta^4 = 6,$$
$$2\eta^1 + 3\eta^2 + 5\eta^3 + 2\eta^4 = 3.$$
