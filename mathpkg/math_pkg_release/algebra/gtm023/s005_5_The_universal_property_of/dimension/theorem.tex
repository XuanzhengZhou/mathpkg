Every system of generators of a non-trivial vector space contains a basis. For a finitely generated non-trivial vector space $E$, every basis consists of the same number of vectors. This number is called the \textbf{dimension} of $E$ and is denoted by $\dim E$. The zero vector space $\{0\}$ is assigned dimension $0$. If $E$ does not have finite dimension, it is called an \textbf{infinite-dimensional} vector space.
