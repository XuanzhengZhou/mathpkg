A continuous function $\Psi$ of one vector which satisfies the parallelogram-identity
$$\Psi(x + y) + \Psi(x - y) = 2(\Psi(x) + \Psi(y))$$
is called a quadratic function. Every quadratic function comes from a symmetric bilinear function $\Phi$ via $\Psi(x) = \Phi(x, x)$, and $\Phi$ is given by the polarization identity. In particular, $\Psi(0) = 0$ and $\Psi(-y) = \Psi(y)$.
