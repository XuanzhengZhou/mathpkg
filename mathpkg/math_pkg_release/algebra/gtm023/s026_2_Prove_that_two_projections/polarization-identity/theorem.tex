Given a symmetric bilinear function $\Phi$ and its associated quadratic function $\Psi(x) = \Phi(x, x)$, the bilinear function can be recovered by the polarization identity:
$$\Phi(x, y) = \frac{1}{2} \left\{ \Psi(x + y) - \Psi(x) - \Psi(y) \right\}.$$
Equivalently,
$$\Psi(x + y) = \Psi(x) + 2\Phi(x, y) + \Psi(y).$$
