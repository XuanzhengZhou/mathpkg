A field is a set $\Gamma$ on which two binary laws of composition, called respectively addition and multiplication, are defined such that

1. $\Gamma$ is a commutative group with respect to the addition.
2. The set $\Gamma - \{0\}$ is a commutative group with respect to the multiplication.
3. Addition and multiplication are connected by the distributive law,
$$(\alpha + \beta)\gamma = \alpha \gamma + \beta \gamma, \quad \alpha, \beta, \gamma \in \Gamma.$$

The rational numbers $\mathbb{Q}$, the real numbers $\mathbb{R}$ and the complex numbers $\mathbb{C}$ are fields with respect to the usual operations.
