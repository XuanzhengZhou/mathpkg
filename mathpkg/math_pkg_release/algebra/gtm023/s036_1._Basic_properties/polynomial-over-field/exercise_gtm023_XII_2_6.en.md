Calculate the greatest common divisors and the least common multiples of the following polynomials over $\mathbb{R}[t]$:

a) $t^5 + t^4 + t^3 + t^2 + t + 1$, $t^6 - 1$
b) $t^3 + 3t^2 + 1$, $t^4 - t + 7$, $7t^2 + 16$
c) $5t^4 - \frac{1}{6}t^3 + t^2 - 3t + 7$, $4t^4 - 17t^3 + 16$, $\frac{1}{10}t^5 - \frac{1}{8}t^2 + 2t + 1$
d) $8t^8 + \sqrt{6}t^4 - \sqrt{2}t^2 - 72 + 2\sqrt{6}$, $2t^8 + t^5 + 5t^4 - 6t^2 - 3t - 15$
e) $3t^4 + 50t^3 - 9t^2 + 84t + 5$, $t^4 + 15t^3 - 29t^2 - 64t + 4$
