The homomorphism $\Phi: f \mapsto f(a)$ induces a monomorphism
$$\bar{\Phi}: \Gamma[t] / \ker \Phi \rightarrow A.$$
In particular, if $A$ is generated by $e$ and $a$, then $\Phi$ (and hence $\bar{\Phi}$) are surjective and so $\bar{\Phi}$ is an isomorphism
$$\bar{\Phi}: \Gamma[t] / \ker \Phi \xrightarrow{\cong} A.$$
