Let $F$ be the functional on $\dot{E} = E \setminus \{0\}$ defined (up to an additive constant) by the path-independent line integral $J$. Then $F$ is homogeneous of degree zero:
$$F(\lambda x) = F(x) \quad \text{for all } x \neq 0 \text{ and all } \lambda \neq 0.$$
Equivalently, the integral $J$ along any path and along the same path scaled by $\lambda$ yields the same value.
