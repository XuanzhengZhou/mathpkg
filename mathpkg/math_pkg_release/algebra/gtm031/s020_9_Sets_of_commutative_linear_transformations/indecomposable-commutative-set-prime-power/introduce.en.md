For a set of commutative linear transformations that is indecomposable (the space cannot be decomposed into a direct sum of proper invariant subspaces), every transformation in the set must have a minimum polynomial that is a prime power. This lemma is a key step toward the simultaneous triangularization theorem for commutative matrices, as it rules out the possibility of splitting the space via primary decomposition of a single operator when the set is indecomposable.
