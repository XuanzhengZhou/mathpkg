If $\Delta$ is a finite division ring with $q$ elements (hence a finite field by Wedderburn's theorem), the general linear group $L(\Delta, n) = \mathrm{GL}(n, q)$ has order $(q^n - 1)(q^n - q) \cdots (q^n - q^{n-1})$. This classical formula counts the number of ways to choose an ordered basis of an $n$-dimensional vector space over $\Delta$: the first basis vector can be any non-zero vector ($q^n - 1$ choices), the second any vector not in the span of the first ($q^n - q$ choices), and so on.
