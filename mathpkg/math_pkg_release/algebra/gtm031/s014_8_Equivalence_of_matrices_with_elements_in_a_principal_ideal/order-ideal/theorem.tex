Let $x$ be any element of the $\mathfrak{o}$-module $\mathfrak{R}$. Then the set $\mathfrak{I}_x$ of elements $\beta \in \mathfrak{o}$ such that $\beta x = 0$ is an ideal. This ideal is called the \text{order ideal} of the element $x$. If $\mathfrak{o}$ is a principal ideal domain, then $\mathfrak{I}_x = (\mu_x)$. Moreover, if $\mathfrak{R} = \Phi[\lambda]$ and $\mathfrak{I}_x \neq (0)$, then $\mu_x$ may be normalized to have leading coefficient $1$.
