Prove the following theorem on matrices: If $(\epsilon_1), (\epsilon_2), \ldots, (\epsilon_r)$ is a set of matrices such that

$$(\epsilon_i)^2 = (\epsilon_i), \quad (\epsilon_i)(\epsilon_j) = 0 \text{ for } i \neq j, \quad \sum_{i=1}^r (\epsilon_i) = 1,$$

then there exists a non-singular matrix $(\mu)$ such that $(\mu)^{-1}(\epsilon_i)(\mu)$ has the form $\operatorname{diag}\{0, \ldots, 0, 1, \ldots, 1, 0, \ldots, 0\}$ where the $1$'s occur in a set of consecutive places on the diagonal, and the number of $1$'s in $(\mu)^{-1}(\epsilon_i)(\mu)$ is the rank of $(\epsilon_i)$.
