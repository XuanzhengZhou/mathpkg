Two square matrices are similar if they represent the same linear endomorphism with respect to different bases of the same vector space. The relation is $(\tilde{\alpha}) = (\mu)(\alpha)(\mu)^{-1}$ for an invertible matrix $(\mu)$. While matrix equivalence (for rectangular matrices) has a simple characterization via rank, the similarity problem for square matrices is much deeper: it amounts to classifying linear transformations up to change of basis. Jacobson illustrates the method with the example of the cyclic permutation matrix and the diagonal matrix of $n$th roots of unity, which are similar over $\mathbb{C}$.
