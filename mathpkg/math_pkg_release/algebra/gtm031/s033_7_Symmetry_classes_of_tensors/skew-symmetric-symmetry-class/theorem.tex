Let $\{Q\}$ be the set of symmetry operators $P(\sigma) - \epsilon_\sigma$ where
$$\epsilon_\sigma = \begin{cases} 1 & \text{if } \sigma \text{ is an even permutation}, \\ -1 & \text{if } \sigma \text{ is odd}. \end{cases}$$
The set $\{Q\}$ determines the class of \textit{skew symmetric} (or \textit{alternating}) tensors. A tensor belongs to this class if it is annihilated by all operators $P(\sigma) - \epsilon_\sigma$, i.e.,
$$P(\sigma)x = \epsilon_\sigma x \quad \text{for all } \sigma \in \mathfrak{S}_r.$$
