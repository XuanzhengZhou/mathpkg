Let $\Delta$ be a division ring and let $n$ be a fixed positive integer. The \emph{vector space of $n$-tuples over $\Delta$} is the set

$$\Delta^{(n)} = \{ (\xi_1, \xi_2, \cdots, \xi_n) \mid \xi_i \in \Delta \}$$

with equality defined by $(\xi_1, \ldots, \xi_n) = (\eta_1, \ldots, \eta_n)$ if and only if $\xi_i = \eta_i$ for $i = 1, 2, \ldots, n$, and with the following operations:

\textbf{Addition:} For $x = (\xi_1, \ldots, \xi_n)$ and $y = (\eta_1, \ldots, \eta_n)$,
$$x + y = (\xi_1 + \eta_1, \xi_2 + \eta_2, \cdots, \xi_n + \eta_n).$$

\textbf{Scalar multiplication (left):} For $\rho \in \Delta$ and $x = (\xi_1, \ldots, \xi_n)$,
$$\rho x = (\rho \xi_1, \rho \xi_2, \cdots, \rho \xi_n).$$

With these operations, $\Delta^{(n)}$ is a finite dimensional vector space over $\Delta$ satisfying axioms A1--A4, S1--S4, and F.
