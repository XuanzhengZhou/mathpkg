Let $\mathfrak{R} = \mathfrak{R}_1 \oplus \mathfrak{R}_2 \oplus \cdots \oplus \mathfrak{R}_s$ be a direct sum decomposition of the vector space $\mathfrak{R}$ into subspaces $\mathfrak{R}_i$ that are invariant relative to a set $\Omega$ of linear transformations. The decomposition is called \text{proper} if each $\mathfrak{R}_i \neq 0$ and $s > 1$.
