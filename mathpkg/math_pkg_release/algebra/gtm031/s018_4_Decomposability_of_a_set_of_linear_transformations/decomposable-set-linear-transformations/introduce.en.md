A set $\Omega$ of linear transformations on a vector space $\mathfrak{R}$ is decomposable if $\mathfrak{R}$ can be written as a direct sum of two or more non-zero $\Omega$-invariant subspaces. Decomposability is strictly stronger than reducibility: every irreducible set is automatically indecomposable, but a reducible set may still be indecomposable, as demonstrated by the single nilpotent Jordan block example.
