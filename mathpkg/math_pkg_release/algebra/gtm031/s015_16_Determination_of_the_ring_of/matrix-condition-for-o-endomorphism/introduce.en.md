When an endomorphism is expressed using a matrix $(\beta)$ with respect to the cyclic generator decomposition, the order ideal condition $\delta_i g_i = 0$ translates into a system of congruences on the matrix entries: each $\beta_{ij}$ in column $j$ is determined only modulo $\delta_j$, and the entries satisfy $\delta_i \beta_{ij} \equiv 0 \pmod{\delta_j}$. This yields the matrix equation $\operatorname{diag}(\delta)(\beta) = (\gamma)\operatorname{diag}(\delta)$ for some matrix $(\gamma)$.
