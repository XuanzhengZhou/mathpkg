Let $\mathfrak{R} = \{f_1\} \oplus \cdots \oplus \{f_t\}$ with order ideals $(\delta_i)$. Let $\mathfrak{M}$ be the set of all $t \times t$ matrices $(\beta) = (\beta_{ij})$ over $\mathfrak{o}$ satisfying
$$\delta_i \beta_{ij} \equiv 0 \pmod{\delta_j}, \quad i, j = 1, \ldots, t,$$
or equivalently, for which there exists a matrix $(\gamma)$ such that
$$\operatorname{diag}(\delta_1, \ldots, \delta_t)\, (\beta) = (\gamma)\, \operatorname{diag}(\delta_1, \ldots, \delta_t).$$
Then $\mathfrak{M}$ is a subring of the full matrix ring $\mathfrak{o}_t$.
