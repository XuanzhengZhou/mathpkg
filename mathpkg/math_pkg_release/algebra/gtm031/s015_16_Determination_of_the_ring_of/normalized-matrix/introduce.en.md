A normalized matrix is a canonical choice of matrix representative for an $\mathfrak{o}$-endomorphism with respect to the invariant factor decomposition. The normalization conditions exploit the divisibility relations $\delta_i \mid \delta_j$ to specify which entries are arbitrary, which are constrained modulo certain elements, and which must be zero, yielding a unique matrix for each endomorphism.
