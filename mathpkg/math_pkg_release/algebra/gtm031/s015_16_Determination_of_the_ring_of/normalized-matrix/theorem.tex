Let $\mathfrak{R} = \{f_1\} \oplus \cdots \oplus \{f_t\}$ with order ideals $(\delta_i)$ satisfying $\delta_j = 0$ if $j > u$ and $\delta_i \mid \delta_j$ for $i \leq j$. A matrix $(\beta) \in \mathfrak{M}$ is called a **normalized matrix** if it satisfies the following four conditions:
\begin{enumerate}
\item For $i \geq j$: $\beta_{ij}$ is arbitrary.
\item For $i \leq u < j$: $\beta_{ij} = 0$.
\item For $i, j > u$: $\beta_{ij}$ is arbitrary.
\item For $i < j \leq u$: $\beta_{ij} \equiv 0 \pmod{\eta_{ij}}$ where $\eta_{ij} = \delta_i^{-1} \delta_j$.
\end{enumerate}
The set of all normalized matrices is denoted by $\mathfrak{U}$.
