Let $g(x, y')$ be a bilinear form on $\mathfrak{R} \times \mathfrak{R}'$. For each fixed $y' \in \mathfrak{R}'$, define $g_{y'}(x) = g(x, y')$. Then $g_{y'} \in \mathfrak{R}^*$ is a linear function on $\mathfrak{R}$. The mapping

$$R: \mathfrak{R}' \to \mathfrak{R}^*, \quad y' \mapsto g_{y'}$$

is a linear transformation. Similarly, for each fixed $x \in \mathfrak{R}$, define $g_x(y') = g(x, y')$. Then $g_x \in (\mathfrak{R}')^*$ is a linear function on $\mathfrak{R}'$, and the mapping

$$L: \mathfrak{R} \to (\mathfrak{R}')^*, \quad x \mapsto g_x$$

is a linear transformation.
