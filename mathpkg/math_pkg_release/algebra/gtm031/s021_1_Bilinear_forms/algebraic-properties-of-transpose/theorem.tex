The mapping $A \mapsto A'$ from $\mathfrak{L}(\mathfrak{R}_1, \mathfrak{R}_2)$ to $\mathfrak{L}(\mathfrak{R}_2', \mathfrak{R}_1')$ satisfies the following algebraic properties:

\begin{enumerate}
\item \textbf{Additivity:} If $A, B \in \mathfrak{L}(\mathfrak{R}_1, \mathfrak{R}_2)$, then
$$(A + B)' = A' + B'.$$

\item If $\mathfrak{R}_1 = \mathfrak{R}_2 = \mathfrak{R}$ and $\mathfrak{R}_1' = \mathfrak{R}_2' = \mathfrak{R}'$, then for $A, B \in \mathfrak{L}(\mathfrak{R}, \mathfrak{R})$,
$$(AB)' = B' A'.$$

\item If $A$ is invertible, then $A'$ is invertible and $(A')^{-1} = (A^{-1})'$.
\end{enumerate}
