Let $\Omega$ be a set of linear transformations in the vector space $\Re$ over a division ring $\Delta$. The trivial subspaces $0$ and $\Re$ are always invariant under $\Omega$.

The set $\Omega$ is called an **irreducible set** (of linear transformations) if the only invariant subspaces relative to $\Omega$ are the trivial subspaces $0$ and $\Re$. One also says that $\Re$ is **irreducible** relative to the set $\Omega$.
