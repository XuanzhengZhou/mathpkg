Let $\Omega$ be a set of linear transformations in a finite dimensional vector space $\Re$ over a division ring $\Delta$, and let $\omega$ be the set of matrices of $\Omega$ relative to a basis $(e_1, e_2, \ldots, e_n)$.

If $\Omega$ is reducible, i.e., there exists a proper invariant subspace $\mathfrak{S} \neq 0, \Re$, then there exists a non-singular matrix $(\mu)$ such that all the matrices of $(\mu)\omega(\mu)^{-1}$ have the same reduced form:

$$\begin{bmatrix}
A_{11} & 0 \\
A_{21} & A_{22}
\end{bmatrix}$$

where $A_{11}$ is an $r \times r$ block, $A_{22}$ is an $(n-r) \times (n-r)$ block, and the upper right $r \times (n-r)$ block is zero.
