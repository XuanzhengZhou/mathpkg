If a Euclidean space $\mathfrak{R}$ is irreducible relative to a commutative set $\Omega$ of linear transformations which are either symmetric or skew, then $\dim \mathfrak{R} \leq 2$. Moreover:
\begin{itemize}
\item If $\Omega$ contains a symmetric $A \neq 0$, then every $A \in \Omega$ is a scalar multiplication, so $\dim \mathfrak{R} = 1$.
\item If $\Omega$ consists only of skew transformations, then either all are zero ($\dim \mathfrak{R}$ arbitrary but irreducible implies $\dim \mathfrak{R} = 1$) or $\Omega$ consists of scalar multiplications and multiples of a single skew transformation $A$ with $A^2 = -\delta^2 1$.
\end{itemize}
