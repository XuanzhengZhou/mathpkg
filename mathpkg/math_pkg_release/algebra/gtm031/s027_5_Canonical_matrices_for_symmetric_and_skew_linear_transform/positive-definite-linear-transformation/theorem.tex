A linear transformation $A$ on a Euclidean space is called \textbf{positive definite} if it is symmetric and if the associated symmetric bilinear form $(xA, y)$ is positive definite:
$$(xA, x) > 0 \quad \text{for all } x \neq 0.$$
It is evident from the definition that a positive definite transformation is semi-definite and 1-1.
