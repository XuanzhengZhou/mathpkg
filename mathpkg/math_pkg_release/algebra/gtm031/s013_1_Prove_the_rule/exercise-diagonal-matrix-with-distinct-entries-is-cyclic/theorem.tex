\text{Show that, if } A \text{ is a linear transformation that has the diagonal matrix } \operatorname{diag}\{\alpha_1, \alpha_2, \dots, \alpha_n\} \text{ where the } \alpha_i \text{ are distinct, then the minimum polynomial of } A \text{ is } \prod_{i=1}^n (\lambda - \alpha_i). \text{ Hence prove } A \text{ is cyclic.}
