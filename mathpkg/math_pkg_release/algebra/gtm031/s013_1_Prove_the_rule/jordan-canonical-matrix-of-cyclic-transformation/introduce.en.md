For a cyclic linear transformation $A$ with cyclic vector $e$, the matrix of $A$ relative to the cyclic basis $(e, eA, \dots, eA^{n-1})$ is the companion matrix of the minimum polynomial. This is called the Jordan canonical matrix of $A$, and it provides the simplest possible matrix representation: the minimum polynomial (and hence the entire structure of $A$) can be read directly from the last row.
