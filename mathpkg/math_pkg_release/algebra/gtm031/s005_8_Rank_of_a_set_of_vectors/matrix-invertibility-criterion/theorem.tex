Let $(\alpha) \in L(\Phi, n)$ be an $n \times n$ matrix with elements in a field $\Phi$, and let $(e_1, e_2, \cdots, e_n)$ be a basis for the $n$-dimensional vector space $\Re$. The matrix $(\alpha)$ is invertible (i.e., belongs to the general linear group) if and only if its determinant is not zero in $\Phi$.
