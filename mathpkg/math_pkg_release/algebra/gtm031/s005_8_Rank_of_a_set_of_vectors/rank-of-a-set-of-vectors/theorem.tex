Let $S = (x_\alpha)$ be an arbitrary subset of the vector space $\Re$ and let $[S]$ denote the subspace spanned by $S$. If $(x_1, x_2, \cdots, x_r)$ is a maximal linearly independent set of vectors chosen from $S$, then every vector in $S$ and hence in $[S]$ is linearly dependent on the $x_i$. Hence $(x_1, x_2, \cdots, x_r)$ is a basis for $[S]$. By the theorem on invariance of dimensionality, $r$ is uniquely determined by $S$, that is, any two maximal linearly independent subsets of a set $S$ have the same cardinal number. This number is called the \textbf{rank} of the set $S$.
