The rank of a set of vectors is defined as the cardinal number of any maximal linearly independent subset of the set. By the invariance of dimensionality, this number is uniquely determined by the set itself—any two maximal linearly independent subsets have the same cardinality. The rank provides a fundamental measure of the "size" of the subspace spanned by the set and generalizes the notion of dimension to arbitrary subsets of a vector space.
