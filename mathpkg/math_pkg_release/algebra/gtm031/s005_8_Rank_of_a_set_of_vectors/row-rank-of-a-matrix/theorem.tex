Let $(\alpha)$ be an arbitrary $r \times n$ matrix with elements in a division ring $\Delta$ and let $(e_1, e_2, \cdots, e_n)$ be an arbitrary ordered basis for $\Re$. Define the row vectors
$$
x_i = \sum_{j=1}^{n} \alpha_{ij} e_j, \quad i = 1, 2, \cdots, r,
$$
of $\Re$. The \textbf{row rank} of $(\alpha)$ is defined to be the rank of the set $(x_1, x_2, \cdots, x_r)$. A different choice of basis yields the same result.
