$k$-fold transitivity is a graded generalization of density in the finite topology: a set of linear transformations is $k$-fold transitive if it can simultaneously send any $l \leq k$ linearly independent vectors to any $l$ target vectors. A set is dense if and only if it is $k$-fold transitive for every finite $k$. This notion provides the inductive bridge needed to prove the Jacobson Density Theorem, showing that two-fold transitivity together with a simple algebraic condition implies full density.
