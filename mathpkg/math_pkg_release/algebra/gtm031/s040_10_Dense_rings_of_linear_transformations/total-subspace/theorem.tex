A subspace $\mathfrak{R}'$ of the conjugate space $\mathfrak{R}^*$ (the space of all linear functions on $\mathfrak{R}$) is called \textbf{total} if for any finite set of linearly independent vectors $y_1, y_2, \cdots, y_m$ in $\mathfrak{R}$, there exist linear functions $\phi_1, \phi_2, \cdots, \phi_m \in \mathfrak{R}'$ such that
$$\phi_i(y_j) = \delta_{ij}, \quad i, j = 1, 2, \cdots, m.$$
