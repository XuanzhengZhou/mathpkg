Let $L$ be a semisimple Lie algebra over an algebraically closed field of characteristic 0. Then $\text{ad}: L \rightarrow \text{Der } L$ is an isomorphism, and each $x \in L$ determines unique elements $x_s, x_n \in L$ such that:
$$x = x_s + x_n, \quad [x_s, x_n] = 0,$$
with $\text{ad } x_s$ semisimple and $\text{ad } x_n$ nilpotent. Moreover, $\text{ad } x = \text{ad } x_s + \text{ad } x_n$ is the usual Jordan decomposition of $\text{ad } x$ in $\text{End } L$. The element $x_s$ is called \text{ad-semisimple} and $x_n$ is \text{ad-nilpotent}.
