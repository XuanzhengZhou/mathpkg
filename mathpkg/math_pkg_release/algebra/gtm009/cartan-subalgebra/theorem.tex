A \text{Cartan subalgebra} (CSA) of a Lie algebra $L$ is a nilpotent subalgebra which equals its own normalizer in $L$:
$$N_L(H) = H.$$
Equivalently, $H$ is a minimal Engel subalgebra of $L$, where an Engel subalgebra is $L_0(\text{ad } x) = \{y \in L \mid (\text{ad } x)^k y = 0 \text{ for some } k\}$ for $x \in L$.
