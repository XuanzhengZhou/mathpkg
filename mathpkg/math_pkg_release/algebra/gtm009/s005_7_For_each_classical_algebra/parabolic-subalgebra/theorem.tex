A subalgebra $P$ of a Lie algebra $L$ is called \textbf{parabolic} if $P$ includes some Borel subalgebra. (In that case $P$ is self-normalizing, by Lemma 15.2B.)

Fix a base $\Delta \subset \Phi$, and set $B = B(\Delta)$. For each subset $\Delta' \subset \Delta$, define the \textbf{standard parabolic subalgebra} $P(\Delta')$ to be the subalgebra of $L$ generated by all $L_{\alpha}$ (where $\alpha \in \Delta$ or $-\alpha \in \Delta'$), along with $H$.
