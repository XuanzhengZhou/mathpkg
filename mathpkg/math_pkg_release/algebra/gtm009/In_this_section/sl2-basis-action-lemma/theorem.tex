Let $V$ be an irreducible $L$-module for $L = \mathfrak{sl}(2, \mathbb{F})$. Choose a maximal vector $v_0 \in V_\lambda$ and set $v_{-1} = 0$, $v_i = \frac{1}{i!} y^i.v_0$ for $i \geq 0$. Then for all $i \geq 0$:

\text{(a)} \quad h.v_i = (\lambda - 2i)v_i,

\text{(b)} \quad y.v_i = (i + 1)v_{i+1},

\text{(c)} \quad x.v_i = (\lambda - i + 1)v_{i-1}.
