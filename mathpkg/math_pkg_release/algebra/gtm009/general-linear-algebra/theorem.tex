Let $V$ be a finite dimensional vector space over $F$, and denote by $\text{End } V$ the set of linear transformations $V \rightarrow V$. Define a new operation
$$[x, y] = xy - yx,$$
called the \text{bracket} of $x$ and $y$. With this operation $\text{End } V$ becomes a Lie algebra over $F$. To distinguish this Lie algebra structure from the associative one, we write $\mathfrak{gl}(V)$ for $\text{End } V$ viewed as a Lie algebra and call it the \text{general linear algebra}.

When a basis of $V$ is fixed, $\mathfrak{gl}(V)$ is identified with the set of all $n \times n$ matrices over $F$, denoted $\mathfrak{gl}(n, F)$. The bracket is given by
$$[e_{ij}, e_{kl}] = \delta_{jk}e_{il} - \delta_{li}e_{kj},$$
where $e_{ij}$ is the matrix with 1 in the $(i,j)$ position and 0 elsewhere.
