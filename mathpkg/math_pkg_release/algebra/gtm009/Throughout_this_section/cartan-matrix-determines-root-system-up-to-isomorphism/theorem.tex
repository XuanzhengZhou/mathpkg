Let $\Phi$ and $\Phi'$ be root systems with bases $\Delta = \{\alpha_1, \ldots, \alpha_\ell\}$ and $\Delta' = \{\alpha'_1, \ldots, \alpha'_\ell\}$. If $\langle \alpha_i, \alpha_j \rangle = \langle \alpha'_i, \alpha'_j \rangle$ for all $1 \leq i, j \leq \ell$, then the bijection $\alpha_i \mapsto \alpha'_i$ extends uniquely to an isomorphism $\phi: E \to E'$ mapping $\Phi$ onto $\Phi'$ and preserving all Cartan integers.
