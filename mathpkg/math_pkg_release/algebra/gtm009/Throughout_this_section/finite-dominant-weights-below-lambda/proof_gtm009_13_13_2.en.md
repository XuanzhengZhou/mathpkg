Since $\lambda + \mu \in \Lambda^+$ and $\lambda - \mu$ is a sum of positive roots, we have:
$$0 \leq (\lambda + \mu, \lambda - \mu) = (\lambda, \lambda) - (\mu, \mu).$$
Thus $(\mu, \mu) \leq (\lambda, \lambda)$, so $\mu$ lies in the compact set $\{x \in E \mid (x, x) \leq (\lambda, \lambda)\}$, whose intersection with the discrete set $\Lambda^+$ is finite.
