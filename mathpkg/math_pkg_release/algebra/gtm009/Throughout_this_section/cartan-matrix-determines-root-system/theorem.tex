Let $\Phi, \Phi'$ be root systems with bases $\Delta = \{\alpha_1, \ldots, \alpha_\ell\}$, $\Delta' = \{\alpha_1', \ldots, \alpha_\ell'\}$. If $\langle \alpha_i, \alpha_j \rangle = \langle \alpha_i', \alpha_j' \rangle$ for all $1 \leq i, j \leq \ell$, then the bijection $\alpha_i \mapsto \alpha_i'$ extends uniquely to an isomorphism $\phi: E \to E'$ mapping $\Phi$ onto $\Phi'$ and preserving all Cartan integers.
