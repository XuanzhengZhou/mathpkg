Let $\Lambda = \{\lambda \in E \mid \langle \lambda, \alpha \rangle \in \mathbb{Z} \text{ for all } \alpha \in \Phi\}$. Elements of $\Lambda$ are called weights. The root lattice is $\Lambda_r = \mathbb{Z}\Phi \subset \Lambda$. A weight $\lambda$ is dominant (relative to a base $\Delta$) if $\langle \lambda, \alpha \rangle \geq 0$ for all $\alpha \in \Delta$. The set of dominant weights is denoted $\Lambda^+$.
