If $\Phi$ is an irreducible root system with base $\Delta$, there exists a unique maximal root $\beta$ (relative to the partial order $\prec$). All coefficients $k_\alpha$ in $\beta = \sum_{\alpha \in \Delta} k_\alpha \alpha$ are strictly positive, and $(\alpha, \beta) \geq 0$ for all $\alpha \in \Delta$, with strict inequality for at least one $\alpha$.
