Let $\Phi$ be an irreducible root system with base $\Delta$. Then $\Phi$ has a unique maximal root $\beta$ (relative to the partial order $\prec$). Moreover, $\beta$ satisfies:
\begin{itemize}
\item All $k_\alpha > 0$ in the expansion $\beta = \sum_{\alpha \in \Delta} k_\alpha \alpha$.
\item $(\alpha, \beta) \geq 0$ for all $\alpha \in \Delta$, with $(\alpha, \beta) > 0$ for at least one $\alpha$.
\end{itemize}
