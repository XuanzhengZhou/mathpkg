This theorem (Theorem 10.1) provides an explicit construction of bases for root systems. Starting from any regular vector $\gamma$ (one not lying on any root hyperplane), the indecomposable positive roots form a base. Conversely, every base arises from some regular vector. This establishes a 1-1 correspondence between bases and Weyl chambers.
