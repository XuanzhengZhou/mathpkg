This fundamental theorem establishes the **existence of bases** for any root system. The proof is constructive: for a regular vector $\gamma$ (one not lying in any root hyperplane $P_\alpha$), the set of indecomposable positive roots forms a base. Conversely, every base arises this way. This yields a 1-1 correspondence between bases and Weyl chambers.
