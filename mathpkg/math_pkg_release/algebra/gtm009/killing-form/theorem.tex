Let $L$ be a Lie algebra. For $x, y \in L$, define
$$\kappa(x, y) = \text{Tr}(\text{ad } x \circ \text{ad } y).$$
Then $\kappa$ is a symmetric bilinear form on $L$, called the \text{Killing form}. It is also \text{associative}:
$$\kappa([xy], z) = \kappa(x, [yz]).$$

A symmetric bilinear form $\beta(x, y)$ is called \text{nondegenerate} if its radical
$$S = \{x \in L \mid \beta(x, y) = 0 \text{ for all } y \in L\}$$
is zero. Because the Killing form is associative, its radical is an ideal of $L$.
