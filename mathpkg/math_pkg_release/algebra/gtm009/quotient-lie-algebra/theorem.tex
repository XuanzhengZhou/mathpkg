Let $I$ be an ideal of a Lie algebra $L$. The \text{quotient algebra} $L/I$ is defined as follows: as a vector space, $L/I$ is the quotient space; the Lie bracket is defined by
$$[x+I, y+I] = [xy] + I.$$
This is well-defined: if $x' = x+u$, $y' = y+v$ with $u, v \in I$, then $[x'y'] = [xy] + ([uy] + [xv] + [uv])$, and the terms in parentheses all lie in $I$, so $[x'y'] + I = [xy] + I$.
