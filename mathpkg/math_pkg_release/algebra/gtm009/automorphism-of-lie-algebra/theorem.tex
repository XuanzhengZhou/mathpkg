An \text{automorphism} of $L$ is an isomorphism of $L$ onto itself. The group of all automorphisms is denoted $\text{Aut } L$.

If $\delta \in \text{Der } L$ is a nilpotent derivation, then the exponential
$$\exp \delta = 1 + \delta + \frac{\delta^2}{2!} + \cdots + \frac{\delta^{k-1}}{(k-1)!}$$
(where $\delta^k = 0$) is an automorphism of $L$. This follows from the identity $\exp \delta(x) \exp \delta(y) = \exp \delta(xy)$, which is proved using the Leibniz rule.

An automorphism of the form $\exp(\text{ad } x)$, with $\text{ad } x$ nilpotent, is called \text{inner}. The subgroup of $\text{Aut } L$ generated by all inner automorphisms is denoted $\text{Int } L$. $\text{Int } L$ is a normal subgroup of $\text{Aut } L$: if $\phi \in \text{Aut } L$, then $\phi(\text{ad } x)\phi^{-1} = \text{ad } \phi(x)$, whence $\phi \exp(\text{ad } x)\phi^{-1} = \exp(\text{ad } \phi(x))$.
