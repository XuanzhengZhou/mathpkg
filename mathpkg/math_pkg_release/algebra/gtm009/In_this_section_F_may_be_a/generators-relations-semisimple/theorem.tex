Let $L$ be a semisimple Lie algebra, $H$ a Cartan subalgebra, $\Phi$ the corresponding root system, and $\Delta = \{\alpha_1, \ldots, \alpha_\ell\}$ a fixed base. Fix a standard set of generators $x_i \in L_{\alpha_i}$, $y_i \in L_{-\alpha_i}$ with $[x_i y_i] = h_i = h_{\alpha_i}$. Then $L$ is generated by $\{x_i, y_i, h_i \mid 1 \leq i \leq \ell\}$, and these generators satisfy the following relations:
\begin{align*}
\text{(S1)} &\quad [h_i h_j] = 0 \qquad (1 \leq i, j \leq \ell);\\
\text{(S2)} &\quad [x_i y_i] = h_i, \quad [x_i y_j] = 0 \text{ if } i \neq j;\\
\text{(S3)} &\quad [h_i x_j] = \langle \alpha_j, \alpha_i \rangle x_j, \quad [h_i y_j] = -\langle \alpha_j, \alpha_i \rangle y_j;\\
\text{(S}_{ij}^+) &\quad (\operatorname{ad} x_i)^{-\langle \alpha_j, \alpha_i \rangle + 1}(x_j) = 0 \qquad (i \neq j);\\
\text{(S}_{ij}^-) &\quad (\operatorname{ad} y_i)^{-\langle \alpha_j, \alpha_i \rangle + 1}(y_j) = 0 \qquad (i \neq j).
\end{align*}
