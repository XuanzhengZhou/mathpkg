The following statements about a left $R$-module $P$ are equivalent:

(a) $P$ is projective;

(b) For each epimorphism $f: {}_R M \rightarrow {}_R N$ the map
$$\operatorname{Hom}_R(P, f): \operatorname{Hom}_R(P, M) \rightarrow \operatorname{Hom}_R(P, N)$$
is an epimorphism;

(c) For each bimodule structure ${}_R P_S$ the functor
$$\operatorname{Hom}_R({}_R P_S, -): {}_R\mathbf{M} \rightarrow {}_S\mathbf{M}$$
is exact;

(d) For every exact sequence
$$M' \xrightarrow{f} M \xrightarrow{g} M''$$
in ${}_R\mathbf{M}$ the sequence
$$\operatorname{Hom}_R(P, M') \xrightarrow{f_*} \operatorname{Hom}_R(P, M) \xrightarrow{g_*} \operatorname{Hom}_R(P, M'')$$
is exact.
