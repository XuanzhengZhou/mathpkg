If $\Gamma$ is linearly ordered by $\leq$, a $\Gamma \times \Gamma$-square matrix $A = \llbracket a_{\alpha\beta} \rrbracket$ is upper triangular (lower triangular) in case for all $\alpha, \beta \in \Gamma$

$$\alpha > \beta \text{ implies } a_{\alpha\beta} = 0 \quad (\alpha < \beta \text{ implies } a_{\alpha\beta} = 0).$$

Every scalar matrix is both upper and lower triangular.
