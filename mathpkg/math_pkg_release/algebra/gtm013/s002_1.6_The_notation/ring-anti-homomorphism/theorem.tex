Suppose that $R$ and $S$ are rings. Then a function $\phi: R \rightarrow S$ is a ring anti-homomorphism in case $\phi$ is an abelian group homomorphism, $\phi(1) = 1$, and

$$\phi(ab) = \phi(b)\phi(a).$$

Thus, the function $\phi: R \rightarrow S$ is a ring anti-homomorphism if and only if the same function $\phi: R^{\text{op}} \rightarrow S$ is a ring homomorphism.
