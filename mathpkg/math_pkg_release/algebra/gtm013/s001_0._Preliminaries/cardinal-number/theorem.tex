The cardinal numbers are the equivalence classes of the class of all sets under cardinal equivalence. The cardinal number of a set $A$ is denoted by $\operatorname{card} A$:
$$\operatorname{card} A = \{ B \mid \text{there is a bijection } A \to B \}.$$
Given two sets $A$ and $B$, we write $\operatorname{card} A \leq \operatorname{card} B$ if there is an injective function $A \to B$.
