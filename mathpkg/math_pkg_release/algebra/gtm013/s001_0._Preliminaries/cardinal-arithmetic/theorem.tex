Let $A, B, C$ be sets with $A$ infinite.
\begin{enumerate}
\item If $A$ is infinite, then $\operatorname{card} A + \operatorname{card} B = \max\{\operatorname{card} A, \operatorname{card} B\}$.
\item If $A$ is infinite and $B \neq \emptyset$, then $\operatorname{card} A \cdot \operatorname{card} B = \max\{\operatorname{card} A, \operatorname{card} B\}$.
\item $((\operatorname{card} A)^{(\operatorname{card} B)})^{(\operatorname{card} C)} = (\operatorname{card} A)^{(\operatorname{card} B) \cdot (\operatorname{card} C)}$.
\item If $\operatorname{card} B \geq 2$, then $(\operatorname{card} B)^{(\operatorname{card} A)} > \operatorname{card} A$.
\end{enumerate}
Moreover, $\operatorname{card}(\mathcal{P}(A)) = 2^{(\operatorname{card} A)} > \operatorname{card} A$, and the set of finite subsets of any infinite set $A$ has the same cardinality as $A$.
