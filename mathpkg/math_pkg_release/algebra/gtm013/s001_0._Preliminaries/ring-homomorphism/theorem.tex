Let $R$ and $S$ be rings. A function $\phi: R \to S$ is a ring homomorphism if for all $a, b \in R$:
\begin{itemize}
\item $\phi(a + b) = \phi(a) + \phi(b)$,
\item $\phi(ab) = \phi(a)\phi(b)$,
\item $\phi(1_R) = 1_S$.
\end{itemize}
The composition of two ring homomorphisms (where defined) is again a ring homomorphism, and the identity map $1_R: R \to R$ is a ring homomorphism. Thus the collection of rings and ring homomorphisms with the usual composition is a concrete category.
