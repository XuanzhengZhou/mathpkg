A relation $\leq$ on a set $P$ is a partial order on $P$ in case it is:
\begin{itemize}
\item reflexive: $a \leq a$ for all $a \in P$,
\item transitive: $a \leq b$ and $b \leq c$ implies $a \leq c$ for all $a, b, c \in P$,
\item antisymmetric: $a \leq b$ and $b \leq a$ implies $a = b$ for all $a, b \in P$.
\end{itemize}
