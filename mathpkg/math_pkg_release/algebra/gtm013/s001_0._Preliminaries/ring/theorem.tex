A ring is a system $(R, +, \cdot, 0, 1)$ consisting of a set $R$, two binary operations addition $(+)$ and multiplication $(\cdot)$, and two distinct elements $0 \neq 1$ of $R$ such that:
\begin{itemize}
\item $(R, +, 0)$ is an abelian group,
\item $(R, \cdot, 1)$ is a monoid (i.e., a semigroup with identity $1$),
\item multiplication is both left and right distributive over addition:
$$a(b + c) = ab + ac \quad \text{and} \quad (a + b)c = ac + bc \quad \text{for all } a, b, c \in R.$$
\end{itemize}
A ring whose multiplicative structure is commutative is called a commutative ring.
