Associated with a quiver (i.e., a finite directed graph) $\mathcal{Q}$ is the path semigroup algebra $K[\mathcal{Q}]$.

(1) If $I \leq N$ is an ideal and $R = K[\mathcal{Q}]/I$ is finite-dimensional, then $\mathcal{Q}({}_R R) \cong \mathcal{Q}$, and $R$ is a split basic algebra with basic set of idempotents $\{e_1 + I, \ldots, e_n + I\}$.

(2) If $R$ is a split basic algebra with $\mathcal{Q} = \mathcal{Q}({}_R R)$, then there is an ideal $I \leq N$ such that $K[\mathcal{Q}]/I \cong R$. [Hint: The number of arrows in $\mathcal{Q}$ is the $K$-dimension of $J/J^2$ if $J = J(R)$. Now see Exercise (1.15(4)).]

(3) If $\mathcal{Q}$ consists of one arrow and one vertex, then $K[\mathcal{Q}] \cong K[X]$.
