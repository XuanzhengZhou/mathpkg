This theorem characterizes rings that are Morita equivalent to the $n \times n$ block upper triangular matrix ring over a division ring $D$. Specifically, a ring $S$ is Morita equivalent to $\mathbb{UTM}_n(D)$ exactly when it is isomorphic to an $(m_1, \ldots, m_n)$ block upper triangular matrix ring over the same division ring $D$, where the block sizes $m_i$ are positive integers. This result connects the Morita equivalence class of block upper triangular matrix rings with the explicit structural description of rings having a complete set of $n$ primitive idempotents whose associated division rings are all isomorphic to $D$.
