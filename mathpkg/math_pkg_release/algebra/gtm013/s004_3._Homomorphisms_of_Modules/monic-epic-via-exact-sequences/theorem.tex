Let $f: M \to N$ be a homomorphism of left $R$-modules. Then:
\begin{enumerate}
  \item $f$ is monic if and only if
  $$0 \to M \xrightarrow{f} N \xrightarrow{n} \text{Coker}\, f \to 0$$
  is exact.
  \item $f$ is epic if and only if
  $$0 \to \text{Ker}\, f \xrightarrow{i} M \xrightarrow{f} N \to 0$$
  is exact.
\end{enumerate}
