Let $f: M \to N$ and $g: N' \to N$ be $R$-homomorphisms with $g$ a monomorphism. If $\text{Im}\, f \subseteq \text{Im}\, g$, then there exists a unique $R$-homomorphism $h: M \to N'$ such that
$$f = gh.$$
Moreover, $\text{Ker}\, h = \text{Ker}\, f$ and $\text{Im}\, h = g^{-1}(\text{Im}\, f)$, so that $h$ is monic if and only if $f$ is monic, and $h$ is epic if and only if $\text{Im}\, g = \text{Im}\, f$.
