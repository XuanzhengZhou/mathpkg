This follows from (16.5). For part (1), by Corollary 16.5, $Hom_R(\bigoplus U_\alpha, f)$ is epic if and only if $\prod_A Hom_R(U_\alpha, f)$ is epic, which holds if and only if each $Hom_R(U_\alpha, f)$ is epic (see Remark (6.25)). Thus $\bigoplus_A U_\alpha$ is $M$-projective precisely when each $U_\alpha$ is $M$-projective. Part (2) follows by a dual argument.
