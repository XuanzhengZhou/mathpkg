Proposition 2.7 states that if a subset $X$ spans a module $M$, then $M$ is the sum (not just the set of linear combinations) of the cyclic submodules $Rx$ generated by each $x \in X$. This provides a decomposition of any module into cyclic submodules and shows that cyclic modules are the fundamental generators in module theory.
