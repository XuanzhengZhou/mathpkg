Let $M$ be a left $R$-module and let $(M_\alpha)_{\alpha \in A}$ be a family of submodules. Then we say that the submodules $(M_\alpha)_{\alpha \in A}$ \textbf{span} $M$ if $\Sigma_A M_\alpha = M$.

If $X$ is a subset of $M$ such that

$$RX = M,$$

then $X$ is said to \textbf{span} $M$, and $X$ is called a \textbf{spanning set} for $M$.

A module with a finite spanning set is said to be \textbf{finitely spanned}. A module with a single-element spanning set is a \textbf{cyclic module}. Thus a cyclic left module is one of the form $M = R\{x\}$ where $x$ is some element of $M$, and we write

$$M = Rx = \{rx \mid r \in R\}.$$
