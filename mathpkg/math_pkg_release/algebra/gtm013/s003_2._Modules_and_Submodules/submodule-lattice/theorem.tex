If $M$ is a left $R$-module, then the set $\mathcal{S}(M)$ of all submodules of $M$ is a complete modular lattice with respect to the partial order $\leq$ (which coincides with set inclusion). In this lattice, if $\mathcal{A}$ is a non-empty set of submodules, then its join and meet are given by

$$\bigvee \mathcal{A} = \Sigma \mathcal{A} \quad \text{and} \quad \bigwedge \mathcal{A} = \bigcap \mathcal{A},$$

respectively. In particular, if $K$ and $L$ are submodules of $M$, then

$$K + L \quad \text{and} \quad K \cap L$$

are their join and meet, respectively; and if $H$ is another submodule of $M$, then

$$K \leq H \quad \text{implies} \quad H \cap (K + L) = K + (H \cap L).$$
