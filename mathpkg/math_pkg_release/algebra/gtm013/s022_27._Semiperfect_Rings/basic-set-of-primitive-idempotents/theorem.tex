Let $R$ be semiperfect with $J = J(R)$. Then every complete set of orthogonal primitive idempotents for $R$ contains a basic set. Moreover, for pairwise orthogonal primitive idempotents $e_1, \ldots, e_m \in R$, the following are equivalent:
\begin{itemize}
\item[(a)] $e_1, \ldots, e_m$ is a basic set of primitive idempotents for $R$;
\item[(b)] $R e_1, \ldots, R e_m$ is an irredundant set of representatives of the indecomposable projective left $R$-modules;
\item[(c)] $R e_1/J e_1, \ldots, R e_m/J e_m$ is an irredundant set of representatives of the simple left $R$-modules;
\item[(d)] $e_1 + J, \ldots, e_m + J$ generate the simple blocks in the block decomposition of the semisimple ring $R/J$.
\end{itemize}
Finally, these are also equivalent to the corresponding versions of (b) and (c) for right $R$-modules.
