Let $R$ be a semisimple ring. Then $R$ contains a finite set $T_1, T_2, \ldots, T_m$ of minimal left ideals which comprise an irredundant set of representatives of the simple left $R$-modules. Moreover for each such set the homogeneous components
$$\operatorname{Tr}_R(T_i) = R T_i R \quad (i = 1, 2, \ldots, m)$$
are simple artinian rings and $R$ is the ring direct sum
$$R = R T_1 R + \cdots + R T_m R.$$
Finally, $T_i$ is a simple generator for the ring $R T_i R$ and
$$R T_i R \cong \mathbb{M}_{n_i}(D_i)$$
where
$$n_i = c({}_{R T_i R} T_i) \quad \text{and} \quad D_i = \operatorname{End}({}_R T_i)$$
$(i = 1, 2, \ldots, m)$.
