Let the left $R$-module $M$ be the direct sum $M = M' \oplus M''$ of submodules $M'$ and $M''$. Then the restriction map
$$\operatorname{BiEnd}({}_R M) \xrightarrow{\operatorname{Res}} \operatorname{BiEnd}({}_R M')$$
is a ring homomorphism. Moreover,
\begin{itemize}
\item[(1)] If $M'$ generates or cogenerates $M''$, then $\operatorname{Res}$ is injective;
\item[(2)] If $M'$ generates and cogenerates $M''$, then $\operatorname{Res}$ is an isomorphism.
\end{itemize}
