The ring $R$ has a simple left generator if and only if $R$ is isomorphic to the full matrix ring $\mathbb{M}_n(D)$ for some division ring $D$ and some natural number $n$. Moreover, if ${}_R T$ is a simple left generator for $R$, then as a ring
$$R \cong \mathbb{M}_n(D)$$
where $D = \operatorname{End}({}_R T)$ and $n = c({}_R T)$, the composition length.
