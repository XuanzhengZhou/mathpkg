If $\alpha \in \operatorname{Ker} \rho$, then for each $i, j$, $\alpha_{ij} = \iota_i \rho(\alpha) \pi_j = 0$, so $\rho$ is injective. Finally, if $\gamma \in \operatorname{End}({}_R M^{(n)})$, then
$$(x \rho([\iota_k \gamma \pi_l])) \pi_j = \sum_i x \pi_i(\iota_i \gamma \pi_j) = (x \gamma) \pi_j$$
and $\rho$ is an isomorphism. $\square$
