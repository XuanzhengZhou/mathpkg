Let $(M_\alpha)_{\alpha \in A}$ be homologically independent and let $M = \bigoplus_A M_\alpha$. Prove that if each $M_\alpha$ is indecomposable and if $K$ is a non-zero direct summand of $M$, then $K = \bigoplus_B M_\beta$ for some (necessarily unique) $B \subseteq A$.

[Hint: Let $(e_\alpha)_{\alpha \in A}$ be the idempotents in $\operatorname{End}({}_R M)$ for the given decomposition. Let $e = e^2 \in \operatorname{End}({}_R M)$. Then $\alpha \neq \beta$ implies $e_\beta e e_\alpha = 0$.]
