Let $\mathcal{U}$ be a class of modules.

(1) If $M$ is in $\mathrm{Cog}(\mathcal{U})$ (or $\mathrm{FCog}(\mathcal{U})$), and if $g: M' \rightarrow M$ is a monomorphism, then $M'$ is in $\mathrm{Cog}(\mathcal{U})$ (resp. $\mathrm{FCog}(\mathcal{U})$).

(2) If $(M_\alpha)_{\alpha \in A}$ is a (finite) indexed set in $\mathrm{Cog}(\mathcal{U})$ (resp. $\mathrm{FCog}(\mathcal{U})$), then $\Pi_A M_\alpha$ is in $\mathrm{Cog}(\mathcal{U})$ (resp. $\mathrm{FCog}(\mathcal{U})$).
