Let $\theta: {}_R U_S \to {}_R V_S$ be a bimodule homomorphism. Then the following are natural transformations:

(1) $\eta: \operatorname{Hom}_R(V_S, -) \to \operatorname{Hom}_R(U_S, -)$ defined via $\eta_M = \operatorname{Hom}_R(\theta, M)$;

(2) $\nu: \operatorname{Hom}_R(-, U_S) \to \operatorname{Hom}_R(-, V_S)$ defined via $\nu_M = \operatorname{Hom}_R(M, \theta)$;

(3) $\phi: (- \otimes_R U_S) \to (- \otimes_R V_S)$ defined via $\phi_N = N \otimes_R \theta$.

Moreover, if $\theta$ is an isomorphism, then each of the above is a natural isomorphism.
