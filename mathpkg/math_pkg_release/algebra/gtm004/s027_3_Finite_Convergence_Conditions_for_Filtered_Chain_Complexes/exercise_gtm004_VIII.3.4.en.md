# Exercise 3.4 (Comparison Theorem)

Let $\varphi : C \to \bar{C}$ be a morphism of homologically finite filtered chain complexes. Show that if $\varphi_* : E_r \cong \bar{E}_r$ for some $r$, then $\varphi_* : H(C) \cong H(\bar{C})$.
