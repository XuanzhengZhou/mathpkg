# Exercise 2.5

Show that one can define an addition in $E(A, B)$ as follows: Let $B \mapsto E_1 \mapsto A$, $B \mapsto E_2 \mapsto A$ be representatives of two elements $\xi_1, \xi_2$ in $E(A, B)$. Let $\Delta_A: A \mapsto A \oplus A$ be the map defined by $\Delta_A(a) = (a, a)$, $a \in A$, and let $\nabla_B: B \oplus B \mapsto B$ be the map defined by $\nabla_B(b_1, b_2) = b_1 + b_2$, $b_1, b_2 \in B$. Define the sum $\xi_1 + \xi_2$ by

$$\xi_1 + \xi_2 = E(\Delta_A, \nabla_B)(B \oplus B \mapsto E_1 \oplus E_2 \mapsto A \oplus A).$$
