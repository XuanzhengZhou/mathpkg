# Exercise 2.6

Show that if $\alpha_1, \alpha_2 : A' \to A$, then

$$(\alpha_1 + \alpha_2)^* = \alpha_1^* + \alpha_2^* : E(A, B) \to E(A', B),$$

using the addition given in Exercise 2.5. Deduce that $E(A, B)$ admits additive inverses (without using Theorem 2.4).
