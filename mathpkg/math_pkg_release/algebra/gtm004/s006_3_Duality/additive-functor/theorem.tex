A functor $F : \mathfrak{A} \to \mathfrak{B}$ between additive categories is called an \textit{additive functor} if it satisfies any (hence all) of the following: it preserves finite products, preserves finite coproducts, or induces group homomorphisms $F : \mathfrak{A}(A, A') \to \mathfrak{B}(FA, FA')$ for all objects $A, A'$.
