A natural transformation $t : F \to G$ is a \textit{natural equivalence} (or \textit{natural isomorphism}) if $t_X : FX \to GX$ is an isomorphism for each object $X$ in $\mathfrak{C}$. We then write $F \simeq G$. The inverse $t^{-1} : G \to F$ is given by $(t^{-1})_X = (t_X)^{-1}$ and is also a natural equivalence.
