\begin{enumerate}
\item[(i)] If $P$ is projective, then any retract of $P$ is projective.
\item[(ii)] If every object of $\mathfrak{C}$ is the image of a projective object under an epimorphism, then an object $P$ is projective if and only if it is a retract of every projective object mapping epimorphically onto it.
\item[(iii)] Under the same hypothesis, every projective is a retract of a free object.
\end{enumerate}
