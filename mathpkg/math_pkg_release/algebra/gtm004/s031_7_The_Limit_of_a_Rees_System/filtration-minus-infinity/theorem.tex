\textbf{Definition.} Let $C$ be a filtered (graded) differential group. An element of $H_q(C^{(p)})$ has \textit{filtration $-\infty$} if it belongs to $I_{p,q}$, that is, if it is in the image of $H_q(C^{(r)})$ for all $r \leq p$. An element of $H_q(C^{(p)})$ is \textit{stable} if it is non-zero in every $H_q(C^{(r)})$, $r \geq p$. Similar terminology applies to $H_q(C/C^{(p)})$.
