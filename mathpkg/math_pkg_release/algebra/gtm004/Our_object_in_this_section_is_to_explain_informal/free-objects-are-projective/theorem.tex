If $F \dashv U$ is a free-forgetful adjunction and the underlying functor $U: \mathfrak{C} \to \mathfrak{S}$ sends epimorphisms to surjections, then every free object in $\mathfrak{C}$ is projective. This is the case for $\mathfrak{Ab}$, $\mathfrak{M}_{\Lambda}$, and $\mathfrak{G}$; it is false for the category of integral domains, where the inclusion $\mathbb{Z} \subseteq \mathbb{Q}$ is an epimorphism that is not surjective on underlying sets.
