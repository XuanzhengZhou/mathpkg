Given a category $\mathfrak{C}$, the \textbf{opposite category} $\mathfrak{C}^{\text{opp}}$ is defined by:

\begin{enumerate}
\item $\operatorname{Ob}(\mathfrak{C}^{\text{opp}}) = \operatorname{Ob}(\mathfrak{C})$;
\item $\mathfrak{C}^{\text{opp}}(X, Y) = \mathfrak{C}(Y, X)$ for all objects $X, Y$.
\end{enumerate}

Composition in $\mathfrak{C}^{\text{opp}}$ is induced by composition in $\mathfrak{C}$: if $f \in \mathfrak{C}^{\text{opp}}(X, Y) = \mathfrak{C}(Y, X)$ and $g \in \mathfrak{C}^{\text{opp}}(Y, Z) = \mathfrak{C}(Z, Y)$, then $g \circ_{\mathfrak{C}^{\text{opp}}} f = f \circ_{\mathfrak{C}} g$.
