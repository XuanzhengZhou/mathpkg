An inverse of an element $a$ in a set with associative operation and identity $e$ is an element $a'$ satisfying $a'a = aa' = e$. The inverse plays the role of "division" or "undoing" the operation: multiplying an element by its inverse yields the identity. The existence of inverses for all elements is the final axiom that distinguishes a group from a monoid. Once inverses exist (and are unique), negative integer exponents can be defined, extending the power notation to all integers.
