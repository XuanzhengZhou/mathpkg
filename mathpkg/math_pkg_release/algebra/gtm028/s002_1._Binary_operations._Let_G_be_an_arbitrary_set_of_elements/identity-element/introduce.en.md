An identity element $e$ in a set $G$ with a binary operation satisfies $ea = ae = a$ for all $a$ in $G$. It acts as a neutral element that leaves every other element unchanged under the operation. The identity is a central concept in group theory: a monoid is a semigroup with identity, and a group is a monoid where every element has an inverse. Once an identity exists it is unique, and it permits the definition of zero exponents via $a^0 = e$.
