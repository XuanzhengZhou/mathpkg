If a set with a binary operation has an identity element, that identity is unique. The proof is a classic short argument: if $e$ and $e'$ are both identities, then $e = ee' = e'$, where the first equality uses that $e'$ is an identity and the second that $e$ is an identity. This uniqueness is essential because it allows us to speak of "the" identity element of a group or monoid without ambiguity.
