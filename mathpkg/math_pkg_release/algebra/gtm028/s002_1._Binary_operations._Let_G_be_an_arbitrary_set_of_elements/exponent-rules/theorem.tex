Let $G$ be a set equipped with an associative binary operation written multiplicatively. For $a \in G$ and positive integers $m, n$:
\begin{enumerate}
    \item $a^m a^n = a^{m+n}$,
    \item $(a^m)^n = a^{mn}$.
\end{enumerate}
Moreover, if $a$ and $b$ are two elements of $G$ which commute, then any powers of $a$ and $b$ also commute, and
\begin{enumerate}
    \item[3.] $(ab)^n = a^n b^n$.
\end{enumerate}
If $G$ has an identity element $e$ and $a$ has an inverse $a'$, then the rules hold for all integers $m, n$ (including negative exponents, with $a^0 = e$).
