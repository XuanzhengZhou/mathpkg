Let $R$ be a ring. A set $M$ is called a module over $R$ (or an $R$-module) if

(a) $M$ is a commutative group (the group operation will be written as addition).

(b) With every ordered pair $(a, x)$ in which $a \in R$ and $x \in M$ there is associated a unique element of $M$, to be denoted by $ax$, such that the following relations hold:

(1) $a(x + y) = ax + ay$

(2) $(a + b)x = ax + bx$

(3) $(ab)x = a(bx)$,

where $a, b$ are any elements of $R$ and $x, y$ are any elements of $M$.
