In a euclidean domain, the identity element $1$ minimizes $\varphi$ among all non-zero elements, since $1$ divides every element. Associates have equal $\varphi$-values, and an element is a unit if and only if $\varphi(\epsilon) = \varphi(1)$. These properties follow directly from the monotonicity condition E1 and the divisibility properties of units and associates.
