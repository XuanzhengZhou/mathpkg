\text{Let } R \text{ be an integrally closed domain, } K \text{ its quotient field, } K' \text{ a finite algebraic extension of } K, \text{ and } R' \text{ the integral closure of } R \text{ in } K'. \text{ Suppose there exists an element } y \in R' \text{ such that } R' = R + Ry + \cdots + Ry^{n-1} \text{ where } n = [K':K], \text{ so that } R' = R[y] \text{ and } \{1, y, \dots, y^{n-1}\} \text{ is an integral basis of } R' \text{ over } R.

\text{Let } F(X) \text{ be the minimal polynomial of } y \text{ over } K, \text{ and let } \mathfrak{p} \text{ be a prime ideal in } R. \text{ Consider the factorization of } \bar{F}(X) \text{ in } (R/\mathfrak{p})[X]:

$$\bar{F}(X) = \prod_{i=1}^{g} (f_i(X))^{e_i},$$

\text{where the } f_i(X) \text{ are distinct monic irreducible polynomials in } (R/\mathfrak{p})[X]. \text{ Then the prime ideal decomposition of } \mathfrak{p}R' \text{ is:}

$$\mathfrak{p}R' = \prod_{i=1}^{g} \mathfrak{P}_i^{e_i},$$

\text{where } \mathfrak{P}_i = R'\mathfrak{p} + F_i(y)R' \text{ and each } F_i(X) \text{ is a monic polynomial in } R[X] \text{ whose reduction modulo } \mathfrak{p} \text{ is } f_i(X). \text{ The residue field degree of } \mathfrak{P}_i \text{ over } \mathfrak{p} \text{ is } f(\mathfrak{P}_i) = \deg(f_i). \text{ Moreover, } \sum_i e_i \deg(f_i) = n.
