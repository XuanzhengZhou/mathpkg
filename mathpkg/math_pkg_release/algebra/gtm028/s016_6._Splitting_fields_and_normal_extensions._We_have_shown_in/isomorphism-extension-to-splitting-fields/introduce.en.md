This is the principal uniqueness theorem for splitting fields. Given an isomorphism between two base fields and corresponding polynomials, the isomorphism lifts to an isomorphism between their splitting fields. The lifted isomorphism maps the root set of the first polynomial onto the root set of the second, establishing that splitting fields are unique up to isomorphism in the strongest possible sense.
