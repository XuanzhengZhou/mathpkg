A set $G$ which is closed under a given multiplication is called a \textbf{group} if the following conditions (group axioms) are satisfied:

\begin{itemize}
\item[G$_1$.] The set $G$ is not empty.
\item[G$_2$.] If $a, b, c \in G$, then $(ab)c = a(bc)$ (associative law).
\item[G$_3$.] There exists in $G$ an element $e$ such that:
  \begin{enumerate}
  \item[(1)] For any element $a$ in $G$, $ea = a$.
  \item[(2)] For any element $a$ in $G$, there exists an element $a' \in G$ such that $a'a = e$.
  \end{enumerate}
\end{itemize}

The element $e$ is called the \textbf{identity} of $G$, and $a'$ is called an \textbf{inverse} of $a$.
