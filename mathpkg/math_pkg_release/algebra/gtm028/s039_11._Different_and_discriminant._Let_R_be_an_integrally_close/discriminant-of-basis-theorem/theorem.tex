\text{Let } R \text{ be a Dedekind domain, } K \text{ its quotient field, } K' \text{ a finite algebraic separable extension of } K, \text{ and } R' \text{ the integral closure of } R \text{ in } K'. \text{ The discriminant } \mathfrak{d}_{R'/R} \text{ of } R' \text{ over } R \text{ is the norm of the different: } \mathfrak{d}_{R'/R} = N_{K'/K}(\mathfrak{D}_{R'/R}).

\text{For any basis } \{u_1, \dots, u_n\} \text{ of } K' \text{ over } K \text{ contained in } R', \text{ the discriminant } d(u) = \det(T(u_i u_j)) \text{ belongs to } \mathfrak{d}_{R'/R}. \text{ Moreover, } \mathfrak{d}_{R'/R} \text{ is generated by the discriminants of all such bases. In order that } \mathfrak{d}_{R'/R} = R \cdot d(u), \text{ it is necessary and sufficient that } \{u_1, \dots, u_n\} \text{ be an integral basis of } R' \text{ over } R.
