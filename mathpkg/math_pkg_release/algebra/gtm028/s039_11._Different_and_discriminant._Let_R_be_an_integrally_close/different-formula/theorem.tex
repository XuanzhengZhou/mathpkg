\text{Let } R \text{ be a Dedekind domain and } R' \text{ its integral closure in a finite algebraic separable extension } K' \text{ of } K. \text{ Let } \mathfrak{P} \text{ be a proper prime ideal in } R'. \text{ The exponent } m(\mathfrak{P}) \text{ of } \mathfrak{P} \text{ in the different } \mathfrak{D}_{R'/R} \text{ is given by:}

$$m(\mathfrak{P}) = e(\mathfrak{P}) - 1$$

\text{when the ramification is tame (i.e., } \text{char}(R/\mathfrak{p}) \nmid e(\mathfrak{P})\text{), and more generally, the differential exponent can be computed from the complementary module's local structure. The different is } \mathfrak{D}_{R'/R} = \prod_{\mathfrak{P}} \mathfrak{P}^{m(\mathfrak{P})}.
