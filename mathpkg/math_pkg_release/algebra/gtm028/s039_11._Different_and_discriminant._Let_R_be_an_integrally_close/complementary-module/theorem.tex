\text{Let } R \text{ be an integrally closed ring, } K \text{ its quotient field, } K' \text{ a finite algebraic separable extension of } K, \text{ and } R' \text{ an integral extension of } R \text{ admitting } K' \text{ as quotient field. Denote by } T = T_{K'/K} \text{ the trace mapping from } K' \text{ to } K.

\text{The \textbf{complementary module} } \mathcal{C} \text{ of } R' \text{ with respect to } R \text{ is the set of all } z \in K' \text{ such that } T(zR') \subset R. \text{ It is an } R'\text{-module, and since } R \text{ is integrally closed, } T(R') \subset R, \text{ so } \mathcal{C} \text{ contains } R'.
