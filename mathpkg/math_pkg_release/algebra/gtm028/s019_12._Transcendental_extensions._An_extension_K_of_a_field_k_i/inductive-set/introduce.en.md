An inductive set (also called a chain-complete poset) is a partially ordered set in which every totally ordered subset (chain) possesses an upper bound within the set. This property is precisely the hypothesis of Zorn's Lemma, which then guarantees the existence of at least one maximal element. In Section 12, the set of all transcendence sets extending a given one is shown to be inductive under inclusion, allowing Zorn's Lemma to prove the existence of transcendence bases.
