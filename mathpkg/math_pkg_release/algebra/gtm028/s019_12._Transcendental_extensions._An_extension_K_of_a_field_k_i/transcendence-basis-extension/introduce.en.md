Corollary 1 applies Theorem 23 to the specific case of field extensions: any transcendence set $L$ can be extended to a full transcendence basis using elements drawn from a given generating set $S$ of $K$ over $k$ (i.e., a set $S$ such that $K/k(S)$ is algebraic). The extension is disjoint from the original set $L$.
