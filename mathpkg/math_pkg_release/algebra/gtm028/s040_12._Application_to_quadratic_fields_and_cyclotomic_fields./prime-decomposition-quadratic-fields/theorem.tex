\textbf{Theorem 32.} Let $K = \mathbb{Q}(\sqrt{m})$ where $m$ is a square-free integer, and let $R$ be the ring of algebraic integers of $K$. Let $\mathfrak{d}$ be the discriminant of $R$ over $\mathbb{Z}$:
$$\mathfrak{d} = \begin{cases}
m & \text{if } m \equiv 1 \pmod{4},\\
4m & \text{if } m \equiv 2, 3 \pmod{4}.
\end{cases}$$

For an odd prime $p$:
\begin{itemize}
\item $p$ is ramified in $K$ if and only if $p \mid \mathfrak{d}$.
\item If $p \nmid \mathfrak{d}$, then $p$ is decomposed in $K$ if $\left(\frac{m}{p}\right) = 1$, and $p$ is inertial in $K$ if $\left(\frac{m}{p}\right) = -1$.
\end{itemize}

For the prime $2$: $2$ is unramified in $K$ if and only if $m \equiv 1 \pmod{4}$. If $m \equiv 1 \pmod{4}$, then $2$ is decomposed in $K$ if $(m-1)/4$ is even (i.e., $m \equiv 1 \pmod{8}$), and $2$ is inertial in $K$ if $(m-1)/4$ is odd (i.e., $m \equiv 5 \pmod{8}$).
