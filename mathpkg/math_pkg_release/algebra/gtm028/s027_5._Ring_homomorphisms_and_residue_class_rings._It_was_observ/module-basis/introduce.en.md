A basis of a module is a generating set — a collection of elements not contained in any proper submodule. A finite module (or module of finite type) is one possessing a finite generating set, while a cyclic module is generated by a single element. These concepts generalize the notions of finite-dimensional vector spaces and cyclic groups to the module setting. The explicit description of the submodule generated by a set involves finite $R$-linear combinations together with integer multiples, reflecting that modules over arbitrary rings need not be unitary.
