The radical $\sqrt{\mathfrak{A}}$ of an ideal $\mathfrak{A}$ in a ring $R$ is the set of all elements $a \in R$ such that $a^m \in \mathfrak{A}$ for some positive integer $m$. Then $\mathfrak{A} \subset \sqrt{\mathfrak{A}}$ and $\sqrt{\mathfrak{A}}$ is an ideal. Moreover:
$$\sqrt{\sqrt{\mathfrak{A}}} = \sqrt{\mathfrak{A}},$$
$$\sqrt{\mathfrak{A}\mathfrak{B}} = \sqrt{\mathfrak{A} \cap \mathfrak{B}} = \sqrt{\mathfrak{A}} \cap \sqrt{\mathfrak{B}},$$
$$\sqrt{\mathfrak{A}} = R \iff \mathfrak{A} = R,$$
$$\sqrt{\mathfrak{A} + \mathfrak{B}} = \sqrt{\sqrt{\mathfrak{A}} + \sqrt{\mathfrak{B}}}.$$
If $\mathfrak{A}^k \subset \mathfrak{B}$ for some $k$, then $\sqrt{\mathfrak{A}} \subset \sqrt{\mathfrak{B}}$.
