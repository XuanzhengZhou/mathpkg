If $\mathfrak{A}$ and $\mathfrak{B}$ are ideals of $R$, then the quotient $\mathfrak{A} : \mathfrak{B}$ consists of all elements $c$ of $R$ such that $c\mathfrak{B} \subset \mathfrak{A}$. The set $\mathfrak{A} : \mathfrak{B}$ is an ideal and contains $\mathfrak{A}$. In particular, $\mathfrak{A} : R \supset \mathfrak{A}$; but if $R$ has an identity then $\mathfrak{A} : R = \mathfrak{A}$. If $\mathfrak{C}$ is any ideal such that $\mathfrak{C}\mathfrak{B} \subset \mathfrak{A}$, then $\mathfrak{C} \subset \mathfrak{A} : \mathfrak{B}$, and conversely. If $\mathfrak{A} \supset \mathfrak{B}$, then $\mathfrak{A} : \mathfrak{B} = R$, and conversely if $R$ has an identity.
