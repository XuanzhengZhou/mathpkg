\text{Suppose } a \text{ has an inverse } a^{-1} \text{ (so } a^{-1}a = aa^{-1} = 1\text{). Assume } ab = 0 \text{ for some } b \in R. \text{ Multiply on the left by } a^{-1}:

$$a^{-1}(ab) = (a^{-1}a)b = 1 \cdot b = b.$$

\text{On the other hand, } a^{-1} \cdot 0 = 0 \text{ (since } a^{-1} \cdot 0 = 0 \text{ by the property of zero). Therefore } b = 0.

\text{Thus } a \text{ is not a left zero divisor. A symmetric argument (multiplying } ba = 0 \text{ on the right by } a^{-1}\text{) shows that } a \text{ is not a right zero divisor. Hence no unit is a zero divisor.}
