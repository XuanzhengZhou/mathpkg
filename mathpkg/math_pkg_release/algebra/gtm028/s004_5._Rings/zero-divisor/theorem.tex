\text{An element } a \text{ of a ring } R \text{ is called a }

\begin{itemize}
\item \text{\textbf{left zero divisor} if there exists } b \in R, b \neq 0, \text{ such that } ab = 0;
\item \text{\textbf{right zero divisor} if there exists } b \in R, b \neq 0, \text{ such that } ba = 0.
\end{itemize}

\text{The element } 0 \text{ is always both a left and right zero divisor whenever } R \text{ contains elements different from } 0. \text{ In the trivial case of a ring consisting only of the element zero (the nullring), it is convenient to regard } 0 \text{ as a zero divisor also.}
