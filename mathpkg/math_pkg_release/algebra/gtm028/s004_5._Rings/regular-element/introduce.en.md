A regular element in a ring is an element that is not a zero divisor. Regular elements satisfy the cancellation law: if $a$ is regular and $ab = ac$, then $b = c$. In an integral domain, all non-zero elements are regular. The terminology is particularly important in localization theory, where one inverts regular elements.
