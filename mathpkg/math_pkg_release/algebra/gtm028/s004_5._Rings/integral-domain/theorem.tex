\text{A commutative ring with identity and having no proper zero divisors is called an \textbf{integral domain}.}

\text{Equivalently, in an integral domain } R, \text{ if } ab = 0 \text{ then } a = 0 \text{ or } b = 0. \text{ In an integral domain, the cancellation law holds: if } a \neq 0 \text{ and } ab = ac, \text{ then } b = c.
