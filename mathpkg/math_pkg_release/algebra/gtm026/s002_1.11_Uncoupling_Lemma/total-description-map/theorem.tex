Let $(X, \delta)$ be an $\Omega$-algebra. The total description map $\delta^@: X\Omega \longrightarrow X$ is defined by algebraic general recursion:
$$x\delta^@ = x \quad \text{for all } x \in X,$$
$$(p_1 \cdots p_n \omega)\delta^@ = (p_1\delta^@, \ldots, p_n\delta^@)\delta_\omega \quad \text{for all } \omega \in \Omega_n,\ p_i \in X\Omega.$$
