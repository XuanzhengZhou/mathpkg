Proposition 3.25 establishes that the set of isolated elements of a $T$-algebra is contained in every generating set. This is immediate from the definition: if $q$ is isolated and $R$ generates the algebra, then $q$ must belong to $R$, since otherwise $R \subseteq Q - \{q\}$ would generate $q$, contradicting isolation. Consequently, the isolated elements form the unique minimal generating set whenever they themselves generate the algebra -- a property that characterizes extremal-state algebras (Definition 3.26).
