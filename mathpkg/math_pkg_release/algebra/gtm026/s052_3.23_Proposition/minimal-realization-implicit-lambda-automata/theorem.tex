Let $(E, M)$ be an image factorization system in $\mathcal{K}$ such that $X^{@}T$ preserves $E$. Let $I, (Y, \theta)$ be fixed. Then for every $f : IX^{@} \longrightarrow Y$ there exists an implicit $\lambda$-automaton $\bar{M} = (\bar{Q}, \bar{\delta}, \xi, \tau, \bar{\beta})$ such that:
\begin{enumerate}
\item the response of $\bar{M}$ is $f$,
\item the reachability map $r^{\#} : IX^{@}T \longrightarrow \bar{Q}$ is in $E$, and
\item the observability map $\sigma : Q \longrightarrow YX^{@}$ is in $M$.
\end{enumerate}
If $\bar{M}'$ also satisfies these three conditions then $\bar{M}$ and $\bar{M}'$ are isomorphic (i.e., there exists an isomorphism $\psi : (\bar{Q}, \bar{\delta}, \xi) \longrightarrow (\bar{Q}', \bar{\delta}', \xi')$ of $\lambda$-algebras such that $\tau.\psi = \tau'$, $\psi.\bar{\beta}' = \bar{\beta}$).
