Let $(X, d)$ be a metric space and let $x \in X$. The map
$$d(x, -): X \longrightarrow \mathbb{R}, \qquad y \longmapsto d(x, y)$$
where $\mathbb{R}$ is equipped with the usual metric $d_{\mathbb{R}}(a, b) = |a - b|$, is distance decreasing. That is,
$$|d(x, y) - d(x, z)| \leq d(y, z)$$
for all $y, z \in X$.
