(McShane's Theorem) $\mathbb{R}$ with the usual metric $d_{\mathbb{R}}(a, b) = |a - b|$ is an injective object in the category of metric spaces and distance-decreasing maps. That is, for any metric space $(X, d)$, any subset $A \subseteq X$, and any distance-decreasing map $f: A \to \mathbb{R}$, there exists a distance-decreasing extension $g: X \to \mathbb{R}$ such that $g|_A = f$.

The extension is given explicitly by the formula
$$g(x) = \sup\{\, f(a) - d(x, a) : a \in A \,\}.$$
