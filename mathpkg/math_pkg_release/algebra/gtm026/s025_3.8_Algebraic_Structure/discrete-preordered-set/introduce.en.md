A preordered set is discrete when its preorder relation coincides with equality. In a discrete preorder, $x \leqslant y$ forces $x = y$, so the only comparisons possible are trivial self-comparisons. This concept formalizes the situation where the preorder carries no additional information beyond the underlying set structure, and it serves as the fiber-wise condition defining discretely ordered structured categories.
