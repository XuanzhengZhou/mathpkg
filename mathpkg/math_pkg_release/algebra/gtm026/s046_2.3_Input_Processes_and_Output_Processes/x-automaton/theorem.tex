An \textbf{$X$-automaton} is a sextuple $(Q, \delta, I, \tau, Y, \beta)$ where
\begin{itemize}
\item $(Q, \delta)$ is an $X$-dynamics,
\item $I$ is an object of $\mathcal{H}$ (the \textit{initial state object}),
\item $Y$ is an object of $\mathcal{H}$ (the \textit{output object}),
\item $\tau: I \to Q$ is an $\mathcal{H}$-morphism (the \textit{initial state}),
\item $\beta: Q \to Y$ is an $\mathcal{H}$-morphism (the \textit{output morphism}).
\end{itemize}
