The response map is the composite $r \circ \beta: IX^{\sharp} \to Y$ of the reachability map with the output morphism. It captures the external input-output behavior of an automaton: given an element of the free input object $IX^{\sharp}$ (encoding both the initial configuration and the sequence of inputs), it produces the resulting output. The response map abstracts the classical notion of the function computed by a sequential machine.
