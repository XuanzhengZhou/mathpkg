A group can be defined equivalently using a single binary division operation $d$ satisfying a single equation. This remarkable formulation, due to Higman and Neumann (1952), shows that the entire group structure — including the identity element and inverses — can be encoded in one binary operation with one equational axiom. A bijective passage exists between this single-operation formulation and the standard formulation with multiplication, inversion, and unit. However, the empty set satisfies this definition (unlike the standard definition), so a modification adding a distinguished element $e$ with $xed = x$ may be needed to exclude the empty group.
