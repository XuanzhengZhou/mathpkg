If $U: \mathcal{A} \longrightarrow \mathcal{K}$ creates limits then the following statements are true:
\begin{enumerate}
\item $(\mathcal{A}, U) \in \text{Struct}(\mathcal{K})$ and is discretely ordered;
\item If $\mathcal{K}$ is locally small then $\mathcal{A}$ is locally small;
\item If $\mathcal{K}$ is well powered and has kernel pairs then $\mathcal{A}$ is well powered.
\end{enumerate}
