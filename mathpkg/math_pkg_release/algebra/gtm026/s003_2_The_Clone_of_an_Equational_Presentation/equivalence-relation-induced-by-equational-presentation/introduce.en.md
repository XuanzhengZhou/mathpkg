Given an equational presentation $(\Omega, E)$, two $\Omega$-terms $p, q \in A\Omega$ are considered equivalent if they have the same interpretation in every $(\Omega, E)$-algebra under every assignment of the variables in $A$. This defines an equivalence relation $E_A$ on the set of all $\Omega$-terms with variables from $A$. The quotient set $AT = A\Omega / E_A$ consists of equivalence classes of terms, and these classes are presentation-independent in a precise sense. This construction is fundamental for defining the clone and free algebras associated with an equational presentation.
