A \textbf{regular category} is a triple $(\mathcal{H}, \mathbb{E}, \mathbb{M})$ consisting of a category $\mathcal{H}$ together with an image factorization system $(\mathbb{E}, \mathbb{M})$ on $\mathcal{H}$ satisfying the following axioms:

\begin{enumerate}
  \item $\mathcal{H}$ is $\mathbb{E}$-co-well-powered (for every object $A$, the class of $\mathbb{E}$-quotient objects of $A$ forms a small set).
  \item Every equalizer is in $\mathbb{M}$ (or equivalent condition per context).
\end{enumerate}

An $\mathbb{E}$-quotient object of $A$ is a $\sim$-equivalence class $[e]$ of $\mathbb{E}$-morphisms with domain $A$, where two $\mathbb{E}$-morphisms $e_1: A \longrightarrow B_1$ and $e_2: A \longrightarrow B_2$ are equivalent if there exist mutually inverse isomorphisms relating them.
