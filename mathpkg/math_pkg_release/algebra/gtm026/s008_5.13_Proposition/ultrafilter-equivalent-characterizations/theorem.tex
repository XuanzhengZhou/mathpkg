Let $\mathcal{U} \subset 2^X$ be a filter on $X$. The following are equivalent:

\begin{enumerate}
\item $\mathcal{U}$ is an ultrafilter: for all $A \subset X$, $A \in \mathcal{U}$ or $X - A \in \mathcal{U}$.
\item If $A_1 \cup \cdots \cup A_n \in \mathcal{U}$ then some $A_i \in \mathcal{U}$.
\item If $A_1 \cap \cdots \cap A_n$ has nonempty intersection with every element of $\mathcal{U}$ then some $A_i \in \mathcal{U}$.
\item $\mathcal{U}$ cannot be extended to a larger collection with the finite intersection property and $\mathcal{U}$ is nonempty.
\item If $A$ is a subset of $X$ and $A \notin \mathcal{U}$ then $A$ has empty intersection with some element of $\mathcal{U}$.
\end{enumerate}
