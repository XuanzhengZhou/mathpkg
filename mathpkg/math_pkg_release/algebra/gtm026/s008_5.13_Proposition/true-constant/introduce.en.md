A true constant in an algebraic theory $\mathbf{T}$ is an operation symbol $\omega$ whose arity is zero, meaning it requires no arguments. True constants correspond to nullary operations, which single out a distinguished element in each algebra. The terminology distinguishes syntactic constants (arity zero) from semantic constants (operations that happen to be constant in every algebra).
