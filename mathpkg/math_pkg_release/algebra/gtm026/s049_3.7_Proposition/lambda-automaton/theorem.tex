A $\lambda$-automaton with respect to an algebraic theory $\mathbf{T}$ in $\mathcal{K}$, a functor $X \colon \mathcal{K} \longrightarrow \mathcal{K}$, and a distributive law $\lambda \colon TX \longrightarrow XT$, is a $7$-tuple
$$
M = (Q, \delta, I, \tau, Y, \theta, \beta)
$$
where:
\begin{itemize}
\item $Q$ is the state object (in $\mathcal{K}$),
\item $\delta \colon QX \longrightarrow QT$ is the transition morphism,
\item $I$ is the input object,
\item $\tau \colon I \longrightarrow Q$ is the initial state morphism,
\item $(Y, \theta)$ is a $\mathbf{T}$-algebra (the output algebra),
\item $\beta \colon Q \longrightarrow Y$ is the output morphism.
\end{itemize}
