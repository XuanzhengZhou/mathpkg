A $\lambda$-automaton generalizes the classical notion of a deterministic automaton to the setting of fuzzy (algebraic) theories. Given an algebraic theory $\mathbf{T}$, an input functor $X$, and a distributive law $\lambda \colon TX \to XT$ encoding how inputs interact with the theory structure, a $\lambda$-automaton consists of a state object $Q$, a transition morphism $\delta$, an initial state $\tau$, and an output $\beta$ into a $\mathbf{T}$-algebra $(Y,\theta)$. This framework unifies probabilistic automata, tree automata, and linear systems under a single categorical definition, with the distributive law providing the rule for propagating inputs through the algebraic structure of the state transitions.
