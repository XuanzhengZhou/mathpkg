If $X$ is an input process in $\mathcal{K}$ then the algebraic theory in $\mathcal{K}$ induced by the adjointness

$$\text{Dyn}(X) \longleftrightarrow \mathcal{K}$$

is $X^@ = (X^@, \eta, \mu)$ where $\mu = \mu_0^{\#}$. The semantics comparison functor

$$\Phi: \text{Dyn}(X) \longrightarrow \mathcal{K}^{X^@}$$

is given by $(Q, \delta)\Phi = (Q, \delta^{\#})$ where $\delta^{\#}$ is the run map of $(Q, \delta)$.
