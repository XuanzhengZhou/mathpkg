Theorem 2.15 provides a converse to the statement that every input process has a free algebraic theory. It asserts that in a locally small, well-powered category $\mathcal{K}$ possessing small limits, any endofunctor $X: \mathcal{K} \to \mathcal{K}$ that admits a free algebraic theory must be an input process. The proof uses the general adjoint functor theorem to construct free $X$-dynamics, showing that a one-element solution set suffices.
