Let $X : \mathcal{K} \longrightarrow \mathcal{K}$ be a process and let $M = (Q, \delta, I, \tau, Y, \beta)$ be an $X$-automaton, where $Q$ is the state object, $\delta: QX \to Q$ is the dynamics, $I$ is the input object, $\tau: I \to Q$ is the input map, $Y$ is the output object, and $\beta: Q \to Y$ is the output map. A morphism of automata consists of morphisms between the corresponding components that commute with the structure maps $\delta$, $\tau$, and $\beta$. This defines a category of $\mathcal{K}$-objects with structure.

If $(E, M)$ is an image factorization system in $\mathcal{K}$ then, if $X$ is an input process, $M$ is reachable if $r: IX^@ \longrightarrow Q$ is in $E$; if $X$ is an output process, $M$ is observable if $\sigma: Q \longrightarrow YX^@$ is in $M$.
