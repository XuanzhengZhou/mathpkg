Given an operator domain $\Omega = (\Omega_n)_{n \in \mathbb{N}}$, an $\Omega$-algebra is a set $X$ together with an interpretation of each $n$-ary operator label $\omega \in \Omega_n$ as a concrete $n$-ary operation $\delta_\omega : X^n \to X$ on $X$. An $\Omega$-homomorphism between two $\Omega$-algebras is a function $f : X \to Y$ that commutes with all the operations: applying an operation in the source algebra and then mapping via $f$ yields the same result as first mapping each argument via $f$ and then applying the corresponding operation in the target algebra. This generalizes the familiar notion of a group homomorphism to arbitrary operator domains.
