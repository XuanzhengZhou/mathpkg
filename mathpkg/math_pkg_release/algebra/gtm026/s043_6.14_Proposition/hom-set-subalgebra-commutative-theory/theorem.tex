Let $\mathbf{T}$ be an algebraic theory in $\mathbf{Set}$ and let $(Y, \theta)$ be a completely commutative $\mathbf{T}$-algebra. Then for any $\mathbf{T}$-algebra $(X, \xi)$, the set
$$A = \{\, f \colon X \longrightarrow Y \mid f \text{ is a } \mathbf{T}\text{-homomorphism } (X, \xi) \longrightarrow (Y, \theta) \,\}$$
is a $\mathbf{T}$-subalgebra of the power algebra $(Y, \theta)^X$.
