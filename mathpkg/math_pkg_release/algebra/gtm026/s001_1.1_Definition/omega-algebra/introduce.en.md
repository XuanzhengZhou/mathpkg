An $\Omega$-algebra is the concrete realization of an operator domain: given a set $X$ and an operator domain $\Omega$, one assigns to each operation label $\omega \in \Omega_n$ an actual $n$-ary function on $X$. This is the fundamental notion of an algebraic structure in universal algebra. For example, a group (in the sense of 1.1) is an $\Omega$-algebra for the operator domain with $\Omega_0 = \{e\}$, $\Omega_1 = \{i\}$, $\Omega_2 = \{m\}$, where $\delta_e$ picks the unit element, $\delta_i$ is the inversion map, and $\delta_m$ is the group multiplication.
