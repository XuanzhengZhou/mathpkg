For the "nonempty subset" theory $\mathbf{T}$ acting on a set $X$, there is a canonical natural transformation $Q\lambda: QT \times X \to (Q \times X)T$ that sends a pair consisting of a nonempty subset $A \subseteq Q$ and an input element $x \in X$ to the product set $A \times \{x\}$. This map is the key structural ingredient that allows one to lift a multiple-branching (nondeterministic) dynamics $\delta: Q \times X \to QT$ to an ordinary (deterministic) dynamics on the set $QT$ of nonempty subsets of states.
