This theorem describes how the zero set $M_\omega$ of a pseudopolynomial $\omega(u, \mathfrak{z})$ of degree $s$ forms an $s$-sheeted branched covering over the parameter space, with the discriminant locus $D_\omega$ as the branch set. Away from $D_\omega$, the roots $f_1(\mathfrak{z}), \ldots, f_s(\mathfrak{z})$ are distinct holomorphic functions, giving $s$ unbranched sheets. Above points of $D_\omega$, multiple roots appear and the number of distinct points in the fiber drops below $s$. This provides the local description of analytic sets as branched coverings, a fundamental structural result in several complex variables.
