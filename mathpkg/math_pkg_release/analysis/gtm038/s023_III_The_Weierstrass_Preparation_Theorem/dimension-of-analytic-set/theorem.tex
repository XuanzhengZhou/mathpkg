The number thus obtained is denoted by $\dim_{\mathbb{R}}(M)$. The complex dimension of $M$ is
$$\dim_{\mathbb{C}}(M) := \frac{1}{2} \dim_{\mathbb{R}}(M).$$

If $M = \bigcup_{i \in \mathbb{N}} M_i$ is the decomposition of an arbitrary analytic set into irreducible components, then we define
$$\dim_{\mathbb{C}}(M) := \max_{i \in \mathbb{N}} \dim_{\mathbb{C}}(M_i).$$
In particular, if $\dim_{\mathbb{C}}(M_i) = k$ for all $i \in \mathbb{N}$, then we say that $M$ is of pure dimension $k$.
