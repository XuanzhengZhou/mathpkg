If $M$ is an irreducible analytic set in $G$ and $f$ a holomorphic function on $G$ with $f|M \neq 0$, then
$$\dim_{\mathbb{C}}(M \cap \{ \mathfrak{z} : f(\mathfrak{z}) = 0 \}) = \dim_{\mathbb{C}}(M) - 1.$$
For every irreducible component $N \subset M \cap \{ \mathfrak{z} : f(\mathfrak{z}) = 0 \}$ we have $\dim_{\mathbb{C}}(N) = \dim_{\mathbb{C}}(M) - 1$.
