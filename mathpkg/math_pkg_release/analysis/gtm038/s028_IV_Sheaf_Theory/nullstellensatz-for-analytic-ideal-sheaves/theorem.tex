Let $\mathcal{I}(A)$ be the ideal sheaf of an analytic set $A \subset B$. Then
$$
N(\mathcal{I}(A)) = \operatorname{Supp}(\mathcal{O}/\mathcal{I}(A)) = A,
$$
where $N(\mathcal{I}(A))$ denotes the zero set of the ideal sheaf.

However, for an arbitrary ideal sheaf $\mathcal{J} \subset \mathcal{O}$, the reverse equality $\mathcal{I}(N(\mathcal{J})) = \mathcal{J}$ is false in general.
