Let $x_0 \in A$ be given, $U(x_0) \subset X$ a neighborhood and $\varphi: (U, \mathcal{H}) \to (\hat{B}, \mathcal{O})$ an isomorphism. $A \cap U$ is singularity free of codimension $d$ if and only if $\tilde{\varphi}(A \cap U)$ is a regular analytic set in $\hat{B}$ of complex dimension $n - d$. This is equivalent to the existence, for every $z \in \tilde{\varphi}(A \cap U)$, of a neighborhood $V(z) \subset \hat{B}$ and an isomorphism $\psi: (V, \mathcal{O}) \to (B, \mathcal{O})$ such that
$$\tilde{\psi}(\tilde{\varphi}(A \cap U) \cap V) = \{w \in B: w_1 = \cdots = w_d = 0\}.$$
Then $\varphi_1 := \psi \circ \varphi: (\tilde{\varphi}^{-1}(V(\tilde{\varphi}(x_0))), \mathcal{H}) \to (B, \mathcal{O})$ provides the required local isomorphism.
