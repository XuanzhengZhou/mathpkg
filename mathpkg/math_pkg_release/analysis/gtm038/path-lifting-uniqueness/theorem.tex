Let $(G_1, \pi_1), (G_2, \pi_2)$ be domains over $\mathbb{C}^n$, $\psi_\lambda : [0, 1] \rightarrow G_\lambda$ be paths with $\pi_1 \circ \psi_1 = \pi_2 \circ \psi_2$. Additionally, let $f_\lambda$ be holomorphic on $G_\lambda$, $\lambda = 1, 2$. If $(f_1)_{\psi_1(0)} = (f_2)_{\psi_2(0)}$, then also $(f_1)_{\psi_1(1)} = (f_2)_{\psi_2(1)}$.
