For a domain $G \subset \mathbb{C}^n$ with $C^2$ boundary, the **Levi condition** is satisfied at a point $\mathfrak{z}_0 \in \partial G$ if there is an open neighborhood $U = U(\mathfrak{z}_0)$ and a function $\varphi$ on $U$ which satisfies the conditions required by Def. 2.4 so that at $\mathfrak{z}_0$ the function $\varphi$ satisfies

$$L_{\varphi}(\mathfrak{w}) := \sum_{i,j=1}^{n} \varphi_{z_i \bar{z}_j}(\mathfrak{z}_0) w_i \bar{w}_j \geq 0$$

for all $\mathfrak{w} \in T_{\mathfrak{z}_0}^{1,0}(\partial G)$.
