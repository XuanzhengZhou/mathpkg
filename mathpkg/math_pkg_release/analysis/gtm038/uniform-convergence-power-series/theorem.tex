Let $M$ be a subset of $\mathbb{C}^n$, $z_0 \in M$, $f$ a complex function on $M$. One says that the power series $\sum_{v=0}^{\infty} a_v(z - z_0)^v$ converges uniformly on $M$ to $f(z)$ if for each $\varepsilon > 0$ there is a finite set $I_0 \subset \Im$ such that for any finite set $I$ with $I_0 \subset I \subset \Im$,
$$\left| \sum_{v \in I} a_v(z - z_0)^v - f(z) \right| < \varepsilon$$
for all $z \in M$.
