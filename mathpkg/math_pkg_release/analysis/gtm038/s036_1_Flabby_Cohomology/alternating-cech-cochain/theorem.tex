An $\ell$-dimensional (alternating) cochain over $\mathfrak{U}$ with values in $\mathcal{S}$ is a mapping
$$\xi: I^{\ell+1} \rightarrow \bigcup_{(\iota_0, \dots, \iota_\ell)} \Gamma(U_{\iota_0 \dots \iota_\ell}, \mathcal{S})$$
with the following properties:

1. $\xi(\iota_0, \dots, \iota_\ell) \in \Gamma(U_{\iota_0 \dots \iota_\ell}, \mathcal{S})$.
2. $\xi(\iota_{\sigma(0)}, \dots, \iota_{\sigma(\ell)}) = \operatorname{sgn}(\sigma)\xi(\iota_0, \dots, \iota_\ell)$ for $\sigma \in \mathfrak{S}_{\ell+1}$.

The set $C^{\ell}(\mathfrak{U}, \mathcal{S})$ of all $\ell$-dimensional alternating cochains becomes an $R$-module via
$$(\xi_1 + \xi_2)(\iota_0, \dots, \iota_\ell) := \xi_1(\iota_0, \dots, \iota_\ell) + \xi_2(\iota_0, \dots, \iota_\ell)$$
and
$$(r \cdot \xi)(\iota_0, \dots, \iota_\ell) := r \cdot \xi(\iota_0, \dots, \iota_\ell).$$
