If $c_1, \ldots, c_n$ are arbitrary complex numbers then there exists exactly one tangent vector $D \in T_{x_0}$ with
$$D(f) = \sum_{v=1}^{n} c_v \frac{\partial}{\partial x_v}(f)$$
for each function $f$ holomorphic at $x_0$. In particular, a given tangent vector $D$ is already uniquely determined by its values on the holomorphic functions. In local coordinates $D$ has the representation
$$D = \sum_{v=1}^{n} \operatorname{Re}(D(z_v)) \frac{\partial}{\partial x_v} + \sum_{v=1}^{n} \operatorname{Im}(D(z_v)) \frac{\partial}{\partial y_v}.$$
