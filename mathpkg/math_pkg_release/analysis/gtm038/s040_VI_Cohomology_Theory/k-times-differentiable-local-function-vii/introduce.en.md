A $k$-times differentiable local function at a point $x_0$ on a complex manifold $X$ is defined via a local chart: pulling back the function through a biholomorphic coordinate map yields a function on $\mathbb{C}^n$ that is $k$-times differentiable in the usual sense. This definition is independent of the choice of coordinate system because biholomorphic transition maps are $C^\infty$ and in particular $k$-times differentiable for every $k$. The set $\mathcal{D}_{x_0}^k$ forms a vector space with natural addition and scalar multiplication.
