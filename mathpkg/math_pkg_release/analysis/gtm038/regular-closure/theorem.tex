Let $Y$ be a compact complex manifold containing $\mathbb{C}^n$ as an open dense subset. $Y$ is called a regular closure of $\mathbb{C}^n$ if every polynomial defined on $\mathbb{C}^n$ extends to a meromorphic function on $Y$. If $Y$ is regular, then $Y - \mathbb{C}^n$ is an analytic set of codimension 1.
