Let $f$ be the bijection provided by Theorem 19.2, so that $f(E)$ is null iff $E$ is of first category, and $f^{-1}(F)$ is of first category iff $F$ is null. Since $f$ is a bijection, it preserves all pure set-theoretic properties (cardinality, disjointness, inclusion, etc.). If $P$ holds for nullsets, then applying $f^{-1}$ to all sets mentioned in $P$ yields that $P^*$ holds for sets of first category. Conversely, if $P^*$ holds for first category sets, applying $f$ yields that $P$ holds for nullsets.
