The closure of a set $A$ is the smallest closed set containing $A$, equivalently the intersection of all closed supersets of $A$. It is denoted $\bar{A}$ or $A^{-}$. The closure plays a central role in defining density (a set is dense if its closure is the whole space) and nowhere denseness (a set is nowhere dense if its closure has empty interior).
