A transcendental number is a real number that is not algebraic --- it satisfies no non-zero polynomial equation with integer coefficients. The existence of transcendental numbers was first proved by Liouville in 1844 via an explicit construction; Cantor later gave a non-constructive proof using countability. Famous examples include $e$ and $\pi$.
