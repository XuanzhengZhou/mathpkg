A real-valued function is called measurable if the preimage of every open set is a measurable set. This definition extends the notion of measurability from sets to functions. Equivalently, one may require $f^{-1}(B)$ to be measurable for every Borel set $B$, or restrict attention to a countable base of the topology of $\mathbb{R}$. The indicator function $\chi_E$ of a set $E \subset \mathbb{R}$ is measurable precisely when $E$ is measurable.
