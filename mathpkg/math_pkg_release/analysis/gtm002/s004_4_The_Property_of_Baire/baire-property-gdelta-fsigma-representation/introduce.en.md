Theorem 4.4 refines the characterization of sets with the property of Baire by relating them to the Borel hierarchy. Any Baire-property set can be written as the disjoint union of a $G_\delta$ set and a set of first category, providing a canonical decomposition. The proof exploits the fact that every set of first category is contained in an $F_\sigma$ set of first category (since the closure of a nowhere dense set is nowhere dense). The $F_\sigma$ representation follows by taking complements. Note the role reversal compared to measurable sets, where $G_\delta$ and $F_\sigma$ appear in the opposite roles (Theorem 3.15).
