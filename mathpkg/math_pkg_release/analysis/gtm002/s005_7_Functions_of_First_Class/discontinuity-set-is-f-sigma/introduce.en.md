Theorem 7.1 establishes that for any real-valued function on $\mathbb{R}$, the set of points where the function is discontinuous is always an $F_\sigma$ set. The proof follows directly from the representation $D = \bigcup_{n=1}^{\infty} \{x : \omega(x) \geq 1/n\}$ together with the fact that each set $\{x : \omega(x) < \varepsilon\}$ is open, making its complement closed. This structural result is the starting point for the classification of discontinuity sets.
