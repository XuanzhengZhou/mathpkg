A Baire first class function (or function of Baire class 1) is a pointwise limit of a sequence of continuous functions. These functions form a natural hierarchy above continuous functions. Simple examples, such as $f_n(x) = \max(0, 1 - n|x|)$ converging to the indicator of $\{0\}$, show that first class functions need not be continuous. Baire's theorem (Theorem 7.3) reveals that they cannot be too discontinuous: their discontinuities form only a first category set.
