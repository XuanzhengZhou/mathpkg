Theorem 7.2 provides the converse to Theorem 7.1: every $F_\sigma$ set can be realized as the set of points of discontinuity of some bounded function. The constructive proof writes $E = \bigcup F_n$ with nested closed sets $F_n$ and builds $f$ as a uniformly convergent series $\sum a_n f_n(x)$ where each $f_n$ is discontinuous exactly on $F_n$. This shows that the $F_\sigma$ condition completely characterizes possible discontinuity sets of arbitrary real functions.
