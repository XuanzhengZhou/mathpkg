This is an immediate consequence of Theorem 7.1 and the fact that an $F_\sigma$ set is of first category if and only if its complement is dense.

Indeed, by Theorem 7.1, the set $D$ of points of discontinuity of $f$ is an $F_\sigma$ set. An $F_\sigma$ set is of first category if and only if its complement is dense (since a closed set is nowhere dense exactly when its complement is dense, and a countable union of nowhere dense sets — i.e., a first category $F_\sigma$ set — has dense complement by the Baire category theorem; conversely, if an $F_\sigma$ set has dense complement, write it as a countable union of closed sets, each of which must have dense complement and hence be nowhere dense). Therefore $D$ is of first category if and only if $\mathbb{R} - D$ is dense, i.e., if and only if $f$ is continuous at a dense set of points.
