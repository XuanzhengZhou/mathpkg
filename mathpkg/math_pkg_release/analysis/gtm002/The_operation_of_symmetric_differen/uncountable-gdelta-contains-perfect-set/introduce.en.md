Lemma 5.1 is a structural result about uncountable $G_\delta$ sets in the real line: every such set contains a closed, nowhere dense, measure-zero subset that maps continuously onto the unit interval. This lemma is crucial for Bernstein's construction of a set that is simultaneously non-measurable and lacks the property of Baire.
