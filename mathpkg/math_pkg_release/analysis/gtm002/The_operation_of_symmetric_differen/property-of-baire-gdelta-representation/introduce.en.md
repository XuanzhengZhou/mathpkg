Theorem 4.4 refines the characterization of sets with the property of Baire: every such set can be expressed as the disjoint union of a $G_\delta$ set and a set of first category. Equivalently, by complementation, it can be written as an $F_\sigma$ set minus a set of first category. Note the reversal of roles of $F_\sigma$ and $G_\delta$ compared to the analogous characterization of measurable sets.
