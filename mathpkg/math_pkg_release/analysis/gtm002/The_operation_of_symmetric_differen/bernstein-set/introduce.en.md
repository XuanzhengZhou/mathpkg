A Bernstein set is a subset of the real line such that both the set and its complement intersect every uncountable closed set. Constructed by Felix Bernstein in 1908 using the well-ordering principle, Bernstein sets exhibit extreme pathological behavior: every measurable subset is a nullset and every subset with the property of Baire is of first category.
