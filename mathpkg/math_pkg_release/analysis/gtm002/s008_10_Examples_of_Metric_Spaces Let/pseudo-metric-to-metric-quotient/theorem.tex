Such a space [a pseudo-metric space] can always be made into a metric space by identifying points $x$ and $y$ whenever $\varrho(x, y) = 0$. It is easy to verify that this defines an equivalence relation in $X$, and that the value of $\varrho(x, y)$ depends only on the equivalence classes to which $x$ and $y$ belong. If we take these classes as the elements of a set $\tilde{X}$, then $(\tilde{X}, \varrho)$ is a metric space.
