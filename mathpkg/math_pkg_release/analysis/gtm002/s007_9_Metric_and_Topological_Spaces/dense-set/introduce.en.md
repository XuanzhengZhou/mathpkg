A set $A$ is dense in $X$ if its closure equals the whole space, equivalently if every non-empty open set contains at least one point of $A$. Density is a central concept in topology and analysis, particularly in the Baire category theorem where the complement of a first-category set is shown to be dense in a complete metric space.
