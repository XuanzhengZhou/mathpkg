Let $f$ be any one-to-one mapping of the line onto a Lusin set $L$. Such a mapping exists because a Lusin set has cardinality $\mathfrak{c}$, and any set of cardinality $\mathfrak{c}$ can be put in one-to-one correspondence with the real line. If $E \subseteq \mathbb{R}$ is uncountable, then $f(E)$ is an uncountable subset of the Lusin set $L$. By definition of a Lusin set, every uncountable subset of $L$ is of second category. Hence $f(E)$ is of second category.
