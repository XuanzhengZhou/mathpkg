Proposition 20.3* is the measure-theoretic dual of Proposition 20.3. It states that any linear set of positive outer measure can be decomposed into continuum-many disjoint subsets, each having positive outer measure. A consequence is that any set of positive measure contains continuum-many disjoint non-measurable subsets, which follows from Theorem 5.5 (every set of positive outer measure contains a non-measurable subset).
