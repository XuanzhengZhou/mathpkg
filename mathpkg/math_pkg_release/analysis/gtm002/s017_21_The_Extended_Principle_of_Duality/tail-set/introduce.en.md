A tail set in the product space $X = \{0,1\}^\mathbb{N}$ is a subset whose membership depends only on the infinite tail of a sequence: changing any finite number of coordinates does not affect membership. Equivalently, a tail set factors through every finite initial segment, meaning it can be written as $X^n \times B_n$ for every $n$. Tail sets are the natural domain for zero-one laws, both in probability theory (Kolmogorov) and in descriptive set theory (Baire category), as examined in Theorem 21.4.
