In a complete separable metric space without isolated points, if an automorphism is transitive (has a dense full orbit), then the set of points whose forward iterates alone are dense is actually a residual set — that is, its complement is of first category. This strengthening is proved by expressing the set as a countable intersection of dense open sets constructed from a countable base and the backward iterates of the transformation.
