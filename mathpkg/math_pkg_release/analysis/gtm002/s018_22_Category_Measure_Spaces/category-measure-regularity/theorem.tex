Let $\mu$ be a category measure in a regular Baire space $X$.

\begin{enumerate}
\item For any open set $G$ and $\varepsilon > 0$ there is a closed set $F$ such that $F \subset G$ and $\mu(F) > \mu(G) - \varepsilon$.
\item For any closed set $F$ and $\varepsilon > 0$ there is an open set $G$ such that $F \subset G$ and $\mu(G) < \mu(F) + \varepsilon$.
\end{enumerate}
