If $\mu$ is a category measure in a regular Baire space $X$, then for any set $E$ having the property of Baire,
$$\mu(E) = \mu(\overline{E}) = \mu(E'^{-'})$$
and
$$\mu(E) = \inf\{\mu(G) : E \subset G,\; G \text{ regular open}\} = \sup\{\mu(F) : F \subset E,\; F \text{ regular closed}\}.$$
Every set $E$ with the property of Baire can be written uniquely (modulo first category) as $G \triangle P$, where $G$ is regular open and $P$ is of first category. Writing $G = \phi(E)$, the map $\phi$ selects a representative from each equivalence class of $S$ modulo sets of first category.
