In the space $M$ of measure-preserving automorphisms of the unit square, the set $P$ of transformations under which every point in some basic open set is periodic is of first category. The proof constructs $P_{ij}$ as the set of automorphisms for which all points of the basic open set $U_i$ have period dividing $j$. By composing any $T \in P_{ij}$ with an irrational rotation supported on a small annulus, one obtains an automorphism arbitrarily close to $T$ that belongs to $M \setminus P_{ij}$, showing each $P_{ij}$ is nowhere dense. Consequently, almost every measure-preserving automorphism (in the Baire category sense) has nonperiodic points in every nonempty open set.
