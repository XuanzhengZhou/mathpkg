For any fixed $x$, the oscillation of $f$ at $x$ is defined by
$$\omega(x) = \lim_{\delta \to 0} \omega((x - \delta, x + \delta)),$$
where $\omega((x - \delta, x + \delta))$ is the oscillation of $f$ on the interval $(x - \delta, x + \delta)$. The function $\omega((x - \delta, x + \delta))$ decreases with $\delta$, so the limit exists in the extended real numbers. $\omega(x_0) = 0$ if and only if $f$ is continuous at $x_0$; when $\omega(x_0) > 0$, it measures the size of the discontinuity of $f$ at $x_0$.
