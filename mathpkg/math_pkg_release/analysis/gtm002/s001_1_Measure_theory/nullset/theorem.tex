The length of any interval $I$ is denoted by $|I|$. A set $A \subset \mathbb{R}$ is called a \textit{nullset} (or a \textit{set of measure zero}) if for each $\varepsilon > 0$ there exists a sequence of intervals $\{I_n\}$ such that
$$A \subset \bigcup_{n=1}^{\infty} I_n \quad\text{and}\quad \sum_{n=1}^{\infty} |I_n| < \varepsilon.$$
