This corollary generalizes the decomposition of the whole line to arbitrary subsets. Any set $E \subset \mathbb{R}$ can be written as $E = (E \cap A) \cup (E \cap B)$, where $A$ is of first category and $B$ is a nullset (from the line decomposition theorem). Since $E \cap A$ is a subset of a first category set, it too is of first category, and $E \cap B$ is a subset of a nullset, hence a nullset. This decomposition highlights the duality between measure and category: every set can be split into "small" parts in both senses simultaneously.
