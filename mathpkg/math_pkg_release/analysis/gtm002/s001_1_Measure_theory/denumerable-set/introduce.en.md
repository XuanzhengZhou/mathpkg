A set is denumerable if it has the same cardinality as the natural numbers $\mathbb{N}$. This means there exists a bijection between the set and $\mathbb{N}$, so the elements can be listed as an infinite sequence $a_1, a_2, a_3, \dots$ without repetition. Denumerability is the fundamental notion of countability used throughout measure and category theory.
