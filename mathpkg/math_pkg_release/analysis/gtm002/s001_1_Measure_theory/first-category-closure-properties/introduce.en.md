This theorem records that the class of first category sets forms a $\sigma$-ideal: it is closed under taking arbitrary subsets and countable unions. However, unlike the class of nowhere dense sets, the first category class is not closed under taking closures — the closure of a first category set need not be first category. In fact, a set has first category closure exactly when the set itself is nowhere dense.
