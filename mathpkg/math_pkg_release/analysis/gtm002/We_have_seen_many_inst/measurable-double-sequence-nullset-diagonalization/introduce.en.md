Theorem 21.2 provides a diagonalization result for measurable sets: given a double sequence of nested measurable sets whose intersection along the second index is always a nullset, one can choose indices $n_k(i)$ so that the "diagonal" set $\bigcap_k \bigcup_i E_{i,n_k(i)}$ remains a nullset. This theorem is used to demonstrate the breakdown of the extended duality principle, because the category analogue (replacing "measurable" with "Baire property" and "nullset" with "first category") is false.
