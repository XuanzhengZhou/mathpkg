Let $X_i = \{0,1\}$ for each positive integer $i$, and let $X = \prod_{i=1}^{\infty} X_i$ be the set of all sequences of zeros and ones. A subset $E \subseteq X$ is called a \text{tail set} if whenever $x \in E$ and $y$ differs from $x$ in only a finite number of coordinates, then $y \in E$. Equivalently, letting
$$X^n = \prod_{i=1}^{n} X_i \quad \text{and} \quad Y^n = \prod_{i=n+1}^{\infty} X_i,$$
$E$ is a tail set if and only if $E = X^n \times B_n$ for some $B_n \subseteq Y^n$.
