Let $X = \{0,1\}^{\mathbb{N}}$ be the product space of all sequences of zeros and ones, equipped with the product measure $\mu$ defined via the mapping $g(x) = \sum_{i=1}^{\infty} x_i/2^i$ onto $[0,1]$. If $E \subseteq X$ is a tail set and $E$ is measurable (with respect to $\mu$), then
$$\mu(E) = 0 \quad \text{or} \quad \mu(E) = 1.$$
