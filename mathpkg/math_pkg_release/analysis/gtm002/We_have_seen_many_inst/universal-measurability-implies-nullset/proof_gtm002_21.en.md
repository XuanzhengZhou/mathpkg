This theorem is presented as a restatement of Theorem 5.5. The proof is given in Chapter 5 and relies on the construction of a Bernstein set (an uncountable set that intersects every perfect set but contains no perfect subset) together with the fact that every uncountable analytic set contains a perfect set. See Theorem 5.5 for the complete proof.
