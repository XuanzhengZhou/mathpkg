The set $E_n \subseteq C$ consists of those continuous functions on $[0,1]$ whose right difference quotients are bounded by $n$ at some point $x \in [0, 1-1/n]$. That is, $|f(x+h) - f(x)| \leq nh$ holds for all sufficiently small $h > 0$. The sets $E_n$ form an increasing sequence whose union $E = \bigcup_n E_n$ captures exactly those functions that possess a bounded right difference quotient somewhere in $[0,1)$. Banach showed that each $E_n$ is closed and nowhere dense, so $E$ is of first category — meaning that in the sense of Baire category, almost no continuous function has bounded difference quotients.
