The space $C = C[0,1]$ consists of all real-valued continuous functions defined on the closed unit interval $[0,1]$. It is a Banach space when equipped with the uniform (supremum) metric $\varrho(f,g) = \sup_{x \in [0,1]} |f(x) - g(x)|$. This space serves as the ambient setting for Banach's category-theoretic proof that almost all continuous functions are nowhere differentiable.
