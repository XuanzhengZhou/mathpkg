This proposition, due to Nikolai Lusin (1914), states that any linear set of second category contains a Lusin set — an uncountable subset all of whose uncountable subsets remain of second category. The proof uses the decomposition of the real line into first category sets and selects one point from each nonempty intersection with the given set $E$. The result depends on the continuum hypothesis and is an early example of the measure-category duality phenomenon studied by Sierpiński.
