A Lusin set is an uncountable set of real numbers with the striking property that every uncountable subset is of second category. This notion was introduced by Nikolai Lusin in 1914 and plays a key role in Sierpiński's duality program between measure and category. Lusin sets are intimately tied to the continuum hypothesis: their existence can be proved using CH, and every Lusin set must have Lebesgue measure zero.
