Let $\mathcal{T} = \{\phi(A) - N : A \in S,\; N \in \mathcal{N}\}$ be the topology on $X$ defined by a lower density $\phi$ on a complete finite measure space $(X, S, \mu)$. A set $N \subset X$ is nowhere dense relative to $\mathcal{T}$ if and only if $N \in \mathcal{N}$. Moreover, every nowhere dense set in this topology is closed.
