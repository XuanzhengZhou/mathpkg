Let $\mu$ be a category measure in a regular Baire space $X$.

\begin{enumerate}
\item For any open set $G \subset X$ and any $\varepsilon > 0$, there exists a closed set $F$ such that $F \subset G$ and $\mu(F) > \mu(G) - \varepsilon$.
\item For any closed set $F \subset X$ and any $\varepsilon > 0$, there exists an open set $G$ such that $F \subset G$ and $\mu(G) < \mu(F) + \varepsilon$.
\end{enumerate}
