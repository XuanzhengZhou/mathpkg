A wandering set for a transformation $T$ is a set whose preimages under iterates of $T$ are pairwise disjoint. This concept captures the idea that the transformation "pushes the set away" — no point ever returns to the set under forward iteration. Wandering sets play a fundamental role in the abstract formulation of the Poincaré recurrence theorem, where nondissipative transformations (those having no nontrivial wandering sets) are exactly those with the recurrence property.
