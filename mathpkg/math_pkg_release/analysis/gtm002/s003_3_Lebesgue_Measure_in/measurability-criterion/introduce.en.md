Lemma 3.11 provides a practical criterion for measurability: a set $A$ of finite outer measure is measurable precisely when it can be simultaneously approximated from the inside by a closed set and from the outside by an open set, with the measure of the gap $G-F$ arbitrarily small. This criterion is frequently used to verify measurability of concrete sets.
