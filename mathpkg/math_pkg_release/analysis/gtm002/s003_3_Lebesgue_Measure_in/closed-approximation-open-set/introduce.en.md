Lemma 3.6 shows that every bounded open set can be approximated from the inside by a closed (hence compact, in $\mathbb{R}^r$) set whose outer measure is arbitrarily close to that of the open set. The proof uses the representation of an open set as a countable union of non-overlapping intervals and the additivity of outer measure on disjoint closed intervals.
