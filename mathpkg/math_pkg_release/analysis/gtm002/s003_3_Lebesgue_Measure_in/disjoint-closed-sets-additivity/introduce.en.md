Lemma 3.5 asserts that outer measure is finitely additive on the class of disjoint bounded closed sets. This is proved by induction on $n$ using the separation property established in Lemma 3.4, which guarantees that disjoint bounded closed sets have positive distance between them. The result is a stepping stone toward the full countable additivity of Lebesgue measure on measurable sets.
