Theorem 3.21 establishes that the density mapping $\phi$, which assigns to each measurable set $A$ the set of points where $A$ has density $1$, satisfies five natural properties. It respects the almost-everywhere equivalence relation $\sim$ (where $A \sim B$ means $m(A \triangle B) = 0$), preserves the empty set and the whole space, commutes with finite intersections, and is monotone with respect to inclusion. These properties show that $\phi$ acts as a canonical selection function that picks a well-behaved representative from each equivalence class of measurable sets modulo nullsets.
