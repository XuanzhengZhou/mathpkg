\begin{theorem}[Theorem 3.17]
For any measurable set $A \subset \mathbb{R}^r$, let $A_i = A \cap (-i, i)^r$ for $i = 1, 2, \ldots$. Then

$$
m(A) = \lim_{i \to \infty} m(A_i).
$$

Equivalently, Lebesgue measure is continuous from below: if $A_i \uparrow A$ is an increasing sequence of measurable sets, then $m(A) = \lim_{i \to \infty} m(A_i)$.
\end{theorem}
