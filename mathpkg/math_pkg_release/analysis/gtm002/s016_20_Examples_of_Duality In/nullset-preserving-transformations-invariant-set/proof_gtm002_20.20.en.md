Index the elements of $K$ and $\mathbb{R}$ so that

$$K = \{ T_\alpha : \alpha < \Omega \} \quad \text{and} \quad \mathbb{R} = \{ p_\alpha : \alpha < \Omega \}.$$

Let $A$ be a nullset such that $\mathbb{R} - A$ is of first category. For $0 < \alpha < \Omega$, let $G_\alpha$ be the group generated by the transformations $T_\beta$ with $\beta < \alpha$. Then $G_\alpha$ consists of all products of the form

$$T_{\beta_1}^{k_1} T_{\beta_2}^{k_2} \dots T_{\beta_n}^{k_n},$$

where $\beta_i < \alpha$ and $k_i = \pm 1$ $(i = 1, \dots, n)$, and $n$ is any positive integer. Hence $G_\alpha$ is countable, and each $T$ in $G_\alpha$ is nullset-preserving. For each $T$ in $G_\alpha$, the set $TA$ is a nullset. Hence $A_\alpha = \bigcup \{ TA : T \in G_\alpha \}$ is a nullset. Let

$$E = \{ p_\alpha : \alpha < \Omega,\ p_\alpha \notin A_\alpha \}.$$

Then $E$ has the required properties.
