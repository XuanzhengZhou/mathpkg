Let $X$ be a complete separable metric space without isolated points. If $T$ is a transitive automorphism of $X$, then the set of points $x \in X$ whose positive semiorbit
$$\{x, Tx, T^2x, \ldots\}$$
is dense in $X$ constitutes a residual set in $X$.

More precisely, let $\{U_j\}_{j=1}^{\infty}$ be a countable base for $X$ and define
$$G_j = \bigcup_{n=0}^{\infty} T^{-n} U_j \quad (j = 1, 2, \ldots), \qquad E = \bigcap_{j=1}^{\infty} G_j.$$
Then $x \in E$ if and only if the positive semiorbit of $x$ is dense in $X$, and $E$ is a residual set.
