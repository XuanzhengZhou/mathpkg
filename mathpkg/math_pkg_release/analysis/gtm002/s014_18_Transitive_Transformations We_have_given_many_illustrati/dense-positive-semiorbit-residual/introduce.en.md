If a topological space admits a transitive automorphism, then not only does there exist a single point with a dense orbit, but the set of all points whose forward iterates are dense forms a topologically large (residual) set. This is proved by expressing the set as a countable intersection of dense open sets defined via the inverse images of basis elements. The result illustrates how the Baire category theorem converts the existence of one dense orbit into a genericity statement about almost every point.
