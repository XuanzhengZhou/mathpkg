Theorem 14.3 extends Fubini's nullset theorem to measurable sets: for any plane measurable set $E$, almost all vertical sections $E_x$ are linearly measurable. The proof decomposes $E$ as the union of an $F_\sigma$ set and a nullset. Sections of closed (and hence $F_\sigma$) sets are automatically measurable, and Theorem 14.2 handles the nullset component. This result is a crucial step toward the full Fubini theorem.
