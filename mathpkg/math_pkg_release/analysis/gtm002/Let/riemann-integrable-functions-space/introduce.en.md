The set $R[a,b]$ of Riemann integrable functions with the $L^1$ metric $\sigma$ forms a pseudometric space because two functions differing on a set of measure zero (a null function) satisfy $\sigma(f,g)=0$ without being identical. The quotient construction identifies such functions, yielding the metric space $(\tilde{R},\sigma)$. This is a precursor to the Lebesgue $L^1$ space.
