The space $C[a,b]$ of real-valued continuous functions equipped with the uniform metric $\varrho(f,g) = \sup_{a \leq x \leq b} |f(x) - g(x)|$ is a complete metric space. The proof uses the fact that $\mathbb{R}$ is complete: a Cauchy sequence in $C[a,b]$ yields, for each fixed $x$, a Cauchy sequence of real numbers, which converges pointwise. One then shows that this pointwise limit is actually the uniform limit, and a classical theorem ensures the uniform limit of continuous functions is continuous. This result is foundational in functional analysis; it shows that $C[a,b]$ with the supremum norm is a Banach space.
