Let $C = C[a,b]$ denote the set of all real-valued continuous functions $f$ on the interval $[a,b]$, and define
$$\varrho(f, g) = \sup_{a \leq x \leq b} |f(x) - g(x)|.$$
Then $\varrho$ is a metric on $C$, called the \text{uniform metric}. Convergence in this metric means uniform convergence on $[a,b]$.
