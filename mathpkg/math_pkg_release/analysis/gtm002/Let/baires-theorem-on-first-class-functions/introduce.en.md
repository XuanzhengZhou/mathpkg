Baire's theorem on functions of first class (Theorem 7.3) states that any pointwise limit of continuous functions can be discontinuous only on a set of first category. This fundamental result shows that Baire class 1 functions, while possibly discontinuous, cannot be "too discontinuous" — their set of discontinuities is topologically small (meager). This theorem was historically the motivation for Baire's introduction of the notion of category.
