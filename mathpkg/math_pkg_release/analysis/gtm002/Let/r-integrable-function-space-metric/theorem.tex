Let $R[a,b]$ denote the set of Riemann-integrable functions on $[a,b]$, equipped with the metric
$$\sigma(f, g) = \int_a^b |f(x) - g(x)| \, dx.$$
Since $\sigma(f,g) = 0$ does not imply $f = g$ pointwise (only that $f$ and $g$ differ by a null function), $\sigma$ is only a pseudometric on $R[a,b]$. Identifying functions that differ by a null function yields a metric space $(\tilde{R}[a,b], \sigma)$, called the space of $R$-integrable functions on $[a,b]$.
