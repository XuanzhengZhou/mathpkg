Theorem 13.1 is a striking bridge between category and measure: any first category set in $[0,1]$ can be mapped to a Lebesgue nullset by a homeomorphism of the interval. Furthermore, the set of homeomorphisms achieving this is residual (comeager) in the space $H$ of endpoint-fixing automorphisms — meaning that a "typical" homeomorphism has this property. The proof uses a Baire category argument: the set of homeomorphisms mapping a given nowhere dense set to a small-measure set is open and dense in $H$.
