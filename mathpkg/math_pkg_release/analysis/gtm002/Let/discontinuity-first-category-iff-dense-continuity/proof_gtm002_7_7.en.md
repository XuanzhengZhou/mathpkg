This is an immediate consequence of Theorem 7.1 and the fact that an $F_\sigma$ set is of first category if and only if its complement is dense.

By Theorem 7.1, the set $D$ of points of discontinuity of $f$ is an $F_\sigma$ set. An $F_\sigma$ set in a complete metric space is of first category if and only if its complement is dense (this follows from the Baire category theorem: an $F_\sigma$ set whose complement is not dense would contain an open set, making it of second category). The complement of $D$ is precisely the set of points where $f$ is continuous. Hence $D$ is of first category if and only if the set of continuity points is dense.
