Let $R[a,b]$ denote the set of Riemann-integrable functions on $[a,b]$, with the metric $\sigma(f, g) = \int_a^b |f(x) - g(x)| \, dx$. Identifying any two elements of $R[a,b]$ that differ by a null function (a function $f$ such that $\int_a^b|f(x)| \, dx = 0$) yields a metric space $(\tilde{R}, \sigma)$, called the space of R-integrable functions on $[a,b]$.
