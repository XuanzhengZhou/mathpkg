A function of the first class of Baire (also called a Baire class 1 function) is one that can be expressed as the pointwise limit of a sequence of continuous functions. Such functions need not be continuous themselves; for example, the pointwise limit of $f_n(x) = \max(0, 1 - n|x|)$ is discontinuous at the origin. This class is the first level of the Baire hierarchy of functions and plays a central role in descriptive set theory and real analysis.
