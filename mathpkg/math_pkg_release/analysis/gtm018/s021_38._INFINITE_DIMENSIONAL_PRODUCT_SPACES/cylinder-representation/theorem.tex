A set $E \subset X = \prod_{i=1}^{\infty} X_i$ is a measurable $\{1, \cdots, n\}$-cylinder if and only if there exists a set $A \subset X_1 \times \cdots \times X_n$ such that

$$E = A \times X^{(n)},$$

and $E \in \prod_{i=1}^{\infty} \mathbf{S}_i$ if and only if $A \in \mathbf{S}_1 \times \cdots \times \mathbf{S}_n$.
