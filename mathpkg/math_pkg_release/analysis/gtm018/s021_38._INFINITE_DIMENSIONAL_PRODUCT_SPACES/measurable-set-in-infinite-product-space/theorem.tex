If each $X_i$ is a measure space, with a $\sigma$-ring $\mathbf{S}_i$ of measurable sets and a measure $\mu_i$, and assuming the spaces $X_i$ are totally finite measure spaces such that $\mu_i(X_i) = 1$, $i = 1, 2, \cdots$, then a subset of $\prod_{i=1}^{\infty} X_i$ will be called measurable if it belongs to the $\sigma$-ring $\mathbf{S}$ (which is in fact a $\sigma$-algebra) generated by the class of all measurable rectangles; we shall write

$$\mathbf{S} = \prod_{i=1}^{\infty} \mathbf{S}_i.$$
