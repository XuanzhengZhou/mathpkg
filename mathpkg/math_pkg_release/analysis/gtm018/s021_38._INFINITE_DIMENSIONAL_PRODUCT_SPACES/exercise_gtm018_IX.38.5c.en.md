If for each $x = (x_1, x_2, \cdots)$ in $\bar{X}$ we write $z(x) = \sum_{i=1}^{\infty} x_i / 2^i$, and we form the product space $X = \prod_{i=1}^{\infty} X_i$, then there exists a one-to-one correspondence between $X$ and $X_0$ such that every Borel set in $X_0$ corresponds to a measurable set (i.e. to a set belonging to $\prod_{i=1}^{\infty} \mathbf{S}_i$) in $X$, and such that corresponding sets have equal measures.

*Hint:* if $Y_0$ is the two-point space described in (5) and denoted there by $X_0$, and if $Y_{ij} = Y_0$ for $i = 0, 1, 2, \cdots$ and $j = 1, 2, \cdots$, then $X_i = \prod_{j=1}^{\infty} Y_{ij}$, $i = 0, 1, 2, \cdots$. The correspondence is based on the usual correspondence between doubly infinite sequences, i.e. elements of $X = \prod_{i=1}^{\infty} X_i = \prod_{i=1}^{\infty} \prod_{j=1}^{\infty} Y_{ij}$, and simple sequences, i.e. elements of $X_0 = \prod_{j=1}^{\infty} Y_{0j}$.
