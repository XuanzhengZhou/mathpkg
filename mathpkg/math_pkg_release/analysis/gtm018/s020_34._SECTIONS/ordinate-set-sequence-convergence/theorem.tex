If $\{f_n\}$ is an increasing sequence of non negative functions converging at every point to $f$, then $\{V_*(f_n)\}$ is an increasing sequence of sets whose union is $V_*(f)$. Similarly, if $\{f_n\}$ is decreasing to $f$, then $\{V^*(f_n)\}$ is a decreasing sequence of sets whose intersection is $V^*(f)$.
