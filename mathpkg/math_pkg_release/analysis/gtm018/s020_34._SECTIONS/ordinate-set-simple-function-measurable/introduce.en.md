Exercise (5a) of Section 34 establishes that for a non negative simple function $f$, both the lower ordinate set $V^*(f)$ and the upper ordinate set $V_*(f)$ are measurable subsets of the product space. This follows because each ordinate set can be expressed as a finite union of measurable rectangles. This result is part of an alternative approach to the theory of measurable functions using ordinate sets.
