A class is a set whose elements are themselves sets. Halmos reserves this terminology to distinguish levels of aggregation: points belong to sets, sets belong to classes. The same inclusion and membership notations apply uniformly, simplifying the language of measure theory where one constantly manipulates collections of sets such as rings, $\sigma$-rings, and algebras.
