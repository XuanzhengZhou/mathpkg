If $\mu$ is a totally $\sigma$-finite measure on a $\sigma$-ring $\mathbf{S}$ and if $\nu$ is a signed measure on $\mathbf{S}$ which is absolutely continuous with respect to $\mu$, then there exists an extended real valued, measurable function $f$ on $X$ such that, for every $E$ in $\mathbf{S}$,
$$\nu(E) = \int_E f \,d\mu.$$
The function $f$ is uniquely determined modulo $\mu$-null sets; it is called the Radon-Nikodym derivative of $\nu$ with respect to $\mu$ and is denoted by $d\nu/d\mu$.
