If $T$ is a measurable transformation from a measure space $(X, \mathbf{S}, \mu)$ into a measurable space $(Y, \mathbf{T})$, and if $g$ is an extended real valued measurable function on $Y$, then
$$\int g \, d(\mu T^{-1}) = \int (gT) \, d\mu,$$
in the sense that if either integral exists, then so does the other and the two are equal.
