A measurable transformation $T$ from $(X, \mathbf{S})$ into $(Y, \mathbf{T})$ assigns in an obvious way a set function $\nu$ on $\mathbf{T}$ to every set function $\mu$ on $\mathbf{S}$; $\nu$ is defined for every $F$ in $\mathbf{T}$ by
$$\nu(F) = \mu(T^{-1}(F)).$$
It is convenient and natural to write $\nu = \mu T^{-1}$.
