Let $X = A \cup B$ be a Hahn decomposition with respect to a signed measure $\mu$. Define, for every measurable set $E$,
$$\mu^{+}(E) = \mu(E \cap A), \quad \mu^{-}(E) = -\mu(E \cap B), \quad |\mu|(E) = \mu^{+}(E) + \mu^{-}(E).$$
The set functions $\mu^{+}$, $\mu^{-}$, and $|\mu|$ are called the \textit{upper variation}, \textit{lower variation}, and \textit{total variation} of $\mu$, respectively.

The upper, lower, and total variations of a signed measure $\mu$ are measures, and $\mu(E) = \mu^{+}(E) - \mu^{-}(E)$ for every measurable set $E$. If $\mu$ is [totally] finite or $\sigma$-finite, then so are $\mu^{+}$ and $\mu^{-}$; at least one of $\mu^{+}$ and $\mu^{-}$ is always finite.
