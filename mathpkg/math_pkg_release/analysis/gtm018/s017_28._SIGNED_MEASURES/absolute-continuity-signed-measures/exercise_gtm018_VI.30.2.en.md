Let the measure space $(X, \mathbf{S}, \mu)$ be the unit interval with Lebesgue measure. Write $F = \{x : 0 \leq x \leq \frac{1}{2}\}$, and let $f_1$ and $f_2$ be the functions defined by $f_1(x) = 2\chi_F(x) - 1$ and $f_2(x) = x$. If the set functions $\mu_i$ are defined by $\mu_i(E) = \int_E f_i\,d\mu$, $i = 1, 2$, then $\mu_2 \ll \mu_1$. It is not, however, true that $\mu_2(E) = 0$ whenever $\mu_1(E) = 0$. If $\mu_2$ were defined by $\mu_2(E) = \int_E (f_2 - \frac{1}{2})\,d\mu$, then even this stronger condition would be satisfied.
