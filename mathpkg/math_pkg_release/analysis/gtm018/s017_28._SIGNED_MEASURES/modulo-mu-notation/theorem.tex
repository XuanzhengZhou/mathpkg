If, for each point $x$ of a measurable space $(X, \mathbf{S})$, $\pi(x)$ is a proposition concerning $x$, and if $\mu$ is a signed measure on $\mathbf{S}$, then the symbol
$$\pi(x)\;[\mu] \quad \text{or} \quad \pi\;[\mu]$$
shall mean that $\pi(x)$ is true for almost every $x$ with respect to the measure $|\mu|$.

Thus, for instance, if $f$ and $g$ are two functions on $X$, we write $f = g\;[\mu]$ for the statement that $\{x : f(x) \neq g(x)\}$ is a measurable set of measure zero with respect to $|\mu|$. The symbol $[\mu]$ may be read as ``modulo $\mu$.''
