A necessary and sufficient condition that a set $A$ in a locally compact topological group $X$ be bounded is that there exist a Borel set $E$ of finite measure and a positive number $\epsilon$, with $0 \leq \epsilon < 2\mu(E)$, such that

$$A \subset \{x: \rho(xE,E) \leq \epsilon\},$$

where $\mu$ is a regular Haar measure on $X$ and $\rho(E,F) = \mu(E \Delta F)$.
