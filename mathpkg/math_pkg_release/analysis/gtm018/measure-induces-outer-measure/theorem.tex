If $\mu$ is a measure on a ring $\mathbf{R}$ and if, for every $E$ in $\mathbf{H}(\mathbf{R})$,
$$\mu^*(E) = \inf \left\{ \sum_{n=1}^{\infty} \mu(E_n): E_n \in \mathbf{R}, \; n = 1, 2, \cdots; \; E \subset \bigcup_{n=1}^{\infty} E_n \right\},$$
then $\mu^*$ is an extension of $\mu$ to an outer measure on $\mathbf{H}(\mathbf{R})$; if $\mu$ is [totally] $\sigma$-finite, then so is $\mu^*$. The outer measure $\mu^*$ is called the outer measure induced by the measure $\mu$.
