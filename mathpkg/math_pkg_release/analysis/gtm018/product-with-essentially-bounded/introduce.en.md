The product of an integrable function with an essentially bounded measurable function is integrable. If $|g| \leq c$ a.e., then $|fg| \leq c|f|$ a.e., and the result follows from Theorem C since $c|f|$ is integrable. This is fundamental: $L^\infty$ acts on $L^1$ by multiplication, making $L^1$ a module over $L^\infty$.
