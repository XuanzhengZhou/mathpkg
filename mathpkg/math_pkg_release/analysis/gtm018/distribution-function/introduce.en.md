The distribution function $g(t) = \mu(\{x: f(x) < t\})$ captures the measure-theoretic behavior of a measurable function $f$ on a finite measure space. It is monotone, left-continuous, and determines a Lebesgue-Stieltjes measure that completes the pushforward measure $\nu = \mu \circ f^{-1}$. This connects the abstract measurable function to classical Stieltjes integration.
