This lemma is a key auxiliary result used in Halmos's proof of the Radon-Nikodym Theorem. It guarantees that when a nonzero finite measure $\nu$ is absolutely continuous with respect to another finite measure $\mu$, one can find a positive $\varepsilon$ and a set $A$ of positive $\mu$-measure that is a positive set for the signed measure $\nu - \varepsilon\mu$. The proof uses the Hahn decomposition with respect to $\nu - \frac{1}{n}\mu$ for each $n$ and exploits the fact that $\nu(B_0) \leq \frac{1}{n}\mu(B_0)$ forces $\nu(B_0)=0$ by absolute continuity, while $\nu \not\equiv 0$ ensures some $A_n$ has positive $\mu$-measure.
