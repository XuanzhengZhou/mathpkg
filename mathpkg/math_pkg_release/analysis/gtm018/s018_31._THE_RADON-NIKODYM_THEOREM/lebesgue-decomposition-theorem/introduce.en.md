The Lebesgue Decomposition Theorem states that any totally $\sigma$-finite signed measure $\nu$ can be uniquely decomposed as $\nu = \nu_0 + \nu_1$, where $\nu_0$ is singular with respect to a reference measure $\mu$ and $\nu_1$ is absolutely continuous with respect to $\mu$. Combining this with the Radon-Nikodym Theorem yields the full picture: $\nu_1$ has a density $\frac{d\nu_1}{d\mu}$, while $\nu_0$ is concentrated on a set of $\mu$-measure zero. The proof uses the Radon-Nikodym Theorem applied to $\nu$ with respect to $\mu + \nu$.
