Lusin's theorem states that every Baire function is "almost continuous": given a Baire measure $\mu_0$, a Baire set $E$ of finite measure, and an arbitrary Baire function $f$, one can find a compact Baire subset $C \subset E$ whose complement has arbitrarily small measure and on which $f$ is continuous. The proof proceeds by first handling simple Baire functions via the regularity of the measure (approximating Baire sets from within by compact Baire sets), and then extending to general Baire functions using Egoroff's theorem.
