Suppose that $\{(X_n, S_n, \mu_n)\}$ is a sequence of probability spaces,
$$(X, S) = \bigtimes_{n=1}^{\infty}(X_n, S_n),$$
and $\mu$ is a set function on $F$ such that, for each positive integer $n$, $\mu$ is a probability measure on $F_n$. If, on each $F_n$, $\mu$ is absolutely continuous with respect to the product measure $\bigtimes_{i=1}^{\infty} \mu_i$, then $\mu$ has a unique extension to a probability measure on $S$.
