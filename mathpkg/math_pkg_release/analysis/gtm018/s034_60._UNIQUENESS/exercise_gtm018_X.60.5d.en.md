Prove that the modular function $\Delta$ is continuous. (Hint: if $C$ is a compact set of positive measure and $\epsilon > 0$, then, by regularity, there exists an open set $U$ such that $C \subset U$ and $\mu(U) \leq (1 + \epsilon)\mu(C)$. If $V$ is a neighborhood of $e$ such that $V = V^{-1}$ and $CV \subset U$, and if $x \in V$, then
$$
\Delta(x)\mu(C) = \mu(Cx) \leq \mu(U) \leq (1 + \epsilon)\mu(C)
$$
and
$$
\frac{\mu(C)}{\Delta(x)} = \mu(Cx^{-1}) \leq \mu(U) \leq (1 + \epsilon)\mu(C),
$$
so that $\frac{1}{1 + \epsilon} \leq \Delta(x) \leq 1 + \epsilon$.)
