If $f$ is an extended real valued, measurable function such that $f \geq 0$ a.e. and $f$ is not integrable, we define $\int f d\mu = \infty$. More generally, if at least one of $f^+$ and $f^-$ is integrable, we define $\int f d\mu = \int f^+ d\mu - \int f^- d\mu$, which is always $+\infty$, $-\infty$, or a finite real number (never $\infty - \infty$). The adjective "integrable" is reserved for functions integrable in the original sense.
