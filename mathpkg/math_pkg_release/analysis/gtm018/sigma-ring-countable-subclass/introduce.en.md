Theorem D states that every set belonging to the $\sigma$-ring $\mathbf{S}(\mathbf{E})$ generated by a class $\mathbf{E}$ is actually contained in the $\sigma$-ring generated by some countable subclass of $\mathbf{E}$. This is a fundamental "countability" result that limits the complexity of sets in generated $\sigma$-rings and is essential for many approximation arguments in measure theory.
