For extended real valued functions (taking values in $\mathbb{R} \cup \{\pm\infty\}$), measurability is defined by regarding $\{\infty\}$ and $\{-\infty\}$ as Borel sets. The inverse images of these infinite points must be measurable, in addition to the usual condition for finite values. For the extended Borel sets, the class is no longer generated by semiclosed intervals.
