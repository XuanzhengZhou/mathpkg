A random variable is a real-valued measurable function $f$ on a probability space $(X, \mathbf{S}, \mu)$. The probability that $f$ takes values in an interval $[\alpha, \beta]$ is
$$\mu\left(\{x \in X : \alpha \leq f(x) \leq \beta\}\right).$$
The measurability condition ensures that for every Borel set $B \subseteq \mathbb{R}$, the preimage
$$\{x \in X : f(x) \in B\}$$
belongs to $\mathbf{S}$, so that all probability questions concerning the values of $f$ are answerable.
