The conditional probability of an event $E$ given that an event $F$ has already occurred, denoted $\mu_F(E)$, is defined as
$$\mu_F(E) = \frac{\mu(E \cap F)}{\mu(F)},$$
provided $\mu(F) > 0$. This definition measures the proportion of $F$ that is contained in $E$: it is the ratio of the probability that both events occur to the probability of the conditioning event alone.
