If $A$ is a measurable subset of $X$ and $y \in X$, then $Ay$ is a measurable set. If moreover $\mu(A) > 0$, then $\mu(Ay) > 0$. Similarly, $A^{-1}$ is a measurable set; if $\mu(A) > 0$, then $\mu(A^{-1}) > 0$. Furthermore, if $f$ is a measurable function on $X$, then the function $\hat{f}(x) = f(x^{-1})$ is also measurable.
