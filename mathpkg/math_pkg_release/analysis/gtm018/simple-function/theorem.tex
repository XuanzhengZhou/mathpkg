A **simple function** $f$ on a measurable space $X$ is a finite linear combination of characteristic functions of measurable sets: $f(x) = \sum_{i=1}^{n} \alpha_i \chi_{E_i}(x)$, where each $E_i$ is measurable and each $\alpha_i$ is a finite real number. The values of a simple function are finite real numbers; it takes only finitely many values different from zero, each on a measurable set.
