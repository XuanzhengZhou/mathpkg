If $(X, \mathbf{S}, \mu)$ and $(Y, \mathbf{T}, \nu)$ are totally $\sigma$-finite measure spaces, and if $f$ is an integrable function on $X \times Y$ with respect to $\mu \times \nu$, then
$$\int_{X \times Y} f \,d(\mu \times \nu) = \int_X \left( \int_Y f(x,y) \,d\nu(y) \right) d\mu(x) = \int_Y \left( \int_X f(x,y) \,d\mu(x) \right) d\nu(y).$$
