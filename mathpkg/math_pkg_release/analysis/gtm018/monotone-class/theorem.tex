A non empty class $\mathbf{M}$ of sets is monotone if, for every monotone sequence $\{E_n\}$ of sets in $\mathbf{M}$, we have
$$\lim_n E_n \in \mathbf{M}.$$
The monotone class $\mathbf{M}(\mathbf{E})$ generated by any class $\mathbf{E}$ of sets is the smallest monotone class containing $\mathbf{E}$.
