If $f$ is a non negative, extended real valued, measurable function, its integral is defined by
$$\int f \,d\mu = \sup \left\{ \int g \,d\mu : g \text{ is an integrable simple function, } 0 \leq g \leq f \right\}.$$
A measurable function $f$ is integrable if $\int |f| \,d\mu < \infty$; in this case its integral is defined by $\int f \,d\mu = \int f^+ \,d\mu - \int f^- \,d\mu$, where $f^+ = f \cup 0$ and $f^- = -(f \cap 0)$.
