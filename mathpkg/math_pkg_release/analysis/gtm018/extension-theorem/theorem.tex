If $\mu$ is a $\sigma$-finite measure on a ring $\mathbf{R}$, then there exists a unique measure $\bar{\mu}$ on the $\sigma$-ring $\mathbf{S}(\mathbf{R})$ such that $\bar{\mu}$ is an extension of $\mu$; the measure $\bar{\mu}$ is $\sigma$-finite. The measure $\bar{\mu}$ is called the extension of $\mu$; if $\mu$ is finite, then so is $\bar{\mu}$.
