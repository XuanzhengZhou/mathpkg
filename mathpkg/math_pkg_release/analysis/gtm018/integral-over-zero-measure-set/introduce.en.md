The integral over any set of measure zero is zero. This follows from Theorem B because $\chi_E f = 0$ a.e. when $\mu(E)=0$. Consequently, changing an integrable function on a null set does not affect its integral or its integrability, which is fundamental for the identification of functions in $L^1$.
