**Theorem D.** If $\{f_n\}$ and $\{g_n\}$ are mean fundamental sequences of integrable simple functions which converge in measure to the same measurable function $f$, if the indefinite integrals of $f_n$ and $g_n$ are $\nu_n$ and $\lambda_n$ respectively, and if $\nu(E) = \lim_n \nu_n(E)$ and $\lambda(E) = \lim_n \lambda_n(E)$ for every measurable set $E$, then $\nu = \lambda$.
