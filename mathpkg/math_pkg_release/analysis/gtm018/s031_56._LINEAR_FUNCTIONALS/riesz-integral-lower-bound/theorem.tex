If $\Lambda$ is a positive linear functional on $\mathcal{L}$, if, for every $C$ in $\mathbf{C}$,

$$\lambda(C) = \inf \left\{ \Lambda(f) : C \subset f \in \mathcal{L}_+ \right\},$$

and if $\mu$ is the Borel measure induced by the content $\lambda$, then

$$\int f \, d\mu \leq \Lambda(f)$$

for every $f$ in $\mathcal{L}_+$.
