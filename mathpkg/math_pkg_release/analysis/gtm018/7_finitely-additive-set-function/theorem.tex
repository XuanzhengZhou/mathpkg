An extended real valued set function $\mu$ defined on a class $\mathbf{E}$ is \textbf{finitely additive} if, for every finite, disjoint class $\{E_1, \dots, E_n\}$ of sets in $\mathbf{E}$ whose union is also in $\mathbf{E}$, we have
$$\mu\left(\bigcup_{i=1}^{n} E_i\right) = \sum_{i=1}^{n} \mu(E_i).$$
