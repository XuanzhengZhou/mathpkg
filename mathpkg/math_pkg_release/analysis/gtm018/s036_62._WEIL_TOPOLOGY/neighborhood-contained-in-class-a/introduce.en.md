Theorem B provides a simple but important inclusion: every set of the form $N = \{x : \rho(xE, E) < \epsilon\}$ (which belongs to class $N$) is contained in $A = EE^{-1}$ (which belongs to class $A$). This follows from the observation that $N \subset \{x : xE \cap E \neq \emptyset\} = EE^{-1}$, and is used in establishing that the class $N$ satisfies the neighborhood axioms for a topological group.
