**Exercise 62.6a: Borel set and linearly independent extension**

Let $X$ be a topological group and $X \times X$ the product topological group. A subset $B$ of $X$ is \emph{linearly independent} if the conditions $\sum_{i=1}^{n} r_i x_i = 0$, $x_i \in B$, $i = 1, \dots, n$, with rational numbers $r_i$, imply that $r_1 = \cdots = r_n = 0$.

Show that if $E$ is a Borel set of positive measure in $X \times X$ and if $B$ is a linearly independent set in $X$, of cardinal number smaller than that of the continuum, then there exists a point $(x,y)$ in $E$ such that $B \cup \{x\}$ is a linearly independent set.

(Hint: there exists a value of $y$ such that the section $E^y$ has positive measure and therefore has the cardinal number of the continuum.)
