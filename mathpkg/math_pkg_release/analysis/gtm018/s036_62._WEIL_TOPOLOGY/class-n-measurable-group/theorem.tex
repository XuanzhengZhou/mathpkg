Let $(X, \mathbf{S}, \mu)$ be a fixed measurable group and write $\rho(E, F) = \mu(E \Delta F)$ for any two measurable sets $E$ and $F$. Denote by $N$ the class of all sets of the form $\{x : \rho(xE, E) < \epsilon\}$, where $E$ is a measurable set of positive, finite measure and $\epsilon$ is a real number such that $0 < \epsilon < 2\mu(E)$.
