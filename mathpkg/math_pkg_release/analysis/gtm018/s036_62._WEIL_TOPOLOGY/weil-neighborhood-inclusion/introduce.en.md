Theorem B establishes the fundamental relationship between the two classes used in constructing the Weil topology: every set $N$ in the neighborhood base class $\mathbf{N}$ is contained in the corresponding set $A = EE^{-1}$ from class $\mathbf{A}$. This inclusion ensures that the Weil topology is compatible with the group structure.
