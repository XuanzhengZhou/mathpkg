A measure is called complete if every subset of a null set (a set of measure zero) is measurable, i.e. belongs to the domain ring. Completeness ensures that negligible sets do not pose measurability problems. While not every measure is initially complete, any measure can be completed by enlarging its domain to include all subsets of null sets; this completion process preserves all essential properties of the original measure.
