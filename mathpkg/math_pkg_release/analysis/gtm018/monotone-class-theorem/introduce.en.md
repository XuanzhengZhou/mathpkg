The Monotone Class Theorem (Theorem B) is one of the most useful results in measure theory. It states that the monotone class generated by a ring equals the $\sigma$-ring generated by that ring. This means that to prove a property holds for all sets in $\mathbf{S}(\mathbf{R})$, it suffices to verify it on the ring $\mathbf{R}$ and then show the class of sets satisfying the property is monotone. This technique avoids explicit transfinite or ordinal constructions.
