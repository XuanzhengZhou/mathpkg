If $E \in \mathbf{H}(\mathbf{S})$ and $F \in \mathbf{S}$, we say that $F$ is a measurable kernel of $E$ if $F \subset E$ and if, for every set $G$ in $\mathbf{S}$ for which $G \subset E - F$, we have $\mu(G) = 0$. Loosely speaking, a measurable kernel of $E$ in $\mathbf{H}(\mathbf{S})$ is a maximal set in $\mathbf{S}$ contained in $E$.
