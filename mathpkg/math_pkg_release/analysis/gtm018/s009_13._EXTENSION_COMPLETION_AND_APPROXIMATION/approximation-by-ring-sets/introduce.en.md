Any set in the generated $\sigma$-ring $\mathbf{S}(\mathbf{R})$ with finite measure can be approximated arbitrarily well by a set from the original ring $\mathbf{R}$ in the pseudometric $d(E,F) = \mu(E \Delta F)$. This justifies viewing the extension as a completion and is a key technical tool.
