If $f_i$, $i = 1, \dots, n$ are independent functions on a probability space $(X, S, \mu)$ such that $\int f_i \, d\mu = 0$ and $\int f_i^2 \, d\mu < \infty$ for $i = 1, \dots, n$, and if

$$f(x) = \max_{1 \leq k \leq n} \left| \sum_{i=1}^{k} f_i(x) \right|,$$

then, for every positive number $\epsilon$,

$$\mu(\{x : f(x) \geq \epsilon\}) \leq \frac{1}{\epsilon^2} \sum_{i=1}^{n} \sigma^2(f_i).$$
