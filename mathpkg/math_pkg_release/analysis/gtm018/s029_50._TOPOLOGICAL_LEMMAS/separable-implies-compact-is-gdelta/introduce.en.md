If the locally compact Hausdorff space $X$ is separable, then every compact subset is automatically a $G_\delta$ set. This provides a wide class of spaces (including Euclidean spaces and all separable metric spaces) in which the Baire set theory simplifies, since the class of compact $G_\delta$'s coincides with the class of all compact sets. The result can be proved either by a direct covering argument or as a consequence of Theorem C.
