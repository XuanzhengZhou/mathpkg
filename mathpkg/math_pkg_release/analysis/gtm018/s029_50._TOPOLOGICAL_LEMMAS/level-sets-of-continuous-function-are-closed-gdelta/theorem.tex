If $f$ is a real valued continuous function on $X$ and $c$ is a real number, then each of the three sets

$$\{x: f(x) \geq c\}, \quad \{x: f(x) \leq c\}, \quad \text{and} \quad \{x: f(x) = c\}$$

is a closed $G_\delta$. If, conversely, $C$ is a compact $G_\delta$, then there exists a function $f$ in $\mathfrak{F}$ such that $C = \{x: f(x) = 0\}$.
