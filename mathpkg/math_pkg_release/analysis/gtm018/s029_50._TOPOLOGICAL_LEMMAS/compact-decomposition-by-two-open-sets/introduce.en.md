If a compact set $C$ is covered by two open sets $U$ and $V$, then $C$ can be decomposed into two compact pieces $D$ and $E$ each lying entirely within the corresponding open set. This purely topological decomposition lemma uses the fact that in a locally compact Hausdorff space, disjoint compact sets can be separated by disjoint open sets. The result is a useful tool in measure theory for establishing countable subadditivity of set functions defined on compact sets.
