The $\sigma$-ring generated by the class of all bounded open sets, or, equivalently, the $\sigma$-ring generated by $\mathbf{U}$, coincides with $\mathbf{S}$. (Hint: for every compact set $C$, let $U$ be a bounded open set containing $C$, and consider $U - (U - C)$.)
