If $\mathbf{E}$ is any class of sets, we denote by $\mathbf{H}(\mathbf{E})$ the hereditary $\sigma$-ring generated by $\mathbf{E}$, i.e. the smallest hereditary $\sigma$-ring containing $\mathbf{E}$. The hereditary $\sigma$-ring $\mathbf{H}(\mathbf{E})$ is the class of all sets which can be covered by countably many sets in $\mathbf{E}$; if $\mathbf{E}$ is a non empty class closed under the formation of countable unions, then $\mathbf{H}(\mathbf{E})$ is the class of all sets which are subsets of some set in $\mathbf{E}$.
