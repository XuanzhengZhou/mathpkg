If a strictly positive function integrates to zero over a set, the set must have measure zero. The proof uses $E \cap \{f \geq 1/n\}$: each such set has measure zero (otherwise the integral would be positive), and their union is $E$ up to a null set. This theorem is crucial for establishing uniqueness properties of integrals.
