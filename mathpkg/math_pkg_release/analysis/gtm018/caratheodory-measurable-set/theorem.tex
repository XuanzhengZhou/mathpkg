Let $\mu^*$ be an outer measure on a hereditary $\sigma$-ring $\mathbf{H}$. A set $E$ in $\mathbf{H}$ is $\mu^*$-measurable if, for every set $A$ in $\mathbf{H}$,
$$\mu^*(A) = \mu^*(A \cap E) + \mu^*(A \cap E').$$
Since the reverse inequality follows automatically from subadditivity, it is sufficient to verify that
$$\mu^*(A) \geq \mu^*(A \cap E) + \mu^*(A \cap E')$$
for every $A$ in $\mathbf{H}$.
