**Theorem A.** If $\{f_n\}$ is a sequence of extended real valued, measurable functions on a measurable space $X$, then each of the four functions $h, g, f^*$ and $f_*$, defined by $h(x) = \sup\{f_n(x): n=1,2,\dots\}$, $g(x) = \inf\{f_n(x): n=1,2,\dots\}$, $f^*(x) = \limsup_n f_n(x)$, $f_*(x) = \liminf_n f_n(x)$, is measurable.
