Every measurable function can be approximated pointwise by simple functions. For non-negative $f$, there is an increasing sequence of non-negative simple functions converging to $f$. This is the fundamental approximation theorem that allows integration theory to be built from simple functions via limiting processes, analogous to how Lebesgue integration extends from step functions.
