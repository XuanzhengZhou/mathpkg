**Theorem D (Lebesgue Bounded Convergence).** If $\{f_n\}$ is a sequence of integrable functions which converges in measure to $f$ [or converges to $f$ a.e.], and if $g$ is an integrable function such that $|f_n(x)| \leq |g(x)|$ a.e., $n=1,2,\dots$, then $f$ is integrable and the sequence $\{f_n\}$ converges to $f$ in the mean.
