Let $(X, \mathcal{A}, \mu)$ be any measure space and let $g$ be an element of $\mathfrak{L}_\infty(X, \mathcal{A}, \mu)$. Define $L_g$ on $\mathfrak{L}_1(X, \mathcal{A}, \mu)$ by the rule

$$L_g(f) = \int_X f \bar{g} d\mu.$$

Then $L_g$ is a bounded linear functional on $\mathfrak{L}_1(X, \mathcal{A}, \mu)$ and $\|L_g\| = \|g\|_\infty$.
