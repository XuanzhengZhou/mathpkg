Let $f$ be a relation and let $A$ be a set. We define the image of $A$ under $f$ to be the set

$$f(A) = \{y : (x, y) \in f \text{ for some } x \in A\}.$$

Observe that $f(A) \neq \emptyset$ if and only if $A \cap \text{dom}f \neq \emptyset$. The inverse image of $A$ under $f$ is the set $f^{-1}(A)$.
