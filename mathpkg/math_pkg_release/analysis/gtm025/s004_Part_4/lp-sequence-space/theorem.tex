Let $D$ be any nonvoid set and consider all complex-valued functions $f$ on $D$ such that $\sum_{x \in D} |f(x)|^p < \infty$, where $0 < p < \infty$. If $\mathcal{A}$ is all subsets of $D$ and $\mu$ the counting measure, these functions are the elements of $\mathfrak{L}_p(D, \mathcal{A}, \mu)$, designated $l_p(D)$, or $l_p$ if $D = N$.
