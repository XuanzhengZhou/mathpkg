Let $A$ and $B$ be normed linear spaces and let $T$ be a bounded linear transformation from $A$ into $B$. Then
$$\|T\| = \sup \left\{ \frac{\|T(x)\|}{\|x\|} : x \in A, x \neq 0 \right\} = \sup \{ \|T(x)\| : x \in A, \|x\| = 1 \} = \sup \{ \|T(x)\| : x \in A, \|x\| \le 1 \},$$
and $\|T(x)\| \le \|T\| \cdot \|x\|$ for all $x \in A$.
