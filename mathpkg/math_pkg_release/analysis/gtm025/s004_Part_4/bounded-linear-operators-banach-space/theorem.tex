Let $A$ and $B$ be normed linear spaces over $F$ and let $\mathfrak{B}(A, B)$ denote the set of all bounded linear transformations from $A$ into $B$. Then, with pointwise linear operations and the operator norm, $\mathfrak{B}(A, B)$ is a normed linear space. Moreover $\mathfrak{B}(A, B)$ is a Banach space if $B$ is a Banach space.
