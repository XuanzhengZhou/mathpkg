Let $X$ be a locally compact Hausdorff space and $\iota$ an outer measure as in Section 9. If $A$ is $\iota$-measurable, then
$$\iota(A) = \sup\{\iota(F) : F \subset A,\ F \text{ is compact}\}.$$
If $\iota(A) < \infty$ and $\varepsilon > 0$, there exists compact $F \subset A$ with $\iota(A \cap F') < \varepsilon$.
