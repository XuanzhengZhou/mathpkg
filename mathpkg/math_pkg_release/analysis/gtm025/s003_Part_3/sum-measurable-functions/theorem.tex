Let $f, g$ be $\mathcal{A}$-measurable extended real-valued. Define $h(x) = f(x) + g(x)$ for $x \notin A$ and $h(x) = \beta$ for $x \in A$, where $A = \{f = \infty, g = -\infty\} \cup \{f = -\infty, g = \infty\}$ and $\beta$ is an arbitrary extended real number. Then $h$ is $\mathcal{A}$-measurable.
