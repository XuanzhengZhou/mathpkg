Let $f$ be any complex- or extended real-valued $\mathcal{A}$-measurable function on $X$. There exists a sequence $(s_n)_{n=1}^{\infty}$ of finite-valued, $\mathcal{A}$-measurable simple functions such that
$$|s_1| \leq |s_2| \leq \cdots \leq |s_n| \leq \cdots$$
and $s_n(x) \to f(x)$ for each $x \in X$.

If $f$ is bounded, the $s_n$ can be chosen so that convergence is uniform. If $f \geq 0$, the sequence can be chosen so that $0 \leq s_1 \leq s_2 \leq \cdots \leq f$.
