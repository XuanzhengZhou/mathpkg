Let \(\{A_i\}_{i \in I}\) be any family of sets. The \textbf{Cartesian product} of this family is the set of all functions \(x\) with domain \(I\) such that \(x_i = x(\iota) \in A_i\) for each \(\iota \in I\). Each such function is called a \textbf{choice function} for the family. When \(I = \{1, \ldots, n\}\), we write \(A^n = \{(a_1, \ldots, a_n) : a_k \in A \text{ for } k = 1, \ldots, n\}\).
