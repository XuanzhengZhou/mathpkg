(18.14) Theorem. If $f$ is a real-valued, nondecreasing function on $[a, b]$, then $f'$ is Lebesgue measurable and is a theorem from Chapter 18 (Differentiation) of the book *Real and Abstract Analysis* by Edwin Hewitt and Karl Stromberg (GTM 25). It is part of the development of differentiation.
