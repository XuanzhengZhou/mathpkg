Let $(X, \mathcal{A}, \mu)$ be a measure space and let $\eta$ be a signed measure on $\mathcal{A}$. An extended real-valued, $\mathcal{A}$-measurable function $f$ on $X$ is a derivative of $\eta$ with respect to $\mu$ if there exists a set $B \in \mathcal{A}$ such that:

(i) $f(x)$ is finite for $\mu$-almost all $x$ and $f$ is a Lebesgue-Radon-Nikodym derivative of the $\mu$-continuous part $\eta_a$ with respect to $\mu$;

(ii) $\mu(B) = 0$ and $|\eta_s|(B') = 0$, where $\eta = \eta_a + \eta_s$ is the Lebesgue decomposition;

(iii) $(P, P')$ is a Hahn decomposition for $\eta_s$;

(iv) $f = \infty$ on $B \cap P$ and $f = -\infty$ on $B \cap P'$.
