Let $(X, \mathcal{M}, \mu)$ and $(Y, \mathcal{N}, \nu)$ be $\sigma$-finite measure spaces and let $(X \times Y, \mathcal{M} \times \mathcal{N}, \mu \times \nu)$ be the product measure space. If $f$ is a nonnegative, extended real-valued, $\mathcal{M} \times \mathcal{N}$-measurable function on $X \times Y$, then:

(i) the function $x \to f(x, y)$ is $\mathcal{M}$-measurable for each $y \in Y$;

(ii) the function $y \to f(x, y)$ is $\mathcal{N}$-measurable for each $x \in X$;

(iii) the function $y \to \int_X f(x, y) d\mu(x)$ is $\mathcal{N}$-measurable;

(iv) the function $x \to \int_Y f(x, y) d\nu(y)$ is $\mathcal{M}$-measurable;

(v) the equalities hold:
$$\int_{X \times Y} f(x, y) d\mu \times \nu(x, y) = \int_Y \int_X f(x, y) d\mu(x) d\nu(y) = \int_X \int_Y f(x, y) d\nu(y) d\mu(x).$$
