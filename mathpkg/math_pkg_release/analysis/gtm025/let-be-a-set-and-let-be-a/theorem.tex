Let $X$ be a set and let $\varrho$ be a function from $X \times X$ into $R$ such that for all $x, y, z \in X$ we have:

(i) $\varrho(x, y) \geq 0$;
(ii) $\varrho(x, y) = 0$ if and only if $x = y$;
(iii) $\varrho(x, y) = \varrho(y, x)$;
(iv) $\varrho(x, z) \leq \varrho(x, y) + \varrho(y, z)$ [the triangle inequality].

Then $\varrho$ is called a metric [or distance-function] for $X$; $\varrho(x, y)$ is called the distance from $x$ to $y$, and the pair $(X, \varrho)$ is called a metric space. When no confusion seems possible, we will refer to $X$ as a metric space.
