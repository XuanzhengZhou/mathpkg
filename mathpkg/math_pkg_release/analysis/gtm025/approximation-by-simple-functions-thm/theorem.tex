Let $f$ be any complex or extended real-valued $\mathcal{A}$-measurable function on $X$. There exists a sequence $(s_n)$ of finite-valued, $\mathcal{A}$-measurable simple functions with $|s_1| \leq |s_2| \leq \cdots$ and $s_n(x) \to f(x)$ for each $x \in X$. If $f$ is bounded, the convergence is uniform. If $f \geq 0$, the $s_n$ can be chosen nondecreasing.
