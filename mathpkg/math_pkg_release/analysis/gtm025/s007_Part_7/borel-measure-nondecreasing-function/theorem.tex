Let $\iota$ be a regular Borel measure on $\mathbb{R}$. Define $\alpha(t) = \iota([0,t])$ for $t>0$, $\alpha(0)=0$, $\alpha(t)=-\iota([t,0])$ for $t<0$. Then $\alpha$ is nondecreasing, real-valued, left-continuous on $\mathbb{R}$. Moreover, $\lim_{t\to\infty}\alpha(t) = \iota([0,\infty[)$ and $\lim_{t\to-\infty}\alpha(t) = -\iota(]-\infty,0])$.
