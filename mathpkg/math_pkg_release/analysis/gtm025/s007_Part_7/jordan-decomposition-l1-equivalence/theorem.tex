Let $\nu$ be a complex measure on $(X, \mathcal{A})$ and let $\nu = \sum_{k=1}^{4} \alpha_k \nu_k$ be its Jordan decomposition. A complex-valued function $f$ defined $|\nu|$-a.e. on $X$ is in $\mathfrak{L}_1(X, \mathcal{A}, |\nu|)$ if and only if it is in $\mathfrak{L}_1(X, \mathcal{A}, \nu_k)$ for each $k \in \{1, 2, 3, 4\}$.
