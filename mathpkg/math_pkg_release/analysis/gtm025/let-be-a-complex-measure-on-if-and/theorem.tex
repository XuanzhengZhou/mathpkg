Let $v$ be a complex measure on $(X, \mathcal{A})$. If $f, g \in \mathcal{L}_1(X, \mathcal{A}, |v|)$ and $\alpha \in K$, then

(i) $$\int_X (f + g) dv = \int_X f dv + \int_X g dv$$

and

(ii) $$\int_X \alpha f dv = \alpha \int_X f dv.$$

Thus $$\int_X \cdots dv$$ is a linear functional on $\mathcal{L}_1(X, \mathcal{A}, |v|)$.

This is proved by an obvious computation, which we leave to the reader.
