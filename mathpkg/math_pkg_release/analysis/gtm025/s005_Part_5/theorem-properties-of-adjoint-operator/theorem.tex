The mapping $T \mapsto T^*$ on $\mathfrak{B}(H)$ has the following properties:
\begin{itemize}
\item[(i)] $(T_1 + T_2)^* = T_1^* + T_2^*$;
\item[(ii)] $(\alpha T)^* = \bar{\alpha} T^*$ for $\alpha \in \mathbb{K}$;
\item[(iii)] $(T_1 T_2)^* = T_2^* T_1^*$;
\item[(iv)] $T^{**} = T$;
\item[(v)] $\|T^* T\| = \|T\|^2$.
\end{itemize}
