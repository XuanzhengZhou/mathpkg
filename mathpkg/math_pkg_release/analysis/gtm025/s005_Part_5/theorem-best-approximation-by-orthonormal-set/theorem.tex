Let $\{z_1,\ldots,z_n\}$ be a finite orthonormal set in $H$, and $x \in H$. The function $f(\alpha_1,\ldots,\alpha_n) = \|x - \sum_{k=1}^{n} \alpha_k z_k\|$ attains its minimum precisely when $\alpha_k = \langle x, z_k \rangle$. Moreover $0 \leq \|x\|^2 - \sum_{k=1}^{n} |\langle x, z_k \rangle|^2$.
