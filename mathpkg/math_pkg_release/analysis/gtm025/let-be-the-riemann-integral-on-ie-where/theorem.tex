Let $S$ be the Riemann integral on $\mathfrak{C}_{00}(R)$, i.e., $S = S_{\alpha}$ where $\alpha(x) = x$ for all $x \in R$. Then $S$ has the following properties:

(i) $S(f + g) = S(f) + S(g)$;
(ii) $S(cf) = cS(f)$ for all $c \in K$;
(iii) $S(f) > 0$ if $f \in \mathfrak{C}_{00}^{+}(R)$ and $f \neq 0$;
(iv) $S(f_t) = S(f)$ for all $t \in R$;
(v) $S(f^{\star}) = S(f)$.
