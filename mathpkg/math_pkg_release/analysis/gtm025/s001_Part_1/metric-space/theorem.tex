Let $X$ be a set and let $\varrho$ be a function from $X \times X$ into $R$ such that for all $x, y, z \in X$:
(i) $\varrho(x, y) \geq 0$;
(ii) $\varrho(x, y) = 0$ if and only if $x = y$;
(iii) $\varrho(x, y) = \varrho(y, x)$;
(iv) $\varrho(x, z) \leq \varrho(x, y) + \varrho(y, z)$ [the triangle inequality].
Then $\varrho$ is called a metric (or distance-function) for $X$, and $(X, \varrho)$ is called a metric space.
