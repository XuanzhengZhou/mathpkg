Consider the ring $R[t]$ of all polynomials in the indeterminate $t$ with coefficients in $R$. Let $J = \{(t^2 + 1) p(t) : p(t) \in R[t]\}$. Then $J$ is an ideal in $R[t]$. Let $R[t]/J$ be the set of cosets $p(t) + J$. Addition and multiplication in $R[t]/J$ are defined by
$$(p(t) + J) + (q(t) + J) = (p(t) + q(t)) + J,$$
$$(p(t) + J)(q(t) + J) = (p(t) q(t)) + J.$$
These definitions are unambiguous, and with addition and multiplication so defined, $R[t]/J$ is a field.
