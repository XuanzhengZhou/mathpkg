The Cantor ternary set consists precisely of numbers in $[0,1]$ whose ternary expansion uses only digits $0$ and $2$. This characterization gives a bijection with $\{0,1\}^N$, proving the Cantor set has cardinality $c$ -- it is uncountable despite having Lebesgue measure zero and being nowhere dense.
