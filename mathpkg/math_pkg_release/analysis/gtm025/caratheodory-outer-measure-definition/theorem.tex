Let $X$ be a set. A function $\mu : \mathcal{P}(X) \to [0, \infty]$ is a \textbf{Carathéodory outer measure} if:
(i) $\mu(\varnothing) = 0$;
(ii) $A \subset B$ implies $\mu(A) \leq \mu(B)$ (monotone);
(iii) $\mu(\bigcup_{n=1}^{\infty} A_n) \leq \sum_{n=1}^{\infty} \mu(A_n)$ (countable subadditivity).
