The order type of a linearly ordered set captures its structure up to order isomorphism. When the set is well-ordered, its order type is an ordinal number, which generalizes the natural numbers into the transfinite. For example, \(\operatorname{ord} N = \omega\) and \(\operatorname{ord} Q = \eta\).
