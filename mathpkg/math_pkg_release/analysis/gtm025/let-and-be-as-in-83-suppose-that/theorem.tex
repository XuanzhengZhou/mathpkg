** Let $[a, b], f$, and $\alpha$ be as in (8.3). Suppose that $\Delta, \Delta^* \in \mathcal{D}([a, b])$ and that $\Delta \subset \Delta^*$. Then we have

(i) $L(f, \alpha, \Delta) \leq L(f, \alpha, \Delta^*) \leq U(f, \alpha, \Delta^*) \leq U(f, \alpha, \Delta)$.

**Proof.** The middle inequality is obvious. Let us show that $L(f, \alpha, \Delta) \leq L(f, \alpha, \Delta^*)$. Suppose first that $\Delta^*$ contains just one more point than $\Delta$. Thus if $\Delta = \{a = t_0 < t_1 < \cdots < t_n = b\}$, then

$$\Delta^* = \{a = t_0 < t_1 < \cdots < t_{k-1} < u < t_k < \cdots < t_n = b\}$$
