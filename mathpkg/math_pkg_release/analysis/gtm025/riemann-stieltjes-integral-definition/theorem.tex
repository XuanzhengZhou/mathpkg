Let $f$ be a bounded complex-valued function on $[a, b]$ and $\alpha$ a nondecreasing function on $[a, b]$. For a subdivision $\Delta$ of $[a, b]$, define lower and upper Darboux sums $L(f,\alpha,\Delta)$ and $U(f,\alpha,\Delta)$. Then $f$ is \textbf{Riemann-Stieltjes integrable} with respect to $\alpha$ if for every $\varepsilon > 0$ there exists a subdivision $\Delta$ such that $U(f,\alpha,\Delta) - L(f,\alpha,\Delta) < \varepsilon$.
