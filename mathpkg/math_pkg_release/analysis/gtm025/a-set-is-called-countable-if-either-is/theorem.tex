A set $S$ is called countable if either $S$ is finite or $\bar{S} = \bar{N} = \aleph_0$. Any set that is not countable is uncountable. A set $S$ is countably infinite [or denumerable] if $\bar{S} = \aleph_0$. If $S$ is countably infinite and $f$ is a one-to-one function from $N$ onto $S$, then the sequence $(x_n)$ where $x_n = f(n)$ is called an enumeration of $S$. Note that $x_n \neq x_m$ if $n \neq m$.
