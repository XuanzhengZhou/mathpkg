(4.14) Definition. A set $S$ is called countable if either $S$ is finite or $\bar{S} = \bar{N} = \aleph_0$. Any set that is not countable is uncountab is a definition from Chapter 4 (Unknown Chapter) of the book *Real and Abstract Analysis* by Edwin Hewitt and Karl Stromberg (GTM 25). It is part of the development of analysis.
