For $p = 1$, we obtain equality in (13.7.i) if and only if there is a positive measurable function $\varrho$ such that

$$f(x) \varrho(x) = g(x)$$

almost everywhere on the set $\{x : f(x) g(x) \neq 0\}$. Equality obtains for $1 < p < \infty$ if and only if $Af = Bg$ almost everywhere, where $A$ and $B$ are nonnegative real numbers such that $A^2 + B^2 > 0$.
