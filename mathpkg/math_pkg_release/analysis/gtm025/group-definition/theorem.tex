A set \(G\) together with a binary operation \((x, y) \mapsto xy\) mapping \(G \times G\) into \(G\) is called a \textbf{group} provided that:
\begin{enumerate}
\item[(i)] \(x(yz) = (xy)z\) for all \(x, y, z \in G\) (associative law);
\item[(ii)] there exists \(e \in G\) such that \(ex = x\) for all \(x \in G\) (left identity);
\item[(iii)] for each \(x \in G\) there exists \(x^{-1} \in G\) such that \(x^{-1}x = e\) (left inverse).
\end{enumerate}
If \(xy = yx\) for all \(x, y \in G\), the group is \textbf{Abelian} (commutative).
