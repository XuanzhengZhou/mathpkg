Let $f$ be an extended real-valued function defined on $X$. Suppose that $f^{-1}([a, \infty]) \in \mathcal{A}$ for every $a \in R$, i.e., $\{x \in X : a < f(x) \leq \infty\} \in \mathcal{A}$ for all real numbers $a$. Then $f$ is said to be an $\mathcal{A}$-measurable function. [The reader should notice that this definition closely resembles the definition of lower semicontinuity (7.21.d).] If $X$ is a topological space and $\mathcal{A}$ is the $\sigma$-algebra $\mathcal{B}(X)$ of Borel sets, then any $\mathcal{B}(X)$-measurable function is said to be Borel measurable. If $X = R$ and $\mathcal{A} = \mathcal{M}_{\lambda}$, then an $\mathcal{M}_{\lambda}$-measurable function is called a Lebesgue measurable function. [Notice that the definition of measurable function depends in no way upon any measure, but only upon a particular $\sigma$-algebra.]
