(21.10) Theorem. Notation is as in (21.9). The set function $\mu \times \nu$ is a [countably additive] $\sigma$-finite measure on $M \times N$. is a theorem from Chapter 21 (Hilbert Spaces and Spectral Theory) of the book *Real and Abstract Analysis* by Edwin Hewitt and Karl Stromberg (GTM 25). It is part of the development of functional-analysis.
