If $f$ is a complex-valued function defined on $[a, a + \delta[$, we say that $f$ is right differentiable at $a$ or has a right derivative at $a$ if

$$\lim_{k \downarrow 0} \frac{f(a + h) - f(a)}{h}$$

exists and is a complex number. Left derivatives and two-sided derivatives [which are simply called derivatives] are defined similarly. It is obvious that $f$ has a derivative at $a$ if and only if $\text{Re}f$ and $\text{Im}f$ have finite derivatives at $a$. Then

$$f'(a) = (\text{Re}f)'(a) + i(\text{Im}f)'(a).$$

(17.6) Remarks. (a) There is a slight solemnism in Definitions (17.4) and (17.5), since a real-valued function is certainly complex-valued, and $\pm \infty$ are admitted as values of $f'$ if $f$ is real-valued. The point should cause no trouble, however, and we will not bother with the special terminology that would be needed to remove the difficulty.

(b) If $f'_+(a)$ exists and is finite, then $\lim_{k \downarrow 0} f(a + h) = f(a)$. Similarly, if $f'_-(a)$ exists and is finite, then $\lim_{k \downarrow 0} f(a + h) = f(a)$. Hence if $f$ has a finite derivative at $a$, then it is continuous at $a$.
