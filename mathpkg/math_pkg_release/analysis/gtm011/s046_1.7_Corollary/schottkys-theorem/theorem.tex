For each $\alpha$ and $\beta$, $0 < \alpha < \infty$ and $0 \leq \beta \leq 1$, there is a constant $C(\alpha, \beta)$ such that if $f$ is an analytic function on some simply connected region containing $B(0; 1)$ that omits the values $0$ and $1$, and such that $|f(0)| \leq \alpha$; then $|f(z)| \leq C(\alpha, \beta)$ for all $|z| \leq \beta$.
