A topological space $(X, \mathcal{T})$ is called metrisable if there exists a metric $d$ on $X$ such that the topology $\mathcal{T}$ coincides exactly with the collection of sets that are open with respect to the metric $d$. Not every topological space is metrisable; the text provides an example of a non-metrisable topology by constructing a space where a point is a limit point of a set but no sequence in that set converges to it. Metrisability is an important property because metric spaces enjoy many convenient features (sequential continuity, first-countability, normality) that general topological spaces lack.
