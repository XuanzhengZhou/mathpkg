Let $\{0 = s_0 < s_1 < \ldots < s_n = 1\}$ and $\{0 = t_0 < t_1 < \ldots < t_n = 1\}$ be partitions of $[0, 1]$ such that for $0 \leq i, j \leq n - 1$,

$$F([s_i, s_{i+1}] \times [t_j, t_{j+1}])$$

is contained in a fundamental open set $\Delta_{ij}$ in $\Omega$. (Verify that this is possible using the Lebesgue covering lemma applied to the compact set $[0,1] \times [0,1]$ and the open cover obtained from the evenly covered neighborhoods of the covering space.)
