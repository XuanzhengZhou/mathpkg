Define $f: X \to \mathbb{R}$ by $f(y) = d(x, y)$. Then $f$ is continuous and, by Corollary 5.12, assumes a minimum value on $K$. That is, there is a point $y$ in $K$ with $f(y) \leq f(z)$ for every $z \in K$.
