The set $[a, b]$ is connected (an interval in $\mathbb{R}$), and the continuous image of a connected set is connected. Thus $f([a, b])$ is a connected subset of $\mathbb{R}$, which implies it is an interval. Since $f(a), f(b) \in f([a, b])$, the interval contains all points between them, including $\xi$.
