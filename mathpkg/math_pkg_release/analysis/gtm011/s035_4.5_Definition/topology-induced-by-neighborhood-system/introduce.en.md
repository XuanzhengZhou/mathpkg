Given a neighborhood system $\{\mathcal{N}_x : x \in X\}$ on a set $X$, one can define a topology $\mathcal{T}$ by declaring a set $U$ to be open if for each $x \in U$ there exists a neighborhood $V \in \mathcal{N}_x$ with $V \subset U$. This topology is called the topology induced by the neighborhood system. This construction, formalized in Proposition 4.17, provides a bridge between the neighborhood system approach and the standard open-set definition of a topological space, with the neighborhood system playing the role that open balls play in metric spaces.
