Proof. Suppose $f: [a, b] \to \mathbb{R}$ is a convex function and let $(x_1, y_1)$ and $(x_2, y_2)$ be points in $A$. If $0 \leq t \leq 1$ then, by the definition of convex function, $f(tx_2 + (1-t)x_1) \leq tf(x_2) + (1-t)f(x_1) \leq ty_2 + (1-t)y_1$. Thus $t(x_2, y_2) + (1-t)(x_1, y_1) = (tx_2 + (1-t)x_1, ty_2 + (1-t)y_1)$ is in $A$; so $A$ is convex.
