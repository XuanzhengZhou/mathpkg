If $\Gamma$ is a circle then an orientation for $\Gamma$ is an ordered triple of points $(z_1, z_2, z_3)$ such that each $z_j$ is in $\Gamma$.

Intuitively, these three points give a direction to $\Gamma$. That is we "go" from $z_1$ to $z_2$ to $z_3$. If only two points were given, this would, of course, be ambiguous.
