Let $A$ and $B$ be subsets of a metric space $(X, d)$. Then:
(a) $A$ is open if and only if $A = \text{int } A$;
(b) $A$ is closed if and only if $A = A^-$;
(c) int $A = X - (X - A)^-$; $A^- = X - \text{int } (X - A)$; $\partial A = A^- - \text{int } A$;
(d) $(A \cup B)^- = A^- \cup B^-$;
(e) $x_0 \in \text{int } A$ if and only if there is an $\epsilon > 0$ such that $B(x_0; \epsilon) \subset A$;
(f) $x_0 \in A^-$ if and only if for every $\epsilon > 0$, $B(x_0; \epsilon) \cap A \neq \square$.
