Let $\gamma$ be a rectifiable curve and suppose that $f$ is a function continuous on $\{\gamma\}$. Then:

(a) $\int_{\gamma} f = -\int_{-\gamma} f$;
(b) $\left| \int_{\gamma} f \right| \leq \int_{\gamma} |f| |dz| \leq V(\gamma) \sup \left[ |f(z)| : z \in \{\gamma\} \right]$;
(c) If $c \in \mathbb{C}$ then $\int_{\gamma} f(z) dz = \int_{\gamma + c} f(z - c) dz$.

The proof is left as an exercise.
