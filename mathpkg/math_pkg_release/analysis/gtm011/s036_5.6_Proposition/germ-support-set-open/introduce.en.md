Theorem 5.12 observes that if $F$ is the complete analytic function obtained from a function element $(f, D)$, then the set $G$ of all points $z$ for which some germ $[g]_z$ belongs to $F$ is an open subset of $\mathbb{C}$. This follows because each germ $[g]_z \in F$ is represented by a function $g$ analytic on a disk $B$ about $z$, and the entire disk $B$ is contained in $G$. The result underpins the sheaf-theoretic treatment of analytic continuation.
