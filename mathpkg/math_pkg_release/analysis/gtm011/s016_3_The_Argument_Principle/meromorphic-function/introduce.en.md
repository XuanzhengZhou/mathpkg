A meromorphic function on an open set is a function that is analytic except for isolated poles. This generalizes the notion of a rational function (which has finitely many poles) to allow poles that may accumulate only at the boundary of the domain. The set of all meromorphic functions on $G$, denoted $M(G)$, contains the analytic functions $H(G)$ as a subset and is closed under addition, multiplication, and division (when the denominator is not identically zero). Meromorphic functions are the natural setting for the Argument Principle and Rouché's Theorem.
