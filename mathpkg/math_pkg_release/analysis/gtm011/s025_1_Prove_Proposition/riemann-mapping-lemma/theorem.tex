Let $G$ be a region which is not the whole plane and such that every non-vanishing analytic function on $G$ has an analytic square root. If $a \in G$ then there is an analytic function $f$ on $G$ such that:
\begin{enumerate}
\item[(a)] $f(a) = 0$ and $f'(a) > 0$;
\item[(b)] $f$ is one-one;
\item[(c)] $f(G) = D = \{z : |z| < 1\}$.
\end{enumerate}
