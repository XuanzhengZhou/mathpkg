A collection of germs $\mathcal{F}$ is called a complete analytic function if there is a function element $(f, G)$ such that $\mathcal{F}$ is the set of all germs $[g]_b$ for which there is a point $a$ in $G$ with $[f]_a = [g]_a$ and there is a path in $\mathbb{C}$ from $a$ to $b$ such that $[g]_b$ is obtained from $[f]_a$ by analytic continuation along this path.

To make $\mathcal{F}$ a function, define
$$\mathcal{R} = \{(z, [f]_z): [f]_z \in \mathcal{F}\}$$
and define $\mathcal{F}: \mathcal{R} \to \mathbb{C}$ by $\mathcal{F}(z, [f]_z) = f(z)$.
