If $G$ is an open connected set in $\mathbb{C}$ and $f: G \to \mathbb{C}$ is a continuous function such that $z = \exp f(z)$ for all $z$ in $G$ then $f$ is a branch of the logarithm.

Notice that $0 \notin G$.
