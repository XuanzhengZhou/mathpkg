Let the metric $\rho$ be defined as in (1.4). If $\epsilon > 0$ is given then there is a $\delta > 0$ and a compact set $K \subset G$ such that for $f$ and $g$ in $C(G, \Omega)$,

1.8 $$\sup \{d(f(z), g(z)): z \in K\} < \delta \Rightarrow \rho(f, g) < \epsilon.$$

Conversely, if $\delta > 0$ and a compact set $K$ are given, there is an $\epsilon > 0$ such that for $f$ and $g$ in $C(G, \Omega)$,

1.9 $$\rho(f, g) < \epsilon \Rightarrow \sup \{d(f(z), g(z)): z \in K\} < \delta.$$
