Let $G$ be a region in $\mathbb{C}$. Define the conjugate region $G^* = \{\bar{z} : z \in G\}$. The region $G$ is said to be \text{symmetric with respect to the real axis} if $G = G^*$. For such a region, define
$$G_+ = \{z \in G : \text{Im } z > 0\}, \quad G_- = \{z \in G : \text{Im } z < 0\}, \quad G_0 = \{z \in G : \text{Im } z = 0\}.$$
If $G$ is connected and symmetric, then $G_0 = G \cap \mathbb{R}$ is nonempty and contains an open interval of the real line.
