This corollary guarantees that every zero of a non-identically-zero analytic function on a region has finite multiplicity. That is, near a zero $a$, the function can be factored as $f(z) = (z-a)^n g(z)$ where $g$ is analytic and $g(a) \neq 0$. The integer $n$ is called the multiplicity (or order) of the zero. If all derivatives at $a$ vanished, Theorem 3.7 would force $f \equiv 0$, so there must be a first non-vanishing derivative, which determines the multiplicity.
