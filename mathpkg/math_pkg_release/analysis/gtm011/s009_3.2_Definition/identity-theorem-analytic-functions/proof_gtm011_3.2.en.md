This follows by applying Theorem 3.7 to the analytic function $h = f - g$. If $\{z \in G : f(z) = g(z)\} = \{z \in G : h(z) = 0\}$ has a limit point in $G$, then by condition (c) of Theorem 3.7, $h \equiv 0$, so $f \equiv g$. Conversely, if $f \equiv g$ then $h \equiv 0$ and its zero set is all of $G$, which certainly has limit points in $G$.
