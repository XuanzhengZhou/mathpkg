(a) Let $G$ be a simply connected region, let $a \in G$, and let $f: G \to D = \{z: |z| < 1\}$ be a one-one analytic function such that $f(G) = D$ and $f(a) = 0$. Show that the Green's Function on $G$ with singularity at $a$ is $g_a(z) = -\log|f(z)|$.

(b) Find the Green's Functions for each of the following regions:

(i) $G = \mathbb{C} - (-\infty, 0]$;

(ii) $G = \{z: \text{Re } z > 0\}$;

(iii) $G = \{z: 0 < \text{Im } z < 2\pi\}$.
