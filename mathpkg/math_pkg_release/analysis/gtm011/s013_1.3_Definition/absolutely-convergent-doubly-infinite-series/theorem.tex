If $\{z_n : n = 0, \pm 1, \pm 2, \ldots\}$ is a doubly infinite sequence of complex numbers, $\sum_{n=-\infty}^{\infty} z_n$ is \textbf{absolutely convergent} if both $\sum_{n=0}^{\infty} z_n$ and $\sum_{n=1}^{\infty} z_{-n}$ are absolutely convergent. In this case
$$\sum_{n=-\infty}^{\infty} z_n = \sum_{n=0}^{\infty} z_n + \sum_{n=1}^{\infty} z_{-n}.$$
