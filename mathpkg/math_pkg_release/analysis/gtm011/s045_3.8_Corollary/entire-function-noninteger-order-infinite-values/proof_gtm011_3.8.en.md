If $\alpha \in f(\mathbb{C})$, apply the preceding theorem to $f - \alpha$. Since $f - \alpha$ is entire of order $\lambda$ (a non-integer), the Hadamard factorization theorem implies that $f - \alpha$ has infinitely many zeros. Hence $f$ assumes the value $\alpha$ infinitely many times.
