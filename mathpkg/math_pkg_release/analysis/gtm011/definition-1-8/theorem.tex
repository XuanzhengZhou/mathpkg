Let $\mathcal{F}$ be the set of all functions $f$ analytic on a region containing the closure of the disk $D = \{z: |z| < 1\}$ and satisfying $f(0) = 0$, $f'(0) = 1$. For each $f$ in $\mathcal{F}$ let $\beta(f)$ be the supremum of all numbers $r$ such that there is a disk $S$ in $D$ on which $f$ is one-one and such that $f(S)$ contains a disk of radius $r$. (So $\beta(f) \geq \frac{1}{72}$). Bloch's constant is the number $B$ defined by

$$B = \inf \{ \beta(f) : f \in \mathcal{F} \}.$$

According to Bloch's Theorem, $B \geq \frac{1}{72}$. If one considers the function $f(z) = z$ then clearly $B \leq 1$. However, better estimates than these are known. In fact, it is known that $.43 \leq B \leq .47$. Although the exact value of $B$ remains unknown, it has been conjectured that

$$B = \frac{\Gamma\left(\frac{1}{3}\right) \Gamma\left(\frac{11}{12}\right)}{(1 + \sqrt{3})^{\frac{1}{2}} \Gamma\left(\frac{1}{4}\right)}.$$
