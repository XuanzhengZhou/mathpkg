This corollary follows from the Inverse Function Theorem for analytic functions (Proposition 2.20) by taking $g(z) = e^z$, which is entire with non-vanishing derivative. Since $\exp(\log z) = z$ for any branch of the logarithm, Proposition 2.20 implies the branch is analytic. Differentiating the identity $g(f(z)) = z$ yields $g'(f(z)) f'(z) = 1$, giving $f'(z) = 1/z$.
