The \textbf{principal branch of the logarithm} is the branch of the logarithm function defined on $\mathbb{C} \setminus \{z \in \mathbb{R} : z \leq 0\}$ (the complex plane slit along the non-positive real axis). It is denoted by $\log z$, and unless otherwise stated, $\log z$ always refers to the principal branch.

For $z = re^{i\theta}$ with $r > 0$ and $-\pi < \theta < \pi$, the principal branch is given by

$$\log z = \ln r + i\theta.$$
