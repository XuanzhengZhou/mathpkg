The Perron function associated with a continuous boundary function $f$ is defined as the pointwise supremum of all subharmonic functions whose limsup on the boundary does not exceed $f$. This theorem proves that the Perron function is always harmonic, regardless of whether the Dirichlet Problem is solvable for the region. It constitutes the first half of Perron's method for solving the Dirichlet Problem; the second half is showing that the Perron function attains the boundary values, which requires additional geometric conditions (barriers).
