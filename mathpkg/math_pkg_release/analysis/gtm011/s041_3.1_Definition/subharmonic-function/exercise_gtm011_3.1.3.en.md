(This exercise is difficult.) If $G$ is a region and if $f: \partial_\infty G \to \mathbb{R}$ is a continuous function, let $u_f$ be the Perron Function associated with $f$. This defines a map $T: C(\partial_\infty G; \mathbb{R}) \to \operatorname{Har}(G)$ by $T(f) = u_f$. Prove:

(a) $T$ is linear (i.e., $T(a_1 f_1 + a_2 f_2) = a_1 T(f_1) + a_2 T(f_2)$).

(b) $T$ is positive (i.e., if $f(a) \geq 0$ for all $a \in \partial_\infty G$ then $T(f)(z) \geq 0$ for all $z \in G$).

(c) $T$ is continuous. Moreover, if $\{f_n\}$ is a sequence in $C(\partial_\infty G; \mathbb{R})$ such that $f_n \to f$ uniformly, then $T(f_n) \to T(f)$ uniformly on $G$.

(d) If the Dirichlet Problem can be solved for $G$, then $T$ is one-one. Is the converse true?
