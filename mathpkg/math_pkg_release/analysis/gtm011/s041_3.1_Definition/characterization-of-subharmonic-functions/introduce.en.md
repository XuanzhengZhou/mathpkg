This theorem gives an equivalent characterization of subharmonicity: a continuous function is subharmonic precisely when subtracting any harmonic function yields a function that satisfies the Maximum Principle. The forward direction follows from the fact that the difference of subharmonic functions is subharmonic; the reverse direction uses the solvability of the Dirichlet Problem on a disk (via the Poisson integral) to compare $\varphi$ with the harmonic function having the same boundary values.
