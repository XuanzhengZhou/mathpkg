Let $G$ be a region and let $\varphi: G \to \mathbb{R}$ be a continuous function. $\varphi$ is a \textbf{superharmonic function} if whenever $\bar{B}(a; r) \subset G$,

$$\varphi(a) \geq \frac{1}{2\pi} \int_{0}^{2\pi} \varphi(a + re^{i\theta}) \, d\theta.$$

Equivalently, $\varphi$ is superharmonic if and only if $-\varphi$ is subharmonic.
