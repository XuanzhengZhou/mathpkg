Let $(X, \rho)$ be a covering space of the space $\Omega$. If $F: [0, 1] \times [0, 1] \rightarrow \Omega$ is a continuous function with $F(0, 0) = \omega_0$ and if $x_0$ is any point in $X$ with $\rho(x_0) = \omega_0$, then there is a unique continuous function $\tilde{F}: [0, 1] \times [0, 1] \rightarrow X$ such that $\tilde{F}(0, 0) = x_0$ and $\rho \circ \tilde{F} = F$.
