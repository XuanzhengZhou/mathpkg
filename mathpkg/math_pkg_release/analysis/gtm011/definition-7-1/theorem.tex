If $\Omega$ is a topological space then a covering space of $\Omega$ is a pair $(X, \rho)$ where $X$ is a connected topological space and $\rho$ is a continuous function of $X$ onto $\Omega$ such that: for each $\omega$ in $\Omega$ there is a neighborhood $\Delta$ of $\omega$ such that each component of $\rho^{-1}(\Delta)$ is open, and $\rho$ maps each of these components homeomorphically onto $\Delta$. Such an open set $\Delta$ is called fundamental and $\Delta$ is properly covered.

Both $(\mathbb{C}, \exp)$ and $(\mathbb{C} - \{0\}, z^n)$ are covering spaces
