A function $\gamma: [a, b] \to \mathbb{C}$, for $[a, b] \subset \mathbb{R}$, is of \textit{bounded variation} if there is a constant $M > 0$ such that for any partition $P = \{a = t_0 < t_1 < \ldots < t_m = b\}$ of $[a, b]$,

$$v(\gamma; P) = \sum_{k=1}^{m} |\gamma(t_k) - \gamma(t_{k-1})| \leq M.$$

The \textit{total variation} of $\gamma$, $V(\gamma)$, is defined by

$$V(\gamma) = \sup \{v(\gamma; P): P \text{ a partition of } [a, b]\}.$$
