Let $\gamma_0, \gamma_1 : [0, 1] \to G$ be two closed rectifiable curves in a region $G$; then $\gamma_0$ is \textit{homotopic} to $\gamma_1$ in $G$ if there is a continuous function $\Gamma : [0, 1] \times [0, 1] \to G$ such that
$$
\begin{cases}
\Gamma(s, 0) = \gamma_0(s) & \text{and} \quad \Gamma(s, 1) = \gamma_1(s) \quad (0 \leq s \leq 1) \\
\Gamma(0, t) = \Gamma(1, t) \quad (0 \leq t \leq 1).
\end{cases}
$$
We write $\gamma_0 \sim \gamma_1$.
