Let $G = \{z = x+iy: a < x < b\}$ be a vertical strip in $\mathbb{C}$ and let $G^-$ be its closure. Suppose $f: G^- \to \mathbb{C}$ is continuous and $f$ is analytic in $G$. Define $M: [a, b] \to \mathbb{R}$ by

$$M(x) = \sup \{ |f(x+iy)| : -\infty < y < \infty \},$$

and suppose $|f(z)| < B$ for all $z$ in $G$ for some constant $B$. Then $\log M(x)$ is a convex function on $[a, b]$.
