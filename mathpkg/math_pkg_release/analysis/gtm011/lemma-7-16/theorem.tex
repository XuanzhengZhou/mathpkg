Let $S = \{z : a \leq \text{Re } z \leq A\}$ where $0 < a < A < \infty$.

(a) For every $\epsilon > 0$ there is a $\delta > 0$ such that for all $z$ in $S$

$$\left| \int_{\alpha}^{\beta} e^{-t} t^{z-1} dt \right| < \epsilon$$

whenever $0 < \alpha < \beta < \delta$.

(b) For every $\epsilon > 0$ there is a number $\kappa$ such that for all $z$ in $S$

$$\left| \int_{\alpha}^{\beta} e^{-t} t^{z-1} dt \right| < \epsilon$$

whenever $\beta > \alpha > \kappa$.
