The zeta function can be defined to be meromorphic in the plane with only a simple pole at $z = 1$ and $\text{Res}(\zeta; 1) = 1$. For $z \neq 1$ $\zeta$ satisfies Riemann's functional equation.

Since $\Gamma(1-z)$ has a pole at $z = 1, 2, \ldots$ and since $\zeta$ is analytic at $z = 2, 3, \ldots$ we know, from Riemann's functional equation, that

$$\zeta(1-z) \sin \left(\frac{1}{2} \pi z\right) = 0$$

for $z = 2, 3, \ldots$. Furthermore, since the pole of $\Gamma(1-z)$ at $z = 2, 3, \ldots$ is simple, each of the zeros of (8.15) must be simple. Since $\sin \left(\frac{1}{2} \pi z\right) = 0$ whenever $z$ is an even integer, $\zeta(1-z) = 0$ for $z = 3, 5, \ldots$. That is $\zeta(z) = 0$ for $z = -2, -4, -6, \ldots$. Similar reasoning gives that $\zeta$ has no other zeros outside the closed strip $\{z: 0 \leq \text{Re} z \leq 1\}$.
