Let $G \subset \mathbb{C}$ be open and let $H(G)$ be the collection of analytic functions on $G$. A set $\mathcal{F} \subset H(G)$ is called \textbf{locally bounded} if for each point $a \in G$ there exist constants $M > 0$ and $r > 0$ such that for all $f \in \mathcal{F}$,

$$|f(z)| \leq M \quad \text{for } |z-a| < r.$$

Equivalently, $\mathcal{F}$ is locally bounded if for each $a \in G$ there exists $r > 0$ such that

$$\sup \{|f(z)| : |z-a| < r,\; f \in \mathcal{F}\} < \infty.$$

That is, about each point $a \in G$ there is a disk on which $\mathcal{F}$ is uniformly bounded.
