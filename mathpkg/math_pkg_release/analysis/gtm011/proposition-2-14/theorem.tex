Let $f$ be an entire function of finite order $\lambda$. If $\epsilon > 0$ then $|f(z)| < \exp (|z|^{\lambda+\epsilon})$ for all $z$ with $|z|$ sufficiently large; and $a$ $z$ can be found, with $|z|$ as large as desired, such that $|f(z)| \geq \exp (|z|^{\lambda-\epsilon})$.

Although the definition of order seems a priori weaker than the conclusion of Theorem 2.6, they are, in fact, equivalent. The reader is asked to show this for himself in Exercise 3

Exercises

1. Let $f(z) = \sum c_n z^n$ be an entire function of finite genus $\mu$; prove that

$$\lim_{n \to \infty} c_n (n!)^{1/(\mu+1)} = 0.$$

(Hint: Use Cauchy's Estimate.)

2. Let $f_1$ and $f_2$ be entire functions of finite orders $\lambda_1$ and $\lambda_2$ respectively. Show that $f = f_1 + f_2$ has finite order $\lambda$ and $\lambda \leq \max (\lambda_1, \lambda_2)$. Show that $\lambda = \max (\lambda_1, \lambda_2)$ if $\lambda_1 \neq \lambda_2$ and give an example which shows that $\lambda < \max (\lambda_1, \lambda_2)$ with $f \neq 0$.

3. Suppose $f$ is an entire function and $A, B, a$ are positive constants such that there is a $r_0$ with $|f(z)| \leq \exp(A|z|^a + B)$ for $|z| > r_0$. Show that $f$ is of finite order $\leq a$.

4. Prove that if $f$ is an entire function of order $\lambda$ then $f'$ also has order $\lambda$.

5. Let $f(z) = \sum c_n z^n$ be an entire function and define the number $\alpha$ by

$$\alpha = \liminf_{n \to \infty} \frac{-\log |c_n|}{n \log n}$$
