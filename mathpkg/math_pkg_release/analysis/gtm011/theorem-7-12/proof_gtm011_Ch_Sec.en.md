Proof. It was shown above that if $n(\gamma; 0) = n(\sigma; 0)$ then $\gamma \sim \sigma$. Conversely, if $\gamma \sim \sigma$ then the Abstract Monodromy Theorem implies that the liftings $\tilde{\gamma}$ and $\tilde{\sigma}$ such that $\tilde{\gamma}(0) = \tilde{\sigma}(0) = 0$ have the same end point. That is, $n(\gamma; 0) = n(\sigma; 0)$.
