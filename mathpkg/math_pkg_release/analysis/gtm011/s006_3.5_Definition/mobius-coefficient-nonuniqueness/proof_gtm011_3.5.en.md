For any $\lambda \in \mathbb{C} \setminus \{0\}$,

$$\frac{(\lambda a)z + (\lambda b)}{(\lambda c)z + (\lambda d)} = \frac{\lambda(az + b)}{\lambda(cz + d)} = \frac{az + b}{cz + d} = S(z),$$

since the factor $\lambda$ cancels in both numerator and denominator. Hence the four coefficients are only defined up to a common scalar multiple.
