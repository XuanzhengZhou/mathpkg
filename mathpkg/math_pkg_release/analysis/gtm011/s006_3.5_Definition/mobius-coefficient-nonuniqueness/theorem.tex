Let $S(z) = \frac{az+b}{cz+d}$ be a Möbius transformation. For any non-zero complex number $\lambda$, the same mapping is given by
$$S(z) = \frac{(\lambda a)z + (\lambda b)}{(\lambda c)z + (\lambda d)}.$$
Thus the coefficients $a, b, c, d$ are not unique; they are determined only up to a common non-zero scalar multiple.
