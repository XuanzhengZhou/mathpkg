A set $\mathcal{F} \subset H(G)$ is locally bounded if for each point $a$ in $G$ there are constants $M$ and $r > 0$ such that for all $f$ in $\mathcal{F}$,

$$|f(z)| \leq M, \text{ for } |z-a| < r.$$

Alternately, $\mathcal{F}$ is locally bounded if there is an $r > 0$ such that

$$\sup \{|f(z)|: |z-a| < r, f \in \mathcal{F}\} < \infty.$$

That is, $\mathcal{F}$ is locally bounded if about each point $a$ in $G$ there is a disk on which $\mathcal{F}$ is uniformly bounded. This immediately extends to the requirement that $\mathcal{F}$ be uniformly bounded on compact sets in $G$.
