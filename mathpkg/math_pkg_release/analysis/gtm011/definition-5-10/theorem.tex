An elementary factor is one of the following functions $E_p(z)$ for $p = 0, 1, \ldots$:

$$E_0(z) = 1 - z,$$
$$E_p(z) = (1 - z) \exp \left( z + \frac{z^2}{2} + \ldots + \frac{z^p}{p} \right), \quad p \geq 1.$$

The function $E_p(z/a)$ has a simple zero at $z = a$ and no other zero. Also if $b$ is a point in $\mathbb{C} - G$ then $E_p\left(\frac{a-b}{z-b}\right)$ has a simple zero at $z = a$ and is analytic in $G$. These functions will be used to manufacture analytic functions with prescribed zeros of prescribed multiplicity, but first an inequality must be proved which will enable us to apply Theorem 5.9 and obtain a convergent infinite product.
