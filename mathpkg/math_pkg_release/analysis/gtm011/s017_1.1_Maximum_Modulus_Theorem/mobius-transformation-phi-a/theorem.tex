If $|a| < 1$, the Möbius transformation $\varphi_a$ is defined by

$$\varphi_a(z) = \frac{z - a}{1 - \bar{a}z}.$$

$\varphi_a$ is analytic for $|z| < |a|^{-1}$ and satisfies $\varphi_a(\varphi_a(z)) = z$ for $|z| < 1$, so it is an involution mapping the open unit disk $D = \{z : |z| < 1\}$ one-to-one onto itself. For any real $\theta$, $|\varphi_a(e^{i\theta})| = 1$, so $\varphi_a$ maps the unit circle onto itself.
