Let $G$ be a region in $\mathbb{C}$ and $f$ an analytic function on $G$. Suppose there is a constant $M$ such that

$$\limsup_{z \to a} |f(z)| \leq M$$

for all $a$ in $\partial_{\infty} G$. Then $|f(z)| \leq M$ for all $z$ in $G$. Moreover, if $|f(a)| = M$ for some $a$ in $G$, then $f$ is a constant function.
