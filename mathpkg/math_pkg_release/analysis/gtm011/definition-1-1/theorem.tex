If $G$ is an open subset of $\mathbb{C}$ then a function $u: G \to \mathbb{R}$ is harmonic if $u$ has continuous second partial derivatives and

$$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0.$$

This equation is called LAPLACE’S EQUATION.

We also review the following facts about harmonic functions.

(1) (Theorem III.2.29) A function $f$ on a region $G$ is analytic iff $\text{Re } f = u$ and $\text{Im } f = v$ are harmonic functions which satisfy the Cauchy-Riemann equations.

(2) (Theorem VIII.2.2(j)) A region $G$ is simply connected iff for each harmonic function $u$ on $G$ there is a harmonic function $v$ on $G$ such that $f = u + iv$ is analytic on $G$.
