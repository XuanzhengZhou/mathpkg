1.1 Definition from Conway's "Functions of One Complex Variable I."
If $G$ is an open subset of $\mathbb{C}$ then a function $u: G \to \mathbb{R}$ is harmonic if $u$ has continuous second partial derivatives and

$$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0.$$

This equation is called LAPLACE’S EQUATION.
