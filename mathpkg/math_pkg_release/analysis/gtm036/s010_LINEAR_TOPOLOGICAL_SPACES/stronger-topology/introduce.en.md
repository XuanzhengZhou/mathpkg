A topology $\mathcal{T}$ is stronger (or finer) than $\mathcal{U}$ if every $\mathcal{U}$-open set is also $\mathcal{T}$-open, i.e., $\mathcal{T} \supseteq \mathcal{U}$. Stronger topologies have more open sets, making it easier for nets to converge and harder for spaces to be compact. The collection of all topologies on a fixed set $X$ forms a complete lattice ordered by inclusion.
