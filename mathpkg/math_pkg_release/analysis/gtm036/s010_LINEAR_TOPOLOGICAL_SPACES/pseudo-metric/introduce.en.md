A pseudo-metric on a set $X$ is a distance function satisfying symmetry, non-negativity, and the triangle inequality, but allowing distinct points to have distance zero. Pseudo-metrics generalize metrics by relaxing the positivity condition $d(x,y) = 0 \implies x = y$. Every pseudo-metric induces a topology whose base consists of open balls $B(x,r) = \{y : d(x,y) < r\}$, but the topology does not determine the pseudo-metric.
