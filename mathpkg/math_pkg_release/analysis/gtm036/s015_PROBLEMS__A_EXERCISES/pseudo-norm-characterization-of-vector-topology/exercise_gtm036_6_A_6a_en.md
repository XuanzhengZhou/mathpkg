(a) If $d$ is an invariant pseudo-metric for $E$ such that each sphere about $0$ is circled and $q(x) = d(x,0)$, then $q$ has the following properties:

(i) $q(x) \geq 0$ for all $x \in E$ and $q(0) = 0$;
(ii) $q(x + y) \leq q(x) + q(y)$ for all $x, y \in E$;
(iii) $q(ax) \leq q(x)$ whenever $|a| \leq 1$;
(iv) $\lim_{n \to \infty} q(x/n) = 0$ for all $x \in E$.

Conversely, if $q$ is a functional on $E$ satisfying (i) through (iv), then $d(x,y) = q(x - y)$ is a pseudo-metric for $E$ defining a vector topology.
