A sufficient condition for a cone to be generating is that its radial kernel is non-empty. The radial kernel of a convex set is the set of points from which the set is absorbing in every direction. When the cone has this property, it necessarily spans the entire space, meaning every element of $E$ can be written as a difference of two cone elements.
