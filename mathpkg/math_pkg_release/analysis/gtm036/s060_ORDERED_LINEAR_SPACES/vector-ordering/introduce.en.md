A vector ordering is a partial ordering on a real linear space that is compatible with the linear structure. Specifically, it is translation-invariant (order is determined by the positive elements), homogeneous with respect to non-negative scalars, and additive on the positive cone. There is a natural one-to-one correspondence between vector orderings and cones: every cone determines a vector ordering, and conversely, the set of non-negative elements of a vector ordering forms a cone.
