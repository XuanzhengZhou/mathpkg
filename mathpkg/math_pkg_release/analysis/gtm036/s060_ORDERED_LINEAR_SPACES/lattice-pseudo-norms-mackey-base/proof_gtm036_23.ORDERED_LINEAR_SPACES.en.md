This result is stated in the introductory summary without proof. The full proof appears later in the chapter. The key idea is that every $m(E, E^\star)$-continuous pseudo-norm can be shown to be equivalent (in the sense of generating the same topology) to a lattice pseudo-norm, using the fact that the order dual $E^\star$ consists precisely of functionals that are differences of positive functionals, and these functionals are bounded on order-bounded sets.
