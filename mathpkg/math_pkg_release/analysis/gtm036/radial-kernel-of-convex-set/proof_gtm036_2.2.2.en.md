Let $v = ax + (1 - a)y$, with $0 < a \leq 1$. For any fixed $z$ in $E$ there is a positive number $r$ such that $x + bz \in A$ whenever $0 < b < r$. Since $A$ is convex, $a(x + bz) + (1 - a)y = v + abz \in A$ for all $b$ such that $0 < b < r$. Hence $v + cz \in A$ if $0 < c < ar$, and it follows that $v$ is in the radial kernel of $A$.
