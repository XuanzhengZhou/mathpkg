For each $p$ with $1 \leq p < \infty$, there exists a Fr\'{e}chet Montel space $E_{p}$ which contains a closed subspace $F_{p}$ such that the quotient $E_{p}/F_{p}$ is topologically isomorphic to $l_{p}$.

Explicitly, let $I$ be the set of positive integers, let $A = I \times I$, and let $\Omega = \{\omega_{n} : n = 1, 2, \cdots\}$ where
$$\omega_{n}(i,j) = \begin{cases} j^{n} & \text{if } i \leq n \\ i^{n} & \text{if } i > n. \end{cases}$$
Define $E_{p} = E^{p}(\Omega)$ as in the construction of 22F.
