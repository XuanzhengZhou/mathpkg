Let $E$ be a linear space over $\mathbb{R}$. The \textbf{open line segment} joining two points $x, y \in E$, denoted by $(x:y)$, is the set of all points of the form $a x + (1 - a) y$ where $a \in \mathbb{R}$ and $0 < a < 1$. That is,
$$
(x : y) = \{ a x + (1 - a) y \mid a \in \mathbb{R},\ 0 < a < 1 \} = [x : y] \setminus \{x, y\}.
$$
