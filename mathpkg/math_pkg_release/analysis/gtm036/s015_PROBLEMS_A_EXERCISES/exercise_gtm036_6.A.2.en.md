If $E$ is a real linear space and $\mathcal{T}$ is a topology for $E$ such that the family of all convex neighborhoods of $0$ is a base for the neighborhood system of $0$, and if $x + y$ and $ax$ are continuous in each variable separately, then $\mathcal{T}$ is a vector topology.
