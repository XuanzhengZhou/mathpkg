A subset $A$ of a linear space $E$ is midpoint convex iff $\frac{1}{2}(x + y)$ is in $A$ whenever $x$ and $y$ are in $A$.

(a) The following conditions are equivalent:

(i) $A$ is midpoint convex;
(ii) $A + A = 2A$;
(iii) $a_1A + a_2A + \cdots + a_nA = A$ whenever $a_1, a_2, \cdots, a_n$ is any finite collection of non-negative dyadic rationals whose sum is $1$.

(b) A midpoint convex set of real numbers is convex if it is either open or closed.
