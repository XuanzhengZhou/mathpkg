A direct consequence of Theorem 2.2 is that the radial kernel of any convex set is itself convex. Moreover, the radial kernel operator is idempotent on convex sets: taking the radial kernel of the radial kernel yields the same set. This makes the radial kernel a well-behaved algebraic analogue of the topological interior, which is similarly idempotent.
