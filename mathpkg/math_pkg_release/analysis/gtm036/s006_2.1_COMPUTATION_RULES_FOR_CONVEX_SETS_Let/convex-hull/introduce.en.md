The convex hull of a subset $A$ of a linear space is the smallest convex set containing $A$. It admits an explicit description as the set of all finite convex combinations of points from $A$ — that is, all finite sums $\sum a_i x_i$ with non-negative real coefficients $a_i$ summing to one. The notation $\langle A \rangle$ is used throughout the text to denote the convex hull. This construction is fundamental to the study of convexity in linear topological spaces.
