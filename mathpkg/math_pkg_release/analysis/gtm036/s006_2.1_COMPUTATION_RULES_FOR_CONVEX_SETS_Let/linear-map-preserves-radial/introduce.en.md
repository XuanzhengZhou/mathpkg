The property of a set being radial at a point is preserved under surjective linear maps: if $A$ is radial at $x$ and $f$ is a linear map onto $F$, then the image $f[A]$ is radial at $f(x)$. This fact is a purely algebraic observation that follows directly from the definitions and the surjectivity of $f$. It is useful for transporting radial properties between spaces, particularly when studying quotient spaces and linear functionals.
