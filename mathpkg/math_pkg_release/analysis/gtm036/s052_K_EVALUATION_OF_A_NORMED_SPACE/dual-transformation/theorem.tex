\text{Each continuous linear transformation } T \colon E \to F \text{ of a linear topological space } E \text{ into a linear topological space } F \text{ induces, in a natural way, an adjoint (dual) transformation } T^* \colon F^* \to E^* \text{ defined on the dual spaces. The transformation } T^* \text{ is given by } \langle x, T^*(f) \rangle = \langle T(x), f \rangle \text{ for all } x \in E,\ f \in F^*.
