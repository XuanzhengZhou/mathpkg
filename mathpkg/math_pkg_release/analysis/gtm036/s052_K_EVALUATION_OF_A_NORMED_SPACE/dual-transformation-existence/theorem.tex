\text{Let } E \text{ and } F \text{ be paired linear spaces and let } G \text{ and } H \text{ be paired linear spaces. A linear transformation } T \colon E \to G \text{ has a dual } T' \colon H \to F \text{ if and only if } T \text{ is } w(E,F)\text{-}w(G,H) \text{ continuous. If } T \text{ has a dual } T', \text{ then } T' \text{ is necessarily } w(H,G)\text{-}w(F,E) \text{ continuous. Moreover, the dual } T' \text{ of } T \text{ is unique if and only if } E \text{ distinguishes members of } F.
