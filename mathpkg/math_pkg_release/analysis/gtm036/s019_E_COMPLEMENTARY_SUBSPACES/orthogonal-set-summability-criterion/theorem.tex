If a subset $\{x_\alpha : \alpha \in A\}$ of a Hilbert space is orthogonal (any two distinct elements are orthogonal), then it is summable if and only if the subset $\{\|x_\alpha\|^2 : \alpha \in A\}$ of the real line is summable. Moreover, if $x = \sum \{x_\alpha : \alpha \in A\}$, then $\|x\|^2 = \sum \{\|x_\alpha\|^2 : \alpha \in A\}$.
