If a subset $\{x_\alpha : \alpha \in A\}$ of a Hilbert space is summable, then the set of indices $\alpha$ for which $x_\alpha \neq 0$ is countable. Furthermore, if those indices $\alpha$ for which $x_\alpha \neq 0$ are enumerated arbitrarily as $\alpha_1, \alpha_2, \cdots$, then $\sum \{x_\alpha : \alpha \in A\} = \lim_{n \to \infty} \sum \{x_{\alpha_r} : 1 \leq r \leq n\}$.
