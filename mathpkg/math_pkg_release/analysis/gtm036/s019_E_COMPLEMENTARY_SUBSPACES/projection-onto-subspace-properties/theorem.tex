If $F$ is a closed linear subspace of a Hilbert space $H$, then the projection $P$ onto $F$ is a continuous linear transformation with the properties:
\begin{enumerate}
\item[(i)] $P \circ P = P$ (that is, $P$ is idempotent).
\item[(ii)] $(P(x), y) = (x, P(y))$ for all $x, y \in H$ (that is, $P$ is self-adjoint).
\item[(iii)] $F = \{x : P(x) = x\}$.
\end{enumerate}
