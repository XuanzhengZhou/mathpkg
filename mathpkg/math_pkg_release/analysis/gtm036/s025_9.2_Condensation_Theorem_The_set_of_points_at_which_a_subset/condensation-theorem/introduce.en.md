The Condensation Theorem provides a structural description of the set of points at which a subset of a topological space is of the second category. This set is exactly the closure of an open set $V$, and the part of $A$ outside $V$ is of the first category. It is a fundamental result in Baire category theory connecting local category properties to global topological structure.
