For a pointwise convergent sequence of continuous functions from a topological space to a pseudo-metric space, the set of points where the sequence is equicontinuous forms a residual set (its complement is of the first category). Consequently, the pointwise limit function is continuous at almost every point in the sense of Baire category. The proof uses the Baire category theorem via the construction of closed sets $K_{n,k}$ expressing approximate equicontinuity conditions.
