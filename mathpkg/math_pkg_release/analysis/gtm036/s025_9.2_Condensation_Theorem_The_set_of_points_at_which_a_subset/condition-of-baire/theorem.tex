A subset $A$ of a topological space $X$ \text{satisfies the condition of Baire} if and only if there is an open set $U$ such that both $U \setminus A$ and $A \setminus U$ are of the first category in $X$. Equivalently, $A$ differs from an open set by a set of the first category.

Every closed set satisfies the condition of Baire, because a closed set equals the union of its interior and its boundary, and the latter is nowhere dense. The complement of any set satisfying the condition also satisfies it. The family of all sets satisfying the condition is closed under countable unions and countable intersections. In particular, every Borel set satisfies the condition of Baire.
