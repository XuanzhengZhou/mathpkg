In 17.7 it is proved that a locally convex space $E$ is complete if and only if every hyperplane in its adjoint is weak* closed whenever its intersection with every equicontinuous set $C$ is weak* closed in $C$. The Krein-\v{S}mulian theorem asserts that the same is true with ``convex set'' in place of ``hyperplane'' if $E$ is metrizable (see 22.6). Hypercompleteness (see 13F) is equivalent to the same property, this time for convex circled sets: a locally convex space is hypercomplete if and only if every convex circled subset of its adjoint is weak* closed whenever its intersection with every equicontinuous set is weak* closed in that set.
