Let $\{e_i: i = 1, 2, \dots\}$ be a linearly independent set in $E$. Define

$$A_n = \left\{\sum_{i=1}^{n} a_i e_i : |a_i| \leq n^{-1},\; i = 1, \dots, n\right\}$$

and let $A = \bigcup_{n=1}^{\infty} A_n$. Let $B$ be a subspace complementary to the subspace generated by $A$. Then there is no circled set $C$ radial at $0$ such that $C + C \subset A + B$.

Since any local base for a vector topology must contain, for each neighborhood $U$ of $0$, a circled neighborhood $C$ of $0$ with $C + C \subset U$, and $A + B$ is a neighborhood of $0$ in the putative topology (being a union of sets each of which is circled and radial at $0$), the failure to find such a $C$ shows that the family of all circled sets radial at $0$ cannot be a local base for any vector topology.
