In an infinite-dimensional linear space, one might hope to construct a strongest vector topology by taking all circled sets that are radial at the origin as a local base. This proposition shows that this naive approach fails: the family of all circled, radial-at-0 sets does not form a local base for any vector topology. The proof constructs a specific counterexample using a linearly independent set and a carefully chosen union to violate the additive property required of a vector topology local base.
