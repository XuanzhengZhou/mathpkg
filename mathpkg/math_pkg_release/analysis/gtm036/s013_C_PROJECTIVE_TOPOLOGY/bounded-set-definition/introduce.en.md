In a linear topological space, a set is bounded if it is absorbed by every neighborhood of the origin. This generalizes the familiar notion of boundedness in normed spaces, where a set is bounded if it is contained in some ball. Bounded sets play a critical role in the characterization of normable spaces: a linear topological space is pseudo-normable precisely when it possesses a bounded convex neighborhood of zero.
