A linear topological space $E$ is \textbf{evaluable} (also called \textit{symmetric}, \textit{infratonnel\'e}, \textit{quasi-tonnel\'e}, or \textit{quasi-barrelled}) if and only if $E$ is locally convex and the evaluation map $I \colon E \to E^{**}$ is continuous.

Equivalently, $E$ is evaluable if and only if each bound absorbing barrel in $E$ is a neighborhood of $0$, or equivalently, if and only if each strongly bounded subset of $E^*$ is equicontinuous.
