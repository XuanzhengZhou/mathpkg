In a linear topological space $E$, any midpoint convex set that contains a subset of the second category with the Baire property necessarily has non-empty interior. This corollary follows from the main theorem on midpoint convex neighborhoods by dropping the requirement that the contained second-category Baire set be symmetric. It provides a useful sufficient condition for a midpoint convex set to have interior points, with applications in the study of topological vector spaces.
