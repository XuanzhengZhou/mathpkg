In a linear topological space, a midpoint convex set $A$ (satisfying $A + A = 2A$) that contains a symmetric subset of the second category with the Baire property must be a neighborhood of the origin. This theorem is a fundamental result linking algebraic conditions (midpoint convexity) with topological conclusions (being a neighborhood) via category arguments. It serves as a crucial tool for deducing continuity and openness properties in linear topological spaces.
