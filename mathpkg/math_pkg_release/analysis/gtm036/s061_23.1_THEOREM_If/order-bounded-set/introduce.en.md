In an ordered linear space, a subset $A$ is order bounded if it is both bounded above and bounded below with respect to the vector ordering. That is, there exist elements $x$ and $y$ such that $x \leq a \leq y$ for every $a \in A$. This concept is fundamental to the study of ordered topological vector spaces and vector lattices, where it interacts with topological boundedness in important ways.
