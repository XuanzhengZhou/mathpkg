Let $(E, \mathcal{T})$ be a linear topological space.
\begin{enumerate}
\item[(i)] If $A$ and $B$ are bounded subsets of $E$ and $a$ is a scalar, then each of the sets $aA$, $A^{-}$, $A + B$, and $A \cup B$ is bounded.
\item[(ii)] A sufficient condition that a linear function be continuous is that the image of some neighborhood of $0$ be bounded. This condition is also necessary if the range space is pseudo-normable.
\item[(iii)] Each continuous linear function is bounded. If the domain space is pseudo-normable, then a linear function is bounded if and only if it is continuous.
\end{enumerate}
