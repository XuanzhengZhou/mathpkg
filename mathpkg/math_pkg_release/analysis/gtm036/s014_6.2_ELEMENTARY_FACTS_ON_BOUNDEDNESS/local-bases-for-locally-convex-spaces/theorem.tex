In each locally convex linear topological space $E$ there is a local base $\mathcal{U}$ such that
\begin{enumerate}
\item[(i)] the sets in $\mathcal{U}$ are convex circled sets each of which is radial at $0$;
\item[(ii)] the intersection of two sets in $\mathcal{U}$ contains a set in $\mathcal{U}$;
\item[(iii)] for each $U$ in $\mathcal{U}$ and each scalar $a \neq 0$, $aU \in \mathcal{U}$; and
\item[(iv)] each set in $\mathcal{U}$ is closed.
\end{enumerate}
Conversely, if $E$ is a linear space, then any non-void family $\mathcal{U}$ of subsets which satisfies (i), (ii), and (iii) is a local base of a unique locally convex topology for $E$.
