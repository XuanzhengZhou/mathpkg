A Hamel base is an algebraic basis for a linear space: every nonzero vector can be expressed uniquely as a finite linear combination of basis elements with nonzero coefficients. Unlike orthonormal bases in Hilbert spaces, Hamel bases use only finite linear combinations. The maximal principle guarantees that every linear space possesses a Hamel base.
