Let $E$ be a real or complex linear space, $F$ a linear subspace, $A$ a convex circled set which is radial at $0$, and let $f$ be a linear functional on $F$ such that $|f(x)| \leq 1$ for $x$ in $A \cap F$. Then there is a linear functional $\overline{f}$ on $E$, an extension of $f$, such that $|\overline{f}(x)| \leq 1$ on $A$.
