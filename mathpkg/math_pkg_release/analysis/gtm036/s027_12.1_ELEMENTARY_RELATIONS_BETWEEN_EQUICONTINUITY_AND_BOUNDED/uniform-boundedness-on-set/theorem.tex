The family $F$ of linear functions, each from a linear topological space $E$ to a linear topological space $H$, is said to be \emph{uniformly bounded} on a set $A \subset E$ if and only if the set $F[A] = \{f(x) : f \in F, x \in A\}$ is a bounded subset of $H$.

Equivalently, $F$ is uniformly bounded on $A$ if and only if $F$ is bounded relative to the topology of uniform convergence on $A$.

If $F$ is uniformly bounded on $A$, then $F$ is pointwise bounded on $A$. The converse is false in general.
