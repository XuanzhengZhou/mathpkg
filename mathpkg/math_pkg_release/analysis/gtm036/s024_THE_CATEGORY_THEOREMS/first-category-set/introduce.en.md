A subset $A$ of a topological space $X$ is said to be **of the first category** in $X$ (also called *meager* or *exhaustible*) if it can be expressed as a countable union of nowhere dense subsets of $X$. The notion captures the idea of a "small" or "thin" set from the topological perspective. A set is defined to be of the first category (absolutely) if it is of the first category in itself. The complement of a first category notion is the second category: a set that cannot be written as such a countable union.
