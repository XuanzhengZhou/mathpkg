In each locally convex linear topological space $E$ there is a local base $\mathcal{U}$ such that: (i) the sets in $\mathcal{U}$ are convex circled sets each of which is radial at $0$; (ii) the intersection of two sets in $\mathcal{U}$ contains a set in $\mathcal{U}$; (iii) for each $U$ in $\mathcal{U}$ and each scalar $a 
eq 0$, $aU \in \mathcal{U}$. Conversely, if $E$ is a linear space, then any non-void family $\mathcal{U}$ of subsets which satisfies (i), (ii), and (iii) is a local base of a unique locally convex topology for $E$.
