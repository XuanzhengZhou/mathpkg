A linear topological space is a linear space $E$ with a topology such that addition and scalar multiplication are each continuous simultaneously in both variables; more precisely, such that each of the following maps is continuous:
\begin{enumerate}
\item[(a)] the map of the product, $E \times E$, with the product topology, into $E$, which is given by $(x, y) \mapsto x + y$ for $x$ and $y$ in $E$;
\item[(b)] the map of the product, $K \times E$, of the scalar field and $E$, with the product topology, into $E$, which is given by $(a, x) \mapsto a x$ for $a$ in $K$ and $x$ in $E$.
\end{enumerate}
The topology of a linear topological space is called a vector topology. Thus a topology for a linear space is a vector topology if and only if addition and scalar multiplication are continuous, and a linear topological space is a linear space with a vector topology.
