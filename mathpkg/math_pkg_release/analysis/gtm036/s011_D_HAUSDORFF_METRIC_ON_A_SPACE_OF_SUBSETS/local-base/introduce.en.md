In a linear topological space, the entire topology is determined by the neighborhoods of the origin. A local base (also called a system of nuclei) is a base for the neighborhood system at $0$. Since translation is a homeomorphism, the neighborhood system at any point $x$ is exactly the family of translates by $x$ of the neighborhoods of $0$. Consequently, a set is open if and only if each of its points can be translated into some member of the local base.
