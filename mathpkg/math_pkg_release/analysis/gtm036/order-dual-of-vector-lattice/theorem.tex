A linear functional $f$ on a vector lattice $E$ belongs to the order dual $E^{\star}$ if and only if $f$ is bounded on order bounded sets. Under the dual ordering $E^{\star}$ is a vector lattice, and, for an $f$ in $E^{\star}$, $f^{+} = f \lor 0$ is given by $f^{+}(x) = \sup \{f(u): 0 \leq u \leq x\}$ for each positive element $x$ in $E$.
