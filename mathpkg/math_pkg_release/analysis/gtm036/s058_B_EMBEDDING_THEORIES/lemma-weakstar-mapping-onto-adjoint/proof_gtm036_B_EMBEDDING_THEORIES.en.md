Choose a co-base $\{B_n: n = 0, 1, \cdots\}$ of the equicontinuous subsets of $E^*$ so that each $B_n$ is convex, circled, weak* compact and weak* metrizable, and so that $B_n \subset B_{n+1}$ for each $n$. There is a continuous function $f_n$ on the Cantor set in $[0,1]$ onto $B_n$ with the weak* topology; see, for example, Kelley [5] Ch. V, O. Define $f$ by $f(2n + s) = f_n(s)$ whenever $s$ belongs to the Cantor set, and by continuity and linearity elsewhere.
