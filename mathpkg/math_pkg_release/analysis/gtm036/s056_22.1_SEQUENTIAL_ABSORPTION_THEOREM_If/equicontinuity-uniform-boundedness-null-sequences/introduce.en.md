This corollary to the Sequential Absorption Theorem provides a useful sequential characterization of equicontinuity for families of linear operators on pseudo-metrizable spaces. Rather than checking the topological definition of equicontinuity directly, one only needs to verify uniform boundedness on null sequences. This simplifies equicontinuity proofs in practice, especially in functional analysis where sequential arguments are often more natural.
