This technical lemma is the critical step in identifying the strongest topology which agrees with the weak* topology on equicontinuous subsets of the dual space. It constructs a countable set $S$ (arranged as a null sequence) whose polar is contained in a given set $W$ satisfying a weak* openness condition. The proof uses an inductive construction and relies on a pre-lemma about finite subsets of neighborhoods.
