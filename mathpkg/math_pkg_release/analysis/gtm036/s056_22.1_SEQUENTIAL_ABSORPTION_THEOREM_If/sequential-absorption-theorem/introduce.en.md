In a pseudo-metrizable linear topological space, the purely sequential notion of absorbing all null sequences is equivalent to the topological notion of being a neighborhood of zero. This theorem bridges the gap between sequential convergence and the topological structure of the space, establishing that a subset which "traps" every sequence tending to zero must necessarily contain an open set around zero. The result is fundamental for connecting sequential arguments to the topology of pseudo-metrizable spaces.
