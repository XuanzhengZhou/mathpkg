Let $D$ be a domain in the complex plane and $x$ a mapping of $D$ into a sequentially complete locally convex Hausdorff space $E$. The function $x$ is analytic in $D$ if and only if, for each $z \in D$,
$$\lim_{\zeta \to 0} \frac{x(z + \zeta) - x(z)}{\zeta}$$
exists. The function $x$ is weakly analytic in $D$ if and only if it is analytic relative to the weak topology, or equivalently, if and only if for each $f \in E^*$, the scalar function $x_f$ defined by $x_f(z) = f(x(z))$ is analytic in $D$.
