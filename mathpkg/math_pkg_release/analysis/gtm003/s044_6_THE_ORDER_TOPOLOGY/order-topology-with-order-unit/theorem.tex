Let $L$ be an Archimedean ordered vector space over $\mathbb{R}$, possessing an order unit $e$. Then:
\begin{enumerate}
\item $(L, \mathfrak{T}_0)$ is an ordered topological vector space which is normable.
\item $\mathfrak{T}_0$ is the finest locally convex topology on $L$ for which the positive cone $C$ is normal.
\item The following assertions are equivalent:
  \begin{enumerate}
  \item[(a)] $(L, \mathfrak{T}_0)$ is complete.
  \item[(b)] Each positive sequence of type $l^1$ in $L$ is order summable.
  \end{enumerate}
\end{enumerate}
