Let $L$ be an ordered vector space over $\mathbb{R}$. The \textit{order topology} $\mathfrak{T}_0$ of $L$ is defined to be the finest locally convex topology on $L$ for which every order interval $[x, y] \subset L$ is bounded.

A subset $W \subset L$ is a $0$-neighborhood for $\mathfrak{T}_0$ if and only if $W$ is convex and absorbs every order interval $[x, y] \subset L$. In particular, $W$ is necessarily radial, since $\{x\} = [x, x]$ for each $x \in L$.
