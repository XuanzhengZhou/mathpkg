A cone in a topological vector space is a convex cone with vertex at the origin that contains the origin. This concept provides the foundation for studying order structures in topological vector spaces. Cones give rise to order relations: if $C$ is the positive cone of an ordering of $L$, then the set $(x + C) \cap (y - C)$ corresponds to the order interval $\{z : x \leq z \leq y\}$. The theory of convex cones is central to the duality theory developed in Chapter V.
