A $C$-saturated set is a subset of a vector space that equals its own $C$-saturated hull, where $C$ is a given cone. The $C$-saturated hull $[A]$ of a set $A$ is defined as $(A + C) \cap (A - C)$. This construction generalizes the notion of order intervals: when $C$ is the positive cone of an ordering, $[x, y] = (x + C) \cap (y - C)$ is precisely the order interval $\{z : x \leq z \leq y\}$. The $C$-saturated hull operator preserves convexity and circledness, and lifts to filters.
