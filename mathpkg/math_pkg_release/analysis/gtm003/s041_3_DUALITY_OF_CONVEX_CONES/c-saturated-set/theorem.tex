Let $L$ be a vector space and $C$ a cone in $L$. For any $x, y \in L$, define
$$[x, y] = (x + C) \cap (y - C).$$
For any subset $A \subset L$, define the $C$-saturated hull of $A$ as
$$[A] = (A + C) \cap (A - C) = \bigcup \{[x, y] : x \in A, y \in A\}.$$
A subset $B \subset L$ is called $C$-saturated if $B = [B]$.

For any $A \subset L$, $[A]$ is the intersection of all $C$-saturated subsets containing $A$. The operator $A \mapsto [A]$ is monotone: $A \subset B$ implies $[A] \subset [B]$. Moreover, $[A]$ is convex if $A$ is convex, and $[A]$ is circled (with respect to $\mathbb{R}$) if $A$ is circled. If $\mathfrak{F}$ is a filter (or filter base) in $L$, then $\{[F] : F \in \mathfrak{F}\}$ is a filter base in $L$, whose corresponding filter is denoted by $[\mathfrak{F}]$.
