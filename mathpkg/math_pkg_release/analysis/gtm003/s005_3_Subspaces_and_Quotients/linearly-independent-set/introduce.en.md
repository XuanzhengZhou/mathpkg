A linearly independent set in a vector space is a subset where no nontrivial finite linear combination of its elements yields the zero vector. Equivalently, each element of the set is not expressible as a finite linear combination of the others. By convention, the empty set is considered linearly independent. The notion of linear independence is fundamental to defining bases and dimension in vector spaces.
