A function $|\cdot|: K \to \mathbb{R}$ is called an \text{absolute value} (or \text{valuation}) if for all $\lambda, \mu \in K$:
\begin{enumerate}
\item[(1)] $|\lambda| = 0$ is equivalent with $\lambda = 0$.
\item[(2)] $|\lambda + \mu| \leq |\lambda| + |\mu|$.
\item[(3)] $|\lambda\mu| = |\lambda|\,|\mu|$.
\end{enumerate}
