Consider the canonical inclusions $E \subset E'' \subset E'^*$. An element $x'' \in E'^*$ belongs to the bidual $E''$ if and only if $x''$ is continuous on every equicontinuous subset of $E'$, where each equicontinuous subset carries the weak topology $\sigma(E', E)$ (which coincides on equicontinuous sets with the topology induced by $\beta(E', E)$).

Equivalently, $x'' \in E''$ if and only if the restriction of $x''$ to each equicontinuous subset $H \subset E'$ is $\sigma(E', E)|_H$-continuous.
