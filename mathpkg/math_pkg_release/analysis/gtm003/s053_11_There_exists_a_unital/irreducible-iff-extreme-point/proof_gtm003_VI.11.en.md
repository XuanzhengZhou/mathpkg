\textit{Proof sketch (Exercise).} One direction uses part (a): if $\phi$ is not extreme, write $\phi = \lambda \phi_p + (1-\lambda)\phi_q$ as a convex combination, and use the corresponding states to construct a nontrivial projection in the commutant of $\pi_\phi(A)$, showing the GNS representation is not irreducible. Conversely, if $(\pi_\phi, H_\phi)$ is not irreducible, the commutant contains a nontrivial projection, which can be used to decompose $\phi$ as a convex combination, contradicting extremality. See also Dixmier~\cite{dixmier1977} for a complete proof.
