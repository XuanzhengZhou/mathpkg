\textit{Proof sketch (Exercise).} The center of $W(G)$ consists of those operators commuting with all $u_g$ ($g \in G$). An element in the center corresponds to an $\ell^\infty$-function on the set of conjugacy classes. If every non-identity conjugacy class is infinite, the only such bounded functions that belong to $W(G)$ are constants, so the center is trivial. Conversely, if some non-identity element has a finite conjugacy class, one can construct a non-trivial central element. Since $W(G)$ is finite by part (a) and is a factor under the ICC condition, it must be of type $\mathrm{II}_1$.
