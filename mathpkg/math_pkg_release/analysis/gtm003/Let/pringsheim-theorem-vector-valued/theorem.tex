Let $E$ be an ordered, semi-complete l.c.s. over $C$ such that the positive cone $C$ is weakly normal. If $a_n \in C$ $(n = 0, 1, \ldots)$ and $\sum_{n=0}^{\infty} a_n z^n$ has radius of convergence 1, then the analytic function represented by the power series is singular at $z = 1$. If this singularity is a pole, it is of maximal order on $|z| = 1$.
