A topological vector lattice is a vector lattice and a Hausdorff t.v.s. over $R$ that is locally solid. A locally convex vector lattice (l.c.v.l.) is a topological vector lattice whose topology is locally convex. A Fr\'echet lattice is a l.c.v.l. which is an (F)-space. A Banach lattice is a complete normed lattice.
