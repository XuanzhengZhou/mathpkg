All locally convex topologies on $F$ compatible with a given duality $\langle F, G \rangle$ have the same bounded sets. A set bounded for the weak topology $\sigma(F, G)$ is automatically bounded for the Mackey topology $\tau(F, G)$ and hence for all intermediate consistent topologies.
