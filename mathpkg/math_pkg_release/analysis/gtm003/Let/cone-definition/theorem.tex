Let $L$ be a vector space (over $R$ or $C$). By a cone in $L$ we understand a convex cone $C$ of vertex 0 such that $0 \in C$. For any pair $(x, y) \in L \times L$, write $[x, y] = (x + C) \cap (y - C)$. For any subset $A \subset L$, define $[A] = (A + C) \cap (A - C) = \bigcup \{[x, y]: x \in A, y \in A\}$. A subset $B \subset L$ is $C$-saturated if $B = [B]$; $[A]$ is the $C$-saturated hull of $A$.
