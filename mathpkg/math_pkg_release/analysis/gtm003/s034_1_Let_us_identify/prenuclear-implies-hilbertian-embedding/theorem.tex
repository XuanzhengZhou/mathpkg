\text{Let } E \text{ be a l.c.s. on which every continuous semi-norm is prenuclear. There exists a neighborhood base } \mathfrak{U} \text{ of } 0 \text{ in } E \text{ such that for each } U \in \mathfrak{U}, \tilde{E}_U \text{ is norm isomorphic with a Hilbert subspace of } L^2(\mu), \text{ where } \mu \text{ is a positive Radon measure on a suitable closed equicontinuous subset of } E'_\sigma.
