\text{Let } L \text{ be a t.v.s. over } K.

\text{(i) For each } x_0 \in L \text{ and each } \lambda_0 \in K,\; \lambda_0 \neq 0, \text{ the mapping } x \mapsto \lambda_0 x + x_0 \text{ is a homeomorphism of } L \text{ onto itself.}

\text{(ii) For any subset } A \text{ of } L \text{ and any base } \mathfrak{U} \text{ of the neighborhood filter of } 0 \in L, \text{ the closure } \bar{A} \text{ is given by } \bar{A} = \bigcap \{A + U : U \in \mathfrak{U}\}.

\text{(iii) If } A \text{ is an open subset of } L, \text{ and } B \text{ is any subset of } L, \text{ then } A + B \text{ is open.}

\text{(iv) If } A, B \text{ are closed subsets of } L \text{ such that every filter on } A \text{ has an adherent point (in particular, such that } A \text{ is compact), then } A + B \text{ is closed.}

\text{(v) If } A \text{ is a circled subset of } L, \text{ then its closure } \bar{A} \text{ is also circled.}
