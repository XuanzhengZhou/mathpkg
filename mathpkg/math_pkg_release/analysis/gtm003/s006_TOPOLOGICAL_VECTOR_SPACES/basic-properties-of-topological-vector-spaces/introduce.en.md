Proposition 1.1 collects the most immediate consequences of the topological vector space axioms. It shows that translations and nonzero dilations are homeomorphisms, gives a neighborhood-base characterization of the closure of any subset, and establishes basic stability properties: the sum of an open set with any set is open, and the sum of a closed set with a compact (or more generally, a set on which every filter has an adherent point) closed set is closed. The final part states that the closure of a circled set is again circled. These facts are used constantly throughout the theory.
