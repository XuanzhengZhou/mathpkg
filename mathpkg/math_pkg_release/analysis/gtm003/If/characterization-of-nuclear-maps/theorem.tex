A linear map $u \in \mathcal{L}(E, F)$ between Banach spaces is nuclear if and only if it admits a representation

$$u(x) = \sum_{n=1}^{\infty} \lambda_n f_n(x) y_n$$

where $\sum_{n=1}^{\infty} |\lambda_n| < +\infty$, $\{f_n\}$ is an equicontinuous sequence in $E'$ (equivalently, bounded in norm), and $\{y_n\}$ is a sequence contained in a convex, circled, and bounded subset $B$ of $F$ for which $F_B$ is complete.
