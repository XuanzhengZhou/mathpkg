In ordered vector spaces with the Riesz decomposition property, the positive cone is normal for the order topology. This is proved by showing that if $0 \leq y \leq x$ and $x$ belongs to a $0$-neighborhood of the inductive limit, then $y$ belongs to the same neighborhood, using the decomposition property repeatedly.
