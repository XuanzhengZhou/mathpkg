For each convex, circled $0$-neighborhood $U$ in a nuclear space $E$, there exists a $0$-neighborhood $V \subset U$ such that $\tilde{E}_V$ is norm isomorphic to a subspace of $\ell^p$ for any fixed $p$ with $1 \leq p < \infty$. In particular, each $\tilde{E}_U$ is isomorphic to a subspace of a Hilbert space (taking $p = 2$).
