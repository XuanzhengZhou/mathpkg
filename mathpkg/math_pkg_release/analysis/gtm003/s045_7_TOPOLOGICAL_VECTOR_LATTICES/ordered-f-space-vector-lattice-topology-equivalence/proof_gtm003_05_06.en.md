If $\mathfrak{T} = \mathfrak{T}_0$, then $C$ is normal by (6.4). Conversely, if $C$ is normal, then $E' = C' - C'$ by (3.3), and $C' - C' = E^+$ by (5.5) (for $C$ is closed in $(E, \mathfrak{T})$ and $E = C - C$); since $E$ is a Mackey space by (IV, 3.4), the assertion follows from (6.4) (c).
