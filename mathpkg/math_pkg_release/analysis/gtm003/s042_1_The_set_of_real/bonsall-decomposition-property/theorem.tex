A normed space $L$ (over $\mathbb{R}$) with a cone $C$ is said to have the \textbf{decomposition property} (after Bonsall) if each $z$ with $\|z\| \leq 1$ can be approximated with given accuracy by differences $x - y$, where $x \in C$, $y \in C$ and $\|x\| \leq k$, $\|y\| \leq k$ for a fixed constant $k > 0$.
