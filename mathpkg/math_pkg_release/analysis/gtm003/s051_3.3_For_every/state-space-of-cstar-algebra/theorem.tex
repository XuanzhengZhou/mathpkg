The set $S(A)$ of all states of a $C^*$-algebra $A$ is called the \textit{state space} of $A$. If $A$ is unital with unit $e$, then $S(A) = \{\phi \in A'_+ : \phi(e) = 1\}$ is the intersection of the $\sigma(A', A)$-closed hyperplane $\{\phi \in A' : \phi(e) = 1\}$ with the cone $A'_+$; hence $S(A)$ is $\sigma(A', A)$-compact.
