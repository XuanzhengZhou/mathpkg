In any $C^*$-algebra $A$, the intersection of the positive cone $A_+$ with the open unit ball $U_0$ is directed upward with respect to the canonical order $\leq$. Moreover, this directed set serves as an approximate unit for the algebra, providing a fundamental structural link between the order and the approximate identities in $C^*$-algebras. This result is essential for many arguments that require approximating arbitrary elements by positive elements of norm less than one.
