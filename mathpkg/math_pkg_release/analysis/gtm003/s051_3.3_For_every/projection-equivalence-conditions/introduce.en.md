In a $C^*$-algebra, the order relation $q \leq p$ between projections is equivalent to the algebraic condition $pq = qp = q$, which in turn is equivalent to $p - q$ being a projection. This result provides both an order-theoretic and an algebraic characterization of the partial order on projections, which is fundamental for the study of spectral families and resolutions of the identity.
