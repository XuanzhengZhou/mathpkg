A metric space $(X, d)$ is a set $X$ together with a function $d: X \times X \to \mathbb{R}_{\geq 0}$ satisfying: (i) $d(x, y) = 0 \iff x = y$; (ii) $d(x, y) = d(y, x)$; (iii) $d(x, z) \leq d(x, y) + d(y, z)$. The topology is generated by open balls $B_\varepsilon(x) = \{y \in X : d(x, y) < \varepsilon\}$.
