Let $X$ be a Hausdorff topological space. For $X$ to be compact, each of the following conditions is necessary and sufficient:

\begin{itemize}
\item[(a)] A family of closed subsets of $X$ has non-empty intersection whenever each finite subfamily has non-empty intersection.
\item[(b)] Each filter on $X$ has a cluster point.
\item[(c)] Each ultrafilter on $X$ converges.
\end{itemize}
