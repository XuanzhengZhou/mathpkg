The text mentions Tychonov's theorem without proof. The standard proof using ultrafilters: Let $X = \prod_\alpha X_\alpha$ where each $X_\alpha$ is compact. Let $\mathfrak{U}$ be an ultrafilter on $X$. For each $\alpha$, the image filter $\pi_\alpha(\mathfrak{U})$ (generated by $\{\pi_\alpha(U) : U \in \mathfrak{U}\}$) is an ultrafilter base on $X_\alpha$. Since $X_\alpha$ is compact, this ultrafilter converges to some point $x_\alpha \in X_\alpha$. Then $\mathfrak{U}$ converges to the point $x = (x_\alpha)$ in the product space: any basic product neighborhood of $x$ is $\prod_\alpha V_\alpha$ where $V_\alpha = X_\alpha$ for all but finitely many $\alpha$, and each $V_\alpha$ is a neighborhood of $x_\alpha$; since $\pi_\alpha(\mathfrak{U})$ converges to $x_\alpha$, each $V_\alpha$ contains some $\pi_\alpha(U_\alpha)$ with $U_\alpha \in \mathfrak{U}$, and the finite intersection (over the relevant $\alpha$) of these $U_\alpha$ belongs to $\mathfrak{U}$ and maps into the basic neighborhood. By the ultrafilter characterization of compactness, $X$ is compact.
