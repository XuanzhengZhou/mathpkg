Let $H$ be a Hilbert space, not reduced to $\{0\}$. An operator $u \in \mathcal{L}(H)$ is called a \text{partial isometry} if $u$ restricted to $(\ker u)^\perp$ is an isometry. Thus if $F_1$ and $F_2$ are closed subspaces of $H$ with the same Hilbert dimension $d$, there exists a partial isometry $u$ satisfying $u(F_1) = F_2$, $u(F_1^\perp) = \{0\}$. It follows that $u^*u$ is a projection (the orthogonal projection $H \to (\ker u)^\perp$), and similarly $uu^*$ is the orthogonal projection $H \to (\ker u^*)^\perp$; in particular, $u^*$ is also a partial isometry.
