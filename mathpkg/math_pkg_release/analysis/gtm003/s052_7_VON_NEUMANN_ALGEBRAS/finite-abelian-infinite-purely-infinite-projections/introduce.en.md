The type classification of von Neumann algebras is built on a hierarchy of projection properties. A projection is finite if it is not equivalent to a proper sub-projection of itself, abelian if the corner algebra $pWp$ is commutative, infinite if it is not finite, and purely infinite if every non-zero sub-projection is infinite. Type I algebras are those whose identity decomposes into abelian projections; type III algebras are those where the identity is purely infinite. Types II$_1$ and II$_\infty$ occupy the intermediate regime.
