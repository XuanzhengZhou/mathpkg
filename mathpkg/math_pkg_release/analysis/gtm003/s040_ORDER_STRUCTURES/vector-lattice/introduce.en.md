A vector lattice (also called a Riesz space) is an ordered vector space in which every pair of elements has both a supremum and an infimum. This rich algebraic-order structure allows the definition of absolute values, disjointness, and orthogonal complements. Vector lattices form the central object of study in the theory of ordered vector spaces, generalizing function spaces with pointwise order.
