A directed set is an ordered set in which every pair (equivalently, every finite subset) has an upper bound. Directed sets generalize sequences by providing index sets for nets (also called Moore-Smith sequences), which are used to study convergence in general topological spaces where sequences are insufficient. Important examples include the natural numbers $\mathbb{N}$, the set of neighborhoods of a point ordered by reverse inclusion, and the finite subsets of a set ordered by inclusion.
