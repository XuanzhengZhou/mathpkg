A totally ordered set (also called linearly ordered or a chain) is an ordered set in which any two elements are comparable: for every $x, y$, either $x \leq y$ or $y \leq x$. This contrasts with partially ordered sets where some elements may be incomparable. The real numbers $\mathbb{R}$ under the usual order form a totally ordered set, as do the integers $\mathbb{Z}$ and the ordinals.
