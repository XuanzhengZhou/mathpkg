A maximal element $x$ of an ordered set $(X, \leq)$ is one that is not strictly less than any other element: if $x \leq y$, then necessarily $x = y$. Maximal elements need not be greatest elements (there may be other elements incomparable to $x$). Zorn's Lemma states that in an ordered set where every chain has an upper bound, there exists at least one maximal element, making this concept central to existence proofs across mathematics.
