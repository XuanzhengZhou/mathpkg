For any pair $(x, y)$ in an ordered set $(X, \leq)$, the notations $\sup(x, y)$ and $\inf(x, y)$ denote the supremum and infimum of the two-element set $\{x, y\}$, provided they exist. These binary operations are the building blocks of lattice theory: an ordered set is a lattice precisely when $\sup(x, y)$ and $\inf(x, y)$ exist for every pair.
