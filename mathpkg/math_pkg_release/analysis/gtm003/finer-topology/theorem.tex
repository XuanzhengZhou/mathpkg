If $X$ is a set and $\mathfrak{T}_1, \mathfrak{T}_2$ are topologies on $X$, $\mathfrak{T}_2$ is finer than $\mathfrak{T}_1$ (or $\mathfrak{T}_1$ is coarser than $\mathfrak{T}_2$) if every $\mathfrak{T}_1$-open set is $\mathfrak{T}_2$-open. This amounts to the inclusion $\mathfrak{G}_1 \subset \mathfrak{G}_2$ in $\mathfrak{P}(\mathfrak{P}(X))$ for the respective families of open sets $\mathfrak{G}_1, \mathfrak{G}_2$.
