If $L$ is a vector space, a linear manifold (or affine subspace) in $L$ is a subset which is a translate of a subspace $M \subset L$, that is, a set $F$ of the form $x_0 + M$ for some $x_0 \in L$. $F$ determines $M$ uniquely while it determines $x_0$ only modulo $M$. A hyperplane in $L$ is a maximal proper affine subspace of $L$; the corresponding subspace has codimension 1.
