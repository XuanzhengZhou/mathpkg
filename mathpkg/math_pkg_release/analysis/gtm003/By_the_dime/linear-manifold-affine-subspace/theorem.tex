If $L$ is a vector space, a linear manifold (or affine subspace) in $L$ is a subset which is a translate of a subspace $M \subset L$, that is, a set $F$ of the form $x_0 + M$ for some $x_0 \in L$. $F$ determines $M$ uniquely while it determines $x_0$ only mod $M$: $x_0 + M = x_1 + N$ if and only if $M = N$ and $x_1 - x_0 \in M$. Two linear manifolds $x_0 + M$ and $x_1 + N$ are said to be parallel if either $M \subset N$ or $N \subset M$. The dimension of a linear manifold is the dimension of the subspace of which it is a translate.
