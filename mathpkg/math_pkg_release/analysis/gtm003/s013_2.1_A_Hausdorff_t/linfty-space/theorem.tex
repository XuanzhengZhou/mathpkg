Let $(X, \Sigma, \mu)$ be a measure space. $\mathscr{L}^\infty$ denotes the vector space of $\mu$-essentially bounded $\Sigma$-measurable functions on $X$; a $\Sigma$-measurable function is $\mu$-essentially bounded if its equivalence class modulo $\mathcal{N}_\mu$ contains a bounded function. Then $\mathscr{L}^\infty / \mathcal{N}_\mu$ is algebraically isomorphic with $M(X, \Sigma) / (\mathcal{N}_\mu \cap M(X, \Sigma))$, where $M(X, \Sigma)$ is the space of bounded $\Sigma$-measurable functions. The latter quotient space is a Banach space, usually denoted by $L^\infty(\mu)$. Moreover, $L^\infty(\mu)$ is norm-isomorphic to $\mathcal{C}(X)$ for a suitable compact space $X$.
