A uniform space is a set $X$ equipped with a uniformity $\mathfrak{U}$, a filter on $X \times X$ satisfying: (i) each $U \in \mathfrak{U}$ contains the diagonal $\Delta$; (ii) if $U \in \mathfrak{U}$ then $U^{-1} \in \mathfrak{U}$; (iii) for each $U \in \mathfrak{U}$ there exists $V \in \mathfrak{U}$ with $V \circ V \subset U$. Every uniform space is completely regular.
