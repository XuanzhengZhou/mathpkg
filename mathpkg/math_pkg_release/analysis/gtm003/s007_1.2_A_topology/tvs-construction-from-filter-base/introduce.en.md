This corollary to Theorem 1.2 provides a practical method for constructing topological vector spaces: any filter base satisfying the three algebraic-topological conditions (subadditive, radial-and-circled, scaling) uniquely determines a t.v.s. topology. This is the key constructive principle used throughout the book to build t.v.s. topologies from specified $0$-neighborhood bases.
