This proposition provides the fundamental algebraic description of hyperplanes: a subset $H$ is a hyperplane precisely when it is the level set of a non-zero linear form. In other words, every hyperplane is of the form $f^{-1}(\alpha)$ for some non-zero $f \in L^*$ and scalar $\alpha$. The linear form $f$ is determined up to a non-zero scalar multiple, reflecting the fact that different equations can define the same hyperplane. In the context of topological vector spaces, the continuity of $f$ determines whether the hyperplane is closed.
