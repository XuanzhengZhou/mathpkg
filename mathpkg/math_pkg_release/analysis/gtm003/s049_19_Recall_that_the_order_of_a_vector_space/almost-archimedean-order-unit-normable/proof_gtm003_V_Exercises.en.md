See the hint in the text: the order unit $e$ defines the Minkowski functional $p_e(x) = \inf\{\lambda > 0 : -\lambda e \leq x \leq \lambda e\}$. By the almost Archimedean property, $p_e$ is a norm. Equip $L$ with this norm; then $\mathfrak{T}_O$ coincides with the norm topology. The dual equalities follow from the order unit property and the fact that $\mathfrak{T}_O$ is the finest locally convex topology for which the order intervals are bounded (V, Section 2).
