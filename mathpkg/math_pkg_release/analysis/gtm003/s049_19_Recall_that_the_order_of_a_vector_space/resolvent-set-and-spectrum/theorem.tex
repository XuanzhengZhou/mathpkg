For $x \in A$, where $A$ is a unital Banach algebra, the \emph{resolvent set} $\rho(x)$ is the set of all $\lambda \in \mathbf{C}$ for which $\lambda e - x$ is invertible. The \emph{spectrum} $\sigma(x) := \mathbf{C} \setminus \rho(x)$ is compact and non-void. The \emph{resolvent} is $R(\lambda, x) := (\lambda e - x)^{-1}$ for $\lambda \in \rho(x)$.
