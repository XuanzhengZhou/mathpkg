If $(X, \Sigma, \mu)$ is a measure space (Chapter II, Section 2, Example 2), the (complex) Banach space $L^\infty(\mu)$ becomes a $C^*$-algebra if multiplication is defined (on equivalence classes) by pointwise multiplication of representatives, the involution again being given by functional conjugation. This is an example of a (commutative) $W^*$-algebra, i.e., a $C^*$-algebra whose underlying normed space is a dual Banach space.
