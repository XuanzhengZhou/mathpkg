A positive element in a C*-algebra $A$ is a self-adjoint element whose spectrum is contained in the non-negative real numbers. The set $A_+$ of all positive elements forms a proper, closed, convex, and normal cone, endowing $A$ with a natural order structure. Generally, $A_+$ is not invariant under multiplication unless the elements commute.
