**(i)** By Theorem 3.1, there exists $d \in A$ such that $0 \leq y - x = d^* d$. Thus
$$c^*(y - x)c = c^* d^* d c = (dc)^* dc \geq 0,$$
whence the first assertion follows. For the second, observe that $\sigma(\|y^*y\| e - y^*y) \subset \mathbf{R}_+$ implies $0 \leq y^*y \leq \|y\|^2 e$, so by the first assertion,
$$x^* y^* y x \leq \|y\|^2 x^* x.$$

**(ii)** Follows from the spectral radius formula and the fact that for any $a \in A_+$, $\|a\| = \sup \sigma(a)$. If $|x| \leq |y|$, then $\sigma(|x|) \subset [0, \| |y| \|]$ by the order relation on spectra, hence $\|x\| = \||x|\| \leq \||y|\| = \|y\|$.

**(iii)** Using the Gelfand representation (or the continuous functional calculus), in the commutative $C^*$-algebra generated by $x$, $y$, and $e$, the square root function $t \mapsto \sqrt{t}$ is continuous and operator monotone on $\mathbf{R}_+$, so $0 \leq x \leq y$ implies $0 \leq \sqrt{x} \leq \sqrt{y}$.

**(iv)** If $0 < x \leq y$, then $y$ is bounded below by the invertible positive element $x$, hence $y$ is also invertible. From $x \leq y$, multiplying on both sides by $y^{-1/2}$ (using part (i)) yields $y^{-1/2} x y^{-1/2} \leq e$. The spectrum of $y^{-1/2} x y^{-1/2}$ is contained in $(0, 1]$, and since inversion is order-reversing on $(0, \infty)$, taking inverses gives $y^{1/2} x^{-1} y^{1/2} \geq e$. Multiplying by $y^{-1/2}$ from both sides yields $y^{-1} \leq x^{-1}$.
