This lemma establishes that every positive element in a C*-algebra admits a unique positive square root. Using the continuous functional calculus via the Gelfand representation (Theorem 2.2), the square root function $t \mapsto \sqrt{t}$ on $\sigma(x) \subset \mathbb{R}_+$ is applied to obtain $y = \sqrt{x}$.
