Let $\{L_\alpha : \alpha \in \mathbf{A}\}$ be a family of vector spaces over the same scalar field $K$. The Cartesian product $L = \prod_{\alpha \in \mathbf{A}} L_\alpha$ is a vector space over $K$ with addition and scalar multiplication defined componentwise:
$$
x + y = (x_\alpha + y_\alpha), \qquad \lambda x = (\lambda x_\alpha)
$$
for $x = (x_\alpha)$, $y = (y_\alpha) \in L$ and $\lambda \in K$.

If each $(L_\alpha, \mathfrak{T}_\alpha)$ is a topological vector space over $K$, then $L$ equipped with the product topology $\mathfrak{T} = \prod_{\alpha \in \mathbf{A}} \mathfrak{T}_\alpha$ is again a topological vector space. The space $(L, \mathfrak{T})$ is called the \textbf{product} of the family $\{L_\alpha(\mathfrak{T}_\alpha) : \alpha \in \mathbf{A}\}$.

Moreover, $(L, \mathfrak{T})$ is a Hausdorff space if and only if each factor $(L_\alpha, \mathfrak{T}_\alpha)$ is Hausdorff, and $(L, \mathfrak{T})$ is a complete uniform space if and only if each factor is complete.
