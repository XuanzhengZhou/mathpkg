An equicontinuous set of linear mappings is a family for which the continuity estimates are uniform: given any neighborhood of zero in the target, a single neighborhood of zero in the domain works simultaneously for all maps in the family. This notion generalises the classical concept of equicontinuity for families of functions to the setting of topological vector spaces and is fundamental to the Banach-Steinhaus theorem.
