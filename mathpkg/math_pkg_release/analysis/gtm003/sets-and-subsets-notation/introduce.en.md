The definition of sets and subsets establishes the fundamental notation used throughout all of mathematics. It defines set membership ($x \in X$), subset relations ($X \subset Y$), and set equality via mutual inclusion. The set-builder notation $\{x \in X : (p)x\}$ allows constructing subsets by specifying a property $(p)$ that elements must satisfy. Additional notations include the complement $Y \sim X$, the empty set $\varnothing$, singletons $\{x\}$, and the power set $\mathfrak{P}(X)$ of all subsets of $X$.
