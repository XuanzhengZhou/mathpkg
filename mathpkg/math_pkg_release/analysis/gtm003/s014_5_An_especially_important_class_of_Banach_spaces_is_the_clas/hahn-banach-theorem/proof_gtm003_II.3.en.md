The case $f = 0$ is trivial, so we assume that $f(x) \neq 0$ for some $x \in M$.

By Chapter I, Proposition 1.2, the convex, circled sets
$$V_n = \{x \in L : p(x) < n^{-1}\}, \quad n \in \mathbb{N},$$
form a $0$-neighborhood base for a topology $\mathfrak{T}$ under which $L$ is a topological vector space. This is the locally convex topology generated by the single seminorm $p$.

Define $H = \{x \in M : f(x) = 1\}$. Then $H$ is a hyperplane in $M$ and a linear manifold in $L$. Let $A = V_1 = \{x \in L : p(x) < 1\}$. Since $|f(x)| \leq p(x)$ for all $x \in M$, we have $H \cap A = \varnothing$: for if $x \in H$, then $1 = f(x) = |f(x)| \leq p(x)$, so $p(x) \geq 1$, hence $x \notin A$.

The set $A$ is convex, circled, and open in the topology $\mathfrak{T}$. By Mazur's theorem (Theorem 3.1), there exists a closed hyperplane $H_1$ in $L$ (with respect to $\mathfrak{T}$) containing $H$ and not intersecting $A$. Since $H_1$ is a hyperplane and $H \subset H_1 \cap M$, the subspace $H_1 \cap M$ has codimension at most $1$ in $M$. But $H \subset H_1 \cap M$ and $H$ has codimension $1$ in $M$, so either $H_1 \cap M = M$ or $H_1 \cap M = H$.

If $H_1 \cap M = M$, then $M \subset H_1$. But then $0 \in M \subset H_1$, which would mean $H_1$ is a subspace hyperplane. Since $H \subset H_1$, we would have $A \cap H_1 = \varnothing$ and $0 \in H_1$, which gives a contradiction because $A$ is a neighborhood of $0$ (so every subspace containing $0$ must intersect $A$ non-trivially -- but $A$ is supposed to not intersect $H_1$).

Therefore $H_1 \cap M = H$. Then $H_1$ defines a linear form $f_1$ on $L$ (unique up to a scalar multiple) whose kernel is the subspace parallel to $H_1$. Choosing the normalization so that $f_1 = f$ on $M$, we obtain a linear form $f_1$ extending $f$ to $L$.

Since $H_1 \cap A = \varnothing$ and $A = \{x : p(x) < 1\}$, for any $x \in L$ with $p(x) < 1$ we have $f_1(x) \neq 1$. More precisely, the condition $H_1 \cap A = \varnothing$ means that $f_1(x) = 1$ implies $p(x) \geq 1$. Equivalently, $p(x) < 1$ implies $f_1(x) \neq 1$. Using the linearity and the fact that $A$ is circled, one shows that $|f_1(x)| \leq p(x)$ for all $x \in L$.
