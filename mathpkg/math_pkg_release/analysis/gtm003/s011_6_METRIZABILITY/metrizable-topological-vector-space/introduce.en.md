A topological vector space is metrizable when its topology can be generated by a metric. This is a natural compatibility condition between the algebraic and topological structures, generalizing the geometry of normed spaces. The key subtlety is that the metric uniformity need not be translation-invariant, so one must distinguish between the metric uniformity and the canonical t.v.s. uniformity. Determining metrizability of a given t.v.s. is a central problem in the theory, with the main criterion given by Theorem 6.1.
