A mapping $f: X \to Y$ assigns to each $x \in X$ a unique element $f(x) \in Y$. The set $f(A) = \{f(x) : x \in A\} \subset Y$ for $A \subset X$. The inverse image of $B \subset Y$ is $f^{-1}(B) = \{x \in X : f(x) \in B\}$. The composition of $f: X \to Y$ and $g: Y \to Z$ is $g \circ f: x \mapsto g(f(x))$. A map $f$ is injective if $f(x_1) = f(x_2) \implies x_1 = x_2$; surjective if $f(X) = Y$; bijective if both.
