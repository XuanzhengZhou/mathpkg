A vector space $L$ over a (valuated) field $K$ is an abelian group $(L, +)$ together with a scalar multiplication $K \times L \to L$, $(\lambda, x) \mapsto \lambda x$, satisfying: $\lambda(x + y) = \lambda x + \lambda y$, $(\lambda + \mu)x = \lambda x + \mu x$, $\lambda(\mu x) = (\lambda \mu)x$, $1 \cdot x = x$.
