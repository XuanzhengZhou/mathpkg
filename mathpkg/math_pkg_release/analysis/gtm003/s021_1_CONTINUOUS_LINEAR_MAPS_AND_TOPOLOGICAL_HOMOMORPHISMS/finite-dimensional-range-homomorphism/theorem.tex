Let $L$ be a topological vector space over $K$ and let $u: L \to M$ be a continuous linear map into a topological vector space $M$ such that $u(L)$ is a separated (Hausdorff) topological vector space of finite dimension. Then $u$ is a topological homomorphism.

Equivalently, for such a map $u$, the following are equivalent:
\begin{enumerate}
\item[(a)] $u$ is a topological homomorphism.
\item[(b)] $u^{-1}(0)$ is closed in $L$.
\item[(c)] $u$ is open onto $u(L)$.
\end{enumerate}
