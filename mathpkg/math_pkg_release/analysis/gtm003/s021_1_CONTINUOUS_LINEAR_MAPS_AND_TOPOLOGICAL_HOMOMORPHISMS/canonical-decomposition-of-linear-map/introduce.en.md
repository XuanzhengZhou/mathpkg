Every linear map between topological vector spaces admits a canonical factorization through its null space and its image. The map factors as a quotient map onto the quotient by the null space, followed by a bijective algebraic homomorphism, followed by the inclusion of the image. This decomposition is fundamental for analyzing topological properties: the original map is a topological homomorphism if and only if the middle bijective map is an isomorphism of topological vector spaces.
