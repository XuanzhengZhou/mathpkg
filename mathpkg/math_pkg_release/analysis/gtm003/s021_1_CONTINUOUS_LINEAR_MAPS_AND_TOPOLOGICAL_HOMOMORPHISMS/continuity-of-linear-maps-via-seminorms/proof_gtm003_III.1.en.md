If the condition holds, then for any $\varepsilon > 0$, the set $U = \{x: c p_i(x) \leq \varepsilon, \, i = 1, \ldots, n\}$ is a $0$-neighborhood in $L$, and $u(U) \subset \{y: q(y) \leq \varepsilon\}$. Hence $u$ is continuous at $0$, therefore continuous.

Conversely, if $u$ is continuous, then for any $q \in \mathcal{Q}$, the set $\{y \in M: q(y) \leq 1\}$ is a $0$-neighborhood in $M$. By continuity at $0$, there exists a $0$-neighborhood $U$ in $L$ such that $u(U) \subset \{y: q(y) \leq 1\}$. But every $0$-neighborhood in the topology generated by $\mathcal{P}$ contains a set of the form $\{x: c p_i(x) \leq 1, \, i = 1, \ldots, n\}$ for some $c > 0$ and $p_1, \ldots, p_n \in \mathcal{P}$. Then for all $x$ with $c p_i(x) \leq 1$, we have $q(u(x)) \leq 1$. By homogeneity, for any $x \in L$, let $t = c \sup_i p_i(x)$; if $t > 0$, then $c p_i(x/t) \leq 1$, so $q(u(x/t)) \leq 1$, i.e., $q(u(x)) \leq t = c \sup_i p_i(x)$. If $t = 0$, then $c p_i(n x) \leq 1$ for all $n$, hence $q(u(n x)) \leq 1$ for all $n$, implying $q(u(x)) = 0$.
