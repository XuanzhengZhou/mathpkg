The proof begins by invoking Lemma 9.10. Since the closed set is nondiscrete, it contains a countable discrete subset $D$ with a limit point in the set. By Lemma 9.10, $D$ contains a copy of $N$ whose Stone-Čech remainder lies in $\beta X - vX$. The closure of this copy of $N$ in $\beta X$ is homeomorphic to $\beta N$, and since the original set is closed, it contains this closure, hence a copy of $\beta N$. The cardinality statement follows from $|\beta N| = 2^c$.
