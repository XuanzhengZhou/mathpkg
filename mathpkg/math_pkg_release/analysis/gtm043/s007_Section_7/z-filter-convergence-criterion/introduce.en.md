This proposition provides a useful criterion for convergence of $z$-filters in completely regular spaces. A $z$-filter $\mathcal{F}$ converges to $p$ precisely when it contains all zero-set-neighborhoods of $p$. This reformulation is possible because in a completely regular space, every neighborhood of a point contains a zero-set-neighborhood, making the zero-set-neighborhood filter a convenient tool for studying convergence.
