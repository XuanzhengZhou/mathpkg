A space $X$ is pseudocompact when the rings $C(X)$ and $C^*(X)$ coincide -- that is, every continuous real-valued function on $X$ is automatically bounded. Every compact space is pseudocompact, but the converse is false (a counterexample is given later in the book). Pseudocompactness is a central concept because it provides an algebraic characterization of a topological property via the function ring.
