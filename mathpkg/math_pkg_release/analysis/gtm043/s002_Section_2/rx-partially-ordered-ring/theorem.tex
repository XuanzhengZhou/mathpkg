The ring $\mathbf{R}^X$ with the pointwise partial ordering is a partially ordered ring. That is:
\begin{itemize}
\item The ordering is invariant under translation: $f + h \geq g + h$ if and only if $f \geq g$.
\item The product of nonnegative functions is nonnegative: if $f \geq 0$ and $g \geq 0$, then $fg \geq 0$.
\end{itemize}
