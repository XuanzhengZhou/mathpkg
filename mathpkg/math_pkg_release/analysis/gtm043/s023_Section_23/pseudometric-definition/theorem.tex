A pseudometric on a set $X$ is a function $d$ on $X \times X$ into $\mathbf{R}$, satisfying, for all $x, y, z \in X$:

\begin{itemize}
\item[(i)] $d(x, y) \geq 0$;
\item[(ii)] $d(x, x) = 0$;
\item[(iii)] $d(x, y) = d(y, x)$; and
\item[(iv)] $d(x, z) \leq d(x, y) + d(y, z)$ (triangle inequality).
\end{itemize}

Thus, a pseudometric differs from a metric only in that $d(x, y) = 0$ need not imply $x = y$.
