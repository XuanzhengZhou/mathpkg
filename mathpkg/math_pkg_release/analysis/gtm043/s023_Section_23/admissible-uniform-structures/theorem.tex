A uniform structure $\mathcal{D}$ on a topological space $X$ is called \textit{admissible} if the uniform topology coincides with the given topology.

For each $f \in C(X)$, let $\psi_f$ be defined by
$$\psi_f(x, y) = |f(x) - f(y)|.$$

The structure generated by all $\psi_f$, for $f \in C(X)$, is denoted by $\mathcal{C} = \mathcal{C}(X)$; the structure generated by all $\psi_f$, for $f \in C^*(X)$, is denoted by $\mathcal{C}^* = \mathcal{C}^*(X)$. In each case, the pseudometrics $\psi_f$ form only a subbase. A convenient base consists of all pseudometrics of the form
$$\psi_{f_1} \vee \cdots \vee \psi_{f_n}.$$
