Let $[X; \mathcal{D}]$ be a uniform space. A $z$-filter $\mathcal{F}$ is said to be a \textit{Cauchy $z$-filter} if for every $d \in \mathcal{D}$ and $\epsilon > 0$, $\mathcal{F}$ contains a set of $d$-diameter $< \epsilon$ — briefly, $\mathcal{F}$ contains arbitrarily small sets. It suffices to let $d$ range over a base for $\mathcal{D}$.
