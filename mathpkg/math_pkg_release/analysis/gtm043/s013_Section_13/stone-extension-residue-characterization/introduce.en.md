Theorem 7.6 characterizes the value of the Stone extension $f^*(p)$ at a point $p \in \beta X$ in terms of the residue class $M^p(f)$ in the residue class field $C(X)/M^p$. The value is $\infty$ precisely when the residue class is infinitely large (unbounded in the canonical copy of $\mathbf{R}$), and equals a real number $r$ precisely when the residue class is infinitely close to $r$. This theorem bridges the topological behavior of the Stone extension with the algebraic structure of the residue class fields of $C(X)$.
