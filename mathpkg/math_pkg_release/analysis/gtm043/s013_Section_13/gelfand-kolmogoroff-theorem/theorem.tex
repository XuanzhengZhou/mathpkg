\text{For the maximal ideals in } C(X), \text{ we have}
$$
M^p = \{ f \in C(X) : p \in \operatorname{cl}_{\beta X} Z_X(f) \} \quad (p \in \beta X).
$$
\text{The correspondence } p \mapsto M^p \text{ is one-one from } \beta X \text{ onto the set of all maximal ideals in } C(X).\\
M^p \text{ is free or fixed according as } p \in \beta X - X \text{ or } p \in X; \text{ and, in the latter case, } M^p = M_p.
