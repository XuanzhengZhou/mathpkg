A space is zero-dimensional if every basic cover has a refinement by a partition of the space (equivalently, $\dim X = 0$). If a cover $\{U_k\}_{k \leq s}$ has a refinement by some partition $\mathcal{W}$, then it has a refinement by a partition $\{V_k\}_{k \leq s}$ such that $V_k \subset U_k$ for each $k$.
