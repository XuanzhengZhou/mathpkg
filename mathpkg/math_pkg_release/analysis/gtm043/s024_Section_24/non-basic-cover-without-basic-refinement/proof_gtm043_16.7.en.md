Suppose that $\mathcal{V} = \{X - Z_k\}_{k \leq s}$ refines $\mathcal{U}$, where each $Z_k$ is a zero-set. For each $k$, $Z_k$ contains either $F$ or $F$. Define
$$Z = \bigcap \{Z_k: Z_k \supset F\}, \quad \text{and} \quad Z = \bigcap \{Z_k: Z_k \supset F\}.$$
Then $Z \supset F$, and $Z \supset F$. Since $F$ and $F$ are not contained in disjoint zero-sets, we have
$$\bigcap_{k \leq s} Z_k = Z \cap Z \neq \emptyset.$$
Therefore $\mathcal{V}$ is not a cover.
