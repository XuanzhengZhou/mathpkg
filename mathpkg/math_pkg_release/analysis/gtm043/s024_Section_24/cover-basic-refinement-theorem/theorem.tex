Let $\mathcal{V} = \{V_k\}_{k \leq s}$ be a cover of $X$. If $\mathcal{V}$ is basic, or if $X$ is normal, then there exist a basic refinement $\mathcal{W} = \{W_k\}_{k \leq s}$ of $\mathcal{V}$ such that $W_k \subset V_k$ for each $k$, and $\{W_1, \cdots, W_k, V_{k+1}, \cdots, V_s\}$ is still a cover of $X$ for each $k$.
