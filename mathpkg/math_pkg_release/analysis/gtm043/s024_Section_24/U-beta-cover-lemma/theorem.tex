For $U \subset X$, define $U^\beta = \beta X - \operatorname{cl}_{\beta X}(X - U)$. If $\mathcal{U} = \{U_k\}_{k \leq s}$ is a basic cover of $X$, then $\mathcal{U}^\beta = \{U_k^\beta\}_{k \leq s}$ is a cover of $\beta X$, of the same order as $\mathcal{U}$. Moreover, $U^\beta \cap X = U$, so $U^\beta \subset \operatorname{cl}_{\beta X} U^\beta = \operatorname{cl}_{\beta X} U$.
