The set $\mathfrak{M}^* = \mathfrak{M}^*(X)$ of all maximal ideals in $C^*(X)$ is in one-one correspondence with $\beta X$ via the mapping $p \mapsto M^{*p}$. The Stone topology on $\mathfrak{M}^*$ is defined by transferring the topology of $\beta X$ via this bijection. Intrinsically, for each $f \in C^*(X)$, define
$$\mathfrak{E}^*(f) = \{M \in \mathfrak{M}^* : f \in M\}.$$
These sets form a base for the closed sets of the Stone topology. The set $\mathfrak{M}^*$ endowed with this topology is called the \textbf{structure space} of $C^*(X)$.
