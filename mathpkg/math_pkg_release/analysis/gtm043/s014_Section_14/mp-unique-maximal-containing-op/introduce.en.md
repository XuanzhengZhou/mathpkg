This theorem establishes that the $z$-ideal $O^p$ is contained in exactly one maximal ideal, namely $M^p$. The proof splits into necessity and sufficiency. For necessity, one shows that if an ideal containing $O^p$ is contained in $M^p$ and $M^p$ is the only maximal ideal containing it, then any function $f$ whose cozero-set meets every zero-set in the ideal must belong to the ideal. For sufficiency, one uses the Stone-Čech compactification to separate $p$ from any other point $q$ via a bounded function.
