For each point $p \in \beta X$, the set $O^p$ is defined as
$$O^p = \{f \in C(X) : \mathrm{cl}_{\beta X} Z(f) \text{ is a neighborhood of } p\}.$$
Equivalently, $f \in O^p$ if and only if there exists a neighborhood $V$ of $p$ in $\beta X$ such that $Z(f) \supset V \cap X$. The set $O^p$ is a $z$-ideal in $C(X)$ contained in $M^p$. For $p \in X$, $O^p$ coincides with the ideal $O_p$.
