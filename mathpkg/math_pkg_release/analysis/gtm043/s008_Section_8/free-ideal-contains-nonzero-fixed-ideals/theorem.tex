Every free ideal $I$ in $C(X)$ or $C^*(X)$ contains nonzero fixed ideals. More precisely, there exists a nonzero function $h \in I$ with $Z(h) \neq \emptyset$, and then the principal ideal $(h) \subseteq I$ is a nonzero fixed ideal.

In particular, if $I$ is free, it contains a nonzero function $f$. If $Z(f) \neq \emptyset$, take $h = f$. If $Z(f) = \emptyset$, let $r$ be any value assumed by $f$; then $h = f \cdot (f - r)$ satisfies $Z(h) \neq \emptyset$ and $h \in I$.
