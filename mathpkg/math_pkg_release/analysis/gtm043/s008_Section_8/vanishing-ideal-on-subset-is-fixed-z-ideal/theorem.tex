For any nonempty subset $S \subseteq X$, the set
$$I_S = \{f \in C(X) : f[S] = \{0\}\}$$
is a fixed ideal in $C(X)$; in fact, it is a $z$-ideal. Moreover, $\bigcap Z[I_S] \supseteq \overline{S}$, so the ideal is nonzero as long as $S$ is not dense in $X$. Its intersection with $C^*(X)$ is a fixed ideal in $C^*(X)$.
