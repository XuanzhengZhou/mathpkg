Let a partial ordering relation be defined on the ring $A$. Then $A$ is called a **partially ordered ring** provided that
$$a \geq b \text{ implies } a + x \geq b + x \text{ for all } x, \text{ and}$$
$$a \geq 0 \text{ and } b \geq 0 \text{ implies } ab \geq 0.$$

The following facts are evident: $a \geq b$ if and only if $a - b \geq 0$.
