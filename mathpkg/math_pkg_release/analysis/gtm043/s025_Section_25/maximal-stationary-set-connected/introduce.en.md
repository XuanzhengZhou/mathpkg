Lemma 16.31 states that for a compact space $X$, every maximal stationary set of an algebraic subring $A$ (a maximal subset on which all functions in $A$ are constant) is connected. The proof expresses such a stationary set as an intersection of sets of the form $\{x: g(x) \leq \epsilon\}$ for nonnegative $g \in A$, and uses the square-root closure property of algebraic subrings to conclude connectedness.
