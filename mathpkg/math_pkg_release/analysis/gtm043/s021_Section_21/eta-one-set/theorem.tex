An ordered set $E$ is called an \emph{$\eta_1$-set} if for any countable subsets $A$ and $B$ of $E$, with $A < B$ ($a < b$ for all $a \in A, b \in B$), there exists $v \in E$ satisfying $A < v < B$.

With $A$ or $B$ empty, this implies no countable subset is coinitial or cofinal. The definition is equivalent to: $E$ has more than one element, and whenever $A$ and $B$ are nonempty countable subsets with $A < B$, there exists $v \in E$ with $A < v < B$.
