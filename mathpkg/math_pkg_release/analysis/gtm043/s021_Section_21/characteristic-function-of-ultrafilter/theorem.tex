Let $\mathcal{U}$ be an ultrafilter on a set $X$. Its characteristic function $\chi_{\mathcal{U}}$ is defined on the set of all subsets of $X$ by:
$$\chi_{\mathcal{U}}(A) = 1 \quad \text{if} \quad A \in \mathcal{U}, \qquad \chi_{\mathcal{U}}(A) = 0 \quad \text{if} \quad A \notin \mathcal{U}.$$

The function $\chi_{\mathcal{U}}$ satisfies $\chi_{\mathcal{U}}(X) = 1$ and
$$\chi_{\mathcal{U}}(A \cup B) = \chi_{\mathcal{U}}(A) + \chi_{\mathcal{U}}(B) \quad (A \cap B = \emptyset).$$
Thus $\chi_{\mathcal{U}}$ is a nonzero, finitely additive $\{0,1\}$-valued function on the family of all subsets of $X$.
Conversely, any nonzero, finitely additive $\{0,1\}$-valued function $\mu$ on the power set of $X$ defines an ultrafilter $\mathcal{U} = \{A : \mu(A) = 1\}$.
