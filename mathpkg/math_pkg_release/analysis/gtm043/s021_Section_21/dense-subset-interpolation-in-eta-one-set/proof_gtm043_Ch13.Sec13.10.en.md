Given countable $A < B$ in $F$, first apply the $\eta_1$ property to get $v_1 \in F$ with $A < v_1 < B$. Then apply it again to get $v_2 \in F$ with $v_1 < v_2 < B$ (or symmetrically if $A$ is nonempty).

Since $S$ is dense in $F$, there exists $v \in S$ with $v_1 < v < v_2$. Then $A < v_1 < v < v_2 < B$, so $A < v < B$.
