Every $\{0,1\}$-valued measure is automatically $\mathfrak{m}$-additive for all nonmeasurable cardinals $\mathfrak{m}$. Failure of $\mathfrak{m}$-additivity would produce a nontrivial measure on an index set of size $\mathfrak{m}$, contradicting nonmeasurability.
