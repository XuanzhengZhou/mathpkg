For $C' \subset C(X)$, write $Z[C'] = \{Z(f) : f \in C'\}$ for the family of zero-sets of functions in $C'$. The family of all zero-sets in $X$ is denoted
$$Z(X) = Z[C(X)] = Z[C^*(X)].$$

Note that $C$ and $C^*$ yield the same zero-sets: if $f \in C$ and $g = |f| \wedge 1$, then $g \in C^*$ and $Z(g) = Z(f)$.
