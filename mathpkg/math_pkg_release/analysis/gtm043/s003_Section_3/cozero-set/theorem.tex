A \emph{cozero-set} is the complement of a zero-set, i.e., an open set of the form
$$X - Z(f) = \operatorname{pos} |f| = \{x \in X : f(x) \neq 0\}.$$

For any $f \in C(X)$, the sets
$$\operatorname{pos} f = \{x: f(x) > 0\}, \quad \operatorname{neg} f = \{x: f(x) < 0\} = \operatorname{pos}(-f)$$
are cozero-sets. Indeed,
$$\{x: f(x) \geq 0\} = Z(f \wedge 0) = Z(f - |f|),$$
$$\{x: f(x) \leq 0\} = Z(f \vee 0) = Z(f + |f|).$$
