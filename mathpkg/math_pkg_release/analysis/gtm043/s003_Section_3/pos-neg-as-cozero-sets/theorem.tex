For any $f \in C(X)$, the sets
$$\operatorname{pos} f = \{x: f(x) > 0\}, \qquad \operatorname{neg} f = \{x: f(x) < 0\} = \operatorname{pos}(-f)$$
are cozero-sets. Moreover,
$$\{x: f(x) \geq 0\} = Z(f \wedge 0) = Z(f - |f|),$$
$$\{x: f(x) \leq 0\} = Z(f \vee 0) = Z(f + |f|),$$
and conversely $X - Z(f) = \operatorname{pos} |f|$.
