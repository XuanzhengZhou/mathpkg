If $X$ is not pseudocompact, then $C(X) \neq C^*(X)$, so there exists an unbounded function $h \in C(X)$. Applying Corollary 1.20 with $E = X$, we obtain a $C$-embedded copy of $\mathbf{N}$ contained in $X$. Conversely, if $X$ contains a $C$-embedded copy of $\mathbf{N}$, the unbounded identity function on $\mathbf{N}$ extends to an unbounded function on $X$, so $X$ is not pseudocompact.
