If $f$ is a unit of $C(X)$, then $f^{-1} \in C(X)$ exists, so $f(x) \cdot f^{-1}(x) = 1$ for all $x$, hence $f(x) \neq 0$ for all $x$, i.e., $Z(f) = \emptyset$. Conversely, if $Z(f) = \emptyset$, then $f(x) \neq 0$ for all $x \in X$, so the function $1/f$ is defined and continuous on $X$, yielding $f^{-1} \in C(X)$.
