A subspace $S$ is $C$-embedded in $X$ if every continuous real-valued function on $S$ (bounded or not) extends continuously to $X$. This is a stronger condition than $C^*$-embedding, which only requires extension of bounded functions. Notable examples: $\mathbf{R} - \{0\}$ is not $C^*$-embedded in $\mathbf{R}$, while $\mathbf{N}$ is $C$-embedded in $\mathbf{R}$.
