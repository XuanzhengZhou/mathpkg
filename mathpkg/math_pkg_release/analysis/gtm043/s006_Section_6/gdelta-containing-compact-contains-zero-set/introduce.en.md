Theorem 3.11(b) provides a powerful tool for relating $G_{\delta}$ sets and zero-sets in the presence of compactness. Given a compact set $S$ contained in a $G_{\delta}$ set $A = \bigcap_n U_n$, each open set $U_n$ can be separated from $S$'s complement by a zero-set-neighborhood of $S$, and the countable intersection of these zero-sets yields a zero-set $F$ with $S \subset F \subset A$. Important consequences: every compact $G_{\delta}$ is a zero-set, and every $G_{\delta}$-point (a point that is a $G_{\delta}$ set) is a zero-set.
