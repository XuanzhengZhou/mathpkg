A collection $\mathcal{B}$ of closed sets is a \emph{base for the closed sets} if every closed set in $X$ is an intersection of members of $\mathcal{B}$.

Equivalently, $\mathcal{B}$ is a base if whenever $F$ is closed and $x \in X - F$, there is a member of $\mathcal{B}$ that contains $F$ but not $x$.
