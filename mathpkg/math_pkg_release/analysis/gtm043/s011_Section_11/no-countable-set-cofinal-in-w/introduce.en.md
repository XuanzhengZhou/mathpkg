In the space $W = W(\omega_1)$ of all countable ordinals, no countable subset can be cofinal. This follows from a cardinality argument: if $S$ were a countable cofinal set, then $W = \bigcup_{\sigma \in S} W(\sigma)$, and since each $|\sigma| \leq \aleph_0$, we would have $\aleph_1 = |W| \leq |S| \cdot \aleph_0 = \aleph_0$, a contradiction. This property is essential for establishing that every continuous function on $W$ is eventually constant.
