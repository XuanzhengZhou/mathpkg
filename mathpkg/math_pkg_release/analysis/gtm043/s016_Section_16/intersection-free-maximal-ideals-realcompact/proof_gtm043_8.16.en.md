If $$f$$ has compact support, then, of course, it belongs to every free maximal ideal.

Conversely, if $$f$$ has noncompact support, there exists a point $$p$$ that belongs to $$\operatorname{cl}_{\beta X}(X - Z(f))$$ but not to $$X$$. As $$X$$ is realcompact, there is a function $$h$$ in $$C(X)$$ for which $$h^*(p) = \infty$$. Then $$h$$ is unbounded on $$X - Z(f)$$, and so, by Corollary 1.20, this set contains a noncompact, closed set $$S$$ that is $$C$$-embedded in $$X$$. By Theorem 1.18, the disjoint sets $$S$$ and $$Z(f)$$ are completely separated; hence they have disjoint closures in $$\beta X$$. For any point $$q$$ in $$\operatorname{cl}_{\beta X}S - X$$, then, we have $$q \notin \operatorname{cl} Z(f)$$. (Such a point exists, as $$S$$ is closed in $$X$$ but not compact.) Therefore, by the Gelfand-Kolmogoroff theorem, $$f$$ does not belong to the free maximal ideal $$M^q$$.
