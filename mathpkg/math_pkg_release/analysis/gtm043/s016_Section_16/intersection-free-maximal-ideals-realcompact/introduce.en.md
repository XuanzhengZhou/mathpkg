For a realcompact space $$X$$, the intersection of all free maximal ideals in the ring $$C(X)$$ of continuous real-valued functions is precisely the set $$C_K(X)$$ of functions having compact support. Functions with compact support trivially belong to every free maximal ideal. The converse uses the realcompactness to construct, for a function with noncompact support, a point $$p \in \beta X \setminus X$$ belonging to the closure of its cozero-set, from which an unbounded function $$h$$ yields a $$C$$-embedded noncompact closed set, eventually placing the function outside some free maximal ideal via the Gelfand-Kolmogoroff theorem.
