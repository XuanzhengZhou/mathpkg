Let $\mathfrak{s}$ be a homomorphism from $C(Y)$ into $C(X)$. Then the element $e = \mathfrak{s}\mathbf{1}$ is an idempotent in $C(X)$, hence the characteristic function of an open-and-closed set $E = e^{\leftarrow}(1)$ in $X$. The element $e$ is the unity element of the image ring $\mathfrak{s}[C(Y)]$, and for every $g \in C(Y)$,

$$(\mathfrak{s}g)[X - E] = \{0\}.$$

If $E \neq \emptyset$, there exists a unique continuous mapping $\tau$ of $E$ into $vY$ such that for all $x \in E$ and $g \in C(Y)$,

$$(\mathfrak{s}g)(x) = g^v(\tau x).$$

The analogous result holds for $C^*$ in place of $C$.
