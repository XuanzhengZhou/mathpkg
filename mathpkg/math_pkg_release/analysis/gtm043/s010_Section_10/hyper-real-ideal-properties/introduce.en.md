This set of propositions characterizes when residue classes in hyper-real fields become infinitely large. The key result is that $|M(f)|$ is infinitely large for some maximal ideal $M$ exactly when $f$ is unbounded on $X$. Moreover, $|M(f)|$ is infinitely large for every free maximal ideal precisely when $f$ is unbounded on every noncompact zero-set. These results connect the nonarchimedean nature of hyper-real fields with the topological properties of the underlying space $X$.
