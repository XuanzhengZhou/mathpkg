When the residue class field modulo a maximal ideal $M$ is \emph{not} real, we call it a \emph{hyper-real field} and refer to $M$ as a \emph{hyper-real ideal}.

Recall that a totally ordered field is said to be \emph{archimedean} if for every element $a$, there exists $n \in \mathbf{N}$ such that $n \geq a$. A nonarchimedean field is characterized by the presence of \emph{infinitely large} elements, that is, elements $a$ such that $a > n$ for every $n \in \mathbf{N}$. An element $b$ is \emph{infinitely small} if it is positive but smaller than $1/n$ for every $n \in \mathbf{N}$; $b$ is infinitely small if and only if $1/b$ is infinitely large.

Every archimedean field is embeddable in $\mathbf{R}$. It follows that every hyper-real field is nonarchimedean. Thus, the residue class field is archimedean if and only if it is real.
