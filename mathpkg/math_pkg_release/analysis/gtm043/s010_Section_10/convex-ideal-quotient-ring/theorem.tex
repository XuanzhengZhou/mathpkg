Let $I$ be an ideal in a partially ordered ring $A$. Then $A/I$ is a partially ordered ring, with the order defined by
$$
I(a) \geq 0 \iff \text{there exists } x \geq 0 \text{ in } A \text{ such that } a \equiv x \pmod{I},
$$
if and only if $I$ is a convex ideal.

Whenever we say that a residue class ring is partially ordered, or totally ordered, we always mean that it is an ordered ring whose order is induced as above. The canonical mapping of $A$ onto $A/I$ is order-preserving, i.e., if $a \geq b$, then $I(a) \geq I(b)$.
