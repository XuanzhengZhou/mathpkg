If $b \in P^a$, then $b^n = sa$ for some $s$ and $n$, so $b^{2n} = s^2 a^2 < a$. Hence $|b| < a^{1/n}$. Conversely, $|b| < a^{1/n}$ implies $b \in P^a$ since $P^a$ is prime and convex. For $P_a$: if $b \in P_a$, then $|b| < a^n$ for all $n$ since $P_a$ is convex and contains no power of $a$. If $|b| < a^n$ for all $n$, then $a \notin P^{|b|}$, so $b \in P_a$.
