A nonnegative real-valued function $\sigma$ on a [real] linear space $\mathcal{E}$ is a pseudonorm on $\mathcal{E}$ if it satisfies the following conditions:

(i') $\sigma(0) = 0$,
(ii') $\sigma(\lambda x) = |\lambda| \sigma(x)$ for every $x$ in $\mathcal{E}$ and every [real] scalar $\lambda$,
(iii) $\sigma(x + y) \leq \sigma(x) + \sigma(y)$ for all $x, y$ in $\mathcal{E}$.

It is readily seen that if $\sigma$ is a pseudonorm on a linear space $\mathcal{E}$, then

$$\rho_0(x, y) = \sigma(x - y), \quad x, y \in \mathcal{E},$$

$$\rho_0(x + z, y + z) = \rho_0(x, y)$$
