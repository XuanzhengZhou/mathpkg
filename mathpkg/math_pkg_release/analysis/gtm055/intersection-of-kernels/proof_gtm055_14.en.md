By Proposition 14.7 the given topology on $\mathcal{E}$ is induced by a family $\{\sigma_\gamma\}$ of pseudonorms. Hence the first assertion is an immediate consequence of Proposition 14.5. Moreover, if $M$ is closed and $x_0 \notin M$, then there exist indices $\gamma_1, \ldots, \gamma_n$ and a positive number $\varepsilon$ such that if we set $\sigma = \sigma_{\gamma_1} \vee \cdots \vee \sigma_{\gamma_n}$, then the ball $\{x \in M : \sigma(x - x_0) < \varepsilon\}$ is disjoint from $M$. Hence there exists a linear functional $f$ on $\mathcal{E}$ such that (17) is satisfied and such that $f$ is bounded with respect to $\sigma$. But then $f$ is continuous on $\mathcal{E}$. Finally, if $\mathcal{E}$ is Hausdorff as a topological space, then $\{0\}$ is a closed set in $\mathcal{E}$. Hence if $x_0 \neq 0$ there exists a functional $f$ in $\mathcal{E}^*$ such that $f(x_0) \neq 0$, and the result follows.
