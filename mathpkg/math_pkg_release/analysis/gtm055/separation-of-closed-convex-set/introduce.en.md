Proposition 14.15 is a geometric Hahn-Banach theorem in locally convex spaces. Any closed convex set $C$ can be strictly separated from any point $x_0 \notin C$ by a closed real hyperplane defined by a continuous linear functional. Consequently, every closed convex set equals the intersection of all closed half-spaces that contain it.
