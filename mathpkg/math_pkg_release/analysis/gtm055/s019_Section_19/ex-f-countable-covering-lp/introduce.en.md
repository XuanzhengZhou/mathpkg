Exercise F shows that for any countable measurable cover of $X$, the union of the $L_p$ spaces supported on each covering set spans $L_p(X)$ for $1 \leq p < \infty$. This is because any function in $L_p$ can be approximated by functions with support in finitely many members of the cover. The result fails for $p = \infty$ since uniform limits of functions with restricted support may not have the same support.
