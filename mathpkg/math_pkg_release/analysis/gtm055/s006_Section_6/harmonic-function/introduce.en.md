A harmonic function is a twice continuously differentiable function satisfying Laplace's equation. In complex analysis, the real and imaginary parts of any analytic function are harmonic, establishing a deep connection between analytic function theory and potential theory. Conversely, on simply connected domains, every harmonic function is the real part of some analytic function.
