A mapping $f$ of one partially ordered set into another is \textit{monotone increasing} [resp. \textit{decreasing}] if $x \leq y$ implies $f(x) \leq f(y)$ [resp. $f(x) \geq f(y)$].

A mapping $f$ is \textit{strictly monotone increasing} [resp. \textit{decreasing}] if $x < y$ implies $f(x) < f(y)$ [resp. $f(x) > f(y)$].
