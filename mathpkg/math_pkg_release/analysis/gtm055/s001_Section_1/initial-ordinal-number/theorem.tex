If $c$ is an infinite cardinal number, then the \textit{initial ordinal number} of $c$ is the smallest ordinal number $\alpha$ such that $\operatorname{card} \alpha = c$. In particular, $\omega$, the smallest infinite ordinal number, is the initial ordinal number of $\aleph_0$. The first uncountable ordinal number, i.e., the initial ordinal number of the smallest cardinal number exceeding $\aleph_0$, is denoted by $\Omega$. Thus the initial segment $W(\Omega)$ coincides with the set of all countable ordinal numbers. Every countable subset of $W(\Omega)$ is bounded in $W(\Omega)$.
