A function $s$ on a set $X$ of the form
$$s = \sum_{i=1}^{n} \alpha_i \chi_{E_i},$$
where $E_1, \ldots, E_n$ are subsets of $X$ and $\alpha_1, \ldots, \alpha_n$ are complex numbers, is called a \textit{simple function} on $X$.

The partition of $X$ determined by $\{E_1, \ldots, E_n\}$ consists of all sets of the form
$$A_1 \cap \cdots \cap A_n$$
where each $A_i$ is either $E_i$ or $X \setminus E_i$. On each set $F$ of this partition, $s$ is constant, and the value $\beta_F$ of $s$ on $F$ is
$$\beta_F = \sum_{F \subset E_i} \alpha_i.$$
