A simple function is a finite linear combination of characteristic functions with complex coefficients. On each atom of the coarsest partition that refines all the supporting sets $E_i$, the simple function is constant, and its value is the sum of the coefficients $\alpha_i$ for those $E_i$ that contain the atom. Simple functions are the building blocks of Lebesgue integration.
