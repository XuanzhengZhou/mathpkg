The \textit{extended real number system} is the set $\mathbb{R} \cup \{+\infty, -\infty\}$ with the ordering extended so that $-\infty < t < +\infty$ for every real number $t$. Addition is extended by:
$$t + (\pm \infty) = (\pm \infty) + t = \pm \infty$$
for every finite $t$, while $(\pm \infty) + (\mp \infty)$ and $(\pm \infty) - (\pm \infty)$ remain undefined.

Multiplication is extended by:
$$t(\pm \infty) = (\pm \infty)t = \begin{cases} \pm \infty, & t > 0, \\ 0, & t = 0, \\ \mp \infty, & t < 0, \end{cases}$$
and $(\pm \infty)(\pm \infty) = +\infty$, $(\pm \infty)(\mp \infty) = -\infty$.

For an extended real number $t$, define $t^+ = t \lor 0$, $t^- = -(t \land 0)$, and $|t| = t^+ + t^-$. The ordinary real numbers $\mathbb{R}$ are called \textit{finite} (real) numbers.
