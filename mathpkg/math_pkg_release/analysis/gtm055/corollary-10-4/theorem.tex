Let $\mu$ be a Borel measure on a locally compact Hausdorff space $X$, and let $\mathcal{H}$ denote the collection of all compact subsets of $X$. If every set $K$ in $\mathcal{H}$ is outer regular with respect to $\mu$, or if every topologically bounded open set in $X$ is inner regular with respect to $\mu$, then every set in the $\sigma$-ring $S(\mathcal{H})$ is regular with respect to $\mu$. In particular, if $X$ is $\sigma$-compact, then a Borel measure $\mu$ on $X$ is regular if and only if every compact subset of $X$ is outer regular with respect to $\mu$ (or, equivalently, if and only if every topologically bounded open subset of $X$ is inner regular with respect to $\mu$).
