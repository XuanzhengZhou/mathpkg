Let $K$ be a compact subset of $\mathbb{C}$ and let $\gamma$ be an oriented envelope of $K$ in an open set $U \supset K$. For any $\mathcal{E}$-valued function $\Phi$ that is locally analytic on $U$, the Cauchy integral representation
$$\Phi(\lambda) = \frac{1}{2\pi i} \int_{\gamma} \frac{\Phi(\zeta)}{\zeta - \lambda} d\zeta$$
holds for all $\lambda$ in some neighborhood of $K$. Moreover, for any complex measure $\xi$ supported on $K$, the Fubini theorem for vector-valued integrals yields
$$\int_K \Phi(\lambda) d\xi(\lambda) = \frac{-1}{2\pi i} \int_{\gamma} \Phi(\zeta) \left[ \int_K \frac{d\xi(\lambda)}{\zeta - \lambda} \right] d\zeta.$$
