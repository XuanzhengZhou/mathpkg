If $(X, \mathbf{S}, \mu)$ is a measure space, then the collection $\mathcal{L}_\infty(X) = \mathcal{L}_\infty(X, \mathbf{S}, \mu)$ of all essentially bounded, measurable, complex-valued functions on $X$ is a linear space (with respect to pointwise linear operations), and setting

$$\| f \|_\infty = \operatorname{ess sup}_X |f|$$

defines a pseudonorm on $\mathcal{L}_\infty(X)$.
