Let $\mathcal{E}$ and $\mathcal{F}$ be Banach spaces and let $T$ be an element of $\mathcal{L}(\mathcal{E}, \mathcal{F})$. If an $\mathcal{E}$-valued mapping $\Phi$ is integrable over an interval $[a, b]$ with respect to an integrator $\alpha$, then the $\mathcal{F}$-valued mapping $T \circ \Phi$ is also integrable over $[a, b]$ with respect to $\alpha$, and

$$T \left( \int_a^b \Phi(t) d\alpha(t) \right) = \int_a^b T(\Phi(t)) d\alpha(t).$$
