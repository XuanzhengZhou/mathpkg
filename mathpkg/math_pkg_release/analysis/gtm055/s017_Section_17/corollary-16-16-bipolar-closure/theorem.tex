If $\langle \mathcal{E}, \mathcal{F} \rangle$ is a dual pair and $M$ is an arbitrary subset of $\mathcal{E}$, then
$$(^0(M^0))^0 = M^0.$$

If $A$ is an absolutely convex subset of $\mathcal{E}$, then $^0(A^0)$ coincides with the closure of $A$ in the weak topology. In particular, if $A$ is an absolutely convex set in $\mathcal{E}$ that is closed in the weak topology, then $^0(A^0) = A$.
