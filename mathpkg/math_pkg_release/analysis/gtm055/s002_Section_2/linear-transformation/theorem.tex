Let $\mathcal{E}$ and $\mathcal{F}$ be linear spaces over the same scalar field. A mapping $T : \mathcal{E} \to \mathcal{F}$ is a \textbf{linear transformation} if $T(\alpha x + \beta y) = \alpha T x + \beta T y$ for all $x, y \in \mathcal{E}$ and scalars $\alpha, \beta$. The \textbf{range} $\mathcal{R}(T) = \{T x : x \in \mathcal{E}\}$ is a linear submanifold of $\mathcal{F}$, and the \textbf{kernel} or null space $\mathcal{K}(T) = \{x \in \mathcal{E} : T x = 0\}$ is a linear submanifold of $\mathcal{E}$.
