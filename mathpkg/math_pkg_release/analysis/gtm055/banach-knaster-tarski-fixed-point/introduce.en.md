The Banach–Knaster–Tarski Lemma states that every monotone increasing mapping from a complete lattice to itself has a fixed point. The proof constructs the fixed point as the infimum of the set of elements $x$ satisfying $\varphi(x) \leq x$. This result is fundamental in fixed-point theory and has applications in set theory (e.g., the Cantor–Bernstein Theorem) and computer science (denotational semantics).
