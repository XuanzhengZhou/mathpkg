If $\mu$ is a signed measure on a measurable space $(X, S)$, and if $X = A \cup B$ is a Hahn decomposition of $X$ with respect to $\mu$, then the set functions defined (unambiguously) on $S$ by

$$\mu_+(E) = \mu(E \cap A) \quad \text{and} \quad \mu_-(E) = -\mu(E \cap B), \quad E \in S,$$

are called the positive and negative variations of $\mu$, respectively.
