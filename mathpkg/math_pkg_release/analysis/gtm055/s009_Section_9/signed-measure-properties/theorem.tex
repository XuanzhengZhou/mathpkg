If $\mu$ is a signed measure on a measurable space $(X, \mathbf{S})$, and if $\mu(E) < +\infty$ for some measurable set $E$, then $\mu(F) < +\infty$ for every measurable set $F \subset E$. Similarly, if $\mu(E) > -\infty$, then $\mu(F) > -\infty$ for every measurable $F \subset E$. In particular, if $\mu(E)$ is finite, then $\mu$ is finite-valued on measurable subsets of $E$. If $\mu(E) = +\infty$ $[\mu(E) = -\infty]$ for any measurable $E$, then $\mu(F) > -\infty$ $[\mu(F) < +\infty]$ for every measurable $F$.
