Lemma 8.4 records basic consistency properties of signed measures. Since a signed measure cannot take opposite infinite values on disjoint sets, the sign of infinity is globally determined: if $+\infty$ is attained anywhere, $-\infty$ is never attained, and vice versa. Moreover, signed measure values propagate downward: a set of finite signed measure can have only measurable subsets of finite signed measure.
