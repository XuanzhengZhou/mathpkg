A value on a (real or complex) linear space $\mathcal{E}$ is a nonnegative real-valued function $v$ on $\mathcal{E}$ having the following properties:

(i) $v(x) = 0$ if and only if $x = 0$,
(ii) $v(-x) = v(x)$ for all $x$ in $\mathcal{E}$,
(iii) $v(x + y) \leq v(x) + v(y)$ for all $x, y$ in $\mathcal{E}$.
(The inequality (iii) is known as the triangle inequality for $v$.)

It follows from properties (ii) and (iii) in the above definition that

$$v(x - y) = v(y - x) \quad \text{and} \quad v(x - z) \leq v(x - y) + v(y - z)$$

for all vectors $x, y, z$ in a linear space $\mathcal{E}$ equipped with a value $v$. Hence the function $\rho(x, y) = v(x - y)$ is a metric on $\mathcal{E}$, called the metric defined by $v$.
