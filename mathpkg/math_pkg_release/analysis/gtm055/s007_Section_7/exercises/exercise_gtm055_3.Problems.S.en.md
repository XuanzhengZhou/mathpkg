(i) Let $\ell$ be a line in $\mathbb{C}$, and let $\mathcal{K} = \{D_0, \ldots, D_n\}$ be a circle chain such that the center $\alpha_i$ of each disc $D_i$ lies on $\ell$. Show that if $\{(D_0, f_0), \ldots, (D_n, f_n)\}$ is an analytic continuation along $\mathcal{K}$, then there exists a (unique) analytic function $f$ on the union $\Delta = D_0 \cup \cdots \cup D_n$ that extends each $f_i$, $i = 0, \ldots, n$. (If the centers $\alpha_i$ all lie on the line segment $\sigma(\alpha_0, \alpha_n)$, such an analytic continuation is known as an analytic continuation along the directed line segment $\sigma(\alpha_0, \alpha_n)$.)

(ii) If $\mathcal{K}_1, \ldots, \mathcal{K}_p$ is a sequence of circle chains, and if there is given an analytic continuation along each of these circle chains, we say that this sequence of analytic continuations is chained if the last function element in the analytic continuation along $\mathcal{K}_i$ coincides with the first function element in the analytic continuation along $\mathcal{K}_{i+1}$, $i = 1, \ldots, p - 1$.

(Hint for (i): Deal first with the case in which the centers $\alpha_i$ are arranged in order in one direction or the other along $\ell$. Begin by showing that if a disc $D_i$ in $\mathcal{K}$ is wholly contained in some other disc in $\mathcal{K}$, then the function element $(D_i, f_i)$ can be removed from the analytic continuation without prejudice to any hypothesis. Hence one may assume without loss of generality that no disc in $\mathcal{K}$ is contained in any other. Show that if this is the case, and if $D_i \cap D_k \neq \emptyset$ for some $i$ and $k$ with $i < k - 1$, then $D_i \cap D_{k-1} \neq \emptyset$.)
