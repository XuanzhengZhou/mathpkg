Let $\Delta$ be a simply connected domain, and let $f$ be an analytic function on $\Delta$ with no zeros in $\Delta$. Show that there exists an analytic determination of $\log f(\lambda)$ on $\Delta$, that is, an analytic function $\log f(\lambda)$ such that $\exp\{\log f(\lambda)\} \equiv f(\lambda)$ on $\Delta$.

In particular, if $0 \notin \Delta$, then there is an analytic determination of $\log \lambda$ on $\Delta$, unique up to an additive constant of the form $2k\pi i$, $k \in \mathbb{Z}$. (For example, if $C$ denotes an arbitrary closed, connected, unbounded subset of $\mathbb{C}$ that contains $0$, then there is an analytic determination of $\log \lambda$ on each component of $\mathbb{C} \setminus C$.)

Conclude that if $\Delta$ is a simply connected domain not containing $0$, then there exists a real harmonic function $\arg \lambda$ on $\Delta$ (unique up to an additive constant of the form $2k\pi$, $k \in \mathbb{Z}$) satisfying the condition $\lambda \equiv |\lambda| e^{i \arg \lambda}$.
