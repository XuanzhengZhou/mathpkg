On a simply connected domain, every real-valued harmonic function $u$ is the real part of some analytic function $f = u + iv$. The harmonic conjugate $v$ can be obtained by integrating the differential $-\frac{\partial u}{\partial y}\,dx + \frac{\partial u}{\partial x}\,dy$, which is exact precisely because $\Delta$ is simply connected and $u$ is harmonic (so the Cauchy-Riemann integrability condition holds). This result establishes a fundamental link between harmonic functions and analytic functions on simply connected domains.
