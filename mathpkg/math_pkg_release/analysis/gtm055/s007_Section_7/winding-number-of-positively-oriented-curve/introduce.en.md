A positively oriented closed curve with respect to a domain has winding number identically equal to one on that domain. This result formalizes the geometric intuition that a curve traced with the domain always on the left winds exactly once around each point of the domain. The proof constructs auxiliary arcs connecting the curve to a small circle in the domain, decomposing the configuration into simply closed arcs with zero winding number at interior points, and using the additivity of winding numbers.
