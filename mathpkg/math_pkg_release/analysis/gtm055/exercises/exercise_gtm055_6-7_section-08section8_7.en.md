G. For an arbitrary nonempty collection $\%$ of subsets of a set $X$ let $\% \neq \%$ denote the collection of all differences $E \setminus F$, where $E$ and $F$ belong to $\%$. Furthermore, let $\%_\sigma$ denote the collection of all unions of countable subcollections of $\%$, and let $\%^* = (\% \neq \%)_\sigma$. In terms of this operation it is possible to describe the $\sigma$-ring $S(\%)$ generated by $\%$. We suppose, without loss of generality, that $\emptyset \in \%$. Let $\%^{(0)} = \%$, and, for a given countable ordinal number $\alpha$, suppose $\%^{(\xi)}$ already defined for every ordinal number $\xi$ less than $\alpha$. Then if $\alpha$ is not a limit ordinal we set $\%^{(\alpha)} = (\%^{(\xi)})^*$, where $\alpha = \xi + 1$, while if $\alpha$ is a limit ordinal we set $\%^{(\alpha)} = \bigcup_{\xi < \alpha} \%^{(\xi)}$. (See Problem 1V for a discussion of transfinite definitions.) Show that if $\Omega$ denotes, as usual, the smallest uncountable ordinal number (Prob. 1W), then $S(\%) = \bigcup_{\xi < \Omega} \%^{(\xi)}$.
