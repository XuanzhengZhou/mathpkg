Let $M_0$ be the linear submanifold generated by $\mathcal{M}$ and $x_0$. Define $g$ on $M_0$ by $g(y + \lambda x_0) = \lambda$ for $y \in \mathcal{M}$. Then $g(\mathcal{M}) = (0)$, $g(x_0) = 1$. The set $H = \{z \in M_0 : g(z) = 1\}$ coincides with the coset $\mathcal{M} + x_0$, so $d(0, H) = d$. If $z \in M_0$ and $\|z\| < d$, then $g(z) \neq 1$, and it follows that $|g(z)| < 1$, so $\|g\| = 1/d$. The result follows by applying the Hahn-Banach theorem to extend $g$ to $\mathcal{E}$ preserving the norm.
