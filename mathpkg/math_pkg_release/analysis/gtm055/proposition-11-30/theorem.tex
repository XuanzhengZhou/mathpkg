The collection of all pseudonorms on a given linear space $\mathcal{E}$ is a directed set in the usual ordering of real-valued functions. If $\sigma_1, \ldots, \sigma_n$ are pseudonorms on $\mathcal{E}$, then both $\sigma_1 + \cdots + \sigma_n$ and $\sigma_1 \vee \cdots \vee \sigma_n$ are pseudonorms that dominate $\sigma_1, \ldots, \sigma_n$ in that ordering. If $\mathcal{T}_i$ denotes the topology induced on $\mathcal{E}$ by $\sigma_i$, $i = 1, \ldots, n$, then both $\sigma_1 + \cdots + \sigma_n$ and $\sigma_1 \vee \cdots \vee \sigma_n$ induce the topology $\mathcal{T} = \mathcal{T}_1 \vee \cdots \vee \mathcal{T}_n$.
