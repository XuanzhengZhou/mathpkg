If $\mathcal{E}$ is a topological linear space, then all translations $x \rightarrow x + x_0$ and all dilatations $x \rightarrow \alpha_0 x$ ($\alpha_0 \neq 0$) are homeomorphisms of $\mathcal{E}$ onto itself. Thus, in particular, $x_0 + V$ is a neighborhood of $x_0$ in $\mathcal{E}$ if and only if $V$ is a neighborhood of 0.
