This definition extends the notions of measurability and integrability to functions that are only defined almost everywhere with respect to a measure $\mu$. A function $f$ defined on a subset $A \subseteq X$ is defined a.e. if the complement of its domain is contained in a null set. Such a function is measurable $[\mu]$ if it agrees a.e. with some everywhere-defined measurable function $g$, and integrable $[\mu]$ if it agrees a.e. with some everywhere-defined integrable function $g$. The integral of $f$ is then defined as the integral of $g$. This formalism justifies identifying functions that differ only on sets of measure zero.
