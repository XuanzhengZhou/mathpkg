A family of vectors $\{x_\gamma\}_{\gamma \in \Gamma}$ in a Banach space $\mathcal{E}$ is weakly summable if $\sum_{\gamma \in \Gamma} |f(x_\gamma)| < +\infty$ for every continuous linear functional $f \in \mathcal{E}^*$. A weakly summable family need not be norm summable in general, but it defines a bounded linear operator $T: \mathcal{E}^* \to \ell_1(\Gamma)$ via $(Tf)_\gamma = f(x_\gamma)$. When $\mathcal{E}$ does not contain a copy of $c_0$, weak summability implies norm summability.
