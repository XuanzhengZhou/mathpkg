Runge's Theorem (Proposition 18.16) is a fundamental result in complex analysis stating that every locally analytic function on an open set $U \subset \mathbb{C}$ can be uniformly approximated on compact subsets by rational functions whose poles lie in a prescribed set $P$, provided $P$ meets every connected component of $\hat{\mathbb{C}} \setminus U$. The proof constructs an exhaustion of $U$ by compact sets $K_n$ and applies Proposition 18.14 to each $K_n$. A key special case is polynomial approximation on simply connected domains (where $P = \{\infty\}$).
