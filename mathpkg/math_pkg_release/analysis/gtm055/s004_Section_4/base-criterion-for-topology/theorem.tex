A collection $\mathcal{B}$ of subsets of a set $X$ forms a base for a topology on $X$ if and only if every point of $X$ lies in some element of $\mathcal{B}$ (briefly: $\mathcal{B}$ covers $X$) and the intersection of every pair of sets belonging to $\mathcal{B}$ is a union of sets belonging to $\mathcal{B}$. The only property that a collection $\mathcal{S}$ of subsets of $X$ must possess in order to be a subbase for a topology on $X$ is that it cover $X$.
