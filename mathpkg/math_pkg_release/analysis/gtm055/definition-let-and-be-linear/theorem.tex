Let $\mathcal{E}$ and $\mathcal{F}$ be linear spaces, and let $\psi$ be a bilinear functional on $\mathcal{E} + \mathcal{F}$. The triple $(\mathcal{E}, \mathcal{F}, \psi)$ is a dual pair if the following two conditions are satisfied:

(i) For each nonzero $x$ in $\mathcal{E}$ there is a vector $y$ in $\mathcal{F}$ such that $\psi(x, y) \neq 0$,

(ii) For each nonzero $y$ in $\mathcal{F}$ there is a vector $x$ in $\mathcal{E}$ such that $\psi(x, y) \neq 0$.

When dealing with a dual pair $(\mathcal{E}, \mathcal{F}, \psi)$, it is customary to supress the symbol for the distinguished bilinear functional $\psi$, writing $\psi(x, y) = \langle x, y \rangle$ instead. Likewise it will be convenient to write $\langle \mathcal{E}, \mathcal{F} \rangle$ for the dual pair consisting of $\mathcal{E}$ and $\mathcal{F}$ equipped with the bilinear functional $\langle , \rangle$.

If $\langle \mathcal{E}, \mathcal{F} \rangle$ is a dual pair then for each $y$ in $\mathcal{F}$ the function $f_y$ defined on $\mathcal{E}$ by setting $f_y(x) = \langle x, y \rangle$ is a linear functional, and it is the substance of condition (ii) in the above definition that the linear transformation $y \rightarrow f_y$ is an isomorphism of $\mathcal{F}$ into the full algebraic dual $\mathcal{E}'$ of $\mathcal{E}$.
