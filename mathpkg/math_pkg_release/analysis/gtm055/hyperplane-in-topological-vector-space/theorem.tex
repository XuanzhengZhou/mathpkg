If $\mathcal{E}$ is either a real or complex topological vector space, then a hyperplane in $\mathcal{E}$ is any set of the form
$$H = \{x \in \mathcal{E} : f(x) = c\},$$
where $c$ is a real number and $f$ is a continuous nonzero real linear functional on $\mathcal{E}$ (regarded as a real topological vector space).
