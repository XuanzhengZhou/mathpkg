If $X$ is a locally compact Hausdorff space (cf. Problem 3V), a subset $S$ of $X$ is said to be topologically bounded if its closure $S^-$ is compact. (Note that a topologically bounded subset of a locally compact metric space is a bounded set, but that the converse need not hold.)
