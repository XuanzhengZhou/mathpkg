If $f$ is a measurable extended real-valued function defined and nonnegative a.e. $[\mu]$ on a measure space $(X, \mathbf{S}, \mu)$, then the set function

$$v_f(E) = \int_E f \, d\mu, \quad E \in \mathbf{S},$$

is a measure on $(X, \mathbf{S})$ called the indefinite integral of $f$ with respect to $\mu$. If $g$ is an arbitrary measurable function that is either integrable with respect to $v_f$ or nonnegative, then

$$\int_X g \, dv_f = \int_X gf \, d\mu.$$
