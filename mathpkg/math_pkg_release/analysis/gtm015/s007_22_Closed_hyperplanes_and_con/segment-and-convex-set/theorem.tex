If $x, y \in E$, the segment from $x$ to $y$ is the set of all vectors of the form $\alpha x + (1 - \alpha) y$, where $0 \leq \alpha \leq 1$; such a vector is called an internal point of the segment if $0 < \alpha < 1$. A subset $A$ of $E$ is said to be convex if, for any $x, y \in A$, it contains the segment from $x$ to $y$.
