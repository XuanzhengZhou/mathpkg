In a real pre-Hilbert space, the inner product $(x, y)$ defined by

$$(x, y) = \frac{1}{4}\bigl\{\|x + y\|^2 - \|x - y\|^2\bigr\}$$

satisfies the following conditions for all vectors $x, y, z$ and all real scalars $\alpha$:

\begin{enumerate}
\item[(i)] $(x, x) = \|x\|^2$; thus $(x, x) \geq 0$ for all $x$, and $(x, x) = 0$ iff $x = \theta$;
\item[(ii)] $(y, x) = (x, y)$;
\item[(iii)] $(x + y, z) = (x, z) + (y, z)$;
\item[(iv)] $(\alpha x, y) = \alpha(x, y)$ for all real $\alpha$.
\end{enumerate}

Consequently,

\begin{enumerate}
\item[(v)] $(x, y + z) = (x, y) + (x, z)$;
\item[(vi)] $(x, \alpha y) = \alpha(x, y)$ for all real $\alpha$.
\end{enumerate}

Thus $(\cdot, \cdot)$ is a symmetric, positive definite, bilinear form on $E$.
