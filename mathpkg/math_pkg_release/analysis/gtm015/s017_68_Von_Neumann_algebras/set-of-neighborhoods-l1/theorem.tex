\begin{definition}
The set $\mathcal{V}$ of all neighborhoods of $e$ in $G$ is directed by reverse inclusion. For each $V \in \mathcal{V}$, let $\mathcal{F}_V$ denote the set of all nonnegative continuous functions on $G$ with compact support contained in $V$ and integral equal to $1$. The family $(\mathcal{F}_V)_{V \in \mathcal{V}}$ forms the basis for an approximate identity in $L^1(G)$.
\end{definition}
