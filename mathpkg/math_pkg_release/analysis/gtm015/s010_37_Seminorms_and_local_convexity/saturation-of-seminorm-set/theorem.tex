If $E$ is a vector space over $\mathbb{K}$ and $\mathcal{F}$ is any set of seminorms on $E$, then a seminorm $p$ belongs to the saturation $\mathcal{F}^*$ if and only if there exist finitely many seminorms $p_1, \ldots, p_n \in \mathcal{F}$ such that
$$p \leq p_1 + \cdots + p_n.$$
(Here $\mathcal{F}^*$ denotes the set of all seminorms on $E$ that are continuous for the topology $\tau(\mathcal{F})$ generated by $\mathcal{F}$.)
