# Proof of A convex set is closed if and only if it is weakly closed

Let $(E, \tau)$ be a locally convex, separated TVS over $\mathbb{K}$, and let $A$ be a convex subset of $E$.

Let us suppose $\mathbb{K} = \mathbb{C}$ (for $\mathbb{K} = \mathbb{R}$ the proof is even simpler). Both $\tau$ and $\sigma(E, E')$ are locally convex, separated topologies on $E$; they determine the same continuous linear forms (38.11), therefore the same continuous $\mathbb{R}$-linear forms (22.6), therefore the same closed convex sets (34.3).

By (34.3), in any locally convex TVS, a convex set is closed if and only if it is the intersection of the closed half-spaces determined by the continuous linear forms that bound it. Since $\tau$ and $\sigma(E, E')$ have the same continuous linear forms, a convex set $A$ is $\tau$-closed iff it is $\sigma(E, E')$-closed.
