Let $E$ be a TVS over $\mathbb{K}$ and write $E'$ for the vector space of all continuous linear forms on $E$ (with pointwise linear operations).
\begin{enumerate}
\item[(i)] If $A \subset E$ is bounded, then for each $f \in E'$, $f(A)$ is bounded in $\mathbb{K}$; the formula
$$p_A(f) = \sup \{|f(x)| : x \in A\} \quad (f \in E')$$
defines a seminorm $p_A$ on $E'$.
\item[(ii)] If $\mathfrak{S}$ is any set of bounded subsets of $E$, the $\mathfrak{S}$-topology on $E'$ is defined to be the locally convex topology generated by the family of seminorms $\{p_A : A \in \mathfrak{S}\}$.
\end{enumerate}
