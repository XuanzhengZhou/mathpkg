A closed set $\sigma$ containing the spectrum of $T$ is a spectral set precisely when the operator norm of every rational function $f(T)$ (with poles off $\sigma$) is bounded by the sup norm of $f$ on $\sigma$. This norm inequality is the defining property that makes spectral sets useful for extending functional calculi.
