For a self-adjoint element of a $C^*$-algebra, positivity can be characterized by the numerical status: $a \geq 0$ iff every state takes a nonnegative value at $a$. This provides a state-space characterization of the algebraic order, linking the cone of positive elements to evaluations by positive linear functionals.
