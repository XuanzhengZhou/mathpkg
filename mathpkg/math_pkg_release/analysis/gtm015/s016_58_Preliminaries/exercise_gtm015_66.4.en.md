# Exercise 4

Suppose $T \in \mathcal{L}(H)$, $\alpha \in \mathbb{C}$, and $\beta > 0$.

(i) The closed disc $\{\lambda \in \mathbb{C} : |\lambda - \alpha| \leq \beta\}$ is a spectral set for $T$ iff $\|T - \alpha I\| \leq \beta$.

(ii) The set $\{\lambda \in \mathbb{C} : |\lambda - \alpha| \geq \beta\}$ is a spectral set for $T$ iff $\alpha \notin \sigma(T)$ and $\|(T - \alpha I)^{-1}\| \leq 1/\beta$.
