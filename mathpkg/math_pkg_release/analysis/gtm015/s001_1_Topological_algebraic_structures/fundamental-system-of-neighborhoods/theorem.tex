In a topological space, a class $\mathcal{B}$ of subsets is a fundamental system of neighborhoods of a point $x$ if $\mathcal{B}$ is a base for the filter of neighborhoods of $x$. That is:
\begin{enumerate}
\item[(i)] every set in $\mathcal{B}$ is a neighborhood of $x$, and
\item[(ii)] every neighborhood of $x$ contains some set in $\mathcal{B}$.
\end{enumerate}
A set $B$ is called a basic neighborhood of $x$ if it belongs to some fundamental system of neighborhoods of $x$.
