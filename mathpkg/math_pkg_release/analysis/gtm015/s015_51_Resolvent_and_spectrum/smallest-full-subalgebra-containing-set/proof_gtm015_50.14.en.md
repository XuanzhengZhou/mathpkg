# Proof of Smallest Full Subalgebra Containing a Set

**Proof.** (i) The intersection of all full subalgebras of $A$ containing $T$ is a full subalgebra (the intersection of full subrings is a full subring, and it is clearly a linear subspace). This intersection is the smallest full subalgebra containing $T$, call it $B$. Since $T''$ is a full subring containing $T$ (50.13) and is a linear subspace, $T''$ is a full subalgebra containing $T$, hence $B \subset T''$.

(ii) If $A$ is a Banach algebra, then $\bar{B}$ is a closed full subalgebra (50.12). If $C$ is any closed full subalgebra containing $T$, then $B \subset C$, so $\bar{B} \subset \bar{C} = C$. Thus $\bar{B}$ is the smallest such.
