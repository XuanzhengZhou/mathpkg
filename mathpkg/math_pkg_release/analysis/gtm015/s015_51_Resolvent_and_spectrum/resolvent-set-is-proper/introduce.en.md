The resolvent set cannot be the entire complex plane. The proof uses Liouville's theorem: if $\rho(x) = \mathbb{C}$, the entire function $F(\lambda) = f((\lambda 1 - x)^{-1})$ would be bounded and constant for every continuous linear form $f$, leading to a contradiction via the Hahn-Banach theorem. Consequently, the spectrum $\sigma(x)$ is nonempty.
