\text{Every normed space may be regarded as a dense linear subspace of a Banach space.}

\text{More precisely, if } E \text{ is a normed space, let } x \mapsto x'' \text{ be the canonical mapping of } E \text{ into } E''. \text{ The image } E_0 \text{ of } E \text{ is a linear subspace of } E''. \text{ Let } F \text{ be the closure of } E_0 \text{ in } E''; \text{ then } F \text{ is a Banach space and } E \cong E_0 \text{ is a dense linear subspace of } F.
