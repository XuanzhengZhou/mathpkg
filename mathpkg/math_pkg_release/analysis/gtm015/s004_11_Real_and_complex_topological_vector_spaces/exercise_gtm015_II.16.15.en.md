# Exercise 16.15

Let $T$ be a nonempty set. A function $x: T \to \mathbb{K}$ is said to be bounded if there exists a constant $M$ such that $|x(t)| \leq M$ for all $t \in T$. Denote by $\mathcal{B}_{\mathbb{K}}(T)$ the set of all bounded functions $x$. With the pointwise linear operations

$$(x + y)(t) = x(t) + y(t), \quad (\lambda x)(t) = \lambda x(t),$$

$\mathcal{B}_{\mathbb{K}}(T)$ is a vector space over $\mathbb{K}$. Define

$$\|x\|_{\infty} = \sup \{|x(t)| : t \in T\}$$

for $x \in \mathcal{B}_{\mathbb{K}}(T)$. Then $(\mathcal{B}_{\mathbb{K}}(T), \|x\|_{\infty})$ is a Banach space.
