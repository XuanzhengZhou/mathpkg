# Exercise 9.2

Let $Y$ denote the solid cylinder $Y = \{(z, t) \in \mathbb{C} \times \mathbb{R} : |z| \leq 1, 0 \leq t \leq 1\}$. Let

$$A(Y) = \{f \in C(Y) : \text{for each } t, f(z, t) \text{ is analytic in } |z| < 1\}.$$

Show that $\check{S}(A(Y)) = \{(z, t) : |z| = 1, 0 \leq t \leq 1\}$.
