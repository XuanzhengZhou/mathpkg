% Algebra Of Locally Holomorphic Functions
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

$\mathcal{H}(X) = \{f \in C(X) \mid \exists \text{ neighborhood } U \text{ of } X \text{ and } \exists F \in H(U) \text{ with } F = f \text{ on } X\}$.
