% Polynomially Convex Set Definition
% From GTM035, Section s007
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

A compact set $X \subset \mathbb{C}^n$ is polynomially convex if for each $z \notin X$ there exists a polynomial $P$ with $|P(z)| > \max_X |P|$.
