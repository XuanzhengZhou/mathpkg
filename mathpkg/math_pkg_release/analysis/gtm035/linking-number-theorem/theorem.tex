% Linking Number Theorem
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

For $K$ polynomially convex in $\mathbb{C}^n$, the linking number of $\hat{X} \setminus X$ with $X$ gives topological obstructions to the structure of the polynomial hull.
