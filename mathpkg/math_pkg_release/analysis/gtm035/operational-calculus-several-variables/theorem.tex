% Operational Calculus Several Variables
% From GTM035, Section s009
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $\mathfrak{A}$ be a Banach algebra and $x_1, \ldots, x_n \in \mathfrak{A}$. Let $\Omega$ be an open set containing $\sigma(x_1, \ldots, x_n)$. Then there exists a map $\tau: H(\Omega) \to \mathfrak{A}$ such that for all $F \in H(\Omega)$,
$$\widehat{\tau(F)} = F(\hat{x}_1, \ldots, \hat{x}_n) \text{ on } \mathcal{M}.$$
