Suppose $(S, d)$ is a metric space. The \textbf{ball} of radius $r > 0$ about $x \in S$ is
$$B_r(x) = \{y \mid y \in S, \, d(x, y) < r\}.$$

A subset $A \subset S$ is a \textbf{neighborhood} of $x \in S$ if $A$ contains some ball $B_r(x)$ centered at $x$. A subset $A$ is \textbf{open} if it is a neighborhood of each of its points. A point $x$ is a \textbf{limit point} of $A$ if every ball $B_r(x)$ contains points of $A$ other than $x$. A subset $A$ is \textbf{closed} if it contains all its limit points.

A subset $A \subset S$ is open if and only if its complement is closed.
