A function $f: A \to B$ is said to be \textbf{1-1} (one-to-one) or \textbf{injective} if it assigns distinct elements of $B$ to distinct elements of $A$:
$$x, y \in A \text{ and } x \neq y \implies f(x) \neq f(y).$$

A function $f: A \to B$ is said to be \textbf{onto} or \textbf{surjective} if for each element $y \in B$, there is some $x \in A$ such that $f(x) = y$.

A function $f: A \to B$ which is both 1-1 and onto is said to be \textbf{bijective}.
