# Exercise 2

Let \(f: [0, 1] \to \mathbb{R}\) be defined by: \(f(0) = 0\), \(f(x) = \sin(1/x)\) if \(x \neq 0\). Show that \(f\) is integrable.

*Hint:* The function is bounded and has a single discontinuity at \(x = 0\). Use Proposition 2.5 and the fact that a function continuous on a closed interval is integrable.
