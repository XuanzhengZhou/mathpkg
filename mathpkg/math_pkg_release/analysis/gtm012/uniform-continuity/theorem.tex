A function $f: S \to S'$ between metric spaces is \textit{uniformly continuous} if for each $\varepsilon > 0$ there exists $\delta > 0$ such that:
\[
d'(f(x), f(y)) < \varepsilon \quad \text{whenever } d(x, y) < \delta,
\]
where $\delta$ depends only on $\varepsilon$, not on the particular points $x, y$.
