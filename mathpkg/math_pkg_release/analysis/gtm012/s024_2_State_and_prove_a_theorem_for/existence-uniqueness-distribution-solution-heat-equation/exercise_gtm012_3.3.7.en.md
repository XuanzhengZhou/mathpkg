# Exercise 7

The *backwards heat equation* is the equation (1) considered for $t \leq 0$, with initial (or "final") condition (2). Is it reasonable to expect that solutions for this problem will exist? Specifically, given $G \in \mathcal{P}'$, will there always be a family of distributions $(F_t)_{t < 0} \subset \mathcal{P}'$ such that

$$\frac{d}{dt} F_t \bigg|_{t=s} = D^2 F_s, \quad \text{all } s < 0,$$

$$F_t \rightarrow G (\mathcal{P}') \quad \text{as } t \rightarrow 0?$$
