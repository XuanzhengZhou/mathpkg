A basis is a minimal spanning set or, equivalently, a maximal linearly independent set. Every vector in the space has a unique representation as a linear combination of basis vectors, with the coefficients providing coordinates. In a finite-dimensional vector space, all bases have the same number of elements, which defines the dimension. A basis establishes an isomorphism between the vector space and $\mathbb{R}^n$ (or $\mathbb{C}^n$) via the coordinate map.
