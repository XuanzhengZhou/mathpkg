Suppose $(S, d)$ and $(S', d')$ are metric spaces. A function $f: S \to S'$ is said to be continuous at the point $x \in S$ if for each $\varepsilon > 0$ there is a $\delta > 0$ such that

$$d'(f(x), f(y)) < \varepsilon \quad \text{if } d(x, y) < \delta.$$

In particular, if $S$ and $S'$ are subsets of $\mathbb{R}$ or of $\mathbb{C}$ with the usual metrics, the condition is

$$|f(y) - f(x)| < \varepsilon \quad \text{if } |y - x| < \delta.$$

An equivalent sequential characterization: $f$ is continuous at $x$ if and only if $f(x_n) \to f(x)$ whenever $(x_n)_{n=1}^{\infty}$ is a sequence in $S$ converging to $x$.
