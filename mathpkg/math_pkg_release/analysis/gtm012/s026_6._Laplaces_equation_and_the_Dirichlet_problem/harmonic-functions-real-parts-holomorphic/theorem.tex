\textbf{Theorem 6.2.} Let $u$ be a harmonic function defined on an open set containing $(x_0, y_0)$. Then there exist $\varepsilon > 0$ and a holomorphic function $f$ defined for $|z - z_0| < \varepsilon$, where $z_0 = x_0 + iy_0$, such that

$$u(x, y) = \operatorname{Re} f(x + iy)$$

for all $(x, y)$ with $|x + iy - z_0| < \varepsilon$. Moreover, $f$ can be expressed as a convergent power series

$$f(z) = \sum_{n=0}^{\infty} a_n (z - z_0)^n,$$

where the coefficients are determined by the Fourier coefficients of the restriction of $u$ to a small circle centered at $(x_0, y_0)$.
