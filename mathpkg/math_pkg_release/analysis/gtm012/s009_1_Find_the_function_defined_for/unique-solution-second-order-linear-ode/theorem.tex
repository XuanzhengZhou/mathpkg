For any $b, c, d_0, d_1 \in \mathbb{C}$ and any continuous function $h: \mathbb{R} \to \mathbb{C}$, there is a unique twice continuously differentiable function $f: \mathbb{R} \to \mathbb{C}$ such that
$$f(0) = d_0, \quad f'(0) = d_1, \quad f''(x) + bf'(x) + cf(x) = h(x), \quad x \in \mathbb{R}.$$
Let $a_1, a_2 \in \mathbb{C}$ be the roots of $z^2 + bz + c = (z - a_1)(z - a_2)$. Then the unique solution is given by
$$f(x) = d_0E(a_2x) + \int_0^x E(a_2(x-t))g(t)\,dt,$$
where
$$g(x) = (d_1 - a_2d_0)E(a_1x) + \int_0^x E(a_1(x-t))h(t)\,dt.$$
