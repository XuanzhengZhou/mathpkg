# Exercise 2

Let $f, g$ be the functions such that

$$f(0) = 1, \quad f'(0) = 0, \quad f'' + b f' + c f = 0,$$

$$g(0) = 0, \quad g'(0) = 1, \quad g'' + b g' + c g = 0.$$

Show that for any constants $d_1, d_2 \in \mathbb{C}$ the function $h = d_1 f + d_2 g$ is a solution of

$$h'' + b h' + c h = 0. \tag{*}$$

Show that conversely if $h$ is a solution of $(*)$ then there are unique constants $d_1, d_2 \in \mathbb{C}$ such that $h = d_1 f + d_2 g$.
