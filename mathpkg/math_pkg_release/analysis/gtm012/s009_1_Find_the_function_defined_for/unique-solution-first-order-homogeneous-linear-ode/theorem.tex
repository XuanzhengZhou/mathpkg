For each $a, c \in \mathbb{C}$ there is a unique continuously differentiable function $f: \mathbb{R} \to \mathbb{C}$ such that
$$f(0) = c, \quad f'(x) = af(x), \quad x \in \mathbb{R}.$$
This function is
$$f(x) = cE(ax) = c \sum_{n=0}^{\infty} \frac{a^n x^n}{n!},$$
where $E(z) = \sum_{n=0}^{\infty} z^n/n!$.
