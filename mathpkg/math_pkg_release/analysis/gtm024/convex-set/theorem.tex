The \emph{line segment} joining $x$ and $y$ is $[x, y] = \{\alpha x + (1-\alpha)y : 0 \leqslant \alpha \leqslant 1\}$. A set $A \subset X$ is \emph{star-shaped} with respect to $p \in A$ if $[p, x] \subset A$ for all $x \in A$. $A$ is \emph{convex} if it is star-shaped with respect to each of its elements.
