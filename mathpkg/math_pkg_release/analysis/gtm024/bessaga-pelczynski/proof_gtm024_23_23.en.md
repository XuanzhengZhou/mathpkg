If $J_V$ is an isomorphism, $U(J_V(X)) = U(V^*) \cap J_V(X)$ is weak*-closed in $V^*$. So $U(X)$ is $\sigma(X, V)$-compact. Conversely, if $U(X)$ is $\sigma(X, V)$-relatively compact for total $V$, then $J_V(X)$ is weak*-dense and weak*-closed in $V^*$, so $J_V$ is surjective.
