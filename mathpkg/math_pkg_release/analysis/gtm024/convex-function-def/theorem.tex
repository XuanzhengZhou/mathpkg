Let $A$ be a convex subset of a real linear space $X$. A function $f: A \to \mathbb{R}$ is convex if its epigraph $\operatorname{epi}(f) = \{(x,t) \in A \times \mathbb{R} : t \geqslant f(x)\}$ is a convex subset of $X \times \mathbb{R}$. Equivalently, $f(tx + (1-t)y) \leqslant tf(x) + (1-t)f(y)$ for all $x, y \in A$, $0 < t < 1$.
