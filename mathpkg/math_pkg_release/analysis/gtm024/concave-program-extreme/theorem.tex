Let $(A, f)$ be a finite-dimensional concave program where $A$ is line-free and closed, and $f$ is lower semicontinuous on $A$. If a solution exists in $A$, then there is a solution in $\operatorname{ext}(A)$. Conversely, any solution to $(\operatorname{ext}(A), f)$ is also a solution to $(A, f)$, provided $f$ is bounded below on $A$.
