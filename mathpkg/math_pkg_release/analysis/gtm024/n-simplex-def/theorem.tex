A finite set $\{x_0, x_1, \ldots, x_n\} \subset X$ is affinely independent if $\{x_1 - x_0, \ldots, x_n - x_0\}$ is linearly independent. The convex hull of such a set is an $n$-simplex with vertices $x_0, \ldots, x_n$. Each point in the $n$-simplex can be uniquely expressed as a convex combination of the vertices; the coefficients are the barycentric coordinates.
