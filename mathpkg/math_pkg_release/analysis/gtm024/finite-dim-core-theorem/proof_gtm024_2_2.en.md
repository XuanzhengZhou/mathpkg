If $\operatorname{aff}(A) = X$, then $A$ contains an $n$-simplex whose intrinsic core is contained in $\operatorname{cor}(A)$, so $\operatorname{cor}(A) \neq \emptyset$. Conversely, if $p \in \operatorname{cor}(A)$ and $x \in X$, there exists $\varepsilon > 0$ with $[p, p + \varepsilon(x-p)] \subset A$. With $\lambda = (\varepsilon-1)/\varepsilon$, we have $x = \lambda p + (1-\lambda)(p + \varepsilon(x-p)) \in \operatorname{aff}(A)$.
