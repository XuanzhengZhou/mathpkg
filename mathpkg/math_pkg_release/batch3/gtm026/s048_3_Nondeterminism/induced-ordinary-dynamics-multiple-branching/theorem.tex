Each multiple branching dynamics $\delta: Q \times X \longrightarrow QT$ induces the ordinary dynamics $\delta^\bullet: QT \times X \longrightarrow QT$ defined by:

$$\delta^\bullet = QT \times X \xrightarrow{Q\lambda} (Q \times X)T \xrightarrow{\delta^\#} QT$$

Here:
\begin{itemize}
\item $Q\lambda$ is the canonical map $(A, x) \mapsto A \times \{x\}$,
\item $\delta^\#$ is the unique union-preserving extension of $\delta : Q \times X \to QT$ to a $\mathbf{T}$-homomorphism $\delta^\# : (Q \times X)T \to QT$.
\end{itemize}

\text{Thus $\delta^\bullet$ is an ordinary (deterministic) $X$-dynamics on the state object $QT$.}
