A fuzzy theory in $\mathbf{Set}$ where for each set $Q$, $QT$ is the set of nonempty subsets of $Q$. The algebras are sets with base point, corresponding to the "subset" measure of certainty.

Fuzzy morphisms are maps $Q \to Q'T$ assigning to each element of $Q$ a nonempty subset of $Q'$.

\text{This is a subtheory of the theory described in 1.3.5.}
