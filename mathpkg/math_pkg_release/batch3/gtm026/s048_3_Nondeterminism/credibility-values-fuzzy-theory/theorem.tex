Let $M$ be a monoid "of credibility values" and let $T_M$ be the associated algebraic theory in $\mathbf{Set}$ as in 1.3.

Given a fuzzy morphism $\alpha : A \longrightarrow B \times M$, if $a\alpha = (b, m)$ then "$a\alpha = b$ with credibility $m$."

The monoid multiplication determines how credibilities combine.

\text{This example is apt for $M$ the unit interval under multiplication.}
