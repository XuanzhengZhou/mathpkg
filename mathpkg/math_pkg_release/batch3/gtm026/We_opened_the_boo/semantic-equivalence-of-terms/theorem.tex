For each set $A$, define an equivalence relation $E_A$ on $A\Omega$ by
$$
E_A = \{(p, q) \mid \text{for all } (\Omega, E)\text{-algebras } (X, \delta) \text{ and all functions } f: A \to X,\ pf^{\#} = qf^{\#}\}.
$$
The quotient set $A\Omega/E_A$ is denoted by $AT$ and is called the \textit{free $(\Omega, E)$-algebra generated by $A$}. The canonical projection is $A\rho: A\Omega \twoheadrightarrow AT$.
