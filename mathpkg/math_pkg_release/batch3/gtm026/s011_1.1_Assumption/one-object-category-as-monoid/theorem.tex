If a category $\mathcal{K}$ has only one object, then the set $M$ of all morphisms of $\mathcal{K}$ consists solely of endomorphisms of the unique object (where an endomorphism is a morphism whose domain and codomain coincide). Composition is an everywhere-defined associative operation on $M$ and the identity map of the unique object is a two-sided unit for this operation. Thus $M$ is a monoid.
