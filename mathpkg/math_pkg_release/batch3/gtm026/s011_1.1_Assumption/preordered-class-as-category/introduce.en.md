Every preordered class can be viewed as a category whose objects are the elements of the class and where a morphism $A \to B$ exists exactly when $A \leqslant B$. Transitivity of the relation corresponds to composition, and reflexivity provides identity morphisms. This identification allows category-theoretic language to be applied to order-theoretic situations.
