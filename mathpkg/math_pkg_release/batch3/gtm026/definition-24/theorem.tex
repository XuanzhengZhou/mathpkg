1. $E = E(X, \xi)$ is a subgroup of bijections of $X$.
2. Every element of $E$ is injective.
3. $E$ is a minimal $T$-algebra.
4. If $(Y, \theta)$ is in the Birkhoff subcategory, $\text{SMP}(X, \xi)$, generated by $(X, \xi)$ then the subalgebra generated by any element of $Y$ is minimal.
5. The subalgebra generated by any element of $(X, \xi) \times (X, \xi)$ is minimal.
