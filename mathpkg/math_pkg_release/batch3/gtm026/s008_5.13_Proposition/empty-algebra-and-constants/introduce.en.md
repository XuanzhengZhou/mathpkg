Proposition 5.13 relates the existence of an empty $\mathbf{T}$-algebra to whether the theory $\mathbf{T}$ admits zero constants. If $\varnothing T = \varnothing$, the empty set carries a $\mathbf{T}$-algebra structure. Conversely, if $\varnothing T \neq \varnothing$ (so the empty set is not a $\mathbf{T}$-algebra), then every operation that behaves as a constant in all algebras must in fact be a true constant, i.e., have arity zero.
