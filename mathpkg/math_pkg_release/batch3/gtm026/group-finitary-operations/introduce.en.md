A group is a fundamental algebraic structure defined by a set equipped with a binary multiplication, a unary inversion, and a distinguished identity element. These operations must satisfy three equational axioms: associativity of multiplication, the identity property, and the inverse property. The definition uses parenthesis-free Polish notation (e.g., $xymzm$ instead of $((x,y)m,z)m$) which allows the equational axioms to be written without any auxiliary symbols such as parentheses or commas. This finitary equational presentation makes groups the prototypical example of an equationally-definable class in universal algebra.
