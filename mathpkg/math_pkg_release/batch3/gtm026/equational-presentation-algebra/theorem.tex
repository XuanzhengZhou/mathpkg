Let $\Omega$ be an operator domain. An $\Omega$-equation is a pair $\{e_1, e_2\}$ of $\Omega$-terms in a set $V$ of variables. Define the interpretation map $r^{\#}: V\Omega \longrightarrow X$ for each $V$-tuple $r: V \longrightarrow X$ in an $\Omega$-algebra $(X, \delta)$ by algebraic general recursion:
$$v r^{\#} = v r, \quad (p_1 \cdots p_n \omega) r^{\#} = (p_1 r^{\#}, \ldots, p_n r^{\#}) \delta_{\omega}.$$

The algebra $(X, \delta)$ satisfies $\{e_1, e_2\}$ if $e_1 r^{\#} = e_2 r^{\#}$ for all $r: V \longrightarrow X$.

An equational presentation is a pair $(\Omega, E)$ where $\Omega$ is an operator domain and $E$ is a set of $\Omega$-equations. An $(\Omega, E)$-algebra is an $\Omega$-algebra that satisfies every equation in $E$.
