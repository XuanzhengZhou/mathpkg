For an arbitrary set $X$, regard the elements of $2^X$ as subsets of $X$ via the characteristic function bijection $A \leftrightarrow \chi_A$. A typical basic open set in the Tychonoff topology for $X$ copies of the discrete space $2$ is
$$
B_{(G, H)} = \{A \subset X : A \cap G = H\}
$$
where $G$ is a finite subset of $X$ and $H \subset G$.
