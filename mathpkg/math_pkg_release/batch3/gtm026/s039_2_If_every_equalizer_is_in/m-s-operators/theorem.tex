Let $\mathcal{A}$ be a class of objects. Define:
\begin{itemize}
\item $\mathsf{M}(\mathcal{A})$ = the class of all objects admitting an $\mathsf{M}$-morphism into some element of $\mathcal{A}$,
\item $\mathsf{S}(\mathcal{A})$ = the class of all objects admitting a split epimorphism from some element of $\mathcal{A}$,
\item $\mathsf{P}(\mathcal{A})$ = the class of all objects isomorphic to a product of elements of $\mathcal{A}$.
\end{itemize}
Then $\mathsf{MP}(\mathcal{A})$ is the quasivariety generated by $\mathcal{A}$ (i.e., $\mathcal{A} \subset \mathsf{MP}(\mathcal{A})$, $\mathsf{MP}(\mathcal{A})$ is a quasivariety, and if $\mathcal{A} \subset \mathcal{B}$ and $\mathcal{B}$ is a quasivariety then $\mathsf{MP}(\mathcal{A}) \subset \mathcal{B}$). Similarly, $\mathsf{SMP}(\mathcal{A})$ is the smallest quasivariety $\mathcal{B}$ containing $\mathcal{A}$ and closed under $\mathsf{S}$ in the sense that $\mathsf{S}(\mathcal{B}) \subset \mathcal{B}$.
