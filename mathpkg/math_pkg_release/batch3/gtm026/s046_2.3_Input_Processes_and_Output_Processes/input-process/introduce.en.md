A process $X$ is called an input process when the forgetful functor $U: \operatorname{Dyn}(X) \to \mathcal{K}$ possesses a left adjoint. This adjointness condition guarantees the existence of free $X$-dynamics, which provides the algebraic mechanism needed to define the reachability map (the unique dynamorphic extension of the initial state). Intuitively, an input process allows one to freely generate the trajectory of an automaton from an initial configuration.
