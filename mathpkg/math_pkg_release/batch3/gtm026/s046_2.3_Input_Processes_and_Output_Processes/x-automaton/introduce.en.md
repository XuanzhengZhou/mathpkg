An $X$-automaton extends an $X$-dynamics $(Q, \delta)$ with additional input-output structure: an initial state object $I$ with initial state morphism $\tau: I \to Q$, and an output object $Y$ with output morphism $\beta: Q \to Y$. This generalizes the classical automaton where $I$ is a one-point set (the unique initial state), $Y$ is the output alphabet, $\tau$ picks the start state, and $\beta$ labels each state with an output symbol. The dynamics morphism $\delta: QX \to Q$ encodes the transition function into the categorical framework.
