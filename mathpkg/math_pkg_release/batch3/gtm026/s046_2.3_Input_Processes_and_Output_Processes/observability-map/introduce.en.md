The observability map $\sigma: (Q, \delta) \to (YX_{\sharp}, YL)$ is the unique $X$-dynamorphism coextending the output morphism $\beta: Q \to Y$ into the cofree $X$-dynamics over $Y$. It encodes the observable behavior of each state: given a state $q \in Q$, the image $\sigma(q)$ is a function that assigns to each possible input word the output the automaton would produce when started from state $q$. Two states are indistinguishable (not observable) precisely when they have the same image under $\sigma$.
