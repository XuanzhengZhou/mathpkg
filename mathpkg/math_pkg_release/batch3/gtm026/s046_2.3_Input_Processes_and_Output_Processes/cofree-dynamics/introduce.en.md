When $X$ is an output process (the forgetful functor $U$ has a right adjoint), the cofree $X$-dynamics over any object $A$ is denoted $(AX_{\sharp}, AL; A\Lambda)$. This triple consists of the object $AX_{\sharp}$, a dynamics morphism $AL: AX_{\sharp} X \to AX_{\sharp}$, and the counit $A\Lambda: AX_{\sharp} \to A$ of the adjunction. In the classical Set case $X = - \times X_0$, we have $AX_{\sharp} = A^{(X_0^*)}$, the set of all functions from the free monoid $X_0^*$ to $A$, equipped with the left-translation dynamics and evaluation-at-empty-string counit.
