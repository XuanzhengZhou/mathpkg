The reachability map $r: (IX^{\sharp}, I\mu_0) \to (Q, \delta)$ is the unique $X$-dynamorphism extending the initial state $\tau: I \to Q$ from the free $X$-dynamics over the initial state object $I$. It encodes how the states of an automaton can be reached from the initial configuration by applying sequences of inputs. In the classical case $X = - \times X_0$, this map sends a pair $(i, w)$ consisting of an initial parameter $i \in I$ and an input word $w \in X_0^*$ to the state reached after processing $w$ starting from $\tau(i)$.
