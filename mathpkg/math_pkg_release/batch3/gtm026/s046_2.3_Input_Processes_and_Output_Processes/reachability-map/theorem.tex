Assume $X$ is an input process and $I$ is the initial state object of an $X$-automaton. The \textbf{reachability map} is the unique dynamorphic extension of the initial state $\tau: I \to Q$, given as the $X$-dynamorphism
$$r: (IX^{\sharp}, I\mu_0) \longrightarrow (Q, \delta)$$
obtained via the universal property of the free dynamics $(IX^{\sharp}, I\mu_0; I\eta)$.
