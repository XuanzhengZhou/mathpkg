Assume $X$ is an input process. Let $r: (IX^{\sharp}, I\mu_0) \to (Q, \delta)$ be the reachability map and $\beta: Q \to Y$ the output morphism of an $X$-automaton. The \textbf{response map} is the $\mathcal{K}$-morphism
$$r \circ \beta: IX^{\sharp} \longrightarrow Y.$$
It sends an input trajectory (an element of $IX^{\sharp}$) to the corresponding output in $Y$.
