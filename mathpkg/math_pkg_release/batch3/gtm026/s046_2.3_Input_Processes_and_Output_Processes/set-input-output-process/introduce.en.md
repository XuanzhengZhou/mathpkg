The endofunctor $X = - \times X_0$ on $\mathbf{Set}$ provides the prototypical example of a process that is simultaneously an input process and an output process. The free dynamics over $A$ is $A \times X_0^*$ (pairs of an element of $A$ and an input word), with dynamics appending a new input symbol to the word. The cofree dynamics is $A^{(X_0^*)}$ (functions from input words to $A$), with dynamics given by left translation on words. This example shows that the classical theory of sequential automata in $\mathbf{Set}$ is a special case of the categorical framework.
