3.33 “Type Theory”. Let $(n, k)$ be a pair of non-negative integers. An $(n, k)$-structure is a pair $(K, s)$ where $K$ is a set and $s$ is an element of $K^n P^k$ where $K^n$ is the $n$-fold cartesian power and $P^k$ is the $k$th iterate of the power set operator, $KP = 2^K$. For example, a topology of open subsets of $K$ is a $(1, 2)$-structure, a uniform structure of entourages on $K$ is a $(2, 2)$-structure, a ternary operation $K^3 \longrightarrow K$ is (via its graph) a $(4, 1)$-structure, a partially ordered set is a $(2, 1)$-structure, and a set with base point is a $(1, 0)$-structure.
