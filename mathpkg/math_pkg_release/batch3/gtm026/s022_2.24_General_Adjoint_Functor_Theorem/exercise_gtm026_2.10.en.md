Let $U: \mathcal{A} \rightarrow \mathbf{Set}$ be a set-valued functor. $U$ is representable if there exists an $\mathcal{A}$-object $A$ such that $U$ is naturally equivalent to $\mathcal{A}(A, -)$; show that this representing object $A$ is unique up to isomorphism. Prove that $U$ is representable if and only if there exists a free object over $1$ with respect to $U$. More generally, for any functor $U: \mathcal{A} \rightarrow \mathcal{B}$ and object $B$ of $\mathcal{B}$, show that there exists a free object over $B$ with respect to $U$ if and only if $\mathcal{B}(B, (-)U): \mathcal{A} \rightarrow \mathbf{Set}$ is representable.
