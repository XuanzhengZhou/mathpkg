A $(P, \Omega)$-algebra assigns to each sort $p \in P$ a carrier set $X(p)$, and to each operation symbol $\omega$ of arity $(p_1, \ldots, p_n) \to p_{n+1}$ an actual function between the corresponding carrier sets. This is the many-sorted analogue of an algebra for a single-sorted algebraic theory, where the sorts are indexed by the set $P$ rather than being a single underlying set.
