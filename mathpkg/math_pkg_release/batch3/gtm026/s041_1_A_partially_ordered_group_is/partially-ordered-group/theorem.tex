A \textit{partially ordered group} is a triple $(X, s, \leqslant)$ where $(X, s)$ is a group, $(X, \leqslant)$ is a partially ordered set, and the following laws hold:
\begin{itemize}
\item if $x \leqslant y$ and $a \leqslant b$ then $xa \leqslant yb$;
\item if $x \leqslant y$ then $y^{-1} \leqslant x^{-1}$.
\end{itemize}
