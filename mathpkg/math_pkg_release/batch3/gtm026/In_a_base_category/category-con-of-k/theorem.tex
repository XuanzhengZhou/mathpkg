Let $\mathcal{K}$ be a fixed base category. The category $\text{Con}(\mathcal{K})$ is defined as follows:
\begin{itemize}
  \item \textbf{Objects:} Concrete categories $(\mathcal{A}, U)$ over $\mathcal{K}$ (where $\mathcal{A}$ is a category and $U: \mathcal{A} \to \mathcal{K}$ is faithful).
  \item \textbf{Morphisms:} Homomorphisms $H: (\mathcal{A}, U) \to (\mathcal{B}, V)$ of concrete categories over $\mathcal{K}$.
  \item \textbf{Composition:} Functor composition.
  \item \textbf{Identities:} Identity functors on the underlying categories.
\end{itemize}
Note that $\text{Con}(\mathcal{K})$ need not be locally small even if $\mathcal{K}$ is.
