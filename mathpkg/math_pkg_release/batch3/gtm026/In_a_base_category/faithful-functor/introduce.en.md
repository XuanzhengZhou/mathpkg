A functor $U: \mathcal{A} \to \mathcal{B}$ is faithful if it is injective on each hom-set; that is, distinct $\mathcal{A}$-morphisms between the same two objects are mapped to distinct $\mathcal{B}$-morphisms. Faithfulness is a fundamental property in category theory; it ensures that the functor does not collapse distinct morphisms into the same morphism. Concrete categories (over a base category $\mathcal{K}$) are defined precisely as pairs $(\mathcal{A}, U)$ where $U: \mathcal{A} \to \mathcal{K}$ is faithful.
