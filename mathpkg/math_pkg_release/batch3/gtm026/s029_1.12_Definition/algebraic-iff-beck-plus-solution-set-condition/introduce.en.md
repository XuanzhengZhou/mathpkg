For a functor $U: \mathcal{A} \to \mathcal{K}$ where $\mathcal{K}$ is locally small with small limits, $U$ is algebraic precisely when it is a Beck functor satisfying the solution set condition. If $\mathcal{A}$ additionally possesses a cogenerator and either $\mathcal{A}$ or $\mathcal{K}$ is well-powered, then the solution set condition is automatic and $U$ is algebraic exactly when it is Beck. This result synthesizes the adjoint functor theorems (2.2.24, 2.2.29) with the characterizations 1.9, 1.16, and 1.19.
