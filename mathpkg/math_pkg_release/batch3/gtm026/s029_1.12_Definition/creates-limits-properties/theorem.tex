If $U: \mathcal{A} \longrightarrow \mathcal{K}$ creates limits then the following statements are true:

\begin{enumerate}
\item $(\mathcal{A}, U) \in \text{Struct}(\mathcal{K})$ (2.3.3) and is discretely ordered (2.3.9);
\item If $\mathcal{K}$ is locally small then $\mathcal{A}$ is locally small;
\item If $\mathcal{K}$ is well powered (2.2.29(2)) and has kernel pairs then $\mathcal{A}$ is well powered.
\end{enumerate}
