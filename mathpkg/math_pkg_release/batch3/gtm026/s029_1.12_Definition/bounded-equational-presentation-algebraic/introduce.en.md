This theorem proves that any bounded equational presentation (one where the arities of operations are bounded by some cardinal) yields an algebraic forgetful functor. Unlike Theorem 1.26, tractability is not required. The key to the proof is constructing a solution set using cardinal arithmetic: the subalgebra generated by a set $S$ has cardinality bounded by a cardinal $\alpha$ depending only on $\Omega$ and $S$. The proof uses transfinite induction to build the subalgebra and cardinal arithmetic to bound its size.
