Let $U: \mathcal{A} \longrightarrow \mathbf{Set}$ be a set-valued functor.
\begin{enumerate}
\item $U$ is algebraic if and only if $U$ creates limits, $U$ creates quotients of congruences, and $U$ satisfies the solution set condition at $K$ for every set $K$.
\item If $\mathcal{A}$ has a cogenerator, $U$ is algebraic if and only if $U$ creates limits and $U$ creates quotients of congruences.
\end{enumerate}
