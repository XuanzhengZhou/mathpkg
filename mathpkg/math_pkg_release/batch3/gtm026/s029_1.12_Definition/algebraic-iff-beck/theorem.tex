Let $U: \mathcal{A} \longrightarrow \mathcal{K}$ be a functor and assume that $\mathcal{K}$ is locally small and has small limits.
\begin{enumerate}
\item $U$ is algebraic if and only if $U$ is Beck and $U$ satisfies the solution set condition at $K$ for every object $K$ of $\mathcal{K}$.
\item If $\mathcal{A}$ has a cogenerator and if either $\mathcal{A}$ or $\mathcal{K}$ is well powered, $U$ is algebraic if and only if $U$ is Beck.
\end{enumerate}
