For each set $A$, define the insertion-of-the-variables map $\eta_A: A \longrightarrow AT$ by $\langle a, \eta_A \rangle = [a]$ (the equivalence class of the variable $a$). Then for every $(\Omega, E)$-algebra $(X, \delta)$ and every function $f: A \longrightarrow X$, there exists a unique $\Omega$-homomorphism $\hat{f}: AT \longrightarrow (X, \delta)$ such that the following diagram commutes:
\[
\begin{CD}
A @>{\eta_A}>> AT \\
@V{f}VV @VV{\hat{f}}V \\
X @= X
\end{CD}
\]
That is, $\eta_A \cdot \hat{f} = f$.
