For each set $A$, define an equivalence relation $E_A$ on $A\Omega$ by
$$E_A = \{(p, q): \text{for all } (\Omega, E)\text{-algebras } (X, \delta) \text{ and all functions } f: A \longrightarrow X,\; p f^\# = q f^\#\}$$
where $f^\# : A\Omega \longrightarrow X$ is the interpretation map extending $f$. The quotient set $A\Omega / E_A$ is denoted by $AT$.
