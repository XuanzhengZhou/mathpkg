Let $A$ be an arbitrary category and let $U: A \longrightarrow \mathbf{Set}$ be a functor. For each set $X$, define $U^X: A \longrightarrow \mathbf{Set}$ by $U^X(-) = U(-)^X$.

If $n$ is a cardinal, $U$ is \textbf{tractable at $n$} if the class of all natural transformations from $U^n$ to $U$ is a small set.
