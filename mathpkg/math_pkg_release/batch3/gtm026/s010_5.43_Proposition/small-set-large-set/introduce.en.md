To resolve paradoxes like Burali-Forti and Russell, set theory introduces a "one universe" framework (see Mac Lane '71, I.6). A universe $U$ models the set of all sets. Small sets are elements of $U$; large sets are subcollections of $U$ too big to be elements of $U$. This distinction ensures that set-theoretic constructions behave well: for instance, the class of $n$-ary operations of any equational presentation may be a large set unless bounded.
