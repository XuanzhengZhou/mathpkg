An ordinal is a set that is hereditarily $\in$-transitive: both the set itself and all its elements are $\in$-transitive. Ordinals form the backbone of von Neumann's set-theoretic construction of the natural numbers and transfinite numbers, where each ordinal is precisely the set of all smaller ordinals.
