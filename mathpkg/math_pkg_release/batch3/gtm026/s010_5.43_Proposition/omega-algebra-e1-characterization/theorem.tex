An $\Omega$-algebra $(X, \delta)$ satisfies $E_1$ if and only if there exists a (necessarily unique) function $\xi: XT \longrightarrow X$ such that for all $\alpha \in \Omega_n$ and all $f: n \longrightarrow X$, the following diagram commutes in the appropriate sense:

$$\langle \alpha, fT \rangle \xi = \langle f, \delta_\alpha \rangle$$
