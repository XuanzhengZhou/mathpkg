If $x$ and $y$ are ordinals, then exactly one of $x \in y$, $x = y$, $y \in x$ holds (using the axiom of regularity). The class $\mathbf{Ord}$ of all ordinals is linearly ordered by $x \leqslant y$ if $x = y$ or $x \in y$.

If $X$ is a nonempty set of ordinals, then $\bigcap \{x : x \in X\}$ is an ordinal and belongs to $X$; in particular, $X$ has a least element.

For ordinals $x$ and $y$, $x \leqslant y$ holds if and only if $x \subset y$.
