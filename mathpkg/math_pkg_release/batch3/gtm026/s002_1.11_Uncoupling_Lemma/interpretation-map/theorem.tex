Let $\Omega$ be an operator domain, and let $(X, \delta)$ be an $\Omega$-algebra. For each $V$-tuple $r: V \longrightarrow X$ there is an interpretation map $r^\#$ defined by
$$r^\# = r\Omega \circ \delta^@ : V\Omega \longrightarrow X.$$

Equivalently, $r^\#$ can be defined directly by algebraic general recursion:
$$v r^\# = v r \quad \text{for all } v \in V,$$
$$(p_1 \cdots p_n \omega) r^\# = (p_1 r^\#, \ldots, p_n r^\#) \delta_\omega.$$

If $\{e_1, e_2\}$ is an $\Omega$-equation, we say that $(X, \delta)$ satisfies $\{e_1, e_2\}$ if $e_1 r^\# = e_2 r^\#$ for all $r: V \longrightarrow X$.

If $(\Omega, E)$ is an equational presentation, an $(\Omega, E)$-algebra is an $\Omega$-algebra which satisfies $E$, that is, satisfies every equation in $E$. The class of all $(\Omega, E)$-algebras is called an equational class or variety.
