An open set in an abstract $\beta$-algebra $(X, \xi)$ provides a topology $\mathcal{T}$ whose convergence map is exactly $\xi$. This definition is equivalent to the condition $A \xi^{-1} \subset A^*$, but is more convenient for establishing immediate properties. The collection of all open sets forms a topology on $X$, and the ultrafilter convergence structure of the $\beta$-algebra recovers the topological convergence relation $\mathfrak{U} \xrightarrow{\mathcal{T}} \mathfrak{U}\xi$.
