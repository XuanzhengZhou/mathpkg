A word over a set $A$ is a finite nonempty sequence of elements from $A$. This fundamental combinatorial notion serves as the basis for constructing terms in universal algebra. Given an operator domain $\Omega$, an $\Omega$-word extends this idea by allowing symbols from both the variable set $A$ and the operator labels $\Omega_n$. These mixed words are the raw syntactic expressions from which well-formed $\Omega$-terms are later carved out. The disjoint union $A + |\Omega|$ ensures that variable symbols and operator symbols are kept formally distinct even if the underlying sets overlap.
