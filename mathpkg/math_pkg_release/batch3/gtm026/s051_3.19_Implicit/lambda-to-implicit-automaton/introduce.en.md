Given a $\lambda$-automaton $M = (Q, \delta, I, \tau, Y, \theta, \beta)$, the $M^*$ construction of Section 3.9 produces an implicit $\lambda$-automaton by taking the state object $QT$ (the free $\mathbf{T}$-algebra on $Q$), equipping it with the $\lambda$-algebra structure $(QT, \delta^*, Qm)$, and using the same input $I$, initial state map $\tau$, output $Y$, and the transformed output map $\beta^*$. This proposition establishes that the response of the original $\lambda$-automaton coincides with the response of the associated implicit $\lambda$-automaton, providing the first half of the equivalence between the two notions of automaton.
