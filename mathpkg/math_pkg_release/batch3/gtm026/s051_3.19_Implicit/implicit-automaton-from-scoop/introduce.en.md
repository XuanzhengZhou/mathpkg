This proposition provides the converse to 3.20: given an implicit $\lambda$-automaton $\bar{M}$ and any scoop $(Q, i, c)$ of its $\mathbf{T}$-algebra state object $\bar{Q}$, one can construct a $\lambda$-automaton $M$ whose state object is the generating subobject $Q$. The transition map $\delta$ is obtained by composing $\bar{\delta}$ with the scoop morphisms $iX$ and $c$, the initial state map $\tau$ factors $\bar{\tau}$ through $c$, and the output map $\beta$ factors $\bar{\beta}$ through $i$. The resulting $\lambda$-automaton has the same response as the original implicit $\lambda$-automaton, completing the equivalence between the two formalisms.
