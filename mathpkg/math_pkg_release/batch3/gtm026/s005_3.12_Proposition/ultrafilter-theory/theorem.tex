The Ultrafilter Theory in $\mathbf{Set}$ is defined as follows. For a set $X$, let $XT$ be the set of all ultrafilters on $X$. Define:
\begin{itemize}
\item $X\eta: X \longrightarrow XT$ by $\langle x, X\eta \rangle = \mathrm{prin}(x)$, where $\mathrm{prin}(x)$ is the principal ultrafilter on $x$ defined by $\mathrm{prin}(x) = \{A \subset X : x \in A\}$.
\item For $f: X \longrightarrow Y$, $fT: XT \longrightarrow YT$ is defined by $\langle \mathcal{A}, fT \rangle = \{B \subset Y : Bf^{-1} \in \mathcal{A}\}$.
\item $X\mu: XTT \longrightarrow XT$ is defined by $\langle \bar{\mathcal{A}}, X\mu \rangle = \{A \subset X : \{\mathcal{A} \in XT : A \in \mathcal{A}\} \in \bar{\mathcal{A}}\}$.
\end{itemize}
This defines an algebraic theory in monoid form $(T, \eta, \mu)$.
