Proposition 1.4 establishes that every contractible coequalizer is automatically an absolute coequalizer. The proof is straightforward: given the contraction data $(d, d')$, the image under any functor $H$ retains the same commutative relations, so the coequalizer property is preserved. This result provides a convenient sufficient condition for absoluteness and is used in the proof of Beck's characterization theorem.
