Let $\mathcal{C}$ be in Struct($\mathbf{Set}$). Prove that $\mathcal{C}$ is isomorphic to topological spaces and continuous maps in Struct($\mathbf{Set}$) if and only if there exists a ``Sierpinski space'' $S$ with underlying set $\{0, 1\}$ satisfying:
(a) Every family $f_i: X \rightarrow S$ has an optimal lift.
(b) The supremum map Sup: $S^I \rightarrow S$ is admissible for every set $I$.
(c) The infimum map Inf: $S^I \rightarrow S$ is admissible for every finite set $I$.
(d) $S$ is a Sierpinski object in $\mathcal{C}$.
