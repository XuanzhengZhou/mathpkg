Let $\mathcal{C}$ be in $\operatorname{Struct}(\mathbf{Set})$. Prove that $\mathcal{C}$ is isomorphic to topological spaces and continuous maps in $\operatorname{Struct}(\mathbf{Set})$ if and only if there exists a "Sierpinski space" $S$ with underlying set $\{0, 1\}$ satisfying the following five conditions:

(a) Every family $f_i: X \to S$ has an optimal lift.

(b) The supremum map $\operatorname{Sup}: S^I \to S$ is admissible for every set $I$.

(c) The infimum map $\operatorname{Inf}: S^I \to S$ is admissible for every finite set $I$.

(d) $S$ is a Sierpinski object in $\mathcal{C}$ as defined in exercise 3.
