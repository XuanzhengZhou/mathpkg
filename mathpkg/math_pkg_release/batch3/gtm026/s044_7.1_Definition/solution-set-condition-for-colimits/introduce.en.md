This theorem gives a necessary and sufficient condition for a locally small category $\mathcal{K}$ with small limits to have all small colimits: the constant-diagram embedding functor $(\tilde{\cdot}): \mathcal{K} \to \mathcal{K}^{\Delta}$ must satisfy the solution set condition. By Theorem 7.2, a colimit is precisely a free object over the diagram with respect to this embedding, so the General Adjoint Functor Theorem can be applied. The result bridges the existence of colimits to the well-understood solution set condition from adjoint functor theory.
