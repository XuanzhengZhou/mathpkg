A \emph{diagram scheme} $\Delta$ consists of a set $N(\Delta)$ of nodes and, for each ordered pair $(i, j)$ of nodes, a set $\Delta(i, j)$ of edges. A \emph{diagram} $(\Delta, D)$ in a category $\mathcal{K}$ is an assignment of an object $D_i$ of $\mathcal{K}$ to each node $i \in N(\Delta)$ and a $\mathcal{K}$-morphism $D_\alpha: D_i \longrightarrow D_j$ to each edge $\alpha \in \Delta(i, j)$.
