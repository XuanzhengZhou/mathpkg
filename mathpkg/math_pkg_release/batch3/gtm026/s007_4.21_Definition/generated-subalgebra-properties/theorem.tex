For a $T$-algebra $(X, \xi)$ and subsets $A, B \subset X$:

1. $A \subset \langle A \rangle$.
2. If $A \subset B$ then $\langle A \rangle \subset \langle B \rangle$.
3. $\langle A \rangle$ is a $T$-subalgebra of $(X, \xi)$.
4. If $B$ is a $T$-subalgebra, then $B = \langle B \rangle$.
5. $\langle \langle A \rangle \rangle = \langle A \rangle$ (idempotence).
6. Whenever $A$ is contained in a $T$-subalgebra $B$, $\langle A \rangle \subset B$ (minimality).
