An $\Omega$-algebra is a pair $(X, \delta)$ where $X$ is a set and $\delta$ assigns to each $\omega$ in $\Omega_n$ an $n$-ary operation
$$\delta_\omega : X^n \longrightarrow X$$

Given $\Omega$-algebras $(X, \delta)$ and $(Y, \gamma)$, an $\Omega$-homomorphism from $(X, \delta)$ to $(Y, \gamma)$ is a function $f: X \longrightarrow Y$ such that for all $\omega \in \Omega_n$ and all $x_1, \ldots, x_n \in X$,
$$f(\delta_\omega(x_1, \ldots, x_n)) = \gamma_\omega(f(x_1), \ldots, f(x_n))$$
