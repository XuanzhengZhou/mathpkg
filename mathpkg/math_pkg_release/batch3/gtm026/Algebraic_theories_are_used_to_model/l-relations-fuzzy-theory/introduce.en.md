Following Goguen (1967), any complete distributive lattice $L$ induces a fuzzy theory $(T, e, \circ)$. The theory assigns to each set $A$ the set $L^A$ of all $L$-valued predicates on $A$. The unit $e$ maps each element to its characteristic function (taking value $1$ at the element and $0$ elsewhere), while composition is defined using the lattice operations: the image of $c$ under the composite $\alpha \circ \beta$ is the supremum over all intermediate elements $b$ of the meet $b\alpha \wedge c\beta$. The distributivity condition ensures that this composition is associative, giving a well-defined fuzzy algebraic theory.
