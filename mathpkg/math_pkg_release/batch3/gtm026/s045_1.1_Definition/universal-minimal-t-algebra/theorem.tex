A $T$-algebra $(X, \xi)$ is a \textit{universal minimal $T$-algebra} if it satisfies:

1. $(X, \xi)$ is minimal.
2. If $(Y, \theta)$ is a minimal $T$-algebra there exists a $T$-homomorphism from $(X, \xi)$ to $(Y, \theta)$ (such homomorphisms must be onto by 1.2).
3. Every $T$-endomorphism $(X, \xi) \longrightarrow (X, \xi)$ is an isomorphism.

Any algebra satisfying the first two properties is isomorphic to the universal minimal one if it exists.
