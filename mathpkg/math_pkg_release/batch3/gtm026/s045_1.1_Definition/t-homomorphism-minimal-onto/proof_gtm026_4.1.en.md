*Proof.* If $B$ is a nonempty subalgebra of $(Y, \theta)$, then $A = Bf^{-1}$ is a nonempty subalgebra of $(X, \xi)$ since $U^T$ creates pullbacks. If $(X, \xi)$ is minimal and $f$ is onto, then $A$ being a nonempty subalgebra forces $A = X$, whence $B = f(A) = f(X) = Y$, so $(Y, \theta)$ is minimal.

Similarly, if $X$ is nonempty, then $\operatorname{Im}(f)$ is a nonempty subalgebra of $(Y, \theta)$. If $(Y, \theta)$ is minimal, this forces $\operatorname{Im}(f) = Y$, so $f$ is onto. $\square$
