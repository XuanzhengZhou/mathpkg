Let $(n, k)$ be a pair of non-negative integers. An $(n, k)$-structure is a pair $(K, s)$ where $K$ is a set and $s$ is an element of $K^n P^k$, where $K^n$ is the $n$-fold cartesian power and $P^k$ is the $k$th iterate of the power set operator, $KP = 2^K$.

For example:
\begin{itemize}
\item A topology of open subsets of $K$ is a $(1, 2)$-structure.
\item A uniform structure of entourages on $K$ is a $(2, 2)$-structure.
\item A ternary operation $K^3 \longrightarrow K$ is (via its graph) a $(4, 1)$-structure.
\item A partially ordered set is a $(2, 1)$-structure.
\item A set with base point is a $(1, 0)$-structure.
\end{itemize}
