Every field $\mathfrak{F}$ can be built up from its prime field $\mathfrak{P}$ by a succession of simple extensions (algebraic or transcendental). More precisely, if $\mathfrak{F}$ is countable, enumerate its elements as $\theta_1, \theta_2, \theta_3, \ldots$ and set $\mathfrak{F}_0 = \mathfrak{P}$, $\mathfrak{F}_i = \mathfrak{F}_{i-1}(\theta_i)$. Then $\mathfrak{F} = \bigcup_i \mathfrak{F}_i$, and each $\mathfrak{F}_i$ is obtained from $\mathfrak{F}_{i-1}$ by a simple transcendental or simple algebraic extension. For uncountable fields, the same result holds using transfinite induction (well-ordering).
