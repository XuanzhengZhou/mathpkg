If $a$ and $b$ are commuting elements of a ring (i.e., $ab = ba$), then for any positive integer $n$,
$$(a + b)^n = a^n + \binom{n}{1}a^{n-1}b + \binom{n}{2}a^{n-2}b^2 + \cdots + b^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k,$$
where the binomial coefficient $\binom{n}{k}$ is an integer given by
$$\binom{n}{k} = \frac{n!}{k!(n-k)!}.$$
