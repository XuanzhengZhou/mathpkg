**($\Rightarrow$)** If $\mathfrak{A}$ is a division ring, then by definition it contains more than one element and $\mathfrak{A}^*$ forms a subgroup of the multiplicative semi-group. Hence $\mathfrak{A}^*$ contains an identity element $1$. Since $1 \cdot 0 = 0 = 0 \cdot 1$, the element $1$ is an identity for the whole ring, so $\mathfrak{A}$ contains an identity $\neq 0$. Moreover, every non-zero element has a two-sided multiplicative inverse in $\mathfrak{A}^*$, so the group of units of $\mathfrak{A}$ is exactly $\mathfrak{A}^*$.

**($\Leftarrow$)** If $\mathfrak{A}$ contains an identity $1 \neq 0$ and the group of units of $\mathfrak{A}$ is precisely $\mathfrak{A}^*$, then every non-zero element is a unit (has a multiplicative inverse). Since $1 \neq 0$, the ring has more than one element. Thus the non-zero elements form a multiplicative group, satisfying the definition of a division ring.
