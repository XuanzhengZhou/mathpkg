An element $a$ of a ring $\mathfrak{A}$ is called \emph{right quasi-regular} if there exists $b \in \mathfrak{A}$ such that $a \circ b = a + b - ab = 0$. The element $b$ is called a \emph{right quasi-inverse} of $a$. Similarly, $a$ is \emph{left quasi-regular} if there exists $c$ such that $c \circ a = 0$. An element is \emph{quasi-regular} if it is both left and right quasi-regular.
