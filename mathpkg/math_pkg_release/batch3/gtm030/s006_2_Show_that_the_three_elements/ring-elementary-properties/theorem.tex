In a ring $\mathfrak{A}$:

\begin{enumerate}
\item $-(a + b) = -a - b \equiv -a + (-b)$.
\item For any integer $n$, the rules for multiples hold:
  $$n(a + b) = na + nb,$$
  $$(n + m)a = na + ma,$$
  $$(nm)a = n(ma).$$
\item The generalized associative laws hold for addition and for multiplication.
\item The generalized commutative law holds for addition.
\end{enumerate}
