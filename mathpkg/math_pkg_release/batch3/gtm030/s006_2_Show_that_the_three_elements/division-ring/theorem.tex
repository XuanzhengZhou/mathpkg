A ring $\mathfrak{A}$ is called a \emph{division ring} (also called a \emph{quasi-field}, \emph{skew field}, or \emph{sfield}) if it contains more than one element and the set $\mathfrak{A}^*$ of non-zero elements forms a subgroup of the multiplicative semi-group. Thus:
\begin{enumerate}
\item $\mathfrak{A}^*$ contains an identity element $1$, which is an identity for the whole ring.
\item For every $a \neq 0$, there exists $a^{-1} \in \mathfrak{A}$ such that $aa^{-1} = 1 = a^{-1}a$.
\end{enumerate}
