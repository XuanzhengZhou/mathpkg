A ring $\mathfrak{A}$ is an integral domain if and only if the restricted cancellation laws of multiplication hold: for all $a, b, c \in \mathfrak{A}$,
\begin{enumerate}
\item If $ab = ac$ and $a \neq 0$, then $b = c$ (left cancellation).
\item If $ba = ca$ and $a \neq 0$, then $b = c$ (right cancellation).
\end{enumerate}
