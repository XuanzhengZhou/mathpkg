We know that $1 \in \mathfrak{R}$. If $k_1, k_2 \in \mathfrak{R}$, then $(k_1k_2)\eta = (k_1\eta)(k_2\eta) = 1'1' = 1'$. Hence $k_1k_2 \in \mathfrak{R}$. Also if $k \in \mathfrak{R}$, then $k^{-1}\eta = (k\eta)^{-1} = 1'^{-1} = 1'$ and $k^{-1} \in \mathfrak{R}$. This proves that $\mathfrak{R}$ is a subgroup. Finally if $a$ is arbitrary in $\mathfrak{G}$ and $k \in \mathfrak{R}$, then $(a^{-1}ka)\eta = (a^{-1}\eta)(k\eta)(a\eta) = (a\eta)^{-1}1'(a\eta) = 1'$ so that $a^{-1}ka \in \mathfrak{R}$. Hence $\mathfrak{R}$ is invariant.
