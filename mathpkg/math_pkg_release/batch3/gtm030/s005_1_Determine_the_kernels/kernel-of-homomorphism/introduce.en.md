The kernel of a group homomorphism $\eta: \mathfrak{G} \to \mathfrak{G}'$ is the set of all elements in the domain that map to the identity element $1'$ of the target group. It is denoted $\eta^{-1}(1')$. The kernel measures how far the homomorphism is from being injective: a homomorphism is injective if and only if its kernel is trivial.
