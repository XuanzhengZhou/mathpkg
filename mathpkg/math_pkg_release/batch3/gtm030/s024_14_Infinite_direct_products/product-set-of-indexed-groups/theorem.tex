Let $J = \{\alpha\}$ be an index set, let $\{\mathfrak{G}\}$ be a collection of groups, and let $\alpha \mapsto \mathfrak{G}_{\alpha}$ be a single-valued mapping of $J$ onto $\{\mathfrak{G}\}$. The **product set** $\prod_{\alpha \in J} \mathfrak{G}_{\alpha}$ of the $\mathfrak{G}_{\alpha}$ is the set of all elements of the form $(\cdots g_{\alpha} \cdots)$ with the property that the element in the $\alpha$-place belongs to $\mathfrak{G}_{\alpha}$. Equivalently, the elements of $\tilde{\Pi} = \prod_{\alpha \in J} \mathfrak{G}_{\alpha}$ are the single-valued mappings $\alpha \mapsto g_{\alpha}$ of $J$ such that for each $\alpha \in J$, the image element $g_{\alpha}$ is in the associated group $\mathfrak{G}_{\alpha}$. If $g$ denotes an element of $\tilde{\Pi}$, we use the functional notation $g(\alpha)$ for the image element $g_{\alpha}$.
