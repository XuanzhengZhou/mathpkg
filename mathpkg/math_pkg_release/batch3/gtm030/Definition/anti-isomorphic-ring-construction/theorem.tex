For any ring $\mathfrak{A}$ with addition $+$ and multiplication $\cdot$, define a new multiplication $\times$ by
$$a \times b = b \cdot a.$$
Then $\mathfrak{A}$ with the same addition and $\times$ is a ring, and the identity mapping is an anti-isomorphism of $\mathfrak{A}, +, \cdot$ onto $\mathfrak{A}, +, \times$.
