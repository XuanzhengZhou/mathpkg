Since $\eta$ is a homomorphism of the additive group of $\mathfrak{A}$, the image $\mathfrak{A}\eta$ is a subgroup of the additive group of $\mathfrak{A}'$. Since $(a\eta)(b\eta) = (ab)\eta$, the set $\mathfrak{A}\eta$ is closed under multiplication; hence it is a subring.

If $\mathfrak{A}$ has an identity $1$, then $1' = 1\eta$ satisfies $1' \cdot a\eta = (1a)\eta = a\eta$ and $a\eta \cdot 1' = (a1)\eta = a\eta$, so $1'$ is an identity for $\mathfrak{A}\eta$. If $u$ is a unit with inverse $v$, then $(u\eta)(v\eta) = (uv)\eta = 1\eta = 1'$ and $(v\eta)(u\eta) = (vu)\eta = 1\eta = 1'$, so $u\eta$ is a unit with inverse $v\eta$.

It may happen that $1\eta = 0$, but in this case $\mathfrak{A}\eta = 0$. In particular, if $\mathfrak{A}$ is a division ring, then either $\mathfrak{A}\eta = 0$ or $\mathfrak{A}\eta$ is also a division ring; for if $a\eta \neq 0$, then $a$ is a unit, hence $a\eta$ is a unit.
