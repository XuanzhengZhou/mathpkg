This theorem establishes the fundamental algebraic properties of the complement operation in a Boolean algebra. Complements are unique, the complement map is an involution (applying it twice returns the original element), and it satisfies De Morgan's laws, which relate complements of joins to meets of complements and vice versa. These properties make the complement operation an order-reversing bijection.
