A Boolean ring is a ring in which every element is idempotent ($a^2 = a$). From this single condition, it follows that the ring is commutative and every element satisfies $2a = 0$ (all elements have additive order dividing $2$). Boolean rings with identity are in one-to-one correspondence with Boolean algebras: from a Boolean algebra one obtains a Boolean ring using symmetric difference as addition and meet as multiplication, and conversely.
