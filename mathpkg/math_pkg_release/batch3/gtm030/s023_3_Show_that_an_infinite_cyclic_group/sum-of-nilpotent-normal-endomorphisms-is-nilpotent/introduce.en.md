For an indecomposable $M$-group satisfying both chain conditions, the set of nilpotent normal $M$-endomorphisms is closed under addition (whenever the sum is an endomorphism). The proof is a clever application of Corollary 1: if $\eta_1 + \eta_2$ were not nilpotent, it would be an automorphism, leading to the equation $\lambda_1 + \lambda_2 = 1$ with both $\lambda_i$ nilpotent, which is shown to be impossible by expanding $(\lambda_1 + \lambda_2)^m$ using the binomial theorem and the commutativity $\lambda_1\lambda_2 = \lambda_2\lambda_1$.
