Given a direct decomposition $\mathfrak{G} = \mathfrak{G}_1 \times \mathfrak{G}_2 \times \cdots \times \mathfrak{G}_n$, define $\epsilon_i$ to be the endomorphism of $\mathfrak{G}$ obtained by putting together the endomorphisms

$$x_1 \rightarrow 1, \cdots, x_{i-1} \rightarrow 1, x_i \rightarrow x_i, x_{i+1} \rightarrow 1, \cdots, x_n \rightarrow 1.$$

Then for any $x = x_1x_2 \cdots x_n$ with $x_i \in \mathfrak{G}_i$,

$$x\epsilon_i = (x_1x_2 \cdots x_n)\epsilon_i = x_i.$$

The projections $\epsilon_i$ satisfy the characteristic properties:

$$\epsilon_i^2 = \epsilon_i, \quad \epsilon_i\epsilon_j = 0 \text{ for } i \neq j \quad \text{(orthogonal idempotents)}$$

$$\epsilon_1 + \epsilon_2 + \cdots + \epsilon_n = 1.$$
