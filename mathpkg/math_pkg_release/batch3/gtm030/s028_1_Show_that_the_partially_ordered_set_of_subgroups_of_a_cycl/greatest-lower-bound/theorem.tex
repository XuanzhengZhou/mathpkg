Similar definitions and remarks apply to lower bounds. An element $\ell$ of a partially ordered set $S$ is a \text{lower bound} for the subset $A$ if $\ell \leq a$ for every $a \in A$. The element $\ell$ is a \text{greatest lower bound} (g.l.b.) if $\ell$ is a lower bound and $\ell \geq w$ for any lower bound $w$ of $A$. If a greatest lower bound exists, it is unique.
