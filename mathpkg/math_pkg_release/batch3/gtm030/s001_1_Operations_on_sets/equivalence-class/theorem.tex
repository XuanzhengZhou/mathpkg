Let $\sim$ be an equivalence relation on a set $S$. For $a \in S$, the equivalence class determined by (or containing) $a$ is the subset
$$\bar{a} = \{ b \in S \mid b \sim a \}.$$
It is the maximal collection of mutually equivalent elements containing $a$: if $c$ is any element equivalent to some $b \in \bar{a}$, then $c \in \bar{a}$.
