Let $\alpha : S \to T$ be a mapping. If $a' \in T$ and $a \in S$ is such that $a\alpha = a'$, then the equivalence class $\bar{a}$ under the equivalence relation $x \sim y \iff x\alpha = y\alpha$ is precisely the set of elements of $S$ that are mapped into $a'$. This set is called the inverse image of $a'$ and is denoted $\alpha^{-1}(a')$.
