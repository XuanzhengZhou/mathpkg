Consider the multiples $x|b|$ of $|b|$ that are $\leq a$. The collection $M$ of these multiples is not vacuous since $-|a||b| \leq -|a| \leq a$. Hence, the set $M$ has a greatest member $h|b|$. Then $h|b| \leq a$ so that $a = h|b| + r$ where $r \geq 0$. On the other hand $(h+1)|b| = h|b| + |b| > h|b|$. Hence $(h+1)|b| > a$ and $h|b| + |b| > h|b| + r$. Thus, $r < |b|$. We now set $q = h$ if $b > 0$ and $q = -h$ if $b < 0$. Then $h|b| = qb$ and $a = qb + r$ as required.
