$$
\text{Let } \mathfrak{M} \text{ be an } \mathfrak{A}\text{-module. An } \mathfrak{A}\text{-subgroup } \mathfrak{N} \text{ of } \mathfrak{M}
\text{ is a subgroup of the additive group of } \mathfrak{M} \text{ that is closed under scalar multiplication:}
\text{ for all } a \in \mathfrak{A} \text{ and } x \in \mathfrak{N}, \text{ we have } ax \in \mathfrak{N}.
$$
$$
\text{The image } \mathfrak{M}\eta \text{ of a module under a homomorphism } \eta \text{ is a submodule.}
\text{ The kernel } \mathfrak{R} \text{ of a module homomorphism is a submodule.}
$$
$$
\text{The submodules of the left module of the ring } \mathfrak{A} \text{ are precisely the left ideals of } \mathfrak{A}.
$$
