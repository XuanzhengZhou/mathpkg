$$
\text{The additive group of a ring } \mathfrak{A} \text{ can be used as the group part of a left module.}
\text{ Take the product of } a \text{ in the ring } \mathfrak{A} \text{ by } x \text{ in the additive group }
\mathfrak{M} = (\mathfrak{A}, +) \text{ to be the ring product } ax.
\text{ Conditions } 2_l \text{ and } 3_l \text{ hold by ring axioms, hence } (\mathfrak{A}, +) \text{ is a left } \mathfrak{A}\text{-module.}
$$
$$
\text{We refer to this module as the left module of the ring } \mathfrak{A}.
\text{ Its submodules are precisely the left ideals of } \mathfrak{A}.
$$
