$$
\text{If } \mathfrak{M} \text{ is a cyclic and unitary } \mathfrak{A}\text{-module,}
\text{ then } \mathfrak{M} \text{ contains an element } x \text{ such that every element of } \mathfrak{M}
\text{ is a multiple } ax \text{ of } x \text{ with } a \in \mathfrak{A}.
\text{ In particular, } x \text{ itself has the form } ex \text{ for a suitable } e \in \mathfrak{A}.
$$
