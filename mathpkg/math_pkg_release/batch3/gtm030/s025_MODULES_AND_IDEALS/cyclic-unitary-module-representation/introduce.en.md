If an $\mathfrak{A}$-module $\mathfrak{M}$ is both cyclic (generated by a single element) and unitary ($\mathfrak{A}\mathfrak{M} = \mathfrak{M}$), then there exists an element $x \in \mathfrak{M}$ such that every element of $\mathfrak{M}$ is of the form $ax$ for some $a \in \mathfrak{A}$. This is a direct consequence of Theorem 1 on generator representation in unitary modules, specialized to the case where the generating set is a singleton. Moreover, the generator $x$ itself can be expressed as $x = ex$ for some idempotent-like element $e \in \mathfrak{A}$.
