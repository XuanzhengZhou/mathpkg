$$
\text{Let } \mathfrak{M} \text{ be an } \mathfrak{A}\text{-module and let } \mathfrak{U} \text{ be a two-sided ideal of } \mathfrak{A}
\text{ such that } ux = 0 \text{ for all } u \in \mathfrak{U} \text{ and } x \in \mathfrak{M}.
\text{ Then } \mathfrak{M} \text{ can be regarded as an } \mathfrak{A}/\mathfrak{U}\text{-module via the well-defined product}
$$
$$
(a + \mathfrak{U})x = ax.
$$
$$
\text{This composition satisfies } 1_l, 2_l, \text{ and } 3_l,
\text{ making } \mathfrak{M} \text{ into an } \mathfrak{A}/\mathfrak{U}\text{-module.}
$$
