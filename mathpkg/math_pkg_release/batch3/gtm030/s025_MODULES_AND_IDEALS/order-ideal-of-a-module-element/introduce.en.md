For an element $x$ in an $\mathfrak{A}$-module $\mathfrak{M}$, the order ideal $\mathfrak{J}_x$ is defined as $\{c \in \mathfrak{A} \mid cx = 0\}$, i.e., the set of ring elements that annihilate $x$. It arises naturally as the kernel of the $\mathfrak{A}$-homomorphism $a \mapsto ax$ from $\mathfrak{A}$ into $\mathfrak{M}$. The image of this map, $\mathfrak{A}x = \{ax \mid a \in \mathfrak{A}\}$, is the cyclic submodule generated by $x$, and by the fundamental homomorphism theorem, $\mathfrak{A}x \cong \mathfrak{A} / \mathfrak{J}_x$. The order ideal generalizes the notion of the order of a group element to the module setting.
