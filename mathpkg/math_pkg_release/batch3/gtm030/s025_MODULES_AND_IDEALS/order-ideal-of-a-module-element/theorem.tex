$$
\text{Let } x \text{ be any element of an } \mathfrak{A}\text{-module } \mathfrak{M}
\text{ and consider the mapping } a \mapsto ax \text{ of } \mathfrak{A} \text{ into } \mathfrak{M}.
\text{ This is an } \mathfrak{A}\text{-homomorphism since }
ba \mapsto (ba)x = b(ax).
$$
$$
\text{The set } \mathfrak{A}x = \{ax \mid a \in \mathfrak{A}\} \text{ is a submodule of } \mathfrak{M}
\text{ and the kernel } \mathfrak{J}_x \text{ of the mapping is a left ideal of } \mathfrak{A}.
\text{ By definition, } \mathfrak{J}_x = \{c \in \mathfrak{A} \mid cx = 0\}.
\text{ We call } \mathfrak{J}_x \text{ the order ideal of the element } x.
$$
$$
\text{By the fundamental theorem of module homomorphisms: } \mathfrak{A}x \cong \mathfrak{A} / \mathfrak{J}_x.
$$
