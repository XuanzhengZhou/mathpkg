$$
\text{A right module is a commutative group } M \text{ with an operator set } \mathfrak{A}
\text{ that is a ring, where the mapping } a \mapsto a_r \text{ of } a \in \mathfrak{A}
\text{ into the associated endomorphism of } \mathfrak{M} \text{ is a ring homomorphism.}
$$
$$
\text{Denoting the endomorphism associated with } a \text{ by } a_r \text{ and the product by } xa,
\text{ so that } xa_r = xa, \text{ the assumptions are:}
$$
$$
1_r.\quad (x + y)a = xa + ya
$$
$$
2_r.\quad x(a + b) = xa + xb
$$
$$
3_r.\quad x(ab) = (xa)b.
$$
$$
\text{If } \mathfrak{A} \text{ is arbitrary and } \mathfrak{A}' \text{ is a ring anti-isomorphic to } \mathfrak{A},
\text{ then any left } \mathfrak{A}\text{-module can be regarded as a right } \mathfrak{A}'\text{-module}
\text{ and any right } \mathfrak{A}\text{-module can be regarded as a left } \mathfrak{A}'\text{-module.}
$$
