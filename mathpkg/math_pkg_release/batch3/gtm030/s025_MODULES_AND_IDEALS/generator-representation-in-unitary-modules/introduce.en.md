In a unitary $\mathfrak{A}$-module $\mathfrak{M}$ (where $\mathfrak{A}\mathfrak{M} = \mathfrak{M}$), every element can be expressed as a finite linear combination of generators with coefficients in $\mathfrak{A}$. Specifically, if $X$ is a generating set for $\mathfrak{M}$, then any $x \in \mathfrak{M}$ can be written as $\sum_{i=1}^r a_i x_i$ with $a_i \in \mathfrak{A}$ and $x_i \in X$. This theorem simplifies the description of elements in unitary modules and is foundational for understanding finitely generated modules, where a finite generating set suffices to represent all elements as such linear combinations.
