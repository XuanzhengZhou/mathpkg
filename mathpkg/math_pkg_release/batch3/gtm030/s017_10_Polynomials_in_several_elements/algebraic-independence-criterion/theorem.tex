Let $\mathfrak{A}$ be a commutative integral domain with identity. Elements $u_1, u_2, \cdots, u_r$ of an extension ring are algebraically independent over $\mathfrak{A}$ if and only if each $u_k$ ($k = 1, 2, \cdots, r$) is transcendental over $\mathfrak{A}[u_1, u_2, \cdots, u_{k-1}]$. Equivalently, the only polynomial relation

$$\sum d_{i_1 i_2 \cdots i_r} u_1^{i_1} u_2^{i_2} \cdots u_r^{i_r} = 0 \quad (d_{i_1 \cdots i_r} \in \mathfrak{A})$$

is the trivial one where all coefficients are zero.
