An integral domain $\mathfrak{A}$ is a \text{Euclidean domain} if there exists a function $\delta : \mathfrak{A} \setminus \{0\} \to \mathbb{N}_0$ (non-negative integers) satisfying:
\begin{enumerate}
\item $\delta(a) \leq \delta(ab)$ for all $a, b \neq 0$ in $\mathfrak{A}$,
\item For any $a, b \in \mathfrak{A}, b \neq 0$, there exist $q, r \in \mathfrak{A}$ such that $a = bq + r$ where either $r = 0$ or $\delta(r) < \delta(b)$.
\end{enumerate}
