Let $a, b, c \in \mathfrak{S}$ and set $r = (a, (b, c))$. Then $r \mid a$ and $r \mid (b, c)$, so that $r \mid b$ and $r \mid c$. Also, if $s \mid a, b, c$, then $s \mid a$ and $s \mid (b, c)$, so that $s \mid (a, (b, c)) = r$. This shows that $r = (a, (b, c))$ is a g.c.d. of $a, b$ and $c$. A similar argument by induction holds for more than three elements. Moreover, it is also clear that $((a, b), c)$ is equally a g.c.d. of $a, b$ and $c$.
