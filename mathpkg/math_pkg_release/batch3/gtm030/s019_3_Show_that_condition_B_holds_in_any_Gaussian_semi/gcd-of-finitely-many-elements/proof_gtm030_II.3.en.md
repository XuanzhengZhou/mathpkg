Let $a, b, c \in \mathfrak{S}$. Set $r = (a, (b, c))$. Then $r \mid a$ and $r \mid (b, c)$, so $r \mid b$ and $r \mid c$. Also if $s \mid a, b, c$, then $s \mid a$ and $s \mid (b, c)$, so $s \mid (a, (b, c)) = r$. Thus $r$ is a g.c.d. of $a, b, c$. A similar inductive argument works for more than three elements.
