With each left ideal $\mathfrak{N}$ of $\mathfrak{A}[x]$ and each $j = 0, 1, 2, \dots$ we associate the set $\mathfrak{J}_j(\mathfrak{N})$ of elements $b \in \mathfrak{A}$ such that there exists an element
$$bx^j + b_{j-1}x^{j-1} + \cdots + b_0$$
in $\mathfrak{N}$. Then $\mathfrak{J}_j(\mathfrak{N})$ is a left ideal in $\mathfrak{A}$. Moreover, if $bx^j + b_{j-1}x^{j-1} + \cdots + b_0 \in \mathfrak{N}$, then
$$bx^{j+1} + b_{j-1}x^j + \cdots + b_0x = x(bx^j + b_{j-1}x^{j-1} + \cdots + b_0) \in \mathfrak{N}.$$
Hence
$$\mathfrak{J}_0(\mathfrak{N}) \subseteq \mathfrak{J}_1(\mathfrak{N}) \subseteq \mathfrak{J}_2(\mathfrak{N}) \subseteq \cdots.$$

Since $\mathfrak{A}$ satisfies the ascending chain condition, there exists an integer $N$ such that $\mathfrak{J}_N(\mathfrak{N}) = \mathfrak{J}_{N+1}(\mathfrak{N}) = \cdots$. Set $\mathfrak{J}(\mathfrak{N}) = \bigcup_{j=0}^\infty \mathfrak{J}_j(\mathfrak{N}) = \mathfrak{J}_N(\mathfrak{N})$. Since $\mathfrak{J}(\mathfrak{N})$ is a left ideal of $\mathfrak{A}$, it is finitely generated; let $b_{N1}, b_{N2}, \dots, b_{Nm_N}$ be generators. Choose for each $b_{Ni}$ a polynomial $f_{Ni}(x) = b_{Ni}x^N + \cdots \in \mathfrak{N}$ of degree $\leq N$. Similarly, for $j = 0, 1, \dots, N-1$, let $b_{j1}, \dots, b_{jm_j}$ be generators of $\mathfrak{J}_j(\mathfrak{N})$ and $f_{ji}(x)$ corresponding polynomials of degree $\leq j$.

We show that $\mathfrak{N}$ is generated by $\{f_{ji}\}$. Let $g = c_r x^r + c_{r-1}x^{r-1} + \cdots \in \mathfrak{N}$.

If $r \leq N$: then $c_r \in \mathfrak{J}_r(\mathfrak{N})$, so $c_r = a_{r1}b_{r1} + \cdots + a_{rm_r}b_{rm_r}$ for suitable $a_{ri} \in \mathfrak{A}$. Hence $g - \sum a_{ri} f_{ri}(x)$ is a polynomial in $\mathfrak{N}$ of degree $< r$.

If $r > N$: then $c_r \in \mathfrak{J}_r(\mathfrak{N}) = \mathfrak{J}_N(\mathfrak{N})$, so $c_r = a_{r1}b_{N1} + \cdots + a_{rm_N}b_{Nm_N}$. Hence $g - \sum a_{ri} x^{r-N} f_{Ni}(x)$ is a polynomial in $\mathfrak{N}$ of degree $< r$.

In either case, the degree is reduced. By induction on $r$, $g$ lies in the ideal generated by the $f_{ji}$. Thus $\mathfrak{N}$ is finitely generated, so $\mathfrak{A}[x]$ satisfies the ascending chain condition.
