Let $\mathfrak{M}$ be a module with fixed generators $x_1, x_2, \cdots, x_r$. For any submodule $\mathfrak{N}$ of $\mathfrak{M}$ and each $j = 1, 2, \cdots, r$, define $\mathfrak{J}_j(\mathfrak{N})$ to be the set of elements $b \in \mathfrak{A}$ for which there exists an element
$$bx_j + b_{j+1}x_{j+1} + \cdots + b_r x_r$$
in $\mathfrak{N}$. Then $\mathfrak{J}_j(\mathfrak{N})$ is a left ideal and $\mathfrak{J}_j(\mathfrak{N}) \subseteq \mathfrak{J}_j(\mathfrak{P})$ for all $j$ if $\mathfrak{N} \subseteq \mathfrak{P}$.

If $\mathfrak{N} \subseteq \mathfrak{P}$ and $\mathfrak{J}_j(\mathfrak{N}) = \mathfrak{J}_j(\mathfrak{P})$ for all $j$, then $\mathfrak{N} = \mathfrak{P}$.
