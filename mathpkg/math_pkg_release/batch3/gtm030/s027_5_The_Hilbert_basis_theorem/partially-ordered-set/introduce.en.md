A partially ordered set (or poset) is a set together with a binary relation $\geq$ that is reflexive, antisymmetric, and transitive. Examples include: the integers with the usual ordering, the positive integers ordered by divisibility ($a \geq b$ if $a \mid b$), the power set of any set ordered by reverse inclusion, and the set of subgroups of a group ordered by inclusion. If every pair of elements in a poset is comparable, the poset is called a linearly ordered set or a chain.
