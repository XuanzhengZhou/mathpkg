For elements $a, b$ of a Boolean algebra $B$, the \emph{symmetric difference} is defined by
$$a + b = (a \cap b') \cup (a' \cap b).$$
Equivalently,
$$a + b = (a \cup b) \cap (a \cap b)'.$$
In the case of subsets of a set $S$, the symmetric difference $U + V$ is the set of elements belonging to either $U$ or $V$ but not to both.
