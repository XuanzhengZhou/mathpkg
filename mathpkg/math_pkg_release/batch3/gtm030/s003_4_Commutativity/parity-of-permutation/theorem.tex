Let $\alpha$ be a permutation with cycle decomposition into disjoint cycles of lengths $r, s, \cdots, u$. Define
$$N(\alpha) = (r - 1) + (s - 1) + \cdots + (u - 1).$$
Then $\alpha$ is a product of $N(\alpha)$ transpositions. If $N(\alpha)$ is even (odd), then any factorization of $\alpha$ as a product of transpositions contains an even (odd) number of factors.

This is proved using the formulas:
$$(a c_1 c_2 \cdots c_h b d_1 \cdots d_k)(ab) = (a c_1 \cdots c_h)(b d_1 \cdots d_k)$$
$$(a c_1 \cdots c_h)(b d_1 \cdots d_k)(ab) = (a c_1 \cdots c_h b d_1 \cdots d_k)$$
which imply $N(\alpha(ab)) = N(\alpha) \pm 1$, and the fact that $N(1) = 0$.
