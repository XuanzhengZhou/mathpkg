A \text{group} is a semigroup that has an identity and in which every element is a unit. Thus a group is a system consisting of a set $\mathfrak{G}$ and a binary composition in $\mathfrak{G}$ such that:

\begin{enumerate}
\item $(ab)c = a(bc)$ (associative law).
\item There exists an element $1 \in \mathfrak{G}$ such that $a1 = a = 1a$ for all $a \in \mathfrak{G}$.
\item For each $a \in \mathfrak{G}$ there is an element $a^{-1} \in \mathfrak{G}$ such that $aa^{-1} = 1 = a^{-1}a$.
\end{enumerate}
