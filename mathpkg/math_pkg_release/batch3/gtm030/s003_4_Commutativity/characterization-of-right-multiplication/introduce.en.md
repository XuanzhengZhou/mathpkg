A transformation of a group's underlying set that commutes with all left multiplications must itself be a right multiplication. This characterization is a powerful structural result: the centralizer of the set of left multiplications in the full symmetric group on the group's underlying set is exactly the set of right multiplications, and vice versa. The proof is elegantly simple: apply the transformation to the identity element.
