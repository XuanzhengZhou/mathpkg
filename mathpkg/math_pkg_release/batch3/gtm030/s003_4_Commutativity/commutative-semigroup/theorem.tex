If $ab = ba$ holds for every pair of elements $a, b$ in a semigroup $\mathfrak{S}$, then $\mathfrak{S}$ is called a \text{commutative} semigroup. In this case, if $a_i a_j = a_j a_i$ for all $i, j = 1, 2, \cdots, n$, then for any permutation $1', 2', \cdots, n'$ of $1, 2, \cdots, n$,

$$a_{1'} a_{2'} \cdots a_{n'} = a_1 a_2 \cdots a_n.$$
