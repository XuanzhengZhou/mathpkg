A lattice is \text{distributive} if for all $a, b, c$:
$$a \cap (b \cup c) = (a \cap b) \cup (a \cap c).$$
This condition is equivalent to its dual:
$$a \cup (b \cap c) = (a \cup b) \cap (a \cup c).$$
Examples include the lattice of all subsets of a set (with union and intersection) and the lattice of positive integers under divisibility (with GCD as join, LCM as meet).
