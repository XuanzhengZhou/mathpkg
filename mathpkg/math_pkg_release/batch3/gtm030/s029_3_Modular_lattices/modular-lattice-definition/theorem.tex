A lattice is called \textit{modular} (or \textit{Dedekind}) if it satisfies the condition

$$L_5: \quad \text{If } a \geq b, \text{ then } a \cap (b \cup c) = b \cup (a \cap c).$$

Since $b = a \cap b$ when $a \geq b$, the right-hand side can be written as $(a \cap b) \cup (a \cap c)$, showing that modularity amounts to the distributive law restricted to triples $a, b, c$ with $a \geq b$.
