This exercise asks to prove Birkhoff's characterization theorem for distributive and modular lattices: a lattice is distributive if and only if it contains no sublattice isomorphic to the pentagon $N_5$ or the diamond $M_3$, and a lattice is modular if and only if it contains no sublattice isomorphic to $N_5$.
