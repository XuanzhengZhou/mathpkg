A lattice is called \text{modular} (or \text{Dedekind}) if it satisfies the modular law:

$$L_5: \quad \text{If } a \geq b, \text{ then } a \cap (b \cup c) = b \cup (a \cap c).$$

A lattice is \text{distributive} if it satisfies the full distributive law:

$$a \cap (b \cup c) = (a \cap b) \cup (a \cap c)$$

for all $a, b, c$. Every distributive lattice is modular, but the converse does not hold in general.
