Let $E_1$ be a subspace of the $n$-dimensional vector space $E$. Then
$$\dim E_1 \leq \dim E.$$
If equality holds, then $E_1 = E$.

Moreover,
$$\dim E = \dim E_1 + \dim(E/E_1).$$

If $E = E_1 \oplus E_2$ is a direct sum, then
$$\dim E = \dim E_1 + \dim E_2.$$

For any two subspaces $E_1, E_2 \subset E$,
$$\dim(E_1 + E_2) = \dim E_1 + \dim E_2 - \dim(E_1 \cap E_2).$$
