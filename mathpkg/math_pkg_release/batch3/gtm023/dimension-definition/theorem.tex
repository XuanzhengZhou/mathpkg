Recall that every system of generators of a non-trivial vector space contains a basis. A finitely generated non-trivial vector space has a finite basis. Every basis of such a space $E$ consists of the same number of vectors. This number is called the dimension of $E$ and is denoted by $\dim E$. $E$ is called a finite-dimensional vector space. If $E = (0)$, we set $\dim E = 0$. If $E$ does not have finite dimension, it is called an infinite-dimensional vector space.
