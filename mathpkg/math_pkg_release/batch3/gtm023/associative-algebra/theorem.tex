An algebra $A$ is called associative if
$$x(yz) = (xy)z \quad x, y, z \in A$$
and commutative if
$$xy = yx \quad x, y \in A.$$

For any algebra $A$, the opposite algebra $A^{\operatorname{opp}}$ is defined by $(xy)^{\operatorname{opp}} = yx$. If $A$ is associative then so is $A^{\operatorname{opp}}$. If $A$ is commutative, $A^{\operatorname{opp}} = A$.

A unit element (or identity) in an algebra is an element $e$ such that $xe = ex = x$ for all $x \in A$. If a unit exists, it is unique.
