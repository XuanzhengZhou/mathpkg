The function $\varphi \mapsto \|\varphi\| = \sup_{\|x\|=1} \|\varphi x\|$ is a norm-function on the space $B(E;E)$ of bounded linear transformations of a normed space $E$. That is, it satisfies:

\begin{itemize}
\item $N_1$: $\|\varphi\| \geq 0$ and $\|\varphi\| = 0$ only if $\varphi = 0$.
\item $N_2$: $\|\varphi + \psi\| \leq \|\varphi\| + \|\psi\|$.
\item $N_3$: $\|\lambda \varphi\| = |\lambda| \cdot \|\varphi\|$.
\end{itemize}
