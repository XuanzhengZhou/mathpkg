Let $E$ be a normed linear space and define $\varrho(x, y) = \|x - y\|$. Then:

\begin{enumerate}
\item $\varrho(x, y) > 0$ if $x \neq y$ (positive definiteness)
\item $\varrho(x, y) \leq \varrho(x, z) + \varrho(z, y)$ (triangle inequality)
\item $\varrho(x, y) = \varrho(y, x)$ (symmetry)
\end{enumerate}

Hence $\varrho$ is a metric in $E$, defining the \textbf{norm topology}. Moreover, the linear operations are continuous in this topology, so $E$ becomes a topological vector space.
