Let $E$ be a real linear space. A \textbf{norm-function} in $E$ is a real-valued function $\|\cdot\| : E \to \mathbb{R}$ satisfying:

\begin{itemize}
\item $N_1$: $\|x\| \geq 0$ for every $x \in E$, and $\|x\| = 0$ only if $x = 0$.
\item $N_2$: $\|x + y\| \leq \|x\| + \|y\|$ for all $x, y \in E$.
\item $N_3$: $\|\lambda x\| = |\lambda| \cdot \|x\|$ for all $x \in E$, $\lambda \in \mathbb{R}$.
\end{itemize}
