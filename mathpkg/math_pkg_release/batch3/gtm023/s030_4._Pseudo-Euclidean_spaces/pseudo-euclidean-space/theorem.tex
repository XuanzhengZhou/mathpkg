A \text{pseudo-Euclidean space} is a real linear space $E$ in which a non-degenerate indefinite bilinear function is given. This bilinear function is called the \text{inner product} and is denoted by $(\ ,\ )$. The index of the inner product is called the \text{index} of the pseudo-Euclidean space.

A vector $x \neq 0$ is called
\begin{itemize}
  \item \text{space-like}, if $(x, x) > 0$
  \item \text{time-like}, if $(x, x) < 0$
  \item a \text{light-vector}, if $(x, x) = 0$
\end{itemize}

The \text{light-cone} is the set of all light-vectors.

Two vectors $x$ and $y$ are called \text{orthogonal} if $(x, y) = 0$. The light-cone consists of all vectors which are orthogonal to themselves.
