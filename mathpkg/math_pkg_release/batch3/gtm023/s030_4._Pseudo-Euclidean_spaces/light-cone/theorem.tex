The \text{light-cone} is the set of all light-vectors, i.e., the set
$$\{x \in E \mid x \neq 0,\ (x, x) = 0\}.$$

A vector $x \neq 0$ is called:
\begin{itemize}
  \item \text{space-like}, if $(x, x) > 0$
  \item \text{time-like}, if $(x, x) < 0$
  \item a \text{light-vector}, if $(x, x) = 0$
\end{itemize}

The light-cone consists of all vectors which are orthogonal to themselves:
$$(x, x) = 0 \iff x \text{ is a light-vector}.$$
