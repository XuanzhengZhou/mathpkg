Let $E$ be an $n$-dimensional pseudo-Euclidean space of index $n-1$. Let $z$ be a fixed time-like unit vector, giving the orthogonal decomposition
$$E = \operatorname{span}\{z\} \oplus z^\perp.$$
Then the following properties hold:

\begin{enumerate}
  \item Two time-like vectors are never orthogonal.
  \item A time-like vector is never orthogonal to a light-vector.
  \item Two light-vectors are orthogonal if and only if they are linearly dependent.
  \item The orthogonal complement of a light-vector is an $(n-1)$-dimensional subspace of $E$ in which the inner product is positive semidefinite and has rank $n-2$.
\end{enumerate}
