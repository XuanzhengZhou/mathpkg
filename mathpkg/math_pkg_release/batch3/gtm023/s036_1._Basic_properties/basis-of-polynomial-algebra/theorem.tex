The polynomial $(0, 1, 0, \ldots)$ is denoted by $t$. Then
$$t^k = \underbrace{(0, \ldots, 0, 1, 0, \ldots)}_{k} \quad k = 0, 1, \ldots.$$
Every polynomial $f$ can be uniquely written as
$$f = \sum_{v=0}^{\infty} \alpha_v t^v \quad \alpha_v \in \Gamma$$
where only finitely many $\alpha_v$ are different from zero. The elements $t^k$ ($k = 0, 1, \ldots$) are linearly independent and form a basis of the vector space $\Gamma[t]$.
