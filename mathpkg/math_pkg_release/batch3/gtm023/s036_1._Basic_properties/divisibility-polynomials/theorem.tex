Let $f$ and $g$ be non-zero polynomials. $g$ divides $f$ (or $f$ is a multiple of $g$) if there is a polynomial $h$ such that $f = g \cdot h$. In this case we write $g \mid f$. $f$ is a multiple of $g$ if and only if it is contained in the ideal $I_g$ generated by $g$. If $h \mid g$ and $g \mid f$, then $h \mid f$. Two monic polynomials which divide each other are equal.
