(Euclidean algorithm) Let $f, g$ be polynomials with $g \neq 0$. Write
$$f = p_1 g + g_1, \quad g_1 = 0 \text{ or } \deg g_1 < \deg g.$$

If $g_1 \neq 0$, write
$$g = p_2 g_1 + g_2, \quad g_2 = 0 \text{ or } \deg g_2 < \deg g_1.$$

Show that the repeated application of this process yields an explicit calculation of the greatest common divisor of $f$ and $g$. (This method is called the Euclidean algorithm).
