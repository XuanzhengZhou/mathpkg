Let $\mu = f_1^{k_1} \cdots f_r^{k_r}$ be the prime factorization of the minimum polynomial of $a$, and set $g = f_1 \cdots f_r$. Then $f(a)$ is nilpotent if and only if $g$ divides $f$. In particular, if all exponents $k_i = 1$ (i.e., $\mu$ is squarefree), then $\Gamma(a)$ has no non-zero nilpotent elements.
