The map $i(\alpha) = (\alpha, 0, \ldots)$ is clearly injective since $i(\alpha) = i(\beta)$ implies $(\alpha, 0, \ldots) = (\beta, 0, \ldots)$ and thus $\alpha = \beta$. It is a homomorphism because $i(\alpha + \beta) = (\alpha+\beta, 0, \ldots) = (\alpha, 0, \ldots) + (\beta, 0, \ldots) = i(\alpha) + i(\beta)$ and $i(\alpha\beta) = (\alpha\beta, 0, \ldots)$. For the product, using the convolution formula, $\gamma_0 = \alpha\beta$ and $\gamma_k = 0$ for $k \geq 1$, so $i(\alpha\beta) = i(\alpha)i(\beta)$. Thus $i$ is an injective algebra homomorphism.
