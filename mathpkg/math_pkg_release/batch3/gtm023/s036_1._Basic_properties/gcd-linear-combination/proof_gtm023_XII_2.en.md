By definition, $I_f = I_{f_1} + \cdots + I_{f_r}$ (generalizing the two-polynomial case). Every element $h \in I_f$ can be written as $h = \sum_{i=1}^{r} f_i g_i$ for some polynomials $g_i$. Since $f \in I_f$, $f$ itself must admit such a representation.
