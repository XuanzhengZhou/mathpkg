Let $\mathcal{I}$ denote the set of all ideals in $\Gamma[t]$, which is a lattice with respect to the partial order given by inclusion. Let $\mathcal{P}$ denote the set of all monic polynomials together with the zero polynomial. Define a partial order in $\mathcal{P}$ by setting $f \leq g$ whenever $f$ divides $g$. Then $\mathcal{I}$ and $\mathcal{P}$ are anti-isomorphic lattices.
