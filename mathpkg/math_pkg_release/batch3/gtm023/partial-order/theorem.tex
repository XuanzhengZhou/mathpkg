Let $A$ be a set and assume that for some pairs $X, Y$ ($X \in A, Y \in A$) a relation, denoted by $X \leq Y$, is defined which satisfies:

(i) $X \leq X$ for every $X \in A$ (Reflexivity)
(ii) if $X \leq Y$ and $Y \leq X$ then $X = Y$ (Antisymmetry)
(iii) If $X \leq Y$ and $Y \leq Z$, then $X \leq Z$ (Transitivity).

Then $\leq$ is called a partial order in $A$.
