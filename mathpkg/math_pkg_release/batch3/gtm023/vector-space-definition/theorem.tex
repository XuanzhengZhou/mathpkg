A vector (linear) space, $E$, over the field $\Gamma$ is a set of elements $x, y, \ldots$ called vectors with the following algebraic structure:

I. $E$ is an additive group; there is a fixed mapping $E \times E \to E$ denoted by
$$(x, y) \to x + y$$
satisfying the following axioms:
\begin{align}
&\text{I.1.} && (x + y) + z = x + (y + z) \quad \text{(associative law)} \\
&\text{I.2.} && x + y = y + x \quad \text{(commutative law)} \\
&\text{I.3.} && \text{there exists a zero-vector } 0 \text{ such that } x + 0 = 0 + x = x \text{ for every } x \in E. \\
&\text{I.4.} && \text{To every vector } x \text{ there is a vector } -x \text{ such that } x + (-x) = 0.
\end{align}

II. There is a fixed mapping $\Gamma \times E \to E$ denoted by
$$(\lambda, x) \to \lambda x$$
satisfying the axioms:
\begin{align}
&\text{II.1.} && (\lambda \mu)x = \lambda (\mu x) \quad \text{(associative law)} \\
&\text{II.2.} && (\lambda + \mu)x = \lambda x + \mu x \quad \text{and} \quad \lambda(x + y) = \lambda x + \lambda y \quad \text{(distributive laws)} \\
&\text{II.3.} && 1 \cdot x = x \quad (1 \text{ unit element of } \Gamma)
\end{align}

$\Gamma$ is called the coefficient field, and its elements are called scalars.
