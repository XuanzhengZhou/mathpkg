The set $GL(E)$ of all linear automorphisms (bijective linear transformations) of a vector space $E$ forms a group under composition $(\varphi, \psi) \mapsto \psi \circ \varphi$, satisfying:
\begin{enumerate}
\item $\chi \circ (\psi \circ \varphi) = (\chi \circ \psi) \circ \varphi$ (associative law)
\item There exists an identity element $\iota_E$ (the identity map) such that $\iota_E \circ \varphi = \varphi \circ \iota_E = \varphi$
\item For each $\varphi \in GL(E)$, the inverse $\varphi^{-1}$ satisfies $\varphi^{-1} \circ \varphi = \varphi \circ \varphi^{-1} = \iota_E$
\end{enumerate}
