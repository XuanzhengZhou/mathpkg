Let $E$ be a real linear space. A norm-function is a real-valued function $\|\cdot\|$ satisfying:
\begin{align}
N_1:&\quad \|x\| \geq 0 \text{ for every } x \in E, \text{ and } \|x\| = 0 \text{ only if } x = 0. \\
N_2:&\quad \|x + y\| \leq \|x\| + \|y\|. \\
N_3:&\quad \|\lambda x\| = |\lambda| \cdot \|x\|.
\end{align}

A linear space with a norm is a normed linear space. The distance $\varrho(x, y) = \|x - y\|$ is a metric defining the norm topology.
