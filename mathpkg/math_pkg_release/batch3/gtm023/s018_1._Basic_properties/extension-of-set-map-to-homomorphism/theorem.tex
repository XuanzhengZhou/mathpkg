Let $\varphi_0: S \to B$ be an arbitrary set map from a system of generators $S$ of an algebra $A$ into an algebra $B$. Then $\varphi_0$ can be extended to a homomorphism $\varphi: A \to B$ if and only if
$$
\sum_{(v)} \lambda^{v_1 \dots v_p} \varphi_0 x_{v_1} \dots \varphi_0 x_{v_p} = 0 \quad \text{whenever} \quad \sum_{(v)} \lambda^{v_1 \dots v_p} x_{v_1} \dots x_{v_p} = 0.
$$
