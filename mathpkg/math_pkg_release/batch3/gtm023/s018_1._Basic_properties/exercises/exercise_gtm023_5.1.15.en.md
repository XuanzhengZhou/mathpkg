Define an equivalence relation in the set of all linearly independent $n^2$-tuples $(\varphi_1, \ldots, \varphi_{n^2})$, $\varphi_v \in L(E; E)$, in the following way: $(\varphi_1, \ldots, \varphi_{n^2}) \sim (\psi_1, \ldots, \psi_{n^2})$ if and only if there exists an element $\chi \in GL(E)$ such that $\psi_v = \chi \varphi_v \chi^{-1}$ for all $v$. Show that each equivalence class corresponds to a basis of $L(E; E)$ of the type described in Problem 14.
