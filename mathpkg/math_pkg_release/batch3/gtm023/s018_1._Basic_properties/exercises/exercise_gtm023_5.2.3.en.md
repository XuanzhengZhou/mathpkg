Let $S$ be the set of step functions $f: [0,1] \to \mathbb{R}$. A function $f$ is a step function if there exists a decomposition $0 = t_0 < t_1 < \cdots < t_n = 1$ such that $f$ is constant on each interval $t_{i-1} < t < t_i$ ($i = 1, \ldots, n$). Define addition and multiplication pointwise. Show that this makes $S$ into a commutative associative algebra with identity.
