For a linear mapping $\varphi: E \to F$, the \textbf{kernel} (or nullspace) is
$$\ker \varphi = \{x \in E \mid \varphi x = 0\},$$
and the \textbf{image} (or range) is
$$\operatorname{Im} \varphi = \{\varphi x \mid x \in E\}.$$
$\ker \varphi$ is a subspace of $E$ and $\operatorname{Im} \varphi$ is a subspace of $F$. $\varphi$ is injective if and only if $\ker \varphi = \{0\}$, and surjective if and only if $\operatorname{Im} \varphi = F$.
