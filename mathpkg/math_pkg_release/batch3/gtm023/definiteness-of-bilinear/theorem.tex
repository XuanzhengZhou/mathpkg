A symmetric bilinear function $\Phi$ is called positive definite if $\Phi(x) = \Phi(x, x) > 0$ for all $x \neq 0$. It is positive semidefinite if $\Phi(x) \geq 0$ for all $x$.

A positive semidefinite bilinear function that is not positive definite is necessarily degenerate: if $\Phi(x_0) = 0$ with $x_0 \neq 0$, then the Schwarz inequality gives $\Phi(x_0, y)^2 \leq \Phi(x_0)\Phi(y) = 0$, so $\Phi(x_0, y) = 0$ for all $y$.

Negative definite and negative semidefinite functions are defined analogously. $\Phi$ is indefinite if $\Phi(x)$ takes both positive and negative values.
