Let $\varphi: E \times F \rightarrow \Gamma$ be a bilinear function which respects $G$-gradations. Then $\varphi$ is non-degenerate if and only if each $\varphi_k = \varphi|_{E_k \times F_k}$ is non-degenerate. Consequently, a scalar product which respects $G$-gradations determines a scalar product between each pair $(E_k, F_k)$, making $E_k$ and $F_k$ dual.
