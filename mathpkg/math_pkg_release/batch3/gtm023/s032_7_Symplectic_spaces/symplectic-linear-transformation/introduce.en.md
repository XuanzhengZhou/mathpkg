A symplectic linear transformation (or linear symplectomorphism) is a linear map that preserves the symplectic inner product. In matrix terms, relative to a symplectic basis, a symplectic transformation is represented by a symplectic matrix $M$ satisfying $M^{\mathsf{T}} J M = J$, where $J = \begin{pmatrix} 0 & I \\ -I & 0 \end{pmatrix}$. The set of all symplectic linear transformations of a $2n$-dimensional symplectic space forms the symplectic group $\mathrm{Sp}(2n, \Gamma)$, which is a classical Lie group. Symplectic transformations arise naturally in Hamiltonian mechanics as the linear part of canonical transformations.
