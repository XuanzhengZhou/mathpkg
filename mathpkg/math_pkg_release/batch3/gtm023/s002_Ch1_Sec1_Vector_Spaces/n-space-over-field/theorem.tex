The set $\Gamma^n = \underbrace{\Gamma \times \cdots \times \Gamma}_{n}$ of $n$-tuples $x = (\xi^1, \ldots, \xi^n)$ with addition
$$(\xi^1, \ldots, \xi^n) + (\eta^1, \ldots, \eta^n) = (\xi^1 + \eta^1, \ldots, \xi^n + \eta^n)$$
and scalar multiplication
$$\lambda(\xi^1, \ldots, \xi^n) = (\lambda \xi^1, \ldots, \lambda \xi^n)$$
is a vector space, called the **$n$-space over $\Gamma$**.
