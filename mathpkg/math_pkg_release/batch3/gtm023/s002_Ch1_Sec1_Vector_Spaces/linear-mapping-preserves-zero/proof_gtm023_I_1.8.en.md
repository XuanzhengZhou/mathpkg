From additivity: $\varphi(0) = \varphi(0+0) = \varphi(0) + \varphi(0)$, hence $\varphi(0) = 0$ by subtracting $\varphi(0)$ from both sides.

If $\sum_i \lambda^i x_i = 0$ is a linear relation, then applying $\varphi$ and using preservation of linear combinations,
$$\sum_i \lambda^i \varphi x_i = \varphi\left(\sum_i \lambda^i x_i\right) = \varphi(0) = 0.$$
