Given the basis transformation

$$\bar{x}_1 = 2x_1 - x_2 - x_3$$
$$\bar{x}_2 = -x_2$$
$$\bar{x}_3 = 2x_2 + x_3$$

find all the vectors which have the same components with respect to the bases $x_\mu$ and $\bar{x}_\mu$ ($\mu = 1, 2, 3$).
