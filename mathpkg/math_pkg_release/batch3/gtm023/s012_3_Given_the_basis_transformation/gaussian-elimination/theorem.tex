Let

$$\alpha_1^1 \xi^1 + \cdots + \alpha_n^1 \xi^n = \eta^1$$
$$\vdots$$
$$\alpha_1^m \xi^1 + \cdots + \alpha_n^m \xi^n = \eta^m$$

be a system of $m$ linear equations in $n$ unknowns. Then, after possibly reordering equations and unknowns, the system can be reduced by successive elimination to an equivalent triangular system whose solution has the form

$$\xi^v = \sum_{\mu=r+1}^{n} \lambda_\mu^v \xi^\mu + \varrho^v \quad (v = 1, \dots, r)$$

where the $\xi^v\ (v = r+1, \dots, n)$ are arbitrary parameters, provided no contradiction arises during the elimination. If a contradiction occurs, the system has no solution.
