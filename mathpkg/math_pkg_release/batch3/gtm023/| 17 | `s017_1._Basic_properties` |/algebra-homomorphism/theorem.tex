Suppose $B$ is a second algebra. Then a linear mapping $\varphi: A \rightarrow B$ is called a homomorphism (of algebras) if $\varphi$ preserves products; i.e.,

$$\varphi(x y) = \varphi x \cdot \varphi y.$$

A homomorphism that is injective (resp. surjective, bijective) is called a monomorphism (resp. epimorphism, isomorphism). If $B = A$, $\varphi$ is called an endomorphism.
