Let $\varphi : A \rightarrow B$ be any homomorphism of associative algebras and $S$ be a system of generators for $A$. Then $\varphi$ determines a set map $\varphi_0 : S \rightarrow B$ by

$$\varphi_0 x = \varphi x, \quad x \in S.$$

The homomorphism $\varphi$ is completely determined by $\varphi_0$.
