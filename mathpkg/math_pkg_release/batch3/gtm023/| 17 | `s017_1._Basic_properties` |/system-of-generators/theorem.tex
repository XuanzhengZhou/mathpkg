If $A$ is an associative algebra, a subset $S \subset A$ is called a system of generators of $A$ if each vector $x \in A$ is a linear combination of products of elements in $S$,

$$x = \sum_{(v)} \lambda^{v_1 \dots v_p} x_{v_1} \dots x_{v_p}, \quad x_{v_i} \in S, \lambda^{v_1 \dots v_p} \in \Gamma.$$
