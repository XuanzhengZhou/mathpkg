From formula (2.42) and (2.48) we obtain that

$$\operatorname{Im} \varphi^* \subset (\operatorname{Im} \varphi^*)^\perp = (\ker \varphi)^\perp.$$

On the other hand, let $f \in (\ker \varphi)^\perp$. Then $f$ vanishes on $\ker \varphi$, so $f$ induces a linear function $\bar{f}$ on $E/\ker \varphi$. Since $\varphi$ induces an isomorphism $E/\ker \varphi \cong \operatorname{Im} \varphi$, $\bar{f}$ corresponds to a linear function $g$ on $\operatorname{Im} \varphi$. Extend $g$ to a linear function $\tilde{g}$ on $F$. Then $\varphi^* \tilde{g} = f$, showing that $(\ker \varphi)^\perp \subset \operatorname{Im} \varphi^*$. Hence equality holds.
