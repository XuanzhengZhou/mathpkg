It is easy to verify that the correspondence $y^* \mapsto \varphi^* y^*$ defined by

$$(\varphi^* y^*)(x) = \langle y^*, \varphi x \rangle$$

determines a linear mapping. Moreover, the relation

$$\langle \varphi^* y^*, x \rangle = (\varphi^* y^*)(x) = \langle y^*, \varphi x \rangle$$

shows that $\varphi^*$ is dual to $\varphi$. If $F^* = L(F)$ as well, then for $f \in L(F)$ we have

$$(\varphi^* f)(x) = \langle f, \varphi x \rangle = f(\varphi x) = (f \circ \varphi)(x),$$

so $\varphi^* f = f \circ \varphi$.
