A determinant function in an $n$-dimensional vector space $E$ is a skew symmetric $n$-linear function from $E$ to $\Gamma$.

In every $n$-dimensional vector space $E$ ($n \geq 1$) there exist non-zero determinant functions.

**Proposition III:** Let $E$ be an $n$-dimensional vector space and fix a non-zero determinant function $\Delta$ in $E$. Then every skew symmetric $n$-linear map $\varphi$ from $E$ to a vector space $F$ determines a unique vector $b \in F$ such that
$$\varphi(x_1, \ldots, x_n) = \Delta(x_1, \ldots, x_n) \cdot b.$$

**Corollary:** Every determinant function is a scalar multiple of $\Delta$. Thus the space of determinant functions is 1-dimensional.
