Let $E$ be a real vector space and $\Phi$ be a bilinear function in $E \times E$. The bilinear function $\Phi$ is called symmetric if
$$\Phi(x, y) = \Phi(y, x) \quad x, y \in E.$$

Given a symmetric bilinear function $\Phi$, the (non-linear) function $\Psi$ defined by $\Psi(x) = \Phi(x, x)$ is called the associated quadratic function. Then $\Phi$ is uniquely determined by $\Psi$ via the polarization identity:
$$\Phi(x, y) = \frac{1}{2} \left\{ \Psi(x + y) - \Psi(x) - \Psi(y) \right\}.$$
