A symmetric bilinear function $\Phi$ on a real vector space satisfies $\Phi(x, y) = \Phi(y, x)$. Every symmetric bilinear function determines a quadratic function $\Psi(x) = \Phi(x, x)$, and conversely $\Phi$ can be recovered from $\Psi$ via the polarization identity. In coordinates, $\Phi$ is represented by a symmetric matrix $\alpha_{v\mu} = \Phi(x_v, x_\mu)$.
