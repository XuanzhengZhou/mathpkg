Show $\psi \mapsto \exp \psi$ has properties: 1. $\exp(\psi_1 + \psi_2) = \exp \psi_1 \circ \exp \psi_2$ if $\psi_2 \circ \psi_1 = \psi_1 \circ \psi_2$. 2. $\exp(-\psi) = (\exp \psi)^{-1}$. 3. $\exp 0 = \iota$. 4. $\widetilde{\exp \psi} = \exp \tilde{\psi}$. 5. $\det \exp \psi = e^{\operatorname{tr} \psi}$. Conclude $\exp \psi$ is selfadjoint if $\psi$ is selfadjoint, and a proper rotation if $\psi$ is skew.
