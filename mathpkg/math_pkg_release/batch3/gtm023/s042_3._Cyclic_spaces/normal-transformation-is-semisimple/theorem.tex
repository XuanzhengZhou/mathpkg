Let $E$ be an inner product space. Then a linear transformation $\varphi$ is normal if and only if
\begin{enumerate}
  \item[(i)] the generalized eigenspaces are mutually orthogonal,
  \item[(ii)] the restriction of $\varphi$ to each generalized eigenspace is homothetic.
\end{enumerate}
In particular, a normal transformation is semisimple.
