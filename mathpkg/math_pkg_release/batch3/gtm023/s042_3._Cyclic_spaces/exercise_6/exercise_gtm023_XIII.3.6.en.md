Define a linear transformation $\varphi$ of $E$ by
$$\varphi = \varphi_1 \oplus \cdots \oplus \varphi_s.$$

a) Prove that $E$ is cyclic if and only if each $\varphi_j$ is cyclic and the $\mu_j$ are relatively prime.

b) Conclude that if $E$ is cyclic, then each $F_j$ is a sum of generalized eigenspaces for $\varphi$.

c) Prove that if $E$ is cyclic and $a = a_1 + \cdots + a_s$ with $a_j \in F_j$, then $a$ generates $E$ if and only if $a_j$ generates $F_j$ ($j=1, \ldots, s$).
