Let $E$ be a real vector space which is irreducible with respect to $\varphi$. Then the polynomial $f$ in $\mu = f^k$ has one of two forms:
\begin{enumerate}
  \item $f = t - \lambda$ (real eigenvalue case), giving Jordan blocks of the form $(13.39)$;
  \item $f = t^2 + \alpha t + \beta$ with $\alpha^2 - 4\beta < 0$ (complex conjugate pair), giving $2 \times 2$ blocks
  $$A_j = \begin{pmatrix} 0 & 1 \\ -\beta & -\alpha \end{pmatrix}.$$
\end{enumerate}
