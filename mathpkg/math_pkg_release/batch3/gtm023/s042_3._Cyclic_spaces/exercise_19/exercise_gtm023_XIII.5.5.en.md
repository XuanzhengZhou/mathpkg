a) Define $C^3(\varphi)$ to be the set of all linear transformations in $E$ commuting with every transformation in $C^2(\varphi)$. Prove that
$$C^3(\varphi) = C(\varphi).$$

b) Prove that $C^2(\varphi) = C(\varphi)$ if and only if $E$ is cyclic.
