Let an affine mapping $x' = \tau x + b$ of the affine space $A$ onto itself be given. Then the image of a quadric
$$Q: \Phi(x) + 2f(x) = \alpha$$
is the quadric $Q'$ defined by the equation
$$Q': \Psi(x) + 2g(x) = \beta,$$
where
$$\Psi(x) = \Phi(\tau^{-1}x),$$
$$g(x) = -\Phi(\tau^{-1}x, \tau^{-1}b) + f(\tau^{-1}x),$$
$$\beta = -\Phi(\tau^{-1}b) + 2f(\tau^{-1}b) + \alpha.$$
These relations satisfy
$$\Psi(\tau x + b) + 2g(\tau x + b) - \beta = \Phi(x) + 2f(x) - \alpha,$$
showing that a point $x \in A$ is contained in $Q$ if and only if the point $x' = \tau x + b$ is contained in $Q'$.
