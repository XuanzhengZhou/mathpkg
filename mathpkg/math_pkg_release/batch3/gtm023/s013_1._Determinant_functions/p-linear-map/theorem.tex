Let $E$ and $F$ be vector spaces over a field $\Gamma$. A map $\Phi: E^p \to F$ is called \text{$p$-linear} if it is linear with respect to each argument; i.e., for all $\lambda, \mu \in \Gamma$,
$$\Phi(x_1, \ldots, \lambda x_i + \mu y_i, \ldots, x_p) = \lambda \Phi(x_1, \ldots, x_i, \ldots, x_p) + \mu \Phi(x_1, \ldots, y_i, \ldots, x_p).$$
