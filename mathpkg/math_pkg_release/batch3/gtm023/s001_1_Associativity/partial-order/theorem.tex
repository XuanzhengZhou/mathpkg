Let $A$ be a set and assume that for some pairs $X, Y$ ($X \in A, Y \in A$) a relation, denoted by $X \leq Y$, is defined which satisfies the following conditions:

\begin{itemize}
  \item[(i)] $X \leq X$ for every $X \in A$ \quad (Reflexivity)
  \item[(ii)] If $X \leq Y$ and $Y \leq X$ then $X = Y$ \quad (Antisymmetry)
  \item[(iii)] If $X \leq Y$ and $Y \leq Z$, then $X \leq Z$ \quad (Transitivity).
\end{itemize}

Then $\leq$ is called a \textbf{partial order} in $A$.
