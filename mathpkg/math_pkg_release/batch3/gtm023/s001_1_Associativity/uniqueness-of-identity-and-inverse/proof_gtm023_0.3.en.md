If $e$ and $e'$ are both identity elements, then $e = ee' = e'$, so the identity is unique. If $x^{-1}$ and $x'$ are both inverses of $x$, then
$$x^{-1} = x^{-1}e = x^{-1}(xx') = (x^{-1}x)x' = ex' = x',$$
so the inverse is unique.
