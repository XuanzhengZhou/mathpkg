An upper bound of a subset within a partially ordered set is an element that is greater than or equal to every element of the subset. Upper bounds need not belong to the subset itself and need not be unique. The concept is essential in Zorn's lemma, which states that if every chain has an upper bound, then the partially ordered set contains a maximal element.
