A maximal element of a partially ordered set is an element that is not strictly less than any other element. Unlike a greatest element, a maximal element need not be comparable to all elements; there may be elements incomparable to it. The distinction between maximal and greatest elements is crucial in applications of Zorn's lemma, which guarantees the existence of maximal elements under certain conditions.
