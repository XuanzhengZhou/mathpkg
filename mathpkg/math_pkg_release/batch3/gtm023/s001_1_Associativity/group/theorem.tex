A \textbf{group} is a set $G$, together with a binary law of composition
$$\mu: G \times G \rightarrow G$$
which satisfies the following axioms ($\mu(x, y)$ will be denoted by $xy$):

\begin{enumerate}
  \item \textbf{Associativity:} $(xy)z = x(yz)$
  \item \textbf{Identity:} There exists an element $e$, called the identity, such that
  $$xe = ex = x.$$
  \item \textbf{Inverse:} To each element $x \in G$ corresponds a second element $x^{-1}$ such that
  $$xx^{-1} = x^{-1}x = e.$$
\end{enumerate}
