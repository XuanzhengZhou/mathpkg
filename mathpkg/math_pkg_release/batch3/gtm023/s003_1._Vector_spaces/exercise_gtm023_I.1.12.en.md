Consider the set of polynomial functions $f: \mathbb{R} \to \mathbb{R}$,

$$f(x) = \sum_{i=0}^{n} \alpha_i x^i.$$

Make this set into a vector space as in Example 3 (mappings from a set to a vector space), and construct a natural basis.
