A linear mapping (or linear transformation) is a function between vector spaces that preserves the vector space structure: it respects addition and scalar multiplication. Equivalently, it preserves all finite linear combinations. Linear mappings are the structure-preserving maps (homomorphisms) in the category of vector spaces. A linear mapping from a vector space to its coefficient field is called a linear function or linear form. Every linear mapping sends the zero vector to the zero vector, and linear relations among domain vectors are preserved in the codomain.
