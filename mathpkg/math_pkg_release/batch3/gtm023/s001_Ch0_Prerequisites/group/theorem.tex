A **group** is a set $G$, together with a binary operation $\mu: G \times G \to G$ (where $\mu(x,y)$ is denoted $xy$), satisfying three axioms:
1. **Associativity:** $(xy)z = x(yz)$ for all $x,y,z \in G$.
2. **Identity:** There exists an element $e \in G$ (the identity) such that $xe = ex = x$ for all $x \in G$.
3. **Inverse:** For each $x \in G$, there exists $x^{-1} \in G$ such that $xx^{-1} = x^{-1}x = e$.
