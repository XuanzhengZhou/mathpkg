Consider a linear transformation $\varphi$ of an $n$-dimensional linear space $E$. A vector $a \neq 0$ of $E$ is called an eigenvector of $\varphi$ if
$$\varphi a = \lambda a.$$
The scalar $\lambda$ is called the corresponding eigenvalue.

$\lambda$ is an eigenvalue of $\varphi$ if and only if
$$\det(\varphi - \lambda\iota) = 0.$$
A linear transformation need not have eigenvectors; for example, a rotation in $\mathbb{R}^2$ by $90^\circ$ has no real eigenvectors.
