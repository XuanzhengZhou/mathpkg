A partial order is a binary relation satisfying reflexivity, antisymmetry, and transitivity. A chain (or linearly ordered set) is a partially ordered set where every pair of elements is comparable. An upper bound of a subset is an element greater than or equal to every element of the subset. A maximal element has no strictly larger element in the order.
