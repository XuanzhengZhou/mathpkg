A group is a set $G$, together with a binary law of composition

$$\mu: G \times G \rightarrow G$$

which satisfies the following axioms ($\mu(x, y)$ will be denoted by $xy$):

\begin{enumerate}
\item Associativity: $(xy)z = x(yz)$
\item Identity: There exists an element $e$, called the identity such that
$$xe = ex = x.$$
\item Inverse: To each element $x \in G$ corresponds a second element $x^{-1}$ such that
$$xx^{-1} = x^{-1}x = e.$$
\end{enumerate}

The identity element of a group is uniquely determined and each element has a unique inverse. We also have the relation

$$(xy)^{-1} = y^{-1}x^{-1}.$$
