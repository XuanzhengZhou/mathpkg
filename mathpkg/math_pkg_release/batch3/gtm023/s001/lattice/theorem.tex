Let $A$ be a partially ordered set and let $A_1 \subset A$ be a subset. An element $A \in A$ is called a least upper bound (l.u.b.) for $A_1$ if

\begin{enumerate}
\item $A$ is an upper bound for $A_1$.
\item If $B$ is any upper bound for $A_1$, then $A \leq B$.
\end{enumerate}

A lattice is a partially ordered set in which every finite subset has a least upper bound and a greatest lower bound.
