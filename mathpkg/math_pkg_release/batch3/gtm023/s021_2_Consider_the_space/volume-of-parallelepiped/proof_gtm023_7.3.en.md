Let \(\Delta_1\) be a normed determinant function on the \(p\)-dimensional subspace \(E_1\) generated by \(a_1, \ldots, a_p\). By definition of the volume,

$$V(a_1, \ldots, a_p) = |\Delta_1(a_1, \ldots, a_p)|.$$

Squaring both sides and using the identity (7.23) for normed determinant functions,

$$V(a_1, \ldots, a_p)^2 = \Delta_1(a_1, \ldots, a_p)^2 = \det\big((a_i, a_j)\big) = G(a_1, \ldots, a_p).$$

For the orthogonal component formula, fix an index \(i\) and decompose

$$a_i = \sum_{\nu \neq i} \xi^{\nu} a_{\nu} + h_i, \quad \text{where} \quad (h_i, a_{\nu}) = 0 \;\; (\nu \neq i).$$

By the multilinearity of the determinant function and the fact that \(h_i\) is orthogonal to all \(a_{\nu}\) (\(\nu \neq i\)),

$$V(a_1, \ldots, a_p) = |\Delta_1(a_1, \ldots, a_{i-1}, h_i, a_{i+1}, \ldots, a_p)|,$$

since the \(\sum_{\nu \neq i} \xi^{\nu} a_{\nu}\) term does not contribute (it is a linear combination of columns already present).
