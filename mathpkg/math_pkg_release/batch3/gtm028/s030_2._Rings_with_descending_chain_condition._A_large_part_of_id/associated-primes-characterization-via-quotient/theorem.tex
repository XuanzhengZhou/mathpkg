Let $\mathfrak{a} = \bigcap_i \mathfrak{q}_i$ be an irredundant primary representation of an ideal $\mathfrak{a}$ in a noetherian ring $R$, and let $\mathfrak{p}_i = \sqrt{\mathfrak{q}_i}$. Then the set $\{\mathfrak{p}_i\}$ is exactly the set of prime ideals $\mathfrak{p}$ for which there exists an element $c \in R$, $c \notin \mathfrak{a}$, such that $\mathfrak{a} : (c)$ is primary for $\mathfrak{p}$. In particular, the set of associated prime ideals of $\mathfrak{a}$ is independent of the particular irredundant primary decomposition.
