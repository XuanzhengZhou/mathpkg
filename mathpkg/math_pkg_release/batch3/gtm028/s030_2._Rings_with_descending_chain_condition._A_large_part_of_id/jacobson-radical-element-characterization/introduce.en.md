This lemma provides a useful characterization of the Jacobson radical: an element lies in the intersection of all maximal ideals precisely when $1 + xa$ is invertible for all ring elements $x$. The forward direction uses the maximal ideal test, while the converse argues by contradiction using the existence of a maximal ideal not containing $a$.
