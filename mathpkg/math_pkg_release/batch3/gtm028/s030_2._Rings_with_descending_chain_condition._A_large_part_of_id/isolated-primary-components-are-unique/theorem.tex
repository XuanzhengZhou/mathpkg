Let $\mathfrak{a} = \bigcap_i \mathfrak{a}_i$ be an irredundant primary representation of an ideal $\mathfrak{a}$ in a noetherian ring $R$, and let $\mathfrak{p}_i = \sqrt{\mathfrak{a}_i}$. For each $i$, define $\mathfrak{a}'_i$ as the set of all elements $x \in R$ such that there exists $\pi \notin \mathfrak{p}_i$ with $\pi x \in \mathfrak{a}$. Then $\mathfrak{a}'_i$ is an ideal of $R$ contained in $\mathfrak{a}_i$. If $\mathfrak{a}_i$ is an isolated primary component of $\mathfrak{a}$, then $\mathfrak{a}'_i = \mathfrak{a}_i$. Hence the isolated primary components of $\mathfrak{a}$ are uniquely determined.
