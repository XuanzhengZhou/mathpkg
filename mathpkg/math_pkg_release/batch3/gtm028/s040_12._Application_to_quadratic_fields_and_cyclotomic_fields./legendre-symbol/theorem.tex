For every integer $s$ which is not a multiple of the odd prime $p$, the \textbf{Legendre symbol} $\left(\frac{s}{p}\right)$ is defined as:
$$\left(\frac{s}{p}\right) = 
\begin{cases}
+1 & \text{if the $p$-residue of $s$ is a square in } \mathbb{Z}/(p),\\
-1 & \text{if the $p$-residue of $s$ is not a square in } \mathbb{Z}/(p).
\end{cases}$$
