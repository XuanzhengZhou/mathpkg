The order of an element $a$ in a group $G$ is defined as the order (cardinality) of the cyclic subgroup generated by $a$. If this subgroup is finite with $m$ elements, then $m$ is the order of $a$; otherwise, $a$ has infinite order. The order of an element captures how many distinct powers $a, a^2, a^3, \ldots$ exist before repetition occurs, and by Lagrange's theorem, the order of every element in a finite group divides the order of the group.
