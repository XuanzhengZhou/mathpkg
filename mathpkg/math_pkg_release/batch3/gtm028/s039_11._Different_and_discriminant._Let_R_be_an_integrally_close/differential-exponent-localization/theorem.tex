\text{Let } R \text{ be a Dedekind domain, } R' \text{ its integral closure in a finite algebraic separable extension } K' \text{ of the quotient field } K \text{ of } R. \text{ Let } \mathfrak{p} \text{ be a proper prime ideal in } R, M = R \setminus \mathfrak{p}, \text{ and } \mathfrak{P} \text{ a prime ideal in } R' \text{ lying over } \mathfrak{p} \text{ with differential exponent } m(\mathfrak{P}). \text{ Then the differential exponent of } \mathfrak{P}R'_M \text{ over } R_M \text{ is } m(\mathfrak{P}). \text{ Hence the different } \mathfrak{D}_{R'_M/R_M} = \mathfrak{D}_{R'/R} R'_M.
