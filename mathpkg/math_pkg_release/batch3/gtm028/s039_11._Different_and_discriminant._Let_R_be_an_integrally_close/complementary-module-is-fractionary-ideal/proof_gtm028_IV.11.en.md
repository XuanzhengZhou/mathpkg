By definition of a fractionary ideal, we need only show that $\mathfrak{C}$ is contained in a finite $R'$-module, and for this it will be sufficient to see that it is contained in a finite $R$-module.

Since $K' = R'_R$ (see proof of Theorem 7, §4), there exists a $K$-basis $\{e_1, \ldots, e_n\}$ of $K'$ all the elements of which are in $R'$. We take an element $z$ of $\mathfrak{C}$, and write $z = \sum_i a_i e_i$ with $a_i \in K$. We have

$$T(z e_j) = \sum_i a_i T(e_i e_j) \in R$$

for $j = 1, \ldots, n$. As $K'$ is separable over $K$, the determinant

$$d = \det(T(e_i e_j))$$

is nonzero. Let $(b_{ij})$ be the inverse matrix of $(T(e_i e_j))$, so that $\sum_j T(e_i e_j) b_{jk} = \delta_{ik}$. The entries $b_{jk}$ lie in $K$. Solving the linear system above yields

$$a_i = \sum_j T(z e_j) b_{ji} \in \sum_j R \cdot b_{ji}.$$

Hence each $a_i$ belongs to the $R$-module generated by the $b_{ji}$. Therefore $z = \sum_i a_i e_i$ belongs to the $R$-module generated by the finite set $\{b_{ji} e_i\}$. This shows that $\mathfrak{C}$ is contained in a finite $R$-module, and consequently $\mathfrak{C}$ is a fractionary ideal of $R'$.
