\text{Let } R \text{ be an integrally closed domain, } K \text{ its quotient field, } K' \text{ a finite algebraic separable extension of } K, \text{ and } R' \text{ the integral closure of } R \text{ in } K'. \text{ Let } \mathfrak{p} \text{ be a prime ideal of } R, \mathfrak{P} \text{ a prime ideal of } R' \text{ lying over } \mathfrak{p}. \text{ Denote by } h : R' \to R'/\mathfrak{P} \text{ the canonical homomorphism. Extend } h \text{ to a homomorphism of a suitable extension of } K' \text{ onto a suitable extension of the field } k = R/\mathfrak{p}. \text{ Then:}

$$h(T_{K'/K}(x)) = \sum_i e_i \cdot T_{k_i/k}(h_i(x)),$$
$$h(N_{K'/K}(x)) = \prod_i (N_{k_i/k}(h_i(x)))^{e_i},$$

\text{where } k_i = R'/\mathfrak{P}_i \text{ and } e_i \text{ are the ramification indices.}
