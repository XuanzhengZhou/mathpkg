\text{Let } R \text{ be an integrally closed domain, } K \text{ its quotient field, and } K' = K(y) \text{ a finite algebraic separable extension of } K \text{ of degree } n. \text{ Let } F(X) \text{ be the minimal polynomial of } y \text{ over } K \text{ and let } R' = R[y] \text{ be the ring generated by } y \text{ over } R. \text{ Then the complementary module } \mathcal{C} \text{ of } R' \text{ over } R \text{ is } R' \cdot (1/F'(y)), \text{ so that the different is } \mathfrak{D}_{R'/R} = R' \cdot F'(y).
