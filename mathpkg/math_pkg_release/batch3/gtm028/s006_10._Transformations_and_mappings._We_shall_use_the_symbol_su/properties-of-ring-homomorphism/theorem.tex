**Theorem 3.** If $T$ is a homomorphism of a ring $R$ into a ring $\bar{R}$, then

(a) $0T = \bar{0}$ and $(-a)T = -(aT)$, for any element $a$ in $R$;

(b) $RT$ is a subring of $\bar{R}$;

(c) the kernel $N$ of $T$ is a subring of $R$;

(d) $T$ is an isomorphism if and only if the kernel $N$ of $T$ contains only the element $0$ of $R$.
