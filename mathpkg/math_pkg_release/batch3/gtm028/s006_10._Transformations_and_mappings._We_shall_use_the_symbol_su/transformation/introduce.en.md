A transformation from a set $S$ to a set $\bar{S}$ is a set-valued function: each element $a \in S$ is assigned a subset $aT \subseteq \bar{S}$, which may be empty. This generalizes the usual notion of a function by allowing multiple images (or none) for a single element. Transformations serve as the foundational building block for defining mappings, homomorphisms, and other structure-preserving correspondences throughout algebra.
