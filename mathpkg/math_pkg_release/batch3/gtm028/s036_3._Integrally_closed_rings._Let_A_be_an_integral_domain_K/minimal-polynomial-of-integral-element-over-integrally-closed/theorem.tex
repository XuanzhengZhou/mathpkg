\text{Let } A \text{ be an integrally closed domain, } K \text{ its quotient field, and } x \text{ an element integral over } A \text{ of some overring of } K. \text{ Let } m(X) \text{ be the minimal polynomial of } x \text{ over } K. \text{ Then } m(X) \text{ has all its coefficients in } A; \text{ in other words, the relation } m(x) = 0 \text{ is an integral dependence equation for } x \text{ over } A.
