\text{Let } A \text{ be an integrally closed domain, and } A' \text{ an overring of } A \text{ integral over } A \text{ and such that no non-zero element of } A \text{ is a zero-divisor in } A'. \text{ If } \mathfrak{p} \subset \mathfrak{q} \text{ are prime ideals in } A, \text{ and } \mathfrak{p}' \text{ is a prime ideal in } A' \text{ lying over } \mathfrak{p}, \text{ then there exists a prime ideal } \mathfrak{q}' \text{ in } A' \text{ lying over } \mathfrak{q} \text{ such that } \mathfrak{p}' \subset \mathfrak{q}'.
