\text{Let } A \text{ be an integral domain, } K \text{ its quotient field, and } x \text{ an element of some extension of } K. \text{ If } x \text{ is integral over } A, \text{ then } x \text{ is algebraic over } K. \text{ Let } f(X) = \prod_{i=1}^{n} (X - x_i) \text{ be the minimal polynomial of } x \text{ over } K, \text{ where } \{x_1, \dots, x_n\} \text{ is a complete set of conjugates of } x \text{ over } K. \text{ Then the coefficients of } f(X) \text{ are integral over } A. \text{ In particular, if } A \text{ is integrally closed, then } f(X) \in A[X].
