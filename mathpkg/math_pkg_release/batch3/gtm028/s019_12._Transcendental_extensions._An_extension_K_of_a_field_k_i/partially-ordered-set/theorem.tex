A set $S$ is said to be \textit{partially ordered} if there is given in $S$ a binary relation $\prec$ which is defined for certain pairs $(a, b)$ of elements of $S$ (it is not necessary that the relation $\prec$ be defined for all pairs $(a, b)$ of elements of $S$) and which satisfies the following conditions:
\begin{enumerate}
\item[(1)] $a \prec a$ for any element $a$ of $S$;
\item[(2)] if $a \prec b$ and $b \prec a$, then $a = b$;
\item[(3)] if $a \prec b$ and $b \prec c$, then $a \prec c$.
\end{enumerate}
