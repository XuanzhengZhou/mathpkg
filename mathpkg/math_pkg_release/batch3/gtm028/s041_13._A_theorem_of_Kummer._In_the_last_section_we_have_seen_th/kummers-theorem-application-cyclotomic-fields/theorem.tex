\text{For the cyclotomic field } \mathbb{Q}(\zeta_p) \text{ where } \zeta_p \text{ is a primitive } p\text{-th root of unity (} p \text{ an odd prime), an integral basis is } \{1, \zeta_p, \dots, \zeta_p^{p-2}\}. \text{ The minimal polynomial is the cyclotomic polynomial } F(X) = X^{p-1} + X^{p-2} + \cdots + X + 1.

\text{The only ramified prime is } p. \text{ For a prime } q \neq p, \text{ the decomposition of } q \text{ is determined by the factorization of } X^{p-1} + \cdots + X + 1 \text{ (equivalently } X^p - 1\text{) over } \mathbb{F}_q. \text{ The polynomial factors into } (p-1)/f \text{ distinct irreducible factors of degree } f, \text{ where } f \text{ is the order of } q \text{ modulo } p.
