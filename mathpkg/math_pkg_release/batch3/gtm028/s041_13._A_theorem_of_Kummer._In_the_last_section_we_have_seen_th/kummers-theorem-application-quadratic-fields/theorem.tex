\text{For a quadratic field } \mathbb{Q}(\sqrt{m}) \text{ (with } m \text{ a squarefree integer):}

\text{Case 1: } m \equiv 2,3 \pmod{4}. \text{ Integral basis } \{1, \sqrt{m}\}, \text{ minimal polynomial } F(X) = X^2 - m.

\text{Case 2: } m \equiv 1 \pmod{4}. \text{ Integral basis } \{1, \tfrac{1}{2}(1+\sqrt{m})\}, \text{ minimal polynomial } F(X) = X^2 - X - \tfrac{m-1}{4}.

\text{For a prime } p, \text{ the decomposition of } p \text{ in the ring of integers is determined by the factorization of } X^2 - m \pmod{p} \text{ (or the corresponding polynomial) via Kummer's Theorem.}
