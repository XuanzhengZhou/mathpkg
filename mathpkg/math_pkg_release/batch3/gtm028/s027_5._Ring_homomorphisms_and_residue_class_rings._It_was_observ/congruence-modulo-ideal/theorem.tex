If $N$ is an arbitrary ideal in a ring $R$, and $a, b$ are elements of $R$, then $a \equiv b \pmod{N}$ (read: $a$ is congruent to $b$ mod $N$) if $a - b \in N$. This congruence is an equivalence relation and has the following properties: if $a \equiv b \pmod{N}$ and $c \equiv d \pmod{N}$, then $a + c \equiv b + d \pmod{N}$ and $ac \equiv bd \pmod{N}$. It follows that if $a \equiv b \pmod{N}$ and if $f(X)$ is a polynomial over $R$, then $f(a) \equiv f(b) \pmod{N}$.
