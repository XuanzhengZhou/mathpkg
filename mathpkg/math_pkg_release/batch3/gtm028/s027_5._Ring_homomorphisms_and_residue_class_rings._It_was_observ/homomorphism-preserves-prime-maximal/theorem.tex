Let $T$ be a homomorphism of a ring $R$ onto a ring $R'$ with kernel $N$. If $\mathfrak{A}$ is an ideal in $R$ containing $N$, then $\mathfrak{A}$ is respectively prime or maximal if and only if $\mathfrak{A}T$ is prime or maximal. If $\mathfrak{A}'$ is an ideal in $R'$, then $\mathfrak{A}'$ is respectively prime or maximal if and only if $\mathfrak{A}'T^{-1}$ is prime or maximal.
