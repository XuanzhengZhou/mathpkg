Submodules $M_1, \cdots, M_r$ of an $R$-module $M$ are said to be independent if
$$M_i \cap (M_1 + \cdots + M_{i-1} + M_{i+1} + \cdots + M_r) = (0)$$
for $i = 1, \cdots, r$. The module $M$ is said to be the direct sum of submodules $M_1, \cdots, M_r$ if it is the sum of these submodules and if these submodules are independent. We then write
$$M = M_1 \oplus \cdots \oplus M_r.$$
Equivalently, $M$ is the direct sum if and only if each $x \in M$ can be represented uniquely in the form $x = x_1 + \cdots + x_r$ with $x_i \in M_i$.
