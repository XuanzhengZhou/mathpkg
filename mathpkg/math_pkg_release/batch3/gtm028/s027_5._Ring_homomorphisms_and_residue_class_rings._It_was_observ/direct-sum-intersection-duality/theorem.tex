Suppose the $R$-module $M$ is the direct sum of submodules $M_1, \cdots, M_r$, so that

$$M = M_1 + \cdots + M_r,$$
$$M_i \cap (M_1 + \cdots + M_{i-1} + M_{i+1} + \cdots + M_r) = (0), \quad i = 1, \cdots, r.$$

If we place $N_i = M_1 + \cdots + M_{i-1} + M_{i+1} + \cdots + M_r$, then

$$(0) = N_1 \cap \cdots \cap N_r,$$
$$N_i + (N_1 \cap \cdots \cap N_{i-1} \cap N_{i+1} \cap \cdots \cap N_r) = M,$$
$$M_i = N_1 \cap \cdots \cap N_{i-1} \cap N_{i+1} \cap \cdots \cap N_r.$$

Conversely, given submodules $N_1, \cdots, N_r$ satisfying the first two intersection conditions and defining $M_i$ by the third, then $M = M_1 \oplus \cdots \oplus M_r$ and the $N_i$ are recovered as above.
