Let $R$ be an arbitrary ring and let $\mathfrak{Q}$ be an ideal in $R$, $\mathfrak{Q} \neq R$. Then $\mathfrak{Q}$ is said to be primary if $ab \in \mathfrak{Q}$ and $a \notin \mathfrak{Q}$ imply $b^m \in \mathfrak{Q}$ for some positive integer $m$. Equivalently, $\mathfrak{Q}$ is primary if every zero-divisor in $R/\mathfrak{Q}$ is nilpotent. If $\mathfrak{Q}$ is primary, then its radical $\mathfrak{P} = \sqrt{\mathfrak{Q}}$ is a prime ideal; $\mathfrak{Q}$ is then said to be primary for $\mathfrak{P}$ (or $\mathfrak{P}$-primary). If $\mathfrak{P}^m \subseteq \mathfrak{Q}$ for some $m$, then $\mathfrak{Q}$ is said to be strongly primary and the least $m$ is called the exponent of $\mathfrak{Q}$. A primary ideal has exponent $1$ if and only if it is prime.
