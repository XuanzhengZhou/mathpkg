A composition series of an $R$-module $M$ is a normal series without repetitions for which every proper refinement has repetitions. Equivalently, a normal series
$$M = M_0 > M_1 > M_2 > \cdots > M_r = (0)$$
without repetitions is a composition series if and only if there exist no $R$-submodules between $M_{i-1}$ and $M_i$ for $i = 1, \cdots, r$, i.e., each difference module $M_{i-1} - M_i$ is simple. Not every module has a composition series — for example, the additive group of integers.
