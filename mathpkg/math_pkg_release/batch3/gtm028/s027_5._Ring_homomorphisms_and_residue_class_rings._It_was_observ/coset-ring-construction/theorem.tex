Let $N$ be an ideal in a ring $R$. Define $\bar{R}$ to be the set of cosets $a + N$ ($a \in R$). Define addition of cosets by $(a+N) + (b+N) = (a+b)+N$ and multiplication by
$$(a + N)(b + N) = ab + N.$$
Then $\bar{R}$ is a ring and the mapping $T: R \to \bar{R}$ defined by $aT = a + N$ is a homomorphism of $R$ onto $\bar{R}$ with kernel $N$. Thus every ideal $N$ in $R$ is the kernel of some homomorphism of $R$.
