Let $M$ and $N$ be $R$-modules, with $M_i$ and $N_i$ submodules of $M$ and $N$ respectively, and let $T_i$ be an $R$-homomorphism of $M_i$ into $N_i$. Assume that
$$M = M_1 \oplus \cdots \oplus M_r, \quad N = N_1 + \cdots + N_r.$$
Then there exists one and only one $R$-homomorphism $T$ of $M$ into $N$ which coincides with $T_i$ on $M_i$. If each $T_i$ is onto, so is $T$. If each $T_i$ is an isomorphism and if the sum for $N$ is direct, then $T$ is an isomorphism.
