Let $R$ be a ring with identity and let $\mathfrak{A}$ be an ideal in $R$, $\mathfrak{A} \neq R$. Then:
\begin{itemize}
\item $\mathfrak{A}$ is prime if and only if $R/\mathfrak{A}$ is an integral domain (has no zero-divisors).
\item $\mathfrak{A}$ is maximal if and only if $R/\mathfrak{A}$ is a field.
\end{itemize}
Hence in a ring with identity, any maximal ideal is prime.
