Let $M$ be an $R$-module, $N$ a submodule. Then
$$l(M) = l(N) + l(M - N),$$
where $l(\cdot)$ denotes the length. This is to be interpreted as meaning, in particular, that if either side is infinite, so is the other. Moreover, if $L$ and $N$ are submodules of $M$, then
$$l(L) + l(N) = l(L + N) + l(L \cap N).$$
