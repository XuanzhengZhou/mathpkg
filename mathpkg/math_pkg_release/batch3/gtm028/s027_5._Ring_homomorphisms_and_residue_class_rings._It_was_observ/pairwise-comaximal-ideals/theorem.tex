Let $R$ be a ring with identity. Two ideals $\mathfrak{A}$ and $\mathfrak{B}$ are said to be comaximal if $\mathfrak{A} + \mathfrak{B} = R$. Ideals $\mathfrak{A}_1, \cdots, \mathfrak{A}_n$ are pairwise comaximal if and only if their radicals are. If an ideal $\mathfrak{B}$ is comaximal with each $\mathfrak{A}_i$, then it is comaximal with $\mathfrak{A}_1 \cap \cdots \cap \mathfrak{A}_n$ and $\mathfrak{A}_1 \cdots \mathfrak{A}_n$.

If $\mathfrak{A}_1, \cdots, \mathfrak{A}_n$ are pairwise comaximal, then
$$\mathfrak{A}_1 \cap \cdots \cap \mathfrak{A}_n = \mathfrak{A}_1 \cdots \mathfrak{A}_n.$$
Moreover, if $b_1, \cdots, b_n$ are elements of $R$, then there exists an element $b$ in $R$ such that
$$b \equiv b_i \pmod{\mathfrak{A}_i}, \quad i = 1, \cdots, n.$$
