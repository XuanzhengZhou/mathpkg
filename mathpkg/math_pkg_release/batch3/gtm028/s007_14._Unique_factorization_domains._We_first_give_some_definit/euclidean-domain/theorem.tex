A \text{euclidean domain} $E$ is an integral domain in which with every element $a$ there is associated a definite integer $\varphi(a)$, provided the function $\varphi$ satisfies the following conditions:

\text{E1.} If $b$ divides $a$, then $\varphi(b) \leq \varphi(a)$.

\text{E2.} For each pair of elements $a, b$ in $E$, $b \neq 0$, there exist elements $q$ and $r$ in $E$ such that $a = bq + r$ and $\varphi(r) < \varphi(b)$.
