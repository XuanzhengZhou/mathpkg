For a fixed polynomial $f \in R[X]$, the transformation $y \mapsto f(y)$, where $y \in \Delta$ and $\Delta$ is any ring unitary over $R$, is a mapping of $\Delta$ into itself. This function is denoted by $f_{\Delta}$.

Thus with every polynomial $f \in R[X]$ and with every ring $\Delta$ unitary over $R$, there is associated a function $f_{\Delta}$ on $\Delta$ to $\Delta$. If $\Delta$ is a subring of another ring $\Delta_1$ which is unitary over $\Delta$, then $f_{\Delta_1} = f_{\Delta}$ on $\Delta$.

Any polynomial in $R[X]$ can be thought of as the symbol of a well-defined operation which can be applied to any element $y$ of any given ring $\Delta$ unitary over $R$ and which, if so applied, yields a well-defined function on $\Delta$ to $\Delta$. This operation is performed by substituting $y$ for $X$ in the given polynomial $f$, or $f(X)$. From this point of view the symbol $X$ appears indeed as an indeterminate, or \text{variable}, which can take values in any ring containing $R$.
