Given any set $S$ of regular elements in a ring $R$, the multiplicative system generated by $S$ is obtained by taking all finite products of elements from $S$. This is the smallest multiplicative system containing $S$, and it provides a convenient way to specify a localization by naming only the elements one wishes to invert, rather than listing all possible denominators. In practice, one often localizes at a single element $s$ (i.e., $S = \{s\}$) or at the complement of a prime ideal.
