Let $R$ be an integral domain satisfying $\mathrm{UF}_1$ and let $F$ be its field of fractions. Let $M$ be a multiplicative system in $R$ and let $M'$ be the set of all elements of $R$ which are units in $R_M$.

Define $S$ to be the set of all irreducible elements of $R$ which divide elements of $M$. For the purposes of this correspondence, associate elements are not regarded as distinct elements of $S$.

Let $M_0$ be the multiplicative system generated by $S$. It is clear that $M_0 \subseteq M'$. While $M_0$ may be a proper subset of $M'$, every element of $M'$ is the associate of some element of $M_0$, and therefore $R_{M'} = R_{M_0}$.

We note that $S$ is uniquely determined by $M'$, since $S$ is also the set of all irreducible elements of $R$ which divide elements of $M'$. Hence $S$ is uniquely determined by the given quotient ring $R_M$.

Conversely, given an arbitrary set $S$ of irreducible elements of $R$, let $M_0$ be the multiplicative system generated by $S$. Then $R_{M_0}$ is a well-defined quotient ring of $R$.

The map $R_M \mapsto S$ (where $S$ is the set of irreducible elements dividing elements of $M$) and the map $S \mapsto R_{M_0}$ (where $M_0$ is the m.s. generated by $S$) are mutually inverse. This establishes the required $(1,1)$ correspondence.
