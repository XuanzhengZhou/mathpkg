If $R$ is a field, then it has only the two (improper) ideals $(0)$ and $R1$, for if $\mathfrak{A}$ is an ideal in $R$ and $\mathfrak{A} \neq (0)$, then $\mathfrak{A}$ contains an element $a \neq 0$, hence contains $aa^{-1} = 1$, hence equals $R$.

Conversely, if $R$ has an identity and has only two ideals, then $R$ is a field; for if $a \in R$, $a \neq 0$, then $Ra \neq R0$, hence $Ra = R$, $1 \in Ra$, so $1 = xa$ for some $x \in R$.
