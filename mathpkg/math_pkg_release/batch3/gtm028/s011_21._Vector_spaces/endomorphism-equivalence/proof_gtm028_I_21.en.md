The assertion that $T$ is univalent means that its kernel $K$ is $(0)$, that is, $[K:F] = 0$. The assertion that $T$ is onto means that $VT = V$, that is, $[VT:F] = [V:F]$.

By Theorem 23, $[V:F] = [K:F] + [VT:F]$.

If $T$ is univalent, then $[K:F] = 0$, so $[VT:F] = [V:F]$. Since $VT$ is a subspace of $V$ with the same finite dimension, $VT = V$, i.e., $T$ is onto.

If $T$ is onto, then $[VT:F] = [V:F]$, so $[K:F] = 0$, whence $K = (0)$, i.e., $T$ is univalent.
