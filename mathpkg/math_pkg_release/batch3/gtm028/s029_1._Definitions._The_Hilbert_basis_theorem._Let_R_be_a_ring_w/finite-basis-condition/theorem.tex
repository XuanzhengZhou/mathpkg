Let $R$ be a commutative ring and let $J$ denote the set of integers. The \textbf{finite basis condition} for ideals of $R$ is the property that every ideal $\mathfrak{A}$ of $R$ has a finite basis; that is, $\mathfrak{A}$ contains a finite set of elements $a_1, \dots, a_n$ such that
$$
\mathfrak{A} = Ra_1 + Ra_2 + \cdots + Ra_n + Ja_1 + Ja_2 + \cdots + Ja_n.
$$
Such a set $\{a_1, \dots, a_n\}$ is called a \textbf{basis} (or a set of generators) of $\mathfrak{A}$.
