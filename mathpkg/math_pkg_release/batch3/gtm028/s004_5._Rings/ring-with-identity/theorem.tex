\text{If there exists in the ring } R \text{ an element which is an identity with respect to multiplication, then this element is uniquely determined. If } R \text{ is not a nullring, this element is called the \textbf{identity} of the ring and is denoted by the symbol } 1.

\text{In such a ring, an \textbf{inverse} of } a \text{ is an element } a' \text{ such that } a'a = 1 \text{ and } aa' = 1. \text{ An element having an inverse is called a \textbf{unit}.}
