\text{By the distributive law } R_3 \text{ with } b = 0, \text{ we have:}

$$a(0 + c) = a \cdot 0 + a \cdot c.$$

\text{Since } 0 + c = c \text{ and } a \cdot 0 = 0 \text{ (from the previous property), this gives } a \cdot c = 0 + a \cdot c = a \cdot c, \text{ which does not directly help. Instead, we use the distributive law on } c + (-c) = 0:

$$a(c + (-c)) = a \cdot c + a(-c).$$

\text{The left side equals } a \cdot 0 = 0. \text{ Hence } a \cdot c + a(-c) = 0, \text{ which means } a(-c) = -a \cdot c \text{ by uniqueness of the additive inverse. The second relation } (-c)a = -c a \text{ follows similarly from the right distributive law.}
