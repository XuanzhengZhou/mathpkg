\text{From the first relation } a(-c) = -ac \text{ (established in the previous proposition), replace } a \text{ by } -a:

$$(-a)(-c) = -((-a)c).$$

\text{Using the second relation } (-c)a = -ca \text{ with } c \text{ replaced by } a, \text{ we have } (-a)c = -ac. \text{ Substituting:}

$$(-a)(-c) = -((-a)c) = -(-ac) = ac,$$

\text{where the last equality uses the fact that in the additive group, } -(-x) = x \text{ for any } x.
