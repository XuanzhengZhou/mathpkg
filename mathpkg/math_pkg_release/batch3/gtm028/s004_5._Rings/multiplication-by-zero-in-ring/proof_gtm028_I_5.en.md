\text{By the distributive law } R_3 \text{ with } b = c, \text{ we have:}

$$a(b + (-b)) = ab + a(-b).$$

\text{Since } b + (-b) = 0 \text{ (by the additive group axioms), the left side equals } a \cdot 0. \text{ Hence:}

$$a \cdot 0 = ab + a(-b).$$

\text{Now set } b = c = 0 \text{ in } R_3. \text{ Then } a(0 + 0) = a \cdot 0 + a \cdot 0, \text{ and subtracting } a \cdot 0 \text{ from both sides yields } a \cdot 0 = 0. \text{ Similarly, from the right distributive law } (b + c)a = ba + ca, \text{ we obtain } 0 \cdot a = 0. \text{ Therefore } a \cdot 0 = 0 \cdot a = 0 \text{ for all } a \in R.
