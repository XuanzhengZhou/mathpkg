An associative binary operation is one where the grouping of operations does not affect the result: $(ab)c = a(bc)$ for all $a, b, c$ in $G$. Associativity is the defining property of semigroups and is the first axiom in the definition of a group. Without associativity, expressions like $abc$ would be ambiguous. Once associativity is assumed, one can freely omit parentheses and define powers of elements inductively, which makes algebraic manipulation vastly simpler.
