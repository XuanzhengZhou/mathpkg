Powers of an element $a$ in a set with an associative binary operation are defined inductively: $a^1 = a$ and $a^n = a^{n-1}a$ for $n > 1$. This definition makes sense precisely because associativity eliminates ambiguity in grouping. The inductive definition is the foundation for all exponential notation in algebra, and leads to the familiar rules of exponents $a^m a^n = a^{m+n}$ and $(a^m)^n = a^{mn}$. Once an identity and inverses are introduced, the definition extends to non-positive exponents as well.
