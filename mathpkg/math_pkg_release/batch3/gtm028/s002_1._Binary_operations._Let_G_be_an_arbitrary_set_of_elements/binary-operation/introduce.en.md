A binary operation on a set $G$ is a rule that assigns to each ordered pair $(a,b)$ of elements of $G$ a unique element $c$ in $G$. This is the fundamental building block of algebraic structures — it defines how two elements of a set can be combined to produce a third element of the same set. The operation is conventionally written multiplicatively as $a \cdot b$ or $ab$, and the result is called the product. The function-like interpretation is that a binary operation is a single-valued function $G \times G \to G$ (or to a subset of $G$). An important subtlety is that the operation is not required to be symmetric: $ab$ and $ba$ may be distinct elements.
