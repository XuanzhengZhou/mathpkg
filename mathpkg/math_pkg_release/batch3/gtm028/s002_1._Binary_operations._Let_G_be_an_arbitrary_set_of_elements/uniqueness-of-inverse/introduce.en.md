If an element in a set with an associative operation and identity has an inverse, that inverse is unique. The proof uses associativity: if $a'$ and $a''$ are both inverses of $a$, then $a'' = a''e = a''(aa') = (a''a)a' = ea' = a'$. This uniqueness is fundamental to group theory — it justifies the notation $a^{-1}$ and ensures that "the inverse" is well-defined. Combined with the uniqueness of the identity, it shows that the identity and inverse operations are single-valued functions on a group.
