Let $R$ be a ring with identity, $n$ a positive integer. Let $I$ be the set of non-negative integers, $I_n$ the set of ordered $n$-tuples $(i) = (i_1, \dots, i_n)$ of elements of $I$. For $(j) = (j_1, \dots, j_n)$ in $I_n$, define $(i) + (j) = (i_1 + j_1, \dots, i_n + j_n)$.

A **polynomial** $f$ over $R$ in $n$ indeterminates is a mapping $f: I_n \to R$ such that $(i)f = 0$ for all but a finite number of $n$-tuples $(i)$.

If $f$ and $g$ are two such polynomials, define $h = f + g$ and $k = f \cdot g$ by:
$$(i)h = (i)f + (i)g,$$
$$(i)k = \sum_{(j)+(l)=(i)} (j)f \cdot (l)g.$$

With these operations, the set of all such polynomials forms a ring containing $R$.
