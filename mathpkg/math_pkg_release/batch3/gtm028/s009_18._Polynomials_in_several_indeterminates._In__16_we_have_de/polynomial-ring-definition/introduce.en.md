A ring $S^\prime$ unitary over $R$ is a polynomial ring over $R$ if it is generated as an $R$-algebra by $n$ algebraically independent elements $x_1, \dots, x_n$. This structural definition is equivalent to the concrete mapping definition given earlier, and is useful for identifying when an arbitrary $R$-algebra is isomorphic to a polynomial ring.
