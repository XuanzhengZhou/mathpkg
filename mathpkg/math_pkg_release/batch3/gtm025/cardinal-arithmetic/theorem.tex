For cardinal numbers \(u, v, w\):
\begin{itemize}
\item[(i)] \(u + (v + w) = (u + v) + w\);
\item[(ii)] \(u + v = v + u\);
\item[(iii)] \(u(v + w) = uv + uw\);
\item[(iv)] \((uv)w = u(vw)\);
\item[(v)] \(uv = vu\);
\item[(viii)] \((u^v)^w = u^{vw}\).
\end{itemize}
If \(u \leq v\), then \(u + w \leq v + w, uw \leq vw, u^w \leq v^w\).
