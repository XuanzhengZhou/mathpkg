Suppose that $f \in \mathfrak{L}_1([a, b])$ and that $x \in ]a, b[$. Then $x$ is called a Lebesgue point for $f$ if (18.5.i) holds. The set of all Lebesgue points for $f$ is called the Lebesgue set for $f$.

It is obvious that every point of $]a, b[$ at which $f$ is continuous is a Lebesgue point for $f$. However $f$ need not be continuous anywhere, and yet almost every point is a Lebesgue point. The Lebesgue set plays an important role in the theory of Fourier series and integrals, as we shall see in (18.29) and (21.43).

We now inquire into the "reverse" [not really a converse] of Theorem (18.3). Given a continuous function $\varphi$ that is differentiable a.e., is it true that

$$\varphi(x) - \varphi(a) = \int_a^x \varphi'(t) dt?$$

That is, does the fundamental theorem of the calculus hold with the Riemann integral replaced by the Lebesgue integral and differentiability replaced by differentiability a.e.? The following examples answer this question with an emphatic "no".

(18.7) Example. As in (8.28), let $\psi$ be Lebesgue's singular function. Then $\psi$ is continuous on $[0, 1]$ and it is clear that $\psi'(x) = 0$ for all $x$ in $[0, 1]$ that are not in $P$ [CANTOR's ternary set]. Thus $\psi'(x) = 0$ a.e. It follows that

$$\psi(1) - \psi(0) = 1 \neq 0 = \int_0^1 \psi'(x) dx.$$
