A set \(B \subset X\) is \textbf{linearly independent} over \(F\) if for any finite subset \(\{x_1, \ldots, x_n\} \subset B\) and scalars \(\alpha_1, \ldots, \alpha_n \in F\), the equality \(\alpha_1 x_1 + \cdots + \alpha_n x_n = 0\) implies \(\alpha_1 = \alpha_2 = \cdots = \alpha_n = 0\). A nonvoid linearly independent set \(B\) such that any strictly larger set is not linearly independent is called a \textbf{Hamel basis} for \(X\) over \(F\).
