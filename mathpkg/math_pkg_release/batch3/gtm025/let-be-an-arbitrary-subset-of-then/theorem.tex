Let $A$ be an arbitrary subset of $R$. Then

(i) $\lim_{k \downarrow 0} \frac{\lambda(A \cap x, x + k)}{h} = \lim_{k \downarrow 0} \frac{\lambda(A \cap x - h, x)}{h}$
$= \lim_{k, k \downarrow 0} \frac{\lambda(A \cap x - h, x + k)}{h + k} = 1$

for almost all $x \in A$. If $A$ is $\lambda$-measurable, the limits in (i) are equal to zero for almost all $x \in A'$.$^1$
