Let $f$ and $g$ be $\mathcal{A}$-measurable functions. Let $h$ be defined on $X$ by

$$h(x) = \begin{cases} f(x) g(x) & \text{if } x \notin A \\ \beta & \text{if } x \in A, \end{cases}$$

where $\beta$ is an arbitrary extended real number, and $A = \{x \in X : f(x) = \infty \text{ and } g(x) = -\infty\} \cup \{x \in X : f(x) = -\infty \text{ and } g(x) = \infty\}$. Then $h$ is an $\mathcal{A}$-measurable function.
