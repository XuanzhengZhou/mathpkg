Let $u, v,$ and $w$ be any three cardinal numbers. Then:

(i) $u + (v + w) = (u + v) + w$;
(ii) $u + v = v + u$;
(iii) $u(v + w) = uv + uw$;
(iv) $u(vw) = (uw)w$;
(v) $uv = vu$;
(vi) $u^p u^w = u^{p+w}$;
(vii) $u^{wv}w = (uw)w$;
(viii) $(u^p)w = u^{pw}$;
(ix) $u \leq v$ implies $u + w \leq v + w$;
(x) $u \leq v$ implies $uw \leq vw$;
(xi) $u \leq v$ implies $uw \leq vw$;
(xii) $u \leq v$ implies $uw \leq vw$.
