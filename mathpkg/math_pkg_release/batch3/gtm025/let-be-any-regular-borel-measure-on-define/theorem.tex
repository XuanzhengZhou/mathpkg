Let $\iota$ be any regular Borel measure on $R$. Define $\alpha$ on $R$ by the rule

(i) $\alpha(t) = \begin{cases} \iota([0, t]) & \text{if } t > 0, \\ 0 & \text{if } t = 0, \\ -\iota([t, 0]) & \text{if } t < 0. \end{cases}$

Then $\alpha$ is a nondecreasing, real-valued, left-continuous function on $R$. Also, we have $\lim_{t \to -\infty} \alpha(t) > -\infty$ if and only if $\iota([- \infty, 0]) < \infty$ and $\lim_{t \to \infty} \alpha(t) < \infty$ if and only if $\iota([0, \infty]) < \infty$.
