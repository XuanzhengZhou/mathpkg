Let $a \in \mathbb{R}$ and $\delta > 0$. If $\varphi$ is a real-valued function defined on $]a, a+\delta[$, the \textbf{lower right limit} and \textbf{upper right limit} of $\varphi$ at $a$ are
$$\varliminf_{h \downarrow a} \varphi(h) = \sup\{\inf\{\varphi(h) : a < h < s\} : a < s < a+\delta\},$$
$$\varlimsup_{h \downarrow a} \varphi(h) = \inf\{\sup\{\varphi(h) : a < h < s\} : a < s < a+\delta\}.$$
If $\varphi$ is defined on $]a-\delta, a[$, define the lower left limit and upper left limit similarly:
$$\varliminf_{h \uparrow a} \varphi(h) = \sup\{\inf\{\varphi(h) : t < h < a\} : a-\delta \leq t < a\},$$
$$\varlimsup_{h \uparrow a} \varphi(h) = \inf\{\sup\{\varphi(h) : t < h < a\} : a-\delta \leq t < a\}.$$
