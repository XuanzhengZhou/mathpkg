Let $\gamma$ be a set function on $\mathcal{A}$ satisfying: (i) $0 \leq \gamma(A) \leq \infty$; (ii) $\gamma(A \cup B) = \gamma(A) + \gamma(B)$ for disjoint $A, B$; (iii) if $A_1 \supset A_2 \supset \cdots$ and $\bigcap A_n = \emptyset$, then $\lim \gamma(A_n) = 0$. Define $\bar{\gamma}$ as in (10.36). Prove that the same conclusions hold.
