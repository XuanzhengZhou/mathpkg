Let $(X, \mathcal{A})$ be a measurable space. An extended real-valued $f$ on $X$ is $\mathcal{A}$-measurable if $f^{-1}(]a, \infty]) = \{x : a < f(x) \leq \infty\} \in \mathcal{A}$ for every $a \in \mathbb{R}$. If $\mathcal{A} = \mathcal{B}(X)$ then $f$ is Borel measurable; if $X = \mathbb{R}$ and $\mathcal{A} = \mathcal{M}_\lambda$ then $f$ is Lebesgue measurable.
