A simple function $s$ on $X$ is a function that assumes only finitely many values. If $\operatorname{rng} s = \{\alpha_1, \ldots, \alpha_n\}$ and $A_k = \{x \in X : s(x) = \alpha_k\}$ for $1 \leq k \leq n$, then
$$s = \sum_{k=1}^{n} \alpha_k \xi_{A_k},$$
where $\xi_{A_k}$ denotes the characteristic function of $A_k$.
