Let $X$ be a set and $\mathcal{A}$ an algebra of subsets of $X$. A set function $\mu$ defined on $\mathcal{A}$ is a finitely additive measure if:

(i) $0 \leq \mu(A) \leq \infty$ for all $A \in \mathcal{A}$;

(ii) $\mu(\emptyset) = 0$;

(iii) $\mu(A \cup B) = \mu(A) + \mu(B)$ for disjoint $A, B \in \mathcal{A}$.

If $\mu(\bigcup_{n=1}^{\infty} A_n) = \sum_{n=1}^{\infty} \mu(A_n)$ for pairwise disjoint $(A_n) \subset \mathcal{A}$ with union in $\mathcal{A}$, $\mu$ is countably additive. A measure space is $(X, \mathcal{A}, \mu)$ with $\mathcal{A}$ a $\sigma$-algebra and $\mu$ countably additive. It is finite if $\mu(X) < \infty$, $\sigma$-finite if $X$ is a countable union of finite-measure sets, and degenerate if $\mu$ assumes only $0$ and $\infty$.
