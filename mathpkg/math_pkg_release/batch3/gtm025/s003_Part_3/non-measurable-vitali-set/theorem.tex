There exists $E \subset \mathbb{R}$ that is not $\lambda$-measurable. The construction uses cosets of a countable dense subgroup $H$ of $\mathbb{R}$ and the axiom of choice. From a $\lambda$-measurable set $T$ of finite positive measure, select representatives from cosets intersecting $T$. The resulting set $E$ satisfies that $J + E$ is measurable with measure zero for a countable set $J$, yet $E$ itself is not measurable.
