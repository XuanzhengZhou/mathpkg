A typical member of $A^n$ is, to be sure, a function and as such is a set of $n$ ordered pairs. We follow conventional notation, however, and list the values of such a function as an ordered $n$-tuple. Thus $A^n = \{(a_1, \ldots, a_n) : a_k \in A$ for $k = 1, \ldots, n\}$. Similarly $A^N = \{(a_1, a_2, \ldots) : a_k \in A$ for $k \in N\}$. The set $R^n$ is called Euclidean $n$-space and $K^n$ is called unitary $n$-space.
