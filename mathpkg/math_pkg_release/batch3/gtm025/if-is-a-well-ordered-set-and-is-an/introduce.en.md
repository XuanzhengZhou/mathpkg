(4.43) Theorem. If $A$ is a well-ordered set and $f$ is an order isomorphism from $A$ into $A$, then $x \leq f(x)$ for each $x \in A$. is a theorem from Chapter 4 (Unknown Chapter) of the book *Real and Abstract Analysis* by Edwin Hewitt and Karl Stromberg (GTM 25). It is part of the development of analysis.
