Let $(X, \mathcal{A}, \mu)$ be a measure space and let $F(X, \mathcal{A}, \mu)$ denote the set of all complex-valued functions $\tau$ defined on $\mathcal{A}$ such that

(i) $\sup\{|\tau(A)| : A \in \mathcal{A}\} < \infty$;

(ii) $\tau(A \cup B) = \tau(A) + \tau(B)$ if $A, B \in \mathcal{A}$ and $A \cap B = \varnothing$; and

(iii) $\tau(A) = 0$ if $A \in \mathcal{A}$ and $A$ is locally $\mu$-null.

For such a $\tau$ we define $|\tau|$ on $\mathcal{A}$ [just as in (19.10) and (19.11)] by the rule $|\tau|(A) = \sup\left\{\sum_{j=1}^{n}|\tau(A_j)| : \{A_1, \ldots, A_n\}\right\}$ is a measurable dissection of $A$.

It is easy to show that $|\tau| \in F(X, \mathcal{A}, \mu)$ [cf. (19.12)]. We define the norm of $\tau$ to be the number $\|\tau\| = |\tau|(X)$. One easily verifies that $F(X, \mathcal{A}, \mu)$ is a complex normed linear space with this norm and with setwise linear operations.
