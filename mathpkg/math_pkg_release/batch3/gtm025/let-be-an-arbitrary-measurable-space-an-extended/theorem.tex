Our first goal is to show that just as a function of finite variation can be expressed as a linear combination of four monotone functions, so a complex measure can be expressed as a linear combination of four measures. It is obvious that any complex measure $v$ can be expressed uniquely in the form $v = v_1 + iv_2$ where $v_1$ and $v_2$ are real-valued signed measures; simply let $v_1(E) = \text{Re } v(E)$ and $v_2(E) = \text{Im } v(E)$ for all $E \in \mathcal{A}$. We therefore take up signed measures first.
