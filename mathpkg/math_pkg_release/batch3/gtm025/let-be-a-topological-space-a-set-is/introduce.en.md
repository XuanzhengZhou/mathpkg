(6.53) Definition. Let $X$ be a topological space. A set $A \subset X$ is said to be nowhere dense if $A^{-\circ} = \varnothing$. A set $F \subset X$ is a definition from Chapter 6 (Topology) of the book *Real and Abstract Analysis* by Edwin Hewitt and Karl Stromberg (GTM 25). It is part of the development of topology.
