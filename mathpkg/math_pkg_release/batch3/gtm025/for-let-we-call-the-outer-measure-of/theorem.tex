For $A \subset X$ let $\iota(A) = \bar{I}(\xi_A)$. We call $\iota(A)$ the [outer] measure of $A$. The function $\iota$, defined on $\mathcal{P}(X)$, is known as the [outer] measure induced by $I$. In case $X = R$ and $I = S_\alpha$ for some real-valued nondecreasing function $\alpha$ on $R$, we write $\lambda_\alpha(A) = \bar{S}_\alpha(\xi_A)$ and call $\lambda_\alpha$ the Lebesgue-Stieltjes [outer] measure on $R$ induced by $\alpha$. If $X = R$ and $I = S$ [the Riemann integral], then we write $\lambda(A) = \bar{S}(\xi_A)$ and call $\lambda$ Lebesgue [outer] measure on $R$. For arbitrary $X$ and $I = E_a$ (9.2.c), we write $\varepsilon_a(A) = \bar{E}_a(\xi_A)$ and call $\varepsilon_a$ the unit point mass [or Dirac measure] concentrated at $a$.
