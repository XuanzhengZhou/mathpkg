Let $(X, \mathcal{A}, \mu)$ be a measure space and let $\tau \in F(X, \mathcal{A}, \mu)$.

(i) If $s = \sum_{j=1}^{m} \alpha_j \xi_{A_j}$ is an $\mathcal{A}$-measurable simple function on $X$, define
$$\int_X s \, d\tau = \sum_{j=1}^{m} \alpha_j \tau(A_j).$$

(ii) If $f$ is a bounded, complex-valued $\mathcal{A}$-measurable function on $X$, obtain a sequence $(s_n)_{n=1}^{\infty}$ of $\mathcal{A}$-measurable simple functions on $X$ such that $\|f - s_n\|_u \to 0$. Then define
$$\int_X f \, d\tau = \lim_{n \to \infty} \int_X s_n \, d\tau.$$
This definition is independent of the choice of $(s_n)$.
