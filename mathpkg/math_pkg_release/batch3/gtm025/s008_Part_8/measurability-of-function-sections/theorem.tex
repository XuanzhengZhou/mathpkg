Let $(X, \mathcal{M})$ and $(Y, \mathcal{N})$ be measurable spaces and let $f$ be an $\mathcal{M} \times \mathcal{N}$-measurable function on $X \times Y$. Then

(i) the function $x \to f(x, y)$ is $\mathcal{M}$-measurable for each $y \in Y$;

(ii) the function $y \to f(x, y)$ is $\mathcal{N}$-measurable for each $x \in X$.
