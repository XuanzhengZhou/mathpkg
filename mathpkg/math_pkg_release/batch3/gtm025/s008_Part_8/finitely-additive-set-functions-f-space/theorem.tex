Let $(X, \mathcal{A}, \mu)$ be a measure space and let $F(X, \mathcal{A}, \mu)$ denote the set of all complex-valued functions $\tau$ defined on $\mathcal{A}$ such that

(i) $\sup\{|\tau(A)| : A \in \mathcal{A}\} < \infty$;

(ii) $\tau(A \cup B) = \tau(A) + \tau(B)$ if $A, B \in \mathcal{A}$ and $A \cap B = \varnothing$;

(iii) $\tau(A) = 0$ if $A \in \mathcal{A}$ and $A$ is locally $\mu$-null.

For such a $\tau$ we define $|\tau|$ on $\mathcal{A}$ by the rule
$$|\tau|(A) = \sup\left\{\sum_{j=1}^{n}|\tau(A_j)| : \{A_1, \ldots, A_n\} \text{ is a measurable dissection of } A\right\}.$$
The norm of $\tau$ is $\|\tau\| = |\tau|(X)$. With setwise linear operations and this norm, $F(X, \mathcal{A}, \mu)$ is a complex normed linear space.
