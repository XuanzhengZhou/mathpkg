This lemma provides a powerful characterization of derivatives of signed measures that avoids explicit reference to Lebesgue or Hahn decompositions. Instead, it uses level set inequalities: a measurable function g is a derivative of eta with respect to mu if and only if for every alpha, the upper level set satisfies an inequality with eta and mu, and similarly for lower level sets. This is essential for proving martingale convergence theorems.
