Let $g$ be an extended real-valued, $\mathcal{A}$-measurable function on $X$. For each real number $\alpha$, let $G_\alpha = \{x \in X : g(x) \geq \alpha\}$ and $L_\alpha = \{x \in X : g(x) \leq \alpha\}$. The function $g$ is a derivative of $\eta$ with respect to $\mu$ if and only if:

(i) for every real number $\alpha$ and every $A \in \mathcal{A}$,
$$\eta(G_\alpha \cap A) \geq \alpha \mu(G_\alpha \cap A);$$

(ii) for every real number $\alpha$ and every $A \in \mathcal{A}$,
$$\eta(L_\alpha \cap A) \leq \alpha \mu(L_\alpha \cap A).$$
