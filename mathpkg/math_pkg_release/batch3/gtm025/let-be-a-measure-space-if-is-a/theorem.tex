Let $(X, \mathcal{A}, \mu)$ be a measure space. If $(A_n)_{n=1}^{\infty}$ is a sequence of sets in $\mathcal{A}$ such that $\mu(A_1) < \infty$ and $A_1 \supset A_2 \supset \cdots \supset A_n \supset \cdots$, then

$$\mu\left(\bigcap_{n=1}^{\infty} A_n\right) = \lim_{n \to \infty} \mu(A_n).$$

In particular, if $\bigcap_{n=1}^{\infty} A_n = \emptyset$, then $\lim_{n \to \infty} \mu(A_n) = 0.$
