(11.7) Theorem. Let $\varphi$ be any extended real-valued function defined on $R^{\#}$ such that $\varphi^{-1}([a, \infty]) \cap R$ is a Borel set for is a theorem from Chapter 11 (Measurable Functions) of the book *Real and Abstract Analysis* by Edwin Hewitt and Karl Stromberg (GTM 25).
