** A set $U \subset R$ is said to be open if for each $x \in U$ there is a positive real number $\varepsilon$ such that $]x - \varepsilon, x + \varepsilon[ \subset U.$

Thus we may say informally that a subset $U$ of $R$ is open if for each of its points $x,$ it contains all points $y$ such that $y$ is sufficiently close to $x.$ [The "sufficiently close" depends of course on $x.$] Plainly every open interval is an open set.
