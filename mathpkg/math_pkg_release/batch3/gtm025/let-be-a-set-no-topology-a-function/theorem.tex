Let $X$ be a set [no topology]. A function $\mu$ defined on $\mathcal{P}(X)$ is called a $[CARATHĂ‰ODORY^1]$ outer measure if the following relations hold:

(i) $0 \leq \mu(A) \leq \infty$ for all $A \subset X$;
(ii) $\mu(\emptyset) = 0$;
(iii) $\mu(A) \leq \mu(B)$ if $A \subset B \subset X$;
(iv) $\mu\left(\bigcup_{n=1}^{\infty} A_n\right) \leq \sum_{n=1}^{\infty} \mu(A_n)$ for all sequences $(A_n)_{n=1}^{\infty}$ of subsets of $X$.
