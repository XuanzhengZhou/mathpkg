If a linearly ordered set \(A\) is well ordered, then \(\operatorname{ord} A\) is called an \textbf{ordinal number}. Examples: \(\operatorname{ord} \varnothing = 0\), \(\operatorname{ord}\{1,\ldots,n\} = n\), \(\operatorname{ord} N = \omega\). The order type of \(Q\) (\(\eta\)) is not an ordinal number since \(Q\) is not well ordered.
