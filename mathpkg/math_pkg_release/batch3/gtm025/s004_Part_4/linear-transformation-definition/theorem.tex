Let $A$ and $B$ be linear spaces over $F$. A function $T: A \to B$ is a linear transformation if $T(x+y) = T(x) + T(y)$ and $T(\alpha x) = \alpha T(x)$ for all $x,y \in A$, $\alpha \in F$. If $A,B$ are normed linear spaces, $T$ is bounded if there exists $M \ge 0$ such that $\|T(x)\| \le M\|x\|$ for all $x \in A$. The operator norm $\|T\|$ is the infimum of all such $M$.
