By (13.20), simple functions are dense. Approximate a simple function $\sigma$ by $\varphi \in \mathfrak{C}_{00}$ using Luzin's theorem (11.36). If $\sigma$ has support of finite measure, use Luzin's theorem to find $\varphi \in \mathfrak{C}_{00}$ with $\|\varphi\|_u \leq \|\sigma\|_u$ that differs from $\sigma$ only on a set of small measure, then bound $\|\sigma - \varphi\|_p$.
