Let $\varphi$ be a continuous, real-valued, strictly increasing function defined on $[0, \infty[$ such that $\lim_{u \to \infty} \varphi(u) = \infty$ and $\varphi(0) = 0$. Let $\psi = \varphi^{-1}$. For all $x \in [0, \infty[$ define $\Phi(x) = \int_0^x \varphi(u) du$ and $\Psi(x) = \int_0^x \psi(v) dv$. Then for all $a, b \in [0, \infty[$, $ab \leq \Phi(a) + \Psi(b)$. Equality holds if and only if $b = \varphi(a)$.
