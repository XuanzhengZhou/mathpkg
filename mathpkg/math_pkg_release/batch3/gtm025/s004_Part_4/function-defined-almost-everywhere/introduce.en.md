This definition formalizes the extension of the Lebesgue integral to functions that are defined only almost everywhere (i.e., on a set $E$ whose complement has measure zero). The integral of such a function is defined via any measurable extension to the whole space, and (12.14) guarantees the definition is well-posed and independent of the choice of extension.
