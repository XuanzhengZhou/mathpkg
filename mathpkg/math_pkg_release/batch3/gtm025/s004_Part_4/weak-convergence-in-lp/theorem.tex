Let $(X, \mathcal{A}, \mu)$ be a measure space, $1 \leq p < \infty$, and $f, (f_n) \subset \mathfrak{L}_p$. If $p > 1$, $(f_n)$ converges to $f$ weakly in $\mathfrak{L}_p$ if $\lim_{n \to \infty} \int_X f_n g d\mu = \int_X f g d\mu$ for every $g \in \mathfrak{L}_{p'}$. If $p = 1$, weak convergence means the same for every bounded $\mathcal{A}$-measurable function $g$.
