A \textbf{vector space} (linear space) is an ordered triple \((X, \cdot, F)\) where \(X\) is an additive Abelian group, \(F\) is a field, and \(\cdot : F \times X \to X\) satisfies for all \(\alpha, \beta \in F\) and \(x, y \in X\):
\begin{enumerate}
\item[(i)] \(\alpha(x + y) = \alpha x + \alpha y\);
\item[(ii)] \((\alpha + \beta)x = \alpha x + \beta x\);
\item[(iii)] \(\alpha(\beta x) = (\alpha \beta)x\);
\item[(iv)] \(1x = x\), where \(1\) is the multiplicative identity of \(F\).
\end{enumerate}
