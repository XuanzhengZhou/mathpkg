Let $A$ and $B$ be linear spaces over $F$. A function $T$ from $A$ into $B$ is called a linear transformation [or linear operator] if

(i) $T(x + y) = T(x) + T(y)$

and

(ii) $T(\alpha x) = \alpha T(x)$

for all $x, y \in A$ and $\alpha \in F$. If $A$ and $B$ are normed linear spaces, a linear transformation $T$ from $A$ into $B$ is said to be bounded if there exists a nonnegative real number $M$ such that

(iii) $\| T(x) \| \leq M \| x \|$ for all $x \in A$

[i.e., $T$ is bounded on the unit sphere of $A$]. In this case we define the norm of $T$ to be the infimum of the set of all $M$'s that satisfy (iii), and we write $\| T \|$ for the norm of $T$. This norm is called the operator norm.
