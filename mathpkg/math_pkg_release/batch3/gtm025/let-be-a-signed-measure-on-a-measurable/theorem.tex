Let $v$ be a signed measure on a measurable space $(X, \mathcal{A})$. We have:

(i) if $E, F \in \mathcal{A}$, $|v(E)| < \infty$, and $F \subset E$, then $|v(F)| < \infty$;

(ii) if $A_n \in \mathcal{A}$ ($n = 1, 2, 3, \ldots$) and $A_1 \subset A_2 \subset \cdots \subset A_n \subset \cdots$, then

$$\lim_{n \to \infty} v(A_n) = v\left(\bigcup_{n=1}^{\infty} A_n\right);$$

(iii) if $A_n \in \mathcal{A}$ ($n = 1, 2, 3, \ldots$), if $A_1 \supset A_2 \supset \cdots \supset A_n \supset \cdots$, and $|v(A_1)| < \infty$, then

$$\lim_{n \to \infty} v(A_n) = v\left(\bigcap_{n=1}^{\infty} A_n\right).$$
