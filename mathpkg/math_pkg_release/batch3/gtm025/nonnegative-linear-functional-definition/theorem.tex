Let $X$ be a nonvoid locally compact Hausdorff space. A complex-valued function $I$ on $\mathfrak{C}_{00}(X)$ is called a \textbf{nonnegative linear functional} (Radon measure) if for all $f, g \in \mathfrak{C}_{00}(X)$ and $\alpha \in K$: (i) $I(f+g) = I(f) + I(g)$; (ii) $I(\alpha f) = \alpha I(f)$; (iii) $f \geq 0$ implies $I(f) \geq 0$.
