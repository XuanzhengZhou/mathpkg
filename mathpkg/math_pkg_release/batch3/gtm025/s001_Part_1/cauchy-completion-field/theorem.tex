Let $F$ be an ordered field. Let $\mathfrak{C}$ be the set of all Cauchy sequences in $F$ and $\mathfrak{N}$ the set of all null sequences in $F$. Then $\mathfrak{N}$ is an ideal in the commutative ring $\mathfrak{C}$ (with componentwise operations). Addition and multiplication in $\mathfrak{C}/\mathfrak{N}$ are defined by
$$((a_n) + \mathfrak{N}) + ((b_n) + \mathfrak{N}) = (a_n + b_n) + \mathfrak{N},$$
$$((a_n) + \mathfrak{N})((b_n) + \mathfrak{N}) = (a_n b_n) + \mathfrak{N}.$$
These definitions are unambiguous, and with addition and multiplication so defined, $\mathfrak{C}/\mathfrak{N}$ is a field.
