Let $H$ be a Hilbert space and let $\{z_n\}_{n=1}^{\infty}$ be an orthogonal set in $H$, i.e., $z_n \perp z_m$ for $n \neq m$. Then $\sum_{n=1}^{\infty} z_n$ converges if and only if $\sum_{n=1}^{\infty} \|z_n\|^2 < \infty$. If $\sum_{n=1}^{\infty} z_n = z$, then $\|z\|^2 = \sum_{n=1}^{\infty} \|z_n\|^2$.
