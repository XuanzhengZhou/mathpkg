A similar schematic procedure could be applied to (ii). However, we may use (i) and the previous laws as follows:

$(A \cup B) \cap (A \cup C) = ((A \cup B) \cap A) \cup ((A \cup B) \cap C)$

the complement of $A$ [relative to $X$] to be the set $\{x : x \in X, x \notin A\}$. This set is denoted by the symbol $A'$. If there is any possible ambiguity as to which set is the universal set, we will write $X \cap A'$ for $A'$. Other common notations for what we call $A'$ are $X - A$, $X \setminus A$, $X \sim A$, $CA$, and $A^e$; we will use $A'$ exclusively.
