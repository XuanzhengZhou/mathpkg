Let $f$ be any complex- or extended real-valued $\mathcal{A}$-measurable function defined on $X$. Then there exists a sequence $(s_n)$ of finite-valued, $\mathcal{A}$-measurable, simple functions defined on $X$ such that $|s_1| \leq |s_2| \leq \cdots \leq |s_n| \leq \cdots$ and $s_n(x) \rightarrow f(x)$ for each $x \in X$. If $f$ is bounded, then the functions $s_n$ can be chosen so that the convergence is uniform. If $f \geq 0$, the sequence $(s_n)$ can be chosen so that $0 \leq s_1 \leq s_2 \leq \cdots \leq f$.
