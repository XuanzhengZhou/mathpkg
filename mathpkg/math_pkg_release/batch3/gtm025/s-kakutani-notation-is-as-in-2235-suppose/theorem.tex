[S. KAKUTANI]. Notation is as in (22.35). Suppose that $\eta_n \ll \mu_n$ for all $n$. Then we have either

(i) $\eta \ll \mu$
or
(ii) $\eta \perp \mu$.

Let $f_n$ be a function in $\mathfrak{L}_1^+ (T_n, \mathcal{M}_n, \mu_n)$ such that

(iii) $\int_E f_n d\mu_n = \eta_n(E_n)$

for all $E_n \in \mathcal{M}_n$, i.e., let $f_n$ be a LEBESGUE-RADON-NIKODYM derivative $\frac{d\eta_n}{d\mu_n}$ in the sense of (19.43). Then (i) holds if and only if

(iv) $\prod_{k=1}^{\infty} \left( \int_T f_k^{\frac{1}{2}} d\mu_k \right) > 0$,

and (ii) holds if and only if

(v) $\prod_{k=1}^{\infty} \left( \int_T f_k^{\frac{1}{2}} d\mu_k \right) = 0$.
