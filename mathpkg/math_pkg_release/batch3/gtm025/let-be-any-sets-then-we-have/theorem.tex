Let $A, B, C$ be any sets. Then we have:

(i) $A \cup B = B \cup A;$ (i') $A \cap B = B \cap A;$

(ii) $A \cup A = A;$ (ii') $A \cap A = A;$

(iii) $A \cup \varnothing = A;$ (iii') $A \cap \varnothing = \varnothing;$

(iv) $A \cup (B \cup C) = (A \cup B) \cup C;$ (iv') $A \cap (B \cap C) = (A \cap B) \cap C;$

(v) $A \subset A \cup B;$ (v') $A \cap B \subset A;$

(vi) $A \subset B$ if and only if $A \cup B = B;$ (vi') $A \subset B$ if and only if $A \cap B = A.$

The proof of this theorem is very simple and is left to the reader.
