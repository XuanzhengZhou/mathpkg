Two linearly ordered sets \(A\) and \(B\) are \textbf{order isomorphic} (written \(A \approx B\)) if there exists a one-to-one, order-preserving mapping of \(A\) onto \(B\). If \(A \approx B\), they have the same \textbf{order type}, written \(\operatorname{ord} A = \operatorname{ord} B\). If \(A\) is well ordered, \(\operatorname{ord} A\) is called an \textbf{ordinal number}.
