Let $a \in R$ and $\delta > 0$. If $f$ is a real-valued function defined on $[a, a + \delta[$, define

$$D_+ f(a) = \lim_{h \downarrow 0} \frac{f(a + h) - f(a)}{h}$$

and

$$D^+ f(a) = \lim_{h \downarrow 0} \frac{f(a + h) - f(a)}{h}.$$

If $f$ is a real-valued function defined on $]a - \delta, a[$, define

$$D_- f(a) = \lim_{h \downarrow 0} \frac{f(a + h) - f(a)}{h}$$

and

$$D^- f(a) = \lim_{h \downarrow 0} \frac{f(a + h) - f(a)}{h}.$$

These four extended real numbers are known as the Dini derivates of $f$ at $a$; $D_+ f(a)$ is the lower right derivate, $D^+ f(a)$ is the upper right derivate, $D_- f(a)$ is the lower left derivate, and $D^- f(a)$ is the upper left derivate.
