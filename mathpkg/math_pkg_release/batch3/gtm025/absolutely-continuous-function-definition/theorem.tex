A function $f$ on a subinterval $J \subset R$ is \textbf{absolutely continuous} if for every $\varepsilon > 0$ there exists $\delta > 0$ such that for any finite family of pairwise disjoint intervals $]a_k, b_k[ \subset J$ with $\sum (b_k - a_k) < \delta$, we have $\sum |f(b_k) - f(a_k)| < \varepsilon$.
