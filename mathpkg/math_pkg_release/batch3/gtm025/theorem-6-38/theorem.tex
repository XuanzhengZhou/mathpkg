Lemma (3.10) implies that $X$ contains a maximal cover $\mathcal{V}$. That is, $\mathcal{V}$ is an open cover of $X$, $\mathcal{V}$ has no finite subcover, and if $U$ is any open set not in $\mathcal{V}$, then $\mathcal{V} \cup \{U\}$ admits a finite subcover. Let $\mathcal{W} = \mathcal{V} \cap \mathcal{S}$. Then no finite subfamily of $\mathcal{W}$ covers $X$, and so (ii) implies that $\mathcal{W}$ is not a cover of $X$. Let $x$ be a point in $X \cap (\cup \mathcal{W})'$, and select a set $V$ in the cover $\mathcal{V}$ that contains $x$. Since $\mathcal{S}$ is a subbase, there are sets $S_1, \ldots, S_n$ in $\mathcal{S}$ such that $x \in \bigcap_{j=1}^{n} S_j \subset V$. Since $x \notin (\cup \mathcal{W})$, no $S_j$ is in $\mathcal{V}$. Since $\mathcal{V}$ is maximal, there exists for each $j$ a set $A_j$ which is the union of a finite number of sets in $\mathcal{V}$ such that $S_j \cup A_j = X$. Hence

$$V \cup \bigcup_{j=1}^{n} A_j \supset \binom{\bigcap_{j=1}^{n} S_j}{\bigcap_{j=1}^{n} A_j} = X,$$

and therefore $X$ is a union of finitely many sets from $\mathcal{V}$. This contradicts our choice of $\mathcal{V}$.
