A set \(S\) is called \textbf{countable} if either \(S\) is finite or \(\overline{S} = \overline{N} = \aleph_0\). Any set that is not countable is \textbf{uncountable}. A set \(S\) is \textbf{countably infinite} (or \textbf{denumerable}) if \(\overline{S} = \aleph_0\). If \(S\) is countably infinite and \(f: N \to S\) is a bijection, then the sequence \((x_n)\) where \(x_n = f(n)\) is called an \textbf{enumeration} of \(S\).
