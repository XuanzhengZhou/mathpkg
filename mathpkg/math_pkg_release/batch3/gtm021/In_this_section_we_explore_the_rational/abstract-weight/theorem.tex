Let $\Phi$ be a root system in a Euclidean space $E = \mathbb{R} \otimes_{\mathbb{Z}} X(T)$, with $W$-invariant inner product $(\cdot,\cdot)$. An element $\lambda \in E$ is called an \textbf{abstract weight} if
$$\frac{2(\lambda, \alpha)}{(\alpha, \alpha)} \in \mathbb{Z} \quad \text{for all } \alpha \in \Phi.$$
The set of all abstract weights forms a lattice $\Lambda$ (a free abelian group of rank $\ell = \dim E$), called the \textbf{abstract weight lattice}.
