First show G is commutative: if G = (G,G), then Ad G subset SL(g)=1. Semisimple elements lie in Z(G), which is finite. Then G/Z(G) is connected, 1-dim, has no semisimple except e, so is unipotent, nilpotent, forcing G nilpotent, contradiction. So G is commutative. By Theorem 15.5, G = G_s or G = G_u. If G = G_s, G is a torus, hence isomorphic to G_m. If G = G_u and char K = 0, Lemma 15.1C gives G isomorphic to G_a. If char K = p > 0, x -> x^p has image G^p connected and unequal to G, so G^p = e (exponent p). Theory of vector groups then yields G isomorphic to G_a.
