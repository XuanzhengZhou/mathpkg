Set $N = N_G(U)$. Since $U$ is a maximal connected unipotent subgroup of $N^\circ$, it contains a conjugate of every unipotent element of $N^\circ$ (Theorem 22.2). But $U$ is normal in $N^\circ$, so we conclude that $N^\circ/U$ consists of semisimple elements and hence is a torus (cf. Exercise 21.2). In particular, $N^\circ$ is solvable. Since $B \subset N^\circ$, we get $B = N^\circ$. But $B = N_G(B)$, so finally $N^\circ = N$.
