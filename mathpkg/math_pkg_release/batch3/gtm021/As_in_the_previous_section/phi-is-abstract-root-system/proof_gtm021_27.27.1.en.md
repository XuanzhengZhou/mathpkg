For each $\alpha \in \Phi$, Lemma 24.3 and Corollary 24.3 provide a reflection $\sigma_\alpha \in W$ (the nontrivial element of $W(Z_\alpha, T)$) which sends $\alpha$ to $-\alpha$ and has order 2. For $\sigma_\alpha$ to be a reflection, it should fix pointwise a subgroup of corank 1 in $X(T)$. Select a 1-psg $\lambda$ for which $\sigma_\alpha \lambda = -\lambda$; then the subgroup of $X(T)$ consisting of characters $\chi$ for which $\langle \chi, \lambda \rangle = 0$ does the trick.

(R3) requires that $\sigma_\alpha$ stabilize $\Phi$, but this was observed earlier for arbitrary elements of $W$ (24.1, 26.3).

Once (R4) has been verified, the abstract Weyl group $W(\Phi)$ may be identified with the subgroup of $W$ generated by all $\sigma_\alpha$ ($\alpha \in \Phi$). We can then conclude that $W(\Phi) = W$ by noting that each acts simply transitively on a set of the same cardinality: $W(\Phi)$ does so on the collection of abstract Weyl chambers in $E$ (A.4), while $W$ does so on the set $\mathfrak{B}^T$ (24.1).
