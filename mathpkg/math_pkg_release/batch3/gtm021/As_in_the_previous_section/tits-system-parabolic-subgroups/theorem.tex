Let $(G, B, N, S)$ be a Tits system with Weyl group $W$. For each subset $I \subset S$, let $W_I$ be the subgroup of $W$ generated by $I$ and set $P_I = B W_I B$.
(a) $P_I$ is a subgroup of $G$ (in particular, $P_S = G$).
(b) For $\sigma, \sigma' \in W$, $B\sigma B = B\sigma' B$ if and only if $\sigma = \sigma'$.
