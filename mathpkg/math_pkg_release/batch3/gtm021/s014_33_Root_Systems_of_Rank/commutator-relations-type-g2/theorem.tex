Let $\Phi_{\alpha\beta}$ be of type $G_2$, with positive roots $\alpha, \beta, \alpha+\beta, 2\alpha+\beta, 3\alpha+\beta, 3\alpha+2\beta$. Set $x_{\alpha+\beta} = \operatorname{Ad} n_{\beta}(x_{\alpha})$, $x_{2\alpha+\beta} = \operatorname{Ad} n_{\alpha}(x_{\alpha+\beta})$, $x_{3\alpha+\beta} = -\operatorname{Ad} n_{\alpha}(x_{\beta})$, $x_{3\alpha+2\beta} = \operatorname{Ad} n_{\beta}(x_{3\alpha+\beta})$. Then the commutation formulas hold with all structural constants equal to $\pm 1$ or $\pm 2$, and $(n_{\alpha}n_{\beta})^{6} = (n_{\beta}n_{\alpha})^{6} = e$.
