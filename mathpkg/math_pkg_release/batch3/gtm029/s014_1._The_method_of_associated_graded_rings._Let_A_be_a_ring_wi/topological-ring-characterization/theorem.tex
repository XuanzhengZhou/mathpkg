Let $A$ be a ring. If a system $\Sigma(A)$ of subsets of $A$ satisfies:

(a) The intersection of any two sets of $\Sigma(A)$ contains a third set of that system.

(b) If $U$ is any set in $\Sigma(A)$ then there exists a set $W$ in $\Sigma(A)$ such that $W - W \subset U$ and $W^2 \subset U$.

(c) If $U$ is any set in $\Sigma(A)$, $a$ any element of $U$ and $b$ any element of $A$, then there exists a set $W$ in $\Sigma(A)$ such that $W + a \subset U$ and $Wb \subset U$.

Then there exists one and only one topology in $A$ such that $A$ is a topological ring with respect to that topology and $\Sigma(A)$ is a basis of neighborhoods of $0$.
