Let $A$ be a topological ring and $E$ a topological $A$-module. Let $\Sigma(A)$ be a basis of neighborhoods of the zero of $A$ and let $\Sigma(E)$ be a basis of neighborhoods of the zero of $E$. The system $\Sigma(E)$ enjoys the following properties:

(a') The intersection of any two sets in $\Sigma(E)$ contains a third set of that system.

(b') If $U'$ is any set in $\Sigma(E)$ then there exists a set $W'$ in $\Sigma(E)$ and a set $W$ in $\Sigma(A)$ such that $W' - W' \subset U'$ and $WW' \subset U'$.

(c') If $U'$ is any set in $\Sigma(E)$, $x$ any element of $U'$ and $a$ any element of $A$, then there exists a set $W'$ in $\Sigma(E)$ and a set $W$ in $\Sigma(A)$ such that $W' + x \subset U'$ and $Wx \subset U'$.
