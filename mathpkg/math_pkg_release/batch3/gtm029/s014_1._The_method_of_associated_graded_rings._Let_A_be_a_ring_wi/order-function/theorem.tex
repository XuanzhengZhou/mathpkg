Given any element $x$ of $E$, we denote by $v(x)$ the largest integer $n$ such that $x \in \mathfrak{m}^n E$. For $x \in \bigcap_{n=0}^{\infty} \mathfrak{m}^n E$, we set $v(x) = +\infty$. Then, if $v(x)$ is finite:

$$x \in \mathfrak{m}^{v(x)} E, \quad x \notin \mathfrak{m}^{v(x)+1} E.$$

The function $v$ is called the order function on the module $E$. This definition applies also to the particular case $E = A$.
