If $\mathfrak{p} + \mathfrak{m} = A$, then $\mathfrak{m}^n + \mathfrak{p} = A$ for all $n$, so $\mathfrak{q} + \mathfrak{m}^n = A$ for all $n$ (since $\mathfrak{p} \subset \sqrt{\mathfrak{q}}$). The closure of $\mathfrak{q}$ in the $\mathfrak{m}$-topology is $\bigcap (\mathfrak{q} + \mathfrak{m}^n) = A$, which contains $\mathfrak{q}$ properly (since $\mathfrak{q}$ is primary, $\mathfrak{q} \neq A$). Hence $\mathfrak{q}$ is not closed.

Conversely, if $\mathfrak{p} + \mathfrak{m} \neq A$, then both are contained in a common maximal ideal, so $\mathfrak{p} \supset \mathfrak{m}^s$ for some $s$ (by properties of noetherian rings). Then $\mathfrak{q}$ is closed because the $\mathfrak{m}$-topology and $(\mathfrak{p}+\mathfrak{m})/\mathfrak{p}$-topology coincide.
