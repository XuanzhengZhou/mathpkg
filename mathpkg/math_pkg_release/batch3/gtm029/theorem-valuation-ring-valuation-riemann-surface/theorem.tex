The closure of an element $v$ of $S$ (i.e., the closure of the set $\{v\}$ consisting of the single element $v$) is the set of all valuations $v' \in S$ which are composite with $v$.

Proof. Suppose that $v'$ is composite with $v$, so that we have for the corresponding valuation rings the inclusion $R_{v'} \subset R_v$. If $E(\emptyset)$ is any basic open set such that $v$ belongs to the basic closed set $S - E(\emptyset)$, then $R_v \neq \emptyset$, whence a fortiori $R_{v'} \neq \emptyset$, and thus $v' \in S - E(\emptyset)$. Thus every basic closed set which contains $v$ necessarily contains $v'$, showing that $v'$ belongs to the closure of the set $\{v\}$. On the other hand, assume that $v'$ is not composite with $v$. We can then find an element $x$ of $K$ such that $v'(x)$ is non-negative while $v(x) < 0$. Then if we set $\emptyset = k[x]$ we will have $v \in S - E(\emptyset)$, $v' \notin S - E(\emptyset)$, and consequently $v'$ is not in the closure of the set $\{v\}$. This completes the proof.
