If we set $s(x) = x(1 + ax_s)$, then $C(x, s)$ is the $v^\star$-residue of $x_s$. From $s(xy) = xy(1 + ax_s + ay_s + a^2x_sy_s)$ and from $a^2 \in \mathfrak{M}_v^\star a$, we deduce formula (29). From

$$s(t(x)) = s(x)[1 + s(a)s(x_t)] = x(1 + ax_s)[(1 + a(1 + aa_s)(1 + a(x_t))x_t]$$

$$\equiv x(1 + ax_s + ax_t)(\text{mod. } \mathfrak{M}_v^\star ax),$$

we deduce formula (30).

We have again a pairing, this time between $H_a$ and the multiplicative group $K^\star'$ of $K^\star$, with values in the additive group of $\Delta^\star$.
