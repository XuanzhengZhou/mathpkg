If $\mathfrak{A}$ is a contracted ideal and $q$ is any integer $\geq 1$ then also $m^q\mathfrak{A}$ is a contracted ideal.

It is sufficient to consider the case in which $\mathfrak{A}$ is primary for $m$ since every ideal in $\mathfrak{o}$ is of the form $x\mathfrak{A}$, $x \in \mathfrak{o}$, $\mathfrak{A}$ primary for $m$. We use the notations of Theorem 1, we set $\mathfrak{o}' = \mathfrak{o}'_1 \cap \mathfrak{o}'_2 \cap \cdots \cap \mathfrak{o}'_m$ and $\mathfrak{D} = \mathfrak{o}' m^q\mathfrak{A} \cap \mathfrak{o}$. It is clear that $r(\mathfrak{D}) = q + r$. We have $\mathfrak{o}'_i\mathfrak{D} = \mathfrak{o}'_i m^q\mathfrak{A} = \mathfrak{o}'_i m^{q+r-s}_i\mathfrak{B}_i$, and since $m^{q+r-s}_i\mathfrak{B}_i$ is the contraction of its extended ideal in $\mathfrak{o}'_i$ (Proposition 3, Corollary 2), it follows that if we set

$$\mathfrak{D}_i = \mathfrak{o}'_i\mathfrak{D} \cap \mathfrak{o},$$

then

$$\mathfrak{D}_i = m^{q+r-s}_i\mathfrak{B}_i.$$
