Let $v$ be an open ideal in a semi-local ring $A$, and $x$ be an element of $A$ which is superficial of order $s$ for $v$. Then there exists an integer $c$ such that

$$P_v(n) - P_v(n-s) \leq P_{v/Ax}(n) \leq P_v(n) - P_v(n-s) + P_v(c),$$

for $n$ large enough.
