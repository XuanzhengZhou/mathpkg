The \emph{core} (algebraic interior) of $A \subset X$ is $\operatorname{cor}(A) = \{a \in A : \forall x \in X, \exists \varepsilon > 0 \text{ s.t. } a + \delta x \in A \text{ for } |\delta| \leqslant \varepsilon\}$. The \emph{intrinsic core} $\operatorname{icr}(A)$ is the core relative to $\operatorname{aff}(A)$.
