Let 𝒞 be the class of all convex sets disjoint from B containing A, partially ordered by inclusion. Zorn's lemma gives a maximal C. Set D=X\C. If D is not convex, there exist x,z∈D with (x,z)∩C≠∅. Maximality of C then forces a contradiction via convexity arguments.
