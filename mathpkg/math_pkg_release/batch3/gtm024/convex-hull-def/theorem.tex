For any $A \subset X$, the convex hull $\operatorname{co}(A)$ is the intersection of all convex sets in $X$ that contain $A$. Equivalently, $\operatorname{co}(A) = \{\sum \alpha_i x_i : 0 \leqslant \alpha_i \leqslant 1, \sum \alpha_i = 1, x_i \in A\}$, the set of all convex combinations of points in $A$.
