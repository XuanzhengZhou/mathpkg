Let $E$ be a subset of a convex set $A$ in $X$. $E$ is semi-extremal if $A \setminus E$ is convex, and $E$ is extremal if $x, y \in A$ and $tx + (1-t)y \in E$ for some $t$ $(0 < t < 1)$ entails $x, y \in E$. When $E = \{x_0\}$ is a singleton, these notions coincide and $x_0$ is an extreme point of $A$, written $x_0 \in \operatorname{ext}(A)$.
