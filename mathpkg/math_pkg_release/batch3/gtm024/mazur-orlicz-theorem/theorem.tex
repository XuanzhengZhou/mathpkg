The system $\phi(x_j) \geqslant c_j$ ($j \in J$) and $g \geqslant \phi$, where $g \in \operatorname{Conv}(X)$, has a solution $\phi \in X'$ if and only if for every finite set $\{j_1,\ldots,j_n\} \subset J$ and every set $\{\alpha_1,\ldots,\alpha_n\}$ of non-negative numbers, $\sum_{k=1}^n \alpha_k c_{j_k} \leqslant g(\sum_{k=1}^n \alpha_k x_{j_k})$.
