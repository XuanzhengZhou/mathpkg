A first-order mathematical theory is a triple $\mathcal{T} = (\mathcal{R}, A, C)$ where $\mathcal{I} \in \mathcal{R}$, $A \subseteq P(V, \mathcal{R})$ for some $V \supset C$ with $V - C$ infinite, and $\operatorname{var}(A) = C$. $A$ is the set of (mathematical) axioms, $C$ is the set of (individual) constants. The language is $\mathcal{L}(\mathcal{T}) = \{p \in P(V, \mathcal{R}) \mid \operatorname{var}(p) \subseteq C\}$. A theorem is $p \in \mathcal{L}(\mathcal{T})$ with $A \vdash p$.
