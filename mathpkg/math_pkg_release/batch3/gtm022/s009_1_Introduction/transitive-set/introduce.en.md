A set $n$ is called transitive if every element of an element of $n$ is itself an element of $n$; that is, $x \in y \in n$ implies $x \in n$. Equivalently, $\cup n \subseteq n$. This property is fundamental in the study of ordinals, as every von Neumann ordinal is a transitive set, and the transitive property ensures that the membership relation behaves like a well-ordering on the ordinal numbers.
