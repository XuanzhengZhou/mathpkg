\text{Assuming that at least one set } a \text{ exists, we can form the set}
$$\varnothing = \{x \in a \mid x \neq x\}.$$
\text{Then for all } x \text{ we have } x \notin \varnothing. \text{ This set } \varnothing, \text{ which is called the empty set, is independent of the choice of the set } a \text{ used in its construction.}
