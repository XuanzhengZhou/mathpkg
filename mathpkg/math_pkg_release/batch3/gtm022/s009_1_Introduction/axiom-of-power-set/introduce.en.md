The Axiom of the Power Set (ZF5) asserts that for any set $a$, there exists a set $b$ that contains all subsets of $a$ as elements. Using the Axiom Schema of Subsets (ZF2), one can then isolate exactly the power set $\operatorname{Pow}(a) = \{x \mid x \subseteq a\}$. This axiom is essential for forming Cartesian products and for constructions involving sets of subsets.
