The ordered pair $(x, y)$ is defined set-theoretically via the Kuratowski encoding as $\{\{x\}, \{x, y\}\}$. This definition ensures that $(x, y) = (x', y')$ if and only if $x = x'$ and $y = y'$, thereby capturing the essential property of ordered pairs using only the primitive notion of set membership. This construction is foundational for defining Cartesian products and relations within ZF set theory.
