The Axiom Schema of Subsets (ZF2), also known as the Axiom Schema of Separation or Comprehension, states that for any given set $a$ and any property $\pi$, there exists a set $b$ consisting of exactly those elements of $a$ that satisfy $\pi$. By restricting set formation to subsets of an already-existing set, this axiom avoids Russell's paradox while still allowing the construction of sets defined by properties. The resulting set is unique by the Axiom of Extension (ZF1) and is denoted $\{x \in a \mid x \text{ has } \pi\}$.
