\text{The von Neumann natural numbers are defined by:}
$$0 = \varnothing,$$
$$1 = 0^+ = \varnothing \cup \{\varnothing\} = \{\varnothing\} = \{0\},$$
$$2 = 1^+ = 1 \cup \{1\} = \{0, 1\},$$
$$3 = 2^+ = \{0, 1\} \cup \{2\} = \{0, 1, 2\},$$
\text{and so on. The set } \omega \text{ together with the usual operations of addition and multiplication will be denoted by } N.
