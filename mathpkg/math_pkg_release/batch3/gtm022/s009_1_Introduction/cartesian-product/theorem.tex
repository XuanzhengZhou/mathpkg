\text{(ZF5) allows formation of the cartesian product}
$$a \times b = \{(x, y) \mid x \in a \text{ and } y \in b\}.$$
\text{To show this, we need only produce a set } c \text{ whose members include all the required ordered pairs } (x, y). \text{ But } (x, y) = \{\{x\}, \{x, y\}\}, \text{ and } \{x\} \subseteq a \cup b.
