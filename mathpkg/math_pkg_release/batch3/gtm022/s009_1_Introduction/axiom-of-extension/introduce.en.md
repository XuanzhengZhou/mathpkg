The Axiom of Extension (also known as the Axiom of Extensionality) is the first axiom of Zermelo-Fraenkel set theory (ZF). It asserts that two sets are equal if and only if they contain exactly the same members. This axiom captures the fundamental principle that a set is completely determined by its elements, making membership the sole criterion for set identity.
