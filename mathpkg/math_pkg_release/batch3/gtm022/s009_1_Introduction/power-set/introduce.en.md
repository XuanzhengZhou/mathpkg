The power set $\operatorname{Pow}(a)$ of a set $a$ is the set of all subsets of $a$. Its existence follows from the Axiom of the Power Set (ZF5) combined with the Axiom Schema of Subsets (ZF2). The power set operation is fundamental in set theory, enabling the construction of Cartesian products, function spaces, and the cumulative hierarchy of sets.
