The Axiom of Union (ZF4) asserts that for any set $c$, there exists a set $a$ that contains all elements of the elements of $c$. This axiom is necessary because the earlier axioms (ZF1 and ZF2) allow the formation of intersections but not unions. Using ZF4 together with ZF2, one can then construct the precise union $\cup c$ as a subset of the overset $a$ provided by this axiom.
