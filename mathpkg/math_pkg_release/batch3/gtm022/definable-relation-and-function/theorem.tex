An $n$-ary relation $\rho$ on the model $M$ of $\mathcal{T}$ is definable in $\mathcal{T}$ if $\rho = \psi(p)$ for some $n$-variable formula $p$ of $\mathcal{T}$. A function $f: M^n \to M$ is definable if there is an $(n+1)$-variable formula $p$ such that $f(a_1, \ldots, a_n) = b$ iff $p(a_1, \ldots, a_n, b)$ is true, and $\mathcal{T} \vdash (\forall x_1)\cdots(\forall x_n)(\exists! y) p(x_1, \ldots, x_n, y)$.
