Let $\mathcal{S} = \{B \subseteq A \mid B \text{ is a } T\text{-subalgebra of } A \text{ and } X \subseteq B\}$. The set $\mathcal{S}$ is non-empty since $A \in \mathcal{S}$. Define $\langle X \rangle = \bigcap_{B \in \mathcal{S}} B$. By the proposition that any intersection of $T$-subalgebras is a $T$-subalgebra, $\langle X \rangle$ is a $T$-subalgebra of $A$. Clearly $X \subseteq \langle X \rangle$, so $\langle X \rangle \in \mathcal{S}$. If $C$ is any $T$-subalgebra containing $X$, then $C \in \mathcal{S}$, so $\langle X \rangle \subseteq C$. Thus $\langle X \rangle$ is the unique smallest $T$-subalgebra containing $X$.
