A \textbf{type} $\mathcal{T}$ is a set $T$ together with a function $\operatorname{ar}: T \rightarrow \mathbb{N}$, from $T$ into the non-negative integers. We write $\mathcal{T} = (T, \operatorname{ar})$, or abuse notation and denote the type simply by $T$. For each $n \in \mathbb{N}$, we define $T_n = \{t \in T \mid \operatorname{ar}(t) = n\}$, the set of operation symbols of arity $n$.
