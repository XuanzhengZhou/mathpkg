A type (or signature) in universal algebra is a pair $\mathcal{T} = (T, \operatorname{ar})$ consisting of a set $T$ of operation symbols and an arity function $\operatorname{ar}: T \rightarrow \mathbb{N}$ that assigns to each symbol its number of arguments. This definition overcomes the objection that operations defined merely as functions on a single set cannot distinguish operations of the same arity on different sets. The set $T_n = \{t \in T \mid \operatorname{ar}(t) = n\}$ collects all $n$-ary operation symbols, allowing a unified treatment of algebraic structures with arbitrary operation signatures.
