A \emph{logic} $\mathcal{L}$ is a system consisting of a set $P$ of elements (called \emph{propositions}), a set $\mathcal{V}$ of functions (called \emph{valuations}) from $P$ into some value set $W$, and, for each subset $A$ of $P$, a set of finite sequences of elements of $P$ (called \emph{proofs} from the assumptions $A$).
