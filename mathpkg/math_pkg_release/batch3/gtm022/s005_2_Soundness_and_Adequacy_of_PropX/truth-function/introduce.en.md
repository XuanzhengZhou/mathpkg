A truth function is any function $f: \mathbb{Z}_2^n \to \mathbb{Z}_2$ from $n$-tuples of Boolean values to a Boolean value. These are the semantic counterparts of propositional formulas: every propositional word in $n$ variables induces a truth function via evaluation under valuations, and conversely every truth function can be represented by a propositional formula (a result known as functional completeness). The set of all truth functions on $n$ variables has cardinality $2^{2^n}$.
