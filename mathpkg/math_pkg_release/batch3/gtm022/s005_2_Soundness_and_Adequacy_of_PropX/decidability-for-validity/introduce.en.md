Theorem 3.5 states that the propositional calculus $\text{Prop}(X)$ is decidable for validity: there exists an algorithm (the truth-table method) that determines in a finite number of steps whether any given propositional formula is valid. The algorithm evaluates the associated truth function $\bar{w}(a_1, \ldots, a_n)$ for all $2^n$ assignments of Boolean values to the $n$ variables; the formula is valid if and only if every evaluation yields $1$.
