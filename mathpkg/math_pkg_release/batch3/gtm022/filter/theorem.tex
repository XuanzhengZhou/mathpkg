A set $\mathcal{F}$ of subsets of $I$ is a \emph{filter} on $I$ if:
(i) $\varnothing \notin \mathcal{F}$,
(ii) Every subset of $I$ which contains a set of $\mathcal{F}$ belongs to $\mathcal{F}$,
(iii) Every finite intersection of sets of $\mathcal{F}$ belongs to $\mathcal{F}$.
An \emph{ultrafilter} is a filter satisfying additionally:
(iv) For each $A \subseteq I$, exactly one of $A$ and $I \setminus A$ belongs to $\mathcal{F}$.
