Let $M_n$ denote the $n$th Turing machine and let $f_n = \Psi_{M_n}^{(1,1)}$. The stopping problem — to determine for all integers $n, r$ whether or not there is a computation of $M_n$ beginning with the state $q_0\,\text{code}(r)$ — is recursively insoluble. In particular, there is no Turing machine which computes for each $n$ whether or not $f_n(n)$ exists.
