The language of an elementary standard system $\mathbb{S}$ consists of all formulas in the first-order algebra $P(V, \mathcal{R})$ whose free variables range only over elements of $S$. Interpreting each element of $S$ and $\mathcal{R}$ as itself assigns truth values to these formulas. This language is the medium through which the complete theory $\mathcal{T}(\mathbb{S})$ expresses all truths about $\mathbb{S}$.
