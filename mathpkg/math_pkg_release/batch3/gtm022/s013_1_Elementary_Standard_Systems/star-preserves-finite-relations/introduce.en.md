The star operation $*\rho$, which extends a definable relation $\rho$ from $\mathbb{S}$ to its enlargement, coincides with the original relation exactly when $\rho$ is finite. For finite relations, a description listing all members forces $*\rho = \rho$. For infinite relations, one can construct a concurrent relation that introduces new elements into $*\rho$, so the extension is always proper.
