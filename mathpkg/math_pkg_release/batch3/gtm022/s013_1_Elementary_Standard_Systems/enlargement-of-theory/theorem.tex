Let $\sigma$ be any set of concurrent relations of the standard system $\mathbb{S}$. For each $\rho \in \sigma$, we take a new variable $c_\rho \notin V$ and form $V^\sigma = V \cup \{c_\rho \mid \rho \in \sigma\}$, $P^\sigma = P(V^\sigma, \mathcal{R})$. We put

$$A_\sigma = \{ \rho(x, c_\rho) \mid \rho \in \sigma \text{ and } x \in D_\rho \}.$$

The enlargement of $\mathcal{T} = (\mathcal{R}, A, \mathbb{S})$ with respect to $\sigma$ is the theory $\mathcal{T}^\sigma = (\mathcal{R}, A \cup A_\sigma, \mathbb{S} \cup \{c_\rho \mid \rho \in \sigma\})$. When $\sigma$ is the set of all concurrent relations of $\mathbb{S}$, we call $\mathcal{T}^\sigma$ the full enlargement of $\mathcal{T}$, and denote it by $*\mathcal{T}$.
