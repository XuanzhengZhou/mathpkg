The initial functions form the atomic building blocks from which all recursive functions can be constructed. They consist of three simple primitives: the constant zero function, the successor function that increments by one, and the projection functions that select a coordinate from a tuple. Together with the operations of composition, primitive recursion, and minimalisation, the set of initial functions generates the entire class of recursive functions, providing an algebraic characterisation that avoids explicit construction of Turing machines.
