This theorem provides an algebraic characterisation of the class of recursive functions without any reference to Turing machines. It states that the recursive functions are exactly those that can be built from the three initial functions---zero, successor, and projections---by finitely many applications of composition, primitive recursion, and minimalisation. The result is remarkable because it offers strong support for Church's Thesis: it shows that the class of functions obtained by these simple algebraic operations coincides with the class of Turing-computable functions, suggesting that all reasonable formalisations of effective computability are equivalent.
