Proof. If $\lim s(n) = r$ in the standard sense, then for any $\varepsilon > 0$ there exists $n_0$ such that $|s(n) - r| < \varepsilon$ for $n > n_0$. This property is expressible in $\mathcal{T}(\mathcal{O}^1(\mathbb{R}))$. For any infinite $n$, $n > n_0$ holds for all standard $n_0$, hence $|*s(n) - r| < \varepsilon$ for all standard $\varepsilon > 0$, so $*s(n) \cong r$. The converse follows by similar transfer arguments. $\square$
