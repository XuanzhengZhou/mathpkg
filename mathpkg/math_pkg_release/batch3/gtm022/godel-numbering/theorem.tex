Define a function $G: \{L, R\} \cup \mathfrak{S}^* \cup \mathfrak{Q}^* \to \mathbb{N}$ by $G(L) = 1, G(R) = 3, G(s_i) = 4i + 5, G(q_j) = 4j + 7$. For a string $a_1 \cdots a_r$, $G(a_1 \cdots a_r) = p_1^{G(a_1)} \cdots p_r^{G(a_r)}$ where $p_k$ is the $k$th prime. For a sequence of strings $\sigma_1, \ldots, \sigma_s$, $G(\sigma_1, \ldots, \sigma_s) = p_1^{G(\sigma_1)} \cdots p_s^{G(\sigma_s)}$. The Godel number of a Turing machine $M$ is the Godel number of the unique finite sequence of strings associated with $M$.
