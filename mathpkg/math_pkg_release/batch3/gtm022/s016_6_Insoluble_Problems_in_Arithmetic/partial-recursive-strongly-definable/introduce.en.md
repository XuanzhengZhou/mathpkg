Theorem 6.9 establishes that every partial recursive function is strongly definable within any theory $\mathcal{T}$ that extends recursive arithmetic $N_0$ and has the natural numbers $\mathbb{N}$ as a model. The proof uses the sequence number function (Lemma 6.5) to encode Turing machine computations as formulas within the formal language, together with the pairing functions $P$, $L$, $R$ to handle multiple arguments. This result is the crucial bridge between computability theory and formal arithmetic, showing that sufficiently strong arithmetical theories can "speak about" all computable functions.
