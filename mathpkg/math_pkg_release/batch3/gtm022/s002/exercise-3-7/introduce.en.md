This exercise asks the reader to show that left $R$-modules form a variety. The type includes the group operations (nullary, unary, binary) together with a unary operation for each element of $R$ (scalar multiplication). The laws include the abelian group axioms together with module axioms: $r(x+y) = rx + ry$, $(r+s)x = rx + sx$, $(rs)x = r(sx)$, and $1_R x = x$.
