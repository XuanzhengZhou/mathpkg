For $\mathcal{F}$ $\alpha$-complete, each $f\mathcal{F}$ equals some constant $m$ since $\{J_m = f^{-1}(m)\}$ partitions $I$ into $\alpha$ sets, so some $J_m \in \mathcal{F}$. Conversely, if $\mathcal{F}$ is $\alpha$-incomplete, $\alpha$ is infinite and one constructs a function that is not almost everywhere constant, giving a non-standard element.
