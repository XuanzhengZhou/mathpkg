Let $M$ be a Turing machine with alphabet $\mathfrak{S}$. For $(n_1, \ldots, n_k) \in \mathbb{N}^k$, the input code is $\operatorname{code}(n_1, \ldots, n_k) = s_1^{n_1} s_0 s_1^{n_2} s_0 \cdots s_0 s_1^{n_k}$. $U_M^{(k,\ell)}$ is the set of $(n_1, \ldots, n_k) \in \mathbb{N}^k$ for which $M$, starting in state $q_0 \operatorname{code}(n_1, \ldots, n_k)$, eventually stops with tape $s_1^{m_1} s_0 \cdots s_0 s_1^{m_\ell}$. The corresponding function $\Psi_M^{(k,\ell)}: U_M^{(k,\ell)} \to \mathbb{N}^\ell$ defined by $\Psi_M^{(k,\ell)}(n_1, \ldots, n_k) = (m_1, \ldots, m_\ell)$ is called a partial recursive function.
