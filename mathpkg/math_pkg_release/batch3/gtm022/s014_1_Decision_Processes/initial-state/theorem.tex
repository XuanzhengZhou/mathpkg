A state $[\sigma q \tau]$ is called \textbf{initial} if $q = q_0$.

Here $q_0$ is the distinguished initial internal state of the Turing machine, and $\sigma q \tau$ denotes an instantaneous description with tape contents $\sigma$ and $\tau$ and current internal state $q$ scanning the first symbol of $\tau$ (or a blank if $\tau$ is empty).
