A terminal state is a configuration of a Turing machine for which no quadruple in the machine's program matches the current internal state and scanned symbol. When the machine reaches a terminal state, it halts — no further action is specified. The terminal state thus formalizes the stopping condition corresponding to response type (iv) in the informal description of Turing machine actions.
