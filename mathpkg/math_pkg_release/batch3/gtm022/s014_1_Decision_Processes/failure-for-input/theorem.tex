We say that $M$ \textbf{fails} for the input $[d_0]$ if there is no computation by $M$ beginning with the state $[d_0]$.

For each state $[d_i]$, there is a unique $[d_{i+1}]$ such that $[d_i] \xrightarrow{M} [d_{i+1}]$. Hence failure of $M$ for the input $[d_0]$ means that the sequence of states taken by $M$ and beginning with $[d_0]$ is infinite --- i.e., the machine never stops.
