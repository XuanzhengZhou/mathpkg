Show that there is a Turing machine $U$ with the property that, for each Turing machine $M$ and shortest description $d$ of an initial state, $U$ started in the state $q \operatorname{code}(G(M), G(d))$

\begin{enumerate}
\item[(i)] fails if $M$ fails in the state $d$,
\item[(ii)] computes $G(d_t)$ if $d_t$ is the shortest description of the terminal state reached by $M$ starting from $d$.
\end{enumerate}

A machine such as $U$ is called a \textbf{universal Turing machine}.
