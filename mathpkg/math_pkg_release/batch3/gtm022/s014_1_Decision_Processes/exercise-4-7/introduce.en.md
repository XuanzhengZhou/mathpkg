This exercise asks the reader to construct a universal Turing machine -- a single machine that can simulate the behavior of any other Turing machine given its Gödel number and initial state description. The universal machine fails on inputs where the simulated machine fails, and otherwise computes the Gödel number of the terminal state. This is the theoretical foundation for the stored-program concept in modern computers.
