A Turing machine with (finite) alphabet $\mathfrak{S}$ and (finite) set $\mathfrak{Q} = \{q_0, q_1, \ldots, q_m\}$ of internal states is a subset $M$ of $\mathfrak{Q} \times (\mathfrak{S} \cup \{L, R\}) \times (\mathfrak{S} \cup \{L, R\}) \times \mathfrak{Q}$ such that at most one quadruple begins with any given pair $(q_i, s_j)$. A quadruple $(q_i, s_j, s_t, q_r)$ means: when in state $q_i$ scanning symbol $s_j$, replace $s_j$ by $s_t$ and change to state $q_r$. $(q_i, s_j, R, q_r)$ means move right; $(q_i, s_j, L, q_r)$ means move left. If no quadruple begins with $(q_i, s_j)$, the machine stops.
