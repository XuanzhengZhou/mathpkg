The Halting Problem: there is no Turing machine that can decide, for every machine-input pair, whether the machine will halt. The proof uses a diagonal argument: if such a decision procedure existed, one could construct a machine that halts exactly when it does not halt, a contradiction. This was proved by Turing in 1936 and is the fundamental result of computability theory.
