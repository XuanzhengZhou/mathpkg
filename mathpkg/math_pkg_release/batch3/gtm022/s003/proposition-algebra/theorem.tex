Let $T = \{F, \Rightarrow\}$, where $F$ is a $0$-ary operation (representing falsehood) and $\Rightarrow$ is a binary operation (representing implication). For any set $X$, the proposition algebra $P(X)$ is the free $T$-algebra on the set $X$ of propositional variables.

The other logical connectives are defined as abbreviations:
$$\sim p = p \Rightarrow F \quad \text{(negation)}$$
$$p \lor q = (\sim p) \Rightarrow q \quad \text{(disjunction)}$$
$$p \land q = \sim((\sim p) \lor (\sim q)) \quad \text{(conjunction)}$$
$$p \Leftrightarrow q = (p \Rightarrow q) \land (q \Rightarrow p) \quad \text{(equivalence)}$$
