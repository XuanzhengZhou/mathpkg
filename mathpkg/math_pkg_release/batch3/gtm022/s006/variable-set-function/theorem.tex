For $w \in \tilde{P}(V, \mathcal{R})$, the set $V(w)$ of variables appearing in $w$ is defined inductively: (i) $V(F) = \emptyset$. (ii) If $r \in \mathcal{R}, \text{ar}(r)=n$, and $x_1,\ldots,x_n \in V$, then $V(r(x_1,\ldots,x_n)) = \{x_1,\ldots,x_n\}$. (iii) $V(w_1 \Rightarrow w_2) = V(w_1) \cup V(w_2)$. (iv) $V((\forall x)w) = \{x\} \cup V(w)$.
