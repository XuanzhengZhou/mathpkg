\text{(The Substitution Theorem). Let } X, Y \text{ be any two sets, and let } \varphi: P(X) \rightarrow P(Y) \text{ be a homomorphism of the (free) proposition algebra on } X \text{ into the (free) proposition algebra on } Y. \text{ Let } w = w(x_1, \ldots, x_n) \text{ be any element of } P(X) \text{ and let } A \text{ be any subset of } P(X). \text{ Put } a_i = \varphi(x_i).
