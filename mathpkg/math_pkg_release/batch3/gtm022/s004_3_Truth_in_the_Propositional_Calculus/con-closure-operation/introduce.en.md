The semantic consequence operator $\text{Con}$, which maps a set of assumptions $A \subseteq P(X)$ to the set $\text{Con}(A)$ of all propositions that are true under every valuation making all of $A$ true, is shown to be a closure operation on the proposition algebra $P(X)$. A closure operation on a set satisfies monotonicity (if $A_1 \subseteq A_2$ then $\text{Con}(A_1) \subseteq \text{Con}(A_2)$), extensivity ($A \subseteq \text{Con}(A)$), and idempotence ($\text{Con}(\text{Con}(A)) = \text{Con}(A)$). This lemma establishes that logical consequence behaves like a topological closure, a perspective that will be essential for the completeness theorem relating semantic consequence ($\models$) to syntactic deduction ($\vdash$).
