A type in universal algebra is a set $T$ equipped with an arity function mapping each operation symbol to a non-negative integer. This generalizes the notion of an algebraic signature, specifying the names and arities of all operations in an algebraic system. The set of $n$-ary operation symbols is $T_n = \{t \in T \mid \operatorname{ar}(t) = n\}$.
