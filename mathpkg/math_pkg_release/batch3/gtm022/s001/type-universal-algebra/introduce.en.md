A type (also called a signature or similarity type) in universal algebra specifies a set of operation symbols together with their arities. It is the fundamental data structure that defines what it means to be an algebra of a given type. Each operation symbol is assigned a non-negative integer indicating how many arguments it takes, with $0$-ary operations corresponding to distinguished constants.
