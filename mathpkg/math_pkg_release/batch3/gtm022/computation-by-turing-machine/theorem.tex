A computation by the machine $M$ is a finite sequence $[d_0], [d_1], \ldots, [d_p]$ of states (equivalence classes of instantaneous descriptions) such that $[d_0]$ is initial ($q = q_0$), $[d_p]$ is terminal (no quadruple applies), and $[d_i] \xrightarrow{M} [d_{i+1}]$ for $i = 0, 1, \ldots, p - 1$. We say $M$ fails for the input $[d_0]$ if there is no computation beginning with $[d_0]$.
