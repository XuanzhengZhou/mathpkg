Let $V$ be an infinite set of individual variables, and let $\mathcal{R}$ be a set of relation (predicate) symbols together with an arity function $\text{ar} : \mathcal{R} \rightarrow \mathbb{N}$. Then $\tilde{P}(V, \mathcal{R})$ is the free algebra on the set
$$\{(r, x_1, \ldots, x_n) \mid r \in \mathcal{R},\; x_i \in V,\; n = \text{ar}(r)\}$$
of free generators, of type $\{F, \Rightarrow, (\forall x) \mid x \in V\}$, where $F$ is a $0$-ary operation, $\Rightarrow$ is binary, and each $(\forall x)$ is unary.
