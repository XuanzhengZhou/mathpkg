This exercise asks for a proof that every congruence class $w \in P(V, \mathcal{R})$ has a representative in which each variable is quantified at most once. This is an important technical property: it ensures that in any formula we can avoid repeated binding of the same variable, which simplifies reasoning about substitution and avoids variable capture when applying axioms such as $\mathcal{A}_5$.
