The monad of the finite real number $a$ is $\mu(a) = \{r \in *\mathbb{R} \mid r \cong a\}$. If $R_0$ is the set of finite real numbers and $R_1$ the set of infinitesimals, then $R_0$ is a ring, $R_1$ is an ideal of $R_0$ and $R_0/R_1$ is isomorphic to $\mathbb{R}$. Each monad contains exactly one standard real number.
