The Axiom Schema of Separation (ZF2) states that for any set $a$ and any predicate $\pi(x)$ expressible in the language of $\mathcal{S}$, there exists a set $b$ whose elements are precisely those $x \in a$ that satisfy $\pi(x)$. In the formal language, $\pi(x) \in P(V, \mathcal{R})$ is an element of the predicate set built from the binary relation symbols $\mathcal{I}$ and $\in$ over the variable set $V$. The notation $\pi(x)$ describes the dependence of $\pi$ upon $x$, but does not imply $\text{var}(\pi(x)) = \{x\}$ (other free variables may appear).
