The ordered pair $(a, b)$ is defined via the Kuratowski construction as $c = \{\{a\}, \{a, b\}\}$. This encoding, expressed as an abbreviation in the formal language $\mathcal{S}$, allows ordered pairs to be constructed entirely within set theory, satisfying the fundamental property that $(a, b) = (c, d)$ if and only if $a = c$ and $b = d$.
