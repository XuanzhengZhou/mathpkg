The theory $\mathcal{S}$ is consistent because any proof of a contradiction $F$ in $\mathcal{S}$ has an informal equivalent. Since the book (and much of mathematics) is written in the context of the informal set theory of ordinary mathematics, and all of this would be destroyed if that informal set theory is inconsistent, the consistency of informal set theory is assumed. With this assumption, no proof of $F$ can exist in $\mathcal{S}$, for it would translate to a proof of a contradiction in informal set theory. Hence $\mathcal{S}$ is a consistent theory.
