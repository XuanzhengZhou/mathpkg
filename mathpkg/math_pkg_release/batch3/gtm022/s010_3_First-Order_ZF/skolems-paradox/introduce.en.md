Skolem's Paradox is the observation that while ZF set theory proves the existence of uncountable sets (such as $\mathbb{R}$), the Löwenheim-Skolem Theorem guarantees that ZF has a countable model. The paradox is resolved by distinguishing between "real" (external) and "model" (internal) notions. A model set is at most real-countable, but it is model-countable only if there is a model function from the model natural numbers onto it. The real function that witnesses countability need not be a model function, so sets can be real-countable without being model-countable. Thus no contradiction arises.
