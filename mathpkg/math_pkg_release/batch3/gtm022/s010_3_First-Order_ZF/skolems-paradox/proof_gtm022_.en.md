A theorem of ordinary set theory asserts the existence of uncountable sets, and this theorem (with its proof) can be formalized in the theory $\mathcal{S}$. Hence there exists a countable model of a theory which has as a theorem the existence of uncountable sets. The paradox is resolved when we realise that it arises by using the word "set" in two ways.

Let us distinguish words used in their ordinary sense from the same words used in the sense of the model by using the adjectives *real* or *model* respectively. "$\mathcal{S}$ has a countable model" then becomes "$\mathcal{S}$ has a real countable model," i.e., there is a real function from the real set of natural numbers onto the underlying set of the model. For this model, every model set is at most real countable. But a model set is model countable only if there is a model function from the model set of natural numbers onto it, and the real function which counts it need not be a model function.
