The Zermelo-Fraenkel axioms are formalized as a first-order theory denoted by $\mathcal{S}$. The theory uses two binary relation symbols: $\mathcal{I}$ for equality and $\in$ for membership. No individual constants are introduced. The language is built from an infinite set $V$ of variables, and the predicate set $P(V, \mathcal{R})$ with $\mathcal{R} = \{\mathcal{I}, \in\}$ forms the well-formed formulas. For readability, $x \in y$ is written instead of the formally correct ${\in}(x, y)$, and $x = y$ instead of $\mathcal{I}(x, y)$.
