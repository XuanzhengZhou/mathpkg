The set-builder notation $a = \{x \mid p(x)\}$ is an abbreviation in the formal language $\mathcal{S}$ for the formula $(\forall x)(x \in a \Leftrightarrow p(x))$, where $p(x) \in P(V, \mathcal{R})$ is a predicate. This asserts that $a$ is the set whose elements are exactly those $x$ satisfying the predicate $p$.
