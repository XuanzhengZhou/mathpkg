A \emph{first-order mathematical theory} is a triple $\mathcal{T} = (\mathcal{R}, A, C)$ where $\mathcal{I} \in \mathcal{R}$, $A \subseteq P(V, \mathcal{R})$ for some $V \supseteq C$ such that $V \setminus C$ is infinite, and $\operatorname{var}(A) = C$. The set $A$ is called the set of (mathematical) axioms of $\mathcal{T}$, the set $C$ is called the set of (individual) constants, while the \emph{language} is $\mathcal{L}(\mathcal{T}) = \{p \in P(V, \mathcal{R}) \mid \operatorname{var}(p) \subseteq C\}$. A \emph{theorem} of $\mathcal{T}$ is $p \in \mathcal{L}(\mathcal{T})$ with $A \vdash p$.
