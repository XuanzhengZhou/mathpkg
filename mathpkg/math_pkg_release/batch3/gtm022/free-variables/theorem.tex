Let $w \in P$. We define the set $\operatorname{var}(w)$ of (free) variables of $w$ by $\operatorname{var}(w) = \operatorname{var}(\tilde{w})$, where $\tilde{w} \in \tilde{P}$ is some representative, and:
(i) $\operatorname{var}(F) = \varnothing$,
(ii) $\operatorname{var}(r(x_1, \ldots, x_n)) = \{x_1, \ldots, x_n\}$ for $r \in \mathcal{R}$,
(iii) $\operatorname{var}(\tilde{w}_1 \Rightarrow \tilde{w}_2) = \operatorname{var}(\tilde{w}_1) \cup \operatorname{var}(\tilde{w}_2)$,
(iv) $\operatorname{var}((\forall x)\tilde{w}) = \operatorname{var}(\tilde{w}) - \{x\}$.
For $A \subseteq P$, $\operatorname{var}(A) = \bigcup_{p \in A} \operatorname{var}(p)$.
