(i) $S \subseteq \mathbb{N}^n$ is diophantine if there is a polynomial $P(x_1, \ldots, x_n, y_1, \ldots, y_m)$ in $m+n$ variables with integer coefficients such that $(x_1, \ldots, x_n) \in S$ iff $(\exists y_1, \ldots, y_m) P(x_1, \ldots, x_n, y_1, \ldots, y_m) = 0$.
(ii) A relation $\rho$ on $\mathbb{N}^n$ is diophantine if its truth set is diophantine.
(iii) A function $f: \mathbb{N}^n \to \mathbb{N}$ is diophantine if its graph is diophantine.
