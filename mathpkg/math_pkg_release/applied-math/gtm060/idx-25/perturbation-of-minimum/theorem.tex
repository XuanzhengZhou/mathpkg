Suppose that the function $f(x)$ has a minimum at $x = 0$ and Taylor expansion
$$f(x) = \frac{A x^2}{2} + \cdots, \quad A > 0.$$
Suppose that the function $h(x)$ has Taylor expansion
$$h(x) = B + Cx + \cdots.$$
Then, for sufficiently small $\varepsilon$, the function
$$f_\varepsilon(x) = f(x) + \varepsilon h(x)$$
has a minimum at the point
$$x_\varepsilon = -\frac{C \varepsilon}{A} + O(\varepsilon^2),$$
which is close to zero. In addition,
$$f''_\varepsilon(x_\varepsilon) = A + O(\varepsilon).$$
