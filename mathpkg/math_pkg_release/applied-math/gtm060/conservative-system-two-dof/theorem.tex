A system $\ddot{x} = f(x)$, $x \in E^2$ is said to be conservative if there exists a function $U: E^2 \to \mathbb{R}$ such that $f = -\partial U / \partial x$, i.e.,
$$\ddot{x} = -\frac{\partial U}{\partial x}.$$

\textbf{Remark.} In a system with one degree of freedom it is always possible to introduce the potential energy $U(x) = -\int_{x_0}^x f(\xi)d\xi$. For a system with two degrees of freedom this is not so.
