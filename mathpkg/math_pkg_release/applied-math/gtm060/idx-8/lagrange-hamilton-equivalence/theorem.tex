The system of Lagrange's equations $\dot{\mathbf{p}} = \partial L / \partial \mathbf{q}$, where $\mathbf{p} = \partial L / \partial \dot{\mathbf{q}}$, with a given Lagrangian function $L: \mathbb{R}^n \times \mathbb{R}^n \times \mathbb{R} \to \mathbb{R}$ assumed to be convex with respect to the second argument $\dot{\mathbf{q}}$, is equivalent to the system of $2n$ first-order Hamilton's equations (canonical equations):

$$\dot{\mathbf{p}} = -\frac{\partial H}{\partial \mathbf{q}}, \qquad \dot{\mathbf{q}} = \frac{\partial H}{\partial \mathbf{p}},$$

where $H(\mathbf{p}, \mathbf{q}, t) = \mathbf{p} \dot{\mathbf{q}} - L(\mathbf{q}, \dot{\mathbf{q}}, t)$ is the Legendre transform of the Lagrangian function viewed as a function of $\dot{\mathbf{q}}$.
