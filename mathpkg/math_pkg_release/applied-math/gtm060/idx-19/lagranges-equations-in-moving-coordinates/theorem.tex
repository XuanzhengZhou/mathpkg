If the trajectory $\gamma: q = \Phi(t)$ of Lagrange's equations

$$\frac{d}{dt}\frac{\partial L}{\partial \dot{q}} = \frac{\partial L}{\partial q}$$

is written as $\gamma: Q = \tilde{\Phi}(t)$ in the local coordinates $Q, t$ (where $Q = Q(q, t)$), then the function $\tilde{\Phi}(t)$ satisfies Lagrange's equations

$$\frac{d}{dt}\frac{\partial L'}{\partial \dot{Q}} = \frac{\partial L'}{\partial Q},$$

where $L'(Q, \dot{Q}, t) = L(q, \dot{q}, t)$.
