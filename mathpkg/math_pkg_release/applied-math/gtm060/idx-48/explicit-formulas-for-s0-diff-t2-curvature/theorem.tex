The scalar product, commutator, operation $B$, connection, and curvature of a right-invariant metric on the group $S_0 \text{Diff} T^2$ are given by the following explicit formulas:

$$\langle e_k, e_l \rangle = 0 \quad \text{for } k + l \neq 0,$$

$$\langle e_k, e_{-k} \rangle = k^2 S;$$

$$[e_k, e_l] = (k \wedge l) e_{k+l};$$

$$B(e_k, e_l) = b_{k,l} e_{k+l}, \quad \text{where } b_{k,l} = (k \wedge l) \frac{k^2}{(k+l)^2};$$

$$\nabla_{e_k} e_l = d_{l,k+l} e_{k+l}, \quad \text{where } d_{u,v} = \frac{(v \wedge u)(u \cdot v)}{v^2};$$

$$R_{k,l,m,n} = 0 \text{ if } k + l + m + n \neq 0; \text{ if } k + l + m + n = 0, \text{ then } R_{k,l,m,n} = (a_{ln} a_{km} - a_{lm} a_{kn}) S, \text{ where } a_{uv} = (u \wedge v)^2 / |u + v|.$$

Here $S$ is the area of the torus, $u \wedge v$ is the area of the parallelogram spanned by $u$ and $v$ (with respect to the chosen orientation of $\mathbb{R}^2$), parentheses denote the euclidean scalar product in the plane, and angled brackets denote the scalar product in the Lie algebra.
