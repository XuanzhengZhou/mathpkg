\textbf{Definition.} A system is called conservative if the forces depend only on the location of a point in the system $(F = F(r))$, and if the work of $F$ along any path depends only on the initial and final points:
$$\int_{M_1}^{M_2} (F, dr) = \Phi(M_1, M_2).$$

\textbf{Theorem.} For a system to be conservative it is necessary and sufficient that there exist a potential energy, i.e., a function $U(r)$ such that
$$F = -\frac{\partial U}{\partial r}.$$

\textbf{Theorem.} The total energy of a conservative system $E = T + U$ is preserved under the motion: $E(t_1) = E(t_0)$.
