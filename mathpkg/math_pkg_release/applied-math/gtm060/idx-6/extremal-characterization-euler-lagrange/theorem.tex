The curve $\gamma: x = x(t)$ is an extremal of the functional

$$\Phi(\gamma) = \int_{t_0}^{t_1} L(x, \dot{x}, t)\, dt$$

on the space of curves passing through the points $x(t_0) = x_0$ and $x(t_1) = x_1$, if and only if

$$\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{x}} \right) - \frac{\partial L}{\partial x} = 0$$

along the curve $x(t)$.
