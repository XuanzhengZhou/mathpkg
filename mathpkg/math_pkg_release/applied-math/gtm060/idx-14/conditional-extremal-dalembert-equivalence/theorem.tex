A curve $\mathbf{x} : \mathbb{R} \to M \subset \mathbb{R}^N$ is a conditional extremal of the action functional (i.e., satisfies $\delta_M \Phi = 0$) if and only if it satisfies D'Alembert's equation

$$\left( \ddot{\mathbf{x}} + \frac{\partial U}{\partial \mathbf{x}}, \xi \right) = 0, \quad \forall \xi \in TM_{\mathbf{x}}.$$
