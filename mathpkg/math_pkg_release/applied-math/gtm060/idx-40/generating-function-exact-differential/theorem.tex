Suppose that the $2n$ functions $\mathbf{P}(\mathbf{p}, \mathbf{q})$ and $\mathbf{Q}(\mathbf{p}, \mathbf{q})$ of the $2n$ variables $\mathbf{p}$ and $\mathbf{q}$ give a canonical transformation $g: \mathbb{R}^{2n} \to \mathbb{R}^{2n}$. Then the $1$-form $\mathbf{p} \, d\mathbf{q} - \mathbf{P} \, d\mathbf{Q}$ is an exact differential:
$$\mathbf{p} \, d\mathbf{q} - \mathbf{P} \, d\mathbf{Q} = dS(\mathbf{p}, \mathbf{q}).$$
The function $S$ is called a generating function of the canonical transformation. Conversely, if this form is an exact differential, then the transformation is canonical.
