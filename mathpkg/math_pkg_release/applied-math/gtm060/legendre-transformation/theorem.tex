Let $f(x)$ be a convex function ($f''(x) > 0$). Its Legendre transform is the function $g(p)$ defined by
$$g(p) = \max_x (px - f(x)) = F(p, x(p)),$$
where $F(p, x) = px - f(x)$ and $x(p)$ is the point where $\partial F/\partial x = 0$, i.e., $f'(x) = p$.

For a convex function of a vector variable, $g(p) = \max_x ((p, x) - f(x))$, where $p = \partial f/\partial x$.
