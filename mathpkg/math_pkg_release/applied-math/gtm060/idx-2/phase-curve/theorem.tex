The image of a motion $\varphi: \mathbb{R} \to \mathbb{R}^2$ in the phase plane is called the \textit{phase curve}.
The phase curve is given by the parametric equations
$$x = \varphi(t), \quad y = \dot{\varphi}(t).$$
On each phase curve the value of the total energy is constant; therefore, each phase curve lies entirely in one energy level set $E(x, y) = h$.
