A phase point is a point in the phase plane representing the instantaneous state $(x, \dot{x})$ of a mechanical system.
A motion of the system corresponds to a curve traced by a phase point moving according to the phase velocity vector field.
The uniqueness theorem for ordinary differential equations guarantees that through every phase point there passes exactly one phase curve.
