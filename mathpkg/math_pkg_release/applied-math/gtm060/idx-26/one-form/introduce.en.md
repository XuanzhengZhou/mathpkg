A $1$-form is the simplest kind of exterior differential form: a linear functional on a real vector space $\mathbb{R}^n$. It assigns a real number to each vector in a linear way. The set of all $1$-forms on $\mathbb{R}^n$ forms the dual vector space $(\mathbb{R}^n)^*$, which is itself $n$-dimensional. In coordinates $x_1, \ldots, x_n$, every $1$-form can be expressed as $\omega = a_1 x_1 + \cdots + a_n x_n$ with $a_i \in \mathbb{R}$.
