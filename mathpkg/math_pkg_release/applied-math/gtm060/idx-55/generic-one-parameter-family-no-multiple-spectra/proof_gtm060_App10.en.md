This follows directly from Theorem 1, which asserts that the set of ellipsoids of revolution has codimension at least 2 in the manifold of all ellipsoids. A one-parameter family is a curve (a 1-dimensional submanifold) in this manifold. By transversality, a generic curve can be perturbed to avoid a subset of codimension 2. The interior of any path from the original system to a more rigid system can be removed by an arbitrarily small perturbation from the set of systems with multiple spectra. The case of systems with multiple spectrum is handled by passage to the limit.
