A symplectic action of a Lie group $G$ on a symplectic manifold $(M^{2n}, \omega^2)$ for which single-valued Hamiltonian functions exist defines a two-dimensional cohomology class of the Lie algebra of $G$. This cohomology class is an invariant of the Poisson action and measures the obstruction to the existence of a genuine Hamiltonian action (moment map).
