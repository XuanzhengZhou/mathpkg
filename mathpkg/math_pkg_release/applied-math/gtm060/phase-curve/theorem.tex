The image of a solution $\varphi: \mathbb{R} \rightarrow \mathbb{R}^2$ is called the phase curve. Thus the phase curve is given by the parametric equations

$$x = \varphi(t) \quad y = \dot{\varphi}(t).$$

Through every phase point there is one and only one phase curve. A phase curve could consist of only one point. Such a point is called an equilibrium position. The vector of phase velocity at an equilibrium position is zero.

The law of conservation of energy allows one to find the phase curves easily. On each phase curve the value of the total energy is constant. Therefore, each phase curve lies entirely in one energy level set $E(x, y) = h$.
