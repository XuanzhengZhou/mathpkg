The fundamental lemma of the calculus of variations states that if a continuous function integrates to zero against every test function vanishing at the endpoints, then the function must be identically zero. This simple but powerful result is what allows one to pass from the integral condition $\int f(t)h(t)dt = 0$ for the variation to the pointwise differential equation $f(t) = 0$. The proof constructs a test function localized near any point where $f$ is nonzero.
