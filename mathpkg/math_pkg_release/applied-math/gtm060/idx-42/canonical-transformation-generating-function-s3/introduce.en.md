This theorem completes the classification of generating functions for canonical transformations. While the simplest generating functions $S_1(\mathbf{Q}, \mathbf{q})$ and $S_2(\mathbf{P}, \mathbf{q})$ require certain nondegeneracy conditions on the transformation, the general theorem shows that locally one can always find an appropriate mixed set of coordinates $(P_i, Q_j, \mathbf{q})$ that serves as independent coordinates. The generating function $S_3(P_i, Q_j, \mathbf{q})$ of mixed type then reconstructs the full canonical transformation via partial derivatives, covering all $2^n$ possible choices of which momentum and position coordinates from the target are used.
