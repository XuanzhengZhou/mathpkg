An infinitesimal canonical transformation satisfies Hamilton's differential equations

$$\left. \frac{d\mathbf{P}}{d\varepsilon} \right|_{\varepsilon=0} = -\frac{\partial H}{\partial \mathbf{q}}, \qquad \left. \frac{d\mathbf{Q}}{d\varepsilon} \right|_{\varepsilon=0} = \frac{\partial H}{\partial \mathbf{p}}$$

with hamiltonian function $H(\mathbf{p}, \mathbf{q}) = S(\mathbf{p}, \mathbf{q}, 0)$, where the generating function of the family $g_\varepsilon$ has the form

$$\mathbf{P}\mathbf{q} + \varepsilon S(\mathbf{P}, \mathbf{q}; \varepsilon), \quad \mathbf{p} = \mathbf{P} + \varepsilon \frac{\partial S}{\partial \mathbf{q}}, \quad \mathbf{Q} = \mathbf{q} + \varepsilon \frac{\partial S}{\partial \mathbf{P}}.$$

An infinitesimal canonical transformation is an equivalence class of families $g_\varepsilon$ depending differentiably on $\varepsilon$, two families $g_\varepsilon$ and $h_\varepsilon$ being equivalent if $|g_\varepsilon - h_\varepsilon| = O(\varepsilon^2)$ as $\varepsilon \to 0$.
