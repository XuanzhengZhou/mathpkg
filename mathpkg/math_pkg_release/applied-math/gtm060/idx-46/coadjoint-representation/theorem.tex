Let $\mathfrak{g}^* = T^*G_e$ be the dual vector space to the Lie algebra $\mathfrak{g}$, i.e., the space of real linear functionals on $\mathfrak{g}$. The \textbf{coadjoint representation} $ad^*$ maps each $\xi \in \mathfrak{g}$ to a linear operator on $\mathfrak{g}^*$:

$$ad^*_\xi : \mathfrak{g}^* \to \mathfrak{g}^*,$$

defined as the adjoint of $ad_\xi$:

$$(ad^*_\xi \eta, \zeta) \equiv (\eta, ad_\xi \zeta) \quad \text{for all } \eta \in \mathfrak{g}^*, \zeta \in \mathfrak{g}.$$

It is convenient to use brace notation: $ad^*_\xi \eta = \{\xi, \eta\}$, so that $(\{\xi, \eta\}, \zeta) = (\eta, [\xi, \zeta])$.
