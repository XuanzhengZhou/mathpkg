An unperturbed integrable hamiltonian system
$$H_0(I) = h, \qquad I \in G \subset \mathbb{R}^n, \quad \varphi \bmod 2\pi \in T^n$$
is called nondegenerate if the Hessian matrix of $H_0$ with respect to the action variables is non-singular everywhere (or almost everywhere) in the domain $G$:
$$\det \left| \frac{\partial^2 H_0}{\partial I^2} \right| \neq 0.$$

This condition guarantees that the frequency map $I \mapsto \omega(I) = \frac{\partial H_0}{\partial I}$ is a local diffeomorphism, so that the frequencies change as the action variables change. Consequently, for a fixed set of Diophantine non-resonant frequencies $\omega$, there exists a unique corresponding torus in the unperturbed system, enabling the iterative scheme of Kolmogorov's proof.
