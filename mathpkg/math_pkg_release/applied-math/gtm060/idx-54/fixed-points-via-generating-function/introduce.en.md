For a symplectic diffeomorphism described by a generating function $Xy + S(X, y)$, the auxiliary function $F(x, y) = S(X(x, y), y)$ provides a bridge between fixed points and critical points. The boundary twist condition forces the gradient of $F$ to point inward (or outward) on both boundary circles, guaranteeing at least two critical points by elementary calculus. Thus the search for fixed points reduces to finding critical points of a smooth function on the annulus.
