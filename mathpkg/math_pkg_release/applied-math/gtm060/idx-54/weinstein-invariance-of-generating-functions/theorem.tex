Let $\Phi$ be the generating function of a symplectic diffeomorphism at a fixed point, constructed with the help of some symplectic coordinate system $(x, y)$. Using another symplectic coordinate system $(x', y')$, we construct a generating function $\Phi'$ in the same way.

If the linearization of the symplectic diffeomorphism at the fixed point has no eigenvalues equal to $-1$, then the functions $\Phi$ and $\Phi'$ are equivalent in a neighborhood of the fixed point, in the sense that there is a diffeomorphism $g$ (in general not symplectic) such that

$$\Phi(z) = \Phi'(g(z)) + \text{const}.$$

Reference: A. Weinstein, The invariance of Poincaré's generating function for canonical transformations, Inventiones Mathematicae, 16, No. 3 (1972), 202–214.
