# Proof of Gauge Equivalence of Gravitational Potentials

The gravitational field is defined as $-\vec{\nabla}\phi$. Thus the physical effect -- the gravitational acceleration at each point -- is determined entirely by $\vec{\nabla}\phi$. If $\vec{\nabla}\phi = \vec{\nabla}\hat{\phi}$, then the gravitational fields coincide at every point, so $\phi$ and $\hat{\phi}$ produce identical forces on all test particles. Conversely, if $\phi$ and $\hat{\phi}$ describe the same gravitational effect, then the resulting forces (and hence accelerations) are identical everywhere, which forces $\vec{\nabla}\phi = \vec{\nabla}\hat{\phi}$. On a connected domain, this condition implies $\phi - \hat{\phi}$ is constant, so the potentials differ at most by an additive constant.
