For every timelike future-pointing curve $\gamma: [a, b] \rightarrow M$, let
$$L(\gamma) = \int_{a}^{b} |\dot{\gamma}(u)| \, du$$
be the arc length; thus $L(\gamma) > 0$. Define the proper time function
$$t: \{(p, q) \in M \times M \mid p \ll q\} \rightarrow (0, \infty]$$
by
$$t(p, q) = \sup \left\{ L(\gamma) \mid \gamma \text{ is a future-pointing timelike curve from } p \text{ to } q \right\}$$
for all $p, q \in M$ such that $p \ll q$.
