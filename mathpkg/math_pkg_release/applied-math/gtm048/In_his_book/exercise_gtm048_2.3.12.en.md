Recall that a tensor field $S$ on $M$ is parallel ($\equiv$ covariant constant) iff $D_X S = 0$ for all vector fields $X$.

(a) Show $\partial_4$ and $(1/3)(5\partial_4 + 4\partial_1)$ are both parallel reference frames on Minkowski space.

(b) Show from Definition 1.0.2 of the curvature tensor that if $Z$ is a parallel reference frame on $M$, $\operatorname{Ric}(Z, X) = 0$ for all $X$.

(c) Show there exists no parallel reference frame on Einstein–de Sitter spacetime.

(d) (difficult) Show that in general there is no rigid reference frame defined in any neighborhood of a spacetime point.
