For each pair $x, z \in M$, define the auxiliary function
$$
\delta(x, z) = \left\{ \sum_{\mu=1}^{3} \big( u^\mu z - u^\mu x \big)^2 \right\}^{1/2}.
$$
Thus $\delta(x, z) = \delta(z, x) \geq 0$. This function satisfies:
\begin{enumerate}
  \item[(a)] If $\psi: M \to M$ is an isometry, then $\delta(\psi x, \psi z) = \delta(x, z)$.
  \item[(b)] Denote the level surface $u^4 = 1$ by $\mathcal{B}$. Let $x'$ be the point at which the inextendible integral curve of $\partial_4$ through $x$ intersects $\mathcal{B}$.
\end{enumerate}
