On the normal Schwarzschild spacetime $N$, define $Z = (1 - 2\mu/r)^{-1/2}(\partial/\partial t)$. Then $Z$ is a static, absolute reference frame on $N$. Moreover, $\hat{t}: N \to \mathbb{R}$ is a time function for $Z$ such that $\hat{Z}\hat{t} \to 1$ as $r \to \infty$ iff $d\hat{t} = dt|_N$. An observer $\gamma$ in $N$ is at rest iff $r \circ \gamma$ is constant.
