\text{Define } Z = \left(1 - \frac{2\mu}{r}\right)^{-1/2}\frac{\partial}{\partial t} \text{ on } N. \text{ Then } Z \text{ is a static, absolute reference frame on } N. \text{ A smooth function } \hat{t}: N \to \mathbb{R} \text{ is a time function for } Z \text{ satisfying } Z\hat{t} \to 1 \text{ as } r \to \infty \text{ if and only if } d\hat{t} = dt|_N. \text{ An observer } \gamma \text{ in } N \text{ is at rest in this frame iff } r \circ \gamma = \text{constant} \in (2\mu, \infty).
