If $(M, g')$ is conformally equivalent to $(M, g)$, then the sets of timelike, lightlike, and spacelike vectors coincide for both metrics at each point, since multiplying the metric by a positive scalar function preserves the sign of $g(v, v)$. Consequently, a curve is future-pointing timelike with respect to $g$ if and only if it is future-pointing timelike with respect to $g'$, and similarly for causal curves. Hence $x \ll z$ in $(M, g)$ iff $x \ll' z$ in $(M, g')$, so $\ll = \ll'$. Likewise, $x \leq z$ iff $x \leq' z$, giving $\leq = \leq'$. This is made precise in Exercise 3.7.8.
