Using the same setup as in Exercise 8.1.8(a), show that $\hat{T}$ is diagonalizable if and only if $\hat{T}$ obeys the dominant energy condition, which holds if and only if either $g(E, B) \neq 0$ or $g(E, E) \neq g(B, B)$.
