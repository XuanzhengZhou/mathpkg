A static reference frame is a further specialization of a stationary reference frame. In addition to being stationary (some rescaling is Killing), a static reference frame is irrotational, meaning its observers experience no local rotation of the geometry. The existence of a static reference frame on a spacetime indicates that the gravitational source is both time-independent and non-rotating, representing an extreme idealization useful for modeling non-rotating stars and black holes.
