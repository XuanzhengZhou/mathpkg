Let $Q$ be a reference frame and $\omega$ the 1-form physically equivalent to $Q$. $Q$ is called:
\begin{itemize}
  \item \textbf{locally synchronizable} iff $\omega \wedge d\omega = 0$,
  \item \textbf{locally proper time synchronizable} iff $d\omega = 0$,
  \item \textbf{synchronizable} iff there are $C^\infty$ functions $h$ and $t$ on $M$ such that $h > 0$ and $\omega = -h\,dt$,
  \item \textbf{proper time synchronizable} iff $\omega = -dt$.
\end{itemize}
If $Q$ is synchronizable or proper time synchronizable, any function $t$ as above is called a \textbf{time function} or a \textbf{proper time function} for $Q$, respectively.
