If a projective plane curve has no singular points at all, then Lemma 9.5 applies with an empty set of singularities, and the whole curve is an orientable real 2-manifold. This is an immediate corollary that confirms nonsingular algebraic curves are well-behaved topologically. Together with connectedness, this makes nonsingular curves compact connected orientable surfaces, ready for genus classification.
