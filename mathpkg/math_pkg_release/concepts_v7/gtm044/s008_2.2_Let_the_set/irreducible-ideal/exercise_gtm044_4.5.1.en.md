# Exercise 4.5

Although Remark 4.7 shows that in a Noetherian ring not every irreducible ideal need be prime, it is true that in a Noetherian ring, every irreducible ideal is **primary** in the following sense: An ideal $\mathfrak{a}$ in any ring $R$ is primary provided $(x \notin \mathfrak{a}$ and $xy \in \mathfrak{a}) \Rightarrow (y^m \in \mathfrak{a}$ for some $m \in \mathbb{Z}^+)$. Prove the italicized statement as follows: First, if $\mathfrak{a}, \mathfrak{b}$ are ideals of any ring $R$, define the "quotient" $\mathfrak{a} : \mathfrak{b}$ to be $\{x \in R \mid x\mathfrak{b} \subset \mathfrak{a}\}$.

(a) Show that $\mathfrak{a} : \mathfrak{b}$ is an ideal of $R$, and that $\mathfrak{a} \subset \mathfrak{a} : \mathfrak{b}$.

In (b)–(d) assume that $R$ is Noetherian, and that $\mathfrak{a}$ is not primary.

(b) Show that there are elements $x, y \notin \mathfrak{a}$ with $xy \in \mathfrak{a}$, $y^m \notin \mathfrak{a}$, for all $m \in \mathbb{Z}^+$.

(c) Show that there is an $m_0 \in \mathbb{Z}^+$ such that $\mathfrak{a} : (y^{m_0}) = \mathfrak{a} : (y^{m_0+1})$.

(d) Show that $\mathfrak{a} = (\mathfrak{a}, x) \cap (\mathfrak{a}, y^{m_0})$, and use this to conclude that $\mathfrak{a}$ is not irreducible.
