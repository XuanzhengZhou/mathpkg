% K Diameter Definition
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

For a finite set $S = \{z_1, \ldots, z_k\} \subset \mathbb{C}$, the $k$-diameter $d_k(S)$ is
$$d_k(S) = \left( \prod_{1 \leq i < j \leq k} |z_i - z_j| \right)^{\frac{2}{k(k-1)}}.$$
