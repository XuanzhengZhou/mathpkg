% Fiber Hull Characterization
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $Y$ be a compact set in $\mathbb{C}^2$ lying over the unit circle $\Gamma$. Assume each fiber $Y_\lambda$ ($\lambda \in \Gamma$) is convex. Then each fiber $\hat{Y}_\lambda$ ($|\lambda| < 1$) is also convex and can be characterized by function theory.
