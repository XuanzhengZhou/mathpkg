# Proof of Plurisubharmonic Exhaustion for Polynomially Convex Sets

Let $K$ be a compact polynomially convex set in $\mathbb{C}^n$ with $K \subseteq U$, where $U$ is open and bounded. We construct a smooth strictly plurisubharmonic function $\rho: \mathbb{C}^n \to \mathbb{R}$ and $R > 0$ such that:

(i) $\rho < 0$ on $K$ and $\rho > 0$ on $\mathbb{C}^n \setminus U$;

(ii) $\rho(z) = |z|^2$ for $|z| > R$; and

(iii) $\rho$ is a Morse function on $\mathbb{C}^n$.

**Proof.** We proceed in three steps.

**Step 1: Construction of a strictly plurisubharmonic function satisfying (i).** Choose $C > \max_{z \in K} |z|^2$. Define

$$L = \{z \in \mathbb{C}^n : |z|^2 - C \leq 0\} \setminus U.$$

Then $L$ is compact and disjoint from $K$. For each $x \in L$, since $K$ is polynomially convex, there exists a polynomial $p_x$ such that $|p_x| < 1$ on $K$ and $|p_x(x)| > 2$. By continuity, $|p_x| > 2$ on a neighborhood $N_x$ of $x$. The collection $\{N_x\}_{x \in L}$ covers $L$; by compactness, a finite subcollection $N_{x_1}, \ldots, N_{x_s}$ covers $L$. Set $p_j = p_{x_j}$ for $j = 1, \ldots, s$.

Now define

$$v(z) = \max\{|z|^2 - C, \ |p_1(z)|^2 - 1, \ldots, |p_s(z)|^2 - 1\}.$$

Each function $|z|^2 - C$ and $|p_j(z)|^2 - 1$ is strictly plurisubharmonic (since $|p_j|^2$ is plurisubharmonic and $p_j$ is holomorphic). However, the maximum of finitely many strictly plurisubharmonic functions is merely plurisubharmonic, not strictly so.

To obtain strict plurisubharmonicity, we smooth $v$ by convolution with a suitable mollifier, obtaining a smooth plurisubharmonic function $\tilde{v}$. Then add a small multiple of $|z|^2$ to make it strictly plurisubharmonic: define

$$\rho_1(z) = \tilde{v}(z) + \varepsilon |z|^2$$

for sufficiently small $\varepsilon > 0$. This $\rho_1$ is smooth, strictly plurisubharmonic, and satisfies (i): $\rho_1 < 0$ on $K$ (since $|z|^2 - C < 0$ and $|p_j|^2 - 1 < 0$ on $K$) and $\rho_1 > 0$ on $\mathbb{C}^n \setminus U$ (by construction on $L$).

**Step 2: Extend to satisfy (ii).** Choose $R > 0$ large enough so that $\overline{U} \subseteq B(0, R)$. Let $\chi: \mathbb{R} \to [0,1]$ be a smooth cutoff function with $\chi(t) = 0$ for $t \leq R$ and $\chi(t) = 1$ for $t \geq R+1$. Define

$$\rho_2(z) = (1 - \chi(|z|))\rho_1(z) + \chi(|z|)|z|^2.$$

For $|z| > R+1$, $\chi(|z|) = 1$, so $\rho_2(z) = |z|^2$, satisfying (ii). For $|z| \leq R$, $\chi(|z|) = 0$, so $\rho_2 = \rho_1$, preserving (i). The convex combination preserves strict plurisubharmonicity if $\chi$ is chosen appropriately.

**Step 3: Make $\rho$ a Morse function.** By Morse's Lemma, we can make a small smooth perturbation of $\rho_2$ on a large ball $B(0, R')$ (where $R' \gg R$) to obtain a Morse function $\rho$ whose derivatives uniformly approximate those of $\rho_2$ up to order 2, while preserving properties (i) and (ii). Since strict plurisubharmonicity is an open condition in the $C^2$-topology, $\rho$ remains strictly plurisubharmonic for sufficiently small perturbations.

Thus $\rho$ satisfies all three required properties. $\square$
