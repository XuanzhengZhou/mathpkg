% Pseudoconcave Set Definition
% From GTM035
% Source: Alexander & Wermer, Several Complex Variables and Banach Algebras (GTM 35), 3rd Edition

Let $\Omega$ be a domain in $\mathbb{C}^n$ and let $E$ be a relatively closed subset of $\Omega$. $E$ is called pseudoconcave in $\Omega$ if the open set $\Omega \setminus E$ is pseudoconvex.
