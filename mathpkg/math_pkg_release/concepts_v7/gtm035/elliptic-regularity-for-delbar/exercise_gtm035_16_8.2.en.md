# Exercise 2

With the same notation as Exercise 1, prove that $u_\varepsilon \to u$ in $L^2(\mathbb{C}^n)$ as $\varepsilon \to 0$.

*Hint:* Write $u_\varepsilon(z) - u(z) = \int (u(z - \zeta) - u(z)) \chi_\varepsilon(\zeta) dV(\zeta)$. Apply the Cauchy-Schwarz inequality and Fubini's theorem, then use the fact that translation is continuous in $L^2$.
