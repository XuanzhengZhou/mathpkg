# Exercise 3

With the same notation as Exercise 1, assume additionally that $u$ is continuous in a neighborhood of a closed ball $\overline{B(x_0, R)}$. Prove that $u_\varepsilon \to u$ uniformly on $\overline{B(x_0, R)}$ as $\varepsilon \to 0$.

*Hint:* Since $u$ is uniformly continuous on the compact set $\overline{B(x_0, R + \delta)}$ for small $\delta > 0$, given $\eta > 0$, choose $\varepsilon$ small enough so that $|u(z - \zeta) - u(z)| < \eta$ for all $z \in \overline{B(x_0, R)}$ and $|\zeta| < \varepsilon$. Then use $\int \chi_\varepsilon = 1$.
