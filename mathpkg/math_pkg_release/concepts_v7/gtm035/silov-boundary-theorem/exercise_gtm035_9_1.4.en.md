# Exercise 9.4

Let $\Delta^2 = \{(z, w) \in \mathbb{C}^2 : |z| \leq 1, |w| \leq 1\}$ and

$$A(\Delta^2) = \{f \in C(\Delta^2) : f \in H(\mathcal{O}), \text{ where } \mathcal{O} = \text{interior of } \Delta^2\}.$$

Show that $\check{S}(A(\Delta^2)) = \mathbb{T}^2 = \{(z, w) : |z| = |w| = 1\}$.

Note that here the Šilov boundary is a two-dimensional subset of the three-dimensional topological boundary of $\Delta^2$.
