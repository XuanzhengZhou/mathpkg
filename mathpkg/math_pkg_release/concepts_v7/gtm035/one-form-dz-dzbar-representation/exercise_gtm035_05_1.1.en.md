# Exercise 5.1

Prove Lemma 5.1: If $\omega \in \wedge^1(\Omega)$ is a 1-form on an open set $\Omega \subseteq \mathbb{C}^n$, then $\omega$ admits a unique representation

$$\omega = \sum_{j=1}^{n} a_j \, dz_j + b_j \, d\bar{z}_j$$

where $a_j, b_j \in C^\infty(\Omega)$.

*Hint.* Use the real basis $\{dx_j, dy_j\}_{j=1}^n$ and the identities $dx_j = \tfrac{1}{2}(dz_j + d\bar{z}_j)$, $dy_j = \tfrac{1}{2i}(dz_j - d\bar{z}_j)$.
