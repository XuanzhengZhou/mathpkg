A pregeometry $(M, \operatorname{cl})$ is \text{modular} if for all $U, V \subseteq M$,
$$w \in \operatorname{cl}(U \cup \{v\}) \implies \exists u \in \operatorname{cl}(U) : w \in \operatorname{cl}(\{u, v\}).$$
It is \text{locally modular} if the localization $(M, \operatorname{cl}_a)$, where $\operatorname{cl}_a(X) = \operatorname{cl}(X \cup \{a\})$, is modular for some $a \in M$.
Examples:
\begin{itemize}
\item Trivial pregeometry: $\operatorname{cl}(U) = \bigcup_{u \in U} \operatorname{cl}(\{u\})$.
\item Modular: projective geometry over a division ring (vector spaces, divisible abelian groups with finite $n$-torsion).
\item Non-modular but locally modular: affine geometry over a field.
\item Non-locally-modular: algebraic closure in an algebraically closed field.
\end{itemize}
