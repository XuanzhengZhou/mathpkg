A \text{nonstandard model of the reals} $^*\mathbf{R}$ is any model of $\operatorname{Th}(\mathbf{R}_{\text{field}})$ not isomorphic to the standard reals. Any nonstandard model contains an \text{infinitesimal} $\alpha$ such that $0 < \alpha < 1/n$ for every positive integer $n$. More generally:
\begin{itemize}
\item An element $\gamma \in {}^*\mathbf{R}$ is \text{infinite} if its Dedekind cut has an empty side.
\item Otherwise $\gamma$ is \text{bounded}.
\item The bounded elements $B \subseteq {}^*\mathbf{R}$ form a ring, and the infinitesimals $\mu \subseteq B$ form a maximal ideal.
\item The \text{standard part map} $st: B \to \mathbf{R}$, $r + \alpha \mapsto r$ ($r \in \mathbf{R}, \alpha \in \mu$) is a surjective ring homomorphism.
\end{itemize}
