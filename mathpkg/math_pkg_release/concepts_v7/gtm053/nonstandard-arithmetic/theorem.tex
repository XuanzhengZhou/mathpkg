Let $\mathbf{N} = (N, +, \cdot, 0, 1)$ be the standard structure of nonnegative integers in the language $L_1\mathrm{Ar}$. The theory $\operatorname{Th}(\mathbf{N})$ is called \text{complete arithmetic}. Any model of $\operatorname{Th}(\mathbf{N})$ not isomorphic to $\mathbf{N}$ is called a \text{nonstandard model of arithmetic}. Nonstandard models exist: extend the language by a new constant $c$ and consider
$$\mathcal{E} = \operatorname{Th}(\mathbf{N}) \cup \{ \neg c = \bar{n} : n = 0, 1, \ldots \}.$$
$\mathcal{E}$ is finitely satisfiable; any model reduced to $L_1\mathrm{Ar}$ is nonstandard. In such a model, nonstandard elements are "infinite integers."
