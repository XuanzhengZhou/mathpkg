Terms are the elements of the least set of expressions in a first-order language $L$ satisfying:
\begin{enumerate}
\item[(a)] Variables and constant symbols are (atomic) terms.
\item[(b)] If $f$ is an operation symbol of arity $r$ and $t_1, \ldots, t_r$ are terms, then $f(t_1, \ldots, t_r)$ is a term.
\end{enumerate}
