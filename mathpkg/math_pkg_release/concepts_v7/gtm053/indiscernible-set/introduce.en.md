An indiscernible set is a set of elements that are indistinguishable by first-order formulas — any permutation preserves formulas. Indiscernibles are fundamental in stability theory: in stable theories, every type over a model is based on an indiscernible set, and the average type of an indiscernible set gives a canonical nonforking extension.
