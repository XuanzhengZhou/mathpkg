# Exercise 1.4 (i)

An embedding $h : A \rightarrow B$ of $L$-structures preserves atomic $L$-formulas, i.e., for any atomic $P(x_1, \ldots, x_n)$ for any $\bar{a} \in A^n$,

$$A \vDash P(\bar{a}) \quad \text{iff} \quad B \vDash P(h(\bar{a})).$$
