# Proof of Existence and Uniqueness of Saturated Models

**Theorem 3.11.** Let $T$ be a complete theory.

(i) For every $\kappa \geq \operatorname{card} T$ there exists a $\kappa$-saturated model of $T$ of cardinality $\leq \kappa^+$.

(ii) Any two saturated models of $T$ of the same cardinality are isomorphic.

*Proof.* (i) We construct a $\kappa$-saturated model of $T$ by building an elementary chain of length $\kappa^+$. Assume $T$ has infinite models (the finite case is trivial). Let $\mathbf{A}_0$ be any model of $T$ of cardinality $\leq \kappa$. For successor steps, given $\mathbf{A}_\alpha$, we want to realize all types over subsets of size $<\kappa$. There are at most $\kappa^{<\kappa} = \kappa$ such types (since $|T| \leq \kappa$). By the compactness theorem and the method of diagrams, each type $\tau$ over a set $C \subseteq A_\alpha$ with $|C| < \kappa$ can be realized in an elementary extension of $(\mathbf{A}_\alpha)_{c \in C}$. Taking the union of realizing all such types, we obtain an elementary extension $\mathbf{A}_{\alpha+1}$ of cardinality $\leq \kappa$ in which every type over a subset of $\mathbf{A}_\alpha$ of size $<\kappa$ is realized. For limit ordinals $\delta$, take $\mathbf{A}_\delta = \bigcup_{\alpha < \delta} \mathbf{A}_\alpha$. Iterating this construction $\kappa^+$ times, we obtain an elementary chain $\{\mathbf{A}_\alpha\}_{\alpha < \kappa^+}$. Let $\mathbf{A} = \bigcup_{\alpha < \kappa^+} \mathbf{A}_\alpha$. By regularity of $\kappa^+$, any subset $C \subseteq A$ of size $<\kappa$ is contained in some $A_\alpha$, and the types over $C$ are realized in $\mathbf{A}_{\alpha+1} \subseteq \mathbf{A}$. Thus $\mathbf{A}$ is $\kappa$-saturated, and $|\mathbf{A}| \leq \kappa^+$.

(ii) Let $\mathbf{A}$ and $\mathbf{B}$ be saturated models of $T$ of the same cardinality $\kappa$. Enumerate their elements: $A = \{a_\delta : \delta < \kappa\}$, $B = \{b_\delta : \delta < \kappa\}$. We construct an isomorphism via a back-and-forth argument. Build increasing chains of elementary submodels $\mathbf{A}_\alpha \preceq \mathbf{A}$ and $\mathbf{B}_\alpha \preceq \mathbf{B}$ with isomorphisms $f_\alpha : \mathbf{A}_\alpha \to \mathbf{B}_\alpha$, for $\alpha < \kappa$, such that $a_\alpha \in A_{\alpha+1}$ and $b_\alpha \in B_{\alpha+1}$.

For limit $\delta$, $\mathbf{A}_\delta = \bigcup_{\alpha < \delta} \mathbf{A}_\alpha$, $\mathbf{B}_\delta = \bigcup_{\alpha < \delta} \mathbf{B}_\alpha$, and $f_\delta = \bigcup_{\alpha < \delta} f_\alpha$.

If $\alpha$ is even (forth step): We have $f_\alpha : \mathbf{A}_\alpha \cong \mathbf{B}_\alpha$. We want to include $a_\alpha$ in the domain. Let $C_\alpha$ name the elements of $A_\alpha$. Consider the type $\operatorname{tp}(a_\alpha / C_\alpha)$ in $\mathbf{A}$. By the isomorphism $f_\alpha$, this corresponds to a type over $f_\alpha(C_\alpha)$ in $\mathbf{B}$. Since $\mathbf{B}$ is $\kappa$-saturated and $|C_\alpha| < \kappa$, this type is realized by some $b \in B$. Set $\mathbf{A}_{\alpha+1}$ as the elementary submodel generated by $A_\alpha \cup \{a_\alpha\}$, $\mathbf{B}_{\alpha+1}$ as the elementary submodel generated by $B_\alpha \cup \{b\}$, and extend $f_\alpha$ by mapping $a_\alpha \mapsto b$.

If $\alpha$ is odd (back step): We do the symmetric argument, including $b_\alpha$ in the range this time.

The union $f = \bigcup_{\alpha < \kappa} f_\alpha$ is an isomorphism $\mathbf{A} \cong \mathbf{B}$. $\square$
