# Proof of Theorem 1 (Ballot Theorem)

Let $\eta_1, \ldots, \eta_n$ be a sequence of independent identically distributed random variables taking finitely many values from the set $(0, 1, \ldots)$ and let

$$S_k = \eta_1 + \cdots + \eta_k, \quad 1 \leq k \leq n.$$

Then $\mathsf{P}$-a.s.

$$\mathsf{P}\!\left\{\max_{1 \leq l \leq n} \frac{S_l}{l} \geq 1\right\} = \mathsf{E}\!\left(\frac{S_n}{n}\right)^+. \tag{12}$$

**Proof.** Consider the martingale $\xi = (\xi_k, \mathcal{D}_k)_{1 \leq k \leq n}$ introduced in Example 4, with

$$\xi_k = \frac{S_{n+1-k}}{n+1-k}$$

and $\mathcal{D}_k = \mathcal{D}_{S_{n+1-k}, \ldots, S_n}$ (the $\sigma$-algebra generated by the future observations). Define the stopping time

$$\tau = \min\{1 \leq k \leq n : \xi_k \geq 1\},$$

taking $\tau = n$ on the set $\{\xi_k < 1 \text{ for all } k, 1 \leq k \leq n\} = \{\max_{1 \leq l \leq n}(S_l/l) < 1\}$.

It is clear that $\xi_\tau = \xi_n = S_1 = 0$ on this set, and therefore

$$\left\{\max_{1 \leq l \leq n} \frac{S_l}{l} < 1\right\} = \left\{\max_{1 \leq l \leq n} \frac{S_l}{l} < 1,\; S_n < n\right\} \subseteq \{\xi_\tau = 0\}. \tag{13}$$

Now consider outcomes for which simultaneously $\max_{1 \leq l \leq n}(S_l/l) \geq 1$ and $S_n < n$. Write $\sigma = n+1-\tau$. It is easy to see that

$$\sigma = \max\{1 \leq k \leq n : S_k \geq k\}$$

and therefore (since $S_n < n$) we have $\sigma < n$, $S_\sigma \geq \sigma$, and $S_{\sigma+1} < \sigma+1$. Consequently $\eta_{\sigma+1} = S_{\sigma+1} - S_\sigma < (\sigma+1) - \sigma = 1$, i.e., $\eta_{\sigma+1} = 0$. Therefore $\sigma \leq S_\sigma = S_{\sigma+1} < \sigma+1$, and consequently $S_\sigma = \sigma$ and

$$\xi_\tau = \frac{S_{n+1-\tau}}{n+1-\tau} = \frac{S_\sigma}{\sigma} = 1.$$

Thus on the set $\{\max_{1 \leq l \leq n}(S_l/l) \geq 1\}$ we have $\xi_\tau \geq 1$. Consequently,

$$\mathsf{E}\,\xi_\tau = \mathsf{E}\!\left[\xi_\tau I_{\{\max(S_l/l) \geq 1\}}\right] \geq \mathsf{P}\!\left\{\max_{1 \leq l \leq n} \frac{S_l}{l} \geq 1\right\}.$$

On the other hand, by the optional stopping theorem (Theorem 1 proved earlier in this section), $\mathsf{E}\,\xi_\tau = \mathsf{E}\,\xi_1$. Since

$$\xi_1 = \frac{S_{n+1-1}}{n+1-1} = \frac{S_n}{n},$$

we obtain

$$\mathsf{E}\,\xi_\tau = \mathsf{E}\!\left(\frac{S_n}{n}\right).$$

More precisely, the optional stopping theorem yields $\mathsf{E}(\xi_\tau \mid \mathcal{D}_1) = \xi_1$, and taking the positive part gives

$$\mathsf{P}\!\left\{\max_{1 \leq l \leq n} \frac{S_l}{l} \geq 1\right\} = \mathsf{E}\!\left(\frac{S_n}{n}\right)^+,$$

which is formula (12).

**Application to the classical ballot problem.** Let $\xi_1, \ldots, \xi_n$ be independent Bernoulli random variables with

$$\mathsf{P}(\xi_i = 1) = \mathsf{P}(\xi_i = -1) = \frac{1}{2},$$

$S_k = \xi_1 + \cdots + \xi_k$ and $a, b$ nonnegative integers such that $a - b > 0$, $a + b = n$. Then

$$\mathsf{P}\{S_1 > 0, \ldots, S_n > 0 \mid S_n = a - b\} = \frac{a - b}{a + b}. \tag{15}$$

Indeed, by symmetry,

$$\begin{aligned}
\mathsf{P}\{S_1 > 0, \ldots, S_n > 0 \mid S_n = a - b\}
&= \mathsf{P}\{S_1 < 0, \ldots, S_n < 0 \mid S_n = -(a - b)\} \\
&= \mathsf{P}\{S_1 + 1 < 1, \ldots, S_n + n < n \mid S_n + n = n - (a - b)\} \\
&= \mathsf{P}\{\eta_1 < 1, \ldots, \eta_1 + \cdots + \eta_n < n \mid \eta_1 + \cdots + \eta_n = n - (a - b)\} \\
&= \left[ 1 - \frac{n - (a - b)}{n} \right]^+ = \frac{a - b}{n} = \frac{a - b}{a + b},
\end{aligned}$$

where we have put $\eta_k = \xi_k + 1$ and applied formula (12). This completes the proof. $\square$
