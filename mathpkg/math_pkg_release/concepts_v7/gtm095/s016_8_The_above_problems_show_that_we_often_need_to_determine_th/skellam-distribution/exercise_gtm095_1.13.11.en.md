# Exercise 11

Let $\xi$ be an integer-valued random variable taking values $0, 1, \ldots$. Define $m_{(r)} = \mathbb{E}\,\xi(\xi - 1) \cdots (\xi - r + 1)$, $r = 1, 2, \ldots$. Show that the quantities $m_{(r)}$, which are called factorial moments (of order $r$), can be obtained from the generating function $G_{\xi}(x)$ by the formulas:

$$m_{(r)} = G_{\xi}^{(r)}(1),$$

where $G_{\xi}^{(r)}$ is the $r$th derivative of $G_{\xi}(x)$.
