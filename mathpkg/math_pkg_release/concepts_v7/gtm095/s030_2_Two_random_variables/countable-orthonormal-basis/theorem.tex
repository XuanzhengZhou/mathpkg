A set $\{\eta_1, \eta_2, \ldots\}$ is a countable orthonormal basis for the closed linear manifold $\overline{\mathcal{L}} = \overline{\mathcal{L}}\{\eta_1, \eta_2, \ldots\}$ if it is an orthonormal system and one of the following equivalent conditions holds:

For every $\xi \in \overline{\mathcal{L}}$,

$$\xi = \lim_{n \to \infty} \sum_{i=1}^{n} (\xi, \eta_i) \eta_i \quad \text{(in mean square)},$$

or equivalently, Parseval's identity holds:

$$\|\xi\|^2 = \sum_{i=1}^{\infty} |(\xi, \eta_i)|^2.$$

A Hilbert space of random variables that contains a countable orthonormal basis is called separable.

Conversely, if $\{\eta_1, \eta_2, \ldots\}$ is an orthonormal system and either condition holds for all $\xi \in \overline{\mathcal{L}}$, then the system is a basis for $\overline{\mathcal{L}}$.
