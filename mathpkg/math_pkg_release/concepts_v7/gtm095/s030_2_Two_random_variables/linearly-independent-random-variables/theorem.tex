Random variables $\eta_1, \ldots, \eta_n \in L^2$ are said to be linearly independent if the equation

$$\sum_{i=1}^{n} a_i \eta_i = 0 \quad (\text{P-a.s.})$$

holds only when all coefficients $a_i$ are zero.

If $\eta_1, \ldots, \eta_n$ are linearly independent, then the covariance matrix $\mathbf{R} = \mathsf{E}[\boldsymbol{\eta} \boldsymbol{\eta}^*]$ of the vector $\boldsymbol{\eta} = (\eta_1, \ldots, \eta_n)^*$ has positive Gram determinant $\det \mathbf{R} > 0$, and consequently all eigenvalues $d_i$ of $\mathbf{R}$ are strictly positive. The covariance matrix is symmetric, positive semi-definite, and can be diagonalized by an orthogonal matrix $\mathcal{O}$: $\mathcal{O}^* \mathbf{R} \mathcal{O} = D$, where $D = \operatorname{diag}(d_1, \ldots, d_n)$.
