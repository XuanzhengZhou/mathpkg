$$P(A \mid A) = 1,$$
$$P(\emptyset \mid A) = 0,$$
$$P(B \mid A) = 1, \quad B \supseteq A,$$
$$P(B_1 + B_2 \mid A) = P(B_1 \mid A) + P(B_2 \mid A).$$
