An unbiased estimator $\tilde{T}_n$ is called efficient (in the class of unbiased estimators $T_n$) if for every $\theta \in \Theta$,

$$\operatorname{Var}_\theta \tilde{T}_n = \inf_{T_n} \operatorname{Var}_\theta T_n,$$

where the infimum is taken over all unbiased estimators $T_n$. An efficient estimator achieves the smallest possible variance among all unbiased estimators.
