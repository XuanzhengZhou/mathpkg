Let $\xi$ be a random variable for which $E\xi$ is defined. Then the set function
$$Q(A) = \int_A \xi \, dP, \quad A \in \mathcal{F},$$
is a signed measure --- a countably additive set function representable as $Q = Q^+ - Q^-$ where
$$Q^+(A) = \int_A \xi^+ \, dP, \quad Q^-(A) = \int_A \xi^- \, dP,$$
and at least one of $Q^+(\Omega)$ and $Q^-(\Omega)$ is finite. Moreover, $Q$ is absolutely continuous with respect to $P$ ($Q \ll P$).
