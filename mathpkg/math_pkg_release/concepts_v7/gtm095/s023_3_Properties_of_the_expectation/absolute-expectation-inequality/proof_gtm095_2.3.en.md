# Proof of Absolute Value of Expectation Inequality

**Property C.** $|E\xi| \leq E|\xi|$.

*Proof.* Since $-|\xi| \leq \xi \leq |\xi|$, Properties A and B imply

$$-E|\xi| \leq E\xi \leq E|\xi|,$$

which is equivalent to $|E\xi| \leq E|\xi|$. $\square$
