The concentration function of a random variable $X$ is

$$Q(X; l) = \sup_{x \in \mathbb{R}} P\{x \leq X \leq x + l\}, \quad l \geq 0.$$

Properties:
(a) If $X$ and $Y$ are independent, then $Q(X+Y; l) \leq \min(Q(X; l), Q(Y; l))$ for all $l \geq 0$.
(b) There exists $x'_l$ such that $Q(X; l) = P\{x'_l \leq X \leq x'_l + l\}$, and the distribution function of $X$ is continuous if and only if $Q(X; 0) = 0$.
