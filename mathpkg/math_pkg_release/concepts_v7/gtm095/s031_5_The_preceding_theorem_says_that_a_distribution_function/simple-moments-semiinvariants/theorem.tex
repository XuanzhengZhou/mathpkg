Let $\xi = (\xi_1, \ldots, \xi_k)^*$ be a random vector with index set $I_\xi = \{1, 2, \ldots, k\}$. For $I \subseteq I_\xi$, let $\chi(I) = (\chi_1, \ldots, \chi_k)$ where $\chi_i = 1$ if $i \in I$ and $\chi_i = 0$ if $i \notin I$. Define

$$m_\xi(I) = m_\xi^{(\chi(I))}, \quad s_\xi(I) = s_\xi^{(\chi(I))}.$$

These are the simple moments and simple semi-invariants of the subvector $\xi_I$ of $\xi$.
