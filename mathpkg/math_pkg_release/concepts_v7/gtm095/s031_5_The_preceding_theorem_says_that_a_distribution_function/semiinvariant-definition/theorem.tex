The semi-invariants (cumulants) $s_{\xi}^{(\nu)} = s_{\xi}^{(\nu_1, \ldots, \nu_k)}$ of a random vector $\xi = (\xi_1, \ldots, \xi_k)^*$ with $E|\xi_i|^n < \infty$ are defined as the coefficients in the Taylor expansion of the logarithm of the characteristic function:

$$\log \varphi_{\xi}(t) = \sum_{|\nu| \leq n} \frac{i^{|\nu|}}{\nu!} s_{\xi}^{(\nu)} t^{\nu} + o(|t|^n).$$

If $\xi$ and $\eta$ are independent, then

$$\log \varphi_{\xi+\eta}(t) = \log \varphi_{\xi}(t) + \log \varphi_{\eta}(t),$$

and therefore

$$s_{\xi+\eta}^{(\nu)} = s_{\xi}^{(\nu)} + s_{\eta}^{(\nu)}.$$

It is this additivity property that gives rise to the term "semi-invariant."
