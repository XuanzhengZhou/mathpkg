# Exercise 14

Let $I(A)$ denote the indicator function of the set $A$. Show that

$$I\left( \bigcup_{n=1}^{\infty} A_n \right) = \max_{n \geq 1} I(A_n), \qquad I\left( \bigcap_{n=1}^{\infty} A_n \right) = \min_{n \geq 1} I(A_n).$$
