# Exercise 18

Let $A_n$ converge to $A$ in the sense that $P(A \triangle A^*) = P(A \triangle A_*) = 0$, where $A^* = \limsup A_n$ and $A_* = \liminf A_n$. Show that then

$$P(A \triangle A_n) \to 0 \quad \text{as } n \to \infty.$$
