# Exercise 4

Let $\xi_1, \ldots, \xi_n$ be independent Bernoulli random variables with $P\{\xi_i = 1\} = p > 0$, $P\{\xi_i = -1\} = 1 - p$. Derive the following Bernstein inequality: there is a number $a > 0$ such that

$$P\left\{\left|\frac{S_n}{n} - (2p - 1)\right| \geq \varepsilon\right\} \leq 2e^{-a\varepsilon^2 n},$$

where $S_n = \xi_1 + \cdots + \xi_n$ and $\varepsilon > 0$.
