$$P\{\omega : \xi_n \rightarrow \xi\} = 0,$$

i.e., if the set of sample points $\omega$ for which $\xi_n(\omega)$ does not converge to $\xi$ has probability zero.

This convergence is denoted by $\xi_n \rightarrow \xi$ (P-a. s.), or $\xi_n \rightarrow \xi$ (a. s.), or $\xi_n \xrightarrow{\text{a.e.}} \xi$ or $\xi_n \xrightarrow{\text{a.e.}} \xi$.
