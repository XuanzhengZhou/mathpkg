# Exercise 12

Give an example of a random variable which is **not** infinitely divisible, but whose characteristic function nevertheless does not vanish anywhere on $\mathbb{R}$.

This shows that the non-vanishing of the characteristic function is a necessary but not sufficient condition for infinite divisibility.
