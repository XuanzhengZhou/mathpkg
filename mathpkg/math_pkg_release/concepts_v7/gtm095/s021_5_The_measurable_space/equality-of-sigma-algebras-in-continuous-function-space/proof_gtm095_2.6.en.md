# Proof of the Equality of the $\sigma$-Algebras $\mathcal{B}(C)$ and $\mathcal{B}_0(C)$ on the Space of Continuous Functions

Let $T = [0, 1]$ and let $C \equiv C[0, 1]$ be the space of continuous functions $x = (x_t)$, $0 \leq t \leq 1$. This is a metric space with the uniform metric

$$\rho(x, y) = \sup_{t \in T} |x_t - y_t|.$$

We introduce two $\sigma$-algebras on $C$:

- $\mathcal{B}(C)$ is the $\sigma$-algebra generated by the cylinder sets
  $$\mathcal{I}_{t_1, \ldots, t_n}(I_1 \times \cdots \times I_n) = \{x \in C : x_{t_1} \in I_1, \ldots, x_{t_n} \in I_n\},$$
  where $I_k = (a_k, b_k]$.

- $\mathcal{B}_0(C)$ is the Borel $\sigma$-algebra generated by the open sets with respect to the metric $\rho$.

**Theorem.** $\mathcal{B}(C) = \mathcal{B}_0(C)$.

*Proof.* The proof proceeds by establishing both inclusions.

**Part 1: $\mathcal{B}(C) \subseteq \mathcal{B}_0(C)$.**

It suffices to show that every generating cylinder set of $\mathcal{B}(C)$ belongs to $\mathcal{B}_0(C)$. Consider a basic cylinder set of the form

$$B = \{x \in C : x_{t_0} < b\}.$$

Let $x \in B$. Since $x$ is continuous at $t_0$, there exists $\delta > 0$ such that $x_{t_0} + \delta < b$. Take the open ball $U = \{y \in C : \rho(x, y) < \delta\}$. For any $y \in U$, we have

$$|y_{t_0} - x_{t_0}| \leq \rho(x, y) < \delta,$$

so $y_{t_0} < x_{t_0} + \delta < b$. Hence $U \subseteq B$, showing that $B$ is an open set in the metric $\rho$. Therefore $B \in \mathcal{B}_0(C)$.

Since sets of the form $\{x : x_{t_0} < b\}$ generate the cylinder sets (by taking finite intersections of such sets and their complements), and since $\mathcal{B}_0(C)$ is a $\sigma$-algebra, we conclude

$$\mathcal{B}(C) \subseteq \mathcal{B}_0(C).$$

**Part 2: $\mathcal{B}_0(C) \subseteq \mathcal{B}(C)$.**

It suffices to show that every open ball $B_\rho(x_0, r) = \{y \in C : \rho(x_0, y) < r\}$ belongs to $\mathcal{B}(C)$. Since the rational numbers are dense in $[0, 1]$, we have

$$B_\rho(x_0, r) = \bigcap_{t \in [0, 1] \cap \mathbb{Q}} \{y \in C : |y_t - x_{0,t}| < r\}.$$

(To see this: if $y \in C$ satisfies $|y_t - x_{0,t}| < r$ for all rational $t$, then by continuity of $y$ and $x_0$, the inequality $\sup_{t \in [0,1]} |y_t - x_{0,t}| \leq r$ holds. The strict inequality $\rho(x_0, y) < r$ follows by considering the supremum over a countable dense set.)

Indeed, for any $y \in B_\rho(x_0, r)$, we have $\sup_{t} |y_t - x_{0,t}| < r$, so in particular $|y_t - x_{0,t}| < r$ for all rational $t$. Conversely, if $|y_t - x_{0,t}| < r$ for all rational $t \in [0, 1] \cap \mathbb{Q}$, then by continuity,

$$\sup_{t \in [0,1]} |y_t - x_{0,t}| = \sup_{t \in [0,1] \cap \mathbb{Q}} |y_t - x_{0,t}| \leq r.$$

Now, each set $\{y \in C : |y_t - x_{0,t}| < r\}$ can be written as

$$\{y \in C : y_t < x_{0,t} + r\} \cap \{y \in C : y_t > x_{0,t} - r\},$$

which is a finite intersection of basic cylinder sets (of the form $\{y : y_t < b\}$ and $\{y : y_t > a\} = C \setminus \{y : y_t \leq a\}$). Thus each factor belongs to $\mathcal{B}(C)$.

Since the intersection in the representation of $B_\rho(x_0, r)$ is taken over a countable set ($\mathbb{Q} \cap [0, 1]$ is countable), $B_\rho(x_0, r)$ is a countable intersection of sets in $\mathcal{B}(C)$, hence $B_\rho(x_0, r) \in \mathcal{B}(C)$.

Every open set in the metric space $C$ is a countable union of open balls (since $C$ is separable under the uniform metric), so every open set belongs to $\mathcal{B}(C)$. Consequently, the $\sigma$-algebra generated by open sets satisfies

$$\mathcal{B}_0(C) \subseteq \mathcal{B}(C).$$

**Conclusion.** Combining both inclusions, we obtain $\mathcal{B}(C) = \mathcal{B}_0(C)$. $\square$

**Remark.** This result is significant in the theory of stochastic processes: for a continuous process $(X_t)_{t \in [0,1]}$, the measurability conditions formulated in terms of cylinder sets (which are natural from the viewpoint of finite-dimensional distributions) coincide with those formulated in terms of Borel sets in the uniform topology (which are natural for pathwise properties like continuity and the supremum). In particular, the sets $\{\sup_{t} X_t < C\}$, $\{\inf_{t} X_t > C\}$, and sets of the form $\{X_t \text{ has some property for all } t\}$ are measurable with respect to $\mathcal{B}(C)$.
