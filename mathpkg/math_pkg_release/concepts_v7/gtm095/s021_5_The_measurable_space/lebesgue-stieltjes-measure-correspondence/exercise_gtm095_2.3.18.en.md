# Exercise 18 (Distribution Function Decomposition)

Show that every distribution function $F(x)$ can be represented in the form

$$F(x) = \alpha_1 F_1(x) + \alpha_2 F_2(x) + \alpha_3 F_3(x),$$

where $F_1$ is discrete, $F_2$ is absolutely continuous, $F_3$ is singular, and $\alpha_i$ are nonnegative numbers with $\alpha_1 + \alpha_2 + \alpha_3 = 1$.
