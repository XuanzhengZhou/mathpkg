# Exercise 7

Let $\xi$ and $\eta$ be random variables, both taking the values $1, 2, \ldots, N$. Suppose that $\mathcal{F}_\xi = \mathcal{F}_\eta$. Show that there is a permutation $(i_1, i_2, \ldots, i_N)$ of $(1, 2, \ldots, N)$ such that $\{\omega : \xi = j\} = \{\omega : \eta = i_j\}$ for $j = 1, 2, \ldots, N$.
