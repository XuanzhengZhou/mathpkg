# Proof of $\sigma(\mathcal{D})$-Measurable Functions are Constant on Decomposition Elements

Let us choose a set $D_k$ and show that the $\sigma(\mathcal{D})$-measurable function $\xi$ has a constant value on that set. For this purpose, denote

$$x_k = \sup \left\{ c : D_k \cap \{ \omega : \xi(\omega) < c \} = \varnothing \right\}.$$

Since $\{\omega : \xi(\omega) < x_k\} = \bigcup_{r < x_k, r \in \mathbb{Q}} \{\omega : \xi(\omega) < r\}$, and each set $\{\omega : \xi(\omega) < r\}$ belongs to $\sigma(\mathcal{D})$ (by the measurability of $\xi$), we have $\{\omega : \xi(\omega) < x_k\} \in \sigma(\mathcal{D})$.

Moreover, $D_k \cap \{\omega : \xi(\omega) < x_k\} = \varnothing$ by construction. Since $\sigma(\mathcal{D})$ is generated by the countable partition $\mathcal{D} = \{D_1, D_2, \ldots\}$, the only sets in $\sigma(\mathcal{D})$ are unions of the $D_i$. Therefore $D_k \subseteq \{\omega : \xi(\omega) \geq x_k\}$.

Similarly, for any $\varepsilon > 0$, $D_k \cap \{\omega : \xi(\omega) < x_k + \varepsilon\} \neq \varnothing$, which means $\{\omega : \xi(\omega) < x_k + \varepsilon\} \in \sigma(\mathcal{D})$ must contain $D_k$ (since it intersects $D_k$ and belongs to $\sigma(\mathcal{D})$). Hence $D_k \subseteq \{\omega : \xi(\omega) < x_k + \varepsilon\}$ for all $\varepsilon > 0$, implying $D_k \subseteq \{\omega : \xi(\omega) \leq x_k\}$.

Combining the two inclusions, $\xi(\omega) = x_k$ for all $\omega \in D_k$. Thus

$$\xi(\omega) = \sum_{k=1}^{\infty} x_k I_{D_k}(\omega),$$

where $x_k \in R$, i.e., $\xi(\omega)$ is constant on the elements $D_k$ of the decomposition $\mathcal{D}$, $k \geq 1$.
