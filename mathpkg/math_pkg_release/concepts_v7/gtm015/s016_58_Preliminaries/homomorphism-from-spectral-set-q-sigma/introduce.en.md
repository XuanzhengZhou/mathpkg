When $\sigma$ is a spectral set for $T$, the rational functional calculus defines a unital algebra homomorphism from $Q(\sigma)$ into the bounded operators. The map sends the coordinate function to $T$ and Cauchy kernels to resolvents. This homomorphism is the foundation for extending to the $\sigma$-analytic functional calculus.
