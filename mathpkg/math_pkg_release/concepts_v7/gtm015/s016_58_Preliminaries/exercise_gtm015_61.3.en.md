# Exercise 3

If $f$ is a linear form on a $*$-algebra $A$, define $f^*(a) = (f(a^*))^*$ for $a \in A$; $f^*$ is also a linear form on $A$, called the adjoint of $f$. If $f^* = f$, that is, if $f(a^*) = f(a)^*$ for all $a \in A$, $f$ is said to be self-adjoint; an equivalent condition is that $f$ be real-valued on the self-adjoint elements of $A$. The correspondence $f \mapsto f^*$ is conjugate-linear and involutory: $(f + g)^* = f^* + g^*$, $(\lambda f)^* = \lambda^* f^*$, $f^{**} = f$. Every linear form $f$ has a unique representation $f = g + ih$ with $g$ and $h$ self-adjoint (called the Cartesian decomposition of $f$).
