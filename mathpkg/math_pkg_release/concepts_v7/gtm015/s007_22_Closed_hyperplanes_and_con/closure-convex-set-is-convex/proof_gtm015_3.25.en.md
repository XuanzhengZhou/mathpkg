# Proof of Closure of a convex set in a TVS is convex

Fix $\alpha$, $0 \le \alpha \le 1$. The mapping $u: E \times E \rightarrow E$ defined by $u(x, y) = \alpha x + (1 - \alpha)y$ is continuous, and maps $A \times A$ into $A$, therefore it maps $\overline{A} \times \overline{A} = \overline{A \times A}$ into $\overline{A}$. Since $\alpha$ is arbitrary, it follows that $\overline{A}$ is convex.
