A preordered [ordered] vector space over $\mathbb{R}$ is a pair $(E, \leq)$, where $E$ is a real vector space and $\leq$ is a preordering [ordering] of $E$ that is compatible with the vector space structure: $x \leq y$ implies $x + z \leq y + z$ for all $z$, and $x \leq y$ and $\alpha \geq 0$ imply $\alpha x \leq \alpha y$.
