# Proof of Explicit description of generated convex cone

If $S$ is any subset of $E$, the convex cone generated by $S$ is

$$C = \left\{\sum_{i \in J} \alpha_i y_i : J \text{ nonempty finite}, y_i \in S, \alpha_i > 0\right\}.$$

Proof: Let $C$ be the set of all such sums. Clearly $C$ is a cone: multiplying by $\lambda > 0$ just multiplies all coefficients. Also $C$ is convex: if $x, y \in C$ and $0 < t < 1$, then $tx + (1-t)y$ is also a sum of the same form (combining the coefficients). Therefore $C$ is a convex cone containing $S$. Conversely, any convex cone $K$ containing $S$ must contain all such finite linear combinations with positive coefficients, because $K$ is closed under positive scaling and addition per (27.5). Hence $C$ is precisely the intersection of all convex cones containing $S$.
