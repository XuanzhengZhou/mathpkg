# Exercise 13.8

If $(E, d)$ is a space of type $(F)$ then the metric $d'$ defined by

$$d'(x, y) = \frac{d(x, y)}{1 + d(x, y)}$$

generates the same topology as $d$, and $(E, d')$ is also a space of type $(F)$.
