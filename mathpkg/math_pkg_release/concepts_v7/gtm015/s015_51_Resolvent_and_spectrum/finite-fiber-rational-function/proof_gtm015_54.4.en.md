# Proof of Rational functions have finite fibers

**Proof.** If $\eta = \infty$, the equation $g(\xi) = \infty$ means $\xi$ is a pole of $g$, of which there are only finitely many.

If $\eta \in \mathbb{C}$, write $g = p/q$ in reduced form. The equation $g(\xi) = \eta$ means $p(\xi)/q(\xi) = \eta$, i.e., $p(\xi) - \eta q(\xi) = 0$ and $q(\xi) \neq 0$. The polynomial $p - \eta q$ has finitely many roots (unless it is identically zero, which would imply $g$ is constant, contradicting the hypothesis).
