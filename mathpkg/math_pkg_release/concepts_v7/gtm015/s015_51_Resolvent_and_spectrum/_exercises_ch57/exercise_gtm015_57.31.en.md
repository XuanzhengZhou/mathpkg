# Exercise 57.31

(57.31) If $T \in \mathcal{L}(E)$, $E$ a reflexive Banach space, then
$$\sigma(T) = \sigma_{\text{p}}(T) \cup \sigma_{\text{p}}(T') \cup [\sigma_{\text{ap}}(T) \cap \sigma_{\text{ap}}(T')]$$
if $E$ is a Hilbert space, then
$$\sigma(T) = \sigma_{\text{p}}(T) \cup (\sigma_{\text{p}}(T^*))^* \cup [\sigma_{\text{ap}}(T) \cap (\sigma_{\text{ap}}(T^*))^*]$$.
