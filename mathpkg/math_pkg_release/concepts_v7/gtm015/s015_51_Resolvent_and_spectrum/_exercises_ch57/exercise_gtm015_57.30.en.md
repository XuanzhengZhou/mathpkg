# Exercise 57.30

(57.30) If $T \in \mathcal{L}(E)$, $E$ a Banach space, then
$$\sigma(T) = \sigma_{\text{ap}}(T) \cup \sigma_{\text{p}}(T')$$
if $E$ is a Hilbert space, then $\sigma(T) = \sigma_{\text{ap}}(T) \cup (\sigma_{\text{p}}(T^*))^*$.
