# Proof of A set of seminorms and its saturation generate the same topology

Since $\mathcal{F} \subset \mathcal{F}^*$, obviously $\tau(\mathcal{F}) \subset \tau(\mathcal{F}^*)$. Conversely, assume $p \in \mathcal{F}^*$; it is to be shown that $\tau_p \subset \tau(\mathcal{F})$. Choose $p_1, \ldots, p_M \in \mathcal{F}$ with $p \leq p_1 + \cdots + p_M$, and write $q = p_1 + \cdots + p_M$. In view of (37.11), it will suffice to show that $\tau_q \subset \tau(\mathcal{F})$. Set $r(x) = \max\{p_1(x), \ldots, p_M(x)\}$. Then $r \leq q \leq Mr$, whence $\tau_r = \tau_q$ by (37.11). Since $\tau_r$ is the topology generated by $\{p_1, \ldots, p_M\} \subset \mathcal{F}$, we have $\tau_r \subset \tau(\mathcal{F})$, and therefore $\tau_q \subset \tau(\mathcal{F})$.
