# Proof of Equality of topologies from saturated seminorm families implies equality of families

If $\tau(\mathcal{F}) = \tau(\mathcal{G})$, then a seminorm on $E$ is continuous for $\tau(\mathcal{F})$ if and only if it is continuous for $\tau(\mathcal{G})$; in view of (37.17), this means that $\mathcal{F}^* = \mathcal{G}^*$. But $\mathcal{F} = \mathcal{F}^*$ and $\mathcal{G} = \mathcal{G}^*$ by hypothesis, thus $\mathcal{F} = \mathcal{G}$.
