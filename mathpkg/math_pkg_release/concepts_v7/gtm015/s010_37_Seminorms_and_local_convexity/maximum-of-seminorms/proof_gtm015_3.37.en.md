# Proof of Finite maximum of seminorms is a seminorm

Let $E$ be a vector space over $\mathbb{K}$ and let $p_1, \ldots, p_n$ be seminorms on $E$. Define

$$p(x) = \max\{p_1(x), \ldots, p_n(x)\} \quad (x \in E).$$

It is clear that $p$ is a seminorm: nonnegativity, absolute homogeneity, and the triangle inequality follow from the corresponding properties of each $p_k$.

For each $\varepsilon > 0$,

$$\{x : p(x) \leq \varepsilon\} = \{x : p_k(x) \leq \varepsilon \text{ for } k = 1, \ldots, n\} = \bigcap_{k=1}^n \{x : p_k(x) \leq \varepsilon\},$$

therefore $\{x : p(x) \leq \varepsilon\}$ is a neighborhood of $\theta$ for the topology $\tau$ generated by the set $\{p_1, \ldots, p_n\}$; it follows that $p$ is continuous at $\theta$ for the topology $\tau$. Since $\tau$ is compatible with the vector space structure (37.10), it follows from (37.5) that $\tau_p \subset \tau$ and that $p$ is $\tau$-continuous on $E$.
