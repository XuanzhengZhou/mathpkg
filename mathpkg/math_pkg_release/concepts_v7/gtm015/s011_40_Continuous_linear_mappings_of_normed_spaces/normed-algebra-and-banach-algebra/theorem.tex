\text{An associative algebra over } \mathbb{K} \text{ is a vector space } A \text{ over } \mathbb{K}, \text{ with an associative multiplication } ab \text{ that is distributive with respect to addition and is compatible with scalar multiplication:}

$$\lambda(ab) = (\lambda a)b = a(\lambda b) \quad (a,b \in A,\ \lambda \in \mathbb{K}).$$

\text{A normed algebra over } \mathbb{K} \text{ is an associative algebra } A \text{ over } \mathbb{K} \text{ equipped with a norm } a \mapsto \|a\| \text{ that is submultiplicative:}

$$\|ab\| \leq \|a\| \|b\| \quad (a,b \in A).$$

\text{If, moreover, } A \text{ is complete with respect to the norm metric (i.e., is a Banach space), then it is called a Banach algebra.}
