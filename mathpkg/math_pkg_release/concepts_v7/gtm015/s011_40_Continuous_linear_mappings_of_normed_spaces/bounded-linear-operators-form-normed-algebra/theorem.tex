\text{If } E \text{ is a normed space over } \mathbb{K}, \text{ then } \mathcal{L}(E) = \mathcal{L}(E, E) \text{ is a normed algebra, with unity element the identity operator } I, \text{ and } \|I\| = 1.

\text{If, moreover, } E \text{ is a Banach space, then } \mathcal{L}(E) \text{ is a Banach algebra.}
