A \text{partition of unity} subordinate to an open cover $\mathcal{U} = \{U_{\lambda}\}_{\lambda \in \Lambda}$ is a collection $\{f_{\lambda}\}_{\lambda \in \Lambda}$ of continuous maps $f_{\lambda}: X \to [0,1]$ such that:
\begin{enumerate}
\item For each $x \in X$, the set $\{\lambda \in \Lambda : f_{\lambda}(x) > 0\}$ is finite;
\item $\sum_{\lambda \in \Lambda} f_{\lambda}(x) = 1$ for all $x \in X$;
\item $\operatorname{supp} f_{\lambda} \subset U_{\lambda}$ for each $\lambda$.
\end{enumerate}
