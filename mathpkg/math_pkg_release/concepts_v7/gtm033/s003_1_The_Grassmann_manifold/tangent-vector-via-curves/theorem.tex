A tangent vector to $M$ is sometimes defined as an equivalence class of $C^1$ maps $f: [0,a) \to M$, where $f$ is equivalent to $g: [0,b) \to M$ if $f(0) = g(0)$ and for some (and hence any) chart $(\varphi_i, U_i)$ at $f(0)$,
$$D(\varphi_i f)(0) = D(\varphi_i g)(0).$$
To such an equivalence class we associate the tangent vector (as defined previously)
$$[f(0), i, D(\varphi_i f)(0)].$$
Conversely, to a tangent vector $[x,i,a]$ we can associate the curve $t \mapsto \varphi_i^{-1}(\varphi_i(x) + ta)$.
