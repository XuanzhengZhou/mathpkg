\text{Let } \xi = (p, E, B) \text{ be a vector bundle. A map } h: Q \to E^* \text{ into the Thom space is in }\textit{standard form}\text{ if there is a submanifold } M \subset Q \text{ and a tubular neighborhood } U \subset Q \text{ of } M \text{ such that } U = h^{-1}(E), M = h^{-1}(B), \text{ and the restriction } h|U: U \to E \text{ is a vector bundle map. This implies } h \pitchfork B \text{ and } h(Q - U) = \{\infty\}.
