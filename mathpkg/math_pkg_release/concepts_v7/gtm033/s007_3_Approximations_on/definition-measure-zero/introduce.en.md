This definition introduces the notion of a measure-zero (null) set in $\mathbb{R}^n$ using coverings by $n$-cubes of arbitrarily small total measure. Key properties are that countable unions of measure-zero sets remain measure-zero, and that the property is local (by Lindelöf's principle). This definition, which uses only elementary geometry rather than full Lebesgue measure theory, is sufficient for the Morse-Sard theorem and all transversality applications in differential topology.
