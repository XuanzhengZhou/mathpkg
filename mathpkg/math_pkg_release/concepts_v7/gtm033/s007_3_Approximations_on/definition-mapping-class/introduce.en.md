A $C^r$ mapping class is an abstract framework for local-to-global properties of maps between manifolds. The class assigns to each triple (closed set $L$, open domain $U$, open target $V$) a set of maps satisfying a localization axiom: a map belongs to the class on $L$ if it locally belongs to the class on a cover of $L$. A mapping class is "rich" if it is locally dense and open on compact sets. Examples include transversality to a fixed submanifold. The Globalization Theorem 2.2 shows rich classes are globally dense and open.
