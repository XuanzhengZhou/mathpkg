Let $M$ and $N$ be $C^r$ manifolds, $0 \leqslant r \leqslant \infty$. A $C^r$ mapping class on $(M,N)$ is a function $\mathcal{X}$ whose domain is the set of triples $(L,U,V)$ where $U \subset M$, $V \subset N$ are open, $L \subset M$ is closed, and $L \subset U$. To each triple $\mathcal{X}$ assigns a set of maps $\mathcal{X}_L(U,V) \subset C^r(U,V)$ satisfying the localization axiom:

Given triples $(L,U,V)$, a map $f \in C^r(U,V)$ belongs to $\mathcal{X}_L(U,V)$ provided there exist triples $(L_i,U_i,V_i)$ and maps $f_i \in \mathcal{X}_{L_i}(U_i,V_i)$ such that $L \subset \bigcup L_i$ and $f = f_i$ in a neighborhood of $L_i$, for all $i$.

A mapping class $\mathcal{X}$ is called rich if there are open covers $\mathcal{U}, \mathcal{V}$ of $M, N$ such that whenever $U \subset M$, $V \subset N$ are subsets of elements of $\mathcal{U}, \mathcal{V}$ and $L \subset U$ is compact, then $\mathcal{X}_L(U,V)$ is dense and open in $C^r_W(U,V)$.
