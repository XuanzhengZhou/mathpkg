# Exercise 1(b)

If $\dim X < s - k$, show that under the classification of vector bundles (Theorem 3.4), the induced map

$$i_\#: [X, G_{s,k}] \to [X, G_{s+1,k+1}]$$

corresponds to the stabilization map

$$\sigma: K^k(X) \to K^{k+1}(X), \qquad [\xi] \mapsto [\xi \oplus \varepsilon^1].$$

Here $K^k(X)$ denotes the set of isomorphism classes of $k$-plane bundles over $X$, $\varepsilon^1$ is the trivial line bundle, and $[X, G_{s,k}]$ denotes the set of homotopy classes of maps from $X$ to $G_{s,k}$.
