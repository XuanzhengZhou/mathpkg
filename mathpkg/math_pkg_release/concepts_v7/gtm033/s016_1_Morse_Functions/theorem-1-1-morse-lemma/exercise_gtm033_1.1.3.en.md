# Exercise 3

Let $M \subset \mathbb{R}^q$ be a $C^2$ submanifold and $f: M \to \mathbb{R}$ a $C^2$ map. The set of linear maps $L \in L(\mathbb{R}^q, \mathbb{R})$ such that the map $M \to \mathbb{R}$, $x \mapsto f(x) + L(x)$ is a Morse function, is a Baire set and thus dense. If $M$ is compact it is open and dense.
