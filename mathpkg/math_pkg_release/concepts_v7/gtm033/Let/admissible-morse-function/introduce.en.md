An admissible Morse function is a Morse function on a compact manifold $M$ whose domain is a closed interval $[a,b]$ and whose boundary $\partial M$ coincides with a regular level set $f^{-1}(b)$. This setup ensures that the topology of $M$ can be analyzed by studying how sublevel sets $f^{-1}[a, c]$ change as $c$ increases. Each time $c$ passes a critical value, the homotopy type changes by the attachment of a cell whose dimension equals the index of the critical point. Admissible Morse functions are the fundamental tool for proving Morse inequalities and the CW-complex homotopy type theorem.
