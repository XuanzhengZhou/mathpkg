Let $\mathfrak{B}$ be an abelian category and $(\mathfrak{B}, f)$ the category of filtered objects of $\mathfrak{B}$. The associated graded object functor

$$Gr : (\mathfrak{B}, f) \rightarrow \mathfrak{B}^Z$$

attaches to a filtered object $B$ with filtration $\{B^{(p)}\}$ the graded object whose $p^{\text{th}}$ component is the subquotient

$$(Gr \, B)^p = B^{(p)} / B^{(p-1)}.$$
