For the associated graded object $Gr(M)$ to faithfully represent the filtered homology $M$, the filtration must be both separated (intersection zero) and exhaustive (union equals $M$). Without separation, non-zero elements can be infinitely deep in the filtration and vanish in the associated graded. Without exhaustiveness, some elements never appear in any filtration level. Both conditions ensure that each non-zero $x \in M$ corresponds to a unique homogeneous element in $Gr(M)$, so that the only information lost is the module-extension data between successive filtration levels.
