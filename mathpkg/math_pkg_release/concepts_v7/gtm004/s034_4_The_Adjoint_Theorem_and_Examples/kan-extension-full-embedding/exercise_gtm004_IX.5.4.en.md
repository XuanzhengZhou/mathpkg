# Exercise 5.4

Prove that, if $T: \mathfrak{D} \rightarrow \mathfrak{A}$ is a functor from the cofiltering category $\mathfrak{D}$ to the category $\mathfrak{A}$ with colimits and if $K: \mathfrak{C} \rightarrow \mathfrak{D}$ is a cofinal functor from the cofiltering category $\mathfrak{C}$ to $\mathfrak{D}$, then

$$\varinjlim_{\mathfrak{D}} T = \varinjlim_{\mathfrak{C}} TK.$$

(You should make the nature of this equality quite precise.)
