# Exercise 9.2

Show that if $B_i$ is an essential extension of $A_i$ for $i = 1, 2$, then $B_1 \oplus B_2$ is an essential extension of $A_1 \oplus A_2$.

Extend this to direct sums over any index set $J$: if $B_j$ is an essential extension of $A_j$ for each $j \in J$, then $\bigoplus_{j \in J} B_j$ is an essential extension of $\bigoplus_{j \in J} A_j$.

*Hint:* Use the elementwise characterization of essential extensions (Proposition 9.1).
