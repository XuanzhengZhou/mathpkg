# Exercise 3.5

Let $C$ be a filtered chain complex of abelian groups in which

$$C^{(p)} = 0, \quad p < 0,$$
$$C_q^{(p)} = C_q, \quad p \ge q.$$

Show that if $C$ satisfies the following conditions (i), (ii), (iii), then $E_r^{p,q}$ is finitely generated for all $p, q$:

(i) $H_q(C)$ is finitely generated for all $q$.

(ii) $E_1^{0,q}$ is finitely generated for all $q$.

(iii) For all $p$, if $E_r^{p,0}$ is finitely generated, then $E_r^{p,q}$ is finitely generated for all $q$.

Also show that if $C$ satisfies the conditions (i'), (ii'), (iii'), then $E_r^{p,q}$ is finitely generated for all $p, q$:

(i') $H_q(C)$ is finitely generated for all $q$.

(ii') $E_1^{0,0}$ is finitely generated.

(iii') For all $r$, if $E_r^{p,r}$ is finitely generated, then $E_{r+1}^{p,p+r}$ is finitely generated for all $p$.
