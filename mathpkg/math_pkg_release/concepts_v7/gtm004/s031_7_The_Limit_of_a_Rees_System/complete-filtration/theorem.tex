\textbf{Definition.} A filtration
$$
\cdots \subseteq X^{p-1} \subseteq X^p \subseteq \cdots \subseteq X, \quad -\infty < p < \infty
$$
in an abelian category $\mathfrak{A}$ is:
\begin{itemize}
\item \textit{left complete} if the canonical map $\lambda : \varinjlim_p X^p \to X$ is an isomorphism;
\item \textit{right complete} if the canonical map $\kappa : X \to \varprojlim_p X_p$ is an isomorphism;
\item \textit{complete} if it is both left and right complete.
\end{itemize}

\textbf{Remarks.} A finite filtration is obviously complete. In a category of modules, left completeness is equivalent to $X = \bigcup_p X^p$. Right completeness in a category of modules requires both $\bigcap_p X^p = 0$ and the surjectivity of the natural map $X \to \varprojlim_p X / X^p$.
