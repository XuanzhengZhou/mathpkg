An injective object is the dual of a projective object: any morphism into it can be extended along any monomorphism. In module categories, injective objects are precisely the injective modules. While projectivity corresponds to a lifting property, injectivity corresponds to an extension property. Injective objects are used to construct injective resolutions, which provide an alternative (and often more flexible) approach to defining derived functors.
