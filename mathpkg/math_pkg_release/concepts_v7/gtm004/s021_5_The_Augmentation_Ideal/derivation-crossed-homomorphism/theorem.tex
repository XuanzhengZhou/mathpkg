A function $d : G \rightarrow A$ from the group $G$ into the $G$-module $A$ with the property
$$d(x \cdot y) = dx + x \circ (dy), \quad x, y \in G,$$
is called a \textbf{derivation} (or \textbf{crossed homomorphism}) from $G$ into $A$.

If $d$ is a derivation, then $d(1) = 0$. The set of all derivations $d : G \rightarrow A$ forms an abelian group, denoted by $\operatorname{Der}(G, A)$.
