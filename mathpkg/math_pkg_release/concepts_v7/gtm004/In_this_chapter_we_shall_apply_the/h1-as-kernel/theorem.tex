For an arbitrary right $G$-module $B$,
$$H_1(G, B) = \ker\bigl(\iota_* : B \otimes_{\mathbb{Z}G} IG \to B\bigr),$$
where $\iota_*(b \otimes (x - 1)) = bx - b$ for $b \in B$, $x \in G$.

In particular, if $B$ is a trivial $G$-module, then $\iota_*$ is the zero homomorphism and
$$H_1(G, B) \cong B \otimes_{\mathbb{Z}G} IG \cong B \otimes G_{\mathrm{ab}}.$$
