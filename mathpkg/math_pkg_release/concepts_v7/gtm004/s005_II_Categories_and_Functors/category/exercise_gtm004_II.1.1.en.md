# Exercise 1.1

Show how to represent an ordered set as a category. (Hint: Regard the elements $a, b, \ldots$ of the set as objects in the category, and the instances $a \leq b$ of the ordering relation as morphisms $a \rightarrow b$.) Express in categorical language the fact that the ordered set is directed [16]. Show that a subset of an ordered set, with its natural ordering, is a full subcategory.
