A zero object is an object that is both initial and terminal — there is exactly one morphism from any object to it and exactly one morphism from it to any object. In the category of groups (or abelian groups), any one-element group serves as a zero object. Zero objects, when they exist, are unique up to canonical isomorphism. They provide the categorical basis for defining zero morphisms.
