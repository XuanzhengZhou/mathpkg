# Exercise 1.4

Show that the class of the split extension in $E(A, B)$ is preserved under the induced maps. That is, if $B \to A \oplus B \to A$ represents the split extension, then for any $\alpha: A' \to A$, the pullback extension $\alpha^*$ applied to the split extension yields (the class of) the split extension of $A'$ by $B$; and for any $\beta: B \to B'$, the pushout extension $\beta_*$ applied to the split extension yields (the class of) the split extension of $A$ by $B'$.
