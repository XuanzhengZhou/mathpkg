Let $\mathfrak{U}$ be an abelian category and let $\mathcal{E}$ be a class of epimorphisms in $\mathfrak{U}$. An object $P$ in $\mathfrak{U}$ is called {\it projective relative to} $\varepsilon : B \to C$ (where $\varepsilon \in \mathcal{E}$) if the induced map
\[
\varepsilon_* : \mathfrak{U}(P, B) \to \mathfrak{U}(P, C)
\]
is surjective. The object $P$ is called {\it $\mathcal{E}$-projective} if it is projective relative to $\varepsilon$ for every $\varepsilon \in \mathcal{E}$.

An object $P_1 \oplus P_2$ is projective relative to $\varepsilon$ if and only if both $P_1$ and $P_2$ are projective relative to $\varepsilon$.
