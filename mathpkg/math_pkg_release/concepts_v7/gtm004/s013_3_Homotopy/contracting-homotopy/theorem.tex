\text{In the special case where}~\varphi = 0 : C \rightarrow C~\text{and}~\psi = 1 : C \rightarrow C,~\text{a homotopy}~\Sigma : 0 \Rightarrow 1~\text{is called a \text{contracting homotopy} for}~C.~\text{We are then given homomorphisms}~\Sigma_n : C_n \rightarrow C_{n+1}~\text{with}

$$\partial_{n+1} \Sigma_n + \Sigma_{n-1} \partial_n = 1, \quad n \in \mathbb{Z}.$$

\text{An immediate consequence of the existence of such a contracting homotopy is}~H(C) = 0,~\text{hence the complex}~C~\text{is exact.}
