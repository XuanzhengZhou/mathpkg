Given a module homomorphism $\alpha : A' \to A$, one can lift it to a map $\pi : P' \to P$ between projective presentations, which then induces a map on Ext groups. Lemma 2.1 of Hilton--Stammbach shows that this induced map $\pi^* : \operatorname{Ext}_{\Lambda}^{\varepsilon}(A, B) \to \operatorname{Ext}_{\Lambda}^{\varepsilon'}(A', B)$ is independent of the particular lifting $\pi$ chosen. This well-definedness is essential for making $\operatorname{Ext}_{\Lambda}(-, B)$ into a functor: the induced map depends only on $\alpha$, so one obtains a genuine contravariant functor in the first variable.
