# Exercise 3

Solve for $u$:

$$u''(t) - 3u'(t) + 2u(t) = t^2 - \cos t, \qquad t > 0,$$
$$u(0+) = u'(0+) = 0.$$

(Note: The source text lists $u''(0+) = 0$ as well; this condition is either redundant for a second-order equation or the equation may have been intended as third-order, e.g., $u'''(t) - 3u'(t) + 2u(t) = t^2 - \cos t$.)
