# Exercise 7

Show that the solution of

$$p(D)u(t) = 0, \quad t > 0,$$
$$D^k u(0+) = 0, \quad 0 \leq k < j \text{ and } j < k \leq n-1,$$
$$D^j u(0+) = 1$$

is a linear combination of functions $t^k \exp(z t)$, where $z$ ranges over the roots of $p$.
