A periodic distribution $F$ is \textbf{real} if $F = F^*$, where the complex conjugate $F^*$ is defined by
$$F^*(u) = \bigl(F(u^*)\bigr)^*, \quad u^*(x) = \overline{u(x)}.$$

Any $F \in \mathcal{P}'$ can be written uniquely as
$$F = G + iH, \quad \text{with } G, H \text{ real}.$$
Explicitly,
$$G = \operatorname{Re} F = \tfrac{1}{2}(F + F^*), \qquad H = \operatorname{Im} F = \tfrac{1}{2i}(F - F^*).$$
