Suppose $f: \mathbb{R} \to \mathbb{C}$ is a continuous function such that for some $a \in \mathbb{R}$, $M \in \mathbb{R}$,
$$f(t) = 0, \quad t \leq M,$$
and $e^{-at}f(t)$ is bounded. Define $F_f: \mathcal{L} \to \mathbb{C}$ by
$$F_f(u) = \int_{-\infty}^{\infty} f(t) u(t) \, dt.$$
Then $F_f$ is a continuous linear functional on $\mathcal{L}$, i.e., $F_f \in \mathcal{L}'$.

We say that $F \in \mathcal{L}'$ is a function, or is defined by a function, if there exists a continuous function $f$ satisfying the above conditions such that $F = F_f$.
