The Intermediate Value Theorem (Theorem 1.5) states that a continuous real-valued function on an interval takes every value between its values at the endpoints. The proof uses a bisection argument: repeatedly halve the interval, keeping a subinterval where $c$ lies between the endpoint values, and the nested intervals converge to a point where $f(x_0) = c$ by continuity. This theorem captures the intuitive idea that the graph of a continuous function has no "jumps."
