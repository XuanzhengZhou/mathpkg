In a metric space, a set is open if every point in the set has some open ball around it that is completely contained within the set. Equivalently, a set is a neighborhood of a point $x$ if it contains an open ball centered at $x$. The concepts of open set and neighborhood form the basis of the topological structure induced by a metric.
