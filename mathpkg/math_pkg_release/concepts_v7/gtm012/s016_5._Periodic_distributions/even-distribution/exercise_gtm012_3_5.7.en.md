# Exercise 7

Show that if $F$ is even and $u \in \mathcal{P}$ is odd, then $F(u) = 0$.

Show that if $F$ is odd and $u \in \mathcal{P}$ is even, then $F(u) = 0$.

Recall: $F$ is even if $F = F^\sim$, and $F$ is odd if $F = -F^\sim$, where $F^\sim(u) = F(\tilde{u})$ and $\tilde{u}(x) = u(-x)$. A function $u$ is even if $\tilde{u} = u$, and odd if $\tilde{u} = -u$.
