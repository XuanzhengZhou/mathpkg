The union of two sets $A$ and $B$ is defined as
$$A \cup B = \{x \mid x \in A \text{ or } x \in B \text{ (or both)}\}.$$

More generally, if $J$ is an index set and $\{A_j\}_{j \in J}$ is a family of sets, their union is
$$\bigcup_{j \in J} A_j = \{x \mid x \in A_j \text{ for some } j \in J\}.$$

For a finite family $A_1, A_2, \ldots, A_m$, the union is denoted
$$A_1 \cup A_2 \cup \cdots \cup A_m \quad \text{or} \quad \bigcup_{j=1}^{m} A_j.$$

For a countably infinite family $A_1, A_2, \ldots$, the union is denoted
$$\bigcup_{j=1}^{\infty} A_j.$$
