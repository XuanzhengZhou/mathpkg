# Exercise 2

Show that if $f$ is defined by the power series

$$f(x) = \sum_{n=0}^{\infty} a_n (x - x_0)^n, \quad |x - x_0| < R,$$

then

$$\int_{x_0}^{x} f(t) \, dt = \sum_{n=0}^{\infty} (n+1)^{-1} a_n (x - x_0)^{n+1}, \quad |x - x_0| < R.$$
