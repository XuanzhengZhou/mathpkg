# Exercise 2

Show that the Mean Value Theorem is not true for complex-valued functions, in general, by finding a differentiable function \(f\) such that \(f(0) = 0 = f(1)\) but \(f'(x) \neq 0\) for all \(x \in [0, 1]\).

*Note: The OCR source text for this exercise was truncated after "but". The statement has been completed based on the standard counterexample \(f(x) = e^{2\pi i x} - 1\).*
