If $f$ is holomorphic on an open set containing a simple closed curve $\gamma$ and its interior, then for any $z$ inside $\gamma$,
$$f(z) = \frac{1}{2\pi i} \oint_\gamma \frac{f(w)}{w - z} \, dw.$$
Moreover, $f$ is infinitely differentiable and
$$f^{(n)}(z) = \frac{n!}{2\pi i} \oint_\gamma \frac{f(w)}{(w - z)^{n+1}} \, dw.$$
