# Proof of $(-1) \cdot x = -x$

Using the field axioms, we compute:

$$x + (-1) \cdot x = 1 \cdot x + (-1) \cdot x \quad (\text{since } 1 \cdot x = x \text{ by M3})$$
$$= (1 + (-1)) \cdot x \quad (\text{by the distributive law, DL})$$
$$= 0 \cdot x \quad (\text{since } 1 + (-1) = 0 \text{ by A4})$$
$$= 0 \quad (\text{since } 0 \cdot x = 0 \text{ was proved above}).$$

Thus $x + (-1) \cdot x = 0$, which means $(-1) \cdot x$ is the additive inverse of $x$. By the uniqueness of the additive inverse (which follows from the axioms), we conclude

$$(-1) \cdot x = -x$$

for all $x \in \mathbb{R}$.
