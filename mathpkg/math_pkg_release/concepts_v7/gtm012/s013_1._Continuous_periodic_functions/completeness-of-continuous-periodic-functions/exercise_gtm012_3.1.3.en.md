# Exercise 3

If $u \in \mathcal{C}$, define the real number $|u|'$ by

$$|u|' = (2\pi)^{-1} \int_{0}^{2\pi} |u(x)| \, dx.$$

Show that $|u|'$ is a norm on $\mathcal{C}$ and that $|u|' \leq |u|$.
