Let $X$ be a vector space over the real or complex numbers. A \emph{seminorm} on $X$ is a function $|\cdot|: X \to \mathbb{R}$ satisfying:
\begin{enumerate}
\item $|u| \geq 0$;
\item $|a u| = |a| \, |u|$ for all scalars $a$;
\item $|u + v| \leq |u| + |v|$.
\end{enumerate}
A seminorm is a norm if and only if $|u| = 0$ implies $u = 0$.
