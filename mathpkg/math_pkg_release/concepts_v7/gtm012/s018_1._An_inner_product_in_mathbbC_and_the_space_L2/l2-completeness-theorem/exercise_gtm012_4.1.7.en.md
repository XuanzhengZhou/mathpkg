# Exercise 7

Show that if $F \in \mathbf{L}^2$, there is a sequence $(u_n)_{1}^{\infty}$ of smooth periodic functions such that

$$u_n \to F \quad \text{in } \mathbf{L}^2.$$

In other words, the smooth periodic functions $\mathcal{P}$ are dense in $\mathbf{L}^2$.

*Hint.* First approximate $F$ by a sequence in $\mathcal{C}$, then approximate each continuous function by a smooth function using the Weierstrass approximation theorem or convolution with an approximate identity.
