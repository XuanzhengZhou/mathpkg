\textbf{Theorem 6.1.} Let $F$ be a periodic distribution. There exists a unique function $u(r, \theta)$ defined and smooth in the open unit disc, satisfying

$$r^2 \frac{\partial^2 u}{\partial r^2} + r \frac{\partial u}{\partial r} + \frac{\partial^2 u}{\partial \theta^2} = 0, \quad 0 \leq r < 1,$$

and such that the distributions defined by the functions $u_r(\theta) = u(r, \theta)$ converge to $F$ in the sense of $\mathcal{P}'$ as $r \to 1$. Moreover, if $F = F_g$ where $g$ is a continuous periodic function, then $u_r \to g$ uniformly as $r \to 1$. The functions $u_r$ are given by convolution with the Poisson kernel:

$$u_r = F * P_r.$$
