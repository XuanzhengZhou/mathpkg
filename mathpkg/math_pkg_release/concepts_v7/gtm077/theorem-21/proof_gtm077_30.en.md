# Proof of Theorem 21

This theorem appears in §11 of the book (Theory of Finite Abelian Groups). In a finite Abelian group, the order of each element divides the group order.

Let $G$ be a finite Abelian group of order $n$. For any $g \in G$, the cyclic subgroup $\langle g \rangle$ has order equal to the order of $g$. By Lagrange's theorem, this order divides $|G| = n$. Hence $g^n = e$ for all $g \in G$.

In the context of ideal theory (§33), this theorem is applied to the class group: the class number $h$ is the order of the class group, so every ideal class raised to the $h$th power yields the principal class, i.e., $\mathfrak{a}^h \sim (1)$.
