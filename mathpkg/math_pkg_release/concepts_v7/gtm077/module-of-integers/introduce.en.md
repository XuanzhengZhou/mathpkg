A module of integers is a nonempty subset closed under addition and subtraction — in modern terms, an additive subgroup of $\mathbb{Z}$. This concept provides a unified algebraic framework for many classical number-theoretic constructions. For example, the set of all common multiples of two integers is a module, as is the set of all integer linear combinations $ax + by$. The module concept is central to Dedekind's approach to ideal theory.
