If $(a, n) = 1$, the congruence
$$a a' \equiv 1 \pmod{n}$$
has exactly one solution $a'$ modulo $n$, called the modular inverse of $a$ modulo $n$. By Euler's theorem, one may take $a' = a^{\varphi(n)-1} \pmod{n}$.

The solution of the linear congruence $ax + b \equiv 0 \pmod{n}$ (when $(a,n)=1$) is then
$$x \equiv -a' b \pmod{n}.$$
