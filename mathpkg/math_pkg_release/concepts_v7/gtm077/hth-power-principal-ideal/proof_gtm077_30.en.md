# Proof of Theorem 97

As an immediate consequence of the finiteness of the class number $h$ we obtain from Theorem 21 (which states that in a finite Abelian group, the order of each element divides the group order):

The $h$th power of each ideal in $K$ is a principal ideal.

Indeed, each ideal class in the class group has finite order dividing the class number $h$. Thus for any ideal $\mathfrak{a}$, the ideal $\mathfrak{a}^h$ lies in the principal class, meaning $\mathfrak{a}^h = (\omega)$ for some $\omega \in K$.
