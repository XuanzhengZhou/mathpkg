A system of $m$ elements $A_1, \ldots, A_m$ of a finite Abelian group, with orders $a_1, \ldots, a_m$, is called independent if
$$A_1^{x_1} A_2^{x_2} \cdots A_m^{x_m} = E \implies x_i \equiv 0 \pmod{a_i} \quad \text{for all } i = 1, \ldots, m.$$
The set of all products of powers of $m$ independent elements forms a subgroup of order $a_1 a_2 \cdots a_m$.
