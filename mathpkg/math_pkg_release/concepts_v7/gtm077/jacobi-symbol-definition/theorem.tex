For an odd positive integer $n = p_1^{\alpha_1} p_2^{\alpha_2} \cdots p_r^{\alpha_r}$ and any integer $a$, the Jacobi symbol is defined as
$$\left(\frac{a}{n}\right) = \left(\frac{a}{p_1}\right)^{\alpha_1} \left(\frac{a}{p_2}\right)^{\alpha_2} \cdots \left(\frac{a}{p_r}\right)^{\alpha_r},$$
where each factor on the right is the Legendre symbol. The Jacobi symbol satisfies:
$$\left(\frac{a}{n}\right) = \left(\frac{a'}{n}\right) \text{ if } a \equiv a' \pmod{n}, \qquad
\left(\frac{ab}{n}\right) = \left(\frac{a}{n}\right)\left(\frac{b}{n}\right).$$
