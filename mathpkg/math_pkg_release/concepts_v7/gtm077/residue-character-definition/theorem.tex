A function $\chi(a)$ defined on all integers is called a residue character modulo $n$ if:
\begin{enumerate}
\item[(1)] $\chi(a) = \chi(b)$ whenever $a \equiv b \pmod{n}$,
\item[(2)] $\chi(a)\chi(b) = \chi(ab)$ for all $a, b$,
\item[(3)] $\chi(a) \neq 0$ if $(a, n) = 1$,
\item[(4)] $\chi(a) = 0$ if $(a, n) > 1$.
\end{enumerate}
There are exactly $\varphi(n)$ distinct residue characters modulo $n$, and they correspond to the characters of the group $\mathfrak{R}(n)$.
