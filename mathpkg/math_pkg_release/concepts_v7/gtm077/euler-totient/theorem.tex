Euler's totient function $\varphi(n)$ is defined for positive integers $n$ as the number of residue classes modulo $n$ that are relatively prime to $n$. Equivalently, $\varphi(n)$ is the number of integers $k$ with $1 \leq k \leq n$ and $(k, n) = 1$.

For a prime power $p^k$:
$$\varphi(p^k) = p^k - p^{k-1} = p^k \left(1 - \frac{1}{p}\right).$$
