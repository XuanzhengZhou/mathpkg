# Proof of Lemma (b)

If

$$\mathfrak{a} = (\alpha_1, \ldots, \alpha_q)_k, \quad \mathfrak{A} = (\alpha_1, \ldots, \alpha_q)_K,$$
$$\mathfrak{b} = (\beta_1, \ldots, \beta_s)_k, \quad \mathfrak{B} = (\beta_1, \ldots, \beta_s)_K,$$

then $\mathfrak{a} = \mathfrak{b}$ implies $\mathfrak{A} = \mathfrak{B}$ and conversely.

*Proof.* The first part is obvious: if $\mathfrak{a} = \mathfrak{b}$, then each ideal has the same generators taken from $k$, so when these generators are regarded in $K$, the resulting ideals $\mathfrak{A}$ and $\mathfrak{B}$ coincide.

For the converse, assume $\mathfrak{A} = \mathfrak{B}$. Then each $\beta_j$ belongs to $\mathfrak{A}$, and by Lemma (a) it then also belongs to $\mathfrak{a}$. Likewise each $\alpha_i$ belongs to $\mathfrak{B}$, thus also to $\mathfrak{b}$. Consequently $\mathfrak{a} = \mathfrak{b}$.
