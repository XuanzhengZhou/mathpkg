Theorem 1, often called the Bezout identity in modern terminology, states that the greatest common divisor of two integers can be expressed as an integral linear combination of them. This result shows that the set of all values of the form $ax + by$ (as $x, y$ range over integers) is precisely the set of multiples of $(a,b)$. It is a foundational tool for solving linear Diophantine equations and for proving unique factorization.
