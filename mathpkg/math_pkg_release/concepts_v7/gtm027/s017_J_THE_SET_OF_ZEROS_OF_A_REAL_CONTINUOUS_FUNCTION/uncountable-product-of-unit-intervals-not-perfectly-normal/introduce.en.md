The uncountable product of unit intervals $[0,1]^\Lambda$ provides a classic counterexample showing that perfect normality is not preserved under uncountable products. Even though each factor $[0,1]$ is perfectly normal (being metrizable), the uncountable product fails to be perfectly normal because singletons are not $G_\delta$ sets. This illustrates a fundamental limitation: in an uncountable product, any $G_\delta$ set depends on only countably many coordinates, so it cannot be a singleton, which must constrain all uncountably many coordinates.
