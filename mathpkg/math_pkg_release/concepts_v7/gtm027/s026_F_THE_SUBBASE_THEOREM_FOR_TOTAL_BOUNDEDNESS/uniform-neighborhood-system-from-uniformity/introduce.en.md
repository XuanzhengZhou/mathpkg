# Uniform Neighborhood System from a Uniformity

This is the converse construction to obtaining a uniformity from a uniform neighborhood system. Given any uniformity $\mathfrak{u}$ on $X$, the entourages $U \in \mathfrak{u}$ naturally define neighborhoods $V_U(x) = U[x]$ around each point. The uniformity itself, directed by inclusion, serves as the index set, and the resulting structure $(V, \subset)$ is a uniform neighborhood system whose associated uniformity recovers the original $\mathfrak{u}$. This establishes the equivalence between the two formalisms.
