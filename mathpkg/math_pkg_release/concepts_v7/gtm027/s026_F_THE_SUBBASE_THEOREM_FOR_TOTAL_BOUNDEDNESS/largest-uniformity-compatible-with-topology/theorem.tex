\text{If } (X, \mathfrak{u}) \text{ is a completely regular space, then there is a largest uniformity } \mathfrak{u}^* \text{ for } X \text{ whose topology coincides with the original topology. This uniformity may be described alternately as the smallest uniformity which makes uniformly continuous each continuous map from } X \text{ into a metric space, or each continuous map from } X \text{ into a uniform space. Explicitly, } V \text{ is a member of } \mathfrak{u}^* \text{ iff } V \text{ is a neighborhood of the diagonal in } X \times X \text{ and there is a sequence } \{V_n, n \in \omega\} \text{ of symmetric neighborhoods of the diagonal such that } V_0 \subset V \text{ and } V_{n+1} \circ V_{n+1} \subset V_n \text{ for each } n \in \omega.
