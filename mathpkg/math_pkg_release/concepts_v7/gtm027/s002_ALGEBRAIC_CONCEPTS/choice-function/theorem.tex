Suppose the index set is itself a family $\alpha$ of sets, and that for each $A \in \alpha$ the $A$-th coordinate set is $A$. Then the cartesian product is the family of all functions $x$ on $\alpha$ such that $x_A \in A$ for each $A \in \alpha$. These functions are called \textbf{choice functions} for $\alpha$, since intuitively the function $x$ "chooses" a member $x_A$ from each set $A$.
