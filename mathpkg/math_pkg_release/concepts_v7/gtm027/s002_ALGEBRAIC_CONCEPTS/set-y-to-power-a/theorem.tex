When the coordinate set $X_a$ is a fixed set $Y$ for each $a$ in the index set $A$, the cartesian product is precisely the set of all functions on $A$ to $Y$, denoted $Y^A$. A familiar instance is real Euclidean $n$-space: the set of all real-valued functions on a set consisting of the integers $0, 1, \dots, n-1$, and the $i$-th coordinate of a member $x$ is $x_i$.
