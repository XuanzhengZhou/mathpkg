A standard exercise in general topology: if a topological space is second-countable (its topology admits a countable base), then any base for that topology necessarily contains a countable subfamily that still forms a base. This shows that having a countable base is a property of the topology itself, not merely a feature of a particular choice of basis. The result is useful for establishing that certain topological properties are preserved when passing between different bases.
