A subadditive function is a real-valued function on the non-negative reals satisfying $f(x+y) \leq f(x) + f(y)$. This property generalizes the triangle inequality and is fundamental in analysis and metric space theory. Subadditive functions that are also non-decreasing and vanish only at zero can be used to construct new metrics from existing ones while preserving the metric topology.
