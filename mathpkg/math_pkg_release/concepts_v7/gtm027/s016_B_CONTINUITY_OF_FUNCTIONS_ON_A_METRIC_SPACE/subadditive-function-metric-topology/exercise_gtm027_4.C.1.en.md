# Exercise C.1 — Subadditive Functions and Metric Topology

Let $f: [0, \infty) \to [0, \infty)$ be a continuous function such that:
1. $f(x) = 0$ if and only if $x = 0$,
2. $f$ is non-decreasing,
3. $f(x + y) \leq f(x) + f(y)$ for all $x, y \geq 0$ (subadditivity).

If $(X, d)$ is a metric space and $e(x, y) = f(d(x, y))$, prove that:

(a) $(X, e)$ is a metric space.

(b) The metric topology of $(X, e)$ is identical with that of $(X, d)$.

(A particular case which occurs frequently in the literature: $f(x) = x/(1+x)$.)
