Let $A$ be a well-ordered set and let $[0,1)$ have the usual order. The \textbf{transfinite line} is the set $A \times [0,1)$ endowed with the dictionary (lexicographic) order: $(a,b) < (c,d)$ if and only if $a < c$, or $a = c$ and $b < d$. The space is given the order topology induced by this lexicographic ordering.
