# Exercise E: Properties of the space of countable ordinals

Let $\Omega$ be the first uncountable ordinal and let $\Omega_0 = \{\alpha : \alpha < \Omega\}$ be the set of all countable ordinals, endowed with the order topology. Prove that $\Omega_0$ is locally compact, Hausdorff, satisfies the first axiom of countability, and is sequentially compact, but is not compact.
