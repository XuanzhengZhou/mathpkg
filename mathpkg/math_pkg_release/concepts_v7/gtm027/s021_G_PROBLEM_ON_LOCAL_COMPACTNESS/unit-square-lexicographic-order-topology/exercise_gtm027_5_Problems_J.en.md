# Exercise 5.J: Unit square with dictionary order

Let $X$ be the cartesian product of the closed unit interval $[0,1]$ with itself, ordered by dictionary (lexicographic) order. (That is, $(a,b) < (c,d)$ if and only if $a < c$ or $a = c$ and $b < d$.) Endow $X$ with the order topology. Prove that $X$ is compact, connected, and Hausdorff. Show that it satisfies the first countability axiom but is not separable and is hence not metrizable.
