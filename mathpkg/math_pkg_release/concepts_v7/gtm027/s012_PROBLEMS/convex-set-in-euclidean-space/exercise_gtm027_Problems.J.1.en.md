# Exercise J.1 — Convex Set in Euclidean $n$-Space

A subset $A$ of Euclidean $n$-space $E_n$ is **convex** iff for every pair $x$ and $y$ of points of $A$ and every real number $t$, such that $0 \leq t \leq 1$, the point $tx + (1-t)y$ is a member of $A$. (We define $(tx + (1-t)y)_i = t x_i + (1-t) y_i$.)
