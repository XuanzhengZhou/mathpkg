# Exercise F.2 — Upper Semi-Continuous Functions

A real-valued function $f$ on a topological space $X$ is **upper semi-continuous** iff the set $\{x: f(x) \geq a\}$ is closed for each real number $a$.

Equivalently, $f$ is upper semi-continuous iff $f$ is continuous relative to the upper topology $\mathcal{U}$ for the real numbers.

If $\{S_n, n \in D\}$ is a net of real numbers, then $\limsup \{S_n, n \in D\}$ is defined as the greatest limit point of the net relative to the upper topology $\mathcal{U}$.
