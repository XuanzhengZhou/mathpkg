A real-valued function $g$ on a topological space $X$ is called \emph{lower semi-continuous} if and only if $-g$ is upper semi-continuous.

If $f$ is a bounded real-valued function on $X$, let $f_{-} = -(-f)^{-}$ (where $f^{-}$ denotes the upper semi-continuous regularization). Define the \emph{oscillation} of $f$ by
\[
\omega_f(x) = f^{-}(x) - f_{-}(x) \qquad (x \in X).
\]
Then $\omega_f$ is upper semi-continuous, and $f$ is continuous at $x$ if and only if $\omega_f(x) = 0$.
