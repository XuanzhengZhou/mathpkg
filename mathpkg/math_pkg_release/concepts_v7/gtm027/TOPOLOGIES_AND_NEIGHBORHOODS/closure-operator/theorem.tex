The function which assigns to each subset $A$ of a topological space the value $A^{-}$ is called the \textbf{closure function}, or \textbf{closure operator}, relative to the topology. This operator determines the topology completely, for a set $A$ is closed iff $A = A^{-}$. In other words, the closed sets are simply the sets which are fixed under the closure operator.

The closure of the union of two sets is the union of the closures, for $(A \cup B)^{-}$ is always a closed set containing $A$ and $B$, and therefore contains $A^{-}$ and $B^{-}$ and hence $A^{-} \cup B^{-}$; on the other hand, $A^{-} \cup B^{-}$ is a closed set containing $A \cup B$ and hence also $(A \cup B)^{-}$.
