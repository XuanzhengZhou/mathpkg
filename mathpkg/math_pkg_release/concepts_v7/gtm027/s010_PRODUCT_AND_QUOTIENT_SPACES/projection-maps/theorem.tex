For the Cartesian product $\bigwedge \{X_a : a \in A\}$, which is defined as the set of all functions $x$ on $A$ such that $x_a \in X_a$ for each $a$ in $A$, the set $X_a$ is the $a$-th \textbf{coordinate set}. The \textbf{projection} $P_a$ of the product into the $a$-th coordinate set is defined by
$$P_a(x) = x_a.$$
For a finite product $X \times Y$, the projections are denoted $P_0$ and $P_1$, where $P_0(x,y) = x$ and $P_1(x,y) = y$.
