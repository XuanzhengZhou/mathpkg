The Banach-Kuratowski-Pettis theorem states that if a set $ in a topological group contains a non-meager almost open subset, then the product set ^{-1}$ is a neighborhood of the identity. This is a fundamental result in the category theory of topological groups, generalizing the fact that in a non-meager group, the difference of two non-meager sets contains a neighborhood of the identity. It has deep consequences for automatic continuity and open mapping theorems.
