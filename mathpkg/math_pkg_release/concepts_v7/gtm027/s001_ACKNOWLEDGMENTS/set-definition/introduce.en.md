A set is the fundamental object of set theory — a collection of elements. Kelley begins by synonymizing "set", "class", "family", "collection", and "aggregate", then gives the extensionality principle: two sets are identical precisely when they contain the same elements. This axiomatic definition underpins all subsequent mathematical constructions in the book.
