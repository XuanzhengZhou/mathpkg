A \textbf{linear ordering} (total, complete, or simple ordering) is an ordering $<$ on a set $X$ such that:
\begin{enumerate}
\item[(a)] If $x < y$ and $y < x$, then $x = y$ (antisymmetry);
\item[(b)] If $x$ and $y$ are distinct members of $X$, then either $x < y$ or $y < x$ (totality).
\end{enumerate}
