# Proof of Function Value Outside Domain

PROOF If $x \notin \text{domain } f$, then $\{y: (x,y) \in f\} = 0$, and $f(x) = \mathfrak{u}$ (theorem 24). If $x \in \text{domain } f$, then $\{y: (x,y) \in f\} \neq 0$ and (theorem 35) $f(x)$ is a set.

The foregoing theorem does not require that $f$ be a function.
