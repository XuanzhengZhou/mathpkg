In Kelley's system, ordinals are defined as full classes (transitive sets) connected by the membership relation. This is the von Neumann definition: an ordinal is the set of all smaller ordinals. The key theorems are that membership well-orders every ordinal, ordinals are comparable, and every member of an ordinal is an ordinal. The class $R$ of all ordinal numbers that are sets is itself well-ordered by membership but is not a set (Burali-Forti paradox avoided).
