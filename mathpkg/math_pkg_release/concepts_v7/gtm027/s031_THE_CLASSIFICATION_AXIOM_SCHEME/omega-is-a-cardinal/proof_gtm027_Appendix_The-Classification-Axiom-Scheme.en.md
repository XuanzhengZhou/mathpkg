# Proof of Omega is a Cardinal

Proof If $\omega \approx x$ and $x \in \omega$, then $x \subset x + 1 \subset \omega$, and hence $P(x + 1) = P(x)$. This contradicts the preceding theorem, which states that each integer is a cardinal number.
