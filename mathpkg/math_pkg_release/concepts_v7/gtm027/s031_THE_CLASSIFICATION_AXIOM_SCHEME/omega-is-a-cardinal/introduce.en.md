The set $\omega$ of all integers is itself a cardinal number (the first infinite cardinal, $\aleph_0$). If $\omega$ were equinumerous with a smaller ordinal $x \in \omega$, then $P(x+1) = P(x)$, contradicting the fact that each integer is a cardinal.
