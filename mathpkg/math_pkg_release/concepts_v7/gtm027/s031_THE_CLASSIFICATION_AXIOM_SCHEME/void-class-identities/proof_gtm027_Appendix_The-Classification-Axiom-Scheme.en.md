# Proof of Void Class Identities

**Theorem 17.** $0 \cup x = x$ and $0 \cap x = 0$.

*Proof.* For the first identity: $z \in 0 \cup x$ iff $z \in 0$ or $z \in x$ (by Theorem 4). Since $z \notin 0$ for all $z$ (Theorem 16), this reduces to $z \in x$. Hence $0 \cup x = x$ by the axiom of extent. For the second: $z \in 0 \cap x$ iff $z \in 0$ and $z \in x$. Since $z \notin 0$, the conjunction is always false, so $0 \cap x = 0$.
