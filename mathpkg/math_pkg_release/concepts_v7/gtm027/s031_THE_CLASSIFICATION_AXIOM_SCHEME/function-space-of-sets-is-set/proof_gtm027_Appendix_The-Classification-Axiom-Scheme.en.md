# Proof of Function Space of Sets is a Set

PROOF If $f \in y^x$, then $f \subset x \times y$, which is a set, and hence $f \in 2^{x \times y}$ (theorem 38) and $2^{x \times y}$ is a set. Since $y^x \subset 2^{x \times y}$ it follows from the axiom of subsets that $y^x$ is a set.
