# Proof of Well-Ordering of the Ordinal Plane (R x R)

177 Theorem $\ll \text{well-orders } R \times R$.The proof is a straightforward but tedious application of thefunction $f$ which is $\ll -E$ order preserving in $x \times x$ and $R$, such that either domain $f = x \times x$ or range $f = R$. Since $x \times x$ is a set and $R$ is not, domain $f = x \times x$. We show that, if $(u,v) \in x \times x$, then $f((u,v)) < x$, and the theorem follows. By the preceding theorem the class of all predecessors of $(u,v)$ is a subset of $(\max [u,v] + 1) \times (\max [u,v] + 1)$. If $x = \omega$, then both $u$ and $v$ are finite because $\max [u,v] < x$; by 170, $(\max [u,v] + 1) \times (\max [u,v] + 1)$ is finite, hence $f((u,v))$ has only a finite number of predecessors and $f((u,v)) < x$. If $x \neq \omega$, and $\max [u,v]$ is not finite, then by 174, $P(\max [u,v] + 1) = P(\max [u,v]) < x$ and hence $P(f((u,v))) < x$ and $f((u,v)) < x$.
