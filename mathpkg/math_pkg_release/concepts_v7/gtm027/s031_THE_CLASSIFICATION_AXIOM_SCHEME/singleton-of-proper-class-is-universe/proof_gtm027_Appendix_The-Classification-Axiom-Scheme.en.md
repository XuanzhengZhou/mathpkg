# Proof of Singleton of a Proper Class equals Universe

PROOF If $x$ is a set, then $\{x\}$ is a set and consequently is not equal to $\mathfrak{u}$. If $x$ is not a set, then $x \notin \mathfrak{u}$ and $\{x\} = \mathfrak{u}$ by the definition.
