# Proof of Epsilon Well-Orders Cardinals

150 THEOREM $E$ well-orders $C$.151 DEFINITION $P = \{(x,y) : x \approx y$ and $y \in C\}$.The class $P$ consists of all pairs $(x,y)$ such that $x$ is a set and $y$ is a cardinal number equivalent to $x$. For each set $x$ the cardinal number $P(x)$ is the power of $x$ or the cardinal of $x$.The basic facts needed for the following sequence of results have already been demonstrated.
