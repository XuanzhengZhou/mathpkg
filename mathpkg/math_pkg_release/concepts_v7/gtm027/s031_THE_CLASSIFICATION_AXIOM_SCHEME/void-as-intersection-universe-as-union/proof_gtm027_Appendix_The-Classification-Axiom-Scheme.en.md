# Proof of Void as Intersection and Universe as Union

Proof If $x \in \bigcap u$, then $x$ is a set and since $0 \subset x$ it follows from 33 that 0 is a set. Then $0 \in u$ and each member of $\bigcap u$ belongs to 0. It follows that $\bigcap u$ has no members. Clearly (that is, theorem 26) $\bigcup u \subset u$. If $x \in u$, then $x$ is a set and by the axiom of subsets there is a set $y$ such that, if $x \subset x$, then $x \in y$. In particular $x \in y$, and since $y \in u$ it follows that $x \in \bigcup u$. Consequently $u \subset \bigcup u$ and equality follows.
