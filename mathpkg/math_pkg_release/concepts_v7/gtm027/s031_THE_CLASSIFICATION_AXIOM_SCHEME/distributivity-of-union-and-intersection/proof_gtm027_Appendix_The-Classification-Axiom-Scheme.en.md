# Proof of Distributive Laws for Union and Intersection

8 Theorem $x \cap (y \cup z) = (x \cap y) \cup (x \cap z)$ and $x \cup (y \cap z) = (x \cup y) \cap (x \cup z)$.9 Definition $x \notin y$ if and only if it is false that $x \in y$.10 Definition $\sim x = \{y : y \notin x\}$.The class $\sim x$ is the complement of $x$.
