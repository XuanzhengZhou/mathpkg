In Kelley's system, a set is finite if its cardinal number is an integer (a member of $\omega$). An elegant characterization: a set is finite precisely when it admits a relation that well-orders it in both directions. With the Axiom of Choice, this is equivalent to the Dedekind definition (not equipotent to a proper subset). Key closure properties: finite unions of finite sets, finite products, and power sets of finite sets are all finite.
