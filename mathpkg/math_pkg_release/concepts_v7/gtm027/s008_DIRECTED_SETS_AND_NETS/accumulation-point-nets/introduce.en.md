A point is an accumulation point of a set precisely when there exists a net in the set, excluding the point itself, that converges to it. This characterization bridges the classical definition of accumulation points (every neighborhood intersects the set in a point different from the given one) with net convergence. The forward direction constructs a net by selecting one point from each neighborhood of the accumulation point, indexed by the neighborhood system ordered by inclusion.
