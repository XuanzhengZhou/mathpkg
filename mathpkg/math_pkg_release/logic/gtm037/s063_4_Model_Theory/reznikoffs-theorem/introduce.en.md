Reznikoff's Theorem asserts that every first-order theory admits an independent axiomatization -- that is, a set of axioms none of which is a logical consequence of the others. This result is one of the deepest in Chapter 22, building on a careful cardinality-based construction. The proof stratifies sentences by the number of nonlogical constants they contain and inductively builds an independent set $\Delta_\omega$ that axiomatizes the original theory.
