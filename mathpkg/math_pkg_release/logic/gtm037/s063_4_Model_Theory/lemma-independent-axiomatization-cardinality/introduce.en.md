This lemma shows that any countable set of sentences $\Gamma$ has an independent axiomatization. Given an enumeration $\{\varphi_i : i \in \omega\}$ of $\Gamma$, one constructs sentences $\psi_i = \bigwedge_{j < i} \varphi_j \rightarrow \varphi_i$ and takes those $\psi_i$ that are not logically valid. The resulting set $\Delta$ both axiomatizes $\Gamma$ and is independent, as each $\psi_i$ is not implied by the others. This result is a key step toward Reznikoff's theorem.
