Let $\Gamma = \{\varphi_i : i \in \omega\}$ be a countable set of sentences. For each $i \in \omega$, let $\psi_i$ be the sentence $\bigwedge_{j < i} \varphi_j \rightarrow \varphi_i$, where by convention $\psi_0$ is $\varphi_0$. Let $\Delta = \{\psi_i : \text{not}(\models \psi_i)\}$. Then $\Delta$ independently axiomatizes $\Gamma$.
