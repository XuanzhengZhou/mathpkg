Let $I$ be an infinite set and let $J = \{X \subseteq I : X \text{ is finite}\}$. For each $X \in J$ define $G_X = \{Y \in J : X \subseteq Y\}$. Then there exists an ultrafilter $F$ over $J$ such that $G_X \in F$ for every $X \in J$. Moreover, $|E| = |J|$ where $E = \{G_X : X \in J\}$, and for every infinite $H \subseteq E$, we have $\bigcap H = \varnothing$.
