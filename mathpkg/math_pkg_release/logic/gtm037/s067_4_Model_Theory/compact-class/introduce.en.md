A class $K$ of structures is compact if it satisfies the compactness property relative to $K$: whenever every finite subset of a set $\Gamma$ of sentences has a model belonging to $K$, the entire set $\Gamma$ has a model in $K$. This generalizes the ordinary compactness theorem for first-order logic to subclasses of all structures.
