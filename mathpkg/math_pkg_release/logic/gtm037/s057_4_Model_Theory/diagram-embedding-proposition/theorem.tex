\text{Proposition 19.10.}\quad
\text{Let } \mathfrak{A} \text{ be an } \mathcal{L}\text{-structure, } \mathcal{L}' \text{ an } A\text{-expansion of } \mathcal{L}, \text{ and } (\mathfrak{B}, l_a)_{a \in A} \text{ a model of the } \mathcal{L}'\text{-diagram of } \mathfrak{A}.
\text{Then } l \text{ is an embedding of } \mathfrak{A} \text{ into } \mathfrak{B}.
