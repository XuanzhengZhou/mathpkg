A discrete ordering is a linearly ordered set in which every element (except possibly the first and last) has both an immediate predecessor and an immediate successor. The standard example is the integers $\mathbb{Z}$ with the usual order. In Enderton's text, this definition is introduced as an application of Henkin's Embedding Theorem: since every finite simple ordering is discrete, the theorem immediately yields that any simple ordering can be embedded into a discrete ordering.
