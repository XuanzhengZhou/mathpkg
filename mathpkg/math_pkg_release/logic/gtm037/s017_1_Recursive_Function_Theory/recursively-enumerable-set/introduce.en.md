A set $A \subseteq \omega$ is called recursively enumerable (r.e.) if it is either empty or the range of a recursive function. This formalizes the intuitive notion of an effectively enumerable set — a set whose members can be listed out one after another by an automatic method. The definition admits several equivalent formulations involving elementary functions, primitive recursive functions, partial recursive functions, and the arithmetical hierarchy class $\Sigma_1$.
