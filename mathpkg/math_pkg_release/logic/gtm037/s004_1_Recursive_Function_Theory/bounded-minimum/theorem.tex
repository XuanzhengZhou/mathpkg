Let $R$ be an $m$-ary relation. For all $x_0, \ldots, x_{m-1} \in \omega$, let
$$f(x_0, \ldots, x_{m-1}) = \begin{cases}
\text{the least } y < x_{m-1} \text{ such that } \langle x_0, \ldots, x_{m-2}, y \rangle \in R, \\
\qquad \text{if there is such a } y, \\
0 \quad \text{otherwise.}
\end{cases}$$
The function $f(x_0, \ldots, x_{m-1})$ is denoted by $\mu y < x_{m-1} \, R(x_0, \ldots, x_{m-2}, y)$.
