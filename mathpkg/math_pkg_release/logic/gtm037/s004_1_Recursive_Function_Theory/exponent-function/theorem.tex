If $a = 0$ or $a = 1$, let $(a)_i = 0$ for all $i \in \omega$. If $a > 1$, let $(a)_i$ be the exponent of $p_i$ in the prime decomposition of $a$. That is, if $a = p_0^{\alpha_0} p_1^{\alpha_1} \cdots p_k^{\alpha_k}$ with $\alpha_k > 0$, then $(a)_i = \alpha_i$ for $i \leq k$ and $(a)_i = 0$ for $i > k$.
