The exponent function $(a)_i$ is elementary recursive. This is proved by expressing $(a)_i$ using the bounded minimum operator: $(a)_i$ is the largest $x < a$ such that $p_i^x$ divides $a$ but $p_i^{x+1}$ does not. Since the divisibility relation, the prime function $p_i$, and the bounded minimum operator are all elementary, the exponent function is elementary.
