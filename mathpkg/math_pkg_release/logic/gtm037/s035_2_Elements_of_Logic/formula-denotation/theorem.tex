For any formula $\varphi$ of $\mathcal{L}$ and any $\mathcal{L}$-structure $\mathfrak{A}$, define $\varphi^{\mathfrak{A}} \subseteq {}^\omega A$ as follows: for any $x \in {}^\omega A$,

$$x \in (\sigma = \tau)^{\mathfrak{A}} \quad\text{iff}\quad \sigma^{\mathfrak{A}}[x] = \tau^{\mathfrak{A}}[x];$$

$$x \in (\mathbf{R}\sigma_0 \cdots \sigma_{m-1})^{\mathfrak{A}} \quad\text{iff}\quad (\sigma_0^{\mathfrak{A}}[x], \ldots, \sigma_{m-1}^{\mathfrak{A}}[x]) \in \mathbf{R}^{\mathfrak{A}},$$
where $\rho\mathbf{R} = m$;

$$(\neg\varphi)^{\mathfrak{A}} = {}^\omega A \setminus \varphi^{\mathfrak{A}};$$

$$(\varphi \lor \psi)^{\mathfrak{A}} = \varphi^{\mathfrak{A}} \cup \psi^{\mathfrak{A}};$$

$$(\varphi \land \psi)^{\mathfrak{A}} = \varphi^{\mathfrak{A}} \cap \psi^{\mathfrak{A}};$$

$$x \in (\forall v_i \varphi)^{\mathfrak{A}} \quad\text{iff}\quad \text{for all } a \in A,\; x_a^i \in \varphi^{\mathfrak{A}}.$$

We say that $\varphi$ elementarily defines $\varphi^{\mathfrak{A}}$ in $\mathfrak{A}$. If $x \in \varphi^{\mathfrak{A}}$, we say that $x$ satisfies $\varphi$ in $\mathfrak{A}$, and write $\mathfrak{A} \models \varphi[x]$.
