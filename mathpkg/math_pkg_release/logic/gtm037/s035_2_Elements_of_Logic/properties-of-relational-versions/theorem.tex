Let $\mathcal{L}'$ be a relational version of $\mathcal{L}$ with translation $T$, notation as in Definition 11.26(i).

\begin{enumerate}
\item[(i)] If $\varphi$ is a formula of $\mathcal{L}$ in which no operation symbol occurs, then $\varphi' = \varphi$;
\item[(ii)] Let $\mathfrak{A}$ be an $\mathcal{L}$-structure and $\mathfrak{A}'$ the relational version of $\mathfrak{A}$. Then for any $\mathcal{L}$-formula $\varphi$ and any assignment $x$, $\mathfrak{A} \models \varphi[x]$ iff $\mathfrak{A}' \models \varphi'[x]$.
\end{enumerate}
