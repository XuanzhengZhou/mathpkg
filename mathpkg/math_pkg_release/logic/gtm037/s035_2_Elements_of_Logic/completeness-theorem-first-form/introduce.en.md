Gödel's completeness theorem, in its first form, asserts that every consistent set of first-order sentences possesses a model. The proof combines three key constructions: (1) expand the language with sufficiently many new Henkin constants via the Rich Extension Lemma; (2) extend the consistent set to a complete consistent set (a Lindenbaum-type maximal extension); (3) apply the Henkin Model Existence Theorem to obtain a term model whose universe consists of equivalence classes of closed terms. The cardinality bound $|\operatorname{Fmla}_{\mathcal{L}}|$ follows from the construction. Taking the reduct to the original language yields a model of the original consistent set. This theorem is the cornerstone of first-order model theory.
