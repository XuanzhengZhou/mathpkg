A set of sentences is **rich** if every existential sentence can be witnessed by an individual constant already present in the language: from $\exists\alpha\varphi$ the set proves $\operatorname{Subf}_c^\alpha\varphi$ for some constant $c$. Since $\vdash \operatorname{Subf}_c^\alpha\varphi \rightarrow \exists\alpha\varphi$ holds, richness essentially guarantees that existential quantifiers can be eliminated via constants. A set is **complete** if it proves either $\varphi$ or $\neg\varphi$ for every sentence $\varphi$. Together, rich and complete consistent sets are the key to Henkin's model-existence construction: they provide enough witnesses and decide every sentence, enabling the term model construction.
