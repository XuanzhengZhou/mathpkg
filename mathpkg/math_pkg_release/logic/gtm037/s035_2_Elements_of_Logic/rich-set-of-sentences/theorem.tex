A set $\Gamma$ of sentences of $\mathcal{L}$ is called **rich** if for every sentence of $\mathcal{L}$ of the form $\exists\alpha\varphi$ there is an individual constant $c$ of $\mathcal{L}$ such that

$$\Gamma \vdash \exists\alpha\varphi \rightarrow \operatorname{Subf}_c^\alpha\varphi.$$

A set $\Gamma$ of sentences of $\mathcal{L}$ is called **complete** if for any sentence $\varphi$ of $\mathcal{L}$,

$$\Gamma \vdash \varphi \quad\text{or}\quad \Gamma \vdash \neg\varphi.$$
