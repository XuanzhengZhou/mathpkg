The tree property is one of the key equivalent formulations of weak compactness for strongly inaccessible cardinals. A tree of height $m$ with all levels of size less than $m$ must have a branch of length $m$; the failure of this property at $\aleph_1$ is witnessed by Aronszajn trees. The tree property at uncountable regular cardinals is a large cardinal notion intimately connected with both combinatorial set theory and infinitary model theory.
