Machine $T_1$ is a simple two-state Turing machine. Its purpose is to write a $1$ on the current tape cell if it is $0$, and to halt without modification if it is already $1$. In both cases, the tape head does not move. This machine serves as a basic building block for more complex Turing machine constructions in recursive function theory.
