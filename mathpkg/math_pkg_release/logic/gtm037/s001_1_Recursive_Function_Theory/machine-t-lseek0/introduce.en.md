Machine $T_{l\text{seek}0}$ is a two-state Turing machine that searches leftward on the tape for the first occurrence of the symbol $0$. It moves left across any $1$'s it encounters and halts when it reaches a $0$ cell. This machine is a fundamental building block in the construction of more complex tape-searching procedures in recursive function theory.
