For each operation symbol $\mathbf{O}$ of $\mathcal{L}_{\text{el}}$, either $\#\mathbf{O} \in \omega$ if $\mathbf{O}$ is $0$-ary, or $\#\mathbf{O}$ is an $m$-ary operation on $\omega$ if $\mathbf{O}$ is $m$-ary. In either case, for any $x_0, \ldots, x_{m-1}, y \in \omega$ we have: $\#\mathbf{O} x_0 \cdots x_{m-1} = y$ implies

$$\mathcal{P} \models \mathbf{O}(\Delta x_0, \ldots, \Delta x_{m-1}) = \Delta y.$$
