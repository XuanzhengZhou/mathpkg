A set $\Gamma$ of sentences is $(m,n)$-compact if the existence of models for all subsets of size $< m$ implies the existence of a model for the whole set $\Gamma$. The notion $(m,\infty)$-compactness means this holds for arbitrarily large sets. Ordinary first-order logic is $(\omega,\infty)$-compact, which is the usual compactness theorem.
