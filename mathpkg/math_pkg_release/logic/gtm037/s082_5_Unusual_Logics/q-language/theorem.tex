Given a first-order language $\mathcal{L}$, we can convert $\mathcal{L}$ into a $\mathcal{Q}$-language by adjoining a new symbol $\mathbf{Q}$ which is used syntactically just like the universal quantifier $\forall$. Thus to the rules of formulas is added the stipulation that $\mathbf{Q}v_i\varphi$ is a formula whenever $\varphi$ is.
