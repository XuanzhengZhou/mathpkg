For any $m \geq \aleph_0$ let

$$\pi m = \text{the least } n \text{ such that there is a family } \langle p_\alpha : \alpha < n \rangle \text{ of cardinals } < m \text{ such that } \prod_{\alpha < n} p_\alpha \geq m.$$

Thus $\aleph_0 \leq \pi m \leq m$ for any cardinal $m$. We have $\pi \aleph_0 = \aleph_0$; $\pi \aleph_1 = \aleph_0$; $\pi m^+ \leq m$; $\pi m = m$ if $m$ is strongly inaccessible; $\pi m \leq \operatorname{cf} m$ if $m$ is singular; and $\pi(\exp \aleph_0)^+ \geq \aleph_1$.
