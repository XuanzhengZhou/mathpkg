Let $\mathcal{L}$ be any first-order language. An $\omega$-expansion of $\mathcal{L}$ is an expansion $\mathcal{L}'$ in which a new unary relation symbol $\mathbf{N}$ and new individual constants $\mathbf{c}_0, \mathbf{c}_1, \ldots$ are introduced. An $\mathcal{L}'$-structure is an $\omega$-structure if $\mathbf{N}^{\mathfrak{A}} = \{\mathbf{c}_i^{\mathfrak{A}} : i < \omega\}$ and the function $i \mapsto \mathbf{c}_i^{\mathfrak{A}}$ is one-one.
