The theory $\Gamma_1$ of the structure $(\omega, s, 0)$ in the language with a unary operation symbol $s$ and an individual constant $\mathbf{O}$ is decidable. Moreover, for any formula $\varphi$ there is a quantifier-free combination of basic formulas equivalent to $\varphi$ under $\Gamma_1$, where a formula is \emph{basic} if it has one of the following forms:
$$s^m v_i = \sigma, \quad \sigma \text{ a term not involving } v_i;$$
$$\mathbf{O} = \mathbf{O}.$$
