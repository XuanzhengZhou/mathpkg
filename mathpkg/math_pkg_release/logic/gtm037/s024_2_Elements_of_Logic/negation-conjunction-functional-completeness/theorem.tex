Let $\mathcal{P} = \langle \{n, k\}, P, f, g \rangle$ be a general sentential logic with $fn = 1$, $fk = 2$, and $g$ given by:

$$g_n \langle 0 \rangle = 1, \quad g_n \langle 1 \rangle = 0;$$
$$g_k \langle 1, 1 \rangle = 1, \quad g_k \langle 1, 0 \rangle = g_k \langle 0, 1 \rangle = g_k \langle 0, 0 \rangle = 0.$$

If $P$ is finite, then $\mathcal{P}$ is functionally complete.
