A **general sentential logic** generalizes the standard sentential logic by allowing an arbitrary set $K$ of connectives, each with an arbitrary finite arity $f_k \geq 1$ and a truth function $g_k$ of that arity. The set $P$ serves as the set of sentential symbols (propositional variables). Sentences are built inductively from sentential symbols by applying the connectives. The recursive truth evaluation $h^+$ extends any assignment $h: P \to \{0,1\}$ to all sentences using the truth functions $g_k$. This framework subsumes the usual systems with negation, conjunction, disjunction, and implication as special cases.
