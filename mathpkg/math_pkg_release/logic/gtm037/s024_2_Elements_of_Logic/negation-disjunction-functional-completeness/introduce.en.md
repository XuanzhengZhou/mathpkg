**Theorem 8.44** is the dual counterpart of Theorem 8.43. It states that the set of connectives consisting of negation and disjunction is also functionally complete for any finite set $P$ of sentential symbols. This follows directly by duality from Theorem 8.43: since conjunction is dual to disjunction, and the set $\{\neg, \wedge\}$ is functionally complete, the set $\{\neg, \vee\}$ is also functionally complete. In practice, this means every truth function can be expressed using only negation and disjunction.
