$T_{\text{right}}$ is a simple two-state Turing machine that serves as an elementary example of Turing machine operation. Its sole function is to move the tape one square to the right and then halt, regardless of what symbol is read from the current square. This machine illustrates the basic mechanics of state transitions and tape movement without performing any computation on the tape contents. It is one of several auxiliary machines introduced early in the chapter to build intuition for constructing more complex Turing machines.
