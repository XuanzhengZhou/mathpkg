In the pure theory of equality $\Gamma_3$ (where the language has no nonlogical constants), the only subsets of a nonempty set $A$ that can be defined by a first-order formula (without parameters) are the trivial ones: the empty set $\emptyset$ and the whole set $A$ itself. This corollary follows from the decision method established for $\Gamma_3$, which reduces truth of sentences to checking only finitely many finite structures. The result characterizes the expressive poverty of the pure language of equality.
