Let $L$ be a relational first-order language with equality, $\varphi \in \text{Sent}_{L}$, and $\Gamma$ a set of $L$-sentences such that equality does not appear in any sentence of $\Gamma$. Then
$$\Gamma \models_{L} \varphi \quad \text{iff} \quad \Gamma \models_{L} \bigwedge E\{\varphi\} \rightarrow \varphi,$$
where $E\{\varphi\}$ denotes the translation of $\varphi$ into the language $L^{e}$ in which the equality symbol $=$ is replaced by a new binary relation symbol $E$, together with axioms asserting that $E$ is an equivalence relation compatible with all relation symbols.
