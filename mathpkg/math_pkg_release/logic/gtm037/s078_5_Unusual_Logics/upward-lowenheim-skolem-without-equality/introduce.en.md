Surprisingly, in logic without equality, the upward Lowenheim-Skolem theorem holds in a much stronger form: any structure can be elementarily extended to any larger universe, even finite structures can be extended to infinite ones. The proof constructs the extension by using a projection map that collapses new elements to a fixed element of the original structure, defining relations via this projection.
