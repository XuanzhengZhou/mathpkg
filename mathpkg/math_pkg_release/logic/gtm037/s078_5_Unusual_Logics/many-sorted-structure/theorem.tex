A many-sorted $\mathcal{L}$-structure is a triple $\mathfrak{A} = (A, f, \mathbf{R})$ such that:
\begin{itemize}
\item[(i)] $A$ assigns to each $s \in \mathcal{S}$ a nonempty set $A_s$;
\item[(ii)] $f$ assigns to each operation symbol $\mathbf{O}$ of rank $(s_0, \ldots, s_m)$ a function $f_{\mathbf{O}} : A_{s_0} \times \cdots \times A_{s_{m-1}} \to A_{s_m}$;
\item[(iii)] $\mathbf{R}$ assigns to each relation symbol $\mathbf{R}$ of rank $(s_0, \ldots, s_m)$ a relation $R_{\mathbf{R}} \subseteq A_{s_0} \times \cdots \times A_{s_m}$.
\end{itemize}
