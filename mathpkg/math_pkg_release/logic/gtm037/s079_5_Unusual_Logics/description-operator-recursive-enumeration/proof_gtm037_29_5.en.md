This follows from Theorem 29.20 (elimination of the description operator) together with the recursive enumerability of the set of valid first-order sentences. Given any sentence $\varphi$ involving $\tau$, we can effectively find an equivalent sentence $\varphi^*$ without $\tau$, and then test validity in ordinary first-order logic.
