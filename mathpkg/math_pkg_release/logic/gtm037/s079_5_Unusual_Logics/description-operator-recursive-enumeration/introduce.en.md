Theorem 29.21 states that the set of logically valid sentences in a language with the description operator $\tau$ is recursively enumerable. This follows from the elimination theorem (29.20) and the recursive enumerability of validities in ordinary first-order logic, and justifies the possibility of a complete axiomatization for descriptive logic.
