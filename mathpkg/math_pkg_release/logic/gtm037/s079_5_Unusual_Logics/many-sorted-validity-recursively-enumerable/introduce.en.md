This corollary follows from Corollary 29.29 and the fact that the set of logical consequences of a recursively enumerable set of axioms (here, the sort axioms $\Gamma$) is recursively enumerable in ordinary first-order logic. Since many-sorted validity reduces to $\Gamma \models \varphi^*$, and the set of ordinary validities is r.e., the many-sorted validities form an r.e. set as well. This result justifies the development of complete axiomatic proof systems for many-sorted logic.
