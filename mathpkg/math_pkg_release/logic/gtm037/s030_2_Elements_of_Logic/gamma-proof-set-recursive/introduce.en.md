This proposition establishes that if the set of hypotheses $\Gamma$ has a recursive set of Godel numbers, then the set of Godel numbers of $\Gamma$-formal proofs is also recursive. This means there is an effective procedure to check whether a given sequence of formulas constitutes a valid formal proof from the hypotheses $\Gamma$. The verification reduces to checking, for each formula in the sequence, that it is either a logical axiom, an element of $\Gamma$, or follows from earlier formulas via modus ponens or generalization---all of which are decidable conditions.
