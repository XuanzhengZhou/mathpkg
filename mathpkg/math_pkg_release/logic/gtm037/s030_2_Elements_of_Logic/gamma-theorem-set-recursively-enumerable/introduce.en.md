This theorem is described as "one of the most important results of elementary logic." It states that if a set of axioms $\Gamma$ is recursively enumerable (i.e., can be effectively listed by an algorithm), then the set of all theorems provable from $\Gamma$ is also recursively enumerable. This is the formal counterpart of the intuitive idea that one can mechanically generate all mathematical theorems from a given set of axioms by systematically enumerating all formal proofs and collecting their conclusions. It provides the positive half of the picture concerning the relationship between axiomatic theories and computability, standing in contrast to Godel's incompleteness theorems.
