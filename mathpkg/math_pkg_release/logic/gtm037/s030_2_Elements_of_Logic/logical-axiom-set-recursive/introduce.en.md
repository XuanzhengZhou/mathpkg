This proposition states that the set of Godel numbers of logical axioms is recursive---there exists an effective procedure to decide, given a Godel number, whether the corresponding expression is a logical axiom. Since the logical axioms consist of a finite number of axiom schemas (propositional axioms, quantifier axioms, and equality axioms), checking whether a given formula instantiates one of these schemas is a finite combinatorial verification. This is a crucial step toward establishing that the notion of a formal proof is effectively decidable.
