This proposition establishes the principle of definition by recursion on the set of first-order formulas. Just as the recursion theorem for terms (Proposition 10.13) allows functions to be defined inductively on terms, this result provides the analogous principle for formulas. Given base values on atomic formulas and combining rules for negation, disjunction, conjunction, and universal quantification, there exists a unique function $s$ defined on all formulas satisfying the recursive clauses. This principle is fundamental for defining semantic notions such as truth assignments and satisfaction.
