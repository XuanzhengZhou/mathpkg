For each $m \in \omega \sim 1$ there exist a 1-place elementary function $f$ and an $(m + 2)$-place elementary function $g$ such that for any $m$-ary partial recursive function $h$ there is an $e \in \omega$ such that for all $x_0, \ldots, x_{m-1} \in \omega$,

$$h(x_0, \ldots, x_{m-1}) \simeq f \, \mu u \, [g(e, x_0, \ldots, x_{m-1}, u) = 0].$$
