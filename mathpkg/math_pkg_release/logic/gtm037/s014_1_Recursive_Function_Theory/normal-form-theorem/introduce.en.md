The Normal Form Theorem, due to Kleene, states that every $m$-ary partial recursive function can be represented as $\varphi_e^m$ for some index $e \in \omega$. This result shows that there is a uniform enumeration of all partial recursive functions, and together with the T-predicate, every partial recursive function can be written in a fixed normal form involving a single application of the $\mu$-operator to a primitive recursive predicate.
