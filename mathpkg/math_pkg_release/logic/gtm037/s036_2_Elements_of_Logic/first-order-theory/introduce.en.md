A theory in first-order logic is a deductively closed set of sentences over a given language. That is, a pair $(\Gamma, \mathcal{L})$ forms a theory precisely when $\Gamma$ contains every $\mathcal{L}$-sentence that is a logical consequence of $\Gamma$.

Every $\mathcal{L}$-structure $\mathfrak{A}$ gives rise to its $\mathcal{L}$-theory, which is the set of all $\mathcal{L}$-sentences true in $\mathfrak{A}$. This provides a natural bridge between semantic structures and syntactic theories.

The definition further introduces key relations between theories: extension, conservative extension, and definitional extension. A conservative extension adds no new theorems in the original language, while a definitional extension allows the introduction of new symbols that are definable in terms of the old ones.
