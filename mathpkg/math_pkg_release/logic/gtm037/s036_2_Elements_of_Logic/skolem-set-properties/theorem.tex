Let $L'$ be a Skolem expansion of $L$, and for each formula $\varphi$ let $\varphi^S$ be its Skolem form. Then:

\begin{enumerate}
\item[(i)] $\models \varphi^S \rightarrow \varphi$;
\item[(ii)] if $\mathfrak{A}$ is a model of the Skolem set of $L'$ over $\mathcal{L}$, then $\mathfrak{A} \models \varphi \rightarrow \varphi^S$;
\item[(iii)] if $\Gamma$ is a theory in $\mathcal{L}$ and $\Gamma' = \{\varphi^S : \varphi \in \Gamma\}\text{-Thm}_{\mathcal{L}'}$, then the following conditions hold:
\begin{enumerate}
\item[(a)] if $\mathfrak{A}$ is a model of $\Gamma$ and $\mathfrak{A}'$ is an expansion of $\mathfrak{A}$ to a model of the Skolem set of $L'$ over $\mathcal{L}$, then $\mathfrak{A}'$ is a model of $\Gamma'$;
\item[(b)] if $\mathfrak{A}'$ is a model of $\Gamma'$, then $\mathfrak{A}' \upharpoonright \mathcal{L}$ is a model of $\Gamma$;
\item[(c)] $(\Gamma', \mathcal{L}')$ is a conservative extension of $(\Gamma, \mathcal{L})$;
\end{enumerate}
\item[(iv)] $\varphi$ has a model iff $\varphi^S$ has a model.
\end{enumerate}
