The following functions are elementary:
\begin{itemize}
\item[(i)] $C_m^n$ (for $n \neq 0$), the $n$-ary constant function with value $m$;
\item[(ii)] $\sigma$, the successor function: $\sigma x = x + 1$;
\item[(iii)] $\text{sg}$, the sign function;
\item[(iv)] $\overline{\text{sg}}$, the cosign function;
\item[(v)] exponentiation: $(x,y) \mapsto x^y$;
\item[(vi)] factorial: $x \mapsto x!$;
\item[(vii)] $\mu$, the predecessor function.
\end{itemize}
