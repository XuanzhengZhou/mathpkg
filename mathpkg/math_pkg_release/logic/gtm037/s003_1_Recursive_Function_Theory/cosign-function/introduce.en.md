The cosign function $\overline{\text{sg}}$ (read "cosign") is a unary number-theoretic function that acts as the indicator of zero: it returns $1$ if the argument is $0$, and $0$ otherwise. Together with the sign function $\text{sg}$, it provides a convenient technical tool for constructing piecewise definitions and characteristic functions of sets within the theory of elementary and recursive functions.
