The sign function $\text{sg}$ (read "sign") is a unary number-theoretic function that returns $0$ for input $0$ and $1$ for all positive inputs. Along with its complement $\overline{\text{sg}}$ (the cosign function), it is a basic technical tool for constructing characteristic functions and encoding case distinctions in the elementary and recursive function hierarchy.
