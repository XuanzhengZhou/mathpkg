This theorem, due to Raphael M. Robinson, provides a compact characterization of the primitive recursive functions. Instead of the usual schema of composition and primitive recursion, Robinson showed that primitive recursive functions are precisely those belonging to every class that contains the successor $\sigma$, the excess-over-a-square function $\mathrm{Exc}$, addition $+$, and all projection functions $U_i^n$, and is closed under composition and iteration (where iteration maps $f$ to the function $g(n) = f^n(0)$). This axiomatization eliminates the need for explicit primitive recursion in favor of the simpler iteration operation.
