This lemma shows that the class of recursive functions is at least as large as the class of Turing computable functions. Given a Turing machine $M$ computing an $m$-ary function $f$ with index $e = gM$, the value $f(x_0, \ldots, x_{m-1})$ can be recovered by applying the $V$ function to the result of an unbounded search over the $m$-ary Turing machine computation relation $T_m$. Since $T_m$ is elementary and $V$ is elementary, the function $f$ is expressible via $\mu$-recursion over an elementary relation, hence recursive.
