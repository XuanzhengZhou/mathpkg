\textbf{Proposition 28.2.}
Let $\mathfrak{A}$ be an $\mathcal{L}$-structure and $\mathfrak{m}$ an infinite cardinal. Then the following conditions are equivalent:
\begin{enumerate}
\item[(i)] $\mathfrak{A}$ is $\mathfrak{m}$-saturated (in the sense of Definition 28.1, using only formulas with one free variable).
\item[(ii)] For every $X \subseteq A$ with $|X| < \mathfrak{m}$, if $\mathcal{L}'$ is obtained from $\mathcal{L}$ by adding constants $c_x$ for $x \in X$, and if $\Delta \subseteq \mathrm{Fmla}_{\mathcal{L}'}$ (allowing formulas with arbitrarily many free variables) is such that for each finite $\Delta' \subseteq \Delta$ there is an $a \in {}^\omega A$ with $(\mathfrak{A}, x)_{x \in X} \models \bigwedge \Delta'[a]$, then there exists $a \in {}^\omega A$ such that $(\mathfrak{A}, x)_{x \in X} \models \varphi[a]$ for all $\varphi \in \Delta$.
\end{enumerate}
