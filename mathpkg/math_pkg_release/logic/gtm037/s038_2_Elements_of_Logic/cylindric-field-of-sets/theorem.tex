Let $U$ be a nonempty set and $\alpha$ an ordinal. For each $\kappa < \alpha$ define a one-place operation $C_\kappa$ on subsets of ${}^\alpha U$ by setting, for any $X \subseteq {}^\alpha U$,
$$C_\kappa X = \{x \in {}^\alpha U : x \upharpoonright (\alpha \setminus \{\kappa\}) = y \upharpoonright (\alpha \setminus \{\kappa\}) \text{ for some } y \in X\}.$$
For $\kappa, \lambda < \alpha$ set
$$D_{\kappa \lambda} = \{x \in {}^\alpha U : x_\kappa = x_\lambda\}.$$
Thus $C_\kappa X$ is the generalized cylinder obtained by moving $X$ parallel to the $\kappa$-axis in $\alpha$-space, while $D_{\kappa \lambda}$ is a diagonal hyperplane. An $\alpha$-dimensional cylindric field of sets is a set $A$ of subsets of ${}^\alpha U$ (for some $U$) which is a field of sets in the Boolean sense and which is closed under all operations $C_\kappa$ and contains all sets $D_{\kappa \lambda}$ as elements. A cylindric set algebra of dimension $\alpha$ is a structure
$$\langle A, \cup, \cap, \sim, \varnothing, {}^\alpha U, C_\kappa, D_{\kappa \lambda} \rangle_{\kappa,\lambda < \alpha}$$
in which $A$ is an $\alpha$-dimensional cylindric field of sets and all of the operations are the natural set-theoretic ones.
