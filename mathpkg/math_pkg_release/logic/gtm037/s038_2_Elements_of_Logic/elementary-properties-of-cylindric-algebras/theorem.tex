In any cylindric algebra $\mathfrak{A}$ of dimension $\alpha$, for all $\kappa, \lambda, \mu < \alpha$ and all $x, y \in A$:

\begin{itemize}
\item[(i)] If $c_\kappa x = 0$, then $x = 0$.
\item[(ii)] $c_\kappa 1 = 1$.
\item[(iii)] $c_\kappa c_\kappa x = c_\kappa x$.
\item[(iv)] $c_\kappa x = x$ iff $x \in \operatorname{Rng} c_\kappa$.
\item[(v)] $c_\kappa x \cdot y = 0$ iff $x \cdot c_\kappa y = 0$.
\item[(vi)] $c_\kappa(x + y) = c_\kappa x + c_\kappa y$.
\item[(vii)] If $x \leq y$, then $c_\kappa x \leq c_\kappa y$.
\item[(viii)] $c_\kappa(-c_\kappa x) = -c_\kappa x$.
\item[(ix)] If $\kappa \neq \lambda$, then $d_{\kappa\lambda} \cdot c_\kappa(d_{\kappa\lambda} \cdot x) = d_{\kappa\lambda} \cdot x$.
\end{itemize}
