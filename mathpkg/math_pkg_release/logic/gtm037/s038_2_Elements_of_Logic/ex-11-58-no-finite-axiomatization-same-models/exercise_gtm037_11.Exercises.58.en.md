Let $\Gamma$ be a theory in a language $L$. Suppose that for every finite subset $\Delta$ of $\Gamma$ there is a model of $\Delta$ which is not a model of $\Gamma$. Then there is no finite set $\Lambda$ of sentences which has exactly the same models as $\Gamma$.
