Exercise 11.55 asks to show that the class of groups in which every element has finite order is not axiomatizable in first-order logic. The standard proof uses the compactness theorem: if such a $\Gamma$ existed, one could construct a model with an element of infinite order by adding a new constant and infinitely many sentences asserting the element has order greater than each natural number.
