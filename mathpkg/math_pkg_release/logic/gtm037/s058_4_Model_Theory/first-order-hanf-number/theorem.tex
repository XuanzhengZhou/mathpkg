For any first-order language $\mathcal{L}$, the relation $R$ defined by

$$(\Gamma, \mathfrak{A}, n) \in R \text{ iff } \mathfrak{A} \text{ is an } \mathcal{L}\text{-structure}, \Gamma \subseteq \mathrm{Fmla}_{\mathcal{L}}, \mathfrak{A} \models \Gamma, \text{ and } |A| = n$$

is a Hanf system. Its Hanf number is $\omega$.
