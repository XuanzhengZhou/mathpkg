Any infinite r.e. set is the disjoint union of a creative set and a productive set.

*Hint:* Say $\operatorname{Rng} f = A$. Let $g(n) = f(\mu i (f(i) \neq g(j) \text{ for all } j < n))$. Show that $g[K]$ is creative and $A \setminus g[K]$ is productive.
