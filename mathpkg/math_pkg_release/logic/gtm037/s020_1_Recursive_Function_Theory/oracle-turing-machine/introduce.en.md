An **oracle Turing machine** (or $g$-Turing machine) extends the standard Turing machine model by equipping it with an oracle — an impenetrable black box that answers queries about a fixed function $g : \omega \to \omega$. When the machine enters a special query state with input $x$, the oracle responds by writing $g(x)$ on the tape. The function $g$ may be nonrecursive, so the oracle is not an effective device. This notion forms the foundation of relative computability theory and the study of Turing degrees.
