A function $h$ is \textbf{$g$-recursive} (or \textbf{recursive in $g$}) if it belongs to every class $\mathcal{A}$ satisfying the closure conditions for recursive functions (as in the definition of recursive functions) with the additional requirement that $g \in \mathcal{A}$. More generally, for a set $F$ of functions, a function is \textbf{recursive in $F$} if it belongs to every class $\mathcal{A}$ satisfying the closure conditions with $F \subseteq \mathcal{A}$. The notions of $g$-Turing computable and $g$-recursive are equivalent.
