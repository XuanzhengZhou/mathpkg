Addition of natural numbers is shown to be a primitive recursive function by giving an explicit definition via the primitive recursion schema. The base case $x + 0 = x$ uses the projection function $U_0^1$, and the recursive step $x + \delta y = \delta(x + y)$ uses composition with the successor function $\delta$ and a ternary projection $U_2^3$. This is one of the fundamental examples demonstrating that the class of primitive recursive functions contains ordinary arithmetic.
