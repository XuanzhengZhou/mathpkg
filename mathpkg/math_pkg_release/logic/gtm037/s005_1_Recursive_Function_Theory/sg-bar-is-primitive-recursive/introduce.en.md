The modified sign function $\overline{sg}$ (denoted with an overbar) is defined on natural numbers as $\overline{sg}(x) = 1 \div x$, where $\div$ is truncated subtraction. This function returns 1 when $x = 0$ and 0 for all positive $x$, serving as a useful indicator for zero within primitive recursive constructions. It is primitive recursive since truncated subtraction and the constant function 1 are primitive recursive.
