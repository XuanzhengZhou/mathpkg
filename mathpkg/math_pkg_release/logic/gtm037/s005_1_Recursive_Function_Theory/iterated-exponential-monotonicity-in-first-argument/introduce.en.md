Lemma 2.39 establishes that the iterated exponential $a(m, n)$ is strictly increasing in the first argument $m$ (the base) for any fixed number of iterations $n$, whenever $m \neq 0$. The proof proceeds by induction on $n$, using the elementary inequality $m^k \leq (m+1)^k$ for exponentiation. Together with Lemmas 2.37 and 2.38, this lemma fully characterizes the monotonicity behavior of the iterated exponential function in both arguments.
