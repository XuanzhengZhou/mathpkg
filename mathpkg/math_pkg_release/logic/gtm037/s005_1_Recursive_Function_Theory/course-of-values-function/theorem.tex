Let $f$ be an $m$-ary number-theoretic function. The **course-of-values function** $\tilde{f}$ for $f$ is the $m$-ary function defined, for any $x_0, \ldots, x_{m-1} \in \omega$, by

$$\tilde{f}(x_0, \ldots, x_{m-1}) = \prod \{p_i^{f(x_0, \ldots, x_{m-2}, i)+1} : i < x_{m-1}\}.$$

Here $p_i$ denotes the $i$-th prime number. Note that $\tilde{f}(x_0, \ldots, x_{m-2}, 0) = 1$.
