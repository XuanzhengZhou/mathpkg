Definition 16.9 introduces the first definitional expansion $\mathcal{L}_1$ of the set-theoretic language $\mathcal{L}_{\text{set}}$. It adds four new symbols: the individual constant $0$ representing the empty set, a binary operation symbol $\mathrm{Op}$ representing the operation of adding an element to a set, a unary operation symbol $\{\,\}$ for forming singletons, and a binary operation symbol $\{\,\}$ for forming pairs. Each new symbol comes with a defining axiom expressed in the base language. This is the first step in a series of definitional expansions that ultimately allow Robinson arithmetic $Q$ to be interpreted within the weak set theory $S$.
