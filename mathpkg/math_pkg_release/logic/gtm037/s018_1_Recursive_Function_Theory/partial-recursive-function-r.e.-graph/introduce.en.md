Theorem 6.16 shows that a unary partial function is partial recursive if and only if its graph is a recursively enumerable relation. This is a conceptually clean characterization: it defines partial recursive functions purely in terms of the enumerability of their input-output pairs, without recourse to the detailed machine models or inductive schemata used in the original definition. The result is fundamental to the modern understanding of computable functions.
