Type symbols are the syntactic building blocks of the theory of types, defined by a simple inductive generation: the symbol $0$ is the base type (representing individuals), and if $\tau_1, \ldots, \tau_n$ are already type symbols, then $(\tau_1, \ldots, \tau_n)$ represents the type of an $n$-ary relation between objects of those types. This inductive definition allows the construction of types of arbitrary finite order, with the order of a type measuring how many levels of set/relation formation are involved above the individuals.
