The proof uses the fact that $(\omega, +, \cdot, 0, s)$ is, up to isomorphism, the only model of the weak second-order theory $Q^w$. This is shown by constructing an isomorphism $g : \omega \to A$ for any model $\mathfrak{A}$ of $Q^w$, using the weak second-order induction axiom. The upward Löwenheim–Skolem theorem thus fails. Then the proofs of 15.20 and 15.21 (essentially Tarski's undefinability theorem) can be repeated in this context to obtain the result. Since $Q^w \models \psi$ iff $\models \bigwedge Q^w \rightarrow \psi$, the set of validities is also not recursively enumerable.
