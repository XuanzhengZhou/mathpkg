An effectivized first-order language enriches a standard first-order language with a Gödel numbering $g$ that assigns natural numbers to each symbol, subject to computability conditions. The nonlogical constants of a language $\mathcal{L}$ are precisely the operation symbols and relation symbols. The set $\text{Sym}_{\mathcal{L}}$ collects all symbols: the logical symbols from $\text{Rng } L$, the individual variables from $\text{Rng } v$, and the nonlogical constants. The effectiveness conditions (i)--(iii) ensure that there is an effective method for recognizing expressions in the given language.
