A modal language is a quadruple $\mathcal{P} = (n, c, d, P)$ such that $|\{n, c, d\}| = 3$, $P \neq 0$, and $\{n, c, d\} \cap P = 0$. The symbol $d$ is the necessity symbol. For any expression $\varphi$ we write $\Box \varphi = \langle d \rangle \varphi$.

If $\varphi$ is a sentence, then $\Box \varphi$ expresses that $\varphi$ is not only true, but is true in all possible worlds.
