Let $\mathfrak{A}$ be any Boolean algebra. Let $\mathcal{P} = (n, c, A)$ be a sentential language whose set of propositional variables is $A$ itself. Define $f([\langle a \rangle]) = a$ for each $a \in A$. By Theorem 9.58, $\mathfrak{M}_{\mathcal{P}}$ is freely generated by $\{[\langle a \rangle] : a \in A\}$, so $f$ can be extended to a homomorphism $f^{+}: \mathfrak{M}_{\mathcal{P}} \to \mathfrak{A}$. Since $f^{+}$ maps onto the generating set of $\mathfrak{A}$, it is surjective.

Let $F$ be the filter-kernel of $f^{+}$, i.e., $F = \{x \in M_{\mathcal{P}} : f^{+}(x) = 1\}$. By the homomorphism theorem for Boolean algebras, $\mathfrak{A} \cong \mathfrak{M}_{\mathcal{P}} / F$. Now let $\Gamma = \{\varphi : [\varphi] \in F\}$. Then $\mathfrak{M}_{\Gamma}^{\mathcal{P}} = \mathfrak{M}_{\mathcal{P}} / F$, and consequently $\mathfrak{A} \cong \mathfrak{M}_{\Gamma}^{\mathcal{P}}$.
