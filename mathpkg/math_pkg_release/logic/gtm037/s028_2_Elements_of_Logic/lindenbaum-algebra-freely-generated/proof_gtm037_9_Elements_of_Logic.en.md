First, if $s, t \in P$ and $s \neq t$, then clearly $\nvdash \langle s \rangle \leftrightarrow \langle t \rangle$ (by considering a truth assignment making one true and the other false), hence $[\langle s \rangle] \neq [\langle t \rangle]$. Thus the map $s \mapsto [\langle s \rangle]$ is one-one.

By induction on sentences, $[\varphi]$ is in the subalgebra generated by $X = \{[\langle s \rangle] : s \in P\}$, for each sentence $\varphi$. Thus $X$ generates $\mathfrak{M}_{\mathcal{P}}$.

Now let $f$ be any function mapping $X$ into a Boolean algebra $\mathfrak{A}$. Define $g: \text{Sent}_{\mathcal{P}} \to A$ by recursion:
$$g(\langle s \rangle) = f([\langle s \rangle]),$$
$$g(\neg \varphi) = -g(\varphi),$$
$$g(\varphi \rightarrow \psi) = -g(\varphi) + g(\psi).$$

We claim: if $\vdash \varphi$, then $g(\varphi) = 1$. Let $\Gamma = \{\varphi : g(\varphi) = 1\}$. Each logical axiom belongs to $\Gamma$ (verified by computation in $\mathfrak{A}$), and $\Gamma$ is closed under modus ponens. Hence $\Gamma$ contains all theorems. In particular, if $\varphi \equiv \psi$ (i.e., $\vdash \varphi \leftrightarrow \psi$), then $g(\varphi) = g(\psi)$. Thus $g$ induces a well-defined homomorphism $g^*: \mathfrak{M}_{\mathcal{P}} \to \mathfrak{A}$ extending $f$. This verifies the universal mapping property.
