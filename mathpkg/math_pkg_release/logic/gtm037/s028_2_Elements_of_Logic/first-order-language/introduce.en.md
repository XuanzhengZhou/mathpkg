A first-order language $\mathcal{L}$ is a formal syntactic framework for first-order logic, specified by four components: logical symbols $L$ (negation, disjunction, conjunction, universal quantifier, equality), individual variables $v$ (countably many), operation symbols $\mathcal{O}$ (each with a finite rank), and relation symbols $\mathcal{R}$ (each with a finite rank). The requirement that the ranges and domains of these functions be pairwise disjoint ensures that no symbol plays multiple roles. This definition serves as the foundation for the syntax of first-order logic throughout the rest of the book.
