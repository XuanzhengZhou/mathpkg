This lemma bridges the gap between syntactic definability in a theory and elementary definability in a structure. If a theory $\Gamma$ has the standard model $\mathfrak{A} = (\omega, +, \cdot, s, 0)$ and a function or relation is syntactically definable in $\Gamma$, then the same defining formula also provides an elementary definition of the function or relation in $\mathfrak{A}$. The proof follows directly from the definition of syntactic definability and the fact that $\mathfrak{A}$ is a model of $\Gamma$.
