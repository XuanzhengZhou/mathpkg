For a given nonempty set $A$, the hierarchy of sets $A_\sigma$ provides the semantic universe for interpreting terms and formulas of a type language. At the base type $0$, $A_0$ is simply the set $A$ itself (the domain of individuals). For a compound type $\sigma = (\tau_0, \ldots, \tau_m)$, the set $A_\sigma$ consists of all $m$-ary relations whose $i$-th argument ranges over $A_{\tau_i}$. This inductive construction builds a typed relational hierarchy over $A$, generalizing the single-sorted domain of first-order logic to a type-theoretic framework.
