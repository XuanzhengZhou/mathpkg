Let $A$ be a nonempty set. By induction on the type $\sigma$, define the set $A_\sigma$ of relations of type $\sigma$ on $A$ as follows:

\begin{enumerate}
\item[(13)] $A_0 = A$;
\item[(14)] For a type $\sigma = (\tau_0, \ldots, \tau_m)$, set
  $$A_\sigma = \{R : R \subseteq A_{\tau_0} \times \cdots \times A_{\tau_m}\}.$$
\end{enumerate}
