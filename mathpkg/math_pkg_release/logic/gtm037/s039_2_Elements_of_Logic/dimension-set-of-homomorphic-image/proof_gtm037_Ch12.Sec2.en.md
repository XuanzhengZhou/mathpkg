The proof of Proposition 12.13 is not present in the extracted section text. The result follows from the definition of homomorphism: if $\kappa \notin \Delta fx$, then $c_\kappa' fx = fx$. Using the homomorphism property $fc_\kappa x = c_\kappa' fx$, we obtain $fc_\kappa x = fx$. This implies $c_\kappa x = x$ when $f$ is injective, but in general the dimension set inclusion $\Delta fx \subseteq \Delta x$ requires additional reasoning from the cylindric algebra axioms linking $c_\kappa$, $d_{\kappa\lambda}$, and the homomorphism conditions (i)-(vii).
