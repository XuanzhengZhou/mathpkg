Under a homomorphism $f$ of cylindric algebras, the dimension set of the image cannot increase. This reflects the fact that a homomorphism cannot introduce new "free coordinates" in an element — if $x$ is a cylinder in coordinate $\kappa$ (i.e., $c_\kappa x = x$), then $fx$ remains a cylinder in that same coordinate.
