Given a model $\mathfrak{A}$ of a first-order theory $\Gamma$, the collection of definable relations $\{\varphi^{\mathfrak{A}} : \varphi \in \mathrm{Fmla}_{\mathcal{L}}\}$ forms a cylindric set algebra. Moreover, the map sending each formula's equivalence class $[\varphi]$ to its interpretation $\varphi^{\mathfrak{A}}$ is a homomorphism from the Lindenbaum-Tarski algebra onto this set algebra, establishing the fundamental connection between syntactic formula algebras and semantic set algebras.
