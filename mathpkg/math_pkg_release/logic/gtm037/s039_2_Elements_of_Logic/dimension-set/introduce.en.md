In a cylindric algebra $\mathrm{CA}_\alpha$, the dimension set $\Delta x$ of an element $x$ records the set of coordinates $\kappa < \alpha$ for which $x$ is not a cylinder, i.e., $c_\kappa x \neq x$. This concept generalizes the notion of free variables in a formula, where cylindrification $c_\kappa$ corresponds to existential quantification over $v_\kappa$.
