Lemma 4.18 establishes that the function $f_1(x) = \prod_{i \leq x} p_i^2$ is elementary. Being elementary means the function can be defined from the basic functions (zero, successor, projection) using composition, bounded sum, and bounded product, and its graph is definable by a bounded formula.
