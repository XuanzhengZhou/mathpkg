The function $f_1 x = \prod_{i \leq x} p_i^2$ is defined using a bounded product of the prime function raised to the constant exponent $2$. Since the class of elementary functions is closed under bounded product and the prime function $p_i$ is elementary, $f_1$ is elementary.
