Lemma 4.20 states that for every positive integer $m$, the $m$-ary function $f_2^m$ is elementary. This follows from the recursive definition of $f_2^m$ using the concatenation function $\text{Cat}$ and the fact that $f_1$ is elementary, together with closure properties of the class of elementary functions.
