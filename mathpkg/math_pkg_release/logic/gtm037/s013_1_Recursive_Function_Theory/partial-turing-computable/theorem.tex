Let $f$ be an $m$-ary partial function.

\begin{enumerate}
\item[(i)] $(x_0, \ldots, x_{m-1}) \in \text{Dmn } f$,
\item[(ii)] there is a computation of $M$ beginning with $(F, c_1, n+1)$
\end{enumerate}

are equivalent; and if one of them holds, and $\langle F, c_1, n+1 \rangle, (G_1, a_1, b_1), \ldots, (G_{p-1}, a_{p-1}, b_{p-1})$ is a computation of $M$, then (1)-(3) of 3.9(ii) hold.
