Let $\mathcal{L}$ be a language having a unary relation symbol $\mathbf{P}$, and suppose that $\mathfrak{A}$ is a special $\mathcal{L}$-structure satisfying the following conditions:

(i) $|\mathbf{P}^{\mathfrak{A}}| \geq \aleph_0$;
(ii) for each operation symbol $\mathbf{O}$ of $\mathcal{L}$, the set $\mathbf{P}^{\mathfrak{A}}$ is closed under $\mathbf{O}^{\mathfrak{A}}$.

Let $\mathfrak{B}$ be the $\mathcal{L}$-structure such that $B = \mathbf{P}^{\mathfrak{A}}$, $\mathbf{R}^{\mathfrak{B}} = \mathbf{R}^{\mathfrak{A}} \cap {}^m\mathbf{P}^{\mathfrak{A}}$ for each $m$-ary relation symbol $\mathbf{R}$ of $\mathcal{L}$, and $\mathbf{O}^{\mathfrak{B}} = \mathbf{O}^{\mathfrak{A}} \upharpoonright {}^m\mathbf{P}^{\mathfrak{A}}$ for each $m$-ary operation symbol $\mathbf{O}$. Then $\mathfrak{B}$ is special.
