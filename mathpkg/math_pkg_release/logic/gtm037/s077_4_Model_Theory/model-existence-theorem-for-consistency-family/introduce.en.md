This theorem is the model existence theorem for logic without equality, restricted to relational languages. It states that any set of sentences belonging to a consistency family has a model. The proof uses a Henkin-style construction: (C0)-(C8) guarantee that maximal elements of the consistency family can be used to build a term model. The theorem is simpler than the corresponding result for logic with equality.
