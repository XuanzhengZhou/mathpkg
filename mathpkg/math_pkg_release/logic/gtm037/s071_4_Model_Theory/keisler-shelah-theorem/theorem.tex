\textbf{Theorem 26.42 (Keisler, Shelah).} The following conditions are equivalent:

\begin{enumerate}
\item[(i)] $\mathfrak{A} \equiv_{\mathrm{ee}} \mathfrak{B}$;
\item[(ii)] there exist an index set $I$ and an ultrafilter $D$ over $I$ such that $I\mathfrak{A}/D \cong I\mathfrak{B}/D$.
\end{enumerate}
