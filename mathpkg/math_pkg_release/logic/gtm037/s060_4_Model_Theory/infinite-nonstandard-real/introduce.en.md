In nonstandard analysis, the hyperreal field $^*\mathbb{R}$ contains elements that exceed every standard natural number in absolute value. An element $r \in {}^*\mathbb{R}$ is called *infinite* if $^*|r| > m$ for all $m \in \omega$. The complementary notion is *finite*: an element is finite if it is bounded in absolute value by some standard natural number. This distinction between finite and infinite elements is fundamental to the nonstandard approach to analysis, enabling the rigorous treatment of infinitesimals and the standard part map.
