Proposition 10.93 specializes Lemma 10.92 to the case where $\Gamma$ is a set of sentences. It states that a set $\Gamma$ of sentences is inconsistent precisely when there is a finite disjunction of the negations of some of its members that is provable (without any additional hypotheses). This result is derived from Lemma 10.92 together with 10.91 and is useful when working purely with sentences rather than arbitrary formulas.
