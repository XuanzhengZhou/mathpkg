Theorem 3.6 asserts that the class of recursive functions properly extends the class of primitive recursive functions. In other words, there exist functions that can be computed by the general recursive scheme (which includes the minimalization operator) that cannot be defined using only composition and primitive recursion. This result highlights the essential power added by the unbounded search operator $\mu$.
