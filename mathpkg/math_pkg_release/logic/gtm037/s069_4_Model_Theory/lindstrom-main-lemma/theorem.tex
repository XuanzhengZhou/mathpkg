Let $\mathscr{L}$ be a general logic such that $L_{\text{fo}} \subseteq \mathscr{L}$ but $\mathscr{L} \not\subseteq_{1\text{inf}} L_{\text{fo}}$. Assume also that $\mathscr{L}$ satisfies the downward Löwenheim–Skolem theorem, in the sense that any $\mathscr{L}$-elementary class with an infinite member has a denumerable member.

Furthermore, let $L$ be a first-order language with exactly one nonlogical constant, a unary relation symbol $P$. Then there is a class $\mathbf{K}$ of $L$-structures satisfying the following conditions:

\begin{enumerate}
\item[(i)] every member of $\mathbf{K}$ is infinite;
\item[(ii)] for any $\mathfrak{A} = (A, P^{\mathfrak{A}}) \in \mathbf{K}$, the set $P^{\mathfrak{A}}$ is finite and nonempty;
\item[(iii)] for every $n \in \omega \sim 1$ there is a countable $\mathfrak{A} \in \mathbf{K}$ such that $|P^{\mathfrak{A}}| = n$;
\item[(iv)] $\mathbf{K}$ is $\mathscr{L}$-projective.
\end{enumerate}
