An atom in a Boolean algebra is a minimal nonzero element: it cannot be split into smaller nonzero pieces. A Boolean algebra is atomic if every nonzero element contains an atom below it; the power set algebra of any set is the canonical example, with singletons as atoms. A Boolean algebra is atomless if it has no atoms at all; the algebra of regular open subsets of the rationals (or the interval algebra of $[0,1)$) provides a standard example. Some Boolean algebras are neither atomic nor atomless.
