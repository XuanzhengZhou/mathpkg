Let $\langle \mathfrak{A}_i : i \in I \rangle$ be a system of Boolean algebras,
$$\mathfrak{A}_i = \langle A_i, +_i, \cdot_i, -_i, 0_i, 1_i \rangle$$
for each $i \in I$. By the product of the system $\langle \mathfrak{A}_i : i \in I \rangle$ we mean the structure
$$\prod_{i \in I} \mathfrak{A}_i = \langle B, +, \cdot, -, 0, 1 \rangle$$
where $B = \prod_{i \in I} A_i = \{f : f \text{ is a function with domain } I \text{ and } f_i \in A_i \text{ for all } i \in I\}$ and $+, \cdot, -$ are defined coordinatewise (for arbitrary $f, g \in B, i \in I$):
$$(f + g)_i = f_i +_i g_i,$$
$$(f \cdot g)_i = f_i \cdot_i g_i,$$
$$(-f)_i = -_i f_i.$$
The constants $0, 1$ are the functions with $0_i = 0_i$ and $1_i = 1_i$ for all $i \in I$.
