In a finite Boolean algebra, every nonzero element can be decomposed into a join of finitely many atoms. The proof proceeds by induction: given a nonzero element $x$, if $x$ is not an atom, there exists a strictly smaller nonzero element $y < x$, and by finiteness this process must terminate, yielding an atom below $x$. Consequently, every finite Boolean algebra is isomorphic to the power set algebra of its set of atoms.
