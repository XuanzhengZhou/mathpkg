Let $\Gamma$ be a theory in $\mathcal{L}$ and let $\Gamma'$ be a definitional expansion of $\Gamma$ to a language $\mathcal{L}'$. Then:

(ii) $\Gamma$ is inseparable if and only if $\Gamma'$ is inseparable.

(iii) If $\Gamma$ is essentially undecidable, then $\Gamma'$ is essentially undecidable (the converse holds under an additional hypothesis).

(iv) $\Gamma$ is finitely inseparable if and only if $\Gamma'$ is finitely inseparable.
