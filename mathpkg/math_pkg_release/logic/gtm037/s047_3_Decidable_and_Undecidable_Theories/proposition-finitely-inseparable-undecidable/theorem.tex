Let $\Gamma$ be a finitely inseparable theory in $\mathcal{L}$, and let $C = \{ g^{+}\varphi : \varphi \text{ is a sentence which holds in every finite model of } \Gamma \}$. By 15.7, $g^{+*} \operatorname{Thm}_{\mathcal{L}}$ and $g^{+*} \operatorname{Sent}_{\mathcal{L}} \smallsetminus C$ are effectively inseparable. If $\Gamma$ is decidable, then $g^{+*} \Gamma$ is a recursive set containing $g^{+*} \operatorname{Thm}_{\mathcal{L}}$ and disjoint from $g^{+*} \operatorname{Sent}_{\mathcal{L}} \smallsetminus C$, which is impossible. Thus $\Gamma$ is undecidable.
