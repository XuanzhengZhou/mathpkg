Theorem 15.8 provides a rigorous version of a diagonalization argument for inseparable theories. Given an inseparable theory $\Gamma$, one can construct a recursive function $h$ with the property that any consistent extension $\Delta$ of $\Gamma$ whose set of Gödel numbers equals the domain of the $e$-th partial recursive function fails to decide some sentence $\varphi$, meaning neither $\varphi$ nor its negation belongs to $\Delta$. This is a key technical result underlying proofs of essential undecidability.
