For a given theory $\Gamma$, an $n$-type can be characterized in three equivalent ways. First, as a set $\Delta$ of $n$-variable formulas that is realized in some model of $\Gamma$ (the definition). Second, as a set that is maximal with respect to consistency over $\Gamma$ — no proper superset remains consistent. Third, as the complete set of $n$-variable formulas satisfied by some $n$-tuple in a model of $\Gamma$ whose cardinality is bounded by the number of $n$-variable formulas. This proposition provides the fundamental link between the semantic and syntactic views of types.
