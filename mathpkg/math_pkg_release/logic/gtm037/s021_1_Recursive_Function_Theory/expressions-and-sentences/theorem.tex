Let $\mathcal{P} = (n, c, P)$ be a sentential language. The set $\{n, c\} \cup P$ is the set of symbols of $\mathcal{P}$. An expression of $\mathcal{P}$ is any finite sequence of symbols of $\mathcal{P}$; the empty set (=empty sequence) is admitted. We define one-place and two-place operations $\neg$ and $\rightarrow$ on the set of expressions by setting
$$\neg \varphi = \langle n \rangle \varphi,$$
$$\varphi \rightarrow \psi = \langle c \rangle \varphi \psi,$$
for all expressions $\varphi$ and $\psi$. Recall from 1.11 that $\varphi\psi$ is just the juxtaposition of the two finite sequences $\varphi$ and $\psi$. $\text{Sent}_{\mathcal{P}}$, the set of sentences of $\mathcal{P}$, is the intersection of all classes $\Gamma$ of expressions of $\mathcal{P}$ such that $\langle s \rangle \in \Gamma$ for each $s \in P$ and $\Gamma$ is closed under $\neg$ and $\rightarrow$.
