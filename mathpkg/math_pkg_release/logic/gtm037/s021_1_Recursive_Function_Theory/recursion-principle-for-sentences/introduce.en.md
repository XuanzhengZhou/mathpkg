The recursion principle permits defining functions on the set of sentences by specifying: (1) the value on atomic sentences via $f$, (2) how to compute the value on $\neg\varphi$ from the value on $\varphi$ via $g$, and (3) how to compute the value on $\varphi \rightarrow \psi$ from the values on $\varphi$ and $\psi$ via $h$. The unique readability theorem guarantees that such a recursive definition yields a well-defined total function on all sentences. This principle is fundamental for defining semantic valuations and syntactic operations.
