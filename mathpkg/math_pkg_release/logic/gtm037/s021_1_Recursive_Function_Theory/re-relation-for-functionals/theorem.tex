Now we say that an $(m, n)$-relation $R$ is recursively enumerable (r.e.) provided that there is an $(m+n+1)$-ary recursive relation $S \subseteq {}^{m+n+1}\omega$ such that for all $\mathfrak{A} \in {}^m(\omega) \times {}^n\omega$,
$$\mathfrak{A} \in R \quad \text{iff} \quad \exists x \in \omega [(\mathfrak{A}x, x) \in S].$$
