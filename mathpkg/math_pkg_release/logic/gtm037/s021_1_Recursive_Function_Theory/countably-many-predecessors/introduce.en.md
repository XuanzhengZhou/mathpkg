For any Turing degree $\alpha \in D$, the set $\{\beta \in D : \beta < \alpha\}$ is at most countable. This follows from the fact that there are only countably many Turing machines, and each predecessor corresponds to a function computable by a Turing machine with an oracle from $\alpha$.
