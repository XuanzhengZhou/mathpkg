A sentence admits a construction sequence --- a finite list of expressions ending with the given sentence, where each expression is either atomic or built from earlier expressions using $\neg$ or $\rightarrow$. This provides an algorithmic characterization of well-formedness: a formula is a sentence exactly when it can be built up step-by-step from atomic sentences via the logical operations.
