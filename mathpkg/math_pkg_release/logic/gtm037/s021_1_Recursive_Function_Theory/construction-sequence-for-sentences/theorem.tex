$\varphi \in \text{Sent}_{\mathcal{P}}$ iff there is a finite sequence $\langle \psi_0, \ldots, \psi_{n-1} \rangle$ of expressions such that $\psi_{n-1} = \varphi$ and for each $i < n$, either $\psi_i \in P$ (so $\psi_i = \langle s \rangle$ for some $s \in P$) or $\psi_i = \neg \psi_j$ for some $j < i$ or $\psi_i = \psi_j \rightarrow \psi_k$ for some $j, k < i$.
