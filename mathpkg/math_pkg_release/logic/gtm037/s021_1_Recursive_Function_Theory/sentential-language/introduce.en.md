A sentential (propositional) language consists of three components: a negation symbol $n$, an implication symbol $c$, and a nonempty set $P$ of atomic sentence symbols. The symbols $n$ and $c$ must be distinct from each other and from the atomic symbols. This minimal setup, using Polish (parenthesis-free) notation, suffices to express all truth-functional connectives.
