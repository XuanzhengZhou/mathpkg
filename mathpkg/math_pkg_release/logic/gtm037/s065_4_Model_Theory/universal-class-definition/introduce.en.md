Definition 25.1 formalizes the notions of universal class (axiomatizable by universal sentences) and elementary class (axiomatizable by arbitrary first-order sentences). A universal class is a special case of an elementary class. The notation $\text{Mod } \Gamma$ denotes the class of all models of a set of sentences $\Gamma$.
