(i) Two sets $A$ and $B$ are \textbf{recursively separable} if there is a recursive set $C$ such that $A \subseteq C$ and $B \subseteq \omega \sim C$.

(ii) $A$ and $B$ are \textbf{recursively inseparable} if they are disjoint but not recursively separable.

(iii) $A$ and $B$ are \textbf{effectively inseparable} if they are disjoint and there is a $2$-ary recursive function $f$ such that for all $e$ and $r$, if $A \subseteq \mathrm{Dmn}\,\varphi_e^1$, $B \subseteq \mathrm{Dmn}\,\varphi_r^1$, and $\mathrm{Dmn}\,\varphi_e^1 \cap \mathrm{Dmn}\,\varphi_r^1 = \emptyset$, then $f(e, r) \in \omega \sim (\mathrm{Dmn}\,\varphi_e^1 \cup \mathrm{Dmn}\,\varphi_r^1)$.
