Definition 16.28 introduces $\mathcal{L}_5$ as a definitional expansion of $\mathcal{L}_4$ with three new symbols. The binary operation $(\cdot,\cdot)$ formalizes Kuratowski ordered pairs via $\{\{v_0\}, \{v_0, v_1\}\}$. The unary predicate $\mathbf{Rln}$ characterizes relations as sets consisting entirely of ordered pairs. The unary predicate $\mathbf{Fcn}$ characterizes functions as relations where each domain element is paired with at most one codomain element. This language provides the basic apparatus for formalizing functions and relations within set theory.
