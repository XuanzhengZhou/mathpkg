Lemma 16.26 states that in the language $\mathcal{L}_4$, the successor of any set is never equal to the empty set $0$. This is one of the Peano-like axioms for the natural numbers being developed within the formal set theory. The proof splits into two cases: if $\Omega x$ holds then $\mathbf{s} x = \mathbf{U} x \neq 0$, and if $\Omega x$ fails then $\mathbf{s} x = x \neq 0$ since $\Omega 0$ holds.
