Lemma 16.21 establishes that in the language $\mathcal{L}_3$, if a set $v_0$ is transitive and does not contain itself as an element, and the $\mathbf{U}$-images of $v_0$ and $v_1$ are equal, then $v_0 = v_1$. This is a cancellation property for the $\mathbf{U}$ operation (where $\mathbf{U} x = x \cup \{x\}$) that holds for transitive, well-founded sets. It implies that $\mathbf{U}$ is injective on the class of transitive sets not containing themselves.
