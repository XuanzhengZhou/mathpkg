Corollary 16.15 states that in the formal language $\mathcal{L}_2$, if a set $v_0$ satisfies the predicate $\mathbf{B}$, then the operation $\mathbf{Op}(v_0, v_1) = v_0 \cup \{v_1\}$ produces a set that also satisfies $\mathbf{B}$. This follows immediately from Lemma 16.14 since $\mathbf{B}$ is a weaker condition than $\mathbf{C}$, and the $\mathbf{C}$-closure under $\mathbf{Op}$ implies the corresponding $\mathbf{B}$-closure.
