Definition 16.16 introduces $\mathcal{L}_3$ as a definitional expansion of $\mathcal{L}_2$ with a new unary relation symbol $\mathbf{W}$. The axiom for $\mathbf{W}$ encodes the property that a set is well-founded in the sense that: (i) it contains no infinite descending $\in$-chains, (ii) every nonempty subset has an $\in$-minimal element, and (iii) every nonempty subset has a minimal element with respect to the $\in$-relation. This formalizes the notion of a well-founded set within the object language.
