A Markov algorithm $A$ is a finite list of rows $a_i b_i c_i$ ($i < m$), where $a_i$ and $b_i$ are words in a given alphabet and $c_i \in \{0, 1\}$. A row with $a_i b_i 0$ is indicated $a_i \rightarrow b_i$ (nonterminating substitution), while a row with $a_i b_i 1$ is indicated $a_i \rightarrow \cdot\ b_i$ (terminating substitution). A computation under $A$ is a finite sequence $\langle d_0, \ldots, d_m \rangle$ of words such that for each $i < m - 1$, $(d_i, d_{i+1})$ is a nonterminating algorithmic step, while $(d_{m-1}, d_m)$ is a terminating algorithmic step.
