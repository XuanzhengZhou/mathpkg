The Effective Lindenbaum Extension Theorem states that every consistent recursively enumerable (r.e.) theory can be extended to a complete and decidable theory. This is a constructive, effective version of Lindenbaum's Theorem (Theorem 11.13), which guarantees that every consistent theory can be extended to a complete theory but without the effectiveness condition. The construction proceeds by effectively enumerating all sentences and adding each sentence if it remains consistent with the theory built so far, resulting in a complete theory whose Gödel number set is r.e., hence decidable by Theorem 15.2.
