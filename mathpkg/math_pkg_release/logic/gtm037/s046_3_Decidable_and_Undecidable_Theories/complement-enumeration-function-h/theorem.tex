Define a partial recursive function $h$ as follows: for any $x \in \omega$,

$$h(x) = \mu y \big( x \notin g^{+*} \text{Sent}, \text{ or } x \in g^{+*} \text{Sent} \text{ and there exist } m, n, p \text{ such that }$$
$$y \text{ codes a triple } \langle \text{seq}, m, p \rangle \text{ where } \text{seq} \text{ is the sequence } (f(m,0), \ldots, f(m,n)),$$
$$\text{and } p \text{ is the G\"{o}del number of a proof from } \Gamma \text{ of }$$
$$\bigwedge_{i \leq n} g^{+-1} f(m, i) \rightarrow \neg g^{+-1} x \big).$$

Then $\text{Dmn}\,h = \omega \setminus g^{+*} \Gamma$.
