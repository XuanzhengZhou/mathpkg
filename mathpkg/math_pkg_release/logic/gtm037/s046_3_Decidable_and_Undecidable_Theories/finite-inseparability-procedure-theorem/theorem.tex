If $\Gamma$ is finitely inseparable (i.e., $g^{+*} \text{Thm}_{\mathcal{L}}$ and $M = g^{+*} \{ \varphi : \varphi \text{ is a sentence whose negation holds in some finite model of } \Gamma \}$ are effectively inseparable), then there exists an automatic procedure $P$ such that, given any consistent finitely axiomatizable theory $\Delta$ with the property that every finite model of $\Gamma$ is a model of $\Delta$, the procedure $P$ produces after finitely many steps a sentence $\varphi$ such that $\varphi \notin \Delta$ but $\varphi$ holds in all finite models of $\Delta$.
