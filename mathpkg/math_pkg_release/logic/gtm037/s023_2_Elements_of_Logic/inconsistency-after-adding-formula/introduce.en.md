Theorem 8.26 characterizes when adding a single sentence $\varphi$ to a set $\Gamma$ creates inconsistency: this happens precisely when $\Gamma$ proves the negation of $\varphi$. This result is a key tool in the completeness proof, used repeatedly to establish properties of maximal consistent sets.
