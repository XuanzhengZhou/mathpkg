A set $\Gamma$ of sentences is defined to be consistent if there exists some sentence $\varphi$ that is not semantically entailed by $\Gamma$, i.e., $\Gamma \not\models \varphi$. Equivalently, $\Gamma$ is consistent exactly when it has a model. This definition provides the semantic counterpart to the syntactic notion of consistency.
