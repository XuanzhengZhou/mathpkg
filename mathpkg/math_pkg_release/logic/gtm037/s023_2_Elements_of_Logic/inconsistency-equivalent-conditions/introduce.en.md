Theorem 8.25 gives three equivalent characterizations of inconsistency for a set $\Gamma$ of sentences in sentential logic. In particular, $\Gamma$ is inconsistent if and only if it proves the negation of the tautology $\varphi \to \varphi$ for some (equivalently, for every) sentence $\varphi$. This result provides a syntactic criterion for detecting inconsistency.
