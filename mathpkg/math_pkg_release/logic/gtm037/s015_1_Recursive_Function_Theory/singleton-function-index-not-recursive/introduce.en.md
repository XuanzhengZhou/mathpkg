This is a direct corollary of Rice's theorem. For any fixed unary partial recursive function $f$, the singleton set $F = \{f\}$ is a non-trivial property of partial recursive functions (it is non-empty and not all functions). By Rice's theorem, the index set $\{e : \varphi^1_e = f\}$ is therefore not recursive. This means there is no algorithm to decide whether a given Turing machine computes exactly the function $f$.
