This lemma defines the relation $R_0$ that captures the effect of modifying a single cell on a Turing machine tape. The relation holds when $x$ encodes a tape description $F$, $n$ encodes an integer position $e$, $\varepsilon$ is the new symbol written at that position, and $y$ encodes the resulting tape description $F^e_\varepsilon$ after the modification. This relation is a building block for arithmetizing Turing machine computation steps.
