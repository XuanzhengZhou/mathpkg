This lemma defines $f_2$ as the function that produces the G\"{o}del number of the constant binary sequence consisting of all 1's of length $x+1$. Since $f_2 x = g 1^{(x+1)}$, it encodes a sequence of $x+1$ ones under the G\"{o}del numbering for binary sequences. The function $f_2$ is elementary, providing a basic building block for constructing encoded representations of tape configurations.
