Lemma 10.37 provides the base case for the principle of substitution of equals for equals in first-order logic with equality. Given an operation symbol $O$ of rank $m$, if a single argument $\sigma_i$ equals a term $\tau$, then replacing $\sigma_i$ by $\tau$ in the application of $O$ yields an equal term. The proof relies on the equality axiom schema (10.23) and a suitable tautology together with modus ponens (10.34).
