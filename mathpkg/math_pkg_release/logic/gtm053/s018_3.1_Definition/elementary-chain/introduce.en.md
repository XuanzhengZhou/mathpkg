An elementary chain is an ordinal-indexed increasing sequence of structures where each inclusion is an elementary embedding. At limit stages, the structure is defined as the union of all earlier structures. The key property is that each $A_\alpha$ is an elementary substructure of the limit $A_\delta$ for $\alpha < \delta$, which is established by induction on formula complexity using the Tarski-Vaught criterion.
