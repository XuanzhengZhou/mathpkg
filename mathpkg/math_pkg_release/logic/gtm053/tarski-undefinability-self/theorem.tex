In the language SELF, let $R$ be any set (property) of expressions. Define $|rN(Q)|_R = 1$ if and only if $Q \in R$. Then the set of formulas that are true under this interpretation (the set of $Q$ such that $|Q|_R = 1$) cannot be equal to $R$ for any choice of $R$. Equivalently, truth in SELF is not definable within SELF.
