A \emph{partial Boolean algebra} is a set $B$ together with the following structures on $B$:

\begin{enumerate}
\item[(a)] A reflexive and symmetric binary relation $*$ called ``compatible measurability.'' Instead of $(a, b) \in *$ we write $a * b$.
\item[(b)] Partial binary operations $\vee$ and $\wedge$ defined exactly on pairs $a, b \in B$ with $a * b$.
\item[(c)] A unary operation $'$ defined everywhere on $B$.
\item[(d)] Distinguished elements $0, 1 \in B$.
\end{enumerate}

These structures satisfy the following axiom: for any subset $S \subseteq B$ in which any two elements are compatible (i.e., $a * b$ for all $a, b \in S$), there exists a subset $C \subseteq B$ containing $S$ such that $C$ is closed under all the operations, all elements of $C$ are pairwise compatible, and $C$ with the induced operations forms a Boolean algebra.
