A \emph{modular structure} (or modular lattice) $L$ is a set with binary operations $\wedge$ and $\vee$ that satisfy the following conditions:

\begin{enumerate}
\item[(a)] $\wedge$ and $\vee$ are associative and commutative;
\item[(b)] $a \wedge a = a \vee a = a$ for all $a \in L$ (idempotence);
\item[(c)] If $a \wedge b = b$, then $a \wedge (b \vee c) = b \vee (a \wedge c)$ for all $c \in L$ (modular law).
\end{enumerate}
