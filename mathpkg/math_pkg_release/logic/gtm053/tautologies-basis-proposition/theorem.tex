A finite list, or basis, of tautologies (logical polynomials in three variables $P, Q, R$) can be given with the following property. Let $\mathcal{F}$ be the set of all formulas obtainable from the basis by substitution. Then any tautology in $L$ is deducible from $\mathcal{F}$ using only the rule of deduction MP. The basis includes A0, A1, A2, A3, B1, B2 and additional tautologies C1-C5.
