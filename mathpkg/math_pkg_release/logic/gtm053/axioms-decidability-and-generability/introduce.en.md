This observation about the syntactic properties of axiom systems formalizes the intuitive requirement that axiomhood should be mechanically checkable. Property (a) is characteristic of axioms, while property (b) is characteristic of the set of deducible formulas. Godel's Incompleteness Theorem shows that for sufficiently rich theories, the set of true formulas has neither property.
