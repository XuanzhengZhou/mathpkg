\begin{enumerate}
\item[(a)] Any Godelian set $T$ is the set of $\phi$-true formulas $T_\phi L$ for a suitable interpretation of $L$ in some set $M$ having cardinality $\leqslant \operatorname{card}(\text{alphabet of } L) + \aleph_0$.
\item[(b)] Any set of formulas $\mathcal{E}$ which contains Ax $L$ and is consistent can be imbedded in a Godelian set.
\end{enumerate}
