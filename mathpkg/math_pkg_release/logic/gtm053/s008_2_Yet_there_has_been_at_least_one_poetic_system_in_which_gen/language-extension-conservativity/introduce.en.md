This fundamental theorem justifies the common mathematical practice of introducing new notation for definable objects. If a formula $P'$ provably defines a unique $x$ for given parameters $y_1, \ldots, y_n$, then adding a new function symbol $f$ together with the defining axiom $P'(f(\vec{y}), \vec{y})$ yields a conservative extension: any formula of the original language provable in the extended theory was already provable in the original. The proof uses an explicit translation procedure that eliminates occurrences of $f$ by existential quantification. Manin illustrates this with examples from set theory: introducing the unordered pair symbol $\{\}$, finite ordinals, and the constant $\omega_0$ for the first infinite ordinal — all without changing the deductive power of Zermelo–Fraenkel set theory.
