Gödel's Completeness Theorem establishes the fundamental equivalence between semantic truth and syntactic provability in first-order logic. Part (a) states that every Gödelian set (a complete, consistent, deduction-closed set containing all logical axioms) is exactly the set of formulas true under some interpretation whose domain has cardinality bounded by the alphabet size plus countable infinity. Part (b) states that any consistent set of formulas containing the logical axioms can be embedded in a Gödelian set, meaning that consistency guarantees the existence of a completing extension. The proof constructs a model from constant terms of the language, giving an explicit term model.
