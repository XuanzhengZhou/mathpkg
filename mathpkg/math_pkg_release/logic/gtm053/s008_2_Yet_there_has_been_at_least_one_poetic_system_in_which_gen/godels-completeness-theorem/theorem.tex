Let $L$ be a language in $\mathfrak{L}_1$, let $\phi$ be an interpretation of $L$, and let $T_\phi L$ be the set of $\phi$-true formulas. A set of formulas $\mathcal{E}$ in $L$ is consistent if the set of formulas deducible from $\mathcal{E}$ does not contain a contradiction, i.e., if there is no $P$ such that $\mathcal{E} \vdash P$ and $\mathcal{E} \vdash \neg P$.

\textbf{(a)} Any Gödelian set $T$ is the set of $\phi$-true formulas $T_{\phi}L$ for a suitable interpretation of $L$ in some set $M$ having cardinality $\leqslant \operatorname{card}(\text{alphabet of } L) + \aleph_0$.

\textbf{(b)} Any set of formulas $\mathcal{E}$ which contains $\operatorname{Ax} L$ and is consistent can be imbedded in a Gödelian set.
