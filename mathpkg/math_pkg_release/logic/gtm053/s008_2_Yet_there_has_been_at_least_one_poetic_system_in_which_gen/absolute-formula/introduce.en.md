An absolute formula is one whose truth value does not change when passing from a smaller model $M$ to a larger model $N$ (or vice versa) for any assignment of the free variables within $M$. Absoluteness is a key concept in model theory because it allows one to transfer truth between nested structures. A formula $P = \exists x Q(x)$ often fails to be absolute: it can be true in $N$ (where some witness exists) but false in $M$ (where no witness lies in the smaller domain). The Löwenheim–Skolem theorems are proved by carefully constructing submodels that make certain formulas absolute.
