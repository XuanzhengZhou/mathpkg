Let ${^*\mathbf{R}}$ be a nonstandard model of the reals and $\gamma \in {^*\mathbf{R}}$. Define
$$[\gamma]^- = \{q \in \mathbb{Q} : q \leq \gamma\}, \quad [\gamma]^+ = \{q \in \mathbb{Q} : q > \gamma\}.$$
Then $\gamma$ is called \textbf{bounded} if neither $[\gamma]^+$ nor $[\gamma]^-$ is empty. The set of all bounded elements is denoted $B \subseteq {^*\mathbf{R}}$.
