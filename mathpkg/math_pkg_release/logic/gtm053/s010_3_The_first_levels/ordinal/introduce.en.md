Due to von Neumann, an ordinal is defined intrinsically as a transitive set that is well-ordered by the membership relation $\in$. This elegant definition avoids the need to consider equivalence classes of well-ordered sets, which would form proper classes of classes. Instead, each ordinal serves as a canonical representative of a well-ordering isomorphism type, with the property that $\alpha = \{\beta \mid \beta < \alpha\}$.
