Let $G$ be a function of sets (it is sufficient to assume that $G$ is defined on all sets in the universe) whose values are sets. Then there exists a unique function $F$ on the ordinals such that

$$F(\alpha) = G(\text{the set of values of } F \text{ on the elements of } \alpha).$$

In particular, this equality uniquely determines

$$F(0) = G(\varnothing), \quad F(1) = G(\{F(0)\}), \quad \ldots$$
