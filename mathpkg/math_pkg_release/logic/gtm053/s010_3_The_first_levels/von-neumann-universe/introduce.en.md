The von Neumann universe $V$ is the standard model of Zermelo-Fraenkel set theory, built as a cumulative hierarchy of sets. The construction begins with the empty set $V_0 = \varnothing$ and iterates the power set operation: $V_{n+1} = P(V_n)$. Each level $V_n$ contains all sets of rank less than $n$, and the hierarchy extends transfinitely by taking unions at limit ordinals, yielding $V_{\omega_0} = \bigcup_{n=0}^{\infty} V_n$ and beyond.
