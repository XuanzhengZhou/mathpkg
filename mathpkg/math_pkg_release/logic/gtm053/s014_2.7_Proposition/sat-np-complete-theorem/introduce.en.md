This is the Cook-Levin theorem, stating that the Boolean satisfiability problem (SAT) is NP-complete. That is, every problem in NP can be reduced to SAT in polynomial time. The proof sketches how a Turing machine computation protocol for any NP problem can be encoded as a Boolean satisfiability instance.
