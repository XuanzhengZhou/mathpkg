Let $\mathcal{E}$ be a finite set of formulas in $L$, and let $P$ be a logical polynomial over $\mathcal{E}$. For any map $v: \mathcal{E} \to \{0,1\}$, set $P^v = P$ if $v(P) = 1$ and $P^v = \neg P$ if $v(P) = 0$. Let $\mathcal{E}^v = \{Q^v \mid Q \in \mathcal{E}\}$. Then for any $v$, $\mathcal{F} \cup \mathcal{E}^v \vdash P^v$ (using MP).
