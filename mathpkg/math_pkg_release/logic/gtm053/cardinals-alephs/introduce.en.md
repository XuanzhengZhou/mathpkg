Cardinals (alephs) are the ordinals that serve as canonical measures of the size of sets. Since every set can be well-ordered (by the Axiom of Choice), every set has the same cardinality as a unique aleph. The aleph sequence $\aleph_0, \aleph_1, \aleph_2, \ldots$ provides the standard scale for measuring infinite cardinalities.
