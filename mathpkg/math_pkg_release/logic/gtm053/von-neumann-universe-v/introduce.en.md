The von Neumann universe V is the cumulative hierarchy of sets built by iterating the power set operation through all ordinals. Starting from the empty set, each level V_alpha collects all sets whose elements have already been constructed at earlier levels. The universe V is the intended model of ZFC set theory: every mathematical object can be encoded as a set in V.
