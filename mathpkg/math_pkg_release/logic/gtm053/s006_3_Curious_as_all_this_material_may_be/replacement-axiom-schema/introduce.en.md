The replacement axiom schema is a fundamental axiom scheme of Zermelo-Fraenkel set theory. It asserts that if a formula $P$ defines a functional relation (for each $x$ there exists a unique $y$ satisfying $P$), then the image of any set under this function is again a set.

Formally, the schema states that for any formula $P$ with free variables among those listed, if for all $x$ in a set $u$ there is a unique $y$ satisfying $P(x, y, \vec{z})$, then there exists a set $w$ containing precisely those $y$ that correspond to some $x \in u$ via $P$.

The separation (comprehension) axioms, which allow forming subsets defined by a property, are deducible from replacement together with the axioms of equality. The replacement axioms are stronger than separation and are essential for developing transfinite recursion and the theory of ordinals in ZF.
