A constructible (or definable) function over an algebraically closed field $F$ is a function whose graph is a constructible subset of $F^{n+1}$. An analysis of constructible sets and constructible functions yields a structural result: any constructible function defined everywhere on a constructible subset $V \subseteq F^n$ is piecewise rational. That is, the domain $V$ can be partitioned into finitely many constructible pieces on each of which the function is given by a rational expression.
