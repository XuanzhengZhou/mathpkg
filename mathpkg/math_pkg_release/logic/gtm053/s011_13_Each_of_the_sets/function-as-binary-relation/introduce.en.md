A function is defined as a special kind of binary relation: one in which each first component uniquely determines the second. In the von Neumann universe, maps between sets are identified with their graphs --- the set of ordered pairs $\langle U, f(U) \rangle$. This reduces the notion of function to the primitive $\in$, consistent with the set-theoretic foundation.
