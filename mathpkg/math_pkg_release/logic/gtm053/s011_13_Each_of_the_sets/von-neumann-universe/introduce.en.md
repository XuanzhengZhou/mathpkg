The von Neumann universe $V$ is the cumulative hierarchy of all sets, built up from the empty set by iterating the powerset operation along all ordinal stages. At successor stages, $V_{\alpha+1}$ collects all subsets of the previous stage; at limit stages, it takes the union of all earlier stages. This construction provides the standard model of ZFC set theory and is the foundational universe in which most modern mathematics is formalized.
