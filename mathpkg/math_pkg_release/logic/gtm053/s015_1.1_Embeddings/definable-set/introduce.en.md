A definable set in an $L$-structure $A$ is the set of tuples satisfying a given $L$-formula $P$. Every $L$-formula with $n$ free variables defines a subset of $A^n$, namely the set of all $n$-tuples that make the formula true in $A$. This notion extends naturally to definable relations (viewing subsets of $A^n$ as $n$-ary relations) and definable operations (when the definable set coincides with the graph of a function).
