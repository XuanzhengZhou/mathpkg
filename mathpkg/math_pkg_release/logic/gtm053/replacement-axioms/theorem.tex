For any formula $P(x, y, z_1, \ldots, z_n)$:
$$\forall z_1 \cdots \forall z_n \forall u (\forall x (x \in u \Rightarrow \exists!y P(x, y, z_1, \ldots, z_n)) \Rightarrow \exists w \forall y (y \in w \Leftrightarrow \exists x (x \in u \land P(x, y, z_1, \ldots, z_n))))$$
The hypothesis says that $P$ gives $y$ as a function of $x \in u$; the conclusion says that the image of the set $u$ under this function is a set $w$.
