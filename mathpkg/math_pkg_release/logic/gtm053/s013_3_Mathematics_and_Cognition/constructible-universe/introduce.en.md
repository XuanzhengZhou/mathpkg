The constructible universe L, introduced by Godel, is the smallest transitive model of ZF that contains all ordinals. It is built by a transfinite hierarchy where each stage adds only those subsets that are definable from previous stages using a fixed finite set of fundamental operations. L is the key tool for proving the consistency of AC and GCH with ZF.
