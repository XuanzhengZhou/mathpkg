Let $F$ be any (nonstrictly) monotonic function on a subclass of the regular cardinals that takes values in the class of cardinals and that satisfies: $\operatorname{cf} (\aleph_{F(k)}) > \aleph_k$. Then the assertion "$\forall$ regular $k \in \operatorname{dom} F, 2^{\aleph_k} = \aleph_{F(k)}$" does not contradict the Zermelo--Fraenkel axioms.
