The basic functions are:
$$\operatorname{suc}: \mathbb{Z}^+ \to \mathbb{Z}^+, \quad \operatorname{suc}(x) = x + 1;$$
$$1^{(n)}: (\mathbb{Z}^+)^n \to \mathbb{Z}^+, \quad 1^{(n)}(x_1, \ldots, x_n) = 1, \quad n \geqslant 0;$$
$$\operatorname{pr}_i^n: (\mathbb{Z}^+)^n \to \mathbb{Z}^+, \quad \operatorname{pr}_i^n(x_1, \ldots, x_n) = x_i, \quad n \geqslant 1.$$
