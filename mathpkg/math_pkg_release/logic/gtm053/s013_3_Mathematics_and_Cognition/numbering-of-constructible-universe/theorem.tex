There exists a formula $N(x, y)$ in $L$ with two free variables such that:
(a) For any $X, Y \in V$, $N(X, Y)$ is $V$-true if and only if $X$ is an ordinal and $Y$ is the $X$-th constructible set in the canonical numbering.
(b) $N(x, y)$ is $M$-absolute for any transitive model $M$ of the axioms (without choice). In particular, it is $L$-absolute.
