(a) Given a family $\{X_i\}_{i \in I} \subset V^B$ and elements $a_i \in B$, the function
$$\|Y \in X\| = \bigvee_{i \in I} a_i \wedge \|Y \in X_i\|$$
is a random class equivalent to a random set. Moreover, $\|X_i \in X\| \geqslant a_i$.

(b) Similarly, the function $\|Y \in Z\| = \bigvee_i a_i \wedge \|Y \in X_i\|$ is a random class equivalent to a random set. If $a_i \wedge a_j = 0$ for $i \neq j$, then $\|Z = X_i\| \geqslant a_i$.
