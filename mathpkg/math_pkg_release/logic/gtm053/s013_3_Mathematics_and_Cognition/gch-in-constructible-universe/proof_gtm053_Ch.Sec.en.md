The proof reduces to showing that for each ordinal $\alpha$, $\mathcal{P}(L_{\omega_\alpha}) \cap L \subset L_{\omega_{\alpha+1}}$, which implies $\operatorname{card}^L(\mathcal{P}(\omega_\alpha)) \leqslant \omega_{\alpha+1}$. The key lemma (4.3) states that if $X \subseteq \beta$ is constructible, then $X \in L_\gamma$ for some $\gamma$ with $\operatorname{card}_L(\gamma) = \operatorname{card}_L(\beta)$. This is proved using the Lowenheim-Skolem-type reflection argument (Proposition 4.5), together with the Mostowski collapsing theorem relativized to L.
