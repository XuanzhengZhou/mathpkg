An operator $\operatorname{cl}$ on a set $M$ is \emph{finitary} if for every $U \subseteq M$,
$$\operatorname{cl}(U) = \bigcup \{ \operatorname{cl}(U') : U' \subseteq U,\ U' \text{ finite} \}.$$
Equivalently, for any $v \in \operatorname{cl}(U)$, there exists a finite subset $U' \subseteq U$ such that $v \in \operatorname{cl}(U')$.
