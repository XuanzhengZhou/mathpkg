A pregeometry $(M, \operatorname{cl})$ is \emph{locally modular} if there exists an element $a \in M$ such that the localization $(M, \operatorname{cl}_a)$ is modular.

An example of a locally modular but not modular pregeometry is an affine geometry over a field $K$: a $K$-vector space $V$ with a set $\{v_0, v_1, \ldots, v_n\} \subseteq V$ considered dependent if and only if $\{v_1 - v_0, \ldots, v_n - v_0\}$ are $K$-linearly dependent.
