\begin{enumerate}
\item[(a)] The ordinal that immediately follows a given $\alpha$ is $\alpha \cup \{\alpha\}$, denoted $\alpha + 1$.
\item[(b)] An ordinal $\beta$ is called a \text{limit ordinal} if $\beta \neq \emptyset$ and $\beta \neq \alpha + 1$ for any $\alpha$. The first limit ordinal is $\omega_0$. If $\alpha$ is a limit ordinal, then $\alpha = \bigcup_{\beta < \alpha} \beta$.
\end{enumerate}
