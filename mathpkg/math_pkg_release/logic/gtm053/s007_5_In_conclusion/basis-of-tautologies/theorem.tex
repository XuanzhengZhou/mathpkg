A finite list, or ``basis,'' of tautologies---logical polynomials in three variables $P, Q, R$---can be given with the following property.

Let $L$ be any language in $\mathfrak{L}_1$, and let $\mathcal{F}$ be the set of all formulas in $L$ that can be obtained from the basis tautologies by substituting all possible formulas in place of $P, Q, R$. Then any tautology in $L$ is deducible from $\mathcal{F}$ using only the rule of deduction MP.
