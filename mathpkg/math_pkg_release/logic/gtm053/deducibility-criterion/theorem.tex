Let $\mathcal{E} \supset \text{Ax } L$.
\begin{enumerate}
\item[(a)] A formula $P$ is deducible from $\mathcal{E}$ if and only if either $\mathcal{E}$ is inconsistent, or $P$ is $\phi$-true for all models $\phi$ of $\mathcal{E}$ having cardinality $\leqslant \operatorname{card}(\text{alphabet of } L) + \aleph_0$.
\item[(b)] A formula $P$ is independent of $\mathcal{E}$ if and only if both $\mathcal{E} \cup \{P\}$ and $\mathcal{E} \cup \{\neg P\}$ are consistent; this holds iff there exist models for $\mathcal{E} \cup \{P\}$ and $\mathcal{E} \cup \{\neg P\}$.
\end{enumerate}
