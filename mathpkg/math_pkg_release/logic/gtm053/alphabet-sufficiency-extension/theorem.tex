If $\mathcal{E}$ is consistent and contains Ax $L$, then there exist:
\begin{enumerate}
\item[(a)] a language $L'$ whose alphabet is obtained from the alphabet of $L$ by adding a set of new constants having cardinality $\leqslant \operatorname{card}(\text{alphabet of } L) + \aleph_0$;
\item[(b)] a set of formulas $\mathcal{E}'$ in $L'$ such that $\mathcal{E}' \supset \mathcal{E}$ and the alphabet of $L'$ is sufficient for $\mathcal{E}'$.
\end{enumerate}
