The language SELF has an alphabet consisting of: $E$ (display operator), $r$ (a unary relation symbol), $*$ (quotation symbol), and a countable set of variables. Distinguished expressions are defined by: if $P$ is an expression without $*$, then $*P*$ is a label (name) of $P$. If $P$ is a name of $Q$, then $EP$ is a name of the display of $Q$ (the expression $Q * Q*$). Formulas have the form $rN(Q)$ or $\neg rN(Q)$.
