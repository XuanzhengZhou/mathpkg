A deduction of a formula $P$ from a set of formulas $\mathcal{E}$ (in a language $L$ in $\mathfrak{L}_1$) is a finite sequence of formulas $P_1, \ldots, P_n = P$ with the property that for each $i$, either $P_i$ is a logical axiom (a tautology or quantifier axiom), or $P_i \in \mathcal{E}$, or $P_i$ is a direct consequence of some preceding formulas by MP or Gen. We write $\mathcal{E} dash P$.
