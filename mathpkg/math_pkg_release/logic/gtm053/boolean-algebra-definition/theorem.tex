A Boolean algebra $B$ is a set with a unary operation $'$, two binary operations $\lor$ and $\land$, and distinguished elements $0$ and $1$, satisfying:
\begin{enumerate}
\item[(a)] $(a')' = a$ for all $a \in B$.
\item[(b)] $\land$ and $\lor$ are each associative and commutative.
\item[(c)] $\land$ and $\lor$ are distributive with respect to one another.
\item[(d)] $(a \lor b)' = a' \land b'$, $(a \land b)' = a' \lor b'$ (De Morgan laws).
\item[(e)] $a \lor a = a \land a = a$ (idempotence).
\item[(f)] $1 \land a = a$, $0 \lor a = a$.
\end{enumerate}
