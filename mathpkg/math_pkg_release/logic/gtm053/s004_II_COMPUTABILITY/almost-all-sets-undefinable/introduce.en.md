This corollary follows directly from Proposition 2.12 and Cantor's theorem. Since $|M| < 2^{|M|}$ for any infinite set $M$, and the number of definable sets is bounded by $\operatorname{card}(\text{alphabet}) + \aleph_0$, any finitely describable language can define only a countable number of subsets. Thus, "almost all" subsets of an infinite domain are undefinable. This is the mathematical basis for Skolem's paradox and illustrates the fundamental inadequacy of formal languages for completely capturing intuitive mathematical concepts.
