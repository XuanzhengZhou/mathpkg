A set of formulas $\mathcal{E}$ in a language $L$ of class $\mathfrak{L}_1$ is called Gödelian if it satisfies the following conditions:

(a) $\mathcal{E}$ is complete: for any closed formula $P$, either $P \in \mathcal{E}$ or $\neg P \in \mathcal{E}$.

(b) $\mathcal{E}$ does not contain a contradiction: there is no formula $P$ with both $P \in \mathcal{E}$ and $\neg P \in \mathcal{E}$.

(c) $\mathcal{E}$ is closed with respect to the rules of deduction MP (modus ponens) and Gen (generalization).

(d) $\mathcal{E}$ contains all the logical axioms of $L$, i.e. $\mathrm{Ax}\,L = \{\text{tautologies of } L\} \cup \{\text{quantifier axioms}\} \subset \mathcal{E}$.
