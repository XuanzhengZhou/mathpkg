This proposition establishes a fundamental limitation of formal languages: the number of definable sets is bounded by the cardinality of the language plus countably many. The proof counts the number of possible formulas. If there are at most countably many variables, there are at most $\operatorname{card}(\text{alphabet}) + \aleph_0$ formulas. If there are uncountably many variables, any definable set can be defined using a fixed countable subset of variables. This result explains Skolem's paradox and shows that "most" subsets of an infinite set are undefinable.
