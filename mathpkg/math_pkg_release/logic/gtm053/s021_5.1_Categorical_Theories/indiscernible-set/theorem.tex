A set $I$ (of elements of a model $\mathcal{A}$) is \textit{indiscernible} if for every $n$ and every $L$-formula $P(x_1, \ldots, x_n)$, either

$$\mathcal{A} \models P(i_1, \ldots, i_n) \text{ for all distinct } i_1, \ldots, i_n \in I$$

or

$$\mathcal{A} \models \neg P(i_1, \ldots, i_n) \text{ for all distinct } i_1, \ldots, i_n \in I.$$
