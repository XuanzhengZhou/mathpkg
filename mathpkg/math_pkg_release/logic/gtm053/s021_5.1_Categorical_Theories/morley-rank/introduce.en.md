Morley rank is a dimension-theoretic notion defined for definable sets in $\aleph_0$-stable theories. It assigns an ordinal rank to each definable set, measuring how many times the set can be partitioned into infinitely many disjoint definable subsets of the same rank. Morley rank is the minimal function satisfying the inductive definition, and it provides the foundation for a well-behaved dimension theory in the context of classification theory.
