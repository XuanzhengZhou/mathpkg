The average type of an indiscernible set $I$ over a parameter set $C$ collects all $C$-definable formulas that hold for all but finitely many elements of $I$. This construction is fundamental in stability theory because it yields a complete and nonforking type, providing a canonical way to extend types. In the context of algebraically closed fields, the average type of an algebraically independent set is called the generic type of the field.
