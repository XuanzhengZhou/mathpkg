Let $T$ be a stable theory and $\mathcal{A} \preceq \mathcal{B}$ a sufficiently saturated elementary extension of a small model $\mathcal{A}$. Let $A$ be the set of parameters naming all elements of $\mathcal{A}$. Then every 1-type $p$ over $A$ can be defined by an $A$-indiscernible set $I \subseteq \mathcal{B}$:

\begin{enumerate}
\item[(i)] every $a \in I$ realizes $p$;
\item[(ii)] for every $C$ with $A \subseteq C \subseteq \mathcal{B}$, the average type $\operatorname{Av}_C(I)$ is a nonforking extension of $p$.
\end{enumerate}
