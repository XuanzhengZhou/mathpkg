A \text{partial Boolean algebra} is a set $B$ together with:
\begin{enumerate}
\item[(a)] A reflexive and symmetric binary relation $*$, called compatible measurability ($a * b$).
\item[(b)] Partial binary operations $\vee$ and $\wedge$ and a partial unary operation $'$, all defined only on compatible pairs.
\item[(c)] Distinguished elements $0$ and $1$.
\end{enumerate}
such that each set of pairwise compatible elements is contained in a subset that forms a Boolean algebra under the partial operations.
