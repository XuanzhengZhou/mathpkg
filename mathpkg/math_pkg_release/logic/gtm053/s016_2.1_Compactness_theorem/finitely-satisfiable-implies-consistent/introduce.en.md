This lemma establishes that finite satisfiability implies deductive consistency. It is a crucial stepping stone in the first proof of the Compactness Theorem, which proceeds via Godel's completeness theorem: if a set of sentences is finitely satisfiable, then it is consistent; by completeness, it is therefore satisfiable. The proof is straightforward and relies on the fact that any formal deduction involves only finitely many formulas.
