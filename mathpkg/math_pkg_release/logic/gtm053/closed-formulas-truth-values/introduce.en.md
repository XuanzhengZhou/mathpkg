This corollary of Proposition 2.10 states that sentences (formulas without free variables) have absolute truth values in any interpretation, independent of the particular assignment of values to variables. This is the formal justification for the intuitive notion that a mathematical statement that mentions no free variables should be either determinately true or determinately false in a given structure.
