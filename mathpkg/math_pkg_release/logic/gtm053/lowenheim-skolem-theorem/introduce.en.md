The Lowenheim-Skolem theorem states that from any model N of a theory, one can extract an elementary submodel M of bounded cardinality (relative to the size of the language and an initial subset). The construction adds witnesses for all existential formulas, ensuring that any object whose existence is asserted in N can be found within M.
