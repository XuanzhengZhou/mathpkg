An ultrapower is a special case of an ultraproduct where all factor structures are identical. Given a first-order structure $M$ and an ultrafilter $\mathcal{F}$ on an index set $I$, the ultrapower $M^I / \mathcal{F}$ is formed by taking the direct product of $M$ with itself $|I|$ times and then identifying elements that agree on a set in $\mathcal{F}$. The natural diagonal embedding $\theta$ sends each element $m \in M$ to the equivalence class of the constant function with value $m$, allowing $M$ to be viewed as an elementary substructure of its ultrapower.
