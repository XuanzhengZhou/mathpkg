The \emph{enlargement} of $\mathcal{T} = (\mathcal{R}, A, S)$ with respect to a set $\sigma$ of concurrent relations is the theory $\mathcal{T}^\sigma = (\mathcal{R}, A \cup A_\sigma, S \cup \{c_\rho \mid \rho \in \sigma\})$ where $A_\sigma = \{\rho(x, c_\rho) \mid \rho \in \sigma, x \in D_\rho\}$. A \emph{full enlargement} takes $\sigma$ as all concurrent relations.
