A Turing machine fails for a given input if, when started in that initial state, it never reaches a terminal state -- its sequence of states continues indefinitely. Since the machine's transition is deterministic at each step, failure is equivalent to the computation running forever without halting. This definition captures the possibility that a Turing machine may not terminate on every input, a fundamental limitation that underlies the undecidability results developed later in the chapter.
