A computation by a Turing machine is a finite sequence of states that begins with an initial state (internal state $q_0$) and ends in a terminal state (no applicable quadruple), where each successive state follows from the previous one by a single transition step. This definition formalizes the notion of a successful, terminating execution of the machine on a given input. Crucially, computations are finite by definition — the machine may instead run forever, in which case it fails for that input.
