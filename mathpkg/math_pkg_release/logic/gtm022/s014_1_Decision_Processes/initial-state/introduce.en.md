The initial state of a Turing machine is any instantaneous description in which the machine's internal state is the distinguished start state $q_0$. A computation begins when the machine is placed in an initial state with the input encoded on its tape. The requirement that the machine must be in state $q_0$ to commence operating ensures a well-defined starting point for every computation.
