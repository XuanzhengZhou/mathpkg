An instantaneous description of a Turing machine $M$ with alphabet $\mathfrak{S}$ and set $\mathfrak{Q}$ of internal states is a finite string $s_{\alpha_1} s_{\alpha_2} \cdots s_{\alpha_r} q s_{\beta_1} s_{\beta_2} \cdots s_{\beta_t}$, where $s_{\alpha_i}, s_{\beta_j} \in \mathfrak{S}$ and $q \in \mathfrak{Q}$. It indicates that the machine is in state $q$, scanning the leftmost symbol $s_{\beta_1}$ of $\tau = s_{\beta_1} \cdots s_{\beta_t}$.
