Let $q \in P(X)$ and let $A \subseteq P(X)$. We say $q$ is a deduction from $A$, or $q$ is provable from $A$, and write $A \vdash q$, if there exists a proof of $q$ from $A$ in the propositional calculus. The set of all deductions from $A$ is denoted $\operatorname{Ded}(A)$.

An element $p \in P(X)$ is called a theorem of the propositional calculus if $\emptyset \vdash p$, written simply as $\vdash p$.
