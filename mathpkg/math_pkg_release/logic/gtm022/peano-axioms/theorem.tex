The Peano axioms for the natural numbers $(N, s, 0)$ are:
$P_1$: $0$ is a natural number.
$P_2$: For every $n$, $s(n)$ is a natural number.
$P_3$: $s$ is injective: $s(m) = s(n)$ implies $m = n$.
$P_4$: $0$ is not a successor: $s(n) \neq 0$ for all $n$.
$P_5$ (Induction): If $\pi$ is a property such that $\pi(0)$ and $\pi(n) \Rightarrow \pi(s(n))$ for all $n$, then $\pi(n)$ for all $n$.
