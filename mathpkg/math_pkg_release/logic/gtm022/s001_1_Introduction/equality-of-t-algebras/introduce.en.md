Definition 1.9 specifies that two $T$-algebras $A$ and $B$ are equal precisely when they are built on the same underlying set and have exactly the same operations for every operation symbol $t \in T$. This is a stronger condition than mere isomorphism; it requires identity of both the carrier set and each individual operation function. The definition is motivated by the earlier observation that a zero ring has only one function $R \times R \to R$, yet we still want to distinguish the addition $+$ and multiplication $\times$ as separate operations.
