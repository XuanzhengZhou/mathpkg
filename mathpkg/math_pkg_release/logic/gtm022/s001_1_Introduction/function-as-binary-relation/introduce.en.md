A function $f: A \rightarrow B$ is defined as a special kind of binary relation — specifically, a subset of $A \times B$ with the property that each element of the domain $A$ is paired with exactly one element of the codomain $B$. This set-theoretic definition unifies the concept of function with the previously defined notion of an $n$-ary relation and provides the foundation for defining operations (which are functions on product sets) in universal algebra.
