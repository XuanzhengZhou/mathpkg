\textbf{Theorem 1.7 (The Satisfiability Theorem).} In $\text{Pred}_g(V, \mathcal{R})$, for any set $A \subseteq P(V, \mathcal{R})$ and any $p \in P(V, \mathcal{R})$, the proof-theoretic consequence $A \vdash_g p$ is equivalent to the semantic consequence $A \vDash_g p$; i.e., $A \vdash_g p$ if and only if $v(p) = 1$ for every proper interpretation with $v(A) \subseteq \{1\}$.
