\textbf{Definition 4.1.} Let $\Pi \subseteq P(\mathcal{T})$. We say that $\mathcal{T}$ \textit{admits $\Pi$-reduction of quantifiers} if there is a process which assigns to each $p \in P(\mathcal{T})$ an element $q \in P(\mathcal{T})$ such that
\begin{enumerate}
  \item[(i)] $q$ is a propositional combination of elements of $\Pi$,
  \item[(ii)] $\text{var}(q) \subseteq \text{var}(p)$,
  \item[(iii)] $\mathcal{T} \vdash p \Leftrightarrow q$.
\end{enumerate}
