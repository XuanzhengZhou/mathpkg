The set of all filters on a given set $I$ is partially ordered by inclusion. An ultrafilter is characterized as a maximal element of this poset: if $\mathcal{F}$ is an ultrafilter and $\mathcal{G}$ is a filter with $\mathcal{F} \subseteq \mathcal{G}$, then $\mathcal{F} = \mathcal{G}$. This follows directly from condition (iv) of the ultrafilter definition—any proper superset would have to contain both a set and its complement, violating the filter condition that the empty set is excluded.
