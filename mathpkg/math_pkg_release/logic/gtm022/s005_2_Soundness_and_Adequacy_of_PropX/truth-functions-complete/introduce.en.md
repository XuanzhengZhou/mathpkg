Theorem 3.3 states that the Lindenbaum algebra $L(X_n)$ — the set of equivalence classes of propositional formulas in $n$ variables modulo logical equivalence — coincides exactly with the set of all truth functions $f: \mathbb{Z}_2^n \to \mathbb{Z}_2$. This is the functional completeness theorem for the $\{\Rightarrow, F\}$ propositional calculus: every Boolean function can be expressed as a propositional formula using only implication and the constant false. The proof uses the fact that constant functions are represented ($0 = \bar{F}$, $1 = \overline{F \Rightarrow F}$) and that the set of truth functions is closed under the implication operation.
