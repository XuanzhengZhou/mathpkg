A set $\mathcal{F}$ of subsets of $I$ is a filter if:
(i) $\varnothing \notin \mathcal{F}$.
(ii) If $A \in \mathcal{F}$ and $A \subseteq B \subseteq I$, then $B \in \mathcal{F}$.
(iii) If $A, B \in \mathcal{F}$, then $A \cap B \in \mathcal{F}$.
An ultrafilter is a filter satisfying:
(iv) For each $A \subseteq I$, exactly one of $A$ and $I - A$ belongs to $\mathcal{F}$.
Ultrafilters are the maximal elements of the set of filters ordered by inclusion.
