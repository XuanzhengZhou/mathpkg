The halting (stopping) problem asks whether an arbitrary Turing machine, started on an arbitrary input, eventually halts. This classic result shows that no Turing machine can decide this question for all machine-input pairs. The proof uses a diagonal argument: if a Turing machine could decide the halting problem, one could construct a computable function that differs from every computable function, a contradiction.
