The union of a set $c$, denoted $\cup c$, is the set of all elements that belong to some member of $c$. Its existence is guaranteed by combining the Axiom of Union (ZF4), which provides an overset, with the Axiom Schema of Subsets (ZF2) to restrict to exactly those elements. The binary union $a \cup b$ of two sets is then defined as the union of the pair set $\{a, b\}$.
