The empty set $\varnothing$ is the unique set that contains no elements. It is constructed using the Axiom Schema of Subsets (ZF2) applied to any existing set $a$ with the property $x \neq x$, which no set satisfies. Its uniqueness follows from the Axiom of Extension (ZF1), and it is a subset of every set: $\varnothing \subseteq b$ for all sets $b$.
