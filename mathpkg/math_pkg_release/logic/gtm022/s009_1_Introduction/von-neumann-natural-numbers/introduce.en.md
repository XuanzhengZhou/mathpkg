The von Neumann construction defines the natural numbers purely in terms of sets. Starting with the empty set as $0 = \varnothing$, each successive number $n^+$ is defined as $n \cup \{n\}$, the successor of $n$. Thus $1 = \{0\}$, $2 = \{0, 1\}$, $3 = \{0, 1, 2\}$, and so on. The set $\omega$ is the set of all natural numbers thus constructed, and when equipped with the usual addition and multiplication, it is denoted by $N$. This construction embeds arithmetic within ZF set theory.
