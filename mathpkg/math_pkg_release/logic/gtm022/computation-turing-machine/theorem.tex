A \emph{computation} by the machine $M$ is a finite sequence $[d_0], [d_1], \ldots, [d_p]$ of states such that $[d_0]$ is initial ($q = q_0$), $[d_p]$ is terminal (no applicable quadruple), and $[d_i] \xrightarrow{M} [d_{i+1}]$ for $i = 0, 1, \ldots, p-1$. $M$ \emph{fails} for input $[d_0]$ if no such finite computation exists.
