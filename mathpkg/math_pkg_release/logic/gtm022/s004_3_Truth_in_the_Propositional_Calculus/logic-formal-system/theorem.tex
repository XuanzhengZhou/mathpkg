\text{A logic } \mathcal{L} \text{ is a system consisting of a set } P \text{ of elements (called propositions), a set } \mathcal{V} \text{ of functions (called valuations) from } P \text{ into some value set } W, \text{ and, for each subset } A \text{ of } P, \text{ a set of finite sequences of elements of } P \text{ (called proofs from the assumptions } A).
