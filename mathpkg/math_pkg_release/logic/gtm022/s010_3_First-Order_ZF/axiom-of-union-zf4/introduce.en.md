The Axiom of Union (ZF4) states that for any set $c$, there exists a set $a$ that contains all elements of elements of $c$. Formally, if $x$ belongs to some $b$ that belongs to $c$, then $x$ belongs to $a$. Together with Separation, this yields the union $\bigcup c$.
