Let $\mathcal{T} = (\mathcal{R}, A, C) \supseteq N_0$ and have $\mathbb{N}$ as model. Let $f: U \rightarrow \mathbb{N}$ be a function defined on some subset $U$ of $\mathbb{N}$. We say that $f$ is strongly definable in $\mathcal{T}$ if there is an element $p(x, y) \in P(V, \mathcal{R})$ such that, for all $m, n \in \mathbb{N}$, $\mathcal{T} \vdash p(m, n)$ if and only if $m \in U$ and $f(m) = n$. The definition is extended in the obvious way for functions of several variables.
