A model of the first-order theory $\mathcal{T} = (\mathcal{R}, A, C)$ is a triple $(M, v, \psi)$ where $M$ is a non-empty set, $v: C \to M$ assigns elements to constants, $\psi: \mathcal{R} \to \operatorname{rel}(M)$ assigns relations to relation symbols with $\psi(\mathcal{I})$ being identity, and for all $a \in A$, $a$ is true in $(M, v, \psi)$.
