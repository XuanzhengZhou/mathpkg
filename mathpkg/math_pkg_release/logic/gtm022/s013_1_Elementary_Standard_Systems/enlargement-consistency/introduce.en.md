This theorem establishes that enlarging the complete theory $\mathcal{T}(\mathbb{S})$ by adding new constants that witness concurrent relations always preserves consistency. The proof uses the concurrency condition and the compactness theorem: any finite subset of the new axioms $A_\sigma$ can be satisfied because each concurrent relation $\rho$ provides, for any finite set of arguments, a witness element in $S$.
