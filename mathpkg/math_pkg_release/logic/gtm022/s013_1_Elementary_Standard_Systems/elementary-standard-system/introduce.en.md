An elementary standard system is a set equipped with a distinguished collection of relations on that set, always containing the identity relation. The underlying set and the system itself are conventionally denoted by the same symbol. This concept provides the foundational framework for studying standard mathematical systems (such as the natural numbers, reals, or complex numbers) within first-order predicate calculus.
