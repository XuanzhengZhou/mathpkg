Let $\mathcal{T} = (\mathcal{R}, A, C) \supseteq \mathcal{N}_0$ and have $\mathbb{N}$ as model. Let $f: U \to \mathbb{N}$ be a partial function. $f$ is strongly definable in $\mathcal{T}$ if there is an element $p(x, y) \in P(V, \mathcal{R})$ such that, for all $m, n \in \mathbb{N}$, $\mathcal{T} \vdash p(m, n)$ if and only if $m \in U$ and $f(m) = n$.
