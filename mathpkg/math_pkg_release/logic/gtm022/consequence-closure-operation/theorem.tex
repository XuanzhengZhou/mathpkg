$\operatorname{Con}$ is a closure operation on $P(X)$, that is, it has the properties:
\begin{enumerate}
\item $A \subseteq \operatorname{Con}(A)$,
\item If $A_1 \subseteq A_2$, then $\operatorname{Con}(A_1) \subseteq \operatorname{Con}(A_2)$,
\item $\operatorname{Con}(\operatorname{Con}(A)) = \operatorname{Con}(A)$.
\end{enumerate}
