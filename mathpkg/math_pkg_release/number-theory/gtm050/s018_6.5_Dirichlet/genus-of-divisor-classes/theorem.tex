In Gauss's theory of binary quadratic forms, a \textbf{genus} is the set of all divisor classes that share the same character. If the character of a divisor class $A$ is a tuple of signs (e.g., $++$, $--$, $+-+$, etc.), then the genus containing $A$ consists of all divisor classes whose character equals that of $A$.

Since the character of a product is the product of the characters, all genera contain the same number of divisor classes. If $h$ is the class number and $g$ is the number of genera, then $h = g \cdot n$ where $n$ is the number of classes per genus.
