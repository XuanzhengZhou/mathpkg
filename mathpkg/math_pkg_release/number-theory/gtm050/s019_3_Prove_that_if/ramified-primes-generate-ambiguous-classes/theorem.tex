If $p$ is a prime which ramifies, then $(p, *)^2 \sim I$ and the class of $(p, *)$ is an ambiguous class. In addition, when $D > 0$, $(-1, *)$ lies in an ambiguous class. Therefore any product $(p_1, *)(p_2, *) \cdots (p_k, *)$ in which $p_1, p_2, \ldots, p_k$ are primes which ramify or, when $D > 0$, where $p_1$ may be $-1$, lies in an ambiguous class.
