Expanding the product of conjugate surds in $\mathbb{Z}[\sqrt{-3}]$:

$$(p + q\sqrt{-3})(p - q\sqrt{-3}) = p^2 - pq\sqrt{-3} + pq\sqrt{-3} - q^2(\sqrt{-3})^2 = p^2 - q^2(-3) = p^2 + 3q^2.$$

This is a direct algebraic computation using the multiplication rule $(a+b\sqrt{-3})(c+d\sqrt{-3}) = (ac-3bd) + (ad+bc)\sqrt{-3}$ with $a=p$, $b=q$, $c=p$, $d=-q$.
