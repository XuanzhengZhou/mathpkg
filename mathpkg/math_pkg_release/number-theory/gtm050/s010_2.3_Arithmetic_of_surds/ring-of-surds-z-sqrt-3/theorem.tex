The set of "numbers" of the form
$$a + b\sqrt{-3}$$
where $a$ and $b$ are integers, together with the operations of addition, subtraction, and multiplication defined by
$$(a + b\sqrt{-3}) + (c + d\sqrt{-3}) = (a+c) + (b+d)\sqrt{-3},$$
$$(a + b\sqrt{-3})(c + d\sqrt{-3}) = (ac - 3bd) + (ad + bc)\sqrt{-3},$$
forms a commutative ring with unit $1 = 1 + 0\sqrt{-3}$.
