For a nontrivial Dirichlet character $\chi$ modulo $\lambda$, the value of the $L$-series at $s=1$ can be expressed as:
$$L(1, \chi) = \sum_{n=1}^{\infty} \frac{\chi(n)}{n} = \sum_{\nu=1}^{\lambda-1} c_\nu \log \frac{1}{1 - \alpha^\nu}$$
where $c_\nu$ are integers determined by $\chi$ and $\alpha$ is a primitive $\lambda$-th root of unity. This evaluation plays a crucial role in Dirichlet's class number formula.
