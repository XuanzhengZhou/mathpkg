An element $x + y\sqrt{-c}$ of $\mathbb{Z}[\sqrt{-c}]$ is called a \textbf{unit} if it divides 1, i.e., if there exists another element $u + v\sqrt{-c}$ such that
$$(x + y\sqrt{-c})(u + v\sqrt{-c}) = 1.$$
This is equivalent to the condition $x^2 + cy^2 = \pm 1$.

For $c=1$ (Gaussian integers), the units are $\pm 1, \pm i$. For $c=3$, the units are $\pm 1, \frac{\pm 1 \pm \sqrt{-3}}{2}$ (six units). For real quadratic fields ($c < 0$), there are infinitely many units given by the solutions of Pell's equation.
