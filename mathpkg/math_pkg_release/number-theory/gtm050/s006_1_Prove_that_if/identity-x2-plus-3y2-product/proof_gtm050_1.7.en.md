The identity is verified by direct algebraic expansion:

$$(a^2 + 3b^2)(c^2 + 3d^2) = a^2c^2 + 3a^2d^2 + 3b^2c^2 + 9b^2d^2$$

$$(ac - 3bd)^2 + 3(ad + bc)^2 = (a^2c^2 - 6abcd + 9b^2d^2) + 3(a^2d^2 + 2abcd + b^2c^2)$$
$$= a^2c^2 - 6abcd + 9b^2d^2 + 3a^2d^2 + 6abcd + 3b^2c^2$$
$$= a^2c^2 + 3a^2d^2 + 3b^2c^2 + 9b^2d^2$$

Both expansions yield the same expression, proving the identity.
