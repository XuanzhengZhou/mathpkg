Let $\lambda$ be a prime and let $\alpha$ be a primitive $\lambda$-th root of unity (i.e., $\alpha^\lambda = 1$, $\alpha \neq 1$). A \text{cyclotomic integer} is a number of the form
$$a_0 + a_1\alpha + a_2\alpha^2 + \cdots + a_{\lambda-1}\alpha^{\lambda-1}$$
where $a_0, a_1, \ldots, a_{\lambda-1}$ are ordinary integers. These form a commutative ring under the usual operations, using $\alpha^\lambda = 1$ and $1 + \alpha + \alpha^2 + \cdots + \alpha^{\lambda-1} = 0$.
