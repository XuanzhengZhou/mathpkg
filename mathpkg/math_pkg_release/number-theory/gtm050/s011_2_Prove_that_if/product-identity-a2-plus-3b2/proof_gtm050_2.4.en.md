Expand the right-hand side:

$$(ac - 3bd)^2 + 3(ad + bc)^2 = (a^2 c^2 - 6abcd + 9b^2 d^2) + 3(a^2 d^2 + 2abcd + b^2 c^2)$$
$$= a^2 c^2 - 6abcd + 9b^2 d^2 + 3a^2 d^2 + 6abcd + 3b^2 c^2$$
$$= a^2 c^2 + 3a^2 d^2 + 3b^2 c^2 + 9b^2 d^2$$
$$= a^2(c^2 + 3d^2) + 3b^2(c^2 + 3d^2)$$
$$= (a^2 + 3b^2)(c^2 + 3d^2).$$

Alternatively, using the arithmetic of surds in $\mathbb{Z}[\sqrt{-3}]$:
$$(a + b\sqrt{-3})(c + d\sqrt{-3}) = (ac - 3bd) + (ad + bc)\sqrt{-3}.$$
Taking the norm $N(x+y\sqrt{-3}) = x^2+3y^2$, the multiplicativity of the norm gives the identity.
