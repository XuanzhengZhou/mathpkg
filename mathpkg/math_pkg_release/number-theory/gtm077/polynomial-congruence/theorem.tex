Two integral polynomials $f(x) = c_0 + c_1 x + \cdots + c_k x^k$ and $g(x) = a_0 + a_1 x + \cdots + a_k x^k$ are called congruent modulo $n$, written
$$f(x) \equiv g(x) \pmod{n},$$
if their corresponding coefficients are congruent modulo $n$:
$$c_i \equiv a_i \pmod{n} \quad \text{for all } i = 0, 1, 2, \ldots$$

Note: This is a stronger condition than $f(a) \equiv g(a) \pmod{n}$ for all integers $a$. For example, $x^p \equiv x \pmod{p}$ holds for all $x$ by Fermat's theorem, but the polynomials $x^p$ and $x$ are not congruent modulo $p$ since their coefficients differ.
