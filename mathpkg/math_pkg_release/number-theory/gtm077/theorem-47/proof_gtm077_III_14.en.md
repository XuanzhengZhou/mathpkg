# Proof of Theorem 47: Binomial Congruence modulo $2^\alpha$

**Theorem 47.** With odd $q$ and odd $a$, the congruence $x^q \equiv a \pmod{2^\alpha}$ always has exactly one solution. For $q = 2$ and odd $a$, the congruence $x^2 \equiv a \pmod{2^\alpha}$ is solvable for $\alpha \geq 3$ if and only if it is solvable modulo $8$, that is, if $a \equiv 1 \pmod{8}$, and the number of incongruent solutions in this case is equal to $4$. For $a \equiv 1 \pmod{4}$, the congruence $x^2 \equiv a \pmod{4}$ has two solutions; otherwise it has no solutions for odd $a$.

*Proof.* We use Theorem 45, which describes the structure of $\Re(2^\alpha)$:

- For $\alpha = 1$: $\Re(2) = \{1\}$ is trivial.
- For $\alpha = 2$: $\Re(4) = \{1, 3\} \cong C_2$ is cyclic of order $2$.
- For $\alpha \geq 3$: $\Re(2^\alpha) \cong C_2 \times C_{2^{\alpha-2}}$ is the direct product of a cyclic group of order $2$ (generated by $-1$) and a cyclic group of order $2^{\alpha-2}$ (generated by $5$).

**Case 1: $q$ is odd.** Since the order of $\Re(2^\alpha)$ is $2^{\alpha-1}$, which is a power of $2$, and $q$ is odd, we have $\gcd(q, 2^{\alpha-1}) = 1$. Therefore the $q$-th power map $x \mapsto x^q$ is an automorphism of the group $\Re(2^\alpha)$. Indeed, on each cyclic factor, raising to the $q$-th power is injective (and hence bijective) since $\gcd(q, \text{order}) = 1$. Consequently, for every odd $a$, the congruence $x^q \equiv a \pmod{2^\alpha}$ has exactly one solution modulo $2^\alpha$.

**Case 2: $q = 2$, $\alpha \geq 3$.** Write the group as $\Re(2^\alpha) = \langle -1 \rangle \times \langle 5 \rangle$. Every element can be written uniquely as $(-1)^\varepsilon \cdot 5^t$ with $\varepsilon \in \{0, 1\}$ and $t \in \{0, 1, \ldots, 2^{\alpha-2}-1\}$.

Consider the squaring map. For an element $x = (-1)^\varepsilon \cdot 5^t$:

$$x^2 = ((-1)^\varepsilon \cdot 5^t)^2 = (-1)^{2\varepsilon} \cdot 5^{2t} = 5^{2t}.$$

Thus the square of any element lies in the subgroup generated by $5$. The image of the squaring map is the subgroup $\langle 5^2 \rangle$, which has index $2$ in $\langle 5 \rangle$ and therefore order $2^{\alpha-3}$.

The image consists of those elements $5^{2t}$ for $t = 0, 1, \ldots, 2^{\alpha-3}-1$. Since $5^2 = 25 \equiv 1 \pmod{8}$, every element in the image of the squaring map is congruent to $1$ modulo $8$. Conversely, if $a \equiv 1 \pmod{8}$ and $\gcd(a, 2) = 1$, then $a$ lies in the subgroup generated by $5^2$, so $a$ is a square.

For the number of solutions: if $a \equiv 1 \pmod{8}$, then $a = 5^{2t}$ for some $t$. The equation $x^2 = 5^{2t}$ has solutions $x = \pm 5^t$, and also $x = \pm 5^{t + 2^{\alpha-3}}$ (since $5^{2(t + 2^{\alpha-3})} = 5^{2t} \cdot (5^{2^{\alpha-2}}) \equiv 5^{2t}$). These four solutions are: $5^t$, $-5^t$, $5^{t + 2^{\alpha-3}}$, $-5^{t + 2^{\alpha-3}}$. They are pairwise incongruent modulo $2^\alpha$. No other solutions exist because the kernel of the squaring map has order $4$ (the elements of order dividing $2$ are $\pm 1$ and $\pm (1 + 2^{\alpha-1})$).

If $a \not\equiv 1 \pmod{8}$, then $a$ is not in the image of the squaring map, so the congruence has no solutions.

**Case 3: $q = 2$, $\alpha = 2$ (modulo $4$).** $\Re(4) = \{1, 3\}$. The squares are $1^2 = 1$ and $3^2 = 9 \equiv 1 \pmod{4}$. So $x^2 \equiv 1 \pmod{4}$ has solutions $x \equiv 1, 3 \pmod{4}$ (two solutions), while $x^2 \equiv 3 \pmod{4}$ has no solutions.

In summary:
- $q$ odd, $(a, 2) = 1$: exactly $1$ solution modulo $2^\alpha$.
- $q = 2$, $\alpha \geq 3$, $a \equiv 1 \pmod{8}$: $4$ solutions.
- $q = 2$, $\alpha \geq 3$, $a \not\equiv 1 \pmod{8}$: $0$ solutions.
- $q = 2$, $\alpha = 2$, $a \equiv 1 \pmod{4}$: $2$ solutions; $a \equiv 3 \pmod{4}$: $0$ solutions.
- $q = 2$, $\alpha = 1$: trivial ($1$ solution). $\square$
