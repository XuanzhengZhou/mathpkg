Consider $ax + b \equiv 0 \pmod{n}$ with $d = (a, n) > 1$. The congruence is solvable if and only if $d \mid b$. In this case there are exactly $d$ distinct solutions modulo $n$. They are obtained by solving the reduced congruence $(a/d)x + (b/d) \equiv 0 \pmod{n/d}$, which has a unique solution $x_0$ modulo $n/d$, and taking
$$x = x_0 + \frac{n}{d} y, \quad y = 0, 1, \ldots, d-1.$$
