A function $\chi$ from a finite Abelian group $\mathfrak{G}$ to the complex numbers is called a group character if
$$\chi(AB) = \chi(A) \cdot \chi(B) \quad \text{for all } A, B \in \mathfrak{G}$$
and $\chi$ is not identically zero. It follows that $\chi(E) = 1$ and $\chi(A)$ is always an $h$th root of unity.

**Theorem 30.** There are exactly $h$ distinct characters of a finite Abelian group $\mathfrak{G}$ of order $h$. If $B_1, \ldots, B_r$ is a basis with orders $h_1, \ldots, h_r$, then each character is uniquely determined by assigning arbitrary $h_m$th roots of unity $\zeta_m$ to $\chi(B_m)$, yielding
$$\chi(A) = \zeta_1^{x_1} \cdots \zeta_r^{x_r} \quad \text{for } A = B_1^{x_1} \cdots B_r^{x_r}.$$
