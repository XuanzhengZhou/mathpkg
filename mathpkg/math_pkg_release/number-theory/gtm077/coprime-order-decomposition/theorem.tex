Let $h = a_1 a_2 \cdots a_r$ with pairwise coprime integers $a_i$. Then each element $C$ of the finite Abelian group $\mathfrak{G}$ can be represented uniquely as
$$C = A_1 \cdot A_2 \cdots A_r,$$
where $A_i^{a_i} = E$ for $i = 1, 2, \ldots, r$. The $A_i$ are given by $A_i = C^{h n_i / a_i}$ where the $n_i$ satisfy $\sum_i (h/a_i) n_i = 1$.
