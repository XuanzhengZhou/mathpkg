For an algebraic number field $k$ of degree $n$ with $r_1$ real embeddings, the theta function is defined as
$$\Theta(t_1, \ldots, t_n) = \sum_{\alpha} e^{-\pi \sum_{p=1}^{n} |\alpha^{(p)}|^2 t_p},$$
where $\alpha$ runs through all integers of $k$ and $t_p$ are complex variables with positive real part. This function satisfies a functional equation (Jacobi transformation formula) relating $\Theta(t)$ to $\Theta(1/t)$, which is the key to proving reciprocity for Gauss sums.
