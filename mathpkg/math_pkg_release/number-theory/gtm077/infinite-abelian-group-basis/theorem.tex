For an infinite Abelian group $\mathfrak{G}$, a system of elements $A_i$ (of infinite order) and $T_k$ (of finite orders $h_k$) is called a basis if each element can be represented as
$$C = A_1^{x_1}A_2^{x_2}\cdots T_1^{y_1}T_2^{y_2}\cdots$$
with only finitely many nonzero integer exponents, where the $x_i$ are uniquely determined and the $y_k$ are uniquely determined modulo $h_k$.
