A system $\mathfrak{G}$ of elements $A, B, C, \ldots$ is called a group if:
\begin{enumerate}
\item[(i)] There is a composition rule assigning to elements $A$ and $B$ a unique element $C = AB$ in $\mathfrak{G}$.
\item[(ii)] The associative law holds: $A(BC) = (AB)C$ for all $A, B, C$.
\item[(iii)] Cancellation laws: If $AB = A'B$ then $A = A'$, and if $BA = BA'$ then $A = A'$.
\item[(iv)] For every $A, B$ in $\mathfrak{G}$ there exist $X, Y$ in $\mathfrak{G}$ such that $AX = B$ and $YA = B$.
\end{enumerate}
If $\mathfrak{G}$ contains only finitely many elements ($h$ in number), it is called a finite group of order $h$; otherwise an infinite group. In a finite group, condition (iv) follows from (i) and (iii).
