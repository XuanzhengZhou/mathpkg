The set of numbers $\lambda$ in a number field $K$ for which the trace $\operatorname{Tr}_{K/\mathbb{Q}}(\lambda\alpha)$ is a rational integer for all integers $\alpha$ of $K$ forms an ideal $\mathfrak{M}$ in $K$. The inverse ideal $\mathfrak{D} = \mathfrak{M}^{-1}$ is called the different of $K$. The different is an integral ideal whose prime factors are exactly the ramified primes. The norm of the different equals the absolute value of the discriminant: $N(\mathfrak{D}) = |d|$.
