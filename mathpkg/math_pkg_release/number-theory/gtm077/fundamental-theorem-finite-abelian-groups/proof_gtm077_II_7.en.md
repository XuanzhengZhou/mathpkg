# Proof of Theorem 24/25: Fundamental Theorem of Finite Abelian Groups

**Theorem.** Every finite abelian group \(\mathfrak{G}\) of order \(h\) can be decomposed into a direct product of cyclic subgroups whose orders are prime powers.

**Reduction to the prime-power case.** By Theorem 23, \(\mathfrak{G}\) decomposes into a direct product of subgroups whose orders are the maximal prime-power factors appearing in the prime factorization of \(h\). Thus it suffices to prove the theorem for \(h = p^k\) where \(p\) is prime and \(k \geq 1\).

Let \(\mathfrak{G}\) have order \(p^k\). Every element of \(\mathfrak{G}\) has order \(p^\alpha\) with \(0 \leq \alpha \leq k\).

**Maximal independent system.** Choose a maximal system of independent elements
\[
B_1, B_2, \ldots, B_r
\]
with orders \(h_1 \geq h_2 \geq \cdots \geq h_r > 1\), each a power of \(p\). Such a system exists because the group is finite: start with any element of maximal order, then iteratively adjoin independent elements until no more can be added.

**Claim.** Every element of \(\mathfrak{G}\) is representable as a product of powers of the \(B_i\).

This follows from the three lemmas:

*Lemma (a).* No \(B_i\) is a \(p\)-th power of an element of \(\mathfrak{G}\). Otherwise, replacing it by a \(p\)-th root would yield a larger independent system, contradicting maximality.

*Lemma (b).* If \(B_m\) is replaced by \(A = B_m^u B_{m+1}^{x_{m+1}} \cdots B_r^{x_r}\) with \(u \not\equiv 0 \pmod{p}\), the new system remains maximal independent. (The order of \(A\) equals that of \(B_m\), and every product from the new set equals some product from the old set and conversely.)

*Lemma (c).* If \(C^p\) is representable by the \(B_i\), so is \(C\). Indeed, write \(C^p = B_1^{x_1} \cdots B_r^{x_r}\); if the first non-\\(p\\)\-divisible exponent is \(x_m = u\), replacing \(B_m\) by \(A = B_1^{x_1/p} \cdots B_{m-1}^{x_{m-1}/p} C B_{m+1}^{-x_{m+1}/p} \cdots B_r^{-x_r/p}\) would enlarge the independent system, which is impossible. Hence every \(x_i\) is divisible by \(p\), and \(C = B_1^{x_1/p} \cdots B_r^{x_r/p}\).

Now for any \(A \in \mathfrak{G}\) of order \(p^m\), we have \(A^{p^m} = E\), so \(A^{p^m}\) is trivially representable. By Lemma (c), \(A^{p^{m-1}}\) is representable, and so on by descending induction until \(A\) itself is representable.

**Conclusion.** Every element is uniquely expressed as

\[
C = B_1^{x_1} B_2^{x_2} \cdots B_r^{x_r},
\]

where \(x_i\) runs through a complete residue system modulo \(h_i\). Thus \(\mathfrak{G}\) is isomorphic to the direct product of the cyclic groups generated by the \(B_i\), whose orders are prime powers \(h_i = p^{k_i}\). Moreover \(h = h_1 h_2 \cdots h_r\).

\(\square\)
