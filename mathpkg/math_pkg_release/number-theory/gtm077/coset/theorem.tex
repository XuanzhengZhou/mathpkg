Let $\mathfrak{U} = \{U_1, U_2, \ldots\}$ be a subgroup of $\mathfrak{G}$. For an element $A \in \mathfrak{G}$, the set
$$A\mathfrak{U} = \{A U_1, A U_2, \ldots\}$$
is called a (left) coset of $\mathfrak{U}$ in $\mathfrak{G}$.

**Lemma.** If two cosets $A\mathfrak{U}$ and $B\mathfrak{U}$ have one element in common, then they are identical (as sets). Consequently, the distinct cosets partition $\mathfrak{G}$.
