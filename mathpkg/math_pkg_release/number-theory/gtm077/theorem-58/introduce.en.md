This is Theorem 58 from Lectures on the Theory of Algebraic Numbers by Erich Hecke.

The $n$ numbers $\omega^{(1)}, \ldots, \omega^{(n)}$ in $K(\theta)$ form a fundamental system if and only if there is no linear relation

$$\sum_{i=1}^{n} u_i \omega^{(i)} = 0$$

with coefficients in $k$ except if all $u_i = 0$.
