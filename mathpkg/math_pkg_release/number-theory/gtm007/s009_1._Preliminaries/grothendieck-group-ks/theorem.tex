We denote by $K_+(S)$ the quotient of $S$ by the stable isomorphism relation and if $E \in S$, we denote by $(E)$ the class of $E$ in $K_+(S)$. The operation $\oplus$ defines by passage to quotient a composition law $+$ on $K_+(S)$; this law is commutative, associative, and has for neutral element the class $0$ of the module $0 \in S$. One has

$$(E \oplus E') = (E) + (E').$$

Moreover, if $x, y, z \in K_+(S)$ are such that $x + z = y + z$, one has $x = y$.

We define the group $K(S)$ from the semi-group $K_+(S)$: an element of $K(S)$ is a pair $(x, y)$ with $x, y \in K_+(S)$, two pairs $(x, y), (x', y')$ being identified if and only if $x + y' = y + x'$. The composition law of $K(S)$ is defined by

$$(x, y) + (x', y') = (x + x', y + y').$$

It makes $K(S)$ into a commutative group with neutral element $(0, 0)$. We identify $K_+(S)$ with a subset of $K(S)$ by the map $x \mapsto (x, 0)$. Every element of $K(S)$ can be written in the form $(E) - (F)$ with $E, F \in S$.
