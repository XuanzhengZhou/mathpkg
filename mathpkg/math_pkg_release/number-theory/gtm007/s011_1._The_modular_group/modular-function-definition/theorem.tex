A weakly modular function is called \textbf{modular} if it is meromorphic at infinity.

Concretely, suppose $f$ satisfies $f(z+1) = f(z)$ and is expressed as a function of $q = e^{2\pi i z}$, denoted $\tilde{f}(q)$, which is meromorphic in the punctured disk $0 < |q| < 1$. Then $f$ is meromorphic at infinity if $\tilde{f}$ extends to a meromorphic function at $q = 0$, i.e. $\tilde{f}$ admits a Laurent expansion
$$\tilde{f}(q) = \sum_{n=-\infty}^{+\infty} a_n q^n,$$
where $a_n = 0$ for $n$ sufficiently small (i.e. only finitely many negative powers of $q$).
