Let $f$ be a meromorphic function on the upper half-plane $H$. Then $f$ is a weakly modular function of weight $2k$ (for the modular group $G$) if and only if it satisfies the two relations:
\begin{align}
f(z+1) &= f(z), \\
f(-1/z) &= z^{2k} f(z).
\end{align}
More generally, if $g = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \in \mathbf{SL}_2(\mathbf{Z})$ and $g$ is taken modulo $\{\pm 1\}$, the invariance condition is
$$f(gz) = (cz+d)^{2k} f(z),$$
which is equivalent to the invariance of the differential form $f(z)\,dz^k$ under $G$.
