Two elements $E, E' \in S$ are stably isomorphic if there exists $F \in S$ such that $E \oplus F \simeq E' \oplus F$. Let $K_+(S)$ be the set of stable isomorphism classes, with addition induced by $\oplus$. The Grothendieck group $K(S)$ is defined from $K_+(S)$ by the standard construction: elements are formal differences $(E) - (F)$.
