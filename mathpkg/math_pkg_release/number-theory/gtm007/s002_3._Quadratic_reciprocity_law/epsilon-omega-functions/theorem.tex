If $n$ is an odd integer, let $\varepsilon(n)$ and $\omega(n)$ be the elements of $\mathbb{Z}/2\mathbb{Z}$ defined by:
$$\varepsilon(n) \equiv \frac{n-1}{2} \pmod{2} = \begin{cases} 0 & \text{if } n \equiv 1 \pmod{4} \\ 1 & \text{if } n \equiv -1 \pmod{4} \end{cases}$$
$$\omega(n) \equiv \frac{n^2-1}{8} \pmod{2} = \begin{cases} 0 & \text{if } n \equiv \pm 1 \pmod{8} \\ 1 & \text{if } n \equiv \pm 5 \pmod{8} \end{cases}$$

[The function $\varepsilon$ is a homomorphism of the multiplicative group $(\mathbb{Z}/4\mathbb{Z})^*$ onto $\mathbb{Z}/2\mathbb{Z}$; similarly, $\omega$ is a homomorphism of $(\mathbb{Z}/8\mathbb{Z})^*$ onto $\mathbb{Z}/2\mathbb{Z}$.]
