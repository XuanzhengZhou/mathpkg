Let $f = \sum c(n)q^n$ be a normalized eigenfunction of all Hecke operators with $c(1) = 1$. Then the coefficients satisfy:
\begin{align*}
c(mn) &= c(m)c(n) \quad \text{if } (m, n) = 1,\\
c(p)c(p^n) &= c(p^{n+1}) + p^{2k-1}c(p^{n-1}) \quad \text{if } p \text{ is prime}, n \geq 1.
\end{align*}
In particular, the function $n \mapsto c(n)$ is multiplicative.
