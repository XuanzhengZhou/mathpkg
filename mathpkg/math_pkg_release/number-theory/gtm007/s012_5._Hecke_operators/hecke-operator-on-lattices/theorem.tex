Let $\mathcal{R}$ be the set of lattices of $\mathbf{C}$. For an integer $n \geq 1$, the Hecke operator $T(n)$ is the correspondence on $\mathcal{R}$ which transforms a lattice to the sum (in $X_\mathcal{R}$) of its sublattices of index $n$:
$$T(n)\Gamma = \sum_{(\Gamma : \Gamma') = n} \Gamma' \quad \text{for } \Gamma \in \mathcal{R}.$$
The sum is finite: every sublattice $\Gamma'$ of index $n$ contains $n\Gamma$, and the number of such sublattices equals the number of subgroups of order $n$ in $\Gamma/n\Gamma \cong (\mathbf{Z}/n\mathbf{Z})^2$.
