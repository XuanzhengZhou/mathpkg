For a lattice $\Gamma \subset \mathbb{C}$, the Weierstrass $\wp$-function is
$$\wp_\Gamma(u) = \frac{1}{u^2} + \sum_{\gamma \in \Gamma, \gamma \neq 0} \left(\frac{1}{(u-\gamma)^2} - \frac{1}{\gamma^2}\right).$$
It satisfies $(\wp')^2 = 4\wp^3 - g_2 \wp - g_3$ where $g_2 = 60G_2(\Gamma)$, $g_3 = 140G_3(\Gamma)$.
