Let $m \geq 1$. The group $G(m) = (\mathbb{Z}/m\mathbb{Z})^*$ of invertible elements mod $m$ has order $\phi(m)$. An element $\chi$ of $\widehat{G(m)}$ is called a character modulo $m$. It is viewed as a function on integers prime to $m$, extended to $\mathbb{Z}$ by $\chi(a) = 0$ if $\gcd(a,m) \neq 1$.
