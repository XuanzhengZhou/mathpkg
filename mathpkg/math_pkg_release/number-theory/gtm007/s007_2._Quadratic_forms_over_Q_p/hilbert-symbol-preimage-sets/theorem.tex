Let $k = \mathbb{Q}_p$ and let $k^*/k^{*2}$ be the square-class group, which is an $\mathbf{F}_2$-vector space.

(a) The number of elements in $k^*/k^{*2}$ is $2^r$, where $r = 2$ if $p \neq 2$ and $r = 3$ if $p = 2$.

(b) For $a \in k^*/k^{*2}$ and $\varepsilon = \pm 1$, let $H_a^\varepsilon$ be the set of $x \in k^*/k^{*2}$ such that $(x, a) = \varepsilon$, where $(\cdot, \cdot)$ is the Hilbert symbol. Then:
- If $a = 1$, $H_a^1$ has $2^r$ elements and $H_a^{-1} = \varnothing$.
- If $a \neq 1$, $H_a^\varepsilon$ has $2^{r-1}$ elements.

(c) Let $a, a' \in k^*/k^{*2}$ and $\varepsilon, \varepsilon' = \pm 1$. If $H_a^\varepsilon = H_{a'}^{\varepsilon'}$ and this set is nonempty, then $a = a'$ and $\varepsilon = \varepsilon'$.
