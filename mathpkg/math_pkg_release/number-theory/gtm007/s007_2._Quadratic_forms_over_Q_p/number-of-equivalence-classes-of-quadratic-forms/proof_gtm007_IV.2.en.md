By Theorem 7, the equivalence class of a quadratic form of rank $n$ is determined by the triple $(n, d, arepsilon)$. By Proposition 6, the valid pairs $(d, arepsilon)$ for a given $n \geq 3$ are all of them (subject to the constraints for $n = 1, 2$ described in Proposition 6). The number of possibilities for $d$ (an element of $k^*/k^{*2}$) is $2^r$ where $r = 2$ if $p 
eq 2$ and $r = 3$ if $p = 2$ (Lemma at n° 2.2). For each $d$, $arepsilon$ can be $\pm 1$. However, certain combinations are ruled out when $n$ is small (Proposition 6), but for sufficiently large $n$ all $2^r 	imes 2 = 2^{r+1}$ combinations occur. The values stated follow from $2^{2+1} = 8$ (for $p = 2$) and $2^{1+1} = 4$ (for $p 
eq 2$). For the precise count at each rank $n$, one applies the restrictions of Proposition 6: when $n = 1$ there is exactly $1$ class; when $n = 2$ the case $d = -1, arepsilon = -1$ cannot occur; for $n \geq 3$ all possibilities occur. The stated numbers account for these adjustments.
