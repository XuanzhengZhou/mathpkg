Let $p$ be an odd prime, $q = p^f$, and let $\omega$ be the Teichmüller character on $\mathbf{F}_q$. For an integer $k \ge 0$, write the $p$-adic expansion
$$k = k_0 + k_1 p + k_2 p^2 + \cdots + k_{r} p^{r}, \qquad 0 \le k_i \le p-1,$$
and define the $p$-adic digit sum
$$s_p(k) = k_0 + k_1 + \cdots + k_r.$$
Extend $s_p(k)$ to all integers by $(q-1)$-periodicity.

Let $S(k) = \tau(\omega^{-k})$ be the Gauss sum associated to the character $\omega^{-k}$, and let $\mathfrak{P}$ be a prime ideal above $p$ in the cyclotomic field $\mathbf{Q}(\zeta_{q-1}, \zeta_p)$.

Then for any integer $k \ge 0$,
$$\frac{S(k)}{S(1)^{k}} \equiv \frac{1}{k!} \pmod{\mathfrak{P}}.$$
In particular,
$$\operatorname{ord}_{\mathfrak{P}} S(k) = \frac{s_p(k)}{p-1}.$$
