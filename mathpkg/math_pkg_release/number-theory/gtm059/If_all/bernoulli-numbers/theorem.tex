The \textbf{Bernoulli numbers} $B_k$ ($k = 0, 1, 2, \dots$) are defined by the generating function
$$F(t) = \frac{t}{e^t - 1} = \sum_{k=0}^{\infty} B_k \frac{t^k}{k!}.$$
The first few Bernoulli numbers are:
$$B_0 = 1, \quad B_1 = -\frac{1}{2}, \quad B_2 = \frac{1}{6}, \quad B_3 = 0, \quad B_4 = -\frac{1}{30}.$$
For odd $k > 1$, $B_k = 0$. The function $F(t)$ satisfies $F(-t) = F(t) + t$, so it is "almost even."
