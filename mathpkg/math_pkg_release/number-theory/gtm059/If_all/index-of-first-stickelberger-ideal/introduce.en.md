# Index of the First Stickelberger Ideal

This theorem computes the index of the Stickelberger ideal $\mathcal{P}$ inside the group ring of the Galois group of a cyclotomic field, assuming the conductor is a prime power. The index factors as a product of determinants, each of which is related to the Stickelberger elements. This result is essential for understanding the size of the Stickelberger module and its relationship to the class group of the cyclotomic field. In later chapters, this index is shown to be connected to the order of the minus part of the ideal class group.
