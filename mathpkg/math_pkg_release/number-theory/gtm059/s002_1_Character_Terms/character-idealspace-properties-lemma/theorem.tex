Let $m = p$ be prime and let $\chi$ be a character. Let $t_{\chi}$ be the $\chi$-eigenspace of the ideal generated by all elements $\sigma_a - \chi(a)$ in the group ring $\mathbb{Z}[G]$.

(i) If $\chi = \omega^k$ where $0 \leq k \leq 3$ and $\omega$ is the Teichmüller character, then $t_{\chi} = \mathfrak{p}(\chi)$.

(ii) If $\chi$ is non-trivial and not equal to the Teichmüller character, then $t_{\chi} = 1$.
