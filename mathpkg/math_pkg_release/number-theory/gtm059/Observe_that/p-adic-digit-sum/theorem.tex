Let $p$ be a prime. For an integer $k \geq 0$, write the $p$-adic expansion
$$k = k_0 + k_1 p + k_2 p^2 + \dots + k_t p^t,$$
with digits $0 \leq k_i \leq p-1$ for all $i$.

Define the $p$-adic digit sum $s(k)$ by
$$s(k) = k_0 + k_1 + k_2 + \dots + k_t.$$

Define the $p$-adic digit factorial product $\gamma(k)$ by
$$\gamma(k) = k_0! \, k_1! \, k_2! \, \cdots \, k_t!.$$

For an arbitrary integer $k$, extend the definitions by $(q-1)$-periodicity: write $k \equiv k' \pmod{q-1}$ with $0 \leq k' < q-1$, and set $s(k) = s(k')$ and $\gamma(k) = \gamma(k')$.
