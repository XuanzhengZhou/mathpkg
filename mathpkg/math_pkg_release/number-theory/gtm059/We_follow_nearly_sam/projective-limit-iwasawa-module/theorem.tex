Let $\Gamma \cong \mathbb{Z}_p$ and for each $n \geq 1$, let $\Gamma_n = \Gamma/\Gamma^{p^n}$ be the cyclic quotient of order $p^n$. Let $\{V_n\}$ be an inverse system of modules, where each $V_n$ is a module over the group ring $\mathbb{Z}_p[\Gamma_n]$, with the transition maps $V_{n+1} \to V_n$ compatible with the natural projections $\mathbb{Z}_p[\Gamma_{n+1}] \to \mathbb{Z}_p[\Gamma_n]$. The projective limit

$$V = \varprojlim_n V_n$$

is then naturally a module over the Iwasawa algebra $\Lambda = \mathbb{Z}_p[[X]]$, called a \textbf{projective limit module}.

In the application to ideal class groups, $V_n = C_n$ is the $p$-primary part of the ideal class group of the $n$-th layer of the $\mathbb{Z}_p$-extension, and the projective limit $C = \varprojlim C_n$ is a $\Lambda$-module.
