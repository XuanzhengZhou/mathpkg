Let $K$ be a totally imaginary abelian extension of $\mathbb{Q}$ (a CM field) and let $K^+$ be its maximal real subfield. Let

$$N = [K : \mathbb{Q}], \quad N^+ = [K^+ : \mathbb{Q}] = N/2,$$
$$h = \text{class number of } K, \quad h^+ = \text{class number of } K^+,$$
$$R = \text{regulator of } K, \quad R^+ = \text{regulator of } K^+,$$
$$Q = \left( E : \mu_K E^+ \right) = \text{unit index}.$$

Then the class number $h$ decomposes as

$$h = h^+ \cdot Q \cdot \frac{R}{R^+} \cdot \frac{2^{N/2}}{w}$$

where $w$ is the number of roots of unity in $K$. Equivalently, reading the analytic class number formula for the real subfield $K^+$ and comparing with the formula for $K$, one obtains

$$\frac{h}{h^+} = \frac{Q}{2^{N/2}} \cdot \frac{R}{R^+} \cdot w.$$

The factor $h/h^+$ is called the \textbf{first factor} (or relative class number), while $h^+$ is the \textbf{second factor}. In terms of $L$-series values,

$$\frac{h}{h^+} = Q \cdot w \cdot \frac{\prod_{\chi \text{ odd}} L(1, \chi)}{2^{N/2}}.$$
