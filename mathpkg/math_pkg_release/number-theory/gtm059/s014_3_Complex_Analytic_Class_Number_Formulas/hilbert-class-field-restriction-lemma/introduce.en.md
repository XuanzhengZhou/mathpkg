This lemma describes how the Artin symbol behaves under restriction to the Hilbert class field. For an abelian extension $K/F$, the Artin symbol $(c, K/F)$ for an ideal class $c \in C_F$ restricts to the Hilbert class field $H$ as $(c, H/F)$. More generally, for an ideal class in $C_K$, the restriction of the Artin symbol to $H$ is given by the Artin symbol of the norm. This compatibility is a fundamental tool in class field theory and is used to prove the injectivity of the norm map on the minus eigenspace of the class group.
