Let $p$ be a prime and let $\chi$ be a Dirichlet character mod $p$. Let $B_{1,\chi}$ denote the first generalized Bernoulli number associated to $\chi$.

\begin{enumerate}
\item If $\chi$ is not equal to the Teichm\"uller character $\omega$, then $B_{1,\chi}$ is $p$-integral.
\item If $\chi = \omega$, then $B_{1,\omega} \equiv \frac{1}{p} \pmod{\mathbb{Z}_p}$, i.e., $p B_{1,\omega} \equiv 1 \pmod{p\mathbb{Z}_p}$.
\end{enumerate}
