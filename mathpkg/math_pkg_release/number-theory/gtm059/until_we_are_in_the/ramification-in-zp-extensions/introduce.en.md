This lemma describes the ramification behavior in a $\mathbb{Z}_p$-extension. Despite being an infinite extension, only finitely many primes can ramify, and each ramified prime eventually becomes totally ramified. This property is crucial for Iwasawa theory because it allows one to control the behavior of ideal class groups across the tower of fields using the finiteness of ramification to apply class field theory.
