Let $K_\infty / K$ be a $\mathbb{Z}_p$-extension with intermediate fields $K_n$ ($[K_n : K] = p^n$). We say that **condition IW** (Iwasawa's condition) is satisfied if the following holds:

There are only finitely many primes of $K$ that ramify in $K_\infty / K$, and all ramified primes are totally ramified.

Equivalently, if $S$ denotes the (finite) set of primes of $K$ that ramify in $K_\infty$, then each $\mathfrak{p} \in S$ is totally ramified in $K_n/K$ for all sufficiently large $n$, and no prime outside $S$ ramifies at any level.

We say condition IW is satisfied **with $s$ primes** if exactly $s$ prime ideals of $K$ ramify in the $\mathbb{Z}_p$-extension.
