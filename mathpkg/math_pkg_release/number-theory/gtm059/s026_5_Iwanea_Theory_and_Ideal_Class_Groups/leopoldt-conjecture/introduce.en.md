Leopoldt's conjecture is a fundamental open problem (for general number fields) in algebraic number theory. It asserts that the $p$-adic rank of the unit group equals $r_1 + r_2 - 1$, the same as the ordinary rank given by Dirichlet's unit theorem. The conjecture is known to hold for abelian extensions of $\mathbb{Q}$ (proved by Brumer using Baker's theory of linear forms in logarithms), but remains open for general number fields. Its truth implies that $G_M(K)$ has $\mathbb{Z}_p$-rank exactly $r_2 + 1$, which is a crucial input for the Main Conjecture of Iwasawa theory.
