Let $a$ be a residue mod $p-1$. There exists a unique continuous linear map

$$\Gamma_a: \mathcal{L} \to C(\mathbb{Z}_p, \mathbb{C}_p)$$

satisfying any one of the following three equivalent conditions:

$$\Gamma_a(1) = 1, \quad \Gamma_a\left(\frac{X}{n}\right)(k) = \frac{k^a}{n},$$

and for $f(X) = \sum a_n X^n$, the transform is given by

$$\Gamma_a(f)(k) = \sum a_n \cdot \frac{k^{a+n}}{n!}.$$

This map also satisfies the norm inequality $\|\Gamma_a(f)\| \le \|f\|_p$.
