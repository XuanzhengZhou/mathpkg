Theorem 3.3 provides an explicit finite-sum formula for the Mellin transform $M_L(z_k)$ when the input function $g$ belongs to $L^2$ and the associated power series $h$ satisfies $Dh = g$ with initial condition $h(0) = 0$. The result expresses $M_L(z_k)$ as a simple arithmetic sum involving the values $z_k$. This bridges the analytic definition of the Mellin transform with elementary computational formulas that can be evaluated at integer points.
