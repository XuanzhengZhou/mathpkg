\text{Let } m \text{ be the conductor of the cyclotomic field } \mathbb{Q}(\zeta_m), \text{ and let } R \text{ be a ring containing a root of unity.} \text{ For } a \text{ prime to } m \text{, a cyclotomic unit is defined by}
$$g_a = \frac{\zeta^a - 1}{\zeta - 1},$$
\text{where } \zeta \text{ is a primitive } m\text{-th root of unity. The group } C \text{ of cyclotomic units in } \mathbb{Q}(\zeta_m) \text{ is generated by all such } g_a \text{ with } (a,m)=1. \text{ Up to sign, these units are real when } a \text{ is taken such that } \zeta^a \text{ is fixed under complex conjugation.}
