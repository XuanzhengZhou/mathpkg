The class number $h$ of a cyclotomic field $K$ decomposes into a product $h = h^+ \cdot h^-$, where $h^+$ is the class number of the maximal real subfield (the plus part, or first factor) and $h^- = h/h^+$ is the relative class number (the minus part, or second factor). This decomposition reflects the action of complex conjugation on the ideal class group and is fundamental to the analytic class number formula for cyclotomic fields.
