This lemma is the key inductive step in the proof of Theorem 3.2 (the matrix normal form for relation matrices over the Iwasawa algebra). It states that if a relation matrix is already partially diagonalized (in $(r-1)$-normal form), then one can extend the diagonalization to the $r$-th row using admissible operations, without disturbing the previously diagonalized rows.
