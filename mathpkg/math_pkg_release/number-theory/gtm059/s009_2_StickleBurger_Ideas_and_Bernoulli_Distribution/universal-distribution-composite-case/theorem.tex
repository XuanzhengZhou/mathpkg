Let $N = \prod p_i^{e_i}$ be composite with partial fraction decomposition:

$$\frac{1}{N}\mathbb{Z}/\mathbb{Z} \cong \bigoplus_i \frac{1}{p_i^{e_i}}\mathbb{Z}/\mathbb{Z}.$$

For each prime power $p_i^{e_i} \mid N$, let $T_{p_i^{e_i}}$ be the set of primitive
elements (exact period $p_i^{e_i}$). Define $T_N = \bigcup_i T_{p_i^{e_i}}$.
Then the universal distribution theorem (Theorem 9.1) holds with $T_N$ in place of the
set of primitive elements, subject to the distribution relations among the components.
