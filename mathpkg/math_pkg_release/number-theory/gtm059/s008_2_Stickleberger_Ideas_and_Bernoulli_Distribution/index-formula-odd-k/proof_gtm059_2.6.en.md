We have $R = 2r \cap R$ and $(r : R) = 2^{n+1}$ (by Lemma 1 of this section). Proceeding as in the even case, we write

$$
(R : R) = \left(\frac{r : R}{r : R}\right) B_{n,i}
$$

and then

$$
(r : R : x : R) = (r : R : x : R) B_{n,i} B_{n,i} B_{n,i} = \prod_{i=1}^{n} B_{n,i} (R : r : R),
$$

because $R$ is invertible in $R[G]$. Furthermore, $(r : R : x : R) = (r : 2r : 2r)$, and since $(2r : 1 : r : 1) = 2^{n+1}$ with $i = 1$ free of rank $\phi(N)$, the result follows.
