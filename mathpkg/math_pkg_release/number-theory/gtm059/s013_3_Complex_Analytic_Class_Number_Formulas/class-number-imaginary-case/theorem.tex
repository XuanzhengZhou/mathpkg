Let $K$ be an imaginary abelian extension of $\mathbb{Q}$ and let $K^+$ be its maximal real subfield. Let $h, h^+$ denote the class numbers, $R, R^+$ the regulators, and let $Q = (E : W \cdot E^+)$ be the unit index of $K$. Let $N = [K : \mathbb{Q}]$ and $N^+ = [K^+ : \mathbb{Q}] = N/2$. Then:

$$h = h^+ \cdot Q \cdot w \cdot \prod_{\chi \text{ odd}} \left( -\frac{1}{2} B_{1,\chi} \right)$$

where the product runs over all odd primitive Dirichlet characters $\chi$ of the Galois group $G = \operatorname{Gal}(K/\mathbb{Q})$, and $B_{1,\chi}$ denotes the first generalized Bernoulli number attached to $\chi$.

Equivalently, using the factorization $h = h^+ \cdot h^-$, the relative class number $h^-$ is given by:

$$h^- = Q \cdot w \cdot \prod_{\chi \text{ odd}} \left( -\frac{1}{2} B_{1,\chi} \right).$$

For cyclotomic fields $K = \mathbb{Q}(\zeta_m)$, this formula provides an explicit computation of $h$ from the Bernoulli numbers, which are rational numbers. The class number of the real subfield $h^+$ is generally much more difficult to compute.
