Let $K$ be an abelian extension of $\mathbb{Q}$ with Galois group $G$. Let
$$r_1 = \text{number of real embeddings of } K,$$
$$r_2 = \text{number of pairs of complex conjugate embeddings},$$
$$N = [K : \mathbb{Q}] = r_1 + 2r_2,$$
$$w = w_K = \text{number of roots of unity in } K,$$
$$h = h_K = \text{class number of } K,$$
$$R = R_K = \text{regulator of } K,$$
$$d = d_K = \text{discriminant of } K.$$

Then the analytic class number formula states:
$$\frac{2^{r_1} (2\pi)^{r_2} h R}{w \sqrt{|d|}} = \prod_{\chi \neq 1} L(1, \chi)$$
where the product is taken over all non-trivial primitive Dirichlet characters $\chi$ of $G$.

If $K$ is real (so $r_2 = 0$), then $w = 2$ and the formula reads:
$$\frac{2^{r_1} h R}{w \sqrt{|d|}} = \prod_{\chi \neq 1} L(1, \chi).$$

If $K$ is imaginary, then the product factors further via the maximal real subfield $K^+$.
