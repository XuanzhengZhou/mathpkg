The \textit{Fermat curve} (of exponent $n$) is the projective algebraic curve defined by the homogeneous equation
$$X^n + Y^n = Z^n$$
over a given field. More generally, one also considers diagonal hypersurfaces
$$a_1 X_1^n + a_2 X_2^n + \cdots + a_k X_k^n = 0$$
with coefficients $a_i$ in the base field.
