In the study of ideal class groups of cyclotomic fields, the decomposition into $(\pm 1)$-eigenspaces under complex conjugation is fundamental. Theorem 4.2 of Lang's Cyclotomic Fields states that for $K = \mathbb{Q}(\zeta_p)$, the natural inclusion of the minus eigenspace $C_K^-$ into $C_K$ is injective. This means the norm map $N_{K/K^+}$ does not collapse any non-trivial elements of $C_K^-$. The result is a key ingredient in relating the minus part of the class number to Bernoulli numbers via the analytic class number formula.
