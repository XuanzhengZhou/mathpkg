There exists a unique function $\log_p: \Omega^\times \to \Omega$ such that: (1) $\log_p x$ agrees with the power series definition for $|x-1|_p < 1$; (2) $\log_p(xy) = \log_p x + \log_p y$ for all $x,y \in \Omega^\times$; (3) $\log_p p = 0$. The extension is locally analytic on $\Omega \setminus \{0\}$.
