The $p$-adic gamma function $\Gamma_p: \mathbb{Z}_p \to \mathbb{Z}_p^\times$ satisfies:
(1) $\Gamma_p(s+1) = -s\Gamma_p(s)$ if $s \in \mathbb{Z}_p^\times$, and $\Gamma_p(s+1) = -\Gamma_p(s)$ if $s \in p\mathbb{Z}_p$;
(2) Reflection formula: $\Gamma_p(s)\Gamma_p(1-s) = (-1)^{s_0}$, where $s_0$ is the first digit of $s$ (with $s_0 = p$ if $s \in p\mathbb{Z}_p$);
(3) Multiplication formula: for $m$ not divisible by $p$, $\prod_{h=0}^{m-1} \Gamma_p((s+h)/m) / (\Gamma_p(s)\prod_{h=1}^{m-1}\Gamma_p(h/m)) = m^{1-s_0}(m^{-(p-1)})^{s_1}$.
