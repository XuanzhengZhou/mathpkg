For a smooth projective hypersurface $\tilde{H}_f$ of dimension $n-1$ in $\mathbb{P}^n$ over $\mathbb{F}_q$, the Weil Conjectures assert: (i) Rationality: $Z(\tilde{H}_f/\mathbb{F}_q; T) = P(T)^{\pm 1} / ((1-T)(1-qT)\cdots(1-q^{n-1}T))$ where $P(T) \in 1+T\mathbb{Z}[T]$ has degree $\beta$ (the Betti number); (ii) Functional equation: if $\alpha$ is a reciprocal root of $P(T)$, so is $q^{n-1}/\alpha$; (iii) Riemann Hypothesis: the complex absolute value of each reciprocal root of $P(T)$ is $q^{(n-1)/2}$.
