A norm on a field is a nonnegative real-valued function satisfying positivity, multiplicativity, and the triangle inequality. Norms induce metrics via $d(x,y) = \|x-y\|$. The classical absolute value on $\mathbb{R}$ is an Archimedean norm, while the $p$-adic absolute value introduced later is a non-Archimedean norm — a fundamental distinction that underpins all of $p$-adic analysis.
