The Möbius function is a fundamental arithmetic function used in the Möbius inversion formula. In $p$-adic analysis, it appears in the Artin-Hasse exponential via the formal identity $\exp(X) = \prod_{n=1}^\infty (1-X^n)^{-\mu(n)/n}$, which allows one to express the exponential as an infinite product where most factors have $p$-adic integral coefficients.
