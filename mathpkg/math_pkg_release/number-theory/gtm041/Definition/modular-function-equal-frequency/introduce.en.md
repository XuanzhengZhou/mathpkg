Theorem 2.5 of Apostol states that every non-constant modular function attains each complex value the same number of times in the closure of the fundamental region $R_{\Gamma}$. This is an immediate consequence of Theorem 2.4 applied to the function $f - c$, which is also modular and meromorphic. This result reveals a striking regularity in the value distribution of modular functions and is the key step toward proving that every modular function can be expressed as a rational function of the $j$-invariant.
