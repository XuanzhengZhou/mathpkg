Theorem 6.8 provides an explicit parametrization of all equivalence classes in $\Gamma(n)$. Each class is represented by an upper triangular matrix $\begin{pmatrix} a & b \\ 0 & d \end{pmatrix}$ where $ad = n$, $d > 0$, and $b$ ranges over a complete residue system modulo $d$. This set of representatives is precisely the collection of matrices used in the definition of the Hecke operator $T_n$, thus connecting the algebraic structure of $\Gamma(n)$ to the theory of modular forms.
