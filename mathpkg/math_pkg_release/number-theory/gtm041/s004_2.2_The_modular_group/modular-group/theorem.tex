The \emph{modular group} $\Gamma$ is the set of all M\"obius transformations of the form
$$\tau' = \frac{a\tau + b}{c\tau + d},$$
where $a, b, c, d$ are integers with $ad - bc = 1$.
The group can be represented by $2 \times 2$ integer matrices
$$A = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \quad \text{with} \quad \det A = 1,$$
provided we identify each matrix with its negative, since $A$ and $-A$ represent the same transformation.
If $A = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$ we write
$$A\tau = \frac{a\tau + b}{c\tau + d}.$$
