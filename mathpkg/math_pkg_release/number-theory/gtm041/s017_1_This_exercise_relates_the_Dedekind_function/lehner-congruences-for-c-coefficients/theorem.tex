Let $c(n)$ denote the coefficients in the Fourier expansion of the modular function $j(\tau)$:
$$j(\tau) = \sum_{n=-1}^{\infty} c(n) e^{2\pi i n \tau}.$$
For $\alpha \geq 1$, the following congruences hold:
$$c(2^\alpha n) \equiv 0 \pmod{2^{3\alpha+8}},$$
$$c(3^\alpha n) \equiv 0 \pmod{3^{2\alpha+3}},$$
$$c(5^\alpha n) \equiv 0 \pmod{5^{\alpha+1}},$$
$$c(7^\alpha n) \equiv 0 \pmod{7^\alpha}.$$
