The univalent function $\Phi(\tau)$ is constructed from the ratio $\varphi(\tau)=\Delta(q\tau)/\Delta(\tau)$ by taking the $(q-1)$-th root, so that $\Phi$ has valence one. Using the Dedekind eta function, $\Phi(\tau)=(\eta(q\tau)/\eta(\tau))^r$ with $r=24/(q-1)$ whenever $r$ is an integer. For the primes $q=2,3,5,7,13$, the function $\Phi$ turns out to be automorphic under the congruence subgroup $\Gamma_0(q)$, making it a canonical univalent modular function for these levels. Its expansion at the cusp $\infty$ has the simple form $1/\Phi(\tau)=x^{-1}+I(x)$ with integer-coefficient power series $I(x)$, which is exploited in later sections to express $j_p$ as a polynomial in $\Phi$.
