Given a modular form $f(\tau)$ with Fourier expansion
$$f(\tau) = c(0) + \sum_{n=1}^{\infty} c(n)e^{2\pi i n\tau},$$
the associated Dirichlet series $\varphi(s)$ is defined by
$$\varphi(s) = \sum_{n=1}^{\infty} \frac{c(n)}{n^s},$$
formed with the same coefficients (except for $c(0)$).

If $f \in M_{2k}$, then:
\begin{itemize}
\item $c(n) = O(n^k)$ if $f$ is a cusp form, and $c(n) = O(n^{2k-1})$ if $f$ is not a cusp form.
\item Consequently, $\varphi(s)$ converges absolutely for $\sigma = \operatorname{Re}(s) > k + 1$ if $f$ is a cusp form, and for $\sigma > 2k$ if $f$ is not a cusp form.
\end{itemize}
