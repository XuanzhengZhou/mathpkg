The discriminant $\Delta(\tau)$ and the Eisenstein series $G_{2k}(\tau)$ satisfy the modular transformation law
$$f\left(\frac{a\tau + b}{c\tau + d}\right) = (c\tau + d)^r f(\tau),$$
where $r$ is an integer (the weight) and $\begin{pmatrix} a & b \\ c & d \end{pmatrix}$ is any element of the modular group $\Gamma$. The function $\Delta$ satisfies this with $r = 12$, and $G_{2k}$ satisfies this with $r = 2k$.
