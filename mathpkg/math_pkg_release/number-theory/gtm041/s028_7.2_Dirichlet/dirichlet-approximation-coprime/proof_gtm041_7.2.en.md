By Theorem 7.1 there is a pair $h', k'$ with $0 < k' \leq N$ satisfying

$$\left| \theta - \frac{h'}{k'} \right| < \frac{1}{Nk'}.$$

Let $d = (h', k')$. If $d = 1$ there is nothing to prove. If $d > 1$ write $h' = hd$, $k' = kd$, where $(h, k) = 1$ and $k < k' \leq N$. Then $1/k' < 1/k$ and the inequality becomes

$$\left| \theta - \frac{h}{k} \right| < \frac{1}{Nk'} < \frac{1}{Nk},$$

from which we find $|k\theta - h| < 1/N$.
