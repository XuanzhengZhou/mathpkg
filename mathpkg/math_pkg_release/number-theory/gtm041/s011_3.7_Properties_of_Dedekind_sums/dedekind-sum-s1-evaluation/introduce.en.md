When $h=1$, the Dedekind sum $s(1,k)$ admits a simple closed-form evaluation in terms of $k$ alone. This special value is obtained by direct computation from the definition using the elementary sums $\sum r = k(k-1)/2$ and $\sum r^2 = (k-1)k(2k-1)/6$, yielding a rational function that will be useful in later congruence calculations.
