$$
((x)) = \begin{cases}
x - [x] - \frac{1}{2} & \text{if } x \text{ is not an integer,} \\
0 & \text{if } x \text{ is an integer.}
\end{cases}
$$
This is a periodic function of $x$ with period $1$, and $((-x)) = -((x))$. The function $((x))$ is the same as the Bernoulli periodic function $\bar{B}_1(x)$.
