This lemma provides the crucial link between the partition generating function $F(x)$ and the Dedekind eta function $\eta(\tau)$, describing the behavior of $F$ near each root of unity singularity. The transformation formula is derived from the modular functional equation of $\eta(\tau)$. In the circle method, this formula allows $F$ to be replaced by an elementary function $\psi_{h,k}$ on each Farey arc, with the error term amenable to estimation.
