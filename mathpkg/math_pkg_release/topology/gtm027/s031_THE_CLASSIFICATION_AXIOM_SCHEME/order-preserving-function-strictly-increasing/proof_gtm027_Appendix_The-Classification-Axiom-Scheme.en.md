# Proof of Order-Preserving Functions are Strictly Increasing

PROOF It must be shown that $\{u : u \in x \text{ and } f(u) r u\}$ is void. If not there is an $r$-

proof If $f(u) = f(v)$, then it is impossible that $u r v$ or $v r u$, for in this case $f(u) s f(v)$ or $f(v) s f(u)$. Hence $u = v$ and $f$ is 1-1. If $f(u) s f(v)$, then $u \neq v$, and if $v r u$, then $f(v) s f(u)$, which is a contradiction. Therefore $f^{-1}$ is $s-r$ order preserving.
