# Proof of Restriction of a Function

126 Theorem If $f$ is a function, $f|x$ is a function whose domain is $x \cap (\text{domain } f)$ and $(f|x)(y) = f(y)$ for each $y$ in domain $f|x$.The final theorem on ordinals asserts that (intuitively) it is possible to define a function on an ordinal so that its value at any member of its domain is given by applying a predetermined rule to the earlier values of the function. A little more precisely, given $g$ it is possible to find a unique function $f$ on an ordinal such that $f(x) = g(f|x)$ for each ordinal number $x$. The value $f(x)$ is then completely determined by $g$ and the values of $f$ at ordinal numbers preceding $x$. Application of this theorem is defining a function by transfinite induction. The proof is similar to that of theorem 99 and the same sort of preliminary lemma is needed.
