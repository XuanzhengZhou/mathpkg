The Well-Ordering Theorem, due to Zermelo, states that every set can be well-ordered. The proof uses transfinite induction with the Axiom of Choice: a choice function is used to construct, step by step, an enumeration of the set by ordinals. The process must terminate before exhausting the class of all ordinals (which is not a set), yielding a bijection between the set and some ordinal. This theorem is equivalent to the Axiom of Choice over ZF set theory.
