# Proof of First and Second Coordinates of an Ordered Pair

54 THEOREM If $x$ and $y$ are sets $1^{\text{st}}$ coord $(x,y) = x$ and $2^{\text{nd}}$ coord $(x,y) = y$. If either56 DEFINITION $r$ is a relation if and only if for each member $z$ of $r$ there is $x$ and $y$ such that $z = (x,y)$.A relation is a class whose members are ordered pairs.57 DEFINITION $r \circ s = \{u:$ for some $x$, some $y$ and some $z$, $u = (x,z)$, $(x,y) \in s$ and $(y,z) \in r\}$.The class $r \circ s$ is the composition of $r$ and $s$.To avoid excessive notation we agree that $\{(x,z): \cdots\}$ is to be identical with $\{u:$ for some $x$, some $z$, $u = (x,z)$ and $\cdots\}$. Thus $r \circ s = \{(x,z):$ for some $y$, $(x,y) \in s$ and $(y,z) \in r\}$.
