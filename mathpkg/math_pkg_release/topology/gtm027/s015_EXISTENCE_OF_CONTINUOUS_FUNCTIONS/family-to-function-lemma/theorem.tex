Suppose that for each member $t$ of a dense subset $D$ of the positive reals, $F_t$ is a subset of a set $X$ such that
\begin{enumerate}
\item[(a)] if $t < s$, then $F_t \subset F_s$;
\item[(b)] $\bigcup \{F_t : t \in D\} = X$.
\end{enumerate}
For $x \in X$ let $f(x) = \inf \{t : x \in F_t\}$. Then
$$\{x : f(x) < s\} = \bigcup \{F_t : t \in D \text{ and } t < s\}$$
and
$$\{x : f(x) \leq s\} = \bigcap \{F_t : t \in D \text{ and } t > s\}$$
for each real number $s$.
