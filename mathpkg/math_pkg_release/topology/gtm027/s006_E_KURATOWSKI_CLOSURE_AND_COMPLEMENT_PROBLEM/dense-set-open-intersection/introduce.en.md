A basic but useful fact about dense sets: if $A$ is a dense subset of a topological space $X$, then for any open set $U$, the closure of the intersection $A \cap U$ contains $U$ itself. In other words, every point of an open set $U$ can be approximated by points of $A$ that also lie in $U$. This property is frequently used in arguments involving density and the behavior of open sets under relativization.
