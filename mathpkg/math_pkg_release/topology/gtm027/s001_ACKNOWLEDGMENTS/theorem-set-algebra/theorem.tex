Let $A, B, C$, and $X$ be sets. Then:

(a) $X \sim (X \sim A) = A \cap X$.

(b) (Commutative laws) $A \cup B = B \cup A$ and $A \cap B = B \cap A$.

(c) (Associative laws) $A \cup (B \cup C) = (A \cup B) \cup C$ and $A \cap (B \cap C) = (A \cap B) \cap C$.

(d) (Distributive laws) $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$ and $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$.

(e) (De Morgan formulae) $X \sim (A \cup B) = (X \sim A) \cap (X \sim B)$ and $X \sim (A \cap B) = (X \sim A) \cup (X \sim B)$.
