The Zermelo Postulate states that for any disjoint family of non-empty sets, there exists a set $C$ (a "transversal" or "system of distinct representatives") that picks exactly one element from each member. It follows immediately from the Axiom of Choice applied to the index set $\alpha$ with $X_A = A$ for each $A \in \alpha$, and the choice function's range is the desired set $C$.
