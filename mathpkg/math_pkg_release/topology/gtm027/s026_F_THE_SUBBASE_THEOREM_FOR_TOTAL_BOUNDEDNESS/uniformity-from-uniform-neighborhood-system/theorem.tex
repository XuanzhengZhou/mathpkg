\text{If } (V, \geq) \text{ is a uniform neighborhood system for } X \text{, then the family of all sets of the form } \{(x,y) : y \in V_a(x)\} \text{, for } a \text{ an arbitrary member of the directed set } A \text{, is a base for a uniformity } \mathfrak{u} \text{ on } X \text{. This uniformity, called the uniformity of the system, satisfies:}
\end{proposition}

\begin{enumerate}
\item \text{For each } a \in A \text{, there is } U \in \mathfrak{u} \text{ such that } U[x] \subset V_a(x) \text{ for all } x \in X
\item \text{For each } U \in \mathfrak{u} \text{, there is } a \in A \text{ such that } V_a(x) \subset U[x] \text{ for all } x \in X
\end{enumerate}
