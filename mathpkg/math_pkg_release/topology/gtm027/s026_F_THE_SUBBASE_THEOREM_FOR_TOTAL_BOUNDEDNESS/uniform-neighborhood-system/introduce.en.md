# Uniform Neighborhood System

A uniform neighborhood system provides an alternative axiomatization of uniform spaces using indexed families of neighborhoods rather than entourages (subsets of the product). Given a directed set of indices, each index assigns to every point a neighborhood satisfying axioms analogous to the entourage axioms: each point belongs to its own neighborhood, symmetry, and a "triangle inequality" condition. This approach, due to Weil, yields a theory equivalent to that of uniform spaces, as demonstrated by the propositions that follow.
