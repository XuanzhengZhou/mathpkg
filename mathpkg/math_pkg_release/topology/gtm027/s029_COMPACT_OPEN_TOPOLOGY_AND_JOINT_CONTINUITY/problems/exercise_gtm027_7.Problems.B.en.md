# Exercise B — Convergence of Functions

Let $f$ be a continuous real-valued function on the closed unit interval $[0,1]$ such that $f(0) = f(1) = 0$ and $f$ is not identically zero. Let $g_n(x) = f(x^n)$ for each non-negative integer $n$. Then $\{g_n, n \in \omega\}$ converges pointwise (but not uniformly) to the function $h$ which is identically zero. The union of $\{h\}$ and the set of all $g_n$ is compact relative to the pointwise topology but is not compact relative to the topology of uniform convergence.
