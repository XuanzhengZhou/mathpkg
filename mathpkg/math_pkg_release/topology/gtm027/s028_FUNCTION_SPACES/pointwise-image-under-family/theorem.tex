Let $F$ be a family of functions, each on a set $X$ to a topological space $Y$, and let $A \subset X$. The \textbf{pointwise image of $A$ under $F$} is the set
$$F[A] = \{f(x) : x \in A,\; f \in F\} \subset Y.$$

For a singleton $\{x\}$, the abbreviation
$$F[x] = F[\{x\}] = \{f(x) : f \in F\}$$
is used. If $e_x : F \to Y$ is the evaluation map at $x$, then $e_x[F] = F[x]$.
