An upper semi-continuous function is a real-valued function for which all superlevel sets are closed. This is a weakening of ordinary continuity: a function is continuous if and only if it is both upper and lower semi-continuous. Upper semi-continuity plays an important role in optimization, potential theory, and the study of semicontinuous regularizations.
