# Exercise F.3 — Upper Semi-Continuous Regularization of a Bounded Function

If $f$ is a bounded real-valued function on $X$, then there is a smallest upper semi-continuous function $f^-$ such that $f^- \geq f$. If $\mathcal{V}$ is the family of neighborhoods of a point $x$ and $S_V = \sup \{f(y): y \in V\}$, then

$$f^-(x) = \lim \{S_V, V \in \mathcal{V}, \subseteq\}.$$
