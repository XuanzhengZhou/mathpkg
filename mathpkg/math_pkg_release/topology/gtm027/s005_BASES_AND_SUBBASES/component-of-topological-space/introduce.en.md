A component of a topological space is a maximal connected subset. Components partition the space: every point belongs to exactly one component. By Zorn's lemma or via the construction using the union of all connected sets containing a given point, each connected set is contained in a unique component. Components are always closed sets (Theorem 22). In totally disconnected spaces, such as the rational numbers, each component consists of a single point.
