Let $(Y, \mathfrak{u})$ be a subspace of $(X, \mathfrak{J})$ and let $A$ be a subset of $Y$. Then:
\begin{itemize}
\item $A$ is $\mathfrak{u}$-closed iff $A = Y \cap F$ for some $\mathfrak{J}$-closed set $F$.
\item The $\mathfrak{u}$-closure of $A$ equals $Y \cap A^{-}$, where $A^{-}$ is the $\mathfrak{J}$-closure of $A$.
\item If $Y$ is open in $X$, then each set open in $Y$ is also open in $X$.
\item If $Y$ is closed in $X$, then each set closed in $Y$ is also closed in $X$.
\end{itemize}
