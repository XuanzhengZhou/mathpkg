A point $s$ is a cluster point of a net if and only if $s$ lies in the closure of every tail set $A_n = \{S_m : m \geq n\}$. This theorem provides a convenient topological characterization of cluster points using the closure operator, linking the net-theoretic notion of "frequently in" with the topological notion of adherence. It is a fundamental tool for working with convergence of nets in general topological spaces.
