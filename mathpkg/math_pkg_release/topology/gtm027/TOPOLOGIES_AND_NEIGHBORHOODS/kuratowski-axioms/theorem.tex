A \textbf{closure operator} on $X$ is an operator which assigns to each subset $A$ of $X$ a subset $A^{c}$ of $X$ such that the following four statements, the Kuratowski closure axioms, are true:
\begin{enumerate}
\item[(a)] If $\emptyset$ is the void set, $\emptyset^{c} = \emptyset$.
\item[(b)] For each $A$, $A \subset A^{c}$.
\item[(c)] For each $A$, $A^{cc} = A^{c}$.
\item[(d)] For each $A$ and $B$, $(A \cup B)^{c} = A^{c} \cup B^{c}$.
\end{enumerate}
