# Proof: Generation of Group Uniformities by Invariant Pseudo-Metrics

Let $(G, \mathcal{J})$ be a topological group.

A pseudo-metric $p$ on $G$ is *left-invariant* if $p(ax, ay) = p(x, y)$ for all $a, x, y \in G$. It is *right-invariant* if $p(xa, ya) = p(x, y)$ for all $a, x, y \in G$.

Consider the left uniformity $\mathcal{L}$. A base for $\mathcal{L}$ consists of sets $U_L = \{(x, y) : x^{-1}y \in U\}$ where $U$ is a neighborhood of $e$. For any left-invariant continuous pseudo-metric $p$ on $G$, the set $\{(x, y) : p(x, y) < \varepsilon\}$ is an entourage, and this entourage is left-invariant: if $(x, y)$ belongs to it, then $p(ax, ay) = p(x, y) < \varepsilon$.

Conversely, for each neighborhood $U$ of $e$, one constructs (by 6.N) a left-invariant pseudo-metric $p_U$ continuous on $G \times G$ such that $\{(x, y) : p_U(x, y) < 1\} \subset U_L$. The family of all such pseudo-metrics generates $\mathcal{L}$.

The same argument applied to right translations shows that $\mathcal{R}$ is generated by right-invariant pseudo-metrics.

(See 6.N for the construction of continuous pseudo-metrics on uniform spaces from sequences of entourages.)
