Let $U$ be a connected open set in $\mathbf{R}^3$, and let $Q \in \overline{U}$. Suppose that for each open set $V$ containing $Q$ there is an open set $W$ such that
\begin{enumerate}
\item $Q \in W \subset V$,
\item every closed path in $W \cap U$ is contractible in $V \cap U$.
\end{enumerate}
Then $U$ is \textit{locally simply connected} at $Q$.
