The first part (connectedness of $X$) is trivial: delete from $X$ all components of $X$ except the one which contains $K'$. (Recall that $K$ is connected.)

For the second part, add to $X$ all components of $\mathbf{R}^3 - X$ except the unbounded component (which contains $\operatorname{Bd} N'$). Let $X'$ be the result. Then $\operatorname{Bd} X'$ separates $K'$ from $\operatorname{Bd} N'$ in $\mathbf{R}^3$. Therefore some component $B$ of $\operatorname{Bd} X'$ has the same property (Theorem 30.2). Since $B$ is a compact connected polyhedral 2-manifold in $\mathbf{R}^3$, the set $\mathbf{R}^3 - B$ has only one bounded component, and the latter contains $K'$ (Theorem 26.6). Therefore $B$ is all of $\operatorname{Bd} X'$, as desired.
