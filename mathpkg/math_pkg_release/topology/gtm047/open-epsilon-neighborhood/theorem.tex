In a metric space $[X, d]$, for each $P$ in $X$ and each $\varepsilon > 0$, we define the (open) $\varepsilon$-neighborhood of $P$ as the set

$$N(P, \varepsilon) = \{Q \mid Q \in X \text{ and } d(P, Q) < \varepsilon\}.$$

More generally, for each $M \subset X$, and each $\varepsilon > 0$, the $\varepsilon$-neighborhood of $M$ is

$$N(M, \varepsilon) = \{Q \mid Q \in X \text{ and } d(P, Q) < \varepsilon \text{ for some } P \in M\}.$$
