We need to show that $f^{-1}$ is a mapping. Given $Q_1, Q_2, \ldots$ in $Y$, with $\lim Q_i = Q$, let $P_i = f^{-1}(Q_i)$ and $P = f^{-1}(Q)$. We need to know that $\lim P_i = P$. If not, we have $\lim P_{n_i} = P' \neq P$ for some subsequence $P_{n_1}, P_{n_2}, \ldots$. It follows that $\lim Q_{n_i} = f(P') \neq Q = f(P)$, which is impossible.
