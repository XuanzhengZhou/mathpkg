Let $M$ be a set in $\mathbb{R}^n$, and suppose that $M$, regarded as a space, is triangulable. If there is a homeomorphism $h: \mathbb{R}^n \leftrightarrow \mathbb{R}^n$, such that $h(M)$ is a polyhedron, then $M$ is tamely imbedded (or simply tame). If $M$ is triangulable, but is not tame, then $M$ is wild.
