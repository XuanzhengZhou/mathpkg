**Definition.** A metric space is a pair $[X, d]$, where $X$ is a nonempty set and $d$ is a function $X \times X \rightarrow \mathbb{R}$, subject to the following conditions:

(D.1) $d(P, Q) \geqslant 0$ always.
(D.2) $d(P, Q) = 0$ if and only if $P = Q$.
(D.3) $d(P, Q) = d(Q, P)$ always.
(D.4) (the triangular property) $d(P, Q) + d(Q, R) \geqslant d(P, R)$ always.
