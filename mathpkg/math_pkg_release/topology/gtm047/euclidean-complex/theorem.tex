A Euclidean complex is a collection $K$ of simplexes in a space $\mathbb{R}^m$, such that:

(K.1) $K$ contains all faces of all elements of $K$.
(K.2) If $\sigma, \tau \in K$, and $\sigma \cap \tau \neq \varnothing$, then $\sigma \cap \tau$ is a face both of $\sigma$ and of $\tau$.
(K.3) Every $\sigma$ in $K$ lies in an open set $U$ which intersects only a finite number of elements of $K$.
