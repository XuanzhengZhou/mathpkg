Let $M$ be a set in $\mathbf{R}^n$, and suppose that $M$, regarded as a space, is triangulable. If there is a homeomorphism $h: \mathbf{R}^n \leftrightarrow \mathbf{R}^n$ such that $h(M)$ is a polyhedron, then $M$ is \textit{tamely imbedded} (or simply \textit{tame}). If $M$ is triangulable, but is not tame, then $M$ is \textit{wild}.
