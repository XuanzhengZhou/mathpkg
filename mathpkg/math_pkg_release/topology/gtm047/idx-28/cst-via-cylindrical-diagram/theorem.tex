Let $K$ be a triangulated 3-manifold, let $M = |K|$, and let $S$ be a polyhedral solid torus in $M$. Then $S$ is a CST if and only if there is a PL mapping $\phi: \sigma^2 \times [0, 1] \rightarrow S$, such that $\sigma^2 \times \{0\}$ and $\sigma^2 \times \{1\}$ are mapped onto the same 2-cell in $S$, and $\phi$ is a homeomorphism elsewhere.
