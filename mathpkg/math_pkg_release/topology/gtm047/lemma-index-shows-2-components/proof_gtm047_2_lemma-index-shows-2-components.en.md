Choose axes in general position. For each $P \in \mathbb{R}^2 - J$, define the index Ind $P$ as the number of points of $L_P \cap J$ to the left of $P$, reduced mod 2. The function $f: \mathbb{R}^2 - J \to \{0,1\}, f(P) = \text{Ind } P$, is continuous and both $f^{-1}(0)$ and $f^{-1}(1)$ are nonempty, so $\mathbb{R}^2 - J$ is not connected.
