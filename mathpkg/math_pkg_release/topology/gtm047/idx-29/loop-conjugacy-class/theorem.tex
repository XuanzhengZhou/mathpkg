Let $L: S^1 \rightarrow X$ be a loop in a pathwise connected space $X$. Let $f: [0, 1] \rightarrow S^1$ be a mapping such that $f(0) = f(1) = Q_0 \in S^1$, and such that $f$ is a homeomorphism elsewhere. For each $t \in [0, 1]$, let $p(t) = L(f(t))$. Then $p$ is a closed path in $X$, with base point $Q_0' = L(Q_0)$. Let $P_0 \in X$, and let $q: [0, 1] \rightarrow X$ be a path from $P_0$ to $Q_0'$. Consider the mapping $r = qpq^{-1}$, where the multiplication is end-to-end, as in the definition of the fundamental group. Now $r$ is a closed path in $X$, with base point $P_0$, and determines an element $\bar{r}$ of $\pi(X, P_0)$.

The conjugacy class of $\bar{r}$ in $\pi(X, P_0)$ depends only on $L$. We define

$$L(X) = L(X, P_0)$$

to be the conjugacy class in $\pi(X, P_0)$ that contains every such element $\bar{r}$. If $N$ is a normal subgroup of $\pi(X, P_0)$, then $N$ contains all or none of the elements of $L(X)$.
