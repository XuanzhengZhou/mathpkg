A linear ordering of a set $R$ is a relation $<$, defined on $R$, such that:
\begin{itemize}
\item[(O.1)] $a < a$ never holds.
\item[(O.2)] $a < b < c \Rightarrow a < c$.
\item[(O.3)] For each $a, b \in R$, one and only one of the following conditions holds:
$$a < b, \quad a = b, \quad b < a.$$
\end{itemize}
The pair $[R, <]$ is then called a linearly ordered set. Open intervals in $R$ are defined as in the real number system:
$$(a, b) = \{x \mid x \in R, a < x < b\}.$$
