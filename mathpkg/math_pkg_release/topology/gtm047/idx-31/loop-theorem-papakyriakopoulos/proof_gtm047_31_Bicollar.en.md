Let $D: \Delta \rightarrow M^3$ be a PL singular 2-cell in $M^3$, with $\text{Bd } D = L$, $|L| \subset M^2$, such that $L$ is not contractible in $M^2$. Since $M^2 \subset \text{Int } M^3$, we can force $|D|$ off of $\text{Bd } M^3$, preserving the stated properties of $D$. Thus we may assume hereafter that $|D| \subset \text{Int } M^3$.

Let $W$ be a bicollar neighborhood of $M^2$ in $\text{Int } M^3$. We choose $D$ in general position relative to $\text{Bd } W$, in the sense that (a) the set

$$D^{-1}(|D| \cap \text{Bd } W)$$

is a finite union of disjoint polygons $J_1, J_2, \ldots, J_n$ and (b) each $J_i$ has an annular neighborhood $A_i$ in $\Delta$ such that one of the components of $A_i - J_i$ is mapped by $D$ into $\text{Int } W$, and the other is mapped into $M^3 - W$. Now each $J_i$ bounds a 2-cell $d_i$ in $\Delta$. Some one of these, say, $d_1$, has the property that $D|\text{Bd } d_1$ is not contractible in $M^2$. Taking $\Delta_1$ to be a polyhedral 2-cell approximating $D|d_1$, we obtain the desired conclusion.
