Let $\phi$ be a function $X \rightarrow \mathbb{R}^+$. Suppose that $\phi$ is bounded away from 0 on every compact set, i.e., for every compact set $M$ there is an $\varepsilon_M > 0$ such that $\phi(P) \geqslant \varepsilon_M$ for each $P \in M$. Then $\phi$ is called strongly positive, and we write $\phi \gg 0$ on $X$.
