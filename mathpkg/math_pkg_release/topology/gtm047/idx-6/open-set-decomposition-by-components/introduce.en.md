In any Euclidean space, if two points lie in different connected components of an open set, the open set can be partitioned into two disjoint open subsets each containing one of the points. This follows because every component of an open set in $\mathbb{R}^n$ is itself open (since open balls are connected), so the union of some components is open and the union of the remaining components is also open. This basic topological fact is used in the proof of the Jordan curve theorem to analyze separation properties.
