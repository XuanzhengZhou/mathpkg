This isotopy extension theorem handles the case when the isotopy is defined on an open subset rather than a submanifold. If an isotopy defined on an open set $U$ has an open image in $M 	imes I$, then near any compact subset $A \subset U$, the isotopy can be extended to a global diffeotopy of $M$ with compact support. The openness condition on the image is essential for extending the tangent vector field via a partition of unity.
