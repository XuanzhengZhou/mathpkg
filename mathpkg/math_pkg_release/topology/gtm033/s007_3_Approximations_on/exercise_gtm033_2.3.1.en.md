# Exercise 2.3.1

Let $1 \leqslant r < s \leqslant \infty$. There are $C^s$ manifolds $M, N$ and closed sets $A \subset M$, $B \subset N$ such that $C^s(M,A; N,B)$ is not dense in $C^r(M,A; N,B)$.

*Hint:* Let $A = M$. Suppose $B \subset N$ is a $C^r$ submanifold which is not $C^s$, and $f: M \to B$ is a $C^r$ diffeomorphism.
