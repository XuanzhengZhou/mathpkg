# Exercise 2.4.14

The limit set $L(f)$ of $f: M \to N$ is the set of $y \in N$ such that $y = \lim x_n$ for some sequence $\{x_n\}$ in $M$ which has no convergent subsequence. If $\dim N > 2 \dim M$ and $1 \leqslant r \leqslant \infty$ then embeddings are dense in

$$\mathcal{L} = \{f \in C_S^r(M, N) : f(M) \cap L(f) = \varnothing\}.$$

*Hint:* If $f \in \mathcal{L}$, there is an open set $N_0 \subset N$ containing $f(M)$ such that $f: M \to N_0$ is proper; use Theorem 2.13.
