# Exercise 9.2.4

Let $f: M \to \mathbb{R}$ be a Morse function on a connected compact Riemannian manifold without boundary. Assume the Riemannian metric comes from Morse charts near critical points. Then if $f$ has more than one local minimum, there exist two local minima $a, b$ and a type $1$ critical point $p$, with the following property: one branch of the unstable manifold of $p$ (for $\operatorname{grad} f$) tends to $a$ and the other branch tends to $b$.
