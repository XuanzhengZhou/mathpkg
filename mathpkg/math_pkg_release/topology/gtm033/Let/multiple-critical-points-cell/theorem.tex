Let $f: M \rightarrow [a,b]$ be an admissible Morse function of type $(v_0, \ldots, v_n)$ on a compact manifold. Suppose $f$ has just one critical value $c$, $a < c < b$. Then there are disjoint $k$-cells $e_i^k \subset M - f^{-1}(b)$, $1 \leq i \leq v_k$, $k = 0, \ldots, n$, such that $e_i^k \cap f^{-1}(a) = \partial e_i^k$; and there is a deformation retraction of $M$ onto $f^{-1}(a) \cup \{\bigcup_{i,k} e_i^k\}$.
