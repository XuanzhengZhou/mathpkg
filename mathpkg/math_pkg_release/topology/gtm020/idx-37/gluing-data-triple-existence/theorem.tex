Let $X = X_0 \cup X_1$ with $A = X_0 \cap X_1$, and let $\alpha: \xi_0 \to \xi_1$ be a difference isomorphism over $(X, A)$. Suppose that for $A = B = \emptyset$ the natural maps are isomorphisms. Then there exists a triple $(\xi, u_1, u_0)$ such that $\xi$ is a vector bundle over $X$, $u_i: \xi_i \to \xi|_{X_i}$ is an isomorphism for $i = 0, 1$, and $u_0 \alpha = u_1$ over $A$. Moreover, if $(\eta, v_1, v_0)$ is a second triple with the above properties, there is an isomorphism $w: \eta \to \xi$ with $u_i = w v_i$ for $i = 0, 1$.
