Let $X$ be an $n$-dimensional CW-complex and let $\xi^k$ be a $k$-dimensional $F$-vector bundle over $X$ with $c = \dim_{\mathbf{R}} F$. Set
$$m = \left\langle \frac{n+1}{c} - 1 \right\rangle,$$
where $\langle x \rangle$ denotes the smallest integer $n$ with $x \leq n$. Then each $k$-dimensional vector bundle $\xi^k$ is isomorphic to
$$\eta^m \oplus \theta^{k-m}$$
for some $m$-dimensional vector bundle $\eta$. In particular:
- For real vector bundles ($c=1$): $\xi^k \cong \eta^n \oplus \theta^{k-n}$.
- For complex vector bundles ($c=2$): $m \approx n/2$.
- For quaternionic vector bundles ($c=4$): $m \approx n/4$.
