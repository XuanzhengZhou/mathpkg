Using the elementary calculations of $U_F(1)$, $SU_F(1)$, $SO(2)$, and $O(2)$, the following relations hold:
$$\pi_0(O(n)) = \mathbf{Z}_2 \quad \text{for all } n \geq 1,$$
$$\pi_0(SO(n)) = \pi_0(U(n)) = \pi_0(SU(n)) = \pi_0(Sp(n)) = 0 \quad \text{for all } n \geq 1.$$
These results follow from the inequality $0 \leq c(n+1)-3$, which holds for all $n \geq 1$ when $c = 2, 4$, and for $n \geq 2$ when $c = 1$.
