7. Universal Vector Bundles

In Theorem 3(7.2) vector bundles over paracompact spaces were classified using the universal principal bundle $\alpha_k^n = (E, p, G_k(F^n))$ when $n = +\infty$. By (6.1), $\alpha_k^n = (V_k(F^n), p, G_k(F^n))$ is the universal $U_F(k)$-bundle in dimensions $\leq c(n-k+1) - 2$. By 5(3.3), the fibre bundle $\alpha_k^n[F^k]$ is a vector bundle. We compare $\alpha_k^n[F^k]$ and $\gamma_{kn}$ using the bundle map $f: \alpha_k^n[F^k] \rightarrow \gamma_{kn}$, where

$$f((v_1, \ldots, v_k, y) \bmod U_F(k)) = (\langle v_1, \ldots, v_k \rangle, y_1v_1 + \cdots + y_kv_k)$$

For $a \in U_F(k)$, let $(v_1, \ldots, v_k)a = (v'_1, \ldots, v'_k)$ and $ay = y'$. Then we have $\sum_m y'_m v_m = \sum_{m,j} a_{m,j}y_jv_m = \sum_j y_jv'_j$, and $u$ is a well-defined bundle morphism.
