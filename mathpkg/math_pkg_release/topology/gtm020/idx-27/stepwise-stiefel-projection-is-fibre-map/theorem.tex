The projection $q: V_{k+1}(F^n) \to V_k(F^n)$ defined by forgetting the last vector,

$$q(v_1, \ldots, v_k, v_{k+1}) = (v_1, \ldots, v_k)$$

is a fibre map with fibre $V_1(F^{n-k})$. Specifically, the fibre is a sphere:

$$V_1(F^{n-k}) \cong \begin{cases}
S^{n-k-1} & \text{if } F = \mathbf{R} \\
S^{2n-2k-1} & \text{if } F = \mathbf{C} \\
S^{4n-4k-1} & \text{if } F = \mathbf{H}
\end{cases}$$

Thus for each $k$ there is a fibre bundle

$$V_1(F^{n-k}) \longrightarrow V_{k+1}(F^n) \xrightarrow{q} V_k(F^n).$$
