By Definition (7.4), we have $e(\xi \oplus \eta) = p^{*-1}j^*(U)$, using the above notation. Then we calculate $p^{*-1}j^*(U) = p^{*-1}j^*(q_1^*(U')q_2^*(U'')) = p^{*-1}[p_1^*j_1^*(U')p_2^*j_2^*(U'')] = e(\xi)e(\eta)$. This proves the theorem.
