This proposition provides a generating function identity relating the power sums $s_k^n$ to the polynomial $f(t) = 1 + y_1 t + \cdots + y_n t^n$. When $y_i$ are specialized to the elementary symmetric functions $\sigma_i$, the logarithmic derivative $-t f'(t)/f(t)$ expands to give $\sum s_k^n(\sigma_1, \ldots, \sigma_n)(-t)^k$, where $s_k^n = x_1^k + \cdots + x_n^k$. This identity is essential for relating power sum symmetric functions to elementary symmetric functions in the computation of representation rings.
