A vector bundle with an everywhere-nonzero cross section splits off a line bundle, that is, $\xi = \theta^1 \oplus \eta$. Then $e(\xi) = e(\theta^1)e(\eta) = 0 \cdot e(\eta) = 0$. The Euler class of a trivial bundle is zero.
