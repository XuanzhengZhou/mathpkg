This corollary confirms that the Euler class is a nontriviality detector: if a vector bundle is trivial (isomorphic to a product bundle), then its Euler class must vanish. Conversely, a nonzero Euler class guarantees the bundle is not trivial. This is consistent with the idea that characteristic classes measure the twisting of a bundle.
