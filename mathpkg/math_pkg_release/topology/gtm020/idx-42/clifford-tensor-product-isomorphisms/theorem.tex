There are isomorphisms
$$C \otimes C_k \cong C \otimes C'_k$$
for $k = 1, 2$. For the third case, define an isomorphism
$$w: \mathbf{H} \otimes \mathbf{H} \rightarrow \operatorname{Hom}_{\mathbf{R}}(\mathbf{H}, \mathbf{H}) = \mathbf{R}(4)$$
by the relation $w(x_1 \otimes x_2)x = x_1 x \bar{x}_2$. Then $w$ is an algebra morphism.
