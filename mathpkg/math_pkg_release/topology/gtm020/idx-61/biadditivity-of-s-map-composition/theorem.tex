\text{The composition function}

$$\{X, Y\} \times \{Y, Z\} \to \{X, Z\}$$

\text{defined from the composition maps } [S^n(X), S^n(Y)] \times [S^n(Y), S^n(Z)] \to [S^n(X), S^n(Z)] \text{ is biadditive.}

\text{That is, for any } f, f_1, f_2 \in \{X, Y\} \text{ and } g, g_1, g_2 \in \{Y, Z\},

$$(f_1 + f_2) \circ g = f_1 \circ g + f_2 \circ g$$

\text{and}

$$f \circ (g_1 + g_2) = f \circ g_1 + f \circ g_2.$$
