\text{Let } f: X \to Y \text{ be a map and form the Puppe sequence}

$$X \xrightarrow{f} Y \xrightarrow{\alpha_f} C_f \xrightarrow{\beta_f} SX \xrightarrow{S_f} SY.$$

\text{Then for each space } Z \text{ the following sequences of } S\text{-maps are exact:}

$$\{X, Z\} \leftarrow \{Y, Z\} \leftarrow \{C_f, Z\} \leftarrow \{SX, Z\} \leftarrow \{SY, Z\}$$

\text{and}

$$\{Z, X\} \to \{Z, Y\} \to \{Z, C_f\} \to \{Z, SX\} \to \{Z, SY\}.$$
