\text{Let } X \text{ be a } CW\text{-complex of dimension } n \text{ and } Y \text{ an } r\text{-connected space with } n - 1 < 2r - 1. \text{ Then the natural function}

$$[X, Y] \to \{X, Y\}$$

\text{is a bijection.}

\text{The proof shows that the suspension map } S: [X, Y] \to [S(X), S(Y)] = [X, \Omega S(Y)] \text{ is a bijection under the above hypotheses. The bijective character of } [X, Y] \to [X, \Omega S(Y)] \text{ is established using a Wang-type exact sequence over } S(Y) \text{ for the fibration } ES(Y) \to S(Y).
