For any fixed object $K$ in a category $\mathbf{A}$, the assignment $X \mapsto [X, K]$ (the set of morphisms from $X$ to $K$) together with the precomposition action on morphisms defines a contravariant functor (cofunctor) from $\mathbf{A}$ to the category of sets. This is the fundamental Hom-functor construction, which forms the basis of the Yoneda embedding. The proof is a straightforward verification of the functor axioms using associativity of composition.
