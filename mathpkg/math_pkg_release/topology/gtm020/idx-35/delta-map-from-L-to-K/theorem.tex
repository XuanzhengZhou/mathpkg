There is a well-defined map $\Delta: L(X, A) \rightarrow K(X, A)$ given by
$$\Delta([\alpha: \xi_0 \rightarrow \xi_1]) = \{\xi_0 \oplus \zeta / (\alpha \oplus 1)\} - \{\xi_1 \oplus \zeta\}$$
where $\zeta$ is any vector bundle such that $\xi_1 \oplus \zeta$ is isomorphic to a trivial bundle $\theta^m$, and $\{\eta/\beta\}$ denotes the relative $K$-theory class represented by the isomorphism $\beta: \eta|A \to \theta^m|A$.
