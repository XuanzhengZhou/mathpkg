In a category of paracompact spaces containing $CP^n$ for $0 \leq n \leq +\infty$, for a ring $R$, there exist mutually inverse isomorphisms

$$a: [\text{Vect}_F, H^{\text{ev}}(-, R)]_{+,+} \rightarrow R[[t]]_+$$

$$m: [\text{Vect}_F, H^{\text{ev}}(-, R)]_{+,x} \rightarrow R[[t]]_x^*$$

satisfying $a(\phi + \psi) = a(\phi) + a(\psi)$ and $m(\phi\psi) = m(\phi)m(\psi)$, where $c_1(\xi)$ denotes the image of the first Chern class under $H^2(X, Z) \rightarrow H^2(X, R)$.
