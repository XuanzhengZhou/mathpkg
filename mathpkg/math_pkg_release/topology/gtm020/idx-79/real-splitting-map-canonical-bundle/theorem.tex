Let $h_n: RP^\infty \times \dots \times RP^\infty \rightarrow G_n(R^\infty)$ be the classifying map for the bundle $\gamma \times \dots \times \gamma$, where $\gamma$ is the canonical line bundle over $RP^\infty$. Then $h_n$ is a splitting map for the canonical bundle $\gamma_n$ over $G_n(R^\infty)$ with respect to $Z_2$ cohomology.
