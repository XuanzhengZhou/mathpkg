The cohomology of the following spaces with values in a 2-divisible ring $R$ is:

$$H^*(CP^\infty) = R[c_1(\gamma)], \quad H^*(G_2^+(\mathbf{R}^\infty)) = R[e(\gamma_2^+)], \quad H^*(G_2(\mathbf{R}^\infty)) = R[p_1(\gamma_2)],$$

$$H^*(CP^\infty \times CP^\infty) = R[c_1(\gamma'), c_1(\gamma'')], \quad H^*(G_2(\mathbf{C}^\infty)) = R[c_1(\gamma_2), c_2(\gamma_2)].$$

Moreover, in the following commutative diagram, $w_2(e(\gamma_2^+)) = c_1(\gamma)$, $v_2(p_1(\gamma_2)) = e(\gamma_2^+)^2$, $v_1(c_1(\gamma_2)) = 0$, $v_1(c_2(\gamma_2)) = p_1(\gamma_2)$, $u_2(c_1(\gamma')) = e(\gamma_2^+)$, and $u_2(c_1(\gamma'')) = -e(\gamma_2^+)$.

$$H^*(CP^\infty) \xleftarrow{w_2} H^*(G_2^+(\mathbf{R}^\infty)) \xleftarrow{u_2} H^*(CP^\infty \times CP^\infty)$$

$$H^*(G_2(\mathbf{R}^\infty)) \xleftarrow{v_2} H^*(G_2(\mathbf{C}^\infty))$$
