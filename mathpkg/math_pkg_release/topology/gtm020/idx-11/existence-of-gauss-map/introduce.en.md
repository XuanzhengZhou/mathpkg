This theorem guarantees that every vector bundle over a paracompact base space admits a Gauss map into a suitable $F^m$. The proof uses a partition of unity subordinate to a trivializing cover: on each trivializing open set the bundle map factors through $F^k$, and the partition of unity is used to patch these local maps into a global Gauss map. In the finite cover case, a Gauss map into $F^{kn}$ (rather than $F^\infty$) can be obtained. Combined with Proposition 5.2, this yields a classifying map $B \to G_k(F^m)$ for every vector bundle.
