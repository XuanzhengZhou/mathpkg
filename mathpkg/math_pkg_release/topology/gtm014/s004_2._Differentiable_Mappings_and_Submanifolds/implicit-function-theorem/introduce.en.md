The Implicit Function Theorem (Theorem 2.4) is a fundamental result in multivariable analysis. Given a $C^s$-differentiable map $\phi: U_1 \times U_2 \to \mathbf{R}^l$, if the partial derivative $(d_y \phi)_{(x_0, y_0)}$ has full rank $l$ at a point where $\phi(x_0, y_0) = y_0$, then locally the equation $\phi(x, y) = y_0$ can be solved for $y$ as a $C^s$ function of $x$. This theorem underpins the local analysis of immersions, submersions, and submanifolds.
