The higher-order critical sets $C_i$ form a descending filtration $C \supset C_1 \supset C_2 \supset \cdots$ of the critical set $C[f]$ of a smooth map $f$. A point $p$ belongs to $C_i$ if all partial derivatives of $f$ up to order $i$ vanish at $p$. This stratification is the key technical device in the proof of Sard's Theorem: it allows the critical set to be decomposed into layers $C - C_1$, $C_i - C_{i+1}$, and the deep tail $C_k$ (for large $k$), each of which is shown to have measure-zero image via separate lemmas.
