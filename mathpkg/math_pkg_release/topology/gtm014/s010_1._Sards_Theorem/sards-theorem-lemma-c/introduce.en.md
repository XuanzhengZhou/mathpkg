Lemma C handles the intermediate layers $C_i \setminus C_{i+1}$ of the critical set stratification. For a point $p \in C_i - C_{i+1}$, all partial derivatives up to order $i$ vanish, but some derivative of order $i+1$ does not. This allows the construction of a local diffeomorphism that "flattens" $C_i$ into a hyperplane, reducing the dimension and applying the induction hypothesis. A countable covering argument then yields that $f(C_i - C_{i+1})$ has measure zero for each $i \geq 1$.
