**Notes.** (1) Sard's Theorem can be generalized as follows: Assume that $k > \max(\dim Y - \dim X, 0)$. If $f$ is a $C^k$-differentiable mapping, then the measure of the set of critical values is zero. Since we will be using only smooth mappings in later chapters we will prove only the more restricted version here.

(2) If $\dim X < \dim Y$ then Sard's Theorem follows directly from Proposition 1.6 and the fact that the image of a subset of measure zero has measure zero.

Sard's Theorem is in reality a local theorem and follows from Proposition 1.13 (the local Euclidean form). The proof of Theorem 1.12 proceeds from Proposition 1.13 precisely as the proof of Corollary 1.4 proceeded from Proposition 1.3 — by covering the source manifold $X$ with coordinate charts, applying the local result on each chart, and using the fact that a countable union of measure zero sets has measure zero. The details are left for the reader.
