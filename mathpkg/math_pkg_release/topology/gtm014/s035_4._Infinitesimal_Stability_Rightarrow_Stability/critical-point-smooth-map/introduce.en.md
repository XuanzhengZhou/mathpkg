This definition introduces the notion of a critical point for a smooth map $f \colon X \to Y$ between manifolds: a point $p$ where the differential $(df)_p$ fails to be surjective. The critical point set $\Sigma = \bigcup_{q \in Y} \Sigma_q$ (with $\Sigma_q = f^{-1}(q) \cap \{\text{critical points}\}$) plays a central role in the proof that infinitesimal stability implies stability. By Lemma 1.8, each fiberwise critical set $\Sigma_q$ is finite, which allows the local deformation trivializations obtained from Propositions 4.3 and 4.4 to be patched together in Proposition 4.5.
