\begin{lemma}
Let $A_i$ ($1 \leq i \leq k$) be a finitely generated $C_{p_i}^\infty(X)$-module for each $i$.
Set $S = \{p_1,\ldots,p_k\} \subset X$ and let $C_S^\infty(X) = \bigoplus_{i=1}^{k} C_{p_i}^\infty(X)$.
Let $A = \bigoplus_{i=1}^{k} A_i$ be the direct sum, viewed as a $C_S^\infty(X)$-module with coordinatewise action.
Suppose $A$ is also a module over $C_q^\infty(Y)$ via the homomorphism $f^*: C_q^\infty(Y) \to C_S^\infty(X)$ induced by the coordinatewise ring homomorphisms $f^*: C_q^\infty(Y) \to C_{p_i}^\infty(X)$.
Then $A$ is finitely generated over $C_q^\infty(Y)$ if and only if each $A_i$ is finitely generated over $C_q^\infty(Y)$.
\end{lemma}
