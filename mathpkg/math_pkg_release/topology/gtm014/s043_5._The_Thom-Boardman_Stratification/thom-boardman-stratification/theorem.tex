Let $f: X \to Y$ be a Boardman map. The \textbf{Thom-Boardman stratification} of $X$ partitions $X$ into a disjoint union of the nonsingular set
$$X \setminus \bigcup_{i \neq 0} S_i(f)$$
(where $f$ is an immersion if $\dim X \le \dim Y$, or a submersion if $\dim X \ge \dim Y$) together with all Boardman sets $S_{r_1,\ldots,r_k}(f)$ whose terminal index satisfies $r_k = 0$. On each such Boardman set the restriction
$$f|_{S_{r_1,\ldots,r_k}(f)}: S_{r_1,\ldots,r_k}(f) \to Y$$
is an immersion. The strata are the connected components of these pieces; restricted to each stratum $f$ is either a submersion or an immersion with normal crossings.
