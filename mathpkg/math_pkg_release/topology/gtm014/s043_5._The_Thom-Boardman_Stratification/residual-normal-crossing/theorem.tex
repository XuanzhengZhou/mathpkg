Let $T_1,\ldots,T_s$ be submanifolds of $J^k(X, Y)$ such that each target map $T_j \to Y$ is a submersion. Then the generalized normal crossing condition (5.5) relative to the $T_j$ is satisfied for a residual set of maps $f: X \to Y$.

In particular, taking $T_j = S_{I_j}$ (the universal Boardman subbundles), Theorem 5.2 follows: for a residual set of maps $f$, the NC condition holds.
