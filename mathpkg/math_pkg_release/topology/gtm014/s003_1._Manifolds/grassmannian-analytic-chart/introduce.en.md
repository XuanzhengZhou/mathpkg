The Grassmannian $G_{k,n}$ is given an analytic manifold structure via charts modeled on the vector space $\operatorname{Hom}(V, V^\perp)$ of linear maps. For a fixed reference $k$-plane $V$, a neighborhood $W_V$ consists of all $k$-planes $U$ whose orthogonal projection onto $V$ is a bijection.

The chart map $\rho_V$ sends each such $U$ to the linear map $\pi_{U,V^\perp} \circ \pi_{U,V}^{-1}$ in $\operatorname{Hom}(V, V^\perp)$. Intuitively, this map encodes how $U$ is "tilted" relative to $V$: the inverse of the projection onto $V$ followed by projection onto the orthogonal complement $V^\perp$. The collection of these charts forms an analytic atlas, making $G_{k,n}$ a real analytic manifold.
