An umbilic is called:
\begin{itemize}
\item \textbf{hyperbolic} if its local ring is $\mathbf{R}[x, y]/(x^2, y^2)$,
\item \textbf{elliptic} if its local ring is $\mathbf{R}[x, y]/(x^2 - y^2, xy)$,
\item \textbf{parabolic} if its local ring is $\mathbf{R}[x, y]/(x^2, xy)$.
\end{itemize}
The name for an umbilic is given by the name of the associated quadratic form on $\mathcal{M}/\mathcal{M}^2$.
