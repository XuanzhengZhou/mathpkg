Let $X, Y$, and $B$ be smooth manifolds. Let $W$ be a submanifold of $J_s^k(X, Y)$. Let $W'$ be an open subset of $W$ such that $\overline{W}' \subset W$. Let $G: X \times B \rightarrow Y$ be a smooth mapping. Define $H: X^{(s)} \times B \rightarrow J_s^k(X, Y)$ by $H(x, b) = j_s^k(G_b)(x)$ and assume that $H \mathrel{\perp\!\!\!\perp} W$ on $\overline{W}'$. Then the set

$$\{b \in B \mid j_s^k G_b \mathrel{\perp\!\!\!\perp} W \text{ on } \overline{W}'\}$$

is dense in $B$.
