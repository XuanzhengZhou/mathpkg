Lemma 5.2 is a basic linear algebra fact about the rank of a block matrix. When the upper-left block $A$ is invertible, the rank of the full matrix $S$ equals $k$ precisely when the Schur complement $D - CA^{-1}B$ vanishes. The proof uses an elementary row operation to eliminate the lower-left block $C$, reducing $S$ to a block-triangular form without changing its rank. This lemma is a key computational tool for determining the codimension of the corank-$r$ strata $S_r$ in the 1-jet bundle.
