Lemma 6.5 states that if $W$ is an open set of smooth maps (in the Whitney $C^\infty$ topology), then the set of all $k$-jets of maps in $W$ forms an open subset of the jet bundle $J^k(X,Y)$. This is a technical result about the relationship between the Whitney topology on $C^\infty(X,Y)$ and the manifold topology on jet spaces. The proof constructs an explicit open neighborhood around any given $k$-jet using local coordinates and polynomial perturbations with small coefficients.
