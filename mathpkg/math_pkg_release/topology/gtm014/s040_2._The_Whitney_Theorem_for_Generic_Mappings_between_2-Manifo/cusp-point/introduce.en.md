A cusp point is an $S_1$ singularity of a smooth map between 2-manifolds where the tangent space of the singular set coincides with the kernel of the differential. The classification depends on the order of contact between a kernel vector field and the singular set. When a smooth nonvanishing vector field spanning the kernel has contact of order exactly two with $S_1(f)$, the point is called a simple cusp. This definition sets up Whitney's second theorem, which provides the normal form $(x_1, x_2) \mapsto (x_1, x_1x_2 + x_2^3)$ for simple cusp points.
