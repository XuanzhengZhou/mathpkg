When the target dimension is sufficiently large relative to the source (specifically $\dim Y \geq 2 \dim X + 1$), this proposition provides a clean geometric characterization of stable mappings. Under this dimensional hypothesis, a map is stable exactly when it is a one-to-one immersion. The proof relies on the Whitney Embedding Theorem: any mapping close enough to a given stable map must also be an embedding, so the set of embeddings is open, and stable maps are those equivalent to embeddings.
