For each integer $i$, let $W_{i}$ be a submanifold of $J^{k_{i}}(X, Y)$. Then the set of smooth mappings $f: X \to Y$ for which $j^{k_{i}} f \mathrel{\overline{\cap}} W_{i}$ for all $i$ is dense in $C^{\infty}(X, Y)$. If the number of $W_{i}$'s is finite and each $W_{i}$ is closed, then this set is open as well.
