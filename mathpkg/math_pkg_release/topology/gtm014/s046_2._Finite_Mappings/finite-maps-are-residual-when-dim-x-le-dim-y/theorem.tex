Let $X$ and $Y$ be smooth manifolds with $\dim X \leq \dim Y$. Then the set of finite maps
$$\{ f \in C^\infty(X, Y) \mid \text{$f$ is finite at every } x \in X \}$$
is a residual subset of $C^\infty(X, Y)$ in the Whitney $C^\infty$ topology. In fact, the complement has "infinite codimension" in a suitable sense.
