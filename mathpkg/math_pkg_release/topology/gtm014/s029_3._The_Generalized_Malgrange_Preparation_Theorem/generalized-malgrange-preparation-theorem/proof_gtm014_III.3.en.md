($\Rightarrow$) If $A$ is finitely generated as a $C_q^\infty(Y)$-module, say by $a_1, \ldots, a_k$, then the images $[a_1], \ldots, [a_k]$ span $A / \mathcal{M}_q(Y)A$ as a vector space over $\mathbf{R} \cong C_q^\infty(Y) / \mathcal{M}_q(Y)$. Hence $A / \mathcal{M}_q(Y)A$ is finite dimensional. This is Corollary 3.5.

($\Leftarrow$) Assume $A / \mathcal{M}_q(Y)A$ is a finite dimensional real vector space. Choose elements $a_1, \ldots, a_k \in A$ whose images form an $\mathbf{R}$-basis of $A / \mathcal{M}_q(Y)A$. Let $A' \subseteq A$ be the $C_q^\infty(Y)$-submodule generated by $a_1, \ldots, a_k$ (via the module structure induced by $\phi^*$). Consider the quotient module $Q = A / A'$. Since $A$ is finitely generated over $C_p^\infty(X)$ and $A'$ is finitely generated over $C_q^\infty(Y)$ (hence also over $C_p^\infty(X)$ via $\phi^*$), $Q$ is a finitely generated $C_p^\infty(X)$-module.

By construction, the images of $a_1, \ldots, a_k$ generate $A / \mathcal{M}_q(Y)A$, so
$$Q / \mathcal{M}_q(Y) Q \cong \frac{A}{A' + \mathcal{M}_q(Y)A} = 0.$$

Now we claim that $\mathcal{M}_p(X) Q = Q$. Since $\phi^*(\mathcal{M}_q(Y)) \subseteq \mathcal{M}_p(X)$, we have
$$\mathcal{M}_q(Y) Q \subseteq \mathcal{M}_p(X) Q \subseteq Q.$$

Because $Q / \mathcal{M}_q(Y) Q = 0$, we have $Q = \mathcal{M}_q(Y) Q$, and therefore $Q = \mathcal{M}_p(X) Q$. Since $C_p^\infty(X)$ is a local ring (Lemma 3.2) and $Q$ is a finitely generated $C_p^\infty(X)$-module, Nakayama's Lemma (Lemma 3.5) implies $Q = 0$. Thus $A = A'$, so $A$ is generated by $a_1, \ldots, a_k$ as a $C_q^\infty(Y)$-module.

**Connection to the classical Malgrange Preparation Theorem.** To see that Theorem 3.6 indeed generalizes Theorem 2.1, let $X = \mathbf{R} \times \mathbf{R}^n$, $Y = \mathbf{R}^n$, and $\pi: \mathbf{R} \times \mathbf{R}^n \to \mathbf{R}^n$ be the projection $\pi(t, x) = x$. Take $p = 0 = q$. Let $[F]_0 \in C_0^\infty(\mathbf{R} \times \mathbf{R}^n)$ satisfy $F(t, 0) = t^k g(t)$ with $g(0) \neq 0$, and let $[G]_0$ be any other germ. Consider the $C_0^\infty(\mathbf{R} \times \mathbf{R}^n)$-module
$$A = C_0^\infty(\mathbf{R} \times \mathbf{R}^n) / \langle [F]_0 \rangle$$
where $\langle [F]_0 \rangle$ is the ideal generated by $[F]_0$. One verifies that $A / \mathcal{M}_0(\mathbf{R}^n) A$ is a $k$-dimensional real vector space (spanned by $1, t, \ldots, t^{k-1}$). By Theorem 3.6, $A$ is finitely generated as a $C_0^\infty(\mathbf{R}^n)$-module, which yields the division statement: $G = qF + r$ where $r$ is a polynomial in $t$ of degree $< k$ with coefficients in $C_0^\infty(\mathbf{R}^n)$.
