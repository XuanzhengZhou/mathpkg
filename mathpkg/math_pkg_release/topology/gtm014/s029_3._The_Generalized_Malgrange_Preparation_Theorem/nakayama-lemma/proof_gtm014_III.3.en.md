Let $A$ be a finitely generated $\mathcal{R}$-module with generators $e_1, \ldots, e_n$. Suppose $A = \mathcal{M}A$. Then each generator $e_k$ can be written as a finite sum

$$e_k = \sum_{i=1}^{s} m_i a_i$$

where each $m_i \in \mathcal{M}$ and $a_i \in A$. Since $e_1, \ldots, e_n$ generate $A$, each $a_i$ can be expressed as $a_i = \sum_{j=1}^{n} r_{ij} e_j$ for some $r_{ij} \in \mathcal{R}$. Substituting,

$$e_k = \sum_{i=1}^{s} m_i \sum_{j=1}^{n} r_{ij} e_j = \sum_{j=1}^{n} \left( \sum_{i=1}^{s} m_i r_{ij} \right) e_j = \sum_{j=1}^{n} s_{kj} e_j,$$

where $s_{kj} = \sum_{i=1}^{s} m_i r_{ij} \in \mathcal{M}$. Using the Kronecker delta, we have

$$\sum_{j=1}^{n} (\delta_{kj} - s_{kj}) e_j = 0 \quad \text{for each } k = 1, \ldots, n.$$

This is a system of $n$ linear equations in the $n$ "unknowns" $e_1, \ldots, e_n$. If the matrix $D = (\delta_{ij} - s_{ij})$ is invertible over $\mathcal{R}$, then multiplying by $D^{-1}$ yields $e_1 = \cdots = e_n = 0$, so $A = 0$.

A matrix over a commutative ring with identity is invertible if and only if its determinant is invertible in the ring. Expanding $\det(\delta_{ij} - s_{ij})$ via permutations, the off-diagonal terms are products involving elements of $\mathcal{M}$ and the diagonal gives $1$, so

$$\det(\delta_{ij} - s_{ij}) = 1 + s$$

for some $s \in \mathcal{M}$. In a local ring, the maximal ideal consists precisely of the non-invertible elements: if $t \in \mathcal{M}$, then $t$ cannot be invertible (otherwise $1 = t t^{-1} \in \mathcal{M}$); conversely, if $t \notin \mathcal{M}$, the ideal generated by $t$ is not contained in $\mathcal{M}$ and must be the whole ring, so $t t' = 1$ for some $t'$. Thus $1 + s \notin \mathcal{M}$ (since $1 = (1+s) - s$, if $1+s \in \mathcal{M}$ then $1 \in \mathcal{M}$), so $1 + s$ is invertible. Hence $D$ is invertible and $A = 0$.
