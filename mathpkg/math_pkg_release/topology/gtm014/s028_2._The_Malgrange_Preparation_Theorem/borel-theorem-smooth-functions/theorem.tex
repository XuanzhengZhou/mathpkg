Given any sequence of smooth functions $\{f_k(x)\}_{k=0}^{\infty}$ on $\mathbf{R}^n$, there exists a smooth function $F(t, x)$ such that
$$\frac{\partial^k F}{\partial t^k}(0, x) = f_k(x)$$
for all $k$. The function can be constructed as
$$F(t, x) = \sum_{k=0}^{\infty} \frac{t^k}{k!} \rho(\mu_k t) f_k(x)$$
where $\rho$ is a smooth bump function and $\mu_k \to \infty$ sufficiently fast.
