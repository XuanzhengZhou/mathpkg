Let $P_k(z, \lambda) = z^k + \sum_{i=0}^{k-1} \lambda_i z^i$. Let $G(t, x)$ be smooth near $0 \in \mathbf{R} \times \mathbf{R}^n$. Then there exists a smooth function $\tilde{G}(z, x, \lambda)$ near $0 \in \mathbf{C} \times \mathbf{R}^n \times \mathbf{C}^k$ such that $\tilde{G}(t, x, \lambda) = G(t, x)$ for real $t$ and $\partial \tilde{G}/\partial \bar{z}$ vanishes to infinite order on $\{P_k(z, \lambda) = 0\}$.
