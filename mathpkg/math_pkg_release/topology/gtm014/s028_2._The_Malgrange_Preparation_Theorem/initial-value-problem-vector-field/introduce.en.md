This lemma constructs a smooth solution to the initial value problem $\partial F/\partial t = XF$, $F(0, x) = f(x)$, valid to infinite order at $t=0$. The formal solution $\exp(tX)f$ is regularized using the Borel Theorem bump function technique. This is used in the proof of the smooth Polynomial Division Theorem to construct smooth extensions of functions that vanish to high order on prescribed sets.
