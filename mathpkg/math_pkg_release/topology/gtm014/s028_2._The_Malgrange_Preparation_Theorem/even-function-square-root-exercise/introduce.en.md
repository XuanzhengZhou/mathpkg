This exercise asks to show that any smooth even function $f(x)$ can be written as $g(x^2)$ for some smooth function $g$. The key idea, as suggested by the hint, is to use the technique from the proof of the Borel Theorem (Lemma 2.5): one expands $f$ in a Taylor series around $0$ (which only contains even powers), and then uses bump functions with rapidly increasing scaling factors to construct a smooth $g$ whose Taylor series matches the formal half-power expansion.
