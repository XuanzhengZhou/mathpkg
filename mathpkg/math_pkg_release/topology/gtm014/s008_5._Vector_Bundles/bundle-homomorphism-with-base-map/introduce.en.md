A homomorphism of vector bundles $\phi: E \to F$ is a smooth, fiberwise-linear map that covers a smooth base map $f: X \to Y$ between the base manifolds. This generalizes Definition 5.1(3), which only considered bundles over the same base space. The key example is the Jacobian (differential) of a smooth map $f: X \to Y$, which defines a vector bundle homomorphism $df: TX \to TY$ covering $f$. Locally, such homomorphisms correspond to smooth families of linear maps between the trivialized fibers.
