If $G$ is an abelian group which contains an element $x$ of order $q \neq 1, 2, 3, 4, 6$ then $Wh(G) \neq 0$. In fact let $j, k$ and $a$ be integers such that $j > 1, k > 1, j + k < q, jk = aq \pm 1$. Such integers always exist. Then a non-trivial unit $u$ of $\mathbb{Z}(G)$ is given by
$$u = (1 + x + \ldots + x^{j-1})(1 + x + \ldots + x^{k-1}) - a(1 + x + \ldots + x^{q-1}).$$
