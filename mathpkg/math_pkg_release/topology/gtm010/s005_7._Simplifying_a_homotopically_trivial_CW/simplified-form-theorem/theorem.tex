If $(K, L)$ is a pair of connected finite CW complexes such that $K \searrow L$, then $K$ can be formally deformed rel $L$ to a pair $(J, L)$ in simplified form:
$$J = L \cup \bigcup_{j=1}^a e_j^r \cup \bigcup_{i=1}^a e_i^{r+1}$$
where the boundary map $\partial: \pi_{r+1}(J, J_r) \rightarrow \pi_r(J_r, L)$ (with matrix $(a_{ij})$ with respect to chosen characteristic maps $\{\varphi_i\}, \{\psi_j\}$) is an isomorphism of $\mathbb{Z}\pi_1$-modules. When this holds, $(J, L)$ is said to be a simplified pair.
