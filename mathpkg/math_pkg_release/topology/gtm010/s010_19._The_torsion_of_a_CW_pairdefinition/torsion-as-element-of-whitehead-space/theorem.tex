If $(K, L)$ is a connected CW pair with $K \searrow L$, we define
$$\tau(K, L) = j_x \circ i_*^{-1} (\tau(K, L) \text{ as defined earlier}) \in Wh(L)$$
where $i: L \subset K$ and $x \in L$ is a base point. Also define $\tau(K, L) = \tau(C(\tilde{K}, \tilde{L})_\psi) \in Wh(G)$ where $\psi$ is a choice of base points for identifying $G$ with $\pi_1(L, x)$.
