An element $d$ of a commutative ring $R$ is a \text{greatest common divisor} (g.c.d.) of $a_1, \cdots, a_n$ if $d \mid a_i$ for all $i$ and whenever $e \mid a_i$ for all $i$, then $e \mid d$. A \text{unique factorization domain} (UFD) is an integral domain where every nonzero non-unit has an essentially unique factorization into primes. Every UFD is a g.c.d. domain. $d$ is a g.c.d. iff the intersection of all principal ideals containing $\{a_1,\cdots,a_n\}$ is the principal ideal $(d)$.
