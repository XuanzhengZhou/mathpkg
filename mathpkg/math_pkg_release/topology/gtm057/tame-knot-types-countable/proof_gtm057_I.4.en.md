Any tame knot is equivalent to a polygonal knot in regular position. A polygonal knot in regular position is determined by a finite set of vertices with rational (or integer) coordinates, together with the crossing information (which specifies over/under at each double point). The set of all such finite combinatorial descriptions is countable. Since each tame knot type contains at least one such polygonal representative, there can be at most countably many tame knot types.
