The set $\mathbb{Q}_p$ is defined as the set of equivalence classes of Cauchy sequences of rational numbers with respect to $|\cdot|_p$. Two Cauchy sequences $\{a_i\}$ and $\{b_i\}$ are equivalent if $|a_i - b_i|_p \to 0$. $\mathbb{Q}_p$ is a field with operations defined termwise. $\mathbb{Q}$ embeds via constant sequences. The norm extends by $|a|_p = \lim_{i\to\infty}|a_i|_p$, and $\mathbb{Q}_p$ is complete.
