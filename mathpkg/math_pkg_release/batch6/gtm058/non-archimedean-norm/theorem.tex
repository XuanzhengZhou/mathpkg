A norm is called \text{non-Archimedean} if $\|x + y\| \leq \max(\|x\|, \|y\|)$ always holds. A metric is called non-Archimedean if $d(x, y) \leq \max(d(x, z), d(z, y))$; in particular, a metric is non-Archimedean if it is induced by a non-Archimedean norm. A norm (or metric) which is not non-Archimedean is called \text{Archimedean}.
