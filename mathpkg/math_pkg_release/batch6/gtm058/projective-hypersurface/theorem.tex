If $\tilde{f}(X_0,\ldots,X_n)$ is a homogeneous polynomial, the set of points in $\mathbb{P}_F^n$ at which $\tilde{f}$ vanishes is called the \text{projective hypersurface} $\tilde{H}_f$ defined by $\tilde{f}$. The projective completion of an affine hypersurface $H_f$ is obtained by homogenizing $f$.
