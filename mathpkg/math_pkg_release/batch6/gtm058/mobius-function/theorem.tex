The Möbius function $\mu(n)$ is defined for $n \in \{1,2,3,\ldots\}$ by $$\mu(n) = \begin{cases} 0, & \text{if } n \text{ is divisible by a perfect square } > 1; \\ (-1)^k, & \text{if } n \text{ is a product of } k \text{ distinct prime factors}. \end{cases}$$ A basic fact: $\sum_{d|n} \mu(d) = 1$ if $n=1$ and 0 otherwise.
