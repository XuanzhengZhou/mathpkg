To interpolate values of the Riemann zeta function hBadically, one must fix the residue class $ modulo -1$, because ^{k}$ depends only on $ modulo -1$ for  \in \mathbb{Z}_p^\times$. The function $\zeta_{p,s_0}$ provides a continuous hBadic interpolation for integers  \equiv s_0 \pmod{p-1}$. There are -1$ distinct branches of the hBadic zeta function.
