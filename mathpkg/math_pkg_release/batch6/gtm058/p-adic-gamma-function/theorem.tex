For $p > 2$, the $p$-adic gamma function $\Gamma_p: \mathbb{Z}_p \to \mathbb{Z}_p^\times$ is defined by continuous interpolation of the values at positive integers: $$\Gamma_p(k) = (-1)^k \prod_{\substack{j < k \\ p \nmid j}} j \quad \text{for } k \in \mathbb{N},$$ and extended by continuity: $\Gamma_p(s) = \lim_{k \to s} \Gamma_p(k)$.
