For any nonzero integer $a$, let the $p$-adic ordinal of $a$, denoted $\text{ord}_p a$, be the highest power of $p$ which divides $a$, i.e., the greatest $m$ such that $a \equiv 0 \pmod{p^m}$. If $a = 0$, we agree to write $\text{ord}_p 0 = \infty$. For any rational number $x = a/b$, define $\text{ord}_p x = \text{ord}_p a - \text{ord}_p b$.
