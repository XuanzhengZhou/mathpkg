There exists a unique linear map

$$\Gamma: K[X] \to C(\mathbb{Z}, K)$$

satisfying any one of the following equivalent conditions:

$$\Gamma(1) = 1, \quad \Gamma\left(\frac{X}{n}\right)(k) = \frac{k}{n},$$

$$f^*(k) = \sum f(k) \frac{X^k}{k!}.$$

This map also satisfies:

$$\Gamma(X) f^*(k) = k f(k).$$
