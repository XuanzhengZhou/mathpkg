Let $K$ be a field of characteristic $0$. Let $T$ be the variable on the multiplicative group. Set

$$X = T - 1, \quad T = X + 1.$$

Then $X$ is the corresponding variable on the additive group. Note that

$$X = T - 1 = e^z - 1$$

after the change of variables $T = e^z$.

For any power series $f$ we define the Leopoldt transform $f^*$ as a function on integers $k$:

$$f^*(k) = \sum_{n=0}^{\infty} a_n \frac{k^n}{n!}$$

where $f(X) = \sum a_n X^n$.
