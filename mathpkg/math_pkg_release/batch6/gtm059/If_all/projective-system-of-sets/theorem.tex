Let $\{X_n\}_{n \geq 1}$ be a sequence of finite sets, together with a sequence of surjective maps
$$\pi_{n,m} : X_n \to X_m \quad (n \geq m)$$
satisfying the compatibility condition $\pi_{n,m} \circ \pi_{m,k} = \pi_{n,k}$ for $n \geq m \geq k$. The \textbf{projective limit} (or inverse limit) is the set
$$X = \varprojlim_{n} X_n = \left\{ (x_n)_{n \geq 1} \in \prod_{n} X_n : \pi_{n,m}(x_n) = x_m \text{ for all } n \geq m \right\}.$$
In applications, the index set is often $\mathbb{Z}_{>0}$ ordered by divisibility, and a typical example is the system $\{\mathbb{Z}/p^n\mathbb{Z}\}$ with reduction maps modulo $p^m$.
