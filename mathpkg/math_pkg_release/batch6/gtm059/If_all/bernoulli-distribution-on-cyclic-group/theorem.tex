On the projective system $\{\frac{1}{M}\mathbb{Z}/\mathbb{Z}\}$ (ordered by divisibility), the assignment
$$\varphi_M^{(k)}(x) = M^{k-1} B_k\!\left( \langle x \rangle \right) \qquad \text{for } x \in \frac{1}{M}\mathbb{Z}/\mathbb{Z}$$
defines a distribution of degree $k-1$, where $\langle x \rangle$ denotes the fractional part in $[0,1)$. Equivalently, on the isomorphic projective system $\{\mathbb{Z}/M\mathbb{Z}\}$, the family
$$\psi_M^{(k)}(a) = M^{k-1} B_k\!\left( \frac{a}{M} \right) \qquad \text{for } a \in \mathbb{Z}/M\mathbb{Z}$$
defines a distribution. This distribution is bounded (hence a Bernoulli distribution).
