The \textbf{Bernoulli polynomials} $B_k(X)$ are defined by the generating function
$$F(t, X) = \frac{t e^{tX}}{e^t - 1} = \sum_{k=0}^{\infty} B_k(X) \frac{t^k}{k!}.$$
The Bernoulli numbers are the constant terms: $B_k = B_k(0)$. The first few Bernoulli polynomials are:
$$B_0(X) = 1, \quad B_1(X) = X - \frac{1}{2}, \quad B_2(X) = X^2 - X + \frac{1}{6}.$$
