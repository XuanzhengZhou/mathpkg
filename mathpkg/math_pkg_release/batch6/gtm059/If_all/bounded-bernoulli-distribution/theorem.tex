Let $X = \varprojlim X_n$ be a projective limit, and let $K$ be a complete non-Archimedean valued field. A distribution $\varphi = \{\varphi_n\}$ with values in $K$ is called \textbf{bounded} (or a \textbf{Bernoulli distribution}) if the values
$$|\varphi_n(x)| \quad \text{for } x \in X_n$$
are uniformly bounded, i.e., if there exists a constant $C$ such that $|\varphi_n(x)| \leq C$ for all $n$ and all $x \in X_n$. For a bounded distribution, the integral
$$\int_X f \, d\varphi$$
can be defined for any continuous function $f$ on $X$ by uniform approximation by locally constant functions.
