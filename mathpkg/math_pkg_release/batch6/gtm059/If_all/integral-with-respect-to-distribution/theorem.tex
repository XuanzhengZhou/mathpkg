Let $\varphi = \{\varphi_n\}$ be a distribution on a projective system $\{X_n\}$ with projective limit $X$. For a locally constant function $f$ on $X$ (i.e., $f$ factors through some $X_n$), the integral is defined as
$$\int_X f(x) \, d\varphi(x) = \sum_{x \in X_n} f(x) \varphi_n(x),$$
independent of the choice of $n$ through which $f$ factors. If $\varphi$ is a bounded distribution, then for any continuous function $f$ on $X$, the integral is defined by uniform approximation:
$$\int_X f \, d\varphi = \lim_{n \to \infty} \int_X f_n \, d\varphi$$
where $\{f_n\}$ is a sequence of locally constant functions converging uniformly to $f$.
