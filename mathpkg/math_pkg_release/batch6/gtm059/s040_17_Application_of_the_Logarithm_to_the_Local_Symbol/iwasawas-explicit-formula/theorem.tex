For the formal multiplicative group with $$K = \mathbb{Q}_p$$ and $$\pi = p$$, let $$\xi$$ be a primitive $$p^2$$-th root of unity. Then $$\pi_0 = \xi - 1$$ and

$$\lambda_f(X) = \log(1 + X), \quad \lambda_f(\mathfrak{u}_n) = \frac{1}{\mathfrak{u}_n} = \xi^{-1}.$$

Let $$x \in V_n = \mathbb{G}_m(V_n)$$. Then for $$m \geq 2n + 1$$:

$$(x, \pi_0) = \xi^{-1}, \quad \text{where } x = \frac{1}{\mathfrak{u}_n^2} T_n(\log(1 + x) + D_n \log x).$$

This is Iwasawa's formula.
