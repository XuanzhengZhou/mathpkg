Let $$w_n$$ be the basic Lubin-Tate generators satisfying

$$g(w_{n+1}) = w_n, \quad w_{n+1} + w_n = 0$$

with the relative different obtained by taking the derivative and evaluating at $$w_n$$ and $$w_{n+1}$$:

$$B_n = \mathfrak{D}_{K_n/K}, \quad \text{the different of } K_n \text{ over } K.$$

For $$x \in K_n$$ written as $$x = g(x_n)$$ with $$g(X) = X^p + \text{higher terms}$$ and $$c_n$$ a unit, we have

$$g'(X) = \frac{1}{2} X^p + \frac{1}{4} K(X).$$

Then $$g'(x)$$ is integral if $$n = 0$$, and in any case lies in $$\mathfrak{p}_1^n$$.
