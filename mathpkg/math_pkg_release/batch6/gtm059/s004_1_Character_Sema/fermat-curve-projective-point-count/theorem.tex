Let $\overline{N}$ be the number of points of the Fermat curve $V(d)$ in projective space over the finite field $F = \mathbb{F}_q$. Then
$$\overline{N} = 1 + q - \sum_{\substack{0 < a < d \\ 0 < b < d \\ a + b \not\equiv 0 \pmod{d}}} \chi_{a+b}(-1) \, J(\chi^a, \chi^b),$$
where $\chi_a$ is the truncated character and $J(\chi^a, \chi^b)$ is the Jacobi sum, with the sum taken over the same pairs $(a,b)$ as in Theorem 6.1.
