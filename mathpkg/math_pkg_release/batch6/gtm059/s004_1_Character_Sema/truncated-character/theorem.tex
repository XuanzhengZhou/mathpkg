Let $F = \mathbb{F}_q$ be a finite field, let $d \ge 2$ be an integer such that $d \mid (q-1)$, and let $w: F^* \to \boldsymbol{\mu}_{q-1}$ be the Teichmüller character. Define
$$\chi(u) = w(u)^{(q-1)/d},$$
so that $\chi$ is a multiplicative character of $F^*$ of order $d$.

For an integer $a$ modulo $d$, the truncated character $\chi_a$ is defined for $u \in F$ by:
$$\chi_a(u) = \begin{cases}
\chi(u)^a & \text{if } u \neq 0, \\[4pt]
1 & \text{if } u = 0 \text{ and } a = 0, \\[4pt]
0 & \text{if } u = 0 \text{ and } a \neq 0.
\end{cases}$$
