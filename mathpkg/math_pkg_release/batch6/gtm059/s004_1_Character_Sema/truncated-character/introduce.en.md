The truncated character $\chi_a$ is an extension of the multiplicative character $\chi^a$ from the multiplicative group $F^*$ to the full field $F$. At $u = 0$, the character takes the value $1$ if $a = 0$ (the trivial character) and $0$ otherwise. This convention is standard when using character sums to count solutions to equations over finite fields: the value $0$ at the origin ensures that the sum $\sum_{u \in F} \chi_a(u)$ correctly counts elements according to whether they are $d$-th powers. This definition is used in Lang's treatment of point counting on the Fermat curve.
