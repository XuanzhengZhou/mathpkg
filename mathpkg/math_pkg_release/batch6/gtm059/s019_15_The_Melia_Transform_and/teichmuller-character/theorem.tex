Let $\omega$ be the Teichmüller character. If $p$ is odd, then

$$\omega: \mathbb{Z}_p^{\times} \to \mu_{p-1}$$

is the character such that $\omega(a) \equiv a \pmod{p}$. If $p = 2$, then we define $\omega(a) = \pm 1$ such that $\omega(a) \equiv a \pmod{4}$.

Then we can write uniquely an element $a \in \mathbb{Z}_p^{\times}$ as

$$a = \omega(a)\langle a \rangle,$$

where $\langle a \rangle \equiv 1 \pmod{p}$ if $p$ is odd, and $\langle a \rangle \equiv 1 \pmod{4}$ if $p = 2$.
