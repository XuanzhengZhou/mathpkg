Let $\mu$ be a measure on $\mathbb{Z}_p^{\times}$ and let $u$ be a $p$-adic variable. For any character $\chi$ such that $\chi(x)/c^2$ is not identically $1$, we define the $p$-adic $L$-function $L_p$ by

$$L_p(s, \chi) = \frac{1}{1 - \frac{\chi(c)}{c^2}} \int_{\mathbb{Z}_p^{\times}} \langle x \rangle^{s} \chi(x) \, d\mu(x).$$

By the lemma, this integral is analytic in $s$. The factor in front is analytic in $s$ as well. If $\chi$ is non-trivial, we can select $c$ such that $\chi(c) = 1$, and then the factor is also analytic at $s = 0$.
