Let
$$x = \prod_{i=1}^{n} (1 - \zeta^i) \quad \text{and} \quad x' = \prod_{i=1}^{n} (1 - \zeta'^{\,i}),$$
where $\zeta$ is a primitive $n$-th root of unity. Note that the character $\chi$ is even, so $\chi$ and $-\chi$ occur simultaneously in each product. Therefore the norm from $K$ to the maximal real subfield $F$ of any cyclotomic unit
$$\frac{1}{x} = \prod_{i=1}^{n} \frac{1}{x'}$$
is a unit in $F$, and the group generated by these norms (modulo $\pm 1$) is infinite cyclic, generated by a unit of order $3$ such that
$$x^p = x^0.$$

From Theorem 5.2 we conclude that
$$h_0 = \log v.$$
In $K^*$, since the norm map presents the norms of cyclotomic units in $K^*$, the index relation $h_0 = (E : C)$ follows, completing the proof of Theorem 5.3.
