\text{The integer } h^* \text{ is called the \textit{first factor}, and } h^- \text{ is called the \textit{second factor} of the class number } h \text{ of the cyclotomic field } K. \text{ The quotient } h/h^* \text{ is an integer, and this terminology corresponds to the \textit{plus} and \textit{minus} parts: } h^+ \text{ and } h^-.
