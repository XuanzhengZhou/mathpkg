This lemma provides the inductive step in the proof of the Iwasawa structure theorem. Given a relation matrix already in $(r-1)$-normal form (i.e., the first $r-1$ rows are already diagonalized with distinguished polynomial entries), it asserts that one can extend the diagonalization to an $r$-th row using only admissible operations. The proof proceeds by applying operation (O.1) with respect to the first $r-1$ rows, exploiting that any power of $p$ divides all remaining entries after suitable reductions. By iterating this lemma starting from $r=1$, the entire relation matrix is brought into diagonal form.
