Let $K_\infty / K$ be a $\mathbb{Z}_p$-extension with intermediate fields $K_n$ ($[K_n : K] = p^n$), satisfying condition IW. Let $C_n$ denote the $p$-primary part of the ideal class group of $K_n$. Then

$$C_n \cong C_1 / U_n$$

where $U_n$ is a certain submodule determined by the Galois action. Equivalently, for any $m \geq n$, the norm map $N_{K_m/K_n}: C_m \to C_n$ is surjective and its kernel is controlled by the Iwasawa algebra action.

More precisely, if $C = \varprojlim C_n$ (with respect to norm maps) is the Iwasawa module, then $C_n \cong C / \nu_n C$ where $\nu_n = (1+T)^{p^n} - 1$ in the Iwasawa algebra $\Lambda = \mathbb{Z}_p[[T]]$.
