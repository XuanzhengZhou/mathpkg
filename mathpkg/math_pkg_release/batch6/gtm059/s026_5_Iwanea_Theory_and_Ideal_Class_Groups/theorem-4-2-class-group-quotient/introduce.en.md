Theorem 4.2 establishes the precise relationship between the $p$-primary class groups at different levels of a $\mathbb{Z}_p$-extension. It shows that each $C_n$ is a quotient of the base class group $C_1$, with the kernel controlled by the Galois action. This result is fundamental to Iwasawa's proof that the $p$-part of the class number $h_n$ of $K_n$ grows as $p^{\mu p^n + \lambda n + \nu}$ for large $n$, where $\mu$, $\lambda$, $\nu$ are the Iwasawa invariants.
