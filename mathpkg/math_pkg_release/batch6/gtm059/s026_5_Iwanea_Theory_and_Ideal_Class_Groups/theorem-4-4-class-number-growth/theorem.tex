Let $K_\infty / K$ be a $\mathbb{Z}_p$-extension, and let $h_n$ be the class number of $K_n$. Let $p^{e_n}$ be the exact power of $p$ dividing $h_n$. Then there exist integers $\mu \geq 0$, $\lambda \geq 0$, and $\nu$ (the Iwasawa invariants) such that for all sufficiently large $n$,

$$e_n = \mu p^n + \lambda n + \nu.$$

The invariant $\mu$ vanishes if and only if the Iwasawa module $C = \varprojlim C_n$ is a finitely generated $\mathbb{Z}_p$-module. In particular, for the cyclotomic $\mathbb{Z}_p$-extension, a theorem of Ferrero-Washington states that $\mu = 0$.
