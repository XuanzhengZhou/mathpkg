The unit index $Q = (E : W \cdot E^+)$ is a fundamental invariant of an abelian number field $K$, measuring the ratio of the full unit group to the subgroup generated by roots of unity together with units from the maximal real subfield $K^+$. This index appears naturally in the factorization of the class number formula for imaginary abelian fields, where it relates the class number $h_K$ to the class number $h_{K^+}$ of the real subfield. For cyclotomic fields, the value of $Q$ is intimately connected to the structure of cyclotomic units.
