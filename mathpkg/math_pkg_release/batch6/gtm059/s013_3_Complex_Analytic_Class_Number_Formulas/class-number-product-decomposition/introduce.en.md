Theorem 3.1 of Lang's *Cyclotomic Fields* decomposes the class number $h_K$ of an abelian extension $K/\mathbb{Q}$ into a product of factors involving generalized Bernoulli numbers. The decomposition separates the contribution of the maximal real subfield $K^+$ and the relative factor $h^- = h/h^+$. The proof uses the functional equations of the zeta function and $L$-series, comparing the behavior under the transformation $s \mapsto 1-s$. This decomposition is fundamental for the subsequent $p$-adic analysis of class numbers via Leopoldt's $p$-adic $L$-functions.
