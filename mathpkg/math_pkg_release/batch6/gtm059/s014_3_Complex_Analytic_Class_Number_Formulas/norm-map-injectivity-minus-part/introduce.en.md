For an imaginary abelian extension $K/\mathbb{Q}$ with maximal real subfield $K^+$, complex conjugation $\tau$ generates $\operatorname{Gal}(K/K^+)$. The norm map $N_{K/K^+}$ acts as $1 + \tau$ on ideal classes. This theorem states that the restriction of the norm map to the minus eigenspace $C_K^-$ is injective, which implies that $C_K^-$ embeds into $C_{K^+}$. The result is a foundational step in Iwasawa's work on the minus part of the class number and is proved via class field theory using the Hilbert class field and the Artin symbol.
