For a cyclotomic field $K = \mathbb{Q}(\zeta_m)$, the minus eigenspace $C_K^-$ of the ideal class group under complex conjugation is a subgroup of $C_K$ by definition. This theorem confirms that the inclusion $C_K^- \hookrightarrow C_K$ is injective, which makes the minus part of the class number $h^- = |C_K^-|$ a well-defined integer divisor of the full class number $h_K$. The minus part plays a central role in the analytic class number formula, where it admits an expression in terms of values of Dirichlet $L$-functions.
