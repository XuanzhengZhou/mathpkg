Let $H$ be the Hilbert class field of $K^+$ (the maximal abelian unramified extension of $K^+$). For any ideal class $c$ in $K^+$, the Artin symbol $(c, K/K^+)$ restricted to $H$ corresponds to the Artin symbol for $K^+$. By class field theory, the Artin symbol gives an isomorphism between the ideal class group $C_{K^+}$ and $\operatorname{Gal}(H/K^+)$. The kernel of the norm map $N_{K^+/K}$ corresponds to classes that become principal in $K$. Since $K/K^+$ is ramified at the archimedean primes (complex conjugation), $K$ cannot intersect $H$ nontrivially except at $K^+$ itself. Hence the norm map is injective.
