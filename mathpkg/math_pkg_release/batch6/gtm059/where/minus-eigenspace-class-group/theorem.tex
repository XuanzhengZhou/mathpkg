Let $K$ be a CM-field (in particular, a cyclotomic field) and let $C_K$ be its ideal class group. Let $\tau$ denote complex conjugation. Define the plus and minus eigenspaces:

$$C_K^+ = \{ c \in C_K : c^{\tau} = c \}, \quad C_K^- = \{ c \in C_K : c^{\tau} = c^{-1} \}.$$

Equivalently, $C_K^{\pm} = \{ c \in C_K : c^{1 \pm \tau} = 0 \}$ where the group operation is written multiplicatively.
