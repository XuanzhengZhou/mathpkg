For characters $\chi, \psi$ of the multiplicative group modulo $m$ with $\chi\psi \neq 1$, the Jacobi sum is defined as

$$J(\chi, \psi) = \sum_{a \bmod m} \chi(a)\psi(1-a).$$

Equivalently, if $a_1, a_2$ are integers with $a_1 + a_2 \not\equiv 0 \pmod{m}$, then

$$J(\chi, \psi) = J(\chi^2, \psi^2).$$
