Let $p$ be an odd prime and let $B_k$ denote the $k$-th Bernoulli number. For even integers $k$ with $2 \leq k \leq p-3$, the following congruence holds:

$$\frac{B_k}{k} \equiv \frac{B_{p-1+k}}{p-1+k} \pmod{p}.$$

Equivalently, if $p \nmid B_k$, then the Fermat quotient $(a^{p-1} - 1)/p$ is determined by Bernoulli numbers modulo $p$.
