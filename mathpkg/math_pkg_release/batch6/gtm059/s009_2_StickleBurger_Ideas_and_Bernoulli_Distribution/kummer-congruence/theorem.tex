Let $k$ be an even positive integer with $k \not\equiv 0 \pmod{p-1}$. Let $a$ be any
integer. Then the values

$$\frac{B_k}{k} \pmod{p^a}$$

depend only on $k \bmod (p-1)p^{a-1}$. More precisely, if $k_1 \equiv k_2 \pmod{(p-1)p^{a-1}}$
and $k_1, k_2 \not\equiv 0 \pmod{p-1}$, then

$$(1 - p^{k_1-1}) \frac{B_{k_1}}{k_1} \equiv (1 - p^{k_2-1}) \frac{B_{k_2}}{k_2} \pmod{p^a}.$$
