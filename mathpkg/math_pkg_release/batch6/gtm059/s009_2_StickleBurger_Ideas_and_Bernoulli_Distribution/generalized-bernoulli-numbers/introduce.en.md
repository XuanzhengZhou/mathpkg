Generalized Bernoulli numbers $B_{k,\chi}$ for a Dirichlet character $\chi$ generalize the classical Bernoulli numbers. Introduced by Leopoldt, they interpolate special values of Dirichlet $L$-functions: $L(1-k, \chi) = -B_{k,\chi}/k$ for $k \geq 1$. The Stickelberger element is intimately connected to these numbers, as the Stickelberger ideal annihilates the class group through congruences involving $B_{1,\chi}$. The integral representation using the Bernoulli distribution provides a $p$-adic framework for studying these numbers.
