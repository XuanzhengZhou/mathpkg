This theorem extends the equality between the Hilbert symbol $(x, u)$ and the generalized symbol $[x, \mu[2] + u]$ to the case where the second argument is an arbitrary unit $u$. The result is conditional on the valuation of the parameter $i$ being sufficiently large, with separate bounds for odd $p$ and $p = 2$. The domain $d(x_i)$ is the domain of definition for the Lubin-Tate symbol, and $\mu[2]$ refers to the second level of the unit filtration. The proof reduces to a triangular set of generators for the units, which allows the verification to be reduced to specific cases.
