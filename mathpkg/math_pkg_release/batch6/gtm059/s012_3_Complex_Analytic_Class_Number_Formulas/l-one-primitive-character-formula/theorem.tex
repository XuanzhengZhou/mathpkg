Let $\chi$ be a primitive Dirichlet character modulo $m$. For $\Re(s) > 1$, let

$$L(s, \chi) = \sum_{n=1}^{\infty} \frac{\chi(n)}{n^s}$$

be the Dirichlet $L$-function associated to $\chi$. Let $\zeta = e^{2\pi i / m}$ be a primitive $m$-th root of unity and let

$$\tau(\chi) = \sum_{a=1}^{m} \chi(a) \zeta^a$$

be the Gauss sum of $\chi$. Then

$$L(1, \chi) = \frac{\tau(\chi)}{m} \sum_{a=1}^{m} \overline{\chi}(a) \log(1 - \zeta^a),$$

where the branch of the logarithm is chosen so that $-\pi < \Im(\log(1 - \zeta^a)) < \pi$.
