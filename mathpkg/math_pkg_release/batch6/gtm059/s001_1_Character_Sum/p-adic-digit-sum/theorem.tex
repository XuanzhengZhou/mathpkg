Let $p$ be a prime and let $q$ be a power of $p$. Write any integer $k \ge 0$ in its $p$-adic expansion
$$k = k_0 + k_1 p + k_2 p^2 + \cdots + k_{r} p^{r}, \qquad 0 \le k_i \le p-1.$$
Define the $p$-adic digit sum
$$s_p(k) = k_0 + k_1 + \cdots + k_r.$$
For an arbitrary integer $k$, extend $s_p(k)$ by $(q-1)$-periodicity, so that $s_p(k + (q-1)) = s_p(k)$ for all integers $k$. When the dependence on $q$ is needed, one also writes $s_{p,q}(k)$.
