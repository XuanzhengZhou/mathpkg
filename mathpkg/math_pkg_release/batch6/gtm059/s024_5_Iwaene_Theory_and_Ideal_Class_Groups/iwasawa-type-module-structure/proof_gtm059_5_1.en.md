**Proof.** The argument distinguishes two cases corresponding to those in Theorem 1.2. Case (i) is essentially unchanged: the module $V$ has the same $\mathbb{Z}_p$-free rank and the torsion part decomposes as in the proof of Theorem 1.2, with the exponents satisfying the same asymptotic linear formula up to a constant.

For the modified situation, note that $V$ is of Iwasawa type, so $V$ is a finitely generated $\mathbb{Z}_p$-module modulo a finite submodule. Let $W = V_{\text{tors}}$ be the torsion submodule. An argument similar to that of Theorem 1.2 shows that the $p$-adic valuation of $|V/(y^{p^n}-1)V|$ grows as

$$e_n = \mu p^n + \lambda n + \nu$$

for all sufficiently large $n$, where $\mu, \lambda$ are the Iwasawa invariants and $\nu$ is a constant. The difference from Theorem 1.2 is absorbed into the constant term $v_k$, which accounts for the finite $\mathbb{Z}_p$-module contribution from the definition of Iwasawa type. The finite quotient $V/U_0$ from the definition contributes an additive constant to the exponent formula, proving the theorem.
