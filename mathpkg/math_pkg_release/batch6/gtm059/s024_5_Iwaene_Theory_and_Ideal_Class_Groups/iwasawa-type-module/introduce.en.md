A module of Iwasawa type is a finitely generated torsion module over the Iwasawa algebra $\Lambda = \mathbb{Z}_p[[X]]$ with an additional finiteness condition on subquotients. This definition captures the essential properties of the projective limit of $p$-parts of ideal class groups in a $\mathbb{Z}_p$-extension. The structure theorem for such modules yields an asymptotic growth formula $|V_n| = p^{\mu p^n + \lambda n + \nu}$ for $n$ sufficiently large, where $\mu, \lambda, \nu$ are the Iwasawa invariants.
