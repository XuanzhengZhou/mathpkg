A $\Lambda$-module $V$ is said to be of Iwasawa type if there exist elements $v_1, \ldots, v_k \in V$ such that, letting $U$ be the $\Lambda$-submodule generated by $(v_1, \ldots, v_k)$, the quotient $V/U$ is finite for all layers $n$ in the $\mathbb{Z}_p$-extension. In particular, $V$ is a finitely generated torsion $\Lambda$-module.
