The proof proceeds by showing that the primitive elements generate the whole group using the distribution relation, and then proving linear independence by exhibiting a concrete distribution (the Bernoulli distribution) whose values on primitive elements achieve the maximal possible rank. For the prime power case, an induction argument using the distribution relation reduces any element to a combination of primitive ones. The independence follows because the rank of the Bernoulli distribution on primitive elements equals the number of primitive elements minus one, matching the single relation.
