Let $K$ be a CM field with maximal real subfield $K^+$. Let $N = [K : \mathbb{Q}]$, $h$ the class number of $K$, $h^+$ the class number of $K^+$, $R$ the regulator of $K$, and $R^+$ the regulator of $K^+$. Let $w$ be the number of roots of unity in $K$. Then the unit index $Q = (E : \mu_K E^+)$ satisfies

$$\frac{R}{R^+} = \frac{2^{N/2}}{Q} \cdot \frac{h}{h^+}.$$

In particular, we have the index relation

$$\left( E : \mu_K E^+ \right) = Q = \frac{2^{N/2} \, h \, R^+}{h^+ \, R}.$$

Furthermore, comparing the analytic class number formulas for $K$ and $K^+$ yields

$$\frac{h}{h^+} = \frac{Q}{2^{N/2}} \cdot \frac{R}{R^+},$$

which gives an expression for the relative class number $h/h^+$ (the "first factor") in terms of the unit index and the ratio of regulators.
