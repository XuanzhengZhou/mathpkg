Each entry of the Carlitz--Olson determinant $D_p = (R(gh^{-1}) - R(-gh^{-1}))_{g,h \in S}$ has absolute value at most $1$. The determinant has size $r = \frac{p-3}{2}$. Applying Hadamard's inequality,

$$h_p^+ = |\det D_p| \leq \prod_{i=1}^r \|\text{row}_i\|_2 \leq r^{r/2} = \left(\frac{p-3}{2}\right)^{\frac{p-3}{4}}.$$

This yields an elementary, though rapidly growing, upper bound for the class number of the maximal real subfield $\mathbb{Q}(\zeta_p + \zeta_p^{-1})$.
