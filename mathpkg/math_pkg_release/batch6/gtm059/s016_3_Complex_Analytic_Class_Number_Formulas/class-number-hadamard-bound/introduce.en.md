Since each entry of the Carlitz--Olson determinant lies in $\{-1, 0, 1\}$, Hadamard's classical determinant inequality immediately gives an elementary upper bound on the class number $h_p^+$. Carlitz observed that this bound, though crude, already shows $h_p^+$ grows at most like $p^{p/4}$. The bound can be refined using more sophisticated determinant estimates or by applying the method to a normalized version of the matrix.
