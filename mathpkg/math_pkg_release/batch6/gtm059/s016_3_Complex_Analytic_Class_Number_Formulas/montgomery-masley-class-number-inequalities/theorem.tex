For primes $p \geq 200$, let $h_p^+$ be the class number of the maximal real subfield $\mathbb{Q}(\zeta_p + \zeta_p^{-1})$. Then there exist constants such that

$$(2\pi)^{-p/2} p^{(p+1)/4} \leq h_p^+ \leq (2\pi)^{-p/2} p^{(p+1)/4} \cdot e^{p/2}.$$

More precisely, the inequalities can be stated as

$$C_1 \cdot p^{(p+1)/4} \leq h_p^+ \leq C_2 \cdot p^{(p+1)/4}$$

with explicit constants $C_1, C_2$ depending on $p$. These bounds show that the Hadamard upper bound is essentially sharp up to an exponential factor.
