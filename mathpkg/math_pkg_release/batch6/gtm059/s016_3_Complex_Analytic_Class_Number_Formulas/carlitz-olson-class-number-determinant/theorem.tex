Let $p$ be an odd prime and let $G = (\mathbb{Z}/p\mathbb{Z})^{\times}/\{\pm 1\}$. For an integer $a$ coprime to $p$, let $R(a)$ denote the smallest positive integer in the residue class of $a$ modulo $p$. Define the matrix

$$D_p = \bigl(R(gh^{-1}) - R(-gh^{-1})\bigr)_{g, h \in S}$$

where $S$ is a set of representatives for $G$. Then the class number $h_p^+$ of the maximal real subfield $\mathbb{Q}(\zeta_p + \zeta_p^{-1})$ satisfies

$$\pm h_p^+ = \det D_p.$$

The determinant has size $\frac{p-3}{2}$ and each entry is an integer with absolute value at most $1$.
