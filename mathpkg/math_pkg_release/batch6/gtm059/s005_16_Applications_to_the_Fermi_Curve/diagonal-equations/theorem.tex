A \textit{diagonal equation} over a field $K$ is an equation of the form
$$a_1 X_1^n + a_2 X_2^n + \cdots + a_k X_k^n = 0$$
with coefficients $a_i \in K^\times$. The solution-counting methods developed for the Fermat curve $X^n + Y^n = Z^n$ apply in essentially the same way to diagonal hypersurfaces over finite fields, with the zeta function factorizing in terms of Jacobi sums attached to the coefficient vector $(a_1, \dots, a_k)$.
