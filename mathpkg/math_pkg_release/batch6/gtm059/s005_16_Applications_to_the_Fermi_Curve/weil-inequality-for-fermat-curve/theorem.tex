For the Fermat curve $X^n + Y^n = Z^n$ defined over a finite field $\mathbf{F}_q$ (with $q = p^f$ and $p \nmid n$), the zeta function factorizes as
$$Z(V,T) = \prod_{\chi} L(T,\chi)$$
where the product runs over certain Hecke characters $\chi$ associated to the curve. Each $L$-factor $L(T,\chi)$ is of the form $1 - a_\chi T$ with $|a_\chi| = \sqrt{q}$, from which the Weil bounds on point counts $|N_\nu - (q^\nu + 1)| \leq 2g q^{\nu/2}$ follow.
