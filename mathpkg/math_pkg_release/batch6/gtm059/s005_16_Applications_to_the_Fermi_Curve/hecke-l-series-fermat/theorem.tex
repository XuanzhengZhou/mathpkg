The zeta function $Z(V,T)$ of the Fermat curve $V: X^n + Y^n = Z^n$ over $\mathbf{F}_q$ satisfies
$$Z(V,T) = \prod_{j=0}^{n-1} \prod_{k=0}^{n-1} (1 - J(\chi^j, \chi^k) \, T)$$
where $\chi$ is a multiplicative character of order $n$ on $\mathbf{F}_q^\times$ and $J(\chi^j, \chi^k)$ are Jacobi sums. Each factor corresponds to a Hecke $L$-series, so that up to an elementary factor coming from the zeta function of $\mathbf{Q}(\zeta_n)$, the zeta function factorizes as a product of Hecke $L$-functions.
