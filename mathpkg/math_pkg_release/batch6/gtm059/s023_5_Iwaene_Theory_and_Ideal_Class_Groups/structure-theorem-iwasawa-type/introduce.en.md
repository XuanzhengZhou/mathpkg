Theorem 1.3 extends the asymptotic formula of Theorem 1.2 to modules of Iwasawa type, which need not be finitely generated over $\Lambda$. The proof adapts the case analysis to the Iwasawa type setting, showing that the same asymptotic structure $s_n(V) = \mu p^n + \lambda n + \nu$ holds.
