The proof follows the same lines as Theorem 1.2. Case (i) (free part and $p$-power torsion) is unchanged. For case (ii), an argument similar to that of Theorem 1.2 shows that $s_n(V) = \lambda n + \nu$ for all $n \ge n_0$, where $\lambda$ is the $\mathbb{Z}_p$-rank. Letting $W$ be a suitably chosen submodule, one obtains $s_n(V) = \lambda n + d(n-n_0) + c$ for some constants $c, d$. Since $W$ has the same $\mathbb{Z}_p$-free rank as $V$, the result follows with the appropriate modified constants.
