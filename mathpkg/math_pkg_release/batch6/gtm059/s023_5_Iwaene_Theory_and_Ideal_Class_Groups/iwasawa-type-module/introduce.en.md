A module of Iwasawa type is a $\Lambda$-module $V$ for which there exists a finite set of elements such that the submodule generated by $(\gamma-1)V$ together with those elements has finite index in $V$. This technical condition is used to extend the structure theorem (Theorem 1.2) to modules that may not be finitely generated over $\Lambda$, but still satisfy the conclusion $s_n(V) = \mu p^n + \lambda n + \nu$ with modified constants.
