Let $M$ be a finitely generated $\Lambda$-module, where $\Lambda = \mathbb{Z}_p[[X]]$ is the Iwasawa algebra. Then there exists a quasi-isomorphism (i.e., a $\Lambda$-module homomorphism with finite kernel and cokernel)
$$M \sim \Lambda^r \oplus \bigoplus_{i} \Lambda/(p^{\mu_i}) \oplus \bigoplus_{j} \Lambda/(f_j^{e_j})$$
where:
\begin{itemize}
\item $r \ge 0$ is the $\Lambda$-rank of $M$,
\item each $\mu_i \ge 1$,
\item each $f_j$ is an irreducible distinguished polynomial,
\item each $e_j \ge 1$.
\end{itemize}
The factors $\Lambda^r$, $\Lambda/(p^{\mu_i})$, and $\Lambda/(f_j^{e_j})$ are uniquely determined up to order.
