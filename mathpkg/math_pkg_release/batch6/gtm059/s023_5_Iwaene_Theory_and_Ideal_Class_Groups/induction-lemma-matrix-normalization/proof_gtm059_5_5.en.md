Using operation O1 with respect to each of the first $r-1$ rows, one may assume without loss of generality that any given power of $p$ divides all elements of $R$ as needed for the induction. Among the remaining rows, select an entry with minimal Weierstrass degree. By row/column operations and the Weierstrass Division Theorem, one can clear out all other entries in the same row and column corresponding to this entry, placing a distinguished polynomial in the $(r,r)$ position and zeros elsewhere in the $r$-th row and column. This yields the $r$-normal form while preserving the first $r-1$ diagonal entries. The process relies on the fact that admissible operations correspond to embeddings/surjections with finite kernel/cokernel, so the module up to quasi-isomorphism is unchanged.
