The proof uses class field theory: $C_n$ is isomorphic via the Artin map to the Galois group of the maximal abelian unramified $p$-extension of $K_n$ (the $p$-Hilbert class field). The norm map on ideal classes corresponds to the restriction map on Galois groups. Since $K_n/K_{n-1}$ is totally ramified at the primes above the ramified primes (condition IW), the norm map $C_n \to C_{n-1}$ is surjective with kernel described by the inertia group. Applying Nakayama's Lemma to the projective limit yields the structural relationship.
