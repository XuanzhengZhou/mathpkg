Let $V$ be a finitely generated torsion $\Lambda$-module such that $V_n = V/(\gamma^{p^n}-1)V$ is finite for all $n$. Let $s_n(V) = \operatorname{ord}_p(|V_n|)$ be the $p$-adic order of $|V_n|$. Then there exist integers $\mu, \lambda, \nu$ and an integer $c$ such that for all sufficiently large $n$,
$$s_n(V) = \mu p^n + \lambda n + \nu.$$

More precisely, if $V$ is quasi-isomorphic to
$$\Lambda^r \oplus \bigoplus_i \Lambda/(p^{\mu_i}) \oplus \bigoplus_j \Lambda/(f_j(T))$$
where each $f_j$ is a distinguished polynomial, then:
\begin{itemize}
\item[(i)] If $V = \Lambda/(p^m)$ then $s_n(V) = m p^n$.
\item[(ii)] If $V = \Lambda/(f)$ with $f$ distinguished of degree $d$, and $V_n$ is finite for all $n$, then $s_n(V) = d n + c$ for some constant $c$.
\item[(iii)] For general finitely generated torsion $V$, $s_n(V) = \mu p^n + \lambda n + \nu$ for $n \gg 0$, with $\mu = \sum m_i$, $\lambda = \sum \deg f_j$.
\end{itemize}
