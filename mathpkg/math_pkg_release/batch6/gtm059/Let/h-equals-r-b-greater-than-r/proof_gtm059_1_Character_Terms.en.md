Note that $m + I$ because
$$m = -(i_{n+1} - (1 + m)).$$

Suppose that an element of $R$ lies in $I$, that is
$$\sum i_{n+1} a_{n+1} B > R$$
with $z(i) = \mathbb{Z}$. Then
$$\sum i_{n+1} \left(\frac{h}{m}\right) B > \mathbb{Z}$$
for all $c$, whence
$$\sum i_{n+1} b = 0 \pmod{m},$$
and $\sum i_{n+1} b$ is in $I$. But then
$$\sum i_{n+1} b = \sum i_{n+1} (a-b) + \sum i_{n+1} b$$
is in $I$, thus proving the lemma.
