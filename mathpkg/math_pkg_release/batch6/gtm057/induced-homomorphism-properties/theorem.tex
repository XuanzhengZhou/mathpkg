For a continuous map $f: X \to Y$:
(i) If $\epsilon$ is an identity, then so is $f_*\epsilon$.
(ii) If $\alpha \cdot \beta$ is defined, then $f_*(\alpha \cdot \beta) = f_*\alpha \cdot f_*\beta$.
(iii) If $f = \mathrm{id}_X$, then $f_* = \mathrm{id}_{\pi(X,p)}$.
(iv) If $X \xrightarrow{f} Y \xrightarrow{g} Z$, then $(gf)_* = g_* f_*$.
