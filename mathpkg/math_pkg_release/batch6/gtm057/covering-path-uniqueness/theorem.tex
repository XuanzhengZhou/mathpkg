For any continuous mapping $h: E \to \mathbb{R}/3\mathbb{Z}$ where $E = [0,\sigma] \times [0,\tau]$, and any real number $x \in \mathbb{R}$ with $\phi(x) = h(0,0)$, there exists one and only one continuous function $\bar{h}: E \to \mathbb{R}$ such that $\bar{h}(0,0) = x$ and $h = \phi \circ \bar{h}$.
