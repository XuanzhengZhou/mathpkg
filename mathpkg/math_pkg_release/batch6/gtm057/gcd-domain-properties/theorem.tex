An element $d$ is a \textit{greatest common divisor} (g.c.d.) of $a_1, \ldots, a_n$ in a commutative ring $R$ if $d \mid a_i$ for all $i$ and any common divisor $e$ divides $d$. Any two g.c.d.'s are associates. A \textit{unique factorization domain} (UFD) is an integral domain where every nonzero non-unit factors essentially uniquely into primes. Every UFD is a g.c.d. domain.
