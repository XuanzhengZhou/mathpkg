A convex set can be deformed linearly to any of its points. For any $p$-based loop $a$, the deformation $h_s(t) = (1-s)a(t) + sp$ is a fixed-endpoint family showing $a \simeq e_p$. Thus every loop is equivalent to the constant loop.
