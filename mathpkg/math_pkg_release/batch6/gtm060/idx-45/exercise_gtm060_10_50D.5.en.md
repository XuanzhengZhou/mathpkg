Show that, if a system is nondegenerate, then in any neighborhood of any point there is a conditionally periodic motion with $n$ frequencies, and also with any smaller number of frequencies.

**Hint.** We can take the frequencies $\omega$ themselves instead of the variables $I$ as local coordinates. In the space of collections of frequencies, the set of points $\omega$ with any number of relations $r$ $(0 \leq r < n)$ is dense.
