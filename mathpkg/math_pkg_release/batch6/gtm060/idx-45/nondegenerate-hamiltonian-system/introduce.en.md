In an integrable Hamiltonian system written in action-angle variables $(I, \varphi)$, the frequency vector $\omega(I) = \partial H / \partial I$ depends on the action variables. The system is called nondegenerate when the Hessian of the Hamiltonian with respect to the action variables has nonzero determinant. This condition ensures that the frequency map $I \mapsto \omega(I)$ is a local diffeomorphism, which guarantees that invariant tori with independent frequencies are dense in phase space.
