An integrable Hamiltonian system $H_0(I)$ in action-angle variables $(I, \varphi)$ is called **nondegenerate** if the frequency map
$$\omega(I) = \frac{\partial H_0}{\partial I}$$
is a local diffeomorphism, i.e., if the frequencies are functionally independent. The condition for nondegeneracy is
$$\det\left| \frac{\partial \omega}{\partial I} \right| = \det\left| \frac{\partial^2 H_0}{\partial I^2} \right| \neq 0.$$

In a nondegenerate system, the unperturbed problem determines conditionally-periodic motions with different frequencies on different invariant tori in phase space. The non-resonant tori (those on which the number of independent frequencies is maximal, i.e., $n$) form a dense set of full Lebesgue measure in phase space, while resonant tori form a dense set of measure zero.
