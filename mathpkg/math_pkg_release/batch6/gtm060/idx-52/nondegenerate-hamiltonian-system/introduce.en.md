A Hamiltonian system is nondegenerate if the Hessian of the unperturbed Hamiltonian with respect to the action variables is non-singular. This means the frequency vector varies from torus to torus, so different invariant tori carry incommensurate frequencies. Nondegeneracy is a key hypothesis of Kolmogorov's theorem (KAM theory): it ensures that most invariant tori survive small perturbations. In a nondegenerate system, non-resonant tori form a set of full measure, while resonant tori are dense but of measure zero.
