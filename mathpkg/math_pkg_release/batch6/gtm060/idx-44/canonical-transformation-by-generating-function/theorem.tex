Theorem. Set
$$S(I, q) = \int_{q_0}^{q} p \, dq \bigg|_{H=h(I)}.$$
Then the formulas
$$p = \frac{\partial S(I, q)}{\partial q}, \quad \varphi = \frac{\partial S(I, q)}{\partial I}, \quad H\left(\frac{\partial S(I, q)}{\partial q}, q\right) = h(I)$$
give a canonical transformation $p, q \to I, \varphi$ satisfying the conditions that $I$ is a first integral and $\varphi$ evolves linearly in time.
