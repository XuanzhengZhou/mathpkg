For any partition $$I + J$$ of the set of indices $$(1, \ldots, n)$$ into two disjoint subsets and for any function $$S(x_I, y_J)$$ of $$n$$ variables $$x_i, i \in I$$, $$y_j, j \in J$$, the formulas

$$y_I = -\frac{\partial S}{\partial x_I}, \quad x_J = \frac{\partial S}{\partial y_J}, \quad z = S - x_I\frac{\partial S}{\partial x_I}$$

define a Legendre submanifold of the contact space with contact form $$\omega = x\,dy + dz$$.
