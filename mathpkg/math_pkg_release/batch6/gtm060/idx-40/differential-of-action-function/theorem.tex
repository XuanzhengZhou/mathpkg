The differential of the action function (for a fixed initial point) is equal to
$$dS = \mathbf{p} \, d\mathbf{q} - H \, dt$$
where $\mathbf{p} = \partial L / \partial \dot{\mathbf{q}}$ and $H = \mathbf{p} \dot{\mathbf{q}} - L$ are defined with the help of the terminal velocity $\dot{\mathbf{q}}$ of the trajectory $\gamma$.
