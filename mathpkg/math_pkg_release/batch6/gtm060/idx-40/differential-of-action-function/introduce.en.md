This theorem expresses the differential of the action function $S(\mathbf{q}, t)$ in terms of canonical variables. For a fixed initial point $(\mathbf{q}_0, t_0)$, the variation of the action along an extremal is given by $dS = \mathbf{p} \, d\mathbf{q} - H \, dt$, where $\mathbf{p} = \partial L / \partial \dot{\mathbf{q}}$ is the momentum and $H = \mathbf{p} \dot{\mathbf{q}} - L$ is the Hamiltonian, both evaluated at the terminal point. This result bridges Lagrangian and Hamiltonian mechanics, showing that the action function serves as a generating function for the canonical momentum and Hamiltonian.
