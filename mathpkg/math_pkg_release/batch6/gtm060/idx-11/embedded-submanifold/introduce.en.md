An embedded $k$-dimensional submanifold of $\mathbb{R}^n$ is a subset $M$ locally defined as the common zero set of $n-k$ smooth functions whose gradients are linearly independent at each point. The linear independence condition ensures, via the implicit function theorem, that $M$ has the local structure of $\mathbb{R}^k$ and can be given a differentiable manifold structure. Every abstract manifold can be embedded in some Euclidean space.
