**Proof.** Cf. Section 6B. The proof is identical to the single-particle case: if the work integral is path-independent, define $U(\mathbf{r}) = -\int_{\mathbf{r}_0}^{\mathbf{r}} (\mathbf{F}, d\mathbf{r})$, where the integral is taken along any path from a fixed reference point $\mathbf{r}_0$ to $\mathbf{r}$. Then $\mathbf{F} = -\partial U / \partial \mathbf{r}$. Conversely, if $\mathbf{F} = -\partial U / \partial \mathbf{r}$ for some function $U$, then by the gradient theorem the work integral depends only on the endpoints.
