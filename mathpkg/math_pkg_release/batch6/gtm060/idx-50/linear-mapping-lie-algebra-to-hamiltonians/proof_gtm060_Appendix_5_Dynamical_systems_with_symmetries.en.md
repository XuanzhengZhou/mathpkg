Each element $a \in \mathfrak{g}$ generates a one-parameter subgroup of $G$, which by the Poisson action condition acts as a locally Hamiltonian phase flow on $M$. By assumption, this flow admits a single-valued Hamiltonian function $H_a$. The map $a \mapsto H_a$ is defined up to an additive constant for each $a$.

Choose a basis $\{e_1, \ldots, e_k\}$ of the Lie algebra $\mathfrak{g}$. Select arbitrary constants for the Hamiltonian functions $H_{e_1}, \ldots, H_{e_k}$ corresponding to the basis elements. For any element $a = \sum_i c_i e_i$, define $H_a = \sum_i c_i H_{e_i}$. This yields a linear mapping $a \mapsto H_a$ from $\mathfrak{g}$ to the space of Hamiltonian functions on $M$.
