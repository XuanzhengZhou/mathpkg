On the hypersurface $E^{2n}$ in the contact manifold $M^{2n+1}$, the characteristic vector field $\xi$ satisfies

$$L_\xi \alpha = c\alpha$$

where $c$ is a smooth function in a neighborhood of $x \in E^{2n}$. Moreover, let $\{g^t\}$ be the (local) phase flow of the field $\xi$ and $\eta$ a vector tangent to $E^{2n}$. Set $\eta(t) = g^t_* \eta$ and $y(t) = \alpha(\eta(t))$. Then the function $y$ satisfies the linear differential equation

$$\frac{dy}{dt} = c(t)\, y(t).$$
