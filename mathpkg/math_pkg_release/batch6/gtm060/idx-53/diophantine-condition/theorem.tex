A set of frequencies $\omega = (\omega_1, \ldots, \omega_n)$ satisfies the Diophantine non-resonance condition if there exist constants $C > 0$ and $\nu > 0$ such that
$$|(\omega, k)| > C|k|^{-\nu} \quad \text{for all integral vectors } k \in \mathbb{Z}^n \setminus \{0\},$$
where $(\omega, k) = \sum_{i=1}^n \omega_i k_i$ and $|k| = \sum_{i=1}^n |k_i|$.

If $\nu$ is sufficiently large (say $\nu = n + 1$), then the measure of the set of frequency vectors $\omega$ (lying in a fixed bounded region) for which this condition is violated, is small when $C$ is small.
