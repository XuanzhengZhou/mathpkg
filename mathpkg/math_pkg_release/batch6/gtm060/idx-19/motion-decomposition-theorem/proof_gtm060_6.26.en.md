Set $r(t) = D_t 0$ and $B_t = C_t^{-1} D_t$, where $C_t q = q + r(t)$. Then $B_t 0 = 0$, so $B_t$ is a linear operator preserving the metric and orientation, i.e., a rotation. The decomposition $D_t = C_t B_t$ is unique because $r(t)$ is uniquely determined as $D_t 0$, and then $B_t = C_t^{-1} D_t$ is uniquely determined.
