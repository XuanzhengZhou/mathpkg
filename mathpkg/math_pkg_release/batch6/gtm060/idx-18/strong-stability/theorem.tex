The trivial solution of a Hamiltonian linear system is called \textbf{strongly stable} if it is stable, and if the trivial solution of every sufficiently close linear Hamiltonian system is also stable.

Here the distance between two linear systems with periodic coefficients $\dot{x} = B_1(t)x$ and $\dot{x} = B_2(t)x$ is measured by
$$\int_0^T \|B_1(t) - B_2(t)\| \, dt.$$
