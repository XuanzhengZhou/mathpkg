Let $\dot{x} = f(x, t)$ be a system of differential equations with periodic coefficients of period $T$, and let $A = g^T$ be the mapping at a period.

\begin{enumerate}
\item The point $x_0$ is a fixed point of the mapping $A$ ($A x_0 = x_0$) if and only if the solution with initial conditions $x(0) = x_0$ is periodic with period $T$.
\item The periodic solution $x(t)$ is Liapunov stable (asymptotically stable) if and only if the fixed point $x_0$ of the mapping $A$ is Liapunov stable (asymptotically stable).
\item If the system is linear, i.e., $f(x, t) = f(t)x$ is a linear function of $x$, then $A$ is linear.
\item If the system is Hamiltonian, then $A$ preserves volume: $\det A_* = 1$.
\end{enumerate}
