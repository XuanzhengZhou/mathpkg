Consider the equation
$$\ddot{x} = -\omega^2(1 + \varepsilon a(t))x, \quad \varepsilon \ll 1,$$
where $a(t + 2\pi) = a(t)$ is periodic with period $2\pi$.

All points on the $\omega$-axis (i.e., $\varepsilon = 0$) except the integers and half-integers $\omega = k/2$, $k = 0, 1, 2, \ldots$, correspond to strongly stable systems.
