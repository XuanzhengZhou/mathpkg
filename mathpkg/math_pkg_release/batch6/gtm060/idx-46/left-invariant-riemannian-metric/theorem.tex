A riemannian metric on a Lie group $G$ is called \textbf{left-invariant} if it is preserved by all left translations $L_g$, i.e., the derivative of left translation carries every vector to a vector of the same length. A left-invariant metric is completely determined by a euclidean structure (inner product) on the Lie algebra $\mathfrak{g} = TG_e$, given by a symmetric positive-definite linear operator $A: \mathfrak{g} \to \mathfrak{g}^*$:

$$\langle A\xi, \eta \rangle = \langle A\eta, \xi \rangle, \quad \text{for all } \xi, \eta \in \mathfrak{g}.$$

The metric on $TG_g$ is then defined by left translation: $A_g \xi = L_{g^{-1}}^* A L_{g^{-1}} \xi$.
