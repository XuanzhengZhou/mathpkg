Lagrange'\''s equations for a mechanical system with lagrangian $L(\mathbf{q}, \dot{\mathbf{q}}, t) = T - U$ are

$$\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_i} \right) - \frac{\partial L}{\partial q_i} = 0, \quad i = 1, \ldots, n.$$

Equivalently, using generalized momenta $p_i = \partial L / \partial \dot{q}_i$ and generalized forces $\partial L / \partial q_i$,

$$\dot{p}_i = \frac{\partial L}{\partial q_i}.$$
