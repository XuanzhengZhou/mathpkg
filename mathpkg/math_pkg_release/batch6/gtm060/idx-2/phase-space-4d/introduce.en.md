For a two-degree-of-freedom system, the phase space is four-dimensional, with coordinates $(x_1, x_2, y_1, y_2) = (x_1, x_2, \dot{x}_1, \dot{x}_2)$.
The phase flow is a one-parameter group of diffeomorphisms of this four-dimensional space.
All of phase space is partitioned into phase curves, which project onto the configuration plane $(x_1, x_2)$ as orbits that can intersect even when the corresponding phase curves do not.
