A phase curve could consist of only one point. Such a point is called an \textit{equilibrium position}.
The vector of phase velocity at an equilibrium position is zero.
The point $x = \xi, y = 0$ is an equilibrium position if and only if $\xi$ is a critical point of the potential energy, i.e., $\left.\frac{dU}{dx}\right|_{x=\xi} = 0$.
