The phase plane is the two-dimensional space with coordinates $(x, y) = (x, \dot{x})$ in which the first-order system equivalent to $\ddot{x} = f(x)$ is studied.
Each point of the phase plane represents a state of the mechanical system, and the phase velocity vector field assigns a velocity vector $(y, f(x))$ to each point.
The phase plane provides a geometric picture of all possible motions without solving the differential equation explicitly.
