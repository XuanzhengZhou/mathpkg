A short-wave initial condition for the Schrödinger equation is a function of the form
$$\psi|_{t=0} = \varphi(q) e^{(i/h)s(q)},$$
where:
\begin{itemize}
\item $\varphi(q)$ is a smooth amplitude function, nonzero only inside some bounded region,
\item $s(q)$ is a smooth real-valued phase function,
\item $h$ is the reduced Planck constant, which is taken as a small parameter ($h \to 0$).
\end{itemize}
