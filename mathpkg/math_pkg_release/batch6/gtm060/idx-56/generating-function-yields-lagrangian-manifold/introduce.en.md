This lemma establishes the fundamental correspondence between smooth generating functions and Lagrangian submanifolds in phase space. Given any smooth function $s(q)$ on the configuration space, its gradient defines a map $p(q) = \partial s / \partial q$, whose graph is automatically Lagrangian with respect to the canonical symplectic form. Conversely, any Lagrangian submanifold that projects diffeomorphically onto the configuration space arises from such a generating function. This relationship is central to the construction of short-wave asymptotics and the WKB method.
