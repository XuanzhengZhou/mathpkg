For any smooth function $s(q)$, define the momentum as the gradient of the phase:
$$p(q) = \frac{\partial s}{\partial q}.$$
Then the graph of the function $p(q)$ in the phase space $\mathbb{R}^{2n} = \{(p,q)\}$ is a Lagrangian manifold.

Conversely, if a Lagrangian manifold projects diffeomorphically onto the $q$-space (i.e., it is a graph over the configuration space), then it is given by some generating function $s(q)$ according to the formula $p(q) = \frac{\partial s}{\partial q}$.
