This theorem classifies all generic Poisson structures on a two-dimensional manifold up to local diffeomorphism. On a 2-manifold, the Jacobi identity is automatically satisfied, so any smooth bivector field $f(x,y) \partial_x \wedge \partial_y$ defines a Poisson structure via $\{x,y\} = f(x,y)$. The classification yields a hierarchy of singular points labeled by the ADE type of the zero locus $f=0$: type $A_0$ (nonsingular, $f = y$), type $A_{2k}$ and $A_{2k-1}^a$, type $D_{2k}^{a,b}$ and $D_{2k+1}^a$, and the exceptional types $E_6$, $E_7^a$, $E_8$. The parameters $a, b$ are continuous moduli. A generic Poisson structure can be reduced near each point to either $\{x,y\} = 1$ or $\{x,y\} = y$.
