A symplectic diffeomorphism of the annulus

$$(x, y) \mapsto (X, Y)$$

is defined with the help of the generating function $Xy + S(X, y)$, where the function $S$ is $2\pi$-periodic in $y$. For this to be a diffeomorphism we need that $\partial X / \partial x \neq 0$. Then

$$dS = (x - X)\,dy + (Y - y)\,dX.$$
