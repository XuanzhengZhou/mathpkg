The generating function $Xy + S(X, y)$ provides a way to describe symplectic diffeomorphisms of an annulus via a single scalar function $S$ that is $2\pi$-periodic in the angular variable $y$. This construction encodes both the radial and angular components of the mapping. The differential relation $dS = (x - X)dy + (Y - y)dX$ links the generating function to the original and image coordinates, and reveals that fixed points of the diffeomorphism correspond to critical points of the function $F(x, y) = S(X(x, y), y)$.
