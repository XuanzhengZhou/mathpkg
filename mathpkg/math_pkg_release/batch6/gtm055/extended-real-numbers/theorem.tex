The \textbf{extended real number system} is $\overline{\mathbb{R}} = \mathbb{R} \cup \{+\infty, -\infty\}$ with the following conventions for $t \in \mathbb{R}$:
$$t \pm \infty = \pm \infty, \quad \infty + \infty = +\infty, \quad -\infty - \infty = -\infty,$$
$$t(\pm \infty) = (\pm \infty)t = \begin{cases} \pm \infty, & t > 0, \\ 0, & t = 0, \\ \mp \infty, & t < 0, \end{cases}$$
$$(\pm \infty)(\pm \infty) = +\infty, \quad (\pm \infty)(\mp \infty) = -\infty.$$
The expressions $(\pm \infty) + (\mp \infty)$ and $(\pm \infty) - (\pm \infty)$ remain undefined. For $t \in \overline{\mathbb{R}}$, write $t^+ = t \vee 0$, $t^- = -(t \wedge 0)$, and $|t| = t^+ + t^-$. Elements of $\mathbb{R}$ are called \textbf{finite} (real) numbers.
