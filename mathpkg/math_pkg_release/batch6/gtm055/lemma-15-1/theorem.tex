For any normed space $\mathcal{E}$ and for each $f$ in $\mathcal{E}^*$ the function $\sigma_f$ defined in (1) is a pseudonorm on $\mathcal{E}$. Moreover, if $\mathcal{T}$ denotes an arbitrary linear topology on $\mathcal{E}$, then $f$ is continuous with respect to $\mathcal{T}$ if and only if $\sigma_f$ is. Hence $f$ is continuous with respect to $\mathcal{T}$ if and only if $\mathcal{T}$ refines the pseudometric topology induced by $\sigma_f$, and the weak topology on $\mathcal{E}$ coincides with the topology induced by the family of pseudonorms $\{\sigma_f\}_{f \in \mathcal{E}^*}$.
