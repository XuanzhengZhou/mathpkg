Let $\mathcal{E}$ be a locally convex topological linear space.

(i) For any subset $M \subset \mathcal{E}$, the preannihilator $^a(M^a)$ coincides with the smallest closed linear manifold in $\mathcal{E}$ that contains $M$.

(ii) For any subset $M \subset \mathcal{E}^*$, the annihilator $(^a M)^a$ coincides with the smallest weak$^*$ closed linear submanifold of $\mathcal{E}^*$ that contains $M$.
