If $\{x_{\gamma}\}_{\gamma \in \Gamma}$ and $\{y_{\gamma}\}_{\gamma \in \Gamma}$ are two summable families of vectors in a normed space $\mathcal{E}$, both indexed by the same index set $\Gamma$, and if

$$x = \sum_{\gamma \in \Gamma} x_{\gamma} \quad \text{and} \quad y = \sum_{\gamma \in \Gamma} y_{\gamma},$$

then, for any pair of scalars $\alpha$ and $\beta$,

$$\sum_{\gamma \in \Gamma} (\alpha x_{\gamma} + \beta y_{\gamma}) = \alpha x + \beta y.$$

Furthermore, if $T$ is any bounded linear transformation of $\mathcal{E}$ into some normed space $\mathcal{F}$, then

$$Tx = \sum_{\gamma \in \Gamma} Tx_{\gamma}.$$

Proof. These facts follow immediately from the definition of summability. $\square$
