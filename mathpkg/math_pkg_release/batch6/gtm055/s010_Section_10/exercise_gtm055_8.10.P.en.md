A measure space $(X, \mathbf{S}, \mu)$ is said to be complete if every subset of an arbitrary set of measure zero with respect to $\mu$ is measurable and is, therefore, also a set of measure zero. (Another way to say this is to say that the collection $\mathcal{Z}$ of null sets is a $\sigma$-ideal not only in $\mathbf{S}$, but in the power class on $X$.) If $(X, \mathbf{S}, \mu)$ is complete, the measure $\mu$ is also said to be complete. Show that if $(X, \mathbf{S}, \mu)$ is a complete measure space, and if $f$ is a measurable function on $(X, \mathbf{S})$, then every function $g$ on $X$ such that $g = f$ a.e. $[\mu]$ is also measurable $[\mathbf{S}]$. Similarly, if $(X, \mathbf{S}, \mu)$ is complete and $E$ belongs to $\mathbf{S}$, then every set $F$ that is almost equal $[\mu]$ to $E$ also belongs to $\mathbf{S}$ and we have $\mu(E) = \mu(F)$. If $\mathcal{Z}$ is a null set in a complete measure space $(X, \mathbf{S}, \mu)$, then an arbitrary complex-valued function defined on $\mathcal{Z}$ is measurable on $\mathcal{Z}$. Show that if $\mu^*$ is an outer measure on a set $X$, and if $\mathbf{M}$ denotes the $\sigma$-ring of sets that are measurable with respect to $\mu^*$, then $(X, \mathbf{M}, \mu^*|\mathbf{M})$ is complete.
