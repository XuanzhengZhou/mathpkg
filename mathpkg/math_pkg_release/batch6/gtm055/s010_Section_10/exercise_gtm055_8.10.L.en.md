(i) Show that a bounded function $\varphi$ is integrable over an interval $[a, b]$ with respect to a bounded function $\alpha$ if and only if the following Cauchy criterion is satisfied: Given $\varepsilon > 0$ there exists $\delta > 0$ such that if $\mathcal{P}$ and $\mathcal{P}'$ are arbitrary partitions of $[a, b]$ with mesh $\mathcal{P}, \mathcal{P}' < \delta$, and if $S$ and $S'$ are arbitrary Riemann--Stieltjes sums for $\varphi$ with respect to $\alpha$ based on $\mathcal{P}$ and $\mathcal{P}'$, respectively, then $|S - S'| < \varepsilon$.
