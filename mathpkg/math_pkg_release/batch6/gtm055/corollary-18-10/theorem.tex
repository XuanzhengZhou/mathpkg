If $X$ is a compact Hausdorff space, and if $\mathcal{F}$ is any strongly separating set of functions in $\mathcal{C}(X)$, then the linear submanifold of $\mathcal{C}(X)$ generated algebraically by the set of all finite products of elements of $\mathcal{F}$ and their complex conjugates is dense in $\mathcal{C}(X)$. Likewise, if $\mathcal{F}$ is a strongly separating subset of $\mathcal{C}_{\mathbb{R}}(X)$, the linear submanifold of $\mathcal{C}_{\mathbb{R}}(X)$ generated algebraically by the set of all finite products of elements of $\mathcal{F}$ is dense in $\mathcal{C}_{\mathbb{R}}(X)$.

Proof. In both cases the system described is a self-conjugate strongly separating subalgebra.

Hence the complex polynomial functions $p(t)$ are dense in $\mathcal{C}([a, b])$ and the real polynomial functions are dense in $\mathcal{C}_{\mathbb{R}}([a, b])$.
