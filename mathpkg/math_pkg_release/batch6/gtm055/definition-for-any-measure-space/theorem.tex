For any measure space $(X, \mathbf{S}, \mu)$ the normed spaces $\mathcal{L}_p(X, \mathbf{S}, \mu)$, $1 \leq p \leq +\infty$, are known as the Lebesgue spaces on $X$.

Example A. Let $(X, \mathbf{S}, \mu)$ be a finite measure space, and suppose $1 \leq p \leq p' < +\infty$. If $f$ is an arbitrary element of $\mathcal{L}_p'(X)$, and if we write

$$E = \{x \in X : |f(x)| < 1\},$$

then

$$\int_X |f|^p d\mu = \left( \int_E + \int_{X \setminus E} \right) |f|^p d\mu \leq \mu(X) + \int_X |f|^{p'} d\mu < +\infty.$$
