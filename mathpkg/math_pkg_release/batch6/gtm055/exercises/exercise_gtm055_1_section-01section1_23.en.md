W. If $c$ is an infinite cardinal number, then there is a first, or smallest, ordinal number $\alpha$ such that card $\alpha = c$. This ordinal number is called the initial number of $c$. Thus $\omega$, the smallest infinite ordinal number, is the initial number of $\aleph_0$. The first non-countable ordinal number, i.e., the initial number of the smallest cardinal number exceeding $\aleph_0$, is customarily denoted by $\Omega$. Thus the initial segment $W(\Omega)$ coincides with the set of all countable ordinal numbers. Show that every countable subset of $W(\Omega)$ is bounded in $W(\Omega)$.
