P. A measure space $(X, S, \mu)$ is said to be complete if every subset of an arbitrary set of measure zero with respect to $\mu$ is measurable and is, therefore, also a set of measure zero. (Another way to say this is to say that the collection $Z$ of null sets is a $\sigma$-ideal (Prob. 6I) not only in $S$, but in the power class on $X$.) If $(X, S, \mu)$ is complete, the measure $\mu$ is also said to be complete. Show that if $(X, S, \mu)$ is a complete measure space, and if $f$ is a measurable function on $(X, S)$, then every function $g$ on $X$ such that $g = f$ a.e. $[\mu]$ is also measurable $[S]$. Similarly, if $(X, S, \mu)$ is complete and $E$ belongs to $S$, then every set $F$ that is almost equal $[\mu]$ to $E$ also belongs to $S$ and we have $\mu(E) = \mu(F)$. If $Z$ is a null set in a complete measure space $(X, S, \mu)$, then an arbitrary complex-valued function defined on $Z$ is measurable on $Z$. Show that if $\mu^*$ is an outer measure on a set $X$, and if $M$ denotes the $\sigma$-ring of sets that are measurable with respect to $\mu^*$ (Prob. C), then $(X, M, \mu^* | M)$ is complete.
