E. Let $X$ be a fixed set. For each subset $A$ of $X$, the characteristic function of $A$ is that function $\chi_A$ which takes the value 1 at every point of $A$ and the value 0 at every point of $X \setminus A$. If $A$ and $B$ are subsets of $X$, then $\chi_{A \cap B} = \chi_A \land \chi_B = \chi_A \chi_B$ and $\chi_{A \cup B} = \chi_A \lor \chi_B$. Furthermore, $A \subset B$ if and only if $\chi_A \leq \chi_B$, and $\chi_{A \cup B} = \chi_A + \chi_B$ if and only if $A$ and $B$ are disjoint.
