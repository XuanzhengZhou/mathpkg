S. If $X$ is any set and if $X$ denotes the power class on $X$, then card $X >$ card $X$. (Hint: Suppose there exists a mapping $\varphi$ of $X$ onto $X$. Set $A = \{x \in X : x \notin \varphi(x)\}$, and let $x_0$ be an element of $X$ for which $\varphi(x_0) = A$.)
