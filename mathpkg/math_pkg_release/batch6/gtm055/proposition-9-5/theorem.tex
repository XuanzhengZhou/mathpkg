If $f$ is a measurable extended real-valued function defined and nonnegative a.e. $[\mu]$ on a measure space $(X, \mathbf{S}, \mu)$, then the set function

$$v_f(E) = \int_E f d\mu, \quad E \in \mathbf{S},$$

is a measure on $(X, \mathbf{S})$ called the indefinite integral of $f$ with respect to $\mu$. If $g$ is an arbitrary measurable function that is either integrable with respect to $v_f$ or nonnegative, then

$$\int_X g dv_f = \int_X gf d\mu.$$

$$\varphi(r, t) = re^{it}, \quad (r, t) \in T,$$

The properties of the indefinite integral $v_f$ of a function $f$ with respect to a measure $\mu$ as defined in Proposition 9.5 reflect, quite naturally, the properties of the underlying measure space $(X, S, \mu)$ as well as those of the function $f$. Thus if $f$ is finite-valued (a.e. $[\mu]$), then $v_f$ is easily seen to be $\sigma$-finite whenever $\mu$ is $\sigma$-finite, and locally finite whenever $\mu$ is locally finite; cf. Proposition 8.1. On the other hand, it is possible for $v_f(E)$ to be equal to $+\infty$ even when $\mu(E)$ is finite and $f$ is finite-valued. An important observation concerning indefinite integrals that holds quite generally is the following: For any measurable nonnegative extended real-valued function $f$, $v_f$ vanishes wherever $\mu$ does (Prop. 7.8).
