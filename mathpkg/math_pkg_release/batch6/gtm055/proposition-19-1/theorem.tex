If $\{x_n\}_{n=0}^{\infty}$ and $\{y_n\}_{n=0}^{\infty}$ are two sequences of vectors in a normed space $\mathcal{E}$, and if

$$x = \sum_{n=0}^{\infty} x_n \quad \text{and} \quad y = \sum_{n=0}^{\infty} y_n,$$

then for any pair of complex numbers $\alpha$ and $\beta$,

$$\sum_{n=0}^{\infty} (\alpha x_n + \beta y_n) = \alpha x + \beta y.$$

Likewise, if $T$ is any bounded linear transformation of $\mathcal{E}$ into some normed space $\mathcal{F}$, then $Tx = \sum_{n=0}^{\infty} Tx_n$. Moreover, the counterparts of these assertions are all valid, with appropriate notational changes, for sequences indexed by other systems of the same order type as $\mathbb{N}_0$, such as $\mathbb{N}$, and also for sequences indexed by $\mathbb{Z}$ (in either sense of convergence).
