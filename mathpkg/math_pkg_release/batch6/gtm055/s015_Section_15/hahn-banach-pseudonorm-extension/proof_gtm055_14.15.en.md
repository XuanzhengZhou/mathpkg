The proof follows the same pattern as the classical Hahn-Banach theorem, using Zorn's lemma. One considers the set of all pairs $(\mathcal{N}, g)$ where $\mathcal{N}$ is a linear submanifold containing $\mathcal{M}$ and $g$ is a linear extension of $f$ to $\mathcal{N}$ with $\|g\|_\sigma = \|f\|_\sigma$, partially ordered by inclusion. Zorn's lemma yields a maximal extension, and a one-dimensional extension argument (as in Lemma 14.1) shows that the maximal extension must be defined on all of $\mathcal{E}$.
