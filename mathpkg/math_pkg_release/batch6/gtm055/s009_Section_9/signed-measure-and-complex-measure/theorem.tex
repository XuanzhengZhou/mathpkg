Let $(X, \mathbf{S})$ be a measurable space. An extended real-valued set function $\mu$ on $(X, \mathbf{S})$ is a \textbf{signed measure} if $\mu(\emptyset) = 0$ and $\mu$ is countably additive on $\mathbf{S}$. A complex-valued set function $\zeta$ defined on $\mathbf{S}$ is a \textbf{complex measure} on $(X, \mathbf{S})$ if it is countably additive. If $\mu$ is a signed measure and $E, F$ are disjoint measurable sets, then $\mu(E) + \mu(F)$ must be defined, so $\mu(E)$ and $\mu(F)$ cannot be infinities of opposite sign.
