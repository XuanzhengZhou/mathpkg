If $\mathcal{K}_1, \ldots, \mathcal{K}_p$ is a sequence of circle chains, and if there is given an analytic continuation along each of these circle chains, we say that this sequence of analytic continuations is \textit{chained} if the last function element in the analytic continuation along $\mathcal{K}_i$ coincides with the first function element in the analytic continuation along $\mathcal{K}_{i+1}$, for $i = 1, \ldots, p - 1$.
