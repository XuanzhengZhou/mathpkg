The monodromy theorem is a cornerstone of analytic continuation theory. It states that if a local analytic function $f_0$ on a disc in a simply connected domain $\Delta$ can be continued analytically along every polygonal arc in $\Delta$, then the result of continuation depends only on the endpoint, not on the path. This path-independence allows one to define a global single-valued analytic function on $\Delta$ extending $f_0$. The proof reduces to showing that continuation around any closed polygonal arc returns the original function element, using triangulation of polygonal arcs and induction.
