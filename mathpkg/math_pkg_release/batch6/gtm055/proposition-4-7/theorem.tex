A set $N$ is nowhere dense in a metric space $X$ if and only if, for every nonempty open set $U$ in $X$, there exists a nonempty open set $V$ such that $V \subset U$ and $V \cap N = \emptyset$. Moreover, if $N$ is nowhere dense, it is always possible to arrange for $V$ to be an open ball with arbitrarily small positive radius satisfying the stronger conditions $V^- \subset U$ and $V^- \cap N = \emptyset$.
