If $(X, \mathbf{S}, \mu)$ is a measure space, and if $\mu$ has the property that

$$\mu(E) = \sup \{ \mu(F) : F \in \mathbf{S}, F \subset E, \mu(F) < +\infty \}$$

for every set $E$ in $\mathbf{S}$, then $\mu$ and $(X, \mathbf{S}, \mu)$ are said to be locally finite.
