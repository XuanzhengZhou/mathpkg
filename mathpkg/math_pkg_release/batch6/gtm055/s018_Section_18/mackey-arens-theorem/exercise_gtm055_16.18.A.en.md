(i) Let $\mathcal{E}_1, \ldots, \mathcal{E}_n$ be Banach spaces, and let $\mathcal{E} = \mathcal{E}_1 \oplus_p \cdots \oplus_p \mathcal{E}_n$ for some number $p$, $1 < p < +\infty$. Show that $\mathcal{E}$ is reflexive if and only if each $\mathcal{E}_i$ is reflexive.
