Let $\mathcal{E}$ be a self-conjugate linear space of complex-valued functions on a set $X$ with the property that the real-valued functions in $\mathcal{E}$ form a function lattice (see Example 2H and Problem 1D). Then a linear functional $\varphi$ on $\mathcal{E}$ is positive if $\varphi(f) \geq 0$ for every nonnegative function $f$ in $\mathcal{E}$. Likewise, if $\mathcal{E}$ is a real linear space of real-valued functions that is a function lattice, then a linear functional $\varphi$ on $\mathcal{E}$ is positive if $\varphi(f) \geq 0$ for every nonnegative function $f$ in $\mathcal{E}$.
