Let $f$ be a real-valued function on a closed interval $[a, b]$. For a partition $\mathcal{P} = \{s_i\}_{i=0}^{n}$ of $[a, b]$, define the \textit{positive} and \textit{negative variations} of $f$ over $\mathcal{P}$ by
$$v_+(\mathcal{P}) = \sum_{i=1}^{n} [f(s_i) - f(s_{i-1})]^+, \quad v_-(\mathcal{P}) = \sum_{i=1}^{n} [f(s_i) - f(s_{i-1})]^-.$$
The nets $\{v_+(\mathcal{P})\}$ and $\{v_-(\mathcal{P})\}$ are both monotone increasing (with respect to refinement of partitions). The function $f$ is \textit{of bounded variation} on $[a, b]$ if and only if both nets are bounded above. In that case, define
$$V_+(f; a, b) = \sup_{\mathcal{P}} v_+(\mathcal{P}), \quad V_-(f; a, b) = \sup_{\mathcal{P}} v_-(\mathcal{P}),$$
and the \textit{total variation}
$$V(f; a, b) = V_+(f; a, b) + V_-(f; a, b).$$
Moreover,
$$f(b) - f(a) = V_+(f; a, b) - V_-(f; a, b).$$
