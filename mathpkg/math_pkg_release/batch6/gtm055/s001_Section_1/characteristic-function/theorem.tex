Let $X$ be a fixed set. For each subset $A \subset X$, the \textit{characteristic function} of $A$ is the function $\chi_A$ defined by
$$\chi_A(x) = \begin{cases} 1, & x \in A, \\ 0, & x \in X \setminus A. \end{cases}$$
For subsets $A, B \subset X$,
$$\chi_{A \cap B} = \chi_A \wedge \chi_B = \chi_A \chi_B, \quad \chi_{A \cup B} = \chi_A \vee \chi_B.$$
Moreover, $A \subset B$ if and only if $\chi_A \leq \chi_B$, and $\chi_{A \cup B} = \chi_A + \chi_B$ if and only if $A$ and $B$ are disjoint.
