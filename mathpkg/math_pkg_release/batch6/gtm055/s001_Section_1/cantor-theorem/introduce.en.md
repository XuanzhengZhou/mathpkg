Cantor's theorem states that the cardinality of the power set of any set strictly exceeds the cardinality of the set itself. The proof is a classic diagonal argument: supposing a surjection from $X$ onto $\mathcal{P}(X)$, one constructs the set $A = \{x \in X : x \notin \varphi(x)\}$ and obtains a contradiction by considering whether the preimage of $A$ belongs to $A$. This theorem shows that there is no largest cardinal number.
