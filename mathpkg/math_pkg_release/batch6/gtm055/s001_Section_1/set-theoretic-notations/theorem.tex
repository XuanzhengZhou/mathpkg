Let $\mathbb{N}$ denote the system of positive integers, $\mathbb{N}_0$ the system of nonnegative integers, $\mathbb{Z}$ the system of all integers, $\mathbb{R}$ the real number system, and $\mathbb{C}$ the complex number system. The empty set is denoted by $\varnothing$. For any two sets $X$ and $Y$, the set-theoretic difference is
$$X \setminus Y = \{x \in X : x \notin Y\}$$
and the symmetric difference is
$$X \nabla Y = (X \setminus Y) \cup (Y \setminus X).$$
If $f$ is a mapping of $X$ into $Y$ (notation: $f : X \rightarrow Y$) and $A \subset X$ and $B \subset Y$, then
$$f(A) = \{f(x) : x \in A\}, \quad f^{-1}(B) = \{x \in X : f(x) \in B\}.$$
