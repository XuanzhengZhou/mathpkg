A collection $\mathcal{B}$ of subsets of a set $X$ serves as a base for a topology if it covers $X$ and the intersection of any two base sets is a union of base sets. This proposition provides the necessary and sufficient conditions for a family of sets to generate a topology, forming a fundamental tool in point-set topology. The corresponding condition for a subbase is simply that it covers $X$, without any intersection requirement.
