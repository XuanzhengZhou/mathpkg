Let $\mathcal{C}$ be a collection of subsets of a set $X$, and let $\mathbf{R}$ and $\mathbf{S}$ denote, respectively, the ring and $\sigma$-ring generated by $\mathcal{C}$. If $\mathcal{C}$ is finite, then so is $\mathbf{R}$. What about $\mathbf{S}$? If $\mathcal{C}$ is countable, then so is $\mathbf{R}$, while for $\mathbf{S}$ one has the estimate $\operatorname{card} \mathbf{S} \leq \aleph$ (where $\aleph$ denotes the cardinal number of the continuum). Indeed, $\operatorname{card} \mathcal{C} \leq \aleph$ implies $\operatorname{card} \mathbf{S} \leq \aleph$. (Hint: If $\mathcal{C}$ consists of $k$ sets, then $\mathbf{R}(\mathcal{C})$ cannot contain more than $2^{2^k}$ sets.)
