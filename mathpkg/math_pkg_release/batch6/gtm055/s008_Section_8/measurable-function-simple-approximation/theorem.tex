Let $X$ be a measurable space and let $f$ be a real-valued function defined on $X$. The following are equivalent:
(i) $f$ is measurable,
(ii) There exists a sequence $\{s_n\}$ of measurable simple real-valued functions converging pointwise to $f$,
(iii) There exists a sequence $\{s_n\}$ of measurable simple real-valued functions converging pointwise to $f$ such that:
(a) $|s_n(x)| \leq |f(x)|$ for all $n,x$,
(b) At each $x$, either $0 \leq s_1(x) \leq \cdots \leq s_n(x) \leq \cdots$ or $0 \geq s_1(x) \geq \cdots \geq s_n(x) \geq \cdots$,
(c) If $|f(x)| \leq N$ then $|f(x) - s_n(x)| \leq 1/2^n$ for all $n \geq N$. In particular, convergence is uniform on every set where $f$ is bounded.
