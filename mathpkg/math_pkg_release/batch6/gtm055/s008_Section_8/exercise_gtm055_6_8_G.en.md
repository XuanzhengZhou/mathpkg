For an arbitrary nonempty collection $\mathcal{C}$ of subsets of a set $X$ let $\mathcal{C} \neq \mathcal{C}$ denote the collection of all differences $E \setminus F$, where $E$ and $F$ belong to $\mathcal{C}$. Furthermore, let $\mathcal{C}_\sigma$ denote the collection of all unions of countable subcollections of $\mathcal{C}$, and let $\mathcal{C}^* = (\mathcal{C} \neq \mathcal{C})_\sigma$. In terms of this operation it is possible to describe the $\sigma$-ring $\mathbf{S}(\mathcal{C})$ generated by $\mathcal{C}$. We suppose, without loss of generality, that $\emptyset \in \mathcal{C}$. Let $\mathcal{C}^{(0)} = \mathcal{C}$, and, for a given countable ordinal number $\alpha$, suppose $\mathcal{C}^{(\xi)}$ already defined for every ordinal number $\xi$ less than $\alpha$. Then if $\alpha$ is not a limit ordinal we set $\mathcal{C}^{(\alpha)} = (\mathcal{C}^{(\xi)})^*$, where $\alpha = \xi + 1$, while if $\alpha$ is a limit ordinal we set $\mathcal{C}^{(\alpha)} = \bigcup_{\xi < \alpha} \mathcal{C}^{(\xi)}$. Show that if $\Omega$ denotes, as usual, the smallest uncountable ordinal number, then $\mathbf{S}(\mathcal{C}) = \bigcup_{\xi < \Omega} \mathcal{C}^{(\xi)}$.
