Let $(X, \mathbf{S}, \mu)$ be a measure space.

If $f$ is a nonnegative measurable function on $X$, then $\int f d\mu = 0$ if and only if $f = 0$ a.e. $[\mu]$.

If $f$ is an arbitrary complex-valued measurable function on $X$ such that $f = 0$ a.e. $[\mu]$, then $f$ is integrable and $\int f d\mu = 0$.

If $f$ and $g$ are measurable functions on $X$ such that $f = g$ a.e. $[\mu]$, then $f$ is integrable $[\mu]$ if and only if $g$ is, and in this case $\int f d\mu = \int g d\mu$.
