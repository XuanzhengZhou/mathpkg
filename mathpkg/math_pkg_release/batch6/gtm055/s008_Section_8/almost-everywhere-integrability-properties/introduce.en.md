Proposition 7.8 formalizes the principle that sets of measure zero are negligible for integration. For nonnegative measurable functions, the integral vanishes if and only if the function is zero almost everywhere. Moreover, any function vanishing almost everywhere is automatically integrable with integral zero. In particular, if two measurable functions agree almost everywhere, then one is integrable precisely when the other is, and their integrals coincide. This justifies the common practice of identifying functions that differ only on null sets.
