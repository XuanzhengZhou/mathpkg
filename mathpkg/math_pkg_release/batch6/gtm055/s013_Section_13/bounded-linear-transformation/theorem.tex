A linear transformation $T$ of a normed space $\mathcal{E}$ into another normed space $\mathcal{F}$ is \text{bounded} if there exists a scalar $M \geq 0$ such that

$$\| Tx \| \leq M \| x \|, \quad x \in \mathcal{E}.$$

If $T$ is bounded, there is a smallest $M$ satisfying the inequality, known as the \text{norm} of $T$ and denoted by $\| T \|$. Thus $\| T \|$ is a nonnegative real number such that $\| Tx \| \leq \| T \| \| x \|$ for every vector $x$ in $\mathcal{E}$, and is the smallest number making this inequality valid. Moreover, for $\mathcal{E} \neq (0)$,

$$\| T \| = \sup_{\| x \| \leq 1} \| Tx \| = \sup_{\| x \| < 1} \| Tx \| = \sup_{\| x \| = 1} \| Tx \| = \sup_{x \neq 0} \frac{\| Tx \|}{\| x \|}.$$
