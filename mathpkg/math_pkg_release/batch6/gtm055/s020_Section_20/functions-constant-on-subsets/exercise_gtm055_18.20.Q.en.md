Let us say that a function $f$ on a set $X$ is constant on a nonempty subset $E$ of $X$ if there exists a scalar $\lambda_0$ such that $f(x) = \lambda_0$ for all $x$ in $E$.

(i) Show that if $X$ is a compact Hausdorff space, and if $\{E_{\gamma}\}_{\gamma \in \Gamma}$ is an arbitrary disjoint collection of nonempty subsets of $X$, then the collection $A$ of all those continuous complex-valued functions on $X$ that are constant on each $E_{\gamma}$ is a closed self-conjugate subalgebra of $\mathcal{C}(X)$. Show too that the subset of $A$ consisting of those functions in $A$ that vanish on some one designated set $E_{\gamma_0}$ is also a closed self-conjugate subalgebra of $\mathcal{C}(X)$.

(ii) The subalgebras described in (i) are the most general closed self-conjugate subalgebras of $\mathcal{C}(X)$. To see this let $A$ be an arbitrary closed self-conjugate subalgebra of $\mathcal{C}(X)$. Use $A$ to define an equivalence relation $\sim$ on $X$, setting $x_1 \sim x_2$ if $f(x_1) = f(x_2)$ for every $f$ in $A$. Prove that $A$ coincides with the collection of all those continuous functions on $X$ that are constant on each equivalence class $[x]$, with a possible distinguished zero class $[x]_0$ on which all functions in $A$ vanish.
