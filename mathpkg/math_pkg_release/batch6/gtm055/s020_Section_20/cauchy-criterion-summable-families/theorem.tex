An indexed family $\{x_{\gamma}\}_{\gamma \in \Gamma}$ of vectors in a Banach space $\mathcal{E}$ is summable if and only if for every $\varepsilon > 0$ there exists a finite set of indices $D_{\varepsilon}$ such that $\|s_D\| = \|\sum_{\gamma \in D} x_{\gamma}\| < \varepsilon$ for every finite set of indices $D$ such that $D$ and $D_{\varepsilon}$ are disjoint.
