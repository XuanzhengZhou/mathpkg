Let $X$ be a compact Hausdorff space, let $\xi$ be a regular complex Borel measure on $X$, and let $\varphi$ denote the bounded linear functional on $\mathcal{C}(X)$ determined by $\xi$ via $\varphi(f) = \int_X f \, d\xi$.

(i) $\varphi$ is self-conjugate if and only if $\xi$ is real-valued.
(ii) $\varphi$ is positive if and only if $\xi$ is a (nonnegative) measure.
(iii) $\varphi$ is an algebra homomorphism of $\mathcal{C}(X)$ onto $\mathbb{C}$ if and only if $\xi$ is a Dirac mass.
