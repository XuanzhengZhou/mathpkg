Let $X$ be a compact Hausdorff space, let $\xi$ be a regular complex Borel measure on $X$, and let $\varphi$ denote the bounded linear functional on $\mathcal{C}(X)$ determined by $\xi$.

(i) Verify that $\varphi$ is self-conjugate (Ex. 2H) if and only if $\xi$ is real-valued, and likewise that $\varphi$ is positive if and only if $\xi$ is a (nonnegative) measure.

(ii) Verify, in the same vein, that $\varphi$ is an algebra homomorphism of $\mathcal{C}(X)$ onto $\mathbb{C}$ if and only if $\xi$ is a Dirac mass.
