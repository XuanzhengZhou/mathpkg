For an ordinal number $\alpha$, the ordinal number segment $W(\alpha)$ is the set of all ordinals strictly less than $\alpha$, ordered by membership. It is the canonical well-ordered set of order type $\alpha$, and its cardinality equals the cardinality of $\alpha$. Ordinal segments are the building blocks of transfinite induction and recursion, providing a standard reference for each ordinal.
