If $f$ is an analytic function on a domain $\Delta$, and if for some complex number $\alpha$ not belonging to $\Delta$ there exists a positive number $r$ such that $\Delta$ contains the punctured disc $\{\lambda \in \mathbb{C} : 0 < |\lambda - \alpha| < r\}$, then $\alpha$ is an **isolated singularity** of $f$.
