Let $\Delta'$ be a domain in $\mathbb{R}^2$, and let $f$ be a function in $\mathcal{C}^{(2)}(\Delta')$. Then $f$ is **harmonic** on $\Delta'$ if the differential equation
$$f_{xx} + f_{yy} = 0$$
is satisfied everywhere on $\Delta'$. Similarly, a function $f$ on a complex domain $\Delta$ is harmonic on $\Delta$ if the function $f_0(x, y) = f(x + iy)$ is harmonic on the corresponding real domain $\Delta'$.
