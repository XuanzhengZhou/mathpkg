(Tonelli Theorem). Let $(X, S, \mu)$ and $(Y, T, v)$ be $\sigma$-finite measure spaces. If $f$ is a function that is measurable $[S \times T]$ and nonnegative on $X \times Y$, then

$$\int_X \left[ \int_Y f(x, y)dv(y) \right] d\mu = \int_Y \left[ \int_X f(x, y)d\mu(x) \right] dv$$

$$= \int_{X \times Y} f d(\mu \times v).$$

Thus if $\varphi$ is an arbitrary measurable complex-valued function on $X \times Y$, and if the function $g(x) = \int_Y |\varphi(x, y)|dv(y)$ satisfies the condition $\int_X g d\mu < +\infty$ (or if $h(y) = \int_X |\varphi(x, y)|d\mu(x)$ satisfies the condition $\int_Y h dv < +\infty$), then $\varphi$ is integrable $[\mu \times v]$.
