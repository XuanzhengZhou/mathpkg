Let $\mathcal{E}$ and $\mathcal{F}$ be linear spaces over the same scalar field. For linear transformations $S, T : \mathcal{E} \to \mathcal{F}$, the sum $S + T$ is defined by $(S + T)x = Sx + Tx$, and scalar multiplication by $(\alpha S)x = \alpha(Sx)$. These operations turn the set of all linear transformations of $\mathcal{E}$ into $\mathcal{F}$ into a linear space — the \textbf{full space of linear transformations}. The full space of linear transformations of $\mathcal{E}$ into its scalar field is the \textbf{full algebraic dual} of $\mathcal{E}$.
