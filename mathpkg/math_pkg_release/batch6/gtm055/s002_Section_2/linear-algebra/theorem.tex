A linear space $\mathcal{A}$ equipped with an associative bilinear multiplication $\mathcal{A} \times \mathcal{A} \to \mathcal{A}$ satisfying $$(xy)z = x(yz), \quad x(y+z) = xy + xz, \quad (x+y)z = xz + yz, \quad \alpha(xy) = (\alpha x)y = x(\alpha y)$$ for all $x, y, z \in \mathcal{A}$ and scalars $\alpha$ is an \textbf{(associative, linear) algebra}. If $xy = yx$ for all $x, y$, the algebra is \textbf{commutative} (abelian). An element $1$ with $1x = x1 = x$ for all $x$ is the \textbf{identity} (unit); $\mathcal{A}$ is \textbf{unital} if such $1$ exists. An element $x$ is \textbf{invertible} if there exists $y$ with $xy = yx = 1$, denoted $y = x^{-1}$.
