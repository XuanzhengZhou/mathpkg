Let $\mathcal{E}$ be a linear space and $\mathcal{M}$ a linear submanifold of $\mathcal{E}$. Define the equivalence relation $x \sim y$ if $x - y \in \mathcal{M}$. The equivalence class $[x] = x + \mathcal{M}$ is the \textbf{coset} of $x$ modulo $\mathcal{M}$. The set of all cosets forms a linear space $\mathcal{E} / \mathcal{M}$ (the \textbf{quotient space}) with operations $[x] + [y] = [x + y]$ and $\alpha[x] = [\alpha x]$. The mapping $\pi : \mathcal{E} \to \mathcal{E} / \mathcal{M}$ defined by $\pi(x) = [x]$ is the \textbf{natural projection}.
