A set $\{x_\gamma\}_{\gamma \in \Gamma}$ of vectors in a linear space $\mathcal{E}$ is a \textbf{Hamel basis} for $\mathcal{E}$ if each vector $y \in \mathcal{E}$ is expressible uniquely as a finite linear combination $y = \sum_{\gamma \in \Gamma} \lambda_\gamma x_\gamma$ where $\lambda_\gamma$ are scalars and $\lambda_\gamma = 0$ for all but a finite set of indices. The scalars $\lambda_\gamma$ are the \textbf{coordinates} of $y$ with respect to the basis. Every linear space possesses a Hamel basis.
