By E2, $\Lambda^*$ is a set system if and only if $G(\Lambda^*) = G_0(\Lambda^*)$. Since $\Lambda$ distinguishes vertices iff $\Lambda^*$ is a set system, and $G(\Lambda) \cong G_1(\Lambda)$ iff $G(\Lambda^*) \cong G_0(\Lambda^*)$ (by E1 and the duality ID5), the corollary follows.
