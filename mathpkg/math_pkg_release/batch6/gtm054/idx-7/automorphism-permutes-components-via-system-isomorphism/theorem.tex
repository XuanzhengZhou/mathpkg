Let $\Lambda = \bigoplus_{i=1}^{n} \Lambda_i$ with components $\Lambda_i = (V_i, f_i, E_i)$ for $i = 1, \ldots, n$. If $(p, q) \in G(\Lambda)$, then to each $i \in \{1, \ldots, n\}$ there corresponds some $j \in \{1, \ldots, n\}$ such that
$$q[V_i] = V_j \quad \text{and} \quad p[E_i] = E_j.$$
Moreover, $(p_{|E_i}, q_{|V_i})$ is a system-isomorphism from $\Lambda_i$ to $\Lambda_j$.
