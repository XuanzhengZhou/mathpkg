This proposition describes how an automorphism of a system decomposed as a direct sum of components interacts with the component structure. Any automorphism permutes the components (mapping vertex sets to vertex sets and edge sets to edge sets), and the restriction of the automorphism to each component yields a system-isomorphism between the corresponding components. The proof uses the path structure and property C20 of the system.
