Apply E2 (which states that $\Lambda^*$ is a set system iff $G(\Lambda^*) = G_0(\Lambda^*)$) together with E4. Since $\Lambda$ is a set system by hypothesis, $\Lambda^*$ also satisfies the appropriate conditions. By E4, $\Lambda$ distinguishes vertices iff $G(\Lambda) \cong G_1(\Lambda)$. For a set system, the duality between $G_0$ and $G_1$ under $*$ yields the equivalence to $G_0(\Lambda) \cong G_1(\Lambda)$.
