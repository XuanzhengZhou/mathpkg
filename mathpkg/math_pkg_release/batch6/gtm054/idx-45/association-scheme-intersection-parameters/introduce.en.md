In the theory of partially-balanced incomplete block designs (PBIB-designs), an $m$-class association scheme partitions the unordered pairs of a vertex set $V$ into $m$ symmetric relations $Q_1, \ldots, Q_m$. For each triple of indices $(i, j, k)$, the intersection parameter $p_{jk}^i$ counts how many vertices are simultaneously $j$th-associates of $x$ and $k$th-associates of $y$, given that $x$ and $y$ are $i$th-associates. The key property is that this count depends only on the relation class $i$, not on the particular pair $\{x,y\}$ chosen. These $m^3$ numbers, together with $n_i$ (the number of $i$th-associates per vertex), form the parameter set of the association scheme and satisfy rich algebraic constraints explored in subsequent propositions.
