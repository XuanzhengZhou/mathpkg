If $\{X_1, \ldots, X_m\} \in \mathbb{P}(X)$, then the function $f \mapsto (f_{|X_1}, \ldots, f_{|X_m})$ is a set-isomorphism between $Y^X$ and $Y^{X_1} \times \ldots \times Y^{X_m}$.

Given the cartesian product $X_1 \times \ldots \times X_m$, the $i$th-coordinate projection is the function from $X_1 \times \ldots \times X_m$ into $X_i$ given by $(x_1, \ldots, x_m) \mapsto x_i$.
