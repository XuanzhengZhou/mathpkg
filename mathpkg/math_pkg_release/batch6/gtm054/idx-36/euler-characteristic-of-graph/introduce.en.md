The Euler characteristic $\varepsilon(\Gamma)$ of a graph $\Gamma$ is defined as the maximum Euler characteristic among all surfaces in which $\Gamma$ admits a topological imbedding. An imbedding achieving this maximum is called a simplest imbedding. The Euler characteristic is anti-monotone with respect to subgraphs: if $\Theta$ is a subgraph of $\Gamma$, then $\varepsilon(\Theta) \geq \varepsilon(\Gamma)$. Together with the genus, this invariant provides the topological framework for bounding the chromatic number of graphs via Heawood-type inequalities.
