Let $(V, D)$ be a directed graph with vertex capacity $j$, and let $(X, k)$ be the associated edge-capacitated network constructed via the vertex-splitting transformation (F3). Then, with $e_0 = (x_0^2, x_0^1)$,

$$\max\{h(x_0): h \text{ is a feasible flow in } (V, D)\} = \max\{\bar{h}(e_0): \bar{h} \text{ is a feasible flow in } (X, k)\}.$$
