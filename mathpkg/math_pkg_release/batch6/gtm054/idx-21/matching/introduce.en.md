A matching in a graph is an independent set of edges, i.e., a set $\mathcal{E}' \subseteq \mathcal{E}$ in which no two edges share a common vertex. Matchings are the central objects of study in this chapter. The Main Matching Theorem, proved using the Max-Flow-Min-Cut Theorem, characterizes the maximum size of a matching in a bipartite graph in terms of the deficiency parameters $\delta_1$ and $\delta_2$.
