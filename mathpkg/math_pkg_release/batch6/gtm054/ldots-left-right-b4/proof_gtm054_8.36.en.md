One may regard any $h \in H$ as a function from $V$ into $\{1, \ldots, m\}$. It is clear that $|V| < n(q_1, \ldots, q_m: 1)$ if and only if for all $h \in H$, $|h^{-1}[i]| \leq q_i - 1$ for each $i = 1, \ldots, m$; that is, if and only if $|V| \leq (\sum_{i=1}^{m} q_i) - m$.
