This formula counts the number of surjective functions from an $|X|$-set to a $|Y|$-set using the Principle of Inclusion–Exclusion. It is obtained by considering the system where vertices are functions $f: X \to Y$, blocks are elements of $Y$, and a function belongs to block $y$ if $y$ is not in its image. The number of blocks of size 0 (functions missing no element of $Y$) then gives the surjection count.
