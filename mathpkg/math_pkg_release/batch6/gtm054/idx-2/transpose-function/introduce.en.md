The transpose $f^*$ of an incidence function $f$ reverses the roles of vertices and blocks: for each vertex $x$, $f^*(x)$ is the set of blocks containing $x$. This defines the dual system $\Lambda^* = (E, f^*, V)$, a fundamental construction that interchanges the vertex and block sets. The duality $x \in f(e) \iff e \in f^*(x)$ means that incidence is a symmetric relation.
