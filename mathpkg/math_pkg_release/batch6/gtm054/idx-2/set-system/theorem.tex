If $\Lambda = (V, f, E)$ is a system and $f$ is an injection, then $\Lambda$ is called a \textbf{set system}. If $\mathcal{E} \subseteq \mathcal{P}(V)$ and the inclusion function $j: \mathcal{E} \to \mathcal{P}(V)$ is defined by $j(S) = S$ for each $S \in \mathcal{E}$, then $(V, j, \mathcal{E})$ is a set system. In this case, the function $j$ is suppressed and the set system is denoted simply by the pair $(V, \mathcal{E})$.

Every set system $(V, f, E)$ is system-isomorphic to $(V, f[E])$, where $f[E] \subseteq \mathcal{P}(V)$.
