A set system is a system whose incidence function $f$ is injective, meaning distinct blocks are assigned distinct subsets of vertices. Every set system can be represented canonically as a pair $(V, \mathcal{E})$ where $\mathcal{E} \subseteq \mathcal{P}(V)$, identifying each block with its image under $f$. This representation is the standard way to define graphs, hypergraphs, and other incidence structures in combinatorics.
