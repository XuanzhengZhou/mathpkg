A system is the most general incidence structure studied in combinatorics, consisting of a vertex set $V$, a block set $E$, and an incidence function $f$ assigning to each block a subset of vertices. This definition unifies graphs, multigraphs, hypergraphs, and set systems under a single framework. When $f$ is injective, the system is called a set system; when all blocks have size 2, the system is a multigraph.
