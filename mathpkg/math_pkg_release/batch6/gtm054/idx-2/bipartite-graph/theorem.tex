A graph $(V, \mathcal{E})$ is said to be \textbf{bipartite} if $|V| \leq 1$ or if there exists a partition $\{V_1, V_2\} \in \mathbb{P}_2(V)$ such that $|E \cap V_1| = |E \cap V_2| = 1$ for all $E \in \mathcal{E}$. When we wish to specify the partition we write: $(V, \mathcal{E})$ is a bipartite graph with respect to $\{V_1, V_2\}$.
