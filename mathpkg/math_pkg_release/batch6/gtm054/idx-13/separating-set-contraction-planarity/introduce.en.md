This proposition provides a reduction step for testing planarity of a connected multigraph. If the graph has a minimal separating set $W$ of size at most $3$, planarity can be checked on the two simpler graphs obtained by contracting each component of $\Gamma \setminus W$ (together with $W$) to a single vertex. The key idea is that a planar imbedding can be pieced together from planar imbeddings of these contracted pieces. The proof for $|W| = 3$ is the most involved case and uses explicit construction of imbeddings from cycle bases.
