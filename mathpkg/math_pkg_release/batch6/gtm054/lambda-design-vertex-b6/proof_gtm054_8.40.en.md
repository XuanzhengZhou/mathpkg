The proof is by induction on $a$. Theorem A12 serves as the first step, with $a = 2$. For our induction hypothesis, we assume that the theorem is valid for all $(; a)$-designs where $a$ is some fixed integer at least 2 and no

at least one 0. If $m$ is the number of 0's in $M$, then $v \leq m \leq b$. By our assumption that $b = v$, there must be exactly one 0 in each row and each column. But then $M$ is an incidence matrix of the set system $(V, \mathcal{P}_{v-1}(V))$. We have already noted (A3) that $b = v$ for trivial designs.

The property of being a $(t; a)$-design when both $t \geq 2$ and $a \geq 2$ and one of these values is $> 2$ is very restrictive, as we shall presently see.
