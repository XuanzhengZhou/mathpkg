Let $A$ be a subspace of $\mathcal{P}(V)$.

(a) If $A = \bigoplus_{i=1}^{k} B_i$, then $\Lambda(A) = \bigoplus_{i=1}^{k} \Lambda(B_i)$.

(b) If $\Lambda(A) = \bigoplus_{i=1}^{k} \Lambda_i$ where $\Lambda_i = (V_i, \mathcal{E}_i)$, then $\Lambda_i = \Lambda(A \cap \mathcal{P}(V_i))$ for $i = 1, \ldots, k$. Furthermore $A = \bigoplus_{i=1}^{k} (\Lambda \cap \mathcal{P}(V_i))$.
