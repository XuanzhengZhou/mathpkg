A graph $\Gamma = (V, \mathcal{E})$ is said to be \textbf{almost factorable} if $\Gamma$ has the property that $\Gamma_{(x)}$ admits a $1$-factor for all $x \in V$. In such a case, $\nu_0(\Gamma)$ is necessarily odd, and $\Gamma$ is as close to admitting a $1$-factor as an odd graph can be. For all $n$, the graph $K_{2n+1}$ is almost factorable.
