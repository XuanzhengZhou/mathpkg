The boundary values of the graph-theoretic Ramsey numbers are trivial: $n(1, q_2) = n(q_1, 1) = 1$ for all $q_1, q_2 \geq 1$. This reflects the fact that a graph with zero vertices vacuously satisfies any clique or independence number bound, so the maximum number of vertices in a graph with $\omega(\Gamma) < 1$ (impossible) or $\alpha_0(\Gamma) < 1$ (impossible) is $0$, giving $n = 0 + 1 = 1$. Allowing $q_1$ or $q_2$ to equal $1$ is a convenient convention that simplifies inductive proofs.
