By the graph-theoretic interpretation C1, $n(q_1, q_2) - 1 = \max\{\nu_0(\Gamma): \omega(\Gamma) < q_1 \text{ and } \alpha_0(\Gamma) < q_2\}$. If $q_1 = 1$, then the condition $\omega(\Gamma) < 1$ cannot be satisfied by any nonempty graph, so the maximum is over the empty graph only, giving $\max = 0$. Hence $n(1, q_2) - 1 = 0$, i.e., $n(1, q_2) = 1$. The symmetric argument for $q_2 = 1$ gives $n(q_1, 1) = 1$.
