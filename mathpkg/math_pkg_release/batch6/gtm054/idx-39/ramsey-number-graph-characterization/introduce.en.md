When $s = 2$, the Ramsey number $n(q_1, q_2)$ admits a natural graph-theoretic interpretation. Viewing a $2$-coloring of the $2$-subsets of $V$ as a graph $\Gamma$ and its complement $\Gamma'$, a complete set in color $1$ corresponds to a clique in $\Gamma$, while a complete set in color $2$ corresponds to an independent set (scattering) in $\Gamma$. Thus $n(q_1, q_2) - 1$ equals the maximum possible number of vertices in a graph that contains neither a $q_1$-clique nor a $q_2$-scattering. This characterization makes sense for $q_1 = 1$ or $q_2 = 1$ as well.
