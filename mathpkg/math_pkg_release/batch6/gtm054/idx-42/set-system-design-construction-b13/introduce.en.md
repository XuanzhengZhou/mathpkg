This proposition relaxes the consecutivity requirement for constructing set systems with a given one-sided design-type. While Theorem B12 covers the case where all $t_i$ are consecutive, this result only requires consecutivity from $t_2$ onward, allowing $t_1$ to be non-consecutive with $t_2$. The proof distinguishes two cases: if all $t_i$ are consecutive, one appeals to B12; otherwise, the integers are arranged as $t_1, t_2, t_2+1, \ldots, t_2+(p-2)=t_p$ and a suitable construction yields the desired set system.
