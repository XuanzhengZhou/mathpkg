XIB Generating Functions

If $a, b \in \mathbb{C}^N$, we say that $b(x) = \sum_{j=0}^{\infty} b_j x^j$ is a generating function for the sequence $a$ if there exists a vector space isomorphism $\varphi: \mathbb{C}^N \to \mathbb{C}^N$ such that $\varphi(a) = b$. (Thus $\varphi$ respects the two operations defined by A1 and A2.) Obviously a given sequence may have many different generating functions. In this section we shall confine our attention to the two most commonly used generating functions. If $\varphi$ is the identity function, then $\varphi(a)$ is called the ordinary generating function for $a$. If $\varphi$ is given by $\varphi(a) = \sum_{j=0}^{\infty} (a_j/j!) x^j$, then $\varphi(a)$ is called the exponential generating function for $a$. In the latter case,

$$\varphi(a(x))\varphi(b(x)) = \sum_{k=0}^{\infty} \left( \sum_{j=0}^{k} \binom{k}{j} a_j b_{k-j} \right) x^k/k!$$

while

$$\varphi(a(x)b(x)) = \sum_{k=0}^{\infty} \left( \sum_{j=0}^{k} a_j b_{k-j} \right) x^k/k!$$

and so $\varphi$ is not an algebra-isomorphism.
