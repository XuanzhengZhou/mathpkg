Let $r$ be the rank function of the geometry $\Lambda = (V, \mathcal{M})$ and let $h$ be the lattice-isomorphism in the proof associating geometries with matroid lattices. If $r'$ is the rank function of the lattice $L$ (from the same proof), show that $r(U) = r'(h(U))$ for all closed sets $U$ of $\Lambda$.
