On any semimodular lattice, a rank function can be defined by taking the length of any maximal chain from the bottom element 0 to a given element. Lemma C28 guarantees that all such maximal chains have the same length, making the rank function well-defined. This rank function is compatible with the lattice isomorphism between the closed sets of a geometry and the corresponding matroid lattice.
