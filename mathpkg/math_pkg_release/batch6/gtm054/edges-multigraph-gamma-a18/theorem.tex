Let $F$ be a set of edges of the multigraph $\Gamma$.

(a) $F$ contains a nonempty cocycle if and only if $\nu_{-1}(\Gamma_{(F)}) > \nu_{-1}(\Gamma)$.

(b) $F$ is an elementary cocycle if and only if $\nu_{-1}(\Gamma_{(F)}) > \nu_{-1}(\Gamma)$ and $\nu_{-1}(\Gamma_{(F+e)}) = \nu_{-1}(\Gamma)$ for every $e \in F$, i.e., $F$ is minimal with respect to the property given in (a).
