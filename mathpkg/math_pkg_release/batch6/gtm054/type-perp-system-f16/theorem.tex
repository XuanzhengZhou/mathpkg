This yields the solutions $\rho = \nu_1 = k, \rho^\perp = 2$, for $k = 2, 3, \ldots$.

Case 2: $\rho^\perp = 3$. Then F17 becomes:

$$\rho \geq 3, \quad \nu_1 \geq 2, \quad \frac{1}{\rho} = \frac{1}{6} + \frac{1}{\nu_1}.$$

Clearly we must have $\rho < 6$, i.e., $\rho = 3, 4$, or 5, whence the solutions:

$$\rho = 3, \quad \nu_1 = 6;$$

$$\rho = 4, \quad \nu_1 = 12;$$

$$\rho = 5, \quad \nu_1 = 30.$$
