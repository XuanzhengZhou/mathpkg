A partially-ordered set (or poset) is a set equipped with a partial order relation. The canonical examples in combinatorics include $(\mathcal{P}(U), \subseteq)$, the power set ordered by inclusion, and $(\mathbb{P}(U), \leq)$ where $\leq$ denotes refinement of partitions. Any subset of a poset inherits the partial order by restriction, making posets ubiquitous in combinatorial structures.
