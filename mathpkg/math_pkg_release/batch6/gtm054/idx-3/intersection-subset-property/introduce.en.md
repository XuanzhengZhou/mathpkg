This lemma records two elementary but fundamental properties relating intersection and subset inclusion on the power set $\mathcal{P}(U)$. First, the intersection of any two subsets is always contained in each factor. Second, intersection is monotone with respect to inclusion: if one set is contained in another, intersecting both with a fixed set preserves the inclusion. These properties are used throughout the text to relate the algebraic operations on $\mathcal{P}(U)$ to its order-theoretic structure.
