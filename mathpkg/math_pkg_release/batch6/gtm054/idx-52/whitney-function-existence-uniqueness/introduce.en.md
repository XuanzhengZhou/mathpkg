Exercise F2 asks the reader to prove that any function $w: V \to \mathcal{V}$ from a set $V$ into a vector space $\mathcal{V}$ can be viewed as a Whitney function for some matroid $\Lambda$ on ground set $V$, and that this matroid is unique up to isomorphism. The matroid structure is determined by declaring a subset to be independent when the images under $w$ of its elements are linearly independent in $\mathcal{V}$. More precisely, $A \subseteq V$ is independent in $\Lambda$ if and only if $\{w(x) : x \in A\}$ is linearly independent in $\mathcal{V}$. This exercise establishes the fundamental connection between vector configurations and matroid representability.
