Let $M$ be an incidence matrix of $\Lambda$. By Exercise C15a,
$$M^*M = (k - \lambda^*(2))I + \lambda^*(2)J.$$
By Theorem C16, this quantity is equal to $MM^*$. Again by Exercise C15a, $M^*$ is an incidence matrix of a $(; 1, 2)$-design. Hence $M$ is an incidence matrix of a $(1, 2;)$-design; i.e., $\Lambda$ is a $(1, 2; 1, 2)$-design. Since $k = r < v$, this is exactly a symmetric block design.
