Proposition C6 establishes a bijective correspondence between independence systems and rank structures on a finite set $V$. Given an independence system $(V, \mathcal{I})$, the rank $r(U)$ of a subset $U$ is the maximum cardinality of an independent set contained in $U$. Conversely, given a rank structure $(V, r)$, a set is independent precisely when its cardinality equals its rank. This equivalence is fundamental to matroid theory: it shows that the rank function captures exactly the same information as the collection of independent sets, unifying the two axiomatic approaches.
