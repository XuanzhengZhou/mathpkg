Corollary C7 gives a direct characterization of when adding a single element to a set does not increase its rank in a matroid: this happens precisely when the element belongs to a cycle that is contained in the enlarged set. Equivalently, $r(U \cup \{x\}) = r(U)$ iff $x$ is in the closure (span) of $U$. This corollary connects the rank function directly to the cycle structure of a matroid, bypassing the intermediate independence system and providing a useful criterion for computing rank.
