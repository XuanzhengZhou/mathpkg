Let $\Gamma = (V, \mathcal{E})$ be a graph. The following four statements are equivalent:

(a) $\Gamma$ is color-perfect;
(b) $\Gamma_U$ contains a clump whenever $\varnothing \subset U \subseteq V$;
(c) $\Gamma'$ is color-perfect;
(d) $\Gamma_U$ contains a spread whenever $\varnothing \subset U \subseteq V$.
