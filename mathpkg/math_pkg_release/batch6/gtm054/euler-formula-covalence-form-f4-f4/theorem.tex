A matroid admits a Tutte subspace over $\mathbb{F}$ if and only if it admits a Whitney function over $\mathbb{F}$.

If a matroid admits a Tutte subspace over $\mathbb{F}$ or, equivalently, a Whitney function over $\mathbb{F}$, then it is said to be representable over $\mathbb{F}$.
