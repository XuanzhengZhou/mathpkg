Let $\Gamma = (V, \mathcal{E})$ be a graph. The following four statements are equivalent:
\begin{enumerate}
\item[(a)] $\Gamma$ is color-perfect;
\item[(b)] $\Gamma_U$ contains a clump whenever $\varnothing \subset U \subseteq V$;
\item[(c)] $\Gamma'$ is color-perfect;
\item[(d)] $\Gamma_U$ contains a spread whenever $\varnothing \subset U \subseteq V$.
\end{enumerate}
