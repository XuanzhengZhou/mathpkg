For a $(; 1)$-design $\Lambda$ with blocksize $k$, the value of $\bar{s}$ on $\mathcal{P}_t(V)$ where $t \leq k$ determines the value of $\bar{s}$ on all subsets of $V$ with cardinality less than $t$. In particular,
$$\bar{s}(S) = \frac{1}{\binom{k-i}{t-i}} \binom{v-i}{t-i} \lambda(t)$$
where $i = |S|$. Thus $\bar{s}(S)$ is determined only by $|S| = i$; that is, $\Lambda$ is an $(i;)$-design.
