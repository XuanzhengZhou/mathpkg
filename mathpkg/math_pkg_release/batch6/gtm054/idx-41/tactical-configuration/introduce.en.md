A tactical configuration is the standard term in the combinatorial design literature for a nondegenerate $1$-design — an incidence structure where each vertex appears in a constant number $r$ of blocks and each block contains a constant number $k$ of vertices. Regular graphs of uniform degree provide a natural example: a $\rho$-valent graph is a tactical configuration with replication number $r = \rho$ and blocksize $k = 2$.
