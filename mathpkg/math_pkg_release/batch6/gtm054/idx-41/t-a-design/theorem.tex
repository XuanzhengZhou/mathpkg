Let $\{t_1, \ldots, t_p\}$ and $\{a_1, \ldots, a_q\}$ be sets of integers, at least one of which is nonempty, and suppose that $0 < t_1 < \cdots < t_p < v$ and $0 < a_1 < \cdots < a_q < b$. We say that the system $\Lambda$ is a $(t_1, \ldots, t_p; a_1, \ldots, a_q)$**-design** if $\bar{s}$ is a positive constant on $\mathcal{P}_{t_i}(V)$ for $i = 1, \ldots, p$, and $\overline{s^*}$ is a positive constant on $\mathcal{P}_{a_i}(E)$ for $i = 1, \ldots, q$. For each such design we define functions $\lambda: \{0, t_1, \ldots, t_p\} \to \mathbb{N} \cup \{0\}$ and $\lambda^*: \{0, a_1, \ldots, a_q\} \to \mathbb{N} \cup \{0\}$ by $\lambda(t_i) = \bar{s}(T)$ for $T \in \mathcal{P}_{t_i}(V)$ and $\lambda^*(a_i) = \overline{s^*}(A)$ for $A \in \mathcal{P}_{a_i}(E)$. If $pq > 0$, then $\Lambda$ is **two-sided**. Observe that $\Lambda$ is a $(t_1, \ldots, t_p; a_1, \ldots, a_q)$-design if and only if $\Lambda^*$ is an $(a_1, \ldots, a_q; t_1, \ldots, t_p)$-design.
