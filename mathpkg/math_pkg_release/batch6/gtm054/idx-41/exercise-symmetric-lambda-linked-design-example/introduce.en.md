Exercise A16 asks the reader to verify that a specific set system — consisting of all pairs containing a distinguished vertex $x_0$ together with the single block $V \setminus \{x_0\}$ — is a symmetric $\lambda$-linked design. This provides a concrete example of a nondegenerate, nontrivial $(2;2)$-design that is not a $t$-design for any $t$.
