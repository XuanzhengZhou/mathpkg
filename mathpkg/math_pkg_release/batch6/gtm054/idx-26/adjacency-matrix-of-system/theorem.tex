If $M$ is an incidence matrix of a system $\Lambda = (V, f, E)$, then the matrix $MM^{*} = A = [a_{ij}]$ is called a \textbf{vertex-vertex incidence matrix} (or an \textbf{adjacency matrix}) of $\Lambda$. The set $V = \{x_1, \ldots, x_r\}$ may be indexed so that for $i, j \in \{1, \ldots, r\}$,

$$a_{ij} = |\{e \in E : \{x_i, x_j\} \subseteq f(e)\}| = |f^{*}(x_i) \cap f^{*}(x_j)|.$$
