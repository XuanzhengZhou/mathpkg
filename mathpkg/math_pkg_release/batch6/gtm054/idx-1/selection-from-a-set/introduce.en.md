A selection from $U$ is a function assigning each element a nonnegative integer count, representing "how many times" that element is chosen. Selections generalize subsets by allowing multiple selections of the same element, and their total weight is the sum of all counts. The set of all $m$-selections $\mathbb{S}_m(U)$ is the combinatorial model for multisets.
