For $f: X \rightarrow Y$, the induced function $f: \mathcal{P}(X) \rightarrow \mathcal{P}(Y)$ is given by
$$f[S] = \{f(x): x \in S\}, \quad \text{for all } S \in \mathcal{P}(X).$$
The induced function $f^{-1}: \mathcal{P}(Y) \rightarrow \mathcal{P}(X)$ is given by
$$f^{-1}[T] = \{x: f(x) \in T\}, \quad \text{for all } T \in \mathcal{P}(Y).$$
