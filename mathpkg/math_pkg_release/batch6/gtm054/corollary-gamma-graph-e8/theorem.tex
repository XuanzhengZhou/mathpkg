E9 Corollary. If $\Gamma$ is a graph, then $\Gamma$ is 3-connected if and only if $\Gamma$ is 3-Tutte connected.

We have observed (Exercise E1) that for any submultigraph $\Theta = (U, g, F)$ of $\Gamma$, $A(\Theta) \supseteq A(\Gamma_F)$, and equality holds if $|A(\Theta)| = \tau(\Gamma)$. Hence if $\tau(\Gamma) \leq \nu_1(\Theta) \leq \nu_1(\Gamma) - \tau(\Gamma)$ and $\tau(\Gamma) = |A(\Theta)|$, then $\tau(\Gamma) \leq \nu_1(\Gamma_F) \leq \nu_1(\Gamma) - \tau(\Gamma)$ and $\tau(\Gamma) = |A(\Gamma_F)|$. We note that $\Theta$ consists only of isolated vertices if and only if $F = \varnothing$. We have shown that if $\Gamma$ has no isolated vertices, then $\tau(\Gamma)$ can be determined by considering only those subgraphs of $\Gamma$ which are induced by a nonempty proper subset of $E$.
