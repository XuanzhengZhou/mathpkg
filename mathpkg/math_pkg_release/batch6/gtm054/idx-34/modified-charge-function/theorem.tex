\text{Define } $h: V \rightarrow \mathbb{Q}$ \text{ by } $h = f + g$ \text{, where } $f$ \text{ is a function such that } $\sum_{x \in V} f(x) = 12$ \text{ and } $g$ \text{ is the Appel-Haken discharging function. It follows that }
$$
\sum_{x \in V} h(x) = 12.
$$
\text{One verifies that } $h(x) = 0$ \text{ if } $\rho(x) = 5$ \text{ or } 6.
