\text{The imbedding index } $\iota(\Gamma)$ \text{ of a graph } $\Gamma$ \text{ is defined as }
$$
\iota(\Gamma) = \nu_2(\Gamma) - \dim(\mathcal{L}(\Gamma)),
$$
\text{where } $\nu_2(\Gamma)$ \text{ is the minimum size of a cycle cover of } $\Gamma$ \text{ and } $\mathcal{L}(\Gamma)$ \text{ is the cycle space of } $\Gamma$.
