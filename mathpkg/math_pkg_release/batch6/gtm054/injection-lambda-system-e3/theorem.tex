$f(e) = 1_V[f(e)] = f(p(e))$, for all $e \in E$.

Since $f$ is an injection, $p(e) = e$ for all $e \in E$. Hence $p = 1_E$.

If $\Lambda$ is not a set system, there exists a set $\{e_1, e_2\} \in \mathcal{P}_2(E)$ such that $f(e_1) = f(e_2)$. We define

$$p(e) = \begin{cases} e_2 & \text{if } e = e_1; \\ e_1 & \text{if } e = e_2; \\ e & \text{if } e \in E + \{e_1, e_2\}. \end{cases}$$

Clearly $p$ satisfies E3, and so $(p, 1_V) \in \ker(\pi)$. But $p \neq 1_E$.
