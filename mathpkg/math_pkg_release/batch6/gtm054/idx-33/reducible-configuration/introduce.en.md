A configuration is reducible if, whenever it is immersed in a minimal counterexample to the Four-Color Conjecture (a planar graph with $m$ vertices requiring 5 colors), that counterexample can be reduced to a smaller graph that is 4-colorable, yielding a contradiction. Reducibility is the central technical concept in the Appel-Haken proof: one must show that each configuration in an unavoidable set is reducible.
