Let $\Lambda = (V, \mathcal{M})$ be a matroid.

(a) If $B \in \mathcal{B}(\Lambda)$, then to each $x \in V + B$ there corresponds a unique cycle $M_x \in \mathcal{M}$ such that $x \in M_x \subseteq B + \{x\}$.

(b) If $M \in \mathcal{M}$, then to each $x \in M$ there corresponds a basis $B_x \in \mathcal{B}(\Lambda)$ such that $M \subseteq B_x + \{x\}$.

(c) Let $S \in \mathcal{P}(V)$. Then $S \in \mathcal{S}(\Lambda)$ if and only if to each $x \in V + S$ there corresponds a cycle $M_x$ such that $x \in M_x \subseteq S + \{x\}$.
