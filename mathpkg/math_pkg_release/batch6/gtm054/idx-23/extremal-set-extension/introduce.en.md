This lemma shows that any extremal set $U$ can be extended to a larger extremal set $W$ whose union with its neighborhood covers the entire vertex set $V$. The construction $W = V \setminus N(U)$ preserves extremality while ensuring that every vertex is either in $W$ or adjacent to $W$. The number of odd components does not decrease under this extension. This result is a key structural tool in the proof of the general 1-factor theorem.
