Assume the result (proved in Section VIE): if $\Gamma$ is a triconnected graph and if $\Gamma \perp \Theta$, then $\Theta$ is likewise a triconnected graph. Using this, show that if $\Gamma$ is a triconnected planar graph, then the set of regions of a planar imbedding is unique and that there exists only one connected multigraph $\Theta$ such that $\Gamma \perp \Theta$.
