A clique in a graph $\Gamma$ is a largest possible complete set of vertices — a maximum cardinality subset of vertices in which every pair is adjacent. The clique number $\omega(\Gamma)$ is the cardinality of a clique. This parameter measures the size of the largest complete subgraph contained in $\Gamma$. The clique number is dual to the vertex independence number $\alpha_0(\Gamma)$ via graph complement: $\alpha_0(\Gamma) = \omega(\Gamma')$.
