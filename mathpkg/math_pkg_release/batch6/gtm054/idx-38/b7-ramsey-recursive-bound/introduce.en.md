Theorem B7 provides the core recursive estimate that proves the finiteness of all Ramsey numbers. For two colors ($m=2$), the Ramsey number $n(q_1, q_2 : s)$ is bounded above by $n(p_1, p_2 : s-1) + 1$, where $p_1 = n(q_1-1, q_2 : s)$ and $p_2 = n(q_1, q_2-1 : s)$. This double induction on $s$ and on the parameters $q_i$ (together with the base case B4 for $s=1$) establishes that every Ramsey number exists and is finite.
