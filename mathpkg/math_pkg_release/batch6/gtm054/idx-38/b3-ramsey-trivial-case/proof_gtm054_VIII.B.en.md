With $m=1$, there is only one color. For any set $V$ with $|V| \geq q$, taking $S$ to be any $q$-subset of $V$, we have $\mathcal{P}_s(S) \subseteq h^{-1}[1]$ trivially, since $h$ maps all $s$-subsets to color $1$. Thus $n(q : s) \leq q$. Conversely, if $|V| < q$, then no $q$-subset exists at all, so $n(q : s) = q$.
