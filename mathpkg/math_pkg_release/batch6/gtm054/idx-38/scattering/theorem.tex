A largest independent subset of the vertex set $V$ of a graph $\Gamma$ is called a \textbf{scattering}. Its cardinality is the vertex independence number $\alpha_0(\Gamma)$.

A subset $S \subseteq V$ is independent in $\Gamma$ if and only if $S$ is a complete set in the complement $\Gamma'$. Thus
$$\alpha_0(\Gamma) = \omega(\Gamma').$$
