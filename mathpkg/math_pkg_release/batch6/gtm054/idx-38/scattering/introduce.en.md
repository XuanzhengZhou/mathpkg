A scattering in a graph $\Gamma$ is a maximum-size independent set of vertices — equivalently, a maximum set of mutually non-adjacent vertices. Its cardinality is precisely the vertex independence number $\alpha_0(\Gamma)$. The terminology establishes a duality with cliques: a set is independent in $\Gamma$ if and only if it is complete in the complement $\Gamma'$, yielding the fundamental identity $\alpha_0(\Gamma) = \omega(\Gamma')$.
