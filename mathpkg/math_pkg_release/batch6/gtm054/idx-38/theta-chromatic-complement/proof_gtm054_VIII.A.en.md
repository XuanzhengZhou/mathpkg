A surjection $h: V \to \{1, \ldots, m\}$ has the property that $h^{-1}[i]$ is a complete set for each $i$ if and only if no two vertices in the same fiber $h^{-1}[i]$ are non-adjacent in $\Gamma$, which is equivalent to saying no two vertices receiving the same color are adjacent in $\Gamma'$. Hence $h$ is a surjection of $V$ into $\{1,\ldots,m\}$ whose color classes are complete sets in $\Gamma$ if and only if $h$ is a proper vertex $m$-coloring of $\Gamma'$. Taking the minimum such $m$ yields $\theta(\Gamma) = \chi_0(\Gamma')$.
