A subset $S \subseteq V$ is independent in $\Gamma$ if and only if no two vertices of $S$ are adjacent in $\Gamma$, which means every pair of distinct vertices in $S$ is adjacent in $\Gamma'$. Thus $S$ is independent in $\Gamma$ if and only if $S$ is a complete set in $\Gamma'$. A scattering is a largest independent set in $\Gamma$, and a clique is a largest complete set in $\Gamma'$. Therefore the cardinalities of these maximum sets are equal:

$$\alpha_0(\Gamma) = \omega(\Gamma').$$
