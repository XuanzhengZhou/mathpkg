Proposition A2 establishes the fundamental duality between the vertex independence number $\alpha_0(\Gamma)$ and the clique number $\omega(\Gamma')$ of the complement. A subset of vertices is independent in $\Gamma$ precisely when it is complete in the complement $\Gamma'$, so the maximum size of an independent set in $\Gamma$ equals the maximum size of a complete set in $\Gamma'$.
