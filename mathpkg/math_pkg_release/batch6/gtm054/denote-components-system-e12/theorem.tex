Denote the set of components of a system $\Lambda$ by

$$\{\Lambda_{ij}: i = 1, \ldots, n_j; j = 1, \ldots, r\}$$

where $\Lambda_{ij}$ and $\Lambda_{i'j'}$ are system-isomorphic if and only if $j = j'$. Let $\Lambda_{ij} = (V_{ij}, f_{ij}, E_{ij})$ for each $i, j$. Let $m_{0j} = |\{i: V_{ij} \neq \emptyset\}|$ and $m_{1j} = |\{i: E_{ij} \neq \emptyset\}|$. Then for $h = 0, 1$,

$$G_h(\Lambda) = \bigoplus_{j=1}^{r} (\Pi(m_{hj}) \text{ wreath } G_h(\Lambda_{1j}))$$

(Here $\oplus$ denotes the direct product of permutation groups.)

If $G_0(\Lambda) = \{1_V\}$, then $\Lambda$ is said to be asymmetric. If $\Lambda$ is not asymmetric but if every element of $G_0(\Lambda)$ is a derangement of $V$, then $\Lambda$ is said to be fixed-point free.

The next result was first proved by G. Sabidussi [s.3] for the special case of graphs.
