The power set $\mathcal{P}(U)$ of a set $U$ is the collection of all subsets of $U$. It is a fundamental construction in set theory and combinatorics, forming the basis for studying subsets, Boolean algebras, and cardinality relations. The notation $\mathcal{P}(U)$ and its variants are used throughout the text.
