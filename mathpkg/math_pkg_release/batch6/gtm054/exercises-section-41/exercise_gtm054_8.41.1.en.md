Show that if $\Lambda$ is a $t$-design with design parameter $\lambda$, if $0 \leq j \leq i \leq t$, and if $S \in \mathcal{P}_i(V)$, then the number of blocks whose images meet $S$ in a $j$-set is

$$\sum_{h=0}^{t-j} \frac{(-1)^h i! \lambda(h+j)}{h! j! (i-h-j)!}.$$

All $t$-designs readily generate other $t$-designs, as the next three propositions indicate.
