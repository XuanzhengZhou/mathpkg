This proposition establishes fundamental relationships between several key graph-theoretic parameters: the vertex-covering number of edge sets $\beta_{10}$, the independent edge set number $\alpha_1$, the graph deficiency $\delta$, and the independent vertex set number $\alpha_0$. Part (a) provides an exact identity linking these parameters in the presence of isolated vertices, while part (b) gives bounds that sandwich the independence number $\alpha_0$ between the deficiency and the adjusted vertex-covering number.
