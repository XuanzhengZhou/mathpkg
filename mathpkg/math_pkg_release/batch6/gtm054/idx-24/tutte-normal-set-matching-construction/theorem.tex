Let $\Gamma = (V, \mathcal{E})$ be a graph and let $U \subseteq V$ be a maximal normal set. Then:
$$\delta(\Gamma) = \nu_0(\Gamma) - 2\alpha_1(\Gamma).$$
Moreover, every largest independent set $\mathcal{F}$ of edges of $\Gamma$ can be constructed as follows:

\begin{enumerate}
\item[(a)] Let $\mathcal{F}'$ be a matching of $\{\{x\}: x \in N(U)\}$ in the bipartite graph $B_U$.

\item[(b)] Let $C$ be the vertex set of a component of $\Gamma_U$. Regarding $C$ as a vertex of $B_U$:
\begin{enumerate}
\item[(i)] if $C$ is incident with an edge $E \in \mathcal{F}'$, then let $x_C$ be the vertex in $C$ incident with $E$ in $\Gamma$.
\item[(ii)] if $C$ is incident with no edge in $\mathcal{F}'$, then arbitrarily select $x_C \in C$.
\end{enumerate}

\item[(c)] Since $U$ is normal, there exists a 1-factor $\mathcal{F}_C$ in the component of $\Gamma_U$ with vertex set $C \setminus \{x_C\}$ (or simply in the component itself if no vertex was removed).

\item[(d)] Let $\mathcal{F} = \mathcal{F}' + \sum_C \mathcal{F}_C$.
\end{enumerate}

In particular, this result generalizes Corollary A6 from bipartite graphs to all graphs.
