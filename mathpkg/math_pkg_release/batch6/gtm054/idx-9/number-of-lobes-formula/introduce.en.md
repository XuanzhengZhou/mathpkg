This exercise (starred as [o.1]) provides a formula for computing the number of lobes of a multigraph in terms of a vertex invariant $i(x)$ defined via rank differences. The quantity $i(x)$ measures the increase in the number of connected components when vertex $x$ is removed. The formula generalizes the fact that for trees, the number of leaves plus internal branch points determines the tree structure.
