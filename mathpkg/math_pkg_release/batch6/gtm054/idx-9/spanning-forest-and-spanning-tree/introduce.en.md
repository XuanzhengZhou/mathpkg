A spanning forest of a multigraph is a subgraph that is a forest, contains all vertices of the original multigraph, and has the same number of connected components. A spanning tree is a spanning forest that is connected. Every multigraph contains a spanning forest, and every connected multigraph contains a spanning tree — this follows from Corollary C5, establishing that spanning forests/trees always exist.
