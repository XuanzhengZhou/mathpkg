An $x_0x$-path $x_0, e_1, x_1, \ldots, e_m, x$ is said to be \textbf{unsaturated} with respect to a flow $h$ if

\begin{enumerate}
\item $e_i \neq e_0$ for $i = 1, \ldots, m$;
\item $e_i = (x_{i-1}, x_i) \implies h(e_i) < k(e_i)$ (forward edge not at capacity);
\item $e_i = (x_i, x_{i-1}) \implies h(e_i) > 0$ (backward edge has positive flow).
\end{enumerate}
