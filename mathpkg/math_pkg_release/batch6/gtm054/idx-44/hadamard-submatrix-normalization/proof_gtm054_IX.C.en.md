Starting from C31, $MM^* = qI + (q-1)J$, we derive the following sequence of equivalent equations under the assumptions on $M$ (or equivalently on $H_1 = 2M - J$):

$$4MM^* - (4q - 4)J = 4qI,$$

$$4MM^* - 2(2q - 1)J - 2(2q - 1)J + (4q - 1)J = 4qI - J,$$

$$4MM^* - 2JM - 2JM^* + J^2 = 4qI - J,$$

$$(H_1 + J)(H_1^* + J) - J(H_1 + J) - J(H_1^* + J) + J^2 = 4qI - J.$$

Expanding the left-hand side of the last expression and simplifying yields

$$H_1 H_1^* = 4qI - J,$$

which is C32.
