Let $\Gamma = (V, f, E)$ be a multigraph.
\begin{itemize}
\item $\nu_{-1}(\Gamma)$ denotes the number of components of $\Gamma$.
\item $\nu_0(\Gamma)$ denotes the number of vertices of $\Gamma$.
\item $\nu_1(\Gamma)$ denotes the number of edges of $\Gamma$.
\end{itemize}
When there is no risk of confusion, we write briefly $\nu_{-1}$, $\nu_0$, and $\nu_1$.
