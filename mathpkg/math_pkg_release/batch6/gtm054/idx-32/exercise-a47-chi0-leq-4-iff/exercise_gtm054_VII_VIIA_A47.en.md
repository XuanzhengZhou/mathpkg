Let $\Gamma = (V, \mathcal{E})$.

(a) (Ore, [o.2, p. 122]). Prove that $\chi_0(\Gamma) \leq 4$ if and only if there exists a set $S \subseteq V$ such that $\Gamma_S$ and $\Gamma_{V+S}$ are bipartite.

(b) (Woodall, [w.13]). Prove that $\chi_0(\Gamma) \leq 4$ if and only if there exists a set $\mathcal{F} \subseteq \mathcal{E}$, such that $\Gamma_{(\mathcal{F})}$ and $\Gamma_{(\mathcal{E}+\mathcal{F})}$ are each bipartite.
