Let $\Lambda = (V, f, E)$ be a set system with $V \neq \varnothing$ or $E \neq \varnothing$. The **automorphism group** $G(\Lambda)$ is the set
$$G(\Lambda) = \{(p, q) : p \in \Pi(E);\; q \in \Pi(V);\; q[f(e)] = f(p(e)) \text{ for all } e \in E\},$$
with the operation of componentwise composition
$$(p_2, q_2)(p_1, q_1) = (p_2p_1, q_2q_1).$$
