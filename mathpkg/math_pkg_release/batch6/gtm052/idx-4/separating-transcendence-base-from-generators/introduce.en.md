This theorem guarantees that for any finitely generated separably generated field extension $K/k$, one can extract a separating transcendence base from an arbitrary set of generators. This is a useful technical result for working with function fields of varieties: given any set of rational functions that generate the function field, one can select a subset that forms a separating transcendence base.
