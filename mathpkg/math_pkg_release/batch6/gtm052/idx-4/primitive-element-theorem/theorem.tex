Let $L$ be a finite separable extension field of a field $K$. Then there is an element $\alpha \in L$ which generates $L$ as an extension field of $K$. Furthermore, if $\beta_1, \ldots, \beta_n$ is any set of generators of $L$ over $K$, and if $K$ is infinite, then $\alpha$ can be taken to be a linear combination $\alpha = c_1\beta_1 + \cdots + c_n\beta_n$ of the $\beta_i$ with coefficients $c_i \in K$.
