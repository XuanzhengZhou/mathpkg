Let $K$ be a finitely generated extension field of a field $k$. Then

$$\dim_K \Omega_{K/k} \geqslant \operatorname{tr.deg.} K/k,$$

and equality holds if and only if $K$ is separably generated over $k$. In particular, if $K/k$ is a finite algebraic extension, then $\Omega_{K/k} = 0$ if and only if $K/k$ is separable.
