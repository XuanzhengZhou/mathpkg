Let $B$ be an $A$-algebra. Let $f: B \otimes_A B \rightarrow B$ be the diagonal homomorphism defined by $f(b \otimes b') = bb'$, and let $I = \ker f$. Consider $B \otimes_A B$ as a $B$-module by multiplication on the left. Then $I/I^2$ inherits a structure of $B$-module. Define a map $d: B \rightarrow I/I^2$ by $db = 1 \otimes b - b \otimes 1$ (modulo $I^2$). Then $\langle I/I^2, d \rangle$ is a module of relative differentials for $B/A$.
