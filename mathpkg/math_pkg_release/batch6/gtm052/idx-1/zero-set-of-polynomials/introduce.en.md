The zero set $Z(T)$ is the fundamental geometric object associated to a collection of polynomials. It consists of all points in affine space where every polynomial in $T$ simultaneously vanishes. Because $k[x_1, \ldots, x_n]$ is noetherian, any zero set can be described by finitely many polynomial equations, which makes them computationally and theoretically tractable.
