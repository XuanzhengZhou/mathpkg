There is a one-to-one inclusion-reversing correspondence between algebraic sets in $\mathbf{A}^n$ and radical ideals (i.e., ideals which are equal to their own radical) in $A = k[x_1,\ldots,x_n]$, given by $Y \mapsto I(Y)$ and $\mathfrak{a} \mapsto Z(\mathfrak{a})$. Furthermore, an algebraic set is irreducible if and only if its ideal is a prime ideal.
