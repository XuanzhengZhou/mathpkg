Let $M$ be a finitely generated graded module over a noetherian graded ring $S$. Then there exists a filtration
$$0 = M^0 \subseteq M^1 \subseteq \ldots \subseteq M^r = M$$
by graded submodules, such that for each $i$, $M^i / M^{i-1} \simeq (S/\mathfrak{p}_i)(l_i)$ for some homogeneous prime ideal $\mathfrak{p}_i \subseteq S$ and some integer $l_i \in \mathbf{Z}$. Moreover:
\begin{enumerate}
  \item[(a)] $\mathfrak{p} \supseteq \operatorname{Ann} M$ if and only if $\mathfrak{p} \supseteq \mathfrak{p}_i$ for some $i$.
  \item[(b)] For each minimal prime $\mathfrak{p}$ of $M$, the number of times $\mathfrak{p}$ appears among $\{\mathfrak{p}_1, \ldots, \mathfrak{p}_r\}$ equals the length of $M_{\mathfrak{p}}$ as an $S_{\mathfrak{p}}$-module.
\end{enumerate}
