Let $Y$ be a reduced curve on a nonsingular surface $X$. Then there exists a finite sequence of monoidal transformations $f: X' \to X$ such that:
\begin{itemize}
\item each irreducible component of the strict transform $\tilde{Y}$ of $Y$ is nonsingular,
\item the total transform $f^{-1}(Y)$ is a divisor with normal crossings (each irreducible component is nonsingular, and the only singularities are ordinary double points).
\end{itemize}
The process terminates because blowing up any singularity that is not a node strictly reduces the arithmetic genus: if $\mu_P(Y) \geqslant 3$, then $p_a(\pi^{-1}(Y)) < p_a(Y)$, and there can be only finitely many drops.
