If $|D|$ is very ample, then by (4.0.1) it has no base points and $|D - P|$ has no unassigned base points for every $P$. When we assign a second base point $Q$ (possibly infinitely near $P$), we are adding a new condition that was not already satisfied by all divisors in $|D - P|$, since there are no unassigned base points. Thus the dimension drops by exactly one for each assignment, giving $\dim |D - P - Q| = \dim |D| - 2$. Conversely, if the dimension drops by exactly 2 for any two points, then neither $P$ nor $Q$ can be fixed points of $|D|$, and $Q$ is never forced to be a base point of $|D - P|$. Hence both conditions of (4.0.1) hold.
