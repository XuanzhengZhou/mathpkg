This result reformulates the classical criterion for a complete linear system $|D|$ on a surface to be very ample (II, 7.8.2) using the language of assigned base points. It states that $|D|$ is very ample exactly when it has no base points and, for every point $P$, the linear system $|D-P|$ with one assigned base point has no unintended (unassigned) base points. The separation of tangent directions is captured by the absence of unassigned base points infinitely near $P$.
