Let $P_1, \ldots, P_r$ be points in $\mathbf{P}^2$, no three collinear, and possibly with $P_2$ infinitely near $P_1$. Let $\mathfrak{d} = |2H - P_1 - \ldots - P_r|$ be the linear system of plane conics through $P_1, \ldots, P_r$. Then:
\begin{enumerate}
\item[(a)] if $r \leqslant 5$, then $\dim \mathfrak{d} = 5 - r$;
\item[(b)] if $r = 5$, then there exists a unique conic containing $P_1, \ldots, P_5$, which is necessarily irreducible.
\end{enumerate}
Furthermore, these results remain true if $P_5$ is infinitely near any one of $P_1, \ldots, P_4$.
