Let $\mathfrak{d}$ be the linear system of plane cubic curves with assigned base points $P_1, \ldots, P_r$ in $\mathbf{P}^2$, and assume that no 4 of the $P_i$ are collinear, and no 7 of them lie on a conic. If $r \leqslant 7$, then $\mathfrak{d}$ has no unassigned base points. This result remains true if $P_2$ is infinitely near $P_1$.
