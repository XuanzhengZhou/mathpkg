Let $A$ be a ring, and let $(\text{Spec } A, \mathcal{O})$ be its spectrum with structure sheaf $\mathcal{O}$.

(a) For any $\mathfrak{p} \in \text{Spec } A$, the stalk $\mathcal{O}_\mathfrak{p}$ is isomorphic to the local ring $A_\mathfrak{p}$.

(b) For any $f \in A$, let $D(f) = \{\mathfrak{p} \in \text{Spec } A \mid f \notin \mathfrak{p}\}$. Then $\mathcal{O}(D(f))$ is isomorphic to the localized ring $A_f$.

(c) In particular, $\Gamma(\text{Spec } A, \mathcal{O}) \cong A$.
