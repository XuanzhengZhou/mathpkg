Let $D$ be a divisor on a curve $X$. The \textbf{complete linear system} associated to $D$ is the set of all effective divisors linearly equivalent to $D$, denoted by $|D|$. The elements of $|D|$ are in one-to-one correspondence with the projective space

$$|D| \cong \bigl(H^0(X, \mathcal{L}(D)) \setminus \{0\}\bigr) / k^*,$$

so $|D|$ carries the structure of the set of closed points of a projective space. The dimension of $H^0(X, \mathcal{L}(D))$ is denoted by $l(D)$, so that

$$\dim |D| = l(D) - 1.$$

The number $l(D)$ is always finite.
