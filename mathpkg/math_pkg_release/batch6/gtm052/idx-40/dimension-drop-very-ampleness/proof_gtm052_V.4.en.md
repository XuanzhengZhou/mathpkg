If we observe that the dimension drops by exactly one when we assign a base point which was not already an unassigned base point of a linear system, then this follows directly from (4.0.1). Condition (a) of (4.0.1) says $\dim |D - P| = \dim |D| - 1$ for all $P$. Condition (b) says $\dim |D - P - Q| = \dim |D - P| - 1$, since $Q$ (possibly infinitely near $P$) is not an unassigned base point of $|D - P|$. Together, $\dim |D - P - Q| = \dim |D| - 2$.
