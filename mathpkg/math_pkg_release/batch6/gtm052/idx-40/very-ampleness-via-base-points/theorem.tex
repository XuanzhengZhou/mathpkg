Let $X$ be a surface and let $|D|$ be a complete linear system. Then $|D|$ is very ample if and only if:
\begin{enumerate}
\item[(a)] $|D|$ has no base points, and
\item[(b)] for every $P \in X$, $|D - P|$ has no unassigned base points.
\end{enumerate}
Indeed, $|D|$ separates the points $P$ and $Q$ if and only if $Q$ is not a base point of $|D - P|$, and $|D|$ separates tangent vectors at $P$ if and only if $|D - P|$ has no unassigned base points infinitely near $P$.
