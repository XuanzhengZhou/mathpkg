Let $P_1, \ldots, P_r$ be points in $\mathbf{P}^2$, no three collinear, and let $\mathfrak{d}$ be the linear system of all conics passing through $P_1, \ldots, P_r$. Then:
\begin{enumerate}
\item[(a)] if $r \leqslant 5$, then $\dim \mathfrak{d} = 5 - r$;
\item[(b)] if $r = 5$, there exists a unique conic containing $P_1, \ldots, P_5$, which is necessarily irreducible.
\end{enumerate}
These results remain true if $P_5$ is infinitely near any one of $P_1, \ldots, P_4$.
