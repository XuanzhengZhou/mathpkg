A linear system with assigned base points is a sublinear system $|D - P_1 - \cdots - P_r|$ consisting of divisors in a complete linear system $|D|$ that pass through specified points $P_1, \ldots, P_r$. This language allows studying linear systems on blown-up models of $X$ in terms of linear systems with assigned base points on $X$, including infinitely near points. The convention about multiplicities for infinitely near base points ensures consistency: a point $P$ must be assigned multiplicity at least equal to the sum of assigned multiplicities of base points infinitely near $P$.
