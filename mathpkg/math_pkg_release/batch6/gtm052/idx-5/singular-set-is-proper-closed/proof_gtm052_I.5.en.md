First we show $\operatorname{Sing} Y$ is a closed subset. It is sufficient to show for some open covering $Y = \bigcup Y_i$ of $Y$ that $\operatorname{Sing} Y_i$ is closed for each $i$. Hence by (4.3) we may assume that $Y$ is affine. By (5.2) and the proof of (5.1) we know that the rank of the Jacobian matrix is always $\leq n - r$, where $r = \dim Y$ and $Y \subseteq \mathbf{A}^n$. Hence the set of singular points is the set of points where the rank is $\leq n - r - 1$ (strict inequality). Thus $\operatorname{Sing} Y$ is the algebraic set defined by the ideal generated by $I(Y)$ together with all determinants of $(n - r) \times (n - r)$ submatrices of the Jacobian matrix $\|\partial f_i/\partial x_j\|$. Hence $\operatorname{Sing} Y$ is closed.

To show that $\operatorname{Sing} Y$ is a proper subset of $Y$, we first apply (4.9) to get $Y$ birational to a hypersurface in $\mathbf{P}^n$. Since birational varieties have isomorphic open subsets, we reduce to the case of a hypersurface. It is enough to consider any open affine subset of $Y$, so we may assume that $Y$ is a hypersurface in $\mathbf{A}^n$, defined by a single irreducible polynomial $f(x_1, \ldots, x_n) = 0$.

Now $\operatorname{Sing} Y$ is the set of points $P \in Y$ such that $(\partial f/\partial x_i)(P) = 0$ for $i = 1, \ldots, n$. If $\operatorname{Sing} Y = Y$, then the ideal of $Y$, which is $(f)$, would be contained in the ideal generated by the partial derivatives $\partial f/\partial x_i$. But $\deg(\partial f/\partial x_i) < \deg f$ (assuming characteristic $0$, or more generally that $f$ is separable in its essential variables). This forces $f$ to divide each $\partial f/\partial x_i$, which is impossible since each $\partial f/\partial x_i$ is nonzero (otherwise $f$ would be a $p$-th power in characteristic $p$, contradicting that $f$ defines a reduced hypersurface). Therefore $\operatorname{Sing} Y \subsetneq Y$, and $\operatorname{Sing} Y$ is a proper closed subset.
