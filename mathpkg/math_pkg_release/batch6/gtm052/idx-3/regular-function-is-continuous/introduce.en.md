Lemma 3.1 establishes that every regular function on a variety is continuous with respect to the Zariski topology. This connects the algebraic notion of regularity with the topological structure, and is essential for the definition of morphisms. The proof reduces to showing that the preimage of any point (a closed set in $\mathbf{A}^1$) is closed, using a local representation of the function as a rational expression.
