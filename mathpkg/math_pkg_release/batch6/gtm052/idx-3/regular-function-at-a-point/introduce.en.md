A regular function on a quasi-affine variety is defined locally as a quotient of polynomials. At each point $P$, the function can be expressed as $g/h$ where $h(P) \neq 0$ in some neighborhood. This definition generalizes polynomial functions to allow rational expressions, provided the denominator does not vanish. Regular functions form the foundation for defining morphisms between varieties.
