An additive functor $F: \mathfrak{A} \to \mathfrak{B}$ between abelian categories is \text{effaceable} if for each object $A$ of $\mathfrak{A}$, there exists a monomorphism $u: A \to M$ for some $M$ such that $F(u) = 0$.

$F$ is \text{coeffaceable} if for each object $A$ there exists an epimorphism $u: P \to A$ for some $P$ such that $F(u) = 0$.
