An effaceable functor is one where every object can be embedded into another object in such a way that the functor annihilates the embedding morphism. This seemingly technical condition is the key criterion for universality of $\delta$-functors: if each $T^i$ for $i > 0$ is effaceable, then the $\delta$-functor $(T^i)$ is universal. For derived functors, effaceability follows because every object embeds into an injective object, and higher derived functors vanish on injectives.
