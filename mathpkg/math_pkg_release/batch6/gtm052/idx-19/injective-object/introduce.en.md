An injective object in an abelian category is one for which the contravariant Hom functor $\operatorname{Hom}(\cdot, I)$ is exact, meaning it preserves short exact sequences. Since $\operatorname{Hom}(\cdot, I)$ is always left exact, injectivity is equivalent to the condition that for every monomorphism $A \hookrightarrow B$, the induced map $\operatorname{Hom}(B, I) \to \operatorname{Hom}(A, I)$ is surjective. Injective objects serve as the target building blocks for injective resolutions, which are used to compute right derived functors.
