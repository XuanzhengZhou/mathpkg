Let $\mathfrak{d}$ be a linear system on a variety $X$ corresponding to a subspace $V \subseteq \Gamma(X, \mathcal{L})$, where $\mathcal{L}$ is an invertible sheaf. Then a point $P \in X$ is a base point of $\mathfrak{d}$ if and only if $s_P \in \mathfrak{m}_P \mathcal{L}_P$ for all $s \in V$. In particular, $\mathfrak{d}$ is base-point-free if and only if $\mathcal{L}$ is generated by the global sections in $V$.
