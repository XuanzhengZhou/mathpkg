Let $\mathfrak{d}$ be a linear system on a variety $X$. A point $P \in X$ is a **base point** of $\mathfrak{d}$ if $P \in \operatorname{Supp} D$ for all $D \in \mathfrak{d}$, where $\operatorname{Supp} D$ denotes the union of the prime divisors of $D$. The linear system $\mathfrak{d}$ is **base-point-free** if it has no base points.
