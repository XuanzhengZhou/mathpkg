A morphism $f: X \to Y$ is proper if it is separated, of finite type, and universally closed. A morphism $f: X \to Y$ is closed if the image of any closed subset is closed. A morphism $f: X \to Y$ is universally closed if it is closed, and for any morphism $Y' \to Y$, the base extension $f': X \times_Y Y' \to Y'$ is also closed.
