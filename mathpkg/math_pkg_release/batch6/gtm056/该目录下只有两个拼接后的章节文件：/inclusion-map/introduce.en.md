Given a set $A$ that is a subset of a set $B$, the inclusion map $i: A \to B$ is the canonical embedding defined by $i(x) = x$ for all $x \in A$. It is the simplest example of an injective function and serves as the identity function restricted to the subset $A$. In topology, inclusion maps are fundamental for relating subspaces to ambient spaces and for studying induced homomorphisms on fundamental groups.
