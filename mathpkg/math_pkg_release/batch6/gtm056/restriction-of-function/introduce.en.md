The restriction of a function to a subset is obtained by limiting the domain of the function to that subset while preserving the original mapping rule. If $i: A \hookrightarrow B$ denotes the inclusion map, then $f \mid A = f \circ i$. Restriction is a ubiquitous operation in mathematics, particularly useful when studying local properties, extending functions, or analyzing behavior on subspaces in topology.
