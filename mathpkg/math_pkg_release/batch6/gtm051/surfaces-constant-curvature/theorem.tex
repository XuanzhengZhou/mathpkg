For a surface with a Riemannian metric, the following are equivalent:
(i) $K = \text{const}$;
(ii) In geodesic polar coordinates, $\sqrt{g_{22}}$ depends only on $u^1$;
(iii) In geodesic parallel coordinates, $\sqrt{g_{22}}$ depends only on $u^1$;
(iv) All curves $u^1 = \text{const}$ (in geodesic parallel coordinates) are geodesics.
