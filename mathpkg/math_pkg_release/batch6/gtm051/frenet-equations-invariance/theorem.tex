Let $c: I \to \mathbb{R}^n$ be a curve with Frenet frame $(e_i(t))$ and Frenet matrix $\omega(t)$.
(i) If $B: \mathbb{R}^n \to \mathbb{R}^n$ is an isometry, then $\tilde{c} = B \circ c$ has Frenet frame $\tilde{e}_i = R e_i$ (where $R$ is the orthogonal part of $B$) and $\tilde{\omega}_{ij} = \omega_{ij}$.
(ii) If $\phi: J \to I$ is a change of variables and $\tilde{c} = c \circ \phi$, then the Frenet curvatures satisfy $\tilde{\kappa}_i(s) = \kappa_i(\phi(s))$.
