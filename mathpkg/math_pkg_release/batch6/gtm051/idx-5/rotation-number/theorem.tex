Let $c: [0, \omega] \to \mathbb{R}^2$ be a piecewise smooth, regular, closed curve. Let $0 = b_{-1} = a_0 < b_0 = a_1 < \ldots < b_k = \omega$ be a partition of $[0, \omega]$ into intervals $I_j := [a_j, b_j]$ on which $c_j := c|_{I_j}$ is smooth.

Let $\theta: [0, \omega] \to \mathbb{R}$ be a continuous angle function as in Proposition 2.1.3, with $\theta$ continuous on each $[a_j, b_j]$ and having jumps $\alpha_j$ at the corners $c(a_j) = c(b_{j-1})$.

The \text{rotation number} $n_c$ of $c$ is defined by
$$n_c := \frac{1}{2\pi}(\theta(\omega) - \theta(0)).$$
