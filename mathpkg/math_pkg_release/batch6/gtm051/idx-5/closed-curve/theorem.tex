A curve $c: I = [a, b] \to \mathbb{R}^n$ is \text{closed} if there exists a curve $\tilde{c}: \mathbb{R} \to \mathbb{R}^n$ such that $\tilde{c}|_I = c$ and $\tilde{c}(t + \omega) = \tilde{c}(t)$ for all $t \in \mathbb{R}$, where $\omega = b - a$. The number $\omega$ is the \text{period} of $c$, and $\tilde{c}$ is said to be \text{periodic} with period $\omega$.

Equivalently, a curve $c: [a, b] \to \mathbb{R}^n$ is closed if $c(a) = c(b)$ and $c^{(k)}(a) = c^{(k)}(b)$ for all $k > 0$.
