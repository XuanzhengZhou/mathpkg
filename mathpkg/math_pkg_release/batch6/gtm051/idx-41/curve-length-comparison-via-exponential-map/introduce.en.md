This lemma compares the length of a geodesic segment with any competing curve that can be expressed through the exponential map at the initial point. Under the curvature upper bound $K \leq K_1$, with $\rho = \pi / \sqrt{K_1}$, the exponential map restricted to $B_\rho(0)$ is a local isometry when the tangent space is endowed with a suitable pullback metric. The result then follows from the Gauss Lemma (5.3.4), which asserts that in geodesic polar coordinates, radial geodesics minimize length among all curves with the same endpoints lying within the coordinate chart.
