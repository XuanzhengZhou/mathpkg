\text{Let }c=c(t),\;0\leq t\leq a,\text{ be a unit-speed geodesic on }M.

\text{A vector field }Y(t)\text{ along }c\text{ is a \textbf{Jacobi field} provided }g_{c(t)}(\dot{c}(t),Y(t))=0,\text{ i.e., }Y\text{ is orthogonal to }c,\text{ and }

\frac{\nabla^2 Y}{dt^2}(t) + K\circ c(t)\,Y(t) = 0.

\text{In terms of the Frenet frame }e_1(t),e_2(t)\text{ on }c\text{, writing }Y(t)=y_2(t)e_2(t)\text{, the equation is equivalent to }

\ddot{y}(t) + K\circ c(t)\,y(t) = 0.
