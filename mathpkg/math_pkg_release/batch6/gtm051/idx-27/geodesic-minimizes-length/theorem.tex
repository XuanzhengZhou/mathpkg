\text{Let }c=c(t),\;0\leq t\leq 1,\text{ be a unit-speed geodesic from }p=c(0)\text{ to }q=c(1).

\text{If }b=b(s),\;s_0\leq s\leq s_1,\text{ is any other curve from }p=b(s_0)\text{ to }q=b(s_1),\text{ then }

L(b)\geq L(c)

\text{with equality if and only if there exists a diffeomorphism }t:[s_0,s_1]\to[0,1]\text{ with }dt/ds\geq 0\text{ and }b(s)=c(t(s)).
