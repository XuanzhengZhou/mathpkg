Given any prescribed positive curvature functions $\kappa_1(s), \dots, \kappa_{n-1}(s)$, there exists a unit speed curve in $\mathbb{R}^n$ having these functions as its curvatures. The proof constructs the curve by solving a linear system of ordinary differential equations for an orthogonal matrix-valued function $X(s)$, whose first column is the unit tangent vector. Integrating this tangent vector yields the curve. The key observation is that the skew-symmetry of the coefficient matrix $A(s)$ forces the solution $X(s)$ to remain orthogonal.
