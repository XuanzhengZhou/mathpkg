Let $(e_i(t))$ be a moving $n$-frame along $c$. Then the derivatives $\dot{e}_i(t)$ can be expressed as
$$\dot{e}_i(t) = \sum_j \omega_{ij}(t) e_j(t)$$
where $\omega(t) = (\omega_{ij}(t))$ is a skew-symmetric matrix. If $(e_i(t))$ is a distinguished Frenet frame, then $\omega_{ij}(t) = 0$ for $|i - j| > 1$; i.e., $\omega$ is the tridiagonal matrix with non-zero entries only $\omega_{i,i+1} = -\omega_{i+1,i}$.
