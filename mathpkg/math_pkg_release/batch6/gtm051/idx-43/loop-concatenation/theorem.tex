If $\alpha, \beta \in \Omega(p)$, the concatenated curve $\beta * \alpha$ is defined by

$$\beta * \alpha(t) = \begin{cases} \alpha(2t), & \text{if } 0 \leq t \leq \frac{1}{2} \\[4pt] \beta(2(1 - t)), & \text{if } \frac{1}{2} \leq t \leq 1. \end{cases}$$

That is, $\beta * \alpha$ is the curve $\alpha$ traversed forward followed by $\beta$ traversed backward.
