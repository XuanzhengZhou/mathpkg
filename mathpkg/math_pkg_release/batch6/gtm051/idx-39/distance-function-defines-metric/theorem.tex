Let $M$ be a surface with a Riemannian metric and let $d(p,q)$ be the distance function defined by
$$d(p,q) = \inf\{L(c) \mid c \text{ is a piecewise smooth curve from } p \text{ to } q\}.$$
Then $d(\cdot,\cdot)$ defines a metric on $M$. Moreover, the metric topology induced by $d$ is equivalent to the topology of $M$.
