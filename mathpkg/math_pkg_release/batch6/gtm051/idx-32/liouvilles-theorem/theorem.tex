\text{Let } (U, g) \text{ be a surface with a Liouville line element. Define } \Phi: T^0 U \to \mathbb{R} \text{ by}

$$\Phi(X) := A(u(\pi X)) \cos^2 \theta(X) + B(v(\pi X)) \sin^2 \theta(X),$$

\text{where } \theta(X) \text{ is the angle between } X \text{ and the tangent to the } v\text{-parameter curve, and } T^0 U = \{X \in T U \mid X \neq 0\}.

\text{A curve } c(t) = (u(t), v(t)) \text{ with } \dot{u}(t) \neq 0 \text{ is (after possibly a reparameterization) a geodesic if and only if } \Phi(\dot{c}(t)) = \text{constant}.
