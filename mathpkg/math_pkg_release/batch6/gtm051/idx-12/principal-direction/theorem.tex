Let $f: U \to \mathbb{R}^3$ be a surface. A unit tangent vector $X_0 \in S_u^1 f$ is called a \textbf{principal direction} if $X_0$ is a critical point of the function

$$X \in S_u^1 f \mapsto \kappa(X) = II(X, X) \in \mathbb{R}.$$

If $X_0$ is a principal direction, the value $\kappa(X_0)$ is called a \textbf{principal curvature} of $f$ at $u$.

Note that if $X$ is a principal direction, so is $-X$. There are always at least two linearly independent principal directions, namely the values of $X$ where $\kappa(X)$ takes on a maximum and a minimum on the compact set $S_u^1 f$.
