Two curves $c: I \rightarrow \mathbb{R}^n$ and $\tilde{c}: \tilde{I} \rightarrow \mathbb{R}^n$ are related by a \text{parameter transformation} if there exists a diffeomorphism $\phi: \tilde{I} \rightarrow I$ such that $\tilde{c} = c \circ \phi$. This relationship is an equivalence relation on the set of all curves in $\mathbb{R}^n$. An equivalence class of curves under this relation is called an \text{unparameterized curve}.
