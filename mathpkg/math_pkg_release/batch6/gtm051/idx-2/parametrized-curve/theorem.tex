Let $I \subseteq \mathbb{R}$ be an interval. A parametrized curve in $\mathbb{R}^n$ is a $C^\infty$ mapping $c: I \rightarrow \mathbb{R}^n$. If $I$ is not an open interval, there exists an open interval $I^*$ containing $I$ and a $C^\infty$ mapping $c^*: I^* \rightarrow \mathbb{R}^n$ such that $c = c^*|_I$. The variable $t \in I$ is called the parameter of the curve.
