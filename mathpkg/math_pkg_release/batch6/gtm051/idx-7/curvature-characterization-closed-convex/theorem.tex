A periodic function $\kappa: [0, \omega] \to \mathbb{R}_{\geq 0}$ occurs as the curvature function of a closed convex plane curve $c: I \to \mathbb{R}^2$ if and only if it satisfies the necessary restrictions given by the strengthened four-vertex theorem: either $\kappa$ is a positive constant, or $\kappa$ has exactly two relative maxima and two relative minima with the maximum values strictly exceeding the minimum values.
