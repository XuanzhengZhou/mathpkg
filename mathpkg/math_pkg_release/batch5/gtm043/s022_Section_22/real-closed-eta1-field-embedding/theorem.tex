A real-closed $\eta_1$-field contains a copy of any real-closed field of cardinal $\leq \aleph_1$.

Since every ordered field $E$ can be embedded in its real-closure, and $|E| = |\mathcal{R}E|$, this yields:

A real-closed $\eta_1$-field (of any cardinal) contains a copy of each ordered field of cardinal $\leq \aleph_1$. Moreover, it contains a copy of $\mathbf{R}$ as well.
