We prove first that no countable subset $D$ of the set $E$ of infinitely small elements is cofinal. By Lemma 14.15(a), there exists $c \in C/P$ such that
$$D \leq c \leq \{1/n: n \in \mathbf{N}\}.$$
The second inequality implies $c \in E$. Therefore $2c \in E$, and we have $D \leq c < 2c$. So $D$ is not cofinal.

The $\eta_1$-property follows from Lemma 14.15(a) together with the condition on countable cofinal/coinitial subsets, verifying the defining property of $\eta_1$-sets for any subsets $A < B$ of cardinality $\leq \aleph_0$.
