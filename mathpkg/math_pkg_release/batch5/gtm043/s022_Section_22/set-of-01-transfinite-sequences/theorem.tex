Let $s$ denote the set of all $\{0, 1\}$-valued, transfinite sequences
$$x = (x_\xi)_{\xi < \omega_1}$$
ordered lexicographically.

When $x$ has an immediate successor $x^+$, we shall refer to $x$ as a \emph{lower element} of $s$, and to $x^+$ as an \emph{upper element}. The immediate successor relation is characterized by: there exists an ordinal $\tau < \omega_1$ such that
$$x_\xi = x^+_\xi \quad \text{for } \xi < \tau,$$
$$x_\tau = 0 \text{ and } x^+_\tau = 1,$$
$$x_\xi = 1 \text{ and } x^+_\xi = 0 \quad \text{for } \xi > \tau.$$

Conversely, if $x$ and $x^+$ are as just given, then no element of $s$ lies between them. No upper element is a lower element.
