Section 13.18 defines $Q$ as the set of upper elements of $s$ -- those transfinite sequences whose set of indices where the value is $1$ has a largest member. The family $Q_\alpha$ stratifies $Q$ by bounding this largest index. Key properties (a) and (b) describe the form taken by suprema and infima of subsets of $Q_\alpha$ when the extremum lies outside the subset: the extremum has a tail of zeros (for suprema) or ones (for infima) beyond $\alpha$.
