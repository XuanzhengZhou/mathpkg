The set of all upper elements of $s$ is denoted by $Q$. Thus, $Q$ is the set of all $x \in s$ such that $\{\xi : x_\xi = 1\}$ has a largest member.

For each $\alpha$, we let $Q_\alpha$ denote the set of all $x \in Q$ for which the largest index where $x_\xi = 1$ is at most $\alpha$.

Properties of suprema and infima of subsets of $Q_\alpha$:

(a) Let $x = \sup A$, where $A \subset Q_\alpha$. If $x \notin A$, then $x_\xi = 0$ for all $\xi \geq \alpha$, but $x \notin Q$.

(b) Let $y = \inf B$, where $B \subset Q_\alpha$. If $y \notin B$, then $y_\xi = 1$ for all $\xi \geq \alpha$, but $y$ is not a lower element of $s$.
