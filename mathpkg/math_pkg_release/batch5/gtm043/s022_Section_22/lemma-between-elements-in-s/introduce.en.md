Lemma 13.19 provides crucial interpolation properties within the ordered set $s$. Part (a) states that between any two elements that are not immediate successors, there is always an element of $Q$ (an upper element). Part (b) refines this: between elements of $Q_{\sigma+1}$ one can find an element of $Q_\sigma$. These interpolation results are fundamental tools in proving that $Q$ is an $\eta_1$-set and in the Dedekind completion analysis.
