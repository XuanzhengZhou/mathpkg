Given a set $A \subseteq s$, we construct its supremum $s = (s_\xi)$ inductively. Given $\sigma < \omega_1$, assume that $s_\xi$ has been defined for every $\xi < \sigma$. If there exists $a \in A$ such that $a_\xi = s_\xi$ for all $\xi < \sigma$, and if, in addition, $a_\sigma = 1$, then we define $s_\sigma = 1$; in the contrary case, we put $s_\sigma = 0$. It is straightforward to verify that $s = \sup A$.

The construction of the infimum is analogous.
