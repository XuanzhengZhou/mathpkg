Lemma 13.17 establishes that the lexicographically ordered set $s$ of all $\{0,1\}$-valued transfinite sequences of length $\omega_1$ is Dedekind-complete. The supremum of any set $A$ is constructed by transfinite induction: at each ordinal $\sigma < \omega_1$, if there exists $a \in A$ matching the partial supremum on all $\xi < \sigma$ with $a_\sigma = 1$, then $s_\sigma = 1$; otherwise $s_\sigma = 0$. Since $s$ has both endpoints, it is lattice-complete.
