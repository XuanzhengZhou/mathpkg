**Definition.** An ordered set $E$ is called an $\eta_\alpha$-set if for any subsets $A$ and $B$ of power $< \aleph_\alpha$, with $A < B$, there exists $v \in E$ satisfying $A < v < B$. For $\alpha = 0$ or $1$, this definition agrees with those given previously. An ordered field that is an $\eta_\alpha$-set is called an $\eta_\alpha$-field.

1. All $\eta_\alpha$-sets of cardinal $\aleph_\alpha$ are similar.
2. An $\eta_\alpha$-set contains a copy of every ordered set of cardinal $\leq \aleph_\alpha$.
3. For $\alpha > 0$, all real-closed $\eta_\alpha$-fields of cardinal $\aleph_\alpha$ are isomorphic. The condition $\alpha > 0$ is essential, as is shown by 13C.4.
4. For $\alpha > 0$, any real-closed $\eta_\alpha$-field contains a copy of every ordered field of cardinal $\leq \aleph_\alpha$. Again, the condition $\alpha > 0$ is essential.
