Define $g \in C(\beta X)$ as follows. First, $g$ is to map $D$ one-one onto the set $\{1/n\}_{n \in N}$. Next, $g$ is to vanish at all limit points of $D$. So far, we have a nonnegative, continuous function defined on the compact set $\operatorname{cl} D$. We choose $g$ to be any nonnegative, continuous extension of this function to all of $\beta X$. (This is the construction described in 8.21.) Since $p$ is a limit point of $D$, $g$ vanishes at $p$.

Because $p \notin vX$, there is a nonnegative function $f$ in $C(\beta X)$ that vanishes at $p$, but nowhere on $vX$. Then $Z(f + g)$ meets $\operatorname{cl} D$, but not $vX \cup D$. By Lemma 9.4, $D$ contains a copy $N$ of $N$, and the subset $Z(f + g)$ of $\beta X - vX$ contains $\beta N - N$.
