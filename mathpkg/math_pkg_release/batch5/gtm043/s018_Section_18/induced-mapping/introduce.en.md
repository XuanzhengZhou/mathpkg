Given a mapping $\varphi: A \to B$ between sets, and a codomain set $E$, every function $g: B \to E$ can be precomposed with $\varphi$ to yield a function $g \circ \varphi: A \to E$. This operation defines the induced mapping $\varphi': E^B \to E^A$, where $\varphi'g = g \circ \varphi$. The induced mapping exhibits a fundamental duality: $\varphi'$ is injective precisely when the original $\varphi$ is surjective, and $\varphi'$ is surjective precisely when $\varphi$ is injective, provided $E$ contains more than one element.
