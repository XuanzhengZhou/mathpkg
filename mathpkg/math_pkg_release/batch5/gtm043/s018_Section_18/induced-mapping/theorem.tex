Let $\varphi$ be a given mapping from a set $A$ into a set $B$.
For each mapping $g$ from $B$ into a set $E$,
the composition $g \circ \varphi$ carries $A$ into $E$.
Thus, $\varphi$ induces a mapping $\varphi' : E^B \rightarrow E^A$;
explicitly,
$$\varphi'g = g \circ \varphi.$$

There is a duality between the properties one-one and onto
(provided that $E$ has more than one element):
$\varphi'$ is one-one if and only if $\varphi$ is onto,
and $\varphi'$ is onto if and only if $\varphi$ is one-one.
