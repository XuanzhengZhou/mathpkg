Let $\tau$ be a continuous mapping from $X$ into $Y$,
and $\tau'$ the induced homomorphism $g \rightarrow g \circ \tau$
from $C(Y)$ into $C(X)$ [resp. $C^*(Y)$ into $C^*(X)$].

(a) $\tau'$ is an isomorphism (into) if and only if $\tau[X]$ is dense in $Y$.

(b) $\tau'$ is onto if and only if $\tau$ is a homeomorphism whose image
is $C$-embedded [resp. $C^*$-embedded] in $Y$.
