A \(z\)-filter \(\mathcal{F}\) on \(X\) is called \textit{fixed} if \(\bigcap \mathcal{F} \neq \varnothing\), and \textit{free} if \(\bigcap \mathcal{F} = \varnothing\).

An ideal \(I\) in \(C(X)\) is fixed if and only if \(Z[I]\) is a fixed \(z\)-filter. Consequently, every ideal in \(C(X)\) is fixed if and only if every \(z\)-filter on \(X\) is fixed.
