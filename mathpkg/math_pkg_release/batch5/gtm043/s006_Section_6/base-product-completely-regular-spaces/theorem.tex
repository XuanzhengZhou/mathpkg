When each $X_{\alpha}$ is completely regular, the collection of all finite unions
$$\pi_{\alpha_1}^{\leftarrow}[Z_1] \cup \cdots \cup \pi_{\alpha_n}^{\leftarrow}[Z_n],$$
where $Z_k$ is a zero-set in $X_{\alpha_k}$, is a base for the closed sets in $X$.

Moreover, each such union is a zero-set, because
$$\pi_{\alpha}^{\leftarrow}[Z_{X_{\alpha}}(f)] = Z_X(f \circ \pi_{\alpha}).$$
