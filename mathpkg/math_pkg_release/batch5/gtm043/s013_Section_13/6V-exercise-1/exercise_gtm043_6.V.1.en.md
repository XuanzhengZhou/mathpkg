Assume the continuum hypothesis $(\mathfrak{c} = \aleph_1)$. Then the $\mathfrak{c}$ nonempty open-and-closed sets in $\beta \mathbf{N} - \mathbf{N}$ (see 6S.2,4) can be indexed $(V_\alpha)_{\alpha < \omega_1}$. Define $W_\alpha$ inductively, for $\alpha < \omega_1$, as follows. Assume that $\bigcap_{\sigma \leq \tau} W_\sigma$ is nonempty for each $\tau < \alpha$; then, by 6S.4,7, $\bigcap_{\sigma \leq \alpha} W_\sigma$ contains a set $V_\gamma$. Put $W_\alpha = V_\gamma$ or $V_\alpha \cap V_\gamma$, according as the latter set is empty or not. If $\alpha_n < \omega_1$ for $n \in \mathbf{N}$, and $\alpha = \sup_n \alpha_n$, then $\bigcap_n W_{\alpha_n} \supset W_\alpha$.
