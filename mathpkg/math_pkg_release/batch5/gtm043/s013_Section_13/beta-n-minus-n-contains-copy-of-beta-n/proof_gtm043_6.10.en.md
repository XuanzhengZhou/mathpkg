Trivially, $D$ is a discrete subspace of $\beta \mathbf{N}$; so $D$ is homeomorphic with $\mathbf{N}$. Moreover, $D$ is $C^*$-embedded (in fact, $C$-embedded) in $T$: to extend a function $f$ on $D$ to a continuous function on $T$, simply assign the constant value $f(p_n)$ to each point of $\operatorname{cl} A_n$. It follows that $D$ is $C^*$-embedded in $\beta T$—which is $\beta \mathbf{N}$. By 6.9(a), $\operatorname{cl}_{\beta \mathbf{N}} D = \beta D$. Now, since $D$ is contained in the closed set $\beta \mathbf{N} - \mathbf{N}$, so is $\operatorname{cl} D$; thus, $\beta \mathbf{N} - \mathbf{N} \supset \beta D$. But $\beta D$ is homeomorphic with $\beta \mathbf{N}$. We have proved:

$\beta \mathbf{N} - \mathbf{N}$ contains a copy of $\beta \mathbf{N}$.
