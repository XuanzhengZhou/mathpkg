If $f^*(p) = \infty$, then for each $n \in \mathbf{N}$, $p$ is in the closure of the set

$$Z_n = \{x \in X : |f(x)| \geq n\}.$$

By the Gelfand-Kolmogoroff theorem, $Z_n \in Z[M^p]$. By 5.4(a), $|M^p(f)| \geq n$. Therefore $|M^p(f)|$ is infinitely large. (Cf. 5.7(a).)

Similarly, if $f^*(p) = r$, then $|M^p(f) - r| \leq 1/n$ for each $n$, so that $|M^p(f) - r|$ is infinitely small or zero.
