The mapping $p \mapsto A^p$ is one-one from $\beta X$ onto the set of all $z$-ultrafilters on $X$. But the latter set is in one-one correspondence with the set of all maximal ideals in $C(X)$ (Theorem 2.5). Therefore, corresponding to each point $p$ of $\beta X$, there is a maximal ideal $M^p$ in $C(X)$, determined by

$$Z[M^p] = A^p,$$

and the correspondence $p \mapsto M^p$ is one-one.

Since the maximal ideal $M^p$ is a $z$-ideal, $f \in M^p$ if and only if $Z(f) \in A^p$, hence if and only if $p \in \operatorname{cl}_{\beta X} Z(f)$ (6.5(c)).

Again, $M^p$ is free or fixed according as $p \in \beta X - X$ or $p \in X$; and, in the latter case, $M^p = M_p$.
