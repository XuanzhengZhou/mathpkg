Let $X$ be a compact space. For any family $\mathcal{E}$ of subsets of $X$, the set of all functions in $C(X)$ that are constant on each member of $\mathcal{E}$ is an analytic subring. Conversely, every analytic subring $A$ of $C(X)$ is of this form: $A$ consists of all functions constant on each maximal stationary set of $A$.
