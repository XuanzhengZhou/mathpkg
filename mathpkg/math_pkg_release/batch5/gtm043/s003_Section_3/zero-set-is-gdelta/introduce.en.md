The formula $Z(f) = \bigcap_n \{x : |f(x)| < 1/n\}$ expresses every zero-set as a countable intersection of open sets, showing that every zero-set belongs to the class $G_\delta$ of the Borel hierarchy. The converse holds in normal spaces: every closed $G_\delta$ is a zero-set. In metric spaces the situation is even simpler -- every closed set is a zero-set, being precisely the set of points whose distance from it is zero.
