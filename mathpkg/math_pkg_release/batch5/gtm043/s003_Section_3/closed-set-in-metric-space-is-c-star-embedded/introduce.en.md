Tietze's Extension Theorem states that every bounded continuous real-valued function on a closed subset of a metric space extends continuously to the whole space, i.e., every closed set in a metric space is $C^*$-embedded. This follows from Urysohn's Extension Theorem because in a metric space, every closed set is a zero-set, so disjoint closed sets are completely separated, and closed sets in a closed subspace remain closed in the ambient space.
