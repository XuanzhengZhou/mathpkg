A $C^*$-embedded subspace need not be $C$-embedded. Examples include:
\begin{itemize}
\item $\mathbf{N}$ as a dense, $C^*$-embedded subset of the space $\Sigma$ (4M) that is not $C$-embedded.
\item $\mathbf{N}$ as a closed, $C^*$-embedded subset of the pseudocompact space $\Lambda$ (6P); since $\Lambda$ is pseudocompact, no unbounded function on $\mathbf{N}$ can be extended continuously to $\Lambda$.
\end{itemize}
