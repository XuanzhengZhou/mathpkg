If $f$ is a unit of $C^*(X)$, then $f^{-1} \in C^*(X)$ exists, so $ff^{-1} = 1$, implying $Z(f) = \emptyset$. For the converse, consider the identity function $i$ on $\mathbb{N}$, defined by $i(n) = n$. Then $i \in C(\mathbb{N})$ is unbounded, so $C(\mathbb{N}) \neq C^*(\mathbb{N})$. The function $j = i^{-1}$, defined by $j(n) = 1/n$, belongs to $C^*(\mathbb{N})$ (it is bounded by 1), and $Z(j) = \emptyset$. However, $j^{-1} = i \notin C^*(\mathbb{N})$, so $j$ is a bounded function with empty zero-set that is not a unit of $C^*(\mathbb{N})$.
