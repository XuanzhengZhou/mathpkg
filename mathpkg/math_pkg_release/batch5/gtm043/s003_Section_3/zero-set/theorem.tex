For $f \in C(X)$, the \emph{zero-set} of $f$ is
$$Z(f) = f^{\leftarrow}(0) = \{x \in X : f(x) = 0\}.$$

More generally, for any real number $r$,
$$f^{\leftarrow}(r) = \{x \in X : f(x) = r\},$$
and $\{x : f(x) = r\} = \{x : (f - s)(x) = r - s\}$ for any real $s$, so the family of all such sets can be obtained by fixing $r = 0$.
