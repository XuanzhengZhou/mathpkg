A subspace $S$ of $X$ is \emph{$C^*$-embedded} in $X$ if every function in $C^*(S)$ can be extended to a function in $C(X)$ (equivalently, to a function in $C^*(X)$).

If $S \subset X \subset Y$ and $X$ is $C^*$-embedded in $Y$, then $S$ is $C^*$-embedded in $Y$ if and only if it is $C^*$-embedded in $X$.
