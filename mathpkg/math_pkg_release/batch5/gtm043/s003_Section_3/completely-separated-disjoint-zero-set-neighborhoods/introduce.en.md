When two sets $A$ and $A'$ are completely separated, they not only lie in disjoint zero-sets but actually possess disjoint zero-set-neighborhoods. These are constructed from the separating function $f$ as the sublevel set $\{x : f(x) \leq 1/3\}$ and superlevel set $\{x : f(x) \geq 2/3\}$. This strengthening is useful for extension problems, particularly in the proof of Urysohn's Extension Theorem.
