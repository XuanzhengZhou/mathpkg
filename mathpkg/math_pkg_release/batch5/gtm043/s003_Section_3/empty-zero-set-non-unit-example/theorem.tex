On $\mathbb{N}$, let $i(n) = n$ be the identity function, and let $j(n) = 1/n$. Then:
$$i \in C(\mathbb{N}), \quad j = i^{-1} \in C^*(\mathbb{N}), \quad Z(j) = \emptyset,$$
but $i = j^{-1} \notin C^*(\mathbb{N})$, so $j$ is a function in $C^*(\mathbb{N})$ with empty zero-set that is not a unit of $C^*(\mathbb{N})$.
