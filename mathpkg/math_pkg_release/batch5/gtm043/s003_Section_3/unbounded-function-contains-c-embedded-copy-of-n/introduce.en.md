If a continuous function $h$ on $X$ is unbounded on a subset $E$, then $E$ contains a copy of the natural numbers $\mathbf{N}$ that is $C$-embedded in $X$, and on this copy $h$ approaches infinity. This structural result shows that unboundedness of a single continuous function forces the presence of a well-behaved discrete subspace that witnesses the unboundedness.
