A space $X$ is pseudocompact (every continuous real-valued function on $X$ is bounded) exactly when $X$ contains no $C$-embedded copy of the natural numbers $\mathbf{N}$. This follows directly from Corollary 1.20: if $X$ were not pseudocompact, there would exist an unbounded function $h$, and then $X$ itself would contain a $C$-embedded copy of $\mathbf{N}$ on which $h \to \infty$.
