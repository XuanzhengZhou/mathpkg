A zero-set-neighborhood of a set $A$ is a zero-set that contains $A$ in its interior. This notion refines the concept of a neighborhood by requiring the neighborhood itself to be a zero-set, which is important for extension and separation arguments. For instance, the sets $\{x : f(x) \leq 1/3\}$ from a separating function $f$ are zero-set-neighborhoods.
