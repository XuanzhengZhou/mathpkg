\textbf{Theorem 2.5.}

(a) If $M$ is a maximal ideal in $C(X)$, then $Z[M]$ is a $z$-ultrafilter on $X$.

(b) If $\mathcal{A}$ is a $z$-ultrafilter on $X$, then $Z^{\leftarrow}[\mathcal{A}]$ is a maximal ideal in $C(X)$.

The mapping $Z$ is one-one from the set of all maximal ideals in $C$ onto the set of all $z$-ultrafilters.
