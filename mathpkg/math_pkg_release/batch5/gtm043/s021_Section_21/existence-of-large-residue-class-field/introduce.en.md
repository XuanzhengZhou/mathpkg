For a discrete space $X$ of infinite cardinal $\mathfrak{m}$, there exists a maximal ideal $M$ in $C(X)$ whose residue class field has cardinality strictly exceeding $\mathfrak{m}$. The proof constructs a function dominating any given family of size $\mathfrak{m}$, establishing that no set of that size is cofinal. Since $|C/M| \leq 2^{\mathfrak{m}}$ always holds, the cardinal is determined when no intermediate cardinals exist.
