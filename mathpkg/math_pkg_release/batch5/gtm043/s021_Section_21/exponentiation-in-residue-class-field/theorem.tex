Let $u > 0$ be an element of $C(X)/M$, and let $r > 0$ be any real number. Choose $f \in C(X)$, $f \geq 0$, such that $M(f) = u$. Define $u^r = M(f^r)$ where $f^r(x) = (f(x))^r$.

This definition is independent of the choice of representative $f$. The laws of exponents hold: $u^r u^s = u^{r+s}$, $(u^r)^s = u^{rs}$. If $u < v$ then $u^r < v^r$. If $u$ is infinitely large, so is $u^r$ for every $r > 0$.
