An algebraic extension is one where every element is algebraic over the base field. For infinite base fields, algebraic extensions do not increase cardinality, since the number of polynomials is bounded by $|E|$ and each has finitely many roots. This fact, with the Artin-Schreier theorem, shows the real-closure of an infinite ordered field has the same cardinality.
