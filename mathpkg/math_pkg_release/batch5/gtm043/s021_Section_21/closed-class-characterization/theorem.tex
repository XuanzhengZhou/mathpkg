A nonempty class $\mathfrak{C}$ is closed if and only if, whenever $\mathfrak{m} \in \mathfrak{C}$,
\begin{enumerate}
\item[(i)] $\mathfrak{n} \in \mathfrak{C}$ for all $\mathfrak{n} < \mathfrak{m}$;
\item[(ii)] the sum of any $\mathfrak{m}$ members of $\mathfrak{C}$ is in $\mathfrak{C}$;
\item[(iii)] $2^{\mathfrak{m}} \in \mathfrak{C}$.
\end{enumerate}
