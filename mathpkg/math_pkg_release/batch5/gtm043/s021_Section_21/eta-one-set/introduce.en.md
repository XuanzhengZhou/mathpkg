An $\eta_1$-set is an ordered set satisfying $\aleph_1$-saturation: between any countable left set and countable right set there exists an element. Examples include saturated nonstandard models. Under CH, all $\eta_1$-sets of cardinality $\aleph_1$ are order-isomorphic. The concept originates with Hausdorff's classification of order types ($\eta_\alpha$-sets).
