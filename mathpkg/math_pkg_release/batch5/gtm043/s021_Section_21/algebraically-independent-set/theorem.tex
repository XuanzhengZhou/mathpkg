A set $T$ is said to be \emph{(algebraically) independent} over a field $E$ if $\mathfrak{p}(x_1, \dots, x_n) \neq 0$ whenever $n \in \mathbf{N}$, $\mathfrak{p}$ is a nonzero polynomial over $E$ in $n$ indeterminates, and $x_1, \dots, x_n$ are distinct elements of $T$.

Equivalently, $T$ is independent if each $x \in T$ is transcendental over $E(T - \{x\})$.
