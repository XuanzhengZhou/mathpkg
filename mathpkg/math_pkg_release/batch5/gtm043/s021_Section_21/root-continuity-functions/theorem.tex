Fix $n \in \mathbf{N}$. For each $a = (a_1, \dots, a_n) \in \mathbf{R}^n$, let $\rho_1 a, \dots, \rho_n a$ denote the real parts of the (complex) zeros of
$$\mathfrak{p}_a(\lambda) = \lambda^n + a_1 \lambda^{n-1} + \cdots + a_n,$$
indexed so that $\rho_1 a \leq \cdots \leq \rho_n a$. Each $\rho_k : \mathbf{R}^n \to \mathbf{R}$ is continuous.
