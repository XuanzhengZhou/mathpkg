This lemma states that if a set $E$ can be separated by disjoint neighborhoods from each point of a compact set $F$ individually, then $E$ can be separated from the entire compact set $F$ by disjoint neighborhoods simultaneously. It is a standard compactness argument: finitely many of the separating neighborhoods cover $F$, and the intersection of the corresponding neighborhoods of $E$ provides the required separation. The lemma underlies many separation results in Hausdorff spaces involving compact sets.
