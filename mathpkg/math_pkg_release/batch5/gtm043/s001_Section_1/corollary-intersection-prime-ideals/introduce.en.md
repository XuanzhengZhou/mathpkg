This corollary identifies the intersection of all prime ideals containing a given ideal $I$ with the radical (or nilradical) of $I$ — the set of elements some power of which lies in $I$. In particular, when $I = (0)$, this says the intersection of all prime ideals is the set of nilpotent elements (the nilradical). This is one of the fundamental characterizations linking prime ideals to nilpotence.
