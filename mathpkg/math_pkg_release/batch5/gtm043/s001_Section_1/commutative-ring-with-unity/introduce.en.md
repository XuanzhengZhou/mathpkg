A commutative ring with unity is a ring in which multiplication is commutative and which possesses a multiplicative identity element $1$ satisfying $1 \cdot a = a$ for all $a$. Throughout this text, all rings are assumed to be commutative with unity unless otherwise noted. This assumption simplifies the ideal theory: an ideal $I$ is proper if and only if it does not contain $1$, and the quotient $A/I$ is again a commutative ring with unity.
