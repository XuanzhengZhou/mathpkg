In this text, a neighborhood of a set $E$ is defined as any set whose interior contains $E$. This is a broader definition than the common "open set containing $E$" — neighborhoods need not be open themselves. This convention facilitates arguments involving disjoint neighborhoods of sets and compact sets, which are fundamental to the separation properties of Hausdorff spaces.
