A maximal element of a partially ordered set is an element that is not strictly less than any other element. Crucially, a maximal element need not be comparable with all elements; it is merely that nothing in the set strictly dominates it. This is distinct from a largest element, which must dominate every element in the set. Maximal elements play a central role in applications of Zorn's Lemma and in the definition of maximal ideals in ring theory.
