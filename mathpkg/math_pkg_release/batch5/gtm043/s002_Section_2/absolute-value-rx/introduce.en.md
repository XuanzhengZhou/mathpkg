The absolute value $|f|$ of a function in $\mathbf{R}^X$ is defined intrinsically using the lattice structure: $|f| = f \vee (-f)$, the join of $f$ and its additive inverse. At each point $x$, this yields the ordinary absolute value $|f(x)|$ of the real number $f(x)$. This construction does not reference the topology and is therefore purely algebraic/lattice-theoretic.
