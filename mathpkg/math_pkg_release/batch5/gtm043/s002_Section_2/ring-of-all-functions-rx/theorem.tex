The collection $\mathbf{R}^X$ of all functions from a set $X$ into the real numbers $\mathbf{R}$ forms a commutative ring with unity when equipped with pointwise addition and multiplication. The zero element is the constant function $0$, and the unity element is the constant function $1$.

$$\mathbf{R}^X = \{f \mid f: X \to \mathbf{R}\}$$
