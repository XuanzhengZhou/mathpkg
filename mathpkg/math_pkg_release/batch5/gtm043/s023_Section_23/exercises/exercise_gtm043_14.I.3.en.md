Let $\mathcal{V}$ be a free ultrafilter on the set $\{1/n: n \in \mathbf{N}\}$. The set $P$ of all $f \in C(\mathbf{R})$ for which $Z(f)$ contains a member of $\mathcal{V}$ is a prime $z$-ideal. It is maximal in the set of all $z$-ideals contained properly in $M_0$.
