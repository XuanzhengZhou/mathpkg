A uniform structure $\mathcal{D}$ on $X$ induces a topology on $X$, called the uniform topology, defined as follows: a subbase for the neighborhood system at $x$ consists of the sets

$$\{y : d(x, y) < \epsilon\}$$

$(d \in \mathcal{D}, \epsilon > 0)$. Closures in the uniform topology are given by:

$$\operatorname{cl} A = \bigcap_{d \in \mathcal{D}} \{x : d(x, A) = 0\}.$$

The uniform topology is a Hausdorff topology if and only if $\mathcal{D}$ is a Hausdorff structure.
