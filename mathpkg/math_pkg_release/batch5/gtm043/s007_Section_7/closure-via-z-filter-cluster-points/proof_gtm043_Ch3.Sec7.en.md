Let $\mathcal{F}$ be the $z$-filter of all zero-sets containing $S$. If $p$ is a cluster point of $\mathcal{F}$, then by definition $p$ belongs to the closure of every member of $\mathcal{F}$. Since each zero-set containing $S$ is a member of $\mathcal{F}$ and is closed, $p$ belongs to every zero-set containing $S$. Because the zero-sets form a base for the closed sets in a completely regular space, every closed set containing $S$ is the intersection of zero-sets containing $S$. Hence $p$ belongs to every closed set containing $S$, which is precisely the condition that $p \in \operatorname{cl} S$. Conversely, if $p \in \operatorname{cl} S$, then $p$ belongs to every closed set containing $S$, in particular to every zero-set containing $S$, and hence to the closure of every member of $\mathcal{F}$. Thus $\operatorname{cl} S$ is exactly the set of all cluster points of $\mathcal{F}$.
