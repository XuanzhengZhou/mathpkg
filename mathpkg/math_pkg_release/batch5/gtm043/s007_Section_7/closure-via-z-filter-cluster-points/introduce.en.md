This proposition characterizes the closure of a subset in a completely regular space in terms of $z$-filters. Given a nonempty set $S$, the collection of all zero-sets that contain $S$ forms a $z$-filter, and the closure of $S$ coincides precisely with the set of all cluster points of this $z$-filter. This follows from the fact that zero-sets form a base for the closed sets in any completely regular space.
