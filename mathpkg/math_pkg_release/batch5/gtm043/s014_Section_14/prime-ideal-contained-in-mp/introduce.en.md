This theorem establishes that any prime ideal $P$ contained in a maximal ideal $M^p$ yields a residue class field $C/P$ whose order structure mirrors that of $C/M^p$. Specifically, a function's image in $C/P$ is infinitely large (resp. infinitely close to a real number $r$) precisely when its image in $C/M^p$ has the same property. It follows that $C/P$ contains infinitely large elements if and only if $C/M^p$ does, i.e., if and only if $M^p$ is a hyper-real ideal.
