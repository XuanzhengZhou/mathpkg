The event "$X_n$ converges to 0" can be expressed purely in terms of countable set operations on the events $[|X_k| \geq 1/m]$. This formulation is essential for proving almost sure convergence results using probability estimates, as it translates a pointwise analytic condition into a measurable set whose probability can be bounded.
