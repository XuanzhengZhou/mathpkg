For a sequence $A_n$ of sets:

The \textit{inferior limit} is the set of points belonging to almost all $A_n$:
$$\liminf A_n = \bigcup_{n=1}^{\infty} \bigcap_{k=n}^{\infty} A_k.$$

The \textit{superior limit} is the set of points belonging to infinitely many $A_n$:
$$\limsup A_n = \bigcap_{n=1}^{\infty} \bigcup_{k=n}^{\infty} A_k.$$

Always $\liminf A_n \subset \limsup A_n$. If equality holds, the common set is called the \textit{limit} $\lim A_n$ and $A_n$ is said to converge.
