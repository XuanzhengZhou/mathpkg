Probability theory and measure theory correspond as follows:

\begin{itemize}
\item probability space (pr. space) $\leftrightarrow$ normed measure space
\item elementary event $\leftrightarrow$ point
\item event $\leftrightarrow$ measurable set
\item sure event $\leftrightarrow$ whole space
\item impossible event $\leftrightarrow$ empty set
\item probability (pr.) $\leftrightarrow$ normed measure
\item almost sure (a.s.) $\leftrightarrow$ almost everywhere (a.e.)
\item random variable (r.v.) $\leftrightarrow$ finite numerical measurable function
\item expectation $E$ $\leftrightarrow$ integral $\int$
\end{itemize}
