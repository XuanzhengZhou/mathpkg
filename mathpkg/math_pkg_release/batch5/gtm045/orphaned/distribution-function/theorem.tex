The \textit{distribution function} (d.f.) $F_X$ of a r.v. $X$ is defined by

$$F_X(x) = P_X(-\infty, x) = P[X < x], \quad x \in \mathbb{R}.$$

The d.f. determines the distribution via $P_X[a, b) = F_X(b) - F_X(a)$.

For any finite Borel function $g$, $E g(X) = \int_{\mathbb{R}} g \, dF_X$ in the Lebesgue-Stieltjes sense.
