The distribution function (CDF) translates the set function $P_X$ into a point function $F_X(x) = P[X < x]$, making probability distributions accessible to classical analysis. The Lebesgue-Stieltjes integral with respect to $dF_X$ computes expectations. The d.f. is nondecreasing, left-continuous, with limits 0 at $-\infty$ and 1 at $+\infty$.
