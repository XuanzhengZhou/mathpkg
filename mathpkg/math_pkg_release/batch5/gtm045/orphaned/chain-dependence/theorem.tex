Events $A_j$, where $j$ runs over an ordered countable set, are said to be \textit{chained} if the probability of every $A_j$ given any finite set of the preceding ones depends only upon the last given one. If the distinct values of elementary random variables $X_n$ correspond to chained partitions, then the $X_n$ are said to be \textit{chained} or to form a \textit{chain}.
