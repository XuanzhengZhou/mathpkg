Carathéodory's measurability criterion (the splitting condition) defines which sets are measurable with respect to an outer measure. The remarkable fact is that the collection of sets satisfying this condition always forms a $\sigma$-field on which the outer measure becomes countably additive. This is the key step in the Carathéodory extension theorem.
