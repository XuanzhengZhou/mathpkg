An outer measure is a countably subadditive set function defined on all subsets of a space. It serves as an intermediate construction in the Carathéodory extension procedure: starting from a measure on a field, one constructs an outer measure on all sets, then restricts to the $\sigma$-field of measurable sets to obtain a countably additive extension.
