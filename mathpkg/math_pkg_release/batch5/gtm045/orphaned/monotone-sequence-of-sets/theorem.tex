A sequence $A_n$ of sets is \textit{monotone} if it is either:
\begin{itemize}
\item \textit{nondecreasing}: $A_1 \subset A_2 \subset \cdots$, denoted $A_n \uparrow$;
\item \textit{nonincreasing}: $A_1 \supset A_2 \supset \cdots$, denoted $A_n \downarrow$.
\end{itemize}

Monotone sequences always converge: if $A_n \uparrow$, then $\lim A_n = \bigcup A_n$; if $A_n \downarrow$, then $\lim A_n = \bigcap A_n$.
