The minimal field (or $\sigma$-field) generated by a class $\mathfrak{C}$ is obtained by intersecting all fields (or $\sigma$-fields) containing $\mathfrak{C}$. This constructive definition is crucial for building the Borel $\sigma$-field from intervals, and for the extension of measures from a generating class to the full $\sigma$-field.
