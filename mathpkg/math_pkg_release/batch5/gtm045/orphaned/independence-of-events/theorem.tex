Two events $A_1, A_2$ are said to be \textit{stochastically independent} or, simply, \textit{independent} if

$$PA_1A_2 = PA_1 \cdot PA_2.$$

More generally, events $A_k, k = 1, 2, \cdots, n$ are \textit{independent} if, for every $m \leq n$ and for arbitrary distinct integers $k_1, k_2, \cdots, k_m \leq n$,

$$PA_{k_1}A_{k_2} \cdots A_{k_m} = PA_{k_1} PA_{k_2} \cdots PA_{k_m}.$$
