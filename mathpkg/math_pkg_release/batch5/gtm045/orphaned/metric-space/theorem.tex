A \textit{metric space} $(\Omega, d)$ is a space $\Omega$ with a distance function $d: \Omega \times \Omega \to [0, \infty)$ satisfying:
\begin{itemize}
\item $d(\omega, \omega') = 0 \iff \omega = \omega'$
\item $d(\omega, \omega') = d(\omega', \omega)$
\item $d(\omega, \omega'') \leq d(\omega, \omega') + d(\omega', \omega'')$ (triangle inequality)
\end{itemize}
