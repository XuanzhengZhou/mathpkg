Let $\{A_t, t \in T\}$ be a collection of sets in spaces $\Omega_t$. The \textit{product set} $A_T = \prod_{t \in T} A_t$ is the set of all elements $\omega_T = (\omega_t, t \in T)$ such that $\omega_t \in A_t$ for every $t$.

A \textit{cylinder} in $\Omega_T$ with base $A_{T_N}$ (where $T_N$ is a finite index subset) is the set $A_{T_N} \times \Omega_{T \setminus T_N}$. If the base is itself a product set, it is a \textit{product cylinder} or \textit{interval}.
