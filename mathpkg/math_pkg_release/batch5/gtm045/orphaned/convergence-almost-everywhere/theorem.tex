A sequence $X_n$ of measurable functions \textit{converges almost everywhere} (a.e.) to $X$ if $X_n(\omega) \to X(\omega)$ for all $\omega$ except those in a null set.

\textbf{Egorov's theorem:} If $\mu(\Omega) < \infty$ and $X_n \to X$ a.e., then for every $\epsilon > 0$ there exists a set $A$ with $\mu(A^c) < \epsilon$ such that $X_n \to X$ uniformly on $A$.
