Let $X_1, X_2, \cdots, X_n$ be independent random variables with $E X_k = 0$ and $\sigma^2 X_k < \infty$. Set $S_k = \sum_{j=1}^{k} X_j$. Then for every $\epsilon > 0$,

$$P\left[\max_{1 \leq k \leq n} |S_k| \geq \epsilon\right] \leq \frac{1}{\epsilon^2} \sum_{k=1}^{n} \sigma^2 X_k = \frac{\sigma^2 S_n}{\epsilon^2}.$$
