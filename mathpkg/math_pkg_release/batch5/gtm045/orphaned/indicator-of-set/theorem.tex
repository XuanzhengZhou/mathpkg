The \textit{indicator} $I_A$ of a set $A$ is defined by $I_A(\omega) = 1$ if $\omega \in A$ and $I_A(\omega) = 0$ if $\omega \notin A$.

Key correspondences:
$$I_{A \cap B} = I_A I_B, \quad I_{A+B} = I_A + I_B \text{ (for disjoint sets)},$$
$$I_{\bigcap A_n} = \prod I_{A_n}, \quad I_{\sum A_n} = \sum I_{A_n},$$
$$I_{\liminf A_n} = \liminf I_{A_n}, \quad I_{\limsup A_n} = \limsup I_{A_n}.$$
