For a nonnegative measurable function $X$, the \textit{integral} (or \textit{expectation}) is defined by

$$EX = \int X \, dP = \sup \{ EY : 0 \leq Y \leq X, Y \text{ elementary} \}.$$

For general measurable $X = X^+ - X^-$, $EX = EX^+ - EX^-$ provided at least one of $EX^+, EX^-$ is finite. $X$ is \textit{integrable} if $E|X| < \infty$.
