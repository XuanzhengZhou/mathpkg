The indicator function $I_A$ of an event $A$ is the simplest random variable: it takes value 1 when $A$ occurs and 0 otherwise. Indicators bridge set operations with algebraic operations: intersections become products, disjoint unions become sums, and complements become subtraction from 1. Every simple random variable is a linear combination of indicators of disjoint events.
