A distribution $P$ is \textit{singular} (with respect to Lebesgue measure) if it is concentrated on a set of Lebesgue measure zero: there exists a Borel set $N$ with Lebesgue measure $\lambda(N) = 0$ and $P(N^c) = 0$.

$P$ is \textit{absolutely continuous} if $P(A) = 0$ whenever $\lambda(A) = 0$. By the Radon-Nikodym theorem, this is equivalent to the existence of a density $f \geq 0$ with $\int f \, d\lambda = 1$ such that $P(A) = \int_A f \, d\lambda$.
