$\operatorname{CP}(X, P_0)$ is the set of all continuous loops in $X$ that start and end at the distinguished base point $P_0$. This set serves as the raw material for constructing the fundamental group $\pi(X, P_0)$: elements of the fundamental group are equivalence classes of these closed paths under a suitable homotopy relation.
