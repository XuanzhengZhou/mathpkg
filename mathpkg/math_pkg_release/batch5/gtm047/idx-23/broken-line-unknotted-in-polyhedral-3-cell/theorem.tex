Let $B$ be a broken line in $\mathbf{R}^3$, with endpoints $P$ and $Q$. Then there is a polyhedral 3-cell $C$ such that:
\begin{enumerate}
\item $\text{Int } B \subset \text{Int } C$,
\item $P, Q \in \text{Bd } C$,
\item there is a PLH $\phi: C \leftrightarrow \sigma^2 \times [0, 1]$ such that $B \leftrightarrow R \times [0, 1]$, for some $R \in \text{Int } \sigma^2$.
\end{enumerate}
