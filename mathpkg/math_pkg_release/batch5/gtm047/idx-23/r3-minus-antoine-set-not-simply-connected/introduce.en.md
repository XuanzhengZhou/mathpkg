This theorem establishes the fundamental topological property of Antoine's necklace: the complement of the Antoine set $\mathcal{Q}$ in $\mathbf{R}^3$ is not simply connected. This is the key result distinguishing the Antoine set from a tame Cantor set, whose complement in $\mathbf{R}^3$ would be simply connected. The proof follows immediately from the nontriviality of the fundamental group of $\mathbf{R}^3 - T_1$ and the path contraction theorem.
