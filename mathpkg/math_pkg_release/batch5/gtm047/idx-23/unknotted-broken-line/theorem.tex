Let $B$ be a broken line in $\mathbf{R}^3$ with endpoints $P$ and $Q$, and let $C$ be a polyhedral 3-cell. We say that $B$ is **unknotted in** $C$ if:
\begin{enumerate}
\item $\text{Int } B \subset \text{Int } C$,
\item $P, Q \in \text{Bd } C$, and
\item there is a PLH $\phi: C \leftrightarrow \sigma^2 \times [0, 1]$ such that $\phi(B) = R \times [0, 1]$ for some $R \in \text{Int } \sigma^2$.
\end{enumerate}
