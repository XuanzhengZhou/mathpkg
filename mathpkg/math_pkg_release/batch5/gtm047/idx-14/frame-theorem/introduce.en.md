The Frame Theorem provides a polyhedral approximation result in the plane: any compact set $M$ contained in an open set $U \subset \mathbf{R}^2$ can be sandwiched inside a compact polyhedral set $P$ that still lies within $U$. This is a fundamental technical tool for extending tameness results beyond $1$-spheres to more general compact sets. The theorem is stated here to set up further preliminaries; its proof uses the metric and topological properties of the plane.
