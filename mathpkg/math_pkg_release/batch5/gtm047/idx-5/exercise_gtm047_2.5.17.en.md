Let $M = |K|$ be a connected linear graph. If $M$ contains no polygon, then $M$ is a tree. A set homeomorphic to such a $|M|$ is called a topological tree. If $M = |K|$ is a finite tree (that is, if $M$ is compact and $K$ is finite), then there is a polyhedron $N$ in $\mathbf{R}^2$ such that $M$ and $N$ are homeomorphic.
