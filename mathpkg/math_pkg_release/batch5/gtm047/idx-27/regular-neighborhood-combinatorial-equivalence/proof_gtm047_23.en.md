For each $\sigma^2 \in \partial L$, the set $C^3 = \operatorname{Cl} \left[ N'(\sigma^2) - L \right]$ is a combinatorial 3-cell, intersecting $|L| \cup N(L^1)$ in a 2-cell. By repeated applications of Theorem 10, there is a PLH $h_1: N(L) \leftrightarrow |L| \cup N(L^1)$. Similarly, there is a PLH $h_2: |L| \cup N(L^1) \leftrightarrow |L| \cup N(L^0)$, and there is a PLH $h_3: |L| \cup N(L^0) \leftrightarrow |L|$. Let $h = h_3 h_2 h_1: N(L) \leftrightarrow |L|$.
