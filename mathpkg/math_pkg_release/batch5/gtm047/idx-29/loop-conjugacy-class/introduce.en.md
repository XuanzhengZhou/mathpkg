Given a loop $L: S^1 \to X$ in a pathwise connected space $X$ and a base point $P_0 \in X$, one can associate a conjugacy class $L(X) = L(X, P_0)$ in the fundamental group $\pi(X, P_0)$. This is constructed by choosing a parametrization of $S^1$ to obtain a closed path, then conjugating by a path from $P_0$ to the image of the base point of $S^1$. Although the resulting element of $\pi(X, P_0)$ depends on these choices, its conjugacy class depends only on the loop $L$. This notion is essential for formulating the loop theorem, where the condition $L(B) \cap N = \emptyset$ means that no element of the conjugacy class lies in the normal subgroup $N$.
