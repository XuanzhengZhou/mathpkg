**Theorem 6 (The frame theorem).** Let $M$ be a compact set in $\mathbb{R}^2$, and let $U$ be an open set containing $M$. Then there is a compact polyhedral 2-manifold $N$ with boundary such that $M \subset \operatorname{Int} N$ and $N \subset U$. Moreover, $N$ can be chosen so that it has the same number of components of $\operatorname{Bd} N$ as the number of components of $M$ that separate $\mathbb{R}^2$.
