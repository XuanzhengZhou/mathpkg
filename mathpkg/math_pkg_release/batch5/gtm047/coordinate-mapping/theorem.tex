A coordinate mapping into $X$ is a linear homeomorphism $h: \sigma \rightarrow X$, where $\sigma$ is a Euclidean simplex. Two coordinate mappings $g$ and $h$ are equivalent ($g \sim h$) if $h^{-1}g$ is a simplicial homeomorphism $\sigma_g \leftrightarrow \sigma_h$. The equivalence class $[h]$ is called a PL simplex.
