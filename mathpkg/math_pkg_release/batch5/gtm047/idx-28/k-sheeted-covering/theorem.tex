Let $g: \tilde{M} \rightarrow M$ be a covering. For each $P \in M$, consider the set $g^{-1}(P) \subset \tilde{M}$. Since $M$ is connected, the number of points in $g^{-1}(P)$ depends only on $g$, and is independent of the choice of $P$. If each set $g^{-1}(P)$ is finite, with $k$ elements, then $g$ is called a \emph{$k$-sheeted covering}, or a \emph{$k$-fold covering}.
