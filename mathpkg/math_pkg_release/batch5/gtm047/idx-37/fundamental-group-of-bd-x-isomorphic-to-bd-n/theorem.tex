There is a PL homeomorphism
$$\phi: \operatorname{Bd} N \times (0, 1) \leftrightarrow \operatorname{Int} N' - K'.$$
Let
$$X_1 = X - K', \quad X_2 = \operatorname{Int} N' - \operatorname{Int} X_1,$$
so that
$$X_1 \cup X_2 = \operatorname{Int} N' - K', \quad X_1 \cap X_2 = \operatorname{Bd} X.$$
Then for any base point $P_0$ in $\operatorname{Bd} X$, every PL closed path in $\operatorname{Int} N' - K'$ with base point $P_0$ is equivalent in $\pi(\operatorname{Int} N' - K', P_0)$ to a PL closed path in $\operatorname{Bd} X$. Consequently,
$$\pi(\operatorname{Bd} X) \approx \pi(\operatorname{Bd} N).$$
