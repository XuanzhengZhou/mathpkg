Let $(M, g)$ be a connected Lorentzian manifold and let
$$\mathcal{T} \coloneqq \{ (x, X) \in TM \mid X \text{ is timelike} \}$$
be the set of all timelike tangent vectors. Let $\psi: \mathcal{T} \to \mathcal{T}$ be the antipodal map $\psi(x, X) = (x, -X)$. Then $\mathcal{T} = \mathcal{A} \cup \psi(\mathcal{A})$ where $\mathcal{A}$ is a connected component of $\mathcal{T}$. Moreover, either $\mathcal{A} \cap \psi(\mathcal{A}) = \varnothing$ (so $\mathcal{T}$ has exactly two connected components) or $\mathcal{A} = \psi(\mathcal{A})$ (so $\mathcal{T}$ is connected).
