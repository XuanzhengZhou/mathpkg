This follows directly from Lemma 1.1.3(a) applied to the 1-dimensional subspace $W = \operatorname{span}\{w\}$. If $w$ is timelike, then $\operatorname{span}\{w\}$ is a timelike subspace, so by Lemma 1.1.3(a) its orthogonal complement $(\operatorname{span}\{w\})^\perp = w^\perp$ is spacelike. Conversely, if $w^\perp$ is spacelike, then Lemma 1.1.3(a) implies $(\operatorname{span}\{w\})^{\perp\perp} = \operatorname{span}\{w\}$ is timelike, so $w$ is timelike.
