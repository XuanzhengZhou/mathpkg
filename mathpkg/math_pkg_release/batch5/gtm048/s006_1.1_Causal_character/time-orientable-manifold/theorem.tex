Let $(M, g)$ be a Lorentzian manifold. Then $(M, g)$ is called time orientable if the set
$$\mathcal{T} \coloneqq \{ (x, X) \in TM \mid X \text{ is timelike} \}$$
of all timelike tangent vectors has (exactly) two connected components. If $M$ is simply connected, then $(M, g)$ is time orientable for every Lorentzian metric $g$ on $M$.
