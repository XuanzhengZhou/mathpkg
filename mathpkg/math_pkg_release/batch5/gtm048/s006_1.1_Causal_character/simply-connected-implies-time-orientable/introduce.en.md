This proposition shows that time orientability is automatic when the underlying manifold is simply connected: the topological obstruction to time orientability vanishes. Thus, time orientability is a purely global topological condition involving the Lorentzian metric, not the smooth structure alone. The example of the Lorentzian metric $\hat{g}_1 = du^1 \otimes du^1 - du^2 \otimes du^2$ on $\mathbb{R}^2$ inducing a time-orientable metric on a non-simply-connected quotient illustrates that non-simply-connected manifolds can fail to be time orientable for some metrics. A complete proof is given in Section 8.2.3.
