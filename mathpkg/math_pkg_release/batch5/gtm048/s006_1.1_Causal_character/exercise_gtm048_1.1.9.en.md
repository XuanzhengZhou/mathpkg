Let $e \in V$ be a timelike vector in an $N$-dimensional Lorentzian vector space $(V, g)$. Use an orthonormal basis to prove the following results.

(a) The set of timelike vectors $\mathcal{T}_0$ is an open submanifold with two connected components
$$\mathcal{T}_0^+ = \{ v \in \mathcal{T}_0 \mid g(e, v) < 0 \}, \qquad \mathcal{T}_0^- = \{ v \in \mathcal{T}_0 \mid g(e, v) > 0 \};$$
each component is diffeomorphic to $\mathbb{R}^N$.

(b) If $N \geq 3$, the lightcone $\mathcal{L}_0$ has two connected components
$$\mathcal{L}_0^+ = \{ v \in \mathcal{L}_0 \mid g(e, v) < 0 \}, \qquad \mathcal{L}_0^- = \{ v \in \mathcal{L}_0 \mid g(e, v) > 0 \};$$
each component is diffeomorphic to $\mathbb{R} \times S^{N-2}$.
