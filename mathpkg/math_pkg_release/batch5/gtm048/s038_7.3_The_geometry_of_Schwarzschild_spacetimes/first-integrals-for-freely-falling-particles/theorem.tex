\text{Let } \gamma: \mathcal{E} \to U \text{ be a future-pointing causal geodesic in a Schwarzschild spacetime } U = N \cup B. \text{ Set } \delta = P \circ \gamma \text{ (the projection to } \mathcal{S}^2\text{) and } \rho = r \circ \gamma. \text{ Then there exist unique numbers } E \in \mathbb{R}, J \in [0, \infty), J_x \in [-J, J] \text{ such that:}
\text{(a) } g(\gamma_*, \partial) = -E;
\text{(b) } \rho^4 h(\delta_*, \delta_*) = J^2;
\text{(c) } \rho^2 h(\delta_*, K_x) = J_x.
