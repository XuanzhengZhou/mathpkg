Corollary 7.3.6 establishes the canonical static reference frame on the normal Schwarzschild region $N$. The normalized vector field $Z = (1 - 2\mu/r)^{-1/2} \partial/\partial t$ defines a static absolute reference frame, and the coordinate $t$ is uniquely characterized (up to an additive constant) as the time function for this frame that approaches proper time at infinity.
