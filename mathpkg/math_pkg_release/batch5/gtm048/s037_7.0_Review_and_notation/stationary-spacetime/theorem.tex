Let $Z$ be a reference frame on a spacetime $(M, g)$. $Z$ is defined as \textbf{stationary} iff there is a positive function $f$ on $M$ such that $fZ$ is a Killing vector field. $Z$ is defined as \textbf{static} iff $Z$ is stationary and irrotational. $(M, g)$ is \textbf{stationary} (respectively, \textbf{static}) iff there exists on $M$ a stationary (respectively, static) reference frame $Z$. An observer $\gamma: \mathcal{E} \to M$ is stationary iff he is in some stationary reference frame.
