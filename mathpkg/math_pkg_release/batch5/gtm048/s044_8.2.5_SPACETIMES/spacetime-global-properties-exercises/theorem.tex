Let $(M, g)$ be a spacetime. Then:
\begin{itemize}
\item[(a)] There exists a reference frame $Z$ on $M$.
\item[(b)] If $M$ is compact, then $M$ is not simply connected.
\item[(c)] If $\omega$ is a 1-form physically equivalent to a reference frame on $M$, then $g + 2\omega \otimes \omega$ is a Riemannian metric on $M$.
\item[(d)] There exists a volume element $\chi$ on $M$ such that $\int_M \chi = 1$.
\end{itemize}
