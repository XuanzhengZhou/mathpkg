Let $(\mathbb{R}^4, h)$ be Minkowski spacetime. Let $f: \mathbb{R} \rightarrow \mathbb{R}$ and $g: \mathbb{R} \rightarrow \mathbb{R}$ be smooth functions. Define on $\mathbb{R}^4$ the Lorentz metric $g$ by, for each chart with coordinates $(u^1, u^2, u^3, u^4) = (x, y, \phi, z)$:

$$g = -du^1 \otimes du^1 - du^2 \otimes du^2 - du^3 \otimes du^3 + du^4 \otimes du^4 + F(du^3 - du^4) \otimes (du^3 - du^4)$$

where $F = F(\phi) = f(\phi) \cdot x + g(\phi) \cdot y$. Orient $\mathbb{R}^4$ as usual and time-orient $(\mathbb{R}^4, g)$ by the vector field $Y = \partial_1 + \partial_2 + \partial_3 + \partial_4$. Then $(\mathbb{R}^4, g)$ is a spacetime, called a gravitational plane wave spacetime.
