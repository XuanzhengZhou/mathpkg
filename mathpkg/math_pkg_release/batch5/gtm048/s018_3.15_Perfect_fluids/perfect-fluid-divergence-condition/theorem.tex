Let $Z$ be a reference frame on $M$, and let $\rho$ and $p$ be functions on $M$ satisfying $\rho + p > 0$. Define a $(2,0)$-tensor by
$$\hat{T} = \rho Z \otimes Z + p(\hat{g} + Z \otimes Z).$$
Let $\operatorname{grad} p$ denote the vector field physically equivalent to $dp$. Then $\operatorname{div} \hat{T} = 0$ if and only if:

(a) $\operatorname{div}(\rho Z) = -p \operatorname{div} Z$;

(b) $D_Z Z = -(\rho + p)^{-1}\{(Zp)Z + \operatorname{grad} p\}$.
