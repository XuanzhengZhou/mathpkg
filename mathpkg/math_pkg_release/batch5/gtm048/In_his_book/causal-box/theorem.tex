Let $(\mathbb{R}^4, g, D)$ be Minkowski space. Define the **standard cylinder** $N_0 \subset \mathbb{R}^4$ by
$$N_0 = \left\{(u^1, \ldots, u^4) \;\middle|\; \sum_{\nu=1}^{3} u^\nu u^\nu \leq 1,\; |u^4| \leq 1\right\}.$$
Its boundary manifolds are the imbedded submanifolds:
$$B_1 = \left\{(u^1, u^2, u^3, -1) \;\middle|\; \sum_{\nu} u^\nu u^\nu < 1\right\},$$
$$B_2 = \left\{(u^1, u^2, u^3, 1) \;\middle|\; \sum_{\nu} u^\nu u^\nu < 1\right\},$$
$$B_3 = \left\{(u^1, u^2, u^3, u^4) \;\middle|\; \sum_{\nu} u^\nu u^\nu = 1,\; |u^4| < 1\right\},$$
$$C_1 = \left\{(u^1, u^2, u^3, -1) \;\middle|\; \sum_{\nu} u^\nu u^\nu = 1\right\},$$
$$C_2 = \left\{(u^1, u^2, u^3, 1) \;\middle|\; \sum_{\nu} u^\nu u^\nu = 1\right\}.$$
$B_3$ is timelike; the rest are spacelike. A **causal box** in a spacetime $M$ is a $C^\infty$ imbedding $\phi: N_0 \to M$ preserving orientation, time orientation, and the causal character of each boundary manifold. By abuse of language, $N \equiv \phi N_0$ is also called a causal box.
