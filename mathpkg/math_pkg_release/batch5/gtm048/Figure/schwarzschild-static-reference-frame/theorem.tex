Define $Z = (1 - 2\mu/r)^{-1/2}(\partial/\partial t)$ on the normal Schwarzschild region $N$. Then:

(1) $Z$ is a static, absolute reference frame on $N$.

(2) A function $\hat{t}: N \to \mathbb{R}$ is a time function for $Z$ such that $\hat{Z}\hat{t} \to 1$ as $r \to \infty$ iff $\hat{d}\hat{t} = dt|_N$.

(3) An observer $\gamma$ in $N$ is at rest relative to $Z$ iff $r \circ \gamma$ is constant.
