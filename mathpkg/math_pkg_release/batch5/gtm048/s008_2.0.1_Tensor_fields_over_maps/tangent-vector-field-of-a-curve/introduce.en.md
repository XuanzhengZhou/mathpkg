For a curve $\gamma$, the naive assignment of the tangent vector $\gamma_*u$ to the point $\gamma u$ fails to define an ordinary vector field when $\gamma$ self-intersects, because different parameter values can map to the same point with different tangent directions. The concept of a tensor field over the map $\gamma$ resolves this: $u \mapsto \gamma_*u$ is a perfectly well-defined $(1,0)$ tensor field over $\gamma$, called the tangent vector field.
