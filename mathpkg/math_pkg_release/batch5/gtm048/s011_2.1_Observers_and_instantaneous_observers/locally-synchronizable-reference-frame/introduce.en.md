A reference frame $Q$ is synchronizable if there exists a global time function $t$ whose level hypersurfaces are orthogonal to the observers in $Q$. The definition provides a hierarchy: proper time synchronizable ($\omega = -dt$) implies synchronizable ($\omega = -h\,dt$), which implies locally synchronizable ($\omega \wedge d\omega = 0$). Locally proper time synchronizable ($d\omega = 0$) sits between proper time synchronizable and locally synchronizable.
