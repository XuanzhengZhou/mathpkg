If $\phi$ and $\psi$ are two complex-valued functions on $G$, define

$$(\phi|\psi) = \frac{1}{g} \sum_{t \in G} \phi(t)\psi(t)^*,$$

where $g$ is the order of $G$. This is a Hermitian scalar product: it is linear in $\phi$, semilinear in $\psi$, and $(\phi|\phi) > 0$ for all $\phi \neq 0$.

If $\chi$ is a character, then $\check{\chi} = \chi$ (where $\check{\psi}(t) = \psi(t^{-1})^*$), so $(\phi|\chi) = \langle \phi, \chi \rangle$ for all $\phi$.
