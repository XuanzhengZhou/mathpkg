Let G be a p-group acting on a finite set X, and let X^G be the set of elements of X fixed by G. We have

$$\text{Card}(X) \equiv \text{Card}(X^G) \pmod{p}.$$

Indeed X - X^G is a union of nontrivial orbits of G, and the cardinality of each of these orbits is a power $p^\alpha$ of p, with $\alpha \geqslant 1$; hence Card(X - X^G) is divisible by p.

Let us now apply this lemma to the case X = G with G acting by inner automorphisms. The set X^G is just the center C of G. Thus

$$\text{Card}(C) \equiv \text{Card}(G) \equiv 0 \pmod{p},$$

whence C ≠ {1}, which proves the theorem.
