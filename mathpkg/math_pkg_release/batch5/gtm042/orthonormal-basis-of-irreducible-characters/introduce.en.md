Theorem 6 is the culmination of basic character theory: the irreducible characters not only form an orthonormal set (Theorem 3) but actually span the entire space of class functions. This means every class function on $G$ is a unique linear combination of irreducible characters. A key corollary is that the number of irreducible representations equals the number of conjugacy classes.
