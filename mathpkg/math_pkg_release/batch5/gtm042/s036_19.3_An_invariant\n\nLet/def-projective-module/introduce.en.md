A projective module is a module that is a direct summand of a free module. Equivalently, it satisfies a lifting property with respect to surjective homomorphisms: any map from the projective module to a quotient can be lifted to the original module. This is also equivalent to the exactness of the $\operatorname{Hom}$ functor. Projective modules are fundamental in homological algebra and representation theory.
