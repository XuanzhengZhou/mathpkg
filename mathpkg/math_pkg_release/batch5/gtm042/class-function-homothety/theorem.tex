Let $f$ be a class function on $G$, and let $\rho: G \to \text{GL}(V)$ be a linear representation of $G$. Define $\rho_f = \sum_{t \in G} f(t) \rho_t$. If $V$ is irreducible of degree $n$ and character $\chi$, then $\rho_f$ is a homothety of ratio $\lambda$ given by:

$$\lambda = \frac{1}{n} \sum_{t \in G} f(t) \chi(t) = \frac{g}{n} (f|\chi^*).$$
