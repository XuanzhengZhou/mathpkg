Let $g$ be the order of $G$, and let $(n_1, \ldots, n_h)$ be the degrees of the distinct irreducible representations of $G$. We know from Chapter 2 that $h$ is the number of conjugacy classes of $G$, and that $g = n_1^2 + \cdots + n_h^2$.

If $G$ is abelian, each conjugacy class consists of a single element, so $h = g$. The equality $g = \sum n_i^2$ with $h = g$ terms forces each $n_i = 1$.

Conversely, if all $n_i = 1$, then $h = \sum n_i^2 = g$, so $G$ has $g$ conjugacy classes, meaning each element is its own conjugacy class, hence $G$ is abelian.
