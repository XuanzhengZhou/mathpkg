Let $C_1, \ldots, C_k$ be the distinct conjugacy classes of $G$. A function $f$ on $G$ is a class function if and only if it is constant on each $C_i$; it is thus determined by its values $\lambda_i$ on the $C_i$, and these values can be chosen arbitrarily. Consequently, the dimension of the space $H$ of class functions is equal to $k$, the number of conjugacy classes.

On the other hand, by Theorem 6, the irreducible characters $\chi_1, \ldots, \chi_h$ form a basis of $H$. Hence the dimension of $H$ is also equal to $h$, the number of distinct irreducible characters (up to isomorphism). Therefore $h = k$. $\square$
