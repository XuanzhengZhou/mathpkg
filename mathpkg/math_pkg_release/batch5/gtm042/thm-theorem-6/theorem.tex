Let $C_1, \ldots, C_k$ be the distinct classes of $G$. To say that a function $f$ on $G$ is a class function is equivalent to saying that it is constant on each of $C_1, \ldots, C_k$; it is thus determined by its values $\lambda_i$ on the $C_i$, and these can be chosen arbitrarily. Consequently, the dimension of the space $H$ of class functions is equal to $k$. On the other hand, this dimension is, by th. 6, equal to the number of irreducible representations of $G$ (up to isomorphism). The result follows.
