Theorem 8 provides an explicit formula for the projection onto each isotypic component $V_i$ using only the character $\chi_i$ and the representation $\rho$. This makes the canonical decomposition computationally accessible: one can find $V_i$ without first decomposing $V$ into irreducibles. The uniqueness of the decomposition follows from the uniqueness of the projections.
