The following properties are equivalent for a finite group $G$:

\begin{itemize}
\item[(i)] Each character of $G$ has values in $\mathbb{Q}$.
\item[(i')] Each character of $G$ has values in $\mathbb{Z}$.
\item[(ii)] Two elements of $G$ which generate the same cyclic subgroup are conjugate.
\end{itemize}
