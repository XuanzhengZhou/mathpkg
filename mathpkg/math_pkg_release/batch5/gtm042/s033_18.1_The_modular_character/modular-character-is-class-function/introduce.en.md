The Brauer character $\phi_E$ is constant on conjugacy classes within $G_{\text{reg}}$. This follows from the fact that $t s t^{-1}$ and $s$ have the same eigenvalues (they are similar endomorphisms), so the lifted sums of eigenvalues coincide.
