Corollary 3 follows directly from Brauer's Theorem 42: the irreducible Brauer characters form a basis of the space of class functions on $G_{\text{reg}}$, so the dimension of this space equals the number of irreducible modular characters. But the dimension of the space of class functions on $G_{\text{reg}}$ is also exactly the number of $p$-regular conjugacy classes of $G$. Hence these two numbers coincide — the number of simple $k[G]$-modules equals the number of $p$-regular conjugacy classes.
