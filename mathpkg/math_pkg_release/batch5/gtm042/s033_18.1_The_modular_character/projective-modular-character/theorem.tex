If $F$ is a projective $k[G]$-module, and $\tilde{F}$ is a projective $A[G]$-module whose reduction modulo $\mathfrak{m}$ is $F$, denote by $\Phi_F$ the ordinary character of the $K[G]$-module $K \otimes_A \tilde{F}$. The function $\Phi_F$ is a class function on the whole group $G$ (not just $G_{\text{reg}}$) whose restriction to $G_{\text{reg}}$ coincides with the modular character $\phi_F$.
