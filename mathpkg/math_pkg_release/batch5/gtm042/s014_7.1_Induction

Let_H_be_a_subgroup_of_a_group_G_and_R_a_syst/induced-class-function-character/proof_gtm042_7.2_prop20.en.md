Assertion (ii) has already been proved in Section 3.3, Theorem 12. Assertion (i) can be proved by a direct calculation: for any $g, x \in G$, one computes $\operatorname{Ind}(f)(xgx^{-1})$ using the definition of the induced class function. Alternatively, (i) follows from (ii) together with the observation that every class function is a linear combination of characters. Since characters are class functions and $\operatorname{Ind}$ is linear, the induced function of any class function remains a class function.
