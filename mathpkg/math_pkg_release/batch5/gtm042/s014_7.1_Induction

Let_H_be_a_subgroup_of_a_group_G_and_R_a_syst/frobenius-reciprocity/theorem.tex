Let $H$ be a subgroup of $G$. For a class function $\psi$ on $H$ and a class function $\varphi$ on $G$, we have

$$\langle \psi, \operatorname{Res}_H^G \varphi \rangle_H = \langle \operatorname{Ind}_H^G \psi, \varphi \rangle_G.$$

Equivalently, using the scalar product $(\alpha \mid \beta)$ defined in Section 2.3,

$$(\psi \mid \operatorname{Res} \varphi)_H = (\operatorname{Ind} \psi \mid \varphi)_G.$$

Moreover, the following useful formula holds:

$$\operatorname{Ind}(\psi \cdot \operatorname{Res} \varphi) = (\operatorname{Ind} \psi) \cdot \varphi.$$
