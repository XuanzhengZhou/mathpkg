By the corollary to Theorem 3, there exists an ordered basis $(a_1, \ldots, a_n)$ such that

$$(\sigma; (a_i)) = \operatorname{diag}(x_1, \ldots, x_r, 0, \ldots, 0),$$

with $x_i 
eq 0$, $i = 1, \ldots, r$. Over $\mathbb{C}$, every non-zero number has a square root; let $y_i^2 = x_i^{-1}$ for $i = 1, \ldots, r$. Then

$$(\sigma; (y_1 a_1, \ldots, y_r a_r, a_{r+1}, \ldots, a_n)) = egin{pmatrix} I_r & 0 \ 0 & 0 \end{pmatrix}.$$

Note that the only property of $\mathbb{C}$ used in the proof is that every complex number has at least one square root.
