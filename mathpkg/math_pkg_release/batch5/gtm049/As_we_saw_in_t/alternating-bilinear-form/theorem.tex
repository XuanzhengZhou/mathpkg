A bilinear form $\sigma$ on a vector space $V$ over a field $F$ is called \textbf{alternating} if

$$\sigma(a, a) = 0$$

for all $a$ in $V$. The pair $(V, \sigma)$ is then called an \textbf{alternating} (or \textbf{symplectic}) space.

If $\sigma$ is alternating, then for any $a, b$ in $V$,

$$0 = \sigma(a + b, a + b) = \sigma(a, a) + \sigma(a, b) + \sigma(b, a) + \sigma(b, b)$$

whence $\sigma(a, b) = -\sigma(b, a)$, i.e., $\sigma$ is skew-symmetric.
