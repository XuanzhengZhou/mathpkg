For any vectors $a, b$ in a euclidean space $(V, \sigma)$,
$$\|a\| = +\sqrt{\sigma(a, a)}, \qquad d(a, b) = \|a - b\|.$$

The distance $d$ satisfies the following properties for all $a, b, c \in V$:
\begin{itemize}
\item[D.1.] $d(a, b) = d(b, a)$;
\item[D.2.] $d(a, b) \geq 0$ and $d(a, b) = 0$ if, and only if, $a = b$;
\item[D.3.] $d(a, c) \leq d(a, b) + d(b, c)$ (the triangle inequality);
\item[D.4.] $d(a + c, b + c) = d(a, b)$;
\item[D.5.] for any real $x$, $d(xa, xb) = |x|\, d(a, b)$;
\item[D.6.] $d(a + b, 0)^2 + d(a - b, 0)^2 = 2(d(a, 0)^2 + d(b, 0)^2)$ (the parallelogram law).
\end{itemize}
