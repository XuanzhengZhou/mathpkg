**Proof.** The vector $b \in V$ lies in $(M\varrho)^{\circ}$ if and only if $b f = 0$ for all $f \in M\varrho$, i.e., for all $f = a\varrho$ with $a \in M$. By the definition of $\varrho$, this means $(a\varrho)(b) = 0$ for all $a \in M$. Since $(a\varrho)(b) = \sigma(a, b)$, this is equivalent to $\sigma(a, b) = 0$ for all $a \in M$, i.e., $b \in M^{\perp}(\sigma)$.

The second equality $M^{\top}(\sigma) = (M\tilde{\sigma})^{\circ}$ is proved in exactly the same way, using $\tilde{\sigma}$ in place of $\varrho$.
