This exercise shows that for any non-zero vector $a$ in a finite-dimensional complex vector space, repeatedly applying a linear mapping $f$ generates a sequence $a, af, af^2, \ldots$ whose linear dependence yields a polynomial $p(X)$ such that $ap(f)$ is an eigenvector. This is a key technique used in the proof of Proposition 3 for the existence of eigenvectors over algebraically closed fields.
