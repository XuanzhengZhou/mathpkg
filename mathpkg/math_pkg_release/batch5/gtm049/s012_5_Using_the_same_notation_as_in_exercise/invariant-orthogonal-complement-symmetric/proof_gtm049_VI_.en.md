For any $$a \in M$$ and $$b \in M^\perp$$, we have $$af \in M$$ (since $$M$$ is $$f$$-invariant) and $$\sigma(a, b) = 0$$. Using $$\sigma$$-symmetry of $$f$$:
$$\sigma(a, bf) = \sigma(af, b) = 0.$$
Thus $$bf \in M^\perp$$ for all $$b \in M^\perp$$, so $$M^\perp f \subset M^\perp$$. Equality follows because $$f$$ is one-one on the Euclidean space.
