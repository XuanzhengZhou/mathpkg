Let $$A_0, A_1, A_2, A_3$$ be distinct collinear points with $$\operatorname{cr}(A_0, A_1; A_2, A_3) = x$$, meaning there exist homogeneous vectors $$a_0, a_1$$ such that $$[a_0] = A_0, [a_1] = A_1, [a_0+a_1] = A_2, [xa_0+a_1] = A_3$$.

If $$\pi$$ is induced by a linear automorphism $$g$$, then $$A_i \pi = [a_i g]$$. Hence $$[a_0 g + a_1 g] = A_2 \pi$$ and $$[x a_0 g + a_1 g] = A_3 \pi$$. Thus the cross-ratio with respect to the homogeneous vectors $$a_0 g, a_1 g$$ is again $$x$$.

If $$\pi$$ is a correlation (semi-linear case), the same argument applies using the semi-linear automorphism. Thus cross-ratio is invariant under all collineations.
