Two Cartesian bases $$(a_i), (b_i)$$ of $$(V, \sigma)$$ have the same **orientation** if the Euclidean automorphism mapping $$a_i \mapsto b_i$$ (for all $$i$$) is a rotation. This defines an equivalence relation with exactly two equivalence classes. To **orient** $$(V, \sigma)$$ is to assign one of these two classes to the space.
