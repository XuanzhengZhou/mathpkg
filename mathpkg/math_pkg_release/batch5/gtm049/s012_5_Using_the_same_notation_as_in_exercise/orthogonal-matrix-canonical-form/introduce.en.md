This corollary translates the canonical form for orthogonal automorphisms into the language of matrices: every orthogonal matrix is orthogonally similar to a block diagonal matrix with $$2 \times 2$$ rotation blocks and $$\pm 1$$ entries on the diagonal. The similarity is by an orthogonal matrix $$P$$, so the structure is conjugation within the orthogonal group.
