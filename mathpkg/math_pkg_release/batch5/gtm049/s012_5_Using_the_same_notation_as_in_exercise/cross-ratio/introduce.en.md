The cross-ratio is a fundamental projective invariant of four collinear points. It is defined using homogeneous coordinates: after normalizing so that $$A_0, A_1$$ have coordinate vectors $$a_0, a_1$$ and $$A_2$$ corresponds to $$a_0 + a_1$$, the scalar $$x$$ for which $$A_3$$ corresponds to $$x a_0 + a_1$$ is the cross-ratio. The cross-ratio is invariant under collineations.
