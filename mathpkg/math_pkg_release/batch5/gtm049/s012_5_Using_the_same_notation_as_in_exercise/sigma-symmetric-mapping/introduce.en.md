A linear mapping $f$ is called $\sigma$-symmetric with respect to a symmetric bilinear form $\sigma$ if it satisfies $\sigma(af, b) = \sigma(a, bf)$ for all vectors $a, b$. In the context of a euclidean space $(V, \sigma)$, this is equivalent to $f$ having a symmetric matrix with respect to any cartesian basis. Given a $\sigma$-symmetric $f$, one can define a new symmetric bilinear form $\tau$ by $\tau(a, b) = \sigma(af, b)$, establishing a correspondence between $\sigma$-symmetric mappings and symmetric bilinear forms.
