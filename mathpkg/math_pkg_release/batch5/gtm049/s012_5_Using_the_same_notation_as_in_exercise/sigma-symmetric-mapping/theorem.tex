A linear mapping $f$ is called $\sigma$-symmetric if it satisfies

$$\sigma(af, b) = \sigma(a, bf)$$

for all $a, b$ in $V$.

Conversely, if a symmetric bilinear form $\sigma$ and a linear mapping $f$ satisfying the above equation are given, then one may define a symmetric bilinear form $\tau$ on $V$ by

$$\tau(a, b) = \sigma(af, b).$$

In a euclidean space $(V, \sigma)$, a linear mapping $f$ is $\sigma$-symmetric if, and only if, it has a symmetric matrix with respect to any cartesian basis of $(V, \sigma)$.
