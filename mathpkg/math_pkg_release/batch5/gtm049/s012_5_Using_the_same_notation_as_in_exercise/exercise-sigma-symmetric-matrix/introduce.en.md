This exercise provides the matrix characterization of a $\sigma$-symmetric linear mapping. Given a symmetric bilinear form $\sigma$ with matrix $S$ and a linear mapping $f$ with matrix $A$ relative to the same basis, the condition $\sigma(af, b) = \sigma(a, bf)$ translates to the matrix equation $AS = SA^t$. When the basis is cartesian ($S = I$), this reduces to $A = A^t$, i.e., $A$ is symmetric.
