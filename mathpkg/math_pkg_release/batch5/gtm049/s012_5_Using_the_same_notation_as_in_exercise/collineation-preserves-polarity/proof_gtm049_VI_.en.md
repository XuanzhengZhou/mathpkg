If $$M$$ is any subspace of $$H$$, then $$M^{\perp(\sigma^g)}$$ is the set of all vectors $$b$$ such that $$\sigma^g(M, b) = 0$$. Hence $$M^{\perp(\sigma^g)}g$$ is the set of all vectors $$bg$$ such that $$\sigma(Mg, bg) = 0$$, which means $$M^{\perp(\sigma^g)}g = (Mg)^{\perp(\sigma)}$$. We thus have the formula
$$\perp(\sigma^g)\mathcal{P}(g) = \mathcal{P}(g)\perp(\sigma).$$
We conclude that $$\mathcal{P}(g)$$ preserves $$\perp(\sigma)$$ if, and only if, $$\perp(\sigma^g) = \perp(\sigma)$$, which is equivalent to $$D^g = D$$ (the distance class being preserved), as required.
