A basis $(a_1, \ldots, a_n)$ of a euclidean space $(V, \sigma)$ is called a \text{cartesian basis} if it consists of mutually orthogonal vectors of length $1$, i.e.,
$$\sigma(a_i, a_j) = \delta_{ij} \quad \text{for all } i, j.$$

If $d$ is the distance determined by $\sigma$ and $a = \sum x_i a_i$, $b = \sum y_i a_i$, then
$$d(a, b) = +\sqrt{(x_1 - y_1)^2 + \cdots + (x_n - y_n)^2}.$$

If the vectors $a, b$ are non-zero then the angle $\theta$ between them is given by
$$\cos \theta = \frac{x_1 y_1 + \cdots + x_n y_n}{+\sqrt{(x_1^2 + \cdots + x_n^2)(y_1^2 + \cdots + y_n^2)}}.$$
