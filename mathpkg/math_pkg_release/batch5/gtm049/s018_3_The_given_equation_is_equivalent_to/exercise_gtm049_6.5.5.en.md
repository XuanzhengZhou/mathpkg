The given equation is equivalent to $$b(af^*g) = bf(ag)$$ for all $$a, b$$, i.e., $$af^* = (f(ag))g^{-1}$$ for all $$a$$. Since $$g$$ is semi-linear with respect to complex conjugation this defines a linear mapping $$f^*$$. (In the language of exercise 7, §4.6, p. 83, $$f(ag) = a(ggf^t)$$ and so $$f^* = gf^t g^{-1}$$.)

(i) $$\sigma(af^{**}, b) = \sigma(a, bf^*) = \overline{\sigma(bf^*, a)} = \overline{\sigma(b, af)} = \sigma(af, b)$$ for all $$a, b \in W$$, hence $$f^{**} = f$$.
