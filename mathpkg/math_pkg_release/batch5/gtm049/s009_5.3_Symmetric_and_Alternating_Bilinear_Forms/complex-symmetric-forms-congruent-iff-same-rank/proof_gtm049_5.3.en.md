**PROOF.** Rank is invariant under congruence, so congruent forms have the same rank. Conversely, if two symmetric bilinear forms on a complex vector space have the same rank $r$, then by Proposition 7 each has matrix $\begin{pmatrix} I_r & 0 \\ 0 & 0 \end{pmatrix}$ relative to a suitable basis. The change-of-basis matrix provides the congruence.
