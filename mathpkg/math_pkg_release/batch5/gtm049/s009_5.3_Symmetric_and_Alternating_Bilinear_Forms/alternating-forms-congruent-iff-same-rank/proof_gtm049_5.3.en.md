**PROOF.** If two alternating bilinear forms are congruent, they clearly have the same rank since rank is invariant under congruence. Conversely, if two alternating forms on $V$ have the same rank $2s$, then by Corollary 2, with respect to suitably chosen ordered bases, each has matrix $A_{n,s}$ (where $n = \dim V$). The change-of-basis matrix between these two bases furnishes the required congruence.
