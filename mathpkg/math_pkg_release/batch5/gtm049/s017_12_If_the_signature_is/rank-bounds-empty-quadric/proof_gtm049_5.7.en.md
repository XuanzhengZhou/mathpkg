(i) Over an ordered field, if all coefficients $a_0, \ldots, a_{r-1}$ have the same sign, then a sum $a_0 X_0^2 + \cdots + a_{r-1} X_{r-1}^2$ with not all $X_i = 0$ is a non-zero element of the same sign, hence cannot be zero. Thus $Q(\sigma_M)$ is empty. Conversely, if coefficients have mixed signs, the equation $a_0 X_0^2 + \cdots + a_{r-1} X_{r-1}^2 = 0$ admits non-trivial solutions by balancing positive and negative terms. Since any $r \geq 1$ can be achieved with all coefficients positive, any rank is possible. (ii) Over an algebraically closed field, when $r > 1$, the equation $a_0 X_0^2 + \cdots + a_{r-1} X_{r-1}^2 = 0$ always admits non-trivial solutions: set $X_1 = 1$ and solve for $X_0$ via the square root. Thus $Q(\sigma_M) \neq \emptyset$ for $r > 1$, so only $r = 1$ yields an empty quadric. (iii) Over $\mathbb{R}$, if $r > 2$, the homogeneous quadric equation in $r$ variables with mixed signs always has a non-trivial real solution (by the intermediate value property). If $r = 2$ and $a_0, a_1$ have opposite signs, non-trivial real solutions exist. If $r = 2$ and $a_0 = a_1$ (both positive or both negative), then $a_0 X_0^2 + a_1 X_1^2 = 0$ forces $X_0 = X_1 = 0$, so $Q(\sigma_M)$ is empty. Hence over $\mathbb{R}$, $r \in \{1, 2\}$.
