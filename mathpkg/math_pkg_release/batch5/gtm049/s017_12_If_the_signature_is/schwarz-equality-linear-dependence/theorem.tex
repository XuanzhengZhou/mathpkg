If $d(a, c) = d(a, b) + d(b, c)$ in a euclidean space, then $a, b, c$ are collinear.

Let $u = b - c$, $v = a - b$. Then $\|u + v\|^2 = (\|u\| + \|v\|)^2 = \|u\|^2 + \|v\|^2 + 2\|u\|\|v\|$. Hence $\sigma(u, v) = \|u\| \|v\|$ and, by Schwarz's Inequality, $u, v$ are linearly dependent. Thus $a, b, c$ are linearly dependent.
