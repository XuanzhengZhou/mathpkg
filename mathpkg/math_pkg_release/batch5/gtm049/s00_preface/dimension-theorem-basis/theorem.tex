**Theorem 1.** Let $V$ be a given finite dimensional vector space.

(1) Any two bases of $V$ have the same number of elements.

(2) The following statements concerning the finite subset $S$ of $V$ are equivalent:
\begin{itemize}
\item[(i)] $S$ is a basis of $V$,
\item[(ii)] $S$ is a minimal spanning set of $V$,
\item[(iii)] $S$ is a maximal linearly independent set in $V$.
\end{itemize}

(3) Any set of linearly independent vectors in $V$ can be extended to give a basis of $V$.
