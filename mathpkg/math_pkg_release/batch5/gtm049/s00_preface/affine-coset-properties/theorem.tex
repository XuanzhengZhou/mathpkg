**Theorem 1.** Let $S$ and $T$ be cosets in the vector space $V$.

(1) If $S \subset T$, then $\dim S \leq \dim T$ and $\dim S = \dim T$ implies $S = T$.

(2) If $S \cap T$ is not empty, then

$$\dim(S \cup T) + \dim(S \cap T) = \dim S + \dim T.$$

(3) Assuming $S \cap T$ is not empty, $S$ and $T$ are parallel if, and only if, one contains the other.

Assuming $S \cap T$ is empty, $S$ and $T$ are parallel if, and only if,

$$\dim(S \cup T) = \max(\dim S, \dim T) + 1.$$
