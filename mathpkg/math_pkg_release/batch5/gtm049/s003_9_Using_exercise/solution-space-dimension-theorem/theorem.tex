Let $F$ be a field. The set of all solutions $(x_1, \ldots, x_n)$ in $F^n$ of the homogeneous linear equations
$$\sum_{j=1}^{n} a_{ij} X_j = 0 \quad (i = 1, \ldots, m)$$
is a subspace of $F^n$ of dimension $n - r$, where $r$ is the maximum number of linearly independent rows $(a_{i1}, \ldots, a_{in})$, $i = 1, \ldots, m$.
