\text{If } S = a + M, \; S' = a' + M' \text{ are cosets in vector spaces } V, V' \text{ over fields } F, F', \text{ respectively, an affinity of } \mathcal{A}(S) \text{ onto } \mathcal{A}(S') \text{ is a mapping of the form } \mathcal{A}(t_{-a} g t_{a'}) \text{ where } g \text{ is an isomorphism of } M \text{ onto } M'.
