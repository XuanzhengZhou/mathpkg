Consider the frame of reference $(A_0, \ldots, A_n, U)$. Let $u$ be a homogeneous vector for $U$ (i.e., $[u] = U$) and let $c_0, \ldots, c_n$ be any homogeneous vectors for $A_0, \ldots, A_n$, respectively. Then $(c_0, \ldots, c_n)$ is an ordered basis of $V$ and $u = x_0c_0 + \cdots + x_nc_n$ where $x_i \neq 0$ for $i = 0, \ldots, n$. If we put $a_i = x_i c_i$ then $[a_i] = A_i$ ($i = 0, \ldots, n$) and $u = a_0 + \cdots + a_n$.

If $(a_0, \ldots, a_n), (b_0, \ldots, b_n)$ determine the same frame of reference, then $b_i = z_i a_i$ ($i = 0, \ldots, n$) and $b_0 + \cdots + b_n = z(a_0 + \cdots + a_n)$. Hence $z_0 a_0 + \cdots + z_n a_n = z a_0 + \cdots + z a_n$. Since $(a_0, \ldots, a_n)$ is a basis, we deduce $z_i = z$ for all $i = 0, \ldots, n$. Thus $b_i = z a_i$ for $i = 0, \ldots, n$.
