Proposition 1 characterizes exactly when two vector space isomorphisms induce the same projectivity: they differ by a non-zero scalar factor. If $g = zf$ for some non-zero scalar $z$, then $Mf = Mg$ for every subspace $M$, so $\mathcal{P}(f) = \mathcal{P}(g)$. Conversely, if $\mathcal{P}(f) = \mathcal{P}(g)$, one can show that on each one-dimensional subspace the two isomorphisms differ by a scalar, and a linear independence argument forces all these scalars to be equal.
