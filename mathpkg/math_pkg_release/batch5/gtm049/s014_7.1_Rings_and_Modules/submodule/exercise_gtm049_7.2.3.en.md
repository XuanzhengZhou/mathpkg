If $R$ is an integral domain (see exercise 1) and $M$ is a module over $R$, prove that the set of all torsion elements in $M$ forms a submodule. This result is false without the integral domain property: if $R = \mathbb{Z}/[6]$, show that $(2, 0), (0, 3)$ are torsion elements in $R^2$ but that their sum is not a torsion element.
