A finitely generated module is one that admits a finite set of generators: every element of the module can be expressed as an $R$-linear combination of these generators. This finiteness condition is essential for the structure theorems of module theory---the classification of finitely generated modules over a principal ideal domain is one of the central results of linear algebra. A cyclic module is the simplest case, being generated by a single element, and every cyclic module is isomorphic to a quotient $R/\mathfrak{a}$ of the base ring by an ideal.
