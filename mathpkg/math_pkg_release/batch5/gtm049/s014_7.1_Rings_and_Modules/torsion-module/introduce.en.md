A torsion module is an $R$-module in which every element is annihilated by some non-zero scalar. For example, a module of the form $V_f$ (a vector space $V$ made into an $F[X]$-module via a linear transformation $f$) is always a torsion module because the minimal polynomial of $f$ annihilates the entire space. In contrast, a torsion-free module has no non-zero torsion elements. The structure theorem for finitely generated modules over a PID decomposes any such module into a direct sum of a torsion part and a torsion-free part.
