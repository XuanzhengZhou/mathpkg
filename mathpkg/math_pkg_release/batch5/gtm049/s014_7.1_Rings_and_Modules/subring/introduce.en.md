A subring is a subset of a ring that is closed under the ring operations and satisfies the ring axioms using the same addition and multiplication rules inherited from the parent ring. Subrings provide a way to identify smaller ring structures within a given ring. For example, $m\mathbb{Z}$ (the set of all integer multiples of $m$) is a subring of $\mathbb{Z}$, though it may not possess an identity element even if the parent ring does.
