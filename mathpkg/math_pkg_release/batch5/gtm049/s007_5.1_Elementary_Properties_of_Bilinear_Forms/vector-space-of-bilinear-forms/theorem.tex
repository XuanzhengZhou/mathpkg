Let $\mathcal{B}(V)$ denote the set of all bilinear forms on $V$. Define $\sigma + \tau$ and $x\sigma$ pointwise:

$$(\sigma + \tau)(a, b) = \sigma(a, b) + \tau(a, b),$$
$$(x\sigma)(a, b) = x\sigma(a, b)$$

for any $\sigma, \tau$ in $\mathcal{B}(V)$ and any $x$ in $F$. Then $\mathcal{B}(V)$ is a vector space over $F$.
