Let $\sigma$ be a bilinear form on a vector space $V$. If $(a_1, \ldots, a_n)$ is any ordered basis of $V$, then the rank of $\sigma$ is defined to be the rank of the matrix $(\sigma; (a_i))$.

If $\operatorname{rank} \sigma < n$, $\sigma$ is said to be degenerate. If $\operatorname{rank} \sigma = n$, $\sigma$ is said to be non-degenerate.
