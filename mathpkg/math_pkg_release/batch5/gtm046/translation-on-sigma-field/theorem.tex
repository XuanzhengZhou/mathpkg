A single-valued transformation $T$ on the $\sigma$-field $\mathfrak{B}$ into itself is a translation (by 1) if it preserves all countable set operations:
$$T\left(\bigcup A_j\right) = \bigcup T A_j, \quad T(\Omega) = \Omega, \quad T(\emptyset) = \emptyset.$$
The $k$th iterate $T^k = T \circ T^{k-1}$ with $T^0 = I$ (identity).
