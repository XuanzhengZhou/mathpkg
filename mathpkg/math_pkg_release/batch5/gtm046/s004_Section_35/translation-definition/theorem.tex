A single-valued transformation $T$ on a $\sigma$-field $\mathfrak{B}$ into itself is a translation (by 1) if it preserves all countable operations: for any $A,A_j\in\mathfrak{B}$,
$$T(\bigcup_j A_j)=\bigcup_j TA_j,\quad T(\bigcap_j A_j)=\bigcap_j TA_j,\quad T(A^c)=(TA)^c,\quad T\Omega=\Omega,\quad T\emptyset=\emptyset.$$
$T^k$ is the $k$th iterate, $T^0=I$. The translate of $\xi$ by $k-1$ is $\xi_k=T^{k-1}\xi$ determined by $[\xi_k\in S]=T^{k-1}[\xi\in S]$ for all Borel $S$. $C$ is invariant if $TC=C$.
