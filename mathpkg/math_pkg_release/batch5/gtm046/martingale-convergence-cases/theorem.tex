I. Martingale sequence: $EX_1 = EX_2 = \cdots$; $EX_1^+ \leq EX_2^+ \leq \cdots$; $E|X_1|^r \leq E|X_2|^r \leq \cdots$. Convergence if $\lim EX_n^+ < \infty$ or $\lim EX_n^- < \infty$. II. Submartingale sequence: $EX_1 \leq EX_2 \leq \cdots$. Convergence if $\lim EX_n^+ < \infty$. IV. Submartingale reversed sequence: $\cdots \leq EX_2 \leq EX_1$. Convergence if $EX_1^+ < \infty$.
