If $B^m = [\sup_{j \leq m} X^j/Y^j > b]$ and $Z^n > 0$, then for every integer $n$ and every invariant event $C$: $$\sum_{k=1}^{n} \int_{B_k^m C} \left( \frac{X_k}{Z^n} - b \frac{Y_k}{Z^n} \right) + \sum_{k=n+1}^{n+m} \int_C \left( \frac{X_k}{Z^n} - b \frac{Y_k}{Z^n} \right)^+ \geq 0,$$ provided the sum exists.
