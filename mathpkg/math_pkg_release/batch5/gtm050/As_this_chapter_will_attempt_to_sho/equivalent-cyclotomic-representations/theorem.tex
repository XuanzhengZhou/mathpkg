Two polynomials $f(\alpha)$ and $g(\alpha)$ of degree at most $\lambda-1$ in $\alpha$ with integer coefficients represent the same cyclotomic integer if and only if they differ by an integer multiple of $1 + \alpha + \alpha^2 + \cdots + \alpha^{\lambda-1}$. That is,
$$f(\alpha) = g(\alpha) \quad \Longleftrightarrow \quad f(\alpha) - g(\alpha) = c \cdot (1 + \alpha + \alpha^2 + \cdots + \alpha^{\lambda-1})$$
for some integer $c$.
