Let $h(\alpha)$ be a cyclotomic integer that is not a unit. Then $h(\alpha)$ is prime (i.e., $h(\alpha) \mid f(\alpha)g(\alpha)$ implies $h(\alpha) \mid f(\alpha)$ or $h(\alpha) \mid g(\alpha)$) if and only if its norm $Nh(\alpha)$ is a prime integer $p$.

Moreover, in this case:
\begin{enumerate}
\item $p \equiv 0$ or $1 \pmod{\lambda}$,
\item If $p = \lambda$, then $h(\alpha)$ is a unit multiple of $\alpha - 1$,
\item If $p \equiv 1 \pmod{\lambda}$, then there exists an integer $k$ such that
$$f(\alpha) \equiv g(\alpha) \pmod{h(\alpha)} \iff f(k) \equiv g(k) \pmod{p}$$
for all cyclotomic integers $f(\alpha), g(\alpha)$.
\end{enumerate}
