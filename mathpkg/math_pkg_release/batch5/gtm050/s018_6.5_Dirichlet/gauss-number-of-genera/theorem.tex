Let $D$ be a determinant of binary quadratic forms. Let $m$ be the number of odd prime factors of $D$, and let $\varepsilon$ be defined by:
\begin{itemize}
\item $\varepsilon = 0$ if $D \equiv 1 \pmod{4}$,
\item $\varepsilon = 1$ if $D \equiv 3 \pmod{4}$ or $D \equiv 0 \pmod{8}$,
\item $\varepsilon = 2$ if $D \equiv 4 \pmod{8}$.
\end{itemize}
Then the number of possible characters is $2^{m+\varepsilon}$, and the number of genera (characters that actually occur) is exactly $2^{m+\varepsilon-1}$. Equivalently, at most half of the possible characters actually occur.
