Let $\chi$ be a Dirichlet character modulo a prime $\lambda$, and let $\alpha$ be a primitive $\lambda$th root of unity. For $\nu = 0, 1, \dots, \lambda-2$, define

$$c_\nu = \frac{1}{\lambda} \sum_{j=1}^{\lambda-1} \chi(j) \alpha^{-j\nu}.$$

In particular, $c_0 = 0$ when $\chi$ is nonprincipal, because the sum of the values of $\chi$ is zero. The terms can be reordered using a primitive root $\gamma$ modulo $\lambda$: setting $b_k = c_\nu$ where $\nu \equiv \gamma^k \bmod \lambda$, one has

$$b_k = \sigma^k(b_0)$$

where $\sigma$ is the conjugation $\alpha \mapsto \alpha^\gamma$ acting on cyclotomic integers in the natural way.
