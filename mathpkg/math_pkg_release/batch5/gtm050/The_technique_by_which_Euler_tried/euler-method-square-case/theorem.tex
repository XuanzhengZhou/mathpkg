If $pq$ is a square, then either each factor is itself a square (i.e., $p = r^2$ and $q = s^2$), or the factors are squares multiplied by one and the same number (i.e., $p = mr^2$ and $q = ms^2$).

This principle, which is true for ordinary integers, was extended by Euler without proof to "numbers" of the form $x + y\sqrt{-c}$ in his study of equations $x^2 + cy^2 = \text{square}$.

Euler's derived solution for $x^2 + cy^2 = \text{square}$ is given by $x = cq^2 - p^2$ and $y = 2pq$, so that

$$
x + y\sqrt{-c} = (p + q\sqrt{-c})^2.
$$
