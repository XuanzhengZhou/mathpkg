For a fixed integer $c > 0$, the set
$$\mathbb{Z}[\sqrt{-c}] = \{a + b\sqrt{-c} : a, b \in \mathbb{Z}\}$$
forms a commutative ring with respect to the operations
$$(a + b\sqrt{-c}) + (c + d\sqrt{-c}) = (a+c) + (b+d)\sqrt{-c}$$
$$(a + b\sqrt{-c})(c + d\sqrt{-c}) = (ac - bcd) + (ad + bc)\sqrt{-c}.$$
Elements of this ring are called \textbf{quadratic integers} or \textbf{surds}.
