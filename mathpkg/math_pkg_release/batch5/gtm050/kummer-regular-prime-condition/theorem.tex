An odd prime $\lambda$ is called \text{regular} if it satisfies the following two conditions:
\begin{enumerate}
\item[(A)] The class number of the cyclotomic field $\mathbb{Q}(\alpha)$ (where $\alpha^\lambda = 1$, $\alpha \neq 1$) is not divisible by $\lambda$.
\item[(B)] Every unit congruent to an integer modulo $\lambda$ is a $\lambda$-th power.
\end{enumerate}
Kummer proved that (A) implies (B), so regularity is equivalent to condition (A) alone.
