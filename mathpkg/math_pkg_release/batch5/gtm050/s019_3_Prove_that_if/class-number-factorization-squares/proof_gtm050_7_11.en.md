The factorization $h = gn$ follows from the definition of genera: the divisor class group is partitioned into $g$ genera, each containing the same number $n$ of classes, as shown at the end of Section 7.9.

For $h = sa$, consider the squaring homomorphism on the class group. Let $A_1, A_2, \ldots, A_a$ be a list of divisors with one in each of the $a$ ambiguous classes. Two classes have the same square if and only if $B^2 \sim C^2$, which is equivalent to $(B\bar{C})^2 \sim I$, i.e., $B\bar{C}$ lies in an ambiguous class. This happens exactly when $C \sim A_i B$ for some $i = 1, 2, \ldots, a$. Thus exactly $a$ classes have the same square as the class of any given $B$, so the number of distinct squares is $s = h/a$. Hence $h = sa$.
