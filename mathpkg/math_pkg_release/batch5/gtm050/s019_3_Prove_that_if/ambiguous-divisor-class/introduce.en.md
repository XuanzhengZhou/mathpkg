An ambiguous divisor class is a class in the divisor class group of quadratic integers whose square is the principal class. A divisor $A$ is said to lie in an ambiguous class when it is equivalent to its own conjugate $\bar{A}$, or equivalently $A^2 \sim I$. Gauss discovered that the number of ambiguous classes can be bounded directly without using quadratic reciprocity, and this bound provides crucial information about the possible characters of divisor classes, ultimately enabling a proof of quadratic reciprocity itself.
