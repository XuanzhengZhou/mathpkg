For a fundamental discriminant $D$, the class number $h(D)$ of the quadratic field $\mathbb{Q}(\sqrt{D})$ is given by:
$$h(D) = \frac{w\sqrt{|D|}}{2\pi} L(1, \chi_D) \quad (D < 0)$$
$$h(D) = \frac{\sqrt{D}}{\log \varepsilon} L(1, \chi_D) \quad (D > 0)$$
where $w$ is the number of roots of unity in the field, $\chi_D(n) = \left(\frac{D}{n}\right)$ is the Kronecker symbol, $L(1, \chi_D) = \sum_{n=1}^{\infty} \frac{\chi_D(n)}{n}$, and $\varepsilon$ is the fundamental unit (for $D > 0$).
