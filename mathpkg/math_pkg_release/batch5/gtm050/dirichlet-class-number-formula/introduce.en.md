Dirichlet's class number formula, proved in 1839, is one of the deepest results in number theory. It connects the algebraic invariant $h(D)$ (the class number) with the analytic invariant $L(1,\chi_D)$ (the value of the Dirichlet $L$-series at $s=1$). This formula was the prototype for Kummer's class number formula for cyclotomic fields. The proof uses the Euler product formula, which expresses the zeta function of the quadratic field as a product over primes, and relates this product to the $L$-series. The formula allows explicit computation of class numbers and is the foundation for the analytic approach to algebraic number theory.
