Let $\lambda$ be an odd prime and $\alpha$ a primitive $\lambda$-th root of unity. Then
$$x^\lambda + y^\lambda = (x + y)(x + \alpha y)(x + \alpha^2 y) \cdots (x + \alpha^{\lambda-1} y).$$
In the context of Fermat's equation $x^\lambda + y^\lambda = z^\lambda$, this gives the factorization
$$(x + y)(x + \alpha y)(x + \alpha^2 y) \cdots (x + \alpha^{\lambda-1} y) = z^\lambda$$
which is the starting point for Kummer's approach via ideal factorization.
