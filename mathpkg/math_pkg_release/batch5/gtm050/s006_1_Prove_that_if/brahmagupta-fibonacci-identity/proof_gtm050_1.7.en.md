The identity is verified by direct algebraic expansion:

$$(a^2 + b^2)(c^2 + d^2) = a^2c^2 + a^2d^2 + b^2c^2 + b^2d^2$$

$$(ac - bd)^2 + (ad + bc)^2 = (a^2c^2 - 2abcd + b^2d^2) + (a^2d^2 + 2abcd + b^2c^2) = a^2c^2 + a^2d^2 + b^2c^2 + b^2d^2$$

Both sides are equal to $a^2c^2 + a^2d^2 + b^2c^2 + b^2d^2$, proving the identity. The second form $(a^2 + b^2)(c^2 + d^2) = (ac + bd)^2 + (ad - bc)^2$ is obtained similarly by swapping the roles of $c$ and $d$ in one of the factors.
