Let $\lambda$ be an odd prime. A sufficient condition for Fermat's Last Theorem to be true for the exponent $\lambda$ is that $\lambda$ be a \textbf{regular prime}, i.e., $\lambda$ does not divide the numerators of the Bernoulli numbers $B_2, B_4, \ldots, B_{\lambda-3}$.

Equivalently, $\lambda$ is regular if and only if:
\begin{enumerate}
\item (Condition A) The class number $h$ of the cyclotomic field $\mathbb{Q}(\alpha)$ is not divisible by $\lambda$, and
\item (Condition B) Every unit congruent to an integer modulo $\lambda$ is a $\lambda$-th power.
\end{enumerate}
