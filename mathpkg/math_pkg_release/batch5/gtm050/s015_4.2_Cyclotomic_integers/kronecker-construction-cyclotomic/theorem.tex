The ring of cyclotomic integers for a prime $\lambda$ can be defined algebraically as the quotient ring
$$\mathbb{Z}[\alpha] \cong \mathbb{Z}[X] \big/ \bigl(1 + X + X^2 + \cdots + X^{\lambda-1}\bigr),$$
where $\mathbb{Z}[X]$ is the ring of polynomials in one variable with integer coefficients and the ideal is generated by the cyclotomic polynomial $\Phi_\lambda(X) = X^{\lambda-1} + X^{\lambda-2} + \cdots + X + 1$. Addition and multiplication are defined in the obvious way subject to the single relation $1 + \alpha + \alpha^2 + \cdots + \alpha^{\lambda-1} = 0$.
