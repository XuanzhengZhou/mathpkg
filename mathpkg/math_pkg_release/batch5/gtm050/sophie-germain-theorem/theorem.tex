Let $n$ be an odd prime. If there exists an auxiliary prime $p$ with the properties that:
\begin{enumerate}
\item $x^n + y^n + z^n \equiv 0 \pmod{p}$ implies $x \equiv 0$ or $y \equiv 0$ or $z \equiv 0 \pmod{p}$, and
\item $x^n \equiv n \pmod{p}$ is impossible,
\end{enumerate}
then Case I of Fermat's Last Theorem is true for the exponent $n$ (i.e., $x^n + y^n = z^n$ has no solution with $x, y, z$ not divisible by $n$).
