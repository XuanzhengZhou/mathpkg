This lemma provides the fundamental reduction for attacking Fermat's Last Theorem: if $n = mk$ and a nontrivial integer solution $(x,y,z)$ exists for exponent $n$, then $(x^k, y^k, z^k)$ would be a nontrivial solution for exponent $m$. Thus proving the impossibility for a given exponent $m$ automatically establishes it for all multiples of $m$. This simple observation reduces the infinite family of exponents to a much smaller set of base cases.
