Suppose there exist positive integers $x, y, z$ and a positive integer $m$ such that $x^{4m} + y^{4m} = z^{4m}$. Set $X = x^m$, $Y = y^m$, $Z = z^m$. Then $X, Y, Z$ are positive integers and

$$X^4 + Y^4 = (x^m)^4 + (y^m)^4 = x^{4m} + y^{4m} = z^{4m} = (z^m)^4 = Z^4.$$

This contradicts Fermat's Last Theorem for $n = 4$, which states that no such positive integers $X, Y, Z$ exist. Therefore the original equation $x^{4m} + y^{4m} = z^{4m}$ has no positive integer solutions.
