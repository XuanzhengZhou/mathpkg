If a number which can be written as a sum of two squares is divisible by a number which is also a sum of two squares, then the quotient is itself a sum of two squares. That is, if $a^2 + b^2$ is divisible by $p^2 + q^2$, then $(a^2 + b^2)/(p^2 + q^2)$ can be expressed as $u^2 + v^2$ for some integers $u, v$.
