Direct expansion:

$$(xu - cyv)^2 + c(xv + yu)^2 = (x^2 u^2 - 2cxyuv + c^2 y^2 v^2) + c(x^2 v^2 + 2xyuv + y^2 u^2)$$
$$= x^2 u^2 + c^2 y^2 v^2 + c x^2 v^2 + c y^2 u^2$$
$$= x^2(u^2 + cv^2) + c y^2(c v^2 + u^2)$$
$$= (x^2 + c y^2)(u^2 + c v^2).$$

The norm interpretation follows from defining the "number" $a+b\sqrt{-c}$ and computing:
$$(x + y\sqrt{-c})(u + v\sqrt{-c}) = (xu - cyv) + (xv + yu)\sqrt{-c}.$$
Taking the norm $N(a+b\sqrt{-c}) = a^2 + cb^2$ of both sides yields the identity.
