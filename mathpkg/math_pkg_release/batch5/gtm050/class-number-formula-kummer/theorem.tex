For the cyclotomic field $\mathbb{Q}(\alpha)$ where $\alpha^\lambda = 1$, the class number $h$ can be factored as $h = h_1 \cdot h_2$ where:
\begin{itemize}
\item $h_1$ (the \textbf{first factor}) is given by
$$h_1 = \frac{1}{(2\lambda)^{\mu}} \prod_{j=1}^{\mu-1} |N(\Psi(\beta^{2j}))|$$
where $\mu = (\lambda-1)/2$.
\item $h_2$ (the \textbf{second factor}) is the class number of the maximal real subfield.
\end{itemize}
Moreover, $\lambda$ divides $h_1$ if and only if $\lambda$ divides the numerator of one of the Bernoulli numbers $B_2, B_4, \ldots, B_{\lambda-3}$.
