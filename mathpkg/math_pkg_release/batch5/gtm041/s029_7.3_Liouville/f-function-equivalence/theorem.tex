Let

$$F(t) = 1 + \sum_{r=1}^{n} e^{2\pi i(t\theta_r - \alpha_r)},$$

where $\alpha_1, \ldots, \alpha_n$ and $\theta_1, \ldots, \theta_n$ are arbitrary real numbers. Let

$$L = \sup_{-\infty < t < +\infty} |F(t)|.$$

Then the following two statements are equivalent:

(a) For every $\varepsilon > 0$ there exists a real $t$ and integers $h_1, \ldots, h_n$ such that

$$|t\theta_r - \alpha_r - h_r| < \varepsilon \quad \text{for } r = 1, 2, \ldots, n.$$

(b) $L = n + 1$.
