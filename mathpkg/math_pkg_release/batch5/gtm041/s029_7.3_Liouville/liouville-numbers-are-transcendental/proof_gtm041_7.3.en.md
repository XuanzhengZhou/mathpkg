If a Liouville number $\theta$ were algebraic of degree $n$ it would satisfy both inequality

$$0 < \left| \theta - \frac{h_r}{k_r} \right| < \frac{1}{k_r^r}$$

and

$$\left| \theta - \frac{h_r}{k_r} \right| > \frac{C(\theta)}{k_r^n}$$

for every $r \geq 1$, where $C(\theta)$ is the constant in Theorem 7.5. Therefore

$$0 < \frac{C(\theta)}{k_r^n} < \frac{1}{k_r^r}, \quad \text{or} \quad 0 < C(\theta) < \frac{1}{k_r^{r-n}}.$$

The last inequality gives a contradiction if $r$ is sufficiently large.
