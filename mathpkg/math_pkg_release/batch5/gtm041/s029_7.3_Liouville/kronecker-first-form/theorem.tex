If $\alpha_1, \ldots, \alpha_n$ are arbitrary real numbers, if $\theta_1, \ldots, \theta_n$ are linearly independent real numbers, and if $\varepsilon > 0$ is arbitrary, then there exists a real number $t$ and integers $h_1, \ldots, h_n$ such that

$$|t\theta_i - h_i - \alpha_i| < \varepsilon \quad \text{for } i = 1, 2, \ldots, n.$$
