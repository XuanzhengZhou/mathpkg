Kronecker's theorem (first form) is a deep generalization of Dirichlet's approximation theorem to simultaneous approximation. Given $n$ arbitrary real numbers $\alpha_i$ and $n$ linearly independent real numbers $\theta_i$, there exists a real parameter $t$ and integers $h_i$ making each $|t\theta_i - h_i - \alpha_i|$ arbitrarily small. The proof uses an analytic argument involving the function $F(t) = 1 + \sum e^{2\pi i(t\theta_r - \alpha_r)}$, showing that its supremum $L = n+1$ if and only if the approximation property holds.
