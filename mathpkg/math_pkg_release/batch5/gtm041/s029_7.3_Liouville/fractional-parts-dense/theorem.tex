If $\theta$ is a given irrational number the sequence of numbers $\{n\theta\}$ is dense in the unit interval. That is, given any $\alpha$, $0 \leq \alpha \leq 1$, and given any $\varepsilon > 0$, there exists a positive integer $k$ such that

$$|\{k\theta\} - \alpha| < \varepsilon.$$

Hence, if $h = [k\theta]$ we have $|k\theta - h - \alpha| < \varepsilon$.
