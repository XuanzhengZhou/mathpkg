Theorem 4.1 establishes that for a prime $p$, the congruence subgroup $\Gamma_0(p)$ has finite index in the full modular group $\Gamma$. The generators of $\Gamma$ are $S\tau = -1/\tau$ (inversion) and $T\tau = \tau + 1$ (translation). The theorem shows how every element of $\Gamma$ can be expressed as a product of elements from $\Gamma_0(p)$ and powers of $T$. This finite-index property is essential because it allows modular functions for $\Gamma_0(p)$ to be expressed as polynomials in a basic modular function $\Phi(\tau)$, analogous to how every modular function for $\Gamma$ is a rational function of $j(\tau)$. This representation is the key step in proving congruences for the Fourier coefficients of the modular invariant $j$.
