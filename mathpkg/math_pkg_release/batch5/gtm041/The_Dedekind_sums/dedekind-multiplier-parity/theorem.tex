Let $a, b, c, d$ be integers with $ad - bc = 1$ and $c > 0$. Let $c_1 = c / q$ where $q = (3, c)$. Define
$$\delta = s(a, c) - s(a, c_1) - \frac{a + d}{12c} + \frac{a + d}{12c_1}.$$
Then $r\delta$ is an even integer, where $r = 12$. Equivalently, $12\delta$ is an even integer, so $\delta$ is an integer multiple of $1/6$.
