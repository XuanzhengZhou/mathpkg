The sawtooth function $((x))$, also known as the first Bernoulli periodic function $\bar{B}_1(x)$, is a periodic odd function with period $1$ that captures the fractional part of a real number shifted by $-\tfrac{1}{2}$. It vanishes at integer arguments. This function provides a convenient notation for expressing Dedekind sums and simplifies manipulations involving sums over residue classes.
