The function $((x))$ is defined for real $x$ by
$$((x)) = \begin{cases} x - [x] - \frac{1}{2} & \text{if } x \text{ is not an integer}, \\ 0 & \text{if } x \text{ is an integer}. \end{cases}$$
This function is periodic with period $1$ and satisfies $((-x)) = -((x))$. It coincides with the Bernoulli periodic function $\bar{B}_1(x)$.
