The Jacobi theta function $\vartheta(\tau)$ is defined on the upper half-plane $H$ by the equation
$$\vartheta(\tau) = 1 + 2 \sum_{n=1}^{\infty} e^{\pi i n^2 \tau} = \sum_{n=-\infty}^{\infty} e^{\pi i n^2 \tau}.$$
The function $\vartheta$ is analytic in $H$ and periodic with period $2$, so that $\vartheta(\tau + 2) = \vartheta(\tau)$ for all $\tau \in H$.
