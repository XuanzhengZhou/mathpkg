Let $\tau(n)$ denote Ramanujan's tau function, defined by the Fourier expansion
$$\sum_{n=1}^{\infty} \tau(n) e^{2\pi i n \tau} = (2\pi)^{-12} \Delta(\tau) = e^{2\pi i \tau} \prod_{m=1}^{\infty} (1 - e^{2\pi i m \tau})^{24}.$$
Then for all positive integers $m, n$,
$$\tau(m)\tau(n) = \sum_{d \mid (m,n)} d^{11} \tau\left(\frac{mn}{d^2}\right).$$
In particular, when $(m,n) = 1$, this implies $\tau(m)\tau(n) = \tau(mn)$, so $\tau(n)$ is multiplicative.
