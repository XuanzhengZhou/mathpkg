Let $c(n)$ denote the coefficients in the Fourier expansion of the modular function $j(\tau)$:
$$j(\tau) = \sum_{n=-1}^{\infty} c(n) e^{2\pi i n \tau}.$$
Let $p$ be a prime such that $p^{-1} p \equiv 1 \pmod{13}$, and define $c(x) = 0$ if $x$ is not an integer. Then for all integers $n$:
$$c(13np) + c(13n)c(13p) + p^{-1}c\left(\frac{13n}{p}\right) \equiv 0 \pmod{13}.$$
