Let $q$ be a positive integer and let
$$\varphi(\tau) = \frac{\Delta(q\tau)}{\Delta(\tau)},$$
where $\Delta(\tau)$ is the modular discriminant. The function $\varphi$ has a zero of order $q-1$ at $\infty$ and no further zeros, so its valence is $q-1$. Define $\alpha = 1/(q-1)$ and set
$$\Phi(\tau) = \varphi(\tau)^\alpha.$$

Equivalently, using the Dedekind eta function $\eta(\tau)$, let $r = 24/(q-1)$ (when this is an integer). Then
$$\Phi(\tau) = \left(\frac{\eta(q\tau)}{\eta(\tau)}\right)^r.$$

The function $\Phi$ is analytic and nonzero in the upper half-plane $H$, has a first-order zero at $\infty$, and satisfies
$$\frac{1}{\Phi(\tau)} = \frac{1}{x} + I(x),$$
where $x = e^{2\pi i\tau}$ and $I(x)$ is a power series in $x$ with integer coefficients.
