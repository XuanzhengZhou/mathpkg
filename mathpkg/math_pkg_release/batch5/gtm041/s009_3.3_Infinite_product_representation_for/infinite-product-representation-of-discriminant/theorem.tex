If $\tau \in H$ and $x = e^{2\pi i\tau}$ then
$$\Delta(\tau) = (2\pi)^{12}\,\eta^{24}(\tau) = (2\pi)^{12}\,x\prod_{n=1}^{\infty}(1-x^n)^{24}.$$

Consequently,
$$\sum_{n=1}^{\infty}\tau(n)x^n = x\prod_{n=1}^{\infty}(1-x^n)^{24} \quad \text{whenever } |x| < 1$$
where $\tau(n)$ is Ramanujan's tau function.
