$V_f(\sigma_0)$ denotes the set of values taken by a Dirichlet series $f(s) = \sum_{n=1}^{\infty} a(n)e^{-s\lambda(n)}$ along the vertical line $\sigma = \sigma_0$, provided the series converges absolutely there.

This is the image of the function $t \mapsto f(\sigma_0 + it)$ as $t$ ranges over all real numbers.
Since $f(s)$ is obtained from its Bohr function $F(Z)$ by substituting $Z = \sigma B$, one always has the inclusion $V_f(\sigma_0) \subseteq U_f(\sigma_0)$.
The relationship between $V_f(\sigma_0)$ and $U_f(\sigma_0)$ is made precise by Theorem 8.9.
