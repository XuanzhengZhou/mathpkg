If $\omega_2/\omega_1$ is not real we define

$$J(\omega_1, \omega_2) = \frac{g_2^3(\omega_1, \omega_2)}{\Delta(\omega_1, \omega_2)}.$$

Since $g_2^3$ and $\Delta$ are homogeneous of the same degree we have $J(\lambda \omega_1, \lambda \omega_2) = J(\omega_1, \omega_2)$. In particular, if $\tau \in H$ we have

$$J(1, \tau) = J(\omega_1, \omega_2).$$

Thus $J(\omega_1, \omega_2)$ is a function of the ratio $\tau$ alone. We write $J(\tau)$ for $J(1, \tau)$.
