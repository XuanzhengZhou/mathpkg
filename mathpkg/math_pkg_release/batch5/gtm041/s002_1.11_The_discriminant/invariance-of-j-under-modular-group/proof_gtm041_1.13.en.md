If $\omega_1, \omega_2$ are given periods with nonreal ratio, introduce new periods $\omega_1', \omega_2'$ by the relations

$$\omega_2' = a\omega_2 + b\omega_1, \quad \omega_1' = c\omega_2 + d\omega_1,$$

where $a, b, c, d$ are integers such that $ad - bc = 1$. Then the pair $(\omega_1', \omega_2')$ is equivalent to $(\omega_1, \omega_2)$; that is, it generates the same set of periods $\Omega$. Therefore $g_2(\omega_1', \omega_2') = g_2(\omega_1, \omega_2)$ and $g_3(\omega_1', \omega_2') = g_3(\omega_1, \omega_2)$ since $g_2$ and $g_3$ depend only on the set of periods $\Omega$. Consequently, $\Delta(\omega_1', \omega_2') = \Delta(\omega_1, \omega_2)$ and $J(\omega_1', \omega_2') = J(\omega_1, \omega_2)$.

The ratio of the new periods is

$$\tau' = \frac{\omega_2'}{\omega_1'} = \frac{a\omega_2 + b\omega_1}{c\omega_2 + d\omega_1} = \frac{a\tau + b}{c\tau + d},$$

where $\tau = \omega_2/\omega_1$. Since $J(\omega_1, \omega_2)$ depends only on the ratio $\tau$, we obtain $J(\tau') = J(\tau)$ for all unimodular transformations. $\square$
