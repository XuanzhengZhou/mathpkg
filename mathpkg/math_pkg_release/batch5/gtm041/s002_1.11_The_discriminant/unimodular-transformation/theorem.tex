The equation

$$\tau' = \frac{a\tau + b}{c\tau + d}$$

is called a unimodular transformation if $a, b, c, d$ are integers with $ad - bc = 1$.

An easy calculation shows that

$$\text{Im}(\tau') = \text{Im}\left(\frac{a\tau + b}{c\tau + d}\right) = \frac{ad - bc}{|c\tau + d|^2} \text{Im}(\tau) = \frac{\text{Im}(\tau)}{|c\tau + d|^2}.$$

Hence $\tau' \in H$ if and only if $\tau \in H$.
