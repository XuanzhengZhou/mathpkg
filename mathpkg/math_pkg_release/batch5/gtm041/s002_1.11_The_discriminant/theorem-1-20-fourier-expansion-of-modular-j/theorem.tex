If $\tau \in H$ we have the Fourier expansion

$$12^3 J(\tau) = e^{-2\pi i\tau} + 744 + \sum_{n=1}^{\infty} c(n) e^{2\pi i n\tau}$$

where the coefficients $c(n)$ are integers. The first few values are:

$$c(0) = 744, \quad c(1) = 196884, \quad c(2) = 21493760, \quad c(3) = 864299970,$$
$$c(4) = 20245856256, \quad c(5) = 333202640600, \quad c(6) = 4252023300096,$$
$$c(7) = 44656994071935, \quad c(8) = 401490886656000.$$
