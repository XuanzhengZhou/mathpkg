Theorem 2.5 generalizes the zero-pole counting principle from Theorem 2.4: a non-constant modular function attains every complex value with the same multiplicity in the closure of the fundamental region. This is an immediate corollary of Theorem 2.4 applied to the function $f - c$, which is also modular. The result implies that modular functions are unconstrained in their values — they are as "surjective" on the Riemann sphere as meromorphic functions can be, constrained only by their modular invariance.
