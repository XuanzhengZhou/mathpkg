Euler's partition generating function expresses the ordinary generating function of $p(n)$ as an infinite product over all positive integers. This identity is the cornerstone of the analytic theory of partitions: it connects the combinatorial object $p(n)$ to the Dedekind eta function $\eta(\tau)$ via $F(e^{2\pi i\tau}) = e^{\pi i\tau/12}/\eta(\tau)$, which makes modular methods available for the study of $p(n)$. The proof via expanding each factor as a geometric series and collecting coefficients is a classic combinatorial argument found in Chapter 14 of Apostol's Introduction to Analytic Number Theory.
