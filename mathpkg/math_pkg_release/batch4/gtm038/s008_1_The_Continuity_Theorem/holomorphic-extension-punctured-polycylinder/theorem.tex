Let $n \geqslant 2$ and let $P := \{ z \in \mathbb{C}^n : |z_\nu| < 1 \text{ for } \nu = 1, \ldots, n \}$ be the unit polycylinder. Let $0 \leqslant r_\nu^0 < 1$ for $\nu = 1, \ldots, n$ and set
$$\bar{P}_{r^0} := \{ z \in \mathbb{C}^n : |z_\nu| \leqslant r_\nu^0 \text{ for all } \nu \}.$$
Let $G := P \setminus \bar{P}_{r^0}$. Then every holomorphic function $f$ on $G$ can be extended uniquely to a function holomorphic on $P$.
