Let $X$ be a paracompact complex manifold. A test function on $X$ is an arbitrarily often differentiable function $t: X \to \mathbb{R}$ with compact support.

Let the ring (without $1$) of all test functions be denoted by $\mathcal{T}$. Let $\mathcal{T}$ be the sheaf of germs of test functions on $X$.
