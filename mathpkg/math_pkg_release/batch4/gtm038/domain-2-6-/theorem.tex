For a domain $G \subset \mathbb{C}^n$ with $C^2$ boundary the Levi condition is satisfied at a point $\mathfrak{z}_0 \in \partial G$ if there is an open neighborhood $U = U(\mathfrak{z}_0)$ and a function $\varphi$ on $U$ which satisfies the conditions required by Def. 2.4 so that at $\mathfrak{z}_0 \varphi$ satisfies the Levi condition.
