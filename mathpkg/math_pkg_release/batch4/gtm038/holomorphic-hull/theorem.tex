Let $\mathfrak{G} = (G, \pi, x_0)$ be a domain over $\mathbb{C}^n$, $\mathcal{F}$ a non-empty set of holomorphic functions on $G$. Then the holomorphic hull $H_{\mathcal{F}}(\mathfrak{G}) = (\hat{G}, \hat{\pi}, \hat{x})$ satisfies: $\mathfrak{G} < H_{\mathcal{F}}(\mathfrak{G})$, each $f \in \mathcal{F}$ extends uniquely to $\hat{G}$, and $H_{\mathcal{F}}(\mathfrak{G})$ is maximal with this property.
