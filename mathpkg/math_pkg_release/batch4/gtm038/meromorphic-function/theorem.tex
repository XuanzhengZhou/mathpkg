A meromorphic function on a complex manifold $X$ is a pair $(A, f)$ where $A \subset X$ is a nowhere dense analytic subset and $f$ is a holomorphic function on $X - A$ such that for each $x_0 \in A$ there is a neighborhood $U$ and holomorphic functions $g, h$ on $U$ with $h \not\equiv 0$ and $f = g/h$ on $U - A$.
