1. If $\varphi$ is of type $(p, q)$, then $\bar{\varphi}$ is of type $(q, p)$.
2. If $\varphi$, $\psi$ are of type $(p, q)$, $c \in \mathbb{C}$, then $\varphi + \psi$ and $c \cdot \varphi$ are of type $(p, q)$.
3. If $\varphi$ is of type $(p, q)$, $\psi$ of type $(p', q')$, then $\varphi \wedge \psi$ is of type $(p + p', q + q')$.
