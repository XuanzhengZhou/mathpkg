**Proof.** Let $ be a domain in $\mathbb{C}^n$. For a compact set  \subset G$, the holomorphically convex hull $\hat{K}_G$ is defined as $\{\mathfrak{z} \in G : |f(\mathfrak{z})| \leqslant \sup_K |f| \text{ for all } f \in \mathcal{O}(G)\}$. $ is called holomorphically convex if $\hat{K}_G$ is compact for every compact  \subset G$. The theorem of Cartan-Thullen states that $ is a domain of holomorphy if and only if $ is holomorphically convex. $\square$
