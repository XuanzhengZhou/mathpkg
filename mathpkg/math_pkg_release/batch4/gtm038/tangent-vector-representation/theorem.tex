If $c_1, \ldots, c_n$ are arbitrary complex numbers then there exists exactly one tangent vector $D$ with $D(f) = \sum_{v=1}^{n} c_v \frac{\partial f}{\partial z_v}(x_0) + \overline{c}_v \frac{\partial f}{\partial \overline{z}_v}(x_0)$ for each holomorphic function $f$. A tangent vector is uniquely determined by its values on holomorphic functions.
