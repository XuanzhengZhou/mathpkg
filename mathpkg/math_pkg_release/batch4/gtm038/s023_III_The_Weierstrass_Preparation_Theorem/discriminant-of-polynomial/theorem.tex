Let $f(X) = \prod_{\rho=1}^{s} (X - X_\rho) \in \mathbb{C}[X]$ and let
$$D(X_1, \ldots, X_s) := \prod_{1 \leq \nu < \mu \leq s} (X_\nu - X_\mu)^2.$$
By the fundamental theorem of symmetric polynomials, there is a unique polynomial $Q(Y_1, \ldots, Y_s)$ with integer coefficients such that $D(X_1, \ldots, X_s) = Q(\sigma_1, \ldots, \sigma_s)$. If $f(X) = X^s + a_1 X^{s-1} + \cdots + a_s$, then the discriminant of $f$ is
$$\Delta(f) := Q(a_1, \ldots, a_s).$$
