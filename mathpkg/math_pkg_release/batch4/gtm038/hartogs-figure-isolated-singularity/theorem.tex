Let $n \geqslant 2$, $P := \{ z : |z_v| < 1 \}$ be the unit polycylinder, $0 \leqslant r_v^0 < 1$ for $v = 1, \ldots, n$, $\overline{P}_{r_0} := \{ z : |z_v| \leqslant r_v^0 \text{ for all } v\}$ and $G := P - \overline{P}_{r_0}$. Then every holomorphic function $f$ on $G$ can be extended uniquely to a function holomorphic on $P$.
