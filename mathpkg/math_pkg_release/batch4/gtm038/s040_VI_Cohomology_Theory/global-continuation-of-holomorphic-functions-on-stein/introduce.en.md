Every holomorphic function defined on an analytic subset $A$ of a Stein manifold $X$ can be extended to a global holomorphic function on all of $X$. This is a powerful global continuation result and a fundamental property of Stein manifolds. The proof constructs local extensions via the ideal sheaf $\mathcal{I}(A)$ of the analytic subset, then uses Cartan's Theorem B to glue these local data into a global section.
