The functor $\Gamma : \mathcal{S} \mapsto \Gamma(X, \mathcal{S})$, $\varphi \mapsto \varphi_*$, is a left-exact functor; that is, if
$$\mathcal{O} \rightarrow \mathcal{S}' \xrightarrow{\varphi} \mathcal{S} \xrightarrow{\psi} \mathcal{S}'' \rightarrow \mathcal{O}$$
is exact, then so is
$$0 \rightarrow \Gamma(X, \mathcal{S}') \xrightarrow{\varphi_*} \Gamma(X, \mathcal{S}) \xrightarrow{\psi_*} \Gamma(X, \mathcal{S}'')$$
exact.
