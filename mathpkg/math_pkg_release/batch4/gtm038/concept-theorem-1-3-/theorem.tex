Let $n \geqslant 2$, $P := \{ \tilde{3} | | \tilde{3}| < 1 \}$ be the unit polycylinder, $0 \leqslant r_v^0 < 1$ for $v = 1, \ldots, n$, $\bar{P}_{r_0} := \{ \tilde{3} | | z_v| \leqslant r_v^0 \text{ for all } v \}$ and $G := P - \bar{P}_{r_0}$. Then every holomorphic function $f$ on $G$ can be extended uniquely to a function holomorphic on $P$.
