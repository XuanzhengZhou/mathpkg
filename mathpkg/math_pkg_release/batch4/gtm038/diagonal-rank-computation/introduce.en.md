To establish that the diagonal $D \subset X \times X$ is a complex submanifold, one considers the holomorphic functions $(x,y) \mapsto z_i(x) - w_i(y)$ defined on $X \times X$ using local coordinates. At any point on the diagonal, the rank of the Jacobian of these $n$ functions equals $n$, showing that the diagonal is a regular analytic subset of codimension $n$. This lemma provides the key technical computation needed for Theorem 3.6, which asserts that the diagonal mapping $d: X \to D$ is biholomorphic.
