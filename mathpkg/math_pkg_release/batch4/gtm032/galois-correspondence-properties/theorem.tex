Let $P$ be a field. For a subgroup $G \subseteq \operatorname{Aut}(P)$, let $I(G) = \{\alpha \in P \mid \alpha^s = \alpha, \forall s \in G\}$. For a subfield $\Phi \subseteq P$, let $A(\Phi) = \{s \in \operatorname{Aut}(P) \mid \alpha^s = \alpha, \forall \alpha \in \Phi\}$. Then:

$$(\alpha)\quad G_1 \supseteq G_2 \implies I(G_1) \subseteq I(G_2)$$
$$(\beta)\quad \Phi_1 \supseteq \Phi_2 \implies A(\Phi_1) \subseteq A(\Phi_2)$$
$$(\gamma)\quad G \subseteq A(I(G))$$
$$(\delta)\quad \Phi \subseteq I(A(\Phi))$$
$$(\epsilon)\quad I(A(I(G))) = I(G)$$
$$(\eta)\quad A(I(A(\Phi))) = A(\Phi)$$

A subfield $\Phi$ is Galois in $P$ if and only if $\Phi = I(A(\Phi))$.
