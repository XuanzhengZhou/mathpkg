An ordered abelian group is an abelian group $\Gamma$ with a total order compatible with addition: $\alpha \leq \beta \implies \alpha + \gamma \leq \beta + \gamma$. A valuation $v: K^* \to \Gamma$ (where $\Gamma$ is an ordered group) satisfies $v(ab) = v(a) + v(b)$ and $v(a+b) \geq \min(v(a), v(b))$. The image $v(K^*)$ is the value group of the valuation.
