Let $\Phi$ be of characteristic $p \neq 0$. Then:
\begin{enumerate}
\item[(i)] An algebraic element $\rho$ is separable over $\Phi$ if and only if
  $$\Phi(\rho) = \Phi(\rho^p) = \Phi(\rho^{p^2}) = \cdots.$$
\item[(ii)] If $\rho$ is purely inseparable over $\Phi$, then its minimum polynomial has the form $x^{p^e} - \alpha$ with $\alpha \in \Phi$. Conversely, if $\rho$ satisfies $x^{p^e} = \alpha \in \Phi$ for some $e \geq 0$, then $\rho$ is purely inseparable over $\Phi$.
\end{enumerate}
