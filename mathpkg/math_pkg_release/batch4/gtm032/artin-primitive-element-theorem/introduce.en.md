Theorem 15 (Artin): A finite extension has a primitive element precisely when it has only finitely many intermediate subfields. In a simple extension $\Phi(\theta)$, each intermediate field is generated by coefficients of the minimum polynomial of $\theta$ over that field, which must divide the fixed polynomial $f(x)$, yielding finiteness. Conversely, finiteness of intermediate fields allows constructing a primitive element by considering fields $\Phi(\xi + \alpha\eta)$.
