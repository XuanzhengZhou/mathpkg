Elements $x_1, x_2, \ldots, x_r$ belonging to a commutative ring are called \text{indeterminates} relative to a subring $\mathfrak{A}$ if a relation
$$\sum_{m_i} a_{m_1 \cdots m_r} x_1^{m_1} \cdots x_r^{m_r} = 0, \quad a_{m_1 \cdots m_r} \in \mathfrak{A},$$
where $(m_1, m_2, \ldots, m_r)$ is an $r$-tuple of non-negative integers, can hold only if every $a_{m_1 \cdots m_r} = 0$.
