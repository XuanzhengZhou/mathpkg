Let $\{E_1,E_2,\ldots\}$ generate $\mathcal{B}_0$. Define $f: X \to \{0,1\}^\infty$ by $f(x) = (\chi_{E_1}(x),\ldots)$. $f$ is 1-1 and Borel with respect to both $\mathcal{B}$ and $\mathcal{B}_0$. By 3.3.4, $f(X)$ is analytic; by 3.3.2 applied to $f^{-1}$ (which is Borel because $\mathcal{B}_0 \subseteq \mathcal{B}$), we get $\mathcal{B} = f^{-1}(\text{Borel}) = \mathcal{B}_0$. $\square$
