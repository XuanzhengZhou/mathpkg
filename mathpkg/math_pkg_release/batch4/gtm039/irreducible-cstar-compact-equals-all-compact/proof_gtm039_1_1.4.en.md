By 1.4.1, $\mathcal{A}$ contains a minimal projection $E$. We show $E$ has rank 1. If $\xi,\eta \in E\mathcal{H}$ with $\eta \perp \xi$, then for any $T \in \mathcal{A}$, $ETE = \lambda E$, so $(\eta, T\xi) = \bar{\lambda}(\eta,\xi) = 0$, giving $\eta \perp [\mathcal{A}\xi] = \mathcal{H}$, so $\eta = 0$. Thus $E$ has rank 1. Now any rank-1 projection $F$ with unit vector $f$ can be approximated: approximate $f$ by $T_n e$ with $T_n \in \mathcal{A}$. Then $T_nET_n^* \to F$, so $F \in \mathcal{A}$. Thus $\mathcal{A}$ contains all finite-rank operators, and by closure equals $\mathcal{C}(\mathcal{H})$. $\square$
