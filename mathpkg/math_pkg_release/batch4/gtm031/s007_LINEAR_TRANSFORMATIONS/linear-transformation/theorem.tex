Let $\mathfrak{R}_1$ and $\mathfrak{R}_2$ be vector spaces over the same division ring $\Delta$. A mapping $A$ of $\mathfrak{R}_1$ into $\mathfrak{R}_2$ is called a \text{linear transformation} if

$$
(x + y)A = xA + yA, \quad (\alpha x)A = \alpha(xA)
$$

for all $x, y$ in $\mathfrak{R}_1$ and all $\alpha$ in $\Delta$. The image of an element $x$ under $A$ is denoted $xA$. The image set $\mathfrak{R}_1 A = \{ xA \mid x \in \mathfrak{R}_1 \}$ is a subset of $\mathfrak{R}_2$ (possibly proper).
