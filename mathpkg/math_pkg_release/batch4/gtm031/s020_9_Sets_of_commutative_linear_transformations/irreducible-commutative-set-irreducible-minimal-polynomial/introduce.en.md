When a vector space is irreducible under the action of a commutative set of linear transformations (meaning it has no proper non-zero invariant subspaces), every transformation in the set must have an irreducible minimum polynomial. This result complements the indecomposable case: while indecomposability forces the minimum polynomial to be a prime power, irreducibility forces it to be a prime (irreducible) polynomial itself. Together these lemmas set the stage for the structure theorem on simultaneous triangularization.
