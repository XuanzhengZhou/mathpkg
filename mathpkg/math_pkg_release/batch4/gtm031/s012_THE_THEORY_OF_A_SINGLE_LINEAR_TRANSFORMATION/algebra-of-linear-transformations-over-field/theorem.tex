Let $\mathfrak{R}$ be a vector space over a field $\Phi$ and let $\mathfrak{L} \equiv \mathfrak{L}(\mathfrak{R}, \mathfrak{R})$ be the set of linear transformations of $\mathfrak{R}$ into itself. Then $\mathfrak{L}$ can be regarded as an algebra over $\Phi$:
\begin{enumerate}
\item $\mathfrak{L}$ is a ring;
\item $\mathfrak{L}$ is a vector space over $\Phi$ with addition the same as the ring addition;
\item $$\gamma(AB) = (\gamma A)B = A(\gamma B)$$ holds for all $A, B \in \mathfrak{L}$ and $\gamma \in \Phi$, where $\gamma A$ is the resultant $\gamma_l A = A\gamma_l$.
\end{enumerate}
Moreover, if $\mathfrak{R}$ is $n$-dimensional over $\Phi$, then $\mathfrak{L}$ is $n^2$-dimensional over $\Phi$.
