Let $\Re$ be a Euclidean space and let $(e_1, e_2, \cdots, e_n)$ be any basis for $\Re$. Then there exists a Cartesian basis $(u_1, u_2, \cdots, u_n)$ for $\Re$ constructed by the following inductive procedure (called \textbf{E. Schmidt's orthogonalization process}):

Set $u_1 = (e_1, e_1)^{-1/2} e_1$, so that $(u_1, u_1) = 1$.

Suppose $(u_1, u_2, \cdots, u_k)$ have been determined such that $(u_i, u_j) = \delta_{ij}$ and $[u_1, \ldots, u_i] = [e_1, \ldots, e_i]$ for $i \leq k$. Define

$$f_{k+1} = e_{k+1} - \sum_{i=1}^{k} (e_{k+1}, u_i) u_i.$$

Then $(f_{k+1}, u_i) = 0$ for $i = 1, \ldots, k$. Set

$$u_{k+1} = (f_{k+1}, f_{k+1})^{-1/2} f_{k+1}.$$

Then $(u_1, \ldots, u_{k+1})$ satisfy $(u_i, u_j) = \delta_{ij}$ and $[u_1, \ldots, u_i] = [e_1, \ldots, e_i]$ for $i \leq k+1$. Repeating this process yields a Cartesian basis $(u_1, u_2, \cdots, u_n)$ for $\Re$.
