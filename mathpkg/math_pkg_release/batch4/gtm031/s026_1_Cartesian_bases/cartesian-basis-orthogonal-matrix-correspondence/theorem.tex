Let $(v_1, v_2, \cdots, v_n)$ be a fixed Cartesian basis for the Euclidean space $\Re$.

If $(u_1, u_2, \cdots, u_n)$ is any Cartesian basis and $u_i = \sum_{j=1}^n \sigma_{ij} v_j$, then the transition matrix $(\sigma)$ is an orthogonal matrix, i.e., $(\sigma)(\sigma)' = 1$.

Conversely, if $(\sigma)$ is any orthogonal matrix and $u_i = \sum_{j=1}^n \sigma_{ij} v_j$, then $(u_1, u_2, \cdots, u_n)$ is a Cartesian basis.

Thus there is a one-to-one correspondence between the set of Cartesian bases of $\Re$ and the set of $n \times n$ orthogonal matrices over $\Phi$.
