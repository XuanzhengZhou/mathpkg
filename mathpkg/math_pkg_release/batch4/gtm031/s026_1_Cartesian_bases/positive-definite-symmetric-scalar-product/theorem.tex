A \textbf{positive definite symmetric scalar product} on a real vector space is a real-valued function $(x, y)$ satisfying:

\begin{enumerate}
\item $(x_1 + x_2, y) = (x_1, y) + (x_2, y)$
\item $(x, y_1 + y_2) = (x, y_1) + (x, y_2)$
\item $(\alpha x, y) = \alpha(x, y) = (x, \alpha y)$
\item $(x, y) = (y, x)$
\item $(x, x) > 0$ if $x \neq 0$.
\end{enumerate}
