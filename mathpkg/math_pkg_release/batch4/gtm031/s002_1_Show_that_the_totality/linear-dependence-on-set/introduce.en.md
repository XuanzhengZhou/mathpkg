A vector $x$ is linearly dependent on a set $S$ if it lies in the subspace (or submodule) generated by $S$, i.e., $x$ can be written as a finite linear combination of elements of $S$. This notion satisfies natural transitivity and finiteness properties that are used throughout linear algebra.
