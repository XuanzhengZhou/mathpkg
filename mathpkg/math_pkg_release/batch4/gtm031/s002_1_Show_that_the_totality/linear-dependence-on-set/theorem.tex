A vector $x$ is \textbf{linearly dependent on} a set of vectors $S$ if $x \in [S]$. This is equivalent to saying that $x = \sum_{i=1}^{m} \xi_i x_i$ for suitable $\xi_i \in \Delta$ and suitable $x_i \in S$.

Properties of linear dependence:
\begin{enumerate}
\item If $x$ is linearly dependent on a set $S$, then $x$ is linearly dependent on a finite subset of $S$.
\item $x$ is linearly dependent on the set $S = (x)$.
\item If $x$ is linearly dependent on $S$ and $T$ is a set containing $S$, then $x$ is linearly dependent on $T$.
\item If $x$ is linearly dependent on $S$ and if every $x_\alpha \in S$ is linearly dependent on the set $T$, then $x$ is linearly dependent on $T$.
\end{enumerate}
