A set $S$ of elements of a module $\mathfrak{R}$ is called a set of generators (or a spanning set) if every element of $\mathfrak{R}$ can be expressed as a finite linear combination of elements of $S$ with coefficients in the ring $\mathfrak{o}$. In other words, $[S] = \mathfrak{R}$.
