A set of vectors is linearly independent if the only linear combination yielding the zero vector is the trivial one (all coefficients zero). Linear independence captures the notion of "no redundancy" in a set of vectors. Key properties include: subsets of independent sets remain independent, and every non-zero vector forms a singleton independent set. Linear independence is the essential property that distinguishes a basis from a mere generating set.
