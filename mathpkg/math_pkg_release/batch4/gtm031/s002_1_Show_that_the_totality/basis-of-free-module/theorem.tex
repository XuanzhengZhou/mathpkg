A set $S$ is called a \textbf{basis} of the module $\mathfrak{R}$ if $S$ is a set of generators for $\mathfrak{R}$ and every element $x \in \mathfrak{R}$ can be expressed in one and only one way as a finite linear combination $\sum \xi_i e_{\alpha_i}$ with $e_{\alpha_i} \in S$ and coefficients $\xi_i$ in the ring.
