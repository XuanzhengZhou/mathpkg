If a set $\Omega$ of linear transformations is reducible, i.e., admits a proper nonzero invariant subspace, then this reducibility is reflected in the matrix representation of the transformations. Specifically, one can choose a basis adapted to the invariant subspace so that every matrix of $\Omega$ simultaneously takes a block upper-triangular (reduced) form with a zero block in the upper right corner. This provides a practical algebraic criterion for detecting reducibility.
