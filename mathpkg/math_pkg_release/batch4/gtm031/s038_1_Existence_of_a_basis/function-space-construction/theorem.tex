Let $I$ be any set and $\Delta$ a division ring. Define $\mathfrak{R}$ to be the set of all functions $x: I \to \Delta$ with finite support (i.e., $x(i) \neq 0$ for only finitely many $i \in I$). For $x, y \in \mathfrak{R}$ define $(x + y)(i) = x(i) + y(i)$ and for $\alpha \in \Delta$ define $(x\alpha)(i) = x(i)\alpha$. Then $\mathfrak{R}$ is a right vector space over $\Delta$. The functions $e_i$ defined by $e_i(j) = \delta_{ij}$ form a basis whose cardinal number equals $|I|$. Thus for every cardinal number there exists a vector space with a basis of that cardinality.
