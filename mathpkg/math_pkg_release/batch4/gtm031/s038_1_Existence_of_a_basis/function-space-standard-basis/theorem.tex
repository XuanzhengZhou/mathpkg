Let $\Delta$ be a division ring and $I$ be any set. Let $\Re$ be the set of functions from $I$ to $\Delta$ which vanish at all but a finite number of elements of $I$. For each $i \in I$, define $e_i$ to be the vector such that
$$e_i(j) = \delta_{ij}.$$
Then the set $\{e_i\}_{i \in I}$ is a basis of $\Re$, and the cardinal number of this basis equals the cardinal number of $I$.
