Let $(\alpha)$ be an $r \times n$ matrix with elements $\alpha_{ij} \in \Delta$ (where $\Delta$ is a division ring), and let $(\beta)$ be an $n \times m$ matrix with elements $\beta_{jk} \in \Delta$. The product $(\alpha)(\beta)$ is the $r \times m$ matrix whose $(i,k)$-entry is

$$\sigma_{ik} = \alpha_{i1}\beta_{1k} + \alpha_{i2}\beta_{2k} + \cdots + \alpha_{in}\beta_{nk}.$$

If $(\gamma)$ is an $m \times q$ matrix with elements in $\Delta$, then the associative law holds:

$$[(\alpha)(\beta)](\gamma) = (\alpha)[(\beta)(\gamma)].$$
