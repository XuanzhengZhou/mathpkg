Let $\Omega$ be a set of linear transformations on a finite-dimensional vector space $\mathfrak{R}$ over a field. A **composition series** for $\Omega$ is a chain of invariant subspaces
$$0 = \mathfrak{S}_0 \subset \mathfrak{S}_1 \subset \cdots \subset \mathfrak{S}_t = \mathfrak{R}$$
such that for each $i = 1, \dots, t$:
\begin{enumerate}
\item $\mathfrak{S}_i$ is an invariant subspace of $\mathfrak{R}$ relative to $\Omega$, and
\item $\mathfrak{S}_i$ is irreducible over $\mathfrak{S}_{i-1}$ (i.e., the factor space $\mathfrak{S}_i/\mathfrak{S}_{i-1}$ has no proper nonzero invariant subspace relative to the set of induced linear transformations $\bar{\Omega}$).
\end{enumerate}

If $\mathfrak{R}$ itself is irreducible, then $0 \subset \mathfrak{R}$ is a composition series.
