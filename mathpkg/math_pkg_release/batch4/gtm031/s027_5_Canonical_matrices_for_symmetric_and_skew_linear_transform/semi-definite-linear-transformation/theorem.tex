A symmetric linear transformation $A$ on a Euclidean space is called \textbf{semi-definite} (or non-negative semi-definite) if
$$(xA, x) \geq 0 \quad \text{for all } x.$$

For any linear transformation $A$, the transformation $B = AA'$ is semi-definite since
$$(xB, x) = (xAA', x) = (xA, xA) \geq 0.$$
Moreover, $B$ and $A$ have the same null-space; consequently $B$ is positive definite if and only if $A$ is 1-1.
