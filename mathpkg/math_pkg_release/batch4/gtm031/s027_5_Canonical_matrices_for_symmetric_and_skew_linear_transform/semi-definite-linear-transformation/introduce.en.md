A semi-definite (or non-negative) linear transformation is a symmetric transformation whose associated quadratic form is everywhere non-negative. Unlike positive definite transformations, semi-definite transformations may have a nontrivial kernel. The canonical example is $AA'$ for any linear transformation $A$, which is always semi-definite and shares the same null-space as $A$, so it is positive definite precisely when $A$ is injective.
