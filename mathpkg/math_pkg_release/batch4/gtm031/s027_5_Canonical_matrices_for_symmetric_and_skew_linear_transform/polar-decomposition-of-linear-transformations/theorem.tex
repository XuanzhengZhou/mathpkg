Any linear transformation $A$ on a Euclidean space can be written as
$$A = PO$$
where $P$ is a semi-definite transformation and $O$ is an orthogonal transformation. The transformation $P$ is uniquely determined, while $O$ is unique if and only if $A$ is 1-1.

In the 1-1 case, $P = \sqrt{AA'}$ is positive definite and $O = P^{-1}A$ is orthogonal.
