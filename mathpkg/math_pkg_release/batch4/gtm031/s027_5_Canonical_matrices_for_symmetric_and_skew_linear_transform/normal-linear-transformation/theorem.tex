A linear transformation $A$ on a Euclidean space is called \textbf{normal} if it commutes with its transpose:
$$AA' = A'A.$$

Every linear transformation $A$ can be uniquely written as $A = B + C$ where
$$B = \frac{1}{2}(A + A') \quad \text{(symmetric part)}, \qquad C = \frac{1}{2}(A - A') \quad \text{(skew part)}.$$
Then $A$ is normal if and only if $B$ and $C$ commute. Special cases of normal transformations include symmetric, skew, and orthogonal linear transformations.
