If $A$ is a normal linear transformation, one can find a unitary basis for the vector space such that the matrix of $A$ has the canonical form

$$\operatorname{diag}\{ \rho_1, \rho_2, \cdots, \rho_n \}.$$

Then if $S(\lambda) = \gamma_0 + \gamma_1\lambda + \gamma_2\lambda^2 + \cdots$ is a power series with radius of convergence $r > |\rho_i|$ for $i = 1, 2, \cdots, n$, the linear transformation $S(A)$ is defined and has the matrix

$$\operatorname{diag}\{S(\rho_1), S(\rho_2), \cdots, S(\rho_n)\}$$

relative to the given unitary basis. Since $S(A)$ has a diagonal matrix relative to a unitary basis, this transformation is normal.
