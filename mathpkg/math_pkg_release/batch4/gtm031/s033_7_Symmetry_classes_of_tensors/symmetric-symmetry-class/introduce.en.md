The symmetric symmetry class is determined by the set of operators $\{P(\sigma) - 1 \mid \sigma \in \mathfrak{S}_r\}$. A tensor in this class is invariant under all permutations of its factors -- it is fully symmetric. This class is one of the two fundamental symmetry classes alongside the skew-symmetric (alternating) class.
