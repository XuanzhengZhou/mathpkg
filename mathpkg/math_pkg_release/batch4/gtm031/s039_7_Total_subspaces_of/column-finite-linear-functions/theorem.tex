\textbf{Definition.} The subset $\mathfrak{R}_0^*$ of $\mathfrak{R}^*$ consists of linear functions $f$ whose matrices relative to the basis $(e_i)$ are \textit{column finite} in the sense that $f(e_i) \neq 0$ for only a finite number of $i$. This collection constitutes a subspace of the conjugate space. For each $i$ we define a linear function $e_i^*$ by
$$e_i^*(e_k) = \delta_{ik}.$$
Thus $e_i^*$ is the linear function whose matrix has a $1$ in the $i$ place and $0$'s elsewhere. The $e_i^*$ form a basis for $\mathfrak{R}_0^*$. Consequently $\dim \mathfrak{R}_0^* = \dim \mathfrak{R}$.
