Let $\Delta$ be a division ring. A \emph{vector space} (or \emph{linear space}) over $\Delta$ consists of an abelian group $\mathfrak{R}$ under an operation of addition and a multiplication of elements of $\mathfrak{R}$ by elements of $\Delta$ on the left (scalar multiplication) satisfying the following axioms:

\textbf{A1.} $(x + y) + z = x + (y + z)$ for all $x, y, z \in \mathfrak{R}$.

\textbf{A2.} $x + y = y + x$ for all $x, y \in \mathfrak{R}$.

\textbf{A3.} There exists an element $0 \in \mathfrak{R}$ such that $x + 0 = x$ for all $x \in \mathfrak{R}$.

\textbf{A4.} For any $x \in \mathfrak{R}$ there exists an element $-x \in \mathfrak{R}$ such that $x + (-x) = 0$.

\textbf{S1.} $\alpha(x + y) = \alpha x + \alpha y$ for all $\alpha \in \Delta$, $x, y \in \mathfrak{R}$.

\textbf{S2.} $(\alpha + \beta)x = \alpha x + \beta x$ for all $\alpha, \beta \in \Delta$, $x \in \mathfrak{R}$.

\textbf{S3.} $(\alpha\beta)x = \alpha(\beta x)$ for all $\alpha, \beta \in \Delta$, $x \in \mathfrak{R}$.

\textbf{S4.} $1x = x$ for all $x \in \mathfrak{R}$, where $1$ is the identity element of $\Delta$.
