Let $E_1, E_2, \ldots, E_r$ be a finite set of orthogonal projections (so $E_i^2 = E_i$ and $E_i E_j = 0 = E_j E_i$ for $i \neq j$). Set

$$E = \sum_{i=1}^r E_i, \quad E_{r+1} = 1 - E.$$

Then $\{E_1, E_2, \ldots, E_r, E_{r+1}\}$ is a supplementary orthogonal set of projections, i.e.,

$$E_i^2 = E_i, \quad E_i E_j = 0 \text{ for } i \neq j, \quad \sum_{i=1}^{r+1} E_i = 1.$$
