Let $\sum_{i=1}^{n} \xi_i \alpha_{ij} = 0$, $j = 1, 2, \dots, m$, be a left-handed system of homogeneous equations and let the row rank of $(\alpha)$ be $\rho$. Then there exist $n - \rho$ linearly independent solutions $(\beta_1^{(k)}, \beta_2^{(k)}, \dots, \beta_n^{(k)})$, $k = 1, 2, \dots, n - \rho$, such that any solution of the system is a left linear combination of these solutions.
