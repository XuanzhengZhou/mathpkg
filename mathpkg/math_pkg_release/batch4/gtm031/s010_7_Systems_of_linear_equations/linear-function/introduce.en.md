A linear function (or linear functional) on a left vector space $\mathfrak{R}$ over a division ring $\Delta$ is a linear transformation from $\mathfrak{R}$ into the one-dimensional vector space $\Delta$. Linear functions form the foundation of duality theory: the set of all linear functions on $\mathfrak{R}$ constitutes the conjugate (dual) space $\mathfrak{R}^*$.
