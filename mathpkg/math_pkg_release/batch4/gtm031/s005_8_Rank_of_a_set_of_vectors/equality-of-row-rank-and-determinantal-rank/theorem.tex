Let $(\alpha)$ be an $r \times n$ matrix with elements in a division ring $\Delta$, and let $x_1, x_2, \cdots, x_\rho$ form a basis for the set of row vectors $x_i = \sum_{j=1}^{n} \alpha_{ij} e_j$, $i = 1, 2, \cdots, r$. Then the determinantal rank of $(\alpha)$ equals the row rank $\rho$. Moreover, applying the same arguments to right vector spaces shows that the column rank and the determinantal rank are also equal.
