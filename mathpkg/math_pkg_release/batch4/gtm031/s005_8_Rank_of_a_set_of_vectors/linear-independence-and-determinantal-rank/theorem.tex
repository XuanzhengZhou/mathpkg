Let $(\alpha)$ be an $r \times n$ matrix with elements in a division ring $\Delta$, and let $(e_1, e_2, \cdots, e_n)$ be an ordered basis for the vector space $\Re$. Define the vectors
$$
x_i = \sum_{j=1}^{n} \alpha_{ij} e_j, \quad i = 1, 2, \cdots, r.
$$
The vectors $x_1, x_2, \cdots, x_r$ are linearly independent if and only if $(\alpha)$ is of determinantal rank $r$.
