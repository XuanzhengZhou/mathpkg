We have shown that the determinantal rank equals the row rank $\rho$. If we apply the same arguments to right vector spaces, we can show that the column rank and the determinantal rank are equal. Therefore, the row rank equals the column rank (both being equal to the determinantal rank). This holds in particular in the commutative case, where row rank and column rank coincide.
