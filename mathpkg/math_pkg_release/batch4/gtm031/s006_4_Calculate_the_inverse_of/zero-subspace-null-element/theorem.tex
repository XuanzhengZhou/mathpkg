In the lattice $L$ of all subspaces of a vector space $\mathfrak{R}$, the subspace consisting of the zero vector only, denoted $0$, acts as the null element. Specifically, for every subspace $\mathfrak{S}$:
$$
\mathfrak{S} + 0 = \mathfrak{S} = 0 + \mathfrak{S}
$$
and
$$
\mathfrak{S} \cap 0 = 0 = 0 \cap \mathfrak{S}.
$$
