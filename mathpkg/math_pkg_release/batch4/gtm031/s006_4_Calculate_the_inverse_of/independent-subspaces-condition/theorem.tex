Let $\mathfrak{S}_1, \mathfrak{S}_2, \cdots, \mathfrak{S}_r$ be subspaces of a vector space $\mathfrak{R}$. A necessary and sufficient condition that these subspaces be independent is that for each $i = 1, 2, \cdots, r$,
$$
\mathfrak{S}_i \cap (\mathfrak{S}_1 + \cdots + \mathfrak{S}_{i-1} + \mathfrak{S}_{i+1} + \cdots + \mathfrak{S}_r) = 0.
$$
That is, each $\mathfrak{S}_i$ intersects the sum of all the other subspaces only in the zero vector. When this condition holds, every representation of a vector $y \in \sum \mathfrak{S}_i$ as $y = y_1 + y_2 + \cdots + y_r$ with $y_i \in \mathfrak{S}_i$ is unique.
