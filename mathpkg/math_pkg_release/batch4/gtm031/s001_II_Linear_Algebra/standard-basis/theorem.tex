The \emph{standard basis} of $\Delta^{(n)}$ consists of the vectors

$$e_i = (0, 0, \cdots, 0, 1, 0, \cdots, 0), \qquad i = 1, 2, \cdots, n,$$

where the $1$ (the identity element of $\Delta$) appears in the $i$-th position and $0$'s appear elsewhere.

Every vector $x = (\xi_1, \xi_2, \cdots, \xi_n) \in \Delta^{(n)}$ can be expressed uniquely as the linear combination

$$x = \sum_{i=1}^{n} \xi_i e_i = \xi_1 e_1 + \xi_2 e_2 + \cdots + \xi_n e_n.$$

Consequently, $\{e_1, e_2, \cdots, e_n\}$ is a finite basis for $\Delta^{(n)}$, verifying that $\Delta^{(n)}$ satisfies axiom F and is therefore finite dimensional.
