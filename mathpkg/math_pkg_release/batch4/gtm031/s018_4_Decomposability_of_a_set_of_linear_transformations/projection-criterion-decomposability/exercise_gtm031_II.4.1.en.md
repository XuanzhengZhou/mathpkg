Show that, if $\mathfrak{R}$ is irreducible (indecomposable) relative to a subset of $\Omega$, then it is irreducible (indecomposable) relative to $\Omega$. Use this to prove that the set of linear transformations corresponding to the set of triangular matrices is an indecomposable set.
