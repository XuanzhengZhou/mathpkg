A vector space $\mathfrak{R}$ is called \text{decomposable} relative to a set $\Omega$ of linear transformations if $\mathfrak{R}$ admits a proper decomposition $\mathfrak{R} = \mathfrak{R}_1 \oplus \mathfrak{R}_2 \oplus \cdots \oplus \mathfrak{R}_s$ where each $\mathfrak{R}_i \neq 0$, $s > 1$, and each $\mathfrak{R}_i$ is invariant under $\Omega$. In this case $\Omega$ is called a \text{decomposable set} of linear transformations.
