Let $\Omega$ be an arbitrary set of linear transformations and let $E$ be any projection such that $EA = EAE$ holds for every $A$ in $\Omega$. As in $\S 2$ define the linear transformation $A_E$ of $\overline{\mathfrak{R}} = \mathfrak{R} / \mathfrak{R}E$ into $\mathfrak{R}E$ by $\bar{x} A_E = x(AE - EA)$. Prove that, if there exists a linear transformation $D$ of $\overline{\mathfrak{R}}$ into $\mathfrak{R}E$ such that $A_E = \bar{A} D - DA$ holds for all $A$, then $\mathfrak{R}$ has a complementary invariant subspace.
