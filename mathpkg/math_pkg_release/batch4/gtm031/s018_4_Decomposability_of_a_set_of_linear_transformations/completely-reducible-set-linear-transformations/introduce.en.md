A set $\Omega$ of linear transformations is completely reducible if every $\Omega$-invariant subspace admits a complementary $\Omega$-invariant subspace. This is a strong condition on the lattice $L_{\Omega}$ of invariant subspaces. While it always holds when $\Omega$ consists only of the identity (since then $L_{\Omega}$ is the full subspace lattice $L$), and also when $\Omega$ is irreducible, the general case is nontrivial. Remarkably, this potentially infinite set of conditions can be replaced by a single equivalent condition: the space decomposes as a direct sum of irreducible invariant subspaces (Theorem 2).
