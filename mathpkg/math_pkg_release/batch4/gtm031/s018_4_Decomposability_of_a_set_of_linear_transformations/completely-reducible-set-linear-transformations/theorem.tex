Let $\Omega$ be a set of linear transformations on a vector space $\mathfrak{R}$ and let $L_{\Omega}$ denote the lattice of $\Omega$-invariant subspaces of $\mathfrak{R}$. The set $\Omega$ is called \text{completely reducible} if $L_{\Omega}$ is complemented, i.e., for every invariant subspace $\mathfrak{S}$ there exists an invariant subspace $\mathfrak{S}'$ such that $\mathfrak{R} = \mathfrak{S} \oplus \mathfrak{S}'$.
