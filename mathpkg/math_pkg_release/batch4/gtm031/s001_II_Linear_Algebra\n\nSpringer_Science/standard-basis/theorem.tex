\text{In the vector space $\Delta^{(n)}$ of $n$-tuples over a division ring $\Delta$,}
\text{define the vectors}
$$e_i = (0, 0, \dots, 0, 1, 0, \dots, 0), \quad i = 1, 2, \dots, n$$
\text{where the $1$ appears in the $i$-th position. Then for any}
$x = (\xi_1, \xi_2, \dots, \xi_n) \in \Delta^{(n)}$, \text{we have}
$$x = \sum_{i=1}^{n} \xi_i e_i = \xi_1 e_1 + \xi_2 e_2 + \cdots + \xi_n e_n$$
\text{and this representation is unique: if $\sum \xi_i e_i = \sum \eta_i e_i$,}
\text{then $\xi_i = \eta_i$ for all $i$.}
\text{The set $\{e_1, e_2, \dots, e_n\}$ is called the \textbf{standard basis} of $\Delta^{(n)}$.}
