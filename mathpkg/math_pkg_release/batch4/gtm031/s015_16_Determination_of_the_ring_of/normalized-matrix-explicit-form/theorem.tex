Let $\mathfrak{R} = \{f_1\} \oplus \cdots \oplus \{f_t\}$ with order ideals $(\delta_i)$ where $\delta_j = 0$ for $j > u$ and $\delta_i \mid \delta_j$ for $i \leq j$. A normalized matrix $(\beta) \in \mathfrak{U}$ has the following explicit form:
\begin{enumerate}
\item For $i \geq j$: $\beta_{ij}$ is arbitrary (no restriction beyond being in $\mathfrak{o}$).
\item For $i \leq u$, $j > u$: $\beta_{ij} = 0$.
\item For $i, j > u$: $\beta_{ij}$ is arbitrary.
\item For $i < j \leq u$: let $\eta_{ij} = \delta_i^{-1} \delta_j$. Then $\beta_{ij} \equiv 0 \pmod{\eta_{ij}}$.
\end{enumerate}
These conditions determine a unique normalized matrix for each endomorphism $B \in \mathfrak{B}$.
