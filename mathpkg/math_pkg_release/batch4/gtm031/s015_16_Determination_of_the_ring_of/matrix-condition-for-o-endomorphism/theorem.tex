Let $\mathfrak{R} = \{f_1\} \oplus \{f_2\} \oplus \cdots \oplus \{f_t\}$ with order ideals $(\delta_i)$. Let $B \in \mathfrak{B}$ be given by $f_i B = \sum_{j=1}^{t} \beta_{ij} f_j$ with matrix $(\beta) = (\beta_{ij})$. The condition $\epsilon_i \mid \delta_i$, equivalently $\delta_i g_i = 0$, is equivalent to the congruences
$$\delta_i \beta_{ij} \equiv 0 \pmod{\delta_j}, \quad i, j = 1, 2, \ldots, t.$$
Equivalently, there exists a matrix $(\gamma) = (\gamma_{ij})$ such that
$$\operatorname{diag}(\delta_1, \ldots, \delta_t) \, (\beta) = (\gamma) \, \operatorname{diag}(\delta_1, \ldots, \delta_t).$$
