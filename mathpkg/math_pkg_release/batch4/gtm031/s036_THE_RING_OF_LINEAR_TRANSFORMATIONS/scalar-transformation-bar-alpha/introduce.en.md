For a fixed basis $(e_1, \dots, e_n)$ of a vector space $\mathfrak{R}$ over a division ring $\Delta$, each element $\alpha \in \Delta$ induces a linear transformation $\bar{\alpha}$ that acts by right multiplication by $\alpha$ on each basis vector: $e_i \bar{\alpha} = \alpha e_i$. The set $\bar{\Delta}$ of all such transformations forms a subring of the endomorphism ring $\mathfrak{L}$ isomorphic to $\Delta$. These transformations serve as the "scalar matrices" in the decomposition $A = \sum \bar{\alpha}_{ij} E_{ij}$, providing the coefficients for the matrix representation of linear transformations.
