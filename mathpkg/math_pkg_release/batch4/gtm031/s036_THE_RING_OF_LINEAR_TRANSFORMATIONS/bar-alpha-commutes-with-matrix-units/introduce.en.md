The scalar transformation $\bar{\alpha}$ commutes with every matrix unit $E_{ij}$. This is a simple but crucial fact: checking on basis vectors shows that both $\bar{\alpha}E_{ij}$ and $E_{ij}\bar{\alpha}$ send $e_i$ to $\alpha e_j$ and all other $e_r$ to $0$. This commutation property allows the coefficients $\bar{\alpha}_{ij}$ to be freely moved past the matrix units in the expression $A = \sum \bar{\alpha}_{ij} E_{ij} = \sum E_{ij} \bar{\alpha}_{ij}$.
