Let $x = \sum \xi_i e_i$ be an arbitrary vector in $\mathfrak{R}$. Then

$$\alpha x = \sum (\alpha \xi_i) e_i$$

while

$$x \bar{\alpha} = \left(\sum \xi_i e_i\right) \bar{\alpha} = \sum \xi_i (e_i \bar{\alpha}) = \sum (\xi_i \alpha) e_i.$$

Thus $\bar{\alpha} = \alpha_l$ if and only if $\alpha \xi_i = \xi_i \alpha$ for all $i$ and for all choices of $\xi_i \in \Delta$. Since the $\xi_i$ range over all elements of $\Delta$, this is equivalent to the condition that $\alpha$ commutes with every element of $\Delta$, i.e., $\alpha \in Z(\Delta)$.
