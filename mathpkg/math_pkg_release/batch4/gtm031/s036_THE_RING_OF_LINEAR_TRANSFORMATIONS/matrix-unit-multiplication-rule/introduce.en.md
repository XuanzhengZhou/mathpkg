The matrix units $E_{ij}$ multiply according to the rule $E_{ij} E_{kl} = \delta_{jk} E_{il}$, which mirrors ordinary matrix multiplication: the product is nonzero only when the column index of the first factor matches the row index of the second. Additionally, the sum of the diagonal matrix units $\sum E_{ii}$ equals the identity transformation. These two properties characterize the matrix units as a system of matrix-like idempotents and nilpotents in the endomorphism ring $\mathfrak{L}$.
