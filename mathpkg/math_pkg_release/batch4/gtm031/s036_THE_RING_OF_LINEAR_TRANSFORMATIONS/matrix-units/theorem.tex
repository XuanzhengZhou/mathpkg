Let $(e_1, e_2, \cdots, e_n)$ be a basis for a finite dimensional vector space $\mathfrak{R}$ over a division ring $\Delta$. For $i, j = 1, 2, \cdots, n$, the \textbf{matrix unit} $E_{ij}$ is the linear mapping $E_{ij}: \mathfrak{R} \to \mathfrak{R}$ defined by

$$e_r E_{ij} = \delta_{ir} e_j, \quad r = 1, 2, \cdots, n.$$
