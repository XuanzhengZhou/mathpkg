For the first distributive law, take any $x \in \mathfrak{R}_1$:
$$x(A(B + C)) = (xA)(B + C) = (xA)B + (xA)C = x(AB) + x(AC) = x(AB + AC).$$
Since this holds for all $x \in \mathfrak{R}_1$, we have $A(B + C) = AB + AC$.

For the second distributive law, take any $x \in \mathfrak{R}_1$:
$$x((B + C)D) = ((xB + xC))D = (xB)D + (xC)D = x(BD) + x(CD) = x(BD + CD).$$
Note that $x(B + C) = xB + xC$ by definition of the sum of linear transformations. Since this holds for all $x \in \mathfrak{R}_1$, we have $(B + C)D = BD + CD$.
