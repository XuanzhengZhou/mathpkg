All Hamel bases of a given linear space have the same cardinality, which defines the dimension of the space. The proof proceeds via a replacement argument: for finite bases, a one-by-one replacement shows that any linearly independent set has at most as many elements as a basis. For infinite bases, a cardinal arithmetic argument using the union of finite support sets establishes the equality of cardinalities.
