A set of vectors is linearly independent if the only way to express the zero vector as a finite linear combination of distinct elements is with all coefficients zero. This is equivalent to requiring that every vector in the span has a unique representation as a linear combination of the independent set. Linear independence is the key notion underlying bases and dimension, distinguishing minimal generating sets from redundant ones.
