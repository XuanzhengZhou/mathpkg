If $A$ and $B$ are subsets of a linear space $E$, then $A + B$ is defined to be the set of all sums $x + y$ for $x \in A$ and $y \in B$. If $a$ is a scalar, then $aA$ denotes the set of all elements $a x$ for $x \in A$, and $-A$ is an abbreviation for $(-1)A$, which coincides with the set of all points $-x$ with $x \in A$.
