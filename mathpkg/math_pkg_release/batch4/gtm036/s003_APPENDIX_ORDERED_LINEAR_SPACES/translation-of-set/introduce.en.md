Translating a subset $A$ of a linear space by a vector $x$ produces the set $x + A = \{x + a : a \in A\}$. Translations are fundamental to affine geometry: a linear manifold is precisely the translate of a linear subspace. Translations preserve many geometric properties (convexity, dimension) while shifting the origin.
