This result is stated in the introductory summary without proof. The full proof appears later in Section 23 (see the discussion of the order dual). The key idea: if $f = f^+ - f^-$ with $f^+, f^-$ positive, then $f$ is bounded on order intervals $[a,b] = \{x : a \leq x \leq b\}$ because $|f(x)| \leq f^+(|x|) + f^-(|x|)$ for such $x$. Conversely, if $f$ is bounded on order-bounded sets, one constructs $f^+$ and $f^-$ via a Riesz-type decomposition.
