A \textbf{vector ordering} on a real linear space $E$ is a partial ordering $\geq$ satisfying:
\begin{enumerate}
  \item $x \geq y$ if and only if $x - y \geq 0$;
  \item If $x \geq 0$ and $t$ is a non-negative scalar, then $tx \geq 0$;
  \item If $x \geq 0$ and $y \geq 0$, then $x + y \geq 0$.
\end{enumerate}
