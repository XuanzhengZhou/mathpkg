Let $E$ be an ordered linear space with positive cone $C$. Then $C$ generates $E$ (i.e., $C - C = E$) if and only if for each $x \in E$ there exists a positive element $u \in C$ such that $u \geq x$.

\textit{Proof sketch.} If $C$ generates $E$, write $x = u - v$ with $u, v \in C$; then $u = x + v \geq x$. Conversely, if for each $x$ there exists $u \in C$ with $u \geq x$, then $x = u - (u - x)$ where $u, u-x \in C$, hence $x \in C - C$.
