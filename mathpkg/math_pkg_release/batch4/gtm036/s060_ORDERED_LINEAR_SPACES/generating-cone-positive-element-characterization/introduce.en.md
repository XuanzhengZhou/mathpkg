A cone $C$ generates the space $E$ if and only if every element $x \in E$ has a positive upper bound. Geometrically, the condition $u \geq x$ means $u - x \in C$. This characterization simplifies the algebraic condition $C - C = E$ to an order-theoretic one: the positive cone must be "large enough" that every element sits below some positive element in the order.
