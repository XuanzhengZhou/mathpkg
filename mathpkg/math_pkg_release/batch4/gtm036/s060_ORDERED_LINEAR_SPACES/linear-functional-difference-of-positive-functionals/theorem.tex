Let $E$ be an ordered normed linear space. Each continuous linear functional on $E$ is the difference of two positive linear functionals if and only if there exists an equivalent norm on $E$ that is monotonic (i.e., $0 \leq x \leq y \implies \|x\| \leq \|y\|$). In particular, this holds if the positive cone of $E$ is normal in the sense of Krein.
