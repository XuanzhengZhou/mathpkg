The proof is not given in detail in this introductory section. The statement is noted as: "in particular, if the radial kernel of $C$ is non-void, then $C - C = E$." The full argument would rely on the definition of the radial kernel: a point $x_0$ is in the radial kernel of $C$ if for every $y \in E$ there exists $\varepsilon > 0$ such that $x_0 + ty \in C$ for $|t| < \varepsilon$. Using this absorbing property, one can represent any element of $E$ as a difference of elements from $C$.
