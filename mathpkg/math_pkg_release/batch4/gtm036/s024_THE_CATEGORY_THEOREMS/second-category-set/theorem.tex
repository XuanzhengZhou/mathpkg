A set $A$ is \text{of the second category in $X$} (inexhaustible) if and only if it is not of the first category in $X$; that is, $A$ cannot be expressed as the union of a countable family of sets each of which is nowhere dense in $X$.
A set $A$ is said to be of the second category if and only if $A$ is of the second category in itself (that is, in $A$).
