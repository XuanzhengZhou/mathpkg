A set $A$ is \text{of the first category in $X$} (meager, exhaustible) if and only if $A$ is the union of a countable family of sets each of which is nowhere dense in $X$:
\[
A = \bigcup_{n=1}^{\infty} A_n, \qquad \text{where each } A_n \text{ is nowhere dense in } X.
\]
A set $A$ is said to be of the first category if and only if $A$ is of the first category in itself (that is, in $A$).
