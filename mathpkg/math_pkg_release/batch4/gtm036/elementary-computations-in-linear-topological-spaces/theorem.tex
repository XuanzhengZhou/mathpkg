Let $E$ be a linear topological space, and let $\mathcal{U}$ be the family of neighborhoods of $0$. Then:
(i) the closure of $x + A$ is $x + A^{-}$ for each vector $x$ and each subset $A$ of $E$; the closure of $aA$ is $aA^{-}$ for each non-zero scalar $a$;
(ii) if $A$ and $B$ are subsets of $E$, then $A^{-} + B^{-} \subset (A + B)^{-}$;
(iii) $A + U$ is open for any open set $U$ and any set $A$;
(iv) if $C$ is compact and $D$ is closed, then $C + D$ is closed;
(v) $A^{-} = igcap \{A + V : V \in \mathcal{U}\}$;
(vi) if $C$ is compact and $U$ is open with $C \subset U$, there is $V \in \mathcal{U}$ such that $C + V \subset U$.
