In a locally convex linear topological space, the family of all closed bounded (respectively, closed totally bounded) convex circled subsets serves as a co-base for the class of all bounded sets (respectively, totally bounded sets). This means that every bounded (or totally bounded) set can be enlarged to a closed, convex, circled set that remains bounded (or totally bounded). This application of the co-base terminology provides a structural characterization of these important classes of sets in locally convex spaces.
