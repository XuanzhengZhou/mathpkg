Let $E$ be a linear topological space, $A$ a convex subset of $E$, and $f$ a linear functional on $E$.

\text{(i)} The restriction $f|A$ is continuous if and only if for each real number $a$, the sets
$$\{x \in A: f(x) \leq a\} \quad\text{and}\quad \{x \in A: f(x) \geq a\}$$
are closed in $A$.

\text{(ii)} If $a$ is a real number such that $f^{-1}[a] \cap A \neq \varnothing$, then $f^{-1}[a] \cap A$ is homeomorphic to $f^{-1}[0] \cap A$.

\text{(iii)} If $f^{-1}[0] \cap A$ is closed in $A$, then $r^{-1}[0] \cap A \cap B$ is closed in $A$, where $r$ is the real part of $f$ and $B = \{x \in E: f(x) \text{ is real}\}$.

\text{(iv)} If $g$ is a symmetric sublinear functional on $E$ and $A$ is a convex set such that $(x - y)/2 \in A$ whenever $x, y \in A$, then
$$g(x) - g(y) = 2g\!\left(\frac{x - y}{2}\right)$$
for all $x, y \in A$.
