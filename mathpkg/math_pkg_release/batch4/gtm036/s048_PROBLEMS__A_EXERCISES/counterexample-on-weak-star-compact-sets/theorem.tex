Let $E$ be the set of sequences $x = \{\xi_n\}$ with at most a finite number of non-zero terms, with norm $\|x\| = \sum |\xi_n|$. Let $f_n(x) = \xi_n$. Then the set $\{2^{2n}f_n : n \in \mathbb{N}\} \cup \{0\}$ is weak* compact but is not strongly bounded. Consequently, its weak* closed convex extension is not weak* compact. Moreover, the weak* topology coincides with the Mackey topology $m(E^*, E)$, and $E^*$ with the Mackey topology is not sequentially complete.
