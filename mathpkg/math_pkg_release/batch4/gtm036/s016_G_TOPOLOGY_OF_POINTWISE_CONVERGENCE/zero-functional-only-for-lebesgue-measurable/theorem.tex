Let $X = [0,1]$ and let $\mu$ be Lebesgue measure on $X$. Then the convex extension of $U_n$ is $S(X, \mu)$ for each $n$. Consequently, the only continuous linear functional on $S(X, \mu)$ is the functional which is identically zero. Moreover, there is no non-negative linear functional on $S(X, \mu)$ which is not identically zero.
