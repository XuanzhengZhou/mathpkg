This exercise shows that for real linear topological spaces, an additive mapping that sends bounded sets to bounded sets is automatically homogeneous, hence linear, provided the target space is Hausdorff. The result is a useful criterion: additivity plus boundedness implies scalar homogeneity, eliminating the need to check $T(ax) = aT(x)$ separately.
