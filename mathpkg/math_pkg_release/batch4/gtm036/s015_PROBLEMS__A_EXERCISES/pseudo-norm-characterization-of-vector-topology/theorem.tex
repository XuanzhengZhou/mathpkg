If $d$ is a translation-invariant pseudo-metric for a linear space $E$ such that each sphere about $0$ is circled and $q(x) = d(x,0)$, then $q$ satisfies:
\begin{enumerate}
\item[(i)] $q(x) \geq 0$ for all $x \in E$ and $q(0) = 0$;
\item[(ii)] $q(x + y) \leq q(x) + q(y)$ for all $x, y \in E$;
\item[(iii)] $q(ax) \leq q(x)$ whenever $|a| \leq 1$; and
\item[(iv)] $\lim_{n \to \infty} q(x/n) = 0$ for all $x \in E$.
\end{enumerate}
Conversely, if $q$ satisfies (i)--(iv), then $d(x,y) = q(x - y)$ defines a translation-invariant pseudo-metric for a vector topology.
