This exercise gives an axiomatic characterization of pseudo-norms that define vector topologies. Starting from a translation-invariant pseudo-metric with circled spheres, the associated gauge function $q(x) = d(x,0)$ satisfies subadditivity, a restricted homogeneity condition ($|a| \leq 1 \implies q(ax) \leq q(x)$), and a continuity condition at zero along rays. Conversely, any functional satisfying these four axioms induces a vector topology via $d(x,y) = q(x-y)$.
