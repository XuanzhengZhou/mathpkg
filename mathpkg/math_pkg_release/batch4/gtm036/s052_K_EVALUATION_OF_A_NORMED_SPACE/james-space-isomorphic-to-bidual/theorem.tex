\text{With the norm given by }
|z|^2 = \sup \left\{ \sum_{i=1}^n \left| \zeta_{p(2i-1)}' - \zeta_{p(2i)}' \right|^2 + \left| \zeta_{p(2n+1)}' \right|^2 \right\},
\text{ where the supremum is taken over all } n \text{ and all permutations } p \text{ of the positive integers, } E^* \text{ is isomorphic to } E. \text{ This norm is equivalent to the dual norm given by }
\|z\|^2 = \sup \left\{ \sum_{i=1}^n \left| \zeta_{p(2i-1)}' - \zeta_{p(2i)}' \right|^2 + \left| \zeta_{p(2n+1)}' \right|^2 \right\}.
