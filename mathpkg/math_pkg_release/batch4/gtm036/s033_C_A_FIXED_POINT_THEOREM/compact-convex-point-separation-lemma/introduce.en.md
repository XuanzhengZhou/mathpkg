This lemma states that if a point $x_0$ can be strongly separated from each individual point $y$ of a compact convex set $A$ by (possibly different) continuous linear functionals, then there exists a single continuous linear functional on the real restriction of the space that strongly separates $x_0$ from the entire set $A$. The compactness of $A$ is essential to pass from pointwise separation to uniform separation over the whole set. This result serves as a key step in proving the fixed point theorem for affine continuous maps on compact convex subsets of linear topological spaces.
