In a linear topological space, any subset that is of the second category and satisfies the condition of Baire necessarily contains a translate of a symmetric set that also enjoys these two properties. This lemma provides a key structural insight used in the study of midpoint convex neighborhoods: it allows one to pass from an arbitrary second-category Baire set to a symmetric one via translation, thereby setting up the hypotheses needed for the main theorem.
