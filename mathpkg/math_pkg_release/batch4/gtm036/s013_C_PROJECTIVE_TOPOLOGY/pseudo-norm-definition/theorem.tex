A \textbf{pseudo-norm} $p$ for a linear space $E$ is a subadditive, absolutely homogeneous, non-negative functional on $E$ (without the requirement that $p(x) = 0$ implies $x = 0$). A linear topological space is \textbf{pseudo-normable} if and only if there is a pseudo-norm whose topology is that of the space. Two pseudo-norms are \textbf{equivalent} if and only if they induce the same topology.
