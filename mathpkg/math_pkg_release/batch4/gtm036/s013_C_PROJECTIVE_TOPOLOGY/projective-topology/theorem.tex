Let $X$ be a linear space and $\mathcal{F}$ a class of linear functions, each member $f$ of $\mathcal{F}$ mapping $X$ into a linear topological space $Y_f$. Then the \textbf{projective topology} for $X$ is the weakest topology on $X$ making each member of $\mathcal{F}$ continuous. This projective topology is a vector topology for $X$.
