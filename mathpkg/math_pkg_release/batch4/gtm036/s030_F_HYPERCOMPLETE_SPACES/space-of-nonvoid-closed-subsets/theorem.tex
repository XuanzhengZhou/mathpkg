Let $E$ be a linear topological space. Denote by $\mathcal{E}$ the set of all non-void closed subsets of $E$, endowed with the Hausdorff uniformity. The Hausdorff uniformity on $\mathcal{E}$ satisfies the Hausdorff separation axiom, because the uniformity distinguishes a set from its closure (and by restricting to closed sets, this ambiguity is removed).
