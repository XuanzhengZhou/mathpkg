Let $E$ be a linear topological space and let $\mathcal{U}$ be a local base of $E$. For each $U \in \mathcal{U}$, define
$$V_U = \{(A,B): A \subset B + U \text{ and } B \subset A + U\}.$$
Then $\{V_U: U \in \mathcal{U}\}$ is a base for a uniformity on the set of non-void subsets of $E$, called the \text{Hausdorff uniformity}. When $E$ is metrizable, this is precisely the uniformity of the Hausdorff metric.
