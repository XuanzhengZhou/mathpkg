Let $E$ be a Hausdorff linear topological space with topology $\mathcal{T}$, and let $\mathcal{U}$ be a stronger vector topology. Then $(E, \mathcal{U})^\wedge$ can be embedded in $(E, \mathcal{T})^\wedge$ if and only if every $\mathcal{U}$-Cauchy net which is $\mathcal{T}$-convergent is $\mathcal{U}$-convergent to the same limit.
