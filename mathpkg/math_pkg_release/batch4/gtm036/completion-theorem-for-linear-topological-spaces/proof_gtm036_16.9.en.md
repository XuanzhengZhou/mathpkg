The theorem follows from the approximation property established in 16.8. If the linear extension of the union of the members of $\mathcal{A}$ is $F$, then each linear functional on $F$ which is $w(F,E)$-continuous on each member of $\mathcal{A}$ can be approximated uniformly on members of $\mathcal{A}$ by elements of the canonical image $T[E]$. Hence, $E$ is complete relative to $\mathcal{T}_A$ if and only if each such functional is already represented by an element of $E$ (i.e., is $w(F,E)$-continuous on $F$). Equivalently, this is the case if and only if each hyperplane which intersects every member of $\mathcal{A}$ in a $w(F,E)$-closed set is $w(F,E)$-closed.
