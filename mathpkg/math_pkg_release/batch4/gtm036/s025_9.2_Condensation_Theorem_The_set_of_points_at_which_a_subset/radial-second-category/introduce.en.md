A corollary of the difference theorem: in a linear topological space of the second category, any radial set (a set whose scaled translates eventually cover every point) must itself be of the second category. Furthermore, if such a set also satisfies the condition of Baire, then its difference set $A - A$ is a neighborhood of the origin. This provides an important criterion for recognizing second category sets in topological vector spaces.
