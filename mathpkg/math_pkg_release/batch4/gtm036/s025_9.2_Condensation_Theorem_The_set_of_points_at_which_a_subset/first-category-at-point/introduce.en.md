The notion of first category (meagerness) is refined to a pointwise property: a set $A$ is of the first category at $x$ if $A$ is meager in some neighborhood of $x$. This localization allows the study of where a set fails to be "large" in the sense of Baire category. The set of points where $A$ is of the first category forms an open set, which is central to the condensation theorem.
