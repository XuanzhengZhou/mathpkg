This corollary provides a sufficient condition for continuity (resp. openness) of an additive map in terms of boundedness (resp. second category) of the image of a "large" set. If an additive map sends a second-category Baire set to a bounded set, the map is continuous; dually, if it sends a bounded set to a second-category Baire set, it is open.
