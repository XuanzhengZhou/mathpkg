A set satisfies the condition of Baire (or has the Baire property) if it differs from an open set by a first category set. This property is closed under complementation, countable unions, and countable intersections, forming a $\sigma$-algebra containing all Borel sets. The condition of Baire is a regularity property essential for many deep results in descriptive set theory and topological vector spaces, including the difference theorem and Banach's subgroup theorem.
