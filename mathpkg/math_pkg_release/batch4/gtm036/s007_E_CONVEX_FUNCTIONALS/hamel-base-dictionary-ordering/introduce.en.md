Every infinite-dimensional real linear space admits a total vector ordering, constructed via a Hamel basis. Given a well-ordered Hamel basis $B$ without a greatest element, one defines the "Z-A dictionary ordering": express each nonzero vector as a finite linear combination of basis elements, and declare it positive if the coefficient of the last basis element (in the well-ordering) is positive. This construction demonstrates that vector orderings exist in abundance on any infinite-dimensional space.
