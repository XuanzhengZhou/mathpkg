Given an arbitrary vector topology $\mathcal{T}$ on a linear space $E$, one can construct a locally convex topology $\mathcal{U}$ by taking all convex circled $\mathcal{T}$-neighborhoods of $0$ as a local base. This topology $\mathcal{U}$ is the strongest locally convex topology weaker than $\mathcal{T}$. A key property is that $\mathcal{T}$ and $\mathcal{U}$ admit exactly the same continuous linear functionals: every $\mathcal{T}$-continuous linear functional is bounded on some convex circled $\mathcal{T}$-neighborhood of $0$, hence $\mathcal{U}$-continuous. Consequently, local convexity is both sufficient and, in this derived sense, necessary for the existence of nontrivial continuous linear functionals.
