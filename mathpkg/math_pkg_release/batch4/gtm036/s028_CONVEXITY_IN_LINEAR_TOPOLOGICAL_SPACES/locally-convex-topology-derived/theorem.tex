If $\mathcal{T}$ is an arbitrary vector topology for a linear space $E$, then the family of all convex circled $\mathcal{T}$-neighborhoods of 0 is a local base for a locally convex topology $\mathcal{U}$, called the locally convex topology derived from $\mathcal{T}$. The topology $\mathcal{U}$ is the strongest locally convex topology weaker than $\mathcal{T}$, and $\mathcal{T}$ and $\mathcal{U}$ yield exactly the same class of continuous linear functionals.
