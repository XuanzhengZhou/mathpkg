Let $T$ be a linear function on a linear topological space $E$ to a linear topological space $G$, let $F$ be the null space of $T$, and let $S$ be the induced map of $E/F$ into $G$. Then $S$ is continuous (open) if and only if $T$ is continuous (open). The map $T$ is continuous and open if and only if $S$ is a topological isomorphism from $E/F$ onto $G$.
