The Krein-Smulian theorem is a fundamental result in the theory of locally convex spaces. It characterizes metrizable complete spaces via a weak* closure property: a convex set in the dual is weak* closed if and only if its intersection with each equicontinuous set is weak* closed in that set. This generalizes the hyperplane version proved in Theorem 17.7. The analogous property for convex circled sets characterizes hypercomplete spaces.
