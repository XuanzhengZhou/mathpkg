Let $E$ and $F$ be Hausdorff linear topological spaces and let $T$ be a linear mapping of $E$ into $F$. Then $T$ is called \emph{completely continuous} if and only if there is some neighborhood $U$ of zero which is mapped by $T$ into a compact set. For Banach spaces, this coincides with the usual definition of a completely continuous mapping as one which maps bounded sets into totally bounded sets. A completely continuous mapping is continuous.
