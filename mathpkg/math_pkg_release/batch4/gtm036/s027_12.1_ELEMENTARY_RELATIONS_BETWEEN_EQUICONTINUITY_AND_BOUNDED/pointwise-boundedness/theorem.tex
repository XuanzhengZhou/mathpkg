The family $F$ of linear functions, each from a linear topological space $E$ to a linear topological space $H$, is said to be \emph{pointwise bounded} on a set $A \subset E$ if and only if for each point $x$ of $A$ it is true that the set $F[x]$ of all points $f(x)$, for $f$ in $F$, is a bounded subset of $H$.

Equivalently, $F$ is pointwise bounded on $A$ if and only if $F$ is uniformly bounded on $\{x\}$ whenever $x \in A$.

The family of all sets of the form $\{f: f(x) \in V\}$, where $x \in A$ and $V$ is a neighborhood of $0$ in $H$, is a subbase for the neighborhood system of $0$ relative to the topology of pointwise convergence on $A$. It follows that $F$ is pointwise bounded on $A$ if and only if it is bounded relative to the topology of pointwise convergence on $A$.
