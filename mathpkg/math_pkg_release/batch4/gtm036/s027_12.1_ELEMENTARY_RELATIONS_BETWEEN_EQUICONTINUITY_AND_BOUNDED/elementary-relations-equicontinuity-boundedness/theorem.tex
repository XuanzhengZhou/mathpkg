Let $F$ be a family of linear functions, each on a linear topological space $E$ to a linear topological space $H$. Then:
(i) If $F$ is equicontinuous, then $F$ is uniformly bounded on each bounded set (equivalently, $F$ is bounded relative to the topology of uniform convergence on bounded sets).
(ii) $F$ is uniformly bounded on a set $A$ if and only if $F[A]$ is bounded, if and only if $F$ is bounded relative to the topology of uniform convergence on $A$.
(iii) If $F$ is uniformly bounded on $A$, then $F$ is pointwise bounded on $A$, which is equivalent to $F$ being bounded relative to the topology of pointwise convergence on $A$.
Consequently, (i) $\implies$ (ii) $\implies$ (iii) $\implies$ (iv).
