A set $U$ is a neighborhood of a point $x$ in a linear topological space $E$ if and only if $-x + U$ is a neighborhood of $0$. A base for the neighborhood system of $0$ is called a local base (or a system of nuclei). The topology is completely determined by any local base $\mathcal{U}$, for a set $V$ is open if and only if for each $x$ in $V$ there is a $U$ in $\mathcal{U}$ such that $x + U \subset V$.
