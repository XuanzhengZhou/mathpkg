Let $E$ be a locally convex space.
\begin{enumerate}
\item[(i)] $E$ is evaluable if and only if the evaluation map $I \colon E \to E^{**}$ is continuous, and this is the case if and only if each strongly bounded subset of $E^*$ is equicontinuous, or, equivalently, if and only if each bound absorbing barrel in $E$ is a neighborhood of $0$.
\item[(ii)] Each barrelled space and each bound space is evaluable, and each evaluable space is a Mackey space.
\item[(iii)] Each quotient space of an evaluable space is evaluable.
\item[(iv)] Each sequential completion of an evaluable space is a barrelled space.
\item[(v)] Products and direct sums of evaluable spaces are evaluable.
\end{enumerate}
