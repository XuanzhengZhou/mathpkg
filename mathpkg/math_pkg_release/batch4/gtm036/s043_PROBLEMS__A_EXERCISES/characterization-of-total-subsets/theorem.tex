A subset $A$ of a linear topological space $E$ is total if and only if each continuous linear functional vanishing on $A$ is identically zero. Equivalently, if $N_x = \{f \in E^* : f(x) = 0\}$, then $A$ is total if and only if $\bigcap \{N_x : x \in A\} = \{0\}$. Moreover, $A$ is total if and only if the subspace generated by $A$ is $w(E, E^*)$-dense in $E$.
