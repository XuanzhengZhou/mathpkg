This follows directly from Theorem 19.4. By that theorem, $\mathcal{T}^b = m(E, E^b)$. The topology $\mathcal{T}$ is a bound topology precisely when $\mathcal{T} = \mathcal{T}^b$. Hence $\mathcal{T} = m(E, E^b)$. This means that $E^b$, the space of all bounded linear functionals, coincides with the continuous dual $E'$, so every bounded linear functional is continuous. Moreover, $\mathcal{T} = m(E, E')$, so $(E, \mathcal{T})$ is a Mackey space. Conversely, if each bounded linear functional is continuous, then $E^b \subset E'$; and if $(E, \mathcal{T})$ is a Mackey space, then $\mathcal{T} = m(E, E')$. Since $E^b$ is always contained in $E'$, and under the hypothesis they coincide, we have $\mathcal{T} = m(E, E^b) = \mathcal{T}^b$, so the topology is bound.
