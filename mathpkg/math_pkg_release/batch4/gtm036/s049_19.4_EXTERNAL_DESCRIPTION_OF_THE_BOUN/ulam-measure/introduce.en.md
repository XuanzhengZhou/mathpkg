An Ulam measure is a nontrivial, atomless, two-valued countably additive measure defined on the full power set of a set $A$. Such a measure witnesses that the cardinal of $A$ is a measurable cardinal. It is a classical result of Ulam that the smallest cardinal admitting an Ulam measure, if one exists, is strongly inaccessible and exceeds $2^{\aleph}$ for all smaller cardinals $\aleph$. The existence of Ulam measures is independent of ZFC.
