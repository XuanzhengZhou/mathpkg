Let $E$ be a linear space with an inductive topology, determined by a family $\{E_t, f_t : t \in A\}$, each $E_t$ being a locally convex space.
\begin{enumerate}
    \item[(a)] If each $E_t$ is barrelled, so is $E$; if each $E_t$ is a bound space, so is $E$; if each $E_t$ is fully complete and $E$ is Hausdorff, then $E$ is fully complete.
    \item[(b)] If $T$ is a continuous linear map of $E$ into a fully complete space $F$, then $T$ is a weak homomorphism if and only if its graph is closed. In particular, $T^{-1}(0)$ is closed; $F/T^{-1}(0)$ is then an inductive limit of a sequence of fully complete spaces, and, if $T = S \circ Q$, (a) applies to $S^{-1}$.
\end{enumerate}
