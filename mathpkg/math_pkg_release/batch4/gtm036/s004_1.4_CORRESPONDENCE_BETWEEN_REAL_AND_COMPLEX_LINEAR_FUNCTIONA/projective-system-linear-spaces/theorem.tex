A projective system (inverse system) of linear spaces consists of the following:
\begin{itemize}
\item an index set $A$ directed by a partial ordering $\geq$;
\item a linear space $E_t$ for each $t$ in $A$;
\item for each pair of indices $s$ and $t$ with $t \geq s$, a canonical linear transformation $P_{st}$ from $E_t$ into $E_s$ with the property that if $t \geq s \geq r$, then $P_{rt} = P_{rs} \circ P_{st}$, and the property that $P_{ss}$ is the identity transformation for all $s$ in $A$.
\end{itemize}
The spaces $E_t$ are to be thought of intuitively as approximations to a limit space, the accuracy of the approximation increasing as the indices increase. The canonical maps $P_{st}$ relate the various approximations.
