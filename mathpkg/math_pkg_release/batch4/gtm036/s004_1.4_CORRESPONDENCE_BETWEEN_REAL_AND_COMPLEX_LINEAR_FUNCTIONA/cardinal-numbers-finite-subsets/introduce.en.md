This exercise from Chapter 1 of Kelley-Namioka concerns cardinal arithmetic: showing that the family of all finite subsets of an infinite set has the same cardinality as the set itself. This is a standard result in set theory that follows from the fact that the set of all finite sequences from an infinite set has the same cardinality as the set.
