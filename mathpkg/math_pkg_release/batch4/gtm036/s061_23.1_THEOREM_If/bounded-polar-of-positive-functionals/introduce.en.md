This theorem links the functional decomposition property to the boundedness of the polar set $B_0$. It shows that if every continuous linear functional decomposes into positive parts, then the polar of the set $B$ of unit-norm positive functionals is bounded. The proof uses the Hahn-Banach separation theorem (theorem 14.2) to show that $B_0 \subset 2[(S + C) \cap (S - C)]$, which together with Lemma 23.4 implies the cone is normal. This result bridges the functional-analytic and geometric approaches to ordered spaces.
