Observe that $x^+$ and $x^-$ are always disjoint, for $(x \vee 0) \wedge [-(x \wedge 0)]$ may be written, using the translation invariance, as

$$[(x \vee 0 + x \wedge 0) \wedge 0] - x \wedge 0 = x \wedge 0 - x \wedge 0 = 0.$$

On the other hand, if $x$ and $y$ are arbitrary disjoint positive elements, then

$$(x - y)^+ = (x - y) \vee 0 = x \vee y - y = x - x \wedge y = x;$$

a similar calculation shows that $(x - y)^- = y$.
