A corollary of the decomposition theorem for positive functionals: if every continuous linear functional can be written as the difference of two continuous positive functionals, then the space admits an equivalent pseudo-norm that is monotonic with respect to the ordering. That is, if $x, y \in C$ and $x \geq y$, then the pseudo-norm satisfies $p(x) \geq p(y)$. This provides a geometric compatibility between the topology and the order structure.
