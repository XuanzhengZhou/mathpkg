Theorem 2.2 establishes a fundamental geometric property of the radial kernel for convex sets: if $x$ lies in the radial kernel of a convex set $A$ and $y$ is any point in $A$, then the entire half-open segment $[x:y)$ (including $x$ but excluding $y$) is contained in the radial kernel. This shows that the radial kernel of a convex set is "star-shaped from the interior" — one can move from an interior point towards any boundary point while staying in the radial kernel. A key consequence is that the radial kernel of a convex set is itself convex and equals its own radial kernel.
