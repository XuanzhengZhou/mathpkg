For a subset $A$ of a linear space, the following conditions are equivalent:
\begin{itemize}
\item[(i)] $A$ is midpoint convex;
\item[(ii)] $A + A = 2A$;
\item[(iii)] $a_1A + a_2A + \cdots + a_nA = A$ whenever $a_1, a_2, \cdots, a_n$ is any finite collection of non-negative dyadic rationals whose sum is $1$.
\end{itemize}
