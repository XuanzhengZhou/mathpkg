A subset $P$ of a real linear space $E$ is called a cone if it satisfies:
\begin{itemize}
\item $P + P \subset P$;
\item $aP \subset P$ for each non-negative scalar $a$.
\end{itemize}
Moreover, $P - P$ is a linear subspace. The condition $x = y$ whenever $x \geq y$ and $y \geq x$ (relative to the ordering induced by $P$) is equivalent to $P \cap (-P) = \{0\}$. The set $P \cap (-P)$ is always a real linear subspace.
