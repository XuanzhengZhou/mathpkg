Let $\geq$ be a partial ordering of a real linear space $E$; that is, $\geq$ is a relation such that $x \geq x$ for each vector $x$ of $E$ and such that $x \geq z$ whenever $x \geq y$ and $y \geq z$. Suppose further that the ordering is translation invariant, in the sense that $x + z \geq y + z$ whenever $x \geq y$. Then the ordering is determined by the set $P$ of all vectors $x$ such that $x \geq 0$, for $x \geq y$ if and only if $x - y \geq 0$.

Conversely, if $P$ is any subset of $E$ such that $0 \in P$ and $P + P \subset P$, then a partial ordering which is translation invariant is defined by agreeing that $x \geq y$ if and only if $x - y \in P$; moreover, $P$ is precisely the set of vectors which are greater than or equal to $0$ relative to this ordering.

If the ordering also satisfies $ax \geq ay$ for all non-negative scalars $a$ whenever $x \geq y$, then the set $P$ of non-negative vectors satisfies $P + P \subset P$ and $aP \subset P$ for each non-negative scalar $a$. Conversely, a set $P$ with these two properties is precisely the set of all non-negative vectors relative to the ordering $x \geq y$ if and only if $x - y \in P$.
