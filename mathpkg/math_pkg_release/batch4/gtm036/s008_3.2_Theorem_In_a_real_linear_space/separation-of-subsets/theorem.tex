Two subsets $A$ and $B$ of a real linear space can be \textit{separated} if there are complementary half-spaces which contain $A$ and $B$ respectively.

A linear functional $f$ is said to \textit{separate} $A$ and $B$ if and only if $f$ is not identically zero and
$$\sup\{f(x) : x \in A\} \leq \inf\{f(x) : x \in B\}.$$
Clearly $f$ separates $A$ and $B$ if and only if $-f$ separates $B$ and $A$.

The linear functional $f$ \textit{strongly separates} $A$ and $B$ if the inequality above is strong; that is, if
$$\sup\{f(x) : x \in A\} < \inf\{f(x) : x \in B\}.$$
