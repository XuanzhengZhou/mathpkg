An ordered linear space (partially ordered linear space) is a real linear space $E$ with a vector ordering $\geq$ such that: $x \geq y$ if and only if $x - y \geq 0$; if $x \geq 0$ and $t$ is a non-negative scalar, then $tx \geq 0$; and if $x \geq 0$ and $y \geq 0$, then $x + y \geq 0$. The positive cone is $\{x : x \geq 0\}$.
