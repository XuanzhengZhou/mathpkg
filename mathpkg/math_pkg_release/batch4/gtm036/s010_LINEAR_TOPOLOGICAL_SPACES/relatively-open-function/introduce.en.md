A function $f: X \to Y$ is relatively open if it maps open sets of $X$ to sets that are open in the relative topology of the image $f[X] \subseteq Y$. This is weaker than being an open map into $Y$: the image of an open set need only be open in $f[X]$, not in all of $Y$. A function is a homeomorphism onto its image precisely when it is continuous, injective, and relatively open.
