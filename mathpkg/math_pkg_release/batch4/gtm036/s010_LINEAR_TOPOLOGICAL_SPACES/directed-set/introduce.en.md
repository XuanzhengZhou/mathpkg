A directed set is a preordered set $(A, \geq)$ in which every pair of elements has an upper bound. Directed sets serve as the index domains for nets (generalized sequences) in Moore-Smith convergence theory. Classical examples include the natural numbers with the usual order, the neighborhood system of a point ordered by reverse inclusion, and the set of finite subsets of a given set ordered by inclusion.
