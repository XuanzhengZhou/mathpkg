A net (or Moore-Smith sequence) is a function from a directed set into a topological space. A net $\{x_\alpha\}_{\alpha \in A}$ converges to a point $x$ if it is eventually in every neighborhood of $x$. Nets generalize sequences and provide a complete characterization of topological concepts: a set is closed iff it contains limits of all convergent nets; a function is continuous iff it preserves net convergence.
