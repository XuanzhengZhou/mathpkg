If $\mathcal{T}$ and $\mathcal{U}$ are two topologies for a set $X$ such that each $\mathcal{U}$-open set is $\mathcal{T}$-open, then the topology $\mathcal{T}$ is stronger (larger, finer) than $\mathcal{U}$, and $\mathcal{U}$ is weaker (smaller, coarser) than $\mathcal{T}$. Equivalently, $\mathcal{T}$ is stronger than $\mathcal{U}$ if and only if $\mathcal{T} \supset \mathcal{U}$.
