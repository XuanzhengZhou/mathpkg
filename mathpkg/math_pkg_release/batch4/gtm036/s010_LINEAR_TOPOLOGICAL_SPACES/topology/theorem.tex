A topology for a set $X$ is a family $\mathcal{T}$ of subsets of $X$, to be called open sets, such that the void (empty) set, the set $X$ itself, the union of arbitrarily many open sets, and the intersection of finitely many open sets, are open. The set $X$ with the topology $\mathcal{T}$ is a topological space, denoted by $(X, \mathcal{T})$.
