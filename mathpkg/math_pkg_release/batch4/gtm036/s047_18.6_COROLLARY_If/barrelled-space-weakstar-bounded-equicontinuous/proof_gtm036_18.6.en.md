Let $\mathcal{T}$ be a locally convex topology for $E$. Then $(E, \mathcal{T})$ is a barrelled space if and only if each $\mathcal{T}$-barrel is a $\mathcal{T}$-neighborhood of $0$, which is the case if and only if each $w(E, E^*)$-barrel is a $\mathcal{T}$-neighborhood of $0$, which is true if and only if the polar of each $w(E^*, E)$-bounded subset of $E^*$ is a $\mathcal{T}$-neighborhood of $0$, and this condition holds if and only if each $w(E^*, E)$-bounded set is $\mathcal{T}$-equicontinuous.
