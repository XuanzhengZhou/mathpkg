This corollary demonstrates a fundamental limitation of first-order logic: no set of first-order sentences can characterize exactly the class of finite structures. The proof is a classic application of the compactness theorem. One expands the language with infinitely many new constants and adds axioms asserting they are pairwise distinct. Any finite subset of these axioms plus the alleged finiteness-characterizing set $\Gamma$ is satisfiable (choose a sufficiently large finite structure). By compactness, the entire set is satisfiable, yielding an infinite model of $\Gamma$, a contradiction. This result illustrates that compactness, a consequence of completeness, has profound implications for definability in first-order logic.
