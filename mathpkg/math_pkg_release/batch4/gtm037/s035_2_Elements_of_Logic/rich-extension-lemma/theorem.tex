Let $\mathcal{L}$ be any first-order language and let $\Gamma$ be a consistent set of sentences in $\mathcal{L}$. Let $\mathcal{L}'$ be an expansion of $\mathcal{L}$ obtained by adjoining $|\operatorname{Fmla}_{\mathcal{L}}|$ new individual constants. Then there is a consistent rich set $\Delta$ of sentences of $\mathcal{L}'$ such that $\Gamma \subseteq \Delta$.
