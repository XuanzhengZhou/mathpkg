Let $\mathcal{L}, \mathcal{L}', T$ be as in Definition 11.26(v) (effective relational version). For any $m \in \omega$, define

$$f(m) = g'^+\varphi' \quad\text{if } m = g^+\varphi \text{ for some } \varphi \in \operatorname{Fmla}_{\mathcal{L}} \text{ (where } \varphi' \text{ is the } \mathcal{L}',T\text{-translate of } \varphi),$$
$$f(m) = 0 \quad\text{otherwise.}$$

Then $f$ is a recursive function.
