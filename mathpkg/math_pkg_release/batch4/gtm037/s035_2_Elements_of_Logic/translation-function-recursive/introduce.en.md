This proposition asserts that when working with effectivized languages, the syntactic translation mapping a formula of $\mathcal{L}$ to its counterpart $\varphi'$ in the relational version $\mathcal{L}'$ is a recursive function on Gödel numbers. Given a Gödel number $m = g^+\varphi$ of an $\mathcal{L}$-formula, the function $f$ returns $g'^+\varphi'$, the Gödel number of the translated formula in $\mathcal{L}'$; on inputs that do not code a formula, $f$ returns $0$. The proof appeals to weak Church's thesis: the recursive definition of $\varphi'$ is clearly effective, and the mapping can be implemented by a Turing machine or partial recursive function. This result is important for metamathematical arguments where syntactic operations must be represented within formal arithmetic.
