A language $\mathcal{L}' = (L', v', \mathcal{O}', \mathcal{R}')$ is an **expansion** of a language $\mathcal{L} = (L, v, \mathcal{O}, \mathcal{R})$ provided that $L = L'$, $v = v'$, $\mathcal{O} \subseteq \mathcal{O}'$, and $\mathcal{R} \subseteq \mathcal{R}'$. In this case $\mathcal{L}$ is a **reduct** of $\mathcal{L}'$.

If $\mathcal{L}' = (L', v', \mathcal{O}', \mathcal{R}', g')$ and $\mathcal{L} = (L, v, \mathcal{O}, \mathcal{R}, g)$ are effectivized first-order languages, we additionally require $g \subseteq g'$ and call $\mathcal{L}'$ an **effective expansion** of $\mathcal{L}$.

If $\mathfrak{A} = (A, f, R)$ is an $\mathcal{L}$-structure and $\mathfrak{B} = (B, f', R')$ is an $\mathcal{L}'$-structure, we say $\mathfrak{A}$ is the $\mathcal{L}$-**reduct** of $\mathfrak{B}$ (or $\mathfrak{B}$ is an $\mathcal{L}'$-expansion of $\mathfrak{A}$) provided $A = B$, $f \subseteq f'$, and $R \subseteq R'$. We write $\mathfrak{A} = \mathfrak{B} \upharpoonright \mathcal{L}$.
