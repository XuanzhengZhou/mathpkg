Let $\Gamma$ be a set of sentences in a first-order language $\mathcal{L}$, and let $\mathfrak{A}$ be a model of $\Gamma$. Then $\{\varphi^{\mathfrak{A}} : \varphi \in \mathrm{Fmla}_{\mathcal{L}}\}$ is an $\omega$-dimensional field of sets. Let $\mathfrak{B}$ be the associated cylindric set algebra. Then the function $f$ defined by $f([\varphi]) = \varphi^{\mathfrak{A}}$ for each $\varphi \in \mathrm{Fmla}_{\mathcal{L}}$ is a homomorphism from $\mathfrak{M}_{\Gamma}^{\mathcal{L}}$ onto $\mathfrak{B}$.
