Let $\mathfrak{A}$ and $\mathfrak{B}$ be $\mathrm{CA}_{\alpha}$'s. A \textit{homomorphism} from $\mathfrak{A}$ into $\mathfrak{B}$ is a function $f \colon A \to B$ such that for all $x, y \in A$ and $\kappa, \lambda < \alpha$,
\begin{enumerate}
\item[(i)] $f(x + y) = fx +' fy$;
\item[(ii)] $f(x \cdot y) = fx \cdot' fy$;
\item[(iii)] $f(-x) = -' fx$;
\item[(iv)] $f0 = 0'$;
\item[(v)] $f1 = 1'$;
\item[(vi)] $f c_{\kappa} x = c_{\kappa}' fx$;
\item[(vii)] $f d_{\kappa\lambda} = d_{\kappa\lambda}'$.
\end{enumerate}
\end{definition}
A homomorphism is called a \textit{homomorphism onto} if it is surjective, an \textit{isomorphism into} if it is injective, and an \textit{isomorphism onto} if it is bijective. We write $\mathfrak{A} \cong \mathfrak{B}$ if there is an isomorphism of $\mathfrak{A}$ onto $\mathfrak{B}$.
