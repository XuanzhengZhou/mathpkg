A class $K$ of $\mathcal{L}$-structures is compact if it satisfies the compactness property internally: whenever every finite subset of a set $\Gamma$ of sentences has a model belonging to $K$, the entire set $\Gamma$ has a model in $K$. This generalizes the usual compactness theorem for first-order logic, which states that the class of all $\mathcal{L}$-structures is compact.
