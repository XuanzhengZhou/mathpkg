Let $K$ be a class of $\mathcal{L}$-structures.
\begin{itemize}
\item[(i)] $\Theta_{\rho}K = \{\varphi \in \text{Sent}_{\mathcal{L}} : \varphi \text{ holds in each member of } K\}$; we call $\Theta_{\rho}K$ the \textit{theory} of $K$. $\Theta_{\rho}\mathfrak{A} = \Theta_{\rho}\{\mathfrak{A}\}$.
\item[(ii)] $K$ is \textit{elementarily closed} iff for all $\mathfrak{A}, \mathfrak{B}$, if $\mathfrak{A} \equiv_{\text{ee}} \mathfrak{B} \in K$ then $\mathfrak{A} \in K$.
\item[(iii)] $K$ is a \textit{projective} (or pseudo-elementary) class iff there is an expansion $\mathcal{L}'$ of $\mathcal{L}$ and an elementary class $L$ in $\mathcal{L}'$ such that $K = \{\mathfrak{A} \upharpoonright \mathcal{L} : \mathfrak{A} \in L\}$.
\item[(iv)] $K$ is \textit{compact} iff for all $\Gamma \subseteq \text{Sent}_{\mathcal{L}}$, if every finite subset of $\Gamma$ has a model in $K$, then $\Gamma$ has a model in $K$.
\end{itemize}
