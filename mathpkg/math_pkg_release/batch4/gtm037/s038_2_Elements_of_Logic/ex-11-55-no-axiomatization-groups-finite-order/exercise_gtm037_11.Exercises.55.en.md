Let $K$ be the class of all groups in which every element is of finite order, and let $L$ be an appropriate first-order language. Show that there is no set $\Gamma$ of sentences of $L$ such that $K = \{ \mathfrak{A} : \mathfrak{A} \text{ is a model of } \Gamma \}$.
