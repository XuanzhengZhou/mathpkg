Exercise 11.58 states that if every finite subset of a theory $\Gamma$ has a model that is not a model of the full $\Gamma$, then $\Gamma$ cannot be replaced by a finite set of sentences with exactly the same models. This follows from the compactness theorem: any finite set $\Lambda$ equivalent to $\Gamma$ would contradict the given condition.
