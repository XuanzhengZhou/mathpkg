Exercise 11.57 states that any first-order sentence true in all non-Archimedean ordered fields is also true in all ordered fields. The proof uses the fact that any ordered field can be embedded in a non-Archimedean one (e.g., by taking an ultrapower or by adjoining an infinitesimal element), so a sentence failing in some ordered field would also fail in some non-Archimedean ordered field.
