Let $\Delta = \{\varphi : \text{every model of } \Gamma \text{ is a model of } \varphi\}$. It is easy to check, using truth tables for the logical axioms, that $\Gamma \subseteq \Delta$, every logical axiom is in $\Delta$, and $\Delta$ is closed under detachment. Hence all $\Gamma$-theorems are in $\Delta$. The lemma follows.
