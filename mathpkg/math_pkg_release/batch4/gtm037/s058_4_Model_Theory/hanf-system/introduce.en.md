A Hanf system is a very general set-theoretic framework that abstracts the upward Löwenheim–Skolem phenomenon. It is defined as any ternary relation $R$ whose first projection is a set (as opposed to a proper class). This simple condition is sufficient to guarantee the existence of a Hanf number: a cardinal bound above which any witnessing of the relation can be extended to arbitrarily large witnesses. The framework applies not only to first-order logic but also to many non-linguistic contexts, based on the set-class distinction in set theory.
