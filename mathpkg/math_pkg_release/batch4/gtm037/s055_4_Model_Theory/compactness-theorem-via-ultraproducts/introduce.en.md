The Compactness Theorem is a fundamental result in first-order logic stating that if every finite subset of a set of sentences has a model, then the entire set has a model. This proof via ultraproducts constructs a large ultraproduct from the models of all finite subsets, using an ultrafilter over the set of finite subsets of the original theory. The resulting ultraproduct satisfies every sentence in the original set by Corollary 18.30 (Łoś's Theorem).
