The successor function $s(x) = x + 1$ is one of the simplest number-theoretic functions, and it is shown to be Turing computable by constructing an explicit Turing machine. The machine proceeds by first copying the input, appending a stroke, and then moving left to the final position. This establishes that the most basic arithmetic operation is representable within the Turing model of computation.
