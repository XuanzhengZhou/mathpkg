The projection functions $U_i^n(x_0, \ldots, x_{n-1}) = x_i$ select the $i$-th argument from an $n$-tuple of inputs. Lemma 3.11 establishes that every such projection function is Turing computable, which is a necessary ingredient for showing that the class of recursive functions is subsumed by Turing computable functions.
