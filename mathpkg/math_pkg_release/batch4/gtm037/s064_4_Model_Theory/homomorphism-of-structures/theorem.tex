Let $\mathfrak{A}$ and $\mathfrak{B}$ be $\mathcal{L}$-structures. A \text{homomorphism} from $\mathfrak{A}$ into $\mathfrak{B}$ is a function $f$ mapping $A$ into $B$ such that:

(i) if $\mathbf{O}$ is an operation symbol, say of rank $m$, and if $a \in {}^mA$, then

$$f\mathbf{O}^{\mathfrak{A}}(a_0, \ldots, a_{m-1}) = \mathbf{O}^{\mathfrak{B}}(f a_0, \ldots, f a_{m-1});$$

(ii) if $R$ is a relation symbol, say of rank $m$, and if $a \in {}^mA$, then

$$R^{\mathfrak{A}}(a_0, \ldots, a_{m-1}) \text{ implies } R^{\mathfrak{B}}(f a_0, \ldots, f a_{m-1}).$$
