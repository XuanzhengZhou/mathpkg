A language $\mathcal{L}$ is \text{algebraic} if it has no relation symbols. In any language, an atomic equality formula will also be called an \text{equation}. An (equational) \text{identity} is the universal closure of an equation. A \text{variety} is a class of $\mathcal{L}$-structures which is the class of all models of a set of identities, in an algebraic language. For any class $\mathbf{K}$ of $\mathcal{L}$-structures, let

$$\mathbf{SK} = \{ \mathfrak{A} : \mathfrak{A} \to \mathfrak{B} \in \mathbf{K} \text{ for some } \mathfrak{B} \};$$

$$\mathbf{PK} = \{ \mathfrak{A} : \mathfrak{A} \cong \prod_{i \in I} \mathfrak{B}_i \text{ for some } \mathfrak{B} \in {}^I\mathbf{K} \}.$$
