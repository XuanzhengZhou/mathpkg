Let $\mathcal{L}$ be a type language and $\mathfrak{A}$ an $\mathcal{L}$-structure. A \textbf{frame} for $\mathfrak{A}$ consists of two mappings $F, G$ whose domains are the set of all appropriate type symbols and which satisfy the following conditions:

\begin{enumerate}
\item[(16)] $F^0 = A = G^0$;
\item[(17)] For a type $\sigma = (\tau_0, \ldots, \tau_m)$, with $m > 0$ we have
  $$\varnothing \neq F^\sigma \subseteq \{f : f \text{ is a function and } f : G^{\tau_0} \times \cdots \times G^{\tau_{m-1}} \to G^{\tau_m}\}.$$
\end{enumerate}
