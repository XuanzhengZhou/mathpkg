A frame for an $\mathcal{L}$-structure $\mathfrak{A}$ provides a restricted semantic universe for higher-order quantification, weakening the standard (full) semantics. The mapping $G$ assigns to each type $\tau$ a set $G^\tau$ of admissible relations (a subset of $A_\tau$), while the mapping $F$ assigns to each type $\tau \neq 0$ a nonempty set of admissible functions from product spaces of $G$-domains into $G$-domains. At the base type, $F^0 = G^0 = A$. This frame-based semantics, also known as Henkin semantics or general models, is the key device for obtaining a completeness theorem for higher-order logic: by restricting the range of higher-order quantifiers, the logic becomes recursively axiomatizable.
