The satisfaction relation $(\mathfrak{A}, F, G) \models \varphi[x]$ defines truth of a formula $\varphi$ in a structure $\mathfrak{A}$ equipped with a frame $(F, G)$, relative to an assignment $x$. The key departure from standard (full) semantics is that relation quantifiers $\forall P_i^\tau$ range only over $G^\tau$ (the set of admissible relations), not over the full power set $A_\tau$. Likewise, operation quantifiers $\forall v_i^\tau$ range over the admissible function set $F^\tau$. This restriction makes the logic recursively axiomatizable, enabling a completeness theorem analogous to first-order logic.
