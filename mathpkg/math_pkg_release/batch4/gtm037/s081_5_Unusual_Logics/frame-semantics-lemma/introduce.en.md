Lemma 30.7 establishes the equivalence between satisfaction in a frame semantics (for type theory) and satisfaction in an associated first-order structure $\mathfrak{A}_G^*$. This is a fundamental technical lemma for reducing the semantics of type theory to ordinary first-order model theory, using an encoding of higher-type relations as first-order objects.
