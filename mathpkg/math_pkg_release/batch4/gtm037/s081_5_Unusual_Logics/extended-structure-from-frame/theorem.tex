With each $\mathcal{L}$-structure $\mathfrak{A}$ and each frame $G$ for $\mathfrak{A}$ we associate an $\mathcal{L}''$-structure $\mathfrak{A}_G^*$. Define:

$$A_G^* = \bigcup_{\tau \text{ a type}} G^\tau;$$

$$R_i^{\mathfrak{A}_G^*} = R_i^{\mathfrak{A}};$$

$$T_i^{\mathfrak{A}_G^*} = G^\tau \quad \text{(where } \tau \text{ is the type of } T_i\text{)};$$

$$S^{\mathfrak{A}_G^*} = \{(U_0, \ldots, U_m, V) : U_0, \ldots, U_m, V \in A_G^*, \; (U_0, \ldots, U_m) \in V \in G^\tau\},$$

where $\tau = (\sigma_0, \ldots, \sigma_m)$.
