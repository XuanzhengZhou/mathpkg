Craig's interpolation theorem extends to $L_{\omega_1\omega}$: if $\varphi$ implies $\psi$, there is an interpolant $\chi$ whose nonlogical symbols appear in both $\varphi$ and $\psi$. The proof follows the same pattern as for first-order logic, using the model existence theorem for $L_{\omega_1\omega}$.
