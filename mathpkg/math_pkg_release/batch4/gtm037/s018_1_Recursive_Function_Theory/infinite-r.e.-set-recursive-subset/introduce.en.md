Every infinite recursively enumerable set contains an infinite recursive subset. The proof constructs such a subset by enumerating the elements of the r.e. set via a recursive function and selecting an increasing subsequence using the unbounded $\mu$-operator. This result shows that although r.e. sets may not be decidable, they always contain a decidable infinite part.
