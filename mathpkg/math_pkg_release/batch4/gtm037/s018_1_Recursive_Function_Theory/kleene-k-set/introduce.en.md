Kleene's $K$ set is the canonical example of a recursively enumerable set that is not recursive. It is defined via the $T_1$ predicate, which encodes the halting of partial recursive functions. The set $K$ serves as the foundation for diagonalization arguments in recursion theory, analogous to the halting problem in computability theory.
