The class of recursively enumerable sets is closed under images by partial recursive functions. That is, if $A \subseteq \omega$ is r.e. and $f$ is a partial recursive function, then the set $f[A] = \{f(x) : x \in A \cap \operatorname{Dmn} f\}$ is also r.e. This closure property is fundamental for establishing that r.e. sets arise naturally from partial recursive computations.
