Kleene's $K$ set is the prototypical example of a recursively enumerable set that is not recursive. The proof uses a diagonal argument: $K \in \Sigma_1$, so it is r.e.; if it were recursive, its complement would also be r.e., which leads to a contradiction via the definition of $K$ using the $T_1$ predicate. This result establishes that the class of r.e. sets strictly contains the class of recursive sets.
