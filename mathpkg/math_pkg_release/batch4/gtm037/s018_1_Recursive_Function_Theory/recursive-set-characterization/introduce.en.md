A set $A \subseteq \omega$ is recursive if and only if both $A$ and its complement $\omega \setminus A$ are recursively enumerable. The nontrivial direction $(ii) \Rightarrow (i)$ shows that a set whose membership can be positively verified and whose non-membership can also be positively verified is actually decidable. This result, also known as Post's theorem for sets, is fundamental to understanding the relationship between decidability and semi-decidability.
