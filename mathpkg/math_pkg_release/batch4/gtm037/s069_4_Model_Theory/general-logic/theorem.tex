A \emph{general logic} is a weak general logic $\mathscr{L}$ which satisfies the following conditions:

\begin{enumerate}
\item[(1)] Every $\mathscr{L}$-elementary class is closed under isomorphism.
\item[(2)] If $\mathcal{L}, \mathcal{L}', f$ are as in (i), $f$ is onto, and $\mathbf{K}$ is an $\mathscr{L}, \mathcal{L}$-elementary class, then $\{ \mathfrak{A}_f : \mathfrak{A} \in \mathbf{K} \}$ is an $\mathscr{L}, \mathcal{L}'$-elementary class.
\end{enumerate}
