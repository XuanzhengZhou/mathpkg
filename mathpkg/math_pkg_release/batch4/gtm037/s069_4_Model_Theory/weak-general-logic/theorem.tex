A \emph{weak general logic} is a class $\mathscr{L}$ of quadruples $(\mathcal{L}, \Gamma, \mathfrak{A}, \varphi)$ such that $\mathcal{L}$ is a first-order language with no operation symbols and only finitely many relation symbols, $\Gamma$ is a class, $\mathfrak{A} \in \mathbf{S}_{\mathcal{L}}$, and $\varphi \in \Gamma$, such that $\Gamma$ is uniquely determined by $\mathcal{L}$ alone, i.e., if $(\mathcal{L}, \Gamma, \mathfrak{A}, \varphi)$ and also $(\mathcal{L}, \Gamma', \mathfrak{A}', \varphi')$ are in $\mathscr{L}$, then $\Gamma = \Gamma'$.

We write $\mathfrak{A} \models_{\mathscr{L}, \mathcal{L}} \varphi$ when $(\mathcal{L}, \Gamma, \mathfrak{A}, \varphi) \in \mathscr{L}$. An $\mathscr{L}, \mathcal{L}$-\emph{sentence} is an object $\varphi$ such that $\varphi \in \Gamma$, where $(\mathcal{L}, \Gamma, \mathfrak{B}, \psi) \in \mathscr{L}$ for some $\mathfrak{B}, \psi$. For any $\mathscr{L}, \mathcal{L}$-sentence $\varphi$ we let

$$\operatorname{Mod}_{\mathscr{L}, \mathcal{L}} \varphi = \{ \mathfrak{A} : \mathfrak{A} \models_{\mathscr{L}, \mathcal{L}} \varphi \}.$$

A class $\mathbf{K}$ is an $\mathscr{L}$-\emph{elementary class} provided that $\mathbf{K} = \operatorname{Mod}_{\mathscr{L}, \mathcal{L}} \varphi$ for some $\mathcal{L}$ and $\varphi$; we also call it an $\mathscr{L}, \mathcal{L}$-elementary class.
