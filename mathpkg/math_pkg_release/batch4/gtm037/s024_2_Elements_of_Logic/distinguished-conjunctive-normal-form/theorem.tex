Let $S = \langle n, c, \{s_0, \ldots, s_{m-1}\} \rangle$ be a sentential logic, with $m \in \omega$. Let $\varphi$ be a sentence which is not a tautology. Then there is a function $\psi$ such that:

\begin{enumerate}
\item[(i)] the domain of $\psi$ is $(p + 1) \times (m + 1)$ for some $p \in \omega$;
\item[(ii)] for each $i \leq p$ and each $j \leq m$, $\psi_{ij} = \langle s_j \rangle$ or $\psi_{ij} = \neg \langle s_j \rangle$;
\item[(iii)] $\vdash \varphi \leftrightarrow \bigwedge_{i \leq p} \bigvee_{j \leq m} \psi_{ij}$.
\end{enumerate}
