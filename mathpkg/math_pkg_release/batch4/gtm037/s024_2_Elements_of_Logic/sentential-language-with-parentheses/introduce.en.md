A **sentential language with parentheses** is an alternative formulation of sentential logic that uses explicit parenthesis symbols ($l$ for left parenthesis, $r$ for right parenthesis) instead of prefix (Polish) notation. The system consists of four distinct special symbols ($n$ for negation, $c$ for implication, $l$ and $r$ for parentheses) and a nonempty set $P$ of sentential symbols. Sentences are built using infix notation with parentheses: $\neg \varphi$ for negation and $(\varphi \rightarrow \psi)$ for implication. This approach more closely mirrors standard mathematical notation and is investigated in Exercise 8.47.
