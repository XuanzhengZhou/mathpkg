**Theorem 8.45** demonstrates that it is possible for a single binary connective to be functionally complete. The connective $s$ described here is the Sheffer stroke (NAND): $g_s(x,y) = 0$ only when both inputs are $1$, and $1$ otherwise. Since negation can be defined as $\varphi \mid \varphi$ (where $\mid$ denotes the Sheffer stroke) and conjunction as $(\varphi \mid \psi) \mid (\varphi \mid \psi)$, the set containing only the Sheffer stroke is functionally complete by reduction to Theorem 8.43. This result is of particular interest because it shows that an entire sentential logic can be built from a single connective.
