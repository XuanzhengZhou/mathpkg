Let $\mathcal{P} = \langle \{s\}, P, f, g \rangle$ be a general sentential logic such that $fs = 2$ and $g$ is given by

$$g_s \langle 1, 1 \rangle = 0,$$
$$g_s \langle 0, 1 \rangle = g_s \langle 1, 0 \rangle = g_s \langle 0, 0 \rangle = 1.$$

If $P$ is finite, then $\mathcal{P}$ is functionally complete.
