By duality from Theorem 8.43. Negation and conjunction form a functionally complete set by Theorem 8.43. Since disjunction is the dual of conjunction (via De Morgan's laws: $\varphi \vee \psi \equiv \neg(\neg \varphi \wedge \neg \psi)$), any truth function expressible using $\neg$ and $\wedge$ can be expressed using $\neg$ and $\vee$. Thus $\{\neg, \vee\}$ is also functionally complete.
