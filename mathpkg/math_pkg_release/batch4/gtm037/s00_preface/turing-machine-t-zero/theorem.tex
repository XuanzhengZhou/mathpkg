$T_0$ is the following Turing machine:
\[
\begin{array}{cccc}
0 & 0 & 4 & 0 \\
0 & 1 & 0 & 0
\end{array}
\]
That is, $T_0$ has a single state $0$, which is also the initial state. If the scanned symbol is $0$, the machine performs action $4$ (stop). If the scanned symbol is $1$, the machine performs action $0$ (write $0$ on the scanned square) and remains in state $0$.
