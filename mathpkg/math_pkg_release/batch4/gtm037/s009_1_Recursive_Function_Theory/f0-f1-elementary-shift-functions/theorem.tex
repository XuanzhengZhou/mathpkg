For any $x \in \omega$, let
$$f_0 x = \begin{cases} x + 2 & \text{if } x \text{ is even}, \\ 0 & \text{if } x = 1, \\ x - 2 & \text{if } x \text{ is odd and } x > 1, \end{cases}$$
$$f_1 x = \begin{cases} x - 2 & \text{if } x \text{ is even and } x > 0, \\ 1 & \text{if } x = 0, \\ x + 2 & \text{if } x \text{ is odd}. \end{cases}$$

$f_0$ and $f_1$ are elementary. For any $e \in \mathbb{Z}$ we have $f_0 g e = g(e + 1)$ and $f_1 g e = g(e - 1)$.
