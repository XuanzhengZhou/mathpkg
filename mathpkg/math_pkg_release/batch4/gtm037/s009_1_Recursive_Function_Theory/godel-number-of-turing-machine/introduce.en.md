This definition assigns a G\"{o}del number $gM$ to each Turing machine $M$ by encoding its instruction quadruples as prime power products. Each instruction is encoded using exponents for the current state, the read symbol, the write symbol, and the direction. The numbering arithmetizes Turing machines into the natural numbers, enabling metamathematical reasoning about computability within arithmetic.
