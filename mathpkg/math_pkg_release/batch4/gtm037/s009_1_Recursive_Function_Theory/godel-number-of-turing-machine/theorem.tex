Let $\mathbb{E}$ be the set of even numbers. Let $T$ be the class of all Turing machines. If $M$ is a Turing machine, with notation as in 1.1, we let the G\"{o}del number of $M$, $gM$, be the number
$$\prod_{i < 2m} p_i^{t_i},$$
where, for each $i < 2m$,
$$t_i = 2^{c[(i+2)/2]} \cdot 3^{(x_B)(i+1)} \cdot 5^{v(i+1)} \cdot 7^{d(i+1)}.$$
