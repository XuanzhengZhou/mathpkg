This definition assigns a G\"{o}del number $gh$ to each finite binary sequence $h$. The encoding uses the first $\operatorname{Dmn}(h)$ prime numbers, with exponent $h_i + 1$ ensuring that each position's value (0 or 1) is encoded as exponent 1 or 2 respectively. This provides an injective mapping from finite binary sequences into the natural numbers, enabling arithmetic operations on sequences to be expressed as arithmetic operations on their codes.
