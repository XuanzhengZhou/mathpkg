This definition encodes a Turing machine tape description $F$ as a single natural number $gF$ by interleaving the values on the positive and negative positions of the tape into a single infinite product of prime powers. Even-indexed primes encode the non-negative half-tape, while odd-indexed primes encode the negative half-tape, providing a G\"{o}del numbering of all possible tape configurations.
