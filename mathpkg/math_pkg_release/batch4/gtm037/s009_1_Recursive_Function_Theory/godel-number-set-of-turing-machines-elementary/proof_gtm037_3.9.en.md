For any $x \in \omega$, $x \in g^*T$ iff $\operatorname{lh}(x)$ is odd, $x > 1$, for every $i \leq \operatorname{lh}(x)$ we have $((x)_i)_2 < 5$, for every $i \leq \operatorname{lh}(x)$ there is a $j \leq \operatorname{lh}(x)$ such that $((x)_i)_3 = ((x)_j)_0$, for every $i \leq \operatorname{lh}(x)$, if $i$ is even then $((x)_i)_0 = ((x)_{i+1})_0$, and for all $i, j \leq \operatorname{lh}(x)$, if $i + 2 \leq j$, then $((x)_i)_0 \neq ((x)_j)_0$, and if $i$ is even then $((x)_i)_1 = 0$, while if $i$ is odd, $((x)_i)_1 = 1$.
