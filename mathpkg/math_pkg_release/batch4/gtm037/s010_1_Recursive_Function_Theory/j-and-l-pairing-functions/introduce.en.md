The pairing function $J: \omega \times \omega \to \omega$ encodes an ordered pair $(a, b)$ as a single natural number via the formula $J(a, b) = ((a + b)^2 + b)^2 + a$. The left and right decoding functions are given by $Lx = \text{Exc}[\sqrt{x}]$ and $\text{Exc}\,x = x - [\sqrt{x}]^2$, where $\text{Exc}\,x$ recovers the first component and $Lx$ recovers the second. This pairing scheme is elementary, injective, and provides a bijective encoding essential for Godel numbering and sequence coding in recursive function theory.
