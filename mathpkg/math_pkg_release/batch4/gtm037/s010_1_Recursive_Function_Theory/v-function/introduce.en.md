The function $V$ is defined via bounded minimization over the elementary relation $R_3$, which encodes the step-by-step computation of a Turing machine. For a given natural number $x$, $Vx$ extracts the least $y \leq x$ such that a certain configuration determined by $x$, $y$, and the concatenation function Cat belongs to $R_3$. The fact that $V$ is elementary follows immediately from closure properties of elementary relations under bounded minimization.
