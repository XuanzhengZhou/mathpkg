The $m$-ary Turing machine computation relation $T_m$ is elementary, meaning it belongs to the class of relations definable by formulas with only bounded quantifiers. This result is essential for establishing that computability-theoretic notions can be expressed within the arithmetical hierarchy without unbounded quantification. The proof proceeds by explicitly characterizing membership in $T_m$ using elementary relations involving sequence numbers, bounded quantifiers, and the coding of Turing machine configurations.
