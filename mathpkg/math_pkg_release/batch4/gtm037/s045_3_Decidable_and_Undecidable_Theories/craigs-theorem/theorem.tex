For any theory $\Gamma$ in an elementary effectivized first-order language the following conditions are equivalent:

\begin{enumerate}
\item[(i)] $\Gamma$ is axiomatizable by a set $\Delta$ such that $g^{+*}\Delta$ is elementary;
\item[(ii)] $\Gamma$ is recursively axiomatizable;
\item[(iii)] $g^{+*}\Gamma$ is r.e.
\end{enumerate}
