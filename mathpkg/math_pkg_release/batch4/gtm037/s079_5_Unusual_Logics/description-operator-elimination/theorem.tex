Let $(\mathcal{L}, \tau, \mathbf{O})$ be a descriptive triple, $\Gamma$ a set of sentences, $\sigma$ a term, and $\varphi$ a formula. Then

(i) if $v_k$ is a variable not occurring in $\sigma$, there is a formula $\psi$ not involving $\tau$ with $\text{Fv}\,\psi \subseteq \text{Fv}\,\sigma \cup \{v_k\}$ such that $\Gamma \models \sigma = v_k \leftrightarrow \psi$;

(ii) there is a formula $\varphi^*$ not involving $\tau$ with $\text{Fv}\,\varphi^* \subseteq \text{Fv}\,\varphi$ such that $\Gamma \models \varphi \leftrightarrow \varphi^*$.
