Let $(\mathcal{L}, \varepsilon, \mathbf{O})$ be a choice triple. A choice structure is a pair $(\mathfrak{A}, f)$ such that $\mathfrak{A}$ is an $\mathcal{L}$-structure and $f$ is a choice function for nonempty subsets of $A$. The term evaluation for $\varepsilon v_i \varphi$ is:

$$(\varepsilon v_i \varphi)^{\mathfrak{A}f}x = f\{a \in A : (\mathfrak{A}, f) \models \varphi[x_a^i]\}$$

if this set is nonempty, and $\mathbf{O}^{\mathfrak{A}}$ otherwise.
