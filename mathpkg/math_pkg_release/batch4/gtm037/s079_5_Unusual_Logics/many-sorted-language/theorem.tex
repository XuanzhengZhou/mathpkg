A many-sorted language $\mathcal{L}$ is determined by: a nonempty set $\mathcal{S}$ of sorts; for each $s \in \mathcal{S}$, individual variables of sort $s$: $v_0^s, v_1^s, \dots$; the logical symbols $\neg, \lor, \land, \forall, \equiv$; and nonlogical relation and operation symbols, each with a rank which is a finite sequence of sorts.
