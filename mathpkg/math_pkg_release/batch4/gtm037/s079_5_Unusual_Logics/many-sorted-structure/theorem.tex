Let $\mathcal{L}$ be a many-sorted language. An $\mathcal{L}$-structure is a triple $\mathfrak{A} = (A, f, R)$ such that:
(i) $A$ assigns to each $s \in \mathcal{S}$ a nonempty set $A_s$;
(ii) $f$ assigns to each operation symbol $\mathbf{O}$ of rank $(s_0, \ldots, s_m)$ a function $f_{\mathbf{O}} : A_{s_0} \times \cdots \times A_{s_{m-1}} \rightarrow A_{s_m}$;
(iii) $R$ assigns to each relation symbol $\mathbf{R}$ of rank $(s_0, \ldots, s_m)$ a relation $R_{\mathbf{R}} \subseteq A_{s_0} \times \cdots \times A_{s_m}$.
