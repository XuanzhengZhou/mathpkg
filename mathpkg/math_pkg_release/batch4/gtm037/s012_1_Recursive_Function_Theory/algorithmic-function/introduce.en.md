An algorithmic function is a function on natural numbers that can be computed by a Markov algorithm. The input arguments are encoded as unary strings separated by zeros, with a special marker $2$ indicating the active position. The algorithm terminates when the marker $2$ appears exactly once in the final word, with the output value encoded in unary. This notion captures the intuitive idea of mechanical computability and is equivalent to Turing computability.
