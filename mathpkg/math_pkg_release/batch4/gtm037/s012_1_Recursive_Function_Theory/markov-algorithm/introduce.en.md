A Markov algorithm is a formal model of computation introduced by Andrey Markov Jr., consisting of a finite list of string rewriting rules applied sequentially to an input word. Each rule substitutes one substring for another, and the algorithm terminates when a terminating rule is applied. Markov algorithms are closely related to Post canonical systems and formal grammars, providing an alternative to Turing machines as a foundation for computability theory.
