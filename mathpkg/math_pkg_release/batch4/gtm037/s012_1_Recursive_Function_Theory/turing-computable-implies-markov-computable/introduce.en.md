This theorem establishes that every function computable by a Turing machine can also be computed by a Markov algorithm, demonstrating that Markov's model of string rewriting is at least as powerful as Turing's model. The proof constructs a Markov algorithm that simulates the behavior of a given Turing machine by encoding the machine's configurations as words and mapping each transition rule to a corresponding set of substitution rules. This result is a key step in proving the equivalence of various formalizations of computability.
