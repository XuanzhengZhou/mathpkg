This lemma establishes that the first occurrence relation $R_1$ is elementary, extending the arithmetization of word operations. Together with $R_0$, this shows that the fundamental syntactic relations needed to simulate Markov algorithms in arithmetic are first-order definable. The proof uses the Cat concatenation function and the property that no earlier position contains the target word, both of which can be expressed with bounded quantifiers.
