Let $\mathcal{L}$ be an expansion of $\mathcal{L}_{\text{nos}}$. A theory $\Gamma$ in $\mathcal{L}$ is called a strong theory provided that for every $m$-ary elementary function $f$ there is an $m$-ary operation symbol $\mathbf{O}$ of $\mathcal{L}$ such that for any $x_0, \ldots, x_{m-1}, y \in \omega$ the following conditions hold:

(i) if $f(x_0, \ldots, x_{m-1}) = y$, then $\Gamma \models \mathbf{O}(\mathbf{x}_0, \ldots, \mathbf{x}_{m-1}) = \mathbf{y}$;

(ii) if $f(x_0, \ldots, x_{m-1}) \neq y$, then $\Gamma \models \neg \mathbf{O}(\mathbf{x}_0, \ldots, \mathbf{x}_{m-1}) = \mathbf{y}$.
