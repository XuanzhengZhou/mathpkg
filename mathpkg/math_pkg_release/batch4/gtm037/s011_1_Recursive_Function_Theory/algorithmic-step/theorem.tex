An algorithmic step under a Markov algorithm $A$ (with rows $(a_i, b_i, c_i)$ for $i \leq m$) is a pair $(d, e)$ of words with the following properties:

(i) there is an $i \leq m$ such that $a_i$ occurs in $d$;

(ii) if $i \leq m$ is minimum such that $a_i$ occurs in $d$, and if $(f, a_i, g)$ is the first occurrence of $a_i$ in $d$, then $e = f b_i g$.
