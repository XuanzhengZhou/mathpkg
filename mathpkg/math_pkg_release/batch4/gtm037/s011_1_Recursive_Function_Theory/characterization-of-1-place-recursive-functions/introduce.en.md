Theorem 3.50 provides a compact characterization of the class of 1-place recursive functions. Rather than defining recursive functions via the full schema of composition, primitive recursion, and minimalization, this theorem shows they are precisely those functions belonging to every set $A$ that contains the constant function $\omega$, the excess-over-a-square function $\mathrm{Exc}$, and is closed under the modified composition operator $K_1^1$, the pairing operator $P$, and inversion applied to surjective functions. This result is used to simplify proofs that specific 1-place functions are recursive.
