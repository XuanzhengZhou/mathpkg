Give an example of a binary partial recursive function $f$ such that if $g$ is defined by

$$gx = \text{least } y \text{ such that } f(x, y) \text{ is defined and } f(x, y) = 0,$$

$$gx = \text{undefined if no such } y,$$

then $g$ is not partial recursive (cf. 5.3 and following remarks).
