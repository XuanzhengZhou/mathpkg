The length function $l(a)$ returns the greatest index $i$ such that the $i$-th prime $p_i$ divides $a$. For $a = 0$ or $a = 1$, the length is defined as $0$. When using prime power encoding of sequences, $l(a)$ tells us how many primes appear in the factorization of $a$, which corresponds to the length of the encoded sequence plus one. Together with the exponent function $(a)_i$, the length function provides complete decoding of Godel-numbered sequences.
