The prime function $p(k)$ that returns the $(k+1)$-st prime number is elementary recursive. This is a non-trivial result because the sequence of primes is not obviously computable by elementary operations alone. The proof constructs the relation "x is the next prime after y" and the relation "x is the k-th prime" using bounded quantification, then extracts the k-th prime via the bounded minimum operator, all of which preserve elementarity.
