A class $A$ of number-theoretic functions is closed under primitive recursive operations if it contains the basic primitive recursive functions (the zero function $\varnothing$ and projection functions $U_i^n$) and is closed under composition and the two primitive recursion schemas $R^m$ and $R^0$. This definition mirrors the analogous notion for elementary recursive operations and provides a modular framework: any class closed under primitive recursive operations automatically contains all primitive recursive functions.
