Definition by cases allows constructing an $A$-function from a finite partition of ${}^n\omega$ into $A$-relations and a corresponding family of $A$-functions. When $A$ is closed under elementary recursive operations, the function defined by cases using $A$-relations and $A$-functions is itself an $A$-function. This closure property is essential for building complex recursive functions from simpler components by case distinction.
