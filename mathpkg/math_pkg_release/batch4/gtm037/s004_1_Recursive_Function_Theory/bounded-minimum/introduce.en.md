The bounded minimum operator $\mu y < z \, R(x_0, \ldots, x_{m-2}, y)$ returns the least $y$ less than $z$ that satisfies an $m$-ary relation $R$, or $0$ if no such $y$ exists. This is also called the bounded $\mu$-operator or bounded search operator. It is a fundamental construction in recursive function theory that allows defining functions by searching through a bounded range for a witness satisfying a given relation.
