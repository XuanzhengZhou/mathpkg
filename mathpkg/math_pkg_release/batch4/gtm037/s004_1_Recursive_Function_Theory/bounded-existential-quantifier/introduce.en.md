The bounded existential quantifier preserves $A$-relations when $A$ is closed under elementary recursive operations. Given an $m$-ary $A$-relation $R$, the set of tuples $\langle x_0, \ldots, x_{m-1}\rangle$ for which there exists a witness $y < x_{m-1}$ satisfying $R(x_0, \ldots, x_{m-2}, y)$ is itself an $A$-relation. This closure property is fundamental in recursive function theory and enables the construction of complex relations from simpler ones while staying within the class $A$.
