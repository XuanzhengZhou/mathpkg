If $\mathfrak{A} = \langle A, +, \cdot, -, 0, 1 \rangle$ is a Boolean algebra, then so is

$$\langle A, \cdot, +, -, 1, 0 \rangle.$$

Consequently, if a statement about all Boolean algebras has been proved, then the dual statement obtained by interchanging $+$ and $\cdot$ and also $0$ and $1$ is also true.
