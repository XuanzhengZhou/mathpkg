This theorem establishes that every recursively enumerable set can be obtained as the inverse image of a creative set under a recursive function. In other words, the class of creative sets is universal for the class of r.e. sets via recursive many-one reductions. This result, together with the fact that the Kleene set $K$ is creative, shows that $K$ is $1$-complete for the class of r.e. sets, a fundamental structural result in recursion theory.
