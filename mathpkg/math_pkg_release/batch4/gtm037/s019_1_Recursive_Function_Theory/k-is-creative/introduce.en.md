The Kleene set $K$, which consists of indices $e$ such that the partial recursive function $\varphi_e^1$ halts on input $e$, is a creative set. This is a fundamental result in recursive function theory: $K$ is not only recursively enumerable but also strongly nonrecursive, as its complement $\omega \sim K$ admits a productive function. Specifically, the universal function $U_0^1$ serves as a productive function for $\omega \sim K$, meaning that for any r.e. subset of $\omega \sim K$ we can effectively find an element of the complement outside that subset.
