Hilbert's tenth problem, which asks for an algorithm to determine whether a given Diophantine equation has an integer solution, is unsolvable. This negative solution, obtained by combining the work of Davis, Putnam, Robinson, and Matiyasevich, is a direct consequence of the MRDP theorem: since the halting problem is r.e. but not recursive, and every r.e. set is Diophantine, no algorithm can decide solvability of arbitrary Diophantine equations.
