The MRDP theorem (Matiyasevich-Robinson-Davis-Putnam) establishes that the recursively enumerable relations are exactly the Diophantine relations. This landmark result provides a purely number-theoretic characterization of computability: a set is computably enumerable if and only if it can be defined by an existential polynomial equation over the natural numbers. As a corollary, Hilbert's tenth problem is unsolvable.
