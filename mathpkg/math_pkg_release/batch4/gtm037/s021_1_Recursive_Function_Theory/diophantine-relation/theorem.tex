An $n$-ary relation $R \subseteq {}^n\omega$ is called diophantine if there is a polynomial $P(x_0, \ldots, x_{n-1}, y_0, \ldots, y_{m-1})$ with integral coefficients such that
$$R = \{x \in {}^n\omega : \text{there exist } y_0, \ldots, y_{m-1} \in \omega \text{ such that } P(x_0, \ldots, x_{n-1}, y_0, \ldots, y_{m-1}) = 0\}.$$
