This definition extends the notion of recursive enumerability from ordinary relations on $\omega$ to $(m,n)$-relations involving function arguments. The key idea is that a machine can only access finite initial segments of function inputs, so an r.e. relation is one whose membership can be verified by examining some finite initial segment $\mathfrak{A}x$ of the argument functions together with a number witness $x$.
