Novikov's theorem (1955) provides a negative solution to the word problem for finitely presented groups: there exists a group presented by finitely many generators and relations for which no algorithm can decide whether two given words represent the same group element. This is a landmark application of recursive function theory to classical algebra, demonstrating that fundamental algebraic questions can be algorithmically undecidable.
