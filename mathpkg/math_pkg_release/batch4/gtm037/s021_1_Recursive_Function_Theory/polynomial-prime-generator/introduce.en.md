As a striking corollary of the MRDP theorem, there exists a multivariate polynomial with integer coefficients whose set of positive values (as the variables range over natural numbers) coincides exactly with the set of prime numbers. This provides an unexpected connection between Diophantine definability and the prime numbers, giving a polynomial formula that enumerates all primes.
