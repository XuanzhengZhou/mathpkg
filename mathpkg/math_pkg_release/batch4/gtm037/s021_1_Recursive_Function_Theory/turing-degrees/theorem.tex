This establishes an equivalence relation on the set of all functions mapping $\omega$ into $\omega$. The equivalence classes are called \emph{Turing degrees of unsolvability}. Each equivalence class has at most $\aleph_0$ members (actually exactly $\aleph_0$, as is easily seen), since there are only $\aleph_0$ possible Turing machines with oracles. Clearly then there are $\exp \aleph_0$ degrees. Let $D$ be the set of degrees. For $\alpha$, $\beta \in D$ we write $\alpha \leq \beta$ provided there exist $f \in \alpha$ and $g \in \beta$ with $f$ recursive in $g$. This relation $\leq$ makes $D$ into a partially ordered set. Clearly the degree of recursive functions, denoted by $0$, is the least element of $D$.
