A Diophantine equation is a polynomial equation with integer coefficients, where one seeks integer (or natural number) solutions. Named after Diophantus of Alexandria, these equations are central to number theory. Hilbert's tenth problem asks whether there is an algorithm to determine if an arbitrary Diophantine equation has an integer solution --- a question answered negatively using recursive function theory.
