This proposition establishes that the simultaneous substitution operation on formulas, viewed at the level of Godel numbers, is a recursive (computable) function. Specifically, the function $s$ that takes the Godel number $x$ of a formula $\varphi$ and the Godel numbers $y_0,\dots,y_{m-1}$ of terms $\sigma_0,\dots,\sigma_{m-1}$ and returns the Godel number of $\varphi(\sigma_0,\dots,\sigma_{m-1})$ is recursive. The proof constructs $s$ as a definition by cases from a more elaborate function $s'$ built using the fact that simultaneous substitution can be expressed via iterated ordinary substitution, following the decomposition described after 10.74.
