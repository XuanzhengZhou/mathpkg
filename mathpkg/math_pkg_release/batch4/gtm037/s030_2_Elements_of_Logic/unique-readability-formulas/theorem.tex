$g^{+*} \operatorname{Fmla}$ is recursive. Moreover, for any expression $\varphi$, if $\varphi \in \operatorname{Fmla}$ then exactly one of the following holds:
\begin{enumerate}
\item[(1)] $\varphi$ is an atomic equality formula, i.e., there are terms $\sigma$ and $\tau$ with $\varphi = (\sigma = \tau)$;
\item[(2)] $\varphi$ is an atomic nonequality formula, i.e., there is an $R \in \operatorname{Dmn} \mathcal{R}$, say with $\mathcal{R}R = m$, and there are $\sigma_0, \ldots, \sigma_{m-1} \in \operatorname{Trm}$ such that $\varphi = \langle R \rangle \sigma_0 \cdots \sigma_{m-1}$;
\item[(3)] $\varphi = \neg \psi$ for some formula $\psi$;
\item[(4)] $\varphi = (\psi \vee \chi)$ for some formulas $\psi, \chi$;
\item[(5)] $\varphi = (\psi \wedge \chi)$ for some formulas $\psi, \chi$;
\item[(6)] $\varphi = \forall \alpha \psi$ for some individual variable $\alpha$ and some formula $\psi$.
\end{enumerate}
Furthermore, if $\varphi$ is a formula and $i < \operatorname{Dmn} \varphi$, then $\langle \varphi_0, \ldots, \varphi_{i-1} \rangle$ is not a formula; and the terms, formulas, etc. asserted to exist in (1)--(6) are uniquely determined by $\varphi$.
