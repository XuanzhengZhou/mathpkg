Let $\mathfrak{A}$ be an $\mathcal{L}$-structure, $n = |A| + |\text{Fmla}_\mathcal{L}|$, and let $\{\varphi_\nu : \nu < \mu\}$ be a set of formulas with $\mu < n^\partial$. Suppose that each finite subset can be simultaneously satisfied. Then in a suitable ultraproduct $\mathfrak{A}^I/D$, there exist elements simultaneously realizing all $\varphi_\nu$.
