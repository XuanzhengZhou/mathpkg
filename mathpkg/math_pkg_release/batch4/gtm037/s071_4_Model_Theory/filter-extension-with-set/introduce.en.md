Proposition 26.30 gives a concrete characterization of membership in the filter generated by adding a single set $Z$ to an existing filter $F$. A set $X$ belongs to $\mathcal{F}_g(F \cup \{Z\})$ if and only if it contains the intersection of some member of $F$ with $Z$. This is a crucial tool for stepwise construction of filters and ultrafilters.
