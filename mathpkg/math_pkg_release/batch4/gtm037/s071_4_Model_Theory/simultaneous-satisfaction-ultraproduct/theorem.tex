Let $(F, 0, D)$ be $m$-consistent over $n$ with $|F| > m$, and let $\mathfrak{A}$ be an $L$-structure with $|A| < n^{\partial}$. Let $\langle c_\gamma : \gamma < \rho \rangle$ be a set of new constant symbols with $\rho < n^{\partial}$, and let $\langle \varphi_\nu : \nu < \mu \rangle$ be a set of formulas in the expanded language such that:

\begin{enumerate}
\item[(i)] for all $\nu_0, \nu_1 < \mu$ there is a $\nu < \mu$ such that $\models \varphi_\nu \rightarrow (\varphi_{\nu_0} \wedge \varphi_{\nu_1})$;
\item[(ii)] for each $\nu < \mu$, the set $\{\delta < n : \mathfrak{A} \models (\exists v_0 \varphi_\nu)[\langle a_{v_j \delta} : j < \omega \rangle]\}$ belongs to $D$.
\end{enumerate}

Then there exist elements $b_\delta \in A$ for $\delta < n$ such that for each $\nu < \mu$,
\[
\{\delta < n : \mathfrak{A} \models \varphi_\nu[\langle a_{v_j \delta} : j < \omega \rangle_{b_\delta}^{v_0}]\} \in D.
\]
