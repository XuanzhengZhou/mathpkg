A triple $(F, G, D)$ is $m$-consistent over $n$ if:
(i) $F$ is a set of functions from $n$ to $n^\partial$, $G$ a set of such functions;
(ii) $D$ is a filter over $n$ generated by a set of cardinality $\leq m$;
(iii) for all $\rho < n^\partial$, all one-one $f \in {}^\rho F$, all $\alpha \in {}^\rho(n^\partial)$, all $n \in \omega$, all one-one $h \in {}^n F$ with $\text{Rng}\,h \cap \text{Rng}\,f = 0$, and all $k \in {}^n G$, the set $n \sim \{\beta < n : \forall \gamma < \rho(f_\gamma \beta = \alpha_\gamma) \text{ and } \forall m < n(h_m \beta = k_m \beta)\}$ is not in $D$.
