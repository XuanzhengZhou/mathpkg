Proposition 28.2 shows that the restriction to formulas with a single free variable in the definition of $\mathfrak{m}$-saturation is inessential. If a structure realizes all consistent $1$-types over parameter sets of size less than $\mathfrak{m}$, then it automatically realizes all consistent types in any finite number of variables over the same parameter sets. The proof uses an inductive construction: given a consistent set of formulas in $k$ variables, one builds a realizing $k$-tuple coordinate by coordinate, at each step applying the $1$-type saturation hypothesis to the set of formulas obtained by existentially quantifying out the remaining variables.
