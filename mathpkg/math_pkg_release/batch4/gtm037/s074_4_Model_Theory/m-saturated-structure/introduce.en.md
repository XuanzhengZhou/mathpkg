An $\mathcal{L}$-structure $\mathfrak{A}$ is **$\mathfrak{m}$-saturated** if it realizes every $1$-type over any parameter set $X \subseteq A$ of size less than $\mathfrak{m}$, where the language is expanded with constants for the parameters. Equivalently, any consistent set of formulas in one free variable with parameters from a "small" subset of $A$ is simultaneously satisfiable in $\mathfrak{A}$.

A structure is **saturated** (without qualification) when it is $|A|$-saturated, meaning it realizes all $1$-types over parameter sets strictly smaller than the universe. The restriction to one-variable formulas is inessential (as shown by Proposition 28.2), while the use of parameters from $A$ is essential for the strength of the notion.
