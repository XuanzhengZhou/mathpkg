This corollary of Rice's theorem states that there is no algorithm to decide whether two given indices $x$ and $y$ code for the same partial recursive function. In other words, the equivalence relation of computing the same function is not decidable. This follows directly from Rice's theorem by considering, for any fixed function $f$, the property $F_f = \{\varphi^1_e : \varphi^1_e = f\}$, which is non-trivial. It is a fundamental undecidability result showing that even the most basic question about programs—whether two programs compute the same function—is algorithmically unsolvable.
