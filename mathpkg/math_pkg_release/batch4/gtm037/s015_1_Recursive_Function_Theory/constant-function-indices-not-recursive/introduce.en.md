This corollary of Rice's theorem states that there is no algorithm to decide whether a given partial recursive function (identified by its index) is a constant function. The property of being constant is a non-trivial semantic property of the computed function: some partial recursive functions are constant and some are not. Rice's theorem therefore implies the index set is non-recursive.
