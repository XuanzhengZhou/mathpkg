For each $m > 0$ there exists an $(m+1)$-ary relation $R_m^m \in \Sigma_m$ which is universal for $m$-ary $\Sigma_m$ relations, i.e., for any $m$-ary relation $S \in \Sigma_m$ there is an $e \in \omega$ such that

$$S = \{(x_0, \ldots, x_{m-1}) : (e, x_0, \ldots, x_{m-1}) \in R_m^{m+1}\}.$$

If $R_m^n$ has been defined for all $n$, let

$$R_m^{n+1} = \{(x_0, \ldots, x_n) : \exists y \in \omega [(x_0, \ldots, x_n, y) \notin R_m^{n+1}]\}.$$
