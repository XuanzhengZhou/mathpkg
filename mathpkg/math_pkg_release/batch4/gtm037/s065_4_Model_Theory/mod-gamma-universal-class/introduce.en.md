This definition introduces the fundamental classification of classes of structures in model theory. A universal class is one axiomatizable by universal sentences, while an elementary (or first-order) class is axiomatizable by arbitrary first-order sentences. The notation $\operatorname{Mod} \Gamma$ denotes the collection of all models of the set of sentences $\Gamma$.
