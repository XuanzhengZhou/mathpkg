For any set $\Gamma$ of sentences, $\operatorname{Mod} \Gamma$ is the class of all models of $\Gamma$. A class $K$ is a \textit{universal class} if $K = \operatorname{Mod} \Gamma$ for some set $\Gamma$ of universal sentences. $K$ is an \textit{elementary class} if $K = \operatorname{Mod} \Gamma$ for some set $\Gamma$ of sentences. We say that $\Gamma$ characterizes $\operatorname{Mod} \Gamma$.
