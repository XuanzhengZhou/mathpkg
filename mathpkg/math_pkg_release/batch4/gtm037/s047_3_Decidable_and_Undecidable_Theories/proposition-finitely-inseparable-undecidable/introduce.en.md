This proposition establishes that every finitely inseparable theory is undecidable. The argument uses the effective inseparability (established in 15.7) of the set of Gödel numbers of logical theorems and the set of sentences that fail in some finite model of $\Gamma$. If $\Gamma$ were decidable, its set of Gödel numbers would be a recursive separator of these effectively inseparable sets, which is impossible. This provides another pathway for proving undecidability, alongside the use of essential undecidability.
