Let $\Gamma$ be an inseparable theory. Then there is a recursive function $h$ such that if $\Delta$ is any consistent extension of $\Gamma$ in $\mathcal{L}$ and if $g^{+*} \Delta = \operatorname{Dmn} \varphi_{e}^{1}$, then $h e = g^{+} \varphi$ for some sentence $\varphi$ such that $\varphi \notin \Delta$ and $\neg \varphi \notin \Delta$.
