Definition 16.36 introduces $\mathcal{L}_7$ as a definitional expansion of $\mathcal{L}_6$ with two new symbols. The individual constant $1$ is defined as $\mathbf{U} 0 = \{0\}$, the set-theoretic ordinal $1$. The binary operation $+$ is defined via the ternary relation $\sigma$ from $\mathcal{L}_6$: when $\sigma(v_0, v_1, v_2)$ holds, $v_0 + v_1 = v_2$ (the usual addition); when $\sigma$ fails for an ordinal-like $v_0$, the result is $\{1\}$; when $\sigma$ fails for a non-ordinal $v_0$, the result is $v_0$ itself. This completes the formal definition of addition within the set-theoretic hierarchy.
