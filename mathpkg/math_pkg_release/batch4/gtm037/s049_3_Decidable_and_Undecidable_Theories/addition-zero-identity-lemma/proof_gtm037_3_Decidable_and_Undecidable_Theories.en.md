Let $x$ be given. If $\Omega x$, then $\sigma(x, 0, x)$ by 16.32, so $x + 0 = x$. If $\Omega x$ fails, then there is no $z$ with $\sigma(x, 0, z)$, and hence $x + 0 = x$ is clear from 16.36(ii).
