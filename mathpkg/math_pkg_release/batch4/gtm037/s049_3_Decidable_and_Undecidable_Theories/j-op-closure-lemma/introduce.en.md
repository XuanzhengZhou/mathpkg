In the formal language $\mathcal{L}_1$, if a set $v_0$ satisfies the predicate $\mathbf{J}$ (intersection formation is possible), then the operation $\mathbf{Op}(v_0, v_1) = v_0 \cup \{v_1\}$ also yields a set satisfying $\mathbf{J}$. This lemma establishes that the $\mathbf{J}$-property is preserved under the successor-like $\mathbf{Op}$ operation.
