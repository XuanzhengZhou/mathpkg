Let $\mathcal{L}$ be a first-order language, as in 10.1. Let $P = \{\varphi : \varphi \text{ is a formula of } \mathcal{L} \text{ whose first symbol is either in } \operatorname{Dom} \mathcal{R}, \text{ or is } L_3 \text{ or } L_4\}$. Let $n, c$ be distinct sets not in $P$. Then $\mathcal{P} = (n, c, P)$ is a sentential language. With each sentence $\varphi$ of $\mathcal{P}$ we associate an expression $\varphi'$ of $\mathcal{L}$:

$$\langle \psi \rangle' = \psi \quad \text{for } \psi \in P;$$
$$(\neg \varphi)' = \neg \varphi';$$
$$(\varphi \rightarrow \psi)' = \varphi' \rightarrow \psi'.$$
