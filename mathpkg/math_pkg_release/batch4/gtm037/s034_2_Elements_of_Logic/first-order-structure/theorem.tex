Let $\mathcal{L} = (L, v, \mathbf{O}, \mathcal{R})$ be a first-order language. An \textit{$\mathcal{L}$-structure} is a triple $\mathfrak{A} = (A, f, R)$ such that:
\begin{enumerate}
\item[(i)] $A \neq \emptyset$;
\item[(ii)] $\operatorname{Dom} f = \operatorname{Dom} \mathbf{O}$, and for each $\mathbf{O} \in \operatorname{Dom} \mathbf{O}$, $f_{\mathbf{O}}$ is an $\mathbf{O}_{\mathbf{O}}$-ary operation on $A$ (with simply $f_{\mathbf{O}} \in A$ if $\mathbf{O}_{\mathbf{O}} = 0$);
\item[(iii)] $\operatorname{Dom} R = \operatorname{Dom} \mathcal{R}$, and for each $\mathbf{R} \in \operatorname{Dom} \mathcal{R}$, $R_{\mathbf{R}}$ is an $\mathcal{R}_{\mathbf{R}}$-ary relation on $A$.
\end{enumerate}
