# Definition 29.18: Descriptive Triple and the Description Operator

A descriptive triple $(L, \tau, O)$ enriches a first-order language $L$ with equality by adding a description operator $\tau$ and a distinguished constant $O$. The term $\tau \alpha \varphi$ is intended to denote "the unique $\alpha$ such that $\varphi$, or $O$ if no such $\alpha$ or more than one exists." This definition extends the usual inductive definition of terms and formulas by adding clause (vii), which allows binding a variable in a formula to produce a term. The description operator is analogous to a definite description operator and, like definitions, can be eliminated from theories.
