Let $(\mathcal{L}, \tau, \mathbf{O})$ be a descriptive triple. Terms and formulas are defined simultaneously:
\begin{itemize}
\item[(i)] any variable is a term;
\item[(ii)] if $\mathbf{O}$ is an operation symbol of rank $m$ and $\sigma_0, \ldots, \sigma_{m-1}$ are terms, then $\mathbf{O}\sigma_0\cdots\sigma_{m-1}$ is a term;
\item[(iii)] if $\sigma$ and $\rho$ are terms, then $\sigma = \rho$ is a formula;
\item[(iv)] if $\mathbf{R}$ is a relation symbol, then $\mathbf{R}\sigma_0\cdots\sigma_{m-1}$ is a formula;
\item[(v)] $\neg\varphi$, $\varphi \lor \psi$, $\varphi \land \psi$ are formulas;
\item[(vi)] $\forall\alpha\varphi$ is a formula;
\item[(vii)] if $\alpha$ is a variable and $\varphi$ is a formula, then $\tau\alpha\varphi$ is a term.
\end{itemize}
