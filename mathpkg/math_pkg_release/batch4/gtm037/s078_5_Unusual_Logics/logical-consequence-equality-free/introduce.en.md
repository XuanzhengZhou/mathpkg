# Proposition 29.10: Logical Consequence with Equality-Free Premises

This proposition establishes that when the equality symbol does not appear in the premises $\Gamma$, the logical consequence relation $\Gamma \models_{L} \varphi$ in a language with equality can be reduced to a consequence relation in the language $L^{e}$ where equality is replaced by an equivalence relation $E$. Specifically, $\Gamma \models_{L} \varphi$ holds if and only if $\Gamma$ together with the translation $E\{\varphi\}$ (which replaces $=$ by $E$) logically implies $\varphi$. This is a key technical lemma for reducing questions about logic with equality to logic without equality.
