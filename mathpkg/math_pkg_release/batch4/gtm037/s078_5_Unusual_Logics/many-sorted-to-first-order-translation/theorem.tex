Let $\mathcal{L}$ be a many-sorted language, $\mathcal{L}^*$ the associated first-order language (with unary predicates $\mathbf{P}_s$ for each sort $s$), and $\mathfrak{A}$ an $\mathcal{L}$-structure. Then $\mathfrak{A}^*$ is a model of the sort axioms $\Gamma$. Furthermore, for any assignment $x$ and formula $\varphi$: $\sigma^{\mathfrak{A}}x = \sigma^{\mathfrak{A}^*}x$ and $\mathfrak{A} \models \varphi[x]$ iff $\mathfrak{A}^* \models \varphi^*[x]$.
