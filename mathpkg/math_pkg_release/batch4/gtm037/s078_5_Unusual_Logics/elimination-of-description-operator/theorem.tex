Let $(\mathcal{L}, \tau, \mathbf{O})$ be a descriptive triple, $\Gamma$ a set of sentences, $\sigma$ a term and $\varphi$ a formula. Then:
\begin{itemize}
\item[(i)] if $v_k$ is a variable not occurring in $\sigma$, there is a formula $\psi$ not involving $\tau$ and with $\operatorname{Fv}\psi \subseteq \operatorname{Fv}\sigma \cup \{v_k\}$, such that $\Gamma \models \sigma = v_k \leftrightarrow \psi$;
\item[(ii)] there is a formula $\varphi^*$ not involving $\tau$ and with $\operatorname{Fv}\varphi^* \subseteq \operatorname{Fv}\varphi$ such that $\Gamma \models \varphi \leftrightarrow \varphi^*$.
\end{itemize}
