This fundamental theorem shows that the description operator $\tau$ is eliminable: any formula involving $\tau$ is equivalent (relative to a theory $\Gamma$) to a $\tau$-free formula. The proof proceeds by simultaneous induction on terms and formulas, replacing $\tau v_i \varphi$ with equivalent descriptions using the uniqueness condition.
