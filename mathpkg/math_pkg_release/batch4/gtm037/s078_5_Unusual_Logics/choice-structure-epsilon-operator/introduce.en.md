Hilbert's $\varepsilon$-operator generalizes the description operator: $\varepsilon v_i \varphi$ is interpreted ambiguously as some $v_i$ satisfying $\varphi$, chosen via a choice function. When multiple witnesses exist, the choice function selects one; when none exist, the distinguished constant $\mathbf{O}$ is used. This operator is powerful enough to define quantifiers: $\exists v_i \varphi$ is equivalent to $\varphi(\varepsilon v_i \varphi)$.
