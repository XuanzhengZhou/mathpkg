Let $(\mathcal{L}, \varepsilon, \mathbf{O})$ be a choice triple. A \textit{choice structure} is a pair $(\mathfrak{A}, f)$ where $\mathfrak{A}$ is an $\mathcal{L}$-structure and $f$ is a choice function for nonempty subsets of $A$. The term evaluation is defined by:
$$\left( \varepsilon v_i \varphi \right)^{\mathfrak{A}f} x = f \{ a \in A : (\mathfrak{A}, f) \models \varphi[x_a^i] \} \quad \text{if this set is nonempty};$$
$$\left( \varepsilon v_i \varphi \right)^{\mathfrak{A}f} x = \mathbf{O}^{\mathfrak{A}} \quad \text{otherwise}.$$
