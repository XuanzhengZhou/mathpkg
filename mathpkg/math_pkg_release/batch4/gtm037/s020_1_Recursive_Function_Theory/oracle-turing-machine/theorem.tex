Let $g : \omega \to \omega$ be a function. A \textbf{$g$-Turing machine} is defined like a Turing machine, except that $v_1, \ldots, v_{2m}$ are arbitrary members of $\{0, 1, 2, 3, 4, 5\}$, and the following stipulation is added: if the symbol under scan is $5$ and either $F(e-1) = 0$ or $F(e) = 1$, then the configuration is unchanged; while if the symbol under scan is $5$ and $0\,1^{(x+1)}$ lies on the tape ending at position $e$, then $0\,1^{(x+1)}\,0\,1^{(g(x)+1)}$ is written on the tape ending at position $e' = e + g(x) + 2$, with the rest of the configuration adjusted accordingly. A function is \textbf{$g$-Turing computable} if it is computable by some $g$-Turing machine.
