A **simple set** is a recursively enumerable set whose complement is infinite yet contains no infinite r.e. subset. This theorem, due to Emil Post, shows that such sets indeed exist. The construction uses a universal function for unary primitive recursive functions to build a partial recursive function $f$ whose range is simple. The proof demonstrates that $\operatorname{Rng} f$ is r.e., intersects every infinite r.e. set, and has an infinite complement — establishing a non-recursive, non-creative r.e. set.
