For each of the following determine if the set in question is recursive, r.e., or has an r.e. complement:

(1) $\{x : \text{there are at least } x \text{ consecutive 7's in the decimal representation of } \pi\}$;

(2) $\{x : \text{there is a run of exactly } x \text{ consecutive 7's in the decimal representation of } \pi\}$;

(3) $\{x : \varphi^1_x \text{ is total}\}$;

(4) $\{x : \operatorname{Dmn} \varphi^1_x \text{ is recursive}\}$.
