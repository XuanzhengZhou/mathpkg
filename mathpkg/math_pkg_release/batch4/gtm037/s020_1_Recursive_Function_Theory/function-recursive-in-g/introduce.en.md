A function $h$ is **$g$-recursive** (or recursive in $g$) if the function $g$ is added as an initial function to the inductive generation of the class of recursive functions. Equivalently, $h$ is $g$-Turing computable: it can be computed by a Turing machine equipped with an oracle for $g$. This notion generalizes to a set $F$ of functions, yielding the concept of a function being recursive in $F$. Most results of standard recursion theory carry over to this relativized setting, forming the basis for the study of Turing degrees.
