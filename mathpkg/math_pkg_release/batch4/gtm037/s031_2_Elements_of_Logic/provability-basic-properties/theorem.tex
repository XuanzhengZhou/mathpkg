Let $\Gamma, \Delta \subseteq \text{Fmla}$ and $\varphi, \psi \in \text{Fmla}$. Then

\begin{itemize}
\item[(i)] if $\Gamma \subseteq \Delta$ and $\Gamma \vdash \varphi$, then $\Delta \vdash \varphi$;
\item[(ii)] if $\Gamma \vdash \varphi$, then $\Theta \vdash \varphi$ for some finite subset $\Theta$ of $\Gamma$;
\item[(iii)] if $\Gamma \vdash \chi$ for each $\chi \in \Delta$ and $\Delta \vdash \varphi$, then $\Gamma \vdash \varphi$.
\end{itemize}
