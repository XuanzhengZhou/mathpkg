Let $g^+$ be a Godel numbering of the language $\mathcal{L}_{\text{nos}}$ in weak second-order logic. The set
$$\{g^+ \psi : \psi \text{ is a sentence of } \mathcal{L}_{\text{nos}} \text{ in weak second-order logic and } Q^w \models \psi\}$$
is not elementarily definable in $(\omega, +, \cdot, s, 0)$.

Consequently, the set $\{g^+ \psi : \models \psi\}$ of weak second-order validities is not elementarily definable in $(\omega, +, \cdot, s, 0)$ and, by 14.16, is not recursively enumerable. Hence there is no reasonable proof theory for weak second-order logic.
