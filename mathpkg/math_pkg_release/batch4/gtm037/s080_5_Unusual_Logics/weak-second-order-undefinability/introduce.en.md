Theorem 30.1 establishes a Tarski-style undefinability result for weak second-order logic: the set of consequences of the weak second-order arithmetic $Q^w$ is not elementarily definable. Consequently, the set of weak second-order validities is not recursively enumerable, so there is no hope of a complete proof theory for this logic.
