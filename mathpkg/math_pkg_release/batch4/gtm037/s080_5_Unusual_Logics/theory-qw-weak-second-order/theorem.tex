Formulated in the language $\mathcal{L}_{\text{nos}}$ (the language of number theory with successor $s$), the theory $Q^w$ consists of the theory $Q$ (Robinson arithmetic) together with the weak second-order sentence:
$$Q^w \quad \forall v_0 \exists F_0 [F_0 v_0 \land \forall v_1 (F_0 s v_1 \rightarrow F_0 v_1)].$$

A structure $\mathfrak{A}$ is a model of $Q^w$ if and only if $\mathfrak{A}$ is isomorphic to $(\omega, +, \cdot, s, 0)$.
