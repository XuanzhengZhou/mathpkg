The theory of types is obtained by iterating the process that leads from first-order to second-order logic, yielding third-order, fourth-order, and so on. The union of all finite-order logics forms the theory of types. In this framework, each variable is assigned a type built inductively from a base type $0$ (the type of individuals), and quantification is permitted over objects of any type. The theory of types provides a unified setting in which higher-order quantification of arbitrary finite order can be studied, and it admits a reasonable proof theory with respect to weakened (general) validity, analogous to Henkin semantics.
