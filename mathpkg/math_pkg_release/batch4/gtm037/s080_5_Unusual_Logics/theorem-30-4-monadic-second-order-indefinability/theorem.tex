Let $g^+$ be a Godel numbering of $\mathcal{L}_{\text{nos}}$ in monadic second-order logic. The set
$$\{g^+ \psi : \psi \text{ is a sentence of } \mathcal{L}_{\text{nos}} \text{ in monadic second-order logic and } Q^m \models \psi\}$$
is not elementarily definable in $(\omega, +, \cdot, s, 0)$.

Consequently, the set $\{g^+ \psi : \models \psi\}$ of monadic second-order validities is not elementarily definable in $(\omega, +, \cdot, s, 0)$ and is not recursively enumerable. Hence there is no reasonable proof theory for monadic second-order logic.
