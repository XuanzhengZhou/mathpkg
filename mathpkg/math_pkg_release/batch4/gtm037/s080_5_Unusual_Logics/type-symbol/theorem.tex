The \textbf{type symbols} are built up from the symbol $(0)$ as follows:
\begin{enumerate}
\item[(1)] $0$ is a type symbol;
\item[(2)] If $\tau_1, \ldots, \tau_n$ are type symbols, then $(\tau_1, \ldots, \tau_n)$ is a type symbol (representing the type of an $n$-ary relation between objects of types $\tau_1, \ldots, \tau_n$).
\end{enumerate}
