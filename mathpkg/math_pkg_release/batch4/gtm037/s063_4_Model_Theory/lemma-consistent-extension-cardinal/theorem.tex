Suppose $\Gamma, \Delta \subseteq \text{Sent}_{\mathcal{L}}$ and:
\begin{enumerate}
\item[(i)] $|\Gamma| \leq |\Delta|$, and $\Gamma \cap \Delta = \emptyset$;
\item[(ii)] $\Delta$ is independent;
\item[(iii)] $\Gamma \subseteq \{\varphi : \Delta \models \varphi\}$.
\end{enumerate}
Then there is an independent set $\Omega \supseteq \Delta$ such that $\Gamma \subseteq \{\varphi : \Omega \models \varphi\}$.
