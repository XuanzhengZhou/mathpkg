The generalized product is a unified framework for constructing product-like structures in model theory, introduced by Feferman and Vaught. It involves two languages: $\mathcal{L}_{\text{fac}}$ for the factor structures and $\mathcal{L}_{\text{ind}}$ for the index set (typically equipped with a Boolean algebra structure). The definition associates to each formula $\varphi$ of $\mathcal{L}_{\text{fac}}$ the set of indices $i$ where $\varphi$ holds. Ordinary direct products arise as a special case when the index structure is the full power set Boolean algebra.
