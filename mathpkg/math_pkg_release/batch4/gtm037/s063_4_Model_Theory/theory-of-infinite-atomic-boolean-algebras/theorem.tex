$\Gamma_{\text{At}}^\infty$ is the extension of $\Gamma_{\text{At}}$ with the following additional axioms (for each $m \in \omega \sim 1$):
$$\exists v_0 \cdots \exists v_{m-1} \bigwedge_{i < j < m} \neg(v_i = v_j).$$
Thus $\mathfrak{A}$ is a model of $\Gamma_{\text{At}}^\infty$ iff $\mathfrak{A}$ is an infinite atomic Boolean algebra.
