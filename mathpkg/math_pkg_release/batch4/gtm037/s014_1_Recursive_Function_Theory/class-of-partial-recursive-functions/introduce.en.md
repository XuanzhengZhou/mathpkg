The class of partial recursive functions is defined as the smallest class of partial functions that contains the successor function and all projection functions, and is closed under composition, primitive recursion, and minimalization. This definition, given as an intersection of all such classes, ensures that partial recursive functions are precisely those functions computable by a Turing machine or any equivalent model of computation.
