Let $f$ be an $(m + 1)$-ary partial function. An $m$-ary partial function $g$ is obtained from $f$ by minimalization provided that for all $x_0, \ldots, x_{m-1} \in \omega$,

$$g(x_0, \ldots, x_{m-1}) \simeq \text{least } y \text{ such that } \forall z \leq y \, ((x_0, \ldots, x_{m-1}, z) \in \text{Dmn } f) \text{ and } f(x_0, \ldots, x_{m-1}, y) = 0.$$

We then write $g(x_0, \ldots, x_{m-1}) \simeq \mu y \, (f(x_0, \ldots, x_{m-1}, y) = 0)$.
