The $\mu$-operator, or minimalization, is a fundamental scheme for generating partial recursive functions. Given an $(m+1)$-ary partial function $f$, the $m$-ary function $g$ obtained by minimalization returns the least $y$ such that $f(x_0,\ldots,x_{m-1},y)=0$, provided $f$ is defined on all earlier arguments. Unlike primitive recursion, minimalization can produce partial functions that are not total, making it essential for defining the class of partial recursive functions.
