There is a Turing machine $M$ with the following property. If $f$ is any unary partial Turing computable function and a Turing machine $N$ computes it, and if $e$ is the Gödel number of $N$, then if $0 \, 1^{(e+1)} \, 0 \, 1^{(x+1)} \, 0$ is placed upon an otherwise blank tape ending at $-1$ and if $M$ is started at $0$, then $M$ will stop iff $x \in \text{Dmn } f$, and if $M$ stops, the final tape description is $0 \, 1^{(fx+1)} \, 0$.
