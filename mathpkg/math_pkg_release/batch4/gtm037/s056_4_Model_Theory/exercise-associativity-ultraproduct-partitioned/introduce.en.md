This exercise establishes an associativity property for ultraproducts. Given a partition of the index set $I$ into blocks $J_k$ and ultrafilters $F_k$ on each block along with an ultrafilter $G$ on the block set $K$, the iterated ultraproduct (first within each block, then across blocks) is isomorphic to the direct ultraproduct over $I$ with the composite ultrafilter $H$ defined in Exercise 18.42.
