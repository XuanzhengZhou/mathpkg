This exercise constructs an ultrafilter $H$ over an index set $I$ from a family of ultrafilters over the blocks of a partition of $I$ and an ultrafilter over the set of blocks. Specifically, a subset $T$ of $I$ belongs to $H$ if the set of blocks $k$ for which $T$ intersects $J_k$ in an $F_k$-large set is $G$-large. The result is a fundamental construction for iterated ultraproducts.
