This theorem establishes a fundamental bridge between syntactic definability in formal theories and computability. If a function $f$ is syntactically definable within a consistent, recursively axiomatizable theory $\Gamma$ that contains the distinctness of numerals, then $f$ must be recursive. The result holds analogously for relations, and provides a method for exhibiting logical characterizations of recursiveness.
