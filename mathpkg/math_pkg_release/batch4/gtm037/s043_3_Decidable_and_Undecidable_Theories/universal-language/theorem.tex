$\mathcal{L}_{\text{un}}$ (the universal language) is an effective first-order language with no operation symbols but such that for each $m \in \omega \setminus \{0\}$ and each $n \in \omega$ there is an $m$-ary relation symbol $\mathbf{R}_n^m$ (with $\mathbf{R}_n^m \neq \mathbf{R}_{n'}^{m'}$ if $m \neq m'$ or $n \neq n'$).
