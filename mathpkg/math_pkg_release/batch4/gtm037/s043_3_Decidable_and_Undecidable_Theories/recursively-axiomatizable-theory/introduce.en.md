A theory $\Gamma$ in an effectivized first-order language $\mathcal{L}$ is recursively axiomatizable if it can be axiomatized by a set of sentences whose Gödel numbers form a recursive set. This notion effectivizes the classical concept of axiomatization by requiring that the set of axioms be computationally decidable. Every finitely axiomatizable theory is automatically recursively axiomatizable, since any finite set of Gödel numbers is recursive.
