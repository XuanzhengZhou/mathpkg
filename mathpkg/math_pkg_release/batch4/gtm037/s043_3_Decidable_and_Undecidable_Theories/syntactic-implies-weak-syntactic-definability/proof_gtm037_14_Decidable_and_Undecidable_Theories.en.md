If $R$ is syntactically defined in $\Gamma$ by a formula $\varphi$, then by the definition of syntactic definability, for any $x_0, \ldots, x_{n-1} \in \omega$, we have $\langle x_0, \ldots, x_{n-1} \rangle \in R$ if and only if $\Gamma \models \varphi(\mathbf{x}_0, \ldots, \mathbf{x}_{n-1})$. The condition $\Gamma \models \varphi(\mathbf{x}_0, \ldots, \mathbf{x}_{n-1})$ is exactly the defining condition for weak syntactic definability, so $\varphi$ also weakly syntactically defines $R$ in $\Gamma$.
