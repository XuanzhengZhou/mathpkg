This theorem is a completeness result for Boolean algebras: every Boolean algebra is isomorphic to a Lindenbaum quotient algebra $\mathfrak{M}_{\Gamma}^{\mathcal{P}}$, where $\mathcal{P}$ is a sentential language and $\Gamma$ is a set of sentences. The proof uses the free algebra property (Theorem 9.58): given any BA $\mathfrak{A}$, take $\mathcal{P}$ with $A$ as propositional variables, extend the identity map to a homomorphism from $\mathfrak{M}_{\mathcal{P}}$ onto $\mathfrak{A}$, then take $\Gamma$ to be the kernel. The quotient $\mathfrak{M}_{\mathcal{P}}/\text{kernel}$ is isomorphic to $\mathfrak{A}$.
