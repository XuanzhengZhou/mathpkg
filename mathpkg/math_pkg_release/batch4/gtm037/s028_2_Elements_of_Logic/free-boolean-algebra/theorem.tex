$\mathfrak{A}$ is freely generated by $X$ provided:

(i) $X$ generates $\mathfrak{A}$;

(ii) if $\mathfrak{B}$ is any Boolean algebra and $f$ is any function mapping $X$ into $B$, then $f$ can be extended to a homomorphism of $\mathfrak{A}$ into $\mathfrak{B}$.

If $\mathfrak{A}$ is freely generated by $X$, we say that $\mathfrak{A}$ is a free Boolean algebra with free generators $X$.
