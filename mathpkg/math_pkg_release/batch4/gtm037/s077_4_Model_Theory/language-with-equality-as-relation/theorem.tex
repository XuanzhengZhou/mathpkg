If $\mathcal{L}$ is a first-order language with equality, we let $\mathcal{L}^{\text{e}}$ be the first-order language without equality in which equality is treated as a new binary relation symbol. Thus $\mathcal{L}^{\text{e}}$ is obtained from $\mathcal{L}$ by moving the equality symbol from the list of logical constants to the list of nonlogical constants.
