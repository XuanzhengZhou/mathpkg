This definition introduces the notation $\mathcal{L}^{\text{e}}$ for the language obtained by treating equality as an ordinary binary relation symbol rather than a logical constant. If $\mathfrak{A}$ is an $\mathcal{L}$-structure, then $(\mathfrak{A}, =)$ is a natural kind of $\mathcal{L}^{\text{e}}$-structure. However, in arbitrary $\mathcal{L}^{\text{e}}$-structures, equality can be interpreted as any binary relation whatsoever, not necessarily the true equality relation.
