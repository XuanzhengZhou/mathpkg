An $\mathcal{L}$-structure $\mathfrak{A}$ is special if and only if either $|A|$ is a successor cardinal, and $\mathfrak{A}$ is saturated, or else $|A|$ is a limit cardinal and there is a sequence $\langle \mathfrak{B}_m : m < |A| \rangle$ such that:

(i) each $\mathfrak{B}_m$ is $m^+$-saturated;
(ii) $\mathfrak{B}_m \preceq \mathfrak{B}_n$ whenever $m < n < |A|$;
(iii) $\mathfrak{A} = \bigcup_{m < |A|} \mathfrak{B}_m$.
