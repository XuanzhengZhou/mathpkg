The **unbounded minimalization** operator, denoted $\mu$, transforms a special $m$-ary function $f$ into an $(m-1)$-ary function $k$ by selecting the least natural number $y$ for which $f(x_0, \ldots, x_{m-2}, y) = 0$. The notation $\mu y\,(f(x_0, \ldots, x_{m-2}, y) = 0)$ expresses this least witness. Unlike bounded minimalization (used in primitive recursion), this operator is unbounded: there is no a priori upper bound on the search for $y$. Together with composition and primitive recursion, minimalization completes the generating operations for the class of general recursive functions.
