If $f$ is a special $m$-ary function, we let

$$k(x_0, \ldots, x_{m-2}) = \text{the least } y \text{ such that } f(x_0, \ldots, x_{m-2}, y) = 0.$$

We write

$$\mu y\,(f(x_0, \ldots, x_{m-2}, y) = 0)$$

for $k(x_0, \ldots, x_{m-2})$. The operation of passing from $f$ to $k$ is called the \textbf{operation of (unbounded) minimalization}.
