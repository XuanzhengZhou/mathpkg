**Lemma 3.5** asserts the existence of a general recursive function that is universal for the class of unary primitive recursive functions. The proof constructs an auxiliary binary function $h$ via a recursion scheme that, while not itself a primitive recursion, can be shown to be general recursive. This lemma is a key step toward separating the primitive recursive functions from the general recursive functions: it provides the universal function needed to apply the diagonal argument of Theorem 3.4, thereby demonstrating the existence of a recursive function that is not primitive recursive.
