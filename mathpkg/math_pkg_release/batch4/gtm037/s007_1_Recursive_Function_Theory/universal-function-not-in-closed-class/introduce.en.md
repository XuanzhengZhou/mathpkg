**Theorem 3.4** is a diagonal argument: if $A$ is a class of number-theoretic functions closed under elementary recursive operations and $f$ is universal for unary members of $A$, then $f$ cannot belong to $A$ itself. The proof constructs $g(m) = f(m,m) + 1$, which belongs to $A$ by closure properties, and then derives a contradiction from the universality assumption. This is an instance of the Cantor diagonal argument and shows, for example, that no class closed under elementary recursive operations can contain a universal function that is itself a member of the class.
