A binary number-theoretic function $f$ is called **universal for unary members of a collection $A$** if every unary function $g \in A$ can be obtained from $f$ by fixing its first argument to some index $m$. That is, $f(m, n) = g(n)$ for all $n$. In computability theory, universal functions serve as enumerations: the first argument acts as an index, and $f$ encodes all unary functions of the class. This concept is central to diagonal arguments showing that certain classes cannot contain their own universal functions.
