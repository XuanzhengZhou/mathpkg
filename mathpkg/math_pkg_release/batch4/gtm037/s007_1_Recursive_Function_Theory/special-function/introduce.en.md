A number-theoretic function $f$ of $m$ arguments ($m > 1$) is called **special** if its last argument can be chosen to make the function value zero, regardless of the values of the preceding arguments. Formally, for every tuple of the first $m-1$ arguments there exists some $y \in \omega$ such that $f(x_0, \ldots, x_{m-2}, y) = 0$. This property is the precondition for applying the unbounded minimalization operator, which selects the least such $y$. The specialness condition ensures that the search for a root of $f$ in its last argument always terminates.
