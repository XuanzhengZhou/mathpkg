The iterated exponential function $a(m, n)$ is defined by $a(m, 0) = m$ and $a(m, n+1) = m^{a(m,n)}$, representing $m$ raised to the $m$ power iterated $n$ times. Lemma 2.36 states that this function is primitive recursive. Although exponentiation is elementary (by Lemma 2.8(v)), iterated exponentiation is not elementary because it grows faster than any elementary function (see Lemma 2.44). This function plays a role analogous to the Ackermann function in demonstrating that the class of primitive recursive functions properly extends the class of elementary functions.
