The course-of-values function $\tilde{f}$ encodes the entire sequence of values $\langle f(x_0, \ldots, x_{m-2}, 0), \ldots, f(x_0, \ldots, x_{m-2}, x_{m-1} - 1) \rangle$ into a single natural number using prime power coding (Gödel numbering). Each value $f(x_0, \ldots, x_{m-2}, i)$ is encoded as the exponent of the $i$-th prime $p_i$, with $+1$ added to distinguish zero values. This construction is fundamental for proving closure properties of primitive recursive function classes, particularly for course-of-values recursion.
