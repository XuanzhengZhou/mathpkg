Lemma 2.38 shows that for any base $m > 1$, the iterated exponential $a(m, n)$ is strictly increasing with the iteration count $n$. This reflects the fact that each additional exponentiation strictly increases the value because $m^{a(m,n)} > a(m,n)$ when $m > 1$. This property is fundamental for comparing the growth rates of iterated exponentials with different bases and iteration depths.
