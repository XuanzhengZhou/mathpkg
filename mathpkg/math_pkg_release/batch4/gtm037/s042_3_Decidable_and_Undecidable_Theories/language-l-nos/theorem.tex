$\mathcal{L}_{\text{nos}}$ is a fixed effective first-order language with the following nonlogical constants:

\begin{itemize}
\item $O$ an individual constant,
\item $s$ a unary operation symbol,
\item $+, \cdot$ binary operation symbols.
\end{itemize}

For each $n \in \omega$, let $\Delta(n + 1) = s\Delta n$ (inductively), with $\Delta 0 = O$; frequently we write $\mathbf{n}$ in place of $\Delta n$. For terms $\sigma, \tau$, the expression $\sigma \leq \tau$ is used as an abbreviation for the formula $\exists v_i(v_i + \sigma = \tau)$, where $i$ is the least integer such that $v_i$ does not occur in $\sigma$ or in $\tau$.
