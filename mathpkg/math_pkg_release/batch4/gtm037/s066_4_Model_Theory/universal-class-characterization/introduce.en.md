Corollary 25.9 gives a concise two-part characterization of universal classes: a class $K$ of structures is universal if and only if it is closed under substructures ($SK = K$) and is axiomatizable by some set $\Gamma$ of sentences ($K = \mathrm{Mod}\,\Gamma$). This follows from Theorem 25.2 (Łoś's theorem on universal classes) and provides a practical criterion for verifying whether a given class of structures is universal.
