Proposition 2.8 establishes that many common number-theoretic functions are elementary. The list includes the constant functions, the successor, sign, and cosign functions, as well as exponentiation, factorial, and the predecessor function. The proof proceeds by showing each function can be obtained from the basic elementary functions using operations that preserve elementariness -- composition, summation, and multiplication.
