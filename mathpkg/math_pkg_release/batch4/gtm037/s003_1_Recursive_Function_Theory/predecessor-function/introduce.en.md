The predecessor function $\mu$ maps each natural number to its predecessor, with the convention that the predecessor of $0$ is $0$. This is a basic elementary function (item (iii) in the list following Proposition 2.8) and serves as a fundamental building block in recursive function theory. Together with the successor function $\sigma$, it provides the basic arithmetic operations on natural numbers.
