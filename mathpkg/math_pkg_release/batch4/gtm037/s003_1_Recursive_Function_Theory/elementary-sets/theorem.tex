The following sets are elementary:
\begin{itemize}
\item[(i)] $0$ (the empty set), with characteristic function $\chi_0 = C_0^1$;
\item[(ii)] $\omega$ (the set of all natural numbers), with characteristic function $\chi_\omega = C_1^1$;
\item[(iii)] For any $x \in \omega$, the singleton $\{x\}$ is elementary, with characteristic function
$$\chi_{\{x\}}(y) = \overline{\text{sg}}(|x - y|) = \overline{\text{sg}}(|C_x^1 y - U_0^1 y|)$$
for all $y \in \omega$.
\end{itemize}
