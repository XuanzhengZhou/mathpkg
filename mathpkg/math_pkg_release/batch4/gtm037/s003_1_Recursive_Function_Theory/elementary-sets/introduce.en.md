Proposition 2.10 establishes that the most basic sets of natural numbers -- the empty set, the full set $\omega$, and every singleton $\{x\}$ -- are all elementary. This means that membership in each of these sets can be decided by an elementary function. The result is a simple but foundational observation: the most basic effectively decidable sets are indeed captured by the class of elementary relations.
