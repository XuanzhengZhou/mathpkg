The function $g$ can be expressed as a composition:
$$g = K_m^m(f; U_{\pi 0}^m, \ldots, U_{\pi(m-1)}^m).$$
Since $\mathcal{A}$ contains the projection functions $U_j^m$ (as basic elementary functions 2.1(3)) and is closed under composition, $g\in\mathcal{A}$.
