A number-theoretic function $f$ is elementary if and only if there is a finite sequence $\langle g_0,\ldots,g_{k-1}\rangle$ of number-theoretic functions such that $g_{k-1}=f$, and for each $i<k$ one of certain conditions holds: $g_i$ is one of the basic elementary functions (successor $+$, and the functions listed in Definition 2.1), or $g_i$ is obtained from earlier functions in the sequence by the operations of composition, summation, or multiplication.
