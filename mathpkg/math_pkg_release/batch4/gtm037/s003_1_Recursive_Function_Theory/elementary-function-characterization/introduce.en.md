Proposition 2.3 gives a characterization of elementary number-theoretic functions. A function is elementary precisely when it can be obtained as the last term of a finite sequence of functions, where each function in the sequence is either one of the basic elementary functions (the constant zero function, the successor function, and the projection functions) or is obtained from earlier functions by the permitted operations (composition, summation, and multiplication). This provides a generative or inductive description of the class of elementary functions.
