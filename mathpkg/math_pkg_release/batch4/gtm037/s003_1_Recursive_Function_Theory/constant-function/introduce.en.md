The constant function $C_m^n$ is the $n$-ary number-theoretic function that always returns the fixed value $m$, regardless of its inputs. These functions serve as basic building blocks in the theory of recursive and elementary functions. For $n \neq 0$, every constant function $C_m^n$ is elementary, as shown by an inductive argument using the zero function, successor, and addition.
