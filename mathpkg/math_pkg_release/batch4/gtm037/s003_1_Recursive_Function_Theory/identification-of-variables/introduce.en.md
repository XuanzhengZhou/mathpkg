Proposition 2.5 states that if a class is closed under elementary recursive operations, then identifying variables (replacing the last argument of an $m$-ary function with its first argument) produces an $(m-1)$-ary function still in the class. By combining this with Proposition 2.4 (permutation of variables), variables can be identified in an arbitrary number of places. For example, if $f$ is 3-ary elementary, then $f(x,x,y)$ and $f(x,y,x)$ are also elementary.
