\text{Theorem 19.13 (Henkin's Embedding Theorem).}\quad
\text{Let } \mathfrak{A} \text{ be an } \mathcal{L}\text{-structure and let } \mathbf{K} \text{ be a class of } \mathcal{L}\text{-structures closed under isomorphism.}
\text{Suppose that every finitely generated substructure of } \mathfrak{A} \text{ can be embedded into some member of } \mathbf{K}.
\text{Then } \mathfrak{A} \text{ can be embedded into some member of } \mathbf{K}.
