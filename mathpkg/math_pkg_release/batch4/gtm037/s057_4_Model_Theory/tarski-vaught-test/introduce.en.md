The Tarski--Vaught test gives a useful criterion for checking whether one structure is an elementary substructure of another, while only needing to evaluate satisfaction in the *larger* structure. If $\mathfrak{A} \subseteq \mathfrak{B}$, then $\mathfrak{A} \preceq \mathfrak{B}$ exactly when every existential statement witnessed in $\mathfrak{B}$ using parameters from $A$ can be witnessed by some element already in $A$. This eliminates the need to check satisfaction in $\mathfrak{A}$ directly, which is particularly convenient for constructing elementary substructures via Skolem hull arguments (as in the downward Lowenheim--Skolem theorem).
