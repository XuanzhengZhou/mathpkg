A modal language extends a sentential language by adding a necessity symbol $d$, written as $\Box$ (box operator). The language is defined as a quadruple $(n, c, d, P)$ where $n$ is negation, $c$ is implication, $d$ is necessity, and $P$ is the set of sentence symbols. For any expression $\varphi$, $\Box\varphi$ intuitively means that $\varphi$ is necessarily true, i.e., true in all possible worlds.
