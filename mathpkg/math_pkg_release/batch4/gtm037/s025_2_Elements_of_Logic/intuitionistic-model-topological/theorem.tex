Given a sentential language $\mathcal{P}$, an intuitionistic model for $\mathcal{P}$ is a function $f$ mapping $P$ into the collection of open sets in some topological space $X$. Given such a function $f$, we define

$$f^{+} \langle s \rangle = fs \text{ for all } s \in P;$$
$$f^{+} \neg \varphi = \text{interior of } (X \sim f^{+} \varphi);$$
$$f^{+} (\varphi \wedge \psi) = f^{+} \varphi \cap f^{+} \psi;$$
$$f^{+} (\varphi \rightarrow \psi) = \text{interior of } ((X \sim f^{+} \varphi) \cup f^{+} \psi).$$
