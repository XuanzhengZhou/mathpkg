For any tape description $F$ and any $e \in \mathbb{Z}$,
\begin{enumerate}
\item[(i)] if $Fe = 0$, then $\langle (F, 0, e), (F_1^e, 0, e) \rangle$ is a computation of $T_1$;
\item[(ii)] if $Fe = 1$, then $\langle (F, 0, e) \rangle$ is a computation of $T_1$.
\end{enumerate}
$T_1$ writes a $1$ if a $1$ is not there, but does not move the tape.
