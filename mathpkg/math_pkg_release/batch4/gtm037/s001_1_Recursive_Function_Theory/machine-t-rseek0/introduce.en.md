Machine $T_{r\text{seek}0}$ is a two-state Turing machine that searches rightward on the tape for the first occurrence of the symbol $0$. It moves right across any $1$'s it encounters and halts when it reaches a $0$ cell. Together with $T_{l\text{seek}0}$, it provides bidirectional search capability for the symbol $0$ in the construction of more complex Turing machines.
