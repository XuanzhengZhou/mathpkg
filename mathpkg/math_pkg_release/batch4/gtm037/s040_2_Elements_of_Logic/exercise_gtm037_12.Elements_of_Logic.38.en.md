If $\mathfrak{A}$ is a $\mathrm{CA}_\alpha$, $\alpha < \omega$, and $|A| > 1$, then $\mathfrak{A} \times \mathfrak{A}$ (understood in the natural sense) is not isomorphic to a cylindric set algebra. Hint: in a cylindric set algebra $\mathfrak{B}$ of dimension $\alpha$, if $x \neq 0$ then $c_0 \cdots c_{\alpha-1}x = 1$.
