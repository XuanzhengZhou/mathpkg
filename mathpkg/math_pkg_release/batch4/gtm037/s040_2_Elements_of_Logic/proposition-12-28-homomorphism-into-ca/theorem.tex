Let $\mathcal{L} = (L, v, 0, \mathcal{R})$ be a first-order language with no operation symbols. Suppose $\mathfrak{A}$ is a $\mathrm{CA}_\omega$, and $f \colon \operatorname{Dmn} \mathcal{R} \to A$ is a function such that $\Delta f_{\mathbf{R}} \subseteq \mathcal{R}_{\mathbf{R}}$ for all $\mathbf{R} \in \operatorname{Dmn} \mathcal{R}$. Then there is a homomorphism $g$ from $\mathfrak{M}_0^{\mathcal{L}}$ into $\mathfrak{A}$ such that $g[\mathbf{R}v_0 \cdots v_{m-1}] = f_{\mathbf{R}}$ for each $\mathbf{R} \in \operatorname{Dmn} \mathcal{R}$ (with, say, $\mathcal{R}_{\mathbf{R}} = m$).
