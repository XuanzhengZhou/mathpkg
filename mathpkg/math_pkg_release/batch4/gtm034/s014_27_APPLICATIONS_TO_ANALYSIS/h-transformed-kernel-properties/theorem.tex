Let $Q$ be a transient kernel on $S$ and $h$ a positive $Q$-regular function on $S$.

\textbf{P2} $Q^h$ is a transient kernel on $S$. Moreover, if we define its iterates by
$$Q_0^h(x,y) = \delta(x,y), \quad Q_{n+1}^h(x,y) = \sum_{t \in S} Q_n^h(x,t)Q^h(t,y),$$
and its Green function by
$$g^h(x,y) = \sum_{n=0}^{\infty} Q_n^h(x,y), \quad x,y \in S,$$
then this coincides with the function $g^h(x,y)$ defined in D4. Furthermore, the constant function $e(x) \equiv 1$ is $Q^h$-regular:
$$\sum_{y \in S} Q^h(x,y) = 1, \quad x \in S.$$
