\textbf{E3 (Theorem).} Let $P(x,y)$ be the transition function of a recurrent aperiodic random walk in one dimension. Then the Wiener-Hopf equation
$$\sum_{y=0}^{\infty} P(x,y)f(y) = f(x), \quad x = 0, 1, 2, \dots,$$
has a unique non-negative solution. It is given by
$$f(x) = u(0) + u(1) + \cdots + u(x), \quad x \geq 0,$$
where $u(x)$ is the function defined in D18.2.
