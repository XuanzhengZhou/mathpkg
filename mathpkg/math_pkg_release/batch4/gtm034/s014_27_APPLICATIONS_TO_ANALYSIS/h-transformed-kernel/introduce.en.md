Given a transient kernel $Q$ and a strictly positive $Q$-regular function $h$, the $h$-transform produces a new transient kernel $Q^h$ by reweighting each transition $Q(x,y)$ by the ratio $h(y)/h(x)$. The corresponding Green function transforms analogously: $g^h(x,y) = g(x,y)h(y)/h(x)$. This device, known today as the Doob $h$-transform, is a cornerstone of probabilistic potential theory. Its key property -- that the constant function $e(x) \equiv 1$ is $Q^h$-regular -- makes it an essential tool for reducing questions about arbitrary regular functions to the study of constants.
