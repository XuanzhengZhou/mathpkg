Three elementary facts about the function classes defined in D2 are collected here. First, every $Q$-potential is automatically $Q$-excessive, because the Green function satisfies $Q g(x,y) - g(x,y) = -\delta(x,y)$. Second, the minimum of two excessive functions remains excessive by a simple inequality argument. Third, potentials decay to zero under repeated application of $Q$: $Q_n f \to 0$ pointwise as $n \to \infty$, because the tail of the Green function series vanishes. These observations are used repeatedly in the proofs of T1 and P3.
