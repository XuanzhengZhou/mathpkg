Let $Q$ be a transient kernel on $S \times S$, and let $f$ be a $Q$-excessive function.

\textbf{P1} The following elementary properties hold:

\begin{enumerate}
\item[(1)] Potentials are excessive.
\item[(2)] The minimum of two excessive functions is excessive.
\item[(3)] If $f(x)$ is a potential, then
$$\lim_{n \to \infty} Q_n f(x) = 0, \quad x \in S.$$
\end{enumerate}
