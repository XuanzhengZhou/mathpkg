If $Q$ is a transient kernel on $S \times S$, then a non-negative function $f(x)$ on $S$ is called (a) $Q$-regular, (b) $Q$-excessive, (c) a $Q$-potential, if

(a) $$Qf(x) = f(x), \quad x \in S,$$

(b) $$Qf(x) \leq f(x), \quad x \in S,$$

(c) $$f(x) = \sum_{y \in S} g(x,y)\psi(y) < \infty, \quad x \in S$$

where $\psi(x) \geq 0$ for $x \in S$. In case (c) $f$ is called the potential of the charge $\psi$.
