Proposition P5 establishes an exponential-type lower bound for the return probabilities $F_n$ of a strongly aperiodic recurrent one-dimensional random walk. The elegant combinatorial proof uses cyclic permutations of the increments to construct a polygonal path lying above its chord, ensuring a first return at time $n$. This bound is essential for controlling the error terms in the subsequent estimates P6-P8 that prove Kesten's theorem T2.
