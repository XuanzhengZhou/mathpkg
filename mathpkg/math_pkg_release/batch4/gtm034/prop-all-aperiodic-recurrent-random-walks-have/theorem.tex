All aperiodic recurrent random walks have the property that $a(x) > 0$ for every $x \neq 0$ with one single exception. This exception is left- or right-continuous random walk whose variance $\sigma^2$ is infinite.

In the left-continuous case with $\sigma^2 = \infty$

$$a(x) = 0 \text{ for } x \geq 0 \quad \text{and} \quad a(x) > 0 \text{ for } x < 0,$$

and in the right-continuous case the situation is of course reversed.

$$\lim_{x \to +\infty} \frac{a(x)}{x} = 0$$
