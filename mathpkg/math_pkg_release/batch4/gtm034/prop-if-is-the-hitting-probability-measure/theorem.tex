If $H_B(x,y)$ is the hitting probability measure of a finite, nonempty subset $B \subset R$ for a random walk of type II, then both limits

(a) $\lim_{x \to -\infty} H_B(x,y) = H_B(-\infty,y)$

(b) $\lim_{x \to +\infty} H_B(x,y) = H_B(+\infty,y)$

exist for every $y$ in $B$.
