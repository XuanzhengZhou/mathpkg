If $P(x,y)$ is a random walk with state space $R$, then the random walk with transition function $P^*(x,y) = P(y,x)$ and with the same state space is called the reversed random walk. We shall denote by $G^*(x,y), Q_n^*(x,y), H_A^*(x,y), \Pi_A^*(x,y), g_A^*(x,y), \text{etc.}, \ldots$ the functions defined in D1 for the reversed random walk.
