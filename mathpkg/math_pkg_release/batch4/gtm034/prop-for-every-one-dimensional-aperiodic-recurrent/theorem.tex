For every one-dimensional aperiodic recurrent random walk,

(a) $\lim_{n \to \infty} [a_n(x) + a_n(-x)] < \infty$ exists for each $x$ in $R$. Furthermore

(b) $\lim_{z \to +\infty} \frac{1}{x} \lim_{n \to \infty} [a_n(x) + a_n(-x)] = \frac{2}{\sigma^2} (= 0 \text{ if } \sigma^2 = +\infty)$,

where

$$\sigma^2 = \sum_{z=-\infty}^{\infty} x^2 P(0,x).$$
