A random walk $P(x,y)$ defined on $R$ is aperiodic if $R = \bar{R}$.

If a random walk is periodic (i.e., not aperiodic), then the problem is badly posed. In other words, if $R \neq \bar{R}$, then the random walk is defined on the wrong group, or on a coset of the wrong group if it starts at a point $x$ not in $\bar{R}$. It will always be possible to reduce problems where $R \neq \bar{R}$ to the aperiodic case, for the simple reason that in every case $\bar{R}$ happens to be group theoretically isomorphic to $R$ of some dimension $d \geq 0$. (See Chapter II, section 7, for a more detailed discussion of such considerations.)
