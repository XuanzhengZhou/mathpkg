One-dimensional random walk with finite absolute mean $m$ is recurrent if and only if the mean $\mu$ is zero.

The ideas of this section are now applied to obtain an amusing generalization of certain properties of Bernoulli random walk.

E1 It was shown in E1.2 that Bernoulli random walk with $p > q$ has the property that

(1) $$G(0,x) = \frac{1}{p - q} \text{ for } x \geq 0.$$

This result can be extended to arbitrary right-continuous random walk with positive mean in a surprisingly interesting fashion; we shall assume that

$$P(0,x) = 0 \text{ for } x \geq 2, \quad 0 < \mu = \sum_{x=-\infty}^{1} xP(0,x)$$

and prove that for such a random walk

(2) $$G(0,0) = G(0,1) = G(0,2) = \cdots = \frac{1}{\mu}.$$

This is a clear-cut generalization of (1) where $\mu$ happens to have the value $p - q$.
