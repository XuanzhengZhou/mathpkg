For aperiodic transient random walk in dimension \(d = 1\) with finite first absolute moment \(m = \sum |x| P(0,x) < \infty\) and mean \(\mu = \sum x P(0,x)\):
\[
\lim_{x \to +\infty} G(0,x) = \frac{1}{\mu}, \quad \lim_{x \to -\infty} G(0,x) = 0 \quad \text{if } \mu > 0,
\]
with limits reversed if \(\mu < 0\) (and \((-\mu)^{-1}\) replacing \(\mu^{-1}\)).
