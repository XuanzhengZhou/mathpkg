(The skeptical reader will observe that we are indeed about to introduce the notion of measure—but measure of a very simple sort, the total number of elementary events being countably infinite.)

Let us define as the set of "elementary events" the set of sequences $\omega$ of the form $\omega = \{x_0, x_1, x_2, \ldots, x_{n-1}, x_n\}$ where $x_0 = x$, and where $x_1, x_2, \ldots, x_n$ may assume any value in $R$. Since $R$ is countable, this set of sequences, which we denote $\Omega_n$, is also countable. With each $\omega$ in $\Omega_n$ we associate the measure

$$p(\omega) = P(x, x_1)P(x_1, x_2) \ldots P(x_{n-1}, x_n).$$

It follows from D1 and P1 that

$$\sum_{[\omega] \in \Omega_n, x_n = y} p(\omega) = P_n(x,y), \quad \sum_{\omega \in \Omega_n} p(\omega) = \sum_{y \in R} P_n(x,y) = 1.$$

On the other hand, if

$$A_k = [\omega \mid \omega \in \Omega_n; x_1 \neq y, x_2 \neq y, \ldots, x_{k-1} \neq y, x_k = y], \quad 1 \leq k \leq n,$$

then the sets $A_k$ are disjoint subsets of $\Omega_n$, and it is obvious from D2 that

$$F_k(x,y) = \sum_{\omega \in A_k} p(\omega), \quad 1 \leq k \leq n.$$

The $A_k$ being disjoint, one obtains

$$\sum_{k=1}^{n} F_k(x,y) \leq \sum_{\omega \in \Omega_n} p(\omega) = 1.$$
