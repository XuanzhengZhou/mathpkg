The non-negative solutions of

(1) $$\sum_{y \in R} P(x,y)f(y) - f(x) = \delta(x,0), \quad x \in R$$

are exactly the following functions: if $d = 2$ or if $d = 1$ and $\sigma^2 = \infty$, then

$$f(x) = a(x) + c, \text{ where } c \geq 0;$$

if $d = 1$ and $\sigma^2 < \infty$, then

$$f(x) = a(x) + \alpha x + c$$

where $-(\sigma^2)^{-1} \leq \alpha \leq (\sigma^2)^{-1}$ and $c \geq 0.$
