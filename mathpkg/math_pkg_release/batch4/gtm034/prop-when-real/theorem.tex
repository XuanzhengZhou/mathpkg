$$\left| \sum_{1}^{\infty} \frac{t^k}{k} E[z^{S_k}; S_k > 0] \right| \leq \sum_{1}^{\infty} \frac{t^k}{k} P[S_k > 0] \leq \ln \frac{1}{1-t}$$
