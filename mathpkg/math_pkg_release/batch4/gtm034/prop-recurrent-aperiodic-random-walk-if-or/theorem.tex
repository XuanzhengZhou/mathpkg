Recurrent aperiodic random walk. If $d = 2$ or if $d = 1$ and $\sigma^2 = \infty$, then $a(x)$ is a non-negative solution of

$$\sum_{y \neq 0} P(x,y)f(y) - f(x) = 0, \quad x \neq 0.$$

If $d = 1$ and $\sigma^2 < \infty$, then the function $f(x) = a(x) + \alpha x$ is a non-negative solution of (1) if and only if the real parameter $\alpha$ lies in the interval $-(\sigma^2)^{-1} \leq \alpha \leq (\sigma^2)^{-1}$.
