Let the random walk $\mathbf{x}_n$ start at $\mathbf{x}_0 = 0$. Define $\mathbf{T}_N$ as the first time outside $[-N,N]$, and for a fixed point $y \in \mathbb{R}$ define the occupation count
$$\mathbf{N}_N(y) = \sum_{k=0}^{\mathbf{T}_N - 1} \delta(y, \mathbf{x}_k),$$
the number of visits to $y$ before leaving the interval $[-N,N]$.
