For a one-dimensional random walk, the interval Green function $g_N$ and the half-line Green function $g$ satisfy the decomposition
$$g_N(x,y) = g(x,y) - \sum_{k=1}^{\infty} R_N(x,k) g(N+k, y), \quad x, y \in [0,N],$$
where $g(x,y) = g_B(x,y)$ with $B = \{x \mid -\infty < x \leq -1\}$ is the Green function of the half-line.
