For simple symmetric random walk restricted to $[0,N]$, the $(N+1) \times (N+1)$ matrix $Q_N$ has eigenvalues
$$\lambda_k = \cos \frac{k+1}{N+2} \pi, \quad k = 0, 1, \ldots, N,$$
with associated eigenvectors
$$v_k(x) = \sqrt{\frac{2}{N+2}} \sin \frac{k+1}{N+2} \pi (x+1), \quad k = 0, 1, \ldots, N, \quad 0 \leq x \leq N,$$
satisfying
$$Q_N v_k = \lambda_k v_k, \quad (v_k, v_m) = \sum_{x=0}^{N} v_k(x) v_m(x) = \delta(k,m).$$
