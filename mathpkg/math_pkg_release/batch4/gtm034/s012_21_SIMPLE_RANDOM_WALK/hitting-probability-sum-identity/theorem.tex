For a one-dimensional random walk with mean $\mu = 0$, the hitting probabilities for the interval $[0,N]$ satisfy:
$$\sum_{k=1}^{\infty} R_N(x,k) + \sum_{k=1}^{\infty} L_N(x,k) = 1, \quad 0 \leq x \leq N,$$
$$\sum_{k=1}^{\infty} (N+k) R_N(x,k) + \sum_{k=1}^{\infty} (-k) L_N(x,k) = x, \quad 0 \leq x \leq N.$$
