For a random walk with restricted transition function $Q_N$, the \textbf{Green function of the interval} $[0,N]$ is defined by
$$g_N(x,y) = \sum_{n=0}^{\infty} Q_N^n(x,y), \quad x,y \in [0,N].$$

Equivalently, in the terminology of the Green function of a general set,
$$g_N(x,y) = g_{R-[0,N]}(x,y).$$
