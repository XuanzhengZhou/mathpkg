If $P(x,y)$ is the transition function of one-dimensional random walk with mean $\mu = 0$, and variance $\sigma^2 = m_2 < \infty$, then

$$\lim_{n \to \infty} \sum_{z < \sqrt{n}\sigma t} P_n(0,x) = F(t),$$

where

$$F(t) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^{t} e^{-x^2/2} dx, \quad -\infty < t < \infty.$$
