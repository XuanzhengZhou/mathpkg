The Lebesgue integral satisfies these seven properties:

(1) If $c$ is a constant function,

$$\int_E cd\mu = c\mu(E).$$

(2) If $f$ and $g$ are measurable functions whose integrals are defined on $E$ and if $f(x) \leq g(x)$ for all $x \in E$, then

$$\int_E fd\mu \leq \int_E gd\mu.$$

(3) If $f$ is integrable on $E$ and if $c$ is a real number, then $cf$ is integrable and $$\int_E cfd\mu = c \int_E fd\mu.$$

(4) If $f$ is measurable and $\mu(E) = 0$, then $$\int_E fd\mu = 0.$$

(5) If $E'$ and $E$ are measurable sets with $E' \subset E$ and if $f$ is a function for which $$\int_E fd\mu$$ is defined, then $$\int_E' fd\mu$$ is defined. In particular,

$$\int_{E'} f^+ d\mu \leq \int_E f^+ d\mu$$
