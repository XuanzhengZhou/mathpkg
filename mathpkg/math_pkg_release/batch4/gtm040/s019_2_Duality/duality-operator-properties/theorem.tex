Let $X$ and $Y$ be square matrices, row vectors, or column vectors indexed by the set of states for a Markov chain $P$. Suppose $\hat{P}$ is the $\alpha$-dual of $P$. Then
\begin{enumerate}
\item[(1)] $\text{dual dual } X = X$.
\item[(2)] $\text{dual}(X + Y) = \text{dual } X + \text{dual } Y$.
\item[(3)] $\text{dual}(cX) = c \, \text{dual } X$.
\item[(4)] $\text{dual}(XY) = \text{dual } Y \; \text{dual } X$.
\item[(5)] $\text{dual } I = I$ and $\text{dual } 0 = 0$.
\item[(6)] If $X \geq 0$, then $\text{dual } X \geq 0$; and if $X > 0$, then $\text{dual } X > 0$.
\item[(7)] If $X \geq Y$, then $\text{dual } X \geq \text{dual } Y$; and if $X > Y$, then $\text{dual } X > \text{dual } Y$.
\item[(8)] If $f$ is a $P$-superregular (or subregular) column vector, then $\text{dual } f$ is a $\hat{P}$-superregular (or subregular) row vector; and if $\beta$ is a $P$-superregular (or subregular) row vector, then $\text{dual } \beta$ is a $\hat{P}$-superregular (or subregular) column vector.
\item[(9)] $\text{dual } 1 = \alpha$ and $\text{dual } \alpha = 1$. The measure $\alpha$ is $\hat{P}$-superregular.
\item[(10)] If $\lim_n X^{(n)} = X$, then $\lim_n \text{dual } X^{(n)} = \text{dual } X$.
\end{enumerate}
