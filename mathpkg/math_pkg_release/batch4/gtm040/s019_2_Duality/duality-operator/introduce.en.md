The duality operator extends the $\alpha$-dual construction from transition matrices to arbitrary matrices, row vectors, and column vectors indexed by the state space. For any matrix or vector $X$, the dual is defined as $\text{dual } X = D X^T D^{-1}$, where $D$ is the diagonal matrix with entries $1/\alpha_i$. This operator formalizes the correspondence between row vectors (measures) and column vectors (functions), converting proofs about functions into proofs about measures, and provides a powerful algebraic tool for deriving new identities by duality.
