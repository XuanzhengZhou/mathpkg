By Lemma 6-6,

$$P^E = T + U \left( \sum Q^n \right) R.$$

Hence, using the properties of the dual operator from Proposition 6-14,

$$\text{dual } P^E = \text{dual } T + (\text{dual } R) \left( \sum (\text{dual } Q)^n \right) (\text{dual } U)$$

$$= \hat{T} + \hat{U} \left( \sum \hat{Q}^n \right) \hat{R}$$

$$= \hat{P}^E.$$

The last equality follows from applying Lemma 6-6 to the dual chain $\hat{P}$ with the same set $E$.
