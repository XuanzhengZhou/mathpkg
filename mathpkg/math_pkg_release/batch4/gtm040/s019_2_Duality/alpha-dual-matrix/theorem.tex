Let $P$ be an arbitrary Markov chain transition matrix and suppose there exists a positive finite-valued $P$-superregular measure $\alpha$. The $\alpha$-dual matrix of $P$ is a matrix $\hat{P}$ defined by
$$\hat{P}_{ij} = \frac{\alpha_j P_{ji}}{\alpha_i}.$$
Let $D$ be a diagonal matrix with diagonal entries $1/\alpha_i$. We note that $\hat{P} = D P^T D^{-1}$.
