If $\mu \in \mathcal{G}_V$, $\iota = (\kappa^0, \kappa^1, \ldots) \in \Omega$, then

$$
\frac{\mu([\kappa^0, \kappa^1, \ldots, \kappa^n])}{\nu([\kappa^0, \kappa^1, \ldots, \kappa^n])}
= \frac{\mu([\kappa^m, \ldots, \kappa^n])}{\nu([\kappa^m, \ldots, \kappa^n])},
\qquad 1 \leq m \leq n < \infty,
$$

where $\nu \in \mathcal{G}_V$ is a fixed reference measure.
