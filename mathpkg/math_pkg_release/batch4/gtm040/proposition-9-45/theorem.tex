In a normal chain

$$G_{ik} \frac{\alpha_j}{\alpha_k} + G_{kj} - G_{ij} = kN_{ij}$$

$$C_{ik} \frac{\alpha_j}{\alpha_k} + C_{kj} - C_{ij} = kN_{ij}$$

$$G_{ik} \frac{\alpha_i}{\alpha_k} + G_{ki} = kN_{ii}$$

$$C_{ik} \frac{\alpha_i}{\alpha_k} + C_{ki} = kN_{ii}.$$

PROOF: The first two expressions follow from Theorem 9-7 and Definition 9-24. For the other two, set $i = j$.
