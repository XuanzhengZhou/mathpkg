Lemma 4-19 presents five identities that connect the hitting probability matrix $H$ and the first-return probability matrix $\bar{H}$. These identities describe the probabilities of various return and visit events: repeated returns to a state, returns before hitting another state, returns via a specific state, first reaching after multiple returns, and the probability of being in a state many times. All five proofs rely on the strong Markov property (Theorem 4-10), demonstrating its power in analyzing Markov chain path probabilities.
