Proposition 4-30 resolves the question of when closed classes are recurrent: finiteness suffices. If a closed equivalence class has only finitely many states, it must be recurrent. The proof uses a counting argument: if the class were transient, the expected total number of steps spent in the class would be finite, but since the class is closed the process is trapped there forever, yielding a contradiction. For infinite closed classes, as demonstrated by the basic example (random walk), recurrence is not guaranteed.
