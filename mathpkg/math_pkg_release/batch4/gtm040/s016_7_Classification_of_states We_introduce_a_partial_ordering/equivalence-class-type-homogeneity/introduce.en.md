Proposition 4-24 is the fundamental result of state classification: within each communication equivalence class, every state shares the same type (recurrent or transient). This is the culmination of the preceding lemmas. If one state in a class is recurrent, Lemma 4-23 forces all communicating states to also be recurrent. Conversely, if one state is transient, the entire class must be transient. This justifies speaking of "recurrent classes" and "transient classes" rather than individual recurrent or transient states.
