Lemma 8-64 provides two practical sufficient conditions under which the equality $h = Nf$ holds in the potential theory of denumerable Markov chains. The first condition involves equilibrium sets: if the random variable $z$ vanishes on all paths that avoid a given equilibrium set $E$, then the representation holds. The second condition applies to absorbing chains where $z$ vanishes on paths originating in the absorbing state.
