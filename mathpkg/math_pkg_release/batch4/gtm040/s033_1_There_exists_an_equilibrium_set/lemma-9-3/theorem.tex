Let $P$ be any Markov chain with a positive superregular measure $\alpha$. If $i$ and $j$ communicate, then for all $n$,
$$\frac{N_{ij}^{(n)}}{N_{jj}^{(n)}} \leq 1, \qquad \frac{N_{ij}^{(n)}}{N_{ii}^{(n)}} \leq \frac{\alpha_j}{\alpha_i},$$
and
$$\lim_n \frac{N_{ij}^{(n)}}{N_{jj}^{(n)}} = H_{ij}, \qquad \lim_n \frac{N_{ij}^{(n)}}{N_{ii}^{(n)}} = \frac{\alpha_j}{\alpha_i} \hat{H}_{ji}.$$
