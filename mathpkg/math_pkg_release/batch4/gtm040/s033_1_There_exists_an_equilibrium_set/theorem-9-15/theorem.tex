If $f$ is a function with $\alpha f = 0$ and if
$$\lim_{n \to \infty} [(I + P + \cdots + P^{n-1})f]_0$$
exists and is finite for some state $0$, then $f$ is a charge. If $g$ is its potential, then:
\begin{enumerate}
\item $g = {}_0 N f + g_0 \mathbf{1}$.
\item $f = (I - P)g$.
\item $P^n g \to 0$.
\item $B^E g = g$ if the support of $f$ is contained in $E$.
\item $f_E = (I - P^E)g_E$ if the support of $f$ is contained in $E$.
\end{enumerate}
