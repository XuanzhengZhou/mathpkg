If $(h_n, \mathcal{R}_n)$ is a nonnegative supermartingale, then there is a unique representation
$$(h_n, \mathcal{R}_n) = (r_n, \mathcal{R}_n) + (g_n, \mathcal{R}_n)$$
of $(h_n, \mathcal{R}_n)$ as the sum of a martingale and a potential. In this representation:
\begin{itemize}
\item The martingale $(r_n, \mathcal{R}_n)$ is a nonnegative martingale.
\item The potential $(g_n, \mathcal{R}_n)$ is a pure potential.
\item $r_n = \lim_k \mathbf{M}[h_{n+k} \mid \mathcal{R}_n]$.
\item $(g_n, \mathcal{R}_n)$ is the difference between a martingale and a process consisting of an increasing sequence of random variables.
\end{itemize}
