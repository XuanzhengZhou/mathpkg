Operations on stochastic processes $(f_n, \mathcal{R}_n)$ are defined by:
\begin{enumerate}
\item $(f_n, \mathcal{R}_n) \leq (g_n, \mathcal{R}_n)$ if $f_n \leq g_n$ a.e.\ for all $n$.
\item $a(f_n, \mathcal{R}_n) + b(g_n, \mathcal{R}_n) = (a f_n + b g_n, \mathcal{R}_n)$.
\item $\lim_k (f_n^{(k)}, \mathcal{R}_n) = (f_n, \mathcal{R}_n)$ if $\lim_k f_n^{(k)} = f_n$ a.e.\ for all $n$.
\end{enumerate}
