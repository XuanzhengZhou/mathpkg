If $P$ is a normal Markov chain, then the row vector $\nu = G_{0\cdot}$ (the $0$th row of the potential matrix) satisfies the admissibility conditions (1), (2), and (3). Moreover, for any finite set $E$ containing $0$,
$$\nu_E (I - P^E) = \lambda^E_E - \delta_0,$$
where $\lambda^E$ is the normalized entrance measure for the set $E$.
