Let $\nu$ be an admissible row vector and let $\bar{\nu}_j = \nu_j + \delta_{0j}$. Set $S^\nu = \{i \mid \nu_i > 0\}$ and define the matrix $Q^\nu$ by
$$Q_{ij}^\nu = \frac{\nu_j P_{ji}}{\nu_i + \delta_{0i}} \quad \text{for } i, j \in S^\nu \cup \{0\},$$
with all other entries taken to be $0$. The non-diagonal entries $N_{0j}^\nu$ of the fundamental matrix of $Q^\nu$ satisfy
$$N_{0j}^\nu = \frac{\bar{\nu}_j N_{j0}}{\bar{\nu}_0} = \bar{\nu}_j = \nu_j + \delta_{0j}.$$
When $\bar{\nu} > 0$, the process governed by $Q^\nu$ is the $\bar{\nu}$-dual of $Q$.
