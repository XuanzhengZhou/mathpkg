The fine boundary function of a non-negative $\pi$-integrable regular function $h$ is the restriction to the exit boundary $B_e$ of the limiting function $f$ obtained from the martingale convergence results. Since the limit function $f$ vanishes identically on the state space $S$, its nontrivial behavior is entirely captured on the boundary $B_e$, making it a natural boundary object for the Markov chain. This notion provides a rigorous connection between the potential-theoretic properties of regular functions and the exit boundary of the process.
