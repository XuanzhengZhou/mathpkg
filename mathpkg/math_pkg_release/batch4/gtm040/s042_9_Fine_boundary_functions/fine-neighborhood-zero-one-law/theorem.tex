For $x \in B_e$, a set $A \subseteq S^*$ containing $x$ is a fine neighborhood of $x$ if and only if
\[
\Pr^{K(\cdot, x)}\left[x_n \in A \text{ from some time on}\right] > 0.
\]
Moreover, for any set $A \subseteq S^*$, the probability
\[
\Pr^{K(\cdot, x)}\left[x_n \in A \text{ from some time on}\right]
\]
can only take the values $0$ or $1$ (a zero-one law).
