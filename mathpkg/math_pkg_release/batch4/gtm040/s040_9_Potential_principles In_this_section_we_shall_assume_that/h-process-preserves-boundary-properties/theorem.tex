Let $h \geq 0$ be a finite-valued normalized $P$-superregular function. If $x \in S^{h*}$, then
\begin{enumerate}
\item $K^h(\cdot, x)$ is normalized if and only if $K(\cdot, x)$ is normalized.
\item $K^h(\cdot, x)$ is regular for $P^h$ if and only if $K(\cdot, x)$ is $P$-regular.
\item $K^h(\cdot, x)$ is minimal for $P^h$ if and only if $K(\cdot, x)$ is minimal for $P$.
\end{enumerate}
Hence $B_e^h = B^h \cap B_e$ and $\overline{S^h} \subset \bar{S}$.
