If $h \geq 0$ is a finite-valued $P$-superregular function such that $\pi h = 1$, the \text{$h$-process} is the Markov chain with state space $S$ and measure $\Pr^h$ defined by
$$\Pr^h[x_0 = c \land x_1 = d \land \cdots \land x_n = j] = \pi_c P_{cd} \cdots P_{ij} h_j.$$
The transition matrix $P^h$ satisfies
$$P_{ij}^h = \begin{cases} P_{ij} h_j / h_i & \text{for } i, j \in S^h, \\ 0 & \text{for } i \in S^h, j \in S - S^h, \end{cases}$$
where $S^h = \{i \in S \mid h_i > 0\}$.
