Fix a function $f \geq 0$ such that $Nf$ is everywhere positive and finite-valued. For $i$ and $j$ in $S$, the Martin entrance kernel is defined as

$$J(i,j) = \frac{N_{ij}}{(Nf)_{i}}.$$

The measure $J(i,\cdot)$ is $P$-regular everywhere except at $i$, where it is strictly superregular, and satisfies $J(i,\cdot)f = 1$ for all $i$. The function $J(\cdot, j)$ is bounded by $N_{jj}/(Nf)_{j}$.

The Martin entrance metric is defined as

$$d'(j,j') = \sum_i w_i (Nf)_{i} |J(j,i) - J(j',i)|,$$

where the $w_i$ are positive weights such that $\sum w_i N_{ii}$ is finite.
