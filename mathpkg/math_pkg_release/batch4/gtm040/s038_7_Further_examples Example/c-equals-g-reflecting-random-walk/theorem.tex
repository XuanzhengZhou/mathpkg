For the reflecting random walk with $p = \frac{1}{2}$, the $C$ matrix coincides with the $G$ matrix:

$$C_{ij} = G_{ij} \quad \text{for all } i, j.$$

In fact, for any recurrent process of this type where $P = \hat{P}$ (the reverse chain has the same transition matrix), $G$ exists if and only if $C$ exists, and when they exist, $C = G$.
