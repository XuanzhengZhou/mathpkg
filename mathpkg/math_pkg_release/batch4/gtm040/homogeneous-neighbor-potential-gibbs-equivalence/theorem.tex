Let $U'$ and $U''$ be homogeneous neighbor potentials such that $u'_k = u''_k = 0$ for all $k$ and
$u'_{ij} = \ln Q'_{ij}$, $u''_{ij} = \ln Q''_{ij}$ with $Q'_{ij}, Q''_{ij} > 0$. Then $U'$ and $U''$ determine the same Gibbs fields if and only if there exist constants $c > 0$ and $h_i > 0$ ($i \in S$) such that
$$
Q'_{ij} = c \, \frac{h_j}{h_i} \, Q'_{ij} \quad \text{for all } i, j \in S.
$$
Equivalently, $U'$ and $U''$ determine the same Gibbs fields iff
$$
\sum_{B \subset A \subset D \subset \bar{A}} (-1)^{|A-B|} \Delta_D(\iota^B) = 0 \quad \text{for every } A = \{m\}, \{m, m+1\},
$$
where $\Delta_A = U'_A - U''_A$.
