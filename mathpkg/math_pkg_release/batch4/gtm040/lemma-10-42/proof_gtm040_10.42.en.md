**Proof:** Since $h$ is bounded, $h \leq c1$ for some

for all Borel sets $C$. Since $h$ is regular, $\mu^h(S) = 0$ and we may take $f$ to vanish on $S$.

By Theorem 10-41 we have

$$h_i = \int_S K(i, x) d\mu^h(x)$$

$$= \int_S K(i, x) f(x) d\mu(x)$$

$$= \int_{S^*} K(i, x) f(x) d\mu(x).$$

Now the argument in the proof of Proposition 10-21 shows that

$$\int_{S^*} K(i, x) f(x) d\mu(x) = \int_\Omega f(x_v(\omega)) d\Pr_i(\omega).$$

Thus if $\Pr_\pi[x_0 = j \wedge \dots \wedge x_n = i] > 0,$

$$M_\pi[f(x_v) \mid x_0 = j \wedge \dots \wedge x_n = i] = M_i[f(x_v)] = h_i.$$

Similarly, if $b$ denotes the absorbing state in the enlarged chain, then

$$M_\pi[f(x_v) \mid x_0 = j \wedge \dots \wedge x_n = b] = 0 = h_b.$$

That is, if $\mathcal{R}_n$ is the partition generated by $\{x_0, \dots, x_n\}$, then

$$M_\pi[f(x_v) \mid \mathcal{R}_n] = h(x_n).$$

On one hand, the Borel field generated by the $\mathcal{R}_n$ is $\mathcal{G}$, and on the other hand $f(x_v)$ is measurable over $\mathcal{G}$, since it is the composition of a Borel function and a function for which the inverse image of every Borel set is in $\mathcal{G}$. By Proposition 3-18,

$$\Pr[\lim_{n \to \infty} h(x_n) = f(x_v)] = 1.$$
