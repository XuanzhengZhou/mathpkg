In the basic example $P$, if $g = Nf$ is a potential with charge $f$, then
$$g_i = (Nf)_i = \frac{1}{\beta_\infty} \sum_{j=0}^{\infty} \beta_j f_j - \frac{1}{\beta_i} \sum_{j=0}^{i-1} \beta_j f_j.$$
In the reverse chain $\hat{P}$, if $g = \hat{N}f$, then
$$g_i = \frac{1}{\beta_\infty} \sum_{j=0}^{\infty} \beta_j f_j - \sum_{j=i+1}^{\infty} f_j.$$
In both cases, $g$ is finite-valued if $\sum \beta_j |f_j| < \infty$. For the forward chain, $\lim_i g_i = 0$; for the reverse chain, $g_i$ increases monotonically to $\frac{\beta|f|}{\beta_\infty}$ when $f \geq 0$.
