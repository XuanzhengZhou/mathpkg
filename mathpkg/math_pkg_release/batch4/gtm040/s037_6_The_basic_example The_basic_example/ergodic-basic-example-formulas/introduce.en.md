The ergodic basic example admits an extraordinarily complete set of explicit formulas for all potential-theoretic quantities. The stationary distribution is proportional to $\beta$, and the entry probabilities, Green function, potential kernel $C$, dual kernel $\hat{C}$, and mean hitting times $M_{ij}$ are all expressed in closed form using the cumulative sums $\sigma_i$. These formulas make the ergodic basic example the central testing ground for verifying general theorems and for constructing counterexamples to plausible but false conjectures.
