In the null recurrent basic example, the Green function and potential kernel are given by
$$\hat{G}_{ij} = {^i\hat{\lambda}_j} \, {^i N_{jj}} = \begin{cases} 1 & \text{if } j > i, \\ 0 & \text{if } j \leq i, \end{cases}$$
and
$$C_{ij} = \frac{\beta_j}{\beta_i} \hat{G}_{ji} = \begin{cases} \frac{\beta_j}{\beta_i} & \text{if } j < i, \\ 0 & \text{if } j \geq i. \end{cases}$$
Moreover, $C = G$ in the null case. Note that $-C$ (or $-G$) is obtained in the transient case by letting $\beta_\infty = +\infty$ in the formula for $N$.
