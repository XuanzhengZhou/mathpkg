The basic example is a denumerable Markov chain $P$ on the nonnegative integers with transition probabilities
$$P_{i,i+1} = p_{i+1}, \quad P_{i,0} = q_{i+1} = 1 - p_{i+1},$$
where $0 < p_i < 1$ for all $i \geq 1$. Setting $\beta_0 = 1$ and
$$\beta_i = p_1 p_2 \cdots p_i,$$
the chain satisfies $\beta_i > 0$ and $\beta_i \to \beta_\infty$ as $i \to \infty$.
