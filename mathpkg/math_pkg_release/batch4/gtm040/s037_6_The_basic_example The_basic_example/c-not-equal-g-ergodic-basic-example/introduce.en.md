While the potential kernel $C$ and the Green function $G$ coincide in the null recurrent basic example, they are never equal in the ergodic case. This negative result illustrates a fundamental difference between null recurrent and positive recurrent potential theory. The proof is by contradiction: assuming $C = G$ forces $\beta_j$ to be constant, which is incompatible with the recurrence requirement that $\beta_j \to 0$.
