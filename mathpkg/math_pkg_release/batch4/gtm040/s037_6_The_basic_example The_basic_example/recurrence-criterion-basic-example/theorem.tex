The basic example $P$ is recurrent if and only if $\beta_n \to 0$ as $n \to \infty$ (i.e., $\beta_\infty = 0$). When recurrent, $\beta$ is regular and we may choose $\alpha = \beta$.

The chain is null recurrent if additionally $\sum_i \beta_i = +\infty$, and ergodic (positive recurrent) if $\sum_i \beta_i < \infty$. In the ergodic case, the stationary distribution is
$$\alpha_j = \frac{\beta_j}{\sum_k \beta_k}.$$
