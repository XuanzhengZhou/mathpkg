In an ergodic chain, the condition $\alpha f = 0$ is not sufficient for $f$ to be a charge. Specifically, there exists a function $h \geq 0$ with $\alpha h = +\infty$ and $f = (I-P)h$ such that $\alpha f = 0$, but $f$ is not a charge.

For example, in the basic example with $p_i = (i/(i+1))^{3/2}$ (so $\beta_i = (i+1)^{-3/2}$ and the chain is ergodic), let $h_i = \sqrt{i+1}$. Then $h \geq 0$, $\alpha h = +\infty$, and for $f = (I-P)h$ we have $\alpha f = 0$. Yet
$$\liminf_n (P^n h) \geq A h = +\infty$$
by Fatou's Theorem, so $(I + P + \cdots + P^{n-1})f = (I - P^n)h$ does not converge, hence $f$ is not a charge.
