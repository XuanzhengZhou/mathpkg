For the forward chain $P$ of the basic example, the equilibrium sets are precisely the finite sets. All non-empty finite sets have capacity $\beta_\infty$. Infinite sets are not equilibrium sets; for any infinite set $E$, we have $h^E = 1$ and $e^E = (I-P)h^E = 0$, so $h^E$ is not a potential.

For the reverse chain $\hat{P}$, every set (finite or infinite) is an equilibrium set.
