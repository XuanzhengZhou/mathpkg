The equilibrium sets of the basic example highlight a fundamental asymmetry between the forward and reverse chains. For $P$, only finite sets have finite capacity and admit equilibrium potentials, while infinite sets always have full hitting probability. For $\hat{P}$, every set is an equilibrium set regardless of cardinality. The capacity of any non-empty finite set for $P$ is exactly $\beta_\infty$, the limiting value of the $\beta$ sequence.
