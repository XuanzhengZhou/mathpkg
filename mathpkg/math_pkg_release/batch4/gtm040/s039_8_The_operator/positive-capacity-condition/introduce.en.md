This proposition characterizes when a finite set $E$ containing the reference state $0$ has positive capacity. The condition is that there exists a state $j$ in $E$ that can be reached from $0$ (${}^0\lambda_j > 0$) and from which the chain can escape back to $E$ ($\hat{\lambda}_j^E > 0$). The proof uses an expression of $k(E)$ involving occupation times and hitting probabilities.
