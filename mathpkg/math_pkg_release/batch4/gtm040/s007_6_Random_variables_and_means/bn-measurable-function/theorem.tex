If $f(\omega)$ is an extended-real-valued function defined on the space $\Omega$ of a stochastic process and if $\Omega$ has the property that for some $n$, $f(\omega)$ is measurable with respect to the Borel field $\mathcal{B}_n$, then $f$ is a random variable because $\mathcal{B}_n \subset \mathcal{B}$. Such a function is said to be $\mathcal{B}_n$-\text{measurable}. For the special case in which $\Omega$ is sequence space, a function $f$ is $\mathcal{B}_n$-measurable for some $n$ if its values depend only on a bounded number of outcomes.
