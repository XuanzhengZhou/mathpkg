A denumerable set of statements $\{q_i\}$ about a stochastic process is said to be a \text{set of alternatives} if the truth sets $Q_i$ of the statements form a partition. For every random variable $f$ whose mean exists, the relation
$$M[f] = \sum_i \int_{Q_i} f \, d\mu$$
holds, since the integral is a completely additive set function.
