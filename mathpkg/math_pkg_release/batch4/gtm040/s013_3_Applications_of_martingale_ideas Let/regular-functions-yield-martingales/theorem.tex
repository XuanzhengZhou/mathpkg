Let $h$ be a $P$-regular function and let $h(x_n)$ denote $h_j$ if $x_n = j$. Suppose $\mathbf{M}[|h(x_n)|]$ is finite. Then $(h(x_n), \mathcal{R}_n)$ is a martingale, where $\mathcal{R}_n$ is the cross partition
$$\mathcal{R}_n = \mathcal{R}^{x_0} \otimes \mathcal{R}^{x_1} \otimes \cdots \otimes \mathcal{R}^{x_n}.$$
