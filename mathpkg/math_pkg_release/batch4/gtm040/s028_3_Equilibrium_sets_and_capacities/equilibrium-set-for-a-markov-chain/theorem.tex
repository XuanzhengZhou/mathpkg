A set $E$ is an \textbf{equilibrium set} for $P$ if there is a pure potential which assumes the value 1 at every point of $E$ and which has support in $E$. Such a potential is called an \textbf{equilibrium potential} for $E$. A set $E$ is a \textbf{dual equilibrium set} for $P$ if there is a pure potential measure with support in $E$ which equals $\alpha$ on $E$.
