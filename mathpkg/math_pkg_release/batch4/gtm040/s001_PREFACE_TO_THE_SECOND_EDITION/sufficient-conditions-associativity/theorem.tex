Let $A$, $B$, and $C$ be matrices such that one of the following holds:
\begin{enumerate}
  \item $A$ has only finitely many non-zero entries in each row,
  \item $C$ has only finitely many non-zero entries in each column, or
  \item $B$ has all entries finite and both $A$ and $C$ have all entries bounded in absolute value.
\end{enumerate}
Then $A(BC) = (AB)C$, provided the products are well defined.
