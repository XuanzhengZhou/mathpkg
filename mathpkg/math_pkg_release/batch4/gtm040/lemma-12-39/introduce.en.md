This lemma establishes that any strictly positive matrix $Q'$ on a finite state space admits a positive eigenvector $\bar{h}$ with positive eigenvalue $\bar{c}$. Using this eigenvector, one constructs a stochastic matrix $Q$ from $Q'$ via the transformation $Q_{ij} = Q'_{ij}\bar{h}_j/(\bar{c}\bar{h}_i)$. This result is a crucial step toward proving uniqueness of the Gibbs field for a given equivalence class $[Q']$, as it provides the canonical stochastic representative $Q$ within each equivalence class of strictly positive matrices.
