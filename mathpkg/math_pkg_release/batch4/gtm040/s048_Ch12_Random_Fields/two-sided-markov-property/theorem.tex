A denumerable stochastic process $\{x_n\}$ satisfies the \textbf{two-sided Markov property} if
$$\Pr[x_n = i_n \mid x_k = i_k; k \in \{m, m+1, \ldots, M\} - \{n\}]$$
$$= \begin{cases} \Pr[x_0 = i_0 \mid x_1 = i_1] & \text{if } 0 = n < m \\ \Pr[x_n = i_n \mid x_{n-1} = i_{n-1}, x_{n+1} = i_{n+1}] & \text{if } m < n < M \\ \Pr[x_M = i_M \mid x_{M-1} = i_{M-1}] & \text{if } m < n = M. \end{cases}$$
