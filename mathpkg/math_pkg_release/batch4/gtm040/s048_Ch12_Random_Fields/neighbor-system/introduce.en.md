A neighbor system on a parameter set $T$ is a symmetric, irreflexive relation that assigns to each site a finite set of neighbors. This combinatorial structure encodes the dependency graph of a Markov random field: the conditional distribution at a site depends only on the values at neighboring sites. The closed neighborhood $\bar{a} = \{a\} \cup \partial a$ and its extension $\bar{A}$ to subsets play a central role in formulating the Markov field property. Important examples include the integer lattices $\mathbb{Z}^d$ with nearest-neighbor adjacency.
