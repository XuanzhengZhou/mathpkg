Given $\iota = \{i_t\}$ and $A \subset T$, the modification $\iota^A = \{i_t^A\}$ of $\iota$ has values
$$i_t^A = \begin{cases} i_t & \text{for } t \in A \\ 0 & \text{otherwise.} \end{cases}$$
For any $a \in T$ and $B \subset T - \{a\}$,
$$\ln \frac{\mu(\{\iota^B\})}{\mu(\{\iota^{B+a}\})} = \sum_{D \subset T} U_D(\iota^B) - \sum_{D \subset T} U_D(\iota^{B+a}).$$
