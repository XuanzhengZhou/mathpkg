Lemma 12-10 establishes the M\"{o}bius inversion formula for set functions defined on subsets of a finite set. This classical combinatorial identity relates a set function $\Phi$ to its partial sum $\Psi$ via alternating signs: $\Phi(A) = \sum_{B \subset A} (-1)^{|A-B|} \Psi(B)$ if and only if $\Psi(A) = \sum_{B \subset A} \Phi(B)$. The lemma is a crucial tool in the proof of the canonical potential representation theorem (Theorem 12-12), where it is used to invert the relationship between probabilities and potentials.
