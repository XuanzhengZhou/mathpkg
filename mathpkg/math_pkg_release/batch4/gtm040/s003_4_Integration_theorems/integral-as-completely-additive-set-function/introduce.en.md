This theorem establishes that the indefinite Lebesgue integral defines a completely additive (countably additive) set function. That is, for any measurable set $E$ expressed as a countable disjoint union $E = \bigcup_{n=1}^{\infty} E_n$, we have $\int_E f \, d\mu = \sum_{n=1}^{\infty} \int_{E_n} f \, d\mu$. The proof proceeds by first establishing the result for characteristic functions, then simple functions, and finally general measurable functions via approximation.
