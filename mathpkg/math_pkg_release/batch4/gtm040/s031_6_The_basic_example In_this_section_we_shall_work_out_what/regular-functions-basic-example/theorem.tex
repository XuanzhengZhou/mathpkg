For the basic example Markov chain $P$ (in the transient case, $\beta_\infty > 0$):

\textbf{(i)} The only $P$-regular functions are the constant functions. That is, if $r$ satisfies $r = Pr$, then $r_i = c$ for all $i$ and some constant $c$.

\textbf{(ii)} Dually, the only $\hat{P}$-regular signed measures are the multiples of $\beta$. That is, if a row vector $\mu$ satisfies $\mu = \mu \hat{P}$, then $\mu_i = c \beta_i$ for some constant $c$.

\textbf{(iii)} $P$ has no non-zero regular signed measures (since $\hat{P}1 \neq 1$, the dual of (i) fails).

\textbf{(iv)} $\hat{P}$ has no non-zero regular functions (the dual of (ii) fails, since $\sum_j \hat{P}_{0j} = 1 - \beta_\infty < 1$).

Consequently, the non-negative superregular functions of $P$ are precisely the pure potentials plus non-negative constants, whereas for $\hat{P}$ only pure potentials are non-negative superregular functions.
