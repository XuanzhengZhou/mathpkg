Proposition 6-14: Let $X$ and $Y$ be square matrices, row vectors, or column vectors indexed by the set of states for a Markov chain $P$. Suppose $\hat{P}$ is the $\alpha$-dual of $P$. Then

(1) dual dual $X = X$.
(2) dual $(X + Y) = \text{dual } X + \text{dual } Y$.
(3) dual $(cX) = c \text{dual } X$.
(4) dual $(XY) = \text{dual } Y \text{dual } X$.
(5) dual $I = I$ and dual $0 = 0$.
(6) If $X \geq 0$, then dual $X \geq 0$; and if $X > 0$, then dual $X > 0$.
(7) If $X \geq Y$, then dual $X \geq \text{dual } Y$; and if $X > Y$, then dual $X > \text{dual } Y$.
(8) If $f$ is a $P$-superregular (or subregular) column vector, then dual $f$ is a $\hat{P}$-superregular (or subregular) row vector; and if $\beta$ is a $P$-superregular (or subregular) row vector, then dual $\beta$ is a $\hat{P}$-superregular (or subregular) column vector.
(9) dual $1 = \alpha$ and dual $\alpha = 1$. The measure $\alpha$ is $\hat{P}$-superregular.
(10) If $\lim_n X^{(n)} = X$, then $\lim_n \text{dual } X^{(n)} = \text{dual } X$.
