Let $h_j$ be the statement about a Markov chain that state $j$ is eventually reached. Let $n_j$ be the number of times in state $j$, and $t_j$ be the time to reach state $j$. Let $f_j^{(k)}$ be the statement that $t_j(\omega) = k$.

Define the shifted quantities $\bar{h}_j, \bar{n}_j, \bar{t}_j$, and $\bar{f}_j^{(k)}$ as follows:
$$\bar{h}_j: \quad h_j \text{ is true for } \omega_1$$
$$\bar{n}_j(\omega) = n_j(\omega_1)$$
$$\bar{t}_j(\omega) = t_j(\omega_1) + 1$$
$$\bar{f}_j^{(k)}: \quad \bar{t}_j(\omega) = k.$$

Define the matrices:
$$H_{ij} = \Pr\nolimits_i[h_j]$$
$$N_{ij} = M_i[n_j]$$
$$F_{ij}^{(k)} = \Pr\nolimits_i[f_j^{(k)}]$$
$$\bar{H}_{ij} = \Pr\nolimits_i[\bar{h}_j]$$
$$\bar{N}_{ij} = M_i[\bar{n}_j].$$
