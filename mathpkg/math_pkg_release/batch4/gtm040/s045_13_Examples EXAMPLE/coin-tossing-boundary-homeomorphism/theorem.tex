For the coin tossing process on state space $S = \{(m,i) \mid 0 \leq i \leq m\}$ with transition probabilities $P_{(m,i),(m+1,i)} = P_{(m,i),(m+1,i+1)} = \frac{1}{2}$, the Martin kernel is

$$K((m,i), t) = 2^m t^i (1 - t)^{m-i}, \qquad t \in [0,1].$$

No point of $S$ is a limit point. The exit boundary $B$ is homeomorphic to the unit interval $[0,1]$ under this correspondence, and every point of $B$ is extreme (i.e., $B_e = B$).
