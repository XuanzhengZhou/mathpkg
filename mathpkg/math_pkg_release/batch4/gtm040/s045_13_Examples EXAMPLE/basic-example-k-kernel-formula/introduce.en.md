The basic example is a denumerable Markov chain in which all pairs of states communicate. When the starting measure $\pi$ is a unit mass at state $0$, the Martin kernel $K(i,j)$ takes a particularly simple piecewise form. This explicit formula makes it possible to compute the metric on the state space for the exit boundary directly, revealing that the metric space $(S,d)$ is isometric to a subset of the real line.
