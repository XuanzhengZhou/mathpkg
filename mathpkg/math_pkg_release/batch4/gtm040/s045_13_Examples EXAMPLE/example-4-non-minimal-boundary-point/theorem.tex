There exists a denumerable Markov chain whose exit boundary contains a point $b_\infty$ such that the associated Martin kernel $K(\cdot, b_\infty)$ is regular and normalized but is not minimal (not an extreme boundary point). Specifically, the chain on state space

$$S = \{a_i, a'_i, b_i \mid i = 0, 1, 2, \ldots\}$$

with suitable transition probabilities has exactly three boundary points $a_\infty$, $a'_\infty$, $b_\infty$, and

$$K(\cdot, b_\infty) = \frac{1}{2}K(\cdot, a_\infty) + \frac{1}{2}K(\cdot, a'_\infty),$$

so that $B_e = \{a_\infty, a'_\infty\}$ are the extreme boundary points.
