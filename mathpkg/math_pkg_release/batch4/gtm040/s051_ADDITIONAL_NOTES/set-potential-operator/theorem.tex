Let $U_E$ denote $U_h$ when $h$ is the characteristic function of a set $E$. Then

$$\left(U_E f\right)_i = M_i \left[ \sum_{1 \leq n \leq t_E} f(x_n) \right],$$

where $t_E$ is the first return time to the set $E$. When $f$ is $1$ on $E$ and $0$ elsewhere, the right side reduces to the probability that the chain started in state $i$ ever returns to the set $E$.
