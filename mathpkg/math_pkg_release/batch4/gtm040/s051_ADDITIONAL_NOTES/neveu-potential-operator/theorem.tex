For a function $h$ on $S$ with $0 \leq h \leq 1$, define

$$U_h = \sum_{n=0}^{\infty} (P D_{1-h})^n P = \sum_{n=0}^{\infty} P (D_{1-h} P)^n,$$

where $D_h$ is the diagonal matrix with $h_i$ as the $i$th diagonal entry.

When $h = 1$, $U_h = P$. When $h = 0$, $U_h = P + P^2 + P^3 + \cdots$. In the general case with $f \geq 0$,

$$\left(U_h f\right)_i = M_i \left[ \sum_{n=0}^{\infty} \left(1 - h(x_1)\right)\left(1 - h(x_2)\right) \cdots \left(1 - h(x_{n-1})\right) f(x_n) \right].$$
