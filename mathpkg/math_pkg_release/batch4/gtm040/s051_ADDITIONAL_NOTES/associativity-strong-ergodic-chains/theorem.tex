For a strong ergodic chain, if the charge $f$ is bounded, then associativity holds:

$$A(0Nf) = (A^0N)f.$$

In particular, if the potential $g$ is bounded (since $f = (I - P)g$), then the relation holds. The key observation is $\alpha^0N1 \leq \alpha^0 N 1 = M_{\alpha 0} < \infty$. Consequently, for a strong ergodic chain, $g = -Gf$ whenever the charge $f$ is bounded.
