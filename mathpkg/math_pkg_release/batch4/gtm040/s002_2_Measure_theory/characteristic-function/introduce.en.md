The characteristic function (or indicator function) of a set $A$ takes value 1 on $A$ and 0 elsewhere. It is the fundamental building block for simple functions. A characteristic function $\chi_A$ is measurable if and only if the set $A$ is measurable. This connection between sets and functions is central to the Lebesgue theory of integration.
