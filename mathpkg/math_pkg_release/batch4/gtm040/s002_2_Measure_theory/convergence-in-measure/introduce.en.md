Convergence of sets with respect to the outer measure pseudometric provides the notion of approximation used in the Carathéodory extension theorem. The class $\mathcal{F}^*$ is defined as all sets that can be approximated in this sense by sets in the original field $\mathcal{F}$. This notion of convergence ensures that the limit of sets preserves the basic set operations and measure-theoretic properties.
