Constant functions are the simplest examples of measurable functions. If $f(x) = a$ identically, then the set $\{x \mid f(x) < c\}$ is either the empty set $\varnothing$ (if $a \geq c$) or the whole space $X$ (if $a < c$), both of which are measurable sets in any Borel field.
