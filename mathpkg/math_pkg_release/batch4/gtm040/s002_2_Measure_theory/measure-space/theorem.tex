A \textit{measure} is a non-negative, completely additive set function defined on a Borel field of subsets of a set $X$. The triple $(X, \mathcal{B}, \mu)$, where $\mathcal{B}$ is a Borel field and $\mu$ is a measure defined on $\mathcal{B}$, is called a \textit{measure space}. If $\mu(X) = 1$, then the space is called a \textit{probability space}.
