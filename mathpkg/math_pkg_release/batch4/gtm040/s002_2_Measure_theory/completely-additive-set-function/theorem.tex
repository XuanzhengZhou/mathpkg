An additive set function $\rho$ defined on a field of sets $\mathcal{F}$ is said to be \textit{completely additive} if $\rho(\bigcup_{n=1}^{\infty} A_n) = \sum_{n=1}^{\infty} \rho(A_n)$ holds whenever $\{A_n\}$ is a sequence of disjoint sets in $\mathcal{F}$ whose union $\bigcup_{n=1}^{\infty} A_n$ is also a set of $\mathcal{F}$.
