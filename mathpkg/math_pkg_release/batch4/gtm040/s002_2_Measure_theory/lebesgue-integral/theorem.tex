In a measure space $(X, \mathcal{B}, \mu)$, let $E$ be a measurable set and $s = \sum_{n=1}^{N} c_n \chi_{A_n}$ be a non-negative measurable simple function. Set
$$I_E(s) = \sum_{n=1}^{N} c_n \mu(A_n \cap E).$$
For any non-negative measurable function $f$, define the \textit{Lebesgue integral} of $f$ on $E$ with respect to $\mu$ by
$$\int_E f \, d\mu = \sup I_E(s),$$
where the supremum is taken over all simple functions $s$ satisfying $0 \leq s \leq f$.
