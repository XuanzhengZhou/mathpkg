An additive set function assigns values to sets in such a way that the value of a disjoint union equals the sum of the values of the parts. This is the fundamental property that distinguishes measures from arbitrary set functions. An additive set function automatically satisfies $\rho(\varnothing) = 0$, finite additivity for disjoint families, and the inclusion-exclusion identity $\rho(A \cup B) + \rho(A \cap B) = \rho(A) + \rho(B)$.
