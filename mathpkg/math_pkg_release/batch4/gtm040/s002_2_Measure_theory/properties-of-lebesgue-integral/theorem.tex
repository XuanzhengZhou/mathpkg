The Lebesgue integral satisfies these seven properties:
\begin{enumerate}
\item[(1)] If $c$ is a constant function, $\int_E c \, d\mu = c \mu(E)$.
\item[(2)] If $f$ and $g$ are measurable functions whose integrals are defined on $E$ and if $f(x) \leq g(x)$ for all $x \in E$, then $\int_E f \, d\mu \leq \int_E g \, d\mu$.
\item[(3)] If $f$ is integrable on $E$ and $c$ is a real number, then $cf$ is integrable and $\int_E cf \, d\mu = c \int_E f \, d\mu$.
\item[(4)] If $f$ is measurable and $\mu(E) = 0$, then $\int_E f \, d\mu = 0$.
\item[(5)] If $E'$ and $E$ are measurable sets with $E' \subset E$ and $\int_E f \, d\mu$ is defined, then $\int_{E'} f \, d\mu$ is defined.
\item[(6)] If $f$ is integrable on $E$ and $E = E_1 \cup E_2$ with $E_1$, $E_2$ disjoint measurable sets, then $\int_E f \, d\mu = \int_{E_1} f \, d\mu + \int_{E_2} f \, d\mu$.
\item[(7)] If $f$ is integrable on $E$, then $|f|$ is integrable and $|\int_E f \, d\mu| \leq \int_E |f| \, d\mu$.
\end{enumerate}
