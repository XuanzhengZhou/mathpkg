Let $P$ be a chain which represents an electric circuit. Then the following is a necessary and sufficient condition for $P$ to be recurrent: If terminal 0 is kept at a unit voltage, if all terminals outside a finite set $F$ are grounded, and if $F$ is allowed to increase to $S$, then the current at terminal 0 tends to zero. Furthermore a necessary and sufficient condition for $P$ to be ergodic is that $\sum_{i,j} c_{ij} < \infty$.
