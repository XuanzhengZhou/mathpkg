In a probability space $(\Omega, \mathcal{B}, \mu)$, a statement whose truth set has measure one is said to be \textbf{almost always}, \textbf{almost everywhere}, \textbf{almost surely true}, or \textbf{true a.e.} Correspondingly, a statement with truth set of measure zero is \textbf{almost surely false}, and the negation $\sim p$ of a statement $p$ which is almost surely false is almost surely true.
