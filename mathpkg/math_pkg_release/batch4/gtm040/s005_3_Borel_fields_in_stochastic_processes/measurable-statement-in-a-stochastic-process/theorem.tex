Probabilities are numbers assigned to statements about stochastic processes. Formally, the probability of a statement $p$ is defined to be the measure of its truth set: $$\Pr[p] = \mu(P),$$ where $P = \{\omega \mid p\}$. If the set $P$ is not in the Borel field on which $\mu$ is defined, then $\Pr[p]$ is undefined. Statements for which $\Pr[p]$ is defined are called \textbf{measurable statements}.
