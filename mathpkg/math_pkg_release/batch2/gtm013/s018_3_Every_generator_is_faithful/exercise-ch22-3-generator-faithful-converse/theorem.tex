Every generator is faithful. Concerning a converse, prove that:
\begin{enumerate}
    \item[(1)] A projective module $P_R$ is a generator iff for each ideal $I$ of $R$, if $PI = P$, then $I = R$.
    \item[(2)] Faithful finitely generated projective modules need not be generators.
    \item[(3)] If $R$ is commutative and $P_R$ is finitely generated projective, then $P_R$ is a progenerator iff it is faithful.
\end{enumerate}
