In the category of left $R$-modules:
\begin{enumerate}
\item[(1)] $M$ is injective if and only if $M = E(M)$;
\item[(2)] If $M \trianglelefteq N$, then $E(M) = E(N)$;
\item[(3)] If $M \leq Q$ with $Q$ injective, then $Q = E(M) \oplus E$;
\item[(4)] If $\bigoplus_A E(M_\alpha)$ is injective (e.g., if $A$ is finite), then
$$E\left(\bigoplus_A M_\alpha\right) = \bigoplus_A E(M_\alpha).$$
\end{enumerate}
