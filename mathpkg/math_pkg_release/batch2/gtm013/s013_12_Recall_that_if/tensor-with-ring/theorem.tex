For each right module $M_R$, there is an $R$-isomorphism $\eta: M \otimes_R R \to M$ such that
$$\eta(m \otimes r) = mr, \quad \eta^{-1}(m) = m \otimes 1.$$
For each left module ${}_R N$, there is an $R$-isomorphism $\mu: R \otimes_R N \to N$ such that
$$\mu(r \otimes n) = rn, \quad \mu^{-1}(n) = 1 \otimes n.$$
If ${}_S M_R$ (resp. ${}_R N_S$) is a bimodule, then $\eta$ (resp. $\mu$) is a bimodule isomorphism.
