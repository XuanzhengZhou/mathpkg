For a ring $R$, the following are equivalent:
\begin{enumerate}
\item[(a)] $R$ is left coherent;
\item[(b)] Every direct product of flat right $R$-modules is flat;
\item[(c)] For every set $A$, the right $R$-module $R^A$ is flat;
\item[(d)] For every set $A$ and every set $B$, $(R^{(B)})^A$ is flat.
\end{enumerate}
