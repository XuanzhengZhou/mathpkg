Let $M$ and $N$ be left $R$-modules, let $X \subseteq M$ span $M$, and let $f: M \to N$ be an $R$-homomorphism.

\begin{enumerate}
  \item $\text{Im}\, f$ is spanned by $f(X)$.
  \item If $g: M \to N$ is another $R$-homomorphism, then $f = g$ if and only if $f(x) = g(x)$ for all $x \in X$.
\end{enumerate}
