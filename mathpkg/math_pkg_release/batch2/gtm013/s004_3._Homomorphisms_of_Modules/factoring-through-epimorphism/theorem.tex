Let $f: M \to N$ and $g: M \to M'$ be $R$-homomorphisms with $g$ an epimorphism. If $\text{Ker}\, g \subseteq \text{Ker}\, f$, then there exists a unique $R$-homomorphism $h: M' \to N$ such that
$$f = hg.$$
Moreover, $\text{Ker}\, h = g(\text{Ker}\, f)$ and $\text{Im}\, h = \text{Im}\, f$, so that $h$ is monic if and only if $\text{Ker}\, g = \text{Ker}\, f$, and $h$ is epic if and only if $f$ is epic.
