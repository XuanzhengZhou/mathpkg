A left $R$-module $M$ is cyclic if and only if it is isomorphic to $R/I$ for some left ideal $I$ of $R$. More precisely, for each $x \in M$, the cyclic submodule $Rx$ satisfies
$$Rx \cong R / l_R(x),$$
where $l_R(x) = \{r \in R \mid rx = 0\}$ is the left annihilator of $x$. Conversely, if $I$ is a left ideal of $R$, then $R/I$ is a cyclic left $R$-module spanned by $1 + I$, and $l_R(1 + I) = I$.
