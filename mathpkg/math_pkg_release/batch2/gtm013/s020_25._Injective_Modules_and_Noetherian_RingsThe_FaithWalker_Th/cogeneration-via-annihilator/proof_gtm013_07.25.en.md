It is easy to check that

$$\ell_R(r_M(I)) = \bigcap \{\operatorname{Ker} f \mid f: {}_R R \to M \text{ with } I \leq \operatorname{Ker} f\}.$$

Hence

$$\ell_R(r_M(I))/I = \operatorname{Rej}_{R/I}(M).$$

Thus $M$ cogenerates $R/I$ (i.e., $\operatorname{Rej}_{R/I}(M) = 0$) if and only if $\ell_R(r_M(I)) = I$, which means $I$ is the annihilator of the subset $r_M(I) \subseteq M$.
