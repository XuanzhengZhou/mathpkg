A module $M$ is $c$-generated in case there is an indexed set $(x_\gamma)_{\gamma \in C}$ that spans $M$ with $c = \operatorname{card} C$. Thus (see (8.1)) if $C$ is a set, then $M$ is $\operatorname{card} C$-generated if and only if $M$ is an epimorphic image of the free module $R^{(C)}$. Every epimorphic image of a $c$-generated module is $c$-generated. In general, submodules of $c$-generated modules need not be $c$-generated.
