A module $M$ is said to be $c$-generated for a cardinal number $c$ if it can be spanned by a set of size $c$, or equivalently if it is an epimorphic image of the free module $R^{(C)}$ with $|C| = c$. This notion generalizes finite generation ($n$-generated for $n \in \mathbb{N}$) to arbitrary cardinals. While epimorphic images preserve $c$-generation, submodules of $c$-generated modules need not be $c$-generated in general. The concept is crucial for the Faith-Walker theorem, where a uniform cardinal bound on the size of indecomposable injective summands characterizes noetherian rings.
