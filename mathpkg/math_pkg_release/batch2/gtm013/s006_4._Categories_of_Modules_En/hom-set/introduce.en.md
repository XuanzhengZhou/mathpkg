For two left $R$-modules $M$ and $N$, the collection of all $R$-module homomorphisms from $M$ to $N$ forms a set (as opposed to a proper class), since each homomorphism is a function from the set $M$ to the set $N$. This set is denoted $\operatorname{Hom}_R(M, N)$. By Proposition 4.1, it carries a natural abelian group structure under pointwise addition. The Hom-set notation is the foundation for the categorical treatment of modules: the category $_R\mathbf{M}$ of left $R$-modules has the modules as objects and $\operatorname{Hom}_R(M, N)$ as the set of morphisms from $M$ to $N$.
