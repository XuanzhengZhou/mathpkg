Let $\mathcal{U}$ and $\mathcal{V}$ be classes of modules.

(1) If $\mathcal{V}$ is contained in $\mathrm{Gen}(\mathcal{U})$ (resp. $\mathrm{FGen}(\mathcal{U})$), then so is $\mathrm{Gen}(\mathcal{V})$ (resp. $\mathrm{FGen}(\mathcal{V})$).

(2) If $\mathcal{V}$ is contained in $\mathrm{Cog}(\mathcal{U})$ (resp. $\mathrm{FCog}(\mathcal{U})$), then so is $\mathrm{Cog}(\mathcal{V})$ (resp. $\mathrm{FCog}(\mathcal{V})$).
