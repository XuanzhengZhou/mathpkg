Let $\mathcal{U}$ be a class of modules. A module $M$ is **(finitely) generated by $\mathcal{U}$** (or $\mathcal{U}$ **(finitely) generates** $M$) in case there is a (finite) indexed set $(U_\alpha)_{\alpha \in A}$ in $\mathcal{U}$ and an epimorphism

$$\oplus_A U_\alpha \rightarrow M \rightarrow 0.$$

If $\mathcal{U} = \{U\}$ is a singleton, then we simply say that $U$ (finitely) generates $M$; this means that there is an epimorphism

$$U^{(A)} \rightarrow M \rightarrow 0$$

for some (finite) set $A$.
