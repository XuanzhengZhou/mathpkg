Theorem 8.1 establishes the fundamental connection between the classical element-based notion of a spanning set and the categorical notion of generation. If a module $M$ has a spanning set $X$, then the free module $R^{(X)}$ maps epimorphically onto $M$ via the map sending each basis element to the corresponding spanning element. In categorical language, this says that the regular module $R$ generates $M$, since $R^{(X)}$ is a direct sum of copies of $R$.
