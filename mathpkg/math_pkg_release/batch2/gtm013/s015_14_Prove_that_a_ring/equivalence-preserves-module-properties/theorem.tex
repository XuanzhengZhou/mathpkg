Let $R$ and $S$ be equivalent rings via an equivalence $F: {}_R \mathbf{M} \to {}_S \mathbf{M}$, and let $M$ and $M'$ be left $R$-modules. Then:

(1) $M$ is simple (semisimple) if and only if $F(M)$ is simple (semisimple);
(2) $M$ is finitely generated (finitely cogenerated) if and only if $F(M)$ is finitely generated (finitely cogenerated);
(3) $M$ is artinian (noetherian) if and only if $F(M)$ is artinian (noetherian);
(4) $c(M) = c(F(M))$; that is, $M$ and $F(M)$ have the same composition length;
(5) $M$ is indecomposable if and only if $F(M)$ is indecomposable.
