Let $F: {}_R \mathbf{M} \to {}_S \mathbf{M}$ be a category equivalence. Then:

(1) A pair $(M, (p_\alpha)_{\alpha \in A})$ is a direct product of $(M_\alpha)_{\alpha \in A}$ if and only if $(F(M), (F(p_\alpha))_{\alpha \in A})$ is a direct product of $(F(M_\alpha))_{\alpha \in A}$;

(2) A pair $(M, (i_\alpha)_{\alpha \in A})$ is a direct coproduct of $(M_\alpha)_{\alpha \in A}$ if and only if $(F(M), (F(i_\alpha))_{\alpha \in A})$ is a direct coproduct of $(F(M_\alpha))_{\alpha \in A}$;

(3) $U$ generates (cogenerates) $M$ if and only if $F(U)$ generates (cogenerates) $F(M)$;

(4) $U$ is a generator (a cogenerator) (faithful) if and only if $F(U)$ is a generator (a cogenerator) (faithful);

(5) A monomorphism (epimorphism) $f: M \to M'$ is essential (superfluous) if and only if $F(f): F(M) \to F(M')$ is essential (superfluous);

(6) $f: M \to M'$ is an injective envelope (projective cover) if and only if $F(f): F(M) \to F(M')$ is an injective envelope (projective cover).
