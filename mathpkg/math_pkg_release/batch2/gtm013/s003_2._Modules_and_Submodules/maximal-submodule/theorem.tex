By convention in module theory:
\begin{itemize}
\item \textbf{maximal submodule} means \textbf{maximal proper submodule};
\item \textbf{minimal submodule} means \textbf{minimal non-zero submodule}.
\end{itemize}

For example, $M$ is simple (hence non-zero) if and only if $M$ is both a minimal and a maximal submodule of itself (with respect to the proper/non-zero conventions).
