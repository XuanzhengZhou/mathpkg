A spanning set $X$ of a module $M$ is a subset such that $RX = M$, meaning every element of $M$ can be expressed as an $R$-linear combination of elements of $X$. A module is finitely spanned if it admits a finite spanning set, and cyclic if it can be spanned by a single element. These concepts generalize the notions of spanning sets, finite-dimensionality, and one-dimensionality from vector spaces to modules over arbitrary rings.
