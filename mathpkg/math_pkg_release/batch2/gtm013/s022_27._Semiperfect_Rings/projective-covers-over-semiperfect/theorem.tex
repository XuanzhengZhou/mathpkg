Let $R$ be semiperfect with basic set of idempotents $e_1, \ldots, e_m$ and let ${}_R M$ be finitely generated. Then $M/JM$ is finitely generated and semisimple, so there exist unique non-negative integers $k_1, \ldots, k_m$ such that
$$M/JM \cong (R e_1/J e_1)^{(k_1)} \oplus \ldots \oplus (R e_m/J e_m)^{(k_m)}.$$
Set
$$P = R e_1^{(k_1)} \oplus \ldots \oplus R e_m^{(k_m)}.$$
Then $P$ is finitely generated projective with $P/JP \cong M/JM$, and there exists a projective cover $P \rightarrow M \rightarrow 0$, which is unique up to isomorphism.
