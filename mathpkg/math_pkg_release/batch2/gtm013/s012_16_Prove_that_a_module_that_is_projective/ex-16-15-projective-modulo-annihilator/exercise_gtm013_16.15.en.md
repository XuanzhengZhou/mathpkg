A module ${}_R U$ is **projective (injective) modulo its annihilator** in case $U$ is a projective (injective) $R/\ell_R(U)$ module.

(1) Prove that these are equivalent: (a) ${}_R U$ is projective modulo its annihilator; (b) $U$ is a projective $R/I$ module for some ideal $I \subseteq \ell_R(U)$; (c) $U$ is $U^A$-projective for every set $A$.

(2) Prove that these are equivalent: (a) ${}_R U$ is injective modulo its annihilator; (b) $U$ is an injective $R/I$ module for some ideal $I \subseteq \ell_R(U)$; (c) $U$ is $U^A$-injective for every set $A$.
