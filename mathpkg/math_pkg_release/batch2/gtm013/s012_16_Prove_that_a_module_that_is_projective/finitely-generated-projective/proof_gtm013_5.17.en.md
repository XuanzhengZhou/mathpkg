**Proof.** A module ${}_R P$ is finitely generated if and only if for some natural number $n$, there is an epimorphism

$$R^{(n)} \\rightarrow P \\rightarrow 0.$$

But by (17.2) this epimorphism splits if and only if $P$ is projective.
