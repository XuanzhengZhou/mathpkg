A ring is left primitive if and only if it is isomorphic to a dense ring of linear transformations of a vector space. In other words, a ring $R$ is left primitive if and only if there is a division ring $D$ and a bimodule ${}_R T_D$ with ${}_R T$ faithful such that for every finite $D$-linearly independent set $x_1, \ldots, x_n \in T$ and every $y_1, \ldots, y_n \in T$ there is an $r \in R$ such that
$$r x_i = y_i \quad (i = 1, \ldots, n).$$
