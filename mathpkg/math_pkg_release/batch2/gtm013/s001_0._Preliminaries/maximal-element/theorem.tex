Let $P$ be a poset. An element $m \in P$ is maximal in $P$ if for all $x \in P$, $x \geq m$ implies $x = m$. Dually, $m$ is minimal in $P$ if $x \leq m$ implies $x = m$. A greatest element of $P$, if it exists, is maximal (and unique); conversely, a poset may have many maximal elements and no greatest element.
