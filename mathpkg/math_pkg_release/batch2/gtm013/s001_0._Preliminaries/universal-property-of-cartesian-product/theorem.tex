Let $(X_{\alpha})_{\alpha \in A}$ be an indexed set of non-empty sets, let $Y$ be a set, and for each $\alpha \in A$, let $f_{\alpha}: Y \to X_{\alpha}$. Then there is a unique function $f: Y \to \prod_{\alpha \in A} X_{\alpha}$ such that $\pi_{\alpha} \circ f = f_{\alpha}$ for each $\alpha \in A$.
