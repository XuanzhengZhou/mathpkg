A concrete category is a category $C = (\mathcal{C}, \operatorname{mor}_C, \circ)$ together with a function $u$ that assigns to each object $A \in \mathcal{C}$ a set $u(A)$ (its underlying set) such that each morphism $f \in \operatorname{mor}_C(A, B)$ is a function $f: u(A) \to u(B)$, the composition $\circ$ is the usual composition of functions, and $1_A$ is the identity function on $u(A)$. An isomorphism $f: A \to B$ in a concrete category is a bijection $f: u(A) \to u(B)$.
