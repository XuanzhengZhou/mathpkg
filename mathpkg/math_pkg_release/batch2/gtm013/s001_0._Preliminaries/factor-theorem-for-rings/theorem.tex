Let $\phi: R \to S$ and $\phi': R \to S'$ be ring homomorphisms with $\phi'$ surjective, and let $K = \operatorname{Ker} \phi$ and $K' = \operatorname{Ker} \phi'$. If $K' \subseteq K$, then there is a unique ring homomorphism $\psi: S' \to S$ such that $\psi \circ \phi' = \phi$. Moreover, $\psi$ is injective if and only if $K = K'$.
