A matrix is row finite if each row contains only finitely many nonzero entries; it is column finite if each column has only finitely many nonzero entries. These finiteness conditions ensure that matrix multiplication, which involves sums over intermediate indices, produces well-defined finite sums. The set of all $\Gamma \times \Lambda$ matrices over $R$ forms an abelian group under entrywise addition.
