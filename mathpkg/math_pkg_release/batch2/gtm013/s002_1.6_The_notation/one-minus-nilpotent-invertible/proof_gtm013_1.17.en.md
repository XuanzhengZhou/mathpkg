Let $x^n = 0$. Compute the product:

$$(1 - x)(1 + x + \cdots + x^{n-1}) = (1 + x + \cdots + x^{n-1}) - (x + x^2 + \cdots + x^n) = 1 - x^n = 1 - 0 = 1.$$

Similarly,

$$(1 + x + \cdots + x^{n-1})(1 - x) = (1 + x + \cdots + x^{n-1}) - (x + x^2 + \cdots + x^n) = 1 - x^n = 1.$$

Thus $1 + x + \cdots + x^{n-1}$ is a two-sided inverse of $1 - x$.
