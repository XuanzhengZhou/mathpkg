For an additive covariant functor $T$, the induced map $\operatorname{Hom}_R(M,N) \to \operatorname{Hom}_S(T(M), T(N))$ is not just a set map but an abelian group homomorphism. This follows directly from the additivity condition $T(f+g) = T(f) + T(g)$ together with the covariance of $T$. It means that additive functors respect the full pre-additive structure of module categories.
