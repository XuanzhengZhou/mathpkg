If $\mu_0$, $\mu_1$, and $\mu_2$ are totally finite measures, and if
$$d\mu_0 = f_1 \, d(\mu_0 + \mu_1) = f_2 \, d(\mu_0 + \mu_2) = f \, d(\mu_0 + \mu_1 + \mu_2),$$
then we have, almost everywhere with respect to $\mu_0 + \mu_1 + \mu_2$,
$$f(x) = \begin{cases}
\dfrac{f_1(x) f_2(x)}{f_1(x) + f_2(x) - f_1(x) f_2(x)} & \text{if } f_1(x) f_2(x) \neq 0, \\[10pt]
0 & \text{if } f_1(x) = f_2(x) = 0.
\end{cases}$$
