If $\mu$ is a totally $\sigma$-finite measure and if $\nu$ is a totally $\sigma$-finite signed measure such that
$$\nu(E) = \int_E f \, d\mu$$
for every measurable set $E$, we write
$$f = \frac{d\nu}{d\mu} \quad \text{or} \quad d\nu = f \, d\mu.$$

The function $\frac{d\nu}{d\mu}$ is called the \text{Radon-Nikodym derivative} (or \text{Radon-Nikodym integrand}) of $\nu$ with respect to $\mu$. It is unique only almost everywhere with respect to $\mu$.
