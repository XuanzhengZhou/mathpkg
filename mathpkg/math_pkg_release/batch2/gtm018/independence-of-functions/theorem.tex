If $(X, \mathbf{S}, \mu)$ is a probability measure space (i.e. $\mu(X) = 1$), a finite set $\{f_1, \cdots, f_n\}$ of measurable functions is independent if, for every Borel set $M_1, \cdots, M_n$,
$$\mu(\{x: f_1(x) \in M_1, \cdots, f_n(x) \in M_n\}) = \prod_{i=1}^{n} \mu(\{x: f_i(x) \in M_i\}).$$
An arbitrary family of measurable functions is independent if every finite subfamily is independent.
