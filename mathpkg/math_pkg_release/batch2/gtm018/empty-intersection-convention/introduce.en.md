When the index set is empty, the intersection is defined as the entire space $X$. This convention ensures that the identity $(\bigcup E_\gamma)' = \bigcap E_\gamma'$ remains valid for all index sets, including the empty one, and is consistent with the monotonicity property that smaller index sets yield larger intersections.
