If $f$ is a real valued function on a set $X$ and $M$ is any subset of the real line, the **inverse image** of $M$ under $f$ is $f^{-1}(M) = \{x : f(x) \in M\}$. The mapping $f^{-1}$ preserves all set operations: $f^{-1}(\bigcup_n M_n) = \bigcup_n f^{-1}(M_n)$ and $f^{-1}(M - N) = f^{-1}(M) - f^{-1}(N)$.
