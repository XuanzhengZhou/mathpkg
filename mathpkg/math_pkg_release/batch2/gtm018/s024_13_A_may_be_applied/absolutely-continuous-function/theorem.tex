A real valued function $f$ of a real variable is called \textit{absolutely continuous} if to every positive number $\epsilon$ there corresponds a positive number $\delta$ such that

$$\sum_{i=1}^{n} |f(b_i) - f(a_i)| < \epsilon$$

for every finite, disjoint class $\{(a_i, b_i): i = 1, \dots, n\}$ of bounded open intervals for which $\sum_{i=1}^{n}(b_i - a_i) < \delta$.
