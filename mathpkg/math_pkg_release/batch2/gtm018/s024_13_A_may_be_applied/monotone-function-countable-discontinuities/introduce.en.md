This classic result of real analysis states that a monotone function can have at most countably many points of discontinuity. Using the measure-theoretic correspondence developed in this section, the proof becomes immediate: represent the monotone function as a distribution function of a finite Borel measure, and note that each discontinuity corresponds to an atom (a point of positive measure), of which there can be only countably many.
