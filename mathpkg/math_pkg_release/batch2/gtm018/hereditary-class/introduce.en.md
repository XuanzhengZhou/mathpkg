A hereditary class contains all subsets of each of its members. Outer measures are defined on hereditary $\sigma$-rings, ensuring that if a set is "small" enough to be measured, all its subsets are also in the domain. The hereditary $\sigma$-ring generated by a class $\mathbf{E}$ is denoted $\mathbf{H}(\mathbf{E})$.
