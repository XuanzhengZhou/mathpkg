If $\mu^*$ is an outer measure on a hereditary $\sigma$-ring $\mathbf{H}$ and if $\bar{\mathbf{S}}$ is the class of all $\mu^*$-measurable sets, then every set of outer measure zero belongs to $\bar{\mathbf{S}}$ and the set function $\bar{\mu}$, defined for $E$ in $\bar{\mathbf{S}}$ by $\bar{\mu}(E) = \mu^*(E)$, is a complete measure on $\bar{\mathbf{S}}$. The measure $\bar{\mu}$ is called the measure induced by the outer measure $\mu^*$.
