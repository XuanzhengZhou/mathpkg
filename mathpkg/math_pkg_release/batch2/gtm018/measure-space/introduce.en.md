A measure space is a triple $(X, \mathbf{S}, \mu)$: a set $X$, a $\sigma$-ring $\mathbf{S}$ of "measurable" subsets, and a measure $\mu$ on $\mathbf{S}$. This is the fundamental setting for integration theory. When $\mathbf{S}$ contains $X$, it is an algebra (or $\sigma$-algebra for countably additive cases).
