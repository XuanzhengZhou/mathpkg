Theorem C shows that if a group element $y$ leaves every set in an invariant $\sigma$-ring $\mathbf{T}$ invariant up to measure zero (i.e., $ho(yE,E) = 0$ for all $E \in \mathbf{T}$), then any integrable function measurable with respect to $\mathbf{T}$ is invariant under translation by $y$. This connects measure-theoretic and functional invariance.
