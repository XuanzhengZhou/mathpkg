If $\mathbf{T}$ is an invariant $\sigma$-ring generated by its sets of finite measure and containing at least one bounded set of positive measure, if $\mathbf{E}$ is a class of sets dense in the metric space of sets of finite measure in $\mathbf{T}$, and if $$Y = \{y : \rho(yE, E) = 0,\ E \in \mathbf{E}\},$$ then $Y$ is a compact invariant subgroup of $X$.
