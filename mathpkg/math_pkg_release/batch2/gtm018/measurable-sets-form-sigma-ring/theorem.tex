If $\mu^*$ is an outer measure on a hereditary $\sigma$-ring $\mathbf{H}$ and if $\bar{\mathbf{S}}$ is the class of all $\mu^*$-measurable sets, then $\bar{\mathbf{S}}$ is a $\sigma$-ring. If $A \in \mathbf{H}$ and if $\{E_n\}$ is a disjoint sequence of sets in $\bar{\mathbf{S}}$ with $\bigcup_{n=1}^{\infty} E_n = E$, then
$$\mu^*(A \cap E) = \sum_{n=1}^{\infty} \mu^*(A \cap E_n).$$
