If $\mu$ is a measure on a $\sigma$-ring $\mathbf{S}$, then the class $\bar{\mathbf{S}}$ of all sets of the form $E \Delta N$, where $E \in \mathbf{S}$ and $N$ is any subset of a set of measure zero in $\mathbf{S}$, is a $\sigma$-ring, and the set function $\bar{\mu}$ defined by $\bar{\mu}(E \Delta N) = \mu(E)$ is a complete measure on $\bar{\mathbf{S}}$. The measure $\bar{\mu}$ is called the completion of $\mu$.
