A set $E$ in $\hat{\mathbf{S}}$ is outer regular if $\hat{\mu}(E) = \inf \{\hat{\mu}(U): E \subset U \in \hat{\mathbf{U}}\}$; inner regular if $\hat{\mu}(E) = \sup \{\hat{\mu}(C): E \supset C \in \hat{\mathbf{C}}\}$; and regular if it is both. A measure $\hat{\mu}$ is regular if every set in $\hat{\mathbf{S}}$ is regular.
