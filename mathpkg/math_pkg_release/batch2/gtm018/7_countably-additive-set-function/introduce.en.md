A countably additive set function (also called $\sigma$-additive) extends finite additivity to countably infinite disjoint sequences. This is the key property that distinguishes genuine measures from merely finitely additive set functions. Countable additivity is the defining property of a measure and is essential for the development of integration theory, as it enables limit theorems such as the monotone convergence theorem.
