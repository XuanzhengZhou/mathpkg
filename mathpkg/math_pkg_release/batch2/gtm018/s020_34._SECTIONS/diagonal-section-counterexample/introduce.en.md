Exercise (2) of Section 34 provides a counterexample showing that the converse of Theorem A is false: it is possible for every section of a set to be measurable while the set itself is not measurable. Taking $X = Y$ as an uncountable set with the $\sigma$-ring of countable subsets, the diagonal $D = \{(x,y) : x = y\}$ has every section either empty or a singleton (hence measurable), but $D$ itself is not in the product $\sigma$-ring.
