If $E$ is any measurable set in $X \times Y$, and if $\alpha$ and $\beta$ are real numbers such that $\alpha > 0$, prove that the set
$$\{ (x,y) : (x, \alpha y + \beta) \in E \}$$
is a measurable subset of $X \times Y$. (Hint: the conclusion is true if $E$ is a measurable rectangle, and the class of all sets for which the conclusion is true is a $\sigma$-ring.)
