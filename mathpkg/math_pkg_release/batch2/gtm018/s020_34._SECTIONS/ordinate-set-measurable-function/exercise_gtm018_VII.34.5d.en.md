If $f$ is a non negative measurable function, prove that $V^*(f)$ and $V_*(f)$ are measurable sets. (Hint: if $f$ is bounded, then there exist sequences $\{g_n\}$ and $\{h_n\}$ of simple functions such that
$$0 \leq g_n \leq g_{n+1} \leq f \leq h_{n+1} \leq h_n, \quad n = 1, 2, \dots$$
and such that $\lim_{n \to \infty} g_n = \lim_{n \to \infty} h_n = f$.)
