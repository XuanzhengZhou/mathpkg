Exercise (5d) of Section 34 generalizes Exercise (5a) from simple functions to all non negative measurable functions. Using approximation by simple functions from above and below (sequences $g_n$ and $h_n$), together with the continuity properties established in (5c), one proves that both the lower ordinate set $V^*(f)$ and the upper ordinate set $V_*(f)$ are measurable for any non negative measurable function $f$.
