The monotone class generated by a class $\mathbf{E}$ of sets, denoted $\mathbf{M}(\mathbf{E})$, is the smallest monotone class that contains $\mathbf{E}$. It is defined as the intersection of all monotone classes containing $\mathbf{E}$. The monotone class theorem establishes that for any ring $\mathbf{R}$, the generated monotone class coincides with the generated $\sigma$-ring: $\mathbf{M}(\mathbf{R}) = \mathbf{S}(\mathbf{R})$.
