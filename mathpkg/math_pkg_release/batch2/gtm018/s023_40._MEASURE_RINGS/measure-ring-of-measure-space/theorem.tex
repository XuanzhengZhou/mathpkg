If $(X, \mathbf{S}, \mu)$ is a measure space, we shall use the symbol $\mathbf{S}(\mu)$ to denote the $\sigma$-ring $\mathbf{S}$ with equality interpreted modulo $\mu$. If two sets $E$ and $F$ in $\mathbf{S}$ are such that $\mu(E \Delta F) = 0$, then we shall consider them equal and we shall write $E = F[\mu]$.

If $E_n = F_n[\mu]$, $n = 1, 2, \cdots$, then

$$E_1 - F_1 = E_2 - F_2 \text{ and } \bigcup_{n=1}^{\infty} E_n = \bigcup_{n=1}^{\infty} F_n[\mu],$$

so that even with the altered concept of equality, $\mathbf{S}$ is a Boolean $\sigma$-ring with respect to the familiar set operations. If $E = F[\mu]$, then $\mu(E) = \mu(F)$, so that even with the altered concept of equality, the measure $\mu$ is unambiguously defined on $\mathbf{S}$. Since the statements $\mu(E) = 0$ and $E = 0[\mu]$ are obviously equivalent, after the alteration of the concept of equality, $\mu$ becomes a positive measure.
