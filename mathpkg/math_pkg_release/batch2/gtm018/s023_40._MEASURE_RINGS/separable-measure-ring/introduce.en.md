A measure ring is called separable if its associated metric space $\mathbf{s}(\mu)$ is separable (i.e., contains a countable dense subset). Separability is a crucial hypothesis in the isomorphism theorem for measure algebras. If a measure ring is generated by a countable set of elements of finite measure, then it is separable -- this provides a practical criterion for verifying separability in applications.
