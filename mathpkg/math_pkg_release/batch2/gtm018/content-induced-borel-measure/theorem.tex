If $\mu^*$ is the outer measure induced by a content $\lambda$, then the set function $\mu$ defined for every Borel set $E$ by $\mu(E) = \mu^*(E)$ is a regular Borel measure. Moreover, $\mu(C) \leq \lambda(C)$ for every compact set $C$, and $\mu(C) = \lambda(C)$ if and only if $\lambda$ is regular.
