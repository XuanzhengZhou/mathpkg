A transformation is a function defined on a set $X$ taking values in a set $Y$. For any subset $E$ of $X$, the image $T(E)$ is the set of all values $T(x)$ for $x \in E$. For any subset $F$ of $Y$, the inverse image $T^{-1}(F)$ consists of all points in $X$ whose image lies in $F$. This concept generalizes the notion of a function and provides the foundational language for studying mappings between measure spaces.
