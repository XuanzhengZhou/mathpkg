Given a compact set $C$ contained in an open set $U$, one can always interpose a compact $G_\delta$ set $C_0$ and a $\sigma$-compact open set $U_0$ between them: $C \subset U_0 \subset C_0 \subset U$. The construction uses a separating function from Theorem B whose sublevel and superlevel sets provide the desired sandwich. This result is foundational for the theory of Baire sets, as it shows that every compact set can be approximated from inside by $\sigma$-compact open Baire sets and from outside by compact $G_\delta$ Baire sets.
