In a locally compact Hausdorff space, a compact set $C$ and a closed set $F$ that are disjoint can be completely separated by a continuous function $f$ taking values in $[0,1]$. The function vanishes identically on $C$ and equals $1$ on $F$. This is a variant of Urysohn's lemma specialized to the setting where one set is compact; the proof uses the complete regularity of the space and a finite open cover argument to construct the separating function via a product of local separating functions.
