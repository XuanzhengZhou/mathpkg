If $\mathbf{E}$ is any class of subsets of $X$, the set of all those points of $X$ which belong to every set of the class $\mathbf{E}$ is called the intersection of the sets of $\mathbf{E}$; it will be denoted by
$$\bigcap \mathbf{E} \quad \text{or} \quad \bigcap \{E : E \in \mathbf{E}\}.$$
If the index set $\Gamma$ is empty, we shall make the convention that
$$\bigcap_{\gamma \in \Gamma} E_\gamma = X.$$
