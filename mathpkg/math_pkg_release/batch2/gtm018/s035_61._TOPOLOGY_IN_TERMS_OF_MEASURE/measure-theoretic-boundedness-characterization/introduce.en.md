This theorem provides a purely measure-theoretic characterization of bounded subsets of a locally compact topological group: a set is bounded precisely when it is contained in a sublevel set $\{x: \rho(xE,E) \leq \epsilon\}$ for some Borel set $E$ of finite measure and some $\epsilon$ strictly less than $2\mu(E)$. This illustrates how topological boundedness can be described entirely in terms of the Haar measure.
