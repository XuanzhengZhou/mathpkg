A function $f$ defined on a measurable set $E$ is measurable on $E$ if $E \cap f^{-1}(M)$ is measurable for all Borel sets $M$. A measurable function on the whole space is automatically measurable on every measurable subset. If the whole space $X$ is measurable, the requirement simplifies to $f^{-1}(M)$ being measurable for every Borel set $M$.
