If $\mu$ is a set function on the ring $\mathbf{F}$ of all measurable finite dimensional subsets of $X = \times_{n=1}^{\infty} X_n$ (where each $X_n$ is the unit interval) such that, for each positive integer $n$, $\mu$ is a probability measure on the $\sigma$-ring $\mathbf{F}_n$ of all measurable $\{1,\ldots,n\}$-cylinders, then $\mu$ has a unique extension to a probability measure on $\mathbf{S}$.
