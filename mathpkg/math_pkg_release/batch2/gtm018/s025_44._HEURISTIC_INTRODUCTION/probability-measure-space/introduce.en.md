The summary definition of probability theory: numerical probability is a measure $\mu$ on a Boolean $\sigma$-algebra $\mathbf{S}$ of subsets of a set $X$ such that $\mu(X) = 1$. This unifies all the previous requirements: events are sets in $\mathbf{S}$, probabilities are values of $\mu$, finite and countable additivity hold, the impossible event has measure $0$ and the certain event has measure $1$. This definition embeds probability theory firmly within the framework of measure theory developed in the preceding chapters.
