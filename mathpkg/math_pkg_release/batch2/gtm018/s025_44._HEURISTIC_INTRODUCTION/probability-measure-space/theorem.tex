Numerical probability is a measure $\mu$ on a Boolean $\sigma$-algebra $\mathbf{S}$ of subsets of a set $X$ such that $\mu(X) = 1$. That is, a probability space is a triple $(X, \mathbf{S}, \mu)$ where:
\begin{itemize}
\item $X$ is a set (the sample space, or certain event);
\item $\mathbf{S}$ is a $\sigma$-algebra of subsets of $X$ (the family of events);
\item $\mu: \mathbf{S} \to [0,1]$ is a countably additive measure with $\mu(X) = 1$.
\end{itemize}
Equivalently, $\mu$ is a measure on $(X, \mathbf{S})$ normalized so that the total measure of the space is $1$.
