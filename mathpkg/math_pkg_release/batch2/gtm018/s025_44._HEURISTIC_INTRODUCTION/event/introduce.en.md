An event is the principal undefined term in the theory of probability. Intuitively, an event is one of the possible outcomes of a physical experiment. In the measure-theoretic formulation, events correspond to sets in a Boolean $\sigma$-algebra of subsets of a fixed set $X$, with $X$ representing the certain event and $\emptyset$ the impossible event. For example, in the experiment of rolling a six-sided die, the event "the number $x$ is even" corresponds to the set $\{2,4,6\}$.
