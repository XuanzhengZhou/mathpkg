The impossible event, denoted by $0$ or $\emptyset$, is the event that never occurs. The certain event, denoted by $X$, is the event that always occurs. In the set-theoretic model, $X$ is the entire underlying set and $\emptyset$ is the empty set. For the experiment of rolling a six-sided die,
$$X = \{1,2,3,4,5,6\}.$$
The impossible event is $0 = \emptyset$, representing the event that the number showing is not any of the first six positive integers.
