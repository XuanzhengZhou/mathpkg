For any two events $E$ and $F$, their intersection $E \cap F$ is the event that occurs if and only if both $E$ and $F$ occur simultaneously. In set-theoretic terms, $E \cap F$ is the set-theoretic intersection. For example, if $E = \{2,4,6\}$ (even) and $F = \{1,2,3\}$ (less than 4), then $E \cap F = \{2\}$.
