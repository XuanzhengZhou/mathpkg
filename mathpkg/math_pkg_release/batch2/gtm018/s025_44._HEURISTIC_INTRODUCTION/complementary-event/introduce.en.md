The complementary event $E'$ of an event $E$ is the event that occurs precisely when $E$ does not occur. This is the simplest algebraic operation on events. In the set-theoretic model, $E'$ is the complement of $E$ relative to the certain event $X$. For example, if $E = \{2,4,6\}$ is the event that a die roll yields an even number, then $E' = \{1,3,5\}$ is the event that the roll yields an odd number.
