Probability is introduced as a numerically valued set function $\mu$ on the Boolean $\sigma$-algebra of events. The value $\mu(E) \in [0,1]$ is interpreted as the long-run proportion of trials in which $E$ occurs. This intuitive interpretation motivates the requirements that $\mu$ be non-negative, that it satisfy additivity for disjoint events, and that $\mu(X) = 1$ for the certain event $X$.
