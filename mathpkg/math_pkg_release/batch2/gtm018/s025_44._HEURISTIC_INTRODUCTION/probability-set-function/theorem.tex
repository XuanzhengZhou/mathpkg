Probability is a numerically valued function $\mu$ defined on events $E$, i.e., on the sets of a Boolean $\sigma$-algebra. The number $\mu(E)$ should give information about the occurrence habits of $E$. If in a large number of repetitions, $E$ occurs a fraction $p$ of the time, then we write $\mu(E) = p$. Consequently:
\begin{itemize}
\item $\mu(E)$ is a non-negative real number,
\item $\mu(E) \in [0,1]$, the unit interval,
\item $\mu(X) = 1$ for the certain event $X$.
\end{itemize}
