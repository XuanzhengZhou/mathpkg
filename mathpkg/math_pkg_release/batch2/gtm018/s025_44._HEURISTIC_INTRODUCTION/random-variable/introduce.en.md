A random variable is a function whose numerical values are determined by the outcome of a random experiment. In the measure-theoretic formulation, a random variable is a measurable function $f$ on the probability space $(X, \mathbf{S}, \mu)$. For example, the function $f(x) = x$ on the die-rolling sample space $X = \{1,2,3,4,5,6\}$ is a random variable representing the number showing on the die. The measurability requirement ensures that one can compute probabilities such as $\mu(\alpha \leq f \leq \beta)$ for any interval $[\alpha, \beta]$.
