If $\{E_n\}_{n=1}^{\infty}$ is an infinite sequence of pairwise disjoint events in the Boolean $\sigma$-algebra $\mathbf{S}$, then
$$\mu\left(\bigcup_{n=1}^{\infty} E_n\right) = \sum_{n=1}^{\infty} \mu(E_n).$$
This condition of countable additivity (or $\sigma$-additivity) is a further restriction on the set function $\mu$, beyond finite additivity, and is essential for modern probability theory.
