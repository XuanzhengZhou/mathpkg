We have $\mu(F) = \mu(F - E) + \mu(E)$. If exactly one of the summands is infinite, then so is $\mu(F)$; if they are both infinite, then (since $\mu$ assumes at most one of the values $+\infty$ and $-\infty$) they are equal and again $\mu(F)$ is infinite. Only one possibility remains, namely that both summands are finite, and this proves that every measurable subset of a set of finite signed measure has finite signed measure.
