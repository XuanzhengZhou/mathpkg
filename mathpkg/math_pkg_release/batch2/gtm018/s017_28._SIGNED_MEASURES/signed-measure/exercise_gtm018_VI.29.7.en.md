If $\mu$ is a signed measure and $f$ is a measurable function such that $f$ is integrable with respect to $|\mu|$, then we may write, by definition,
$$\int f\,d\mu = \int f\,d\mu^+ - \int f\,d\mu^-.$$
This integral has many of the essential properties of the "positive" integrals discussed in Chapter V. If $\mu$ is a finite signed measure, then, for every measurable set $E$,
$$|\mu|(E) = \sup \left| \int_E f\,d\mu \right|,$$
where the supremum is extended over all measurable functions $f$ such that $|f| \leq 1$.
