A complex measure on the class of all measurable sets of a measurable space is a set function $\mu$ such that, for every measurable set $E$, $\mu(E) = \mu_1(E) + i\mu_2(E)$, where $i = \sqrt{-1}$, and where $\mu_1$ and $\mu_2$ are signed measures in the sense of this section. Are Theorems A, B, and C true for complex measures?
