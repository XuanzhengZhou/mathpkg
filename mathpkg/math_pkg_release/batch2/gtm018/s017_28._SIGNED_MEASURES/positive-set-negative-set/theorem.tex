A measurable set $E$ is \textit{positive} (with respect to $\mu$) if, for every measurable set $F$, $E \cap F$ is measurable and $\mu(E \cap F) \geq 0$.

A measurable set $E$ is \textit{negative} (with respect to $\mu$) if, for every measurable set $F$, $E \cap F$ is measurable and $\mu(E \cap F) \leq 0$.

The empty set is both positive and negative in this sense.
