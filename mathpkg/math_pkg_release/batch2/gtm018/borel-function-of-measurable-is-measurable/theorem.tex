**Theorem B.** If $\phi$ is an extended real valued Borel measurable function on the extended real line such that $\phi(0) = 0$, and if $f$ is an extended real valued measurable function on a measurable space $X$, then the function $\tilde{f}$, defined by $\tilde{f}(x) = \phi(f(x))$, is a measurable function on $X$.
