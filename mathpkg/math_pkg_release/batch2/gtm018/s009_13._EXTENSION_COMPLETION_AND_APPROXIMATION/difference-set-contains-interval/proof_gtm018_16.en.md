If $E$ contains an open interval, the result is trivial. Otherwise, by Theorem A, find open $U$ with $\bar{\mu}(E \cap U) \geq \frac{3}{4}\mu(U)$. For $|x| < \frac{1}{2}\mu(U)$, the sets $E \cap U$ and $(E \cap U) + x$ intersect, so $x \in D(E)$. Thus $(-\frac{1}{2}\mu(U), \frac{1}{2}\mu(U)) \subset D(E)$.
