Vitali's classic construction using the Axiom of Choice shows that not every subset of $\mathbb{R}$ is Lebesgue measurable. One representative from each coset of the dense subgroup $A = \{n + m\xi\}$ yields a set with inner measure 0 whose translates cover the line, forcing a contradiction if it were measurable.
