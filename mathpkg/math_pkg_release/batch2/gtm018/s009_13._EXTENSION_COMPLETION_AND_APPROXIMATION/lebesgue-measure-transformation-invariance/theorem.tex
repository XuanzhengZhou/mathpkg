Let $T(x) = \alpha x + \beta$ with $\alpha,\beta \in \mathbb{R}$, $\alpha \neq 0$, and $T(E) = \{ \alpha x + \beta : x \in E \}$. Then

$$\mu^*(T(E)) = |\alpha| \mu^*(E) \quad \text{and} \quad \mu_*(T(E)) = |\alpha| \mu_*(E).$$

$T(E)$ is a Borel set or Lebesgue measurable set iff $E$ is, respectively.
