For any $a$, $\{a\} = \bigcap_{n=1}^{\infty} [a, a + 1/n)$, so $\mu(\{a\}) = \lim_n \mu([a, a+1/n)) = \lim_n 1/n = 0$. Each singleton is a Borel set; since Borel sets form a $\sigma$-ring and $\mu$ is countably additive, every countable set is a Borel set of measure zero.
