A finitely additive set function extends the notion of additivity from two sets to any finite collection of pairwise disjoint sets. This property generalizes the additive property by requiring additivity for an arbitrary finite number of disjoint sets whose union remains in the domain class. Every measure is finitely additive, as can be seen by appending countably many copies of the empty set to a finite disjoint collection.
