If $(X, \mathbf{S})$ is a measurable space (i.e. a set $X$ and a $\sigma$-ring $\mathbf{S}$ of subsets of $X$), a real valued function $f$ on a set $E$ in $\mathbf{S}$ is called measurable (on $E$) if, for every real number $c$, the set
$$\{x \in E: f(x) < c\}$$
belongs to $\mathbf{S}$. An extended real valued function is called measurable if it satisfies the same condition and, additionally, the sets $\{x: f(x) = +\infty\}$ and $\{x: f(x) = -\infty\}$ are measurable.
