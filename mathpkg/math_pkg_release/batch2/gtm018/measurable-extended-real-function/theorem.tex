An extended real valued function $f$ is **measurable** if the one-point sets $\{+\infty\}$ and $\{-\infty\}$ of the extended real line are regarded as Borel sets, and for every Borel set $M$ of real numbers, each of the three sets $f^{-1}(\{+\infty\})$, $f^{-1}(\{-\infty\})$, and $N(f) \cap f^{-1}(M)$ is measurable.
