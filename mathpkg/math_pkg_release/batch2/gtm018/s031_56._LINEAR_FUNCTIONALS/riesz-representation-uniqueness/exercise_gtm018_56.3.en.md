If $\mu_0$ is a Baire measure and $\Lambda(f) = \int f \, d\mu_0$ for every $f$ in $\mathcal{L}$, define

$$\lambda_*(U) = \sup \left\{ \Lambda(f) : U \supset f \in \mathcal{L}_+ \right\}$$

for every $U$ in $\mathbf{U}$ (bounded open sets), and

$$\mu^*(E) = \inf \left\{ \lambda_*(U) : E \subset U \in \mathbf{U} \right\}$$

for every $\sigma$-bounded set $E$. Show that $\mu^*(E) = \mu_0(E)$ for every Baire set $E$.
