If $\Lambda$ is a positive linear functional on $\mathcal{L}$, if, for every $C$ in $\mathbf{C}$,

$$\lambda(C) = \inf \{ \Lambda(f) : C \subset f \in \mathcal{L}_+ \},$$

and if $\mu$ is the Borel measure induced by the content $\lambda$, then, corresponding to every compact set $C$ and every positive number $\epsilon$, there exists a function $f_0$ in $\mathcal{L}_+$ such that $C \subset f_0$, $f_0 \leq 1$, and

$$\Lambda(f_0) \leq \int f_0 \, d\mu + \epsilon.$$
