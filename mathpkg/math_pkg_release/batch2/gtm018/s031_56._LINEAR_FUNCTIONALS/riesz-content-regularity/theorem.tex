If $\Lambda$ is a positive linear functional on $\mathcal{L}$, and if for every compact set $C$ in $\mathbf{C}$,

$$\lambda(C) = \inf \left\{ \Lambda(f) : C \subset f \in \mathcal{L}_+ \right\},$$

then $\lambda$ is a regular content on $\mathbf{C}$ (i.e., $\lambda$ is finite, nonnegative, monotone, subadditive, additive on disjoint compact sets, and regular). Moreover, if $\mu$ is the Borel measure induced by the content $\lambda$, then for every compact set $C$ and every bounded open set $U$ containing $C$,

$$\mu(C) \leq \Lambda(f)$$

for every $f \in \mathcal{L}_+$ with $C \subset f$ (meaning $f(x) = 1$ on $C$ and $f(x) = 0$ outside $U$). Consequently, $\mu(U) = \sup \{\mu(C) : C \subset U,\ C \in \mathbf{C}\} \leq \Lambda(f)$.
