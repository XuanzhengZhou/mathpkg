A positive linear functional on the class $\mathcal{L}$ of continuous functions vanishing at infinity is a linear functional $\Lambda$ that sends every nonnegative function $f \in \mathcal{L}_+$ to a nonnegative real number. This positivity condition is the essential hypothesis for the Riesz Representation Theorem: every positive linear functional on $\mathcal{L}$ arises as integration against a uniquely determined regular Borel measure. Positivity implies monotonicity: if $f \geq g$, then $\Lambda(f) \geq \Lambda(g)$.
