If $f$ is a measurable function on a totally finite measure space $(X,\mathbf{S},\mu)$ and, for every Borel set $M$, $\nu(M) = \mu(f^{-1}(M))$, then $\nu$ is a measure on the Borel sets. If $f$ is finite valued, the **distribution function** $g$ of $f$ is defined by $g(t) = \mu(\{x : f(x) < t\})$. It is monotone increasing, continuous on the left, $g(-\infty) = 0$, $g(\infty) = \mu(X)$. If $g$ is continuous, the Lebesgue-Stieltjes measure induced by $g$ is the completion of $\nu$.
