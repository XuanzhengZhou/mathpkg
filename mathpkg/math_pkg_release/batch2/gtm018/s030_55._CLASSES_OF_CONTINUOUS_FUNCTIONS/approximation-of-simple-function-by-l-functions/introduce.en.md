Theorem D establishes that any simple function whose supporting sets are compact Baire sets can be approximated arbitrarily well in the $L^1$ norm by a function in $\mathcal{L}$ (a continuous function with compact support). The proof uses the existence of bounded open Baire sets separating disjoint compact sets and Urysohn-type functions in $\mathcal{L}$ that equal $1$ on a given compact set and vanish outside a given open neighborhood. Together with Theorem C (Lusin's theorem), this shows that $\mathcal{L}$ is $L^1$-dense in the space of integrable Baire functions.
