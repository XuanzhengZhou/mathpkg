Let $X$ be a locally compact Hausdorff space. The class $\mathcal{L}(X)$, or simply $\mathcal{L}$, is the set of all real-valued continuous functions on $X$ which vanish outside a compact set. Equivalently, $\mathcal{L}$ is the class of all continuous functions $f$ on $X$ for which the set
$$N(f) = \{x : f(x) \neq 0\}$$
is bounded. If $X$ is not compact and $X^*$ is the one-point compactification of $X$ by $x^*$ (the point at infinity), then $\mathcal{L}$ is the class of all continuous functions which vanish in a neighborhood of infinity.
