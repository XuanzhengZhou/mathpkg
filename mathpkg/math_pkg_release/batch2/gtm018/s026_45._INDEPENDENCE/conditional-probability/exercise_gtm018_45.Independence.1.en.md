If $F$ is a measurable set of positive measure in a probability space $(X, \mathbf{S}, \mu)$, and if, for every measurable set $E$,

$$\mu_F(E) = \frac{\mu(F \cap E)}{\mu(F)},$$

then $\mu_F$ is a probability measure on $\mathbf{S}$ such that $\mu_F(F) = 1$; the sets $E$ and $F$ are independent if and only if $\mu_F(E) = \mu(E)$. The number $\mu_F(E)$ is called the conditional probability of $E$ given $F$.
