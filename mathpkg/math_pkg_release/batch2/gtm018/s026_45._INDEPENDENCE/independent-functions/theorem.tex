If $\{f\}$ is a finite or infinite class of real valued measurable functions on a probability space $(X, \mathbf{S}, \mu)$, the functions of the set $\{f\}$ are \text{(stochastically) independent} if

$$\mu\left(\bigcap_{i=1}^{n} \{x : f_i(x) \in M_i\}\right) = \prod_{i=1}^{n} \mu\left(\{x : f_i(x) \in M_i\}\right)$$

for every finite subset $\{f_i : i = 1, \dots, n\}$ of distinct functions in $\{f\}$ and every finite class $\{M_i : i = 1, \dots, n\}$ of Borel sets on the real line. An equivalent way of expressing this condition is to say that if, for each $f$ in $\{f\}$, $M_f$ is a Borel set on the real line, then, for every possible choice of the Borel sets $M_f$, the sets of the class $\mathbf{E} = \{f^{-1}(M_f) : f \in \{f\}\}$ are independent.
