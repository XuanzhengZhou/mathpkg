If $\{f_{ij} : i = 1, \dots, k; \, j = 1, \dots, n_i\}$ is a set of independent functions, if $\phi_i$ is a real valued, Borel measurable function of $n_i$ real variables, $i = 1, \dots, k$, and if

$$f_i(x) = \phi_i\bigl(f_{i1}(x), \dots, f_{in_i}(x)\bigr),$$

then the functions $f_1, \dots, f_k$ are independent.
