An outer measure is a countably subadditive, monotone, non-negative set function vanishing at the empty set, defined on a hereditary $\sigma$-ring. Outer measures are the principal tool for extending measures from rings to larger classes: Carathéodory's method uses the outer measure induced by a measure to define measurability and obtain a complete measure extension.
