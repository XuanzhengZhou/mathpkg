An a.e. finite valued, measurable function $f$ on a measure space $(X,\mathbf{S},\mu)$ is **integrable** if there exists a mean fundamental sequence $\{f_n\}$ of integrable simple functions which converges in measure to $f$. The **integral** of $f$ is $\int f d\mu = \lim_n \int f_n d\mu$. This value is finite and uniquely determined. The integral over a set $E$ is $\int_E f d\mu = \int \chi_E f d\mu$.
