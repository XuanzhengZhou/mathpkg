The support $N(f)$ of an integrable function has $\sigma$-finite measure. This follows because $f$ is approximated by simple functions, each with finite-measure support, and the set where $f \neq 0$ but all approximants vanish must be null (by Theorem E). This property ensures integrable functions are concentrated on $\sigma$-finite sets.
