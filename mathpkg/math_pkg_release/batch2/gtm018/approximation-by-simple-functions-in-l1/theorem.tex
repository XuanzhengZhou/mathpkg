**Theorem A.** If $\{f_n\}$ is a mean fundamental sequence of integrable simple functions which converges in measure to the integrable function $f$, then $\rho(f, f_n) = \int |f - f_n| d\mu \to 0$ as $n \to \infty$. Hence, to every integrable function $f$ and every $\epsilon > 0$, there corresponds an integrable simple function $g$ such that $\rho(f, g) < \epsilon$.
