An essentially bounded measurable function is integrable over any set of finite measure. This follows from Theorem D: $\chi_E$ is integrable (since $\mu(E) < \infty$) and $f$ is essentially bounded, so $\chi_E f$ is integrable. This gives a broad sufficient condition for integrability: bounded functions on finite measure spaces are integrable.
