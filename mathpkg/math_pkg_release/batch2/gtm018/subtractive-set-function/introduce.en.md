A subtractive set function satisfies the natural formula $\mu(F - E) = \mu(F) - \mu(E)$ for the measure of a difference, provided the subtracted set has finite measure (to avoid $\infty - \infty$). Every measure is subtractive, a property used constantly when manipulating measures of set differences.
