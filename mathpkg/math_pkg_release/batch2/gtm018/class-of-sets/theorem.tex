In addition to sets of points we shall have frequent occasion to consider also sets of sets. We shall always use the word class for a set of sets. The same notations and terminology will be used for classes as for sets. Thus, if $E$ is a set and $\mathbf{E}$ is a class of sets, then
$$E \in \mathbf{E}$$
means that the set $E$ belongs to the class $\mathbf{E}$; if $\mathbf{E}$ and $\mathbf{F}$ are classes, then
$$\mathbf{E} \subset \mathbf{F}$$
means that every set of $\mathbf{E}$ belongs also to $\mathbf{F}$, i.e. that $\mathbf{E}$ is a subclass of $\mathbf{F}$.
