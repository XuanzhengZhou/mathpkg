A simple function is a finite linear combination of characteristic functions of measurable sets,
$$f(x) = \sum_{i=1}^{n} \alpha_i \chi_{E_i}(x),$$
where each $E_i$ is a measurable set of finite measure. A simple function is integrable if each $\alpha_i \neq 0$ implies $\mu(E_i) < \infty$; its integral is defined by
$$\int f \,d\mu = \sum_{i=1}^{n} \alpha_i \mu(E_i).$$
