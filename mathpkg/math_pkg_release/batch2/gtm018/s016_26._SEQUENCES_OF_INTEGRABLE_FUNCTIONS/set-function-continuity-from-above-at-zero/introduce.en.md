This definition captures the property that a finite-valued set function $\nu$ on a class of sets vanishes along any decreasing sequence of sets whose limit is the empty set. For a decreasing sequence $\{E_n\}$ with $\lim_n E_n = \emptyset$, continuity from above at $0$ requires that $\nu(E_n) \to 0$. This condition is satisfied by finite measures and by indefinite integrals of non-negative integrable functions (by 9.E).
