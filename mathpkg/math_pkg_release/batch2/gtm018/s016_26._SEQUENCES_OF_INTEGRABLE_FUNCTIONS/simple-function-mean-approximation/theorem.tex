If $\{f_n\}$ is a mean fundamental sequence of integrable simple functions which converges in measure to the integrable function $f$, then

$$\rho(f, f_n) = \int |f - f_n| \, d\mu \to 0 \quad \text{as } n \to \infty;$$

hence, to every integrable function $f$ and to every positive number $\epsilon$, there corresponds an integrable simple function $g$ such that $\rho(f, g) < \epsilon$.
