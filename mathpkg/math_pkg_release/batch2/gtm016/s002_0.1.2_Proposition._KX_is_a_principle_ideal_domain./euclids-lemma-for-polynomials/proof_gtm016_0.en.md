Let $d(X) = \gcd(f(X), g(X))$. Since $f(X)$ is irreducible, $d(X)$ is a unit or an associate of $f(X)$. If associate, $f(X) \mid g(X)$. If unit, $1 = f(X)a(X) + g(X)b(X)$. Multiply by $h(X)$: $h(X) = f(X)a(X)h(X) + g(X)h(X)b(X)$. Since $f(X) \mid g(X)h(X)$, $f(X) \mid h(X)$.
