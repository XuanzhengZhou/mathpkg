If $s \in k_{\text{Alg}}^G$, then for any root $s'$ of $f_s(X)$, there is $\sigma \in G$ with $\sigma(s) = s'$, so $s = s'$. Conversely, if $f_s(X)$ has only one root, then for any $\sigma \in G$, $\sigma(s)$ is a root of $f_s(X)$, hence $\sigma(s) = s$.
