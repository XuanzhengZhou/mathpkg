$\Delta$ and $\varepsilon$ are $K$-linear mappings such that:

1. $\sum_i {}_r x \otimes {}_r(x_i) \otimes (x_i)_r = \sum_i {}_r(x) \otimes (x)_r \otimes x_i$ for $x \in H(K/k)$ ($\Delta$ is coassociative);
2. $x = \sum_i \varepsilon({}_i x)x_i = \sum_i \varepsilon(x_i) \, {}_i x$ for $x \in H(K/k)$ ($\varepsilon$ is a coidentity);
3. $\Delta(I) = I \otimes I$ and $\Delta(xy) = \sum_{i,j} {}_j x \, {}_i y \otimes x_j y_i$ for $x, y \in H(K/k)$ ($\Delta$ preserves products);
4. $\varepsilon(I) = I$ and $\varepsilon(xy) = \varepsilon(x)\varepsilon(y)$ for $x, y \in H(K/k)$ ($\varepsilon$ preserves products).
