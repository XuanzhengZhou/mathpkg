If $s$ is radical over $k$, then $s^{p^e} \in k$. For any $t = \sum a_i s^i \in k(s)$, $t^{p^e} = \sum a_i^{p^e} s^{ip^e} \in k$, so $t$ is radical. For separable, similar reasoning using the fact that the separable degree equals the number of $k$-homomorphisms.
