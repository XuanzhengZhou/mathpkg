By Theorem 0.2.2, $G = \prod G_{p_i}$ (internal direct product). Each $G_{p_i}$ has an element $g_i$ whose order equals $\operatorname{Exp} G_{p_i}$. Let $|g_i| = p_i^{e_i}$. Then $g = \prod g_i$ has order $\prod p_i^{e_i}$, which equals $\operatorname{Exp} G$.
