Suppose $\mathcal{F} \subset C(G, \Omega)$ is equicontinuous at each point of $G$; then $\mathcal{F}$

and Proposition 1.16 implies there are functions $f_1, \ldots, f_n$ in $\mathcal{F}$ such that for each $f$ in $\mathcal{F}$ there is at least one $f_k$ with

$$\sup \{d(f(z), f_k(z)) : z \in K\} < \frac{\epsilon}{3}.$$

But since each $f_k$ is continuous there is a $\delta, 0 < \delta < R$, such that $|z - z_0| < \delta$ implies that

$$d(f_k(z), f_k(z_0)) < \frac{\epsilon}{3}$$

for $1 \leq k \leq n$. Therefore, if $|z - z_0| < \delta$, $f \in \mathcal{F}$, and $k$ is chosen so that (1.24) holds, then

$$d(f(z), f(z_0)) \leq d(f(z), f_k(z)) + d(f_k(z), f_k(z_0)) + d(f_k(z_0), f(z_0))$$

$$< \epsilon$$

That is, $\mathcal{F}$ is equicontinuous at $z_0$.
