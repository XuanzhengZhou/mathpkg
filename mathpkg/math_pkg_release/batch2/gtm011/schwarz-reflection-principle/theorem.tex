Let $G$ be a region such that $G = G^*$ (symmetric with respect to the real axis). If $f: G_+ \cup G_0 \to \mathbb{C}$ is a continuous function which is analytic on $G_+$ and if $f(x)$ is real for $x$ in $G_0$, then there is an analytic function $g: G \to \mathbb{C}$ such that $g(z) = f(z)$ for $z$ in $G_+ \cup G_0$.
