A set $\mathcal{F} \subset C(G, \Omega)$ is normal if each sequence in $\mathcal{F}$ has a subsequence which converges to a function $f$ in $C(G, \Omega)$.

This of course looks like the definition of sequentially compact subsets, but the limit of the subsequence is not required to be in the set $\mathcal{F}$. The next proof is left to the reader.
