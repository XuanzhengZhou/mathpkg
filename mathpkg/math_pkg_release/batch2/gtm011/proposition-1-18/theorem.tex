$\left(\prod_{n=1}^{\infty} X_n, d\right)$, where $d$ is defined by (1.17), is a metric space. If $\xi^k = \{x_n^k\}_{n=1}^{\infty}$ is in $X = \prod_{n=1}^{\infty} X_n$ then $\xi^k \rightarrow \xi = \{x_n\}$ iff $x_n^k \rightarrow x_n$ for each $n$. Also, if each $(X_n, d_n)$ is compact then $X$ is compact.
