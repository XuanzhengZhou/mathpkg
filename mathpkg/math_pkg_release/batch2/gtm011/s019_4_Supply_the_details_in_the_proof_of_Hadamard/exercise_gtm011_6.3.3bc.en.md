Let $f: [a, b] \to \mathbb{R}$ be a function. Show that $f$ is convex if and only if for any $a \leq x < u < y \leq b$,
\[
\frac{f(u) - f(x)}{u - x} \leq \frac{f(y) - f(x)}{y - x}.
\]

Let $f: [a, b] \to \mathbb{R}$ be a function. Show that $f$ is convex if and only if for any $a \leq x < u < y \leq b$,
\[
\frac{f(u) - f(x)}{u - x} \leq \frac{f(y) - f(u)}{y - u}.
