Let $G$ be the open unit disk $B(0; 1)$ and let $K = \{z : \frac{1}{4} \leq |z| \leq \frac{3}{4}\}$. Show that there is a function $f$ analytic on some open subset $G_1$ containing $K$ which cannot be approximated on $K$ by functions in $H(G)$.

*Remarks.* This problem is concerned with the following question. Given a compact set $K$ contained in an open set $G_1 \subset G$, can functions in $H(G_1)$ be approximated on $K$ by functions in $H(G)$? Exercise 2 says that for an arbitrary choice of $K$, $G$, and $G_1$ this is not true. Exercise 4 below gives criteria for a fixed $K$ and $G$ such that this can be done for any $G_1$. Exercise 3 is a lemma which is useful in proving Exercise 4.
