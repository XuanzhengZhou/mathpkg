If $G$ is an open subset of $\mathbb{C}$ such that $\mathbb{C}_{\infty} - G$ is connected then for each analytic function $f$ on $G$ there is a sequence of polynomials $\{p_n\}$ such that $f = \lim p_n$ in $H(G)$.

$$|p(w)| \leq \max \{|p(z)| : z \in K\}.$$

That is, if the right hand side of this inequality is denoted by $\|p\|_K$, then

$$\hat{K} = \{w : |p(w)| \leq \|p\|_K \text{ for all polynomials } p\}.$$

If $K$ is an annulus then $\hat{K}$ is the disk obtained by filling in the interior hole. In fact, if $K$ is any compact set the Maximum Modulus Theorem gives that $\hat{K}$ is obtained by filling in any "holes" that may exist in $K$.
