Let $\gamma$ be a rectifiable curve and let $K$ be a compact set such that $K \cap \{\gamma\} = \square$. If $f$ is a continuous function on $\{\gamma\}$ and $\epsilon > 0$ then there is a rational function $R(z)$ having all its poles on $\{\gamma\}$ and such that

$$\left| \int_{\gamma} \frac{f(w)}{w-z} \, dw - R(z) \right| < \epsilon$$

for all $z$ in $K$.
