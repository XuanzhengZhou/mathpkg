Prove Carathéodory's Inequality whose statement is as follows: Let $f$ be analytic on $\bar{B}(0; R)$ and let

$$M(r) = \max\{|f(z)| : |z| = r\}, \quad A(r) = \max\{\operatorname{Re} f(z) : |z| = r\};$$

then for $0 < r < R$, if $A(r) \geq 0$,

$$M(r) \leq \frac{R + r}{R - r} \left[A(R) + |f(0)|\right].$$

(Hint: First consider the case where $f(0) = 0$ and examine the function $g(z) = f(Rz) [2A(R) + f(Rz)]^{-1}$ for $|z| < 1$.)
