A topology on a set $X$ is a collection $\mathcal{T}$ of subsets of $X$, called open sets, that contains the empty set and $X$ itself, is closed under finite intersections, and is closed under arbitrary unions. This definition abstracts the three essential properties of open sets in a metric space. A set $X$ together with a topology $\mathcal{T}$ is called a topological space $(X, \mathcal{T})$, and the study of topological spaces forms the foundation of point-set topology.
