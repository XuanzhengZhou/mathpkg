A Möbius transformation $S(z) = \frac{az+b}{cz+d}$ can be considered as a mapping defined on the extended complex plane $\mathbb{C}_\infty$ by setting
$$S(\infty) = \frac{a}{c}, \qquad S\!\left(-\frac{d}{c}\right) = \infty,$$
provided $c \neq 0$. If $c = 0$, then $S(\infty) = \infty$. Note that we cannot have $a = 0 = c$ or $d = 0 = c$ simultaneously with $ad - bc \neq 0$.
