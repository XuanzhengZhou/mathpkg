In an $a$-star shaped open set $G$, every closed rectifiable curve can be continuously shrunk to the constant curve at $a$ while staying entirely within $G$. The homotopy is given by the linear deformation $\gamma_t(s) = a + t(\gamma(s) - a)$, which uses the star-shaped property to ensure all intermediate curves lie in $G$. This proposition provides the cleanest path to Cauchy's Theorem: in star-shaped regions, every closed curve is homotopic to a point, so integrals of analytic functions over such curves vanish.
