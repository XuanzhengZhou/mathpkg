A set $G$ is \textit{convex} if given any two points $a$ and $b$ in $G$ the line segment joining $a$ and $b$, $[a, b]$, lies entirely in $G$.

The set $G$ is \textit{star shaped} if there is a point $a$ in $G$ such that for each $z$ in $G$, the line segment $[a, z]$ lies entirely in $G$. We say that $G$ is \textit{$a$-star shaped} if $[a, z] \subset G$ whenever $z \in G$.

Clearly each convex set is star shaped but the converse is false.
