A set $G$ is convex if the straight line segment between any two points of $G$ stays inside $G$. A weaker condition is star-shaped: there exists a "center" point $a \in G$ such that every line segment from $a$ to any other point in $G$ lies in $G$. Every convex set is star-shaped (any point can serve as center), but star-shaped sets need not be convex. Star-shaped sets are important because every closed rectifiable curve in an $a$-star shaped region is homotopic to the constant curve at $a$, which provides a clean path to proving Cauchy's Theorem in such regions.
