Let $G$ be a region such that $G = G^*$ (symmetric with respect to the real axis), and let
$$G_+ = \{z \in G : \text{Im } z > 0\}, \quad G_0 = \{z \in G : \text{Im } z = 0\}.$$
If $f: G_+ \cup G_0 \to \mathbb{C}$ is a continuous function which is analytic on $G_+$, and if $f(x) \in \mathbb{R}$ for all $x \in G_0$, then there exists an analytic function $g: G \to \mathbb{C}$ such that $g(z) = f(z)$ for all $z \in G_+ \cup G_0$. The extension is unique.
