The Poisson kernel satisfies the following:
\begin{itemize}
\item[(a)] $\displaystyle \frac{1}{2\pi} \int_{-\pi}^{\pi} P_r(\theta) \, d\theta = 1$;
\item[(b)] $P_r(\theta) > 0$ for all $\theta$, $P_r(-\theta) = P_r(\theta)$, and $P_r$ is periodic in $\theta$ with period $2\pi$;
\item[(c)] $P_r(\theta) < P_r(\delta)$ if $0 < \delta < |\theta| \leq \pi$;
\item[(d)] for each $\delta > 0$, $\displaystyle \lim_{r \to 1^-} P_r(\theta) = 0$ uniformly in $\theta$ for $\pi \geq |\theta| \geq \delta$.
\end{itemize}
