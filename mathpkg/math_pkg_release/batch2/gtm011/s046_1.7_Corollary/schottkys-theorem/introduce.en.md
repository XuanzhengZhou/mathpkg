Schottky's Theorem provides a quantitative bound for analytic functions that omit the values $0$ and $1$. Given such a function $f$ on a disk, if $|f(0)|$ is bounded by $\alpha$, then $|f(z)|$ is uniformly bounded on any smaller concentric disk $|z| \leq \beta$ by a constant $C(\alpha, \beta)$ depending only on $\alpha$ and $\beta$, not on the particular function $f$. This theorem is a key ingredient in proving the Little Picard Theorem.
