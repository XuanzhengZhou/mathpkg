Let $C$ be a component of $G$ and let $x_0 \in C$. Since $G$ is open there is an $\epsilon > 0$ with $B(x_0; \epsilon) \subset G$. By Lemma 2.6, $B(x_0; \epsilon) \cup C$ is connected and so it must be $C$. That is $B(x_0; \epsilon) \subset C$ and $C$ is, therefore, open.

To see that the number of components is countable let $S = \{a + ib : a \text{ and } b \text{ are rational and } a + bi \in G\}$. Then $S$ is countable and each component of $G$ contains a point of $S$, so that the number of components is countable.
