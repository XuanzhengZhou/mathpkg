(a) If $S = \{z : a \leq \text{Re } z \leq A\}$ where $1 < a < A < \infty$ then the integral

$$\int_0^\infty (e^t - 1)^{-1} t^{z-1} dt$$

converges uniformly on $S$.

(b) If $S = \{z : \text{Re } z \leq A\}$ where $-\infty < A < \infty$ then the integral

$$\int_1^\infty (e^t - 1)^{-1} t^{z-1} dt$$

converges uniformly on $S$.
