This definition extends the winding number to curves that are not necessarily rectifiable. The curve $\gamma$ is radially projected onto the unit circle $\Gamma = \{z : |z| = 1\}$ via $\gamma_1(t) = \gamma(t)/|\gamma(t)|$. Since $\exp(2\pi i \cdot): \mathbb{R} \to \Gamma$ is a covering map, the path lifting property yields a unique lift $\tilde{\gamma}$ of $\gamma_1$ with $\tilde{\gamma}(0) = 0$. The winding number is then the endpoint $\tilde{\gamma}(1)$, which measures the net number of counterclockwise revolutions of $\gamma$ about the origin. This definition agrees with the integral formula for rectifiable curves.
