For a given power series $\sum_{n=0}^{\infty} a_n(z-a)^n$ define the number $R$, $0 \leq R \leq \infty$, by

$$\frac{1}{R} = \lim \sup |a_n|^{1/n},$$

then:

(a) if $|z-a| < R$, the series converges absolutely;

(b) if $|z-a| > R$, the terms of the series become unbounded and so the series diverges;

(c) if $0 < r < R$ then the series converges uniformly on $\{z: |z-a| \leq r\}$. Moreover, the number $R$ is the only number having properties (a) and (b).
