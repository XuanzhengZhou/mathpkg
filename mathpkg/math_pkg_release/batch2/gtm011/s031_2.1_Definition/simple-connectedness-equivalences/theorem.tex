Let \(G\) be an open connected subset of \(\mathbb{C}\). Then the following are equivalent:

\begin{enumerate}
\item[(a)] \(G\) is simply connected.
\item[(b)] \(n(\gamma; a) = 0\) for every closed rectifiable curve \(\gamma\) in \(G\) and every point \(a\) in \(\mathbb{C} \setminus G\).
\item[(c)] \(\mathbb{C}_{\infty} \setminus G\) is connected.
\item[(d)] For every analytic function \(f\) on \(G\) there is a sequence of polynomials that converges to \(f\) uniformly on compact subsets of \(G\).
\item[(e)] For every closed rectifiable curve \(\gamma\) in \(G\) and every analytic function \(f\) on \(G\), \(f\) can be uniformly approximated on \(\{\gamma\}\) by polynomials.
\item[(f)] Every closed rectifiable curve in \(G\) is homologous to zero.
\item[(g)] Every analytic function \(f: G \to \mathbb{C}\) that never vanishes has an analytic logarithm.
\item[(h)] Every analytic function \(f: G \to \mathbb{C}\) that never vanishes has an analytic square root.
\item[(i)] Every harmonic function on \(G\) has a harmonic conjugate.
\item[(j)] \(G\) is homeomorphic to the open unit disk \(D = \{z : |z| < 1\}\).
\end{enumerate}
