Again let \(h\) be the meromorphic function from part (a). Prove that for \(z \neq a_1, a_2, \ldots\) and \(\gamma\) any rectifiable curve in \(\mathbb{C} \setminus \{a_1, a_2, \ldots\}\),

\[
f(z) = \exp\left( \int_{\gamma} h \right)
\]

defines an analytic function on \(\mathbb{C} \setminus \{a_1, a_2, \ldots\}\) with \(f'/f = h\). (That is, the value of \(f(z)\) is independent of the curve \(\gamma\) and the resulting function \(f\) is analytic.)
