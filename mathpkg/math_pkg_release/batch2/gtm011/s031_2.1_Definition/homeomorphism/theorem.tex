Let \(X\) and \(\Omega\) be metric spaces. A **homeomorphism** between \(X\) and \(\Omega\) is a continuous map \(f: X \to \Omega\) which is one-one, onto, and such that \(f^{-1}: \Omega \to X\) is also continuous.

If \(f: X \to \Omega\) is one-one, onto, and continuous then \(f\) is a homeomorphism if and only if \(f\) is open (or, equivalently, \(f\) is closed).

If there is a homeomorphism between \(X\) and \(\Omega\) then the metric spaces \(X\) and \(\Omega\) are said to be **homeomorphic**.
