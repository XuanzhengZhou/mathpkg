This proposition gives an explicit contour integral formula for the inverse of an injective analytic function on a disk. For each value $\omega$ in the image $\Omega = f[B(a;R)]$, the preimage $f^{-1}(\omega)$ is expressed as a contour integral involving $f'$ and $f - \omega$ over the boundary circle. The formula is derived by applying the Generalized Argument Principle (Theorem 3.6) with $g(z) = z$ to the function $f(z) - \omega$, which has exactly one simple zero inside the disk (by injectivity). This result provides a constructive way to compute inverse function values using contour integration.
