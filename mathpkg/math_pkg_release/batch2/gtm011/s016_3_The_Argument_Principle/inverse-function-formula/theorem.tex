Let $f$ be analytic on an open set containing $\overline{B}(a; R)$ and suppose that $f$ is one-one on $B(a; R)$. If $\Omega = f[B(a; R)]$ and $\gamma$ is the circle $|z - a| = R$, then $f^{-1}(\omega)$ is defined for each $\omega$ in $\Omega$ by the formula
$$
f^{-1}(\omega) = \frac{1}{2\pi i} \int_{\gamma} \frac{z f'(z)}{f(z) - \omega}\, dz.
$$
