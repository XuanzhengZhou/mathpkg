A function $f$ is \text{meromorphic} on an open set $G$ if there exists a set $A \subseteq G$ such that:
\begin{enumerate}
\item $A$ has no limit point in $G$;
\item $f$ is analytic on $G \setminus A$;
\item $f$ has a pole at each point of $A$.
\end{enumerate}
The points of $A$ are called the \text{poles} of $f$. The class of all meromorphic functions on $G$ is denoted by $M(G)$. Note that $H(G) \subseteq M(G)$, i.e., every analytic function is meromorphic (with $A = \emptyset$).
