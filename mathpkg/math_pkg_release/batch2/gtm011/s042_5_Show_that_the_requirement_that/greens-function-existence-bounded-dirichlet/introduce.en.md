This theorem guarantees the existence of a Green's Function on any bounded Dirichlet Region. The construction is elegant: solve the Dirichlet Problem with boundary values $f(z) = \log|z-a|$ to obtain a harmonic function $u$, then set $g_a(z) = u(z) - \log|z-a|$. The logarithmic singularity is thereby cancelled by $u$, and the boundary condition $u|_{\partial G} = \log|z-a|$ ensures $g_a$ vanishes on the boundary.
