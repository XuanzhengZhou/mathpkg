*Proof.* If $G \neq \mathbb{C}$ the result is clear since $\mathbb{C}_\infty - G$ has only one component (which contains $\infty$ and is non-degenerate), so by Corollary 4.17, $G$ is a Dirichlet Region. If $G = \mathbb{C}$ then the result is trivial since $\partial_\infty \mathbb{C} = \emptyset$. $\square$
