An entire function has finite order if its maximum modulus $M(r) = \max_{|z|=r} |f(z)|$ is bounded by $\exp(r^a)$ for some $a > 0$ and all large $r$. The order $\lambda$ is the infimum of all such exponents $a$. This definition is a priori weaker than the genus bound from Theorem 2.6, but Hadamard's Factorization Theorem will later show that finite order and finite genus are equivalent, with $\mu \leq \lambda$.
