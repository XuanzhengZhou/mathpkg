Let $f(z) = \sum c_n z^n$ be an entire function and define the number $\alpha$ by
$$\alpha = \liminf_{n \to \infty} \frac{-\log |c_n|}{n \log n}.$$

**(a)** Show that if $f$ has finite order then $\alpha > 0$. (Hint: If the order of $f$ is $\lambda$ and $\beta > \lambda$ show that $|c_n| \leq r^{-n} \exp(r^\beta)$ for sufficiently large $r$, and find the minimum value of this expression.)

**(b)** Suppose that $0 < \alpha < \infty$ and show that for any $\epsilon > 0$, $\epsilon < \alpha$, there is an integer $p$ such that $|c_n|^{1/n} < n^{-(\alpha-\epsilon)}$ for $n > p$.
