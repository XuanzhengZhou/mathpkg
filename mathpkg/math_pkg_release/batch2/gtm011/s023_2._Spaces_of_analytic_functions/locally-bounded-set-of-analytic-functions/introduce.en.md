A family $\mathcal{F}$ of analytic functions on an open set $G$ is locally bounded if around each point of $G$ there is a disk on which all functions in $\mathcal{F}$ are uniformly bounded by some constant. This property is central to the theory of normal families: as Montel's Theorem shows, local boundedness is equivalent to normality for families of analytic functions.
