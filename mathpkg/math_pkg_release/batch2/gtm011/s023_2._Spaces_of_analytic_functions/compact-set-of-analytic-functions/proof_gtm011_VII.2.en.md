Since $H(G)$ is a metric space, compactness is equivalent to sequential compactness. By Montel's Theorem (2.9), $\mathcal{F}$ is normal (every sequence has a convergent subsequence) if and only if $\mathcal{F}$ is locally bounded. Adding the condition that $\mathcal{F}$ is closed ensures that the limit of every convergent subsequence belongs to $\mathcal{F}$. Thus a closed and locally bounded set is sequentially compact, hence compact. Conversely, a compact set in any metric space is closed and (by Montel's Theorem, since compact implies sequentially compact implies normal) must be locally bounded.
