Let $\varphi$ be a subharmonic function on a region $G$ and let $\bar{B}(a; r) \subset G$. Define $\varphi'$ on $G$ by:
\begin{enumerate}
\item[(i)] $\varphi'(z) = \varphi(z)$ if $z \in G \setminus B(a; r)$;
\item[(ii)] $\varphi'$ is the continuous function on $\bar{B}(a; r)$ which is harmonic in $B(a; r)$ and agrees with $\varphi(z)$ for $|z - a| = r$.
\end{enumerate}
Then $\varphi'$ is subharmonic on $G$.
