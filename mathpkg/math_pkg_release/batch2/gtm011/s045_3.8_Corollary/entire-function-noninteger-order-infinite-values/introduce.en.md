This corollary connects the order of growth of an entire function to its value distribution. If an entire function $f$ has finite order $\lambda$ that is not an integer, then the Hadamard factorization forces $f$ to take every complex value infinitely many times. In particular, such a function cannot miss any value entirely (unlike $e^z$, which has order 1 — an integer — and misses 0). The proof is a one-line application of the preceding theorem: for any $\alpha \in f(\mathbb{C})$, consider $f - \alpha$, which has the same order $\lambda$, and apply the factorization theorem to conclude it has infinitely many zeros.
