Proof. Let $S: \mathbb{C}_\infty \rightarrow \mathbb{C}_\infty$ be defined by $Sz = (z, z_2, z_3, z_4)$; then $S^{-1}(\mathbb{R}) = \text{the set of } z \text{ such that } (z, z_2, z_3, z_4) \text{ is real. Hence, we will be finished if we can show that the image of } \mathbb{R}_\infty \text{ under a Möbius transformation is a circle.}$

Let $Sz = \frac{az + b}{cz + d}$; if $z = x \in \mathbb{R}$ and $\omega = S^{-1}(x) \neq \infty$ then $x = S\omega$ implies that $S(\omega) = S(\omega)$. That is,

$$\frac{a\omega + b}{c\omega + d} = \frac{a\bar{\omega} + \bar{b}}{c\bar{\omega} + d}$$

Cross multiplying this gives

3.11 $(a\bar{c} - \bar{a}c) |\omega|^2 + (a\bar{d} - \bar{b}c)\omega + (b\bar{c} - d\bar{a})\bar{\omega} + (b\bar{d} - \bar{b}d) = 0.$$

If $a\bar{c}$ is real then $a\bar{c} - \bar{a}c = 0$; putting $\alpha = 2(a\bar{d} - \bar{b}c), \beta = i(b\bar{d} - \bar{b}d)$ and multiplying (3.11) by $i$ gives

3.
