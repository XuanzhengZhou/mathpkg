Let $\sigma: [c, d] \to \mathbb{C}$ and $\gamma: [a, b] \to \mathbb{C}$ be rectifiable paths. The path $\sigma$ is equivalent to $\gamma$ if there is a function $\varphi: [c, d] \to [a, b]$ which is continuous, strictly increasing, and surjective with $\varphi(c) = a$, $\varphi(d) = b$ such that $\sigma = \gamma \circ \varphi$.
