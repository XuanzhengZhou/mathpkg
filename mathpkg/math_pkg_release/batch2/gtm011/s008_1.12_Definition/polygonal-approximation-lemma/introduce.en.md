Lemma 1.19 is a technical approximation result fundamental to the proof of the Fundamental Theorem of Calculus for line integrals (Theorem 1.18). It asserts that the line integral of a continuous function along any rectifiable path can be approximated arbitrarily well by the integral along a polygonal path with the same endpoints. The proof proceeds by considering two cases: when $G$ is a disk (Case I), and when $G$ is an arbitrary open set (Case II), using compactness of the trace to stay within $G$.
