Let $\gamma$ be a rectifiable curve and suppose that $f$ is a function continuous on $\{\gamma\}$. Then:

\begin{enumerate}
\item[(a)] $\displaystyle \int_{\gamma} f = -\int_{-\gamma} f$;
\item[(b)] $\displaystyle \left| \int_{\gamma} f \right| \leq \int_{\gamma} |f| \, |dz| \leq V(\gamma) \sup \left[ |f(z)| : z \in \{\gamma\} \right]$;
\item[(c)] If $c \in \mathbb{C}$ then $\displaystyle \int_{\gamma} f(z) \, dz = \int_{\gamma + c} f(z - c) \, dz$.
\end{enumerate}
