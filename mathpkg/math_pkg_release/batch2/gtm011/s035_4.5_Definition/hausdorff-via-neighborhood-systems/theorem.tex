If $\{\mathcal{N}_x : x \in X\}$ is a neighborhood system on $X$ and $\mathcal{T}$ is the induced topology, then $(X, \mathcal{T})$ is a Hausdorff space if and only if for any two distinct points $x$ and $y$ in $X$ there is a set $U$ in $\mathcal{N}_x$ and a set $V$ in $\mathcal{N}_y$ such that $U \cap V = \emptyset$.
