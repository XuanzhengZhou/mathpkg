If $\Gamma$ is a circle then an \textit{orientation} for $\Gamma$ is an ordered triple of points $(z_1, z_2, z_3)$ such that each $z_j$ is in $\Gamma$.

Given an orientation $(z_1, z_2, z_3)$ for $\Gamma$, the \textit{right side} of $\Gamma$ is defined to be

$$\{z : \operatorname{Im} (z, z_1, z_2, z_3) > 0\},$$

and the \textit{left side} of $\Gamma$ is defined to be

$$\{z : \operatorname{Im} (z, z_1, z_2, z_3) < 0\}.$$
