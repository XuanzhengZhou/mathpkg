The function $\rho$ defined on $C(G, \Omega)$ via the compact exhaustion $\{K_n\}$ satisfies all metric axioms. Symmetry $\rho(f,g) = \rho(g,f)$ is immediate from the definition. The triangle inequality follows because each $\rho_n$ is a pseudometric and the transformation $t \mapsto t/(1+t)$ is subadditive on $[0,\infty)$. Positive definiteness uses the fact that $G = \bigcup_n K_n$: if $\rho(f,g) = 0$, then $\rho_n(f,g) = 0$ for all $n$, so $f$ and $g$ agree on every $K_n$ and hence on all of $G$.
