$C(G, \Omega)$ denotes the set of all continuous functions from an open set $G \subseteq \mathbb{C}$ into a complete metric space $(\Omega, d)$. This space is never empty since it always contains the constant functions. When $\Omega$ is $\mathbb{C}$ or the extended complex plane $\mathbb{C}_{\infty}$, the space $C(G, \Omega)$ is richly populated: every analytic function on $G$ belongs to $C(G, \mathbb{C})$, and every meromorphic function on $G$ belongs to $C(G, \mathbb{C}_{\infty})$. The space $C(G, \Omega)$ serves as the ambient setting for studying compactness and convergence of analytic and meromorphic functions in Chapter VII.
