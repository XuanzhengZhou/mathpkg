A branch of the logarithm is a continuous function defined on a connected open subset of $\mathbb{C}\setminus\{0\}$ that serves as a right inverse to the exponential function. Because the exponential function is periodic with period $2\pi i$, there are infinitely many solutions to $e^z = w$, making the logarithm inherently multi-valued. Selecting one continuous choice yields a branch. If $f$ is one branch on a connected set $G$, then $f(z) + 2\pi k i$ is also a branch for any integer $k$, and conversely any two branches on $G$ differ by an integer multiple of $2\pi i$.
