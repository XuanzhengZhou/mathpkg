This result characterizes all complex numbers $z$ satisfying $e^z = w$ for a given non-zero $w$. Writing $z = x + iy$ and using $|w| = e^x$ together with $y = \arg w + 2\pi k$, one obtains an infinite discrete set of solutions parameterized by an integer $k$. This multi-valued nature of the inverse of the exponential function is the fundamental obstacle to defining a single-valued complex logarithm, motivating the introduction of branches of the logarithm.
