Let $f$ be analytic in the region $G$ except for the isolated singularities $a_1, a_2, \ldots, a_m$. If $\gamma$ is a closed rectifiable curve in $G$ such that $\gamma \approx 0 \pmod{G}$ and $\gamma$ does not pass through any $a_k$, then
$$\int_{\gamma} f = 2\pi i \sum_{k=1}^{m} n(\gamma; a_k) \operatorname{Res}(f; a_k).$$
