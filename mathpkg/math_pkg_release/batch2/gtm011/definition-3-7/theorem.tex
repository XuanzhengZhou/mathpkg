If $f$ is a meromorphic function on the region $G$ then define $\mu(f)$: $G \to \mathbb{R}$ by

$$\mu(f)(z) = \frac{2|f'(z)|}{1 + |f(z)|^2}$$

whenever $z$ is not a pole of $f$, and

$$\mu(f)(a) = \lim_{z \to a} \frac{2|f'(z)|}{1 + |f(z)|^2}$$

if $a$ is a pole of $f$.

It follows that $\mu(f) \in C(G, \mathbb{C})$.

The reason for introducing $\mu(f)$ is as follows: If $f: G \to \mathbb{C}_\infty$ is meromorphic then for $z$ close to $z'$ we have that $d(f(z), f(z'))$ is approximated by $\mu(f)(z) |z - z'|$. So if a bound can be obtained for $\mu(f)$ then $f$ is a Lipschitz function. If $f$ belongs to a family of functions and $\mu(f)$ is uniformly bounded for $f$ in this family, then the family is a uniformly Lipschitz set of functions.
