The Identity Theorem states that two analytic functions defined on a region (a connected open set) are identical if and only if the set of points where they agree has a limit point inside the region. This is an immediate consequence of applying Theorem 3.7 to the difference $f - g$. The theorem provides a powerful uniqueness principle in complex analysis: the values of an analytic function on a set with an accumulation point completely determine the function on the whole region.
