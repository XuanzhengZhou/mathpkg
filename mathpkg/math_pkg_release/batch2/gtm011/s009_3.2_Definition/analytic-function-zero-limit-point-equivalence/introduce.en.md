This theorem gives three equivalent conditions for an analytic function on a connected open set (a region) to be identically zero: the function vanishes everywhere; all derivatives vanish at a single point; or its zero set has a limit point inside the domain. This is a powerful uniqueness result: if two analytic functions agree on a set with a limit point in the region, they are identical. The proof uses power series expansions and the connectedness of the domain to propagate the vanishing property.
