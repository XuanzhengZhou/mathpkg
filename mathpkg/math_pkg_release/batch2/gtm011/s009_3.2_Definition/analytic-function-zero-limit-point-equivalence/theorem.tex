Let $G$ be a connected open set and let $f: G \to \mathbb{C}$ be an analytic function. Then the following are equivalent statements:

\begin{enumerate}
\item[(a)] $f \equiv 0$;
\item[(b)] there is a point $a$ in $G$ such that $f^{(n)}(a) = 0$ for each $n \geq 0$;
\item[(c)] $\{ z \in G : f(z) = 0 \}$ has a limit point in $G$.
\end{enumerate}
