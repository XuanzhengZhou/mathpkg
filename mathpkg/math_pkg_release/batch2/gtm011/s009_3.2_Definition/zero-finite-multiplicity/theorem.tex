If $f$ is analytic on an open connected set $G$ and $f$ is not identically zero, then for each $a$ in $G$ with $f(a) = 0$ there is an integer $n \geq 1$ and an analytic function $g: G \to \mathbb{C}$ such that $g(a) \neq 0$ and

$$f(z) = (z-a)^n g(z)$$

for all $z$ in $G$. That is, each zero of $f$ has finite multiplicity.
