The collection $\{D_0, D_1, \ldots, D_n\}$ of open disks is called a chain of disks if $D_{j-1} \cap D_j \neq \emptyset$ for $1 \leq j \leq n$. If $\{(f_j, D_j): 0 \leq j \leq n\}$ is a collection of function elements such that $\{D_0, D_1, \ldots, D_n\}$ is a chain of disks and $f_j(z) = f_{j-1}(z)$ for $z$ in $D_{j-1} \cap D_j$, $1 \leq j \leq n$, then $\{(f_j, D_j): 0 \leq j \leq n\}$ is called an analytic continuation along a chain of disks. If $\{(f_j, D_j): 0 \leq j \leq n\}$ is an analytic continuation along a chain of disks and if $D_0 = D_n$, need it follow that $f_0 = f_n$?
