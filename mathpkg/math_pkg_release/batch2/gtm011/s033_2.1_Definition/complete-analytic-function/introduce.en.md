A complete analytic function is the global object obtained by taking all germs that can be reached from a given function element via analytic continuation along arbitrary paths. It encapsulates the maximal analytic extension of a local analytic function. To formally view it as a function, one constructs the Riemann domain $\mathcal{R}$ consisting of pairs $(z, [f]_z)$ and defines the evaluation map $\mathcal{F}(z, [f]_z) = f(z)$. This construction provides the foundation for the sheaf-theoretic and Riemann surface viewpoints of analytic continuation, which are developed further when a topology is introduced on $\mathcal{R}$.
