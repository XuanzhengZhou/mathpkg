A function element is a pair $(f, G)$ consisting of a region $G$ in the complex plane and an analytic function $f$ defined on $G$. This concept packages an analytic function together with its domain of definition, which is essential for discussing analytic continuation. When two function elements agree on the intersection of their domains, they represent the same local analytic behavior. Function elements serve as the building blocks for the theory of germs and analytic continuation developed in this chapter.
