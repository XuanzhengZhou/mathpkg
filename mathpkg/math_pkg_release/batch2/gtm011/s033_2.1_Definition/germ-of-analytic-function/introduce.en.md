The germ $[f]_a$ of an analytic function $f$ at a point $a$ is the equivalence class of all function elements $(g, D)$ that agree with $f$ on some neighborhood of $a$. This concept captures the local analytic behavior of a function at a specific point, independent of the particular domain on which a representative is defined. Germs are the central objects in the theory of analytic continuation, as they allow one to track how local analytic data propagates along paths. Note that a germ is a collection of function elements, not a function element itself, and comparing germs at different points is not meaningful.
