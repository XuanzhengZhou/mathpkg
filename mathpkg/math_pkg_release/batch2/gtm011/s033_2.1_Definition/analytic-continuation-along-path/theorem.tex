Let $\gamma: [0, 1] \to \mathbb{C}$ be a path and suppose that for each $t$ in $[0, 1]$ there is a function element $(f_t, D_t)$ such that:
\begin{enumerate}
\item[(a)] $\gamma(t) \in D_t$;
\item[(b)] for each $t$ in $[0, 1]$ there is a $\delta > 0$ such that if $|s - t| < \delta$ then $\gamma(s) \in D_t$ and $[f_s]_{\gamma(s)} = [f_t]_{\gamma(s)}$.
\end{enumerate}
Then $(f_t, D_t)$ is called the analytic continuation of $(f_0, D_0)$ along the path $\gamma$; the collection $\{(f_t, D_t): 0 \leq t \leq 1\}$ is called an analytic continuation along $\gamma$.
