Using the Puppe sequence of Chap. 10, one proves that there is a sequence of cofunctors $\tilde{K}^{-p}$ and $\tilde{KO}^{-p}$ satisfying all the axioms for a cohomology theory on the category of finite CW-complexes except the dimension axiom (see Eilenberg and Steenrod [1]). These cohomology groups are defined for negative integers.

Using the periodicity theorems $\tilde{K}^{-p}(X) \cong \tilde{K}^{-p-2}(X)$ and $\tilde{KO}^{-p}(X)$ and $\tilde{KO}^{-p}(X) \cong \tilde{KO}^{-p-8}(X)$, we can systematically define $\tilde{K}^p$ and $\tilde{KO}^p$ for all integers $p$.
