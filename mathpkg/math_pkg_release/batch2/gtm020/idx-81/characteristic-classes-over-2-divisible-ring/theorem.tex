Let $R$ be a $2$-divisible ring. Each characteristic class $\phi \colon \mathrm{Vect}_R^n \to H^*(-, R)$ is of the form
$$\phi(X)\xi = q(p_1(\xi), \ldots, p_r(\xi)),$$
where $q(y_1, \ldots, y_r)$ is a polynomial uniquely determined by $\phi$, $p_i(\xi)$ is the $i$th Pontrjagin class, and $n$ equals $2r$ or $2r + 1$. If $\mathrm{Vect}_+^n$ denotes the cofunctor of oriented $n$-dimensional real vector bundles, each characteristic class $\phi \colon \mathrm{Vect}_+^n \to H^*(-, R)$ is of the form
$$\phi(X)\xi = q(p_1(\xi), \ldots, p_{r-1}(\xi), e(\xi))$$
for $n = 2r$, or
$$\phi(X)\xi = q(p_1(\xi), \ldots, p_r(\xi))$$
for $n = 2r + 1$, where $q$ is a polynomial uniquely determined by $\phi$.
