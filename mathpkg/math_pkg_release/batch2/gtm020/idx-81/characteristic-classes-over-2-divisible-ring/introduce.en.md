This theorem classifies all characteristic classes of real vector bundles with values in a $2$-divisible ring. In the unoriented case, every characteristic class is a polynomial in the Pontrjagin classes, with the polynomial uniquely determined by the class. For oriented bundles of even dimension $n = 2r$, the Euler class $e(\xi)$ enters as an additional generator alongside the first $r-1$ Pontrjagin classes, while for odd dimension the Euler class does not appear.
