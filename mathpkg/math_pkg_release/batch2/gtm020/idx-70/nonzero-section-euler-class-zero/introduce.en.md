This theorem provides a geometric interpretation of the Euler class: if a vector bundle admits a continuous choice of nonzero vector in each fibre (an everywhere-nonzero cross section), then the Euler class must be zero. The proof follows from the fact that such a bundle splits off a trivial line bundle, together with the multiplicativity of the Euler class and the vanishing of the Euler class for trivial bundles.
