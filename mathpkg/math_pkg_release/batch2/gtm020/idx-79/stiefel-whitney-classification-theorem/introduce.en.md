The classification theorem for $Z_2$-valued characteristic classes of real vector bundles states that every such characteristic class is uniquely expressible as a polynomial in the Stiefel-Whitney classes. This result is the real counterpart to the complex classification theorem using Chern classes and formal power series. It follows from the computation of $H^*(BO(n), Z_2)$ as the polynomial ring $Z_2[w_1, \ldots, w_n]$, showing that Stiefel-Whitney classes form a complete set of universal characteristic classes for real vector bundles with $Z_2$ coefficients.
