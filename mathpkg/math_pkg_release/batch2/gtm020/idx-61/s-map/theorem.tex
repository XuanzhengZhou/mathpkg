\text{Let } X, Y \text{ be topological spaces. The direct limit of the sequence of abelian groups}

$$[S^2(X), S^2(Y)] \to [S^3(X), S^3(Y)] \to \cdots \to [S^n(X), S^n(Y)] \to \cdots$$

\text{is denoted by } \{X, Y\}. \text{ An element of } \{X, Y\} \text{ is called an } S\text{-map from } X \text{ to } Y.

\text{The suspension functor } S: \{X, Y\} \to \{S(X), S(Y)\} \text{ is an isomorphism.}
