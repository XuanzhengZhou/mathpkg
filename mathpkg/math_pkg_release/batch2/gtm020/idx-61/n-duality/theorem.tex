\text{An } n\text{-pairing is a map } u: X \wedge X' \to S^n. \text{ Such a map defines two group morphisms}

$$u_Z: \{Z, X'\} \to \{X \wedge Z, S^n\}, \quad u_Z(\{f\}) = \{u(1 \wedge f)\}$$

\text{and}

$$u_Z': \{Z, X\} \to \{Z \wedge X', S^n\}, \quad u_Z'(\{g\}) = \{u(g \wedge 1)\}.$$

\text{For } Z = S^k \text{ we write } u_k \text{ and } u^k \text{ respectively.}

\text{An } n\text{-pairing } u: X \wedge X' \to S^n \text{ is called an } n\text{-duality provided } u_k: \{S^k, X'\} \to \{X \wedge S^k, S^n\} \text{ is an isomorphism for all integers } k.
