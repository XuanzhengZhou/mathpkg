\text{Let } u: X \wedge X' \to S^n \text{ be an } n\text{-duality between finite } CW\text{-complexes. Then for each finite } CW\text{-complex } Z \text{ the following group morphisms are isomorphisms:}

$$u_Z: \{Z, X'\} \to \{X \wedge Z, S^n\}$$

\text{and}

$$u_Z': \{Z, X\} \to \{Z \wedge X', S^n\}.$$

\text{The proof uses induction on the number of cells of } Z \text{ and the Puppe sequence for the last cell attached to } Z.
