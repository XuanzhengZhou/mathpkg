There are isomorphisms
$$
\mathbf{C} \otimes C_k \cong \mathbf{C} \otimes C'_k
$$
for $k = 1, 2$ in Proposition (5.4). For the third, we have an isomorphism
$$
w: \mathbf{H} \otimes \mathbf{H} \to \operatorname{Hom}_{\mathbf{R}}(\mathbf{H}, \mathbf{H}) = \mathbf{R}(4)
$$
defined by $w(x_1 \otimes x_2)x = x_1 x \bar{x}_2$. This is an algebra morphism and an epimorphism since $\dim \mathbf{H} \otimes \mathbf{H} = 16 = \dim \mathbf{R}(4)$.
