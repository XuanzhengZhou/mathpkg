This proposition provides a practical criterion for determining when two fibre bundles (or vector bundles) over the same base $B$ are isomorphic. Given atlases for both bundles over a common open covering $\{V_i\}$, the bundles are isomorphic if and only if their transition functions are equivalent, meaning there exist maps $r_i: V_i \rightarrow G$ conjugating one set of transition functions to the other: $g'_{i,j} = r_i^{-1} g_{i,j} r_j$. The proof constructs the isomorphism explicitly by modifying the local chart isomorphisms using the maps $r_i$, and the converse follows from the uniqueness of transition functions associated to an atlas.
