This theorem completes Dold's quartet of local-to-global results. It states that a bundle morphism which is a fibre homotopy equivalence over each set in a numerable covering is globally a fibre homotopy equivalence. This result is used in Chapter 16 of the book in a mild form. The proof, like those of Theorems A.6 and A.8, is reduced to an application of the section extension property (Theorem A.4) via an appropriate mapping space construction.
