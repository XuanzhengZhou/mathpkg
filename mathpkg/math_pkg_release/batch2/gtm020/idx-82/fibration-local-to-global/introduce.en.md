This theorem states that the property of being a fibration satisfies Dold's local-to-global principle. If a bundle restricts to a fibration over each set in a numerable covering of the base space, then it is a fibration over the entire base space. The proof reduces this statement to an application of the section extension property (Theorem A.4) via a mapping space construction.
