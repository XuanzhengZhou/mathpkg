If there is an immersion of $\mathbf{RP}^n$ into $\mathbf{R}^{n+1}$, then $n = 2^r - 1$ or $n = 2^r - 2$ for some $r$. If there is an embedding of $\mathbf{RP}^n$ into $\mathbf{R}^{n+1}$, then $n = 2^r - 1$ for some $r$. For $n = 2^r$, there is no immersion of $\mathbf{RP}^n$ into $\mathbf{R}^{2n-2}$ and no embedding into $\mathbf{R}^{2n-1}$.
