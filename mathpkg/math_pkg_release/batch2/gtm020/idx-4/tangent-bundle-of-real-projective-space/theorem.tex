The tangent bundle $\tau(\mathbb{R}P^n)$ can be viewed as the quotient of $\tau(S^n)$. A point of $\mathbb{R}P^n$ is a two-element set $\pm b = \{b, -b\}$, where $b \in S^n$, and a point of $E(\tau(\mathbb{R}P^n))$ is a two-element set $\pm (b, x) = \{(b, x), (-b, -x)\}$, where $(b, x) \in \tau(S^n)$. The projection is $p(\pm (b, x)) = \pm b$.
