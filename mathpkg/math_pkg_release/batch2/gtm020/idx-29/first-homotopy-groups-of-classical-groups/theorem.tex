The fundamental groups of the classical groups are:
$$\pi_1(Sp(1)) = \pi_1(Sp(n)) = 0 \quad \text{for } 1 \leq n \leq +\infty,$$
$$\pi_1(SO(3)) = \pi_1(SO(n)) = \pi_1(O(n)) = \mathbf{Z}_2 \quad \text{for } 3 \leq n \leq +\infty.$$
The key to these calculations is the homeomorphism $SO(3) \cong \mathbf{R}P^3$.
