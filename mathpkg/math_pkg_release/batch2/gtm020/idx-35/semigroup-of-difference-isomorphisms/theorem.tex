Let $S(X, A)$ (or, more precisely, $S_F(X, A)$) denote the semigroup of isomorphism classes of difference isomorphisms of $F$-vector bundles over $(X, A)$. The commutative semigroup structure on $S(X, A)$ is defined using the Whitney sum operation:
$$\alpha \oplus \beta: \xi_0 \oplus \eta_0 \rightarrow \xi_1 \oplus \eta_1$$
for $\alpha: \xi_0 \rightarrow \xi_1$ and $\beta: \eta_0 \rightarrow \eta_1$. If $f: (Y, B) \rightarrow (X, A)$ is a map, and if $\alpha: \xi_0 \rightarrow \xi_1$ is a difference isomorphism, then $f^*(\alpha): f^*(\xi_0) \rightarrow f^*(\xi_1)$ is a difference isomorphism over $(Y, B)$. This operation is compatible with isomorphisms and Whitney sums.
