(1) $\gamma^0(x) = 1$ and $\gamma^1(x) = x$ for each $x \in R$.

(2) For each $x, y \in R$, $\gamma^k(x + y) = \sum_{i+j=k} \gamma^i(x)\gamma^j(y)$.

(3) $\gamma^k(x) = \lambda^k(x) + \sum_{i < k} a_{i,k}\lambda^i(x)$ and $\lambda^k(x) = \gamma^k(x) + \sum_{i < k} b_{i,k}\gamma^i(x)$,

where $a_{i,k}$ and $b_{i,k}$ are integers.
