A characteristic class is stable if it factors through the group completion (K-theory) of the semigroup of vector bundles. This proposition gives a simple criterion: a characteristic class valued in a cofunctor $H$ is stable precisely when its values on all bundle classes are invertible elements in $H$. For finite CW-complexes, it suffices to check that the characteristic class takes the neutral value on trivial bundles.
