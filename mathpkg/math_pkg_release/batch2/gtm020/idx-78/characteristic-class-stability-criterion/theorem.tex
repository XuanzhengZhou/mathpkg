A characteristic class $\phi$ with values in a cofunctor $H$ is stable if and only if $\phi(\xi)$ is invertible for each bundle class $\xi$. If $\phi$ is defined over the category of finite CW-complexes and if $\phi(\theta^q)$ is the neutral element in $H(X)$ for each trivial bundle class $\theta^q$, then $\phi$ is stable.
