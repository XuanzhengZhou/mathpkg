Let $\mathbf{A}$ be a category, and let $K$ be an object in $\mathbf{A}$. For each morphism $f: Y \rightarrow X$, let $[f, K]$ denote the function that assigns to each morphism $u \in [X, K]$ the morphism $uf \in [Y, K]$. Let $[-, K]$ denote the function that assigns to each $X$ in $\mathbf{A}$ the set $[X, K]$ and to each morphism $f: Y \rightarrow X$ the function $[f, K]: [X, K] \rightarrow [Y, K]$. Then $[-, K]: \mathbf{A} \rightarrow \mathbf{ens}$ is a cofunctor.
