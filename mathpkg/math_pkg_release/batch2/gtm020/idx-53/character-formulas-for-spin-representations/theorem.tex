In $R\,\text{Spin}(2r+1)$,
$$\Delta = \prod_{1 \leq j \leq r} (\alpha_j^{1/2} + \alpha_j^{-1/2}) = \sum_{\varepsilon(j) = \pm 1} \alpha_1^{\varepsilon(1)/2} \cdots \alpha_r^{\varepsilon(r)/2}.$$

In $R\,\text{Spin}(2r)$,
$$\Delta^+ = \sum_{\varepsilon(1) \cdots \varepsilon(r) = 1} \alpha_1^{\varepsilon(1)/2} \cdots \alpha_r^{\varepsilon(r)/2},$$
$$\Delta^- = \sum_{\varepsilon(1) \cdots \varepsilon(r) = -1} \alpha_1^{\varepsilon(1)/2} \cdots \alpha_r^{\varepsilon(r)/2}.$$

In $RSO(2r)$,
$$\lambda_r^\pm = \sum_{i(1) \leq \cdots \leq i(r),\; \varepsilon(1) \cdots \varepsilon(r) = \pm 1} \alpha_{i(1)}^{\varepsilon(1)} \cdots \alpha_{i(r)}^{\varepsilon(r)}.$$
