Let $M^n$ be an $n$-dimensional manifold with maximal atlas $A$. The tangent bundle $\tau(M^n)$ is the $n$-dimensional vector bundle defined by transition functions
$$D(\psi \phi^{-1})(\phi(x)) \in \mathrm{GL}(n, \mathbb{R})$$
for $x \in U \cap V$ and $(U, \phi), (V, \psi) \in A$. The relation
$$D(\eta \phi^{-1})(\phi(x)) = D(\eta \psi^{-1})(\psi(x)) D(\psi \phi^{-1})(\phi(x))$$
holds by the chain rule, so the set of functions $D(\psi \phi^{-1})(\phi(x))$ is a family of transition functions for a vector bundle.
