The Chern polynomials $c_q(x) = c_q(x_{1,1}, \ldots, x_{n,n})$ for $0 \leq q \leq n$ are defined by the relation
$$
R_x(t) = \det(I + Xt) = \sum_{0 \leq q \leq n} c_q(x) t^q,
$$
where $X = (x_{i,j})$ is an $n \times n$ matrix. In particular, $c_0(x) = 1$, $c_1(x) = \text{Tr}(X)$, and $c_n(x) = \det(X)$. When $X$ is diagonal with eigenvalues $\lambda_1, \ldots, \lambda_n$, we have
$$
c_q(x) = \sum_{1 \leq i_1 < \cdots < i_q \leq n} \lambda_{i_1} \cdots \lambda_{i_q}.
$$
