The Chern polynomials $c_q(x)$ are fundamental $GL_n(k)$-invariant polynomials in the matrix entries, defined via the characteristic polynomial $\det(I + Xt)$. They form a set of algebraically independent generators of the ring of invariant polynomials $\text{Inv}(n)$. For a diagonalizable matrix, $c_q(x)$ is the $q$-th elementary symmetric function in the eigenvalues. These polynomials are the algebraic foundation for defining Chern characteristic classes of vector bundles via the Chern-Weil homomorphism.
