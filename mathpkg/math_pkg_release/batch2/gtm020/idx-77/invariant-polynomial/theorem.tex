Let $k$ be a field. A polynomial $P(x_{1,1}, \ldots, x_{n,n}) = P(X)$ in the $n^2$ matrix elements of an $n \times n$ matrix $X$ is called an \textbf{invariant polynomial} if for all $A \in \text{GL}_n(k)$,
$$
P(A X A^{-1}) = P(X).
$$
The set of all invariant polynomials is denoted
$$
\text{Inv}(n) = k[x_{1,1}, \ldots, x_{n,n}]^{\text{GL}_n(k)}.
$$
