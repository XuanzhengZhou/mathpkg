Let $\xi$ be an $n$-dimensional vector bundle. By Theorem (2.5), there exist unique classes $x_i(\xi) \in H^{ci}(B, K_c)$ for $i = 1, \ldots, n$ such that

$$a_{\xi}^n = -\sum_{1 \leq i \leq n} x_i(\xi) \, a_{\xi}^{n-i}.$$

Define the total class $x(\xi) = 1 + x_1(\xi) + \cdots + x_n(\xi)$.

For a real vector bundle $\xi$, the $i$th Stiefel-Whitney class of $\xi$, denoted $w_i(\xi)$, is $x_i(\xi) \in H^i(B(\xi), \mathbb{Z}_2)$.

For a complex vector bundle $\xi$, the $i$th Chern class of $\xi$, denoted $c_i(\xi)$, is $x_i(\xi) \in H^{2i}(B(\xi), \mathbb{Z})$.
