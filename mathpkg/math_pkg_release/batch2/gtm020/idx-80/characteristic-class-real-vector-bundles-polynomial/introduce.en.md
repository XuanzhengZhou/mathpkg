Every characteristic class for real $n$-plane bundles taking values in mod 2 cohomology can be expressed uniquely as a polynomial in the Stiefel-Whitney classes. That is, for each $\phi$ there exists a unique polynomial $q$ in $n$ variables over $\mathbb{Z}_2$ such that evaluating $q$ on $w_1(\xi), \dots, w_n(\xi)$ yields $\phi(\xi)$. This theorem establishes Stiefel-Whitney classes as the universal characteristic classes for real vector bundles in mod 2 cohomology.
