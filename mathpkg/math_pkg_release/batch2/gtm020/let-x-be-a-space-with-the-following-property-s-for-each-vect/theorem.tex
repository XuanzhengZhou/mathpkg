Then $\xi \oplus \theta^m \oplus \theta^q = \xi \oplus \theta^{m+q}$ and $\eta \oplus \theta^n \oplus \theta^q = \eta \oplus \theta^{n+q}$ are isomorphic and $\xi$ and $\eta$ are $s$-equivalent. Conversely, if $\xi \oplus \theta^n$ and $\eta \oplus \theta^m$ are isomorphic, we have $\alpha(\xi \oplus \theta^n) = \alpha(\eta \oplus \theta^m)$. Since $\alpha(\zeta \oplus \theta^q) = \alpha(\zeta)$, we have $\alpha(\zeta) = \alpha(\eta)$. This proves the theorem.

3.9 Example. The tangent bundle $\tau(S^n)$ to $S^n$ is trivial only for $n = 1, 3$, or 7, and $\tau(S^n)$ is $s$-trivial for all $n$ since $\tau(S^n) \oplus \theta^l \cong \theta^{n+1}$.
