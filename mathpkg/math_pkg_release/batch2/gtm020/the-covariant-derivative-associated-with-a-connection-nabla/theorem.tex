6.3 Remark. The covariant derivative satisfies the following relations as $a'$ function of $(\xi, s) \mapsto \nabla_\xi s$:

(1) For $\xi \in \Gamma(T(M))$ and $s \in \Gamma(E)$ the function $\nabla_\xi s$ is C-linear, and

(2) for a smooth function $f$ we have $\nabla_{f\xi} s = f \nabla_\xi s$ and

$$\nabla_\xi(fs) = f \nabla_\xi s + \xi(f)s.$$
