The Gram-Schmidt map $GS$ transforms an arbitrary linearly independent $k$-tuple of vectors in $F^n$ into an orthonormal $k$-frame (a point in the Stiefel variety $V_k(F^n)$) that spans the same $k$-dimensional subspace. It is defined by iteratively orthogonalizing and normalizing each vector with respect to the previously constructed orthonormal vectors. This map is a fundamental tool for constructing local trivializations of the Stiefel bundle over the Grassmannian.
