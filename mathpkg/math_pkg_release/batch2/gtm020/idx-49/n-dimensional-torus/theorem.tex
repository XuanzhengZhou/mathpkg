The $n$-dimensional torus, denoted $T^n$, is the quotient topological group $\mathbf{R}^n / \mathbf{Z}^n$. A torus is any topological group isomorphic to an $n$-dimensional torus for some $n$. For $n \neq m$, the tori $T^n$ and $T^m$ are nonisomorphic, so the dimension of a torus is well-defined. A topological group is a torus if and only if it is a compact, abelian, connected Lie group.
