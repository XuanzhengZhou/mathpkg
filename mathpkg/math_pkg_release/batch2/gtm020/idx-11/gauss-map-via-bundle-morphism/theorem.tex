If $(u, f): \xi^k \rightarrow \gamma_k^m$ is a vector bundle morphism that is an isomorphism when restricted to any fibre of $\xi^k$, then $q u: E(\xi^k) \rightarrow F^m$ is a Gauss map. Conversely, if $g: E(\xi^k) \rightarrow F^m$ is a Gauss map, there exists a vector bundle morphism $(u, f): \xi^k \rightarrow \gamma_k^m$ such that $q u = g$.
