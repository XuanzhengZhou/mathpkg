A path-connected space $X$ is called an Eilenberg-MacLane space $K(\Pi, n)$ provided

$$\pi_i(X) = \begin{cases} \Pi & \text{for } i = n \\ 0 & \text{for } i \neq n \end{cases}$$

In other words, there is one nonvanishing homotopy group and it is in degree $n$. This means that $\Pi$ must be abelian for $n > 1$, but $\Pi$ can be any group for $n = 1$, and for $n = 0$ we can choose $\Pi$ to be any set. Since a $K(\Pi, n)$ is path connected, we can use any base point for $n > 0$.
