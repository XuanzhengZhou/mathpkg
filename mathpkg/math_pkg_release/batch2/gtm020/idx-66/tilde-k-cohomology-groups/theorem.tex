$$
\tilde{K}^{-p}(X) = \tilde{K}(S^p X), \quad
\widetilde{KO}^{-p}(X) = \widetilde{KO}(S^p X) \qquad (p \geq 0)
$$

Using the Puppe sequence, the cofunctors $\tilde{K}^{-p}$ and $\widetilde{KO}^{-p}$ satisfy all axioms for a cohomology theory on the category of finite CW-complexes except the dimension axiom. These cohomology groups are defined for negative integers. Using the periodicity theorems

$$
\tilde{K}^{-p}(X) \cong \tilde{K}^{-p-2}(X), \qquad
\widetilde{KO}^{-p}(X) \cong \widetilde{KO}^{-p-8}(X),
$$

we can systematically define $\tilde{K}^p$ and $\widetilde{KO}^p$ for all integers $p$.
