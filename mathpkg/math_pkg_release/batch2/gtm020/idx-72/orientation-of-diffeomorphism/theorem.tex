Let $f: U \to V$ be a diffeomorphism of class $C^r$ for $r \geq 1$ of open connected subsets of $\mathbf{R}^n$. It is said that $f$ \textbf{preserves orientation}, denoted $O(f) = +1$, provided
$$\det(Df)(x) > 0 \quad \text{for all } x \in U$$
and that $f$ \textbf{reverses orientation}, denoted $O(f) = -1$, provided
$$\det(Df)(x) < 0 \quad \text{for all } x \in U.$$
