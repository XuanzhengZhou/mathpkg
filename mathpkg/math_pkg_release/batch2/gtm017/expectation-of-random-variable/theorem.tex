Let $X$ be a random variable with probability distribution $P(X = a_i) = p_i$, $i = 1, 2, \ldots$. The \textbf{expectation} $EX$ is defined for random variables with $\sum_i |a_i| p_i < \infty$ as
$$EX = \sum_{i=1}^{\infty} a_i p_i.$$

The \textbf{$n$-th moment} of $X$ is $EX^n = \sum_i a_i^n p_i$, whenever $\sum_i |a_i|^n p_i < \infty$.

The expectation operator $E$ is linear:
$$E(\alpha X + \beta Y) = \alpha EX + \beta EY.$$
