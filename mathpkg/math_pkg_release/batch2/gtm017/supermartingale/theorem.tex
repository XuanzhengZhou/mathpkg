A sequence $\{X_n\}$ is a \text{supermartingale} relative to $\{\mathfrak{B}_n\}$ if $E[|X_n|] < \infty$ and
$$X_m \geq E[X_n \mid \mathfrak{B}_m], \quad m < n,$$
almost surely. Equivalently, the process tends to decrease on average. If $\{X_n\}$ is a supermartingale, then $\{-X_n\}$ is a submartingale.
