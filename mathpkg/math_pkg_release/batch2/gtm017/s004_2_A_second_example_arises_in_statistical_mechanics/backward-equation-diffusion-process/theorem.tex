For a diffusion process, $F(\tau, \xi; t, x)$ satisfies the backward parabolic PDE:
$$\frac{\partial F}{\partial \tau} + \frac{1}{2} a(\tau, \xi) \frac{\partial^2 F}{\partial \xi^2} + b(\tau, \xi) \frac{\partial F}{\partial \xi} = 0.$$
If a smooth density $f$ exists, it satisfies the same equation in $(\tau, \xi)$.
