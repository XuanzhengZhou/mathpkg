For a weakly stationary process with continuous positive spectral density $f$ on $[-\pi, \pi]$ ($f(-\pi)=f(\pi)$), the one-step prediction error using the infinite past is $\sigma^2(f) = \lim_m \min_{c_j} E|X_n - \sum_{j=1}^m c_j X_{n-j}|^2 = 2\pi \exp\{\frac{1}{2\pi} \int_{-\pi}^\pi \log f(\lambda) d\lambda\}$.
