A weakly stationary solution to $\sum_{k=0}^m a_k X_{n-k} = \xi_n$ exists iff $a(e^{-i\lambda}) = \sum a_k e^{-ik\lambda}$ has no zeros on the unit circle. If a solution exists, it is unique with absolutely continuous spectral density $f_x(\lambda) = \frac{\sigma^2}{2\pi} / |a(e^{-i\lambda})|^2$ and representation $X_n = \int_{-\pi}^\pi e^{in\lambda} a(e^{-i\lambda})^{-1} dZ_\xi(\lambda)$.
