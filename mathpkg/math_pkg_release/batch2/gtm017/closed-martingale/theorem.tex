A martingale $X = (X_n)$ with respect to $\{\mathcal{B}_n\}$ is \textbf{closed} if there exists a random variable $Y$ with $E|Y| < \infty$ such that
$$X_n = E[Y \mid \mathcal{B}_n]$$
with probability one.

A martingale is closed if and only if it is uniformly integrable. In this case, $X_n \to Y$ in mean and almost surely.
