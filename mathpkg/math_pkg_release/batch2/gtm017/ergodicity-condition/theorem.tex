A necessary and sufficient condition for a stationary process to be ergodic is
$$\lim_{n \to \infty} \frac{1}{n} \sum_{j=0}^{n-1} P(B \cap T^{-j} A) = P(A) P(B)$$
for any two measurable sets $A, B$.

For a stationary Markov chain: the chain is ergodic if and only if it is irreducible.
