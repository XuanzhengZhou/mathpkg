The \textbf{joint distribution function} of random variables $X_1, \ldots, X_k$ is
$$F(x_1, \ldots, x_k) = P(\{w \mid X_1(w) \leq x_1, \ldots, X_k(w) \leq x_k\}).$$

Properties:
\begin{enumerate}
\item $0 \leq F(x_1, \ldots, x_k) \leq 1$
\item $\lim_{x_i \to -\infty} F = 0$ for each $i$; $\lim_{x_1,\ldots,x_k \to \infty} F = 1$
\item $F$ is nondecreasing in each coordinate and right-continuous
\item The $k$-th order difference $\Delta_{h_1}\cdots\Delta_{h_k} F \geq 0$
\end{enumerate}
