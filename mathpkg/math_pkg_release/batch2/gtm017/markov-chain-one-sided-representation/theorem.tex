Let $\{X_n\}$ be a finite-state stationary Markov chain with all transition probabilities positive. Then $X_n$ can be represented as a one-sided function of independent uniformly distributed (on $[0,1]$) random variables $\xi_n$:
$$X_n = f(\xi_n, \xi_{n-1}, \ldots).$$
The construction uses the conditional distribution function $F(y|y')$ of the continuous-state extension $Y_n = X_n - U_n$ and the inverse transform method.
