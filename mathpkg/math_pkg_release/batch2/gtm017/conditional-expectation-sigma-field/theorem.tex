Let $Y(w)$ be an integrable random variable on $(\Omega, \mathfrak{F}, P)$ and $\mathfrak{B} \subset \mathfrak{F}$ a sub-sigma-field. The \text{conditional expectation} $E[Y|\mathfrak{B}](w)$ is the $\mathfrak{B}$-measurable function satisfying
$$\int_B Y(w) \, dP = \int_B E[Y|\mathfrak{B}](w) \, dP \quad \text{for all } B \in \mathfrak{B}.$$

Equivalently, $E[Y|\mathfrak{B}](w) = \int Y(w') P(dw'|\mathfrak{B})(w)$. For the sigma-field $\mathfrak{F}_X$ generated by a random variable $X$, we write $E[Y|X] = E[Y|\mathfrak{F}_X]$.
