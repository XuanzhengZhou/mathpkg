Let $A_1, A_2, \ldots$ be a sequence of independent events on a probability space. Then
$$P(A_i \text{ i.o.}) = P\left(\bigcap_{n=1}^{\infty} \bigcup_{i=n}^{\infty} A_i\right)$$
can only take the values $0$ or $1$. Specifically:
\begin{itemize}
\item $P(A_i \text{ i.o.}) = 0$ if $\sum_{i=1}^{\infty} P(A_i) < \infty$.
\item $P(A_i \text{ i.o.}) = 1$ if $\sum_{i=1}^{\infty} P(A_i) = \infty$.
\end{itemize}
