\textbf{Theorem 2:} Let $A = (a_{i,j})$ be an irreducible matrix with non-negative elements and $B = (b_{i,j})$ a matrix with complex elements, $|b_{i,j}| \leq a_{i,j}$ for all $i, j = 1, \ldots, n$. Let $\lambda$ be the maximal eigenvalue of $A$ and $\beta$ an arbitrary eigenvalue of $B$. Then $|\beta| \leq \lambda$.

Equality $|\beta| = \lambda$ holds if and only if $B = e^{i\varphi} D A D^{-1}$ where $D$ is a diagonal matrix with diagonal entries of absolute value one.
