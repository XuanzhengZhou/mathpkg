Let $X_t$, $-\infty < t < \infty$, satisfy a uniform mixing condition. If for some fixed $L$ with $0 < L < \infty$ and every $t$, the process $Y_t(\omega)$ is measurable with respect to the $\sigma$-field generated by $\{X_u(\omega) : t - L \leq u \leq t\}$, then $Y_t$ also satisfies a uniform mixing condition.
