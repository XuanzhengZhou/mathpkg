A strictly stationary process is ergodic (metrically transitive) if and only if for any two measurable sets $A, B$,
$$\lim_{n \to \infty} \frac{1}{n} \sum_{j=0}^{n-1} P(T^{-j}A \cap B) = P(A) P(B).$$

Equivalently, the process is ergodic if and only if
$$\lim_{n \to \infty} \frac{1}{n} \sum_{j=0}^{n-1} I_A(T^j w) = P(A)$$
for every measurable set $A$, where $I_A$ is the indicator function of $A$.
