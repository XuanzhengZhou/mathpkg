An event $A$ is called \text{invariant} (under the shift $T$) if $A = T^{-1}A$ up to a set of $P$-measure zero. The collection of invariant events forms a sub-sigma-field $\mathfrak{G}$.

A function $f$ is \text{invariant} if $f(w) = f(Tw)$ almost everywhere. Equivalent: the set $\{w \mid f(w) \leq x\}$ is invariant for every real $x$.

The stationary process (or the measure $P$) is called \text{metrically transitive} if the invariant sigma-field $\mathfrak{G}$ is trivial, i.e., every invariant event has probability $0$ or $1$. Equivalently, every invariant function is constant almost everywhere.
