Given random variables $X_1, \ldots, X_k$, their joint characteristic function is
$$\varphi(t_1, \ldots, t_k) = E\left[\exp\left(i \sum_{j=1}^{k} t_j X_j\right)\right] = \int \cdots \int e^{i \sum t_j x_j} \, dF(x_1, \ldots, x_k).$$
If $X_1, \ldots, X_k$ are independent, then $\varphi(t_1, \ldots, t_k) = \prod_{j=1}^{k} \varphi_j(t_j)$.
