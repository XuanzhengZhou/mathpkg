A \textbf{random} or \textbf{stochastic process} $\{X_t(w), t \in T\}$ is an indexed family of random variables $X_t(w)$ on a probability space with the index $t$ ranging over some parameter set $T$.
\end{itemize}

Examples of parameter sets: $T = \{kh; k = 0, \pm 1, \ldots\}$ (discrete time), $T = \mathbb{R}$ (continuous time), $T \subset \mathbb{R}^k$ (multidimensional parameter).
\end{itemize}

The joint distribution functions of any finite number of random variables satisfy:
\begin{enumerate}
\item Permutation invariance: $F(X_{t_{Q1}}, \ldots, X_{t_{Qk}})(x_{Q1}, \ldots, x_{Qk}) = F(X_{t_1}, \ldots, X_{t_k})(x_1, \ldots, x_k)$
\item Consistency: $F(X_{t_1}, \ldots, X_{t_k})(x_1, \ldots, x_j, \infty, \ldots, \infty) = F(X_{t_1}, \ldots, X_{t_j})(x_1, \ldots, x_j)$
\end{enumerate}
