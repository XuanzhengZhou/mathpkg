\textbf{Borel-Cantelli Lemma (Zero-One Law):} Let $A_1, A_2, \ldots$ be a sequence of independent events. Then $P(A_i \text{ i.o.})$ (the probability that infinitely many $A_i$ occur) can only assume the values 0 and 1.

Specifically:
\begin{itemize}
\item If $\sum_{i=1}^{\infty} P(A_i) < \infty$, then $P(A_i \text{ i.o.}) = 0$.
\item If $\sum_{i=1}^{\infty} P(A_i) = \infty$, then $P(A_i \text{ i.o.}) = 1$.
\end{itemize}

Here $A_i \text{ i.o.} = \bigcap_{n=1}^{\infty} \bigcup_{i=n}^{\infty} A_i$.
