A random variable $Y$ has the \textbf{Poisson distribution} with parameter $\lambda > 0$ if
$$P(Y = r) = \frac{\lambda^r}{r!} e^{-\lambda}, \quad r = 0, 1, 2, \ldots.$$

The Poisson distribution is obtained from the binomial distribution by the limiting argument:
$$\lim_{m \to \infty} \binom{m}{r} p^r q^{m-r} = \frac{\lambda^r}{r!} e^{-\lambda},$$
where $mp = \lambda$ is held constant.
