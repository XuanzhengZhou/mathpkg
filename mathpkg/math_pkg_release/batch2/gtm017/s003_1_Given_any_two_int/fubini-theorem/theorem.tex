Let $m = m_1 \times m_2$ be the product measure on $\mathfrak{F}_1 \times \mathfrak{F}_2$. If $f(\omega_1, \omega_2)$ is integrable with respect to $m$, then
$$\int_{\Omega_1 \times \Omega_2} f(\omega_1, \omega_2) \, dm = \int_{\Omega_1} \left( \int_{\Omega_2} f(\omega_1, \omega_2) \, dm_2 \right) dm_1 = \int_{\Omega_2} \left( \int_{\Omega_1} f(\omega_1, \omega_2) \, dm_1 \right) dm_2.$$
If $f$ is nonnegative, the equality holds without the assumption of integrability (Tonelli's theorem). If one of the iterated integrals is infinite, all of them are.
