Let $X_1(\omega), \ldots, X_k(\omega)$ be $k$ random variables on the probability space $(\Omega, \mathfrak{F}, P)$. The function
$$F(x_1, \ldots, x_k) = P[\{\omega \mid X_1(\omega) \leq x_1, \ldots, X_k(\omega) \leq x_k\}]$$
is called the joint distribution function of $X_1, \ldots, X_k$.
Properties: (i) $0 \leq F(x_1, \ldots, x_k) \leq 1$; (ii) $\lim_{x_i \to -\infty} F(x_1, \ldots, x_k) = 0$; (iii) $\lim_{x_1, \ldots, x_k \to \infty} F(x_1, \ldots, x_k) = 1$. $F$ is nondecreasing in each coordinate. The $k$-th order difference $\Delta_{h_1} \cdots \Delta_{h_k} F(x_1, \ldots, x_k) = P[\{\omega \mid x_i < X_i(\omega) \leq x_i + h_i, i = 1, \ldots, k\}] \geq 0$. $F$ is continuous from the right in each variable separately.
