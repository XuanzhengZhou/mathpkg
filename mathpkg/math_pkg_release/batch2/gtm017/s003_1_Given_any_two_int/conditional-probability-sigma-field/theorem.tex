Let $(\Omega, \mathfrak{F}, P)$ be a probability space and $\mathfrak{B}$ a sub-sigma-field of $\mathfrak{F}$. For a fixed set $A \in \mathfrak{F}$, the conditional probability of $A$ given $\mathfrak{B}$ is a function $P(A|\mathfrak{B})(\omega)$, measurable with respect to $\mathfrak{B}$, satisfying
$$\int_B P(A|\mathfrak{B})(\omega) \, dP = P(A \cap B)$$
for every $B \in \mathfrak{B}$. This function is uniquely defined up to a set of $P$ measure zero. It satisfies: (i) $0 \leq P(A|\mathfrak{B})(\omega) \leq 1$; (ii) $P(\Omega|\mathfrak{B})(\omega) = 1$; (iii) for disjoint $A_i$, $P(\bigcup A_i|\mathfrak{B})(\omega) = \sum P(A_i|\mathfrak{B})(\omega)$, except on a $P$-null set.
