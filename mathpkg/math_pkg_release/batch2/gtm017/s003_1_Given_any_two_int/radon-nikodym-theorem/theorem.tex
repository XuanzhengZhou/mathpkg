If $m_1$ is absolutely continuous with respect to $m_2$ on a sigma-field $\mathfrak{F}$, then there exists a non-negative measurable function $f(\omega)$ (the Radon-Nikodym derivative, denoted $dm_1/dm_2$), uniquely defined up to a set of $m_2$ measure zero, such that for every $A \in \mathfrak{F}$,
$$m_1(A) = \int_A f(\omega) \, dm_2.$$
