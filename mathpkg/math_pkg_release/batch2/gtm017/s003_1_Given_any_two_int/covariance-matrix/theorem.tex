The covariance matrix of random variables $X_1, \ldots, X_k$ is the $k \times k$ matrix
$$\mathbf{R} = (r_{u,v}), \quad u, v = 1, \ldots, k,$$
where $r_{u,v} = E[(X_u - m_u)(X_v - m_v)]$. The covariance matrix is always symmetric and positive semidefinite. For any row vector $\mathbf{t}$,
$$\mathbf{t} \mathbf{R} \mathbf{t}' = E\left[\left(\sum_{j=1}^{k} t_j (X_j - m_j)\right)^2\right] \geq 0.$$
