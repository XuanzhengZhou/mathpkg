Given any two integrable functions $X(w)$, $Y(w)$ and any two real numbers $\alpha, \beta$, the linear combination $\alpha X(w) + \beta Y(w)$ is integrable and its integral is given by
$$\int [\alpha X(w) + \beta Y(w)] \, dP = \alpha \int X(w) \, dP + \beta \int Y(w) \, dP.$$
In probabilistic notation:
$$E[\alpha X + \beta Y] = \alpha E[X] + \beta E[Y].$$
