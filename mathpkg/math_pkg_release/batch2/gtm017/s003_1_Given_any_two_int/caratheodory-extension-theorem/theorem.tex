Let $m$ be a non-negative set function defined on a field $\mathbb{C}$. If $m$ satisfies:
(1) $m(A) \geq 0$ for $A \in \mathbb{C}$,
(2) For any countable collection of disjoint sets $A_1, A_2, \ldots \in \mathbb{C}$ with $\bigcup A_i \in \mathbb{C}$, $m(\bigcup A_i) = \sum m(A_i)$,
then $m$ can be extended to a measure on the sigma-field $\mathscr{F}(\mathbb{C})$ generated by $\mathbb{C}$. The extension is unique if $m$ is sigma-finite on $\mathbb{C}$. Moreover, for any $A \in \mathscr{F}(\mathbb{C})$ with $m(A)$ finite and any $\varepsilon > 0$, there exists $B \in \mathbb{C}$ such that $m(A \ominus B) < \varepsilon$, where $A \ominus B = (A - B) \cup (B - A)$.
