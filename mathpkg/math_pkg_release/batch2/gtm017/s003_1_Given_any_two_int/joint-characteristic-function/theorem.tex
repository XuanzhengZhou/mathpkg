Given random variables $X_1, \ldots, X_k$, their joint characteristic function is
$$\varphi(t_1, \ldots, t_k) = E \exp\left(i \sum_{j=1}^{k} t_j X_j\right) = \int \cdots \int \exp\left(i \sum t_j x_j\right) dF(x_1, \ldots, x_k).$$
If $X_1, \ldots, X_k$ are independent: $\varphi(t_1, \ldots, t_k) = \prod \varphi_j(t_j)$. The characteristic function of the sum $\sum X_j$ of independent random variables is $\prod \varphi_j(t)$.
