Let $\mathfrak{B}$ be a sub-sigma-field of $\mathfrak{F}$. For an integrable random variable $Y$, the conditional expectation $E(Y|\mathfrak{B})(\omega)$ is a $\mathfrak{B}$-measurable function satisfying
$$\int_B Y(\omega) \, dP = \int_B E(Y|\mathfrak{B})(\omega) \, dP$$
for every $B \in \mathfrak{B}$. It is uniquely defined up to a set of $P$ measure zero. Equivalently, $E(Y|\mathfrak{B})(\omega) = \int Y(\omega') P(d\omega'|\mathfrak{B})(\omega)$. For a random variable $X$, $E[Y|\mathfrak{F}_X]$ is the conditional expectation given the sigma-field generated by $X$.
