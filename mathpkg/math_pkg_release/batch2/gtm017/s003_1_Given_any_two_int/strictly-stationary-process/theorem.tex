Consider a random process $\{X_t(\omega), t \in T\}$ indexed by a parameter set $T$ on which an additive group operation is defined. The process is strictly stationary if, for any positive integer $m$, any $t_1, \ldots, t_m \in T$, and any shift $h \in T$, the random variables
$$(X_{t_1}, \ldots, X_{t_m}) \quad \text{and} \quad (X_{t_1+h}, \ldots, X_{t_m+h})$$
have the same joint probability distribution. Equivalently, the probabilistic structure is invariant under parameter translation. The finite-dimensional joint distributions depend on $t_1, \ldots, t_m$ only through the differences $t_2 - t_1, \ldots, t_m - t_1$.
