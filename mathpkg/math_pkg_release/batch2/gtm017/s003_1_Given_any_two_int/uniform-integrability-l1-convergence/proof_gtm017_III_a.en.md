Fix $\varepsilon > 0$. By uniform integrability, choose $a$ large so $\int_{B_{n,a}} |X_n| dP < \varepsilon/3$ for all $n$. On $\{|X_n| < a\}$, the sequence is bounded, so by bounded convergence $\int_{|X_n|<a} |X - X_n| dP \to 0$. Split the integral: $\int |X - X_n| dP = \int_{|X_n|<a} |X - X_n| dP + \int_{|X_n| \geq a} |X - X_n| dP$. The first term tends to zero as $n \to \infty$. The second term is bounded by $\int_{B_{n,a}} |X_n| dP + \int_{B_{n,a}} |X| dP$, both small by uniform integrability and Fatou's lemma.
