A stochastic process is a collection of random variables indexed by a parameter (usually time or space), all defined on a common probability space. The parameter set can be discrete (time series), continuous (diffusion processes, Brownian motion), or multidimensional (random fields). For each fixed $\omega$, $t \mapsto X_t(\omega)$ is a sample path; for each fixed $t$, $\omega \mapsto X_t(\omega)$ is a random variable.
