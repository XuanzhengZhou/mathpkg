A random or stochastic process $\{X_t(\omega), t \in T\}$ is an indexed family of random variables $X_t(\omega)$ on a probability space, with the index $t$ ranging over a parameter set $T$. Typical parameter sets include: $T = \mathbb{Z}$ or $\{kh : k = 0, \pm 1, \ldots\}$ (discrete time), $T = \mathbb{R}$ or $[0, \infty)$ (continuous time), $T = \mathbb{R}^k$ (multidimensional parameter, e.g., spatial processes).
