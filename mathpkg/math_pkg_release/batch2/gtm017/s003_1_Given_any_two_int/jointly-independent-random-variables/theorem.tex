The random variables $X_1(\omega), \ldots, X_k(\omega)$ are called jointly independent if for any real numbers $x_1, \ldots, x_k$,
$$P(X_1 \leq x_1, \ldots, X_k \leq x_k) = P(X_1 \leq x_1) \cdots P(X_k \leq x_k).$$
Equivalently, the joint distribution function factors as the product of the marginal distribution functions:
$$F(x_1, \ldots, x_k) = F_1(x_1) F_2(x_2) \cdots F_k(x_k).$$
