Lebesgue measure is the unique translation-invariant measure on the real line that assigns to each interval its length. Constructed via Caratheodory's extension theorem starting from intervals, it is the canonical example of a measure extending a natural set function on a field to the full Borel sigma-field. Lebesgue measure on the real line is sigma-finite but not finite, while restricted to $(0,1]$ it is a probability measure. It serves as the foundation for Lebesgue integration theory.
