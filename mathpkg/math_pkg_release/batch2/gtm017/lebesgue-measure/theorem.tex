\textbf{Lebesgue measure} on $(0,1]$ is obtained by taking $\Omega = \{w \mid 0 < w \leq 1\}$, $\mathbb{C}$ as the field generated by intervals $(\alpha, \beta]$, and $m((\alpha, \beta]) = \beta - \alpha$. The extension via Carathéodory's theorem yields Lebesgue measure on the completed sigma-field.

Lebesgue measure on the whole real line is a sigma-finite (but not finite) measure.
