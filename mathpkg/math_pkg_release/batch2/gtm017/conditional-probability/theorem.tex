The conditional probability of an event $A_1$ given that another event $A_2$ has occurred is defined by
$$P(A_1 \mid A_2) = \frac{P(A_1 \cap A_2)}{P(A_2)}$$
provided $P(A_2) > 0$. If $P(A_2) = 0$, $P(A_1 \mid A_2)$ can be taken as any number between zero and one.

Two events $A_1, A_2$ are said to be \text{independent} if
$$P(A_1 \mid A_2) = P(A_1),$$
or equivalently
$$P(A_1 \cap A_2) = P(A_1) P(A_2).$$

A denumerable collection of events $A_1, A_2, \ldots$ is a collection of \text{independent events} if every finite subcollection $A_{k_1}, \ldots, A_{k_m}$ satisfies
$$P(A_{k_1} A_{k_2} \cdots A_{k_m}) = \prod_{i=1}^{m} P(A_{k_i}).$$
