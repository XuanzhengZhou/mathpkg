Let $A = (a_{i,j})$ be an irreducible matrix with non-negative elements and $B = (b_{i,j})$ a matrix with complex elements satisfying $|b_{i,j}| \leq a_{i,j}$ for all $i, j$. Let $\lambda$ be the maximal eigenvalue of $A$ and $\beta$ an arbitrary eigenvalue of $B$. Then $|\beta| \leq \lambda$. Moreover, $|\beta| = \lambda$ if and only if $|b_{i,j}| = a_{i,j}$ for all $i, j$, in which case $B = e^{i\varphi} D A D^{-1}$ for some diagonal unitary matrix $D$.
