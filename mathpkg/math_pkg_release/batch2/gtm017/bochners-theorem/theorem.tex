A function $\varphi(t)$ is the characteristic function of a probability distribution if and only if:
\begin{enumerate}
\item $\varphi(0) = 1$,
\item $\varphi$ is continuous at $0$,
\item $\varphi$ is positive definite: for any finite set $t_1, \ldots, t_k$ and complex $c_1, \ldots, c_k$,
  $$\sum_{i,j=1}^{k} c_i \bar{c}_j \varphi(t_i - t_j) \geq 0.$$
\end{enumerate}
