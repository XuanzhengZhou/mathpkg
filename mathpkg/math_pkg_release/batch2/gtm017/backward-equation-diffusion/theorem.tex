For a diffusion process, the conditional distribution function $F$ satisfies the \textbf{backward equation} (in the backward variables $\tau, \xi$):
$$\frac{\partial F}{\partial \tau} + \frac{1}{2} a(\tau, \xi) \frac{\partial^2 F}{\partial \xi^2} + b(\tau, \xi) \frac{\partial F}{\partial \xi} = 0.$$

If $F$ has a smooth density $f$, then $f$ also satisfies the same equation.
