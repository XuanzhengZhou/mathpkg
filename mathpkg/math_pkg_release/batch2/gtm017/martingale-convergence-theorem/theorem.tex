If $\{X_n\}$ is a submartingale with $\sup_n E[|X_n|] < \infty$, then $X_n$ converges almost surely to an integrable random variable $X_\infty$. In particular, this holds for martingales and non-negative supermartingales.

As an example: if $X_n = E[f \mid \mathfrak{B}_n]$ where $\mathfrak{B}_n \uparrow \mathfrak{B}_\infty$, then $X_n \to E[f \mid \mathfrak{B}_\infty]$ almost surely and in $L^1$.
