Given a metric space $(X, d)$ and a continuous proper function $f: X \to \mathbf{R}$, one can construct a new metric $d'$ by adding the absolute difference of the $f$-values to the original distance. The resulting metric $d'$ is always complete and induces the same topology as $d$. This lemma provides a general technique for proving that certain manifolds are metrizable by a complete metric, by first constructing a proper function (such as an embedding into Euclidean space) and then applying this construction.
