Let $X$ be a smooth vector field on $\mathbf{R}^n$. Given any smooth function $f$ on $\mathbf{R}^n$, there exists a smooth function $F(t, x)$ near $0 \in \mathbf{R} \times \mathbf{R}^n$ satisfying:
(a) $F(0, x) = f(x)$,
(b) $\frac{\partial F}{\partial t} = XF$ to infinite order at $t = 0$.
The formal solution is $F(t, x) = e^{tX} f$, regularized via bump functions as in the Borel Theorem.
