Let $X$ be a smooth vector field on $\mathbf{R}^n$ and let $f$ be a smooth function defined on a neighborhood of $0$ in $\mathbf{R}^n$. Then there exists a smooth function $F(t, x)$ defined on a neighborhood of $0$ in $\mathbf{R} \times \mathbf{R}^n$ satisfying:
\begin{enumerate}
\item $F(0, x) = f(x)$ for all $x$ near $0$,
\item $\displaystyle \frac{\partial F}{\partial t}(t, x) = X F(t, x)$ for all $(t, x)$ near $0$.
\end{enumerate}
