Let $G(t, x)$ be a smooth function defined on a neighborhood of $0$ in $\mathbf{R} \times \mathbf{R}^n$, and let
$$P_k(z, \lambda) = z^k + \sum_{i=0}^{k-1} \lambda_i z^i$$
be the generic monic polynomial of degree $k$. Then there exists a smooth extension $\tilde{G}(z, x, \lambda)$ of $G$ to a neighborhood of $0$ in $\mathbf{C} \times \mathbf{R}^n \times \mathbf{C}^k$ such that:
\begin{enumerate}
\item $\tilde{G}(t, x, \lambda) = G(t, x)$ for all real $t$ (with arbitrary $\lambda$),
\item $\partial \tilde{G} / \partial \bar{z}$ vanishes to infinite order on the set $\{P_k(z, \lambda) = 0\}$,
\item $\partial \tilde{G} / \partial \bar{z}$ vanishes to infinite order for real $z$.
\end{enumerate}
