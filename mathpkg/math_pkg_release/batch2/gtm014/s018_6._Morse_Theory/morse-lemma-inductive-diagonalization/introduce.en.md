Lemma 6.12 is the core technical result used to prove the Morse Lemma. It provides an inductive diagonalization procedure: starting with a function whose Hessian at the critical point $0$ is already diagonal with $\pm 1$ entries, it constructs smooth coordinate functions $h_i$ that transform the function into a pure sum of squares $\pm h_1^2 \pm \cdots \pm h_n^2$. The proof proceeds by induction on $r$ (the number of variables already diagonalized), using an algebraic manipulation to separate one square at a time while maintaining smoothness.
