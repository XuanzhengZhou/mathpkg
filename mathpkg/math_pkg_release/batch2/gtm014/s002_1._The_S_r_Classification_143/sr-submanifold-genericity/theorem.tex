For a generic smooth mapping $f: X \to Y$, the set $S_r(f)$ is a submanifold of $X$, and
$$
\operatorname{codim} S_r(f) = \operatorname{codim} S_r = r^2 + er,
$$
where $e = |\dim X - \dim Y|$ and $S_r \subset J^1(X,Y)$ is the submanifold of 1-jets of corank $r$.

Moreover, the set of mappings for which $j^1 f \pitchfork S_r$ (for all $r$) is residual in $C^\infty(X,Y)$.
