Define an equivalence relation $\sim$ on $\mathbf{R}^{n+1} \setminus \{0\}$ by
\[
(x_0,\ldots,x_n) \sim (x'_0,\ldots,x'_n) \quad \text{iff} \quad \exists\, c \in \mathbf{R} \setminus \{0\} \text{ such that } x_i = c x'_i \text{ for all } i = 0,\ldots,n.
\]
The real projective $n$-space $\mathbf{P}^n$ is the quotient space
\[
\mathbf{P}^n = (\mathbf{R}^{n+1} \setminus \{0\}) / {\sim}.
\]
Equivalently, $\mathbf{P}^n$ can be identified with the Grassmannian $G_{1,n+1}$, the space of $1$-dimensional subspaces of $\mathbf{R}^{n+1}$.
