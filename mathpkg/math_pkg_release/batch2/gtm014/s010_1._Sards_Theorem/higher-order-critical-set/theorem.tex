Let $f: U \to \mathbf{R}^m$ be a smooth mapping with $U \subset \mathbf{R}^n$ open, and let $C = C[f]$ be the set of critical points of $f$. For $i = 1, 2, \ldots$, define
$$C_i = \left\{ p \in C \;\middle|\; \frac{\partial^{|\alpha|} f_l}{\partial x^\alpha}(p) = 0 \text{ whenever } 0 < |\alpha| \leq i \text{ and } 1 \leq l \leq m \right\}.$$
Thus $C_i$ consists of those critical points at which all partial derivatives of all component functions of $f$ up to order $i$ vanish.
