Let $T_a: \mathbb{R}^n \to \mathbb{R}^n$ be translation by $a$ in $\mathbb{R}^n$, i.e., $T_a(x) = x + a$. Given an open set $B$ in $\mathbb{R}^n$, there exists a diffeomorphism $\eta: \mathbb{R}^n \to \mathbb{R}^n$ such that $\eta = T_a$ on $B$ and $\eta = \mathrm{id}_{\mathbb{R}^n}$ outside of some compact set.
