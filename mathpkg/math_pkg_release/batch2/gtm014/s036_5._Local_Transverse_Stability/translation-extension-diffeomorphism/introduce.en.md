This proposition shows that any translation on $\mathbb{R}^n$ can be "damped" to the identity outside a compact set while remaining a diffeomorphism. The construction uses a smooth bump function $\rho$ that equals $1$ on a large ball containing $B$ and has compact support, and sets $\eta(x) = x + \rho(x)a$. By choosing the support small enough (or equivalently scaling the bump function), the derivative $d\eta = I + a \otimes d\rho$ remains everywhere invertible, so $\eta$ is an immersion. Lemma 5.2 then implies $\eta$ is a diffeomorphism.
