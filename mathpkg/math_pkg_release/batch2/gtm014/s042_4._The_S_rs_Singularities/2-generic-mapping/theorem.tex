A smooth map $f: X \to Y$ is called \emph{2-generic} if for all $r, s$, the second jet extension $j^2 f$ is transverse to the universal singularity set $S_{r,s}$:

$$j^2 f \overline{\cap} S_{r,s} \quad \text{for all } r, s.$$

The Transversality Theorem guarantees that the set of 2-generic mappings is residual in $C^\infty(X, Y)$.
