A fold point is the simplest type of stable singularity for mappings between manifolds of different dimensions. When $f: X \to Y$ is transverse to the corank-1 singularity set $S_1 \subset J^1(X, Y)$, the singular set $S_1(f)$ is a submanifold. A point $x \in S_1(f)$ is a fold if the tangent space to the singular set and the kernel of the derivative span the full tangent space — geometrically, the singular set meets the fiber of the derivative transversally.
