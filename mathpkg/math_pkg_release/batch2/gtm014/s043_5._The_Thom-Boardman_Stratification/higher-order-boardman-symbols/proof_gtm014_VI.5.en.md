Higher order Boardman symbols $S_{r_1, r_2, \ldots, r_k}$ are defined inductively. If $f: X \to Y$ is $(k-1)$-generic, then $S_{r_1, \ldots, r_{k-1}}(f)$ is a submanifold of $X$. The symbol $S_{r_1, \ldots, r_k}(f)$ is defined as the set of points in $S_{r_1, \ldots, r_{k-1}}(f)$ where the restricted map $f: S_{r_1, \ldots, r_{k-1}}(f) \to Y$ drops rank by $r_k$. Boardman proved that for a residual set of maps, this process can be continued indefinitely, and each $S_{r_1, \ldots, r_k}$ is a fiber subbundle of $J^k(X, Y)$. The process terminates when $r_k = 0$, at which point the restricted map becomes an immersion.
