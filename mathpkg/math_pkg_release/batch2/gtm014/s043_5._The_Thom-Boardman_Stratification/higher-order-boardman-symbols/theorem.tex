Let $f: X \to Y$ be a 2-generic smooth map between smooth manifolds. The higher-order Boardman symbol $S_{i,j,k}(f)$ is defined inductively as the set of points in $S_{i,j}(f)$ where the restricted map
$$f|_{S_{i,j}(f)}: S_{i,j}(f) \to Y$$
drops rank by $k$. If $S_{i,j,k}(f)$ is a submanifold of $X$, one may similarly define $S_{i,j,k,\ell}(f)$, and so on, yielding symbols $S_{r_1,r_2,\ldots,r_k}(f)$ for any finite sequence of integers $r_1 \ge r_2 \ge \cdots \ge r_k \ge 0$, provided each intermediate set is a submanifold.
