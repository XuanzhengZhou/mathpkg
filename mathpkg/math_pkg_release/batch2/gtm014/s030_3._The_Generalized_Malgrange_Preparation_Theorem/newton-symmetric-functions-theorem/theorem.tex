Let $f(x_1, \ldots, x_n)$ be a smooth symmetric function on $\mathbf{R}^n$. Let $g_i(x_1, \ldots, x_n)$ be the elementary symmetric polynomials. Then there exists a smooth function $h$ on $\mathbf{R}^n$ such that
$$f(x_1, \ldots, x_n) = h(g_1(x), \ldots, g_n(x)).$$
In particular, every smooth even function $f(x)$ can be written as $f(x) = g(x^2)$ for some smooth $g$.
