Given a pair of vector spaces $V$ and $W$, let $L^r(V, W)$ be the set of linear maps of $V$ into $W$ which drop rank by $r$ (i.e., linear maps whose corank is exactly $r$).

For each $p \in G(s, V)$, where $s = \dim V - r$ and $G(s, V)$ is the Grassmannian of $s$-dimensional subspaces, let $Q_p = V / K_p$ where $K_p$ is the kernel corresponding to $p$. We have a projection $\pi_p: V \to Q_p$ inducing a transpose map
$$\pi_p^*: \operatorname{Hom}(Q_p, W) \to \operatorname{Hom}(V, W).$$
The image of $\pi_p^*$ is the set of elements in $\operatorname{Hom}(V, W)$ of corank $\geq r$. Moreover, $\pi_p^*$ maps $L^0(Q_p, W)$ bijectively onto the elements in $\operatorname{Hom}(V, W)$ which are precisely of corank $r$, yielding the identification
$$L^r(V, W) \cong \bigcup_{p \in G(s, V)} L^0(Q_p, W).$$
This identification provides $L^r(V, W)$ with a manifold structure.
