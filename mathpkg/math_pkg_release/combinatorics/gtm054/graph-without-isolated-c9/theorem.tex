For any graph without isolated vertices, the following inequalities hold:

(a) $\alpha_0 \leq \beta_{10} \leq \alpha_0 + \alpha_1$;
(b) $\alpha_1 \leq \beta_{01} \leq 2\alpha_1$;
(c) $\alpha_1 \leq \nu_0 - \alpha_0 \leq 2\alpha_1$.

Furthermore, if equality holds in any one of the left-hand inequalities above, then it holds in all three of the left-hand inequalities. If equality holds in any one of the right-hand inequalities or in the inequality $\delta \leq \alpha_0$ (cf. C6b), then it holds in all three right-hand inequalities and $\delta = \alpha_0$.
