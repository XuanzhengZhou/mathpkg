Let $\Lambda = (V, \mathcal{A})$ be a set system satisfying A12, and let $\mathcal{B}$ be the collection of largest sets in $\mathcal{A}$. The greedy algorithm yields a heaviest set in $\Lambda$ for every weight function if and only if the following two conditions hold:

(a) For every $A \in \mathcal{A}$, there exists $B \in \mathcal{B}$ such that $A \subseteq B$.

(b) $(V, \mathcal{B})$ is an exchange system.
