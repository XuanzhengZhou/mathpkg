An independent set of vertices of a graph which meets every clique is called a spread. A complete set which meets every scattering is called a clump. Clearly if $S$ is a spread in a graph $\Gamma$, then $S$ is a clump in $\Gamma'$. Moreover, if $S$ is a spread in $\Gamma$ and $C$ is a clump in $\Gamma$, then $\omega(\Gamma_S) = \omega(\Gamma) - 1$ and $\alpha_0(\Gamma_C) = \alpha_0(\Gamma) - 1$.
