An unsaturated path is a path in a network whose forward edges are not yet at full capacity and whose backward edges carry positive flow, with the additional condition that it does not use the distinguished edge $e_0$. This concept is central to the constructive proof of the Max-Flow Min-Cut Theorem: if an unsaturated path from $x_0$ to $y_0$ exists, the flow through $e_0$ can be augmented; otherwise, a minimum cut has been identified.
