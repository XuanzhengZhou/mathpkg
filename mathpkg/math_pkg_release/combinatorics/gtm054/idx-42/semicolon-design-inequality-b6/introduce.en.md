This theorem generalizes Theorem A12 and establishes a fundamental inequality for $(;a)$-designs: the number of vertices $v$ is at least the number of blocks $b$ when $a \geq 2$, provided no vertex is incident with every block. When $a \geq 3$, equality $v = b$ characterizes trivial designs — those whose incidence matrix corresponds to the set system $(V, \mathcal{P}_{v-1}(V))$ of all $(v-1)$-subsets. The proof proceeds by induction on $a$, with Theorem A12 serving as the base case $a = 2$.
