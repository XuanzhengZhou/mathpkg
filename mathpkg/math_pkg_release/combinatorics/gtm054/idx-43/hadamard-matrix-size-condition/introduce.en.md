The order $m$ of a Hadamard matrix must be $1$, $2$, or a multiple of $4$. This necessary condition is proved by analyzing the column types in the first three rows of a normalized Hadamard matrix and using the orthogonality constraints to derive the congruence $m \equiv 0 \pmod{4}$ for $m \geq 4$.
