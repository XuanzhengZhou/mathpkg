Let $M$ be an incidence matrix of a design $\Lambda$.
(a) Let $J$ denote a matrix (of appropriate dimensions) wherein every entry is $1$. Let $n_1, n_2 \in \mathbb{N}$ such that $0 < n_2 < n_1 < v$. Prove:
$$M^*M = (n_1 - n_2)I + n_2J$$
if and only if $\Lambda$ is a $(; 1, 2)$-design with $k = n_1$ and $\lambda^*(2) = n_2$.
(b) If $\Lambda$ is a $(; 1, 2)$-design, then
$$\det(M^*M) = [k + (b - 1)\lambda^*(2)](k - \lambda^*(2))^{b-1}.$$
