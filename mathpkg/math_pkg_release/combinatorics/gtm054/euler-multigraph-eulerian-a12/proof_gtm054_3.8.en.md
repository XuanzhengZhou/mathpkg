Let $\Gamma = (V, f, E)$ be a multigraph. By IIA8, $E \in \mathcal{X}(\Gamma)$ if and only if $\mathcal{X}^{\perp}(\Gamma)$ is even, i.e., if and only if each vertex has even valence. If $\Gamma$ is connected and if $E$ is a cycle, then by Proposition A10, $\Gamma$ is an Euler circuit. Conversely, if $\Gamma$ has an Euler circuit, it is connected. Moreover, the set $E$ of the edges of the Euler circuit is a cycle.

The traditional proof of Euler’s Theorem is constructive. The algebraic proof, however, has the advantage that it shows that Euler’s Theorem and the following well-known result are really “dual” to one another.
