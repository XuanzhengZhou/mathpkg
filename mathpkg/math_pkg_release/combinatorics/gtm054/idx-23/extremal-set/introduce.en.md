An extremal set in a graph is the natural analogue, for arbitrary graphs, of a critical set in bipartite graphs. It is a vertex subset whose deficiency achieves the maximum possible value $\delta(\Gamma)$. The relationship between extremal sets and critical sets is explored in subsequent propositions, forming the foundation for the general theory of 1-factors.
