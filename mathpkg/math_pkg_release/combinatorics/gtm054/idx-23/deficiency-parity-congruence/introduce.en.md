This lemma establishes a parity property of the deficiency function. Since $|U|$ is odd if and only if the induced subgraph $\Gamma_U$ has an odd number of components with odd vertex sets, the difference $|U| - \|U\|$ is always even. Consequently, $|U \cup N(U)| - \delta(U)$ is always even, a fact that proves useful in structural arguments about 1-factors.
