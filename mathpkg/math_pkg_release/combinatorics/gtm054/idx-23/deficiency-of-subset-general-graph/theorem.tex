For a graph $\Gamma = (V, \mathcal{E})$ and a subset $U \in \mathcal{P}(V)$, let $\|U\|$ denote the number of components of the induced subgraph $\Gamma_U$ that contain an odd number of vertices. The deficiency of $U$ is defined as

$$\delta(U) = \|U\| - |N(U)|.$$

If $U$ is an independent set, then $\|U\| = |U|$, recovering the bipartite definition of deficiency as a special case.
