This corollary specializes E4 to set systems: a set system distinguishes its vertices precisely when its point permutation group $G_0(\Lambda)$ is isomorphic to its edge permutation group $G_1(\Lambda)$. The isomorphism is as abstract groups. Together with E2, this gives a clean algebraic criterion for the distinguishing-vertices property.
