Consider an $st$-path $s = s_0, s_1, \ldots, s_n = t$ in $\Lambda$, and assume $s \in V$. One easily verifies that for any $(p, q) \in G(\Lambda)$, the sequence $q(s_0), p(s_1), q(s_2), \ldots$ is also a path in $\Lambda$. From this together with property C20 (which characterizes when a system is a direct sum), we immediately deduce that for each component index $i$, there exists $j$ such that $q[V_i] = V_j$ and $p[E_i] = E_j$, and the restricted pair $(p_{|E_i}, q_{|V_i})$ is a system-isomorphism from $\Lambda_i$ to $\Lambda_j$.
