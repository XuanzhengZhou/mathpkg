Denote the set of components of a system $\Lambda$ by
$$\{\Lambda_{ij} : i = 1, \ldots, n_j; \; j = 1, \ldots, r\}$$
where $\Lambda_{ij}$ and $\Lambda_{i'j'}$ are system-isomorphic if and only if $j = j'$. Let $\Lambda_{ij} = (V_{ij}, f_{ij}, E_{ij})$ for each $i, j$. Let
$$m_{0j} = |\{i : V_{ij} \neq \varnothing\}|, \qquad m_{1j} = |\{i : E_{ij} \neq \varnothing\}|.$$
Then for $h = 0, 1$,
$$G_h(\Lambda) = \bigoplus_{j=1}^{r} \bigl( \Pi(m_{hj}) \text{ wreath } G_h(\Lambda_{1j}) \bigr),$$
where $\oplus$ denotes the direct product of permutation groups.
