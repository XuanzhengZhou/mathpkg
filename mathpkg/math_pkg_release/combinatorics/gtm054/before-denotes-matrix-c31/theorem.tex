$MM^* = qI + (q - 1)J$.

Let $H_1 = 2M - J$ where, as before, $J$ denotes the matrix whose every entry is 1. Then $H_1$ is a $(4q - 1) \times (4q - 1)$ matrix whose entries are $\pm 1$ and whose line sums are $-1$. Beginning with C31 we derive the following sequence of equations, which are equivalent under our assumptions on $M$ or on $H_1$:

$$4MM^* - (4q - 4)J = 4qI$$
$$4MM^* - 2(2q - 1)J - 2(2q - 1)J + (4q - 1)J = 4qI - J$$
$$4MM^* - 2JM - 2JM^* + J^2 = 4qI - J$$
$$(H_1 + J)(H_1^* + J) - J(H_1 + J) - J(H_1^* + J) + J^2 = 4qI - J$$
