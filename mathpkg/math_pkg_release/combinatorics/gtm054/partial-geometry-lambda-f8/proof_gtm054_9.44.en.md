It is immediate from F1 that $\lambda_1 = 1$ and $\lambda_2 = 0$. Each point $x \in V$ is incident with exactly $r$ lines, each of which contains exactly $k - 1$ points other than $x$. Thus each point has precisely $n_1 = r(k - 1)$ first associates. Substituting this value and the value of $v$ from F5d into $n_2 = v - n_1 - 1$ (cf. E1), we obtain $n_2 = (k - t)(k - 1)(r - 1)/t$.
