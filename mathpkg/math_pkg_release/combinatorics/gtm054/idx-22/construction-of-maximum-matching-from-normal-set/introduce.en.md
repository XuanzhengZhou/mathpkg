Theorem B10 provides a constructive characterization of all maximum independent edge sets (largest matchings) in a graph that admits a normal set. Given a normal set $U$, one constructs a bipartite graph $B_U$ between the components of $\Gamma_U$ and the singletons of the neighbor set $N(U)$. Every maximum matching of the original graph can be built by combining a matching in $B_U$ with 1-factors from each component of $\Gamma_U$, possibly omitting one vertex from components not matched in $B_U$.
