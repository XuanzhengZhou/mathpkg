A vertex-covering set is a subset $S \subseteq V \cup \mathcal{E}$ of the vertices and edges of a graph such that every vertex of positive degree is incident (in the extended sense) with some element of $S$. This property is upward-closed with respect to set inclusion, meaning that any superset of a vertex-covering set remains a vertex-covering set. Vertex-covering sets are one of the key structural concepts for studying matchings in bipartite graphs.
