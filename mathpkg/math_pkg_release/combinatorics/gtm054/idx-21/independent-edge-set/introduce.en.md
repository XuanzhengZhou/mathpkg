An independent edge set is an independent subset of the edge set $\mathcal{E}$ of a graph, meaning no two distinct edges in the set share a common endpoint. Such a set is also called a matching. The size of a largest independent edge set in a graph $\Gamma$ defines the edge-independence number $\alpha_1(\Gamma)$, a fundamental parameter in matching theory.
