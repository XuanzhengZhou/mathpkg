An independent vertex set, also known as an internally stable set, is a subset of the vertex set $V$ of a graph in which no two distinct vertices are adjacent. This is the classical notion of an independent set restricted to vertices. The size of a largest independent vertex set in a graph $\Gamma$ defines the vertex-independence number $\alpha_0(\Gamma)$, a fundamental graph invariant.
