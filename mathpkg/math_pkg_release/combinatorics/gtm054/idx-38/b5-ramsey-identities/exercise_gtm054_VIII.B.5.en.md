\textbf{Exercise B5.}
Show that:
(a) If $m \geq 2$ and $1 \leq s \leq q_i$ for $i = 1,\ldots,m$, then
$$n(q_1, \ldots, q_{m-1} : s) = n(q_1, \ldots, q_{m-1}, s : s).$$
In particular, if $m = 2$, then $n(q, s : s) = q$.

(b) If $\pi \in \Pi(\{1, \ldots, m\})$ is a permutation, then
$$n(q_1, \ldots, q_m : s) = n(q_{\pi(1)}, \ldots, q_{\pi(m)} : s).$$
