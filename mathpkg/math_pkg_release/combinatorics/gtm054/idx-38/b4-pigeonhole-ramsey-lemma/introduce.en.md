Lemma B4 identifies the Ramsey number for $s=1$ (coloring singletons, i.e., vertices) with the classical pigeonhole principle. If we want to guarantee a set of $q_i$ elements all receiving color $i$ for some $i$, then the threshold is $\sum q_i - m + 1$. This is the familiar formula: placing $(\sum q_i - m)$ objects into $m$ boxes can avoid having $q_i$ in box $i$ for all $i$, but adding one more forces this to happen for some box.
