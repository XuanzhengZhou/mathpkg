Let integers $q_1, \ldots, q_m, s$ be given such that
$$1 \leq s \leq q_i \quad (i = 1, \ldots, m)$$
where $m \geq 1$. Let $V$ be any set, and let $H$ be the set of all functions
$$h: \mathcal{P}_s(V) \to \{1, 2, \ldots, m\}.$$

The \textbf{Ramsey number} $n(q_1, \ldots, q_m : s)$, when it exists, is the least integer $n$ such that for any set $V$ with $|V| \geq n$ and any $h \in H$, there exists an index $i \in \{1, \ldots, m\}$ and a subset $S \subseteq V$ with $|S| = q_i$ such that $\mathcal{P}_s(S) \subseteq h^{-1}[i]$.
