The least integer $m$ such that there exists a surjection
$$h: V \to \{1, 2, \ldots, m\}$$
with the property that $h^{-1}[i]$ is a complete set for each $i = 1, \ldots, m$ is denoted by $\theta(\Gamma)$. Equivalently, $\theta(\Gamma)$ is the minimum number of complete sets needed to partition (or cover) the vertex set $V$.
