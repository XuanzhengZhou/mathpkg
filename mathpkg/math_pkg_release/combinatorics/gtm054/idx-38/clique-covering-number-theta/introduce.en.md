The clique covering number $\theta(\Gamma)$ is the minimum number of complete subsets into which the vertex set of $\Gamma$ can be partitioned. Formally, it is the least $m$ for which a coloring-function $h: V \to \{1,\ldots,m\}$ exists such that each color class $h^{-1}[i]$ is a complete set. This parameter is dual to the vertex chromatic number $\chi_0$ via complementation: $\theta(\Gamma) = \chi_0(\Gamma')$.
