Proposition B3 states the trivial case of Ramsey numbers: when there is only one color ($m=1$), the Ramsey number $n(q : s)$ is simply $q$. This means that any set of size at least $q$ necessarily contains a $q$-subset whose $s$-subsets are all colored with that single color, which is vacuously true.
