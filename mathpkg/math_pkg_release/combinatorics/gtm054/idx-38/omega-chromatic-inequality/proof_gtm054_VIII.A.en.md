In a proper vertex coloring of $\Gamma$, no two vertices of the same complete set receive the same color, because by definition all vertices in a complete set are pairwise adjacent. Hence a clique, being a largest complete set, requires $|\text{clique}|$ distinct colors. Therefore $\omega(\Gamma) \leq \chi_0(\Gamma)$.
