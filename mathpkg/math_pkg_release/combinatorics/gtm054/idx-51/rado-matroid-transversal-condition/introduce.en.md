Rado's condition generalizes Phillip Hall's marriage theorem to the setting of matroids. Given a system $\Lambda = (V, f, E)$ and a matroid $\Theta$ on the same vertex set $V$, the proposition characterizes precisely when there exists a system of distinct representatives (an LDR) whose image forms an independent set in the matroid. The necessary and sufficient condition is a rank inequality: for every subset $F$ of the index set $E$, the rank (in $\Theta$) of the union of the corresponding sets $f(e)$ must be at least $|F|$. When $\Theta$ is the trivial matroid $(V, \varnothing)$ whose rank function is cardinality, this reduces to Hall's condition $|\bigcup_{e \in F} f(e)| \geq |F|$.
