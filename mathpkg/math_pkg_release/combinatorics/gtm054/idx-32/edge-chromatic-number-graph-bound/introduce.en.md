This corollary of Vizing's theorem (A37) establishes the classical bounds for the edge chromatic number of any simple graph: it is either equal to the maximum degree $\hat{\rho}$ or one greater. Both bounds can be attained: even circuits achieve the lower bound $\chi_1 = \hat{\rho}$, while odd circuits achieve the upper bound $\chi_1 = \hat{\rho} + 1$. This dichotomy is a central result in edge coloring theory.
