Let the multigraphs $\Gamma$ and $\Theta$ have a common edge set $E$. We say that $\Gamma$ is orthogonal to $\Theta$, and write $\Gamma \perp \Theta$ if $\mathcal{X}(\Gamma) = \mathcal{X}^{\perp}(\Theta)$.

Some consequences of the assertion $\Gamma \perp \Theta$ are immediate. First of all, $\Theta \perp \Gamma$. Secondly, $E = \text{Fnd}(\mathcal{X}^{\perp}(\Gamma)) = \text{Fnd}(\mathcal{X}(\Gamma))$, and so neither $\Gamma$ nor $\Theta$ may have an isthmus. Thirdly, if neither graph has isolated vertices, it then follows from B3 that $\Gamma$ is biconnected if and only if $\Theta$ is biconnected. Fourthly, if $\Gamma \perp \Theta'$ also holds, then $\Theta$ and $\Theta'$ have identical cycle space and cocycle space.

Because we have given a combinatorial definition for planarity, we obtain a short proof of Whitney’s characterization of planar multigraphs.
