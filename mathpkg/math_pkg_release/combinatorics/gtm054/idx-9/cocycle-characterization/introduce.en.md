This proposition characterizes edge sets of a multigraph that contain nonempty cocycles in terms of the rank invariant $\nu_{-1}$. Specifically, removing the edges in $F$ increases the number of connected components if and only if $F$ contains a nonempty cocycle. The second part characterizes elementary cocycles as minimal edge sets with this property — a set $F$ is an elementary cocycle exactly when removing it increases the component count, but removing $F$ together with any single additional edge does not further increase it.
