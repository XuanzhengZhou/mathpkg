A spanning coforest is the dual notion to a spanning forest. While removing the edges of a spanning forest from the edge set yields a spanning coforest, these two submultigraphs are complementary. The edge set of a spanning coforest — a dendroid in Tutte's terminology — provides a maximal set of edges containing no cocycle, dual to how a spanning forest provides a maximal set containing no cycle.
