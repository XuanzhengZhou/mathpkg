Let $\Gamma = (V, f, E)$ be a connected multigraph, let $W$ be a minimal separating set such that $|W| \leq 3$. Let $U_1$ be the vertex set of a component of $\Gamma_{(W)}$, and let $U_2 = V \setminus W \setminus U_1$. Then $\Gamma$ is planar if and only if the contraction $\Gamma_j$ obtained by contracting $U_j$ to a single vertex is planar for $j = 1, 2$.
