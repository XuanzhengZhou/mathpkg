(Inversion Formulas). Let $\mu$ be the Möbius function of $(U, \leq)$. For any selection $s \in \mathbb{S}(U)$ and for all $x \in U$,

(a) $s(x) = \sum_{u \in U} \mu(x, u) \bar{s}(u)$ if $(U, \leq)$ has a maximum element;

(b) $s(x) = \sum_{u \in U} \mu(u, x) \bar{s}(u)$ if $(U, \leq)$ has a minimum element.
