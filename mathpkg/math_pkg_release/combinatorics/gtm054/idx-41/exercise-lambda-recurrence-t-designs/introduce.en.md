Exercise A9 asks the reader to prove a fundamental recurrence relation for the parameter $\lambda$ of a $t$-design. The formula $\lambda(i+1) = \frac{k-i}{v-i} \lambda(i)$ shows how the number of blocks containing an $(i+1)$-subset relates to the number containing an $i$-subset. Since $k \leq v$, this implies $\lambda$ is a decreasing function of $i$.
