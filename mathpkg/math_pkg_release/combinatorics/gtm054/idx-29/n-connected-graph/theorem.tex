\textbf{Definition.} A graph $\Gamma$ is \textit{$n$-connected} if $|V| \geq n+1$ and for every $S \subseteq V$ with $|S| < n$, the subgraph $\Gamma_{(S)}$ is connected. Thus $\Gamma$ is connected if and only if $\Gamma$ is $1$-connected. If $\Gamma$ is $n$-connected and $0 \leq m \leq n$, then $\Gamma$ is $m$-connected.
