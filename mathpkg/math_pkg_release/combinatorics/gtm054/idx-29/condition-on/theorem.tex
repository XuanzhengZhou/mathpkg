\textbf{Definition.} A multigraph $\Gamma$ satisfies \textit{condition $O_n$} $(n = 1, 2, \ldots)$ if $|V| \geq 2n$ and, given any set $\{a_1, \ldots, a_n, z_1, \ldots, z_n\} \in \mathcal{P}_{2n}(V)$, there exists an openly-disjoint $n$-family $\{\Pi_1, \ldots, \Pi_n\}$ of paths such that $\Pi_i$ is an $a_iz_i$-path for $i = 1, \ldots, n$. Clearly $O_n$ implies $O_m$ if $n > m$.
