If $X$ is finite and $h \in X^X$, then $h$ is a surjection if and only if $h$ is an injection.

If $S \subseteq X$ and $h \in X^X$, we say $h$ fixes $S$ if $h[S] \subseteq S$. If $h_{|S} = 1_S$, we say $h$ fixes $S$ pointwise.

If $*$ is a binary operation on $Y$, then $*$ induces a binary operation on $Y^X$ which is also denoted by $*$. Thus

$$(f_1 * f_2)(x) = f_1(x) * f_2(x), \quad \text{for all } f_1, f_2 \in Y^X, x \in X.$$

Note that if $*$ on $Y$ enjoys any of the properties of associativity, commutativity, or existence of an identity, then that property is also enjoyed by $*$ on $Y^X$.

One final important convention: henceforth, all arbitrarily chosen sets will be finite unless explicitly stated otherwise.
