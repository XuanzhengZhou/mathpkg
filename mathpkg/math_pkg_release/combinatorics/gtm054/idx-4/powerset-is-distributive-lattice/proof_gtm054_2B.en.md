For any $A, B, C \subseteq U$, the distributive law $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$ holds in set theory. Since meet in $(\mathcal{P}(U), \subseteq)$ is $\cap$ and join is $\cup$, this verifies condition (a) of Proposition B32, so $(\mathcal{P}(U), \subseteq)$ is a distributive lattice. Any Boolean lattice, being isomorphic to a sublattice of a power set lattice, inherits distributivity.
