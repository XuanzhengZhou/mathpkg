If $*$ is a binary operation on $Y$, then $*$ induces a binary operation on $Y^X$ given by
$$(f_1 * f_2)(x) = f_1(x) * f_2(x), \quad \text{for all } f_1, f_2 \in Y^X, x \in X.$$
If $*$ on $Y$ enjoys associativity, commutativity, or existence of an identity, then that property is also enjoyed by $*$ on $Y^X$.
