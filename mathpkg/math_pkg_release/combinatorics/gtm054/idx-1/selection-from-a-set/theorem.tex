For a set $U$, a function $s \in \mathbb{N}^U$ is called a selection from $U$. If $x \in U$, the number $s(x)$ is the number of times $x$ is selected by $s$. The cardinality (weight) of the selection is
$$|s| = \sum_{x \in U} s(x).$$
If $|s| = m$, $s$ is an $m$-selection. The set of all $m$-selections from $U$ is $\mathbb{S}_m(U)$, and $\mathbb{S}(U) = \bigcup_{m=0}^{\infty} \mathbb{S}_m(U) = \mathbb{N}^U$.
