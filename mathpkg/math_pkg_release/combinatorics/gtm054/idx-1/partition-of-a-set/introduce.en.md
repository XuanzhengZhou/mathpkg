A partition decomposes a set into a collection of nonempty, pairwise disjoint subsets (cells) that together cover the whole set. The number of cells determines the partition type: $\mathbb{P}_m(U)$ collects all partitions with exactly $m$ cells. Partitions correspond bijectively to equivalence relations.
