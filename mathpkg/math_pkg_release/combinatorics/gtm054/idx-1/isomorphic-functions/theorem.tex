Functions $f: B \rightarrow U$ and $g: C \rightarrow W$ are isomorphic if there exist bijections $p: B \rightarrow C$ and $q: U \rightarrow W$ such that $g p = q f$. In this case $(p, q)$ is called a function-isomorphism from $f$ to $g$.

If $p$ alone satisfies $g p = f$, then $p$ is a partition-isomorphism. If $q$ alone satisfies $q f = q$, then $q$ is a selection-isomorphism.

If $f$ and $g$ have the same codomain ($U = W$) and there exists a bijection $p: B \rightarrow C$ with $f = g p$, then $p$ is a set-isomorphism from $f$ to $g$.
