The set $\mathbb{Q}^X$ of all functions from a set $X$ to the rational numbers $\mathbb{Q}$ is a commutative algebra under the usual operations of pointwise addition, multiplication, and scalar multiplication:

\begin{align*}
(h_1 + h_2)(x) &= h_1(x) + h_2(x) &&\text{for all } x \in X,\\
(h_1h_2)(x) &= h_1(x)h_2(x) &&\text{for all } x \in X,\\
(\eta h_1)(x) &= \eta h_1(x) &&\text{for all } \eta \in \mathbb{Q}, x \in X.
\end{align*}
