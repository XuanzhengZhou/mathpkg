(Marshall Hall [h.1], 1948). Let $q \in \mathbb{N} + \{0\}$. Let the system $\Lambda = (V, f, E)$ satisfy

$$D \subseteq E \Rightarrow |D| \leq \left| \bigcup_{e \in D} f(e) \right|,$$

and suppose that all blocks have size at least $q$. The number of LDR’s admitted by $\Lambda$ is at least:

(a) $q!$ if $q < |E|$;
(b) $q! / (q - |E|)!$ if $|E| \leq q$.
