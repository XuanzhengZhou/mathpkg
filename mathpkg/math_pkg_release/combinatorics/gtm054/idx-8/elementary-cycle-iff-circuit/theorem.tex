If $Z = \{e_1, \ldots, e_k\}$ is an elementary cycle in $\mathcal{L}(\Gamma)$, then the submultigraph $\Gamma_Z$ formed by the edges of $Z$ is a circuit. Conversely, if $\Delta$ is a circuit in $\Gamma$ with edge set $Z$, then $Z$ is an elementary cycle in $\mathcal{L}(\Gamma)$.

[Note: The converse is partial. Not every cycle is the edge set of a single circuit; e.g., the edges of two disjoint circuits form a cycle but not a single circuit.]
