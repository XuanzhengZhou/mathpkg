This lemma shows that the collection of all transversals of a set system $\Lambda = (V, f, E)$ forms an independence system on $V$. The proof uses the bipartite graph representation of $\Lambda$ and the matching matroid of that graph: a set $T \subseteq V$ is a transversal if and only if it is contained in a matched set, which is equivalent to being independent in the matching matroid restricted to $V$. This lemma is the foundation for defining the transversal matroid.
