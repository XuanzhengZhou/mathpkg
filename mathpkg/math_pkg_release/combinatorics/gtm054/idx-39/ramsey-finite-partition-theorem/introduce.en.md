This proposition provides a finitary version of Schur's theorem using Ramsey numbers. For any $m$-partition of the initial segment $\{1, 2, \ldots, k\}$ where $k$ is sufficiently large relative to the Ramsey number $n(3, \ldots, 3: 2)$ with $m$ arguments all equal to $3$, there exists a part containing two elements whose sum also lies in that same part. The proof uses the difference function $f(\{a,b\}) = |a-b|$ to reduce the additive problem to a Ramsey-type coloring problem on $2$-subsets.
