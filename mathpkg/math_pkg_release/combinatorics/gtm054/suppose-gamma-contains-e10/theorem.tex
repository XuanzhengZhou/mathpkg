Suppose $\Gamma$ contains no isolated vertex. Then $\Gamma$ is $k$-separated if and only if there exists $F \subset E$, $F \neq \varnothing$, such that
(a) $k \leq \min\{|F|, |E + F|\}$;
(b) $k = |A(\Gamma_F)| = |A(\Gamma_{E+F})|$.

We observe that
$$|A(\Gamma_F)| = \nu_0(\Gamma) - |\{x: f^*(x) \subseteq F\}| - |\{x: f^*(x) \subseteq E + F\}|.$$
