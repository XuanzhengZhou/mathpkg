For any set $U$, refinement is a partial order on $\mathbb{P}(U)$.

Thus each of $\mathcal{P}(U), \mathbb{S}(U)$, and $\mathbb{P}(U)$ admits a partial order in a rather natural way. Having defined various operations on these objects in the first chapter, let us observe how they relate to these objects as partially-ordered sets.
