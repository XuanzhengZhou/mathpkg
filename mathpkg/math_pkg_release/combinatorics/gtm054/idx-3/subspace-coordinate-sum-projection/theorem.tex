Let $\mathcal{A}$ be a subspace of $\mathcal{P}(U)$.

(a) If $\mathcal{A} = \mathcal{B} \oplus \mathcal{C}$ and if $B = \operatorname{Fnd}(\mathcal{B})$, then $\pi_B[\mathcal{A}] = \mathcal{P}(B) \cap \mathcal{A} = \mathcal{B}$.

(b) Conversely, if $B \in \mathcal{P}(U)$ and if $\pi_B[\mathcal{A}] = \mathcal{P}(B) \cap \mathcal{A}$, then
$$\pi_{U \setminus B}[\mathcal{A}] = \mathcal{P}(U \setminus B) \cap \mathcal{A},$$
and
$$\mathcal{A} = (\mathcal{P}(B) \cap \mathcal{A}) \oplus (\mathcal{P}(U \setminus B) \cap \mathcal{A}) = \pi_B[\mathcal{A}] \oplus \pi_{U \setminus B}[\mathcal{A}].$$
