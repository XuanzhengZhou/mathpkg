Lemma B8 establishes monotonicity properties for the product operation on partitions with respect to the refinement order. The product of two partitions is always a refinement of each factor, mirroring how intersection is always contained in each factor (B7). Moreover, if one partition refines another, multiplying both by a fixed partition preserves this refinement relationship. These properties demonstrate that the partition product operation is well-behaved with respect to the partial order structure on $\mathbb{P}(U)$.
