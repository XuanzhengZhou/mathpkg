Suppose $\Gamma = (V, f, E)$ is of Type I. Since $\nu_0 = 2$ and every edge must be incident with some vertex, $f(e) = V$ for all $e \in E$. Since $\nu_1 = k$, the edge set has exactly $k$ elements, each joining the two vertices. This multigraph is uniquely determined up to isomorphism. Existence is verified by constructing such a multigraph explicitly.

For Type II, $\rho = 2$ means the graph is $2$-valent. With $\nu_1 = k$, the graph is a disjoint union of cycles. Since the graph has an $m$-covalent imbedding and is connected (as it belongs to the classification of connected isovalent planar multigraphs), it must be a single $k$-cycle, which is uniquely determined.
