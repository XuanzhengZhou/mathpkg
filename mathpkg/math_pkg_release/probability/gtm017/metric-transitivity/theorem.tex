An event $A$ is \textbf{invariant} if $A = T^{-1}A$ up to a set of measure zero. The process (or its measure $P$) is \textbf{metrically transitive} if the sigma-field of invariant sets is trivial (consisting only of sets of measure 0 or 1).

Equivalently, every invariant function is constant almost everywhere.
\end{enumerate}
