The construction takes $\Omega$ as the space of all functions $w: T \to \mathbb{R}$ and defines the probability measure on cylinder sets using the given consistent finite-dimensional distributions. The set function is countably additive on the field of cylinder sets (a consequence of the consistency conditions and a compactness argument). Carathéodorys theorem extends it uniquely to the generated sigma-field. $X_t(w) = w(t)$ defines the process.
