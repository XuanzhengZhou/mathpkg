Given a consistent family of finite-dimensional distribution functions indexed by a parameter set $T$, there exists a probability space $(\Omega, \mathfrak{F}, P)$ and a stochastic process $\{X_t(\omega), t \in T\}$ on it such that the given distributions are precisely the finite-dimensional joint distributions of the process. The sigma-field $\mathfrak{F}$ is generated by cylinder sets $\{\omega \mid \omega(t_1) \leq x_1, \ldots, \omega(t_k) \leq x_k\}$, and $P$ is obtained by extending the set function via Caratheodory's theorem.
