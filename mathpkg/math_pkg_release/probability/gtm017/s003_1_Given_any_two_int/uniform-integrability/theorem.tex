Let $X_n(w)$, $n = 1, 2, \ldots$, be a sequence of integrable functions with $X(w) = \lim_{n \to \infty} X_n(w)$. Define
$$B_{n,a} = \{w \mid |X_n(w)| \geq a\}.$$
The functions $X_n(w)$ are said to be uniformly integrable if
$$\int_{B_{n,a}} |X_n(w)| \, dP \to 0$$
uniformly in $n$ as $a \to \infty$.
