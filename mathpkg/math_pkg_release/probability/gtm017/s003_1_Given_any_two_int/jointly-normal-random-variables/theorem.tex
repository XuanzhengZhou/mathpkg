The random variables $X_1, \ldots, X_k$ are said to be jointly normally distributed if every linear combination $t_1 X_1 + \cdots + t_k X_k$ is a normally distributed random variable. The joint characteristic function is
$$\varphi(t_1, \ldots, t_k) = \exp\left\{-\frac{1}{2} \mathbf{t} \mathbf{R} \mathbf{t}' + i \mathbf{t} \mathbf{m}'\right\},$$
where $\mathbf{m} = (m_1, \ldots, m_k)$ is the mean vector and $\mathbf{R} = (r_{u,v})$ is the covariance matrix. When $\mathbf{R}$ is nonsingular, the joint density is
$$f(x_1, \ldots, x_k) = \frac{1}{(2\pi)^{k/2} |\mathbf{R}|^{1/2}} \exp\left\{-\frac{1}{2} (\mathbf{x} - \mathbf{m}) \mathbf{R}^{-1} (\mathbf{x} - \mathbf{m})'\right\}.$$
