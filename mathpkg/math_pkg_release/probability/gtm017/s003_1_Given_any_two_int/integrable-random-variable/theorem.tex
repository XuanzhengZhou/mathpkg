Define the non-negative and negative parts of a random variable $X(w)$:
$$X_+(w) = \max(X(w), 0), \quad X_-(w) = X_+(w) - X(w).$$
A random variable $X(w)$ is said to be integrable if both $X_+(w)$ and $X_-(w)$ are integrable (as non-negative random variables). In that case,
$$\int X(w) \, dP = \int X_+(w) \, dP - \int X_-(w) \, dP.$$
