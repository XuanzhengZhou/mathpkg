Consider an experiment $\mathscr{A}$ with a finite number of elementary outcomes $A_1, \dots, A_n$ and corresponding probabilities $p_i > 0$, $\sum p_i = 1$. The \textbf{entropy} $H(\mathscr{A})$ is a number that measures the uncertainty associated with $\mathscr{A}$.

The entropy satisfies:
\begin{enumerate}
\item $H(p_1, \dots, p_n)$ is a continuous function of $p_1, \dots, p_n$.
\item $H(1/n, \dots, 1/n)$ is a nondecreasing function of $n$.
\item (Additivity) $H(\mathscr{B} \vee \mathscr{A}) = H(\mathscr{A}) + H_{\mathscr{A}}(\mathscr{B})$.
\item (Consistency) $H(p_1, \dots, p_n) = H(p_1, \dots, p_n, 0)$.
\end{enumerate}
