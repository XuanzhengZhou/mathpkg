The \textbf{distribution function} of a random variable $X$ is the function
$$F(x) = P(X \leq x) = \sum_{a_i \leq x} p_i.$$

Properties of a distribution function $F(x)$:
\begin{enumerate}
\item $F(x)$ is nondecreasing
\item $\lim_{x \to -\infty} F(x) = 0$, $\lim_{x \to \infty} F(x) = 1$
\item $F$ is continuous from the right
\end{enumerate}
