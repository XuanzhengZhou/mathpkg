Let $X(m)$ be a Markov chain and $Y(m) = f(X(m))$ where $f$ collapses the state space into sets $S_\alpha$. Assume no $S_\alpha$ is a single-entry set. Then $Y(m)$ is Markovian for all initial distributions if and only if for all $\alpha, \beta$,
$$\sum_{j \in S_\beta} p_{i,j} = C_{S_\alpha, S_\beta}$$
is constant for all $i \in S_\alpha$. For reversible chains, this condition is both necessary and sufficient.
