Let $\mu = \varphi'(1) = \sum k q_k$ be the expected number of offspring per individual. The probability of eventual extinction $e = \lim_{t \to \infty} P(X_t = 0)$ satisfies $e = \varphi(e)$.

\begin{itemize}
\item If $\mu \leq 1$ (except the trivial case $q_1 = 1$), then $e = 1$ (certain extinction).
\item If $\mu > 1$, there is a unique solution $\zeta$, $0 \leq \zeta < 1$, of $e = \varphi(e)$, and $e = \zeta < 1$ (positive probability of survival).
\end{itemize}

The expected population size at time $n$ is $EX_n = \mu^n$.
