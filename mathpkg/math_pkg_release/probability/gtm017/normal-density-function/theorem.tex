If $X_1, \ldots, X_k$ are jointly normal with mean vector $\mathbf{m}$ and nonsingular covariance matrix $\mathbf{R}$, their joint density function is
$$f(x_1, \ldots, x_k) = \frac{1}{(2\pi)^{k/2} |\mathbf{R}|^{1/2}} \exp\left\{-\frac{1}{2} (\mathbf{x} - \mathbf{m}) \mathbf{R}^{-1} (\mathbf{x} - \mathbf{m})'\right\}.$$
This follows from the fact that the standardized variables $\mathbf{Z} = (\mathbf{X} - \mathbf{m})\mathbf{R}^{-1/2}$ are independent standard normal.
