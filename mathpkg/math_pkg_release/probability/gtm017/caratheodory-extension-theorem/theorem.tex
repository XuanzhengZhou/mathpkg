Let $m$ be a non-negative set function defined on a field $\mathbb{C}$. If $m$ is countably additive on $\mathbb{C}$, then $m$ can be extended uniquely to a measure on the sigma-field $\mathcal{F}(\mathbb{C})$ generated by $\mathbb{C}$, provided $m$ is sigma-finite on $\mathbb{C}$.

Furthermore, any set $A \in \mathcal{F}(\mathbb{C})$ can be approximated arbitrarily well by sets of $\mathbb{C}$: given $\varepsilon > 0$, there exists $B \in \mathbb{C}$ such that
$$m(A \ominus B) < \varepsilon,$$
where $A \ominus B = (A - B) \cup (B - A)$ is the symmetric difference.
