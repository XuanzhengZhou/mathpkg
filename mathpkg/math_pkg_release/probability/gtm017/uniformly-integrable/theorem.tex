A sequence of integrable functions $\{X_n(w)\}$ is \textbf{uniformly integrable} if
$$\int_{B_{n,a}} |X_n(w)| dP \to 0$$
uniformly in $n$ as $a \to \infty$, where $B_{n,a} = \{w \mid |X_n(w)| \geq a\}$.

If $X_n(w)$ are uniformly integrable and $X(w) = \lim X_n(w)$, then
$$\lim_{n \to \infty} \int |X(w) - X_n(w)| dP = 0.$$
