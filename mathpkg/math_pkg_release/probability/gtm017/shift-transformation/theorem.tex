For a strictly stationary discrete-parameter process $\{X_n\}$, the probability space can be taken as $\Omega = \{\ldots, w_{-1}, w_0, w_1, \ldots\}$ with $X_n(w) = w_n$. The \text{shift transformation} $T: \Omega \to \Omega$ is defined by
$$(Tw)_n = w_{n+1}, \quad n = 0, \pm 1, \ldots.$$

The inverse $T^{-1}$ satisfies $(T^{-1}w)_n = w_{n-1}$. By stationarity, $T$ is \text{measure-preserving}:
$$P(T^{-1}A) = P(A) \quad \text{for all measurable } A.$$
