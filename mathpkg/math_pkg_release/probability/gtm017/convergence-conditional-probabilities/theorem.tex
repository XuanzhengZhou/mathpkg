Let $\mathcal{C}_n$ be an increasing sequence of subsigma-fields of $\mathcal{B}$ with $\mathcal{C}$ the subsigma-field generated by $\bigcup \mathcal{C}_n$. For any event $A$,
$$P(A \mid \mathcal{C}_n) \to P(A \mid \mathcal{C})$$
almost everywhere as $n \to \infty$.

More generally, for a bounded measurable function $f$,
$$E(f \mid \mathcal{C}_n) \to E(f \mid \mathcal{C})$$
almost everywhere.
\end{enumerate}
