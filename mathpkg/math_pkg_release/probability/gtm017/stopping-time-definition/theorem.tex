A random variable $N$ taking values in $\{0, 1, \ldots, \infty\}$ is a \textbf{stopping time} with respect to an increasing family of Borel fields $\{\mathcal{B}_n\}$ if
$$\{w \mid N(w) = n\} \in \mathcal{B}_n$$
for all $n$ (and also for $n = \infty$ with $\mathcal{B}_\infty = \mathcal{B}(\mathcal{B}_n, n = 0, 1, \ldots)$).

The Borel field $\mathcal{B}_T$ associated with a stopping time $T$ consists of all events $A$ such that $A \cap \{T \leq n\} \in \mathcal{B}_n$.
