A necessary and sufficient condition for a function $\varphi(t)$ to be a characteristic function is that $\varphi(0) = 1$, $\varphi(t)$ is continuous, and the nonnegative definiteness condition holds:

$$\sum_{i,j=1}^{n} \varphi(t_i - t_j) \lambda_i \bar{\lambda}_j \geq 0$$

for all $n \geq 1$, all real $t_1, \ldots, t_n$, and all complex $\lambda_1, \ldots, \lambda_n$.

The necessity follows from

$$\sum_{i,j=1}^{n} \varphi(t_i - t_j) \lambda_i \bar{\lambda}_j = \int_{-\infty}^{\infty} \left| \sum_{k=1}^{n} \lambda_k e^{it_k x} \right|^2 dF(x) \geq 0.$$
