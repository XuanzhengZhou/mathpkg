Bochner's theorem characterizes characteristic functions among continuous functions via positive definiteness: a continuous function $\varphi$ with $\varphi(0)=1$ is a characteristic function if and only if the matrices $(\varphi(t_i - t_j))_{i,j}$ are positive semi-definite for all choices of points $t_1,\ldots,t_n$. The necessity is immediate from the representation $\varphi(t) = \int e^{itx} dF(x)$, while the sufficiency (Bochner's theorem) is deeper. This is the analogue for characteristic functions of Bochner's criterion for Laplace transforms via complete monotonicity.
