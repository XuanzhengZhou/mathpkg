A random variable $T$ is infinitely divisible if and only if its characteristic function has the form $\varphi(t) = \exp \psi(t)$ with

$$\psi(t) = it\beta - \frac{t^2\sigma^2}{2} + \int_{-\infty}^{\infty} \left( e^{itx} - 1 - \frac{itx}{1+x^2} \right) \frac{1+x^2}{x^2} \, d\lambda(x),$$

where $\beta \in \mathbb{R}$, $\sigma^2 \geq 0$, and $\lambda$ is a measure on $(\mathbb{R}, \mathcal{B}(\mathbb{R}))$ with $\lambda\{0\} = 0$.
