The Euler numbers $(e_0, e_1, e_2, \ldots)$ are defined by the exponential generating function

$$\frac{2e^x}{e^{2x} + 1} = \sum_{n=0}^{\infty} e_n \frac{x^n}{n!}.$$

The first few values are $e_0 = 1$, $e_1 = 0$, $e_2 = -1$, $e_3 = 0$, $e_4 = -5$, $e_5 = 0$, $e_6 = -61$, $e_7 = 0$, $e_8 = 1385, \ldots$
