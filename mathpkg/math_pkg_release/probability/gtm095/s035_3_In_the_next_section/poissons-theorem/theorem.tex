$$P(\xi_{nk} = 1) = p_{nk}, \quad P(\xi_{nk} = 0) = q_{nk}$$

with $p_{nk} + q_{nk} = 1$. Suppose that

$$\max_{1 \leq k \leq n} p_{nk} \to 0, \quad \sum_{k=1}^{n} p_{nk} \to \lambda > 0, \quad n \to \infty.$$

Then, for each $m = 0, 1, \ldots,$

$$P(S_n = m) \to \frac{e^{-\lambda} \lambda^m}{m!}, \quad n \to \infty.$$
