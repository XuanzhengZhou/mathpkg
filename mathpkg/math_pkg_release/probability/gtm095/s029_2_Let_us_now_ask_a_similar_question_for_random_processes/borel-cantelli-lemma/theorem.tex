Let $A_1, A_2, \ldots$ be events on a probability space $(\Omega, \mathcal{F}, P)$. Let $\{A_n \text{ i.o.}\} = \limsup A_n$ denote the event that infinitely many of the $A_n$ occur.

\textbf{Borel–Cantelli Lemma.}

(a) If $\sum_{n=1}^{\infty} P(A_n) < \infty$, then $P(A_n \text{ i.o.}) = 0$.

(b) If the events $A_1, A_2, \ldots$ are independent and $\sum_{n=1}^{\infty} P(A_n) = \infty$, then $P(A_n \text{ i.o.}) = 1$.
