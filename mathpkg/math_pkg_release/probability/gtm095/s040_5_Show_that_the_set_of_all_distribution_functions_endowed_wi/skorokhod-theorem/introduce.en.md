The Skorokhod representation theorem establishes a deep connection between weak convergence (convergence in distribution) of probability measures and almost sure convergence of random elements. It asserts that whenever a sequence of random elements $X_n$ converges to $X$ in distribution, one can construct — on a single, common probability space — a new sequence $X^*_n$ and a limit $X^*$ that have exactly the same individual distributions as the originals, but now converge almost surely. This "method of a single probability space" is a fundamental coupling technique that allows one to replace weak convergence arguments with pathwise almost-sure arguments. For the case $E = \mathbb{R}$, the construction uses quantile functions on the unit interval with Lebesgue measure.
