Let $F = F(x)$ be a distribution function on $\mathbb{R}$ with characteristic function $\varphi(t) = \int_{-\infty}^{\infty} e^{itx} \, dF(x)$. Then for any $a > 0$,
$$\int_{|x| \geq 1/a} dF(x) \leq K \cdot \frac{1}{a} \int_{0}^{a} \bigl[ 1 - \operatorname{Re} \varphi(t) \bigr] \, dt,$$
where
$$\frac{1}{K} = \inf_{|y| \geq 1} \left( 1 - \frac{\sin y}{y} \right) = 1 - \sin 1 \geq \frac{1}{7},$$
so that the inequality holds with $K = 7$.
