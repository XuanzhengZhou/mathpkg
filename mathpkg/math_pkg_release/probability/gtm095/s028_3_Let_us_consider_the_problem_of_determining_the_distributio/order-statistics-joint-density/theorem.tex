Let $X_1, \ldots, X_n$ be independent identically distributed random variables with density $f(x)$ and distribution function $F(x)$. Let $X^{(1)} \leq X^{(2)} \leq \cdots \leq X^{(n)}$ be the order statistics. Then:

(a) The density of the $k$th order statistic $X^{(k)}$ is

$$f_{X^{(k)}}(x) = n f(x) C_{n-1}^{k-1} [F(x)]^{k-1} [1 - F(x)]^{n-k}.$$

(b) The joint density of all order statistics is

$$f(x^1, \ldots, x^n) = \begin{cases} n! \, f(x^1) \cdots f(x^n), & x^1 < x^2 < \cdots < x^n, \\ 0, & \text{otherwise}. \end{cases}$$
