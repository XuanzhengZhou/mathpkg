Show that the Green function $G(x,y)$ determines the random walk (i.e., two transient random walks with the same Green function must have the same transition function). (Hint: Show that $G(x,y)$ determines $\Pi_A(x,y)$, and then let the set $A$ "grow very large" so that $\Pi_A(x,y) \to P(x,y)$.)
