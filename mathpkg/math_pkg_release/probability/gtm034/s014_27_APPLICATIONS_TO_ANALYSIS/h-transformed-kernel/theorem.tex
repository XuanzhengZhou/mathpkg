Let $Q$ be a transient kernel on $S$ and let $h(x)$ be a positive $Q$-regular function on $S$.

\textbf{D4} The $h$-transformed kernel $Q^h$ and $h$-transformed Green function $g^h$ are defined by
$$Q^h(x,y) = \frac{Q(x,y)h(y)}{h(x)}, \quad x,y \in S,$$
$$g^h(x,y) = \frac{g(x,y)h(y)}{h(x)}, \quad x,y \in S.$$
