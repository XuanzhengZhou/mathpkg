This proposition confirms that the $h$-transform is well-behaved: $Q^h$ remains a transient kernel whose iterates and Green function are consistent with the transformed expressions from D4. The crucial consequence is that under $Q^h$, the constant function $e(x) \equiv 1$ becomes regular. This reduction is the key technique in the proof of P3, where an arbitrary regular function $h$ transforms the kernel so that the constant function replaces $h$ as the object of study. The proof is straightforward from the definitions and is omitted in the text.
