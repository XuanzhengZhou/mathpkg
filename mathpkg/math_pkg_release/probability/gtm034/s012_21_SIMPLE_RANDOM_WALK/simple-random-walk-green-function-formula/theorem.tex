For simple symmetric random walk on $[0,N]$, the Green function and hitting probabilities are given by the exact formulas
$$\frac{1}{2(N+2)} g_N(x,y) = R\left(\frac{x+1}{N+2}, \frac{y+1}{N+2}\right),$$
$$R_N(x) = \frac{x+1}{N+2}, \quad L_N(x) = R_N(N-x) = \frac{N-x+1}{N+2},$$
where $R(x,y) = \min(x,y) - xy$ is the kernel defined in the boundary value problem equivalence.
