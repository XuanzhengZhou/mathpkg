This theorem provides an alternative and elegant formulation of the exit time limit theorem. Instead of describing the distribution of the exit time directly, it describes the probability that the maximum absolute value of the first $n$ partial sums stays below a threshold $\sigma x \sqrt{n}$. The equivalence follows from the observation that the event $\max_{1 \leq k \leq n} |\mathbf{S}_k| \leq \sigma x \sqrt{n}$ is the same as $\mathbf{T}_{[\sigma x \sqrt{n}]} > n$. The original proof by Erdős and Kac (1946) used an invariance principle to reduce the problem to the corresponding result for Brownian motion, known since Bachelier.
