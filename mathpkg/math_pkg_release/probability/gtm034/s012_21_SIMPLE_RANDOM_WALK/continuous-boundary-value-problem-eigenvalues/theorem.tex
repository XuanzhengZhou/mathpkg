The boundary value problem
\[
\frac{d^2}{dx^2}f(x) + \mu f(x) = 0, \quad 0 \leq x \leq 1, \quad f(0) = f(1) = 0,
\]
is equivalent to the integral equation
\[
f(x) = \mu \int_0^1 R(x,y) f(y) \, dy, \quad 0 \leq x \leq 1,
\]
with symmetric kernel \(R(x,y) = \min(x,y) - xy\). The eigenvalues are \(\mu_k = \pi^2 k^2\), \(k = 1,2,\dots\), and the eigenfunctions \(\phi_k(x) = \sqrt{2}\sin(\pi k x)\) form an orthonormal set.
