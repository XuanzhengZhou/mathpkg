The moments of the distribution $F(x)$ with density $f(x) = F'(x)$ are
$$m_p = \int_0^\infty x^p f(x) \, dx = p! \frac{\pi}{2} \left( \frac{8}{\pi^2} \right)^{p+1} \sum_{k=0}^{\infty} (-1)^k \left( \frac{1}{2k+1} \right)^{2p+1}, \quad p \geq 0.$$

Moreover, if $G(x)$ is any distribution function such that $\int_0^\infty x^p \, dG(x) = m_p$ for all $p = 0, 1, 2, \ldots$, then $G(x) = F(x)$. This follows from the fact that $\sum_{k=0}^{\infty} m_k r^k / k! < \infty$ for all $r < \pi^2/8$.
