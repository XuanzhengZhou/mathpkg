$$x_n = X_1 + X_2 + \cdots + X_n, \quad n \geq 1$$

where the $X_i$ are independent complex random variables with $|X_i| = 1$, whose arguments are equidistributed between 0 and $2\pi$. Develop the requisite Fourier analysis for spherically symmetric functions (see [5], Ch. 9) to conclude that

for $n \geq 1, r > 0$. When $n \geq 2$ the right-hand side is a continuous function of $r$, and it follows that

$$P[|x_n| < 1] = P[|x_n| \leq 1] = 1/(n + 1).$$
