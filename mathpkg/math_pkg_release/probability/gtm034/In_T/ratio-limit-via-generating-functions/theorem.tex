For aperiodic recurrent random walk in one dimension with finite variance $\sigma^2 < \infty$, the ratio limit

$$\lim_{t \to 1} \frac{1 - E_0[t^{T_x}]}{1 - E_0[t^{T_0}]} = a(-x)$$

holds for all $x \neq 0$, which implies

$$\lim_{n \to \infty} \frac{P_x[T > n]}{P_0[T > n]} = a(x), \quad x \neq 0.$$
