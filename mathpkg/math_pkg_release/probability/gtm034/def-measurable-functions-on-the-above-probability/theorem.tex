Measurable functions on the above probability space are called random variables. In particular we denote the random variables $\omega_k$ by

$$\omega_k = X_k(\omega) = X_k, \quad k = 1, 2, \ldots,$$

and their sums by

$$S_0 = 0, \quad S_n = X_1 + \cdots + X_n, \quad n \geq 1.$$

If $f(\omega) = f$ is a random variable, we write for each $x \in R$

$$P[f = x] = P[\omega | f(\omega) = x]$$

and if $f(\omega) = f$ is in addition integrable, then

$$\int_{\Omega} f(\omega) dP(\omega) = E[f]$$

is called its expectation (expected value). Finally, when $A \in \mathcal{F}$, the symbol $E[f;A]$ will denote the expectation $E[f(\omega)\varphi_A(\omega)]$ where $\varphi_A(\omega) = 1$ for $\omega \in A$ and zero otherwise.
