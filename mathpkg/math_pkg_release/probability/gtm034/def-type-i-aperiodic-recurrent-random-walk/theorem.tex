Type I: Aperiodic recurrent random walk in one dimension, with $\sigma^2 = \sum x^2 P(0,x) = +\infty$;

Type II: Aperiodic recurrent random walk in one dimension, with $\sigma^2 = \sum x^2 P(0,x) < \infty$.

It will become apparent that, although to a certain extent type I and type II exhibit the same behavior, the difference between these two cases is sufficiently sharp to require different methods of proof, even when the final results are the same! Thus it will be expedient to treat the two types separately after a certain point.
