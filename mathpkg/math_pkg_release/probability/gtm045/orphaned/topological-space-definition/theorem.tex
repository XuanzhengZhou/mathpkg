A \textit{topological space} $(\Omega, \mathcal{O})$ is a space $\Omega$ together with a class $\mathcal{O}$ of subsets called \textit{open sets} satisfying:
\begin{itemize}
\item $\emptyset, \Omega \in \mathcal{O}$
\item Finite intersections and arbitrary unions of open sets are open.
\end{itemize}
A set is \textit{closed} if its complement is open. A \textit{neighborhood} of a point is any open set containing it.
