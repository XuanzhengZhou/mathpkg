For a triangular array of row-wise independent r.v.'s $\{X_{nk}: 1 \leq k \leq k_n\}$, the \textit{uniformly asymptotically negligible} (uan) condition is:

$$\max_{1 \leq k \leq k_n} P[|X_{nk}| \geq \epsilon] \to 0 \quad \text{for every } \epsilon > 0.$$

Equivalently, $\max_k |\varphi_{nk}(u) - 1| \to 0$ uniformly on bounded $u$-intervals.
