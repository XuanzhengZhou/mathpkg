For a triangular array $\{X_{nk}\}$ of row-wise independent r.v.'s with zero means and variances $\sigma_{nk}^2 = E X_{nk}^2$, set $s_n^2 = \sum_k \sigma_{nk}^2$. The \textit{Lindeberg condition} is: for every $\epsilon > 0$,

$$\frac{1}{s_n^2} \sum_k \int_{|x| \geq \epsilon s_n} x^2 \, dF_{nk}(x) \to 0 \quad \text{as } n \to \infty.$$
