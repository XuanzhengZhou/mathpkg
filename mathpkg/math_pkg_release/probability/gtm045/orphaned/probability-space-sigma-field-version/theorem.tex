A \textit{probability space} $(\Omega, \alpha, P)$ consists of:
\begin{itemize}
\item the sure event $\Omega$,
\item a (nonempty) $\sigma$-field $\alpha$ of events,
\item a probability $P$ on $\alpha$ (normed, nonnegative, $\sigma$-additive).
\end{itemize}

$P\emptyset = 0$, $PA \leq PB$ when $A \subset B$, $P(\bigcup A_j) \leq \sum PA_j$, and

$$P(\liminf A_n) \leq \liminf PA_n \leq \limsup PA_n \leq P(\limsup A_n).$$

If $\lim A_n$ exists, then $P(\lim A_n) = \lim PA_n$.
