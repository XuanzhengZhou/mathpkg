A linear combination $X = \sum_{j} x_j I_{A_j}$ of a countable number of indicators of disjoint events $A_j$ is an \textit{elementary random variable}. If $j$ varies over a finite set, then $X$ reduces to a simple random variable.

The expectation $EX$ is defined by $EX = \sum_{j} x_j PA_j$ when the right-hand side is absolutely convergent. Then $EX$ exists and is finite if and only if $E|X|$ exists and is finite.
