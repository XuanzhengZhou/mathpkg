A regular conditional probability given $\mathcal{B}$ is a function $Q(\omega, A)$ such that: (i) for each $\omega$, $Q(\omega, \cdot)$ is a probability measure on $\mathcal{A}$; (ii) for each $A \in \mathcal{A}$, $Q(\cdot, A)$ is $\mathcal{B}$-measurable; (iii) $Q(\omega, A) = P^{\mathcal{B}}A$ up to a $P_{\mathcal{B}}$-equivalence.
