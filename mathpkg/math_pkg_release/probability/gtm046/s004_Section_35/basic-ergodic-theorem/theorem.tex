Let $\{X_n\},\{Y_n\}$ be r.v. sequences with $Y_n>0$, $Y^n=\sum_{k=1}^n Y_k\uparrow\infty$, $X^n=\sum_{k=1}^n X_k$. If
\begin{itemize}
\item[(i)] $\int|X_{n+k}|/Y^n\to 0$ for every fixed $k$ as $n\to\infty$,
\item[(ii)] $\int Y_{n+k}/Y^n\to 0$ for every fixed $k$ as $n\to\infty$,
\end{itemize}
then $X^n/Y^n$ converges a.s. The limit is invariant. The theorem extends to $Z^n$ replacing $Y^n$ if $\liminf\int_C Y^n/Z^n=0\Rightarrow P(C)=0$ for invariant $C$.
