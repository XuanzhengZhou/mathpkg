Let $\{X_n\}$, $\{Y_n\}$ be sequences of r.v.'s with $Y_n>0$, $Y^n=\sum_{k=1}^n Y_k\uparrow\infty$. Set $X^n=\sum_{k=1}^n X_k$. Let $B^m=[\sup_{j\leq m} X^j/Y^j>b]$. If $Z^n>0$, then for every integer $n$ and invariant event $C$,
$$\sum_{k=1}^{n}\int_{B_k^m C}(X_k/Z^n-bY_k/Z^n)+\sum_{k=n+1}^{n+m}\int_C(X_k/Z^n-bY_k/Z^n)^+\geq 0,$$
provided the sum exists.
