Let $h \geq 0$ be a finite-valued normalized $P$-super-regular function. If $x$ is in $S^{h^*}$, then

(1) $K^h(\cdot, x)$ is normalized if and only if $K(\cdot, x)$ is normalized.

(2) $K^h(\cdot, x)$ is regular for $P^h$ restricted to $S^h$ if and only if $K(\cdot, x)$ is $P$-regular.

(3) $K^h(\cdot, x)$ is minimal for $P^h$ restricted to $S^h$ if and only if $K(\cdot, x)$ is minimal for $P$.

Hence $B_e^h = B^h \cap B_e$ and $\overline{S^h} \subset \bar{S}$.
