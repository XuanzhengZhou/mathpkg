Let $z$ be a nonnegative random variable, and define $z^{(1)}$ as the shift of $z$ by one step. Let
$$h_i = M_i[z], \quad f_i = M_i[z - z^{(1)}],$$
with $z^{(n)}$ defined recursively. Then $z \geq z^{(1)} \geq z^{(2)} \geq \cdots \geq 0$, so $z^{(\infty)} = \lim_n z^{(n)}$ exists and is finite.

The function $h$ satisfies $h = Nf$ if and only if $z^{(\infty)} = 0$ almost everywhere.
