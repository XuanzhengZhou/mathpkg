If $P$ is a transient chain whose states communicate and if $\alpha > 0$ is superregular, show that
\begin{enumerate}
\item[(a)] $\hat{P}$ stops (disappears) a.e. if and only if $\alpha$ is a potential measure.
\item[(b)] The mean stopping time is finite if and only if $\alpha$ is a charge.
\end{enumerate}

\textit{Hint:} Adjoin an absorbing state to $\hat{P}$.
