The second condition is just the first for $E$, the set of all transient states. For the first condition, on every path which does not pass through $E$, $z(\omega) = 0$ by hypothesis. Hence $z^{(\infty)}(\omega) = 0$ for all $\omega$ not passing through $E$. For paths that do pass through $E$, since $E$ is an equilibrium set, the process eventually leaves $E$ and never returns; thus after sufficiently many shifts the path no longer passes through $E$, so $z^{(n)}(\omega) \to 0$ on these paths as well. Therefore $z^{(\infty)} = 0$ almost everywhere, and by Lemma 8-63, $h = Nf$.
