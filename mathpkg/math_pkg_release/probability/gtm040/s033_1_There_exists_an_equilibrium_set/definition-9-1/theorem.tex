Let $P$ be a recurrent chain which is either null or noncyclic ergodic, with regular measure $\alpha$.

\begin{itemize}
\item If $\mu$ is a signed measure with $\mu \mathbf{1}$ finite and if
$$\nu = \lim_n [\mu(I + P + \cdots + P^{n-1})]$$
exists and is finite-valued, then $\mu$ is called a \emph{left charge} with \emph{potential measure} $\nu$ and \emph{total charge} $\mu \mathbf{1}$.

\item If $f$ is a function with $\alpha f$ finite and if
$$g = \lim_n [(I + P + \cdots + P^{n-1})f]$$
exists and is finite-valued, then $f$ is called a \emph{right charge} with \emph{potential function} $g$ and \emph{total charge} $\alpha f$.
\end{itemize}

The \emph{support} of a charge is the set on which the charge is not zero; the support of a potential is the support of its charge.
