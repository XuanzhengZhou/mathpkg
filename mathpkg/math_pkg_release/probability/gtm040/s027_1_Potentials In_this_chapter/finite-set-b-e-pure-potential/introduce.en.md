Lemma 8-22 states that balayage onto a finite set $E$ transforms any non-negative superregular function $h$ into a pure potential with support contained in the finite set $E$. This follows because $B^E h$ is a finite linear combination of columns of $B^E$, each of which is a potential (by Proposition 8-20), and the non-negative superregular property is preserved. This lemma is key for approximating general potentials by finite-support potentials.
