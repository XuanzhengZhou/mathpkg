A column vector $h$ indexed by the countable state space $S$ is called continuous if it admits an extension to a continuous function on the entire Martin compactification $*S$. The extension is necessarily unique since $S$ is dense in $*S$. This definition is central to the boundary theory of denumerable Markov chains, as it allows algebraic operations on column vectors to be interpreted through continuous functions on the compactified state space.
