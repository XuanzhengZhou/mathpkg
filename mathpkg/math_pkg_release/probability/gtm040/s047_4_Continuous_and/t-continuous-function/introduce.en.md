A $T$-continuous function is a continuous function $h$ on the Martin compactification $*S$ that is invariant under the linear transformation $T$, defined by integrating against the boundary measure $\beta^x$. The operator $T$ maps continuous functions to bounded functions on $*S$, and the $T$-continuous functions form the core class for which the second convergence theorem (Theorem 11-13) applies. When every boundary point is extreme, every continuous function is automatically $T$-continuous.
