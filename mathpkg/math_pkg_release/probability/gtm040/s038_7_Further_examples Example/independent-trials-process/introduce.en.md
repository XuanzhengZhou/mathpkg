An independent trials process is a simple denumerable Markov chain in which the transition probability from any state $i$ to state $j$ depends only on the destination state $j$, not on the origin. Each row of the transition matrix $P$ is identical, with $P_{ij} = p_j$ where $\{p_j\}$ is a probability distribution over the state space. This is one of the simplest nontrivial Markov chain models, and it often serves as a basic example for illustrating chain properties such as idempotence of the transition matrix.
