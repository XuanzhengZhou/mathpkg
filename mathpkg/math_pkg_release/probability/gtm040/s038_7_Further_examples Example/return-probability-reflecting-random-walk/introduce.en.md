This proposition gives the exact distribution of the return time to state $0$ for the reflecting $p$-$q$ random walk on the nonnegative integers. From state $0$, the chain returns in one step with probability $q$ (by stepping to the left, which reflects back to $0$). Longer returns take an even number of steps: a path returning in $2n$ steps must have $n$ right steps and $n$ left steps, and the number of such paths is given by the Catalan number $\frac{1}{n}\binom{2n-2}{n-1}$, each occurring with probability $(pq)^n$.
