For the reflecting $p$-$q$ random walk on the nonnegative integers with $p = q = \frac{1}{2}$, the potential matrix $G$ is given by:

$$G_{ij} = \begin{cases} 2(j-i) & \text{if } j > i, \\ 0 & \text{if } j \leq i. \end{cases}$$

More generally, $iN_{jj} = 2j$ for $j > 0$, and $iN_{jj} = 2(j-i)$ for $j > i$.
