When the reflecting random walk is recurrent ($p \leq \frac{1}{2}$), the $r$-th moment of the return time $\bar{t}_0$ can be expressed as an infinite series involving the return probabilities. The formula combines the one-step return probability $q$ with an infinite sum over even return times $2n$, weighted by $(2n)^r$ times the Catalan-weighted path count. The convergence of this series depends on the value of $4pq$, which determines whether the chain is null or positive recurrent.
