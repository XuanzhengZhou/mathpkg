For subsets $A$, $B$, and $C$ of $X$, the symmetric difference satisfies:
$$A \oplus B = B \oplus A,$$
$$A \oplus A = \varnothing,$$
and
$$A \oplus B \subset (A \oplus C) \cup (B \oplus C).$$
Consequently, if $d(A,B) = \mu(A \oplus B)$ for the outer measure $\mu$, then $d$ is a metric on the class of sets modulo sets of $\mu$-measure zero.
