Let $\rho$ be a set function defined on a field of sets $\mathcal{F}$ taking values in the extended real numbers. The set function $\rho$ is said to be \textit{additive} if $\rho(A \cup B) = \rho(A) + \rho(B)$ whenever $A$ and $B$ are disjoint sets of $\mathcal{F}$ such that $A \cup B$ is in $\mathcal{F}$.
