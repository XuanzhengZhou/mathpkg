A measurable function is one for which the preimage of every interval $(-\infty, c)$ is a measurable set. This definition is the foundation of Lebesgue integration theory. The text notes that the condition $f(x) < c$ can be replaced by $f(x) \leq c$, $f(x) > c$, or $f(x) \geq c$ with equivalent effect, so that sets of the form $\{x \mid c \leq f(x) \leq d\}$ are always measurable for measurable functions.
