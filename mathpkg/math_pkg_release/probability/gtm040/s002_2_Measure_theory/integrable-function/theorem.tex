If $f$ is an arbitrary measurable function with positive part $f^+$ and negative part $f^-$, and if the integrals of $f^+$ and $f^-$ are not both infinite, the integral of $f$ is defined by
$$\int_E f \, d\mu = \int_E f^+ \, d\mu - \int_E f^- \, d\mu.$$
The function $f$ is said to be \textit{integrable} on the set $E$ if $\int_E f^+ \, d\mu$ and $\int_E f^- \, d\mu$ are both finite.
