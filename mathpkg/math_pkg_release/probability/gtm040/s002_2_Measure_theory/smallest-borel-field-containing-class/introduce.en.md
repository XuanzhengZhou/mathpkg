Given any collection $\mathcal{C}$ of subsets of $X$, one can always form the smallest Borel field (sigma-algebra) that contains $\mathcal{C}$, by intersecting all Borel fields that contain $\mathcal{C}$. The existence is guaranteed because the power set of $X$ is itself a Borel field containing $\mathcal{C}$. This construction is fundamental for generating measure-theoretic structures from simpler collections.
