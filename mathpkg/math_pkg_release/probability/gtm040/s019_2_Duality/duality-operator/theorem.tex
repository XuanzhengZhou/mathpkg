Let $P$ be a transition matrix, and let $\alpha$ be a positive finite-valued superregular measure. Let $Y$ be any square matrix, let $\beta$ be any row vector, and let $f$ be any column vector all indexed by the set of states. Define $D$ to be a diagonal matrix whose $i$-th diagonal entry is $1/\alpha_i$. The duality operator is defined by
$$\text{dual } X = D X^T D^{-1},$$
where $X^T$ denotes the transpose of $X$. When $X$ is a row vector $\beta$, it is treated as a $1 \times n$ matrix; when $X$ is a column vector $f$, it is treated as an $n \times 1$ matrix.
