A $\mathcal{B}_n$-measurable function is a random variable whose values depend only on the first $n+1$ outcomes of a stochastic process. On sequence space, this means the function's value is determined by a bounded number of coordinates. Every $\mathcal{B}_n$-measurable function is automatically a random variable because the Borel field $\mathcal{B}_n$ is contained in the full Borel field $\mathcal{B}$ of the process. This concept formalizes the intuitive notion of a quantity that can be determined by observing only finitely many steps of the process.
