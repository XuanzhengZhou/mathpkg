Let $P$ be a stochastic matrix and ${}^0N$ its fundamental matrix relative to a fixed reference state $0$. Proposition 9-18 asserts that if for each column $j$ the limit $\lim_n (P^n {}^0N)_{ij}$ exists for at least one row $i$, then the entire matrix limit $\lim_n (P^n {}^0N)$ exists and has constant columns (i.e., each column is a scalar multiple of the column vector of all ones). Moreover, these constant column values are finite. This result is a key technical building block for defining normal chains.
