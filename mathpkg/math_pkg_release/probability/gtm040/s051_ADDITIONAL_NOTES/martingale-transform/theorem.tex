Let $(f_n, \mathcal{F}_n)$ be a martingale and let $d_n$ be the sequence of differences defined by $d_0 = f_0$, $d_n = f_n - f_{n-1}$ for $n \geq 1$, so that $f_n = d_0 + \cdots + d_n$. If $v_n$ is measurable with respect to $\mathcal{F}_{n-1}$, then the sequence $g_n$ defined by

$$g_n = \sum_{k=0}^{n} v_k d_k$$

is called a \textbf{transform} of $f_n$. It is a martingale if $M[|g_n|] < \infty$ for all $n$.
