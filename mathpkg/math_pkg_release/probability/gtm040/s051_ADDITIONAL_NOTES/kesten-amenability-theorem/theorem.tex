Let $P$ be a symmetric transition matrix on a countable group, viewed as a linear operator on square-summable sequences. The group admits such a $P$ with spectral radius equal to $1$ if and only if the group is amenable; i.e., there exists a non-zero left-invariant positive linear functional on the bounded functions on the group.

If the spectral radius is less than $1$, then the fundamental matrix $N$ not only has finite entries but also is a bounded operator on square-summable sequences. Consequently, recurrence implies spectral radius $1$.
