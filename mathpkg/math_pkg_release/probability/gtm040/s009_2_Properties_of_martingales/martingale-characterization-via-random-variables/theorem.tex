Let $f_0, f_1, \ldots$ be a sequence of denumerable-valued random variables with $M[|f_j|]$ finite. Setting $\mathcal{R}_n = \mathcal{R}^{f_0} \otimes \cdots \otimes \mathcal{R}^{f_n}$, the sequence $\{f_n\}$ forms a martingale if and only if $$M[f_{n+1} \mid f_0 \wedge f_1 \wedge \cdots \wedge f_n] = f_n.$$
