Let $P$ be an arbitrary Markov chain. Then

(1) $h^E = B^E 1$.
(2) $h^E = e^E + Ph^E$ and hence $e^E = (I - P)h^E$.
(3) $s^E = 1$ if and only if $h^E = 1$ and $P 1 = 1$.
(4) If $E \subset F$, then $h^E \leq h^F$ and $s^E \leq s^F$.
(5) $s^E = B^Es^E$.
(6) If $E \subset F$, then $B^F B^E = B^E$.
(7) $s^E = \lim P^n h^E$.
