This proposition establishes the monotonicity of matrix multiplication with respect to entrywise inequalities. If a matrix inequality holds entrywise, multiplying both sides by a non-negative matrix preserves the inequality (for $\geq$), while multiplying by a positive matrix preserves strict inequality (for $>$), assuming finiteness of the products. These properties are essential for comparing matrices and establishing bounds in Markov chain theory.
