Let $V$ be a neighbor potential and let $\mathcal{G}_V$ denote the set of all Gibbs fields with potential $V$. When $|\mathcal{G}_V| = \infty$ we say that there is \emph{phase multiplicity} (or \emph{phase transition}) for $V$.

A measure $\mu \in \mathcal{G}_V$ is \emph{extreme} if whenever $\mu = \alpha \mu_1 + (1 - \alpha)\mu_2$ with $\mu_1, \mu_2 \in \mathcal{G}_V$ and $0 < \alpha < 1$, then $\mu_1 = \mu_2 = \mu$. The class of extreme elements of $\mathcal{G}_V$ is denoted by $\mathcal{E}_V$.
