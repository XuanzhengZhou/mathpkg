This proposition establishes that the restricted $K$-matrix $K_E$ is invertible precisely when the capacity $k(E)$ is positive. Moreover, it provides an explicit expression for the matrix that $K_E$ inverts, relating $P^E$, $l_E^E$, and $\lambda_E^E$. This result is the gateway to computing $P^E$ from $K_E$, which is much more practical for infinite chains than the $T + UNR$ formula.
