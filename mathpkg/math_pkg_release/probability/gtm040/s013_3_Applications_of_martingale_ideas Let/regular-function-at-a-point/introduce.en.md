A function $f$ is said to be regular at a point $j$ if its value at that state equals the $P$-weighted average of its values at all states reachable in one step, i.e., $f_j = (Pf)_j$. This is a pointwise version of the global $P$-regularity condition $f = Pf$. A function is $P$-regular if and only if it is regular at every point $j$ in the state space.
