Let $h$ be a function on the state space of a Markov chain with transition matrix $P$. If $h$ is superregular (i.e., $h \geq Ph$) and $\mathbf{M}[|h(x_n)|]$ is finite, then $(h(x_n), \mathcal{R}_n)$ is a supermartingale. If $h$ is subregular (i.e., $h \leq Ph$) and $\mathbf{M}[|h(x_n)|]$ is finite, then $(h(x_n), \mathcal{R}_n)$ is a submartingale.
