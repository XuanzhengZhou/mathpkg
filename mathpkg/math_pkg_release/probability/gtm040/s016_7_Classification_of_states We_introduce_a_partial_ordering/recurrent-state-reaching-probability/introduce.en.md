Lemma 4-23 is a key structural result: if a recurrent state $i$ can reach a state $j$ (i.e., $R(i,j)$), then the hitting probabilities are both 1 in both directions. This means that from a recurrent state, any reachable state is almost surely visited, and from that state, the original recurrent state is also almost surely revisited. This lemma is essential for proving that all states in an equivalence class share the same type (recurrent or transient), as it establishes the two-way almost-sure reachability within recurrent classes.
