Proposition 4-28 provides the complementary result to Proposition 4-27. While recurrent states are visited infinitely often, any finite collection of transient states is visited only finitely many times, almost surely. The proof uses a simple pigeonhole argument: if a finite set were visited infinitely often with positive probability, some state in that set would be visited infinitely often, which would imply infinite $N_{ij}$, contradicting Corollary 4-21 for transient states.
