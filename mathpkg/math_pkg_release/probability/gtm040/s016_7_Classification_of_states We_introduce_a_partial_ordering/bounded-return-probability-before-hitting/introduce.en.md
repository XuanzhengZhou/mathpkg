Lemma 4-22 states that if $j$ is reachable from $i$ (with $i \neq j$), then the probability of returning to $i$ before ever hitting $j$ is strictly less than 1. Equivalently, if the process starts at $i$, there is a positive probability of hitting $j$ before any return to $i$. This lemma is crucial for proving that recurrent classes are closed and maximal, since it precludes the possibility of a recurrent state being able to reach another state from which return is impossible.
