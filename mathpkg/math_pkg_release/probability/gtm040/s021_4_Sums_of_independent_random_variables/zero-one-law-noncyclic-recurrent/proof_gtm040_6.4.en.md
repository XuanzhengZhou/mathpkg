Form the ladder process $P^*$ of Proposition 6-32, which is a sums of independent random variables chain on the integers whose outcome functions are the return times to state $s$ in the original chain $P$.

We first show that for any two states $i$ and $j$ of $P^*$, there exists a state $k$ that can be reached from both $i$ and $j$. For this purpose it suffices to show that from state 0 it is possible to reach all sufficiently large states, since $P^*$ represents sums of independent random variables. Now the set of states reachable from 0 is precisely the additive semigroup generated by $\{k : p_k > 0\}$. Since $P$ is noncyclic, by Lemma 6-35 the greatest common divisor of this set is 1, and it is a standard number-theoretic fact that the semigroup generated by a set of integers with gcd 1 contains all sufficiently large integers.

It follows from the Hewitt-Savage zero-one law for sums of independent random variables that the probability that the $P^*$ process visits a given state infinitely often is either 0 or 1 and is independent of the starting distribution $\pi$.

Now if both
$$\Pr_{\pi}[x_n = s 	ext{ for infinitely many } n \in E] = 0$$
and
$$\Pr_{\pi}[x_n = s 	ext{ for infinitely many } n \in F] = 0,$$
then since $E \cup F = \mathbb{N}_0$, we would have
$$\Pr_{\pi}[x_n = s 	ext{ infinitely often}] = 0,$$
which contradicts the recurrence of $P$ (which guarantees $\Pr_{\pi}[x_n = s 	ext{ infinitely often}] = 1$). Therefore at least one of the two probabilities must be 1.
