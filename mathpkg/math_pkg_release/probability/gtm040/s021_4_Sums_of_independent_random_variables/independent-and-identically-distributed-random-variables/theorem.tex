The denumerable-valued random variables $y_1, y_2, y_3, \ldots$ defined on $\Omega$ are \textit{independent} if, for every finite collection of sets $E_1, E_2, \ldots, E_m$ of reals, it is true that

$$\Pr[y_{n_k}(\omega) \in E_k \text{ for } k = 1, \ldots, m] = \prod_{k=1}^{m} \Pr[y_{n_k}(\omega) \in E_k].$$

The random variables are \textit{identically distributed} if, for any $m$ and $n$ and for any set $E$ of reals, it is true that

$$\Pr[y_m(\omega) \in E] = \Pr[y_n(\omega) \in E].$$
