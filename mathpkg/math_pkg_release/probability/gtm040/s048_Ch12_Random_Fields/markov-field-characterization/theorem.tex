For a random field $\{x_t\}$ on a finite set $T$ with neighbor system $\partial$, the following are equivalent:
\begin{enumerate}
\item[(1)] $\{x_t\}$ is a Markov field.
\item[(2)] For every $a \in T$ and every finite $\Lambda \supset \bar{a}$,
$$\mu_a^\Lambda(\iota) = \mu_a^{\bar{a}}(\iota).$$
\item[(3)] For every $a \in T$ and all configurations $\iota, \iota'$ that agree on $\partial a$,
$$\mu_a^T(\iota) = \mu_a^T(\iota').$$
\end{enumerate}
