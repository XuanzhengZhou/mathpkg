The basic example is the central running example used throughout the potential theory of denumerable Markov chains. It is a birth-death-like process on the nonnegative integers where from each state $i$ the chain either moves to $i+1$ with probability $p_{i+1}$ or returns to $0$ with probability $q_{i+1}$. The parameters $\beta_i = \prod_{k=1}^i p_k$ completely determine the asymptotic behavior and recurrence properties of the chain.
