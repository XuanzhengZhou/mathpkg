The potential kernel of the null basic example has an elegant triangular form: the Green function $\hat{G}_{ij}$ equals 1 when $j > i$ and 0 otherwise, reflecting the deterministic leftward drift of the reverse chain. The potential kernel $C$ is lower-triangular with entries $\beta_j/\beta_i$, and remarkably coincides with the Green function $G$ in the null case. This explicit structure makes the null basic example an invaluable testing ground for general potential-theoretic results.
