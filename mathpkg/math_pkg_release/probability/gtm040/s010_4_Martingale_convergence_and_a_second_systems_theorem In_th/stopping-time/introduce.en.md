A stopping time is a random time that is almost surely finite. It serves as the key concept for the Optional Stopping Theorem (Second Systems Theorem): given a process $\{f_n\}$ and a stopping time $t$, one evaluates $f_t$ at the random time when the process stops. The evaluation $f_t$ is itself a random variable, defined by setting $f_t(\omega) = f_n(\omega)$ on the set where $t(\omega) = n$.
