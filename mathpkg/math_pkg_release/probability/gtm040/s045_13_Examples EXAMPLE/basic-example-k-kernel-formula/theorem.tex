For the basic example chain with $\pi$ chosen as a unit mass at $0$, the Martin kernel $K(i,j)$ is given by

$$K(i,j) = \frac{N_{ij}}{N_{0j}} = \begin{cases} 1 & \text{if } i \leq j \\[4pt] \dfrac{\beta_i - \beta_\infty}{\beta_i} & \text{if } i > j, \end{cases}$$

where $N_{ij} = \begin{cases} \frac{\beta_j}{\beta_\infty} & \text{if } i \leq j \\ \frac{\beta_j}{\beta_\infty} - \frac{\beta_j}{\beta_i} & \text{if } i > j \end{cases}$ and $\beta_i$, $\beta_\infty$ are as defined in Chapter 5.
