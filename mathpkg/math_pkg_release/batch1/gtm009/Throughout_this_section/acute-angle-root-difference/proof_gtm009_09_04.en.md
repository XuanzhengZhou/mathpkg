The second assertion follows from the first by replacing $\beta$ with $-\beta$. Since $(\alpha, \beta)$ is positive if and only if $\langle\alpha, \beta\rangle$ is positive, Table 1 shows that one of $\langle\alpha, \beta\rangle$, $\langle\beta, \alpha\rangle$ equals 1.

If $\langle\alpha, \beta\rangle = 1$, then $\sigma_\beta(\alpha) = \alpha - \langle\alpha, \beta\rangle \beta = \alpha - \beta \in \Phi$ by (R3).

If $\langle\beta, \alpha\rangle = 1$, then $\sigma_\alpha(\beta) = \beta - \alpha \in \Phi$, hence $\sigma_{\beta-\alpha}(\beta-\alpha) = \alpha - \beta \in \Phi$. $\square$
