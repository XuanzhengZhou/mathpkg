This proposition from Section 11.1 shows that the Cartan matrix (or equivalently, the Dynkin diagram) completely determines the root system up to isomorphism. This is the key fact that makes the classification of root systems via Dynkin diagrams possible. The proof constructs the isomorphism explicitly using the Weyl group action.
