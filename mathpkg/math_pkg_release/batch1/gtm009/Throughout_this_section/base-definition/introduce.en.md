A base of a root system is a subset that serves as both a vector space basis and a set of generators for all roots with integer coefficients of uniform sign. This concept is fundamental to the entire structure theory of root systems. The simple roots in a base allow one to define positive/negative roots, heights, and the fundamental Weyl chamber. Every root system has a base, and all bases are conjugate under the Weyl group.
