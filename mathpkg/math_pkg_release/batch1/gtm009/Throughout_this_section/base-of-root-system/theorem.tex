A subset $\Delta$ of a root system $\Phi$ is called a base if:
(B1) $\Delta$ is a basis of the euclidean space $E$,
(B2) Each root $\beta \in \Phi$ can be written as $\beta = \sum_{\alpha \in \Delta} k_\alpha \alpha$ with integral coefficients $k_\alpha$ all nonnegative or all nonpositive.
Roots in $\Delta$ are called simple. If all $k_\alpha \geq 0$ (resp. $\leq 0$), $\beta$ is called positive (resp. negative). The height of $\beta$ is $\operatorname{ht}\beta = \sum k_\alpha$.
