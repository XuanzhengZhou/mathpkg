The map $\text{ad}: L \rightarrow \mathfrak{gl}(L)$ defined by $x \mapsto \text{ad } x$, where
$$\text{ad } x(y) = [xy],$$
is called the \text{adjoint representation} of $L$. It is a homomorphism of Lie algebras:
$$[\text{ad } x, \text{ad } y] = \text{ad }[xy].$$
The kernel of $\text{ad}$ is the center $Z(L) = \{x \in L \mid [xy] = 0 \text{ for all } y \in L\}$.
