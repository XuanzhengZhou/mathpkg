For $v \in V_\lambda$, compute using the commutation relations $[h, x] = 2x$ and $[h, y] = -2y$:

$$h.(x.v) = [h, x].v + x.h.v = 2x.v + x.(\lambda v) = (2 + \lambda)x.v,$$

so $x.v \in V_{\lambda+2}$. Similarly,

$$h.(y.v) = [h, y].v + y.h.v = -2y.v + y.(\lambda v) = (\lambda - 2)y.v,$$

so $y.v \in V_{\lambda-2}$.
