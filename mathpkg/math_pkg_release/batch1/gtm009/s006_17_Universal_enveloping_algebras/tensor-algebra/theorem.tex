Let $V$ be a finite dimensional vector space over $F$. Set $T^0V = F$, $T^1V = V$, $T^2V = V \otimes V$, \dots, $T^mV = V \otimes \dots \otimes V$ ($m$ copies). Define
$$
\mathfrak{T}(V) = \prod_{i=0}^{\infty} T^iV,
$$
with associative product defined on homogeneous generators by
$$
(v_1 \otimes \dots \otimes v_k)(w_1 \otimes \dots \otimes w_m) = v_1 \otimes \dots \otimes v_k \otimes w_1 \otimes \dots \otimes w_m \in T^{k+m}V.
$$
Then $\mathfrak{T}(V)$ is an associative graded algebra with $1$, generated by $1$ along with any basis of $V$. It is called the \text{tensor algebra} on $V$.
