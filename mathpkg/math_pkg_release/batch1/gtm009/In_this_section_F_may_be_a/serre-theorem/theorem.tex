Let $\Phi$ be a root system with fixed base $\Delta = \{\alpha_1, \ldots, \alpha_\ell\}$ and Cartan matrix $(c_{ij}) = (\langle \alpha_j, \alpha_i \rangle)$. Let $L$ be the Lie algebra with generators $\{x_i, y_i, h_i \mid 1 \leq i \leq \ell\}$ and relations:
\begin{align*}
\text{(S1)} &\quad [h_i h_j] = 0;\\
\text{(S2)} &\quad [x_i y_i] = h_i, \quad [x_i y_j] = 0 \; (i \neq j);\\
\text{(S3)} &\quad [h_i x_j] = c_{ij} x_j, \quad [h_i y_j] = -c_{ij} y_j;\\
\text{(S}_{ij}^+) &\quad (\operatorname{ad} x_i)^{-c_{ji}+1}(x_j) = 0 \quad (i \neq j);\\
\text{(S}_{ij}^-) &\quad (\operatorname{ad} y_i)^{-c_{ji}+1}(y_j) = 0 \quad (i \neq j).
\end{align*}
Then $L$ is a finite-dimensional semisimple Lie algebra with Cartan subalgebra $H = \sum F h_i$ and root system $\Phi$.
