If $L$ is any Lie algebra, and $x \in L$, we call $x$ \text{ad-nilpotent} if $\operatorname{ad} x$ is a nilpotent endomorphism of $L$. The condition for $L$ to be nilpotent can be rephrased as: For some $n$, $(\operatorname{ad} x_1)(\operatorname{ad} x_2) \cdots (\operatorname{ad} x_n)(y) = 0$ for all $x_i, y \in L$. In particular, $(\operatorname{ad} x)^n = 0$ for all $x \in L$.
