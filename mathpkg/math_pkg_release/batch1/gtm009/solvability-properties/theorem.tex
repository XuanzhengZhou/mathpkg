Let $L$ be a Lie algebra.

\begin{enumerate}
\item[(a)] If $L$ is solvable, then so are all subalgebras and homomorphic images of $L$.
\item[(b)] If $I$ is a solvable ideal of $L$ such that $L/I$ is solvable, then $L$ itself is solvable.
\item[(c)] If $I$, $J$ are solvable ideals of $L$, then so is $I+J$.
\end{enumerate}
