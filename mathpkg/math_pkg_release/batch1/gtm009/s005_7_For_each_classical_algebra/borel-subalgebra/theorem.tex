A \textbf{Borel subalgebra} of a Lie algebra $L$ is a maximal solvable subalgebra of $L$.

For a semisimple Lie algebra $L$ with Cartan subalgebra $H$ and root system $\Phi$, fix a base $\Delta$ and a set of positive roots. The \textbf{standard Borel subalgebra} (relative to $H$ and $\Delta$) is:

$$B(\Delta) = H + \sum_{\alpha > 0} L_{\alpha}$$

with nilpotent radical $N(\Delta) = \sum_{\alpha > 0} L_{\alpha}$. Then $B(\Delta)$ is a Borel subalgebra of $L$, and its derived algebra is $N(\Delta)$.
