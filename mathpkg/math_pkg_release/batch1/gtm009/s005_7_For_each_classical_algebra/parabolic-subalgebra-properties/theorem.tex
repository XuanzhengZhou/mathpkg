Let $L$ be a semisimple Lie algebra with Cartan subalgebra $H$, root system $\Phi$, and fixed base $\Delta$. Set $B = B(\Delta)$. For each subset $\Delta' \subset \Delta$, define $P(\Delta')$ to be the subalgebra of $L$ generated by all $L_\alpha$ ($\alpha \in \Delta$ or $-\alpha \in \Delta'$), along with $H$.

\begin{enumerate}
  \item[(a)] $P(\Delta')$ is a parabolic subalgebra of $L$ (called \textbf{standard} relative to $\Delta$).
  \item[(b)] Every parabolic subalgebra of $L$ containing $B(\Delta)$ has the form $P(\Delta')$ for some $\Delta' \subset \Delta$.
  \item[(c)] Every parabolic subalgebra of $L$ is conjugate under $\mathcal{E}(L)$ to one of the $P(\Delta')$.
\end{enumerate}
