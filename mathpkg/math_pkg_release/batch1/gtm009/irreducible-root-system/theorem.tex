A root system $\Phi$ is called \text{irreducible} if it cannot be decomposed as $\Phi = \Phi_1 \cup \Phi_2$ with $\Phi_1 \perp \Phi_2$ (i.e., $(\alpha, \beta) = 0$ for all $\alpha \in \Phi_1, \beta \in \Phi_2$). Otherwise $\Phi$ is \text{reducible}.

Every root system decomposes uniquely as a disjoint union of irreducible root systems (its irreducible components). The irreducible root systems correspond precisely to the simple Lie algebras.
