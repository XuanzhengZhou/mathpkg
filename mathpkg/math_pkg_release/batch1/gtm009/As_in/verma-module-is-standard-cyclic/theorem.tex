For each $\lambda \in H^*$, the Verma module $Z(\lambda) = \mathfrak{U}(L) \otimes_{\mathfrak{U}(B)} D_\lambda$ is a standard cyclic module of weight $\lambda$. Moreover, as a $\mathfrak{U}(N^-)$-module (where $N^- = \bigoplus_{\alpha \prec 0} L_\alpha$), $Z(\lambda)$ is isomorphic to $\mathfrak{U}(N^-)$ itself.
