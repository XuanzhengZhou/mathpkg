Let $L$ be a Lie algebra. For each $x \in L$, the map $\text{ad } x: L \rightarrow L$ defined by
$$y \mapsto [xy]$$
is an endomorphism of $L$ and satisfies $\text{ad } x \in \text{Der } L$, since the Jacobi identity can be rewritten as
$$[x[yz]] = [[xy]z] + [y[xz]].$$
Derivations of this form are called \text{inner derivations}. All other derivations are called \text{outer derivations}.

The map $L \rightarrow \text{Der } L$ sending $x$ to $\text{ad } x$ is called the \text{adjoint representation} of $L$.
