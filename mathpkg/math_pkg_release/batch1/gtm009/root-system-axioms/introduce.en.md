A root system is an abstract finite set of vectors in a Euclidean space satisfying four axioms: it spans the space, is closed only under sign reversal (not scaling), is invariant under reflections across hyperplanes orthogonal to its elements, and has integrality of Cartan integers. Root systems are completely classified by Dynkin diagrams, yielding the types $A_\ell, B_\ell, C_\ell, D_\ell, E_6, E_7, E_8, F_4, G_2$. The Weyl group generated by reflections is a finite Coxeter group.
