The proposition **If $G = \{ \langle x, y \rangle | [x = 0 \land y = a] \lor [x \neq 0 \land \sup(\mathcal{D}(x)) \neq \mathcal{D}(x) \land y = H'x' \sup(\mathcal{D}(x))] \lor [x \neq 0 \land \sup(\mathcal{D}(x)) = \mathcal{D}(x) \land y = \cup \mathcal{W}(x)]\}$$** (Proposition 7.44.) is a fundamental notion in axiomatic set theory, introduced in Chapter 7 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
