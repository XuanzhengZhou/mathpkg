Proposition 5.21 shows that the universal class $V$ is not a set, i.e., $V$ is a proper class. The proof is by contradiction: if $V$ were a set, then since $V = V$, we would have $V \in V$, but Corollary 5.19 (derived from Regularity) asserts that no set contains itself.
