Since $V = V$ is trivially true, if $V$ were a set, then by definition of $V$ (as the class of all $x$ such that $x = x$), we would have $V \in V$. But Corollary 5.19 asserts that $a \notin a$ for every set $a$. Taking $a = V$ yields $V \notin V$, a contradiction. Therefore $V$ is not a set, i.e., $\Pr(V)$.
