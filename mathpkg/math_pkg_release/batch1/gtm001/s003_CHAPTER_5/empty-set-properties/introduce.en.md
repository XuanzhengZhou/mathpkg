Proposition 5.17 establishes two fundamental properties of the empty set. First, no individual belongs to $0$: $\forall x (x \notin 0)$. Second, a set $a$ is nonempty if and only if there exists some $x$ belonging to $a$. Together these characterize $0$ as the unique set with no elements.
