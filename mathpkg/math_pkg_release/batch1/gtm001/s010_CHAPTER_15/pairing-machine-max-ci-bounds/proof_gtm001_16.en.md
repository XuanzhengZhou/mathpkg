**Proof.** (1) $f(\alpha) = J''\alpha^\omega$ is increasing, so $\alpha \leq J''\alpha^\omega$. If $\alpha_i = \max(\boldsymbol{\alpha})$, then $\alpha_i \leq J''\alpha_i^\omega \leq J(\boldsymbol{\alpha})$. (2) $\alpha = J(\langle \alpha_0, \ldots \rangle)$ gives $\max(\alpha_i) \leq \alpha$, so $C_i(\langle \alpha \rangle) = \alpha_i \leq \alpha$.
