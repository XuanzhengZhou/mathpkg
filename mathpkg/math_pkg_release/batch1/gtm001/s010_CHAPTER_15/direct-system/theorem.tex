Let $\langle I, \leq \rangle$ be a directed set. $\Pi = \langle \eta_i, \pi_{ij} \rangle_{i \leq j}$ is a direct system iff
\begin{enumerate}
\item $\pi_{ij} : \eta_i \rightarrow \eta_j$ is order preserving for $i \leq j$,
\item $\pi_{ii}$ is the identity,
\item $\pi_{jk} \circ \pi_{ij} = \pi_{ik}$ if $i \leq j \leq k$.
\end{enumerate}
