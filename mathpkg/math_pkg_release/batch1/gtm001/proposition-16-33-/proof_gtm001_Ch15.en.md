(1) Since the function $f(\alpha) = J“\alpha^@$ is increasing, it follows that $\alpha \leq J“\alpha^@$ for all $\alpha$. Thus if $\alpha_i = \max(\alpha)$, then $\alpha_i \leq J“\alpha_i^@ \leq J(\alpha)$.

(2) Let $\alpha = J(\langle \alpha_0, \ldots, \alpha_{n-1} \rangle)$. Then by (1), $\max(\alpha_0, \ldots, \alpha_{n-1}) \leq \alpha$ and so $C_i(\langle \alpha \rangle) = \alpha_i \leq \alpha$ for each $i < n$.
