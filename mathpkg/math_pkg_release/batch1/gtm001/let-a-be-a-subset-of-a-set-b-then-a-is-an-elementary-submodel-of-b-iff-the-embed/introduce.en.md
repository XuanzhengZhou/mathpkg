The definition **Let $A$ be a subset of a set $B$. Then $A$ is an elementary submodel of $B$ iff the embedding map of $A$ into $B$ is an elementary embedding. We write $A \prec B$ which is read “$A$ is an elementary submodel of $B$.”** (Definition 17.25.) is a fundamental notion in axiomatic set theory, introduced in Chapter 17 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
