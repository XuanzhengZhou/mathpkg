To formalize the forcing language within the ground model $M$, each symbol of $\mathcal{L}(M, \mathbf{G})$ is assigned a code in $M$ -- typically an ordered pair encoding the symbol type and its data. Logical connectives receive fixed codes, while constants $\mathbf{k}$ for $k \in M$ are coded using the set $k$ itself. This coding allows formulas and terms to be represented as sets in $M$, enabling the definition of the forcing relation within $M$.
