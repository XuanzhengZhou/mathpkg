$\Pi = \langle \langle \delta_i, \alpha_i, P_i \rangle, \pi_{ij} \rangle_{i,j \in I}$ is a $\kappa$-direct limit system if:

(1) $\langle I, \leq \rangle$ is a directed partially ordered set,

(2) $\langle \delta_i, \pi_{ij} \rangle_{i \leq j}$ is a direct system of medium $M$-maps,

(3) if $i < j$, then $\pi_{ij} : \langle \delta_i, \alpha_i, P_i \rangle \rightarrow \langle \delta_j, \alpha_j, P_j \rangle$ is an acceptable map such that $\sup \{\pi_{ij}(v) + 1 \mid v < \delta_i \} < \delta_j$,

(4) $\delta_i < \kappa$, and

(5) $\{\alpha_i \mid i \in I\}$ is cofinal in $\kappa$.
