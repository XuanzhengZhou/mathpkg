This theorem establishes that if every set is constructible ($V = L$), then the Generalized Continuum Hypothesis holds. The proof shows that $\mathcal{P}(\aleph_\alpha) \subseteq L_{\aleph_\alpha + 1}$ for every ordinal $\alpha$, using the collapsing property of the $L$-machine $M$. Given any subset $a \subseteq \aleph_\alpha$, one constructs a term whose denotation equals $a$ and whose rank is bounded by $\aleph_\alpha + 1$, thereby bounding the power set.
