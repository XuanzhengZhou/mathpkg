The theorem **If $| \vdash \varphi$ and if $A$ is a nonempty class that satisfies each nonlogical axiom that occurs in some proof of $\varphi$ then** (Theorem 12.6.) is a fundamental notion in axiomatic set theory, introduced in Chapter 11 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
