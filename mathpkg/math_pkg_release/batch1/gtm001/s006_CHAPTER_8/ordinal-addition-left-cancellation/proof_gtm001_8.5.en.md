**Proof.** From Proposition 8.4, $\alpha < \beta \rightarrow \gamma + \alpha < \gamma + \beta$ and $\beta < \alpha \rightarrow \gamma + \beta < \gamma + \alpha$. Therefore $\gamma + \alpha = \gamma + \beta$ implies $\alpha \not< \beta$ and $\beta \not< \alpha$, so by trichotomy of ordinals, $\alpha = \beta$. The converse is trivial: if $\alpha = \beta$, then clearly $\gamma + \alpha = \gamma + \beta$.
