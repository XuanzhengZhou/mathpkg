(By transfinite induction on $\gamma$). If $\alpha \le \beta$, then $\alpha + 0 \le \beta + 0$. If $\alpha + \gamma \le \beta + \gamma$ then $\alpha + (\gamma + 1) \le \beta + (\gamma + 1)$. If $\gamma \in K_{II}$ and $(\forall \delta < \gamma)[\alpha + \delta \le \beta + \delta]$ then $\alpha + \gamma = \bigcup_{\delta < \gamma} (\alpha + \delta) \le \bigcup_{\delta < \gamma} (\beta + \delta) = \beta + \gamma.$
