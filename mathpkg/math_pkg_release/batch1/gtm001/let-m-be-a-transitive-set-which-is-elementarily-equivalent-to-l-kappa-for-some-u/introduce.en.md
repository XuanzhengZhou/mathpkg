The lemma **Let $M$ be a transitive set which is elementarily equivalent to $L_\kappa$ for some uncountable cardinal in $L$. If $\pi: \langle \delta, \alpha, P \rangle \rightarrow \langle \delta', \alpha', P' \rangle$ is an acceptable map such that $\langle \delta, \alpha, P \rangle, \langle \delta', \alpha', P' \rangle \in M$, then $\pi \in M$** (Lemma 17.11.) is a fundamental notion in axiomatic set theory, introduced in Chapter 17 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
