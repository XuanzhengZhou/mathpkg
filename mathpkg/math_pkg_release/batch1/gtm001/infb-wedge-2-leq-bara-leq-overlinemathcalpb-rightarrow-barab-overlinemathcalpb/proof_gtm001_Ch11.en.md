By Proposition 10.44 $\overline{\mathcal{P}(b)} = 2^b$. Therefore if $\bar{a} \leq \overline{\mathcal{P}(b)}$, then $\bar{a} \leq 2^b$ and so

$$\bar{a}^b \leq (2^b)^b = 2^{b \times b} = 2^b = \overline{\mathcal{P}(b)}$$

i.e., $\bar{a}^b \leq \overline{\mathcal{P}(b)}$.

On the other hand if $2 \leq \bar{a}$, then $2^b \leq \bar{a}^b$. Therefore

$$\bar{a}^b = \overline{\mathcal{P}(b)}$$
