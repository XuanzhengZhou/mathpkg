The theorem **If $|\vdash [\varphi \leftrightarrow (\forall x)\psi(x)]$ and $|\vdash [\varphi \leftrightarrow (\exists x)\eta(x)]$, if $A$ is a non-empty class that satisfies each nonlogical axiom in some proof of $\varphi \leftrightarrow (\forall x)\psi(x)$ and some proof of $\varphi \leftrightarrow (\exists x)\eta(x)$, if $\psi(x)$ Abs $A$ and if $\eta(x)$ Abs $A$ then** (Theorem 13.8.) is a fundamental notion in axiomatic set theory, introduced in Chapter 11 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
