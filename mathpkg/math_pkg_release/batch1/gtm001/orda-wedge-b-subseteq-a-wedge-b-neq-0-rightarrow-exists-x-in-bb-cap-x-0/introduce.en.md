The proposition **Ord($A$) $\wedge B \subseteq A \wedge B \neq 0 \rightarrow (\exists x \in B)[B \cap x = 0]$** (Proposition 7.5.) is a fundamental notion in axiomatic set theory, introduced in Chapter 7 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
