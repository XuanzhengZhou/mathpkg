The proposition **$\alpha \beta = 0 \leftrightarrow \alpha = 0 \vee \beta = 0$** (Proposition 8.22.) is a fundamental notion in axiomatic set theory, introduced in Chapter 8 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
