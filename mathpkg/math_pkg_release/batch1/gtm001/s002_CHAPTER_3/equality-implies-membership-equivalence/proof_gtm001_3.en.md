\text{From Axiom 1: } a = b \land a \in c \rightarrow b \in c.
\text{From Proposition 3.2(2): } a = b \rightarrow b = a.
\text{Thus } b = a \land b \in c \rightarrow a \in c.
\text{Therefore } a = b \rightarrow [a \in c \leftrightarrow b \in c].
