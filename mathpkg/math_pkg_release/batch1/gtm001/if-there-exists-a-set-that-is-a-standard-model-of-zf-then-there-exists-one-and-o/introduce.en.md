The theorem **If there exists a set that is a standard model of ZF then there exists one and only one set $M_0$ such that** (Theorem 18.1.) is a fundamental notion in axiomatic set theory, introduced in Chapter 17 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
