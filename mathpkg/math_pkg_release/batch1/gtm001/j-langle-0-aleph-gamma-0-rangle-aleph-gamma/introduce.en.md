The proposition **$J' \langle 0, \aleph_\gamma, 0 \rangle = \aleph_\gamma.$** (Proposition 15.12.) is a fundamental notion in axiomatic set theory, introduced in Chapter 15 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
