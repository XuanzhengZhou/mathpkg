A universal element of a set-valued functor $H: C \to \mathbf{Set}$ is an element $e \in Hr$ from which every other element can be obtained uniquely by applying $H$ to some arrow. Universal elements provide an equivalent formulation of representable functors: $H$ is representable if and only if it has a universal element. This concept bridges the gap between universal arrows and the Yoneda lemma.
