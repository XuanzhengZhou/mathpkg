If $\langle r, u: c \rightarrow Sr\rangle$ is a universal arrow from $c$ to $S$, then the object $r$ is uniquely determined up to (unique) isomorphism in $D$. More precisely, if $\langle r', u': c \rightarrow Sr'\rangle$ is another such universal arrow, there is a unique isomorphism $\varphi: r \rightarrow r'$ such that $S\varphi \circ u = u'$.
