A \text{groupoid} is a category in which every arrow is invertible. A typical example is the \text{fundamental groupoid} $\pi(X)$ of a topological space $X$: an object of $\pi(X)$ is a point $x$ of $X$, and an arrow $x \rightarrow x'$ of $\pi(X)$ is a homotopy class of paths $f$ from $x$ to $x'$, where a path $f$ is a continuous function $f: I \rightarrow X$ (with $I = [0, 1]$ the closed interval) satisfying $f(0) = x$ and $f(1) = x'$.
