A terminal object is an object that receives exactly one arrow from every other object in the category. This is the simplest example of a universal property: it is defined by the existence and uniqueness of morphisms. The uniqueness of arrows $t \to t$ forces the only endomorphism of a terminal object to be the identity. Any two terminal objects are canonically isomorphic, so terminal objects are "essentially unique".
