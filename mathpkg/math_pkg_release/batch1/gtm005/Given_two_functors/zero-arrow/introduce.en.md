The zero arrow between any two objects in a category with a null object is the unique arrow factoring through the null object. The zero arrow acts as an absorbing element for composition: composing any arrow with a zero arrow (on either side, when defined) yields a zero arrow. This provides an additive structure that is fundamental for the theory of abelian categories.
