A null object is an object that is simultaneously initial and terminal. Such an object provides a canonical "zero" in the category. Examples include the trivial group in $\text{Grp}$, the zero module in $R\text{-Mod}$, and the one-point set in $\text{Set}_*$. The existence of a null object allows one to define zero arrows between any two objects.
