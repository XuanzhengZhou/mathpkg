An arrow $e: a \rightarrow b$ is \text{invertible} in a category $C$ if there is an arrow $e': b \rightarrow a$ in $C$ with $e' \circ e = 1_a$ and $e \circ e' = 1_b$. If such an $e'$ exists, it is unique, and is written as $e' = e^{-1}$. By the usual proof, $(e_1 e_2)^{-1} = e_2^{-1} e_1^{-1}$, provided the composite $e_1 e_2$ is defined and both $e_1$ and $e_2$ are invertible.
