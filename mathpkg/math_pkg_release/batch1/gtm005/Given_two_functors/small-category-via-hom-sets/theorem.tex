A \emph{small category} is given by: a set of objects; for each ordered pair $\langle a, b \rangle$ a set $\hom(a, b)$; for each triple $\langle a, b, c \rangle$ a composition function $\hom(b, c) \times \hom(a, b) \to \hom(a, c)$; and for each $b$ an identity $1_b \in \hom(b, b)$, satisfying associativity, unit laws, and the disjointness axiom $\hom(a, b) \cap \hom(a', b') = \emptyset$ when $\langle a, b \rangle \neq \langle a', b' \rangle$.
