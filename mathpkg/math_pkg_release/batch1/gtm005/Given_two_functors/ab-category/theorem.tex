An \emph{$\mathbf{Ab}$-category} (or \emph{preadditive category}) is a category $A$ in which each hom-set $A(a, b)$ is an additive abelian group and for which composition is bilinear: for arrows $f, f': a \to b$ and $g, g': b \to c$,
$$(g + g') \circ (f + f') = g \circ f + g \circ f' + g' \circ f + g' \circ f'.$$
