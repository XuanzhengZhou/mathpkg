If $z$ is initial, there is a unique map $z \rightarrow z$, hence $1_z = 0$ and $A(z,z) = 0$. If $1_z = 0$, then any $f : b \rightarrow z$ has $f = 1_z f = 0 f = 0 : b \rightarrow z$, so there is a unique arrow, namely $0$, from $b$ to $z$, and $z$ is terminal. The rest follows by duality.

When there is a null (i.e., initial and terminal) object $z$ in the $\text{Ab}$-category $A$, the unique maps $b \rightarrow z$ and $z \rightarrow c$ are the zero elements of $A(b,z)$ and $A(z,c)$ respectively.
