In an Ab-category, the algebraic (additive) structure forces the categorical notions of initial and terminal objects to coincide. This proposition shows that any initial object is automatically terminal, and vice versa, making it a null (zero) object. The key insight is that the identity arrow on such an object must equal the zero arrow, which is a purely algebraic characterization.
