For any small category $\mathcal{C}$, the functor category $\mathbf{Sets}^{\mathcal{C}^{\mathrm{op}}}$ (the category of presheaves on $\mathcal{C}$) has a subobject classifier. The object $\Omega \in \mathbf{Sets}^{\mathcal{C}^{\mathrm{op}}}$ is defined by: for each object $c \in \mathcal{C}$,
$$
\Omega(c) = \{\, \text{sieves on } c \,\},
$$
where a \textit{sieve} on $c$ is a set of morphisms with codomain $c$, closed under precomposition. The universal monomorphism $\mathsf{true} : 1 \rightarrowtail \Omega$ selects, for each $c$, the maximal sieve (the set of all morphisms into $c$).
