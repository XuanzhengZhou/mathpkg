In any metacategory, each object has exactly one identity arrow. If $1_b$ and $1'_b$ both satisfy the unit law for object $b$, then $1_b = 1_b \circ 1'_b = 1'_b$. This observation justifies the common practice of identifying an object with its identity arrow.
