See canonical definition at: \texttt{left-c-object}.

An \textbf{internal diagram} on an internal category $C$ is a synonym for a left $C$-object. It consists of an object $H$ with a projection $\pi : H \to C_0$ and an action $\mu : C_1 \times_{C_0} H \to H$ satisfying the unit and associative laws as defined in \texttt{left-c-object}.
