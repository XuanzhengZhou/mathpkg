A category object in the category $\mathbf{Grp}$ of groups is exactly a group object in the category $\mathbf{Cat}$ of small categories. That is, the structure of an internal category in $\mathbf{Grp}$ — where $C_0$ and $C_1$ are groups and the structure maps $i, d_0, d_1, \gamma$ are group homomorphisms — is equivalent to giving a small category $C$ together with a functor $m : C \times C \to C$ that is associative and admits an inverse, making $C$ a group object in $\mathbf{Cat}$.
