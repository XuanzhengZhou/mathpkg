An object of $(B \times C)^{\mathrm{op}}$ is a pair $\langle b, c \rangle$, which is also an object of $B^{\mathrm{op}} \times C^{\mathrm{op}}$. An arrow $\langle f, g \rangle: \langle b', c' \rangle \to \langle b, c \rangle$ in $(B \times C)^{\mathrm{op}}$ corresponds to an arrow $\langle f, g \rangle: \langle b, c \rangle \to \langle b', c' \rangle$ in $B \times C$, where $f: b \to b'$ in $B$ and $g: c \to c'$ in $C$. This is the same data as an arrow $\langle f^{\mathrm{op}}, g^{\mathrm{op}} \rangle: \langle b, c \rangle \to \langle b', c' \rangle$ in $B^{\mathrm{op}} \times C^{\mathrm{op}}$. The composition rules match under this correspondence, giving the required isomorphism.
