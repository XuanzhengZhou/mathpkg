For categories $C$ and $D$ with small hom-sets and a functor $S: D \to C$, there is a bijection between universal arrows $\langle r, u: c \to Sr \rangle$ from $c$ to $S$ and natural isomorphisms
$$\varphi: D(r, -) \cong C(c, S-).$$
Given $\varphi$, set $u = \varphi_r(1_r): c \to Sr$; then $\langle r, u \rangle$ is a universal arrow from $c$ to $S$. Conversely, given a universal arrow $\langle r, u \rangle$, the map $f' \mapsto Sf' \circ u$ defines a natural isomorphism.
