To each Hausdorff space $Y$, the \textbf{Kelleyfication} $KY$ is the compactly generated Hausdorff space with the same underlying set as $Y$, whose closed sets are those subsets $A \subset Y$ such that $A \cap C$ is closed in $Y$ for every compact subset $C \subset Y$. The identity function $\varepsilon_Y: KY \to Y$ is continuous. This defines a functor $K: \mathbf{Haus} \to \mathbf{CGHaus}$ right adjoint to the inclusion $\mathbf{CGHaus} \hookrightarrow \mathbf{Haus}$, making $\mathbf{CGHaus}$ a coreflective subcategory of $\mathbf{Haus}$.
