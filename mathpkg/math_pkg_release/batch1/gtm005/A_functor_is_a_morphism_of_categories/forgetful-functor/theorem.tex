A functor which simply ``forgets'' some or all of the structure of an algebraic object is called a \textbf{forgetful functor} (or \textbf{underlying functor}). For example, the forgetful functor $U: \mathbf{Grp} \rightarrow \mathbf{Set}$ assigns to each group $G$ the set $UG$ of its elements (forgetting the multiplication and hence the group structure), and assigns to each morphism $f: G \rightarrow G'$ of groups the same function $f$, regarded just as a function between sets.
