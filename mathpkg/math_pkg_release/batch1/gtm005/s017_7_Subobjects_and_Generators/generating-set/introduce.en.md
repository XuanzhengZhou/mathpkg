A generating set (or generator) in a category is a collection of objects that can distinguish distinct parallel arrows via morphisms from objects in the set. If two arrows $h, h': c \to d$ are different, there must exist an object $s$ in the generating set and a morphism $f: s \to c$ such that $hf \neq h'f$. This is a "separating" condition — the term "generates" is traditional but "separates" would be more descriptive. The concept is dual to that of a cogenerating set and plays a key role in the adjoint functor theorems.
