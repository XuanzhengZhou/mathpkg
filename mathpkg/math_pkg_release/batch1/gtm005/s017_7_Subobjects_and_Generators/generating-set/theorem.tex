\text{A set $S$ of objects of the category $C$ is said to \textbf{generate} $C$ when to any parallel pair $h, h': c \to d$ of arrows of $C$, $h \neq h'$ implies that there is an $s \in S$ and an arrow $f: s \to c$ with $hf \neq h'f$.}

\text{This definition includes the case of a single object $s$ generating a category $C$ (then called a \textbf{generator}).}

\text{Examples: any one-point set generates $\mathbf{Set}$; $\mathbb{Z}$ generates $\mathbf{Ab}$ and $\mathbf{Grp}$; $R$ generates $R\text{-}\mathbf{Mod}$. The set of finite cyclic groups is a generator for the category of all finite abelian groups (or of all torsion abelian groups).}
