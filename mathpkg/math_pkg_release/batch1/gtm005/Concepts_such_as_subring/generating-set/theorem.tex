A set $S$ of objects of a category $\mathcal{C}$ is said to \textbf{generate} $\mathcal{C}$ when to any parallel pair $h, h' : c \to d$ of arrows of $\mathcal{C}$, $h \neq h'$ implies that there is an $s \in S$ and an arrow $f : s \to c$ with $h f \neq h' f$.

(The term "generates" is well established but poorly chosen; "separates" would have been better.)

This definition includes the case of a single object $s$ generating a category $\mathcal{C}$. For example, any one-point set generates $\mathbf{Set}$, $\mathbb{Z}$ generates $\mathbf{Ab}$ and $\mathbf{Grp}$, and $R$ generates $R\text{-}\mathbf{Mod}$. The set of finite cyclic groups is a generator for the category of all finite abelian groups (or of all torsion abelian groups).
