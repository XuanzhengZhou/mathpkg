Let the category $\mathcal{A}$ be small-complete, with small hom-sets, and a small cogenerating set $Q$, while every set of subobjects of an object $a \in \mathcal{A}$ has a pullback (and hence has an intersection). Let the category $\mathcal{X}$ have small hom-sets. If $G : \mathcal{A} \to \mathcal{X}$ is a continuous functor (preserves all small limits), then $G$ has a left adjoint.
