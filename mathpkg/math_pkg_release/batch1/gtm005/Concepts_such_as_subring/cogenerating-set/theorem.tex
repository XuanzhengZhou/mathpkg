Dually, a set $Q$ of objects is a \textbf{cogenerating set} for the category $\mathcal{C}$ when to every parallel pair $h \neq h' : a \to b$ of arrows of $\mathcal{C}$ there is an object $q \in Q$ and an arrow $g : b \to q$ with $g h \neq g h'$. A single object $q$ is a \textbf{cogenerator} when $\{q\}$ is a cogenerating set.

For example, any two-point set is a cogenerator in $\mathbf{Set}$.
