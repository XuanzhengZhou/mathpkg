A spanning arrow $f : x \to G a$ (relative to a functor $G$) is one that does not factor through any proper subobject of $a$. This notion is a technical tool used in constructing small solution sets for the Adjoint Functor Theorem. For instance, in the category of algebras of a given type, an arrow $f : S \to G A$ spans $A$ if all elements of $A$ are generated from the image $f(S)$ by iterated operations. The set of such spanning arrows from a given set $S$ is small, and forms a solution set for the forgetful functor from algebras to sets, establishing the existence of free algebras.
