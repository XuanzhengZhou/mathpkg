A functor $G: A \to X$ has a left adjoint if and only if both:
\begin{itemize}
\item[(i)] $G$ preserves all limits which exist in $A$;
\item[(ii)] For each $x \in X$, $\mathrm{Lim}\big(Q: (x \downarrow G) \to A\big)$ exists in $A$.
\end{itemize}
When this is the case, a left adjoint $F$ is given on each $x \in X$ by
$$Fx = \mathrm{Lim}\big(Q: (x \downarrow G) \to A\big).$$
