Given functors $K: M \rightarrow C$ and $T: M \rightarrow A$, the right Kan extension $\operatorname{Ran}_K T: C \rightarrow A$, when it exists, is given by the end formula
\[
(\operatorname{Ran}_K T)(c) = \int_{m \in M} A(\hom_C(c, Km), Tm).
\]
Dually, the left Kan extension $\operatorname{Lan}_K T$ is given by a coend formula
\[
(\operatorname{Lan}_K T)(c) = \int^{m \in M} \hom_C(Km, c) \otimes Tm.
\]
