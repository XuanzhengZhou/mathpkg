The category $\mathbf{Set}_*$ of pointed sets is a basic example of a category whose objects carry additional structure. An object is a set together with a distinguished base point, and morphisms are functions that preserve the base point. This category appears frequently in algebraic topology (for pointed spaces) and serves as a simple model for categories of structured objects.
