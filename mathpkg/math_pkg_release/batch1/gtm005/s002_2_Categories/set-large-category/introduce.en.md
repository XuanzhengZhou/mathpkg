The category $\mathbf{Set}$ of all small sets is constructed by fixing a Grothendieck universe $U$. A set is declared small precisely when it is an element of $U$, and $\mathbf{Set}$ has the set $U$ itself as its set of objects, with all functions between small sets as arrows. This device resolves the set-theoretic paradox that the collection of all sets is not itself a set, allowing category theory to treat $\mathbf{Set}$ as a legitimate (though large) category.
