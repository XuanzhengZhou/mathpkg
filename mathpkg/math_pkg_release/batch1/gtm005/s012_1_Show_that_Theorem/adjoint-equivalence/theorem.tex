A functor $S: A \to C$ is an equivalence of categories if and only if $S$ is full, faithful, and each object $c \in C$ is isomorphic to $Sa$ for some object $a \in A$ (i.e., $S$ is essentially surjective on objects). In this case, there exists a functor $T: C \to A$ and natural isomorphisms $\eta: I_C \cong ST$, $\varepsilon: TS \cong I_A$ such that $\langle T, S; \eta, \varepsilon \rangle: C \to A$ is an adjoint equivalence.
