An augmented simplicial object in a category $X$ is a functor $S' : \Delta_+^{\text{op}} \rightarrow X$, where $\Delta_+$ is the category of all finite ordinals including the empty ordinal. A simplicial object $S$ may be augmented (i.e., extended to a functor $S'$) by finding one object $S_{-1} \in X$ and one arrow $\varepsilon : S_0 \rightarrow S_{-1}$ satisfying the appropriate compatibility conditions.
