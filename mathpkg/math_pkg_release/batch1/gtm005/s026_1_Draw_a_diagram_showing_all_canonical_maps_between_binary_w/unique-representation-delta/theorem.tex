In $\Delta$, any arrow $f : n \rightarrow n'$ has a unique representation

$$f = \delta_{i_1} \circ \cdots \circ \delta_{i_k} \circ \sigma_{j_1} \circ \cdots \circ \sigma_{j_h},$$

where the ordinal numbers $h$ and $k$ satisfy $n - h + k = n'$, while the strings of subscripts $i$ and $j$ satisfy

$$n' > i_1 > \cdots > i_k \geq 0, \quad 0 \leq j_1 < \cdots < j_h < n-1,$$

and where

$$\delta_i^n = 1_i + \eta + 1_{n-i} : n \rightarrow n+1, \quad \sigma_i^n = 1_i + \mu + 1_{n-i-1} : n+1 \rightarrow n.$$
