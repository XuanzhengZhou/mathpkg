A closed category $V$ is a symmetric monoidal category in which each functor $- \square b : V \rightarrow V$ has a specified right adjoint $(\cdot)^b : V \rightarrow V$. For example, $\langle \mathbf{Ab}, \otimes, \mathbb{Z} \rangle$ is closed; the adjoint is given for abelian groups $A$ and $B$ as $A^B = \hom(B, A)$, the abelian group of all morphisms $B \rightarrow A$.
