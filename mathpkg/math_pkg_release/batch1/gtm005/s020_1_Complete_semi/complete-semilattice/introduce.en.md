A complete semi-lattice is a partially ordered set in which every subset (not just finite subsets) has a least upper bound, called the supremum. This is a stronger condition than a join-semilattice, which requires suprema only for finite subsets. Complete semi-lattices play a key role in the theory of monads because they arise naturally as the algebras of the covariant power set monad $\mathcal{P}$ on $\mathbf{Set}$. The empty set has a supremum, which serves as the bottom element, and the set $Q$ itself has a supremum, which serves as the top element, so every complete semi-lattice is actually a complete lattice.
