Every (small) complete semi-lattice $Q$ is a $\mathcal{P}$-algebra for the power set monad $\langle \mathcal{P}, \eta, \mu \rangle$ on $\mathbf{Set}$, with the structure map $h : \mathcal{P}Q \to Q$ given by $hS = \sup S$ for each $S \subset Q$, and the partial order defined by $x \leq y$ iff $h\{x, y\} = y$.
