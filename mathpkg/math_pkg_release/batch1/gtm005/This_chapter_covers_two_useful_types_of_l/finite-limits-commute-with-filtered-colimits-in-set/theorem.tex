In the category $\mathbf{Set}$, finite limits commute with filtered colimits. That is, for any bifunctor $F : P \times J \rightarrow \mathbf{Set}$ with $P$ a finite category and $J$ a filtered category, the canonical map

$$\kappa : \operatorname{Colim}_j \operatorname{Lim}_p F(p,j) \rightarrow \operatorname{Lim}_p \operatorname{Colim}_j F(p,j)$$

is an isomorphism.
