For any functor $S: C^{\text{op}} \times C \rightarrow X$, a limit of the induced functor $S^{\S}: C^{\S} \rightarrow X$ is an end of $S$. That is, a cone $\tau: x \rightarrow S^{\S}$ is exactly the same thing as a wedge $\omega: x \mathbin{\dot{\rightarrow}} S$, so

$$\operatorname{Lim} S^{\S} \cong \int_c S(c, c)$$

whenever either side exists.
