For each functor $T: C \rightarrow X$, let $S$ be the composite functor

$$C^{\text{op}} \times C \xrightarrow{Q} C \xrightarrow{T} X$$

where $Q$ is the second projection $\langle c, c' \rangle \mapsto c'$. Then $\langle e, \tau: e \rightarrow T \rangle$ is a limit for $T$ in $X$ if and only if $\langle e, \tau: e \mathbin{\dot{\rightarrow}} S \rangle$ is an end for $S$ in $X$. Consequently, there is an isomorphism

$$\int_c S(c, c) = \int_c Tc \cong \operatorname{Lim} T,$$

valid when either the end or the limit exists, carrying the ending wedge to the limiting cone.
