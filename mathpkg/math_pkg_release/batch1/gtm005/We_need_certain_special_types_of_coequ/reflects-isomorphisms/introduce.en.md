A functor reflects isomorphisms (equivalently, is conservative) if the only way an arrow's image under the functor can be invertible is if the original arrow was already invertible. This is a basic property of forgetful functors from algebraic categories: a homomorphism whose underlying function is bijective must itself be an isomorphism.
