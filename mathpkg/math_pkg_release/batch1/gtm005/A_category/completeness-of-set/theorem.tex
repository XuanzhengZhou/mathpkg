If the category $J$ is small, any functor $F : J \rightarrow \mathbf{Set}$ has a limit which is the set $\operatorname{Cone}(*, F)$ of all cones $\sigma : * \rightarrow F$ from the one point set $*$ to $F$, while the limiting cone $v$, with

$$v_j : \operatorname{Cone}(*, F) \rightarrow F_j, \quad \sigma \mapsto \sigma_j,$$

is for each $j$ that function sending each cone $\sigma$ to the element $\sigma_j \in F_j$.
