A monoid can be viewed as a category with exactly one object, where the arrows correspond to the elements of the monoid and composition of arrows corresponds to the monoid operation. Conversely, for any category and any fixed object in it, the set of endomorphisms (arrows from that object to itself) forms a monoid under composition. This perspective unifies algebraic and categorical notions.
