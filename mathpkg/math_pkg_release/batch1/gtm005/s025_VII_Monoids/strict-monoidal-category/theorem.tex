A \textbf{strict monoidal category} $\langle B, \square, e \rangle$ is a category $B$ together with a bifunctor $\square : B \times B \to B$ which is associative,
$$\square(\square \times 1) = \square(1 \times \square) : B \times B \times B \to B,$$
and with an object $e$ which is a left and right unit for $\square$,
$$\square(e \times 1) = \mathrm{id}_B = \square(1 \times e).$$
The bifunctor $\square$ assigns to each pair of objects $a, b \in B$ an object $a \square b$ of $B$ and to each pair of arrows $f : a \to a', g : b \to b'$ an arrow $f \square g : a \square b \to a' \square b'$, satisfying the interchange law
$$1_a \square 1_b = 1_{a \square b}, \quad (f' \square g') (f \square g) = (f' f) \square (g' g).$$
The associative law means $\square$ is associative both for objects and arrows; the unit law means $e \square c = c = c \square e$ for objects $c$ and $1_e \square f = f = f \square 1_e$ for arrows $f$.
