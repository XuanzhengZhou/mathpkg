The covariant hom-functor $C(a, -): C \to \mathbf{Set}$ is one of the most important examples of a functor in category theory. For a fixed object $a$ in a category $C$ with small hom-sets, it sends each object $b$ to the set of all morphisms $\hom(a, b)$ and each morphism $k: b \to b'$ to the post-composition map $k_*: f \mapsto k \circ f$. This construction captures how the object $a$ "sees" the rest of the category through morphisms originating from $a$. The hom-functor is the basis for the Yoneda lemma and the theory of representable functors.
