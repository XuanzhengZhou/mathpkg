Let $\Sigma$ be any statement with free variables $f, g, \ldots$ in the elementary theory of an abstract category. Then $\Sigma$ is true for arrows $f, g, \ldots$ of a category $C$ if and only if the dual statement $\Sigma^*$ is true for the arrows $f^{\text{op}}, g^{\text{op}}, \ldots$ of the opposite category $C^{\text{op}}$. In particular, a sentence $\Sigma$ is true in $C^{\text{op}}$ if and only if the dual sentence $\Sigma^*$ is true in $C$.
