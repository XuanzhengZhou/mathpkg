In an $\textit{Ab}$-category, each hom-set carries the structure of an additive abelian group and composition is bilinear. Proposition 1 establishes that the categorical notions of initial and terminal objects coincide in this setting: an object is initial if and only if it is terminal, which occurs precisely when its identity morphism equals the zero morphism. Such an object is called a null object and serves as the zero object in the additive structure of the category.
