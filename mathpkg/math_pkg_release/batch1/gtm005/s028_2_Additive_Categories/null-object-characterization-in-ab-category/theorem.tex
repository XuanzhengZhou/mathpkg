In an $\textit{Ab}$-category $A$, the following properties of an object $z$ are equivalent:
\begin{itemize}
\item[(i)] $z$ is initial;
\item[(ii)] $z$ is terminal;
\item[(iii)] $1_z = 0 : z \to z$;
\item[(iv)] the abelian group $A(z,z)$ is the zero group.
\end{itemize}
In particular, any initial (or any terminal) object in $A$ is a null object.
