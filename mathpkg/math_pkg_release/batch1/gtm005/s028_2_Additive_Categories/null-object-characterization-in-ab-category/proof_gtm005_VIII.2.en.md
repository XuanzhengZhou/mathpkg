If $z$ is initial, there is a unique map $z \to z$, hence $1_z = 0$ and $A(z,z) = 0$. If $1_z = 0$, then any $f : b \to z$ has $f = 1_z f = 0 f = 0 : b \to z$, so there is a unique arrow (namely $0$) from $b$ to $z$, and $z$ is terminal. The rest follows by duality.
