For a continuous map $f: X \to Y$ with $Gf = k \circ t$ for some function $k$, where $t: GX \to MS$ is a surjection onto a quotient set, there exists a functor $M$ sending a set function $t: GX \to S$ to the set $S$ equipped with the quotient topology induced by $t$ (the finest topology making $t$ continuous). Then $k: MS \to Y$ is continuous, and

$$(X \downarrow \mathbf{Top}) (Mt, f) \cong (GX \downarrow \mathbf{Set}) (t, Gf)$$

with unit the identity map, so $M$ is left-adjoint-right-inverse to $X \downarrow G$.
