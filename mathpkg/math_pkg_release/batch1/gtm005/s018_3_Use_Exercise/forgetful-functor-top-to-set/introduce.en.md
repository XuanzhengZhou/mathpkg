The forgetful functor $G: \mathbf{Top} \to \mathbf{Set}$ strips a topological space of its topology, retaining only the underlying set of points. It is faithful because distinct continuous maps remain distinct as set functions. Remarkably, $G$ has both a left adjoint (discrete topology) and a right adjoint (indiscrete topology), which implies it preserves all limits and all colimits.
