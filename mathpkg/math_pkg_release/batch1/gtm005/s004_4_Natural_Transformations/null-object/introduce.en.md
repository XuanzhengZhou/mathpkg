A null object (or zero object) is simultaneously initial and terminal. It provides a canonical 'zero' arrow between any two objects. Categories with null objects (such as Ab, R-Mod, and Set$_*$) admit a rich theory of kernels and cokernels, forming the foundation for abelian categories.
