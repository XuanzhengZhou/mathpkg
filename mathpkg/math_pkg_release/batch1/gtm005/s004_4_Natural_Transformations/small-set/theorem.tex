Fix a universe $U$. A set $u$ is called \textbf{small} if $u \in U$. The universe $U$ itself is the set of all small sets. A function $f: u \rightarrow v$ is small when $u$ and $v$ are small sets. A category $C$ is \textbf{small} if the set of objects of $C$ and the set of arrows of $C$ are both small sets.
