A \emph{product} of a family of objects $\{a_i\}_{i \in I}$ indexed by a set $I$ is an object $\prod_i a_i$ together with projections $p_i: \prod_i a_i \to a_i$ such that for any object $c$ and family of arrows $f_i: c \to a_i$, there is a unique arrow $h: c \to \prod_i a_i$ with $p_i \circ h = f_i$ for all $i$.
