A \emph{forgetful functor} (or \emph{underlying functor}) $U: \mathcal{A} \rightarrow \mathbf{Set}$ is a functor which assigns to each object $A$ of an algebraic category $\mathcal{A}$ the underlying set $UA$ of its elements, and to each morphism $f: A \rightarrow A'$ of $\mathcal{A}$ the same function $f$, regarded just as a function between sets.
