Given a monad $\langle T, \eta, \mu \rangle$ in a category $X$, consider the category $\mathbf{Adj}(T)$ whose objects are all adjunctions $\langle F, G, \eta, \varepsilon \rangle : X \to A$ which define the monad $\langle T, \eta, \mu \rangle$ in $X$, and whose arrows are those maps of adjunctions (in the sense of \S IV.7) which are the identity on $X$. Then the Kleisli adjunction $\langle F_T, G_T, \eta, \varepsilon_T \rangle : X \to X_T$ is an initial object in this category.
