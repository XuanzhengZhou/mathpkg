An \textbf{abelian category} is an additive category in which:
\begin{enumerate}
\item Every arrow has a kernel and a cokernel;
\item Every monic arrow is a kernel of some arrow (i.e., every monic is normal);
\item Every epic arrow is a cokernel of some arrow (i.e., every epic is conormal).
\end{enumerate}
In an abelian category, every arrow $f$ factors as $f = m \circ e$ where $m$ is monic and $e$ is epic, and the First Isomorphism Theorem holds.
