Let $A$ be any category which satisfies the following axioms:
\begin{itemize}
\item[(i)] $A$ has a null object.
\item[(ii')] $A$ has binary products and binary coproducts.
\item[(iii)] Every arrow has a kernel and a cokernel.
\item[(iv)] Every monic arrow is a kernel, and every epi arrow is a cokernel.
\end{itemize}
Then the formula (2.6) for addition of arrows can be used to introduce an addition in each hom-set $A(a, b)$, and with this addition $A$ becomes an abelian category. That is, the abelian group structure on hom-sets need not be given a priori but arises from the categorical structure.
