Given a pullback square

$$\begin{CD}
s @>f'>> d \\
@V{g'}VV @VV{g}V \\
b @>f>> c
\end{CD}$$

in an abelian category, if $f$ is epi then $f'$ is also epi. Moreover, the kernel $k = \ker f$ factors as $k = g' k'$ for a unique arrow $k'$ which is the kernel of $f'$. In particular, given a short exact sequence $a \rightarrow b \rightarrow c$, any arrow $g : d \rightarrow c$ to the right-hand end object yields by pullback a short exact sequence $a \rightarrow s \rightarrow d$ with the same kernel object $a$.
