If $G: A \to X$ has a left adjoint $F$ with counit $\varepsilon: FG \to 1_A$, then $\operatorname{Ran}_G 1_A$ exists, is equal to $F$ with counit $\varepsilon$, and is preserved by any functor whatever. More generally, for any functor $H: A \to C$, one has $HF = \operatorname{Ran}_G H$, with unit $H\varepsilon$, and this Kan extension is also absolute.
