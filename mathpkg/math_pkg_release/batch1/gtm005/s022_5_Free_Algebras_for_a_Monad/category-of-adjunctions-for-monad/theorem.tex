Given a monad $\langle T, \eta, \mu \rangle$ in a category $X$, consider the category $\mathbf{Adj}(T)$ whose objects are all adjunctions $\langle F, G, \eta, \varepsilon \rangle : X \rightleftharpoons A$ whose associated monad in $X$ is $\langle T, \eta, \mu \rangle$, and whose arrows are maps of adjunctions (in the sense of § IV.7) that restrict to the identity on $X$. This category has an initial object — the Kleisli adjunction $F_T \dashv G_T : X \rightleftharpoons X_T$ — and a terminal object — the Eilenberg–Moore adjunction $F^T \dashv G^T : X \rightleftharpoons X^T$.
