For any category $C$ with small hom-sets, each covariant hom-functor
$$C(c, -) : C \rightarrow \mathbf{Set}$$
preserves all limits which exist in $C$; in particular, all small limits.

More generally, if $C$ has hom-sets in a category $\mathbf{Ens}$ of sets in which $\mathbf{Ens}(X, Y)$ consists of all functions from $X$ to $Y$, then each hom-functor $C(c, -) : C \rightarrow \mathbf{Ens}$ preserves all limits which exist in $C$.

Consequently,
$$\operatorname{Lim} C(c, F-) \cong C(c, \operatorname{Lim} F)$$
for any functor $F : J \rightarrow C$ having a limit. Dually, the contravariant hom-functor $C(-, c) = C^{\text{op}}(c, -) : C^{\text{op}} \rightarrow \mathbf{Set}$ carries small colimits in $C$ to limits in $\mathbf{Set}$.
