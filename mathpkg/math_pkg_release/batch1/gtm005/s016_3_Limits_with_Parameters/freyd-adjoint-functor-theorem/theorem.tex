Let $A$ be a small-complete category with small hom-sets. A functor $G : A \rightarrow X$ has a left adjoint if and only if it preserves all small limits and satisfies the following

\textbf{Solution Set Condition.} For each object $x \in X$ there exists a small set $I$ and an $I$-indexed family of arrows $f_i : x \rightarrow G a_i$ (with $a_i \in A$) such that every arrow $h : x \rightarrow G a$ can be written as a composite $h = G t \circ f_i$ for some index $i \in I$ and some arrow $t : a_i \rightarrow a$ in $A$.
