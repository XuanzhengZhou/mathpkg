A metagraph is the simplest categorical structure, consisting of objects and arrows together with the assignment of a domain and codomain to each arrow. It provides the underlying directed graph upon which a metacategory is built by adding identity arrows and a composition operation. This notion is introduced by Mac Lane as a precursor to the full definition of a metacategory, emphasizing that the structure of a category can be described purely axiomatically without any recourse to set theory.
