If $S: D \to C$ is a functor and $c$ an object of $C$, a \textbf{universal arrow from $c$ to $S$} is a pair $\langle r, u \rangle$ consisting of an object $r$ of $D$ and an arrow $u: c \to Sr$ of $C$, such that for every pair $\langle d, f \rangle$ with $d$ an object of $D$ and $f: c \to Sd$ an arrow of $C$, there is a unique arrow $f': r \to d$ of $D$ with $Sf' \circ u = f$. Equivalently, $\langle r, u \rangle$ is an initial object in the comma category $(c \downarrow S)$.
