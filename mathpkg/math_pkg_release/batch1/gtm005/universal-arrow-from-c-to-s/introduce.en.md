A universal arrow from an object $c$ to a functor $S: D \to C$ is an initial object in the comma category $(c \downarrow S)$. It consists of an object $r \in D$ and an arrow $u: c \to Sr$ through which every arrow from $c$ to any $Sd$ factors uniquely. Universal arrows unify many constructions: free algebras, completions, bases, and tensor products are all examples.
