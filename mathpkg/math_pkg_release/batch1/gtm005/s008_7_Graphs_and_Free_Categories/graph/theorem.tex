A \textbf{(directed) graph} $G$ consists of a set $O$ of objects (vertices), a set $A$ of arrows $f$ (edges), and a pair of functions $\partial_0, \partial_1: A \rightrightarrows O$:
\[
A \xrightarrow{\partial_0}_{\partial_1} O, \quad \partial_0 f = \text{domain } f, \quad \partial_1 f = \text{codomain } f.
\]
