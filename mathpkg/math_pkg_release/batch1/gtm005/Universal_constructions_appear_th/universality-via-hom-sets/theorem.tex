Let $C$ and $D$ be categories with small hom-sets. For a functor $S \colon D \to C$ and an object $c \in C$, a pair $\langle r, u \colon c \to Sr \rangle$ is a universal arrow from $c$ to $S$ if and only if the function sending each arrow $f' \colon r \to d$ of $D$ into $Sf' \circ u \colon c \to Sd$ of $C$ defines a natural isomorphism

$$
\varphi \colon D(r, -) \xrightarrow{\cong} C(c, S -)
$$

of functors $D \to \mathbf{Set}$.
