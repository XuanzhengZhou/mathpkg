Let $*$ denote any one-point set and let $D$ have small hom-sets. If $\langle r, u \colon * \to Kr \rangle$ is a universal arrow from $*$ to $K \colon D \to \mathbf{Set}$, then the function

$$
\psi_d \colon D(r, d) \to Kd, \qquad f' \mapsto (Kf')u(*)
$$

defines a natural isomorphism $\psi \colon D(r, -) \cong K$, i.e., a representation of $K$ with representing object $r$.
