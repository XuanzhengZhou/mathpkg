If $D$ is a category and $H \colon D \to \mathbf{Set}$ a functor, a \text{universal element of the functor $H$} is a pair $\langle r, e \rangle$ consisting of an object $r \in D$ and an element $e \in Hr$ such that for every pair $\langle d, x \rangle$ with $x \in Hd$ there is a unique arrow $f \colon r \to d$ of $D$ with $(Hf)e = x$.

The notion "universal element" is a special case of the notion "universal arrow": if $*$ is the set with one point, then any element $e \in Hr$ can be regarded as an arrow $e \colon * \to Hr$ in $\mathbf{Set}$. Thus a universal element $\langle r, e \rangle$ for $H$ is exactly a universal arrow from $*$ to $H$.
