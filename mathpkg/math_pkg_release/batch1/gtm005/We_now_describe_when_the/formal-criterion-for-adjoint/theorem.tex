A functor $G: A \to X$ has a left adjoint if and only if both

\begin{enumerate}
  \item[(i)] $G$ preserves all limits which exist in $A$;
  \item[(ii)] For each $x \in X$, $\mathrm{Lim}(Q: (x \downarrow G) \to A)$ exists in $A$.
\end{enumerate}

When this is the case, a left adjoint $F$ is given on each $x \in X$ as

$$Fx = \mathrm{Lim}\bigl(Q: (x \downarrow G) \to A\bigr).$$
