(Representables and adjoints; Benabou.) Let $C$ have small hom-sets, and construct from each $K: C \to \mathbf{Set}$ the category $C_K$ obtained by adjoining to $C$ one new object $\infty$ with new hom-sets $C_K(\infty, c) = Kc$, $C_K(\infty, \infty) = *$, the one-point set and $C_K(c, \infty) = \emptyset$, the empty set, with appropriate composition. Let $J_K: C \to C_K$ be the inclusion. Prove that $K$ is representable if and only if $J_K$ has a left adjoint.
