Let $K: M \to C$ and $T: M \to A$ be functors. If $M$ has small hom-sets and $A$ is complete, then for each $c \in C$ the functor

$$(c \downarrow K) \xrightarrow{Q} M \xrightarrow{T} A$$

has a limit in $A$, with limiting cone $\lambda$. Define

$$Rc = \lim\bigl(TQ: (c \downarrow K) \to A\bigr),$$

and for each $g: c \to c'$, let $Rg$ be the unique arrow commuting with the limiting cones. These formulas define a functor $R: C \to A$, and for each $n \in M$, the components $\lambda_{1_{Kn}} = \varepsilon_n$ of the limiting cones define a natural transformation $\varepsilon: RK \to T$. Then $\langle R, \varepsilon \rangle$ is a right Kan extension of $T$ along $K$.
