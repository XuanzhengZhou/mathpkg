An $O$-graph is a graph (in the sense of a directed multigraph) whose object set is fixed to $O$. It consists of a set $A$ of arrows together with domain and codomain functions $\partial_0, \partial_1 : A \to O$. This notion is used as a stepping stone to describe categories algebraically: a category with object set $O$ is an $O$-graph equipped with composition and identity operations satisfying associativity and unit axioms. Morphisms of $O$-graphs are required to preserve domains and codomains.
