The \textbf{metacategory of sets} has:
\begin{itemize}
\item Objects: all sets;
\item Arrows: all functions $f: X \rightarrow Y$ with specified domain $X$ and codomain $Y$;
\item Identity: for each set $S$, the identity function $1_S : S \rightarrow S$ defined by $s \mapsto s$ for all $s \in S$;
\item Composition: for $f: X \rightarrow Y$ and $g: Y \rightarrow Z$, the composite $g \circ f : X \rightarrow Z$ is defined by $(g \circ f)(x) = g(f(x))$ for all $x \in X$.
\end{itemize}
Here a function is understood to have a specified domain and codomain; thus the inclusion function $S \hookrightarrow Y$ (for $S \subset Y$) is distinct from the identity $1_S$ unless $S = Y$.
