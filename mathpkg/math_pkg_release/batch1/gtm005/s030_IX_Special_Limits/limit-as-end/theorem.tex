\text{Proposition 3.} For any functor $T : C \to X$, define $S : C^{\mathrm{op}} \times C \to X$ as the composite
$$
C^{\mathrm{op}} \times C \xrightarrow{Q} C \xrightarrow{T} X
$$
where $Q$ is projection onto the second factor. Then $\langle e, \tau : e \Rightarrow T \rangle$ is a limit of $T$ if and only if $\langle e, \tau : e \xrightarrow{\bullet} S \rangle$ is an end of $S$. Consequently,
$$
\int_c T(c) \cong \varprojlim T,
$$
so every limit can be written as an integral (an end) with a dummy contravariant variable.
