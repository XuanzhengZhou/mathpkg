(Naturality by separation of arguments.) Given $b \in B$, a functor $S : (C \times D)^{\mathrm{op}} \times C \times D \to B$, and a function $\beta$ assigning to $c \in C$, $d \in D$ an arrow $\beta_{c,d} : S(c, d, c, d) \to b$ of $B$, show that $\beta : S \xrightarrow{\bullet} b$ is dinatural if and only if it is dinatural in $c$ (for each fixed $d$) and dinatural in $d$ (for each fixed $c$). State the dual result.
