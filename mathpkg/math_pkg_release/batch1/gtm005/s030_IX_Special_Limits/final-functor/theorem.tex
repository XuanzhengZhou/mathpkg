A functor $L : J' \to J$ is \text{final} if for each $k \in J$ the comma category $(k \downarrow L)$ is nonempty and connected. This means: for each $k \in J$ there exists an object $j' \in J'$ and an arrow $k \to L j'$, and any two such arrows can be embedded in a finite commutative diagram connecting them. A subcategory is called final when the corresponding inclusion functor is final.
