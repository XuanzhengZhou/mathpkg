Since the invariants $(d, \varepsilon)$ take values in a set of size $2 \times 2 = 4$ for $p \neq 2$ and $2 \times 2 \times 2$ constraints for $p = 2$ (with some restrictions), the number of classes is bounded. For large $n$, all admissible combinations occur.
