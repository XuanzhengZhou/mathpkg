Theorem 5 gives the elementary evaluation of the Legendre symbol for $x = 1, -1, 2$. The case $x=1$ is trivial. The case $x=-1$ uses $\varepsilon(p)$ which distinguishes $p \equiv 1 \pmod{4}$ from $p \equiv 3 \pmod{4}$. The case $x=2$ uses $\omega(p)$ which depends on $p \bmod 8$: $2$ is a square modulo $p$ exactly when $p \equiv \pm 1 \pmod{8}$. Together with the quadratic reciprocity law (Theorem 6), these allow computation of any Legendre symbol.
