The sigma invariant $\sigma(E) \in \mathbf{Z}/8\mathbf{Z}$ is defined as the residue class modulo 8 of $u.u$, where $u$ is an element of $E$ such that $u.x \equiv x.x \pmod{2}$ for all $x \in E$. A change $u \mapsto u + 2x$ changes $u.u$ by a multiple of 8, so the class is well-defined. For type II (even) modules, $\sigma(E) = 0$. This invariant is crucial for studying lattices modulo 8.
