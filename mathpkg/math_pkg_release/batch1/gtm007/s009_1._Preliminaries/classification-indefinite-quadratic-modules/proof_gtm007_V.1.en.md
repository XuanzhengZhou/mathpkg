Since the isomorphism class of an indefinite type I module is determined up to isomorphism by its rank and index (cf. Theorem 4), and the same is true for type II (cf. Theorem 5), we conclude that any two indefinite elements of $S$ having the same rank, index, and type are isomorphic.
