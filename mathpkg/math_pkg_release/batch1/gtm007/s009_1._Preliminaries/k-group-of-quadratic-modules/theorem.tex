Two elements $E, E' \in S$ are called \textbf{stably isomorphic} if there exists $F \in S$ such that $E \oplus F \simeq E' \oplus F$. This is an equivalence relation. The set of stable isomorphism classes forms the Grothendieck group $K(S)$, defined as the group completion of the commutative semigroup $K_+(S)$: an element of $K(S)$ is represented by a formal difference $(E) - (F)$ with $E, F \in S$, where $(E) - (F) = (E') - (F')$ if and only if $E \oplus F' \simeq E' \oplus F$ stably.
