Let $E \in S_n$ be an indefinite unimodular lattice. If $n \leq 4$, then the associated quadratic form $f$ represents $0$ nontrivially over $\mathbf{Q}$. More precisely:
\begin{itemize}
\item[(i)] $n = 2$: $f$ represents $0$ since the signature is $(1,1)$, so $d(E) = -1$ and $-d(E)$ is a square in $\mathbf{Q}$.
\item[(ii)] $n = 3$: $f$ represents $0$ in all $\mathbf{Q}_p$ ($p \neq 2$), in $\mathbf{R}$, hence in $\mathbf{Q}$ by the Hasse-Minkowski theorem.
\item[(iii)] $n = 4$: if $d(E) = 1$, the same local argument works; if $d(E) = -1$, the form represents $0$ in $\mathbf{Q}_2$ as well, so by Hasse-Minkowski it represents $0$ in $\mathbf{Q}$.
\item[(iv)] $n \geq 5$: $f$ represents $0$ in $\mathbf{Q}$.
\end{itemize}
