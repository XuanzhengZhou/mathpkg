**Lemma.** Let $k = \mathbf{Q}_p$ be the $p$-adic field and let $k^*/k^{*2}$ be the square class group, viewed as an $\mathbf{F}_2$-vector space.

a) The number of elements in $k^*/k^{*2}$ is $2^r$, where
$$r = 2 \text{ if } p \neq 2, \qquad r = 3 \text{ if } p = 2.$$

b) If $a \in k^*/k^{*2}$ and $\varepsilon = \pm 1$, let $H_a^\varepsilon$ be the set of $x \in k^*/k^{*2}$ such that the Hilbert symbol $(x, a) = \varepsilon$. Then:
- If $a = 1$, $H_a^1$ has $2^r$ elements and $H_a^{-1} = \varnothing$.
- If $a \neq 1$, $H_a^\varepsilon$ has $2^{r-1}$ elements.

c) Let $a, a' \in k^*/k^{*2}$ and $\varepsilon, \varepsilon' = \pm 1$. Assume that $H_a^\varepsilon \cap H_{a'}^{\varepsilon'} \neq \varnothing$. Then $H_a^\varepsilon \cap H_{a'}^{\varepsilon'} = H_a^\varepsilon$ if and only if $(a, a') = \varepsilon \varepsilon'$; otherwise the intersection has $2^{r-2}$ elements.
