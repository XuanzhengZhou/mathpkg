The Ramanujan tau function $\tau(n)$ satisfies the Hecke multiplicativity relations:
\begin{align*}
\tau(nm) &= \tau(n)\tau(m) \quad \text{if } (n, m) = 1,\\
\tau(p)\tau(p^n) &= \tau(p^{n+1}) + p^{11}\tau(p^{n-1}) \quad \text{if } p \text{ is prime}, n \geq 1.
\end{align*}
For primes, one has the recurrence $\tau(p^{n+1}) = \tau(p)\tau(p^n) - p^{11}\tau(p^{n-1})$ for $n \geq 1$.
