Let $f$ be a modular function of weight $2k$. The Hecke operator $T(n)$ acts on $f$ via the lattice correspondence with an additional numerical factor $n^{2k-1}$:
$$T(n)f(z) = n^{2k-1} \sum_{\substack{ad = n, a \geq 1 \\ 0 \leq b < d}} d^{-2k} f\left(\frac{az + b}{d}\right).$$
This formula is derived from the action of the correspondence $T(n)$ on the set of lattices $\mathcal{R}$, identifying a modular function $f$ of weight $2k$ with a function on $\mathcal{R}$ satisfying the homogeneity condition $f(\lambda\Gamma) = \lambda^{-2k} f(\Gamma)$.
