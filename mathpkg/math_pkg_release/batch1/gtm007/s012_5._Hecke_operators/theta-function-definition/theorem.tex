Let $\Gamma$ be a quadratic module (a lattice with an integral positive definite quadratic form). For $m \geq 0$, let $r_\Gamma(m)$ be the number of elements $x \in \Gamma$ with $(x,x)/2 = m$. The theta function $\theta_\Gamma$ is defined on the upper half-plane $H$ by
$$\theta_\Gamma(z) = \sum_{m=0}^{\infty} r_\Gamma(m) q^m = \sum_{x \in \Gamma} q^{(x,x)/2} = \sum_{x \in \Gamma} e^{\pi i z (x,x)},$$
where $q = e^{2\pi iz}$. The function $\theta_\Gamma$ is holomorphic on $H$.
