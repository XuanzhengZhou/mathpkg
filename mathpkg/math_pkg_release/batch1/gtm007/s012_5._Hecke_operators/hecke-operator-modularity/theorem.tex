Let $f$ be a weakly modular function of weight $2k$. Then the function $T(n)f$ is weakly modular of weight $2k$. It is holomorphic on the upper half-plane $H$ if $f$ is holomorphic on $H$. Moreover, the Hecke operators satisfy:
\begin{align*}
T(m)T(n)f &= T(mn)f \quad \text{if } (m, n) = 1,\\
T(p)T(p^n)f &= T(p^{n+1})f + p^{2k-1}T(p^{n-1})f \quad \text{if } p \text{ is prime}, n \geq 1.
\end{align*}
