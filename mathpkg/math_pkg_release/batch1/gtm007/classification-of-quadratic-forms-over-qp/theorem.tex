Two quadratic forms over $\mathbb{Q}_p$ are isomorphic if and only if they have the same dimension, the same discriminant (in $\mathbb{Q}_p^*/\mathbb{Q}_p^{*2}$), and the same Hasse invariant $\varepsilon_p \in \{\pm 1\}$.

The Hasse invariant of a quadratic form $Q = a_1 X_1^2 + \cdots + a_n X_n^2$ is defined by:
$$\varepsilon_p(Q) = \prod_{i < j} (a_i, a_j)_p$$
where $(a_i, a_j)_p$ denotes the Hilbert symbol in $\mathbb{Q}_p$.
