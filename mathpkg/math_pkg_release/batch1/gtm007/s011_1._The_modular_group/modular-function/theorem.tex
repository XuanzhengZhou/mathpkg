Let $f$ be a weakly modular function of weight $2k$ on the upper half-plane $H$. Using the periodicity $f(z+1) = f(z)$, one can express $f$ as a function $\tilde{f}$ of $q = e^{2\pi i z}$, which is meromorphic in the punctured disk $0 < |q| < 1$. If $\tilde{f}$ extends to a meromorphic (resp. holomorphic) function at the origin $q = 0$, one says that $f$ is \textbf{meromorphic (resp. holomorphic) at infinity}. A weakly modular function that is meromorphic at infinity is called a \textbf{modular function}. When $f$ is holomorphic at infinity, one sets $f(\infty) = \tilde{f}(0)$.
