The \textbf{Ramanujan $\tau$-function} is defined by the Fourier expansion of the discriminant modular form:
$$
\Delta(z) = \sum_{n=1}^\infty \tau(n) q^n, \quad q = e^{2\pi i z}.
$$
Explicitly, $\tau(1) = 1$, $\tau(2) = -24$, $\tau(3) = 252$, $\tau(4) = -1472$, etc. The function $\tau(n)$ is multiplicative: $\tau(mn) = \tau(m)\tau(n)$ when $\gcd(m,n) = 1$.
