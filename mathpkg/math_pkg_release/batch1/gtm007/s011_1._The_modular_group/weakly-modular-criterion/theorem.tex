A meromorphic function $f: \mathcal{H} \to \mathbf{C}$ is weakly modular of weight $k$ for $\Gamma = \operatorname{SL}_2(\mathbf{Z})$ if and only if it satisfies:
\begin{enumerate}
\item[(i)] $f(z+1) = f(z)$ (invariance under $T$),
\item[(ii)] $f(-1/z) = z^k f(z)$ (transformation under $S$).
\end{enumerate}
Since $T$ and $S$ generate $\Gamma$, these two conditions are sufficient. For weight $k=0$, a weakly modular function is simply a $\Gamma$-invariant meromorphic function on $\mathcal{H}$.
