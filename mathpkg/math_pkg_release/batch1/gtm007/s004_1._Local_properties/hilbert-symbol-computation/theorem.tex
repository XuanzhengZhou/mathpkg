If $k = \mathbf{R}$, we have
$$
(a, b) = 1 \text{ if } a \text{ or } b \text{ is } > 0, \quad (a, b) = -1 \text{ if } a \text{ and } b \text{ are } < 0.
$$

If $k = \mathbf{Q}_p$ and we write $a, b$ in the form $p^\alpha u, p^\beta v$ where $u$ and $v$ belong to the group $\mathbf{U}$ of $p$-adic units, we have:
$$
(a, b) = (-1)^{\alpha \beta \varepsilon(p)} \left( \frac{u}{p} \right)^\beta \left( \frac{v}{p} \right)^\alpha \quad \text{if } p \neq 2,
$$
$$
(a, b) = (-1)^{\varepsilon(u) \varepsilon(v) + \alpha \omega(v) + \beta \omega(u)} \quad \text{if } p = 2,
$$
where $\left( \frac{u}{p} \right)$ denotes the Legendre symbol $\left( \frac{\bar{u}}{p} \right)$ with $\bar{u}$ the reduction of $u$ modulo $p$, and $\varepsilon(u)$, $\omega(u)$ denote respectively the class modulo $2$ of $\frac{u-1}{2}$ and of $\frac{u^2-1}{8}$.
