A \emph{character modulo $m$} is an element $\chi$ of the dual $\widehat{G(m)}$ of $G(m) = (\mathbb{Z}/m\mathbb{Z})^*$. It can be viewed as a function defined on the set of integers prime to $m$, with values in $\mathbb{C}^*$, satisfying $\chi(ab) = \chi(a)\chi(b)$. It is extended to all of $\mathbb{Z}$ by setting $\chi(a) = 0$ if $a$ is not prime to $m$.
