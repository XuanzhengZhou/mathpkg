Standard elements of $S$:
\begin{itemize}
\item $I_+$ is the module $\mathbb{Z}$ with form $x \cdot x = x^2$ (rank 1, index 1, discriminant 1).
\item $I_-$ is the module $\mathbb{Z}$ with form $x \cdot x = -x^2$ (rank 1, index -1, discriminant -1).
\item $U$ is the module $\mathbb{Z}^2$ with matrix $\begin{pmatrix}0&1\\1&0\end{pmatrix}$ (rank 2, index 0, discriminant -1, type II).
\end{itemize}
