Let $k$ be an integer. A meromorphic function $f$ on $H$ is weakly modular of weight $2k$ if $f(gz) = (cz+d)^{2k} f(z)$ for all $g = \begin{pmatrix}a&b\\c&d\end{pmatrix} \in \mathbf{SL}_2(\mathbb{Z})$. Since $S$ and $T$ generate $G$, this is equivalent to $f(z+1) = f(z)$ and $f(-1/z) = z^{2k}f(z)$.
