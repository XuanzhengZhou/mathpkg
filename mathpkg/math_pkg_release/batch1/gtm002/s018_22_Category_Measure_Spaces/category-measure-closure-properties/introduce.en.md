Theorem 22.3 establishes that for a category measure, the measure of any Baire-property set equals the measure of its closure. The canonical representative map $\phi$ that selects a regular open set from each equivalence class modulo first category satisfies conditions analogous to those of the Lebesgue lower density (Theorem 3.21), suggesting a general method for converting a measure space into a category measure space.
