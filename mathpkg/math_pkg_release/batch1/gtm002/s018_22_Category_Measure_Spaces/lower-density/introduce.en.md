A lower density is a selection function on a measure space that picks a canonical measurable representative from each equivalence class modulo nullsets. It respects the Boolean algebra structure: equivalent sets get the same representative, empty and full sets are fixed, and the map preserves intersections and inclusions. The Lebesgue density function $A \mapsto \{x : d(x, A) = 1\}$ is the prototypical example.
