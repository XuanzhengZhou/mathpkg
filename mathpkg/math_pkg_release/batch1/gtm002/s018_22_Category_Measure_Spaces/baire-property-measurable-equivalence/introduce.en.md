Theorem 22.7 establishes the second key identification: in the density topology, a set has the property of Baire exactly when it is measurable. Together with Theorem 22.6 (identifying nowhere dense with nullset), this makes $(X, \mathcal{T}, S, \mu)$ a category measure space, where the topological and measure-theoretic structures perfectly mirror each other.
