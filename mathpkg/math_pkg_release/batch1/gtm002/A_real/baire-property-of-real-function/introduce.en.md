A real-valued function has the property of Baire if the preimage of every open set has the property of Baire (as a subset of $\mathbb{R}$). Equivalently, one may require this for all sets in a countable base, or for all Borel sets. The indicator function $\chi_E$ of a set $E \subset \mathbb{R}$ has the property of Baire if and only if the set $E$ itself has the property of Baire.
