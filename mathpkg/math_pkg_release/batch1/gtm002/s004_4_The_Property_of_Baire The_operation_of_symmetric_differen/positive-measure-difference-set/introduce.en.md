Theorem 4.8 is a classical result due to Steinhaus: any set of positive Lebesgue measure, or any second-category set with the property of Baire, has the property that its difference set $A - A$ contains an interval around zero. This theorem provides a crucial link between the algebraic structure of the real line and the topological/measure-theoretic largeness of subsets. It is used in constructing non-measurable sets (Vitali's construction) and in harmonic analysis.
