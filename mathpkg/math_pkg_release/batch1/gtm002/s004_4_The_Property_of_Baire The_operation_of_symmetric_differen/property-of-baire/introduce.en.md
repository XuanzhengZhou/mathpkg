A set has the property of Baire if it differs from some open set by only a set of first category (a "meager" set). This property defines a class of sets that is analogous to the class of measurable sets in measure theory, with first category sets playing the role of nullsets. The class of sets with the property of Baire forms a $\sigma$-algebra containing all open sets and all sets of first category, and it includes all Borel sets.
