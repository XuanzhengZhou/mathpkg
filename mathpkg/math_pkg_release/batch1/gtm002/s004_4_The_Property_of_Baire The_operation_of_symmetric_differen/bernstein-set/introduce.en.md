A Bernstein set is a subset of the real line constructed via transfinite induction using the axiom of choice. It has the remarkable property that both the set and its complement meet every uncountable closed set. This construction, due to F. Bernstein (1908), provides an example of a set that is simultaneously non-measurable in the sense of Lebesgue and lacks the property of Baire, demonstrating that neither class of "nice" sets includes the other.
