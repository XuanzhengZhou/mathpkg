This lemma, used in Bernstein's construction of a non-measurable set, states that every uncountable $G_{\delta}$ set in $\mathbb{R}$ contains a nowhere dense closed subset of measure zero that can be continuously mapped onto $[0,1]$. The proof uses the set of condensation points and a Cantor-like inductive construction of a family of nested closed intervals. The lemma implies in particular that every uncountable closed set (and hence every perfect set) has cardinality $\mathfrak{c}$.
