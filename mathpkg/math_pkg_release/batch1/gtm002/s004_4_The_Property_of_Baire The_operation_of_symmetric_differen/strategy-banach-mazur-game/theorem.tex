A **strategy** for player $(B)$ in the Banach-Mazur game is a sequence of functions $f_1, f_2, \ldots$ where $f_n$ assigns to each possible history $(I_0, I_1, \ldots, I_{2n-1})$ a closed interval $I_{2n} = f_n(I_0, I_1, \ldots, I_{2n-1}) \subset I_{2n-1}$. The strategy is **winning** if $\bigcap I_n \subset B$ for every sequence $\{I_n\}$ consistent with the strategy. A strategy for $(A)$ is defined analogously.
