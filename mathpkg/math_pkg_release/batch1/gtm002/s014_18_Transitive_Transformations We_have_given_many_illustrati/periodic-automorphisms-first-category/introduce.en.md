In the space of measure-preserving automorphisms of the unit square, those that have periodic points in every non-empty open set form a set of first category. This means that the typical measure-preserving automorphism fails to be periodic on any open subset -- it has nonperiodic points everywhere. The proof constructs, for any automorphism with periodic behavior on some small disk, an arbitrarily close perturbation (via an irrational rotation on a concentric annulus) that destroys that periodicity, showing that each $P_{ij}$ is nowhere dense.
