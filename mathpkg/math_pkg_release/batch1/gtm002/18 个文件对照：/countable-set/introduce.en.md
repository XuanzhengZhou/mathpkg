A countable set is a set that is either finite or denumerable (countably infinite). This is the most fundamental size classification in set theory: a set is countable if its cardinality does not exceed that of the natural numbers. The class of countable sets forms a $\sigma$-ideal, meaning it is closed under taking subsets and countable unions. Countable sets serve as the prototypical "small" sets against which measure (nullsets) and category (first category sets) are compared.
