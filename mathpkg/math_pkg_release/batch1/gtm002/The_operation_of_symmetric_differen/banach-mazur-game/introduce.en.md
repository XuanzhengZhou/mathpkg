The Banach-Mazur game is an infinite two-player topological game invented by Stanislaw Mazur around 1928. Players alternately choose nested closed intervals, and player A wins if the intersection point belongs to a prescribed set A. The game provides a game-theoretic characterization of Baire category: the second player has a winning strategy precisely when the set A is of first category.
