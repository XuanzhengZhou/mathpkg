A set has the property of Baire if it differs from an open set by only a set of first category (a meager set). This concept is central to descriptive set theory and provides a topological analogue of measurability in measure theory. The class of sets with the property of Baire forms a $\sigma$-algebra containing all Borel sets and all sets of first category.
