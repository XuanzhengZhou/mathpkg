A Vitali set is a set containing exactly one element from each coset of the rational numbers in the real line. Constructed in 1905 by Giuseppe Vitali, it was the first example of a set of real numbers that is not Lebesgue measurable, demonstrating that the Axiom of Choice implies the existence of non-measurable sets.
