Let $X_i$ consist of the two elements $0$ and $1$, and let $X = \prod_{i=1}^{\infty} X_i$ be the set of all sequences of zeros and ones. A subset $E$ of $X$ is called a \text{tail set} if whenever $x \in E$ and $y$ differs from $x$ in only a finite number of coordinates, then $y \in E$. Equivalently, membership in a tail set depends only on the tail of a sequence $\{x_i\}$: letting

$$X^n = \prod_{i=1}^{n} X_i \quad \text{and} \quad Y^n = \prod_{i=n+1}^{\infty} X_i,$$

$E$ is a tail set if and only if it can be written in the form $E = X^n \times B_n$ for each $n$, where $B_n \subset Y^n$.
