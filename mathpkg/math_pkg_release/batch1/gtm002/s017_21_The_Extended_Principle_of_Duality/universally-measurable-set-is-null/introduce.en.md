This theorem formalizes a fundamental analogy between Lebesgue measure and the Baire category on the real line. If a set of real numbers has the property that every one of its subsets is Lebesgue measurable, then the set itself must be negligible in the measure sense (a nullset). The dual statement holds for the property of Baire: if every subset of a set has the Baire property, then the set is topologically small (of first category). This duality motivates the extended principle of duality between measure and category explored in this chapter.
