Let $C$, or $C[a, b]$, denote the set of all real-valued continuous functions $f$ on the interval $[a, b]$, and define

$$\varrho(f, g) = \sup_{a \leq x \leq b} |f(x) - g(x)|.$$

It is easy to verify that $\varrho$ is a metric in $C$; in particular, the triangle inequality follows from the fact that

$$|f(x) - h(x)| \leq |f(x) - g(x)| + |g(x) - h(x)| \leq \varrho(f, g) + \varrho(g, h)$$

for all $x$ in $[a, b]$. Convergence in this metric means uniform convergence on $[a, b]$. For this reason, $\varrho$ is called the uniform metric.
