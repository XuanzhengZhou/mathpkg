A null function in the context of Riemann integration is a Riemann-integrable function whose absolute value has integral zero. In the space $R[a,b]$ with the integral metric $\sigma(f,g) = \int_a^b |f-g|$, two functions $f$ and $g$ satisfy $\sigma(f,g) = 0$ precisely when $f-g$ is a null function. Identifying null functions (functions that differ by a null function) is the essential step in constructing the metric space $(\tilde{R}, \sigma)$ of R-integrable functions from the pseudo-metric space $R[a,b]$.
