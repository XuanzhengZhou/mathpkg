Let $\mathcal{K}$ be any class of one-to-one nullset-preserving transformations of $\mathbb{R}$ with $|\mathcal{K}| = \mathfrak{c}$. Then there exists a linear set $E \subseteq \mathbb{R}$ of first category and of power $\mathfrak{c}$ such that $TE \triangle E$ is a countable set for every $T \in \mathcal{K}$.
