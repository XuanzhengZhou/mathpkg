By Proposition 20.1, $E$ contains a Lusin set $L$. Since $|L| = \mathfrak{c}$, we have $|L| = |\mathbb{R}|$. Moreover, $|\mathbb{R}| = |\mathbb{R}^2|$, so there exists a one-to-one mapping $f : \mathbb{R}^2 \to L$.

For each $x \in \mathbb{R}$, define $L_x = f(\{x\} \times \mathbb{R}) \subseteq L \subseteq E$. Then the sets $L_x$ are pairwise disjoint (since $f$ is one-to-one and the slices $\{x\} \times \mathbb{R}$ are disjoint). There are $\mathfrak{c}$ such sets (one for each $x \in \mathbb{R}$). Each $L_x$ is an uncountable subset of the Lusin set $L$, hence each $L_x$ is of second category.
