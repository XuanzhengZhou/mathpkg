The first statement is obvious. To prove the second, note that if $A_1$ and $A_2$ are nowhere dense, then for each interval $I$ there is an interval $I_1 \subset I - A_1$ and an interval $I_2 \subset I_1 - A_2$. Hence $I_2 \subset I - (A_1 \cup A_2)$. This shows that $A_1 \cup A_2$ is nowhere dense. Finally, any open interval contained in $A'$ is also contained in $A^{-'}$, so the closure of a nowhere dense set is nowhere dense.
