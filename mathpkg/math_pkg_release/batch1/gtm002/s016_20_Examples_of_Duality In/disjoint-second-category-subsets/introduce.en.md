Proposition 20.3 states that any linear set of second category can be decomposed into $\mathfrak{c}$ (the cardinality of the continuum) pairwise disjoint subsets, each also of second category. The proof uses a Lusin set: embed a Lusin set into $E$ via a bijection, then use the fact that the line and the plane have the same cardinality to decompose the Lusin set into $\mathfrak{c}$ disjoint subsets. This proposition further illustrates the richness of the category structure under the continuum hypothesis.
