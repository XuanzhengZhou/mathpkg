Proposition 20.3* is the measure-theoretic dual of Proposition 20.3. It states that any linear set of positive outer measure contains $\mathfrak{c}$ pairwise disjoint subsets, each also of positive outer measure. A consequence is that any set of positive measure contains $\mathfrak{c}$ disjoint non-measurable subsets (since every set of positive outer measure contains a non-measurable subset, by Theorem 5.5). By Zorn's lemma, this family extends to a maximal disjoint class; its complement must have measure zero, yielding a decomposition of $E$ into $\mathfrak{c}$ disjoint non-measurable subsets.
