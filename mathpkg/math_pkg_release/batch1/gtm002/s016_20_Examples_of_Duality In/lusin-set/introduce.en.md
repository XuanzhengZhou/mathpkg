A Lusin set is an uncountable subset of the real line with the property that every uncountable subset of it is of second category. Equivalently, no uncountable subset of a Lusin set can be covered by countably many first category sets. Lusin sets were introduced by Nikolai Lusin in 1914 and provide a striking example of the duality between measure and category: every Lusin set must have Lebesgue measure zero, since every subset of the line is the union of a nullset and a set of first category. The construction of a Lusin set depends on the continuum hypothesis.
