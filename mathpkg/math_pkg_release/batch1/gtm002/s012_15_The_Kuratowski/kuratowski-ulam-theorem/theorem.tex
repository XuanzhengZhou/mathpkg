\text{If } E \text{ is a plane set of first category, then } E_x \text{ is a linear set of first category for all } x \text{ except a set of first category. If } E \text{ is a nowhere dense subset of the plane } X \times Y \text{, then } E_x \text{ is a nowhere dense subset of } Y \text{ for all } x \text{ except a set of first category in } X.
