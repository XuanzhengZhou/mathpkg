For any linear set $A$ of second category having the property of Baire, there exists a $\delta > 0$ such that $(x + A) \cap A$ contains an interval whenever $|x| < \delta$. For any measurable set $A$ with $m(A) > 0$, there exists a $\delta > 0$ such that $(x + A) \cap A \neq \emptyset$ whenever $|x| < \delta$.
