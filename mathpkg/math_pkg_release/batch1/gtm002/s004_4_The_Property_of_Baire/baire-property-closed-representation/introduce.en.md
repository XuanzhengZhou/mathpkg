Theorem 4.1 establishes a dual characterization of the property of Baire: a set has the Baire property if and only if it can be represented as the symmetric difference of a closed set and a set of first category. This parallels the open-set definition and demonstrates that the roles of open and closed sets are symmetric in this context. The proof uses the fact that the boundary of an open set is nowhere dense to convert between the two representations.
