The property of Baire provides a topological analogue of Lebesgue measurability, with sets of first category playing the role of null sets. A set possesses the property of Baire if it differs from an open set only by a set of first category, expressed via symmetric difference. Together with the operations of symmetric difference and intersection, the class of sets having this property forms a commutative ring and, as shown later, a $\sigma$-algebra containing all Borel sets.
