A lower density is a mapping $\phi$ on the $\sigma$-algebra of a measure space that selects a canonical representative from each equivalence class modulo nullsets. It satisfies: (1) $\phi(A)$ is equivalent to $A$ modulo nullsets, (2) equivalent sets have the same representative, (3) $\phi$ preserves empty set and whole space, (4) $\phi$ preserves intersections, and (5) $\phi$ is monotone. The Lebesgue density theorem (Theorem 3.21) provides the canonical example for the Lebesgue measure space. Lower densities are the key tool for topologizing a measure space into a category measure space.
