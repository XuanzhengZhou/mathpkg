A Liouville number is an irrational real number that can be approximated by rationals to arbitrarily high order: for every $n$, there is a rational $p/q$ (with $q > 1$) within distance $1/q^n$. By Liouville's Approximation Lemma, no algebraic number of degree $>1$ can satisfy this property, so every Liouville number is transcendental. Liouville numbers thus provide the first constructive proof of the existence of transcendental numbers. Their set has interesting category and measure properties: it is residual (its complement is of first category) yet has Lebesgue measure zero.
