A set is called denumerable if its elements can be placed in one-to-one correspondence with the natural numbers. In other words, a denumerable set is countably infinite — it has the same cardinality as $\mathbb{N}$. This notion is fundamental in analysis and topology, as it provides the basis for defining countable sets, which in turn underpin the concepts of measure and Baire category.
