Let $H$ denote the set of all automorphisms of $I = [0,1]$ that leave the endpoints fixed, metrized by $\varrho(g, h) = \max_{x \in I} |g(x) - h(x)|$. For any set $A$ of first category in $I$ there exists an $h \in H$ such that $h(A)$ is a nullset; indeed, such automorphisms constitute a residual set in $H$.
