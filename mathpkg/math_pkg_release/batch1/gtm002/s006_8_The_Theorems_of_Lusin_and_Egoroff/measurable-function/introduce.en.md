A real-valued function on the real line is called measurable if the preimage of every open set is a measurable set. This definition extends the notion of measurability from sets to functions. It is equivalent to requiring that $f^{-1}(B)$ be measurable for every Borel set $B$, or for every set in a countable base of the topology. Measurable functions form the natural class of functions for which integration with respect to Lebesgue measure is well-defined.
