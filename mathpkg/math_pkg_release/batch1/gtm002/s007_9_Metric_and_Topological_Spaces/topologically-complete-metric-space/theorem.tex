A metric space $(X, \varrho)$ is \text{topologically complete} if it is homeomorphic to some complete space. If $f$ is a homeomorphism of $(X, \varrho)$ onto a complete space $(Y, \sigma)$, then $\sigma(f(x), f(y))$ is a metric in $X$ topologically equivalent to $\varrho$. Thus a metric space is topologically complete if and only if it can be remetrized (with a topologically equivalent metric) so as to be complete. An important property of such spaces is that the Baire category theorem still holds.
