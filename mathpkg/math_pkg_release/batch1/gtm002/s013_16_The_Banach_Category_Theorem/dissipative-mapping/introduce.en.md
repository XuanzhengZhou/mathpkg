A mapping is dissipative if it admits a wandering set that is not negligible with respect to the $\sigma$-ideal $\mathcal{I}$. In the measure-theoretic setting, this means a wandering set of positive measure; in the category setting, a wandering set of second category. Dissipativity is the key concept that obstructs the recurrence property.
