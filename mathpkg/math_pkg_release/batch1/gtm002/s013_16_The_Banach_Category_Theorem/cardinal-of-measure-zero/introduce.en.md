A cardinal has measure zero if no nontrivial finite measure can be defined on all subsets of a set of that cardinality. This notion, introduced by Marczewski and Sikorski, connects measure theory with large cardinals. Every cardinal less than the first weakly inaccessible cardinal has measure zero, and only exceedingly large cardinals can fail this property (assuming the continuum hypothesis).
