The set $D(E)$ consists of points in $E$ that eventually leave $E$ forever under iteration of $T$ -- the "nonrecurrent" points. The recurrence property is precisely the statement that $D(E)$ is negligible for every measurable set $E$. This definition provides the formal framework for stating the abstract Poincare recurrence theorem.
