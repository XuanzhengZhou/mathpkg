Lemma 3.7 establishes that for a closed set $F$ contained in a bounded open set $G$, the outer measure is additive across the set difference: $m^*(G-F) = m^*(G) - m^*(F)$. This is a key stepping stone in the development of measurability theory, allowing the outer measure of a "ring" $G-F$ to be expressed as the difference of the measures of its constituents.
