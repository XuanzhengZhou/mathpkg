For any class $\mathcal{C}$ of subsets of a set $X$, there exists a smallest $\sigma$-algebra of subsets of $X$ that contains $\mathcal{C}$, namely, the intersection of all $\sigma$-algebras that contain $\mathcal{C}$. This $\sigma$-algebra is called the \textit{$\sigma$-algebra generated by $\mathcal{C}$}.
