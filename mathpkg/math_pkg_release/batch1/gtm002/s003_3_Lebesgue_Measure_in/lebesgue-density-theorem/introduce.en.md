The Lebesgue Density Theorem states that for any measurable set $E \subset \mathbb{R}$, almost every point of $E$ is a point of density $1$. More precisely, if $\phi(E)$ denotes the set of points where the density of $E$ equals $1$, then the symmetric difference $E \triangle \phi(E)$ has measure zero. This theorem reveals a fundamental regularity property of Lebesgue measurable sets: locally, a measurable set fills up "almost all" of the space near almost all of its points.
