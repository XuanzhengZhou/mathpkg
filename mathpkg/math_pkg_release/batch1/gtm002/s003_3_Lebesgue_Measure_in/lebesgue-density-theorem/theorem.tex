\begin{theorem}[Theorem 3.20 — Lebesgue Density Theorem]
For any measurable set $E \subset \mathbb{R}$, let $\phi(E)$ denote the set of points where $E$ has density $1$. Then

$$
m(E \triangle \phi(E)) = 0.
$$

In other words, almost every point of a measurable set is a point of density $1$ of that set.
\end{theorem}
