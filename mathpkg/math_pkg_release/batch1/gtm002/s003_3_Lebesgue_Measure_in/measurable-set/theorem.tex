A set $A \subset \mathbb{R}^r$ is called \textit{(Lebesgue) measurable} if for every $\varepsilon > 0$ there exist a closed set $F$ and an open set $G$ such that
$$
F \subset A \subset G
$$
and
$$
m^*(G - F) < \varepsilon.
$$

For a measurable set $A$, its \textit{Lebesgue measure} $m(A)$ is defined to be $m^*(A)$.
