\begin{theorem}[Theorem 3.15]
A set $A$ is measurable if and only if it can be represented as an $F_{\sigma}$ set plus a nullset (or as a $G_{\delta}$ set minus a nullset). That is,
$$
A = E \cup N = H \setminus M
$$
where $E$ is an $F_{\sigma}$ set, $H$ is a $G_{\delta}$ set, and $N, M$ are nullsets.
\end{theorem}
