A set of first category (also called a meager set) is a countable union of nowhere dense sets. This notion provides a topological analogue of "smallness" distinct from measure-theoretic smallness. The Baire category theorem states that in a topologically complete metric space, the complement of any first category set is dense, so first category sets are "negligible" from a topological perspective.
