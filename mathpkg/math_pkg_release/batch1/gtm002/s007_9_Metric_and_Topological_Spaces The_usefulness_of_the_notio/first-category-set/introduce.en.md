A set is of first category (or meager) if it is a countable union of nowhere dense sets. In the sense of Baire category, first category sets are considered "small" or "negligible." The complement of a first category set is a residual set, which is "large" in the topological sense. This notion parallels that of null sets in measure theory but is purely topological.
