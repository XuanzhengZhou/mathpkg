A set is nowhere dense if its closure has empty interior. In other words, it does not contain any nonempty open set in its closure. Equivalently, every nonempty open set contains a nonempty open subset disjoint from $A$. Nowhere dense sets are the "small" sets used to define Baire category: a countable union of nowhere dense sets is called meager or of first category.
