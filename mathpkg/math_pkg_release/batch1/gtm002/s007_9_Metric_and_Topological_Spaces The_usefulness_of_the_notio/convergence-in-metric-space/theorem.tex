A sequence $x_1, x_2, \ldots$ of points of a metric space $(X, \varrho)$ is said to \textbf{converge} to the point $x$ if $\varrho(x_n, x) \to 0$ as $n \to \infty$. We then write $x_n \to x$. A sequence is \textbf{convergent} if it converges to some point of $X$.

Convergence is a topological property: $x_n \to x$ if and only if every open set that contains $x$ contains all but a finite number of terms of the sequence.
