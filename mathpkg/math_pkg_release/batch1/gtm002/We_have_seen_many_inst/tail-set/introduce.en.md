A tail set in the product space $X = \{0,1\}^{\mathbb{N}}$ is a set whose membership depends only on the asymptotic behavior of sequences: changing finitely many coordinates of a sequence never changes membership. In other words, a tail set $E$ can be expressed as $E = X^n \times B_n$ for every $n$, where $B_n$ is a subset of the tail product $Y^n = \prod_{i=n+1}^{\infty} X_i$. Tail sets are the natural domain of zero-one laws in both measure and category.
