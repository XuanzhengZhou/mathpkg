A set has the property of Baire if it differs from some open set by only a set of first category (a "meager" set). Equivalently, it can be expressed as the symmetric difference of an open set and a first-category set. The class of sets with this property forms a $\sigma$-algebra containing all open sets and all first-category sets, making it the smallest such $\sigma$-algebra. This class is the topological analogue of the class of measurable sets, with first-category sets playing the role of nullsets.
