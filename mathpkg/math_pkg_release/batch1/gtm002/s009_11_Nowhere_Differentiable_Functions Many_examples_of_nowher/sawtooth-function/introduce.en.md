The saw-tooth function $\phi(x) = \min(x - [x], [x] + 1 - x)$ measures the distance from a real number $x$ to the nearest integer. It is continuous, piecewise-linear, and periodic with period $1$, taking values in $[0, 1/2]$. Its graph resembles a sequence of triangular teeth. The function has right derivative $\pm 1$ at every non-integer point. When scaled by a large frequency factor $m$, the function $\phi(mx)$ oscillates rapidly, producing a right derivative of magnitude $m$ almost everywhere. This property is exploited in Banach's proof: adding a small multiple of a rapidly oscillating saw-tooth to any piecewise-linear function produces a function with arbitrarily large one-sided derivative, thereby avoiding the sets $E_n$.
