Let $C$ be the space of continuous functions on $[0, 1]$ equipped with the uniform metric. For each positive integer $n$, let

$$E_n = \{ f \in C : \exists\, x \in [0, 1 - 1/n] \text{ such that } |f(x + h) - f(x)| \leq nh \text{ for all } 0 < h < 1 - x \}.$$

Then each $E_n$ is closed and nowhere dense in $C$. Consequently, the set

$$E = \bigcup_{n=1}^{\infty} E_n$$

is of first category in $C$. This set $E$ consists precisely of those continuous functions that have a bounded right difference quotient at some point of $[0, 1)$. By symmetry (substituting $1 - x$ for $x$), the set of functions with a bounded left difference quotient at some point is also of first category. The union of these two sets, which is still of first category, contains every continuous function that possesses a finite one-sided derivative at any point of $[0, 1]$. Hence, in the sense of Baire category, almost all continuous functions on $[0, 1]$ are nowhere differentiable.
