This lemma shows that each $E_n$ is nowhere dense in $C[0,1]$. The proof is constructive: given any piecewise-linear continuous function $g$ and any $\varepsilon > 0$, one perturbs $g$ by adding a rapidly oscillating saw-tooth function $\varepsilon \phi(m x)$ with sufficiently large frequency $m$. The perturbation ensures the resulting function $h$ has a right derivative numerically exceeding $n$ everywhere, placing $h$ outside $E_n$ while remaining within distance $\varepsilon/2$ of $g$. Together with the previous lemma that $E_n$ is closed, this yields that $E = \bigcup_n E_n$ is of first category.
