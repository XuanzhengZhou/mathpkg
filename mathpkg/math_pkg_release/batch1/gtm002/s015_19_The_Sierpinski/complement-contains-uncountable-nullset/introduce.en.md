Theorem 19.1 establishes that the complement of any Lebesgue nullset on the real line contains a nullset of cardinality $\mathfrak{c}$, and similarly the complement of any set of first category contains a first category set of power $\mathfrak{c}$. This result shows that neither class is "small" in the sense of being contained in a set whose complement is also small. The theorem is an immediate consequence of the fact that every uncountable closed set contains a closed nullset of power $\mathfrak{c}$ (Lemma 5.1) and that the complement of a set of first category contains an uncountable $G_\delta$ set.
