By Theorem 3.18, the complement of a nullset contains an uncountable closed set. By Lemma 5.1, this contains a closed nullset of power $\mathfrak{c}$.

The complement of a set of first category contains an uncountable $G_{\delta}$ set. By Lemma 5.1, this contains a nowhere dense set of power $\mathfrak{c}$.
