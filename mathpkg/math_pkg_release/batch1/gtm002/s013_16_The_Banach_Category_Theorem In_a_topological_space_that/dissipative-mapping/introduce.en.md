A mapping $T$ is dissipative if it has a nontrivial wandering set (one not in the $\sigma$-ideal $\mathcal{I}$), meaning some set of "significant size" is pushed away and never returns under iteration. Otherwise $T$ is nondissipative — every wandering set is negligible in the sense of $\mathcal{I}$. This dichotomy is the key condition in the abstract recurrence theorem: nondissipative mappings are exactly those for which the recurrence property holds.
