Let $X$ be a metric space with a base whose cardinal has measure zero. Let $\mu$ be a nonatomic Borel measure in $X$ such that
\begin{enumerate}
\item every set of infinite measure has a subset with positive finite measure, and
\item every set of measure zero is contained in a $G_\delta$ set of measure zero.
\end{enumerate}
Then $X$ can be represented as the union of a $G_\delta$ set of measure zero and a set of first category.
