In an ordinary Baire space, a set of first category may have a nonempty interior and thus fail to be nowhere dense. However, in a category measure space over a regular Baire space, the condition $\mu(E) = 0 \iff E$ is first category forces every first-category set to be nowhere dense. This is a fundamental structural property that distinguishes category measure spaces.
