A simple but illuminating observation: any nowhere dense closed subset of $[0,1]$ can be transformed into a Lebesgue nullset by stretching the complementary open intervals to fill the unit interval. The explicit homeomorphism $h(x) = m([0,x] \cap G)/m(G)$ maps the dense open set $G$ onto a set of full measure, compressing the original closed nowhere dense set into a nullset. This trivial construction serves as the foundation for the deeper generalization to arbitrary first category sets.
