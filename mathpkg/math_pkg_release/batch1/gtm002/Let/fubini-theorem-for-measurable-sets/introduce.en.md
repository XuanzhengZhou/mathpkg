Theorem 14.3 extends Fubini's theorem from nullsets to general measurable sets in the plane. Using the fact that any measurable set can be expressed as the union of an $F_\sigma$ set and a nullset (Theorem 3.15), the result for nullsets (Theorem 14.2) together with the fact that sections of $F_\sigma$ sets are $F_\sigma$ (hence measurable) yields that almost all sections of a plane measurable set are linearly measurable.
