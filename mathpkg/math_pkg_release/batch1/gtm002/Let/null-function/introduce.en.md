A null function in the context of Riemann-integrable functions is one whose absolute value has zero integral. Under the integral metric $\sigma(f,g) = \int_a^b |f(x)-g(x)| \, dx$, null functions are precisely those at distance zero from the zero function. Identifying functions that differ by a null function turns the pseudo-metric space $R[a,b]$ into the genuine metric space $(\tilde{R}, \sigma)$ of R-integrable functions. This is an early instance of the quotient-by-null-sets construction that later becomes standard in Lebesgue integration theory.
