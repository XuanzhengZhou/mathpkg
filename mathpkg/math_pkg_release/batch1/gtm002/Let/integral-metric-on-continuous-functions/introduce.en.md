The integral metric on $C[a,b]$ is defined as $\sigma(f,g) = \int_a^b |f(x) - g(x)| \, dx$, the $L^1$ distance between continuous functions on a closed interval. Unlike the uniform metric, convergence in this metric is convergence in mean, not pointwise or uniform. As shown in the text, $(C[a,b], \sigma)$ is not a complete metric space, contrasting sharply with the uniform metric on the same set of functions.
