Any pseudo-metric space can be canonically turned into a genuine metric space through a quotient construction. Define an equivalence relation by $x \sim y$ iff $\varrho(x,y) = 0$; then the distance function descends to a well-defined metric on the set of equivalence classes $\tilde{X} = X/{\sim}$. This construction is used throughout analysis: for example, the space $R[a,b]$ of Riemann-integrable functions with the integral metric becomes a proper metric space $(\tilde{R}, \sigma)$ after identifying functions that differ by a null function. The same idea underlies the construction of $L^p$ spaces in measure theory.
