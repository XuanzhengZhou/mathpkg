A pseudo-metric space $(X, \varrho)$ can always be made into a metric space by identifying points $x$ and $y$ whenever $\varrho(x, y) = 0$. This defines an equivalence relation in $X$, and the value of $\varrho(x, y)$ depends only on the equivalence classes to which $x$ and $y$ belong. Taking these equivalence classes as the elements of a set $\tilde{X}$, the pair $(\tilde{X}, \varrho)$ is a metric space.
