Theorem 7.4 provides a simple characterization: a function's set of discontinuity points has first category exactly when the function is continuous on a dense set. This follows immediately from Theorem 7.1 (the discontinuity set is $F_\sigma$) and the general fact that an $F_\sigma$ set is of first category if and only if its complement is dense. The result connects the topological size of the discontinuity set with the distribution of continuity points.
