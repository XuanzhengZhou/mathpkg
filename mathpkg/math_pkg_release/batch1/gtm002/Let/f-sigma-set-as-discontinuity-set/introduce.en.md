Theorem 7.2 provides the converse to Theorem 7.1: not only is every set of discontinuity points an $F_\sigma$ set, but every $F_\sigma$ set can be realized as the discontinuity set of some bounded function. The construction writes $E$ as a union of an increasing sequence of closed sets $F_n$, and uses indicator-like functions with carefully chosen coefficients to produce the desired oscillation pattern.
