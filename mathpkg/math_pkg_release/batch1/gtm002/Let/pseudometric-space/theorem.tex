A set $X$ with a distance function $\varrho$ that satisfies the first three axioms of a metric (non-negativity, symmetry, and triangle inequality) but not necessarily the fourth ($\varrho(x,y) = 0 \implies x = y$) is called a \text{pseudometric space}. Such a space can be made into a metric space by identifying points $x$ and $y$ whenever $\varrho(x,y) = 0$. This defines an equivalence relation in $X$, and the value of $\varrho(x,y)$ depends only on the equivalence classes to which $x$ and $y$ belong. Taking these classes as elements yields a metric space $(\tilde{X}, \varrho)$.
