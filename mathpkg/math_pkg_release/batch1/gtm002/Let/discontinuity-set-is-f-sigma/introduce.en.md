Theorem 7.1 establishes that the set of discontinuity points of any real-valued function on the real line is always an $F_\sigma$ set (a countable union of closed sets). This follows from the observation that the set $\{x : \omega(x) < \varepsilon\}$ is open for any $\varepsilon > 0$. The result places a topological restriction on how badly a function can behave: the discontinuity set must be a countable union of closed sets, which is a fundamental structural result in real analysis.
