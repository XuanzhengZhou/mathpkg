If $\omega(x_0) < \varepsilon$, then $\omega(x) < \varepsilon$ for all $x$ in a neighborhood of $x_0$. Hence the set $\{x : \omega(x) < \varepsilon\}$ is open. The set $D$ of all points at which $f$ is discontinuous can be represented in the form

$$D = \bigcup_{n=1}^{\infty} \{x : \omega(x) \geq 1/n\},$$

hence $D$ is always an $F_\sigma$ set.
