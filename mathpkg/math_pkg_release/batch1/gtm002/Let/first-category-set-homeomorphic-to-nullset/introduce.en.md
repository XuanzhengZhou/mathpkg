Theorem 13.1 is a striking result linking the ideals of first category sets and nullsets: any set of first category in $[0,1]$ can be transformed into a Lebesgue nullset by some homeomorphism of the interval. Moreover, the set of homeomorphisms that accomplish this is residual (comeager) in the space $H$ of automorphisms. The proof uses a category argument: for a first category set $A = \bigcup A_n$, the set of homeomorphisms mapping each $\bar{A}_n$ to a set of arbitrarily small measure is open and dense in $H$.
