By Theorem 3.15, $E$ can be represented as the union of an $F_\sigma$ set $A$ and a nullset $N$. We have $E_x = A_x \cup N_x$ for all $x$. Any section of a closed set is closed, hence $A_x$ is an $F_\sigma$ (and thus measurable) for every $x$. By Fubini's theorem for nullsets (Theorem 14.2), $N_x$ is a nullset for all $x$ except a set of linear measure zero. Therefore, for almost all $x$, $E_x$ is the union of an $F_\sigma$ set and a nullset, hence measurable.
