Theorem 7.4 provides a clean necessary and sufficient condition: a function has a first-category discontinuity set exactly when its points of continuity form a dense subset of $\mathbb{R}$. This follows immediately from Theorem 7.1 and the fact that an $F_\sigma$ set is of first category if and only if its complement is dense. The result elegantly links the topological size of the discontinuity set with the structure of the continuity set.
