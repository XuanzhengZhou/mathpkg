Let $R[a, b]$ denote the set of Riemann integrable functions on $[a, b]$, with the metric $\sigma(f, g) = \int_a^b |f(x) - g(x)| \, dx$. This is a pseudometric space. Identifying functions that differ by a \emph{null function} (a function $f$ such that $\int_a^b |f(x)| \, dx = 0$) yields a metric space $(\tilde{R}, \sigma)$, called the \emph{space of $R$-integrable functions} on $[a, b]$.
