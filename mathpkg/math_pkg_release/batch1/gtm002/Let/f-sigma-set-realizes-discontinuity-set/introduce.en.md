Theorem 7.2 is the converse of Theorem 7.1. It shows that the $F_\sigma$ characterization is sharp: given any $F_\sigma$ set $E$, one can construct a bounded real-valued function whose set of discontinuity points is exactly $E$. The construction uses a series of specially crafted functions supported on closed sets whose union is $E$, with carefully chosen coefficients to control the oscillation at each point of $E$.
