A function of the first Baire class is defined as the pointwise limit of a sequence of continuous functions. Although such functions need not be continuous everywhere, Baire's theorem limits how discontinuous they can be: their discontinuity sets are always of first category. Simple examples include the pointwise limit of $f_n(x) = \max(0, 1 - n|x|)$, which converges everywhere to an indicator function with one discontinuity.
