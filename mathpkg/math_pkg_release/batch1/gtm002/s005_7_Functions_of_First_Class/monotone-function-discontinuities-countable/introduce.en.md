Theorem 7.8 characterizes the discontinuity sets of monotone functions: a monotone function can have at most countably many discontinuities, and conversely any countable set can be realized as the discontinuity set of some monotone function. The forward direction follows because there can be at most $|f(b) - f(a)|/\varepsilon$ points in $(a, b)$ with oscillation at least $\varepsilon$. The converse constructs a monotone function with prescribed jump discontinuities as $f(x) = \sum_{x_i \leq x} \varepsilon_i$ where $\sum \varepsilon_i$ converges.
