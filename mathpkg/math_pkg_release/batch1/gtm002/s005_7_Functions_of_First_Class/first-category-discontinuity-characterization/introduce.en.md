Theorem 7.4 gives a clean characterization: a real-valued function has its discontinuity set of first category if and only if it is continuous on a dense set of points. This follows immediately from Theorem 7.1 (the discontinuity set is $F_\sigma$) and the general fact that an $F_\sigma$ set is of first category precisely when its complement is dense. This result provides a useful necessary and sufficient condition that can be checked by examining the density of continuity points.
