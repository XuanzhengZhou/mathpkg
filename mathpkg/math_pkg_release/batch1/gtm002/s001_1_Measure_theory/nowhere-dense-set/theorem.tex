A set $A \subset \mathbb{R}$ is \textit{nowhere dense} if it is dense in no interval; that is, if every interval has a subinterval contained in the complement of $A$. Equivalently:

\begin{enumerate}
  \item $A$ is nowhere dense if and only if its complement $A'$ contains a dense open set.
  \item $A$ is nowhere dense if and only if $\overline{A}$ (the closure of $A$) has no interior points.
\end{enumerate}
