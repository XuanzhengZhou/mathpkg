\begin{enumerate}
  \item Any subset of a set of first category is of first category.
  \item The union of any countable family of first category sets is of first category.
\end{enumerate}
$$\\text{However, the closure of a set of first category is not in general of first category.}$$
In fact, the closure of a linear set $A$ is of first category if and only if $A$ is nowhere dense.
