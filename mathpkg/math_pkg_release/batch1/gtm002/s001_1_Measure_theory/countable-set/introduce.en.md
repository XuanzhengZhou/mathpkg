A countable set is one whose cardinality does not exceed that of the natural numbers: it is either finite or countably infinite (denumerable). This concept is central to both measure theory and Baire category theory, as both the class of nullsets and the class of first category sets include all countable sets. Examples include the rational numbers $\mathbb{Q}$ (denumerable) and any finite set.
