Let $L$ be a first-order language. Consider the set $\mathbf{S}$ of all $L$-structures of bounded cardinality $\leq \operatorname{card} L + \aleph_0$. Consider the quotient
$$\mathcal{S} = \mathbf{S} / {\equiv},$$
where $\equiv$ denotes elementary equivalence between $L$-structures. For each $L$-sentence $P$, define the basic open set
$$[P] = \{ \mathbf{A} \in \mathbf{S} : \mathbf{A} \vDash P \}.$$
The topology on $\mathcal{S}$ is generated by taking the sets $[P]$ as an open basis. The resulting topological space $\mathcal{S}$ is called the \textbf{Stone space of $L$-structures}.
