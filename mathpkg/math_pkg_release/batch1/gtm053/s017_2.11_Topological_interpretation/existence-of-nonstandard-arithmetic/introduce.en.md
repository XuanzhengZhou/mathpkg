The existence of nonstandard arithmetics is proved by expanding the language with a new constant symbol $c$ and considering the set of sentences $\mathcal{E} = \operatorname{Th}(\mathbf{N}) \cup \{\neg c = \bar{n} : n = 0, 1, \ldots\}$. Any finite subset of $\mathcal{E}$ is satisfiable (by interpreting $c$ as a sufficiently large standard integer), so by the compactness theorem $\mathcal{E}$ has a model. Reducing this model to the original language yields a structure elementarily equivalent to $\mathbf{N}$ but containing an element different from every standard $n$---a nonstandard arithmetic.
