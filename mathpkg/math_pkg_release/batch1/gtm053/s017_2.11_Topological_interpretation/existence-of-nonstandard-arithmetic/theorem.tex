Consider the language $L_1\mathrm{Ar}(c)$ obtained by extending $L_1\mathrm{Ar}$ with a new constant symbol $c$, and the set of sentences
$$\mathcal{E} = \operatorname{Th}(\mathbf{N}) \cup \{ \neg c = \bar{n} : n = 0, 1, \ldots \}.$$
Then $\mathcal{E}$ is finitely satisfiable, hence by the compactness theorem has a model. Any such model, reduced to the language $L_1\mathrm{Ar}$, is a nonstandard arithmetic.
