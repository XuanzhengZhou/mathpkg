For an $L$-sentence $P$, the \textbf{basic open set} $[P]$ in the Stone space $\mathcal{S}$ of $L$-structures is defined as
$$[P] = \{ \mathbf{A} \in \mathbf{S} : \mathbf{A} \vDash P \},$$
where $\mathbf{S}$ is the set of all $L$-structures of bounded cardinality $\leq \operatorname{card} L + \aleph_0$, and the corresponding equivalence class in $\mathcal{S} = \mathbf{S}/{\equiv}$ is taken.
