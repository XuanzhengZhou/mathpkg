Let $\mathbf{N} = (\mathbb{N}, +, \cdot, 0, 1)$ be the standard structure of nonnegative integers in the language of arithmetic $L_1\mathrm{Ar}$. The theory $\operatorname{Th}(\mathbf{N})$, the set of all $L_1\mathrm{Ar}$-sentences true in $\mathbf{N}$, is called \textbf{complete arithmetic}. This is to be distinguished from Peano arithmetic, which is given by a recursive system of axioms and is incomplete.
