In the compact Hausdorff Stone space $\mathcal{S}$ of $L$-structures, every ultrafilter $U$ on a set of points $I \subseteq \mathcal{S}$ has a unique limit point $\lim_U I$. This limit point is exactly the elementary equivalence class of the ultraproduct $\prod_{i \in I} \mathbf{A}_i / U$, where each $\mathbf{A}_i$ is a representative of the corresponding point in $I$. Thus, the ultraproduct construction provides the concrete realization of the abstract topological limit along an ultrafilter.
