Suppose that the axiom of extensionality and the axiom of the empty set are $\phi$-true in a class $N \subset V$ with a binary relation $\varepsilon \subset N \times N$, where $[X] = \{Y \mid Y \varepsilon X\}$ is a set for all $X \in N$. Then there exists a unique transitive class $M$ and a unique isomorphism $f: N \to M$ taking $\varepsilon$ to the standard membership relation $\in$.
