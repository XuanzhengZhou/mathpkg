An $L$-sentence $S$ is a 	ext{logical consequence} of a finite set $E$, written $E \models S$, if $A \models E$ implies $A \models S$ for every $L$-structure $A$. For infinite $E$, $E \models S$ means that there is a finite $E^0 \subset E$ such that $E^0 \models S$.
$S$ is called 	ext{logically valid}, written $\models S$, if $A \models S$ for every $L$-structure $A$.
