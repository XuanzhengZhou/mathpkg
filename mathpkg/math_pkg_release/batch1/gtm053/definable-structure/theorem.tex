A language $L_1$ is 	ext{interpreted} in an $L_0$-structure $A$ if there are $L_0$-formulas defining: a domain set $Q(A)$, an equivalence relation $E(A)$ on $Q(A)$, and for each relation symbol of $L_1$, a corresponding relation on $Q(A)$ preserved by $E(A)$. The interpreted structure has domain $Q(A)/E(A)$ with the induced relations.
