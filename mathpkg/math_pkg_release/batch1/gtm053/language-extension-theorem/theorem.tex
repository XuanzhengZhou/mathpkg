Let $\mathcal{E}' \vdash \exists!x P'(x, y_1, \ldots, y_n)$. Adding a new function symbol $f$ of degree $n$ and the axiom $\forall y_1 \cdots \forall y_n P'(f(y_1,\ldots,y_n), y_1,\ldots,y_n)$ to obtain an extended language $L$ and theory $\mathcal{E}$ is a conservative extension: the set of formulas in the original language $L'$ that are deducible from $\mathcal{E}'$ coincides with the set of $L'$-formulas deducible from $\mathcal{E}$.
