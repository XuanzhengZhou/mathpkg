Let $\mathcal{E}$ be a finite set of formulas in $L$, and let $P$ be a logical polynomial (with a fixed representation) over $\mathcal{E}$. For any map $v : \mathcal{E} \to \{0,1\}$, extend $v$ to $P$ using the same rules that defined the truth function $|\cdot|$ in Section 2.5, and set

$$P^v = \begin{cases} P, & \text{if } v(P) = 1, \\ \neg P, & \text{if } v(P) = 0. \end{cases}$$

Let $\mathcal{E}^v = \{Q^v \mid Q \in \mathcal{E}\}$. Then for any $v$ we have

$$\mathcal{F} \cup \mathcal{E}^v \vdash P^v$$

using only the rule of deduction MP.
