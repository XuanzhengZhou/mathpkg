Let $X$ be a set with a binary relation $<$. The relation is:
\begin{itemize}
\item a \text{partial ordering} if it is irreflexive ($Y \not< Y$) and transitive;
\item a \text{linear ordering} if it is a partial ordering and any two elements are comparable;
\item a \text{well-ordering} if it is a linear ordering and every nonempty subset has a least element.
\end{itemize}
