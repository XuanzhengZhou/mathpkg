Von Neumann's elegant definition of ordinals avoids the need for equivalence classes of well-orderings. Instead, each ordinal is a canonical well-ordered set where the ordering is given by the membership relation itself. This definition makes ordinals into concrete sets: $0 = \emptyset$, $1 = \{\emptyset\}$, $2 = \{\emptyset, \{\emptyset\}\}$, and so on, with each ordinal being the set of all smaller ordinals.
