A set of sentences is finitely satisfiable if every finite subset has a model. This notion is central to the compactness theorem of first-order logic, which states that a finitely satisfiable set of sentences is itself satisfiable (has a model). Finite satisfiability provides the key hypothesis for applying compactness arguments throughout model theory.
