An embedding between $L$-structures is always injective. This basic property follows from the preservation of equality as a logical relation: if two elements map to the same element, the embedding conditions force them to be equal in the domain. More directly, the identity relation $x = y$ can be seen as a binary relation preserved by embeddings, which forces injectivity.
