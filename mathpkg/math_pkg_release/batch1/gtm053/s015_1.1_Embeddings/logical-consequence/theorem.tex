Let $E$ be a set of $L$-sentences. Write $A \vDash E$ if $A \vDash S$ for every $S \in E$.

An $L$-sentence $S$ is a logical consequence of $E$, written $E \vDash S$, if $A \vDash E$ implies $A \vDash S$ for every $L$-structure $A$. For infinite $E$, $E \vDash S$ means there exists a finite $E^0 \subset E$ such that $E^0 \vDash S$.
