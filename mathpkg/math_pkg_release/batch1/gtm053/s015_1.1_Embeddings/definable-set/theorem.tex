For an $L$-structure $A$ and an $L$-formula $P(x_1, \ldots, x_n)$ one defines the set
$$P(A) = \{ \bar{a} \in A^n : A \vDash P(\bar{a}) \}.$$
Sets of this form are called definable.

Since any subset of $A^n$ can be viewed as an $n$-ary relation, $P(\bar{v})$ determines also an $L$-definable relation. If $P(A)$ coincides with the graph of an operation $f : A^{n-1} \rightarrow A$, we say that $f$ is an $L$-definable operation.
