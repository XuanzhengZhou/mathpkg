Formulas are the expressions in a first-order language that represent statements or propositions about the domain of discourse. They are built from atomic formulas (formed by applying relation symbols to terms) using the propositional connectives ($\Leftrightarrow, \Rightarrow, \vee, \wedge, \neg$) and the quantifiers ($\forall, \exists$). The recursive definition ensures that the set of formulas is the smallest set closed under these construction rules.
