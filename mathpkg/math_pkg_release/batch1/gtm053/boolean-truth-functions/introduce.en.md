Boolean truth functions generalize the two-valued truth functions by allowing truth values in an arbitrary Boolean algebra B. When B = {0,1}, they reduce to the standard truth functions. The Boolean-valued approach unifies the semantics of classical logic with the technique of forcing: Cohen's forcing can be viewed as interpreting formulas in a suitable complete Boolean algebra, with "forcing" corresponding to the truth value being 1.
