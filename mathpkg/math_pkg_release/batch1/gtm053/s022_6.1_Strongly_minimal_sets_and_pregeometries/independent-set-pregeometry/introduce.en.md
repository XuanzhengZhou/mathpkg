In a combinatorial pregeometry, a set is independent if none of its elements can be generated by (is in the closure of) the others. This generalizes linear independence in vector spaces and algebraic independence in fields. Independent sets serve as the building blocks for the dimension theory: any maximal independent subset of a set forms a basis, and all bases have the same cardinality (the dimension). The independence notion is central to geometric stability theory's analysis of strongly minimal structures.
