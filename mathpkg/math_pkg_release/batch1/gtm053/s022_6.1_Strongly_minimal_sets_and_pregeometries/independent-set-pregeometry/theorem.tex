A set $U \subseteq M$ is said to be \emph{independent} (with respect to a pregeometry $(M, \operatorname{cl})$) if for every $u \in U$,
$$u \notin \operatorname{cl}(U \setminus \{u\}).$$
Equivalently, no element of $U$ belongs to the closure of the remaining elements. A set is \emph{dependent} if it is not independent.
