The (von Neumann) universe $V$ is the class of sets $\bigcup_{\alpha \in \text{On}} V_{\alpha}$, where the set $V_{\alpha}$ is defined by the following transfinite recursion:

$$V_0 = \varnothing,$$
$$V_{\alpha+1} = \mathcal{P}(V_{\alpha}),$$
$$V_{\alpha} = \bigcup_{\beta < \alpha} V_{\beta}, \quad \text{if } \alpha \text{ is a limit ordinal.}$$
