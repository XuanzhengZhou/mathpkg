Given a well-ordered set $(X, <)$, the initial segment determined by an element $Y$ is the set of all elements strictly less than $Y$, equipped with the restricted ordering. Initial segments play a crucial role in comparing well-ordered sets and in the definition of ordinals, where every ordinal is precisely the set of all smaller ordinals.
