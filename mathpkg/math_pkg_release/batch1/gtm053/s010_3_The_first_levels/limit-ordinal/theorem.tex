An ordinal $\beta$ is called a \textit{limit ordinal} if $\beta \neq \varnothing$ and $\beta \neq \alpha + 1$ for any $\alpha$.

The first limit ordinal $\omega_0$ is isomorphic as a totally ordered set to $\{0, 1, 2, 3, \ldots, n, \ldots\}$.

If $\alpha$ is a limit ordinal, then

$$\alpha = \bigcup_{\beta < \alpha} \beta.$$

The converse is also true.
