Let $X$ be any set on which a binary relation $<$ is given. The relation $<$ is a \textit{well-ordering} of $X$ if it satisfies all three conditions:

\begin{enumerate}
\item[(a)] $Y \not< Y$ for all $Y \in X$.
\item[(b)] $Y < Z$ or $Y = Z$ or $Z < Y$ for any $Y, Z \in X$.
\item[(c)] Every nonempty subset of $X$ has a least element with respect to $<$.
\end{enumerate}
