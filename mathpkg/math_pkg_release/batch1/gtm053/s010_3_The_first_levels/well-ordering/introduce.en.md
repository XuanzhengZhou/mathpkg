A well-ordering is a linear ordering with the additional property that every nonempty subset possesses a least element. This powerful condition enables transfinite induction and recursion, which are fundamental tools in set theory. Well-orderings are the key structural property used to define ordinals and to prove that any set can be well-ordered (Zermelo's theorem).
