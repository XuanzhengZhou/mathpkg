Given an infinite cardinal $\kappa$, a structure $\mathbf{A}$ is called $\kappa$-saturated if for any cardinal $\lambda < \kappa$ and for any expansion $\mathbf{A}_C$ of $\mathbf{A}$ by constant symbols $C = \{c_i : i \leq \lambda\}$, every $1$-type in $\text{Th}(\mathbf{A}_C)$ is realized in $\mathbf{A}_C$.
