A structure $\mathbf{A}$ is $\kappa$-saturated when it realizes every $1$-type over any expansion by fewer than $\kappa$ constant symbols. In other words, for any set $C$ of parameters from $A$ with $|C| < \kappa$, every complete $1$-type consistent with $\text{Th}(\mathbf{A}_C)$ is already realized by some element of $A$. This is a fundamental largeness/completeness property in model theory.
