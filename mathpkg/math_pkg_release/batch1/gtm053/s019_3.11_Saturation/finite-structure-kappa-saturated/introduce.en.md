Every finite structure is trivially $\kappa$-saturated for every cardinal $\kappa$. This holds because in a finite structure there are only finitely many elements, so any $1$-type over finitely many constant symbols names one of those elements and is therefore realized. The saturation condition is vacuous or trivial for finite structures, which underscores that saturation is a meaningful concept only for infinite models.
