Given a saturated model $\mathbf{A}$ of a complete theory $T$, the following two conditions are equivalent:

\begin{enumerate}
  \item[(i)] For any two $n$-tuples $\bar{a}$ and $\bar{b}$ in $\mathbf{A}$,
  $$\text{qftp}(\bar{a}) = \text{qftp}(\bar{b}) \quad \text{iff there is } \pi \in \text{Aut}(\mathbf{A}) \text{ such that } \pi : \bar{a} \mapsto \bar{b}.$$
  \item[(ii)] Any $L$-formula with $n$ free variables is equivalent modulo $T$ to a quantifier-free $L$-formula.
\end{enumerate}
