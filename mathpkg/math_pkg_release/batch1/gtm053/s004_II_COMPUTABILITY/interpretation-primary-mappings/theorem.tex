Let $L$ be a language in $\mathfrak{L}_1$ and $M$ an interpretation set. The primary mappings of an interpretation $\phi$ are:

(a) An interpretation of the constants is a map from the set of constant symbols to $M$ that takes a symbol $c$ to $\phi(c) \in M$.

(b) An interpretation of the operations is a map from the set of operation symbols that takes a symbol $f$ of degree $r$ to a function $\phi(f) : M^r \to M$.

(c) An interpretation of the relations is a map from the set of relation symbols that takes a symbol $p$ of degree $r$ to a subset $\phi(p) \subset M^r$.
