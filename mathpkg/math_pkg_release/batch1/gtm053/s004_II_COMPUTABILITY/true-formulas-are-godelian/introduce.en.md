This proposition establishes that the set of true formulas in any interpretation satisfies the syntactic properties defining a Gödelian set. The converse -- that every Gödelian set is the set of true formulas in some interpretation -- is Gödel's completeness theorem (proved later in the book). Together, these results show that the concept of a Gödelian set is the closest purely syntactic approximation to the semantic notion of truth.
