A \text{modular structure} $L$ is a set with binary operations $\wedge$ and $\vee$ satisfying:
\begin{enumerate}
\item[(a)] $\wedge$ and $\vee$ are associative and commutative.
\item[(b)] $a \wedge a = a \vee a = a$ for all $a \in L$ (idempotence).
\item[(c)] If $a \wedge b = b$, then $(a \vee c) \wedge b = a \vee (c \wedge b)$ (modular law).
\end{enumerate}
The set $B(\mathcal{H})$ of closed subspaces of a Hilbert space with intersection and closed linear sum forms a modular structure.
