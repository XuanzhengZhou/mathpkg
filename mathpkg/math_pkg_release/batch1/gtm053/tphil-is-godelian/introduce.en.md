A set of formulas is called Gödelian if it satisfies the four conditions: completeness, consistency, closure under deduction rules, and containment of logical axioms. This proposition establishes that the set of semantically true formulas in any interpretation always satisfies these syntactic conditions. The converse—that every Gödelian set arises as the set of true formulas in some interpretation—is Gödel's Completeness Theorem, proved in Section 6.
