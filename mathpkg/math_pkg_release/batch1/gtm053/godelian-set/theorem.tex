A set of formulas $\mathcal{E}$ in a language $L$ is called \text{Godelian} if it is complete (for every closed formula $P$, either $P \in \mathcal{E}$ or $\neg P \in \mathcal{E}$), does not contain a contradiction, is closed with respect to the rules of deduction MP and Gen, and contains all the logical axioms Ax $L$.
