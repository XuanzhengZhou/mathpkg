(a) For fixed $\bar{x}$, the function $\bar{h}(\bar{x})$ is a random variable (measurable function).
(b) $\|\bar{h}(\bar{x}) = 0\| = \bigvee_{j \in \mathcal{J}} \|\bar{x} = \bar{x}_j\|$, where $\bar{x}_j$ are coordinate projections indexed by $\mathcal{J} \subset I$ with $\omega_0 < \operatorname{card} \mathcal{J} < \operatorname{card} I$.

Moreover, $\|\bar{x} = \bar{y}\| \wedge \|\bar{h}(\bar{x}) = 0\| \leqslant \|\bar{h}(\bar{y}) = 0\|$ (extensionality condition).
