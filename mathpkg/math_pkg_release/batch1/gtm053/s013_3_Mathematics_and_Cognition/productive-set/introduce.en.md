A productive set is one for which there exists an effective method to produce, given any recursively enumerable subset (presented via its index in a versal family), an element of the set that lies outside that enumerable subset. This notion formalizes the "diagonal" argument in Godel's incompleteness theorem: the set of (numbers of) true arithmetical formulas is productive, meaning that from any enumerable set of truths we can effectively find a new truth not in that set. Productive sets are never recursively enumerable, and their complements are never recursively enumerable either.
