We order triples $\langle i, \alpha, \beta \rangle$, where $i = 0, \ldots, 8$, by:
$$\langle i_1, \alpha_1, \beta_1 \rangle < \langle i_2, \alpha_2, \beta_2 \rangle \Leftrightarrow \text{either } \langle \alpha_1, \beta_1 \rangle < \langle \alpha_2, \beta_2 \rangle,$$
or else $\langle \alpha_1, \beta_1 \rangle = \langle \alpha_2, \beta_2 \rangle$ and $i_1 < i_2$,
where pairs are ordered by: $\langle \alpha_1, \beta_1 \rangle < \langle \alpha_2, \beta_2 \rangle$ iff either $\max(\alpha_1, \beta_1) < \max(\alpha_2, \beta_2)$, or the maxima are equal and $\alpha_1 < \alpha_2$, or $\max$ equal, $\alpha_1 = \alpha_2$, and $\beta_1 < \beta_2$.

Then: (a) Every nonempty class of triples has a unique least element; (b) For any triple, there are only finitely many smaller triples.
