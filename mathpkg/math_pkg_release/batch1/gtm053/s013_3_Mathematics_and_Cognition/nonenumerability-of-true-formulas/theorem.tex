Let $E \subset \mathbb{Z}^+$ be an enumerable but undecidable set, defined by the formula $P(x)$ in $\mathrm{L}_1\mathrm{Ar}$. Then:
(a) The function $n \mapsto \neg P(\bar{n})$ from $\mathbb{Z}^+$ to formulas is recursive.
(b) The set $\{n \mid \neg P(\bar{n}) \text{ is true}\}$ is nonenumerable.
Consequently, the set $T_{\mathrm{L}_1}$ of all true formulas in $\mathrm{L}_1\mathrm{Ar}$ is nonenumerable.
