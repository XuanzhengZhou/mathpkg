The axiom of choice is "true" in $V^B$: for every set $X$, there exists a choice function $Y$ that chooses one element from each nonempty element of $X$. Formally, the axiom has the form $\forall x \exists y(Q \wedge R \wedge S \wedge T)$ where:
$Q$: $y$ is a binary relation;
$R$: $y$ is a function;
$S$: the domain of $y$ is contained in $x$;
$T$: the domain of $y$ coincides with $x$, and $y$ chooses one element from each nonempty element of $x$.
