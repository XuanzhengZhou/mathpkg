The Boolean-valued universe V^B is a generalization of the standard von Neumann universe V. Each set in V^B is a random set -- an extensional function assigning Boolean values (probabilities) to its potential elements. The Boolean truth values of membership and equality are defined recursively. V^B serves as a Boolean-valued model for the Zermelo-Fraenkel axioms.
