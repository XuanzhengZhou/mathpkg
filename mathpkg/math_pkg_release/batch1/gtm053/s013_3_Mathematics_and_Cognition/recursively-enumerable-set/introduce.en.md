A set is recursively enumerable (r.e.) if there is an algorithm that, given an input, halts exactly when the input belongs to the set. Equivalently, there is a program that enumerates all elements of the set (possibly with repetitions). R.e. sets are also called semidecidable: membership can be confirmed in finite time, but non-membership may never be confirmed.
