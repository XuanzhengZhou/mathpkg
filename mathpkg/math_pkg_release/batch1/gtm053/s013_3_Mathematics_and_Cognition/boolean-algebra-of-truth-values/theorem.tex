Let:
$$I = \text{a set};$$
$$\Omega = [0, 1]^I \text{ with Lebesgue measure};$$
$$B = \text{the algebra of measurable sets in } \Omega \text{ modulo sets of measure zero};$$
$$0 = \text{the class of the empty set in } B;$$
$$1 = \text{the class of } \Omega \text{ in } B.$$

Operations in $B$: $a'$ (complement), $a \wedge b$ (intersection), $a \vee b$ (union). These operations give a Boolean algebra structure on $B$.

$B$ satisfies the countable chain condition: if $a_\alpha \wedge a_\beta = 0$ for all distinct $\alpha, \beta$ then $a_\alpha \neq 0$ for at most countably many $\alpha$.
