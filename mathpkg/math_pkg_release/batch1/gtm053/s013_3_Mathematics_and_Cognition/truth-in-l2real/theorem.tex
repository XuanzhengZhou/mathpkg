A formula $P$ in $\mathrm{L}_2\mathrm{Real}$ is said to be "true" if $\|P\|(\xi) = 1$ for all $\xi \in \overline{M}$, where $\|P\|(\xi)$ is the Boolean truth value of $P$ computed in the Boolean algebra $B$ of measurable sets modulo null sets, and $\overline{M}$ is the interpretation class assigning to each variable a random variable and to each function symbol a function on random variables satisfying a natural continuity condition.
