\textbf{Downward Löwenheim–Skolem (restricted form).} For a countable $L$, an infinite $L$-structure $A$, an infinite cardinal $\kappa \leq \operatorname{card}A$, and each $L_{\omega_1,\omega}$-sentence $P$ that holds in $A$, there exists an $L$-substructure $B \subset A$ such that $\operatorname{card}B = \kappa$ and $B \models P$.

The proof uses Skolem functions in the same way as in the first-order case (see 3.4).

\textbf{Failure of the upward theorem.} The upward Löwenheim–Skolem theorem does not hold for $L_{\omega_1,\omega}$. There exist $L_{\omega_1,\omega}$-sentences that have models but no models above a certain cardinality. For example, in the language of arithmetic extended by a unary predicate $N$ and a binary predicate $\epsilon$, one can write an $L_{\omega_1,\omega}$-sentence $Q$ stating that $(N, +, \cdot, 0, 1)$ is a standard arithmetic and that every element of the model belongs to $N$.
