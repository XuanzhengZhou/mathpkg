Types are the central objects of modern model theory. An n-type describes a consistent collection of properties that an n-tuple of elements can satisfy in some model. Complete types provide the finest possible description of an element (up to automorphism in saturated models). The distinction between principal types (generated by a single formula) and nonprincipal types is fundamental to the omitting types theorem and classification theory.
