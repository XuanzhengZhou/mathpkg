Each set of axioms (tautologies, Ax $L$, special axioms of $\mathrm{L}_1\mathrm{Ar}$ and $\mathrm{L}_1\mathrm{Set}$) has the following informal syntactic characteristics:
\begin{enumerate}
\item[(a)] An algorithm can be given that determines whether any given expression is an axiom (decidability).
\item[(b)] A finite number of rules can be given for generating the axioms (generability).
\end{enumerate}
Property (a) is more restrictive than (b); an algorithm can be transformed into a generation rule by enumerating all expressions and testing each.
