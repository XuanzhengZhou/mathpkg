Let $\mathcal{E} \supset \operatorname{Ax} L$.

\textbf{(a)} A formula $P$ is deducible from $\mathcal{E}$ if and only if either $\mathcal{E}$ is inconsistent, or $P$ is $\phi$-true for all models $\phi$ of the set $\mathcal{E}$ having cardinality $\leqslant \operatorname{card}(\text{alphabet of } L) + \aleph_0$.

\textbf{(b)} A formula $P$ is independent of $\mathcal{E}$ if and only if both $\mathcal{E} \cup \{P\}$ and $\mathcal{E} \cup \{\neg P\}$ are consistent; by Theorem 6.2, this is true if and only if $\mathcal{E} \cup \{P\}$ and $\mathcal{E} \cup \{\neg P\}$ have models.
