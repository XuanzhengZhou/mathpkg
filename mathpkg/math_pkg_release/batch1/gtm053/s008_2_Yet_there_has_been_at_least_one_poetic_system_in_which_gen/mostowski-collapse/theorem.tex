Let $N \subset V$ be a subclass, and let $\varepsilon \subset N \times N$ be a binary relation. For any $X \in N$ set $[X] = \{Y \mid Y \varepsilon X\}$. Suppose that $[X] \in V$ for all $X \in N$, i.e., each $[X]$ is a set rather than a class. Consider the interpretation $\phi$ of $\mathrm{L}_1\mathrm{Set}$ in the class $N$ for which $\phi(\in)$ is $\varepsilon$ and $\phi(=)$ is equality. Suppose that the axiom of extensionality and the axiom of the empty set are $\phi$-true. Then there exists a transitive class $M \subset V$ and an isomorphism $f: (N, \varepsilon) \to (M, \in)$.
