Let $\mathcal{E}$ be a set of formulas in a language $L$. The alphabet of $L$ is said to be sufficient for $\mathcal{E}$ if, for each closed formula $\neg \forall x P(x)$ in $\mathcal{E}$ there exists a constant $c_P$ (depending on $P$) such that the formula

$$R_P: \neg \forall x P(x) \Rightarrow \neg P(c_P)$$

belongs to $\mathcal{E}$.

The intuitive meaning of $R_P$ is: "If not all $x$ have the property $P$, then some concrete object $c_P$ can be found that does not have this property."
