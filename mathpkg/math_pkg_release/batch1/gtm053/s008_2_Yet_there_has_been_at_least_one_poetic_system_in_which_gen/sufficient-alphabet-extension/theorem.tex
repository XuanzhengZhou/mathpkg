If $\mathcal{E}$ is consistent and contains $\operatorname{Ax} L$, then there exist:

\textbf{(a)} a language $L'$ whose alphabet is obtained from the alphabet of $L$ by adding a set of new constants having cardinality $\leqslant \operatorname{card}(\text{alphabet of } L) + \aleph_0$;

\textbf{(b)} a set of formulas $\mathcal{E}'$ in $L'$ such that $\mathcal{E}' \supset \mathcal{E}$, $\mathcal{E}'$ is consistent, and the alphabet of $L'$ is sufficient for $\mathcal{E}'$.
