Let $L'$ be a language in $\mathfrak{L}_1$ with equality and with an infinite set of variables, and let $P'(x, y_1, \ldots, y_n)$ be a formula in $L'$ in which $x$ occurs freely. Let $\mathcal{E}'$ be a set of formulas in $L'$ that contains $\operatorname{Ax} L'$, the axioms of equality, and perhaps some special axioms. Suppose that $\mathcal{E}' \vdash \exists! x P'(x, y_1, \ldots, y_n)$. Let $L$ be the language obtained from $L'$ by adding a new $n$-ary function symbol $f$, and let $\mathcal{E}$ be the set of formulas obtained from $\mathcal{E}'$ by adding the axiom

$$P'(f(y_1, \ldots, y_n), y_1, \ldots, y_n).$$

Then the set of formulas in $L'$ that are deducible from $\mathcal{E}'$ coincides with the set of formulas in $L$ that do not contain $f$ and are deducible from $\mathcal{E}$ in $L$.
