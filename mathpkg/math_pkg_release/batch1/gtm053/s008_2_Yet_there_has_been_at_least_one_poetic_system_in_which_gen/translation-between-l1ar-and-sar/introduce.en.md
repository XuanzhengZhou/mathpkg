This proposition establishes that the language of arithmetic $\mathrm{L}_1\mathrm{Ar}$ and Smullyan's language SAr have the same expressive power for defining sets of natural numbers, despite their syntactic differences. SAr allows class terms $x(P(x))$ and formulas $x(P(x))\bar{k}$ that directly name sets, while $\mathrm{L}_1\mathrm{Ar}$ uses a more traditional first-order syntax. The translations preserve truth values and free variables. The translation from $\mathrm{L}_1\mathrm{Ar}$ to SAr uses the fact that addition can be expressed via multiplication and exponentiation ($2^{z} = 2^{x} \cdot 2^{y}$). The reverse translation from SAr to $\mathrm{L}_1\mathrm{Ar}$ is more subtle, requiring the definability of exponentiation in $\mathrm{L}_1\mathrm{Ar}$ (a nontrivial result proved in Part II of the book using the Chinese remainder theorem). This equivalence justifies using SAr as a convenient vehicle for Tarski's and Gödel's theorems.
