It is impossible to introduce a mapping $S^2 \to \{0,1\}$, $\alpha \mapsto |s|(\alpha, t; \omega)$, that satisfies the following two conditions simultaneously for fixed $t$ and $\omega$:

\textbf{(a)} $|s|(\alpha, t; \omega)$ takes only the values $0$ and $1$;

\textbf{(b)} $\sum_{i=1}^{3} |s|(\alpha_i, t; \omega) = 2$ for any mutually orthogonal frame $\{\alpha_1, \alpha_2, \alpha_3\} \subset S^2$.

More precisely, there exists a finite system $\Gamma \subset S^2$ of 117 points with the following property: for any mapping $k: \Gamma \to \{0,1\}$, either there is a frame $\{\alpha_1, \alpha_2, \alpha_3\} \subset \Gamma$ in which $k$ takes the value $0$ exactly once (instead of taking $1$ twice and $0$ once), or else there is a pair of perpendicular directions $\{\alpha_1, \alpha_2\} \subset \Gamma$ on which $k$ equals $0$.
