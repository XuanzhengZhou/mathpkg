In a saturated model of a stable theory, given any small set of parameters $A$, one can find large $A$-indiscernible sets using the Erdős-Rado theorem (or equivalently, Ramsey's theorem for finite colorings) combined with compactness. The key point is that stability ensures that the number of types over $A$ is bounded, which allows one to apply the Erdős-Rado partition relation $\kappa \to (\omega)^{<\omega}_2$ for sufficiently large $\kappa$ to extract an indiscernible sequence from any sufficiently long sequence of elements realizing the same type. The exact argument uses the fact that stability controls the number of $\varphi$-types, which in turn bounds the number of "patterns" that need to be homogenized via Ramsey theory.
