This theorem is a cornerstone of stability theory: every complete 1-type over a small model can be represented by an indiscernible set whose elements all realize the type, and whose average type over any intermediate parameter set gives a nonforking extension. Consequently, the cardinality of a maximal such indiscernible set becomes an important invariant of the type, generalizing notions like transcendence degree in the algebraic setting.
