Since $\mathcal{B}$ is a saturated model of a stable theory, the ubiquity of indiscernibles (see indiscernible-ubiquity-stable) guarantees the existence of a large $A$-indiscernible set $I \subseteq \mathcal{B}$ whose elements all realize $p$. This set can be taken to be maximal with respect to this property. For any $C$ with $A \subseteq C \subseteq \mathcal{B}$, the average type $\operatorname{Av}_C(I)$ is complete because for any $L_C$-formula $Q(x)$, either $Q(i)$ holds for all but finitely many $i \in I$ or $\neg Q(i)$ holds for all but finitely many $i \in I$, by indiscernibility. Moreover, $\operatorname{Av}_C(I)$ extends $p$ and, by a fundamental result of stability theory (the "forking calculus"), the average type of an indiscernible set over a larger parameter set is always a nonforking extension.
