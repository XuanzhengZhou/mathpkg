Let $T$ be an $\aleph_0$-stable theory and $\mathbf{M}$ a universal domain for $T$. Let $\operatorname{Def}_n(\mathbf{M})$ be the collection of all nonempty subsets of $M^n$ definable with parameters in $\mathbf{M}$.

The \textit{Morley rank} is the minimal function $\operatorname{rk} : \operatorname{Def}_n(\mathbf{M}) \to \operatorname{Ord}$ satisfying:

\begin{itemize}
\item $\operatorname{rk} S \geq \alpha + 1$ if and only if there is a countable family $\{S_i : i \in \mathbf{N}\}$ of pairwise disjoint subsets of $S$ with $\operatorname{rk} S_i \geq \alpha$;
\item for a limit ordinal $\delta$, $\operatorname{rk} S \geq \delta$ if and only if $\operatorname{rk} S \geq \alpha$ for all $\alpha < \delta$.
\end{itemize}
