This theorem provides the fundamental instability criterion in classification theory: the presence of a definable linear order on an infinite set forces the theory to be unstable. The reason is that an ordered set of cardinality $\kappa$ can have more than $\kappa$ Dedekind cuts, and each cut gives rise to a distinct complete type. Thus the number of types over a parameter set of size $\kappa$ exceeds $\kappa$, violating $\kappa$-stability for every infinite $\kappa$.
