An indiscernible set is a collection of elements that are mutually indistinguishable by any first-order formula: any two $n$-tuples of distinct elements from the set satisfy exactly the same formulas. Indiscernible sets are fundamental in stability theory because, in stable theories, they are ubiquitous in saturated models and serve as carriers for defining types. A canonical example is an algebraically independent subset of an algebraically closed field.
