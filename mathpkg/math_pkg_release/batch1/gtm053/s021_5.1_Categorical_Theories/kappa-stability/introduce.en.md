$\kappa$-stability generalizes $\aleph_0$-stability to arbitrary infinite cardinals $\kappa$. The definition is uniform: over any parameter set of size $\kappa$, the number of complete 1-types is bounded by $\kappa$. This notion provides the quantitative backbone of stability theory, allowing theories to be classified according to for which cardinals they are stable.
