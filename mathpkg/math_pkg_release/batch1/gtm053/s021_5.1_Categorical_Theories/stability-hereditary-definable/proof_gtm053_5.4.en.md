Let $\mathcal{A}$ be a structure definable in a stable structure $\mathcal{M}$. Every formula in the language of $\mathcal{A}$ translates via the definable interpretation to a formula in the language of $\mathcal{M}$. Consequently, every type in the sense of $\mathcal{A}$ corresponds to a type in the sense of $\mathcal{M}$ (relativized to the definable copy of $\mathcal{A}$). Since $\mathcal{M}$ is stable, the number of types over any parameter set in $\mathcal{M}$ is bounded. The mapping of types from $\mathcal{A}$ to types in $\mathcal{M}$ is injective, so the number of types in $\mathcal{A}$ is bounded by the number of types in $\mathcal{M}$. Thus $\mathcal{A}$ inherits stability.
