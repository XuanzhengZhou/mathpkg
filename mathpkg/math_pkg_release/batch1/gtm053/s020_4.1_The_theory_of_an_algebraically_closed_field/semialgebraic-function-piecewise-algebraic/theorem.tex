Given a definable (i.e., semialgebraic) function $g : R \to R$, its domain can be divided into finitely many open intervals and points such that on each interval or point, $g(x)$ is equal to the $k$-th root (for some $k$) of a polynomial equation $p(x, y) = 0$ with coefficients in $R$.

More precisely: write $p(x, y) = a_0(x)y^n + a_1(x)y^{n-1} + \cdots + a_n(x)$, for some $n > 0$ and $a_i(x) \in R[x]$. Then on each subinterval, $g(x)$ coincides with a specific continuous branch of the algebraic function defined by $p(x, y) = 0$ (e.g., the greatest root, the second greatest, and so on).
