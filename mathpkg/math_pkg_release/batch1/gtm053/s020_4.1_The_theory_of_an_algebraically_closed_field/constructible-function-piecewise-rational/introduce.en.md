This theorem provides a structural description of constructible functions over algebraically closed fields of characteristic zero: every such function is piecewise rational. More precisely, given a constructible function $f$ defined on a constructible set $V$, the domain can be partitioned into finitely many constructible subsets on each of which $f$ coincides with a rational function (a quotient of two polynomials). This result is fundamental for understanding definable functions in algebraically closed fields and is a key ingredient in the proof that injective constructible self-maps of constructible sets are surjective.
