The interpretation set $\overline{M}$ consists of random variables on $\Omega = [0,1]^I$. A point $\xi \in \overline{M}$ assigns to each variable $x$ a measurable function $\bar{x}: \Omega \to \mathbb{R}$ and to each function variable $f$ an operation on random variables satisfying a Lipschitz-type condition with probability one. Truth is Boolean-valued: for a formula $P$, $\|P\| \in B$ is the measure of the set of $\omega$ where $P_\omega$ is true. A formula $P$ is "true" if $\|P\| = 1$.
