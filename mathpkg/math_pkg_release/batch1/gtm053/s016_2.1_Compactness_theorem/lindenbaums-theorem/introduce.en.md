Lindenbaum's theorem asserts that every finitely satisfiable (equivalently, consistent) set of first-order sentences can be extended to a complete finitely satisfiable set. The proof uses Zorn's lemma, and therefore the axiom of choice, to produce a maximal element in the poset of finitely satisfiable supersets ordered by inclusion. This maximal element is necessarily complete: if some sentence $S$ were neither in the set nor its negation, one could adjoin either while preserving finite satisfiability, contradicting maximality.
