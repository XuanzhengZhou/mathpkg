Fix a complete $L$-theory $T$. A set $\tau$ of $L$-formulas $P(\bar{x})$ with $n$ free variables $\bar{x} = (x_1, \ldots, x_n)$ is called an $n$-type (in $T$) if for any $P_1(\bar{x}), \ldots, P_k(\bar{x}) \in \tau$,

$$T \models \exists \bar{x} \bigwedge_{i \leq k} P_i(\bar{x}).$$

That is, every finite conjunction of formulas from $\tau$ is consistent with $T$.
