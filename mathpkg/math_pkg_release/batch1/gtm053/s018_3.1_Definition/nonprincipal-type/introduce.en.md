A nonprincipal type is a type that cannot be generated by a single formula — no principal formula isolates it. The Omitting Types Theorem states that in a countable language, any countable collection of nonprincipal types can be simultaneously omitted in some countable model of $T$. For example, the set of formulas $\{0 < x < 1/n : 0 < n \in \mathbb{N}\}$ is a nonprincipal 1-type in the theory of the real field.
