Given a structure $A$ and an $L$-formula $P$, a Skolem function $g_P$ chooses, for each tuple $(a_1,\ldots,a_{n-1})$, a witness $a$ such that $P(a_1,\ldots,a_{n-1},a)$ holds in $A$, if one exists, and a default element otherwise. Closing a set under all Skolem functions yields an elementary substructure, which is the key construction in the proof of the downward Lowenheim-Skolem theorem.
