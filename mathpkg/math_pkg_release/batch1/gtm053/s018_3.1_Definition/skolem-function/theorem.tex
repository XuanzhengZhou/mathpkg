Let $A$ be an $L$-structure and fix $a_0 \in A$. For each $L$-formula $P(v_1, \ldots, v_n)$, define a function $g_P : A^{n-1} \to A$ by

$$g_P(a_1, \ldots, a_{n-1}) = \begin{cases}
\text{an element } a \in A \text{ such that } A \models P(a_1, \ldots, a_{n-1}, a), & \text{if such exists,} \\
a_0, & \text{if not.}
\end{cases}$$

The functions $g_P$ are called Skolem functions (for the structure $A$).
