By definition, the map $a \mapsto c_a^{B_A}$ is an embedding of $A$ into $B$ if and only if $B_A \models \text{Diag}(A)$, because $\text{Diag}(A)$ records precisely the atomic and negated atomic truths about the named elements, and an embedding preserves all atomic formulas and their negations.

Similarly, $a \mapsto c_a^{B_A}$ is an elementary embedding of $A$ into $B$ if and only if $B_A \models \text{CDiag}(A)$, since $\text{CDiag}(A)$ records every $L_A$-sentence true in $A_A$, and an elementary embedding preserves truth of all first-order sentences.
