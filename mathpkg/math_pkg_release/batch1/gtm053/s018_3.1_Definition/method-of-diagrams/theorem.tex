For an $L$-structure $B$,

(i) there exists an expansion $B_A$ to the language $L_A$ such that $B_A \models \text{Diag}(A)$ if and only if $A \subseteq B$ (i.e., $A$ is isomorphic to a substructure of $B$).

(ii) there exists an expansion $B_A$ to the language $L_A$ such that $B_A \models \text{CDiag}(A)$ if and only if $A \preceq B$ (i.e., $A$ is elementarily embeddable in $B$).
