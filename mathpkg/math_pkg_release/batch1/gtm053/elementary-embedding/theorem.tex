An embedding $h : A 	o B$ is 	ext{elementary} if for any $L$-formula $P(x_1,\ldots,x_n)$ and any $a_1,\ldots,a_n \in A$,
$$A \models P(a_1,\ldots,a_n) 	ext{ iff } B \models P(h(a_1),\ldots,h(a_n)).$$
$A$ is an elementary substructure of $B$ and $B$ an elementary extension of $A$, written $A \preceq B$.
$A$ is 	ext{elementarily equivalent} to $B$, written $A quiv B$, if for any $L$-sentence $P$, $A \models P$ iff $B \models P$.
