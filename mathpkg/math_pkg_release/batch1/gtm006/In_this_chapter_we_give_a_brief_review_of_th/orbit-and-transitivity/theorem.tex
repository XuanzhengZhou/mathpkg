If $\Gamma$ is a permutation group on a set $\mathcal{S}$ then, for any $x \in \mathcal{S}$, the \textbf{orbit} of $x$ under $\Gamma$ is the set
$$x\Gamma = \{x^\gamma \mid \gamma \in \Gamma\}.$$
Clearly $y \in x\Gamma$ if and only if $x \in y\Gamma$ so that the orbits of $\Gamma$ partition $\mathcal{S}$.

If $\mathcal{S}$ is the only orbit of $\Gamma$ then we say that $\Gamma$ is \textbf{transitive} on $\mathcal{S}$. More generally we say that $\Gamma$ is \textbf{$t$-transitive} on $\mathcal{S}$ if for every pair of ordered $t$-tuples of distinct elements in $\mathcal{S}$ there is an element in $\Gamma$ sending the first $t$-tuple onto the second. Thus one-transitive is the same as transitive. We shall often refer to a two-transitive group as being \textbf{doubly transitive}.
