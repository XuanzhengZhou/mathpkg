A \textbf{skewfield} is a non-empty set $K$ with two binary operations called addition and multiplication, so that if $a$ and $b$ are in $K$, then $a+b$ and $ab$ (or $a \cdot b$) are elements in $K$ all satisfying:

\begin{enumerate}
\item[(1)] Both operations are associative; i.e., $a+(b+c)=(a+b)+c$, and $a(bc)=(ab)c$ for all $a,b,c$ in $K$.
\item[(2)] Addition is commutative; i.e., $a+b = b+a$ for all $a,b$ in $K$.
\item[(3)] Each operation has an identity; so there is an element $0$ in $K$ such that $0+b = b+0 = b$ for all $b$ in $K$ and there is an element $1$ in $K$ such that $1b = b1 = b$ for all $b$ in $K$. In addition $0 \neq 1$.
\item[(4)] For each $b$ in $K$ there is an element $-b$ in $K$ such that $b+(-b)=0$, and for each $b \neq 0$ in $K$ there is an element $b^{-1}$ in $K$ such that $bb^{-1}=b^{-1}b=1$. The element $-b$ is called the \textit{additive inverse} of $b$ while $b^{-1}$ is its \textit{multiplicative inverse}.
\item[(5)] $a(b+c) = ab + ac$, and $(a+b)c = ac + bc$ for all $a,b,c$ in $K$.
\end{enumerate}
