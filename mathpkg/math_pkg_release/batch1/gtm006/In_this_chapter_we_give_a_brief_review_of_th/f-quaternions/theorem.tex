Let $F$ be a field and let $K$ be the set of all symbols
$$a + bi + cj + dk$$
where $a,b,c,d$ are in $F$, with addition being coefficient-wise, and with multiplication given by the rule:
\begin{align*}
&\left( a + bi + cj + dk \right) \left( a_1 + b_1 i + c_1 j + d_1 k \right) = \\
&\left( aa_1 - bb_1 - cc_1 - dd_1 \right) + \left( ab_1 + ba_1 + cd_1 - dc_1 \right) i \\
&+ \left( ac_1 - bd_1 + ca_1 + db_1 \right) j + \left( ad_1 + bc_1 - cb_1 + da_1 \right) k.
\end{align*}
Then $K$ is a skewfield which we call the \textbf{$F$-quaternions} or, if $F$ is the reals, merely the \textbf{quaternions}.
