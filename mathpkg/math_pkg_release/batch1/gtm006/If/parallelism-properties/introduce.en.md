In an affine plane, "parallel" is defined as the relation of being equal or disjoint. This lemma establishes that the parallelism relation is well-behaved: parallel lines are exactly those that do not meet, and through any point off a given line there passes exactly one parallel. These properties make parallelism an equivalence relation, partitioning the line set into parallel classes. Each parallel class corresponds to a "point at infinity" in the projective completion.
