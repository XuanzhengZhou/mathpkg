If $n \geq 2$ and $\mathcal{B}$ is a set of $n^2 + n + 1$ points and $n^2 + n + 1$ lines such that any line is incident with $n+1$ points and any two distinct lines have a unique common point, show that $\mathcal{B}$ is a projective plane.
