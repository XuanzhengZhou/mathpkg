Let $e_1, e_2, \dots, e_{n+1}$ be a basis for $V$. Set $E_i = \langle e_i \rangle$ for $i = 1, \dots, n+1$ and $E_{n+2} = \langle e_1 + e_2 + \cdots + e_{n+1} \rangle$. We must verify that no $n+1$ of these $n+2$ points lie in a hyperplane. Suppose to the contrary that some set of $n+1$ of the $E_j$ were contained in a hyperplane. The only case requiring checking is when the omitted point is among $E_1, \dots, E_{n+1}$ (if $E_{n+2}$ is omitted, then $E_1, \dots, E_{n+1}$ correspond to a basis and span the whole space, so they cannot lie in a proper subspace). If $E_{n+2}$ is contained in the span of $n$ of the $E_i$, say $E_1, \dots, E_n$, then $e_1 + \cdots + e_{n+1}$ would be a linear combination of $e_1, \dots, e_n$, forcing $e_{n+1}$ to be a linear combination of $e_1, \dots, e_n$, contradicting that the $e_i$ form a basis. Hence each set of $n+1$ of the $E_j$ spans $V$, i.e., no $n+1$ lie in a hyperplane. Thus $E_1, \dots, E_{n+2}$ is a frame.

Conversely, if $F_1, \dots, F_{n+2}$ is any frame, choose non-zero vectors $f_i$ with $F_i = \langle f_i \rangle$. Since $F_{n+2}$ is not in the hyperplane generated by $F_1, \dots, \hat{F}_k, \dots, F_{n+1}$ for any $k$, we have $f_{n+2} = \sum x_i f_i$ with each $x_i \neq 0$. Setting $e_i = x_i f_i$ yields a basis with the required property. $\square$
