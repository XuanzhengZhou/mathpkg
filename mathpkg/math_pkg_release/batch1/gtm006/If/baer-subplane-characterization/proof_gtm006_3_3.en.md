If $\mathcal{C}$ is a subplane then $\mathcal{C}$ is called a Baer subplane. From the proof of Theorem 3.5 we have the characterization: If $\mathcal{P}_0$ is a subplane of order $m$ inside a finite projective plane $\mathcal{P}$ of order $n$, then $\mathcal{P}_0$ is a Baer subplane if and only if $n = m^2$. The proof follows from the incidence count: every line of $\mathcal{P}$ meets $\mathcal{P}_0$ in either $1$ point (tangent) or $m+1$ points (secant), and the Baer property requires that every line is a secant, which forces $n+1 = (m+1) + (n-m)(1)$ through point-line incidence counting, yielding $n = m^2$.
