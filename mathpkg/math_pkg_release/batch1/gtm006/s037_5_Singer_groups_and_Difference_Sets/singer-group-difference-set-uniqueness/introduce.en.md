This lemma establishes the fundamental "difference" property of a difference set. In a Singer group $\Pi$, every non-identity element can be expressed in exactly one way as $\delta_1\delta_2^{-1}$ (and also as $\delta_3^{-1}\delta_4$) with the $\delta_i$ coming from a fixed difference set $\mathcal{D}$. This property is what justifies the name "difference set" and is the basis for constructing projective planes from group-theoretic data.
