If $\mathcal{P}$ is a projective plane with a Singer group $\Gamma$ and difference set $\mathcal{D}$, show that, for any $\alpha, \beta$ in $\Gamma$, the set $\alpha^{-1} \mathcal{D} \beta$ is also a difference set for $\Gamma$.
