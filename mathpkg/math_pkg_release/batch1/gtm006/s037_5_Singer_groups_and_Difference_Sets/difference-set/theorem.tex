If $\Pi$ is a Singer group of a projective plane $\mathcal{P}$, then for any given point $X$ and line $m$ of $\mathcal{P}$, the subset $\mathcal{D} = \{\pi \in \Pi \mid X^\pi \in m\}$ is called a difference set of $\Pi$. For every $\beta \neq 1$ in $\Pi$, there exist unique pairs $\delta_1, \delta_2$ and $\delta_3, \delta_4$ in $\mathcal{D}$ such that $\beta = \delta_1 \delta_2^{-1} = \delta_3^{-1} \delta_4$.
