Given a Singer group $\Pi$, the points can be identified with elements of $\Pi$, and a line can be represented by the set of group elements sending a base point to that line. This set $\mathcal{D}$ is a difference set: every non-identity element of $\Pi$ can be uniquely expressed as $d_1 d_2^{-1}$ with $d_1, d_2 \in \mathcal{D}$. A finite projective plane of order $n$ with a Singer group corresponds to a difference set of size $n+1$ in a group of order $n^2 + n + 1$.
