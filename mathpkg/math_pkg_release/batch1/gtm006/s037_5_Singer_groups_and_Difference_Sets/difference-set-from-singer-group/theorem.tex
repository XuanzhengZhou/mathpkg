If $\Pi$ is a Singer group of a projective plane $\mathcal{P}$, then for any point $X$ and line $m$, the difference set is $\mathcal{D} = \{\pi \in \Pi \mid X^\pi \in m\}$. This is a subset of $\Pi$ of size $n+1$ (where $n$ is the order of $\mathcal{P}$) satisfying the property that every non-identity element of $\Pi$ can be uniquely expressed as $\delta_1 \delta_2^{-1}$ with $\delta_1, \delta_2 \in \mathcal{D}$.
