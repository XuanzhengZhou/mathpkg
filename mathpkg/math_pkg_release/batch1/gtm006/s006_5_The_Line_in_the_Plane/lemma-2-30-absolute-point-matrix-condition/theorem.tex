If $A = (a_{ij})$ is the matrix for the polarity $\beta$ then the point $\langle x, y, z \rangle$ is absolute (i.e., incident with its image under $\beta$) if and only if:
\begin{itemize}
\item For an orthogonal polarity ($\phi = 1$): $a_{11}x^2 + a_{22}y^2 + a_{33}z^2 + 2a_{12}xy + 2a_{13}xz + 2a_{23}yz = 0$.
\item For a unitary polarity: $a_{11}x x^\phi + \cdots = 0$ with appropriate hermitian conditions.
\end{itemize}
