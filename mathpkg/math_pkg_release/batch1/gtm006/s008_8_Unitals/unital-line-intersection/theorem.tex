Let $\mathcal{U}$ be a unital associated with a unitary polarity with matrix $A$ and involutory automorphism $\phi$. Then:

\begin{enumerate}
\item[(i)] A line meets $\mathcal{U}$ exactly once if and only if the line is absolute.
\item[(ii)] A non-absolute line either meets $\mathcal{U}$ not at all, or meets it in a set of points which is in one-to-one correspondence with the elements of the norm-one subgroup $N$. A non-absolute line $\langle v \rangle + \langle w \rangle$ meets the unital if and only if there exists $z \in K$ satisfying
$$z^{1+\phi} = (w A v'^{\phi})(v A w'^{\phi}) - (v A v'^{\phi})(w A w'^{\phi}).$$
\end{enumerate}
