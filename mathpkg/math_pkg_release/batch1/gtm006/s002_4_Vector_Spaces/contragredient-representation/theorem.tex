If $\alpha$ is a non-singular semi-linear transformation of $V$ onto $V$ with associated automorphism $\sigma$ and matrix $A$ (so $v^\alpha = (x_1^\sigma, \dots, x_n^\sigma)A$), define $\alpha' : V' \to V'$ by $(y_1, \dots, y_n)^{\alpha'} = A^{-1}(y_1^\sigma, \dots, y_n^\sigma)^T$ (as a column vector). Then $\alpha'$ is a non-singular semi-linear transformation of $V'$ with the same automorphism $\sigma$, satisfying $v^\alpha(w') = (v w')^\sigma$. The map $\alpha \mapsto \alpha'$ is an isomorphism of $\Gamma L(V)$ onto $\Gamma L(V')$ sending $GL(V)$ onto $GL(V')$, with $(\alpha')' = \alpha$.
