If $V$ is a left vector space over the skewfield $K$, a linear functional on $V$ is a mapping $f': V \to K$ such that:
\begin{itemize}
\item[(i)] $(v + w)f' = vf' + wf'$ for all $v, w \in V$,
\item[(ii)] $(kv)f' = k(vf')$ for all $v \in V$, $k \in K$.
\end{itemize}
The set $V'$ of all linear functionals is a right vector space over $K$ under the operations $(f' + g')v = f'v + g'v$ and $(f'b)v = (f'v)b$.
