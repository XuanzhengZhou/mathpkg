If $\mathcal{P}$ is a division ring plane and $D_1, D_2$ are division rings which may be used to coordinatize $\mathcal{P}$, then each one may be used to coordinatize $\mathcal{P}$ with the same choice of points for $(\infty), (0)$ and $(0, 0)$. If $S$ is the point which when chosen as $(1, 1)$ gives $D_1$ as coordinatizing division ring, and if $T$ gives rise to $D_2$ in a similar manner then, using Exercise 5.2, $D_1$ is isomorphic to $D_2$ if and only if there is a collineation $	heta$ of $\mathcal{P}$ such that $(\infty)^	heta = (\infty)$, $(0)^	heta = (0)$, $(0, 0)^	heta = (0, 0)$ and $S^	heta = T$. But $	heta$ is an autotopism of $\mathcal{P}$, and thus $D_1$ is isomorphic to $D_2$ if and only if $S$ and $T$ are in the same orbit of the autotopism group of $\mathcal{P}$. This establishes the one-to-one correspondence. $\square$
