Corollary 2 of Theorem 13.1 provides a sharp upper bound for the order of any collineation of a finite projective plane of order $n$: the order $t$ cannot exceed $n^2 + n + 1$, which is the total number of points in the plane. For an abelian collineation group, an orbit is faithful if and only if its length equals the group order, so the bound follows from the fact that a faithful orbit length cannot exceed the total number of points. The bound is best possible, as demonstrated by the projective plane $\mathcal{P}_2(q^n)$ which admits a collineation of order $q^{2n} + q^n + 1$.
