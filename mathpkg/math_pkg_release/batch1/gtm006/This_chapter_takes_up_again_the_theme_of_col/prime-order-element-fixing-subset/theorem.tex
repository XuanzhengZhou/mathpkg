Let $\Gamma$ be a permutation group on a finite set $\mathcal{S}$ and let $p$ be a prime. If $\mathcal{T}$ is a subset of $\mathcal{S}$ such that, for any $t \in \mathcal{T}$, $\Gamma$ contains an element of order $p$ fixing $t$ and no other element of $\mathcal{S}$, then $\mathcal{T}$ is contained in an orbit of $\Gamma$.
