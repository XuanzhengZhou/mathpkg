By Theorem 13.4, $\Pi$ has an equal number of point orbits and line orbits. $\Pi$ is transitive on points if and only if it has exactly one point orbit. By the equality, this holds if and only if $\Pi$ has exactly one line orbit, which is equivalent to $\Pi$ being transitive on lines.
