In a collineation group of a projective plane, two elation flags are connected if they can be joined by a chain of elation flags where each consecutive pair shares a common element (a common centre or a common axis). A set of elation flags is connected if any two flags in it are connected in this sense. The motivation for this definition is that all elations whose centres belong to the same non-trivial connected set must have the same prime order (Lemma 13.7). The trivial connected set consists of a single flag.
