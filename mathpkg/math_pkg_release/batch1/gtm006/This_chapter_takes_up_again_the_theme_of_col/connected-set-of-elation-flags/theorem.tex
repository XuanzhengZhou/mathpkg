Two elation flags $A$ and $C$ of a collineation group $\Pi$ of a projective plane $\mathcal{P}$ are said to be \textit{connected} if there is a sequence $A = F_1, F_2, \dots, F_k = C$ of elation flags such that, for each $i = 1, 2, \dots, k-1$, the flags $F_i$ and $F_{i+1}$ have an element in common.

A set $\mathcal{S}$ of elation flags is \textit{connected} if any two flags in $\mathcal{S}$ are connected. A connected set is called \textit{trivial} if it consists of a flag only.
