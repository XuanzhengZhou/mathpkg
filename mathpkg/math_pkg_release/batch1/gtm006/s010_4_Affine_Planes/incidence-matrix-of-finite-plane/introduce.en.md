The incidence matrix of a finite plane is a $(0,1)$-matrix encoding which points lie on which lines. For a projective plane of order $n$, it is a square matrix of size $n^2+n+1$ with each row and column summing to $n+1$. For an affine plane of order $n$, it has $n^2$ rows and $n^2+n$ columns. The algebraic properties of these matrices (e.g., $AA^T = nI + J$) lead to important non-existence results such as the Bruck-Ryser-Chowla theorem.
