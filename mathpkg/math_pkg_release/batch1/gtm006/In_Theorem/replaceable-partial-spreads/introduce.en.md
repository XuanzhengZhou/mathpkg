Replaceable partial spreads generalize the concept of derivation in translation planes. Two partial spreads are replacements for each other if they cover the same set of points but share no common components. Given a spread whose components contain a replaceable partial spread, replacing that partial spread by any one of its replacements yields a new spread and thus a new translation plane defined on the same set of points. This construction shares the parallel classes of the non-replaced components with the original plane. Derivation is the special case where a set of two components is replaceable.
