Let $K$ be a field with an involutory automorphism $\phi$ and let $F$ be the set of elements fixed by $\phi$. For any $a \neq 0$ in $K$, the equation
$$x^{1+\phi} = a$$
can only have solutions if $a \in F$. If it has solutions, then the set of all solutions is a coset of $N$ in the multiplicative group $K^*$, where $N = \{ n \in K^* \mid n^{1+\phi} = 1 \}$.
