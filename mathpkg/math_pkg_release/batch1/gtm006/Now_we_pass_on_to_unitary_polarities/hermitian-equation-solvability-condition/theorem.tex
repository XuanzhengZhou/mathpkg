Let $a, b, c$ be elements of $K$, where $a, c$ lie in $F$ (the fixed field of $\phi$) and $a \neq 0$. The equation
$$x^{1+\phi}a + x^{\phi}b + x b^{\phi} + c = 0 \tag{*}$$
has solutions only if $b^{1+\phi} - ac = d$ is an element of $F$. Moreover, the number of solutions depends on whether $d = 0$: if $d = 0$, the equation has exactly one solution; if $d \neq 0$, the set of solutions (when non-empty) is in one-to-one correspondence with $N$.
