Let $\mathcal{U}$ be a unital in a projective plane over a field $K$ with involutory automorphism $\phi$, defined by a unitary polarity with matrix $A$ satisfying $A'^{\phi} = A$. Then:

\begin{enumerate}
\item[(i)] A line meets $\mathcal{U}$ exactly once if and only if the line is absolute.
\item[(ii)] A non-absolute line $\langle v \rangle + \langle w \rangle$ either meets $\mathcal{U}$ in no points, or meets it in a set of points which is in one-to-one correspondence with the elements of $N = \{ n \in K^* \mid n^{1+\phi} = 1 \}$. Such a line meets $\mathcal{U}$ if and only if there is a solution to
$$z^{1+\phi} = (w A v'^{\phi})(v A w'^{\phi}) - (v A v'^{\phi})(w A w'^{\phi}).$$
\end{enumerate}
