Let $\mathcal{P}$ be a projective plane coordinatized by a set $R$. Define $T$ by $T(a, b, c) = k$ if and only if $(b, c)$ is on $[a, k]$. Then the following properties hold.

\begin{itemize}
  \item[(A)] $T(a, 0, c) = T(0, b, c) = c$ for all $a, b, c \in R$.
  \item[(B)] $T(a, 1, 0) = T(1, a, 0) = a$ for all $a \in R$.
  \item[(C)] If $a, b, c, d \in R$, $a \neq c$, then there is a unique $x \in R$ such that $T(x, a, b) = T(x, c, d)$.
  \item[(D)] If $a, b, c \in R$ then there is a unique $x \in R$ such that $T(a, b, x) = c$.
  \item[(E)] If $a, b, c, d \in R$, $a \neq c$, then there exist unique $x, y \in R$ such that $T(a, x, y) = b$ and $T(c, x, y) = d$.
\end{itemize}
