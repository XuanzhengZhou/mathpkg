A closed configuration is an incidence structure satisfying the first two axioms of a projective plane but not necessarily the third (non-degeneracy) axiom. Thus every projective plane is a closed configuration, but the converse is false. Closed configurations that fail axiom (iii) are precisely the degenerate planes, which form a very restricted class of structures.
