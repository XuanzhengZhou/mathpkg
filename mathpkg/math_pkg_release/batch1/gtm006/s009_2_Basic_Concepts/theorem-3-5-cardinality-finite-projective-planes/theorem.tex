Let $\mathcal{P}$ be a finite projective plane. Then there exists an integer $n \geq 2$ such that:
\begin{enumerate}
  \item[(i)] Every line contains exactly $n + 1$ points.
  \item[(ii)] Every point is incident with exactly $n + 1$ lines.
  \item[(iii)] The total number of points in $\mathcal{P}$ is $n^2 + n + 1$.
  \item[(iv)] The total number of lines in $\mathcal{P}$ is $n^2 + n + 1$.
\end{enumerate}
