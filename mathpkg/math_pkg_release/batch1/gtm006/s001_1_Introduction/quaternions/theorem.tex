Let $F$ be a field. Extend $F$ to a four-dimensional vector space over $F$ with basis $\{1, i, j, k\}$. Multiplication is defined by the rule:
$$(a + bi + cj + dk)(a_1 + b_1 i + c_1 j + d_1 k) = (aa_1 - bb_1 - cc_1 - dd_1) + (ab_1 + ba_1 + cd_1 - dc_1)i + (ac_1 - bd_1 + ca_1 + db_1)j + (ad_1 + bc_1 - cb_1 + da_1)k.$$
Then the resulting algebra is a skewfield, called the $F$-quaternions. If $F = \mathbb{R}$, they are simply called the quaternions.
