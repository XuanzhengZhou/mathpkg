A permutation group $\Gamma$ on $\mathcal{S}$ is transitive if $\mathcal{S}$ is the only orbit of $\Gamma$. More generally, $\Gamma$ is $t$-transitive if for every pair of ordered $t$-tuples of distinct elements in $\mathcal{S}$ there is an element in $\Gamma$ sending the first $t$-tuple onto the second. Two-transitive groups are called doubly transitive.
