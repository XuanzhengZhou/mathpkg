If $\Gamma$ is a permutation group on a set $\mathcal{S}$ then, for any $x \in \mathcal{S}$, we denote by $\Gamma_x$ the set of all elements in $\Gamma$ which fix $x$; i.e.,
$$\Gamma_x = \{\gamma \in \Gamma \mid x^\gamma = x\}.$$
$\Gamma_x$ is a subgroup of $\Gamma$, called the \text{stabilizer} of $x$ in $\Gamma$.
