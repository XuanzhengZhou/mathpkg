Let $\{\mathcal{A}_\alpha\}$ be a collection of sub-configurations of a configuration $\mathcal{C}$.

\begin{enumerate}
  \item[(i)] If all $\mathcal{A}_\alpha$ are open, then $\bigcap_\alpha \mathcal{A}_\alpha$ is open.
  \item[(ii)] If all $\mathcal{A}_\alpha$ are confined, then $\bigcup_\alpha \mathcal{A}_\alpha$ is confined.
\end{enumerate}
