A closed configuration is a configuration in which every pair of elements of the same type (two points, or two lines) possess a common incident element of the opposite type. Equivalently, it satisfies the definition of a projective plane given in Chapter III, without necessarily containing a quadrangle. Closed configurations that lack a quadrangle are called degenerate; those with a quadrangle are precisely the projective planes.
