A \textbf{latin square of order $n$} is an $n \times n$ matrix with entries from a set $R$ of $n$ distinct elements, which we shall usually take to be $\{0, 1, 2, \dots, n-1\}$, such that each row and each column contains every element of $R$ exactly once.

Thus the Cayley table for any quasigroup provides an example of a latin square.
