An $n$-step extension generalizes the notion of a one-step extension by allowing a chain of $n$ successive one-step extensions. Starting from a configuration $\mathcal{C} = \mathcal{C}_0$, we build a tower $\mathcal{C}_0 < \mathcal{C}_1 < \cdots < \mathcal{C}_n$ where each step adds new elements. This iterative construction is used to generate closed configurations from a given base configuration via full free extensions.
