If $\mathcal{C} = \mathcal{C}_0 < \mathcal{C}_1 < \mathcal{C}_2 < \cdots < \mathcal{C}_n$ is a sequence of configurations such that each $\mathcal{C}_{i+1}$ is a one-step extension of $\mathcal{C}_i$, then $\mathcal{C}_n$ is an $n$-step extension of $\mathcal{C}$. A configuration is an extension of $\mathcal{C}$ if it is an $n$-step extension for some $n \geq 0$.
