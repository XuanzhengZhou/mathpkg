Let $\mathcal{A}$ and $\mathcal{B}$ be configurations in the closed configuration $\mathcal{P}$ such that each is contained in an extension of the other. Then there exists a sub-configuration $\mathcal{C}$ of $\mathcal{P}$ which is a common extension of both $\mathcal{A}$ and $\mathcal{B}$. If $\mathcal{P} = \mathcal{F}(\mathcal{A}) = \mathcal{F}(\mathcal{B})$, then $\mathcal{C}$ can be chosen to be a free extension of both $\mathcal{A}$ and $\mathcal{B}$.
