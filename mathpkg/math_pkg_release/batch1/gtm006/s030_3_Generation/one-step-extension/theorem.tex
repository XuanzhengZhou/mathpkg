A configuration $\mathcal{C}_1$ is a one-step extension of a configuration $\mathcal{C}$ if $\mathcal{C} < \mathcal{C}_1$ (i.e., $\mathcal{C}$ is a sub-configuration of $\mathcal{C}_1$). If $\mathcal{C}$ and $\mathcal{C}_1$ are both sub-configurations of a configuration $\mathcal{P}$, then $\mathcal{C}_1$ is called a one-step extension of $\mathcal{C}$ in $\mathcal{P}$.
