The Pickert coordinatization of a projective plane uses the same point and line assignments as the Hall method but defines a ternary operation $S$ by: $S(m, x, k) = y$ if and only if $(x, y)$ lies on the line $[m, k]$. Addition and multiplication are defined by $a + b = S(1, a, b)$ and $ab = S(a, b, 0)$. The ternary ring is linear if $S(m, x, k) = mx + k$.
