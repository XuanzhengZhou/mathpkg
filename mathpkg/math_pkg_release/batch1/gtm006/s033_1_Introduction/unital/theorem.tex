A \textit{unital} in a finite projective plane of order $n = s^2$ is a set of $s^3 + 1$ points (the absolute points) together with a distinguished set of lines (the non-absolute lines) such that every line meets the point-set in either $1$ or $s+1$ points, and through each point of the unital there passes exactly one line meeting the unital in a single point.
