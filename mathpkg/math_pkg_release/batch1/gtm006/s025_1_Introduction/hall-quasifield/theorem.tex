Let $F$ be a field and $f(s) = s^2 - as - b$ an irreducible quadratic over $F$. Let $H$ be a two-dimensional right vector space over $F$, with basis elements $1$ and $\lambda$, so that $H$ consists of all elements of the form $x + \lambda y$ as $x$ and $y$ vary over $F$.

Define a multiplication on $H$ as follows:

If $y \neq 0$:
$$(x + \lambda y)(z + \lambda t) = xz - y^{-1}tf(x) + \lambda(yz - xt + at).$$

If $y = 0$:
$$x(z + \lambda t) = xz + \lambda(xt).$$

The system $(H, +, \cdot)$ with vector space addition and this multiplication is called a **Hall quasifield**.
