Let $W_1$ and $W_2$ be two-dimensional subspaces of a three-dimensional vector space $V$ over $K$, and let $\alpha$ be an arbitrary non-singular semi-linear transformation from $W_1$ to $W_2$. Then there exists a non-singular semi-linear transformation $\alpha^*$ of $V$ which sends $W_1$ to $W_2$ such that if $w$ is a vector in $W_1$, then $w^{\alpha^*} = w^{\alpha}$. Hence in particular there is an element of $P\Gamma L(V)$ which agrees with the mapping induced by $\alpha$ from the projective line $\mathcal{P}(W_1)$ to the projective line $\mathcal{P}(W_2)$.
