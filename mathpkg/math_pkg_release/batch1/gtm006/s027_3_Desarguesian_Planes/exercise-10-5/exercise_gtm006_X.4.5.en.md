Using the notation of Theorem 10.14, show that $\mathcal{B}(a, b, c)$ and $\mathcal{B}(d, e, f)$ are equal if and only if $F^* a = F^* d$, $F^* a + b = F^* d + e$, and $F^* a + c = F^* d + f$; and that $\mathcal{B}(a, b, c)$ and $\mathcal{B}(d, e, f)$ are "parallel" if and only if $F^* a = F^* d$.
