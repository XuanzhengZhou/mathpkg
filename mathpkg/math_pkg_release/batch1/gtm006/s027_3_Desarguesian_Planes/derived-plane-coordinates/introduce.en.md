Theorem 10.15 provides an explicit description of the planar ternary ring that coordinatizes the derived plane $\mathcal{D}_{\mathcal{S}}(\mathcal{A})$ in terms of the original coordinatization of $\mathcal{A}$. The underlying set $R$ remains the same, but the ternary operation is modified according to the structure of the derivation set $\mathcal{S}$ and its associated Baer subplanes. This theorem is essential for computational work with derived planes, as it gives concrete formulas for the new algebraic operations.
