Theorem 18.1 asserts that in any Boolean-valued model $V^{(B)}$, the interpretation of the power set of $\omega$ is always contained in the check of the power set of $\omega$, with Boolean truth value 1. This is a fundamental consistency result about the relationship between the constructed power set inside $V^{(B)}$ and the externally given power set from the ground model.
