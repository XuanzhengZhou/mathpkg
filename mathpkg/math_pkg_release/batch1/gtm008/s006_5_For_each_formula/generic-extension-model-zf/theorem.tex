If $G$ is $P$-generic over $M$, then $M[G]$ is a standard transitive model of $\mathsf{ZF}$ that has the same order type as $M$. For any formula $\varphi$,

$$
M[G] \models \varphi \leftrightarrow \llbracket \varphi \rrbracket \in G.
$$

Furthermore, if $M$ satisfies the Axiom of Choice, then so does $M[G]$.
