Theorem 9.29 establishes that in the ramified $B$-valued structure $\mathbf{T}_{\alpha}$, distinct constant symbols representing different sets in the well-founded universe receive distinct Boolean truth values for equality: identical constants have equality value $1$, while distinct constants have equality value $0$. This is proved by induction on the rank $\rho$ of the constants, using the fact that if $k_1 \neq k_2$, then by symmetry one contains an element not in the other, and this difference propagates to the Boolean truth values.
