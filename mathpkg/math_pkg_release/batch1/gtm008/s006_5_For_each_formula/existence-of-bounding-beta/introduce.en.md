Theorem 9.22 provides a uniform bound: for any $\alpha$, there exists $\beta$ such that every element of $M$ defined over $M_{\alpha}$ is equal (with value $1$) to some element of $M_{\beta}$. The proof uses the fact that $B^{M_{\alpha}}$ is a set and defines a bounding function $f$ on elements of $B^{M_{\alpha}}$, then takes the supremum.
