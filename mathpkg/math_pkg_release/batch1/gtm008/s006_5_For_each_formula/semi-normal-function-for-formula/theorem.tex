For each formula $\varphi$ of $\mathcal{L}$, there exists a semi-normal function $F$ such that for all $\alpha, \beta$,
$$\beta = F(\alpha) \to (\forall a_1, \ldots, a_n \in M_{\alpha})\left[\llbracket (\exists y)\varphi(a_1, \ldots, a_n, y) \rrbracket = \sum_{a \in M_{\beta}} \llbracket \varphi(a_1, \ldots, a_n, a) \rrbracket\right].$$
