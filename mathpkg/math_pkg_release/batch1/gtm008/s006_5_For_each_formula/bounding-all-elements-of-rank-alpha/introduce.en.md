Theorem 9.23 states that for any ordinal $\alpha$, there exists a set $b \in M$ that contains every element of $M_{\alpha}$ with Boolean truth value $1$. This simple structural property is proved by taking $\varphi(x)$ to be $x = x$ in condition 5 of the construction and using the resulting $b \in M_{\alpha+1}$. In the classical $B = 2$ case, it simply says $M_{\alpha} \subseteq b$.
