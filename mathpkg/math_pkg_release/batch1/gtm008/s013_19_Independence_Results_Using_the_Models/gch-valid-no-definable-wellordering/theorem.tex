Assuming the $GCH$, there is a Boolean algebra $B$ such that the $GCH$ is $B$-valid in $V^{(B)}$ but the statement ``There is a definable well-ordering of $\mathcal{P}(\omega)$'' is not $B$-valid. (Here by ``definable'' we mean ``definable using constants $\check{k}$ as parameters''. Note that $\check{k}$ corresponds to the constant $\underline{k} \in C(M)$ in Theorem 11.7.) ``There is a definable well-ordering of $\mathcal{P}(\omega)$'' is in fact a statement in the language of set theory.
