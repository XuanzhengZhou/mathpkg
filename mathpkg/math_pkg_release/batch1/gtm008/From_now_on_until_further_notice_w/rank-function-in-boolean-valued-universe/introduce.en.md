Theorem 16.26 characterizes the rank function $R(\check{\alpha})$ in the Boolean-valued universe $V^{(\mathbf{B})}$. Statement 1 shows that $R(\check{\alpha})$ is Boolean-valued-equal to $V_\alpha^{(\mathbf{B})} \times \{1\}$, meaning rank in $V^{(\mathbf{B})}$ corresponds to the $\alpha$-th level of the $V^{(\mathbf{B})}$ hierarchy, embedded with truth value $1$. Statement 2 says that any set whose domain lies in $V_\alpha^{(\mathbf{B})}$ is Boolean-valued-contained in $R(\check{\alpha})$. Statement 3 expresses the Boolean truth of $u \subseteq R(\check{\alpha})$ as the join of equality values with sets from a set-sized collection $A_\alpha$, analogous to Theorem 16.25. These results are applications of the power set representation to the cumulative hierarchy.
