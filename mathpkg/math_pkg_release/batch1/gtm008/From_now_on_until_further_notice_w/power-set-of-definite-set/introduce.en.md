Theorem 16.27 gives a precise description of the power set of a definite set $u$ in the Boolean-valued universe. When $u$ is definite, $\mathcal{P}(u)$ is Boolean-valued-equal to $B^{\mathcal{D}(u)} \times \{1\}$, the set of all functions from $\mathcal{D}(u)$ into the Boolean algebra $B$, each assigned truth value $1$. The key insight is that for a definite set, the Boolean truth of $x \subseteq u$ reduces to checking membership values, and any function $f: \mathcal{D}(u) \to B$ can be interpreted as a Boolean-valued subset of $u$. The proof uses the fact that $B^{\mathcal{D}(u)}$ is a subset of $V^{(\mathbf{B})}$ (as a set of characteristic functions) and Theorem 16.25 to translate the subset relationship.
