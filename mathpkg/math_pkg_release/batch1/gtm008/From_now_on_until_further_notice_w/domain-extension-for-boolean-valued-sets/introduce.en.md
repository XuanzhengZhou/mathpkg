In a Boolean-valued universe $V^{(\mathbf{B})}$, each set $u$ has a domain $\mathcal{D}(u)$. This theorem shows that whenever a family of sets $\{u_i\}$ has domains all contained in some common set $d$, we can extend each domain exactly to $d$ while preserving Boolean-valued equality $\llbracket u_i = u'_i \rrbracket = 1$. The extension is defined by setting $u'_i(x) = \llbracket x \in u_i \rrbracket$ for $x \in d$, and the proof uses Theorem 13.4.3 to verify the equality condition. This technical lemma is essential preparation for the Maximum Principle.
