For a uniform (extensional with complete domain) Boolean-valued set $u$, the supremum $\sup(u) = \sum_{x \in \mathcal{D}(u)} u(x)$ is actually attained as $u(x_0)$ for some element $x_0$ in the domain. This is a form of the Maximum Principle specialized to uniform sets. The proof constructs disjoint Boolean components from the values $u(x_\xi)$ and mixes the corresponding domain elements to obtain the witness $x$.
