Theorem 7.9 states that the domain $A$ itself is always definable within the structure $\mathbf{A}$. This is immediate by taking the formula $\varphi(x)$ to be $x = x$, which defines the whole domain. This simple observation ensures that at each successor step of the constructible hierarchy, the previous stage is included as an element of the next stage.
