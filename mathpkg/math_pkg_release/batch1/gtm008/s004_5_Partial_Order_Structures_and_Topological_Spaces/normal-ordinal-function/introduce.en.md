A normal function is a strictly increasing ordinal function that is continuous at limit ordinals. Every normal function is also semi-normal (since strict monotonicity implies $\alpha \leq F(\alpha)$). Normal functions have the important property that they possess arbitrarily large fixed points (ordinals $\beta$ such that $F(\beta) = \beta$). This is a classical result in set theory, related to the Veblen fixed point theorem.
