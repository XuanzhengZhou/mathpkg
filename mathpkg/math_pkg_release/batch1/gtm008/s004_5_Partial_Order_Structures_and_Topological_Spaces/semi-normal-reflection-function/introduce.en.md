Theorem 7.17 strengthens Theorem 7.16 by showing that the witnesses for reflection can be produced uniformly by a semi-normal function $F$. For each formula $\varphi$, there is a semi-normal $F$ such that at every fixed point $\beta = F(\beta)$, the structure $M_\beta$ correctly reflects existential statements about $\varphi$ with parameters from stages below. This uniformity is essential for applying the fixed-point theorem to handle multiple formulas simultaneously.
