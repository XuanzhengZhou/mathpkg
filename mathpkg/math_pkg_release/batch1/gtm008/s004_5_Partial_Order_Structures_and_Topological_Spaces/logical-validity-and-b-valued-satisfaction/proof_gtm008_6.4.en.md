($\Leftarrow$) If $\varphi$ is logically valid, then $\varphi$ is satisfied in every 2-valued structure. Since $2 = \{0, 1\}$ is a complete subalgebra of $B$, every 2-valued structure is (the reduct of) a $B$-valued structure. Hence $\varphi$ is satisfied in every $B$-valued structure.

($\Rightarrow$) Conversely, suppose $\varphi$ is not logically valid, i.e., not satisfied by some 2-valued structure. Equivalently, if $\varphi$ is not satisfied by some $B$-valued structure $A$, i.e., $[\![\varphi]\!]_A = b \neq 1$, then the computation of $[\![\varphi]\!]_A$ requires only a finite number of applications of Definition 6.2 (clauses 5 and 6). By considering the finite Boolean subalgebra of $B$ generated by these finitely many values, and using the fact that every finite Boolean algebra is a subalgebra of a power set algebra (hence admits 2-valued homomorphisms), one obtains a 2-valued structure in which $\varphi$ is false. Therefore $\varphi$ is not logically valid.
