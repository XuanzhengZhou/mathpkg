Definition 5.1: x \neq y \rightarrow (\exists N(x))[y \notin N(x)] \wedge (\exists N(y))[x \notin N(y)] from Partial Order Structures and Topological Spaces of Takeuti and Zaring's "Axiomatic Set Theory" (GTM 8). This definition is part of the foundational development of Boolean-valued models and forcing techniques in set theory.
