The following theorem is a kind of completeness theorem for forcing.

**Theorem 10.6.** If $\langle M, P \rangle$ is a setting for forcing then

$$p \models \varphi \leftrightarrow (\forall G')[G' \text{ is } P\text{-generic over } M \wedge p \in G' \rightarrow M[G'] \models \varphi].$$

**Proof.** Using the one-to-one correspondence between $P$-generic sets over $M$ and $M$-complete homomorphisms from $B$ into $2$ we need only show:

$$p \models \varphi \leftrightarrow (\forall h')[h' : |B| \rightarrow |2| \text{ is an } M\text{-complete homomorphism}$$

$$\wedge h'([p]^{-0}) = 1 \rightarrow h'([\varphi]) = 1].$$

But the right-hand side is equivalent to

$$[p]^{-0} \leq [[\varphi]]$$

which in turn is equivalent to each of the following:

$$p \in [[\varphi]]$$
$$p \models \varphi.$$
