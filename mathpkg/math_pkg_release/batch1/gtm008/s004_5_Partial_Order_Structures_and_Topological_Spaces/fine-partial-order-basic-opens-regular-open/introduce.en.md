Lemma 5.22 states that in a fine partial order structure $P$, every basic open set $[p] = \{x \in P : x \leq p\}$ is regular open, i.e., equals the interior of its own closure. The proof uses the fine property: if $q \notin [p]$, then $q \not\leq p$, so there exists $r \leq q$ incompatible with $p$, which implies $[r] \cap [p] = 0$, so $r \notin [p]^{-}$ and consequently $q \notin [p]^{-0}$.
