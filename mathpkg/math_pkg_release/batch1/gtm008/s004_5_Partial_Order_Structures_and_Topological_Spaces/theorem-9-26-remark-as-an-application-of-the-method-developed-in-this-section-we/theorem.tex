Remark. As an application of the method developed in this section we shall define $V[F]$ by using a ramified language. In §7 $L[K]$ was introduced as the class of all sets that are constructible from $K$. The class $K$ can be identified with its characteristic function $F: K \rightarrow 2$. However we now consider a Boolean-valued set i.e., a function $F: K \rightarrow |\mathbf{B}|$ and regard the class of sets constructible from $K$ in the B-valued sense. If we think of $V$ as the class of 2-valued sets $V[F]$ becomes an extension of $V$ containing new sets.

Assume $K \in V, f_0: K \rightarrow B$ and $\mathcal{L}$ is the language $\mathcal{L}_0(C(V) \cup \{V(\cdot), F(\cdot)\})$ therefore we have individual constants $k$ for each $k \in V$.
