$L_K$ is the class of sets constructible relative to a class $K$. The construction generalizes Godel's $L$: instead of starting from $\emptyset$ and using pure definability, one adds a unary predicate $K(\cdot)$ to the language, and at each successor stage takes all sets definable from the current structure augmented with the interpretation $\bar{K}_\alpha = A_\alpha \cap K$. The union over all ordinals yields $L_K$. Important special cases include $L_a$ (where we can intersect with $a$) and $L[a]$ (where $a$ itself is an element).
