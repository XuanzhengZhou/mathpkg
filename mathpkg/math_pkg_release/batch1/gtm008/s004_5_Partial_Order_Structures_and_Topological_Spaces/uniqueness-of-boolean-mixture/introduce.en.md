Theorem 6.9 establishes the uniqueness of the Boolean mixture $a = \sum_{i \in I} b_i a_i$ in a complete $B$-valued structure. If two elements $a$ and $a'$ both satisfy $b_i \leq [\![a = a_i]\!]$ for a partition of unity $\{b_i\}$, then they are equal with truth value 1. This is a key technical result used to construct complete $B$-valued structures where every partition of unity can be "realized" by an element of the domain.
