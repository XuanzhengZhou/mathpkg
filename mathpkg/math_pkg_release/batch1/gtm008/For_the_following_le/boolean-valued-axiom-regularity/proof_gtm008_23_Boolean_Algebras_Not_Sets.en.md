The proof follows the same pattern as for set-sized Boolean algebras. In $V^{(B)}$ (or $V^{(\tilde{B})}$), each $u$ is constructed at some stage $\alpha$ from functions whose domains are subsets of earlier stages $V_{\xi}^{(B)}$ with $\xi < \alpha$. Thus the Boolean-valued $\in$-relation is well-founded: the "rank" of an element in the Boolean-valued universe strictly decreases along any $\in$-chain. Consequently, the Boolean truth value of the Axiom of Regularity evaluates to 1.
