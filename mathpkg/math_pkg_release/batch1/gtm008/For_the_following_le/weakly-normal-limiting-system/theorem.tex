A weakly normal limiting system is obtained from a normal limiting system by replacing condition 4 with condition 4':
If $x \in P_\alpha$, $y \in P_\beta$, $\alpha < \beta < \kappa$, then
$$\text{Comp}(x, p_{\alpha\beta}(y)) \leftrightarrow \text{Comp}(x, y).$$
If the union $\bigcup_{\alpha < \kappa} P_\alpha$ satisfies the fineness condition
$$(\forall p, q)[q \not\leq p \rightarrow (\exists r)[r \leq q \land \neg \text{Comp}(r, p)]],$$
then the system is normal.
