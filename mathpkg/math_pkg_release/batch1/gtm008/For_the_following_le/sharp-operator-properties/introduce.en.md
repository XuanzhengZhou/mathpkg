Given a complete monomorphism $i: B_0 \to B_1$ between complete Boolean algebras, one can define a left adjoint $\#: B_1 \to B_0$ by $\#(b_1) = \inf\{b_0 \mid b_1 \leq i(b_0)\}$. This "sharp" operator satisfies key algebraic properties: it is order-preserving, it is a left inverse to $i$ on the image, and it interacts well with the Boolean meet operation. The sharp operator is the algebraic counterpart of the open continuous onto map between the associated Stone spaces (partial order structures $P_0, P_1$), and its properties ensure that the topological map induces the original complete monomorphism.
