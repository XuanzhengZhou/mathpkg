This theorem states that for every element $x$ of the Boolean-valued universe, there exists a set $k$ in the ground model such that the Boolean truth value of $x$ being in $M$ is bounded by the truth value of $x$ belonging to $\check{k}$. In other words, any set that is in the ground model (in the Boolean sense) is necessarily contained in some ground model set of sufficiently large rank. This is a technical lemma used to establish further properties of the $M$ predicate.
