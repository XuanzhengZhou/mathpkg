This corollary follows directly from Corollary 17.5 by internalizing the statement. For each cardinal $\alpha$, Corollary 17.5 gives $[\text{Card}(\check{\alpha})] = 1$. Conversely, by Theorem 17.1, if $\alpha$ is not a cardinal then $[\neg \text{Card}(\check{\alpha})] = 1$. Since $B$ satisfies the c.c.c., these hold for all ordinals, so the universal statement $(\forall v)[\text{Card}(v) \leftrightarrow \text{Card}(\check{v})]$ has Boolean value $1$ in $V^{(B)}$.
