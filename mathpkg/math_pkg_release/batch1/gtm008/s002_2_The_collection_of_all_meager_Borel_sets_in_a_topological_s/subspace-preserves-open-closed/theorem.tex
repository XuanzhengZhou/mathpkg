If $\langle X, T \rangle$ is a topological space and $X' \subseteq X$ with subspace topology $T' = \{X' \cap N \mid N \in T\}$, then:
1. If $A$ is open in $T$, then $A \cap X'$ is open in $T'$.
2. If $A$ is closed in $T$, then $A \cap X'$ is closed in $T'$.
3. If $A$ is both open and closed in $T$, then $A \cap X'$ is both open and closed in $T'$.
