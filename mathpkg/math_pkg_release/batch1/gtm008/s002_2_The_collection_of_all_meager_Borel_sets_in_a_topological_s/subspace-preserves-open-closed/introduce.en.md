This result shows that the subspace topology behaves well with respect to the Boolean structure of the original topology: the relativization of an open set is open, of a closed set is closed, and of a clopen set is clopen. The proof is a straightforward verification using the definition $T' = \{X' \cap N \mid N \in T\}$.
