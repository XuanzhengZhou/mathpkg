Let $I$ be the ideal generated by $E$. Then $E \subseteq I$, so every upper bound for $I$ is an upper bound for $E$. Conversely,
$$(\forall b \in I) (\exists b_1, \ldots, b_n \in E) [b \leq b_1 + \cdots + b_n].$$
Therefore every upper bound for $E$ is also an upper bound for $I$.

By Zorn's Lemma, there exists a maximal set $F$ of pairwise disjoint nonzero elements of $I$. By the c.c.c., $\overline{\overline{F}} \leq \omega$. Since $F \subseteq I$, every upper bound for $I$ is an upper bound for $F$.

If $b_0$ is an upper bound for $F$ that is not an upper bound for $I$, then
$$(\exists b_1 \in I) [b_1 \not\leq b_0].$$
Therefore $b_1 - b_0 \in I$. Furthermore, since $(\forall b \in F) [b \leq b_0]$,
$$(\forall b \in F) [b \cap (b_1 - b_0) = 0].$$
Then $F \cup \{b_1 - b_0\}$ is a collection of pairwise disjoint elements of $I$, contradicting the maximality of $F$. Thus every upper bound for $F$ is also an upper bound for $I$.

We have established that $F$ is countable and has the same upper bounds as $E$. For each $f \in F$, choose $e_f \in E$ with $f \leq e_f$ (possible since $F \subseteq I$). Set $D = \{e_f \mid f \in F\}$, which is countable. Then $D$ has the same upper bounds as $F$, hence the same as $E$. Indeed,
$$\sum_{b \in E} b = \sum_{b \in D} b.$$
