**Proof.**

1. $a + a = (a + a)1 = (a + a)(a + a^-) = a + a a^- = a + 0 = a.$

2. $a + ab = a1 + ab = a(1 + b) = a(b^- + b + b) = a(b^- + b) = a1 = a.$

The proofs of the multiplicative properties are left to the reader.
