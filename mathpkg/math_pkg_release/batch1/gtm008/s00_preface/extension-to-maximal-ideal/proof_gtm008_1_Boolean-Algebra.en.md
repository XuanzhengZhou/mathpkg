**Proof.** For any well-ordering $R$ of $|B|$, define by transfinite recursion:
$$I_0 = I.$$
$$I_{\alpha+1} = \text{the ideal generated by } I_{\alpha} \cup \{a\}, \text{ where } a \text{ is the first element of } |B| \text{ for which } a \notin I_{\alpha} \text{ and } -a \notin I_{\alpha}, \text{ if such exists;} = I_{\alpha} \text{ otherwise.}$$
$$I_{\alpha} = \bigcup_{\beta < \alpha} I_{\beta}, \quad \alpha \text{ a limit ordinal.}$$

Let $J = \bigcup_{\alpha \in O_n} I_{\alpha}$. Then $I \subseteq J \subseteq |B|$. If $x, y \in J$, then for some $\alpha, \beta$, $x \in I_{\alpha}$ and $y \in I_{\beta}$. Taking $\gamma = \max(\alpha, \beta)$, we have $x, y \in I_{\gamma}$, so $x + y \in I_{\gamma} \subseteq J$. Also, if $a \in |B|$, then $ax \in I_{\gamma} \subseteq J$. Since $0 \in I \subseteq J$ and $1 \notin I_{\alpha}$ for all $\alpha$ (by construction), $1 \notin J$, so $J$ is proper.

$J$ is maximal because if there existed $a$ with $a \notin J$ and $-a \notin J$, then $a$ would have been added to some $I_{\alpha}$, a contradiction.
