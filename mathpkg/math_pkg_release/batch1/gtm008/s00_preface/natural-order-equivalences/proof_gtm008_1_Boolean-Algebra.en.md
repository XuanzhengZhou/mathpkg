**Proof.** 1. If $a \leq b$, then $a = ab$, so $-a = -(ab) = -a + -b$, giving $(-b)(-a) = -b$, i.e., $-b \leq -a$. The converse is similar.
2. $a \leq b \leftrightarrow a = ab \leftrightarrow a(-b) = 0 \leftrightarrow a - b = 0$.
3. $a \leq b \leftrightarrow a - b = 0 \leftrightarrow -a + b = 1 \leftrightarrow (a \Rightarrow b) = 1$.
