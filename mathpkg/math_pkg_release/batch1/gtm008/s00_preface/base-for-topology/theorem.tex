If $T' \subseteq \mathcal{P}(X)$ is such that $(\forall x \in X)[x \in \bigcup (T')]$ and $(\forall x \in X)(\forall A, B \in T')[x \in A \cap B \rightarrow (\exists C \in T')[x \in C \subseteq A \cap B]]$, then $T \triangleq \{B \subseteq X \mid (\exists C \subseteq T') [B = \bigcup (C)]\}$ is a topology on $X$ and $T'$ is a base for $T$.
