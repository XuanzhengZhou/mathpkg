If $I$ is a proper ideal in the Boolean algebra $B$ then:
\begin{enumerate}
\item $a/I = b/I \leftrightarrow a(-b) + b(-a) \in I$.
\item $a/I = c/I \land b/I = d/I \rightarrow (a + b)/I = (c + d)/I$.
\item $a/I = c/I \land b/I = d/I \rightarrow ab/I = cd/I$.
\item $a/I = b/I \rightarrow -a/I = -b/I$.
\end{enumerate}
