This definition classifies filters in a Boolean algebra. An ultrafilter is a maximal proper filter characterized by containing exactly one of $x$ or $-x$ for each element. A principal filter is generated by a single element. An $M$-complete filter is closed under products of subsets belonging to a model $M$. These classifications are essential for the correspondence between generic sets and ultrafilters (Theorems 2.16 and 2.17).
