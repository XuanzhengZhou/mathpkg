**Proof.** (The source text is partially cut off. The standard proof follows from the Boolean algebra axioms.) To show $(a + b)^- = a^- b^-$, note that $(a + b)(a^- b^-) = (a a^- b^-) + (b a^- b^-) = 0 + 0 = 0$, and $(a + b) + a^- b^- = (a + b + a^-)(a + b + b^-) = 1 \cdot 1 = 1$. By uniqueness of complement, $(a + b)^- = a^- b^-$. The dual statement follows similarly.
