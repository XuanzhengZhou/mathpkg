**Proof.** If $q_1, q_2 \in G$, then $p \leq q_1$ and $p \leq q_2$, so $\{q_1, q_2\}$ is compatible. If $q_1 \in G$, $q_2 \in P$, and $q_1 \leq q_2$, then $p \leq q_1 \leq q_2$, so $q_2 \in G$. If $S \in A$ is dense in $P$, then for any $q \in P$, $[q] \cap S \neq 0$. In particular, $[p] \cap S \neq 0$. But $p$ is minimal, so $[p] = \{p\}$. Hence $p \in S$, and since $p \in G$, we have $G \cap S \neq 0$.
