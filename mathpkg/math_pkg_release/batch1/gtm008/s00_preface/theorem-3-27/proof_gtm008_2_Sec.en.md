If $I$ is the ideal generated by $E$ then $E \subseteq I$. Consequently every upper bound for $I$ is an upper bound for $E$. Conversely

$$(\forall b \in I) \left( \exists b_1, \ldots, b_n \in E \right) \left[ b \leq b_1 + \cdots + b_n \right].$$

Therefore every upper bound for $E$ is also an upper bound for $I$.

From Zorn’s Lemma there exists a maximal set $F$ of disjoint elements of $I$. By the c.c.c. $\bar{F} \leq \omega$. Since $F \subseteq I$ every upper bound for $I$ is an upper bound for $F$. If $b_0$ is an upper bound for $F$ that is not an upper bound for $I$ then

$$(\exists b_1 \in I) \left[ b_1 \not\leq b_0 \right].$$

Therefore $b_1 - b_0 \in I$. Furthermore, since $(\forall b \in F) \left[ b \leq b_0 \right]$$

$$(\forall b \in F) \left[ b \cap \left( b_1 - b_0 \right) = 0 \right].$$

Then $F \cup \{b_1 - b_0\}$ is a collection of pairwise disjoint elements of $I$. But this contradicts the definition of $F$. Thus every upper bound for $F$ is also an upper bound for $I$.
