**Proof.**

1. If $A = A^{-0}$ then $A^0 = A^{-00} = A^{-0} = A$, so $A$ is open.

2. $A = A^0 \iff (X - A) = (X - A^0) \iff (X - A) = (X - A)^-$ (by Theorem 1.22.4).

3. Left to the reader.

4. $A \subseteq B \rightarrow A^- \subseteq B^-$. But $A$ dense in $X$ implies $A^- = X$. Hence $B^- = X$, so $B$ is dense in $X$.
