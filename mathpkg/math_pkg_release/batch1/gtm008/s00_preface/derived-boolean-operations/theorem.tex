If $\langle B, +, \cdot, ^-, 0, 1 \rangle$ is a Boolean algebra then $\forall a, b \in B$:
\begin{enumerate}
\item $(a - b) \triangleq a(-b)$.
\item $(a \Rightarrow b) \triangleq -a + b$.
\item $(a \Leftrightarrow b) \triangleq (a \Rightarrow b)(b \Rightarrow a)$.
\item $(a \leq b) \Leftrightarrow ab = a$.
\end{enumerate}
