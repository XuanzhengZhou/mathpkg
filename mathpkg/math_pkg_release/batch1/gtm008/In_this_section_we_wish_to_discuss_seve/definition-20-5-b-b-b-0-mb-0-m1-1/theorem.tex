$$(\forall b \in B) [b \neq 0 \rightarrow m(b) > 0] \wedge m(1) = 1$$

and

$$(\forall b \in B^{\omega}) \left[ (\forall i, j < \omega) [i \neq j \rightarrow b_i \cdot b_j = 0] \rightarrow m \left( \sum_{i < \omega} b_i \right) = \sum_{i < \omega} m(b_i) \right].$$

Remark. Note that a measure algebra always satisfies the c.c.c. and hence it is complete.
