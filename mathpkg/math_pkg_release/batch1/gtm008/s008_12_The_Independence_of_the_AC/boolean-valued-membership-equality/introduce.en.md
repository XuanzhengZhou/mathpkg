This definition assigns Boolean truth values $[\![u \in v]\!]$ and $[\![u = v]\!]$ to atomic formulas in $V^{(B)}$. The membership value is the supremum (Boolean sum) of $v(y) \cdot [\![u = y]\!]$ over all $y$ in the domain of $v$. The equality value is defined recursively using Boolean implication, encoding the extensionality principle that two sets are equal if they have the same members with the same weights. The recursion is well-founded via an ordinal measure such as the natural sum or Godel's pairing function.
