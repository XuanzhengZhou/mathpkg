This theorem characterizes the ordinal property in the Boolean-valued universe: an element $u$ of $V^{(B)}$ is an ordinal (with maximal Boolean truth value) exactly when it is forced to equal the canonical image of a standard ordinal $\alpha$. The proof uses the fact that for distinct ordinals $\alpha \neq \beta$, the Boolean values $[\![u = \check{\alpha}]\!]$ and $[\![u = \check{\beta}]\!]$ are disjoint, so the set $\{\xi \mid [\![u = \check{\xi}]\!] > 0\}$ is a set (by injectivity into $B$), and an ordinal bound on this set gives the result.
