$\mathbf{B}$ satisfies the $(\omega_\alpha, \omega_\beta)$-WDL iff for every family
$$\{b_{\xi \eta} \mid \xi < \omega_\alpha \land \eta < \omega_\beta\} \subseteq B,$$
$$\prod_{\xi < \omega_\alpha} \sum_{\eta < \omega_\beta} b_{\xi \eta} = \sum_{f \in \omega_\beta^{\omega_\alpha}} \prod_{\xi < \omega_\alpha} \sum_{\eta < f(\xi)} b_{\xi \eta}.$$
