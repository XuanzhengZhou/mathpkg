A function $F: \mathrm{On} \to \mathrm{On}$ is \text{semi-normal} iff:
\begin{enumerate}
\item $(\forall \alpha)[\alpha \leq F(\alpha)]$.
\item $(\forall \alpha, \beta)[\alpha < \beta \to F(\alpha) \leq F(\beta)]$.
\item $(\forall \alpha \in K_{\mathrm{II}})\left[F(\alpha) = \bigcup_{\beta < \alpha} F(\beta)\right]$, where $K_{\mathrm{II}}$ denotes the class of limit ordinals.
\end{enumerate}
