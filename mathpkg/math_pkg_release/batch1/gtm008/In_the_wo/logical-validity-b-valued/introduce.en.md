A closed formula $\varphi$ is true in all $B$-valued structures exactly when it is logically valid (true in all ordinary two-valued structures). The forward direction uses that $2$ is a complete subalgebra of any nontrivial complete Boolean algebra $B$, so every two-valued structure is also a $B$-valued structure. The converse follows from the fact that if $\varphi$ fails in some $B$-valued structure, the failure is witnessed by a finite partial computation, which can be embedded in a two-valued counterexample.
