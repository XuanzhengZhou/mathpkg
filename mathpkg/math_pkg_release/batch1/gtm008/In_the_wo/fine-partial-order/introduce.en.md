A partial order structure is called "fine" if whenever one element does not lie below another, there is a lower bound of the first that is incompatible with the second. This property ensures that the basic open sets $[p]$ of the partial order behave nicely with respect to the regularization operation: in a fine partial order, $[p]^{-0} = [p]$ for every $p$. Many partial orders used in forcing and independence proofs are fine.
