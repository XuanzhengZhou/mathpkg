A function $F: \mathrm{On} \to \mathrm{On}$ is a \text{normal function} iff:
\begin{enumerate}
\item $(\forall \alpha, \beta)[\alpha < \beta \to F(\alpha) < F(\beta)]$.
\item $(\forall \alpha \in K_{\mathrm{II}})\left[F(\alpha) = \bigcup_{\beta < \alpha} F(\beta)\right]$, where $K_{\mathrm{II}}$ denotes the class of limit ordinals.
\end{enumerate}
Every normal function is strictly monotonic. Since $\alpha \leq F(\alpha)$ for every strictly monotonic ordinal function, every normal function is also semi-normal.
