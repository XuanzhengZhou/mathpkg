In Boolean-valued semantics, the truth of a formula is not a simple true/false but an element of a complete Boolean algebra $B$. Logical validity is defined as having truth value $1$ in every $B$-valued structure for every complete Boolean algebra $B$. This generalizes classical Tarskian semantics: when $B$ is the two-element Boolean algebra $\{0, 1\}$, Boolean-valued semantics reduces to ordinary two-valued semantics. The Boolean-valued approach provides a uniform framework for independence proofs in set theory.
