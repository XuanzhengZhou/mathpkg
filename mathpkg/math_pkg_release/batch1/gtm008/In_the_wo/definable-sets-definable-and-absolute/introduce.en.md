The operation $\operatorname{Df}$ is itself definable in the language of set theory, and its computation is absolute for transitive models containing the structure $\mathbf{A}$ as an element. This absoluteness is essential for the internal definability of the constructible hierarchy: each $L_\alpha$ can be defined within any transitive model that contains it, and the construction proceeds uniformly across all models.
