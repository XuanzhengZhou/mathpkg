Any finite collection of semi-normal functions has arbitrarily large common fixed points. The proof constructs an $\omega$-sequence by iterating the pointwise supremum of the functions starting from $\alpha$, then taking the limit. This result is the engine behind the reflection principle: for any finite set of formulas, one can find ordinals $\beta$ where all the corresponding semi-normal functions simultaneously reach fixed points, guaranteeing that truth in the full model $M$ coincides with truth in the substructure $M_\beta$.
