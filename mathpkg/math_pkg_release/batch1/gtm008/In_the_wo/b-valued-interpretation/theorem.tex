Let $\mathcal{L}$ be a first order language with individual constants $c_0, c_1, \ldots, c_i, \ldots$ ($i < \alpha$) and predicate constants $R_0, R_1, \ldots, R_j, \ldots$ ($j < \beta$). A \text{$B$-valued interpretation} of $\mathcal{L}$ is a pair $\langle A, \phi \rangle$ where $A$ is a nonempty set and $\phi$ is a mapping defined on the set of constants of $\mathcal{L}$ satisfying:
\begin{enumerate}
\item $\phi(c_i) \in A$, $i < \alpha$.
\item $\phi(R_j) \colon A^{n_j} \to B$, for $j < \beta$, where $n_j$ is the number of arguments of $R_j$.
\end{enumerate}
