For a set $x$, $\operatorname{Def}(x)$ is the set of all $y \subseteq x$ such that $y$ is definable over the structure $\langle x, \in \rangle$ with parameters from $x$. That is, $y \in \operatorname{Def}(x)$ iff there exists a formula $\varphi(v, u_1, \ldots, u_n)$ and parameters $a_1, \ldots, a_n \in x$ such that

$$y = \{v \in x \mid \langle x, \in \rangle \models \varphi[v, a_1, \ldots, a_n]\}.$$
