In any $B$-valued structure $\mathbf{A} = \langle A, \|{\cdot}\| \rangle$ for a language $\mathcal{L}$ with equality, the standard equality axioms are logically valid (have truth value $1$):
\begin{itemize}
\item $\|x = x\| = 1$
\item $\|x = y \rightarrow y = x\| = 1$
\item $\|x = y \land y = z \rightarrow x = z\| = 1$
\item $\|x_1 = y_1 \land \cdots \land x_n = y_n \rightarrow (P(x_1, \ldots, x_n) \leftrightarrow P(y_1, \ldots, y_n))\| = 1$
\end{itemize}
