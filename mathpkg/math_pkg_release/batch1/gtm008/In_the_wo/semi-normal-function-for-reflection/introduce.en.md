This theorem extracts from the reflection principle a canonical semi-normal function $F$ for each formula $\varphi$, such that at the image points $F(\alpha)$, existential reflection holds for $\varphi$ with parameters from $M_\alpha$. Combined with the common fixed point theorem, this yields that for any finite collection of formulas, there are ordinals where reflection holds simultaneously for all of them.
