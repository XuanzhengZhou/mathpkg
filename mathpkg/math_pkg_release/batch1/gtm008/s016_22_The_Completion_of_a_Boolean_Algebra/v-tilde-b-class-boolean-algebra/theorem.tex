For a complete Boolean algebra $\tilde{B}$ (possibly a proper class), the Boolean-valued universe $V^{(\tilde{B})}$ is defined as:

\begin{enumerate}
\item $V_{0}^{(\tilde{B})} \triangleq 0$.
\item $V_{\alpha}^{(\tilde{B})} \triangleq \{u \in {}^{\tilde{B}} \mathcal{D}(u) \mid (\exists \xi < \alpha)[\mathcal{D}(u) \subseteq V_{\xi}^{(\tilde{B})}]\}$.
\item $V^{(\tilde{B})} \triangleq \bigcup_{\alpha \in \mathrm{On}} V_{\alpha}^{(\tilde{B})}$.
\end{enumerate}

Here $\tilde{B}$ is required to be complete, i.e., $\sup A$ exists in $\tilde{B}$ for every class $A \subseteq \tilde{B}$.$$
