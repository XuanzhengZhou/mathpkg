Let $\kappa$ be a cardinal or $\mathrm{On}$. Let $X_i$ ($i < \kappa$) be topological spaces and define $X^{\alpha} = \prod_{i < \alpha} X_i$ for $\alpha < \kappa$ and $X = \prod_{i < \kappa} X_i$ with the $\lambda$-product topologies. Define $p_{\alpha\beta}$ ($\alpha \leq \beta < \kappa$) to be the projection from $X^{\beta}$ onto $X^{\alpha}$.

Then $\langle \{X^{\alpha} \mid \alpha < \kappa\}, \{p_{\alpha\beta} \mid \alpha \leq \beta < \kappa\} \rangle$ is an o.c.o. inverse system.

Moreover, if $\mathrm{cf}(\kappa) \geq \lambda$, then $X$ is homeomorphic to $\varprojlim_{\alpha \to \kappa} X^{\alpha}$.$$
