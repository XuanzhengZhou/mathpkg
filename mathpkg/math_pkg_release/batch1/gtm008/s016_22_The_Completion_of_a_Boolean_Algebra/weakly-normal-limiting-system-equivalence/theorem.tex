Let $\kappa$ be a cardinal or $\mathrm{On}$. Let $P_0 \subseteq P_1 \subseteq \cdots \subseteq P_{\alpha} \subseteq \cdots$ ($\alpha < \kappa$) be a weakly normal limiting system. If $\bigcup_{\alpha < \kappa} P_{\alpha}$ satisfies

$$(\forall p, q \in \bigcup_{\alpha < \kappa} P_{\alpha})[q \not\leq p \rightarrow (\exists r \in \bigcup_{\alpha < \kappa} P_{\alpha})[r \leq q \land \neg \mathrm{Comp}(r, p)]],$$

then the system is a normal limiting system.$$
