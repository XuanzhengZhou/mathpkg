Theorem 13.15 shows that when $B$ is a complete subalgebra of $B'$, the Boolean-valued universe $V^{(B)}$ embeds naturally into $V^{(B')}$, and the Boolean truth values of atomic formulas are preserved. This is because any function into $B$ is automatically a function into $B'$, and Boolean sums and products over values in $B$ compute identically in both $B$ and $B'$. This embedding result is a key tool for comparing different Boolean-valued models.
