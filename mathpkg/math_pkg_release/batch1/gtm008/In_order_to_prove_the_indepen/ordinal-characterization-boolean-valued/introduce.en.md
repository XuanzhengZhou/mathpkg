Theorem 13.22 characterizes the Boolean-valued ordinals in $V^{(B)}$: the Boolean truth value of "$u$ is an ordinal" equals the supremum over all standard ordinals $\alpha$ of the truth value that $u$ equals $\check{\alpha}$. In other words, any $B$-valued set that satisfies the ordinal property (with Boolean truth value) is a "mixture" of standard ordinals. This is the Boolean-valued analogue of the fact that forcing does not add new ordinals.
