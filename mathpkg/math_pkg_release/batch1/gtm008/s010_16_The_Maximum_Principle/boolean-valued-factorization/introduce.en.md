Corollary 16.21 generalizes Theorem 16.20 by combining the extensional representation theorem with the uniform set factorization. Since every Boolean-valued set can be represented by an extensional set with a complete domain (by moving to $U_\alpha^{(\mathbf{B})}$ via Theorem 16.6), and any such uniform set factors as $b \cdot v$ with $v$ definite, every $u \in V^{(\mathbf{B})}$ is Boolean-equal to $b \cdot v$ for some Boolean scalar $b$ and some definite uniform set $v$. This provides a complete structural decomposition of the Boolean-valued universe.
