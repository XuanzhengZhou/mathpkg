Theorem 16.1 provides a technical tool for working with families of Boolean-valued sets in $V^{(\mathbf{B})}$. Given any family $\{u_i\}$ whose domains are all contained in some set $d$, one can replace each $u_i$ by an equivalent set $u'_i$ that has domain exactly $d$, while preserving Boolean-valued equality $\llbracket u_i = u'_i \rrbracket = 1$. This domain normalization is essential for the proof of the Maximum Principle and subsequent results about Boolean-valued models.
