$\text{Ord}^1(P(t)) = g(t) = \text{Ord}^1(t \simeq k)$, since $\text{rank}(k) \leq \rho(t)$. Moreover,

$$\text{Ord}^2(P(t)) = 1$$

But

$$\text{Ord}^2(t \simeq k) = 0.$$

Hence

$$\text{Ord}(t \simeq k) < \text{Ord}(P(t)).$$
