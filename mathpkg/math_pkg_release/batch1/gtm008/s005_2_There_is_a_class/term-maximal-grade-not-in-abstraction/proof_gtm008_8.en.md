If $t_1 = \hat{x}^\alpha \psi(x^\alpha)$ is an abstraction term such that $t$ occurs in $t_1$ and $t_1$ occurs in $\varphi$, then $t$ occurs in $\psi(x^\alpha)$. Hence by Definition 8.4,

$$g(t_1) = 2\alpha + 2 \leq \text{Ord}^1(\varphi).$$

If $t = k$ for some $k \in K$ then $\text{rank}(k) = \rho(k) < \alpha$.

If $t = \hat{y}^\beta \psi(y^\beta)$ with $\beta < \alpha$ then $g(t) < 2\alpha + 1 < \text{Ord}^1(\varphi)$.

So if $t$ occurs in an abstraction term of $\varphi$ then $\text{Ord}^1(\varphi) > g(t)$, contradicting $\text{Ord}^1(\varphi) = g(t)$.
