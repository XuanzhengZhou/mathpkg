A fundamental result in the theory of vector lattices: the order dual $E^+$ of any vector lattice $E$ is always an order complete vector lattice. This follows from Theorem 1.4 since every vector lattice has the decomposition property (D). Even if $E$ fails to be order complete, its order dual possesses this strong completeness property.
