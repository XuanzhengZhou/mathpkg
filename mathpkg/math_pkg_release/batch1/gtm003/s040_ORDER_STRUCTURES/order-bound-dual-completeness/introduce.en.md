For an ordered vector space whose positive cone is generating and has the decomposition property (D), the order bound dual $E^b$ is always an order complete vector lattice. The proof constructs $\sup(f, 0)$ for each $f \in E^b$ using the decomposition property, showing that suprema exist in the dual. In particular, this implies $E^b = E^+$, so the order bound dual coincides with the order dual.
