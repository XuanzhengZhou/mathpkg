Let $E$ be a vector lattice and let $f, g$ be order bounded linear forms on $E$. For each $x \in E$, we have:
$$\sup(f, g)(|x|) = \sup\{f(y) + g(z): y \geq 0, z \geq 0, y + z = |x|\}$$
$$\inf(f, g)(|x|) = \inf\{f(y) + g(z): y \geq 0, z \geq 0, y + z = |x|\}$$
$$|f|(|x|) = \sup\{f(y - z): y \geq 0, z \geq 0, y + z = |x|\}$$
$$|f(x)| \leq |f|(|x|).$$
In particular, two linear forms $f \geq 0$, $g \geq 0$ are disjoint if and only if for each $x \geq 0$ and each real number $\varepsilon > 0$, there exists a decomposition $x = x_1 + x_2$ with $x_1 \geq 0, x_2 \geq 0$, and such that $f(x_1) + g(x_2) \leq \varepsilon$.
