Let $A$ be a C*-algebra and $A_+ = \{x \in A_{\mathrm{sa}} : \sigma(x) \subset \mathbb{R}_+\}$. Then:
\begin{itemize}
\item[(i)] $A_+$ is a proper, closed, convex, normal cone with vertex $0$.
\item[(ii)] $A_{\mathrm{sa}}$ is a real ordered Banach space with respect to the order generated by $A_+$.
\end{itemize}
