Let $x, y, c$ be elements of $A$.
\begin{itemize}
\item[(i)] $x \leq y$ implies $c^*xc \leq c^*yc$. In particular, $x^*y^*yx \leq \|y\|^2x^*x$.
\item[(ii)] $|x| \leq |y|$ implies $\|x\| \leq \|y\|$.
\item[(iii)] $0 \leq x \leq y$ implies $0 \leq \sqrt{x} \leq \sqrt{y}$.
\item[(iv)] If $A$ is unital, if $x$ is invertible and $0 < x \leq y$, then $y$ is invertible and $0 < y^{-1} \leq x^{-1}$.
\end{itemize}
