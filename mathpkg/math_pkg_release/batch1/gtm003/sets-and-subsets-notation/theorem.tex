Let $X, Y$ be sets. The notation $x \in X$ means "$x$ is an element of $X$", $X \subset Y$ (or $Y \supset X$) means "$X$ is a subset of $Y$", and $X = Y$ means "$X \subset Y$ and $Y \subset X$". If $(p)$ is a proposition, the subset of all $x \in X$ for which $(p)$ is true is denoted by $\{x \in X: (p)x\}$. $x \notin X$ means "$x$ is not an element of $X$". The complement of $X$ relative to $Y$ is $Y \sim X = \{x \in Y: x \notin X\}$. The empty set is denoted by $\varnothing$. The singleton containing $x$ is $\{x\}$. The set of all subsets of $X$ (the power set) is $\mathfrak{P}(X)$.
