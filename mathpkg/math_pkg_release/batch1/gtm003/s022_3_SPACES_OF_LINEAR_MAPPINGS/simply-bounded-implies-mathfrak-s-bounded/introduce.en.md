This result shows that simple boundedness (boundedness for pointwise convergence) implies boundedness for the finer topology of uniform convergence on all convex, circled, bounded, and complete sets. The proof uses the fact that for a simply bounded set \(H\), the intersection \(\bigcap_{u \in H} u^{-1}(V)\) of inverse images of a closed convex circled \(0\)-neighborhood is a barrel, which absorbs each complete convex circled bounded set by the Banach-Steinhaus principle for barrels.
