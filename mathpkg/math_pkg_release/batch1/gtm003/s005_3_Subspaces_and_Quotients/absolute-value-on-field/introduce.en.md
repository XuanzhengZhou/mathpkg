An absolute value (or multiplicative valuation) on a field K is a real-valued function satisfying positive definiteness, the triangle inequality, and multiplicativity. These axioms generalize the familiar absolute value on the real numbers and the p-adic valuations. An absolute value induces a metric and thereby a topology on the field, turning it into a valuated field which serves as the scalar field for topological vector spaces.
