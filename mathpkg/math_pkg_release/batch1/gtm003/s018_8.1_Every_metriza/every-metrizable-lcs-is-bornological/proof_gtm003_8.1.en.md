If $E$ is metrizable, there exists a countable $0$-neighborhood base $\{V_n: n \in \mathbb{N}\}$ by (I, 6.1), which can be chosen to be decreasing. Let $A$ be a convex, circled subset of $E$ that absorbs every bounded set; we must have $V_n \subset nA$ for some $n \in \mathbb{N}$. For if this were false, there would exist elements $x_n \in V_n$ such that $x_n \notin nA$ ($n \in \mathbb{N}$); since $\{x_n\}$ is a null sequence, it is bounded by (I, 5.1), Corollary 2, and hence absorbed by $A$, which is contradictory.
