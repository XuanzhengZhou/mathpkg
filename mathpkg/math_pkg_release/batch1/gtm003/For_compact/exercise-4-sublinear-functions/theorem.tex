Let $\psi$ be a sublinear function on $R^n$. Then:
\begin{itemize}
\item[(a)] Every sublinear function on $R^n$ is continuous.
\item[(b)] If $\psi(x_1, \ldots, x_n) \geq 0$ whenever $x_i \geq 0$ and $\psi(x_1, \ldots, x_n) \leq 0$ if all $x_i \leq 0$, and if $p_i$ are continuous semi-norms on a t.v.s. $E$, then $\psi(p_1, \ldots, p_n)$ is a continuous semi-norm on $E$.
\item[(c)] If additionally $x_i \geq 0$ and $\psi(x_1, \ldots, x_n) = 0$ imply $x_i = 0$, then for $n$ normed spaces $(E_i, p_i)$ the map $(x_1, \ldots, x_n) \to \psi[p_1(x_1), \ldots, p_n(x_n)]$ is a norm on $\prod_i E_i$ generating the product topology.
\end{itemize}
