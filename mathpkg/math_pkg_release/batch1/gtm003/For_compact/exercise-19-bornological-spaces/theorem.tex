Let $\{E_\alpha : \alpha \in A\}$ be a family of l.c.s. over $K$.
\begin{itemize}
\item[(a)] Assume $K_0^A$ is bornological. If $u$ is a linear map on $E = \prod_\alpha E_\alpha$ into a l.c.s. $F$ such that $u$ carries bounded sets onto bounded sets and vanishes on $\dot{E}_\alpha = \{x \in E: x_\beta = 0 \text{ for } \beta \neq \alpha\}$ for each $\alpha$, then $u = 0$.
\item[(b)] If $u: \prod_\alpha E_\alpha \to F$ with $F$ Banach maps bounded sets to bounded sets, then $u$ vanishes on all but finitely many $\dot{E}_\alpha$.
\item[(c)] If $K_0^A$ is bornological, any bounded-linear map from $\prod_\alpha E_\alpha$ to a Banach space factors through a finite product.
\item[(d)] $\prod_\alpha E_\alpha$ is bornological if each $E_\alpha$ and $K_0^A$ are bornological.
\end{itemize}
