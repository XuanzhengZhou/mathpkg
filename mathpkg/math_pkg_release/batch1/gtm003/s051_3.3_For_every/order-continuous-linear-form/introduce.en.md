An order continuous linear form on an ordered vector space preserves suprema of upward directed sets: whenever a directed set has a supremum $z$, the values of the linear form along the set converge to $\varphi(z)$. This concept abstracts the property of normal (order-continuous) functionals in $C^*$-algebra theory and is essential for Sakai's characterization of the predual of a $W^*$-algebra.
