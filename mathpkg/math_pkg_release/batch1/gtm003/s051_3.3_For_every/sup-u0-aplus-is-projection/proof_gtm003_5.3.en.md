Clearly, if $A$ is unital with unit $e$, then $e = \sup(U_0 \cap A_+)$; in fact, $e = \sup\{\lambda e : 0 \leq \lambda < 1\}$. Now suppose $A$ is not unital and let $(u_\lambda)_{\lambda \in \Lambda}$ be an approximate unit of $A$ (see 3.3). Since $0 \leq u_\lambda \leq p$, we have $p^{1/2}u_\lambda p^{1/2} \leq p^2$ by (3.2)(i). Since $u_\lambda p^{1/2} \rightarrow p^{1/2}$, by (3.2)(i), we have

$$p \leq p^2 = p^{1/2}pp^{1/2} \leq \|p\|p,$$

which implies $p = p^2$, i.e., $p$ is a projection.
