A $C^*$-algebra whose underlying normed space is a dual Banach space is called a $W^*$-\textit{algebra}. Every $C^*$-algebra can be viewed as a $C^*$-subalgebra of $\mathcal{L}(H)$ for a suitable Hilbert space $H$, and $\mathcal{L}(H)$ is a $W^*$-algebra because it is the norm dual of the trace class operators $L^1(H)$.
