A $C^*$-subalgebra of a $W^*$-algebra is itself a $W^*$-algebra if and only if it is closed in the canonical weak-* topology $\sigma(W, V)$. This provides a convenient criterion for recognizing $W^*$-subalgebras: one only needs to verify weak-* closedness. The result follows from the bipolar theorem and the fact that a $\sigma(W, V)$-closed subspace of a dual Banach space is itself a dual space.
