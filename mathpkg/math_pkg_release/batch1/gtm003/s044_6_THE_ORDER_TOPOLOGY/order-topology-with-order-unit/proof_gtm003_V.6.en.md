The order interval $[-e, e]$ is convex, circled, and (by the definition of order unit) radial in $L$; since $L$ is Archimedean ordered, the gauge $p_e$ of $[-e, e]$ is a norm on $L$. The topology generated by $p_e$ is finer than $\mathfrak{T}_0$ since $[-e, e]$ is $\mathfrak{T}_0$-bounded, and it is coarser than $\mathfrak{T}_0$, since it is locally convex and $[-e, e]$ absorbs order intervals; hence $p_e$ generates $\mathfrak{T}_0$. Thus $(L, \mathfrak{T}_0)$ is normable.

To see that $C$ is normal for $\mathfrak{T}_0$, let $U$ be a convex circled $0$-neighborhood. Since $[-e, e]$ is $\mathfrak{T}_0$-bounded, there exists $\rho > 0$ with $[-e, e] \subset \rho U$. If we define $V = U \cap (C - C)$, then $V$ is $C$-saturated: if $x \in V$ and $0 \leq y \leq x$, write $x = x_1 - x_2$ with $x_1, x_2 \in C$, then from the decomposition lemma one obtains $y \in V$, proving $C$ is normal via criterion (3.1)(c).

The fact that $\mathfrak{T}_0$ is the finest locally convex topology for which $C$ is normal follows: any locally convex topology $\mathfrak{T}$ with $C$ normal must have $[-e, e]$ bounded (since $e$ is an order unit), hence $\mathfrak{T}$ must be coarser than $\mathfrak{T}_0$ by definition of $\mathfrak{T}_0$.

For the equivalence of (a) and (b): If $(L, \mathfrak{T}_0)$ is complete and $\{x_n\}$ is a positive sequence of type $l^1$, then $\sum_{n=1}^{\infty} \lambda_n < \infty$ and $0 \leq x_n \leq \lambda_n a$ implies $\sum_{p=1}^{n} x_p \leq (\sum_{p=1}^{n} \lambda_p)a$, so the partial sums form a Cauchy sequence in the normed space $(L, p_e)$, hence converge, which implies that $\sup_n \sum_{p=1}^{n} x_p$ exists (since $L$ is Archimedean). Conversely, if every positive sequence of type $l^1$ is order summable, then completeness of $(L, p_e)$ follows by a standard argument using the order unit norm.
