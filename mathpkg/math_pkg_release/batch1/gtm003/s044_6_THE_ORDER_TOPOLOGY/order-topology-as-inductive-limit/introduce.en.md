This proposition describes the order topology for Archimedean ordered vector spaces without an order unit. $(L, \mathfrak{T}_0)$ is expressed as the locally convex inductive limit of the subspaces $L_a$, each of which has $a$ as an order unit and thus falls under the scope of Theorem 6.2. When each $L_a$ is complete, $(L, \mathfrak{T}_0)$ is barreled, which is a desirable property for applications in functional analysis.
