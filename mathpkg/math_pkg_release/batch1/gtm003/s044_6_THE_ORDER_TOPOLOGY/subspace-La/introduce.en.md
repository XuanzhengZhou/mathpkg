The subspace $L_a$ is the linear span of the order interval $[-a, a]$, endowed with the induced order. When $L$ does not possess an order unit, the order topology $\mathfrak{T}_0$ can be described as the locally convex inductive limit of the spaces $(L_a, \mathfrak{T}_{0,a})$, where each $L_a$ has $a$ as an order unit. This provides a way to extend the description given by Theorem 6.2 to spaces without order units.
