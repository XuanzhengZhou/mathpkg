Let $L$ be a vector space over $\mathbb{R}$. A pre-order ``$<$'' on $L$ is said to be \emph{compatible} with the vector structure of $L$ if for all $x, y, z \in L$ and $\lambda \geq 0$:
\begin{itemize}
\item $x < y$ implies $x + z < y + z$ (translation invariance);
\item $x < y$ implies $\lambda x < \lambda y$ (positive homogeneity).
\end{itemize}
