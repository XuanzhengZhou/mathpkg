A weak order unit is an element $x \geq 0$ in a vector lattice $L$ with the property that no non-zero element of $L$ is disjoint from $x$ (equivalently, the infimum of $x$ with the absolute value of any element is zero only for the zero element). This notion, introduced by Freudenthal, serves as a weaker substitute for order units, which are comparatively rare. In an order complete vector lattice of minimal type, weak order units coincide with quasi-interior points of the positive cone (see 7.7).
