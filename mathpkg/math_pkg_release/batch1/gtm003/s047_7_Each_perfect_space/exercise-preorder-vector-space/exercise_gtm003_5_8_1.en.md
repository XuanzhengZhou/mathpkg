A reflexive, transitive binary relation "$<$" on a set $S$ is called a \emph{pre-order} on $S$. A pre-order on a vector space $L$ over $\mathbb{R}$ is said to be \emph{compatible} (with the vector structure of $L$) if $x < y$ implies $x + z < y + z$ and $\lambda x < \lambda y$ for all $\lambda \geq 0$.
