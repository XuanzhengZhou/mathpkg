Let $B_x$ denote the band in $E$ generated by $\{x\}$. Then $x$ is a weak order unit if and only if $B_x = E$, by virtue of (1.3).

$(a) \Rightarrow (b)$: Suppose $x$ is a weak order unit, so $B_x = E$. Let $f$ be a positive linear form on $E$. Since $E$ is of minimal type, $f(x) = 0$ implies $f(B_x) = \{0\}$. But $B_x = E$, so $f = 0$. Hence for any non-zero positive linear form $f$, we must have $f(x) > 0$.

$(b) \Rightarrow (a)$: If $x$ is not a weak order unit, then $B_x \neq E$. By the order completeness and minimal type of $E$, there exists a non-zero positive linear form $f$ on $E$ vanishing on $B_x$, hence $f(x) = 0$. This contradicts (b).

$(a) \Rightarrow (c)$: Since $E$ is of minimal type, $B_x = E$ is equivalent to the assertion that the linear hull of $[0, x]$ is dense in $(E, \mathfrak{T}_0)$ (for the closure of each solid subspace $G$ in $(E, \mathfrak{T}_0)$ contains the band generated by $G$). Hence, if $x$ is a weak order unit, the order interval $[0, x]$ is total in $(E, \mathfrak{T}_0)$. For any topology $\mathfrak{T}$ on $E$ such that $(E, \mathfrak{T})$ is a locally convex vector lattice, $\mathfrak{T}$ is necessarily coarser than $\mathfrak{T}_0$ by (7.3). Therefore $[0, x]$ remains total in $(E, \mathfrak{T})$, so $x$ is a quasi-interior point of the positive cone.

$(c) \Rightarrow (a)$: Clear, since a quasi-interior point is in particular a weak order unit by definition (if $[0, x]$ is total, then $\inf(x, |y|) = 0$ implies $y = 0$).
