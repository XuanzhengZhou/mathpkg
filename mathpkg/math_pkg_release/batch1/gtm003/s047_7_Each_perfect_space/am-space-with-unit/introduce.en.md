An (AM)-space with unit is an (AM)-space that contains a distinguished positive element $u$ whose order interval $[-u,u]$ coincides with the unit ball. The element $u$ is necessarily a (strong) order unit and is unique. The canonical example is $\mathcal{C}_{\mathbb{R}}(X)$ with the constantly-one function as unit. Kakutani's representation theorem (8.5) shows that every (AM)-space with unit is isometrically isomorphic as a Banach lattice to $\mathcal{C}_{\mathbb{R}}(X)$ for a suitable compact space $X$.
