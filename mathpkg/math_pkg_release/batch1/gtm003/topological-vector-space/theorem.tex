Given a vector space $L$ over a (not necessarily commutative) non-discrete valuated field $K$ and a topology $\mathfrak{T}$ on $L$, the pair $(L, \mathfrak{T})$ is called a \textbf{topological vector space} (abbreviated t.v.s.) over $K$ if these two axioms are satisfied:

$$(LT)_1 \quad (x, y) \rightarrow x + y \text{ is continuous on } L \times L \text{ into } L.$$
$$(LT)_2 \quad (\lambda, x) \rightarrow \lambda x \quad \text{is continuous on } K \times L \text{ into } L.$$

Here $L$ is endowed with $\mathfrak{T}$, $K$ is endowed with the uniformity derived from its absolute value, and $L \times L$, $K \times L$ denote the respective topological products.
