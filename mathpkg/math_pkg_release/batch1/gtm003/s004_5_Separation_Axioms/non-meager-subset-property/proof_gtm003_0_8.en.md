The text states this without proof. Proof sketch for the first claim: Let $A \subset X$ be non-meager. If $A$ were meager in itself, then $A = \bigcup_n E_n$ where each $E_n$ is nowhere dense in $A$. A set nowhere dense in $A$ is also nowhere dense in $X$ (since the closure in $X$ of a subset of $A$ that has empty interior in $A$ also has empty interior in $X$ when $A$ has the subspace topology), so $A$ would be meager in $X$, contradiction. For the second claim: the Cantor set is nowhere dense in $\mathbb{R}$ but is a Baire space (as a compact Hausdorff space).
