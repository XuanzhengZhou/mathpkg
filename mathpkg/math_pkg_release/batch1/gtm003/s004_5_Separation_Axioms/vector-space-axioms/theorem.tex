Let $L$ be a set and $K$ a (not necessarily commutative) field. A \textbf{left vector space} structure on $L$ over $K$ consists of:

\begin{itemize}
\item[(1)] $(x + y) + z = x + (y + z)$ for all $x, y, z \in L$ (associativity of addition).
\item[(2)] $x + y = y + x$ for all $x, y \in L$ (commutativity of addition).
\item[(3)] There exists an element $0 \in L$ such that $x + 0 = x$ for all $x \in L$ (existence of zero).
\item[(4)] For each $x \in L$, there exists $z \in L$ such that $x + z = 0$ (existence of additive inverses).
\item[(5)] $\lambda(x + y) = \lambda x + \lambda y$ for all $\lambda \in K$, $x, y \in L$ (distributivity I).
\item[(6)] $(\lambda + \mu)x = \lambda x + \mu x$ for all $\lambda, \mu \in K$, $x \in L$ (distributivity II).
\item[(7)] $\lambda(\mu x) = (\lambda \mu)x$ for all $\lambda, \mu \in K$, $x \in L$ (associativity of scalar multiplication).
\item[(8)] $1x = x$ for all $x \in L$ (identity scalar).
\end{itemize}
