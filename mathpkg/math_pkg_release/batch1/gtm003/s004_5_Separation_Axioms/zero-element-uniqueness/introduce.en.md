The zero element (additive identity) of a vector space is unique. This follows from the group axioms: if $0$ and $0'$ both satisfy $x + 0 = x$ and $x + 0' = x$ for all $x$, then $0 = 0 + 0' = 0'$. The zero element of the vector space is notationally distinguished from the zero of the field $K$, though in practice the same symbol is used.
