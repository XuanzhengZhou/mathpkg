Suppose $0$ and $0'$ are two elements of $L$ both satisfying axiom (3): $x + 0 = x$ and $x + 0' = x$ for all $x \in L$. Then $0 = 0 + 0'$ (taking $x = 0$ in the second identity) and $0 + 0' = 0'$ (taking $x = 0'$ in the first identity). By commutativity of addition (axiom (2)), $0' + 0 = 0 + 0'$, so $0 = 0 + 0' = 0' + 0 = 0'$. Thus the zero element is unique.
