Let $A$ be an algebra. For any $X \subset A$, the subset
$$X^c = \{y \in A : xy = yx \text{ for all } x \in X\}$$
is called the \text{commutant} of $X$. If $A$ is a $C^*$-algebra and $X$ is self-adjoint (i.e., $X = X^*$), then $X^c$ is a $C^*$-subalgebra, containing the unit whenever $A$ is unital. The map $X \mapsto X^c$ is anti-tone with respect to inclusion $\subset$, and $X^c = X^{ccc}$. In particular, if $A = \mathcal{L}(H)$ and $X = X^*$, then $X^c$ is a $C^*$-subalgebra containing $\mathrm{id}_H$ and closed in the weak operator topology.
