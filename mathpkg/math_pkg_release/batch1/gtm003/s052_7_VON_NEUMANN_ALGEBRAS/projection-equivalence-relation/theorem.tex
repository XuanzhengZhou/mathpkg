Let $W$ be a $W^*$-algebra and $P(W)$ the set of its projections. Two projections $p, q \in P(W)$ are called \text{equivalent}, denoted $p \sim q$, if there exists a partial isometry $u \in W$ such that $p = u^*u$ and $q = uu^*$. This defines an equivalence relation on $P(W)$. We write $p \preceq q$ if $p \sim r$ for some $r \leq q$; this relation is a pre-order on $P(W)$.
