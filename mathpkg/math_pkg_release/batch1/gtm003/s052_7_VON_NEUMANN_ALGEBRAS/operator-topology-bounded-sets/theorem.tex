Denote by $\mathfrak{B}_j$ ($j = 1, 2, 3, 4$) the bounded sets for: the norm, $\sigma$-weak, strong, and weak operator topology respectively. Then $\mathfrak{B}_1 = \mathfrak{B}_2$ because $\mathcal{L}(H)$ is the Banach dual of $H \hat{\otimes} H$; $\mathfrak{B}_1 = \mathfrak{B}_3$ by the principle of uniform boundedness; $\mathfrak{B}_3 = \mathfrak{B}_4$ because these topologies have the same dual $H \otimes H$. Moreover, multiplication is $(\mathfrak{B}, \mathfrak{B})$-hypocontinuous and uniformly continuous on $B \times B$ for all bounded sets $B$.
