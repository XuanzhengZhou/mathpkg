Let $W$ be a $W^*$-algebra.
\begin{itemize}
\item A projection $p \in W$ is called \text{finite} if $p \sim q$ and $q \leq p$ implies $p = q$.
\item A projection $p \in W$ is called \text{abelian} if $pWp$ is commutative.
\item $W$ is called \text{of type I} if $W = \{0\}$ or its unit $e$ is the orthogonal sum of a family of abelian projections. (Equivalently: each non-zero central projection majorizes a non-zero abelian projection.)
\item A projection is called \text{infinite} if it is not finite.
\item A projection $p$ is called \text{purely infinite} if $p \geq q > 0$ implies that $q$ is infinite.
\item $W$ itself is called \text{purely infinite} or \text{of type III} if its unit is purely infinite.
\item $W$ is called \text{finite} if its unit $e$ is finite.
\end{itemize}
Each projection equivalent to an abelian projection is itself abelian. Abelian projections are always finite.
