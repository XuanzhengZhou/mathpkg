This lemma establishes that the adjoint operation is strongly continuous on the set of normal operators. The proof uses the identity $(x^* - y^*)(x - y) = x^*x - y^*y + y^*(y - x) + (y^* - x^*)y$ and norm estimates. This result is a key technical ingredient in Kaplansky's density theorem.
