The preimage (or inverse image) $f^{-1}(B)$ of a subset $B \subset Y$ under a mapping $f: X \rightarrow Y$ is the set of all elements of $X$ that map into $B$. This defines an associated map $f^{-1}: \mathfrak{P}(Y) \rightarrow \mathfrak{P}(X)$ from the power set of the codomain to the power set of the domain. For singletons $\{b\}$, one writes $f^{-1}(b)$ as an abbreviation.
