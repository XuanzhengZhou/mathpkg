Let $\{\mathfrak{T}_\alpha : \alpha \in A\}$ be a family of topologies on a set $X$. There exists a finest topology $\mathfrak{T}$ on $X$ which is coarser than each $\mathfrak{T}_\alpha$ ($\alpha \in A$); a set is $\mathfrak{T}$-open if and only if it is $\mathfrak{T}_\alpha$-open for every $\alpha \in A$.
