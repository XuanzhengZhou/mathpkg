\text{A continuous semi-norm } p \text{ on a l.c.s. } E \text{ is called prenuclear if there exists a positive Radon measure } \mu \text{ on an equicontinuous subset } A \subset E'_\sigma \text{ such that } p(x) \leq \int_A |\langle x, x' \rangle| d\mu(x') \text{ for all } x \in E. \text{ Equivalently, a subset } B \subset E' \text{ is prenuclear if } x \mapsto \sup\{|\langle x, b \rangle| : b \in B\} \text{ is a prenuclear semi-norm on } E. \text{ A prenuclear semi-norm is necessarily continuous, and a prenuclear subset of } E' \text{ is necessarily equicontinuous.}
