\text{Let } H_1 \text{ and } H_2 \text{ be Hilbert spaces over } \mathbb{K}. \text{ A linear map } u \in \mathcal{L}(H_1, H_2) \text{ is called a Hilbert-Schmidt transformation if there exists an orthonormal basis } \{x_\alpha : \alpha \in A\} \text{ of } H_1 \text{ such that } \sum_{\alpha} \|u(x_\alpha)\|^2 < +\infty. \text{ Such a map is compact, and } u(x_\alpha) = 0 \text{ save for a countable number of indices } \alpha \in A. \text{ The value of the sum is independent of the choice of the orthonormal basis and equals } \sum_{\beta} \|u^*(y_\beta)\|^2 \text{ for any orthonormal basis } \{y_\beta : \beta \in B\} \text{ of } H_2.
