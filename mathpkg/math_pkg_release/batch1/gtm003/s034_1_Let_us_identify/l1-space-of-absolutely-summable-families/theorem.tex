\text{Let } A \text{ be a fixed non-empty index set and let } E \text{ be a given l.c.s. The set of all absolutely summable families } \mathbf{x} = \{x_\alpha : \alpha \in \mathbf{A}\} \text{ in } E \text{ can be identified with a subspace } S_a \subset E^\mathbf{A}. \text{ Denote by } \mathfrak{U} \text{ any fixed base of convex circled 0-neighborhoods in } E, \text{ and by } r_U \text{ the gauge of } U \in \mathfrak{U}. \text{ The mapping } \mathbf{x} \mapsto p_U(\mathbf{x}) = \sum_\alpha r_U(x_\alpha) \text{ is a semi-norm on } S_a, \text{ and the family } \{p_U : U \in \mathfrak{U}\} \text{ generates a locally convex topology under which } S_a \text{ is denoted by } l^1[\mathbf{A}, E].

\text{Similarly, the set of all summable families can be identified with a subspace } S \subset E^\mathbf{A}, \text{ and for each } U \in \mathfrak{U} \text{ the mapping } \mathbf{x} \mapsto q_U(\mathbf{x}) = \sup \{ \sum_\alpha |\langle x_\alpha, x' \rangle| : x' \in U^\circ \} \text{ is a semi-norm on } S, \text{ generating a locally convex topology.}
