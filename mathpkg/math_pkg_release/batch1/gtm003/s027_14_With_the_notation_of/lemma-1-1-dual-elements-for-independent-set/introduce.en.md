Lemma 1.1 states that given a duality $\langle F, G \rangle$ and any finite linearly independent set $\{y_1, \ldots, y_n\}$ in $G$, one can find elements $x_1, \ldots, x_n$ in $F$ that form a biorthogonal system: $\langle x_i, y_j \rangle = \delta_{ij}$. This algebraic fact is fundamental for studying the weak topology $\sigma(F, G)$ and for characterizing weakly continuous linear forms. The statement holds symmetrically with the roles of $F$ and $G$ interchanged.
