A filter $\mathfrak{F}$ in a t.v.s. $L$ is a Cauchy filter if for each 0-neighborhood $V$ in $L$, there exists $F \in \mathfrak{F}$ such that $F - F \subset V$. A sequence $\{x_n : n \in \mathbb{N}\}$ is a Cauchy sequence if for each 0-neighborhood $V$ in $L$ there exists $n_0 \in \mathbb{N}$ such that $x_m - x_n \in V$ whenever $m \geq n_0$ and $n \geq n_0$.
