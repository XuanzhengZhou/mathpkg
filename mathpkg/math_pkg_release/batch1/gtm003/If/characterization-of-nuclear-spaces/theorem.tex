For a locally convex space $E$, the following are equivalent:
\begin{enumerate}
\item[(a)] $E$ is nuclear.
\item[(b)] Every continuous linear map of $E$ into any Banach space $F$ is nuclear.
\item[(c)] Each convex, circled $0$-neighborhood $U$ in $E$ contains another, $V$, such that the canonical map $\tilde{E}_V \to \tilde{E}_U$ is nuclear.
\end{enumerate}
