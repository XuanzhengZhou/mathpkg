Let $L$ be an Archimedean ordered vector space over $\mathbb{R}$ possessing an order unit $e$. Then $(L, \mathfrak{T}_O)$ is an ordered topological vector space which is normable, and $\mathfrak{T}_O$ is the finest locally convex topology on $L$ for which the positive cone $C$ is normal. Moreover, the following are equivalent:
\begin{enumerate}
\item[(a)] $(L, \mathfrak{T}_O)$ is complete.
\item[(b)] Every positive sequence of type $\ell^1$ in $L$ is order summable.
\end{enumerate}

Here a positive sequence $\{x_n\}$ is of type $\ell^1$ if there exists $a \geq 0$ in $L$ and $(\lambda_n) \in \ell^1$ such that $0 \leq x_n \leq \lambda_n a$ for all $n$.
