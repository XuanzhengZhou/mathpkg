Let $L$ and $M$ be topological vector spaces and $u: L \to M$ a linear map with null space $N = u^{-1}(0)$. The map $u_0: L/N \to u(L)$ defined by $u_0(\hat{x}) = u(x)$ for each equivalence class $\hat{x} \in L/N$ is called the **bijective map associated with $u$**. It is an algebraic isomorphism between the quotient space $L/N$ and the range $u(L)$.
