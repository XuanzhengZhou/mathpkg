This lemma provides the key technical step for Banach's homomorphism theorem (the open mapping theorem for topological vector spaces). If a continuous linear map from a complete metric t.v.s. sends a closed ball around zero to a set whose closure is a neighborhood of zero, then the map is automatically open onto its image. The proof uses completeness to construct convergent series, showing that the image of a slightly larger ball actually contains a neighborhood of zero.
