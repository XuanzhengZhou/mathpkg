Let $L$ and $M$ be topological vector spaces over $K$ and let $u: L \to M$ be a linear map. Set $N = u^{-1}(0)$, the null space of $u$. Then $u$ can be canonically decomposed as

$$L \xrightarrow{\phi} L/N \xrightarrow{u_0} u(L) \xrightarrow{\psi} M,$$

where:
\begin{itemize}
\item $\phi: L \to L/N$ is the canonical (quotient) map,
\item $\psi: u(L) \hookrightarrow M$ is the canonical imbedding,
\item $u_0$ is the algebraic isomorphism which maps each equivalence class $\hat{x} \in L/N$ onto the common image $u(x)$ ($x \in \hat{x}$) of this class under $u$.
\end{itemize}

Thus $u = \psi \circ u_0 \circ \phi$, and $u_0$ is called the **bijective map associated with $u$**.
