\text{Given a vector space } L \text{ over a (not necessarily commutative) non-discrete valuated field } K \text{ and a topology } \mathfrak{T} \text{ on } L, \text{ the pair } (L, \mathfrak{T}) \text{ is called a topological vector space (abbreviated t.v.s.) over } K \text{ if these two axioms are satisfied:}

$$(LT)_1 \quad (x, y) \mapsto x + y \text{ is continuous on } L \times L \text{ into } L.$$
$$(LT)_2 \quad (\lambda, x) \mapsto \lambda x \text{ is continuous on } K \times L \text{ into } L.$$

\text{Here } L \text{ is endowed with } \mathfrak{T}, \; K \text{ is endowed with the uniformity derived from its absolute value, and } L \times L, \; K \times L \text{ denote the respective topological products.}
