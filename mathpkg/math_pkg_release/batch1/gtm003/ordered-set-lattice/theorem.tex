Let $(X, \leq)$ be an ordered set. A subset $A$ is majorized if there exists $a_0 \in X$ such that $a \leq a_0$ for all $a \in A$; $a_0$ is a majorant. Dually for minorized and minorant. $\sup A$ is the least majorant; $\inf A$ is the greatest minorant. $(X, \leq)$ is a lattice if $\sup(x, y)$ and $\inf(x, y)$ exist for all $x, y$. It is totally ordered if $x \leq y$ or $y \leq x$ for all $x, y$.
