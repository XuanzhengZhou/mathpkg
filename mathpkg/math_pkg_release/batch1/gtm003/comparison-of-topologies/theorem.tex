If $\mathfrak{T}_1, \mathfrak{T}_2$ are topologies on a set $X$, $\mathfrak{T}_2$ is finer than $\mathfrak{T}_1$ (or $\mathfrak{T}_1$ is coarser than $\mathfrak{T}_2$) if every $\mathfrak{T}_1$-open set is $\mathfrak{T}_2$-open. For any family $\{\mathfrak{T}_\alpha: \alpha \in A\}$ of topologies on $X$, there exists a finest topology on $X$ which is coarser than each $\mathfrak{T}_\alpha$.
