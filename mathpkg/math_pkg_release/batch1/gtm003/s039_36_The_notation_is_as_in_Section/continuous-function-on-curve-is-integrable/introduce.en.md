In a locally convex space $E$ where the closed convex hull of every compact set remains compact (a property satisfied by all quasi-complete spaces), every continuous $E$-valued function on a rectifiable curve is automatically Riemann integrable. This result parallels the classical scalar case and relies on the compactness of the closed, convex, circled hull of the image $f(\delta)$ to verify the $\sigma(E',E)$-continuity condition in the definition of the vector-valued integral.
