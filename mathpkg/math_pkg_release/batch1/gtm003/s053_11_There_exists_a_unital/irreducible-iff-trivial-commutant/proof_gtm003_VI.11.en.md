\textit{Proof sketch (Exercise).} If $(\pi, H)$ is not irreducible, there exists a nontrivial closed invariant subspace, whose orthogonal projection $p$ belongs to the commutant $\pi(A)^c$ and satisfies $0 \neq p \neq \mathrm{id}_H$. Conversely, if $\pi(A)^c$ contains a projection $p$ with $0 \neq p \neq \mathrm{id}_H$, then $p(H)$ is a nontrivial closed invariant subspace, so $(\pi, H)$ is not irreducible. The full commutant is trivial if and only if no such projection exists.
