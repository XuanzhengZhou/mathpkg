Let $(L, \mathfrak{T})$ be a topological vector space and suppose $L$ is the algebraic direct sum of $n$ subspaces $M_i$ ($i = 1, \ldots, n$). Then $L$ is the topological direct sum $L = M_1 \oplus \cdots \oplus M_n$ (i.e., the canonical map $\psi : \prod_{i=1}^n M_i \to L$ given by $(x_1, \ldots, x_n) \mapsto \sum_{i=1}^n x_i$ is a topological isomorphism) if and only if the associated projections $u_i : L \to M_i$ ($i = 1, \ldots, n$) are continuous.

Moreover, if $n-1$ of these projections are continuous, then the remaining one is automatically continuous, since the identity map $e = u_1 + \cdots + u_n$ is continuous on $L$.
