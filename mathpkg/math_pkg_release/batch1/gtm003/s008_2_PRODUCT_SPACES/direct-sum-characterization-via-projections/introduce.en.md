When a topological vector space decomposes algebraically as a direct sum of subspaces, it is a topological direct sum precisely when all the associated coordinate projections are continuous. This theorem provides a clean criterion: the algebraic decomposition lifts to a topological isomorphism with the product space if and only if each projection is a continuous linear map. An interesting consequence is that continuity of all but one projection already forces continuity of the last one.
