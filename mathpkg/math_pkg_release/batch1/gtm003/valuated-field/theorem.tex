A valuated field $K$ is a field equipped with an absolute value $|\cdot|: K \to \mathbb{R}_{\geq 0}$ satisfying: (i) $|x| = 0 \iff x = 0$; (ii) $|xy| = |x||y|$; (iii) $|x + y| \leq |x| + |y|$. $K$ is non-discrete if there exists $x \in K$ with $0 < |x| < 1$. $K$ is Archimedean if $|n| \to \infty$ as $n \in \mathbb{N}$.
