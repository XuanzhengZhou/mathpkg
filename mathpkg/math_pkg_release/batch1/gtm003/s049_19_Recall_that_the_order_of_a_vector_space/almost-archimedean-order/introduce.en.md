An ordered vector space is almost Archimedean if no non-zero element is bounded in absolute value by arbitrarily small positive multiples of a fixed element. This is a weaker condition than the Archimedean property. Exercise 19 in Schaefer's book explores the relationship between these two notions, including the fact that for almost Archimedean vector lattices whose order dual separates points, the order is necessarily Archimedean.
