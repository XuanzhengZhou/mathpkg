Factor $P(\lambda) - \mu = c \prod_{i=1}^n (\lambda - \lambda_i(\mu))$ over $\mathbf{C}$. Then $P(x) - \mu e = c \prod_{i=1}^n (x - \lambda_i(\mu)e)$. The element $P(x) - \mu e$ is invertible in $A$ if and only if each factor $x - \lambda_i(\mu)e$ is invertible, which occurs precisely when $\mu \notin P(\sigma(x))$. Hence $\sigma(P(x)) = P(\sigma(x))$.
