If $A$ is a locally convex algebra, an element $a \in A$ is called a \emph{spectral element} if $a$ is contained in a spectral subalgebra of $A$. If $a$ is a spectral element of $A$ and $\mu$ is a spectral measure on $X$ such that $a = \mu(f)$, then $f(X)$ is the spectrum of $a$; that is, $f(X) = \sigma(a)$.
