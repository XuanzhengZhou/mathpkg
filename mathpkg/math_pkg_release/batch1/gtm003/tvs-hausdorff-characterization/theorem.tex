A t.v.s. $L$ is a Hausdorff (separated) topological space if and only if $L$ is a separated uniform space. Equivalently, $\bigcap \{U : U \in \mathfrak{U}\} = \{0\}$, where $\mathfrak{U}$ is any neighborhood base of 0 in $L$. An equivalent condition is that for each non-zero $x \in L$, there exists a 0-neighborhood $U$ such that $x \notin U$.
