Let $L$ be a topological vector lattice with positive cone $C = \{x \in L : x \geq 0\}$.

(1) $C$ is closed: The lattice operations are continuous in a topological vector lattice. In particular, the map $x \mapsto x^- = \sup(-x, 0)$ is continuous. Since $C = \{x : x^- = 0\} = (x \mapsto x^-)^{-1}(\{0\})$ and $\{0\}$ is closed, $C$ is closed.

(2) $C$ is normal: A topological vector lattice possesses a base $\mathfrak{U}$ of solid 0-neighborhoods. For any $U \in \mathfrak{U}$, the set $(U + C) \cap (U - C)$ is contained in $U$ because $U$ is solid. This implies that for any 0-neighborhood $U$, there exists another 0-neighborhood $V$ (take $V = U$ itself, since $U$ is solid) such that $(V + C) \cap (V - C) \subset U$. Hence the neighborhood base satisfies the normality condition, so $C$ is normal.

(3) $C$ is a strict $\mathfrak{B}$-cone: Since $L$ is a locally convex vector lattice, its topology is generated by a family of lattice semi-norms. Each lattice semi-norm $p$ satisfies $p(x) \leq p(y)$ whenever $|x| \leq |y|$. For $x \in C$, $|x| = x$, and the order interval $[0, x]$ is bounded in $L$ because for any continuous lattice semi-norm $p$ and any $y \in [0, x]$, we have $0 \leq y \leq x$, hence $|y| = y \leq x = |x|$, so $p(y) \leq p(x)$. Thus each order interval $[0, x]$ with $x \in C$ is bounded. By definition, this means $C$ is a strict $\mathfrak{B}$-cone (where $\mathfrak{B}$ is the family of all bounded subsets of $L$).
