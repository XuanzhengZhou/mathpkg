By (2.6) Corollary, there exists a positive Radon measure $\mu_0$ with $r\mu_0 = u'(\mu_0)$. Since $\{f: \mu_0(|f|) = 0\}$ is a solid subspace invariant under $u$, $\mu_0$ is strictly positive. This implies $\|u\| = r$ iff $u(1) = r1$. For (ii), define $g = f/|f|$ for an eigenfunction $f$ for $r\alpha$; then $v = \alpha^{-1} w^{-1} u w$ where $w(h) = gh$ is a positive operator. The group action follows. (iii) follows from the fact that any eigenfunction for $r\alpha$ is nowhere vanishing, and the multiplicity is 1 by considering $f_0 = |f|$. The remaining parts follow from spectral mapping properties and the structure of $C(X)$.
