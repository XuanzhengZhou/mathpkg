Let $E$ be a Banach lattice whose absolute value extends to the complexification $E_1$, and let $u$ be a positive endomorphism of $E$ with spectral radius $r$, for which there exists a strictly positive linear form $x'_0$ satisfying $u'(x'_0) \leq r x'_0$. If the peripheral point spectrum of $u$ contains $r\alpha$, $|\alpha| = 1$, then it contains $rH$, where $H$ is the cyclic group generated by $\alpha$.
