For $0 \leq i \leq n-1$, define the simplicial subcomplexes
$$\Sigma^{2i-1} = \Sigma_1 * \Sigma_2 * \dots * \Sigma_i \subset \mathbb{R}^{2i}$$
where each $\Sigma_j \cong S^1$ is triangulated as a $p$-gon. The full sphere $\Sigma^{2n-1} = \Sigma_1 * \Sigma_2 * \dots * \Sigma_n$ and the group $G = \langle R_1 * \dots * R_n \rangle \cong \mathbb{Z}_p$ acts simplicially.
