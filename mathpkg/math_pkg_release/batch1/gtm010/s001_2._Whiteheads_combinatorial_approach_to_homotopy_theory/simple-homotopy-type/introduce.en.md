Two finite simplicial complexes have the same simple-homotopy type if they can be connected by a finite zigzag of simplicial expansions and collapses. Since each elementary collapse determines a strong deformation retraction, complexes of the same simple-homotopy type necessarily have the same homotopy type. Whitehead asked whether the converse holds.
