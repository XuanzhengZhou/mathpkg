Assume that the pair $(K, L)$ is in simplified form and has matrix $(a_{ij})$ with respect to some set of characteristic maps. Suppose that the matrix $(a_{ij})$ can be transformed to the matrix $(b_{ij})$ by one of the following operations:
I. $R_i \rightarrow \pm \alpha R_i$ ($\alpha \in \pi_1 \subset \mathbb{Z}\pi_1$) — multiply row on left by trivial unit
II. $R_k \rightarrow R_k + \rho R_i$ ($\rho \in \mathbb{Z}\pi_1$) — add left multiple of one row to another
III. $(a_{ij}) \rightarrow \begin{pmatrix} a_{ij} & 0 \\ 0 & I_q \end{pmatrix}$ — add corner identity matrix
Then there is a simplified pair $(M, L)$ such that $K \wedge M$ rel $L$ and a set of characteristic maps with respect to which $(M, L)$ has the matrix $(b_{ij})$.
