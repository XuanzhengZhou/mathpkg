If two CW complexes are obtained from $L$ by attaching $n$-cells with homotopic attaching maps, then they have the same simple-homotopy type relative to $L$. This fundamental lemma shows that simple-homotopy type depends only on the homotopy class of the attaching map, not its specific representative. A key application: the dunce hat has the simple-homotopy type of a point.
