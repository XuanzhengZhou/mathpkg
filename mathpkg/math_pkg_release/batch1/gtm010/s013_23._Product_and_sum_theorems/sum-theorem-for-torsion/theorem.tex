Suppose $K = K_1 \cup K_2$, $K_0 = K_1 \cap K_2$, $L = L_1 \cup L_2$, $L_0 = L_1 \cap L_2$ and that $f: K \rightarrow L$ is a map which restricts to homotopy equivalences $f_\alpha: K_\alpha \rightarrow L_\alpha$ ($\alpha = 0, 1, 2$). Let $j_\alpha: L_\alpha \rightarrow L$ and $i_\alpha: K_\alpha \rightarrow K$ be the inclusions. Then $f$ is a homotopy equivalence and
(a) $\tau(f) = j_{1*}\tau(f_1) + j_{2*}\tau(f_2) - j_{0*}\tau(f_0)$
(b) If $f$ is an inclusion map,
$$\tau(L, K) = i_{1*}\tau(L_1, K_1) + i_{2*}\tau(L_2, K_2) - i_{0*}\tau(L_0, K_0).$$
