Two continuous maps $f, g: X \to Y$ are said to be homotopic, denoted $f \simeq g$, if one can be continuously deformed into the other. The deformation is witnessed by a homotopy $F: X \times I \to Y$, which interpolates between $f$ (at $t=0$) and $g$ (at $t=1$). Homotopy defines an equivalence relation on the set of maps from $X$ to $Y$, and the resulting equivalence classes are called homotopy classes. This notion is fundamental to algebraic topology, as it provides the equivalence relation with respect to which topological invariants such as homotopy groups are defined.
