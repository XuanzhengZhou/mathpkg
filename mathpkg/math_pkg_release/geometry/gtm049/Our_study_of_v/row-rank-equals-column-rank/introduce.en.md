This fundamental theorem of linear algebra states that for any matrix (not necessarily square), the maximal number of linearly independent rows equals the maximal number of linearly independent columns. This common value is simply called the rank of the matrix. Two proofs are given: one using the symmetry of the canonical form $PAQ = [I_r \; 0; 0 \; 0]$, and another independent of the theory of linear mappings that uses linear dependence relations among rows to bound the column rank.
