A rotation is an orthogonal transformation with determinant $$+1$$. In dimension 2, rotations correspond to the familiar rotation matrices; in dimension 3, they represent proper rotations in space (as opposed to reflections). The set of all rotations forms the rotation group, which is the special orthogonal group $$\operatorname{SO}(V, \sigma)$$.
