Theorem 2 classifies quadrics in euclidean geometry using simultaneous orthogonalization (Theorem 1). Given a quadric $R$ in a euclidean space, one finds a euclidean frame of reference that diagonalizes both the metric form $\sigma$ and the quadric form $\tau$, yielding a canonical coordinate expression for the quadric.
