Let $A_0, A_1, A_2, A_3$ be distinct collinear points in a projective geometry $\mathcal{P}(V)$ over an arbitrary field $F$. Choose homogeneous vectors such that $[a_0] = A_0$, $[a_1] = A_1$, $[a_0 + a_1] = A_2$ and $[x a_0 + a_1] = A_3$. Then $x$ is uniquely determined and is called the cross-ratio $\operatorname{cr}(A_0, A_1; A_2, A_3)$. For any collineation $\pi$, $\operatorname{cr}(A_0\pi, A_1\pi; A_2\pi, A_3\pi) = \operatorname{cr}(A_0, A_1; A_2, A_3)$.
