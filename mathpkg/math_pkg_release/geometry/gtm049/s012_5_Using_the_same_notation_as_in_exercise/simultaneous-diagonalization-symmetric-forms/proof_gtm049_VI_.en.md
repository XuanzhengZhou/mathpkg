Let $$f$$ be the $$\sigma$$-symmetric linear mapping determined by $$\sigma, \tau$$ according to equation (1) on p. 135:
$$\tau(a, b) = \sigma(af, b)$$
for all $$a, b \in V$$. Then by Proposition 4, $$(V, \sigma)$$ has a Cartesian basis $$(a_1, \ldots, a_n)$$ such that
$$a_i f = x_i a_i \quad \text{for} \quad i = 1, \ldots, n.$$
Consequently,
$$\tau(a_i, a_j) = \sigma(a_i f, a_j) = x_i \sigma(a_i, a_j) = x_i \delta_{ij},$$
so $$a_1, \ldots, a_n$$ are mutually orthogonal with respect to $$\tau$$ as required.
