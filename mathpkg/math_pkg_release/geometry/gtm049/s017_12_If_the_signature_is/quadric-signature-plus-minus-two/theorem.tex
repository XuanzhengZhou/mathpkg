If the signature of a non-degenerate quadric in $\mathbb{P}^3$ is $\pm 2$, then the equation of the quadric in a suitable coordinate system is
$$X_0^2 + X_1^2 + X_2^2 - X_3^2 = 0.$$
Moreover, if there were a generator, the plane $X_3 = 0$ would have to meet it in a point (or points) satisfying $X_0^2 + X_1^2 + X_2^2 = 0$, which has no non-trivial real solutions.
