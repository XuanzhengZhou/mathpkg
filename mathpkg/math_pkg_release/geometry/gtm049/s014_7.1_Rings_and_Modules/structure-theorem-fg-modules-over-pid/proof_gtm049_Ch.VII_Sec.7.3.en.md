We note that if $b_1, \ldots, b_l$ generate $M$, then so do
$$b_1, \ldots, b_{l-1}, b_l + q_j b_j,$$
where $q_j \in F[X]$ and $1 \leq j \leq l-1$. Let $k$ be the smallest number of elements needed to generate $M$.

(a) Suppose first that we can find a set $\{b_1, \ldots, b_k\}$ of generators among which no non-trivial relation holds: i.e., such that
$$m_1 b_1 + \cdots + m_k b_k = 0$$
implies $m_1 = \cdots = m_k = 0$. Then
$$M = [b_1] \oplus \cdots \oplus [b_k]$$
and $\mathfrak{a}(b_i) = 0$ for each $i$. The first part of the theorem is now established.

(b) Assume next that every system of $k$ generators gives at least one non-trivial relation. Then there is one such system $\{b_1, \ldots, b_k\}$ for which a relation
$$m_1 b_1 + \cdots + m_k b_k = 0$$
holds with a non-zero coefficient $m_i$ of minimum degree. We may suppose that $i = k$.

We prove first that $m_j$ is a multiple of $m_k$ for $j = 1, \ldots, k-1$. For if $m_j = q_j m_k + r_j$ where $\deg r_j < \deg m_k$, then
$$m_1 b_1 + \cdots + r_j b_j + \cdots + m_k(b_k + q_j b_j) = 0,$$
from which we deduce that $r_j = 0$.

We contend also that if $n_1 b_1 + \cdots + n_k b_k = 0$ is any relation, then $m_k$ divides $n_k$.

Now define
$$a_k = q_1 b_1 + \cdots + q_{k-1} b_{k-1} + b_k.$$
Then (i) shows that $m_k a_k = 0$.

If $M'$ denotes the submodule of $M$ generated by $b_1, \ldots, b_{k-1}$, then
$$M = M' \oplus [a_k].$$
This follows since the relation $n_1 b_1 + \cdots + n_{k-1} b_{k-1} + n_k a_k = 0$ implies that $n_k$ is a multiple of $m_k$ (by (ii)) and so $n_k a_k = 0$ (by (iii)).

The first part of our theorem may now be proved by induction on $k$. We can express $M'$ as a direct sum
$$[a_1] \oplus \cdots \oplus [a_{k-1}]$$
where $\mathfrak{a}(a_1) \subset \cdots \subset \mathfrak{a}(a_{k-1})$.

It only remains to prove that $\mathfrak{a}(a_{k-1}) \subset \mathfrak{a}(a_k)$. We proved under (i) that if we have a relation between $k$ generators involving $m_k$ as a coefficient, then all coefficients are divisible by $m_k$. If now $x \in \mathfrak{a}(a_{k-1})$, then $x a_{k-1} = 0$ and so $x a_{k-1} + m_k a_k = 0$. Hence $m_k$ divides $x$.

In order to prove the uniqueness part of the theorem we suppose that $r$ of the ideals $\mathfrak{a}(a_i)$ are zero ($0 \leq r \leq k$) and define
$$M_1 = [a_1] \oplus \cdots \oplus [a_r], \quad M_2 = [a_{r+1}] \oplus \cdots \oplus [a_k].$$
We assert that $M_1$ is precisely the set of all elements in $M$ annihilated by $0$, i.e., the torsion-free part, while $M_2$ is the torsion submodule.

For the torsion-free case, assume $M = [a_1] \oplus \cdots \oplus [a_k]$ where $\mathfrak{a}(a_i) = 0$ for each $i$. Distinct elements $b_1, \ldots, b_s$ of $M$ are called linearly independent over $F[X]$ if every relation $m_1 b_1 + \cdots + m_s b_s = 0$ implies each $m_i = 0$. The mapping $p_1 a_1 + \cdots + p_k a_k \mapsto (p_1, \ldots, p_k)$ is an $F[X]$-isomorphism of $M$ onto $F[X]^k$. Let $F(X)$ denote the field of quotients. Then linear independence over $F[X]$ is equivalent to linear independence over $F(X)$. It follows that $k$ is the maximum number of linearly independent elements over $F[X]$ and as such is an invariant of $M$.

Thus the integers $r$ and $k$ and the ideals $\mathfrak{a}(a_i)$ are uniquely determined by $M$.
