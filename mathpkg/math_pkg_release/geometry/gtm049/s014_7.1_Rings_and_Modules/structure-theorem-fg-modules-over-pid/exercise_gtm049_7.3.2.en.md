Given two non-zero elements $y, z$ in a principal ideal domain $R$, we say that $y$ divides $z$ if there exists in $R$ an element $x$ such that $z = xy$, i.e., if $Rz \subset Ry$. If $y$ divides $1$ then $y$ is said to be invertible. An element $z$ is called irreducible if it is not invertible and if $z = xy$ implies that $x$ or $y$ is invertible. Prove that $Rz = Ry$ if, and only if, $z$ and $y$ are related by an equation $z = xy$ where $x$ is invertible. Using this and exercise 1, show that any non-zero non-invertible element of $R$ can be expressed as a product of a finite number of irreducible elements.
