A basis of a finitely generated torsion-free $F[X]$-module is a set of elements that simultaneously forms a minimal generating set and a maximal linearly independent set. Unlike vector spaces, not every maximal linearly independent set is a minimal generating set for a torsion-free module. However, a basis guarantees that the module decomposes as a direct sum of cyclic submodules, each isomorphic to the base ring $F[X]$. Bases are essential for representing homomorphisms by matrices with entries in $F[X]$, which leads to the Smith normal form and the solution of the matrix equivalence problem over polynomial rings.
