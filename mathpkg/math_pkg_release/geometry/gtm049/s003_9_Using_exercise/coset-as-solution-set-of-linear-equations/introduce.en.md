This proposition provides the converse to Theorem 3: while Theorem 3 states that the solution set of a consistent system of linear equations is a coset, Proposition 2 asserts that every coset in $F^n$ arises this way. The proof uses the orthogonal complement $M^\perp$ of the subspace $M$ directing the coset. This establishes that affine subspaces of $F^n$ are exactly the solution sets of (possibly inhomogeneous) linear systems.
