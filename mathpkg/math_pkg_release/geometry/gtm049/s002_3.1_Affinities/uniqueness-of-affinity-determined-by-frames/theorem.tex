\text{Let } A = \mathcal{A}(a + M), \; A' = \mathcal{A}(a' + M') \text{ be affine geometries of the same dimension } n \text{ over the same field, where } a + M, a' + M' \text{ are cosets in vector spaces } V, V'. \text{ Let } (P_1, \ldots, P_n, Q) \text{ be a frame of reference for } A \text{ and } (P_1', \ldots, P_n', Q') \text{ a frame of reference for } A'. \text{ Then there exists one and only one affinity } \alpha \colon A \to A' \text{ such that } Q\alpha = Q' \text{ and } P_i\alpha = P_i' \text{ for } i = 1, \ldots, n.
