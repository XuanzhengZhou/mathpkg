\text{Every frame of reference for } \mathcal{P}(V) \text{ is determined by at least one ordered basis of } V. \text{ Two ordered bases } (a_0, \ldots, a_n), (b_0, \ldots, b_n) \text{ of } V \text{ determine the same frame of reference if, and only if, there exists a non-zero scalar } z \text{ such that } b_i = z a_i \text{ for } i = 0, \ldots, n.
