\text{Let } P \text{ and } P' \text{ be projective geometries of the same dimension } n \text{ over the same field } F. \text{ If } (A_0, \ldots, A_n, U) \text{ is a frame of reference for } P \text{ and } (A_0', \ldots, A_n', U') \text{ is a frame of reference for } P', \text{ then there exists one and only one projectivity } \pi \colon P \to P' \text{ such that } A_i\pi = A_i' \text{ for } i = 0, \ldots, n \text{ and } U\pi = U'.
