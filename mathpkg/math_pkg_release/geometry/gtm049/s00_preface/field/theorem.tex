A set $F$ is called a field if $F$ is a set together with rules of addition and multiplication which associate to any two elements $x, y$ in $F$ a sum $x+y$ and a product $xy$, both in $F$, and the following axioms are satisfied:

\begin{itemize}
\item[F.1.] $F$ is a commutative group with respect to addition;
\item[F.2.] the set $F^*$, obtained from $F$ by omitting the zero, is a commutative group with respect to multiplication;
\item[F.3.] $x(y+z) = xy+xz$ and $(y+z)x = yx+zx$ for all $x, y, z$ in $F$.
\end{itemize}
