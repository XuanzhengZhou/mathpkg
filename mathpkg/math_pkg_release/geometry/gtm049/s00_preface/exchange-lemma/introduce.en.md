The Exchange Lemma (also known as the Steinitz Exchange Lemma) states that a spanning set of size $r$ cannot be replaced by a linearly independent set of size $s > r$ while maintaining the spanning property. It is the key technical lemma used to prove that all bases of a finite dimensional vector space have the same cardinality.
