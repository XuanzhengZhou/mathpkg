The angle between two non-zero vectors in a Euclidean space is defined via the cosine function using the ratio $\sigma(a, b) / (\|a\| \|b\|)$. Schwarz's inequality guarantees that this ratio lies in $[-1, 1]$, so the definition is well-posed. This generalizes the familiar notion of angle from $\mathbf{R}^n$ to any abstract Euclidean space.
