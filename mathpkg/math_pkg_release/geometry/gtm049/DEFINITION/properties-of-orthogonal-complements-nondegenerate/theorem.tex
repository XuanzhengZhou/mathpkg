Let $\sigma$ be a non-degenerate bilinear form on a vector space $V$. Then the mappings $\perp(\sigma)$ and $\top(\sigma)$ are bijections of the set of all subspaces of $V$ onto itself and satisfy:

(1) $\dim M^{\perp} = \dim V - \dim M = \dim M^{\top}$;

(2) $M^{\perp \top} = M = M^{\top \perp}$;

(3) $M \subset N$ if and only if $M^{\perp} \supset N^{\perp}$, and $M \subset N$ if and only if $M^{\top} \supset N^{\top}$;

(4) $(M + N)^{\perp} = M^{\perp} \cap N^{\perp}$, $(M + N)^{\top} = M^{\top} \cap N^{\top}$;

(5) $(M \cap N)^{\perp} = M^{\perp} + N^{\perp}$, $(M \cap N)^{\top} = M^{\top} + N^{\top}$.
