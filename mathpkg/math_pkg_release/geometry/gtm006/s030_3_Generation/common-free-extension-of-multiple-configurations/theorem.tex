If $\mathcal{A}_1, \mathcal{A}_2, \ldots, \mathcal{A}_n$ are configurations in the closed configuration $\mathcal{P}$, such that $\mathcal{P} = \mathcal{F}(\mathcal{A}_i)$ for each $i = 1, 2, \ldots, n$, and each $\mathcal{A}_i$ is contained in a free extension of any $\mathcal{A}_j$, then there is a unique configuration $\mathcal{B}$ which is a free extension of all the $\mathcal{A}_i$ such that any configuration which is a free extension of all the $\mathcal{A}_i$ must contain $\mathcal{B}$.
