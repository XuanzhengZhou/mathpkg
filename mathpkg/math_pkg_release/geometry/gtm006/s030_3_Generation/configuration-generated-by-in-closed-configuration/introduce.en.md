The configuration generated by $\mathcal{A}$ in a closed configuration $\mathcal{P}$, denoted $\langle \mathcal{A} \rangle_{\mathcal{P}}$, is defined as the intersection of all closed sub-configurations of $\mathcal{P}$ that contain $\mathcal{A}$. This construction yields the smallest closed configuration containing $\mathcal{A}$, analogous to the notion of a generated substructure in universal algebra. It provides a closure operator on the lattice of sub-configurations of $\mathcal{P}$.
