If $\mathcal{A} < \mathcal{P}$ where $\mathcal{A}$ is a configuration and $\mathcal{P}$ is a closed configuration, then the configuration generated by $\mathcal{A}$ in $\mathcal{P}$, written $\langle \mathcal{A} \rangle_{\mathcal{P}}$, is the intersection of all the closed sub-configurations of $\mathcal{P}$ which contain $\mathcal{A}$. It is also called the closure of $\mathcal{A}$ in $\mathcal{P}$.
