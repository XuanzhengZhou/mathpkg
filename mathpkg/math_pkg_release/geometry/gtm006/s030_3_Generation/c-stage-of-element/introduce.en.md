The $\mathcal{C}$-stage of an element $X$ in the closure $\langle \mathcal{C} angle_{\mathcal{P}}$ measures how many full extension steps are needed to generate $X$ from the base configuration $\mathcal{C}$. Elements of $\mathcal{C}$ itself have stage $0$, while elements appearing first in the full $n$-step extension have stage $n$. This provides a natural grading on the generated configuration and is used in proofs involving induction on the stage.
