Exercise 13.2 asks to prove a sharp bound for abelian collineation groups: if $\Pi$ is abelian and its order is not the maximal value $n^2 + n + 1$, then $|\Pi| \leq n^2$. This refines Corollary 2 of Theorem 13.1 in the abelian case. The idea is that an abelian group acts regularly on each orbit, so if the faithful orbit covers all points ($n^2 + n + 1$ points), then $|\Pi| = n^2 + n + 1$; otherwise the faithful orbit has size at most the number of affine points $n^2$.
