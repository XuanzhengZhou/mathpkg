For a non-trivial connected set of elation flags in a finite projective plane, Lemma 13.7 guarantees that all elations whose centres belong to the set have the same prime order. This unique prime is called the characteristic of the connected set. The concept links the combinatorial structure of connected flags with the group-theoretic orders of the elations they contain, providing a key invariant for the analysis of collineation groups.
