Lemma 13.5 is a general combinatorial result about permutation groups on finite sets. Given a permutation group $\Gamma$ and a prime $p$, if there is a subset $\mathcal{T}$ of the domain such that each $t \in \mathcal{T}$ is the unique fixed point of some element of $\Gamma$ of order $p$, then all elements of $\mathcal{T}$ must lie in the same orbit of $\Gamma$. The proof uses orbit length arguments modulo $p$: if two elements were in different orbits, the orbit cardinalities would yield contradictory congruences mod $p$.
