Let $\Lambda$ be a Frobenius permutation group of order $2n$ on a set $\mathcal{S}$ of $n$ points, where $n$ is odd. Then

(i) given any pair of distinct points $a, b$ of $\mathcal{S}$ there exists an element of order $2$ in $\Lambda$ which interchanges $a$ and $b$.

(ii) if $\Lambda'$ is a subgroup of $\Lambda$ containing $r$ elements of order $2$, $r \geq 1$, then the order of $\Lambda'$ is $2r$.
