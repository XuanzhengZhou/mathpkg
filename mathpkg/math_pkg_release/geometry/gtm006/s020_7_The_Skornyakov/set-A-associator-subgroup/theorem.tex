For any $a, b \in D$, define
$$A(a, b) = \{x \in D \mid [x, a, b] = x[b, a]\}.$$

Then:
\begin{enumerate}
\item $x \in A(a, b) \iff (xa)b = x(ba)$.
\item $0 \in A(a, b)$, so $A(a, b)$ is nonempty.
\item $A(a, b)$ is a subgroup of the additive group of $D$.
\item $A(a, b) = A(b, a)$.
\end{enumerate}
