The result follows from the proof of Theorem 3.5. A finite projective plane of order $n$ has exactly $n^2 + n + 1$ points and $n^2 + n + 1$ lines, with each line containing $n+1$ points. A subplane $\mathcal{P}_0$ of order $m$ contains exactly $m^2 + m + 1$ points. The subplane $\mathcal{P}_0$ is a Baer subplane if and only if every point of $\mathcal{P}$ is incident with at least one line of $\mathcal{P}_0$, which by counting arguments forces $n = m^2$.
