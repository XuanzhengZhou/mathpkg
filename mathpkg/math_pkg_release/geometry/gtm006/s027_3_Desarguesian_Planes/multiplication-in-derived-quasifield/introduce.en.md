Theorem 10.16 specializes the coordinate description of Theorem 10.15 to the case where the original planar ternary ring is a quasifield (so the plane is a translation plane). In this setting, the new multiplication $*$ in the derived PTR admits an explicit formula: for elements with nonzero second component $\beta t$, the product is determined by solving a system of two equations involving auxiliary parameters $\varrho, \sigma$, while for elements with $\beta = 0$, the multiplication reduces to the ordinary scalar multiplication in the original quasifield. This formula is essential for concretely computing in derived planes such as the Hall planes.
