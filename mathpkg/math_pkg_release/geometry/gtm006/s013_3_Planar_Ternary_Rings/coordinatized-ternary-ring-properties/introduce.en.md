When a projective plane is coordinatized by a set $R$, the incidence structure naturally induces a ternary operation $T$ on $R$: the value $T(a,b,c)=k$ means that the point $(b,c)$ lies on the line $[a,k]$. Theorem 5.1 shows that this geometrically defined ternary operation always satisfies five algebraic properties (A)-(E). Any ternary ring satisfying these five axioms is called a planar ternary ring (PTR), and these axioms are the starting point for the algebraic study of projective planes.
