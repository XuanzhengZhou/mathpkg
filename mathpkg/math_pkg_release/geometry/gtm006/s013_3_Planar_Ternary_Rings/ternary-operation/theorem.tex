Let $\mathcal{S}$ be any set. A ternary operation $T$ on $\mathcal{S}$ is a rule which assigns to any three ordered elements $a, b, c$ of $\mathcal{S}$ a unique element $T(a, b, c)$ of $\mathcal{S}$. A non-empty set $\mathcal{S}$ with a ternary operation $T$ is called a ternary ring, denoted by $(\mathcal{S}, T)$.
