If the PTR $(R,T)$ of Theorem 10.15 is a quasifield, the multiplication in $(R,T_1)$ is given by:
$$(\alpha + \beta t) * (\gamma + \delta t) = -\eta - \mu t, \quad \text{if } \beta \neq 0,$$
where
(1) $(\varrho + \sigma t)(-1 + \alpha t) = -\beta t$,
(2) $(\varrho + \sigma t)(\gamma + \eta t) = -\delta - \mu t$,
and by
$$\alpha * (\gamma + \delta t) = \alpha \gamma + (\alpha \delta) t, \quad \text{if } \beta = 0.$$
