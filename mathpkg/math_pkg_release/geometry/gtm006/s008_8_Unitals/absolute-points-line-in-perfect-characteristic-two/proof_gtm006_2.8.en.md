*Proof.* Using a quadratic form representation, the absolute point condition is $a_{11}x^2 + a_{22}y^2 + a_{33}z^2 = 0$. Since every element of $K$ is a square, there exist $b_{ij} \in K$ with $b_{ij}^2 = a_{ij}$. The quadratic form can then be written as
$$(b_{11}x + b_{22}y + b_{33}z)^2 = 0,$$
which is equivalent to the linear equation $b_{11}x + b_{22}y + b_{33}z = 0$. This is precisely the equation of a line. $\square$
