Let $K$ be a field with an involutory automorphism $\phi$ and let $F$ be the set of elements fixed by $\phi$. For any $a \neq 0$ in $K$, the equation
$$x^{1+\phi} = a$$
can only have solutions if $a \in F$, in which case if it has solutions then the set of solutions is a coset of $N$ in the multiplicative group $K^*$.
