A set of mutually orthogonal Latin squares (MOLS) of order $n$ is called complete if it contains the maximum possible number of squares, namely $n-1$. This upper bound follows from simple counting arguments: in any set of MOLS in normal form, there are only $n-1$ possible non-zero entries for the $(1,0)$-position. Complete sets of MOLS are equivalent to finite projective planes of order $n$: given a projective plane, one can construct a complete set of MOLS via the associated planar ternary ring, and conversely, a complete set of MOLS yields a planar ternary ring which coordinatizes a projective plane.
