A \textbf{weak quasifield} $W$ is a set with two binary operations $+, \cdot$ satisfying:
\begin{enumerate}
\item $(W, +)$ is an abelian group;
\item $(W^*, \cdot)$ is a loop, where $W^* = W \setminus \{0\}$;
\item $x(y+z) = xy+xz$ for all $x, y, z \in W$;
\item the additive identity $0$ satisfies $0 \cdot x = 0$ for all $x \in W$.
\end{enumerate}
