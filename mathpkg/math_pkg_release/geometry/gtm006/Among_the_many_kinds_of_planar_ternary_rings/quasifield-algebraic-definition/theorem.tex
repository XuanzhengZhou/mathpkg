A \textbf{quasifield} $Q$ is a set with two binary operations $+, \cdot$ satisfying:
\begin{enumerate}
\item $(Q, +)$ is a group;
\item $(Q^*, \cdot)$ is a loop, where $Q^* = Q \setminus \{0\}$;
\item $x(y+z) = xy+xz$ for all $x, y, z \in Q$;
\item the additive identity $0$ satisfies $0x=0$ for all $x \in Q$;
\item $ax=bx+c$ has a unique solution for $x$, given $a, b, c \in Q$, $a \neq b$.
\end{enumerate}
