A projective plane is desarguesian if and only if every coordinating planar ternary ring is a skewfield, if and only if the plane is $(V,k)$-transitive for all $V$ and $k$. Equivalently, it suffices that the plane is $(V,k)$-transitive for all incident pairs $V$ and $k$, together with $(V,k)$-transitivity for one non-incident pair $V$ and $k$.
