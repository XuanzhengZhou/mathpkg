In an affine plane $\mathcal{A}$:
\begin{enumerate}
\item If $l \parallel h$ then $l$ and $h$ have no common point (unless $l = h$).
\item For any line $m$ and any point $A$ not on $m$, there is a unique line through $A$ parallel to $m$.
\end{enumerate}
Thus parallelism is an equivalence relation on the set of lines of $\mathcal{A}$, and the equivalence classes are called parallel classes.
