Let $A$ be the incidence matrix of a finite projective plane of order $n$. Let $B = AA'$. Then the entries of $B = (b_{ij})$ are given by:
$$b_{ii} = n + 1 \quad \text{for all } i,$$
$$b_{ij} = 1 \quad \text{for all } i \neq j.$$
Equivalently, $AA' = nI + J$, where $I$ is the $v \times v$ identity matrix and $J$ is the $v \times v$ matrix of all ones, with $v = n^2 + n + 1$.
