Exercise 2.7 asks to prove that if $E_1, \dots, E_{n+2}$ is a frame in $\mathcal{P}(V)$, then for any $r < n+1$, no set of $r+1$ points of the frame lies in a subspace of $g$-dimension $r-1$. This is a basic structural property of frames generalizing the fact that in a projective plane no three points of a frame are collinear.
