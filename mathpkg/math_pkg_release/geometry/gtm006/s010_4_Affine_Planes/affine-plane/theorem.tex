An \text{affine plane} $\mathcal{A}$ is a set of points and lines with an incidence relation satisfying:

\begin{enumerate}
\item[(i)] Any two distinct points are incident with exactly one line.
\item[(ii)] Given a point $P$ and a line $l$ not incident with $P$, there exists exactly one line $m$ through $P$ such that $l$ and $m$ have no point in common (i.e., $l \parallel m$). This is \text{Playfair's axiom}.
\item[(iii)] There exist three non-collinear points.
\end{enumerate}

Equivalently, if $\mathcal{P}$ is a projective plane and $l$ is any line of $\mathcal{P}$, then $\mathcal{P}^l$ — the set of points and lines of $\mathcal{P}$ obtained by deleting $l$ and all points incident with it, with incidence inherited from $\mathcal{P}$ — is an affine plane. Conversely, every affine plane arises uniquely in this way.
