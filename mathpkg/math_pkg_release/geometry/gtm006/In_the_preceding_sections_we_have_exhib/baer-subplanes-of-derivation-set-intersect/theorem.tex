Let $\mathcal{S}$ be a derivation set of an affine plane $\mathcal{A} = \mathcal{P}^{\ell_{\infty}}$, and let $\{\mathcal{B}_1, \mathcal{B}_2, \ldots\}$ be the Baer subplanes of $\mathcal{P}$ which belong to $\mathcal{S}$. If $\mathcal{A}_i = \mathcal{B}_i^{\ell_{\infty}}$ for each $i$, then $\mathcal{A}_i \cap \mathcal{A}_j$ consists of $0$ or $1$ points, if $i \neq j$.
