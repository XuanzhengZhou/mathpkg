Let $A$ be an arbitrary (not necessarily open) set in $\mathbb{R}^n$. A mapping $F: A \to \mathbb{R}^m$ is said to be \text{differentiable on $A$} if there exists an open set $U \subset \mathbb{R}^n$ containing $A$ and a mapping $G: U \to \mathbb{R}^n$ such that

$$G|_A = F,$$

and $G$ is differentiable at each $x_0 \in U$.
