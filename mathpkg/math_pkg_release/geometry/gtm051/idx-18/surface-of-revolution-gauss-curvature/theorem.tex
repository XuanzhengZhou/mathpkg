Let a surface of revolution be given by
$$f(u, v) = (h(u) \cos v, h(u) \sin v, k(u)),$$
with the parametrization satisfying $h'^2 + k'^2 = 1$ (unit speed profile curve). Then:
$$g_{11} = 1, \quad g_{12} = 0, \quad g_{22} = h^2,$$
$$h_{11} = -k'h'' + h'k'', \quad h_{12} = 0, \quad h_{22} = hk'.$$

The Gauss curvature is
$$K = \frac{(h'k'k'' - k'k'h'')}{h} = -\frac{h''}{h}.$$

A surface of revolution has constant Gauss curvature $K_0$ if and only if
$$h''(u) + K_0 h(u) = 0,$$
with $h'^2 \leq 1$.
