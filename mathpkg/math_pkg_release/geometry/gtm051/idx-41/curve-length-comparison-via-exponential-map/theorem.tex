Let $K \leq K_1$ on a complete surface $M$. Define $\rho = \pi/\sqrt{K_1}$. Let $c = c(t)$, $0 \leq t \leq a$, be a unit-speed geodesic from $p = c(0)$ to $q = c(a)$ with $a < \rho$. Let $b = b(s)$, $s_0 \leq s \leq s_1$, be another curve from $p$ to $q$ which may be written as

$$b(s) = \exp_p \tilde{b}(s),$$

where $\tilde{b}(s)$ is a curve lying in $B_\rho(0) \subset T_p M$ with $\tilde{b}(s_0) = 0$, $\tilde{b}(s_1) = a\dot{c}(0)$. Then $L(b) \geq L(c)$.
