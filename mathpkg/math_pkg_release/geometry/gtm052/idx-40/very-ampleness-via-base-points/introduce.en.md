This result rephrases the condition (II, 7.8.2) for a complete linear system $|D|$ to be very ample using the language of assigned base points. The reformulation states that $|D|$ must have no base points, and the linear system obtained by subtracting any point $P$ must have no unassigned base points (ordinary or infinitely near). This elegantly captures both point separation and tangent vector separation.
