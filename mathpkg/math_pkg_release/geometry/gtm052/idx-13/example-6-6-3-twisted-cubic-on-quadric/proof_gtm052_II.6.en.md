Let $C$ be the twisted cubic curve parametrized by $(t^3, u^3, t^2u, tu^2)$ in $\mathbf{P}^3$, which lies on $Q$. Consider the quadric cone $Y: yz = w^2$. Then $Y \cap Q = C \cup L$ where $L$ is the line $y = w = 0$. Since $Y \sim 2H$ (where $H$ is a hyperplane) on $\mathbf{P}^3$, the divisor $Y \cap Q$ has type $(2,2)$ on $Q$ via the restriction map $\text{Cl} \mathbf{P}^3 \to \text{Cl} Q$. The line $L$ has type $(1,0)$, so $C = (Y \cap Q) - L$ has type $(1,2)$.

If there existed a surface $Y' \subseteq \mathbf{P}^3$, not containing $Q$, with $Y' \cap Q = C$, then the corresponding divisor would be $rC$ for some integer $r > 0$, which is of type $(r, 2r)$. But any divisor coming from $\text{Cl} \mathbf{P}^3$ via restriction to $Q$ must be of type $(d, d)$ for some $d$, since $\text{Cl} \mathbf{P}^3 \cong \mathbb{Z}$ is generated by $H$ and $H \cap Q$ has type $(1,1)$. This is a contradiction, so no such surface exists.
