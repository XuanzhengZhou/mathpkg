A linear system with assigned base points, denoted $|D - P_1 - \ldots - P_r|$, is a sublinear system of a complete linear system $|D|$ on a surface $X$, consisting of all divisors passing through specified points $P_1, \ldots, P_r$. This language provides a convenient way to describe linear systems on various blown-up models of $X$ in terms of linear systems on $X$ with assigned base points, including infinitely near points. The dimension of the system drops by exactly one for each assigned base point that was not already an unassigned base point.
