Let $X$ be a surface and $|D|$ a complete linear system on $X$. Using the convention that dimensions drop by exactly one when assigning a base point not already unassigned, $|D|$ is very ample if and only if for any two points $P, Q \in X$ (including the case where $Q$ is infinitely near $P$),
$$\dim |D - P - Q| = \dim |D| - 2.$$
