The proof follows from the definition of very ampleness in (II, 7.8.2) translated into the language of assigned base points. A complete linear system $|D|$ separates points $P$ and $Q$ if and only if $Q$ is not a base point of the linear system $|D - P|$ (i.e., there exists a divisor in $|D|$ passing through $P$ but not through $Q$). Similarly, $|D|$ separates tangent vectors at $P$ if and only if $|D - P|$ has no unassigned base points infinitely near $P$, meaning that there exist divisors in $|D|$ having $P$ as a simple point with distinct tangent directions. Combined, these conditions are equivalent to: $|D|$ has no base points, and for every $P \in X$, $|D - P|$ has no unassigned base points.
