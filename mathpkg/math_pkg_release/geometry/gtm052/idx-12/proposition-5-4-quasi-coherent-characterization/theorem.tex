Let $X$ be a scheme. Then an $\mathcal{O}_X$-module $\mathcal{F}$ is quasi-coherent if and only if for every open affine subset $U = \operatorname{Spec} A$ of $X$, there is an $A$-module $M$ such that $\mathcal{F}|_U \cong \tilde{M}$. If $X$ is noetherian, then $\mathcal{F}$ is coherent if and only if the same is true, with the extra condition that $M$ be a finitely generated $A$-module.
