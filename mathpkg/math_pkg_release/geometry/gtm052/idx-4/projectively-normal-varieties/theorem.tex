A projective variety $Y \subseteq \mathbf{P}^n$ is projectively normal (with respect to the given embedding) if its homogeneous coordinate ring $S(Y)$ is integrally closed.

(a) If $Y$ is projectively normal, then $Y$ is normal.
(b) There are normal varieties in projective space which are not projectively normal. For example, let $Y$ be the twisted quartic curve in $\mathbf{P}^3$ given parametrically by $(x, y, z, w) = (t^4, t^3u, tu^3, u^4)$. Then $Y$ is normal but not projectively normal.
