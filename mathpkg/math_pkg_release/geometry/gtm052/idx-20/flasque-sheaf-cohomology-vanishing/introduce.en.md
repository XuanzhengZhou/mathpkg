A flasque sheaf is one for which all restriction maps between open sets are surjective. This property makes flasque sheaves particularly well-behaved with respect to the global sections functor. Proposition 2.5 of Hartshorne shows that flasque sheaves are acyclic: their higher derived functor cohomology groups all vanish. This result is fundamental because it provides a practical tool for computing sheaf cohomology via flasque resolutions, analogous to using injective resolutions.
