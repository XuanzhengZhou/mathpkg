Let $A$ and $B$ be rings, and let $(\text{Spec } A, \mathcal{O}_{\text{Spec } A})$ and $(\text{Spec } B, \mathcal{O}_{\text{Spec } B})$ be their spectra as locally ringed spaces.

(a) $(\text{Spec } A, \mathcal{O}_{\text{Spec } A})$ is a locally ringed space.

(b) If $\varphi: A \to B$ is a ring homomorphism, then $\varphi$ induces a natural morphism of locally ringed spaces
$$(f, f^\#): (\text{Spec } B, \mathcal{O}_{\text{Spec } B}) \to (\text{Spec } A, \mathcal{O}_{\text{Spec } A}),$$
where $f(\mathfrak{p}) = \varphi^{-1}(\mathfrak{p})$ for $\mathfrak{p} \in \text{Spec } B$, and $f^\#$ is the induced sheaf map.

(c) Conversely, any morphism of locally ringed spaces from $\text{Spec } B$ to $\text{Spec } A$ is induced by a unique ring homomorphism $\varphi: A \to B$ as in (b).
