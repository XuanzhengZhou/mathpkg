Let $S$ be a graded ring, and let $\text{Proj } S$ be the projective spectrum with structure sheaf $\mathcal{O}$.

(a) $\text{Proj } S$ is a locally ringed space.

(b) For any homogeneous $f \in S_+$, let $D_+(f) = \{\mathfrak{p} \in \text{Proj } S \mid f \notin \mathfrak{p}\}$. Then $D_+(f)$ is open in $\text{Proj } S$, the open sets $D_+(f)$ cover $\text{Proj } S$, and for each such $f$,
$$\left(D_+(f), \mathcal{O}|_{D_+(f)}\right) \cong \text{Spec } S_{(f)},$$
where $S_{(f)}$ is the subring of elements of degree $0$ in the localized ring $S_f$.

(c) $\text{Proj } S$ is a scheme.
