A \text{complex} $A^\bullet$ in an abelian category $\mathfrak{A}$ is a collection of objects $A^i$, $i \in \mathbb{Z}$, and morphisms $d^i: A^i \to A^{i+1}$, such that $d^{i+1} \circ d^i = 0$ for all $i$. If the objects $A^i$ are specified only in a certain range (e.g., $i \geqslant 0$), then we set $A^i = 0$ for all other $i$. A morphism of complexes $f: A^\bullet \to B^\bullet$ is a collection of morphisms $f^i: A^i \to B^i$ commuting with the differentials.
