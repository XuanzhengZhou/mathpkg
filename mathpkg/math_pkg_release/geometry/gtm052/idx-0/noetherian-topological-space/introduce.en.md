This is a definition from Chapter I, Section §1 of Hartshorne's Algebraic Geometry. A noetherian topological space satisfies the descending chain condition on closed subsets. A key example is affine $n$-space $\mathbf{A}^n$ with the Zariski topology, which is noetherian because the polynomial ring $k[x_1,\ldots,x_n]$ is a noetherian ring.
