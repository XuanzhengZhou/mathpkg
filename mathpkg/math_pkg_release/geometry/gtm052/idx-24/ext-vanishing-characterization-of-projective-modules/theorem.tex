Let $A$ be a ring and $M$ an $A$-module. Then:
\begin{enumerate}
\item[(a)] $M$ is projective if and only if $\operatorname{Ext}^1(M, N) = 0$ for all $A$-modules $N$;
\item[(b)] $\operatorname{pd}(M) \leq n$ if and only if $\operatorname{Ext}^i(M, N) = 0$ for all $i > n$ and all $A$-modules $N$.
\end{enumerate}
