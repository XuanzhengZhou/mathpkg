Lemma 5.1 from Chapter IV, Section 5 of Hartshorne's Algebraic Geometry establishes that for any curve of genus at least 2, the canonical linear system has no base points. This fundamental result ensures that the canonical morphism is defined everywhere on the curve, providing a well-defined map to projective space. The proof is a short application of the Riemann-Roch theorem and the fact that a non-rational curve has no global sections for a single point divisor.
