Lemma 5.1 of Hartshorne's Algebraic Geometry establishes that for any curve of genus $g \geqslant 2$, the canonical linear system $|K|$ is base-point-free. This is a fundamental preliminary result for the study of the canonical morphism. The proof uses the Riemann–Roch theorem together with the fact that a non-rational curve satisfies $\dim|P| = 0$ for any point $P$, to conclude that $\dim|K-P| = g-2 = \dim|K|-1$, which is exactly the condition for $|K|$ to have no base points.
