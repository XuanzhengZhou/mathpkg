An inverse system (also called a projective system) is a family of objects indexed by a directed set, together with compatible transition maps going backward in the index. It provides the algebraic foundation for studying completions, formal schemes, and limits. The definition is purely categorical: for each pair of indices $n' \geq n$, there is a morphism $\varphi_{n'n}$ from the object at index $n'$ to the object at index $n$, satisfying the cocycle condition $\varphi_{n''n} = \varphi_{n'n} \circ \varphi_{n''n'}$.
