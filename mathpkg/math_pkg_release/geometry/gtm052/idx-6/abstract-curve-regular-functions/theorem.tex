Let $C_K$ be the abstract nonsingular curve of a function field $K/k$. For an open subset $U \subseteq C_K$, the ring of regular functions on $U$ is
$$\mathcal{O}(U) = \bigcap_{P \in U} R_P.$$

An element $f \in \mathcal{O}(U)$ defines a function $U \to k$ by $f(P) = f \bmod \mathfrak{m}_P$. If two elements $f, g \in \mathcal{O}(U)$ define the same function, then $f - g \in \mathfrak{m}_P$ for infinitely many $P \in C_K$, so $f = g$. Thus elements of $\mathcal{O}(U)$ can be identified with functions $U \to k$.

By Theorem 6.5, any $f \in K$ is a regular function on some open set $U \subseteq C_K$.
