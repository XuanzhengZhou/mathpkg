Use (IV, Ex. 4.16) to prove the Weil conjectures for elliptic curves. First note that for any $r$,

$$N_r = q^r - (f^r + \hat{f}^r) + 1,$$

where $f = F'$. Then calculate $Z(t)$ formally and conclude that

$$Z(t) = \frac{(1 - ft)(1 - \hat{f}t)}{(1 - t)(1 - qt)}$$

and hence

$$Z(t) = \frac{1 - at + qt^2}{(1 - t)(1 - qt)},$$

where $f + \hat{f} = a_X$. This proves rationality immediately. Verify the functional equation. Finally, if we write

$$1 - at + qt^2 = (1 - \alpha t)(1 - \beta t),$$

show that $|a| \leqslant 2\sqrt{q}$ if and only if $|\alpha| = |\beta| = \sqrt{q}$. Thus the analogue of the Riemann hypothesis is just (IV, Ex. 4.16d).
