Start with any nonsingular projective algebraic threefold. Let $c$ be a curve in $X$ which is nonsingular except for one double point $P$, having distinct tangent directions. In a small analytic neighborhood of $P$, blow up one branch of $c$, then the other. Away from $P$, blow up the curve $c$. The result is a Moishezon manifold $\mathfrak{X}$ which is not algebraic. Indeed, one finds $l_0 \sim 0$ in $\mathfrak{X}$, which is impossible for an algebraic variety: for an algebraic variety, a curve has nonzero intersection number with some surface, but intersection number is defined on homology classes, so $l_0 \sim 0$ cannot happen.
