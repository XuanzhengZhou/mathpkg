It is possible to write

$$Z(t) = \frac{P_1(t) P_3(t) \cdots P_{2n-1}(t)}{P_0(t) P_2(t) \cdots P_{2n}(t)}$$

where $P_0(t) = 1-t$, $P_{2n}(t) = 1-q^n t$, and for $1 \leqslant i \leqslant 2n-1$, $P_i(t)$ is a polynomial with integer coefficients, which can be written

$$P_i(t) = \prod_j (1 - \alpha_{ij} t)$$

where the $\alpha_{ij}$ are algebraic integers with $|\alpha_{ij}| = q^{i/2}$.
