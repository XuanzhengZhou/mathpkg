Let $P$ be an arbitrary point of an irreducible plane curve $C \subset \mathbb{C}_{XY}$ having coordinate ring $\mathbb{C}[x, y]$ and function field $K_C = \mathbb{C}(x, y)$. Then there exists an evaluation of the elements of $K_C$ satisfying (2.5.1) and (2.5.2) and coinciding with the natural evaluation of $\mathbb{C}[x, y]$ at $P$.

If $x$ and $y$ can be expressed as convergent power series in an element $T \in K_C$ of order one, then every element $f \in K_C$ is a Laurent series $f(x, y) = c_m T^m + c_{m+1} T^{m+1} + \ldots$, and the evaluation assigns to $f$ the value
$$\begin{cases} c_0 & \text{if } m = 0 \\ 0 & \text{if } m > 0 \\ \infty & \text{if } m < 0 \end{cases}$$
where $(x, y)$ evaluated at $T = 0$ equals $P \in C$.
