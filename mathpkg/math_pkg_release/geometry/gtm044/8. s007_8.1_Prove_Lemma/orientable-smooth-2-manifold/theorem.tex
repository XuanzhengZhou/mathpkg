Let $M$ be a smooth 2-manifold with coordinate charts $\phi_\alpha$ from open sets of $\mathbb{R}^2$ to $U_\alpha \subset M$ such that each transition map $\phi_\beta^{-1} \circ \phi_\alpha: \phi_\alpha^{-1}(U_\alpha \cap U_\beta) \rightarrow \phi_\beta^{-1}(U_\alpha \cap U_\beta)$ is smooth. The manifold $M$ is \text{orientable} if each such $\phi_\beta^{-1} \circ \phi_\alpha$ is orientation preserving---that is, if
$$\det(J_{\phi_\beta^{-1} \circ \phi_\alpha}(x)) > 0 \quad \text{for each } x \in \phi_\alpha^{-1}(U_\alpha \cap U_\beta).$$
This definition generalizes in the obvious way to real $n$-manifolds.
