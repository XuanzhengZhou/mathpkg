Let $\kappa > \omega$ be a regular cardinal and $B_0 \subseteq B_1 \subseteq \cdots \subseteq B_\alpha \subseteq \cdots (\alpha < \kappa)$ be a direct system of complete Boolean algebras such that $B_\alpha$ is a completion of $\bigcup_{\beta < \alpha} B_\beta$ for limit $\alpha$. Define $B = \bigcup_{\alpha < \kappa} B_\alpha$ and $\bar{B}$ its completion. If $(\forall \alpha < \kappa)[\bar{B}_\alpha < \kappa]$, then:
1. $B$ satisfies the $\kappa$-chain condition.
2. $\bar{B} = B$.
