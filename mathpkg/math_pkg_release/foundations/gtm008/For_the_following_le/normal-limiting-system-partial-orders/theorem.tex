A system $\langle \{P_\alpha \mid \alpha < \kappa\}, \{p_{\alpha\beta} \mid \alpha \leq \beta < \kappa\} \rangle$ is a normal limiting system of partial order structures iff:
1. Each $P_\alpha$ is a partial order structure.
2. The system is an o.c.o. inverse system.
3. For $\alpha \leq \beta \leq \gamma$, $p_{\alpha\gamma} = p_{\alpha\beta} \circ p_{\beta\gamma}$.
4. $p_{\alpha\beta}''[x]_{P_\beta} = [p_{\alpha\beta}(x)]_{P_\alpha}$.
