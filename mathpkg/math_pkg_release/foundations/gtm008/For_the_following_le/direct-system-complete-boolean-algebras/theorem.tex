Let $\kappa$ be a cardinal or $\mathrm{On}$. A system
$$\langle \{\mathbf{B}_{\alpha} \mid \alpha < \kappa\}, \{i_{\alpha\beta} \mid \alpha \leq \beta < \kappa\} \rangle$$
is called a \textbf{direct system of complete Boolean algebras} if and only if $\mathbf{B}_{\alpha}$ ($\alpha < \kappa$) is a complete Boolean algebra and $i_{\alpha\beta}: \mathbf{B}_{\alpha} \to \mathbf{B}_{\beta}$ is a complete monomorphism such that:
\begin{enumerate}
\item $i_{\alpha\alpha}$ is the identity map.
\item $i_{\beta\gamma} \circ i_{\alpha\beta} = i_{\alpha\gamma}$ for $\alpha \leq \beta \leq \gamma < \kappa$.
\end{enumerate}

We assume that $\mathbf{B}_{\alpha}$ is a complete subalgebra of $\mathbf{B}_{\beta}$ if $\alpha \leq \beta < \kappa$. Under this assumption $\bigcup_{\alpha < \kappa} B_{\alpha}$ becomes a Boolean algebra $\mathbf{B}'$ if we define $b_1 + b_2$, $b_1 \cdot b_2$, $-b$ in $\mathbf{B}_{\alpha}$ where $\alpha$ is the least ordinal such that $b_1, b_2 \in \mathbf{B}_{\alpha}$.
