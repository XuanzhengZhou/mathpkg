1. $CA(p) < \aleph_\alpha.$
2. Define $IC(p) = \{q \mid q \text{ is coatomic } \land p \text{ and } q \text{ are incompatible}\}$. Then

$$IC(p) \leq \aleph_\alpha.$$
3. If $p$ and $q$ are incompatible, then there exists $q_0 \in IC(p)$ such that $q \leq q_0.$
4. Let $A$ be a set of coatomic elements with $\bar{A} \leq \aleph_\alpha$. Then

$$\{p \mid CA(p) \subseteq A\} \leq \aleph_\alpha.$$

Remark. Condition 3 implies that the set of all $q$'s that are incompatible with $p$ is

$$\bigcup_{q \in IC(p)} [q].$$
