Theorem 10.3 asserts that the forcing relation $p \Vdash \varphi$ is definable within the language $\mathcal{L}_0$ from the Boolean truth value $\llbracket \varphi \rrbracket$, the Boolean algebra $B$, and the partial order $P$. This definability result is fundamental because it allows one to work with the forcing relation internally within the ground model $M$, without appealing to generic extensions. The definability is uniform for limited formulas (those with bounded quantifiers).
