Theorem 9.35 (and the remark preceding the three theorems after condition 5) states that for quantifier-free formulas $\varphi$, the Boolean truth value computed within the substructure $M_{\alpha}$ coincides with the truth value computed in the full model $M$. This absoluteness property is critical because it ensures that the recursive definition of $\llbracket \varphi \rrbracket$ is well-founded: the truth value of a limited formula at stage $\alpha$ depends only on the elements already constructed up to that stage.
