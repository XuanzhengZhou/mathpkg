The structure $\langle P, \leq \rangle$ is a \text{partial order structure} iff $\leq$ is a subset of $P \times P$ and $\forall x, y \in P$:
\begin{enumerate}
\item $x \leq x$.
\item $x \leq y \land y \leq x \rightarrow x = y$.
\item $x \leq y \land y \leq z \rightarrow x \leq z$.
\end{enumerate}
The relation $\leq$ is a \text{partial ordering} of $P$ iff $\langle P, \leq \rangle$ is a partial order structure.
