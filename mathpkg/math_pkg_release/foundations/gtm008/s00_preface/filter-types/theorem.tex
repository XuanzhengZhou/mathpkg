If $F$ is a filter on the Boolean algebra $B$ then:
\begin{enumerate}
\item $F$ is an \text{ultrafilter} iff $(\forall x \in |B|)[x \in F \leftrightarrow -x \notin F]$.
\item $F$ is a \text{principal filter} iff $(\exists a \in F)[F = \{x \in |B| \mid a \leq x\}]$.
\item $F$ is \text{$M$-complete} iff $(\forall A \in M)\left[A \subseteq F \rightarrow \prod_{a \in A} a \in F\right]$.
\item $F$ is a \text{proper filter} iff $0 \notin F$.
\item $F$ is a \text{trivial filter} iff $F = \{1\}$.
\end{enumerate}
