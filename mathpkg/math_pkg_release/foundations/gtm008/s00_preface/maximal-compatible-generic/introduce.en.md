This theorem shows that generic sets are maximal with respect to compatibility: no new element can be added to $G$ while preserving pairwise compatibility. The proof constructs a dense set from any candidate element $p \notin G$ that would witness the failure of genericity if $p$ were compatible with all of $G$. This maximality is used to prove the bijective correspondence between generic sets and ultrafilters (Theorems 2.19 and 2.20).
