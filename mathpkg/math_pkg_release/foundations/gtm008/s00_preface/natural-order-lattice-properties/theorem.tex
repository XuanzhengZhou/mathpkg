If $\langle B, +, \cdot, ^-, 0, 1 \rangle$ is a Boolean algebra with natural order $\leq$ then $\forall a, b, c \in B$:
\begin{enumerate}
\item $0 \leq a$ and $a \leq 1$.
\item $ab \leq a$ and $ab \leq b$.
\item $[a \leq c \land b \leq c] \rightarrow (a + b) \leq c$.
\item $[c \leq a \land c \leq b] \rightarrow c \leq ab$.
\end{enumerate}
