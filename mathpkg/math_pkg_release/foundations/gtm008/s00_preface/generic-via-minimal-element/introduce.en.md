This theorem provides a simple sufficient condition for genericity: if a partial order has a minimal (least) element $p$, then the upward closure of $p$ forms a generic set. The proof verifies the three genericity conditions: compatibility follows from all elements being above $p$, upward closure is by construction, and meeting dense sets follows because the minimal element $p$ belongs to every dense set.
