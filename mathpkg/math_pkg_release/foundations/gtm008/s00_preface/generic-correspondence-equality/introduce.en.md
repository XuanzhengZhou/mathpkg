This theorem establishes that the correspondence between generic sets and $M$-complete ultrafilters is a bijection. Starting from a generic set $G$, constructing the ultrafilter $F$, and then reconstructing a generic set $G'$ from $F$, one recovers the original $G$. The proof uses the maximal compatibility of generic sets (Theorem 2.18) to show $G \subseteq G'$, and since both are maximal, they must be equal.
