If $\leq$ is a partial ordering of $P$ then:
\begin{enumerate}
\item $(\forall a, b \in P)[\text{Comp}(a, b) \xrightarrow{\Delta} (\exists c \in P)[c \leq a \land c \leq b]]$.
\item $(\forall S \subseteq P)[\text{Comp}(S) \xrightarrow{\Delta} (\forall a, b \in S)[\text{Comp}(a, b)]]$.
\end{enumerate}
