**Proof.**
1. $aa = a$, hence $a \leq a$.
2. If $a \leq b$ and $b \leq a$, then $a = ab$ and $b = ba$. Since multiplication is commutative, $a = ab = ba = b$.
3. If $a = ab$ and $b = bc$, then $a = ab = a(bc) = (ab)c = ac$, so $a \leq c$.
