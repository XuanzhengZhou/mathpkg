Let $A$ be a given class. If $P = \langle P, \leq \rangle$ is a partial order structure and $G \subseteq P$, then $G$ is \text{$P$-generic over $A$} iff:
\begin{enumerate}
\item $G$ is compatible.
\item $p \in G \land q \in P \land p \leq q \rightarrow q \in G$.
\item $(\forall S \in A)[S \text{ is a dense subset of } P \rightarrow G \cap S \neq 0]$.
\end{enumerate}
