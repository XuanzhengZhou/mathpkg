This theorem notes two elementary consequences of the filter axioms. Every filter contains the unit element $1$. Moreover, if $x$ is in a filter, then $x + y$ is also in the filter for any $y$ (since $x \leq x + y$). These properties are dual to the corresponding properties of ideals (Theorem 1.42).
