If $K$ is a class, if $\mathcal{L}(\{K(\cdot)\})$ is the language obtained from $\mathcal{L}_0$ by adding a unary predicate symbol $K(\cdot)$, and if $\langle A_\alpha \mid \alpha \in On \rangle$ is a sequence of structures $A_\alpha = \langle A_\alpha, \bar{K}_\alpha \rangle$ defined by transfinite induction on $\alpha$ by:
\begin{align*}
A_0 &\triangleq 0,\\
\bar{K}_\alpha &\triangleq A_\alpha \cap K,\\
A_{\alpha+1} &\triangleq Df(A_\alpha),\\
A_\alpha &\triangleq \bigcup_{\beta < \alpha} A_\beta, \quad \alpha \in K_{\mathrm{II}},
\end{align*}
then
$$L_K \triangleq \bigcup_{\alpha \in On} A_\alpha.$$
