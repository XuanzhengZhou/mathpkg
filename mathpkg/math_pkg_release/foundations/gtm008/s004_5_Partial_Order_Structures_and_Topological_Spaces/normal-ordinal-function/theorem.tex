A function $F: On \rightarrow On$ is a \textit{normal function} iff:
\begin{enumerate}
\item $(\forall \alpha, \beta)[\alpha < \beta \rightarrow F(\alpha) < F(\beta)]$.
\item $(\forall \alpha \in K_{\mathrm{II}})\left[ F(\alpha) = \bigcup_{\beta < \alpha} F(\beta) \right]$.
\end{enumerate}
where $K_{\mathrm{II}}$ denotes the class of limit ordinals.
