If $\mathcal{L}$ is a first order language with individual constants
$$c_0, c_1, \ldots, c_i, \ldots \quad i < \alpha$$
and predicate constants
$$R_0, R_1, \ldots, R_j, \ldots \quad j < \beta,$$
then a \textit{$B$-valued interpretation} of $\mathcal{L}$ is a pair $\langle A, \phi \rangle$ where $A$ is a nonempty set and $\phi$ is a mapping defined on the set of constants of the language $\mathcal{L}$ satisfying:
\begin{enumerate}
\item $\phi(c_i) \in A$, for $i < \alpha$.
\item $\phi(R_j) \colon A^{n_j} \rightarrow B$, for $j < \beta$, where $n_j$ is the number of arguments of $R_j$.
\end{enumerate}
