$$c_0, c_1, \ldots, c_i, \ldots \quad i < \alpha$$

and predicate constants

$$R_0, R_1, \ldots, R_j, \ldots \quad j < \beta.$$

Then a B-valued interpretation of $\mathcal{L}$ is a pair $\langle A, \phi \rangle$ where $A$ is a nonempty set and $\phi$ is a mapping defined on the set of constants of the language $\mathcal{L}$ satisfying the following,

1. $\phi(c_i) \in A, i < \alpha.$
2. $\phi(R_j) \colon A^{n_j} \rightarrow B$, for $j < \beta$ where $n_j$ is the number of arguments of $R_j$.

Remark. In order to define a truth value for closed formulas of $\mathcal{L}$ under a given B-valued interpretation we first extend $\mathcal{L}$ to a new language $\mathcal{L}^* \triangleq \mathcal{L}(C(A))$ by introducing new individual constants $c_a$ for each $a \in A$.
