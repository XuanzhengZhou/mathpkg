A class $A$ is \textit{definable} in $\mathcal{L}_0$ iff there is a formula $\varphi(x)$ of $\mathcal{L}_0$ containing no free variable other than $x$ such that $A = \{x \mid \varphi(x)\}$. In this case $\varphi$ is called a \textit{defining formula} for $A$. Moreover, if $\varphi$ contains parameters $y_1, \ldots, y_k$ from a given set $c$, then we say $A$ is definable in $\mathcal{L}_0$ \textit{from} $c$.
