We will write $[\psi(a_1, \ldots, a_n)]$ for $[\psi(c_{a_1}, \ldots, c_{a_n})]$, $\bar{R}_j$ for $\phi(R_j)$ and $\bar{c}$ for $\phi(c)$. If $\langle A, \phi \rangle$ is a $B$-valued interpretation then

$$A \triangleq \langle A, \bar{R}_0, \bar{R}_1, \ldots, \bar{c}_0, \bar{c}_1, \ldots \rangle$$

is a $B$-valued structure.

We will occasionally consider several interpretations in the same context. We will write $[\varphi]_A$ to indicate that element of $B$ determined by $\varphi$ and the interpretation $\langle A, \phi \rangle$ of the $B$-valued structure $A$.
