Let $\mathbf{A} = \langle A, \|\cdot\| \rangle$ be a $B$-valued structure for a language $\mathcal{L}$. For a sentence $\varphi$ of $\mathcal{L}$, the truth value $\|\varphi\|$ is defined inductively on the complexity of $\varphi$. A sentence $\varphi$ is \textbf{logically valid} in $\mathbf{A}$ iff $\|\varphi\| = 1$ for every $B$-valued structure $\mathbf{A}$ based on any complete Boolean algebra $B$.
