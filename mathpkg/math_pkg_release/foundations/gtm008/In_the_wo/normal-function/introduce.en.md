A normal function is a strictly increasing ordinal function that is continuous at limit ordinals. Every normal function is also semi-normal. Normal functions are important in set theory because they have arbitrarily large fixed points (the Veblen fixed-point theorem). Examples include the aleph function $\alpha \mapsto \aleph_\alpha$ and the beth function $\alpha \mapsto \beth_\alpha$.
