For any finite set of formulas $\{\varphi_1, \ldots, \varphi_n\}$, there exists a semi-normal function $F$ such that for each $i$,

$$(\forall \alpha)(\exists \beta > \alpha)[\beta = F(\beta) \land (\forall x_1, \ldots, x_{k_i} \in M_\alpha)(\varphi_i^{M}(x) \leftrightarrow \varphi_i^{M_\beta}(x))].$$
