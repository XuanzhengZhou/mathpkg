A class $A$ is \text{definable in $\mathcal{L}_0$} iff there is a formula $\varphi(x)$ of $\mathcal{L}_0$ containing no free variable other than $x$ such that $A = \{x \mid \varphi(x)\}$. In this case $\varphi$ is called a defining formula for $A$. If $\varphi$ contains parameters $y_1, \ldots, y_k$ from a given set $c$, then $A$ is \text{definable in $\mathcal{L}_0$ from $c$}.
