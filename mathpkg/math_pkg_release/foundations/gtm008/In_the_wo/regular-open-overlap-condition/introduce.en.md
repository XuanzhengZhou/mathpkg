This theorem provides a criterion for determining when one open set is contained in the regularization (interior-of-closure) of another. If whenever a regular open set $H$ is disjoint from $G_1$, it is also disjoint from $G_2$, then $G_2$ must be contained in the smallest regular open set containing $G_1$. This is a key technical lemma used in proving that the star-delta closure produces regular open sets.
