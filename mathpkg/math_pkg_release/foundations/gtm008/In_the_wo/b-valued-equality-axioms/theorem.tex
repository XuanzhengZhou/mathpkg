If $\mathcal{L}$ is a first order language with equality, then a \text{$B$-valued interpretation} of $\mathcal{L}$ is a $B$-valued interpretation in the sense of Definition 6.1 that additionally satisfies the following Axioms of Equality:
\begin{enumerate}
\item $[\![c = c]\!] = 1$.
\item $[\![c_1 = c_2]\!] = [\![c_2 = c_1]\!]$.
\item $[\![c_1 = c_2]\!] \cdot [\![c_2 = c_3]\!] \leq [\![c_1 = c_3]\!]$.
\item For each $n$-ary predicate constant $R$:
$$[\![c_1 = c'_1]\!] \cdots [\![c_n = c'_n]\!] \cdot [\![R(c_1, \ldots, c_n)]\!] \leq [\![R(c'_1, \ldots, c'_n)]\!].$$
\end{enumerate}
