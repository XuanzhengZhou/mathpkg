Theorem 16.27 gives an explicit description of the power set of a definite set $u$ in $V^{(\mathbf{B})}$. When $u$ is definite, its power set $\mathcal{P}(u)$ is Boolean-equal to the check of the ground-model set $B^{\mathcal{D}(u)}$ (all functions from $\mathcal{D}(u)$ into the Boolean algebra $B$). This is a set-sized object in the ground model, confirming that the Boolean-valued power set operation remains well-behaved from a set-theoretic perspective. The proof uses Theorem 16.25 to express $\llbracket x \in B^{\mathcal{D}(u)} \times \{1\} \rrbracket$ as the sum of equality values with elements of $A_u$.
