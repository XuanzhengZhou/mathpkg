If $A$ is a set, and if
$$(\forall x, y \in A)[x \neq y \rightarrow (\exists z \in A) \neg [z \in x \leftrightarrow z \in y]]$$
then there exists a transitive set $a$ and a function $f$ such that
$$f: A \xrightarrow{1-1}_{\text{onto}} a$$
and $(\forall x, y \in A)[x \in y \leftrightarrow f(x) \in f(y)]$. Moreover if $b$ is a transitive subset of $A$ then $f \upharpoonright b = I \upharpoonright b$.
