This is a form of the Mostowski Collapse Lemma: any extensional set $A$ can be collapsed to a transitive set $a$ via an $\in$-isomorphism $f$. The function $f$ acts as the identity on any transitive subset of $A$, making this a powerful tool for constructing transitive models.
