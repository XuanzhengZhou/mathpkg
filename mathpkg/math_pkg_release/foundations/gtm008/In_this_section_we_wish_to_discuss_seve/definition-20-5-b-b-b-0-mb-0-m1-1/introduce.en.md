Definition 20.5: (\forall b \in B) [b \neq 0 \rightarrow m(b) > 0] \wedge m(1) = 1 from Weak Distributive Laws of Takeuti and Zaring's "Axiomatic Set Theory" (GTM 8). This definition is part of the foundational development of Boolean-valued models and forcing techniques in set theory.
