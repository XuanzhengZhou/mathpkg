A partial order structure $P$ is called coatomic if the partial order can be recovered from the inclusion relation among the sets of coatomic elements above each element. Specifically, $p \leq q$ holds exactly when every coatomic element above $q$ is also above $p$. This condition implies that coatomic elements determine the order structure, and for any $p \neq 1$, there exists at least one coatomic element above $p$.
