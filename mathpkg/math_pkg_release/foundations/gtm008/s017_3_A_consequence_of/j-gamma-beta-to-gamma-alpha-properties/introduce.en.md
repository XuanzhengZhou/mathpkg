Theorem 24.4 establishes two elementary properties of the projection map $j: \Gamma_\beta \rightarrow \Gamma_\alpha$ in Easton's forcing construction. First, $j$ is order-preserving: if $q \leq p$ in $\Gamma_\beta$, then $j(q) \leq j(p)$ in $\Gamma_\alpha$. Second, the image under $j$ of the basic open set $[p]_{P_\beta}$ equals the basic open set $[j(p)]_{P_\alpha}$ in $P_\alpha$. The second property is key to showing that $j$ is an open continuous map, which induces the complete monomorphism between the corresponding Boolean algebras.
