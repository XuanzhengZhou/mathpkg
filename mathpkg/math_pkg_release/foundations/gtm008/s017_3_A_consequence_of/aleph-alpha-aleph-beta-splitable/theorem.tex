$\mathbf{B}$ is $(\aleph_\alpha, \aleph_\beta)$-splitable if the following condition is satisfied. Suppose $0 < q \leq b$, $q, b \in B$, $\{b_{ij} \mid i \in I \land j \in \aleph_\beta\} \subseteq B$, $\bar{I} \leq \aleph_\alpha$, and $(\forall i \in I)[\sum_{j < \aleph_\beta} b_{ij} = b]$. Then
$$(\exists \bar{q} \in B)(\exists \Lambda \subseteq B)[0 < \bar{q} \leq q \land \bar{\Lambda} \leq \aleph_\alpha \land (\forall q' \in B)$$
$$[0 < q' \leq \bar{q} \rightarrow (\forall i \in I)(\exists p \in \Lambda)(\exists \gamma < \aleph_\beta)[p \cdot q' > 0 \land p \cdot \bar{q} \leq \sum_{j < \gamma} b_{ij}]].$$
