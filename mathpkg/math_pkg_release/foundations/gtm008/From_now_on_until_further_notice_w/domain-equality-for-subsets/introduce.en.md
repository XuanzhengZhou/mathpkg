Theorem 16.23 is an important technical lemma for handling power sets in $V^{(\mathbf{B})}$. If $v$ is a Boolean-valued subset of $u$ (i.e., $\llbracket v \subseteq u \rrbracket = 1$) and both are extensional, then $v$ can be replaced by a set $v'$ with the same domain as $u$ (namely $\mathcal{D}(v') = \mathcal{D}(u)$) while preserving Boolean-valued equality $\llbracket v = v' \rrbracket = 1$. The construction defines $v'(x) = \llbracket x \in v \rrbracket$ on $\mathcal{D}(u)$. This result, together with Theorem 16.24, is crucial because it shows that when treating power sets, we can restrict attention to subsets whose domain equals that of the original set, avoiding class-sized domains.
