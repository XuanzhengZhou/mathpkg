This corollary extends the factorization theorem to all Boolean-valued sets: every $u \in V^{(B)}$ is equal (with Boolean value $1$) to a scalar $b \in B$ multiplied by a set $v$ that is both definite and uniform. This canonical form is achieved by first representing $u$ by an extensional set (Theorem 16.11), then applying the uniform representation theorem (Theorem 16.20). It provides a powerful normal form for Boolean-valued sets.
