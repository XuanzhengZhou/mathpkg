This theorem shows that every Boolean-valued set $v$ can be replaced by an extensional set $u$ on any prescribed superset $d$ of its domain, preserving equality with Boolean value $1$. The construction defines $u(x) = \llbracket x \in v \rrbracket$ for $x \in d$. Since any Boolean-valued set is thus equivalent to an extensional one, many proofs can assume without loss of generality that all sets under consideration are extensional, simplifying Boolean-valued calculations.
