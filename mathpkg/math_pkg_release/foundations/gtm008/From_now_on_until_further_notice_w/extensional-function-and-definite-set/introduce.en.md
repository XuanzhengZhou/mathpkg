An extensional function $g$ on a subset $d$ of $V^{(B)}$ respects the Boolean-valued equality: equal inputs (with respect to the Boolean truth value) yield equal outputs. A definite set $u$ assigns the constant value $1$ to all elements of its domain. The canonical example of a definite set is $\check{x}$ (the standard embedding of a set $x$). These concepts bridge the gap between "real" (external) functions and Boolean-valued (internal) functions.
