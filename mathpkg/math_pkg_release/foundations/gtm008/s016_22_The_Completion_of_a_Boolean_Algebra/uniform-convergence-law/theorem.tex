$\tilde{B}$ satisfies the uniform convergence law (UCL) iff the following condition is satisfied:

Let $I$ be a set and for each $i \in I$, let $\{b_{\alpha i} \mid \alpha \in \mathrm{On}\} \subseteq \tilde{B}$. If

\begin{enumerate}
\item $(\forall \alpha)(\forall \beta)[\alpha < \beta \rightarrow b_{\alpha i} \geq b_{\beta i}]$,
\item $\prod_{\alpha \in \mathrm{On}} b_{\alpha i} = 0$,
\end{enumerate}

then

$$\prod_{\alpha \in \mathrm{On}} \sum_{i \in I} b_{\alpha i} = 0.$$

Roughly, this means that if a nonincreasing sequence $b_{\alpha i}$ ($\alpha \in \mathrm{On}$) of elements of $\tilde{B}$ converges to $0$ for each $i \in I$, then these sequences converge uniformly to $0$.$$
