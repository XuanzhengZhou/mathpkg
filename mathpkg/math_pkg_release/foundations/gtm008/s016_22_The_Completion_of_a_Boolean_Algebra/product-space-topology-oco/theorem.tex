Let $\kappa$ be a cardinal or $\mathrm{On}$. Let $X^{\alpha}$ ($\alpha < \kappa$) be topological spaces and define $X = \prod_{\alpha < \kappa} X^{\alpha}$. The topology on $X$ is defined by the open base

$$T = \{\tilde{G}_{\alpha} \mid \alpha < \kappa \land (G_{\alpha} \text{ is open in } X^{\alpha})\}$$

where $\tilde{G}_{\alpha} = \{f \in X \mid f(\alpha) \in G_{\alpha}\}$. Define $p_{\alpha}: X \to X^{\alpha}$ by $p_{\alpha}(f) = f(\alpha)$.

Then $\langle X, T \rangle$ is a topological space, $p_{\alpha}$ is an o.c.o. map and

$$p_{\alpha\beta} \circ p_{\beta} = p_{\alpha}.$$
