Let $B$ be a Boolean algebra (possibly a proper class) and let $C_{\alpha} = B \cap R(\alpha)$ for $\alpha \in \mathrm{On}$. Then:

\begin{enumerate}
\item $V_{0}^{(B)} \triangleq 0$.
\item $V_{\alpha}^{(B)} \triangleq \{u \in {}^{C_{\alpha}} \mathcal{D}(u) \mid (\exists \xi < \alpha)[\mathcal{D}(u) \subseteq V_{\xi}^{(B)}]\}$.
\item $V^{(B)} \triangleq \bigcup_{\alpha \in \mathrm{On}} V_{\alpha}^{(B)}$.
\end{enumerate}

We also require that $B$ satisfies the weaker condition:

$$(\forall A \subseteq B)[\mathcal{M}(A) \rightarrow (\exists x \in B)[x = \sup A]],$$

i.e., every set-sized subset of $B$ has a supremum in $B$. Moreover,

$$V^{(B)} = \{u \mid \mathcal{D}(u) \subseteq V^{(B)} \land u: \mathcal{D}(u) \rightarrow B\}$$

as in the case of a set $B$. Each $V_{\alpha}^{(B)}$ is a set and $V^{(B)}$ is a class of sets.$$
