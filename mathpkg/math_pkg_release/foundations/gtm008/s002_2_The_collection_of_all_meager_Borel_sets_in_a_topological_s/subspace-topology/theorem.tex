If $\langle X, T \rangle$ is a topological space, if $X' \subseteq X$ and $T' = \{X' \cap N \mid N \in T\}$ then $\langle X', T' \rangle$ is a topological space. Furthermore

1. $\langle X', T' \rangle$ is a compact space if $X'$ is a compact set,
2. $\langle X', T' \rangle$ is a Hausdorff space if $\langle X, T \rangle$ is Hausdorff.
