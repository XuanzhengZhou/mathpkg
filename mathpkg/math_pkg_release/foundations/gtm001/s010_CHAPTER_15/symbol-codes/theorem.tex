Using the pairing machine, symbols are coded by ordinals: $\ulcorner \in \urcorner = J(\langle 0, 0 \rangle)$, $\ulcorner = \urcorner = J(\langle 0, 1 \rangle)$, $\ulcorner \neg \urcorner = J(\langle 0, 2 \rangle)$, $\ulcorner \lor \urcorner = J(\langle 0, 3 \rangle)$, $\ulcorner ( \urcorner = J(\langle 0, 4 \rangle)$, $\ulcorner ) \urcorner = J(\langle 0, 5 \rangle)$, $\ulcorner x_i \urcorner = J(\langle 0, 6 + i \rangle)$, $\ulcorner \exists^\alpha \urcorner = J(\langle 0, \omega + \alpha \rangle)$, $\ulcorner \wedge^\alpha \urcorner = J(\langle 0, \omega + \alpha, \omega + \alpha \rangle)$. For sequences: $\ulcorner s_1, \ldots, s_n \urcorner = J(\langle 1, \ulcorner s_1 \urcorner, \ldots, \ulcorner s_n \urcorner \rangle)$.
