**Proof.** Since $\mathcal{D}(F \upharpoonright \alpha) = \alpha$: $F' \alpha = G'(F \upharpoonright \alpha)$. If $K_3' \alpha = 0$, then $F' \alpha = \mathcal{W}(F \upharpoonright \alpha) = F'' \alpha$. If $K_3' \alpha = n \neq 0$, then $F' \alpha = \mathcal{F}_n((F \upharpoonright \alpha)'K_1' \alpha, (F \upharpoonright \alpha)'K_2' \alpha)$. By Proposition 15.10, $K_1' \alpha < \alpha$ and $K_2' \alpha < \alpha$, so $(F \upharpoonright \alpha)'K_i' \alpha = F'K_i' \alpha$.
