**Proof.** $\mathcal{F}_1(c,d) = \{c,d\}$, $\mathcal{F}_2(c,d) = c \cap E$, $\mathcal{F}_3(c,d) = c - d$, $\mathcal{F}_4(c,d) = c \upharpoonright d$, etc. Each uses only absolute notions (pairing, intersection, difference, restriction).
