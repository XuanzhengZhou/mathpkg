**Proof.** (1) $\mathrm{Od}''b$ is a set of ordinals, so $(\exists \alpha)[\mathrm{Od}''b \subseteq \alpha]$. Let $\beta = J' \langle 0, \alpha, 0 \rangle$. Then $K_3'\beta = 0$, so $F'\beta = F''\beta$, and $\alpha \leq \beta$. If $x = F'\beta$ and $y \in b$, then $y \in L \land \mathrm{Od}'y < \alpha \leq \beta$, so $(\exists \gamma < \beta)[y = F'\gamma]$, hence $y = F'\gamma \in F''\beta = x$. Thus $b \subseteq x \land x \in L$.
