If $a \subseteq \alpha$, then by Corollary 7.54, $(\exists \beta)(\exists f)[f \text{ Isom}_{E, E}(\beta, a)]$. This function $f$ is then a strictly monotone ordinal function and so it follows that $(\forall \gamma \in \beta)[\gamma \leq f' \gamma \leq \alpha]$. Therefore $\beta \leq \alpha$ and $\beta \simeq a$.
