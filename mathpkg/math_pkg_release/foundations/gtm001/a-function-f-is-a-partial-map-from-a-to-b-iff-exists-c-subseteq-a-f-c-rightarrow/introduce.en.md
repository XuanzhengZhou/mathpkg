The definition **A function $f$ is a partial map from $A$ to $B$ iff $(\exists C \subseteq A) [f : C \rightarrow B]$** (Definition 16.2.) is a fundamental notion in axiomatic set theory, introduced in Chapter 15 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
