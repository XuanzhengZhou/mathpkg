By Proposition 5.15, $a - a = 0$ for any set $a$. By Proposition 5.13, $\mathcal{M}(a - A)$ for any set $a$ and any class $A$. Taking $A = a$, we obtain $\mathcal{M}(a - a) = \mathcal{M}(0)$. Therefore the empty set is indeed a set.
