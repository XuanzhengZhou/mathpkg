Zermelo's Schema of Separation asserts that for any set $a$ and any class $A$, the intersection $a \cap A$ is a set. In other words, the subclass of a set defined by any property is itself a set. Unlike the unrestricted Axiom of Abstraction (which leads to Russell's paradox), Separation restricts comprehension to elements already within a given set. Proposition 5.11 derives Separation from Fraenkel's Axiom Schema of Replacement.
