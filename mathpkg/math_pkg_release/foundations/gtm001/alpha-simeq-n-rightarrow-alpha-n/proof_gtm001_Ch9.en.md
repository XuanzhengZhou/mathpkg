If $\alpha \geq \omega$ then $n < \alpha$ and hence $n + 1 \leq \alpha$. If $\alpha \simeq n$ then, since $n < n + 1$, it follows from the Cantor–Schröder–Bernstein theorem that $\alpha \simeq n + 1$. But then $n \simeq n + 1$. This contradicts Corollary 10.21(1) and compels us to conclude that if $\alpha \simeq n$ then $\alpha < \omega$. From Proposition 10.20 it then follows that $\alpha = n$.
