(1) $C$ is closed in $\kappa \overset{\Delta}{\rightarrow} C \subseteq \kappa \wedge (\forall X \subseteq C)[\cup X < \kappa \rightarrow \cup X \in C]$.

(2) $C$ is unbounded in $\kappa \leftrightarrow C \subseteq \kappa \wedge \cup C = \kappa$.

(3) $S$ is stationary in $\kappa$ iff $S \subseteq \kappa$ and $S \cap C \neq 0$ for all closed unbounded subsets $C$ in $\kappa$.
