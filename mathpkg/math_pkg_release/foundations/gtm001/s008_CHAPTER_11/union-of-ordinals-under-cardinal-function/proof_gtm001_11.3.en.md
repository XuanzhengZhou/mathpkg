Since $\mathcal{D}(f)$ is a set, $fa$ is a set of ordinals. Let $eta = \cup(fa)$. One proves that $\overline{eta} = eta$, establishing that $eta$ is a cardinal. The proof uses properties of cardinal bounds and the fact that each $fx$ is in $N$.
