Under the Generalized Continuum Hypothesis, a cardinal is weakly inaccessible if and only if it is inaccessible. The GCH collapses the distinction: for any set $x$ with $|x| < \aleph_\alpha$, we have $|\mathcal{P}(x)| = |x|^+ < \aleph_\alpha$, so the strong limit property follows from the weak limit property.
