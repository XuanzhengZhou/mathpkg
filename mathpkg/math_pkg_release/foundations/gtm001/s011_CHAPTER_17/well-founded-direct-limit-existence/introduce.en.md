This lemma provides sufficient conditions for constructing a well-founded $\kappa$-direct limit system with a prescribed limit ordinal $\mu$. The hypothesis asserts that for all relevant $\eta, \alpha$ and finite $Q$, the order type of the machine closure is bounded below $\kappa$. The construction indexes the system by all triples $\langle \eta, \alpha, Q \rangle$ and uses collapsing maps to define the connecting morphisms.
