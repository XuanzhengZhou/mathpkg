(1) $\pi_{ij} : \eta_i \rightarrow \eta_j$ is an order preserving map for $i \leq j$,

(2) $\pi_{ii}$ is the identity map, and

(3) $\pi_{jk} \circ \pi_{ij} = \pi_{ik}$ if $i \leq j \leq k$.

If each $\pi_{ij} : \eta_i \rightarrow \eta_j$ is a strong $\mathfrak{A}$-map (medium $\mathfrak{A}$-map), then $\Pi$ is a direct system of strong $\mathfrak{A}$-maps (medium $\mathfrak{A}$-maps).
