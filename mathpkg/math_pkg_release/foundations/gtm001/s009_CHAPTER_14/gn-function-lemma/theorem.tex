If $M$ is transitive, almost universal, and closed under the eight fundamental operations, if $A$ is $M$-constructible, if $a \in M$, and if $G_n$ is a function on $a$ defined by
$$G_n'b = \{y \in A \mid \langle y, b \rangle \in Q_n \land (\forall x \in A)[\langle x, b \rangle \in Q_n \rightarrow \mathrm{rank}(y) \leq \mathrm{rank}(x)]\},$$
then:
\begin{itemize}
\item[(1)] $(\exists y \in M)[\cup(G_n''a) \subseteq y]$,
\item[(2)] $(\exists z \in M)[a \cap Q_n^*z = a \cap Q_n^*A]$.
\end{itemize}
