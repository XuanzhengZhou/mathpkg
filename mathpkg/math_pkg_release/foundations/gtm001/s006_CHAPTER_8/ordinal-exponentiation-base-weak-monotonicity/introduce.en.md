If $\alpha < \beta$, then $\alpha^\gamma \le \beta^\gamma$ for any exponent $\gamma$. Exponentiation is weakly monotone in the base. The proof uses transfinite induction on $\gamma$, with the successor case using the left strict monotonicity of ordinal multiplication.
