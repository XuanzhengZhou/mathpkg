If for every $\alpha$ in a set $A$ of ordinals there exists a $\beta$ in a set $B$ such that $\alpha \le \beta$, then the supremum of $A$ is at most the supremum of $B$. This is a useful property for comparing suprema of ordinal sets. Note that the converse is false: $\sup(\omega) = \sup(\omega + 1)$ provides a counterexample.
