The axiom **(Regularity). $(\forall x)[x \neq 0 \rightarrow (\exists y)[y \in x \wedge y \cap x = 0]]$** (Axiom 6) is a fundamental notion in axiomatic set theory, introduced in Chapter 11 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
