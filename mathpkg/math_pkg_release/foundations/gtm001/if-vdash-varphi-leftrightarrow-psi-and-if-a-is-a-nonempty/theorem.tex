Furthermore there exists an $x$ distinct from $a_1, \ldots, a_n$. Then since $\varphi(x)$ is absolute with respect to $A$.

$$a_1, \ldots, a_n \in A \wedge x \in A \rightarrow [\varphi(x) \leftrightarrow \varphi^A(x)].$$

Therefore

$$a_1, \ldots, a_n \in A \rightarrow [(\exists x \in A)\varphi(x) \leftrightarrow (\exists x \in A)\varphi^A(x)]$$

$$\rightarrow [(\exists x \in A)\varphi^A(x) \rightarrow (\exists x)\varphi(x)].$$
