The definition **$\{a, b\} \triangleq \{x \mid x = a \lor x = b\}$** (Definition 5.1.) is a fundamental notion in axiomatic set theory, introduced in Chapter 5 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
