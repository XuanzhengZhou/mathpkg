** If $\gamma \in K_{\text{I}} \wedge \gamma \neq 0$, then $(\exists \delta)[\gamma = \delta +

If $\alpha^{\delta} \leq \beta < \alpha^{\delta+1}$ and $\alpha^{\gamma} \leq \beta < \alpha^{\gamma+1}$ and if $\delta < \gamma$, then $\delta + 1 \leq \gamma$. Hence
$$\beta < \alpha^{\delta+1} \leq \alpha^{\gamma} \leq \beta.$$
Similarly if $\gamma < \delta$, then $\beta < \beta$. Therefore $\delta = \gamma$.
