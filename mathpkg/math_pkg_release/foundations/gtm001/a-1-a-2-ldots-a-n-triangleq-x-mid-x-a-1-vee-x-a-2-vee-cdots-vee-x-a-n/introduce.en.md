The definition **$\{a_1, a_2, \ldots, a_n\} \triangleq \{x \mid x = a_1 \vee x = a_2 \vee \cdots \vee x = a_n\}$** (Definition 5.3.) is a fundamental notion in axiomatic set theory, introduced in Chapter 5 of Takeuti and Zaring's *Introduction to Axiomatic Set Theory* (2nd ed., 1982). It forms part of the formal development of Zermelo-Fraenkel set theory with classes (GB or ZF).
